**Google云端平台**（，简称为**GCP**）是一系列由[Google](../Page/Google.md "wikilink")提供的[云计算服务](../Page/雲端運算.md "wikilink")，在运行[Google搜索](../Page/Google搜索.md "wikilink")和[YouTube](../Page/YouTube.md "wikilink")的服务器上提供[基础设施服务](https://zh.wikipedia.org/wiki/基礎設施即服務 "wikilink")、[平台服务及](../Page/平台即服务.md "wikilink")[无服务器计算环境](https://zh.wikipedia.org/wiki/無伺服器計算 "wikilink")。\[1\]除了提供管理工具外，Google云端平台还提供了一系列模块化云服务，包括：云计算、[数据存储](../Page/電腦數據存貯器.md "wikilink")、[数据分析及](https://zh.wikipedia.org/wiki/数据分析 "wikilink")[机器学习](../Page/机器学习.md "wikilink")等。\[2\]用户注册账号时需要提供[信用卡](../Page/信用卡.md "wikilink")或银行卡支付信息。\[3\]

2008年4月，Google发布了[应用程序引擎](../Page/Google應用服務引擎.md "wikilink")，同时也是其旗下首款云端计算服务，为开发者提供位于Google[数据中心](../Page/数据中心.md "wikilink")的网页应用开发及托管服务。2011年11月，应用程序引擎于正式向大众开放。自其发布以来，Google为云端平台添加了多款服务。

Google云端平台与Google云端平台基础设施、**G Suite**（企业版[Android](../Page/Android.md "wikilink")及[Chrome OS](../Page/Chrome_OS.md "wikilink")）和[机器学习](../Page/机器学习.md "wikilink")及企业映射[应用程序接口](../Page/应用程序接口.md "wikilink")一样，均是\[4\]**Google Cloud**的一部分。

## 产品

[GoogleCloudKubernetes.jpg](https://zh.wikipedia.org/wiki/File:GoogleCloudKubernetes.jpg "fig:GoogleCloudKubernetes.jpg")发布会\]\]Google旗下的云端计算品牌共有超过[90种产品](https://cloud.google.com/products/)。下列是部分关键服务：

### 计算

  - [App Engine](../Page/Google應用服務引擎.md "wikilink") - 部署[Java](../Page/Java.md "wikilink")、[PHP](../Page/PHP.md "wikilink")、[Node.js](../Page/Node.js.md "wikilink")、[Python](../Page/Python.md "wikilink")、[C\#](../Page/C♯.md "wikilink")、[.Net](../Page/.NET框架.md "wikilink")、[Ruby](../Page/Ruby.md "wikilink")及[Go](../Page/Go.md "wikilink")应用程序的[平台服务](../Page/平台即服务.md "wikilink")。
  - [Compute Engine](../Page/Google计算引擎.md "wikilink") - 运行[Windows及](https://zh.wikipedia.org/wiki/Microsoft_Windows "wikilink")[Linux](../Page/Linux.md "wikilink")虚拟机的[基础设施服务](https://zh.wikipedia.org/wiki/基礎設施即服務 "wikilink")。
  - Kubernetes Engine (GKE) - 基于[Kubernetes](../Page/Kubernetes.md "wikilink")的容器即服务。
  - Cloud Functions - 运行以Node.js或Python语言编写的事件驱动代码的。

### 存储及数据库

  - \- 用于存储且集成边缘缓存的。

  - Cloud SQL - 基于[MySQL](../Page/MySQL.md "wikilink")及[PostgreSQL](../Page/PostgreSQL.md "wikilink")的。

  - [Cloud BigTable](../Page/Bigtable.md "wikilink") - 托管式[NoSQL](../Page/NoSQL.md "wikilink")宽列数据库服务。

  - [Cloud Spanner](../Page/Spanner_\(数据库\).md "wikilink") - 水平可扩展的[关系型数据库服务](../Page/关系数据库.md "wikilink")。

  - \- 为网页及掌上应用程序提供的NoSQL数据库。

  - 永久性磁盘 -  为计算引擎虚拟机打造的[区块存储](../Page/块_\(数据存储\).md "wikilink")。

  - Cloud MemoryStore - 基于[Redis](../Page/Redis.md "wikilink")的托管式内存数据存储服务。

### 网络

  - VPC - 管理云端资源[软件定义网络的](../Page/軟體定義網路.md "wikilink")[虚拟私有云](../Page/虛擬私有雲.md "wikilink")。
  - Cloud Load Balancing - 软件定义的托管式网络流量服务。
  - Cloud Armor - 防止[DDoS攻击的](../Page/阻斷服務攻擊.md "wikilink")。
  - Cloud CDN - 基于Google全球分布的节点打造的[内容分发网络](../Page/內容傳遞網路.md "wikilink")。
  - Cloud Interconnect - 将数据中心连接至Google云端平台的服务。
  - Cloud DNS - Google基础设施上的托管式权威[DNS](../Page/域名系统.md "wikilink")。
  - 网络服务层级 - 选择优质或标准层级的网络以优化服务效果。

### 数据分析

  - [BigQuery](../Page/BigQuery.md "wikilink") - 可扩展的托管式企业级分析[数据仓储](../Page/資料倉儲.md "wikilink")。

  - \- 基于[Apache Beam的托管式流数据及批数据处理](../Page/Apache_Beam.md "wikilink")。

  - \- 运行[Apache Hadoop和](../Page/Apache_Hadoop.md "wikilink")[Apache Spark任务的](../Page/Apache_Spark.md "wikilink")[大数据平台](../Page/大數據.md "wikilink")。

  - Cloud Composer - 构建于Apache Airflow上的托管式工作流编排服务。

  - Cloud Datalab - 、[分析](https://zh.wikipedia.org/wiki/分析 "wikilink")、可视化及机器学习工具。这是一款完全托管式的Jupyter Notebook服务。

  - Cloud Dataprep - 基于的数据服务，用于为分析进行可视化探索、清除并准备数据过程。

  - Cloud Pub/Sub - 基于[消息队列](../Page/消息队列.md "wikilink")的可扩展数据接入服务。

  - Cloud Data Studio - 透过仪表盘及汇报来可视化数据的[商业智能](../Page/商业智能.md "wikilink")工具。

### AI 和机器学习

  - Cloud AutoML - 训练并部署自定义机器学习模型的服务。
  - Cloud TPU - Google训练机器学习模型时使用的加速器。
  - Cloud Machine Learning Engine - 基于主流框架训练并构建机器学习模型的托管式服务。
  - Cloud Job Discovery - 基于Google搜索与机器学习的招聘服务。
  - Dialogflow Enterprise -  基于Google的机器学习构建的开发环境。
  - Cloud Natural Language - 基于Google[深度学习](../Page/深度学习.md "wikilink")模型的[文本分析服务](../Page/內容分析法.md "wikilink")。
  - Cloud Speech-to-Text - 基于机器学习的[语音转文字服务](../Page/语音识别.md "wikilink")。
  - Cloud Text-to-Speech - 基于机器学习的[文字转语音服务](../Page/语音合成.md "wikilink")。
  - Cloud Translation API - 实时翻译至不同语言的服务。
  - Cloud Vision API - 基于机器学习的[图像分析服务](https://zh.wikipedia.org/wiki/图像分析 "wikilink")。
  - Cloud Video Intelligence - 基于机器学习的服务。

### 管理工具

  - \- Monitoring, logging, and diagnostics for applications on Google Cloud Platform and AWS.

  - Cloud Deployment Manager  - Tool to deploy Google Cloud Platform resources defined in templates created in [YAML](../Page/YAML.md "wikilink"), [Python](../Page/Python.md "wikilink") or [Jinja2](https://zh.wikipedia.org/wiki/Jinja2 "wikilink").

  - Cloud Console - Web interface to manage Google Cloud Platform resources.

  - Cloud Shell - Browser-based shell command line access to manage Google Cloud Platform resources.

  - Cloud Console Mobile App - [Android](../Page/Android.md "wikilink") and [iOS](../Page/IOS.md "wikilink") application to manage Google Cloud Platform resources.

  - Cloud APIs - [APIs](https://zh.wikipedia.org/wiki/API_\(消歧義\) "wikilink") to programmatically access Google Cloud Platform resources

### 安全性

  - [Cloud Identity](https://cloud.google.com/identity/) - [Single sign-on](../Page/單一登入.md "wikilink") (SSO) service based on [SAML 2.0](../Page/SAML_2.0.md "wikilink") and [OpenID](../Page/OpenID.md "wikilink").
  - Cloud IAM -  (IAM) service for defining policies based on [role-based access control](../Page/以角色為基礎的存取控制.md "wikilink").
  - Cloud Identity-Aware Proxy - Service to control access to cloud applications running on Google Cloud Platform without using a [VPN](../Page/虛擬私人網路.md "wikilink").
  - Cloud Data Loss Prevention API - Service to automatically discover, classify, and redact sensitive data.
  - Security Key Enforcement - [Two-step verification](../Page/多重要素驗證.md "wikilink") service based on a security key.
  - Cloud Key Management Service  - Cloud-hosted key management service integrated with IAM and audit logging.
  - Cloud Resource Manager - Service to manage resources by project, folder, and organization based on the hierarchy.
  - Cloud Security Command Center - Security and data risk platform for data and services running in Google Cloud Platform.
  - Cloud Security Scanner - Automated [vulnerability scanning](https://zh.wikipedia.org/wiki/漏洞 "wikilink") service for applications deployed in App Engine.
  - Access Transparency - Near real-time audit logs providing visibility to Google Cloud Platform administrators.

### 物联网

  - Cloud IoT Core - Secure device connection and management service for [Internet of Things](../Page/物联网.md "wikilink").
  - Edge TPU - Purpose-built ASIC designed to run inference at the edge. As of September 2018, this product is in private beta.
  - Cloud IoT Edge - Brings AI to the [edge computing](../Page/邊緣運算.md "wikilink") layer.

### API 管理

  - Maps Platform - APIs for maps, routes, and places based on Google Maps.
  - Apigee API Platform - Lifecycle management platform to design, secure, deploy, monitor, and scale APIs.
  - API Monetization - Solution for API providers to create revenue models, reports, s, and developer portal integrations.
  - Developer Portal - Self-service platform for developers to publish and manage APIs.
  - API Analytics - Service to analyze API-driven programs through monitoring, measuring, and managing APIs.
  - Apigee Sense -  Enables API security by identifying and alerting administrators to suspicious API behaviors.
  - Cloud Endpoints - An [NGINX](../Page/Nginx.md "wikilink")-based proxy to deploy and manage APIs.

## 服务节点

Each region is an independent geographic area that consists of zones.

A zone is a deployment area for Google Cloud Platform resources within a region. Zones should be considered a single failure domain within a region.

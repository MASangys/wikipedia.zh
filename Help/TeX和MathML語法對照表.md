## 基本

| 項目    | Wiki中的TeX語法         | 對應的MathML語法                       | 預覽                     |
| ----- | ------------------- | --------------------------------- | ---------------------- |
| 未知數   | x                   | <mi>x</mi>                        | \(x\)                  |
| 整數    | 3                   | <mn>3</mn>                        | \(3\)                  |
| 小數    | 3.2                 | <mn>3.2</mn>                      | \(3.2\)                |
| 純文字   | where               | <mtext>where</mtext>              | \(\mbox{where}\)       |
| 自訂運算符 | \\operatorname{op}a | <mo>op</mo><mi>a</mi>             | \(\operatorname{op}a\) |
| 群組化   | {xy}                | <mrow><mi>x</mi><mi>y</mi></mrow> | \({xy}\)               |

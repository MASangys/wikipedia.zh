<td><h2 id="尋找開放授權文字以放入維基百科">尋找開放授權文字以放入維基百科</h2>
<p><a href="https://zh.wikipedia.org/wiki/Wikipedia:维基百科图书馆/资料" title="wikilink">維基百科圖書館內有許多各類廣泛主題的開放近用資源</a>。</p>
<p><strong>哪些是良好的外部來源作品？</strong></p>
<p>最佳的開放來源文本包含以下幾點，足以納入維基百科：</p>
<ul>
<li>文獻回顧或調查；</li>
<li>百科全書或三手來源材料；</li>
<li>關於某方面主題或領域中已知相關知識的研究、調查或總結報告。</li>
</ul>
<p>Try to avoid works which:</p>
<ul>
<li>Are secondary sources, or sections within sources which focus on the author's analysis or conclusions.</li>
<li>Focuses on novel or unusual opinions about a topic; remember Wikipedia is about focusing on established, shared knowledge about topics.</li>
<li>Principally focus on persuasion or argumentation; thought sections of the work may appear neutral in tone, Wikimedia tries to include balance as well as neutrality and sources focused on persuasion frequently omit contrary perspectives.</li>
</ul></td>

<td><h2 id="measuring_reach">Measuring reach</h2>
<p>Tools are available to measure the number of page views Wikipedia articles receive that use text from a range of external sources e.g a website, a section of a website, a specific URL or a publication:</p>
<h4 id="a_small_number_of_articles">A small number of articles</h4>
<p>To measure the page views for a small number of Wikipedia pages where it is known text from a source has been used simply use the <a href="https://tools.wmflabs.org/pageviews">Pageviews tool</a> and enter the names of the articles into the <strong>Pages</strong> field (you can click on the <strong>x</strong> next to the name of an article to remove it from the graph).</p>
<h4 id="a_larger_number_of_articles">A larger number of articles</h4>
<p>To measure page views of all articles using text from a source use the Massviews Analysis Tool:</p>
<ol>
<li><a href="https://tools.wmflabs.org/massviews/">Open the <strong>Massviews Analysis Tool</strong></a>.</li>
<li>Chose the <strong>Dates</strong> you would like to measure.</li>
<li>Change the <strong>Source</strong> to <strong>Search</strong>.</li>
<li>Set the site to <strong>en.wikipedia.org</strong>.</li>
<li>Depending on whether you want results from one source or multiple sources:
<ol>
<li><u><strong>One source:</strong></u> In the search field add <strong>hastemplate:"Free-content attribution" insource:"NAMEOFSOURCE"</strong> (including the quote marks), replacing NAMEOFSOURCE with the source you are searching for. E.g if you would like to measure the page views of articles that reuse text from UNESCO enter <strong>hastemplate:"Free-content attribution" insource:"unesco.org"</strong> which will return <a href="https://tools.wmflabs.org/massviews?platform=all-access&amp;agent=user&amp;source=search&amp;target=hastemplate%3A%22Free-content_attribution%22_insource%3A%22unesco.org%22&amp;start=2017-04-01&amp;end=2017-04-30&amp;project=en.wikipedia.org&amp;sort=views&amp;direction=1&amp;view=list">this result</a>.</li>
<li><u><strong>Multiple sources:</strong></u> In the search field add <strong>hastemplate:"Free-content_attribution"_insource:/SOURCE1|SOURCE2|SOURCE3/</strong> (including the quote marks), replacing SOURCE1, SOURCE2 and SOURCE3 with the sources you would like to search for. You may search for as many sources as you want, simply continue to separate each query with a | symbol. The only limitations are that the SOURCE cannot include a / symbol (searching for web addresses can be done by finding a unique part of the URL and searching for that) and the queries are limited to 300 characters. For example if you would like to measure the page views of articles that reuse text from a series of UNESCO PDFs (using only name of the PDF rather than the whole URL) enter <strong>hastemplate:"Free-content_attribution" insource:/244756e.pdf|232555e.pdf/</strong> which will return <a href="https://tools.wmflabs.org/massviews?platform=all-access&amp;agent=user&amp;source=search&amp;target=hastemplate%3A%22Free-content_attribution%22_insource%3A%2F244756e.pdf%7C232555e.pdf%2F&amp;start=2017-04-01&amp;end=2017-04-30&amp;project=en.wikipedia.org&amp;sort=views&amp;direction=1&amp;view=list">this result</a>.</li>
</ol></li>
<li>Click <strong>Submit</strong>, to receive your results, you should receive a page that looks like <a href="https://tools.wmflabs.org/massviews?platform=all-access&amp;agent=user&amp;source=search&amp;target=hastemplate%3A%22Free-content_attribution%22_insource%3A%22unesco.org%22&amp;start=2017-02-28&amp;end=2017-03-29&amp;project=en.wikipedia.org&amp;sort=views&amp;direction=1&amp;view=list">this example showing text used on English languages Wikipedia from UNESCO</a>.</li>
<li>The data can be viewed as either a <strong>list</strong> or a <strong>chart</strong> and can be downloaded as either a <strong>csv</strong> or <strong>json</strong> file, you can also create a <strong>permalink</strong> (permanent link) to easily access the results again. If you want to set up a URL to a query that will always return results for the past 30 days, simply select <strong>last month</strong> in the date selection options.</li>
</ol></td>

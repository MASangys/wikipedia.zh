[1955年](../Page/1955年.md "wikilink")[2月7日](../Page/2月7日.md "wikilink")，[柬埔寨](../Page/柬埔寨.md "wikilink")举行了关于[诺罗敦·西哈努克](../Page/诺罗敦·西哈努克.md "wikilink")国王在[日内瓦会议上的倡议的](https://zh.wikipedia.org/wiki/日内瓦会议_\(1954年\) "wikilink")[公民投票](../Page/公民投票.md "wikilink")。99.8%的选民赞成了该倡议。\[1\]

## 结果

| 选择                 | 得票          | %       |
| ------------------ | ----------- | ------- |
| 支持                 | 925,667     | 99.8    |
| 反对                 | 1,834       | 0.2     |
| 废票/白票              |             | –       |
| **总计**             | **927,501** | **100** |

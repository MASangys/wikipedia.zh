**1980年大韓民國總統選舉**，於該年8月27日於韓國舉行。總統仍由[統一主體國民會議選出](https://zh.wikipedia.org/wiki/統一主體國民會議 "wikilink")，最終由唯一的候選人全斗煥勝出。\[1\]

## 結果

| Candidate                        | Party     | Votes   | %     |
| -------------------------------- | --------- | ------- | ----- |
| [全斗煥](../Page/全斗煥.md "wikilink") | 無黨籍       | 2,524   | 99.96 |
| 無效票                              | 1         | 0.04    |       |
| **總票數**                          | **2,525** | **100** |       |
| 選舉人數/投票率                         | 2,540     | 99.41   |       |

**1991年馬其頓獨立公投**舉辦於1991年9月8日和3月1日\[1\].這次公投的投票率是75.7%，96.4%的投票人支持馬其頓自南斯拉夫獨立\[2\]。

## 結果

| 選項                  | 票數            | %       |
| ------------------- | ------------- | ------- |
| 支持獨立                | 1,079,308     | 96.4    |
| 反對獨立                | 39,639        | 3.6     |
| 無效/白票               | 13,648        | –       |
| **總計**              | **1,132,595** | **100** |
| 投票人數/投票率            | 1,495,807     | 75.7    |
| 來源: Nohlen & Stöver |               |         |

1996年NBA總決賽是[1995-96 NBA賽季的最後一個系列賽](https://zh.wikipedia.org/wiki/1995-96_NBA賽季 "wikilink")，兩支球隊將爭奪本年的總冠軍。[西部聯盟冠軍](https://zh.wikipedia.org/wiki/西部聯盟_\(NBA\) "wikilink")[西雅圖超音速](https://zh.wikipedia.org/wiki/西雅圖超音速 "wikilink")（64-18）對陣[東部聯盟有](https://zh.wikipedia.org/wiki/東部聯盟_\(NBA\) "wikilink")[主場優勢的](https://zh.wikipedia.org/wiki/主場優勢 "wikilink")[芝加哥公牛](../Page/芝加哥公牛.md "wikilink")（72-10）。比賽采用七場四勝制2-3-2的形式進行，系列賽的第一、二、六（如果需要），七（如果需要）場比賽在有主場優勢的隊伍的主場進行。

公牛以總比數4比2擊敗超音速，總計公牛單季的常規賽和季后赛战绩为87胜13负，勝率為87%，創下NBA史上最佳成績。[-{zh-hans:迈克尔·乔丹;zh-hant:邁克爾·喬丹;zh-tw:麥可·喬丹;zh-hk:米高·佐敦;zh-mo:米高·佐敦;}-贏得本年度的](../Page/迈克尔·乔丹.md "wikilink")[NBA总决赛最有价值球员](../Page/NBA总决赛最有价值球员.md "wikilink")。

## 晉級

  - [芝加哥公牛](../Page/芝加哥公牛.md "wikilink")以4-0总比分击败[奧蘭多魔術赢得](https://zh.wikipedia.org/wiki/奧蘭多魔術 "wikilink")1996年的东部分区决赛。
  - [西雅圖超音速以](https://zh.wikipedia.org/wiki/西雅圖超音速 "wikilink")4-3总比分击败[猶他爵士赢得](https://zh.wikipedia.org/wiki/猶他爵士 "wikilink")1996年的西部分区决赛。

## 賽程表

所有時間為美國時間

  - 第一場－6月5日**公牛 107，** 超音速 90： 公牛以1-0領先
  - 第二場－6月7日**公牛 92，** 超音速 88： 公牛以2-0領先
  - 第三場－6月9日**公牛 108，** 超音速： 86： 公牛以3-0領先
  - 第四場－6月12日**超音速 107，** 公牛 86： 公牛以3-1領先
  - 第五場－6月14日**超音速 89，** 公牛 78： 公牛以3-2領先
  - 第六場－6月16日**公牛 87，** 超音速 75： 公牛以4-2勝出

比賽采用七場四勝制2-3-2的形式進行，系列賽的第一、二、六場比賽在芝加哥公牛主場（[聯合中心](https://zh.wikipedia.org/wiki/聯合中心 "wikilink")）進行。

## 第一場

6月5日，在[聯合中心舉行](https://zh.wikipedia.org/wiki/聯合中心 "wikilink")。

雖然公牛隊進攻能力在今場比賽未能發揮水準，但卻有超水準的防守。公牛隊在第三節偷以兩分領前。最後一節[托尼·库科奇](../Page/托尼·库科奇.md "wikilink")得及[朗·夏巴](https://zh.wikipedia.org/wiki/朗·夏巴 "wikilink")2個偷球使公牛隊勝出。

| :----- | -- | -- | -- | -- | ------- |
| 超音速    | 18 | 30 | 29 | 13 | 90      |
| **公牛** | 24 | 29 | 26 | 28 | **107** |
|        |    |    |    |    |         |

## 第二場

6月7日，在[聯合中心舉行](https://zh.wikipedia.org/wiki/聯合中心 "wikilink")。

第一節超音速以27-23領先。不過超音速再一次在領先之下落敗。不過第二節披頓率領超音速反超前。古高帶領公牛在第三節反超前，總比分是92-88。洛文取得11個進攻籃板。

| :----- | -- | -- | -- | -- | ------ |
| 超音速    | 27 | 18 | 20 | 23 | 88     |
| **公牛** | 23 | 23 | 30 | 16 | **92** |
|        |    |    |    |    |        |

## 第三場

6月9日，在[钥匙竞技场舉行](https://zh.wikipedia.org/wiki/钥匙竞技场 "wikilink")。

## 第四場

6月12日，在[钥匙竞技场舉行](https://zh.wikipedia.org/wiki/钥匙竞技场 "wikilink")。

超音速在籃板上取回優勢，並靠著「手套」Gary Payton看管喬丹，終場超音速107:86擊敗公牛，躲過被剔光頭的命運。

## 第六場

6月16日，在[聯合中心舉行](https://zh.wikipedia.org/wiki/聯合中心 "wikilink")。

公牛隊在[父親節](../Page/父親節.md "wikilink")以4-2擊敗對手，三年前的今天佐敦的父親被殺。勝利的關鍵是[大前鋒](https://zh.wikipedia.org/wiki/大前鋒 "wikilink")[丹尼斯·洛文追平在第二場紀錄](https://zh.wikipedia.org/wiki/丹尼斯·洛文 "wikilink")，共取得11個進攻藍板。

**2-乙基-1-己醇**，分子式C<sub>8</sub>H<sub>18</sub>O。

## 性质

无色有特殊气味液体。可燃。粘度：9.8 mPa·s（20°C）。
溶于约720倍的水，与[醇](../Page/醇.md "wikilink")、[醚和](../Page/醚.md "wikilink")[氯仿等多数](../Page/氯仿.md "wikilink")[有机溶剂互溶](https://zh.wikipedia.org/wiki/有机溶剂 "wikilink")。与水形成共沸混合物，其中水占20％，共沸点99.1°C。

## 制备

1、[丙烯与](../Page/丙烯.md "wikilink")[合成气经](../Page/合成气.md "wikilink")[羰基合成得到](https://zh.wikipedia.org/wiki/羰基合成 "wikilink")[丁醛](../Page/丁醛.md "wikilink")，后者两分子[羟醛缩合得到](../Page/羟醛缩合.md "wikilink")[2-乙基-2-己烯醛](https://zh.wikipedia.org/wiki/2-乙基-2-己烯醛 "wikilink")，最后[加氢](https://zh.wikipedia.org/wiki/加氢 "wikilink")，得到2-乙基-1-己醇。

\[\rm{\ CH_3CH\!=\!CH_2+CO+H_2 \ \xrightarrow [150\!-\!200^\circ C, \ 20MPa]{cat.} \ }\]\(\rm{\ CH_3CH_2CH_2CHO}\)

\[\rm{\ 2CH_3CH_2CH_2CHO \ \xrightarrow{-H_2O} \ }\]\(\rm{\ CH_3CH_2CH_2CH\!=\!C(C_2H_5)CHO}\)

\[\rm{\ CH_3CH_2CH_2CH\!=\!C(C_2H_5)CHO \ \xrightarrow {+2H_2} \ }\]\(\rm{\ CH_3CH_2CH_2CH_2CH(C_2H_5)CH_2OH}\)
2、两分子[乙醛](../Page/乙醛.md "wikilink")[羟醛加成为](https://zh.wikipedia.org/wiki/羟醛加成 "wikilink")[丁醇醛](https://zh.wikipedia.org/wiki/丁醇醛 "wikilink")，失水得[巴豆醛](../Page/巴豆醛.md "wikilink")，[加氢等](https://zh.wikipedia.org/wiki/加氢 "wikilink")[正丁醛](https://zh.wikipedia.org/wiki/正丁醛 "wikilink")，然后两分子正丁醛缩合为[2-乙基-2-己烯醛](https://zh.wikipedia.org/wiki/2-乙基-2-己烯醛 "wikilink")，最后加氢得2-乙基-1-己醇。

\[\rm{\ 2CH_3CHO \ \xrightarrow {NaOH} \ CH_3CH(OH)CH_2CHO \ \xrightarrow {-H_2O} \ }\]\(\rm{\ CH_3CH\!=\!CHCHO \ \xrightarrow [Ni]{H_2} \ CH_3CH_2CH_2CHO}\)

  -
    正丁醛缩合失水以及产物加氢的方程式与（1）中相同。

## 用途

用于制取[鄰苯二甲酸二(2-乙基己基)酯](../Page/鄰苯二甲酸二\(2-乙基己基\)酯.md "wikilink")、[壬二酸二异辛酯和](https://zh.wikipedia.org/wiki/壬二酸二异辛酯 "wikilink")[癸二酸二异辛酯等](https://zh.wikipedia.org/wiki/癸二酸二异辛酯 "wikilink")[增塑剂](https://zh.wikipedia.org/wiki/增塑剂 "wikilink")。也用于合成抗氧化剂、润滑剂、消泡剂、溶剂，用于照相、纸张上浆、胶乳和织物印染等方面。

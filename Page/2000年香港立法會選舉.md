**2000年香港立法會選舉**是[香港特別行政區成立後的第二屆](https://zh.wikipedia.org/wiki/香港特別行政區 "wikilink")[香港立法會](../Page/香港立法會.md "wikilink")選舉，於2000年9月10日順利舉行。

## 背景

由於《[香港基本法](https://zh.wikipedia.org/wiki/香港基本法 "wikilink")》明定[第一屆立法會的任期是兩年](../Page/1998年香港立法會選舉.md "wikilink")，故在1998年產生的立法會於2000年改選。

1999年的[金融風暴](../Page/亞洲金融風暴.md "wikilink")，使[香港經濟](../Page/香港經濟.md "wikilink")走下坡，市民對香港特區政府的不滿愈來愈高。之後在2000年的[鍾廷耀事件發生後](https://zh.wikipedia.org/wiki/港大民調風波 "wikilink")，民主派用了「任用私人、黑箱作業」的口號，去攻擊[董建華](../Page/董建華.md "wikilink")政府施政無能。

在[政治](../Page/政治.md "wikilink")上，可算兩大陣營的席次爭奪戰。[民主黨參照](../Page/民主黨_\(香港\).md "wikilink")[李卓人](../Page/李卓人.md "wikilink")、[梁耀忠](../Page/梁耀忠.md "wikilink")的成功模式，在[新界西分開三張名單出選](https://zh.wikipedia.org/wiki/新界西 "wikilink")，分別由[李永達](../Page/李永達.md "wikilink")、[何俊仁](../Page/何俊仁.md "wikilink")、[陳偉業](../Page/陳偉業.md "wikilink")三人領軍，在[新界東分開兩張名單](https://zh.wikipedia.org/wiki/新界東 "wikilink")，分別由[鄭家富](../Page/鄭家富.md "wikilink")、[黃成智](../Page/黃成智.md "wikilink")領軍。而[民建聯方面](https://zh.wikipedia.org/wiki/民建聯 "wikilink")，在新界西[譚耀宗](../Page/譚耀宗.md "wikilink")為了修補與鄉事派勢力的關係，故與[港進聯的](https://zh.wikipedia.org/wiki/港進聯 "wikilink")[鄧兆棠](../Page/鄧兆棠.md "wikilink")合作參選，而在[香港島](../Page/香港島.md "wikilink")，傳統福建社團推出港進聯的[蔡素玉](../Page/蔡素玉.md "wikilink")加入[民建聯](https://zh.wikipedia.org/wiki/民建聯 "wikilink")，以[程介南](../Page/程介南.md "wikilink")為首合組名單出選。[前綫的](../Page/前綫_\(1996年\).md "wikilink")[何秀蘭](../Page/何秀蘭.md "wikilink")由上次與[劉慧卿](../Page/劉慧卿.md "wikilink")聯手出戰新界東雙雙當選，但轉到香港島參選。而[九龍東的建制派名單也是由民建聯的](https://zh.wikipedia.org/wiki/九龍東 "wikilink")[陳婉嫻](../Page/陳婉嫻.md "wikilink")領軍，加上民建聯觀塘支部主席[陳鑑林](../Page/陳鑑林.md "wikilink")。

在新界東最令人注目的是「長毛」[梁國雄](../Page/梁國雄.md "wikilink")的參選，為立法會選舉帶來新的動力。

## 廢票

### 定義

廢票是指無效的選票\[1\]，如白票、填寫不當、塗污了的選票等，算法是把總投票數字減去有效票數的數字；而廢票率則是廢票數量佔總投票數字的百分比，公式如下：

  - 對每個選區而言：

|                        |
| ---------------------- |
| 有效票數 = ∑(各候選人票數)       |
| 廢票量 = 總票數 - 有效票數       |
| 廢票率 = 廢票量 ÷ 總票數 × 100% |

## 結果、分析與影響

總括而言，這次選舉中，民主派及建制派陣營則各有斬獲。

在新界西方面，新增一席，民主黨分開的三張名單，由李、何、陳三人各有責任區，但由於陳偉業較弱，而民調顯示[李永達](../Page/李永達.md "wikilink")及[何俊仁](../Page/何俊仁.md "wikilink")會當選，故民主黨的支持票流向[陳偉業](../Page/陳偉業.md "wikilink")，以致何俊仁與李永達流失選票，但三人的票數加起來仍不及民建聯名單的第二位候選人，李永達最終落選。而民建聯[譚耀宗](../Page/譚耀宗.md "wikilink")所採用與鄉事派合作的策略是正確的，結果他與[鄧兆棠](../Page/鄧兆棠.md "wikilink")雙雙當選。

新界東方面，由於沒有增加議席，被視為席次保衛戰，民主黨分拆出[鄭家富](../Page/鄭家富.md "wikilink")、[黃成智](../Page/黃成智.md "wikilink")兩張名單，由鄭家富負責大埔、黃成智負責北區，而沙田的負任區則由二人分配。[前綫的](../Page/前綫_\(1996年\).md "wikilink")[劉慧卿](../Page/劉慧卿.md "wikilink")則受到民主黨的投票策略的影響，所得的票數比上屆減少三成左右，使排名第二的[蔡耀昌](../Page/蔡耀昌.md "wikilink")未能當選，相反，黃成智因選舉策略成功而得益，以當選名單的最低票數進入立法會。而長毛[梁國雄](../Page/梁國雄.md "wikilink")的形象與其他民主派人士不同，他與黃成智名單的得票只相差七千票左右，亦是落選名單中得票第一，所以在翌屆選舉中不少人同情梁國雄，「不讓長毛落選」的效應開始發酵。

九龍西方面，新增的一席被[馮檢基](../Page/馮檢基.md "wikilink")奪得。馮檢基得以自臨立會後重返立法會。泛民主派名單的總票數是民建聯名單的三倍。

九龍東方面，新增的一席則落入民建聯的陳鑑林手上，而且[陳婉嫻](../Page/陳婉嫻.md "wikilink")和[陳鑑林](../Page/陳鑑林.md "wikilink")民建聯名單的得票，剛好超越民主黨由[司徒華](../Page/司徒華.md "wikilink")領軍的名單。使陳婉嫻多了當屆「票-{后}-」的美名，是少數泛民、建制正面交鋒下，建制派得票較高的選區。

香港島方面，由於增加一席，而且民權黨的[陸恭蕙](../Page/陸恭蕙.md "wikilink")不再參選，為了爭奪這兩席，民主黨、民建聯以及其他候選人各出奇謀，原本[民建聯](https://zh.wikipedia.org/wiki/民建聯 "wikilink")[程介南](../Page/程介南.md "wikilink")與[蔡素玉](../Page/蔡素玉.md "wikilink")合組的名單穩奪兩席，而民主黨則鬧出[曾健成](../Page/曾健成.md "wikilink")脫黨參選的家變。剛退休的前局長[藍鴻震](../Page/藍鴻震.md "wikilink")則代表[新世紀論壇](../Page/新世紀論壇.md "wikilink")參選、在民間有名的《[三國演義](https://zh.wikipedia.org/wiki/三國演義 "wikilink")》節目解說人，於[1999年香港區議會選舉](../Page/1999年香港區議會選舉.md "wikilink")中擊敗[周梁淑怡](../Page/周梁淑怡.md "wikilink")的[馮兩努](../Page/馮兩努.md "wikilink")參選。[前綫的](../Page/前綫_\(1996年\).md "wikilink")[何秀蘭](../Page/何秀蘭.md "wikilink")由新界東空降到香港島參選，成為混戰之局。

選戰後期，《[蘋果日報](../Page/蘋果日報_\(香港\).md "wikilink")》揭發[程介南](../Page/程介南.md "wikilink")以權謀私的醜聞，使民建聯的選情一度重創，眼見民建聯名單排名第二的[蔡素玉](../Page/蔡素玉.md "wikilink")當選機會愈來愈低。但選舉結果，民建聯得到兩席，程介南、蔡素玉仍能雙雙當選，而民主黨亦沒因[曾健成](../Page/曾健成.md "wikilink")脫黨而嚴重失血，亦奪得兩席，而空降到香港島的[前綫何秀蘭亦取代](../Page/前綫_\(1996年\).md "wikilink")[陸恭蕙](../Page/陸恭蕙.md "wikilink")的一席。

由於程介南宣布就算他當選也會辭職，故這一席需要補選，結果民主黨主席[李柱銘](../Page/李柱銘.md "wikilink")與大律師和其他民主派協調，不推出民主黨第二梯隊的成員出選，推舉前[香港大律師公會](../Page/香港大律師公會.md "wikilink")主席[余若薇](../Page/余若薇.md "wikilink")參選，引來黨內第二梯隊的不滿。而民建聯則推舉原排在民建聯名單第4位的東區支部主席[鍾樹根](../Page/鍾樹根.md "wikilink")出戰，結果在民主派的大力支持下，余若薇以高票勝出，民主派終於在地方選區及選舉委員會的組別奪得過半數席次，足以否決建制派議員所提出的議案，例如「施政報告致謝議案」。

總體而言，在此次選舉中，民主黨的選票並沒有在這次選舉中增加，相比在1998年的選舉，民主黨在新界東也下降了一萬。在九龍西、九龍東也大跌四萬之多，而且在九龍東更被民建聯／工聯會名單超越變成第一。香港島雖然鬧出[曾健成](../Page/曾健成.md "wikilink")脫黨參選的家變，但兩張名單相加的票數也比上次大跌四萬之多。換言之，選民對民主黨有所不滿，或失去了像1998年一樣使民主黨重返立法會般的氣勢。加上選後民主黨的家變內鬥不斷，像[陳偉業](../Page/陳偉業.md "wikilink")等少壯派與主流派的矛盾日漸激化，甚至發生退黨。一切一切使民主黨的聲勢進一步滑落。

反之民建聯在香港島，雖然與福建社團的蔡素玉整合，或許因為程介南的以權謀私風波，民建聯選票也下跌一萬多。九龍東方面，雖然輕微下調，但跌幅無民主黨那麼厲害，還能超越民主黨成為第一。九龍西也輕微下調。但是在新界東，劉江華經營得力，得票數成長一萬多。而新界西亦因與鄉事派整合，得票數比上次成長差不多三萬，亦只以一萬之差落後於民主黨三張名單，但也是新界西的第一高票。

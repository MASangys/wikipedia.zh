**2003年歐洲足協圖圖盃**（**2003 UEFA Intertoto Cup**）從6月21日至8月26日舉行，為期兩個月，比賽每圈皆分為首次回合制，冠軍為[史浩克零四](https://zh.wikipedia.org/wiki/沙爾克04足球俱樂部 "wikilink")、[維拉利爾和](https://zh.wikipedia.org/wiki/比利亞雷亞爾足球俱樂部 "wikilink")[佩魯賈](https://zh.wikipedia.org/wiki/佩魯賈足球會 "wikilink")。

## 賽程

| 2003年6月28日或29日 | 第一圈次回合 |
| 2003年7月5日或6日   | 第二圈首回合 |
| 2003年7月12日或13日 | 第二圈次回合 |
| 2003年7月19日或20日 | 第三圈首回合 |
| 2003年7月26日     | 第三圈次回合 |
| 2003年7月30日     | 準決賽首回合 |
| 2003年8月4日      | 準決賽次回合 |
| 2003年8月12日     | 決賽首回合  |
| 2003年8月26日     | 決賽次回合  |

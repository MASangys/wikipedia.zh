**2008年度香港四台冠軍歌列表**列舉2008年間，於[香港](../Page/香港.md "wikilink")四大電子傳媒機構之[粵語流行音樂](../Page/粵語流行音樂.md "wikilink")流行榜冠軍歌曲。

## 流行榜

所統計之四大電子傳媒機構流行榜包括：

|                                                       |                                                         |         |
| ----------------------------------------------------- | ------------------------------------------------------- | ------- |
| **機構**                                                | **頻道**                                                  | **流行榜** |
| [香港電台](../Page/香港電台.md "wikilink")                    | [香港電台第二台](../Page/香港電台第二台.md "wikilink")                | 中文歌曲龍虎榜 |
| [商業電台](https://zh.wikipedia.org/wiki/商業電台 "wikilink") | [叱咤903](../Page/叱咤903.md "wikilink")                    | 903專業推介 |
| [新城電台](../Page/新城電台.md "wikilink")                    | [新城知訊台](https://zh.wikipedia.org/wiki/新城知訊台 "wikilink") | 新城勁爆流行榜 |
| [電視廣播有限公司](../Page/電視廣播有限公司.md "wikilink")（無綫電視）      | 勁歌金榜                                                    |         |

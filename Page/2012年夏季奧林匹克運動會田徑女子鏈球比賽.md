**2012年夏季奥林匹克运动会田徑比赛**的**女子鏈球**项目于2012年8月8日至8月10日在[倫敦](https://zh.wikipedia.org/wiki/倫敦 "wikilink")[倫敦奧林匹克體育場举行](https://zh.wikipedia.org/wiki/倫敦奧林匹克體育場 "wikilink")。

## 賽程

<small>所有時間為[英國夏令時間](https://zh.wikipedia.org/wiki/歐洲西部夏令時間 "wikilink")（[UTC+1](https://zh.wikipedia.org/wiki/UTC+1 "wikilink")）</small>

| 日期    | 時間       | 階段    |
| ----- | -------- | ----- |
| 8月8日  | 10:00    | 預賽    |
|       |          |       |
| 8月10日 | '''19:35 | '''決賽 |

## 比賽規則

在預賽，各位選手會有三次試擲，如果選手到達指定距離會直接晉級決賽，而少於12名選手到達指定長度則前12名選手會進入決賽。在決賽，前8名選手會有六次試擲，而第9名及以下的選手則有三次試擲。

**2012年夏季奥林匹克运动会田徑比赛**的**男子撐竿跳高**项目于2012年8月8日至8月10日在[倫敦](https://zh.wikipedia.org/wiki/倫敦 "wikilink")[倫敦奧林匹克體育場举行](https://zh.wikipedia.org/wiki/倫敦奧林匹克體育場 "wikilink")。

## 賽程

<small>所有時間為[英國夏令時間](https://zh.wikipedia.org/wiki/歐洲西部夏令時間 "wikilink")（[UTC+1](https://zh.wikipedia.org/wiki/UTC+1 "wikilink")）</small>

| 日期    | 時間       | 階段    |
| ----- | -------- | ----- |
| 8月8日  | 10:00    | 預賽    |
|       |          |       |
| 8月10日 | '''19:00 | '''決賽 |

## 比賽規則

在預賽，各位選手會不斷試跳直到她們達到指定高度或者被淘汰。如果選手到達指定高度會直入決賽。如果少於12名選手到達指定高度則前12名選手會進入決賽。每位選手在每個高度都會有三次試跳。

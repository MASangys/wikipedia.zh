本條目列舉**[2012年夏季奧林匹克運動會男子手球比賽](../Page/2012年夏季奧林匹克運動會男子手球比賽.md "wikilink")**的參賽資格及出線隊伍名單。各項決出奧運資格的賽事在2011年1月至2012年4月期間舉行。

## 概述

### 基本條件

參賽選手須為現行《[奥林匹克宪章](../Page/奥林匹克宪章.md "wikilink")》認可參與[奧林匹克運動會的運動員](https://zh.wikipedia.org/wiki/奧林匹克運動會 "wikilink")\[1\]。

### 總人數限額

本屆男子手球比賽的運動員總人數限額為168人（12隊），每個[奧委會](https://zh.wikipedia.org/wiki/奧委會 "wikilink")（NOC）最多可派出14名運動員參加比賽。其詳細分布見下表\[2\]：

| 參賽資格     | 選手數目    | 隊伍數目   |
| -------- | ------- | ------ |
| 通過資格賽的隊伍 | 154人    | 11隊    |
| 主辦國      | 14人     | 1隊     |
| '''總數    | '''168人 | '''12隊 |

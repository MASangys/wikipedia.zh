本條目列舉**[2012年夏季奧林匹克運動會羽毛球比賽](https://zh.wikipedia.org/wiki/2012年夏季奧林匹克運動會羽毛球比賽 "wikilink")**的參賽資格及運動員名單。

## 概述

### 基本條件

參賽選手須為現行《[奥林匹克宪章](../Page/奥林匹克宪章.md "wikilink")》認可參與[奧林匹克運動會的運動員](https://zh.wikipedia.org/wiki/奧林匹克運動會 "wikilink")\[1\]。

### 最多可得席位分配條件

每個[奧委會最多可在一個比賽項目中派出](https://zh.wikipedia.org/wiki/奧委會 "wikilink")3名或2對獲得參賽資格的運動員參加比賽，每個奧委會在每個項目最多可得席位的分配條件如下\[3\]：

| 項目                            | 條件（根據2012年5月3日世界羽聯公佈的世界排名） | 最多可得席位 |
| ----------------------------- | -------------------------- | ------ |
| 單打項目                          | 在同一項目中，所有選手皆排名1-4位         | 3席     |
| 在同一項目中，所有選手皆排名1-16位           | 2席                         |        |
| 只有1名選手列於2012年5月3日世界羽聯公佈的世界排名中 | 1席                         |        |
| 雙打項目                          | 在同一項目中，所有選手皆排名1-8位         | 4席（2對） |
| 只有1組選手列於2012年5月3日世界羽聯公佈的世界排名中 | 2席（1對）                     |        |

換言之，在條件符合的情況下，一個[奧委會最多可派出男](https://zh.wikipedia.org/wiki/奧委會 "wikilink")、女運動員各9名（男單3名、女單3名、男雙4名、女雙4名及混雙4名）參加比賽\[4\]。

### 積分計算方法

計算奧運積分的時期由2011年5月2日開始，至2012年5月3日結束。球員在此期間進行的110項賽事中，其所得的排名積分將會用作決定他們參加2012年夏季奧林匹克運動會羽毛球比賽的資格（換言之，亦即是2012年5月3日世界羽聯公布的世界排名）。這些賽事包括\[5\]：

  - [湯姆斯杯及](https://zh.wikipedia.org/wiki/湯姆斯杯 "wikilink")[尤伯杯](../Page/尤伯杯.md "wikilink")（各個階段）
  - [蘇迪曼杯](https://zh.wikipedia.org/wiki/蘇迪曼杯 "wikilink")
  - [世界羽毛球錦標賽](https://zh.wikipedia.org/wiki/世界羽毛球錦標賽 "wikilink")
  - [世界羽聯超級系列賽](https://zh.wikipedia.org/wiki/世界羽聯超級系列賽 "wikilink")（Super
    Series）賽事
  - [世界羽聯大獎賽](../Page/世界羽聯大獎賽.md "wikilink")（Grand Prix Gold & Grand
    Prix）賽事
  - 國際挑戰賽（International Challenge）賽事
  - 國際系列賽（International Series）賽事
  - 未來系列賽（Future Series）賽事
  - 各洲別的羽毛球錦標賽
  - 其他世界羽聯理事會批准的綜合性運動會內單項和團體比賽

## 爭議

[中国国家羽毛球队](../Page/中国国家羽毛球队.md "wikilink")在本屆賽事中首次取得5個項目的滿額席位（男女單各3席、雙打項目各2席，共18席）\[14\]。其中，男單選手[陈金](../Page/陈金.md "wikilink")在奧運積分賽期間大部分時間都排在丹麦老将盖德之下位列第5；直至積分賽最後一周，盖德在[印度超級賽八強出局](../Page/2012年印度羽毛球超級賽.md "wikilink")，陈金才能夠確定伦敦奥运的參賽資格\[15\]。

然而，中國羽毛球隊在此間為陈金爭取積分的行徑，卻引來外間廣泛批評。例如，陈金在奥运积分赛每遇上[林丹](../Page/林丹.md "wikilink")、[谌龙](../Page/谌龙.md "wikilink")等队友，对方往往会因伤退赛，被外界质疑为「让球」。英国《[每日电讯报](../Page/每日电讯报.md "wikilink")》指出，中国选手在2011年間進行的比赛中对阵99次，其中20场沒有完成比賽，退赛率达20%；但当中国选手对阵外国选手时，退赛率則只有0.21%，顯示中国隊的退赛動機明顯\[16\]；就連國內的《[中国青年报](../Page/中国青年报.md "wikilink")》也曾以《国羽退赛战术保了金牌丢了精神》为题，批评中国羽毛球队“以违背奥运精神的方式去参加奥运会，夺取奥运金牌，奥运理念出现了偏差”\[17\]。

此外，在2012年2月進行的[湯姆斯盃亞洲區預賽](../Page/2012年湯姆斯盃亞洲區預賽.md "wikilink")，本已憑東道主及衛冕身份得到參賽資格的中國隊，卻不必要地報名參賽以為陈金增加约1000个奥运积分；此舉亦惹來其他参赛队伍的一致声讨\[18\]。

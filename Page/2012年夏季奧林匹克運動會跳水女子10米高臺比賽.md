**2012年夏季奧林匹克運動會跳水比賽**的**女子10米高臺**项目于2012年8月8日至8月9日在[倫敦](https://zh.wikipedia.org/wiki/倫敦 "wikilink")[倫敦水上運動中心](../Page/倫敦水上運動中心.md "wikilink")举行。

## 賽程

<small>所有時間為英國夏令時間
([UTC+1](https://zh.wikipedia.org/wiki/UTC+1 "wikilink")) </small>

| 日期        | 時間        | 賽程      |
| --------- | --------- | ------- |
| 8月8日      | **19:00** | **預賽**  |
| 8月9日      | **10:00** | **準決賽** |
| **19:00** | **決賽**    |         |

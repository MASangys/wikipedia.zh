**2012年夏季奧林匹克運動會跳水比賽**的**女子3米彈板**项目于2012年8月3日至8月5日在[倫敦](https://zh.wikipedia.org/wiki/倫敦 "wikilink")[倫敦水上運動中心](../Page/倫敦水上運動中心.md "wikilink")举行。

## 賽程

<small>所有時間為英國夏令時間 ([UTC+1](https://zh.wikipedia.org/wiki/UTC+1 "wikilink")) </small>

| 日期   | 時間        | 賽程      |
| ---- | --------- | ------- |
| 8月3日 | **14:30** | **預賽**  |
| 8月4日 | **14:30** | **準決賽** |
| 8月5日 | **19:00** | **決賽**  |

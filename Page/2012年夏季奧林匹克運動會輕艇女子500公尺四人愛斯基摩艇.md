**2012年夏季奧林匹克運動會輕艇比賽**的**女子500公尺四人愛斯基摩艇**項目於2012年8月6日至8月8日在[多尼湖舉行](../Page/多尼湖.md "wikilink")。

## 賽程

<small>所有時間為[英國夏令時間](https://zh.wikipedia.org/wiki/歐洲西部夏令時間 "wikilink")（[UTC+1](https://zh.wikipedia.org/wiki/UTC+1 "wikilink")）</small>

| 日期    | 時間       | 階段    |
| ----- | -------- | ----- |
| 8月6日  | 10:39    | 預賽    |
| 11:51 | 準決賽      |       |
|       |          |       |
| 8月8日  | '''10:44 | '''決賽 |

## 比賽規則

比賽規則是根據比賽的輕艇數目來決定。如果比賽的輕艇數目為11至16艘，預賽將會分為兩組。預賽每組的第一名會直接進入決賽A組，其餘選手會進入準決賽。

每組準決賽的頭3名會進入決賽A組，而其餘的則進入決賽B組。

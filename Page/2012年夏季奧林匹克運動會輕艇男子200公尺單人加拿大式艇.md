**2012年夏季奧林匹克運動會輕艇比賽**的**男子200公尺單人加拿大式艇**項目於2012年8月10日至8月11日在[多尼湖舉行](../Page/多尼湖.md "wikilink")，為本次奧運會新增的輕艇小項。

## 賽程

<small>所有時間為[英國夏令時間](https://zh.wikipedia.org/wiki/歐洲西部夏令時間 "wikilink")（[UTC+1](https://zh.wikipedia.org/wiki/UTC+1 "wikilink")）</small>

|       |          |       |

## 比賽規則

比賽規則是根據比賽的輕艇數目來決定。如果比賽的輕艇數目超過24艘，預賽將會分為四組。預賽每組的第1至6名將會進入準決賽，其餘將被淘汰。

每組準決賽的第1至5名及第6名最快時間會進入決賽，其中每組第1至2名及時間最快的2位第3名會進入決賽A組，而其餘則進入決賽B組。

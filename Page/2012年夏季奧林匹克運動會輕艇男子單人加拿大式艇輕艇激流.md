**2012年夏季奧林匹克運動會輕艇比賽**的**男子單人加拿大式艇輕艇激流**項目於2012年7月29日至7月31日在[李爾峽谷白色水上中心](../Page/李爾峽谷白色水上中心.md "wikilink")舉行。

## 賽程

<small>所有時間為英國夏令時間（[UTC+1](https://zh.wikipedia.org/wiki/UTC+1 "wikilink")）</small>

| 日期       | 時間            | 階段  |
| -------- | ------------- | --- |
| 7月29日    | 13:30 & 15:42 | 預賽  |
| 7月31日    | 13:30         | 準決賽 |
|          |               |     |
| '''15:06 | '''決賽         |     |

## 比賽規則

賽事共有17位運動員參與，運動員需按次序和方向通過22個旗門。

本屆奧運的賽程分為三場比賽：預賽滑行二次，時間最快的12位運動員晉級準決賽。在準決賽中只有8人能出線決賽。在決賽中會進行一次性的滑行，成績最佳者為金牌得主。

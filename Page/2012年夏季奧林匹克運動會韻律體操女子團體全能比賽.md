**2012年夏季奥林匹克运动会**的**体操比赛－女子團體全能**项目于2012年8月9日至8月12日在[倫敦](https://zh.wikipedia.org/wiki/倫敦 "wikilink")[溫布萊體育館举行](https://zh.wikipedia.org/wiki/溫布萊體育館 "wikilink")。

## 賽程

<small>所有時間為[英國夏令時間](https://zh.wikipedia.org/wiki/歐洲西部夏令時間 "wikilink")（[UTC+1](https://zh.wikipedia.org/wiki/UTC+1 "wikilink")）</small>

| 日期    | 時間       | 階段    |
| ----- | -------- | ----- |
| 8月9日  | 12:00    | 預賽第一輪 |
| 8月10日 | 12:00    | 預賽第二輪 |
|       |          |       |
| 8月12日 | '''13:30 | '''決賽 |

## 比賽規則

預賽分為兩天進行，第一天將會進行五人球操，第二天會進行三人繩操及二人圈操混合韻律體操，總成績最高的8隊會進入決賽。預賽成績並不會帶入決賽。

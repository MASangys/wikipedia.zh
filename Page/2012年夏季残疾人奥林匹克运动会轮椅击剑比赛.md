**[2012年夏季殘疾人奧林匹克運動會](https://zh.wikipedia.org/wiki/2012年夏季殘疾人奧林匹克運動會 "wikilink")**的**輪椅劍擊比賽**會由9月4日至9月8日在[倫敦展覽中心](../Page/倫敦展覽中心.md "wikilink")舉行，本項目共產生12面金牌。輪椅劍擊在1960年正式成為殘奧會比賽項目。雖然在上屆北京殘奧會取消了5個團體項目，但在今屆殘奧會再次包括了2個團體項目，分別為男子團體花劍及女子團體重劍\[1\]。上屆輪椅劍擊項目獲得最多獎牌的國家是中國，共6金6銀1銅。

## 比賽分級

輪椅劍擊比賽會根據選手的殘障種類及程度分級，確保比賽的公平性及選手能夠與殘障程度接近的對手比賽。殘奧會輪椅劍擊比賽共有2個分級：

  - A級：選手能夠良好地控制軀幹，及他們的殘障對持劍手沒有影響。
  - B級：選手的殘障會影響他們對軀幹及持劍手的控制能力。

更詳細的分級可參考國際輪椅及截肢者體育運動聯會（IWAF）的分級指引\[2\]。

## 比賽項目

本屆殘奧會輪椅劍擊比賽是自2004年再次出現團體項目，共有12個項目。其中男子有7個項目、女子有5個項目。而個人項目亦分為A級及B級。

  - 男子
      - 個人重劍
      - 個人花劍
      - 個人佩劍
      - 團體花劍

<!-- end list -->

  - 女子
      - 個人重劍
      - 個人花劍
      - 團體重劍

## 參賽資格

本屆輪椅劍撃比賽共有100位選手，其中男子名額有60至76個，女子名額有24至40個。而每個國家或地區最多可以有6名男子及4名女子選手參加殘奧會。而每個項目最多可以有2位相同國家的選手，而每個選手最多可以參加兩個個人項目。此外，選手必須在2010年11月1日至2012年2月29日期間參加最少一項IWAS輪椅劍撃比賽\[3\]。

除了東道主國家，所有殘奧會名額都是屬於選手本身。而東道主國家的名額則所屬的殘委會。

此外，每個殘障級別的選手及團體隊伍的數目都設有上限：

| 項目   | 名額  |
| ---- | --- |
| 男子A級 | 30個 |
| 男子B級 | 30個 |
| 女子A級 | 12個 |
| 女子B級 | 12個 |
| 男子團體 | 8隊  |
| 女子團體 | 8隊  |

而每個單項的比賽名額亦設有上限（因每名選手可參加兩項個人比賽）：

| 項目   | 男子名額   | 女子名額   |
| ---- | ------ | ------ |
| 重劍A級 | 20個    | 12個    |
| 重劍B級 | 18個    | 12個    |
| 花劍A級 | 20個    | 12個    |
| 花劍B級 | 18個    | 12個    |
| 佩劍A級 | 20個    |        |
| 佩劍B級 | 18個    |        |
| 團體   | 8隊（花劍） | 8隊（重劍） |

## 比賽規則\[12\]

  - 輪椅劍撃比賽的場地為4米長、1.5米闊。選手的輪椅將會固定在場地，而比賽時只能夠自由移動他們的上半身。而比賽雙方的距離以手臂較短者能夠刺中對方為準。
  - 輪椅劍撃基本上與普通劍擊十分相似，亦分為三個類別：重劍、花劍、佩劍。
  - 重劍（Épée）：是三個類別中最重的劍。得分方法是用劍尖刺中對方上半身任何一部份。但用劍身攻擊則無效。此外，如果雙方同時擊中對手可同時得分。
  - 花劍（Foil）：是三個類別中最輕的劍。得分方法是用劍尖刺中對方頸項以下、大腿以上的身體軀幹部份，但不包括四肢。劍身攻擊亦是無效。
  - 佩劍（Sabre）：得分方法是可以用劍尖刺中或劍身劈中對方上半身，但不包括手腕。
  - 在小組賽，在3分鐘內得到5分者勝出。而淘汰賽，在9分鐘內（共分3節，每節3分鐘）得到15分者勝出。
  - 詳細規則可參考IWAS的官方網頁\[13\]。

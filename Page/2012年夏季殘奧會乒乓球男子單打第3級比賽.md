**[2012年夏季殘奧會](https://zh.wikipedia.org/wiki/2012年夏季殘奧會 "wikilink")**的**男子單打第3級比賽**會由8月30日至9月2日在[倫敦展覽中心舉行](../Page/倫敦展覽中心.md "wikilink")。

## 賽程

<small>所有時間為[英國夏令時間](https://zh.wikipedia.org/wiki/歐洲西部夏令時間 "wikilink")
([UTC+1](https://zh.wikipedia.org/wiki/UTC+1 "wikilink")) </small>

| 日期       | 時間            | 階段     |
| -------- | ------------- | ------ |
| 8月30日    | 09:00 & 19:20 | 小組賽    |
| 8月31日    | 18:00         |        |
| 9月1日     | 12:30         | 八強賽    |
| 18:15    | 準決賽           |        |
| **9月2日** | **10:15**     | **決賽** |

## 比賽規則

每場比賽將採用5局賽制。每場每局以先取得11分的選手獲得勝利。如果比數為10比10，勝利者則為隨後先取得2分領先的選手。

單打項目將採用小組及淘汰混合賽制。每個項目的準決賽勝方將角逐金牌，而敗方則爭取銅牌。

**[2012年夏季殘奧會](https://zh.wikipedia.org/wiki/2012年夏季殘奧會 "wikilink")**的**混合單打BC3級硬地滾球比賽**會由9月5日至9月8日在[倫敦展覽中心舉行](../Page/倫敦展覽中心.md "wikilink")。

## 賽程

<small>所有時間為[英國夏令時間](https://zh.wikipedia.org/wiki/歐洲西部夏令時間 "wikilink")
([UTC+1](https://zh.wikipedia.org/wiki/UTC+1 "wikilink")) </small>

| 日期           | 時間     | 階段  |
| ------------ | ------ | --- |
| 9月5日         | 09:00  | 種子賽 |
| 9月5日 | 11:25 | 第一圈    |     |
| 9月6日         | 10:05  | 第二圈 |
| 17:30        | 八強賽    |     |
| 9月7日         | 16:20  | 準決賽 |
| **16:25**    | **決賽** |     |
| **9月8日**     |        |     |

## 比賽規則

硬地滾球單打比賽會先進行排名賽決定對手，然後進行淘汰賽決定排名及獎牌。

**[2012年夏季殘疾人奧林匹克運動會](https://zh.wikipedia.org/wiki/2012年夏季殘疾人奧林匹克運動會 "wikilink")**的**混合單打BC1級硬地滾球比賽**會由9月5日至9月8日在[倫敦展覽中心](../Page/倫敦展覽中心.md "wikilink")舉行。

## 賽程

<small>所有時間為[英國夏令時間](https://zh.wikipedia.org/wiki/歐洲西部夏令時間 "wikilink") ([UTC+1](https://zh.wikipedia.org/wiki/UTC+1 "wikilink")) </small>

| 日期          | 時間        | 階段     |
| ----------- | --------- | ------ |
| 9月5日        | 09:00     | 種子賽    |
| 12:45～18:50 | 淘汰賽       |        |
| 9月6日        | 16:25     | 八強賽    |
| 9月7日        | 16:20     | 準決賽    |
| **9月8日**    | **13:25** | **決賽** |
|             |           |        |

## 比賽規則

硬地滾球單打比賽會先進行排名賽決定對手，然後進行淘汰賽決定排名及獎牌。

**[2012年夏季殘疾人奧林匹克運動會](https://zh.wikipedia.org/wiki/2012年夏季殘疾人奧林匹克運動會 "wikilink")**的**馬術團體賽**會由8月30日至9月2日在[倫敦格林威治公園舉行](https://zh.wikipedia.org/wiki/倫敦 "wikilink")。

## 賽程

<small>所有時間為[英國夏令時間](https://zh.wikipedia.org/wiki/歐洲西部夏令時間 "wikilink") ([UTC+1](https://zh.wikipedia.org/wiki/UTC+1 "wikilink")) </small>

| --------- | ------------ | -------- |
| 8月30日     | 09:00        | 團體測試賽II級 |
| 14:15     | 團體測試賽Ib級     |          |
| 8月31日     | 09:00        | 團體測試賽IV級 |
| 13:15     | 團體測試賽III級    |          |
| 15:45     | 團體測試賽Ia級     |          |
| 9月1日      | 09:00        | 個人錦標賽II級 |
| 14:45     | 個人錦標賽Ib級     |          |
| **9月2日**  | 09:00        | 個人錦標賽IV級 |
| 13:45     | 個人錦標賽III級    |          |
| **16:15** | **個人錦標賽Ia級** |          |

## 比賽規則

團體賽的排名是根據4位選手在團體測試賽及個人錦標賽中獲得的得分總和來決定。

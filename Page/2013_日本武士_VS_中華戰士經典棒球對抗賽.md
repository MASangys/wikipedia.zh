**2013 日本武士 VS 中華戰士經典棒球對抗賽**，[中華棒協向](https://zh.wikipedia.org/wiki/中華棒協 "wikilink")[日本野球機構提出邀請](https://zh.wikipedia.org/wiki/日本野球機構 "wikilink")，希望[日本](../Page/日本.md "wikilink")派出由職棒球員組成的代表隊，於季後與[中華隊進行友誼賽](../Page/中華成棒隊.md "wikilink")。

在11月8日到11月10日於[台北市](https://zh.wikipedia.org/wiki/台北市 "wikilink")[新莊棒球場](https://zh.wikipedia.org/wiki/新莊棒球場 "wikilink")、[台北市立天母棒球場進行三場友誼賽](https://zh.wikipedia.org/wiki/台北市立天母棒球場 "wikilink")。

## 比賽結果

【GAME 1】 19時37分比賽開始、比賽時間3小時39分鐘、觀眾人數7,277人。

|      |       |      |
| ---- | ----- | ---- |
| 日本武士 | 4 - 1 | 中華戰士 |

【GAME 2】 19時5分比賽開始、比賽時間3小時17分、觀眾人數9,752人。

|      |       |      |
| ---- | ----- | ---- |
| 中華戰士 | 2 - 4 | 日本武士 |

【GAME 3】 19時1分比賽開始、比賽時間2小時52分、觀眾人數8,081人。

|      |       |      |
| ---- | ----- | ---- |
| 日本武士 | 1 - 0 | 中華戰士 |

本條目列舉**[2014年世界羽毛球錦標賽](https://zh.wikipedia.org/wiki/2014年世界羽毛球錦標賽 "wikilink")**的參賽資格及運動員名單。

## 概述

### 比賽項目

本屆比賽包括五個項目：男子單打、女子單打、男子雙打、女子雙打及混合雙打\[1\]。

### 項目名額分配

本屆比賽的運動員總人數限額為400人。各項目的名額分配如下\[2\]：

|    | 男子單打 | 女子單打 | 男子雙打 | 女子雙打 | 混合雙打 | 總數      |
| -- | ---- | ---- | ---- | ---- | ---- | ------- |
|    |      |      |      |      |      |         |
| 總數 | 64   | 48   | 96   | 96   | 96   | **400** |

### 參賽名額限制

大會將按照世界羽聯在2014年4月24日公布的世界排名，決定選手的參賽資格；惟每個世界羽聯的成員協會所派出的選手數目則有所限制，每個成員協會在每個項目最多可得席位的分配條件如下\[3\]：

| 條件（根據2014年4月24日世界羽聯公佈的世界排名） | 最多可得席位 |
| --------------------------- | ------ |
| 在同一項目中，排名第1-8位的選手 / 組合      | 4席     |
| 在同一項目中，排名第9-24位的選手 / 組合     | 3席     |
| 在同一項目中，排名第25-150位的選手 / 組合   | 2席     |

如果同一成員協會中有超過4名選手/4對組合，排名第1-8位，則該成員協會可無須按照其排名，自由選擇派出任何4名選手/4對組合參賽。大會將按照上述安排決定符合參賽資格的選手，如其後參賽名單出現空缺，則由候補名單順序填補，惟同一成員協會中有超過4名選手/4對組合參賽。

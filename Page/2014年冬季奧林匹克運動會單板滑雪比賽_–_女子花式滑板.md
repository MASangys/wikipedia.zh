**2014年冬季奧林匹克運動會單板滑雪比賽 –
女子花式滑板**是於2014年2月6日及2月9日兩天，在[俄羅斯](https://zh.wikipedia.org/wiki/俄羅斯 "wikilink")[索契](../Page/索契.md "wikilink")[玫瑰莊園極限公園舉行的賽事](../Page/玫瑰莊園極限公園.md "wikilink")\[1\]。這也是女子花式滑板項目首次於冬季奧運會上登場。

## 賽程表

<small>所有時間的時區皆為[UTC+4](https://zh.wikipedia.org/wiki/UTC+4 "wikilink")</small>

| 日期        | 時間        | 賽事      |
| --------- | --------- | ------- |
| 2月6日      | **14:00** | **資格賽** |

| **13:15** | **決賽**    |         |

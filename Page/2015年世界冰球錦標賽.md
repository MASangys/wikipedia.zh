}} **2015年国际冰球联合会世界锦标赛**，在[捷克](../Page/捷克.md "wikilink")进行，主办城市为[布拉格](../Page/布拉格.md "wikilink")和[俄斯特拉发](../Page/俄斯特拉发.md "wikilink")。

2015年5月14日决赛结束后，本届赛事的总观众人数达到741700人次，打破了冰球世锦赛观众人数的历史记录。\[1\]

加拿大队在决赛中以6-1完胜俄罗斯队，夺得本届赛事冠军。加拿大队长西尼·克罗斯比成为历史上第26位夺得过斯坦利杯、奥运会冠军和冰球世锦赛冠军的球员，并且是首位在三支冠军队伍中都担任队长的球员。43岁的捷克老将亚罗米尔·雅格尔当选赛事最有价值球员。

## 竞争申办

2010年5月24日, 捷克在国际冰球联合会的申办竞选中得到84票，以62票的优势击败[乌克兰](../Page/乌克兰.md "wikilink")的[基辅](https://zh.wikipedia.org/wiki/基辅 "wikilink")，获得了本届赛事的举办权。\[2\]

举办本届赛事的2个场馆分别为布拉格的O2竞技场（容量为17383人）和俄斯特拉发的ČEZ竞技场（容量为8812人）。这两座球馆同样举办了2004年冰球世锦赛。在俄斯特拉发被选定前，主办方也曾考虑过[比尔森](../Page/比尔森.md "wikilink")、[布尔诺和](https://zh.wikipedia.org/wiki/布尔诺 "wikilink")[帕尔杜比采](../Page/帕尔杜比采.md "wikilink")，甚至还考虑过[斯洛伐克](../Page/斯洛伐克.md "wikilink")首都[布拉迪斯拉发](../Page/布拉迪斯拉发.md "wikilink")。\[3\]\[4\]

## 竞赛规程

参加本届赛事的16支球队被分成2个小组，每个小组的前4名将晋级淘汰赛。 \[5\]

小组赛的积分规则如下：

  - 常规时间获胜的球队得到3分 (**W**)。
  - 在加时赛或互射罚球中获胜的球队获得2分。加时赛时长3分钟，为金球制胜制。如果在加时赛中双方都不能进球，那么比赛进入互射罚球阶段。在淘汰赛阶段，加时赛时长变为10分钟；在决赛，加时赛时长变为20分钟。 (GWS) (**OTW**)。\[6\] Game Winning Shots procedure is as follows: Three different players from each team would take alternate shots. If the game is still tied after this, one player from each team would take alternating shots until one scored and the other missed. Only the decisive goal counted in the result table for group play.\[7\]|group=n}}
  - 在加时赛或互射罚球中落败的球队获得1分 (**OTL**)。
  - 在常规时间落败的球队得0分 (**L**)。

如果小组中有2支或以上的球队积分相同，由以下的方式决定排名:\[8\]

1.  相互交锋战绩
2.  相互交锋的净胜球
3.  相互交锋的进球数
4.  与另一小组排名第一的球队相互交锋的战绩
5.  与另一小组排名第二的球队相互交锋的战绩
6.  国际冰联排名

最终排名： 第1-4名由冠亚军决赛和第三名决赛决定，其他名词由淘汰赛排名、小组赛排名决定。 本届赛事排名第15和16的球队将降入世锦赛IA组。\[9\]

## 球员

每支球队的名单需要包括15-22名非门将球员（前锋和后卫）和2-3名门将。 每支球队都需要在赛事开始前上交一份球员名单，但在任意一场比赛之前，球队在不超过人员限制的情况下可以增加球员（因此，结束NHL季后赛任务的球员可以在赛事的任意时刻加入）。

## 裁判员

国际冰球联合会挑选了16名主裁判和16名边裁执法本届世锦赛。 \[11\]\[12\]

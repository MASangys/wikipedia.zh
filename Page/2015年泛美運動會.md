**第十七屆泛美運動會**（），于2015年7月10日至7月26日在[加拿大](../Page/加拿大.md "wikilink")[安大略省](../Page/安大略省.md "wikilink")[多倫多市舉行](https://zh.wikipedia.org/wiki/多倫多 "wikilink")。

## 有意申辦城市

[泛美體育組織於](https://zh.wikipedia.org/wiki/泛美體育組織 "wikilink")2009年11月6日在[墨西哥](../Page/墨西哥.md "wikilink")[瓜达拉哈拉举行的會議中從以下三個候選城市中選出主辦城市](https://zh.wikipedia.org/wiki/瓜达拉哈拉 "wikilink")：

| 2015年泛美運動會主辦城市甄選結果                                       |
| -------------------------------------------------------- |
| 城市                                                       |
| [多倫多](https://zh.wikipedia.org/wiki/多倫多 "wikilink")\[1\] |
| [利馬](../Page/利馬.md "wikilink")\[2\]                      |
| [波哥大](../Page/波哥大.md "wikilink")                         |

最終由多倫多脫穎而出，獲得是次泛美運動會主辦權。

此外，[委内瑞拉的](https://zh.wikipedia.org/wiki/委内瑞拉 "wikilink")[卡拉卡斯](../Page/卡拉卡斯.md "wikilink")亦曾有意參與角逐\[3\]，但其後卻放棄\[4\]。而[美國](https://zh.wikipedia.org/wiki/美國 "wikilink")[亞拉巴馬州](https://zh.wikipedia.org/wiki/亞拉巴馬州 "wikilink")[伯明翰的市長亦曾提出參與角逐](https://zh.wikipedia.org/wiki/伯明翰_\(亞拉巴馬州\) "wikilink")，但該市卻沒有趕及在截止報名日期前遞交申請。\[5\]

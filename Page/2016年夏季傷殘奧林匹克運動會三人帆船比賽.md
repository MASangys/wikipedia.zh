**[2016年夏季傷殘奧林匹克運動會的三人](https://zh.wikipedia.org/wiki/2016年夏季帕拉林匹克運動會 "wikilink")[帆船比賽](https://zh.wikipedia.org/wiki/2016年夏季傷殘奧林匹克運動會帆船比賽 "wikilink")**在9月12日至17日於[格洛里亞碼頭舉行](../Page/格洛里亞碼頭.md "wikilink")。本項目為男女混合比賽，使用的帆船為索納級重型帆船。

## 比賽模式

傷殘奧林匹克運動會帆船比賽將進行11回合的比賽，並依照排名給予分數，如第1名獲到1分、第2名獲得2分，如此類推。最後，將成績最差的1場減去後的，所有場次的總分數就是最後成績，而最低分的運動員將獲得金牌。

以下時間為南美標準時間（[UTC-3](https://zh.wikipedia.org/wiki/UTC-3 "wikilink")）\[1\]。

| 日期        | 時間        | 回合        |
| --------- | --------- | --------- |
| 9月12日     | 13:30     | 第1、2回合    |
| 9月13日     | 13:30     | 第3、4回合    |
| 9月14日     | 13:30     | 第5、6回合    |
| 9月15日     | 13:30     | 第7、8回合    |
| 9月16日     | 13:30     | 第9、10回合   |
| **9月17日** | **14:00** | **第11回合** |

  - **DNC**：沒有到達比賽場地
  - **DNS**：沒有開始比賽（非DNS及OCS）
  - **OCS**：沒有開始比賽，違反規則30.1（偷步）
  - **ZFP**：違反規則30.2，20%罰分
  - **BFD**：違反規則30.3，取消資格

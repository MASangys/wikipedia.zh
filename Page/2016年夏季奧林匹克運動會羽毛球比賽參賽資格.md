### 最多可得席位分配條件

每個[奧委會最多可在一個比賽項目中派出](https://zh.wikipedia.org/wiki/奧委會 "wikilink")2名或2對獲得參賽資格的運動員參加比賽，每個奧委會在每個項目最多可得席位的分配條件如下\[3\]：

| 項目                            | 條件（根據2016年5月5日世界羽聯公佈的世界排名） | 最多可得席位    |
| ----------------------------- | -------------------------- | --------- |
| 單打項目                          | 在同一項目中，所有選手皆排名1-16位        | 2席（2名運動員） |
| 只有1名選手列於2016年5月5日世界羽聯公佈的世界排名中 | 1席（1名運動員）                  |           |
| 雙打項目                          | 在同一項目中，所有選手皆排名1-8位         | 2席（4名運動員） |
| 只有1組選手列於2016年5月5日世界羽聯公佈的世界排名中 | 1席（2名運動員）                  |           |

換言之，在條件符合的情況下，一個[奧委會最多可派出男](https://zh.wikipedia.org/wiki/奧委會 "wikilink")、女運動員各8名（男單2名、女單2名、男雙4名、女雙4名及混雙4名）參加比賽\[4\]。

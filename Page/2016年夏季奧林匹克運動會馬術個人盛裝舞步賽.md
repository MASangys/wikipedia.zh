**2016年夏季奧林匹克運動會[馬術個人](https://zh.wikipedia.org/wiki/馬術 "wikilink")[盛裝舞步賽](../Page/盛裝舞步.md "wikilink")**于2016年8月10日至15日在[巴西](../Page/巴西.md "wikilink")[里约热内卢奥林匹克马术中心举行](../Page/里约热内卢.md "wikilink")，共有来自25个国家和地区的60名运动员参加此次比赛。\[1\]

## 赛程

以下是此次比赛的赛程：\[2\]

<small>所有時間為巴西時間（[UTC-3](https://zh.wikipedia.org/wiki/UTC-3 "wikilink")）</small>

| 日期             | 时间        | 阶段         |
| -------------- | --------- | ---------- |
| 2016年8月10日     | 10:00     | 大奖赛（第一天）   |
| 2016年8月11日     | 10:00     | 大奖赛（第二天）   |
| 2016年8月12日     | 10:00     | 特大奖赛       |
| **2016年8月15日** | **10:00** | **大奖赛自由式** |

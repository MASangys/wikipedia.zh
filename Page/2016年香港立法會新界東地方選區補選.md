**2016年香港立法會新界東地方選區補選**，是[香港特別行政區成立後第](https://zh.wikipedia.org/wiki/香港特別行政區 "wikilink")7次進行的[香港立法會](../Page/香港立法會.md "wikilink")議席補選，於2016年2月28日舉行\[1\]，以填補於2015年10月1日起辭去立法會議員席位的[湯家驊](../Page/湯家驊.md "wikilink")在[新界東的民選議席空缺](https://zh.wikipedia.org/wiki/新界東 "wikilink")。這是[雨傘革命](../Page/雨傘革命.md "wikilink")後的第二場香港大型選舉，雖然只在部分區域。

根據立法會條例，立法會地區直選議席選舉採用[名單比例代表制](../Page/名單比例代表制.md "wikilink")，然而由於是次補選只須選出一個議員，變相是[單議席單票制](https://zh.wikipedia.org/wiki/單議席單票制 "wikilink")\[2\]。這次補選由[泛民主派](../Page/泛民主派.md "wikilink")的[公民黨](../Page/公民黨_\(香港\).md "wikilink")[楊岳橋](../Page/楊岳橋.md "wikilink")勝出，泛民成功維持在地區直選的過半議席。

## 背景

2015年6月22日，原任立法會新界東民選議員、[泛民主派](../Page/泛民主派.md "wikilink")的[湯家驊](../Page/湯家驊.md "wikilink")在投票否決政改方案後宣佈退出[公民黨](../Page/公民黨_\(香港\).md "wikilink")，並辭去其立法會議員席位，10月1日起生效\[3\]\[4\]。

雖然是次選舉只有一席任期7個月的補選議席，但由於可視作9月全港立法會選舉的參考指標，因此成為各大政黨爭取的目標，初期形成泛民主派和建制派之爭。但在[2016年旺角騷亂](../Page/2016年旺角騷亂.md "wikilink")後，6號候選人、本土派的[本土民主前線](../Page/本土民主前線.md "wikilink")[梁天琦](../Page/梁天琦.md "wikilink")被警方落案起訴暴動罪，知名度大增，令是次選舉增添變數。

## 各政黨立場

  - [民主黨](../Page/民主黨_\(香港\).md "wikilink")：民主黨選舉委員會主席[單仲偕](../Page/單仲偕.md "wikilink")於6月23日表示不會派人參選，並表示由公民黨派人參與補選，以填補湯家驊的席位是合適做法，又指民主黨可提供協助。至於當時被凍結黨員資格的民主黨前立法會議員[黃成智](../Page/黃成智.md "wikilink")表示有意參選，單表示民主黨於6月底會決定是否將黃成智驅逐出黨。一旦他被逐出黨，民主黨只能勸誡他不要參選，其行為與民主黨無關\[5\]。
  - [人民力量](../Page/人民力量.md "wikilink")：人民力量立法會議員[陳偉業](../Page/陳偉業.md "wikilink")表示希望公民黨派出重量級人物出戰補選，免失議席及分組點票否決權，並點名希望[余若薇](../Page/余若薇.md "wikilink")或[吳靄儀](../Page/吳靄儀.md "wikilink")參選\[6\]。
  - [新民黨](https://zh.wikipedia.org/wiki/新民黨 "wikilink")：黨主席[葉劉淑儀](../Page/葉劉淑儀.md "wikilink")於6月23日表示該黨傾向不派員參選\[7\]。

## 廢票

### 定義

廢票是指無效的選票\[22\]，如白票、填寫不當、塗污了的選票等，算法是把總投票數字減去有效票數的數字；而廢票率則是廢票數量佔總投票數字的百分比，公式如下：

對每個選區而言：

|                        |
| ---------------------- |
| 有效票數 = ∑(各候選人票數)       |
| 廢票量 = 總票數 - 有效票數       |
| 廢票率 = 廢票量 ÷ 總票數 × 100% |

## 事件

### 黃成智在電視選舉論壇為方國珊拉票

2016年2月15日，[now新聞台播出的選舉論壇中](https://zh.wikipedia.org/wiki/now新聞台 "wikilink")，二號候選人[黃成智](../Page/黃成智.md "wikilink")在拉票環節時竟替五號候選人[方國珊](../Page/方國珊.md "wikilink")拉票，遭網民質疑動機\[24\]。但及後在[香港電台第一台](../Page/香港電台第一台.md "wikilink")於2月19日舉辦的選舉論壇中，黃成智又推翻當日言論，呼籲選民不要支持方國珊\[25\]。

### 選舉事務處拒絕免費投寄梁天琦選舉郵件

2016年2月16日，[選舉事務處指](https://zh.wikipedia.org/wiki/選舉事務處 "wikilink")[梁天琦](../Page/梁天琦.md "wikilink")的選舉宣傳郵件中，「自治」、「自決前途」等字眼抵觸《[香港特別行政區基本法](https://zh.wikipedia.org/wiki/香港特別行政區基本法 "wikilink")》，故拒絕替其免費投寄。梁天琦批評選舉事務處侵犯《基本法》保障的言論及出版自由，以行政手段干預政治\[26\]。而[公民黨亦在Facebook專頁發表聲明](https://zh.wikipedia.org/wiki/公民黨 "wikilink")，譴責選舉事務處自行詮釋《基本法》，打壓候選人宣傳的權利\[27\]\[28\]。

### 楊岳橋對同性婚姻立場模棱兩可惹爭議

2016年2月22日，[商業電台節目](../Page/香港商業電台.md "wikilink")《[人民大道中](https://zh.wikipedia.org/wiki/人民大道中 "wikilink")》主持[健吾](../Page/健吾.md "wikilink")受博客[記者甲啟發](https://zh.wikipedia.org/wiki/記者甲 "wikilink")，致電詢問候選人對[同性婚姻的看法](https://zh.wikipedia.org/wiki/同性婚姻 "wikilink")\[29\]。七位候選人有五人明確表示反對或保留，當中僅[梁天琦](../Page/梁天琦.md "wikilink")明確表示會「義無反顧」地支持同性婚姻及同志平權，亦支持在立法會內捍衛同志權利；公民黨[楊岳橋](../Page/楊岳橋.md "wikilink")強調支持同志平權，任何人均應享有自由平等的權利，被質疑答非所問、取態模棱兩可，需二度在Facebook澄清\[30\]，清楚表明支持同性婚姻的立場\[31\]\[32\]，但同時又稱「立法有序，需先從[反歧視法做起](https://zh.wikipedia.org/wiki/同性反歧視法 "wikilink")」\[33\]，及後亦未再有回應記者甲對「立法有序」的質疑\[34\]。

### 周浩鼎在電視選舉論壇中突然落淚

2016年2月25日，於無綫電視舉辦的選舉論壇中，民建聯候選人[周浩鼎](../Page/周浩鼎.md "wikilink")發言時哽咽落淚\[35\]，表示希望七名候選人都可以做朋友，「坐下慢慢談，而不是天天你打我，我打你」，當他強調「香港不是這樣的」時，便低頭拿出一張紙巾抹鼻，眼泛淚光，聲稱眼淚「為香港而流」\[36\]。隨後[方國珊](../Page/方國珊.md "wikilink")回應指周浩鼎與其黨友應檢討，不要扼殺市民的意見。另一候選人[梁思豪](../Page/梁思豪.md "wikilink")懷疑周浩鼎假裝落淚以博同情，質疑周輕易流下男兒淚，日後在議會內能否撐得住。

### 錦濤票站遺失投票箱鎖匙

2016年2月28日晚上11時，投票站於投票結束後隨即成為點票站，位於馬鞍山[頌安邨](../Page/頌安邨.md "wikilink")陳震夏中學票站的四個投票箱中，其中一個投票箱的鑰匙不翼而飛。事件引起在場監票員、市民及當區區議員[陳國強不滿及鼓譟](https://zh.wikipedia.org/wiki/陳國強 "wikilink")，擔心有人已偷開票箱並換走箱中選票，甚至質疑該票箱只是「突然出現」\[37\]。午夜12時，票站主任致電警方求助。一批手持[胡椒噴霧](../Page/胡椒噴霧.md "wikilink")、防暴長警棍及警告旗的[機動部隊警員到場](https://zh.wikipedia.org/wiki/機動部隊 "wikilink")，包圍點票區。有在場市民投訴警方阻礙公眾觀看點票過程。凌晨1時，票站職員無視現場人士的反對，將票箱強行開鎖並將內裡具爭議的選票混入三箱正常的選票之中\[38\]。

### 無綫新聞引用民建聯提供之數據報票

### 大學新聞系刊物被拒絕採訪

多家學生媒體包括[香港浸會大學](../Page/香港浸會大學.md "wikilink")傳理學院新聞系學生刊物《新報人》、[香港中文大學新聞傳播學系刊物](https://zh.wikipedia.org/wiki/香港中文大學 "wikilink")《Varsity》和另一刊物《大學線》派出記者在選舉當天到新聞中心所在地調景嶺體育館採訪，但遭拒絕進入館內記者區。學生向選管會投訴，[政府新聞處](../Page/政府新聞處.md "wikilink")回覆說「並非所有已註冊的傳媒可獲安排現場採訪」，有關刊物也非「大眾新聞傳媒機構」，學生記者也不是專業記者，基於場地限制和保安等因素未能接納其採訪要求。《新報人》及中大新傳系學生向[高等法院入稟申請](https://zh.wikipedia.org/wiki/香港高等法院 "wikilink")[司法覆核](../Page/司法覆核.md "wikilink")，質疑處方違反《基本法》賦予之[新聞自由](https://zh.wikipedia.org/wiki/新聞自由 "wikilink")，要求保障採訪權。他們說3本刊物均根據《本地報刊註冊條例》合法註冊，定期出版，與一般其他傳統媒體無分別。\[39\]\[40\]

2018年11月，高等法院頒下判詞拒絕受理本案，判詞指政府自2016年9月起已採取新措施，只容許指定數量學生記者採訪主要官方活動，每次讓每間院校不多於4名新聞系學生參與採訪，過去做法則是任何學生均不能入場採訪，法官認為因此無覆核必要。\[41\]後來《新報人》正式提出上訴，認為「新措施」在入稟時仍未推出，質疑該措施只是為了應付這次司法覆核。\[42\]

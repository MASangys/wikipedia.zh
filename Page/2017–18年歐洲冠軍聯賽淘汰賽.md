**2017–18年歐洲冠軍聯賽淘汰賽**將在2018年2月13日開始並於2018年5月26日舉行[決賽於烏克蘭](../Page/2018年歐洲冠軍聯賽決賽.md "wikilink")[基輔](../Page/基輔.md "wikilink")[NSC奧林匹克國家綜合體育場決定出](../Page/奧林匹克國家綜合體育場.md "wikilink")[2017–18年歐洲冠軍聯賽](../Page/2017–18年歐洲冠軍聯賽.md "wikilink")冠軍後結束。\[1\] 在淘汰賽階段共有16支球隊參賽。\[2\]

## 賽程及抽籤

淘汰賽之賽程如下（所有抽籤儀式會於[歐洲足球協會聯盟](../Page/歐洲足球協會聯盟.md "wikilink")總部[瑞士](https://zh.wikipedia.org/wiki/瑞士 "wikilink")[尼翁舉行](https://zh.wikipedia.org/wiki/尼翁 "wikilink")）：

| 圈數   | 抽籤                                                                                         | 首回合                  | 次回合                |
| ---- | ------------------------------------------------------------------------------------------ | -------------------- | ------------------ |
| 十六強  | 2017年12月11日                                                                                | 2018年2月13–14日／20–21日 | 2018年3月6–7日／13–14日 |
| 半準決賽 | 2018年3月16日                                                                                 | 2018年4月3–4日          | 2018年4月10–11日      |
| 準決賽  | 2018年4月13日                                                                                 | 2018年4月24–25日        | 2018年5月1–2日        |
| 決賽   | 2018年5月26日 於[基輔](../Page/基輔.md "wikilink")[奧林匹克國家綜合體育場](../Page/奧林匹克國家綜合體育場.md "wikilink") |                      |                    |

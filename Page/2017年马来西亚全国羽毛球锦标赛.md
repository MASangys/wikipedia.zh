**2017年马来西亚全国羽毛球锦标赛**為第1屆[马来西亚全国羽毛球锦标赛](../Page/马来西亚全国羽毛球锦标赛.md "wikilink")，是[马来西亚](../Page/马来西亚.md "wikilink")國內最高級別的[羽毛球](../Page/羽毛球.md "wikilink")比賽。本屆賽事於2017年2月19日至25日在[马来西亚](../Page/马来西亚.md "wikilink")[霹雳州](https://zh.wikipedia.org/wiki/霹雳州 "wikilink")的[Dewan Sri Perdana舉行](https://zh.wikipedia.org/wiki/Dewan_Sri_Perdana "wikilink")\[1\]\[2\]。本屆亦是取代前身[马来西亚全国羽毛球大奖赛总决赛](../Page/马来西亚全国羽毛球大奖赛总决赛.md "wikilink")後舉辦的第一屆賽事\[3\]。

## 賽事資料

### 比賽安排

賽事包括：男子單打、女子單打、男子雙打、女子雙打及混合雙打五個項目，分預賽（Qualifying Rounds）及主賽（Main Draw）兩部分\[4\]。

### 比賽日程

大會公布的日程如下\[5\]：

| 賽日  | 日期         | 時間            | 比賽  |
| --- | ---------- | ------------- | --- |
| 第1日 | 2017年2月19日 | 09:00 - 17:00 | 資格賽 |
| 第2日 | 2017年2月20日 | 09:00 - 21:00 | 資格賽 |
| 第3日 | 2017年2月21日 | 09:00 - 17:00 | 資格賽 |
| 第4日 | 2017年2月22日 | 09:00 - 19:00 | 首輪  |
| 第5日 | 2017年2月23日 | 09:00 - 21:00 | 次輪  |
| 第6日 | 2017年2月24日 | 15:30 - 20:00 | 準決賽 |
| 第7日 | 2017年2月25日 | 15:00 - 19:30 | 決賽  |

### 獎金安排

本屆賽事的總獎金達[馬來西亞令吉](../Page/馬來西亞令吉.md "wikilink")27万。按[馬來西亞羽毛球協會](../Page/馬來西亞羽毛球協會.md "wikilink")的指引，獎金分布的安排如下\[6\]：

| 项目   | 奖金       |
| ---- | -------- |
| 男子單打 | RM2万     |
| 女子單打 | RM1万4000 |
| 男子雙打 | RM3萬     |
| 女子雙打 | RM2萬     |
| 混合雙打 | RM2萬     |

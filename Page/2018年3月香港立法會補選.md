[LegCoByElection2018.svg](https://zh.wikipedia.org/wiki/File:LegCoByElection2018.svg "fig:LegCoByElection2018.svg")、[新界東選區](../Page/新界東選區.md "wikilink")和[九龍西選區](../Page/九龍西選區.md "wikilink")的民選議席及[建築、測量、都市規劃及園境界功能界別](../Page/建築、測量、都市規劃及園境界功能界別.md "wikilink")議席分佈地圖\]\] **2018年3月香港立法會補選**是[香港特別行政區成立後第](https://zh.wikipedia.org/wiki/香港特別行政區 "wikilink")8次進行的[香港立法會](../Page/香港立法會.md "wikilink")議席補選，於2018年3月11日舉行\[1\]\[2\]\[3\]，以填補於[香港高等法院裁定自](https://zh.wikipedia.org/wiki/香港高等法院 "wikilink")2016年10月12日起喪失立法會議員就任資格的[羅冠聰](../Page/羅冠聰.md "wikilink")、[游蕙禎](../Page/游蕙禎.md "wikilink")、[梁頌恆](../Page/梁頌恆.md "wikilink")和[姚松炎](../Page/姚松炎.md "wikilink")分別在三個[地方直選即](https://zh.wikipedia.org/wiki/地方選區 "wikilink")[香港島選區](../Page/香港島選區.md "wikilink")、[九龍西選區](../Page/九龍西選區.md "wikilink")和[新界東選區](../Page/新界東選區.md "wikilink")的民選議席及[建築、測量、都市規劃及園境界功能界別](../Page/建築、測量、都市規劃及園境界功能界別.md "wikilink")議席空缺。

提名期從2018年1月16日至1月29日止\[4\]。根據立法會條例，立法會地區直選議席選舉採用[名單比例代表制](../Page/名單比例代表制.md "wikilink")，然而由於是次補選只須選出一個議員，變相是[單議席單票制](https://zh.wikipedia.org/wiki/單議席單票制 "wikilink")。如無意外，當選人將會完成原議員餘下任期至2020年9月30日。

另外兩位被撤銷議員資格的前議員[梁國雄](../Page/梁國雄.md "wikilink")及[劉小麗](../Page/劉小麗.md "wikilink")正等候上诉，预计排期至少半年\[5\]，故是次補選並不包括他們的議席空缺。

## 背景

2016年10月12日，[第六屆香港立法會](../Page/第六屆香港立法會.md "wikilink")舉行宣誓儀式。政府其後分別就[游蕙禎](../Page/游蕙禎.md "wikilink")、[梁頌恆](../Page/梁頌恆.md "wikilink")及[劉小麗](../Page/劉小麗.md "wikilink")、[梁國雄](../Page/梁國雄.md "wikilink")、[羅冠聰](../Page/羅冠聰.md "wikilink")、[姚松炎](../Page/姚松炎.md "wikilink")六人的宣誓入稟，要求法庭頒令各人宣誓無效並撤銷其議員資格。[高等法院裁決六人宣誓無效](https://zh.wikipedia.org/wiki/香港高等法院 "wikilink")，且自宣誓當日起喪失議員資格。劉小麗和梁國雄就裁決提出上訴，其餘四人的司法程序則已完成，需要進行補選。

## 民主派初選

為增加勝算及避免分裂，民主派將於是次補選前進行「初選」，於每地方選區協調出一位候選人，並定立若初選勝出者被政府取消參選資格（DQ）的替補方法。初選由協調民主派選舉安排的組織「[民主動力](../Page/民主動力.md "wikilink")」負責，已於2018年1月13日分別在九龍西及新界東兩地方選區順利舉行。

### 廢票

廢票是指無效的選票\[22\]，如白票、填寫不當、塗污了的選票等，算法是把總投票數字減去有效票數的數字；而廢票率則是廢票數量佔總投票數字的百分比，公式如下：

對每個選區而言：

|                        |
| ---------------------- |
| 有效票數 = ∑（各候選人票數）       |
| 廢票量 = 總票數 - 有效票數       |
| 廢票率 = 廢票量 ÷ 總票數 × 100% |

所有原始數據均摘自香港政府立法會選舉網站\[23\]，依以上定義運用前述公式作簡單計算而得出下表：

## 爭議

### 選舉日期

在補選舉行日期上，[建制派批評補選日子與每年的全國人大政協](https://zh.wikipedia.org/wiki/建制派 "wikilink")[兩會](https://zh.wikipedia.org/wiki/兩會 "wikilink")「撞期」，加上今年正值換屆， 建制派的民建聯主席[李慧琼](../Page/李慧琼.md "wikilink")批評訂立日期的人對「國家的政治傳統認識不足」，民主派的公民黨黨魁[楊岳橋](../Page/楊岳橋.md "wikilink")則指建制派的批評是質疑[選舉管理委員會的獨立性](https://zh.wikipedia.org/wiki/選舉管理委員會 "wikilink")\[24\]。[全國政協委員](https://zh.wikipedia.org/wiki/全國政協委員 "wikilink")、立法會議員、[經民聯副主席](https://zh.wikipedia.org/wiki/經民聯 "wikilink")[林健鋒](../Page/林健鋒.md "wikilink")指撞期令「[上面的人](../Page/北京市.md "wikilink")」不高興，但他同時指港府仍按選管會安排在3月11日舉行補選，顯示「[一國兩制運作良好](https://zh.wikipedia.org/wiki/一國兩制 "wikilink")」。[政務司司長](https://zh.wikipedia.org/wiki/政務司司長 "wikilink")[張建宗](../Page/張建宗.md "wikilink")則指日期是由選管會根據法例和種種因素來決定，政府曾「努力過」，但最終仍尊重選管會決定\[25\]。及後張建宗解釋「努力過」是指政府有努力了解和分析事件\[26\]。

其後，有媒體報道披露補選選擇於3月11日舉行的原因，是因為遷就[香港中學文憑考試](../Page/香港中學文憑考試.md "wikilink")，如果押後舉行便會與中學文憑試開考「撞期」，屆時會難以借用學校作投票場地，但如等到考試結束卻會大大拖延補選時間表\[27\]，若6月才举行補選，新议员只能参与餘下的一个月会期，对新议员不公平。

此外，非建制派在選管会決定補選日期前，一直批評選管会一直未決定補選日期，而议席亦懸空已久。（以往補選日期议席懸空后，在六个月內举行。四个议席实际懸空八个月，两个议席实际上空十六个月，而法律上实际懸空十七个月。）

### 民主派參選人被取消參選資格

[Agnes_Chow_being_interviewed_in_Jan_2018_(2).jpg](https://zh.wikipedia.org/wiki/File:Agnes_Chow_being_interviewed_in_Jan_2018_\(2\).jpg "fig:Agnes_Chow_being_interviewed_in_Jan_2018_(2).jpg")成員，被取消參選資格而未能參選。\]\] 2018年1月25日，傳出民主派港島參選人[周庭](../Page/周庭.md "wikilink")及九龍西參選人[姚松炎](../Page/姚松炎.md "wikilink")可能被取消參選資格。選舉事務處承認，選舉主任曾就三個地區補選諮詢律政司意見，但拒絕披露細節。姚松炎在Facebook指若果再度被取消參選資格是「剝奪政治權利終生」，形容是「威權向法治下戰書」，而周庭被DQ「就是DQ整個政治聯繫的政治權利」。《星島日報》引述消息人士指出，兩人被取消參選資格，是建基於全國人大常委會在2016年就《基本法》第104條進行的釋法。[香港眾志](../Page/香港眾志.md "wikilink")創辦人之一的周庭，以「民主自決」為最高綱領，並將「港獨」列為選項，因而不合乎要求。而姚松炎則因釋法第四條規定，「不得重新安排宣誓」指的是同一屆立法會會期內不得重新安排宣誓，即不具備參加補選資格\[28\]。面對被取消參選資格可能性非常高，民主派開始為替補二人的Plan B做準備。民主黨中委會支持[袁海文](../Page/袁海文.md "wikilink")擔任姚松炎的後備方案。民主黨前成員、南區區議員[區諾軒](../Page/區諾軒.md "wikilink")，料會頂替周庭在港島區參選\[29\]。

2018年1月27日，特區政府發聲明，指已有一名候選人因提倡「民主自決或以公投方式提出包括選擇獨立來處理香港體制等」，不符合《立法會條例》第40(1)(b)(i)條的規定，被選舉主任裁定其提名為無效。聲明強調「鼓吹或推動民主自決或以何種形式提議獨立的人士不可能擁護《基本法》，因此不可能履行立法會議員的職責」\[30\]。同日，香港眾志發表聲明，證實周庭接獲選舉主任[鄧如欣通知](https://zh.wikipedia.org/wiki/鄧如欣 "wikilink")，她的選舉提名無效。選舉主任是以周庭填報其政治聯繫是[香港眾志](../Page/香港眾志.md "wikilink")，而香港眾志以「民主自決」為最高綱領，在《香港眾志：民主自決 此際起航—我們的自決運動路線圖》\[31\] 一文，提及「香港眾志主張透過具憲制效力的前途公投，由香港人共同認受香港主權和憲制。即使香港眾志並不提倡港獨，但為着體現『主權在民』的理念，我們同意公投應該包括獨立和地方自治等選項」。選舉主任指以上顯示周庭沒有真心及真誠擁護《基本法》及效忠香港特區\[32\]\[33\]。香港眾志發聲明，強烈譴責選舉主任作出政治決定，並引述選舉主任在解釋中，列出香港眾志的政綱，但並無如2016年般要求周庭或眾志解釋政治立場，批評這等同剝奪整個組織參選權\[34\]。同日[區諾軒](../Page/區諾軒.md "wikilink")遞交提名表格正式參選港島區補選\[35\]。

2018年1月28日，周庭在《城市論壇》稱香港有言論自由，有權利討論和對《基本法》條文持不同的意見，政府是「思想審查、以言入罪」，禠奪基本政治權利。周庭其後在政府總部外舉行集會時批評今次的「政治清算」目標不止是香港眾志，而是很多香港人的自由意志和選擇權\[36\]。

2018年1月29日，下午2時30分，由於[姚松炎](../Page/姚松炎.md "wikilink")參選資格未得到確認，替補人選[袁海文](../Page/袁海文.md "wikilink")報名參選。下午4時10分，[姚松炎](../Page/姚松炎.md "wikilink")獲口頭確認參選資格，4時14分再獲電郵通知提名有效，[袁海文](../Page/袁海文.md "wikilink")隨即遞交退選通知書\[37\]。

2018年1月31日，[社區網絡聯盟](../Page/社區網絡聯盟.md "wikilink")[劉頴匡](../Page/劉頴匡.md "wikilink")證實被取消參選資格，選舉主任陳婉雯在通知書上指，劉在2016年曾在[Facebook](../Page/Facebook.md "wikilink")上發表支持[香港獨立的言論](https://zh.wikipedia.org/wiki/香港獨立 "wikilink")；而且於2018年1月2日，劉曾指新界東議席不單屬於[梁頌恆](../Page/梁頌恆.md "wikilink")或[梁天琦](../Page/梁天琦.md "wikilink")的，而是屬於所有認同[本土派理念的選民](https://zh.wikipedia.org/wiki/本土派_\(香港\) "wikilink")，陳認為劉用到「本土派」的字眼，並堅持要接替或取代支持香港獨立的[梁頌恆](../Page/梁頌恆.md "wikilink")及[梁天琦](../Page/梁天琦.md "wikilink")，並獲梁頌恆公開支持。對劉於2017年12月在其[面書聲稱](https://zh.wikipedia.org/wiki/面書 "wikilink")「「基於政治現實，本人現正宣佈，本人現時已不再支持香港獨立，並全心全意擁護及熱愛基本法」，選舉主任認為劉這樣聲明純粹因為「政治現實」「會做所有能令自己進入議會的事情」，而且「實際上是承認了（至少在該陳述前）他一直支持香港獨立」\[38\]\[39\]。

2018年2月1日，報名參加新界東補選的沙田區議員[陳國強](../Page/陳國強_\(區議員\).md "wikilink")，被裁定提名無效。選舉主任陳婉雯在通知書提到，陳國強在2016年曾以講者身分參加由[香港民族黨](../Page/香港民族黨.md "wikilink")舉辦的公眾集會，發表有關「香港獨立」的言論；而且，陳國強於2016年發給選舉主任的信件中，清晰陳述主張和推動香港獨立的立場，他其後的行為亦表明支持「港獨」。雖然陳國強於2017年11月起在面書多次表示他不是「港獨派」，表示「香港獨立」是「獨立於共產統治」，但選舉主任認為他「與之前的言行矛盾」，並認為他「以片面之詞宣稱意向改變」只為了參加補選，並不是真心，所以不接納陳國強真心擁護《基本法》，裁定他提名無效\[40\]\[41\]。

### 無綫電視首度拒辦選舉論壇

網媒[眾新聞](../Page/眾新聞.md "wikilink")在2月9日報道[無綫新聞將不會就是次立法會補選舉辦論壇](https://zh.wikipedia.org/wiki/無綫新聞 "wikilink")，有別於往年做法，反觀其他主要電子傳媒如[有線](../Page/香港有線新聞.md "wikilink")、[now](https://zh.wikipedia.org/wiki/now寬頻電視新聞 "wikilink")、[香港電台](../Page/香港電台.md "wikilink")及[香港商業電台](../Page/香港商業電台.md "wikilink")已計劃舉辦論壇。消息指新聞部主管[袁志偉認為新界東的參選人質素參差](https://zh.wikipedia.org/wiki/袁志偉 "wikilink")，會影響節目效果，但選舉條例要求每名候選人有「平等時間」發言，故決定不舉辦選舉論壇。袁志偉回覆時僅指無必要向外解釋採訪政策，一眾候選人均感不滿，新同盟范國威批評無綫不負責任；方國珊亦指此舉對所有候選人都有損失\[42\]。

**2018年度[亞洲小姐競選香港區決賽](https://zh.wikipedia.org/wiki/亞洲小姐競選 "wikilink")**是[亞洲電視數碼媒體](../Page/亞洲電視數碼媒體.md "wikilink")於2018年10月7日舉行的[選美活動](../Page/選美活動.md "wikilink")，並於晚上八時三十分至十時正（均為香港時間）透過[亞視A1台及奇妙電視中文台](../Page/A1台.md "wikilink")（現稱「[香港開電視](../Page/香港開電視.md "wikilink")」）播出。

## 決賽日節目流程

|      |                                                                                                                                                                      |
| ---- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 司儀   | [李志剛](../Page/李志剛_\(香港\).md "wikilink")、[朱慧敏](../Page/朱慧敏.md "wikilink")、[丘雅雯](https://zh.wikipedia.org/wiki/丘雅雯 "wikilink")                                         |
| 評判   | [陳志雲](../Page/陳志雲.md "wikilink")、[顧紀筠](../Page/顧紀筠.md "wikilink")、[陳欣健](../Page/陳欣健.md "wikilink")、[梁佩瑚](../Page/梁佩瑚.md "wikilink")、[陳嘉容](../Page/陳嘉容.md "wikilink") |
| 表演嘉賓 | [李幸倪](../Page/李幸倪.md "wikilink")                                                                                                                                     |

  - 第一節：
      - 由舞蹈員及李幸倪首先踏入舞台，其後十位佳麗於李幸倪歌曲《曌》的歌聲中以泳裝進場。
      - 三位司儀宣布「人氣大獎」得主，並由李幸倪頒發獎項及介紹評判並進行訪問。
  - 第二節：
      - 十位佳麗先進行歌舞表演
      - 由司儀丘雅雯訪問大會排舞師Sunny Wong對佳麗表現的意見
      - 再由另外兩位評判李志剛、朱慧敏訪問五位評判的意見
      - 司儀李志剛宣佈得分最高的五位佳麗為梁雪瑤、張右雨、蘇頌筠、傅美霖、林孜沫
  - 第三節：
      - 由評判向1至5號佳麗作出提問
  - 第四節：
      - 由評判向6至10號佳麗作出提問
  - 第五節：
      - 由李幸倪唱出歌曲《侘寂》及《很堅強》，十位佳麗再次登場
      - 於表演後，再由丘雅雯於後台訪問其中兩位佳麗張右雨及李芷菱
  - 第六節：
      - 由司儀介紹2018年亞洲小姐競選總決賽位於馬來西亞馬六甲的舉辦場地並播映當地宣傳片段
      - 司儀公佈最終結果，並依次由評判梁佩瑚、顧紀筠、陳嘉容分別頒發最上鏡小姐、季軍及亞軍獎項
      - 司儀請出2017年度亞洲小姐競選總冠軍何丹妮，並由何丹妮頒發冠軍獎項
      - 冠軍佳麗於得獎繞場一周，再接受其他佳麗及評判祝賀致意，而司儀亦就「大中華區決賽」及「總決賽」作出預告

## 入圍佳麗個人背景

  - 1號賴嘉瑩：曾為香港排球隊青年軍成員，亦曾參與「環亞美神大賽」。
  - 2號梁雪瑤：曾參選[香港小姐競選並獲面試](https://zh.wikipedia.org/wiki/香港小姐競選 "wikilink")，於談及參選亞洲小姐的原因時，就曾表示參選期間能到韓國跟星級導師學走貓步和跳舞，而感到十分吸引，也是十分好的體驗，而自己也十分希望能在選美比賽中勝出及成為藝人。\[1\]
  - 3號張靜茵：曾參選[香港小姐競選](https://zh.wikipedia.org/wiki/香港小姐競選 "wikilink")，並獲面試。
  - 5號蘇頌筠：曾參選[香港小姐競選](https://zh.wikipedia.org/wiki/香港小姐競選 "wikilink")，並獲面試。
  - 9號李芷菱：曾參選[2018年度香港小姐競選](../Page/2018年度香港小姐競選.md "wikilink")，並獲晉身第二輪面試，她曾表示自己自小就有當演員的夢想，希望能透過不同的渠道可以去實踐夢想。

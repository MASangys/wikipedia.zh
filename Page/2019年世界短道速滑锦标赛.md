**2019年世界短道速滑锦标赛**是第44届[世界短道速滑锦标赛](../Page/世界短道速滑锦标赛.md "wikilink")，于2019年3月8日至10日在保加利亚索菲亚举行，这是索菲亚第三次举办短道速滑世锦赛。\[1\]

## 赛程

所有时间均为[欧洲中部时间](../Page/欧洲中部时间.md "wikilink")（[UTC+2](https://zh.wikipedia.org/wiki/UTC+02:00 "wikilink")）：\[2\]

| 日期        | 时间    | 项目      |
| --------- | ----- | ------- |
| 3月8日      | 10:00 | 资格赛     |
| 3月9日      | 14:02 | 女子1500米 |
| 男子1500米   |       |         |
| 女子500米    |       |         |
| 男子500米    |       |         |
| 3月10日     | 14:02 | 女子1000米 |
| 男子1000米   |       |         |
| 女子3000米   |       |         |
| 男子3000米   |       |         |
| 女子3000米接力 |       |         |
| 男子5000米接力 |       |         |

**2019年世界羽聯世界巡迴賽總決賽**，是第2屆[世界羽聯世界巡迴賽總決賽](../Page/世界羽聯世界巡迴賽總決賽.md "wikilink")，在[2019年世界羽聯世界巡迴賽](../Page/2019年世界羽聯世界巡迴賽.md "wikilink")全部比賽完結後舉行，屬於世界羽聯第二等別第一級賽事。本屆賽事將於2019年12月11日至12月15日在[中國](../Page/中國.md "wikilink")[廣州的](https://zh.wikipedia.org/wiki/廣州 "wikilink")[天河體育中心舉行](https://zh.wikipedia.org/wiki/天河體育中心 "wikilink")，總獎金150萬美元。本屆賽事是[世界羽聯超級系列賽總決賽的延續](https://zh.wikipedia.org/wiki/世界羽聯超級系列賽總決賽 "wikilink")，沿用相同的參賽條件和比賽方式，只邀請各單項年末在[世界羽聯世界巡迴賽](../Page/世界羽聯世界巡迴賽.md "wikilink")排名前八名的選手參加，但每個協會最多兩個席位，而且獲得當年度奧運或世界錦標賽冠軍者直接入選。\[1\]

## 賽事資料

### 賽事安排

賽事包括：男子單打、女子單打、男子雙打、女子雙打及混合雙打五個項目，賽制則分為小組單循環賽及單淘汰賽兩部分\[2\]。

### 比賽日程

大會公布的日程如下\[3\]：

| 賽日  | 日期          | 時間            | 項目  |
| --- | ----------- | ------------- | --- |
| 第1日 | 2019年12月11日 | 11:00 / 18:00 | 小組賽 |
| 第2日 | 2019年12月12日 | 11:00 / 18:00 | 小組賽 |
| 第3日 | 2019年12月13日 | 11:00 / 18:00 | 小組賽 |
| 第4日 | 2019年12月14日 | 11:00 / 18:00 | 準決賽 |
| 第5日 | 2019年12月15日 | 13:00         | 決賽  |

### 積分配置

依據[BWF積分排名系統中的](https://zh.wikipedia.org/wiki/世界羽聯世界排名#2018-2021年賽制 "wikilink")[世界巡迴賽總決賽級別賽事點數分配](../Page/世界羽聯世界巡迴賽.md "wikilink")，本賽事各名次積分配置如下表：

| 冠軍     | 亞軍     | 4強    | 8強    | 16強   |
| ------ | ------ | ----- | ----- | ----- |
| 12,000 | 10,200 | 8,400 | 6,600 | 4,800 |

  - 小組賽第四名，採計16強積分；小組賽第三名採計8強積分。

### 獎金安排

本屆賽事的總獎金為[美金](https://zh.wikipedia.org/wiki/美金 "wikilink")1,500,000元。按[世界羽聯的指引](https://zh.wikipedia.org/wiki/世界羽聯 "wikilink")，獎金分布的安排如下\[4\]：

| 項目   | 獎金（按美元計算） |
| ---- | --------- |
| 冠軍   | 亞軍        |
| 單打項目 | $120,000  |
| 雙打項目 | $126,000  |

**2020年国际汽联世界一级方程式锦标赛**（英语：2020 FIA Formula One World Championship）为[一级方程式赛车的第](https://zh.wikipedia.org/wiki/一級方程式賽車 "wikilink")71届[世界锦标赛赛事](https://zh.wikipedia.org/wiki/一級方程式賽車 "wikilink")，这是由国际[动力运动主管组织](../Page/動力運動.md "wikilink")[国际汽车联盟所主办的最高级别](https://zh.wikipedia.org/wiki/國際汽車聯盟 "wikilink")[开轮式赛车赛事](../Page/開輪式賽車.md "wikilink")。

本屆一級方程式錦標賽的賽程表是有以來最緊湊的，共包含有22場大獎賽賽事，為這項運動歷史上賽事最多、時間最長的一個賽季，且當中有7個、共14場賽事為「背靠背」(Back-to-Back)雙週賽事，為F1擁有最多雙週賽事的賽曆。

### 车队变更

  - [红牛第二车队宣布将车队名称将从Toro](https://zh.wikipedia.org/wiki/红牛第二车队 "wikilink") Rosso更名为红牛旗下的服装品牌。\[1\]
  - [阿尔法罗密欧车队在](https://zh.wikipedia.org/wiki/阿尔法罗密欧车队 "wikilink")2020年1月1日宣布与波兰燃油巨头签下合约，后者将在2020赛季起作为车队的冠名赞助商，车队将更名为Alfa Romeo Racing ORLEN。

### 车手变更

  - 雷诺车队宣布[埃斯特班·奥康将顶替](https://zh.wikipedia.org/wiki/埃斯特班·奥康 "wikilink")[尼科·霍肯伯格搭档](https://zh.wikipedia.org/wiki/尼可·賀根堡 "wikilink")[丹尼尔·里卡多参加](https://zh.wikipedia.org/wiki/丹尼尔·里卡多 "wikilink")2020赛季比赛。\[2\]
  - [罗伯特·库比卡](../Page/罗伯特·库比卡.md "wikilink")宣布2019年赛季结束后离开威廉姆斯车队。\[3\]之后在冬休期他被[阿尔法罗密欧车队选为预备车手](https://zh.wikipedia.org/wiki/阿尔法罗密欧车队 "wikilink")。
  - 加拿大车手[尼古拉斯·拉提菲从](https://zh.wikipedia.org/wiki/尼古拉斯·拉提菲 "wikilink")[二级方程式的DAMS车队加入威廉姆斯车队](https://zh.wikipedia.org/wiki/二级方程式 "wikilink")，担任主力车手。

|    |        |        |        |        |        |        |        |        |        |        |

正赛中拿到最快圈速且在前十名完赛的车手将获得一个积分。

夺冠车手必须完成75%的赛程，方能获得全额积分。若夺冠车手未完成75%的赛程，但已跑完至少两圈，则积分以一半计之。假使积分排名出现同分状况时，将比较车手获得的分站冠军数，决定最终的排行榜名次。

**2021年国际汽联世界一级方程式锦标赛**（英语：2021 FIA Formula One World Championship）为[一级方程式赛车的第](https://zh.wikipedia.org/wiki/一級方程式賽車 "wikilink")72届[世界锦标赛赛事](https://zh.wikipedia.org/wiki/一級方程式賽車 "wikilink")，这是由国际[动力运动主管组织](../Page/動力運動.md "wikilink")[国际汽车联盟所主办的最高级别](https://zh.wikipedia.org/wiki/國際汽車聯盟 "wikilink")[开轮式赛车赛事](../Page/開輪式賽車.md "wikilink")。

本屆一級方程式錦標賽的賽程是自混动时代以来，第一次大规模的规则修改。而且地面效应（文丘里效应）自被禁之后首次重新允许赛车使用，以减小下压力损失及跟车乱流影响。

### 积分系统

| 名次 | **冠軍** | **亞軍** | **季軍** | **第四** | **第五** | **第六** | **第七** | **第八** | **第九** | **第十** |
| -- | ------ | ------ | ------ | ------ | ------ | ------ | ------ | ------ | ------ | ------ |
| 积分 | 25     | 18     | 15     | 12     | 10     | 8      | 6      | 4      | 2      | 1      |
|    |        |        |        |        |        |        |        |        |        |        |

正赛中拿到最快圈速且在前十名完赛的车手将获得一个积分。

夺冠车手必须完成75%的赛程，方能获得全额积分。若夺冠车手未完成75%的赛程，但已跑完至少两圈，则积分以一半计之。假使积分排名出现同分状况时，将比较车手获得的分站冠军数，决定最终的排行榜名次。

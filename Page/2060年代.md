**2060年代**从2060年1月1日开始，到2069年12月31日结束。

## <span data-segmentid="99" class="cx-segment">时间囊</span>

  - 唐哈靈頓探索中心（Don Harrington Discovery
    Center）的發現<span data-segmentid="106" class="cx-segment">氦百週年纪念碑（Helium
    Centennial Time Columns Monument
    ）预计将在1968年起被锁定後過了100年再次打開時間囊。</span>

## <span data-segmentid="108" class="cx-segment">预测</span>

### <span data-segmentid="109" class="cx-segment">2060年</span>

<span data-segmentid="110" class="cx-segment">根据[国际能源机构](https://zh.wikipedia.org/wiki/国际能源机构 "wikilink")（IEA）的预测，全球三分之一的能源可能是[太阳能](https://zh.wikipedia.org/wiki/太阳能 "wikilink")。\[1\]</span>

### <span data-segmentid="113" class="cx-segment">2061年</span>

  - <span data-segmentid="114" class="cx-segment">商业开采[月球元素在经济上是可行的](../Page/月球.md "wikilink")。\[2\]</span>
  - <span data-segmentid="115" class="cx-segment">预计新加坡與馬來西亞水資源協定将到期。</span>
  - <span data-segmentid="117" class="cx-segment">7月28日—[哈雷彗星到达](../Page/哈雷彗星.md "wikilink")[近日点](https://zh.wikipedia.org/wiki/近日点 "wikilink")，也就是离太阳最近的点，上一次到达近日点是在1986年2月9日。</span>

### <span data-segmentid="120" class="cx-segment">2065年</span>

  - <span data-segmentid="121" class="cx-segment">11月11日-[水星凌日](https://zh.wikipedia.org/wiki/水星凌日 "wikilink")</span>
  - <span data-segmentid="123" class="cx-segment">[世界协调时的](https://zh.wikipedia.org/wiki/世界协调时 "wikilink")11月22日12时45分[金星将隐匿](../Page/金星.md "wikilink")[木星](../Page/木星.md "wikilink")。
    </span><span data-segmentid="127" class="cx-segment">从地球上观测将非常困难，因为此时金星和木星与[太阳的距离将只有](../Page/太阳.md "wikilink")7度。
    </span><span data-segmentid="129" class="cx-segment">这是自1818年1月3日</span><span data-segmentid="127" class="cx-segment">後</span><span data-segmentid="129" class="cx-segment">又一次一颗[行星被另一颗行星](../Page/行星.md "wikilink")[掩星](https://zh.wikipedia.org/wiki/掩星 "wikilink")，然而下一次掩星将发生在不到两年后的2067年7月15日。</span>

### <span data-segmentid="130" class="cx-segment">2066年</span>

<span data-segmentid="131" class="cx-segment">[英国预计至少有](https://zh.wikipedia.org/wiki/英国 "wikilink")50万人年龄超过100岁。\[3\]</span>

### <span data-segmentid="132" class="cx-segment">2067年</span>

  - <span data-segmentid="133" class="cx-segment">2月15日—
    假定在此之前[美国版权法中的版权擁有期不再延长](../Page/美国版权法.md "wikilink")，所有1972年2月15日之前的录音製品都将进入美国[公共领域](../Page/公共领域.md "wikilink")。\[4\]</span>
  - <span data-segmentid="136" class="cx-segment">世界协调时的7月15日11时56分[水星将隐匿于](../Page/水星.md "wikilink")[海王星](../Page/海王星.md "wikilink")。
    </span><span data-segmentid="139" class="cx-segment">这种罕见的现象在地球上很难观测到。</span>
  - <span data-segmentid="140" class="cx-segment">10月—[主动搜寻地外文明计划在](../Page/主动搜寻地外文明计划.md "wikilink")[烏克蘭](https://zh.wikipedia.org/wiki/烏克蘭 "wikilink")[葉夫帕托里亞通過](../Page/叶夫帕托里亚.md "wikilink")70米碟形天線發送的[宇宙的呼喚訊號將到它的目的地](../Page/宇宙的呼喚.md "wikilink")，恒星HD
    178428。</span>

### <span data-segmentid="145" class="cx-segment">2068年</span>

  - 發現<span data-segmentid="106" class="cx-segment">氦百週年纪念碑的時間囊在1968年被锁定後過了100年再次打開。</span>
  - <span data-segmentid="148" class="cx-segment">据[未来学家大卫](https://zh.wikipedia.org/wiki/未来学家 "wikilink")·帕斯克（David
    Passig）预测，到2068年将会有一个海底城市。</span>

### <span data-segmentid="150" class="cx-segment">2069年</span>

<span data-segmentid="151" class="cx-segment">11月—1999年<span data-segmentid="140" class="cx-segment">[主动搜寻地外文明计划在](../Page/主动搜寻地外文明计划.md "wikilink")[烏克蘭](https://zh.wikipedia.org/wiki/烏克蘭 "wikilink")[葉夫帕托里亞通過](../Page/叶夫帕托里亚.md "wikilink")70米碟形天線發送的[宇宙的呼喚訊號將到它的目的地](../Page/宇宙的呼喚.md "wikilink")</span>，[奚仲四三合星系統](../Page/奚仲四.md "wikilink")。</span>

## <span data-segmentid="156" class="cx-segment">宗教和精神上的預测</span>

### <span data-segmentid="157" class="cx-segment">2060年</span>

<span data-segmentid="158" class="cx-segment">[艾萨克·牛顿预言](../Page/艾萨克·牛顿.md "wikilink")，根据他对[圣经的解释](https://zh.wikipedia.org/wiki/圣经 "wikilink")，世界将在2060年之前结束。\[5\]\[6\]</span>

### <span data-segmentid="161" class="cx-segment">2062年</span>

<span data-segmentid="162" class="cx-segment">在[新纪元运动中有影响力的](../Page/新纪元运动.md "wikilink")[占星家Dane](https://zh.wikipedia.org/wiki/占星家 "wikilink")
Rudhyar在1972年预言，[宝瓶座年代将在公元](https://zh.wikipedia.org/wiki/宝瓶座年代 "wikilink")2062年开始。\[7\]</span>

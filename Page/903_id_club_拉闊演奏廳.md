《**903 id club
拉闊演奏廳**》是香港歌手[張敬軒和](https://zh.wikipedia.org/wiki/張敬軒 "wikilink")[王菀之的](https://zh.wikipedia.org/wiki/王菀之 "wikilink")[現場專輯](../Page/現場專輯.md "wikilink")，收錄他們於2006年7月18日在[九龍灣國際展貿中心演唱的歌曲](../Page/九龍灣國際展貿中心.md "wikilink")，由[環球唱片先於同年](https://zh.wikipedia.org/wiki/環球唱片 "wikilink")9月28日發行[卡拉OK](../Page/卡拉OK.md "wikilink")
[VCD版本](https://zh.wikipedia.org/wiki/影音光碟 "wikilink")。\[1\]\[2\]然後，於同年10月19日發行卡拉OK
[DVD版本](https://zh.wikipedia.org/wiki/DVD "wikilink")。\[3\]

2015年11月20日，環球唱片發行《903 id club
拉闊演奏廳》的CD版本作為「簡約再生」系列的第四回產品之一。\[4\]\[5\]\[6\]

## 曲目

出自《903 id club 拉闊演奏廳》專輯內頁。\[7\]

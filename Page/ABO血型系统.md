**ABO血型系统**是[人类最早认识也是最为重要的](../Page/人类.md "wikilink")[血型系统](../Page/血型.md "wikilink")。ABO血型由[红细胞膜上的不同](../Page/红细胞.md "wikilink")[抗原所决定](../Page/抗原.md "wikilink")，与人类[输血时发生的](../Page/输血.md "wikilink")[溶血反应密切相关](../Page/溶血反应.md "wikilink")，具有重要的[临床意义](../Page/临床医学.md "wikilink")。ABO抗原也存在于[牛](../Page/牛.md "wikilink")、[羊和一些](../Page/羊.md "wikilink")[猿类等其他动物体内](../Page/猿.md "wikilink")\[1\]。ABO血型可能繼承自數百萬年前一個共同的靈長類祖先，並在傳染病免疫反應具關鍵作用\[2\]。

## 发现史

1900年，[奥地利](../Page/奥地利.md "wikilink")[维也纳大学病理研究所的生物学家](../Page/维也纳大学.md "wikilink")[卡尔·兰德施泰纳首次报道](../Page/卡尔·兰德施泰纳.md "wikilink")，健康人的[血清对一些人类个体的红细胞有凝聚作用](../Page/血清.md "wikilink")。通过混合不同人的血清和红细胞，他发现了A、B、O三种血型\[3\]，他的学生Decastrello和Sturli又于两年后发现了第四种——AB型\[4\]。兰德施泰纳因此获得1930年度[诺贝尔生理学或医学奖](../Page/诺贝尔生理学或医学奖.md "wikilink")。[捷克血清学家](../Page/捷克.md "wikilink")也于1907年独立发现人类血液可分为四种血型\[5\]。但由于当时[通讯不便](../Page/通讯.md "wikilink")，只有兰德施泰纳的成就得到科学界广泛承认，扬斯基则几乎被遗忘，仅在[前苏联等少数国家仍有提及](../Page/前苏联.md "wikilink")。

1910至1911年，[波兰人](../Page/波兰人.md "wikilink")[希尔斯菲尔德和](../Page/希尔斯菲尔德.md "wikilink")[德国人](../Page/德国人.md "wikilink")发现了ABO血型系统的遗传性。1924年，德国数学家[费利克斯·伯恩斯坦进一步阐释了ABO的遗传方式是由同一](../Page/费利克斯·伯恩斯坦.md "wikilink")[基因座上的几个](../Page/基因座.md "wikilink")[等位基因决定的](../Page/等位基因.md "wikilink")\[6\]。[英国人沃特金斯](../Page/英国人.md "wikilink")（Watkins）和摩尔根（Morgan）发现，决定ABO抗原不同的分子基团（称为[抗原决定簇或](../Page/抗原决定簇.md "wikilink")[表位](../Page/表位.md "wikilink")，英文：**epitope**）由[糖类转化而来](../Page/糖类.md "wikilink")，决定A抗原的是N-乙酰半乳糖胺（N-acetylgalactosamine），决定B抗原的是[半乳糖](../Page/半乳糖.md "wikilink")（galactose）\[7\]。1990年，[日本学者山本等首次描述了编码这些抗原决定簇的A](../Page/日本.md "wikilink")、B、O等位基因的DNA结构\[8\]。

## ABO抗原

[ABO_blood_group_diagram_zh.svg](https://zh.wikipedia.org/wiki/File:ABO_blood_group_diagram_zh.svg "fig:ABO_blood_group_diagram_zh.svg")

人类ABO血型系统的抗原合成基于前体物质**H抗原**。H抗原是一种[糖脂](../Page/糖脂.md "wikilink")，基本分子结构是以[糖苷键与多肽链骨架结合的四糖链](../Page/糖苷键.md "wikilink")，即β-D-[半乳糖](../Page/半乳糖.md "wikilink")、β-D-[N-乙酰葡萄糖胺](../Page/N-乙酰葡萄糖胺.md "wikilink")、β-D-半乳糖以及在β-D-半乳糖2-位连接的抗原决定簇α-L-[岩藻糖](../Page/岩藻糖.md "wikilink")。H抗原的决定基因*FUT1*位于[19号染色体](../Page/19号染色体.md "wikilink")，长度超过5000个碱基对，包括3个[外显子](../Page/外显子.md "wikilink")。*FUT1*基因有两个等位基因*H*和*h*，*H*等位基因编码**[岩藻糖转移酶](../Page/岩藻糖转移酶.md "wikilink")**，使岩藻糖与糖链末端的半乳糖相连，形成H抗原。而*h*等位基因无法编码具有活性的岩藻糖转移酶。*hh*纯合子个体在人类中非常罕见，形成所谓[孟买血型](../Page/孟买血型.md "wikilink")。参见[Hh/孟买血型系统](../Page/Hh/孟买血型系统.md "wikilink")。

编码ABO抗原决定簇的是[9号染色体上的](../Page/9號染色體_\(人類\).md "wikilink")*ABO*基因，总长度18000至20000个碱基对，包括7个外显子，其中最大的第7外显子和第6外显子的碱基数占整个编码序列的77%\[9\]。*ABO*基因有三个最主要的等位基因：*I<sup>A</sup>*（*A*）、*I<sup>B</sup>*（*B*）和*i*（*O*），这些等位基因的原初产物是糖基转移酶。*I<sup>A</sup>*等位基因编码α-1,3N-乙酰氨基半乳糖转移酶，能将α-N-乙酰半乳糖胺接到H抗原的β-D-半乳糖上，形成**A抗原**；*I<sup>B</sup>*等位基因编码α-1,3-D-半乳糖转移酶，将α-D-半乳糖接到H抗原的相同位置，形成**B抗原**；*i*等位基因的第6外显子包含一个[核苷酸缺失](../Page/核苷酸.md "wikilink")，导致其编码的蛋白质无法正常表达，从而失去酶活性，因此，O型血的抗原就是未经改变的H抗原。

A、B抗原主要表达在红细胞膜上，以糖蛋白或糖脂的形式，呈树枝状突出细胞表面。在大多数[上皮细胞](../Page/上皮细胞.md "wikilink")、[内皮细胞以及](../Page/内皮细胞.md "wikilink")[体液中](../Page/体液.md "wikilink")，也有A、B抗原的存在。

### 亚型

1930年，人们发现A抗原可以分为A1与A2两个亚型。目前已知，A型至少包括20个亚型，绝大多数是A1和A2（\>99%），其中A1的比例超过80%\[10\]。
1980年以前，人们通常认为红细胞的ABH决定簇主要存在于鞘糖脂分子上，但是，1980年发表的几篇报道指出：糖脂与糖蛋白相比，在ABH抗原决定簇中仅扮演次要角色。红细胞上的多数ABH抗原都位于离子交换蛋、带3蛋白、葡萄糖转运蛋白以及带4.5蛋白的单个高度分枝的多聚N-乙酰氨基乳糖的N-型糖苷键上。

## 血清学

[ABO_blood_type_zh.svg](https://zh.wikipedia.org/wiki/File:ABO_blood_type_zh.svg "fig:ABO_blood_type_zh.svg")

根据红细胞膜上A抗原与B抗原的有无，可以将血液分为4型：如果只存在A抗原，则称为**A型**；只存在B抗原，则称为**B型**；若A与B两种抗原都存在，则称为**AB型**；这两种抗原都没有的，则称为**O型**。

不同血型的血液里可能存在**抗-A**或**抗-B**抗体（也称血凝集素，见右图）。这些抗体通常是[免疫球蛋白](../Page/免疫球蛋白.md "wikilink")**IgM**，无法通过[胎盘从母体进入胎儿血液](../Page/胎盘.md "wikilink")，但会在出生后一年内出现在人体内，产生原因可能是环境中某些与A、B抗原决定簇类似的糖类的刺激。这些抗体遇到与本身血型不合的红细胞时，就会产生溶血反应，例如一个A血型的人如果输入B型血，会立刻导致体内的抗-B抗体与B型红细胞的表面B抗原结合，引发[补体介导的红细胞自溶](../Page/补体.md "wikilink")。相同的抗原-抗体反应也会在B血型的人输入A型血或O型血的人输入其他血型的血液时发生。只有AB血型的人体内没有抗A和抗B，可以接受其他血型的血液，故曾被称为“万能受血者”；而各种血型的人均可以输入O型血，因此O型血的人曾被称为“万能供血者”（见下表）。然而，这种“万能”的说法并不严格。以A血型的受血者输入O型血为例，O型血的红细胞不含B抗原，不会与受血者体内的抗-B反应，但O型血中的抗-A会与受血者血液中带有A抗原的红细胞反应。当输血量较小时，输入的抗-A抗体被受血者的血清充分稀释，其影响可以降低；但如果输血量较大，抗-A无法得到充分稀释，仍可能引发受血者的溶血反应。在這種情況下，一個解決之道是將送血者血液中的抗體分離，否則不应进行异型输血。

| 受血者 | 供血者       |
| --- | --------- |
| A型  | A或O型      |
| B型  | B或O型      |
|     |           |
| AB型 | A、B、AB或O型 |
| O型  | O型        |

## 遗传性

ABO血型系统的遗传是单基因决定的。*ABO*基因位于[9号染色体的长臂上](../Page/9號染色體_\(人類\).md "wikilink")（9q34），有三个主要的等位基因*I<sup>A</sup>*（A）、*I<sup>B</sup>*（B）和*i*（O）。*I<sup>A</sup>*和*I<sup>B</sup>*的产物是具有活性的酶，分别催化A抗原和B抗原的合成，而等位基因*i*的产物不具有酶活性，无法催化A、B抗原合成。由于人类染色体是[双倍体](../Page/双倍体.md "wikilink")，一个人通常只能拥有三个等位基因中的两个，分别来自父母双方。这两个等位基因的类型，即血型的[基因型](../Page/基因型.md "wikilink")（genotype），决定了人类血型的[表现型](../Page/表现型.md "wikilink")（phenotype）。

*I<sup>A</sup>*和*I<sup>B</sup>*对*i*均为显性，故而只有基因型是*ii*的人才有O型血，基因型是*I<sup>A</sup>I<sup>A</sup>*或*I<sup>A</sup>i*的人是A血型，基因型是*I<sup>B</sup>I<sup>B</sup>*或*I<sup>B</sup>i*的人是B血型。而*I<sup>A</sup>*和*I<sup>B</sup>*是[共显性](../Page/共显性.md "wikilink")，因此基因型是*I<sup>A</sup>I<sup>B</sup>*的人具有两种表型，即AB血型。

一般说来，如果父母双方均为O型血，子女必然是O型；如果父母有一方是AB型血，子女不可能是O型；A型和O型血的父母不可能生育B型或AB型血的子女，B型和O型血的父母不可能生育A型或AB型血的子女。因此，ABO血型系统曾经广泛用于[亲子鉴定](../Page/亲子鉴定.md "wikilink")。父母与子女血型的详细关系及子女血型的可能性见下表：

### 异常型

#### *cis*-*AB*

极少数人类个体的一条染色体中同时存在一个***cis*-*AB***基因，它编码的酶可以同时催化A抗原和B抗原的合成。*cis*-*AB*是从一个亲本遗传而来的，因此即便其基因型是*cisAB*
/ *O*，也会表现为AB型，若与O型血的人（基因型*O* /
*O*）生育，子女的血型可能是AB或O，而不像通常AB型血与O型血的父母只能有A或B型血的子女。

## ABO血型在人群中的分布

随着地域、人群的不同，ABO各个血型的频率也有很大区别，常作为研究人类的[民族分化](../Page/民族.md "wikilink")、迁徙史的工具。

### 世界人群

从世界角度来看，*I<sup>B</sup>*等位基因的频率最低，约为16%，在[中亚一带的阿爾泰民族最常见](../Page/中亚.md "wikilink")，而在美洲和[澳大利亚原住民中最为罕见](../Page/澳大利亚.md "wikilink")。*I<sup>A</sup>*等位基因较*I<sup>B</sup>*更为常见，约21%的世界人口携带*I<sup>A</sup>*，但相对高频率的区域比较分散，如北美印第安人黑脚族（30-35%），澳大利亚一些原住民族群（40-53%），[斯堪的納維亞地區北部的](../Page/斯堪的納維亞地區.md "wikilink")[拉普人](../Page/拉普人.md "wikilink")，或者[薩米人](../Page/薩米人.md "wikilink")（50－90%）等，而在中南美的印第安人中，*I<sup>A</sup>*几乎无分布。由于*I<sup>A</sup>*、*I<sup>B</sup>*两种等位基因型频率不高，O血型称为世界上最常見的血型，平均频率达63%，特别是在中南美洲印第安人中，O型血频率接近100%，O血型在澳大利亚原住民和[西歐人群](../Page/西歐.md "wikilink")（特别是[凯尔特人](../Page/凯尔特人.md "wikilink")）中也很常见，在[東歐和中亚的频率则相对较低](../Page/東歐.md "wikilink")。\[11\]
A型在北欧西欧与日本、湖南最多。

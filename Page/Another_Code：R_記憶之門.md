《**Another Code：R
記憶之門**》（，）是一個[Wii](../Page/Wii.md "wikilink")[視覺小說](https://zh.wikipedia.org/wiki/視覺小說 "wikilink")[冒险游戏](../Page/冒险游戏.md "wikilink")。此遊戲由[Cing](../Page/Cing.md "wikilink")開發，[任天堂](../Page/任天堂.md "wikilink")發佈，是《[Another
Code
兩段記憶](../Page/Another_Code_兩段記憶.md "wikilink")》的續作。此遊戲也是Cing首個開發的Wii遊戲。\[1\]

## 劇情

夢想成為音樂家的阿什利·罗宾斯在暑假期間收到遠方父親理查德的邀請。理查德因研究事宜而無法回家，而其研究對象正正是殺死阿什利母親的記憶控制系統，「Another」。為了修復父女之間的關係，理查德因此邀請女兒阿什利與他在朱麗葉湖共渡週末。可是，朱麗葉湖其實是一個被謎團重重包圍的地方，而這些謎團卻與13年前死去的阿什利母親有關。\[2\]

## 遊戲系統

此遊戲是以[視覺小說形式進行](https://zh.wikipedia.org/wiki/視覺小說 "wikilink")，故文字對話佔了遊戲的不少部份。玩家在遊戲中需要操控阿什利解決了一系列謎題，並尋找失去的記憶。阿什利需要與其他角色對話，與不同場境中的事物互動，解開不同的謎題，以獲得更多資訊，並找出謎團的真相。玩家需要使用Wii遙控器點擊事物來與之互動。在解謎時，玩家會得到便攜設備DAS的幫助去解決難題。\[3\]

## 角色

**阿什利·水木·羅賓斯**

  -
    遊戲的[主角](https://zh.wikipedia.org/wiki/主角 "wikilink")，一位16歲的女孩，夢想是成為一位音樂家。在過去的兩年中，她與父親住在一起。她對母親的記憶模糊。

**理查德·羅賓斯**

  -
    阿什利的父親，於JC山谷公司工作。他在妻子死後便留在血愛德華島上研究記憶控制系統長達七年。

**夜子·羅賓斯**

  -
    阿什利的母親，於研究記憶控制系統「Another」時死去。

**傑西卡·羅賓斯**

  -
    理查德的妹妹，高中化學老師，是她養育阿什利。

**丹·麥克斯韋**

  -
    於朱麗葉湖工作的勤奮護林員。

**馬修·克魯索**

  -
    一個離家出走的男孩，他正在尋找其失蹤的父親。

**瑞安·格雷**

  -
    JC山谷公司科學家。科學天才，理查德與夜子同事。

**神秘男子**

  -
    身穿黑色西裝和墨鏡的男子，正在調查朱麗葉湖的污染原因。

**格雷格·戴維斯**

  -
    一位研究茱麗葉湖過去的記者。\[4\]

## 發佈

此遊戲於2008年10月在[任天堂](../Page/任天堂.md "wikilink")秋季發布會上首次亮相。\[5\]於2009年2月5日，此遊戲在日本發佈。\[6\]於2009年6月26日，此遊戲在歐洲發佈。\[7\]與前作不同，此遊戲並沒有在北美及澳洲發佈。

## 評價

此遊戲獲得的評價不錯。[GameRankings得出此遊戲的分數為](https://zh.wikipedia.org/wiki/GameRankings "wikilink")69.21%。\[8\]
日本遊戲雜誌《[Fami通](../Page/Fami通.md "wikilink")》給予此遊戲28/40分。\[9\]而《[任天堂官方雜誌](https://zh.wikipedia.org/wiki/任天堂官方雜誌 "wikilink")》
則給予此遊戲84%的得分。[IGN](../Page/IGN.md "wikilink")給予此遊戲8/10分，並稱讚其畫質、音樂和遊戲系統。\[10\]而[Eurogamer](../Page/Eurogamer.md "wikilink")則給予此遊戲6/10分，稱讚其畫質，但就批評其遊戲費時。\[11\]

## 銷售

此遊戲在發佈後的第一個星期裡是日本第11暢銷遊戲，在該星期裡共售出了近15,000份遊戲。\[12\]此遊戲至今已售出了逾50,000份遊戲。\[13\]

**“Build（开发者大会）”**是由[微软举行的一年一度的开发者大会](../Page/微软.md "wikilink"),面向使用[Windows](../Page/Windows.md "wikilink"),[Windows
Phone](../Page/Windows_Phone.md "wikilink"),[Microsoft
Azure和其他微软技术的](../Page/Microsoft_Azure.md "wikilink")[软件开发人员和](../Page/软件开发.md "wikilink")[web开发人员](../Page/web开发人员.md "wikilink")。第一届大会在2011年举行,它继任了微软之前的开发者大会，[Professional
Developers
Conference](../Page/Professional_Developers_Conference.md "wikilink")(面向Windows操作系统的软件开发、[MIX](../Page/MIX.md "wikilink")、[web开发和微软技术如](../Page/web开发.md "wikilink")[Silverlight和](../Page/Silverlight.md "wikilink")[ASP.net](../Page/ASP.net.md "wikilink"))。

## 大会历届纪要

### Build 2011

Build2011于2011年9月13日至9月16日在[阿纳海姆
(加利福尼亚州)召开](../Page/阿纳海姆_\(加利福尼亚州\).md "wikilink")。这届大会主要侧重于[Windows
8](../Page/Windows_8.md "wikilink"),[Windows Server
2012和](../Page/Windows_Server_2012.md "wikilink")[Microsoft Visual
Studio](../Page/Microsoft_Visual_Studio.md "wikilink")2012；它们的开发者预览版也在大会上发布。与会者还收到了一个运行Windows
8“开发者预览版”的[三星](../Page/三星.md "wikilink")[平板电脑](../Page/平板电脑.md "wikilink")。

### Build 2012

Build 2012于2012年10月30日至11月2日在雷德蒙德的微软校园举行,侧重于最近发布的Windows 8,[Windows
Azure和](../Page/Windows_Azure.md "wikilink")[Windows Phone
8](../Page/Windows_Phone_8.md "wikilink")。与会者收到了附触控键盘的[Surface
RT平板电脑](../Page/Surface_RT.md "wikilink"),一部[诺基亚Lumia
920智能手机](../Page/诺基亚Lumia_920.md "wikilink"),和100GB的免费[OneDrive](../Page/OneDrive.md "wikilink")(当时称为SkyDrive)存储。\[1\]

### Build 2013

Build
2013于2013年6月26日至6月28日在[旧金山Moscone中心举行](../Page/旧金山.md "wikilink")。\[2\]这届大会主要是发布Windows
8的Windows 8.1更新。\[3\]\[4\]每一个与会者都能收到一个[Surface
Pro](../Page/Surface_Pro.md "wikilink"),附带[蓝牙键盘的](../Page/蓝牙.md "wikilink")[宏碁公司Iconia](../Page/宏碁公司.md "wikilink")
W3(第一个8英寸Windows 8平板电脑),一年期限的Adobe Creative
Cloud和100GB的存储[OneDrive](../Page/OneDrive.md "wikilink")(当时称为SkyDrive)。\[5\]

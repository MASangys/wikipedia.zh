**7類雙絞線**（英文：Category 7 cable），是一種雙絞線，為 Class
F纜線標準的非正式名稱，一般稱為CAT-7線，目前未被TIA/EIA承認。此標准定義4對各別遮蔽的雙絞線包覆在一個遮蔽內，是在万兆[以太网和其它向下兼容CAT](../Page/以太网.md "wikilink")-6A／CAT-6的[网络上所使用到的传输线材标准](https://zh.wikipedia.org/wiki/网络 "wikilink")。設計能提供大於100米的10Gbps[萬兆乙太網傳輸](https://zh.wikipedia.org/wiki/兆\(Giga\)乙太網 "wikilink")，可使用常用的[8P8C的](../Page/8P8C.md "wikilink")[連結器插頭](https://zh.wikipedia.org/wiki/連結器 "wikilink")。使用或的[連結器插頭作傳輸時](https://zh.wikipedia.org/wiki/連結器 "wikilink")，传输[頻寬宽度可达到](https://zh.wikipedia.org/wiki/頻寬 "wikilink")600MHz，为[CAT-5e的](https://zh.wikipedia.org/wiki/CAT-5#Category_5e "wikilink")6倍、[CAT-6的](../Page/CAT-6.md "wikilink")2.4倍。

## Class F<sub>A</sub>

'''Class F<sub>A</sub> '''（英文：Augmented Class
F），可稱為CAT-7a線，增強了雙絞線的傳輸能力，傳輸[頻寬可高達](https://zh.wikipedia.org/wiki/頻寬 "wikilink")1000
MHz。能分別於50米內和15米內，提供40Gbps[四萬兆乙太網和](https://zh.wikipedia.org/wiki/100吉比特乙太网 "wikilink")100Gbps[十萬兆乙太網傳輸](https://zh.wikipedia.org/wiki/100吉比特乙太网 "wikilink")。\[1\]\[2\]正式名稱為
ISO 11801 Amendment 1 (2008) 和 ISO 11801 Amendment 2
(2010)，目前未被TIA/EIA承認。

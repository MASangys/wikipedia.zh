《**CODE NAME BLUE**》是[韓國搖滾樂團](https://zh.wikipedia.org/wiki/韓國 "wikilink")[CNBLUE](../Page/CNBLUE.md "wikilink")的第3張[日語](https://zh.wikipedia.org/wiki/日語 "wikilink")[專輯](https://zh.wikipedia.org/wiki/專輯 "wikilink")，在2012年8月29日發行，也是他們首張由[華納唱片 (日本)發行的專輯](../Page/華納音樂集團.md "wikilink")\[1\]。

## 背景

這專輯的發行消息在2012年7月2日與他們第6張單曲《[Come on](../Page/Come_on_\(CNBLUE單曲\).md "wikilink")》的資料同時公開\[2\]。在2012年8月29日以3個不同版本：CD+DVD初回限定盤\[3\]、通常盤\[4\]及Lawson限定盤發行\[5\]。每個版本都使用不同的封面及含有獨有的內容：CD+DVD初回限定盤包括加收《[Where you are](../Page/Where_you_are_\(CNBLUE單曲\).md "wikilink")》英文版本的CD及收錄《Time is Over》[音樂錄影帶](../Page/音樂錄影帶.md "wikilink")與製作花絮的DVD、5張年曆卡及隨附卡序列碼版本D；Lawson限定盤包括了一片收錄基本曲目的CD及一片收錄在2012年2月5日舉行的《〈Where You Are〉 Release live at [MEETS PORT](../Page/MEETS_PORT.md "wikilink") 2012.2.5》音樂會片段及隨附卡序列碼版本E，只在[羅森便利店發售](../Page/羅森_\(便利店\).md "wikilink")；通常盤則只有一片收錄基本曲目的CD，首批版本才附有隨附卡序列碼版本E。

## 專輯內容

《CODE NAME BLUE》13首歌，3首單曲、3首歌曲曾在單曲中發行，另有7首新歌。大部分歌曲由CNBLUE成員[鄭容和](../Page/鄭容和.md "wikilink")及[李宗泫](../Page/李宗泫.md "wikilink")。容和創作了《[In My Head](../Page/In_My_Head_\(CNBLUE單曲\).md "wikilink")》、《Where You Are》（日語與英語版）、《Time is Over》、《Have a Good Night》、《Wake Up》 、《No More》、《Mr. KIA (Know It All)》及《Come On》；宗泫則創作了《No More》、《These Days》、《Get Away》及 《Come On》。

## 銷售榜單成績

《CODE NAME BLUE》專輯以首天開售售出21,982張登上Oricon專輯日榜冠軍位置\[6\]；同時以首周銷售45,151張取得Oricon專輯週榜冠軍位置。這是[華納唱片 (日本)首次有旗下藝人獲得Oricon專輯週榜冠軍](../Page/日本華納音樂.md "wikilink")；也是首次有韓國樂隊以第一張唱片及第三位韓國藝人或樂隊取得Oricon專輯週榜冠軍位置（其餘兩個是[FTIsland](../Page/FTIsland.md "wikilink")的《[FIVE TREASURE ISLAND](../Page/FIVE_TREASURE_ISLAND.md "wikilink")》及[張根碩](../Page/張根碩.md "wikilink")的《[JUST CRAZY](https://zh.wikipedia.org/wiki/JUST_CRAZY "wikilink")》）\[7\]。

**Cloud9IDE** 是一個線上整合開發環境，從2.0版作為開放軟體，到了3.0版本。 它支援多種 語言，其中包括
[C](https://zh.wikipedia.org/wiki/C语言 "wikilink"),
[C++](../Page/C++.md "wikilink"), [PHP](../Page/PHP.md "wikilink"),
[Ruby](../Page/Ruby.md "wikilink"), [Perl](../Page/Perl.md "wikilink"),
[Python](../Page/Python.md "wikilink"),
[JavaScript](../Page/JavaScript.md "wikilink") 與[Node.js和](../Page/Node.js.md "wikilink")
Go 。

幾乎是以JavaScript寫成和使用Node.js 作為後端。 編輯的部分則是使用 Ace.

在2016年7月Cloud9已經被Amazon收購。\[1\] 從2018年3月開始，現有的帳戶依然可以上Cloud9的原網站上使用，但不能創建新的帳號。
Cloud9現在是Amazon網路服務的一部分，因此新用戶必須使用Amazon網路服務帳號來使用Cloud9服務。\[2\]

## 功能

  - 內建终端，使用Node Package Manager以及基本的
    [Unix命令](../Page/Unix实用程序列表.md "wikilink")
  - 片段和識別碼自動完成
  - 多游標，方便同時編輯
  - 实时分析语言JavaScript(标记的共同JavaScript陷阱)
  - 使用變數或函數的名稱重構JavaScript
  - 自動匹配括號
  - 行號 、警告和錯誤
  - 除錯
  - 文件標籤管理
  - 主題
  - 定制的关键绑定，包括预 [Vim](../Page/Vim.md "wikilink")、Emacs的和崇高文字
  - 內建圖像編輯器
  - 透過JSBeautify和CSSLint重新格式化
  - 可以 [拖拽](https://zh.wikipedia.org/wiki/拖放 "wikilink") 文件项目
  - 支援以下程式碼存放：
      - GitHub 
      - Bitbucket
      -  Mercurial repositories 
      - Git repositories 
      - FTP servers
  - 支援開發：
      - Heroku 
      - Joyent 
      - Microsoft Azure 
      - Google App Engine 
      - SFTP/FTP
  - 支持公共和私人项目
  - 插件支援
  - 突出顯示以下的語言的語法： C\#, C/C++, Clojure, CoffeeScript, ColdFusion, CSS,
    Apache Groovy, Java, JavaScript, LaTeX, Lua, Markdown, OCaml, PHP,
    Perl, PowerShell, Python, Ruby, Scala, SCSS, SQL, Textile, X(HTML),
    XML

## 關於

2010年成立，以舊金山以及阿姆斯特丹作為根據地，Cloud9IDE原本是一家私營公司。 

2016年7月14號，Cloud9宣布被Amazon.com收購。\[3\]

## 使用

Cloud9是本机IDE的 [BeagleBone Black](../Page/BeagleBoard.md "wikilink")
[单载计算机](../Page/单板机.md "wikilink")，这主要是编程的扩展node.js
叫Bonescript的。

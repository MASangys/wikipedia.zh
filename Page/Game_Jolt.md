**Game
Jolt**是一个用于下载[免费软件和付费](https://zh.wikipedia.org/wiki/免费软件 "wikilink")[电子游戏的](../Page/电子游戏.md "wikilink")[互联网托管服务](../Page/互联网托管服务.md "wikilink")，也具有[社会化媒体功能](../Page/社会化媒体.md "wikilink")。用户可以直接通过[浏览器或是](../Page/网页浏览器.md "wikilink")[客户端进入此服务](https://zh.wikipedia.org/wiki/客户端 "wikilink")。该网站于2004年1月开始营运\[1\]，由戴维·迪卡尔米（David
'CROS' DeCarmine）所拥有。

## 历史

### 2003年 - 2007年

Game Jolt于2002年以Holo-World的名称开始进行开发\[2\]，并在2003年更名为Game
Jolt\[3\]。网站在2004年7月29日开始公开营运\[4\]并加入了公众账户系统、论坛和聊天室以及多个不同的游戏，所有游戏都已经过各自的创造者允许上传。在2007年，这个网站因不活跃而被撤下\[5\]。

### 2008年–现今

在2008年12月，David再次营运了网站的第二个版本，这次Game
Jolt成为了一个游戏门户。网站已被重新设计并推出了给可下载、[Flash](../Page/Adobe_Flash.md "wikilink")、[Unity和](../Page/Unity_\(游戏引擎\).md "wikilink")[Java游戏的自动化上传系统](../Page/Java_applet.md "wikilink")\[6\]。

广告获利共享在2009年9月已公开发布\[7\]\[8\]，用户将会获得在游戏页面、介绍页和部落格贴文上的广告获利中的30%\[9\]。

网站在2001年中至2012年早期有着大量的自动刷屏，导致社区和拥有者不活跃\[10\]\[11\]\[12\]\[13\]\[14\]\[15\]\[16\]。

Game Jolt的API在2012年7月释出了长达3年的测试版\[17\]\[18\]，让游戏能够和网站成为一体。

Game
Jolt于2013年2月开始接受浏览器基础的[HTML5游戏](../Page/HTML5.md "wikilink")\[19\]\[20\]\[21\]。

在2013年4月，Game
Jolt在公开测试版中发布了一个用户等级系统，其包含了API中的[成就和](https://zh.wikipedia.org/wiki/成就_\(电子游戏\) "wikilink")[得分](https://zh.wikipedia.org/wiki/得分 "wikilink")，同时也准备了一些网站活动让用户累积EXP（[经验值](https://zh.wikipedia.org/wiki/经验值 "wikilink")）\[22\]。戴维·迪卡尔米在8月8日宣布他已开始完全专注开发Game
Jolt\[23\]，并退出Zulily公司的工作。\[24\]

在2013年10月，现已不存在\[25\]的独立游戏相关[聚合器和](../Page/聚合器.md "wikilink")[网志](../Page/網誌.md "wikilink")——Indie
Statik宣布正在与Game Jolt“合作”\[26\]。他们计划将首页中的文章和Game
Jolt主持的游戏文章展示在上述游戏的资料页，并让负责集合的Indie
Statik和负责服务的Game Jolt合作准备游戏门户网站\[27\]，不过最终并没有完成。

Game Jolt果酱于2014年早期作为服务发布，让用户可以创造自己的并整合在主页上\[28\]\[29\]。

新网站的改革测试版于2015年6月公开并之后在该月释出，Game
Jolt宣传了其[响应式设计](../Page/响应式网页设计.md "wikilink")、游戏与游戏新闻文章的自动化屏膜（衡量出该游戏有多近期上传以及有多热门）以及过滤游戏列表的选项，即平台、完善评分和开发阶段\[30\]\[31\]。

于2016年1月，Game
Jolt在[MIT許可證下在](../Page/MIT許可證.md "wikilink")[GitHub发布了客户端的](../Page/GitHub.md "wikilink")[源代码以及网站的前端](../Page/源代码.md "wikilink")\[32\]。

网上市场在2016年4月被公布\[33\]并于5月发布，让开发者们可以在网站售卖自己的游戏\[34\]。

#### 游戏实况主合伙

于2014年11月，Game Jolt宣布了“Indies vs
PewDiePie”的游戏果酱，与知名YouTuber——[Pewdiepie合作](../Page/PewDiePie.md "wikilink")\[35\]\[36\]。开发者必须在给予的一个周末（11月21日至24日）内制作出与主题“玩也有趣，看也有趣”（Fun
to play, fun to
watch）相关的游戏，来搭配游戏实况的娱乐风格\[37\]。用户们之后可以给游戏进行评分，直到计分的12月1日。前十名的游戏奖品为Pewdiepie在频道游玩该游戏以作为给开发者一个推销的意思\[38\]，虽然之后他还是去玩了其他游戏\[39\]。

Game
Jolt在2015年7月与Pewdiepie、[Jacksepticeye和](../Page/Jacksepticeye.md "wikilink")[Markiplier合作主持](../Page/Markiplier.md "wikilink")“独立
VS 玩家”\[40\]。参赛需求为该游戏必须是使用Game Jolt
API计分表的街机游戏，并且必须在7月17日至20日之间制作完成\[41\]。前5名的游戏能够在合作人的YouTube频道上被实况游玩\[42\]\[43\]\[44\]。

## API

Game
Jolt的[应用程序接口](../Page/应用程序接口.md "wikilink")（被普遍简称为**GJAPI**）让任何开发者可以使用支持[HTTP和](https://zh.wikipedia.org/wiki/HTTP "wikilink")[MD5或](../Page/MD5.md "wikilink")[SHA-1的](../Page/SHA-1.md "wikilink")[游戏开发平台](../Page/系统平台.md "wikilink")\[45\]\[46\]。Game
Jolt宣传表示此API可以：

  - 创建多个“记分板”，让玩家收集[高分并展示在游戏资料页](https://zh.wikipedia.org/wiki/得分#遊戲 "wikilink")，同时给自身账户获得EXP\[47\]。
  - 授予玩家[奖杯](https://zh.wikipedia.org/wiki/成就_\(电子游戏\) "wikilink")，让用户的账号获得更多EXP\[48\]。
  - 在Game Jolt的数据服务器上保存游戏数据\[49\]。

## 竞赛

Game Jolt曾经主持了多个不同参赛需求和奖品的官方。“竞赛”和“果酱”不同，开发者若要参加Game
Jolt上的竞赛，必须跟随该竞赛的单一主题，游戏排名选择则是由员工所决定或是由用户投票决定（近期的两个独立
VS竞赛）；但果酱并没有任何需求并只有一个可选主题，但胜利者并没有任何奖品。

### 果酱

会在每个周末进行主持。它不像竞赛般有主题，纯粹开发新游戏或完成[半成品](https://zh.wikipedia.org/wiki/半成品 "wikilink")，并且最后并不会有胜利者。开发者们被鼓励进行[直播](../Page/流媒体.md "wikilink")、发布截图以及在[推特发布贴文讲述任何他们在开发的事物](https://zh.wikipedia.org/wiki/推特 "wikilink")\[70\]\[71\]\[72\]。

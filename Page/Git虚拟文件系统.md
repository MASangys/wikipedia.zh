**GVFS**（全称**Git Virtual File System**，即**Git虚拟文件系统**）是[微软](../Page/微软.md "wikilink")为配合、加速大型[Git](../Page/Git.md "wikilink")仓库的使用而开发的一个[虛擬檔案系統](https://zh.wikipedia.org/wiki/虛擬檔案系統 "wikilink")。\[1\]\[2\]\[3\]

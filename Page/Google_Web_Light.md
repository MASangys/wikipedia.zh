**Google Web
Light**，是[Google提供的一项服务](../Page/Google.md "wikilink")，可在其[Android移动浏览器](../Page/Android.md "wikilink")[Chrome中更快地浏览](https://zh.wikipedia.org/wiki/Chrome "wikilink")。它能检测[2G等慢速網路](../Page/2G.md "wikilink")，并會自動切换到内置数据压缩功能的Google代理服务器。它可以加快開啟以文字為主的的网站。

这种服务在特定國家提供，包括印度尼西亚、巴西\[1\]和印度。\[2\]

## 批评

Google Web Light不經内容过滤器處理。没有内容过滤器，成人內容或其他有害内容可能會顯示給18岁以下用户。\[3\]

Hack
Day，是一项活动，它将开发者、设计师和有创意的人聚集在一起，大家共同做出一些很酷的东西\[1\]。在全世界范围内，有各种各样的人、公司甚至政府机构，在举办类似的活动。

虽然大家都在举办Hack Day性质的活动，但是名字却不尽相同，有一些组织者就没有使用“Hack
Day”这个名字。比如[Yahoo\!为了区分自己的活动](https://zh.wikipedia.org/wiki/Yahoo! "wikilink")，起名叫做Open
Hack Day；还有些组织者，会起一些更贴切兴趣领域的名字，比如“Over The
Air”\[2\]；还有些更深奥的，比如“Mashed08”\[3\]。在中国，由HTML5研究小组发起的Hack
Day活动，命名为HTML5 Code Jam\[4\]。

Hack
Day活动通常会持续48个小时，第一天通常会从解决实际问题的演讲开始，比如如何访问数据库和API等，当然也会有一些励志的话题，鼓励开发者和设计师们发挥想像，做一些平时没有机会尝试的事情。2008年，[BBC](https://zh.wikipedia.org/wiki/BBC "wikilink")
Hack Day（也叫Mashed08 \[5\]）在Alexandra Palace举办，开场的时候就有来自
[NASA](https://zh.wikipedia.org/wiki/NASA "wikilink")\[6\]、[Yahoo\!
Developer
Network](https://zh.wikipedia.org/wiki/Yahoo!_Developer_Network "wikilink")
\[7\]和[backstage.bbc.co.uk](https://zh.wikipedia.org/wiki/backstage.bbc.co.uk "wikilink")\[8\]的演讲。

2005年11月8日，Yahoo\!在公司内部举办了世界第一场Hack
Day活动。同样是Yahoo\!，于2006年9月29日－30日，在加利福尼亚的桑尼维尔举办了Yahoo\!
Campus，这是对外公开的第一次Hack
Day活动，[Beck在这场活动中还做了演出](https://zh.wikipedia.org/wiki/Beck "wikilink")。

Hack
Day活动的特点，和[unconference](https://zh.wikipedia.org/wiki/unconference "wikilink")、[MashPit](https://zh.wikipedia.org/wiki/MashPit "wikilink")、[Foocamp](https://zh.wikipedia.org/wiki/Foocamp "wikilink")、[BarCamp等其他几个类型的活动相近](../Page/BarCamp.md "wikilink")，但基本按照下面的形式操作：

“第一天”

  - 签到
  - 演讲
  - 24小时Hack活动开始

“第二天”

  - 24小时Hack活动结束
  - 大家来介绍各自的作品
  - 颁奖

在最后一天的晚上，或者在星期六晚上的活动休息时，有时候会有娱乐节目，比如在加利福尼亚的Sunnyvale举办的[Yahoo\!](https://zh.wikipedia.org/wiki/Yahoo! "wikilink")
Hack
Day，[Beck就做了演出](https://zh.wikipedia.org/wiki/Beck "wikilink")。2007年，BBC
Hack Day最后一天的晚上，[The Rumble
Strips也做了表演](https://zh.wikipedia.org/wiki/The_Rumble_Strips "wikilink")。

活动的形式有非常大的灵活性，可以根据具体活动的需要作出调整。比如说，你可以将Hack Day简化为Hack
Evening，只要领会这个活动的核心思想，就不用约束在48小时的周期内\[9\]。

2009年，[Yahoo\!在台北举办的Open](https://zh.wikipedia.org/wiki/Yahoo! "wikilink")
Hack Day，邀请了“Hack
Girls”作表演，遭到了很多投诉\[10\]。[Yahoo\!在博客上发表了一篇文章](https://zh.wikipedia.org/wiki/Yahoo! "wikilink")\[11\]，对这件事情可能带来的恶果进行道歉。

## Hack Day一览

### Global Game Jam

GGJ会将你那里有才能的人们召集在一起，由IGDA（国际游戏开发者协会）主办。这是一个独特的机会，大家同心协力，磨砺自己的技艺，为自己的工作提出挑战。参与活动的人与全世界的游戏同好一起工作。我们为一个共同的理想而奋斗，在48小时内创造出一款游戏。我们希望大家能在这次活动中创造出一些极具潜力的原型概念，并在活动结束后仍然会不断完善，发展这些原型概念，让它们成为自己今后工作的内容和方向。我们在2009年开发出的许多游戏都成为了广受赞扬的游戏产品。GGJ是一个开放资源，是跨越不同硬件和软件的合作，它的一切制作项目都受到Creative
Commons（知识共享组织）的授权保护。我们鼓励人们尝试新的创意，并在合理的条件下推动他们为此而努力。当然，我们也鼓励参与者们在活动中合理地吃饭睡觉，好维持自己的体力在最佳状态\[12\]。2011年，Global
Game Jam首次登陆上海，由Coconut Island Studio承办\[13\]\[14\]。

### Shanghai Game Jam

由Coconut Island Studio组办，上海地区性的Game
Jam活动，2011年7月22日，作为上海游戏制作节的一个环节，举办了第一届。\[15\]

### [HTML5 Code Jam](http://blog.sina.com.cn/html5codejam)

HTML5 Code Jam的灵感来自于Global Game Jam，由HTML5研究小组主办。该活动的宗旨，就是将Game
Jam中“激发创意、展示激情、团队合作、快速开发“的精髓和HTML5技术的推广相结合，面向所有对Web技术感兴趣的朋友，让大家可以有机会在48小时内，创造一款自己的游戏。游戏的品质也许无法保证，但是整个策划、开发、合作的过程，将带给参加者非同一般的体验。同时，希望通过这个活动，展示出HTML5更多新奇好玩的特性\[16\]。截至目前，HTML5
Code Jam已经举办了3期\[17\]。

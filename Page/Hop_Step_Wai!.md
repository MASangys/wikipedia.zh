是Aqours的单曲，也是“Aqours Hop\! Step\! Jump\! Project\!”的主题曲，2018年6月30日由[Lantis](../Page/Lantis.md "wikilink")发售。

## 概要

本单曲随“Aqours CLUB CD SET 2018”（普通版）及“Aqours CLUB CD SET 2018 GOLD EDITION”（黄金版）发售，歌曲试听最早在2018年6月4日[TOKYO FM](https://zh.wikipedia.org/wiki/TOKYO_FM "wikilink")《[SCHOOL OF LOCK\!](../Page/SCHOOL_OF_LOCK!.md "wikilink")》节目中的“Aqours LOCK\!”环节公布。单曲的形式与2017年不同，本次除九人合唱版本外，还收录了无伴奏版本和三个官方小队（CYaRon\! 、AZALEA和Guilty Kiss）的版本，而不是2017年的成员独唱\[1\]。与前一年发售的“Aqours CLUB CD SET 2017”同样为期间限定生产，在不同店铺购买可获得不同特典\[2\]。

与2017年不同的是，2018年的套装分为两个版本，两者均附带会员证、记载有2017年度“Aqours CLUB”照片的画册、序列号及抽选券，不同之处如下\[3\]：

  - 外包装略有差异；
  - 普通版会员证为白色，黄金版会员证为金色；
  - 黄金版额外附带三张2017年“Aqours CLUB”网站上的视频影像[DVD以及记载了](https://zh.wikipedia.org/wiki/DVD "wikilink")“Aqours CLUB”2017年度博客文章（Aqours成员亲笔书写）的画册。

另外，2017年度的“Aqours CLUB”会员可在购买2018年套装后继承之前的账号\[4\]。

单曲发售首日以约1.4万张的销量位列[Oricon公信榜](../Page/Oricon公信榜.md "wikilink")日榜第一位\[5\]，初动销量约2.8万张，位列周榜第4\[6\]。

以上表格提供了下列信息：

| 名称        | 提供[隧道中间人](../Page/隧道中间人.md "wikilink")服务的服务商，这通常与一个[互联网服务供应商](../Page/互联网服务供应商.md "wikilink")对应。                                                                                                                                                                                                                                                                                                                                                                                          |
| --------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| PoPs      | PoPs 部署的数量                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| 服务位置      | 服务商提供的[隧道中间人](../Page/隧道中间人.md "wikilink")服务器的地理位置                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| 子网        | What [subnet](https://zh.wikipedia.org/wiki/subnetwork "wikilink") allocation the tunnel broker provides next to the /127 or /64 for the tunnel.                                                                                                                                                                                                                                                                                                                                          |
| 隧道协议      |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|           | Which [tunnelling protocols](https://zh.wikipedia.org/wiki/tunnelling_protocol "wikilink") are provided by the tunnel broker. This has impact on whether one can use a tunnel behind [NAT](https://zh.wikipedia.org/wiki/Network_Address_Translation "wikilink") or in cases where the IPv4 address of the user changes often. Check the relevant protocol descriptions for more details, or see this [tunneling comparison](http://www.sixxs.net/faq/connectivity/?faq=comparison).      |
| 6in4      | [6in4](../Page/6in4.md "wikilink") tunnels IPv6 packets directly inside IPv4, also called 'proto-41' due to the protocol number (41) it uses.                                                                                                                                                                                                                                                                                                                                             |
| TSP       | The [Tunnel Setup Protocol](https://zh.wikipedia.org/wiki/Tunnel_Setup_Protocol "wikilink"). It uses a UDP-based tunnel, which is used for both tunnelling and configuration.                                                                                                                                                                                                                                                                                                             |
| AYIYA     | [AYIYA](https://zh.wikipedia.org/wiki/AYIYA "wikilink") allows users behind a NAT and/or a dynamic address to maintain connectivity even when roaming between networks. It uses a UDP-based tunnel.                                                                                                                                                                                                                                                                                       |
| 心跳包\[13\] | [heartbeat](https://zh.wikipedia.org/wiki/6in4#Dynamic_6in4_tunnels_and_heartbeat "wikilink") is a side-protocol for signalling the current IP address of a user's 6in4 endpoint. This allows the use of a 6in4 tunnels where the user's endpoint is a dynamic IP address.                                                                                                                                                                                                                |
| Features  | RDNS                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| BGP       | Whether the tunnel broker accepts a [BGP](https://zh.wikipedia.org/wiki/Border_Gateway_Protocol "wikilink") peering allowing one to announce one's own address space or see routing information.                                                                                                                                                                                                                                                                                          |
| 多播        | Whether the tunnel broker provides [IPv6 Multicast](https://zh.wikipedia.org/wiki/IPv6_multicast "wikilink") over a tunnel.                                                                                                                                                                                                                                                                                                                                                               |
| 需要注册？     | If registration is optional or required. Registration requires the user provides real details (at minimum an email address, typically also name and address)                                                                                                                                                                                                                                                                                                                              |
| 配置        |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|           | What methods are available for configuring tunnels and subnets provided by the tunnel broker                                                                                                                                                                                                                                                                                                                                                                                              |
| 脚本        | Indicates that the broker provides a script through their website that one can run on a computer for setting up the tunnel.                                                                                                                                                                                                                                                                                                                                                               |
| 手动        | That one can set up the tunnel manually if wanted                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| TSP       | Uses the Tunnel Setup Protocol (TSP) which requires the gogoclient and which can provide configuration details for TSP protocol tunnels.                                                                                                                                                                                                                                                                                                                                                  |
| TIC       | Uses the Tunnel Information and Control (TIC)\[14\] protocol as implemented by [AICCU](https://zh.wikipedia.org/wiki/AICCU "wikilink"), [AVM](https://zh.wikipedia.org/wiki/AVM_GmbH "wikilink") [FRITZ\!Box](https://zh.wikipedia.org/wiki/FRITZ!Box "wikilink"), [Draytek](https://zh.wikipedia.org/wiki/Draytek "wikilink"), [Motorola](https://zh.wikipedia.org/wiki/Motorola "wikilink") and others. This can provide configuration details for static, heartbeat and AYIYA tunnels. |
| URL       | URL indicates that changes to the tunnel parameters can be made on the website (e.g. enable/disable or change the endpoint)                                                                                                                                                                                                                                                                                                                                                               |
| 语言        | Which [languages](https://zh.wikipedia.org/wiki/language "wikilink") are supported by the providers [web interface](https://zh.wikipedia.org/wiki/user_interface "wikilink") and other [communication channels](https://zh.wikipedia.org/wiki/Channel_\(communications\) "wikilink").                                                                                                                                                                                                     |
| 前缀        | Which IPv6 prefix(es) are used for the tunnel broker. If a /32 is indicated, generally only a portion is used for tunnels/subnets for the tunnel broker or the name of the tunnel broker service that is providing the address space.                                                                                                                                                                                                                                                     |
| 服务器实现     | The [Tunnel broker server software](https://zh.wikipedia.org/wiki/Tunnel_broker#Implementations "wikilink") used by the tunnel broker.                                                                                                                                                                                                                                                                                                                                                    |

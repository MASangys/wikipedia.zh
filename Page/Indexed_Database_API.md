**Indexed Database API**（简称**IndexedDB**，以前称WebSimpleDB）是[W3C推荐的一项](../Page/万维网联盟.md "wikilink")[网页浏览器](../Page/网页浏览器.md "wikilink")标准\[1\]，是为提供一个具有[索引的](https://zh.wikipedia.org/wiki/数据库索引 "wikilink")[JSON](../Page/JSON.md "wikilink")对象集合的[事务性本地数据库操作接口](../Page/事務隔離.md "wikilink")。W3C于2015年1月8日发布了IndexedDB接口的最终建议。\[2\]

## 浏览器支持

[Firefox](../Page/Firefox.md "wikilink")（自版本4\[3\]）、[Google Chrome](../Page/Google_Chrome.md "wikilink")（自版本11\[4\]）和Internet Explorer 10的消费者预览及Metro式应用中IndexedDB已经得到初步支持。\[5\]苹果公司已在2014年6月2日的[WWDC](../Page/苹果全球开发者大会.md "wikilink") 2014的主旨发言中宣布将在iOS 8和OS X上的Safari 8中支持。\[6\]

## 使用案例

IndexedDB可以用于浏览器内实现的功能，例如书签，以及Web应用程序，如电子邮件。索引数据库API的一个开源参考实现已经出现（用于测试和实验目的）\[7\]。

## 特性

IndexedDB是一个嵌入在浏览器中的事务数据库。该数据库的管理围绕[JSON](../Page/JSON.md "wikilink")对象集合的概念，这类似[NoSQL](../Page/NoSQL.md "wikilink")数据库[MongoDB](../Page/MongoDB.md "wikilink")与[CouchDB](https://zh.wikipedia.org/wiki/CouchDB "wikilink")。其中每个对象使用插入时生成的键标识。而索引系统优化对存储对象的访问。\[8\]

**JIS X 0201**（旧称 **JIS C 6220**），是（另見[日本工業規格](https://zh.wikipedia.org/wiki/日本工業規格 "wikilink")）订立的8[位元](../Page/位元.md "wikilink")编码系统。由于[日语](../Page/日语.md "wikilink")可使用[片假名](../Page/片假名.md "wikilink")表示，故日语可使用类似[ISO/IEC 8859的单](https://zh.wikipedia.org/wiki/ISO/IEC_8859 "wikilink")[字节](../Page/字节.md "wikilink")编码系统来表达。

“JIS X 0201”的[拉丁字母](../Page/拉丁字母.md "wikilink")部分又称“”（JIS [罗马字](https://zh.wikipedia.org/wiki/日語羅馬字 "wikilink")），而其[假名部分又称](https://zh.wikipedia.org/wiki/假名 "wikilink")“”（JIS 假名）。

上图浅蓝色的是JIS使用范围。[清音片假名置于](https://zh.wikipedia.org/wiki/清音 "wikilink")0xA6至0xDD。0xDE及0xDF分别置放[浊音和](https://zh.wikipedia.org/wiki/浊音 "wikilink")[半浊音符号](https://zh.wikipedia.org/wiki/半浊音 "wikilink")。0xA1-0xA4是[句号](../Page/句号.md "wikilink")、[引号](https://zh.wikipedia.org/wiki/引号 "wikilink")、[頓號](../Page/頓號.md "wikilink")（，用法同中文[逗号](https://zh.wikipedia.org/wiki/逗号 "wikilink")）、0xA5則是[隔音符号](https://zh.wikipedia.org/wiki/隔音 "wikilink")（，用法類似中文[頓號](../Page/頓號.md "wikilink")和[間隔號](https://zh.wikipedia.org/wiki/間隔號 "wikilink")）。

此标准未有定义0xA0，但多数[操作系统](../Page/操作系统.md "wikilink")都將之用作[不换行空格](../Page/不换行空格.md "wikilink")。

日本有另一個標準[JIS X 0207定义](https://zh.wikipedia.org/wiki/JIS_X_0207 "wikilink")[控制字符](../Page/控制字符.md "wikilink")，在JIS X 0201标准中未有定义。。

上图浅黄色的是JIS与ASCII不同的地方。JIS以“[¥](../Page/¥.md "wikilink")”（[日圓](../Page/日圓.md "wikilink")符號）替换“\\”（[反斜線](https://zh.wikipedia.org/wiki/反斜線 "wikilink")）、以“‾”（[上劃線](https://zh.wikipedia.org/wiki/上劃線 "wikilink")）替换“\~”（[波浪號](../Page/波浪號.md "wikilink")）。

“JIS X 0201”的片假名，又称“[半角片假名](https://zh.wikipedia.org/wiki/半角片假名 "wikilink")”，以别于“[JIS X 0208](../Page/JIS_X_0208.md "wikilink")”的“[全角片假名](https://zh.wikipedia.org/wiki/全角 "wikilink")”，因为此字集的字符只占用一字节空间，印刷字型的宽度也只有全角片假名的一半。

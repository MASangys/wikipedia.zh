**Java EE**，Java平台企业版（Java Platform Enterprise Edition），之前称为Java 2 Platform, Enterprise Edition (J2EE)，2018年3月更名为**Jakarta EE**。\[1\]是[Sun公司为企业级应用推出的标准](https://zh.wikipedia.org/wiki/Sun "wikilink")[平台](https://zh.wikipedia.org/wiki/系統平台 "wikilink")。Java平台共分为三个主要版本**Java EE**、[Java SE和](https://zh.wikipedia.org/wiki/Java_SE "wikilink")[Java ME](../Page/Java_ME.md "wikilink")。

[Sun公司在](https://zh.wikipedia.org/wiki/Sun "wikilink")1998年发表[JDK](../Page/JDK.md "wikilink")1.2版本的时候，使用了新名称Java 2 Platform，即“Java2平台”，修改后的[JDK](../Page/JDK.md "wikilink")称为Java 2 Platform Software Developing Kit，即[J2SDK](https://zh.wikipedia.org/wiki/J2SDK "wikilink")。并分为标准版（Standard Edition，[J2SE](https://zh.wikipedia.org/wiki/J2SE "wikilink")），企业版（Enterprise Edition，J2EE），微型版（MicroEdition，[J2ME](https://zh.wikipedia.org/wiki/J2ME "wikilink")）。J2EE便由此诞生。

2005年6月，JavaOne大会召开，SUN公司公开Java SE 6。此时，Java的各种版本已经更名以取消其中的数字“2”：J2EE更名为Java EE, [J2SE更名为](https://zh.wikipedia.org/wiki/J2SE "wikilink")[Java SE](https://zh.wikipedia.org/wiki/Java_SE "wikilink")，[J2ME更名为](https://zh.wikipedia.org/wiki/J2ME "wikilink")[Java ME](../Page/Java_ME.md "wikilink")。

随着[Java](../Page/Java.md "wikilink")技术的发展，J2EE平台得到了迅速的发展，成为[Java](../Page/Java.md "wikilink")语言中最活跃的体系之一。现如今，J2EE不仅仅是指一种标准平台，它更多的表达着一种[软件架构](../Page/软件架构.md "wikilink")和设计思想。

## 组件

Java EE是一系列技术标准所组成的平台，包括：

  - [Applet](https://zh.wikipedia.org/wiki/Applet "wikilink") - Java Applet
  - [EJB](../Page/EJB.md "wikilink") - 企业级[JavaBean](https://zh.wikipedia.org/wiki/JavaBean "wikilink")（Enterprise Java Beans）
  - [JAAS](../Page/JAAS.md "wikilink") - Java Authentication and Authorization Service
  - [JACC](https://zh.wikipedia.org/wiki/JACC "wikilink") - J2EE Authorization Contract for Containers
  - [JAF](https://zh.wikipedia.org/wiki/JAF "wikilink") - Java Beans Activation Framework
  - [JAX-RPC](https://zh.wikipedia.org/wiki/JAX-RPC "wikilink") - Java API for XML-Based Remote Procedure Calls
  - [JAX-WS](../Page/JAX-WS.md "wikilink") - Java API for XML Web Services
  - [JAXM](../Page/JAXM.md "wikilink") - Java API for XML Messaging
  - [JAXP](../Page/JAXP.md "wikilink") - Java XML解析API（Java API for XML Processing）
  - [JAXR](https://zh.wikipedia.org/wiki/JAXR "wikilink") - Java API for XML Registries
  - [JCA](https://zh.wikipedia.org/wiki/JCA "wikilink") - J2EE连接器架构（J2EE Connector Architecture）
  - [JDBC](https://zh.wikipedia.org/wiki/JDBC "wikilink") - Java数据库联接（Java Database Connectivity）
  - [JMS](https://zh.wikipedia.org/wiki/JMS "wikilink") - Java消息服务（Java Message Service）
  - [JMX](../Page/JMX.md "wikilink") - Java Management
  - [JNDI](https://zh.wikipedia.org/wiki/JNDI "wikilink") - Java名稱与目录接口（Java Naming and Directory Interface）
  - [JSF](https://zh.wikipedia.org/wiki/JSF "wikilink") - Java Server Faces
  - [JSP](../Page/JSP.md "wikilink") - Java服务器页面（Java Server Pages）
  - [JSTL](../Page/JSTL.md "wikilink") - Java服务器页面标准标签库（Java Server Pages Standard Tag Library）
  - [JTA](https://zh.wikipedia.org/wiki/JTA "wikilink") - Java事务API（Java Transaction API）
  - [JavaMail](https://zh.wikipedia.org/wiki/JavaMail "wikilink")
  - [Servlet](https://zh.wikipedia.org/wiki/Servlet "wikilink") - Java Servlet [API](https://zh.wikipedia.org/wiki/API "wikilink")
  - [StAX](../Page/StAX.md "wikilink") - Streaming APIs for XML Parsers
  - [WS](https://zh.wikipedia.org/wiki/WS "wikilink") - Web Services

《**Jump up HIGH\!\!**》為[Aqours](../Page/Aqours.md "wikilink")的單曲，在2019年6月30日由[Lantis](../Page/Lantis.md "wikilink")發售\[1\]。該單曲為「**Aqours Jumpin' up Project**」的主題曲\[2\]，隨「**Aqours CLUB CD SET 2019**」一同販售\[3\]。

## 概要

本作是前作單曲《[Believe again/Brightest Melody/Over The Next Rainbow](https://zh.wikipedia.org/wiki/Believe_again/Brightest_Melody/Over_The_Next_Rainbow "wikilink")》推出五個月後的單曲，是[Aqours CLUB CD SET系列的第三彈](https://zh.wikipedia.org/wiki/Aqours_CLUB_CD_SET "wikilink")，並隨「**Aqours CLUB CD SET 2019**」一同販售\[4\]。

單曲標題曲《Jump up HIGH\!\!》是Aqours自2019開展的企劃「**Aqours Jumpin' up Project**」的主題曲\[5\]\[6\]。該單曲的標題、內容及試聽影片已先行於2019年6月3日在官方[YouTube](../Page/YouTube.md "wikilink")頻道公布\[7\]。完整版則在2019年6月9日舉行的演唱會《LoveLive\! Sunshine\!\! Aqours 5th LoveLive\! ～Next SPARKLING～》第二日上首度披露\[8\]。這也是首度有Aqours的樂曲在CD發售前即於演唱會中披露完整版\[9\]，也是LoveLive\!系列的首次。

本單曲與前回的《[Hop Step Wai\!](../Page/Hop_Step_Wai!.md "wikilink")》一樣，收錄了分別由不同小隊演唱的版本\[10\]\[11\]。

該單曲共以兩種型態發售。一種是隨「**Aqours CLUB CD SET 2019**」一同收錄了Aqours粉絲俱樂部「[Aqours CLUB 2019](https://zh.wikipedia.org/wiki/Aqours#Aqours_CLUB "wikilink")」的註冊資格序號及官方相關商品的通常盤，另一種是隨包含前述內容外另外收錄了特別紀念冊及紀念DVD的「PLATINUM EDITION」\[12\]\[13\]。

## 影響

本單曲在2019年6月28日的Oricon單曲日榜排名第一\[14\]，並維持至7月1日。這也讓Aqours的單曲連續四次獲得Oricon單曲日榜第一。在7月8日公布的Oricon單曲週榜中，本單曲以25,327張的推定銷量排名週榜第二，僅次於[IZ\*ONE](../Page/IZ*ONE.md "wikilink")的單曲《[布宜諾斯艾利斯](../Page/布宜諾斯艾利斯_\(單曲\).md "wikilink")》\[15\]，這也是自2014年[舞祭組](../Page/舞祭組.md "wikilink")的《》以來，第二度有星期日發售的單曲獲得週榜第二。在動畫單曲週榜方面，這是首度有星期日發售的單曲獲得週榜第一。單曲綜合分數榜方面，本單曲以25,327分的成績在2019年7月8日公布的單曲綜合分數榜排名第二\[16\]。截至2019年7月，[Aqours](../Page/Aqours.md "wikilink")已有連續20首作品排名於週榜五名以內\[17\]\[18\]，在動畫歌手裡僅次於[水樹奈奈](../Page/水樹奈奈.md "wikilink")的連續25次登上排行榜\[19\]\[20\]\[21\]\[22\]。

在Billboard Japan方面，本單曲在2019年7月8日的[Billboard Japan Hot 100榜上獲得第](../Page/告示牌日本熱門100金曲榜.md "wikilink")16名\[23\]，單曲銷量榜週榜則以約25,480張的銷量獲得第2名\[24\]，Billboard JAPAN HOT Animation週榜則獲得週榜第3名\[25\]。

其他地区：每年四次\[1\] | attempt_restriction = 同一考生一年内最多测验三次，五年内最多五次，一生最多七次（2019年起）\[2\] | regions = 全世界 | language = 英语和西班牙语\[3\] | test_takers = 13.8万人（2018年）\[4\] | fee = 200美元，每次额外写作测验15美元\[5\] | score_users = 美国，加拿大和澳大利亚的法学院 }}  **LSAT**（），即**法学院招生测验**，是1947年设立，由位于[美国](../Page/美国.md "wikilink")[宾夕法尼亚州](../Page/宾夕法尼亚州.md "wikilink")牛顿市的**[法学院招生委员会](https://zh.wikipedia.org/wiki/法学院招生委员会 "wikilink")**（**LSAC**，**L**aw **S**chool **A**dmission **C**ouncil）举办的标准化测验。\[6\]本测验被美国，加拿大以及澳大利亚的法学院用作[法律博士](../Page/法律博士.md "wikilink")（Juris Doctor, J.D.）项目申请的重要参考。

## 考试构成

LSAT考试由五部分构成，分别为五个**单项选择**部分和一个**写作**部分。其中，单项选择包括两个**逻辑推理**部分（Logical Reasoning），一个**阅读理解**部分（Reading Comprehension）一个**分析推理**部分（Analytical Reasoning，又俗称逻辑游戏部分（Logic Games）\[7\]），以及一个不计入分数的实验部分。实验部分可能是逻辑推理，阅读理解或分析推理部分的任意一种，用意是测验考生们对LSAT新题的适应程度。

每个单项选择部分和写作部分占35分钟；写作部分不计入成绩。\[8\]

LSAT的每一个部分都不要求学生掌握任何专业知识，而是更加注重学生的逻辑推理能力。\[9\]

### 单项选择部分

单项选择部分的考试在考场完成。所有单项选择问题都有五个备选项。

#### 逻辑推理

逻辑推理部分考查考生对逻辑论证的分析能力。该部分包括25-26个问题，每一个问题的题干是一个以对话或者句子呈现的论证。考生可能会被要求找出论证的[错误](../Page/謬誤.md "wikilink")，加强或者削弱一个论证，找出论证所依赖的假设，找出论证的论点等等。\[10\]

#### 分析推理

分析推理部分考察考生根据给定的规则对一组元素之间的关系进行整理和分析的能力。其中，要求学生对一组元素进行排列或分组的问题最为常见。该部分包括四组问题，每组问题共用一个题干。\[11\]一个典型的分析推理问题组如下所示：

题干：

电影节共放映A，B，C，D，E，F，G七部电影。电影节持续三天，每天至少放映两部电影。除此以外，电影节还有以下规定：

  - A和C不得在同一天放映。
  - C和D在同一天放映。
  - E和G在同一天放映。
  - F只得在第三天放映。

问题：

假设E和C和F不在同一天放映，请列出A都可以在哪天放映：

  - 第一天
  - 第一天和第二天
  - 第二天和第三天
  - 第一天和第三天
  - 第一天，第二天和第三天

#### 阅读理解

分析推理部分考察考生理解介绍新知识的复杂文章的能力。该部分包含四个问题组，其中三个问题组的题干是一篇较长的文章，另外一个问题组的题干则是两篇较短但是相互关联的文章（例如对同一个社会事件的两篇意见相左的社评）。每个问题组包含5-8个基于文章的问题。考生可能被要求辨别文章的主旨，文章中特定段落或句子在作者的论证所起的作用，文章的结构，特定词语在文章中的意思等。\[12\] 一般来说，阅读文章的题材涉及[艺术](../Page/艺术.md "wikilink")，[社会科学](../Page/社会科学.md "wikilink")，[法律](../Page/法律.md "wikilink")和[自然科学](../Page/自然科学.md "wikilink")四大类。\[13\]

### 写作部分

写作部分考察学生进行完整论证的能力。写作部分的题干通常描述一个困境以及两个可能的解决方案。考生需选择其中一个方案，并对为什么该方案优于另一个方案进行论证。这类写作问题没有正确答案；只要给出的论证合理有力，考生可以任选一个解决方案。LSAC不会对学生的写作进行评分，而是会将作文直接送到考生欲申请的法学院。\[14\]

自2019年起，写作部分和其他考试部分正式分离；学生可在考场参加单项选择部分的考试之后，在自己的电脑上下载LSAC官方的监考软件，并在电脑上完成写作部分的测验。\[15\]

## 考试分数

LSAT的考试分数范围是120-180。考生的分数取决于回答正确的问题数量，每正确回答一个问题，考生得一原始分（raw score），回答错误不扣分。考生的原始分将会根据对照表进行换算以得到考生的最终分数。在提供分数的同时，考生也会得知自己的[百分位数](https://zh.wikipedia.org/wiki/百分位数 "wikilink")，即自己的成绩高于百分之几的考生。以下是2007年6月考试的分数对照表的节选：\[16\]\[17\]\[18\]

| LSAT分数 | 原始分/正确答案数量 | 百分位数  |
| ------ | ---------- | ----- |
| 180    | 99         | 99.9% |
| 175    | 96         | 99.6% |
| 170    | 92         | 97.6% |
| 165    | 86         | 91.7% |
| 160    | 78         | 80.4% |
| 155    | 69         | 63.5% |
| 150    | 59         | 44.6% |

假设一名考生正确地回答了92个问题，那么该生的分数根据对照表应为170分。同时，由表可见，LSAT的分数计算由高到低越来越宽松；从180分扣至175分仅需要多错三题，而从165分扣至160分则需要多错八题；从155分扣至150分需要多错十题。

一般来说，学生可以在考试后三周左右以电子邮件的形式得知自己的分数。\[19\]美国最优秀的法学院，例如被称作“十四巨头（）”的，[美国新闻与世界报道排名前十四位的法学院的录取生普遍拥有](https://zh.wikipedia.org/wiki/美国新闻与世界报道 "wikilink")167分（百分位数94.7%）以上的LSAT成绩。\[20\]

### 分数汇报

与[ACT](../Page/ACT_\(测验\).md "wikilink")，[GRE](../Page/GRE.md "wikilink")等考试不同，考生的所有LSAT分数都会被递送至欲申请的法学院；换言之，考生无法选择不递交任意一次成绩。然而；考生可以选择在参加完考试后六天内（得知分数前）取消自己的分数。考生一旦选择取消，这个决定将是不可逆的。同时，法学院也会得知考生取消分数的操作。\[21\]

然而，参加2019年7月考试的考生可以在得知自己分数后取消自己的成绩，这是因为该次考试是第一次电子化的LSAT，具有一定的实验性质。\[22\]

## 考试次数

LSAT在美国，加拿大和加勒比海地区每测验年（每年6月至次年6月）共安排九次考试（6，7，9，10，11，1，2，3，4月），在全球其他地区则安排四次（6，10，1，3月）。\[24\]

## 考试地区

LSAT在全球设有考场。其中，在东亚地区，LSAT在[北京](../Page/北京市.md "wikilink")（[北京外国语大学](../Page/北京外国语大学.md "wikilink")），[上海](https://zh.wikipedia.org/wiki/上海市 "wikilink")（[上海交通大学](https://zh.wikipedia.org/wiki/上海交通大学 "wikilink")，[上海财经大学](../Page/上海财经大学.md "wikilink")），[广州](../Page/广州市.md "wikilink")（[广东外语外贸大学](../Page/广东外语外贸大学.md "wikilink")），[台北](../Page/臺北.md "wikilink")（[台湾科技大学](../Page/國立臺灣科技大學.md "wikilink")），[香港](../Page/香港.md "wikilink")（[香港考评局](../Page/香港考試及評核局.md "wikilink")[新蒲岗办事处](../Page/新蒲崗.md "wikilink")），[东京](../Page/东京.md "wikilink")（[天普大学日本校区](../Page/天普大學.md "wikilink")）以及[首尔](../Page/首爾.md "wikilink")（[韩国外国语大学](../Page/韓國外國語大學.md "wikilink")）等地设有考场。\[25\]

## 备考

LSAC与[可汗学院共同推出了免费](../Page/可汗學院.md "wikilink")，官方的LSAT备考[课程](https://www.khanacademy.org/prep/lsat?utm_source=lsac-site&utm_medium=lsac&utm_campaign=lsac-launch2018&utm_term=preparing-callout)。同时，LSAC在官网上也免费提供了2007年6月的官方考试。除此以外，LSAC官方也销售历年的真题供考生备考。

市场上也有很多第三方机构和个人提供LSAT的备考服务。\[26\]

## 电子化

从2019年7月开始，LSAT将停止使用传统的填涂式答题卡，而是改在[微软Surface Go平板电脑上进行考试](../Page/Microsoft_Surface.md "wikilink")。同时，LSAC也会为学生准备笔，草稿纸以及触摸笔。

电子化的LSAT考试支持以下功能：

  - 触摸屏
  - 可调节的字体和行距
  - 放大
  - 高对比度文字
  - 反色
  - 颜色滤镜（用以辅助[色盲](../Page/色盲.md "wikilink")考生）
  - 亮度和灰度调节

在变为电子化考试之后，LSAT的各部分题型和考试时间维持不变。LSAC也在官网提供了真实考试所使用的电子监考软件，以帮助学生适应新的考试形式。\[27\]

西班牙语LSAT的题目类型和考试结构与英语LSAT相同，但是分数范围是320–380分，这是为了用于与英语版本相区别。\[28\]

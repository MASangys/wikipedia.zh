**MDN Web Docs**（舊稱Mozilla Developer Network、Mozilla Developer Center，简称MDN）是一個汇集众多[Mozilla基金會](../Page/Mozilla基金會.md "wikilink")產品和網路技術開發文档的免费網站\[1\]。

## 歷史

該計畫始於2005年，最初由Mozilla公司員工Deb Richardson領導。自2006年以來，文档工作由Eric Shepherd領導\[2\]。

網站最初的內容是由[DevEdge提供](https://zh.wikipedia.org/wiki/網景#解散 "wikilink")，但在[AOL收購Netscape後](https://zh.wikipedia.org/wiki/AOL "wikilink")，DevEdge網站也宣佈關閉。為此Mozilla基金會向AOL取得了DevEdge釋出的內容\[3\]\[4\]，同時將DevEdge內容搬移到mozilla.org\[5\]\[6\]。

MDN本身有一個論壇，并在Mozilla IRC網絡上有一個[IRC](../Page/IRC.md "wikilink")頻道\#mdn。MDN由[Mozilla公司](../Page/Mozilla公司.md "wikilink")提供服務器和員工的資助。

2016年10月3日發表的[Brave網頁瀏覽器將MDN作為其](https://zh.wikipedia.org/wiki/Brave_\(網頁瀏覽器\) "wikilink")[搜尋引擎選項之一](https://zh.wikipedia.org/wiki/搜尋引擎 "wikilink")\[7\]。

[MediaWiki-smaller-logo.png](https://zh.wikipedia.org/wiki/File:MediaWiki-smaller-logo.png "fig:MediaWiki-smaller-logo.png") **MediaWiki**是一套[开放源代码](../Page/开放源代码.md "wikilink")的[Wiki引擎](../Page/Wiki軟體.md "wikilink")。第一个版本为1.1，发布于2003年12月。[MediaWiki](../Page/MediaWiki.md "wikilink") 1.32.0是当前的稳定版本，发布于2019年1月。

下表包含了MediaWiki的**版本历史**，展示了该软件所有的发行版本。

## 版本歷史

| -- | -------- |
| 红色 | 旧版本；不再支持 |
| 黄色 | 旧版本；仍受支持 |
| 绿色 | 当前版本     |
| 蓝色 | 未来版本     |

**Microsoft Messenger for Mac**，原名為**MSN Messenger for Mac**，是[微軟Macintosh](https://zh.wikipedia.org/wiki/微軟 "wikilink") Business Unit為[Mac OS X開發的](https://zh.wikipedia.org/wiki/Mac_OS_X "wikilink")[即時通訊軟件](https://zh.wikipedia.org/wiki/即時通訊軟件 "wikilink")。然而與[Windows Live Messenger相比](../Page/Windows_Live_Messenger.md "wikilink")，Microsoft Messenger for Mac仍然缺少了一些功能，如離線訊息及與[Windows之間的對應等](https://zh.wikipedia.org/wiki/Windows "wikilink")。因此如[Adium](../Page/Adium.md "wikilink")、[emesene](https://zh.wikipedia.org/wiki/emesene "wikilink")、[Mercury Messenger及](../Page/Mercury_Messenger.md "wikilink")[aMSN等類似軟體在Mac](https://zh.wikipedia.org/wiki/aMSN "wikilink") OS X上陸續開發使用。

## 版本

3.x版本及更高版本的Microsoft Messenger for Mac使用新的[Aqua圖像介面](../Page/Aqua_\(GUI\).md "wikilink")。而2.x版本和更低版本的圖像介面與Windows Messenger 4.0相似。5.0版本使用brushed metal為主題。

  - 2.5.1版本 是2.5版本的更新,這是一個2.1版本的強制性更新,需使用Mac OS 9.2.2
  - 3.5版本 需使用Mac OS X 10.2.8或更舊
  - 4.0.1版本 需使用Mac OS X 10.2.8或更新
  - 5.1.1版本 需使用Mac OS X 10.3或更新
  - 6.0.3版本 需使用Mac OS X 10.3.9或更新
  - 7.0.0版本 需使用Mac OS X 10.4.9或更新，發佈於2008年4月29日。
  - 7.0.1版本 需使用Mac OS X 10.4或更新，包括小規模的錯誤修復。
  - 7.0.2版本 需使用Mac OS X 10.4或更新，提高與微軟[Office Communicator 2007](https://zh.wikipedia.org/wiki/Office_Communicator_2007 "wikilink") R2的整體素質和對話。
  - 8.0.1版本 需使用Mac OS X 10.5或更新，支持跨平台的视频/音频功能。隨附在[Microsoft Office 2011 for Mac之中](https://zh.wikipedia.org/wiki/Microsoft_Office_for_Mac "wikilink")

## 功能比較

Microsoft Messenger for Mac沒有Windows Live Messenger的以下功能:

  - 離線留言
  - 手寫郵件
  - 螢幕/屏幕共享
  - Nudges
  - Winks
  - Photo backgrounds on current conversation windows
  - 互動遊戲
  - 一些內置表情符號,包括the rabbit emoticon
  - 顯示詳細個人資料
  - Windows Live Spaces 整合
  - Windows Live Today 整合
  - Color schemes
  - 麥克風、視訊聊天 (8 版本之後之功能)

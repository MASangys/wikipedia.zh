**Microsoft Visual Studio
Express**是一套由[微軟公司開發的](https://zh.wikipedia.org/wiki/微軟公司 "wikilink")[免費](../Page/免費軟體.md "wikilink")\[1\][整合開發環境](https://zh.wikipedia.org/wiki/整合開發環境 "wikilink")，是[Microsoft
Visual
Studio產品線上的輕量版本](../Page/Microsoft_Visual_Studio.md "wikilink")。其具有易學易用的特點，而功能稍弱；故其受眾是愛好者和學生，而非專業開發者。

Express版本於Microsoft Visual Studio 2005最初出現。

## 2005至2010版本

Visual Studio 2005、2008、及2010 Express包含以下獨立產品：

  - [Visual Basic](../Page/Visual_Basic_.NET.md "wikilink") Express
  - [Visual C++](https://zh.wikipedia.org/wiki/Visual_C++ "wikilink")
    Express
  - [Visual
    C\#](https://zh.wikipedia.org/wiki/Microsoft_Visual_C_Sharp "wikilink")
    Express
  - Visual [J\#](https://zh.wikipedia.org/wiki/J_Sharp "wikilink")
    Express（僅2005）
  - Visual Web Developer Express
  - Visual Studio Express for [Windows
    Phone](../Page/Windows_Phone.md "wikilink")（僅2010）
  - [SQL Server Express](../Page/SQL_Server_Express.md "wikilink")

## 2012版本

Visual Studio 2012有如下版本釋出：

  - Visual Studio Express 2012 for Web\[2\]\[3\]
  - Visual Studio Express 2012 for Windows 8\[4\]\[5\]
  - Visual Studio Express 2012 for Windows Desktop\[6\]\[7\]
  - Visual Studio Team Foundation Server Express 2012\[8\]
  - Visual Studio Express 2012 for Windows Phone\[9\]

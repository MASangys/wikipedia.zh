**Opera軟件**（；全稱：**Opera Software
AS**），是[挪威](../Page/挪威.md "wikilink")的一家軟件公司，主要以其桌面網頁瀏覽器[Opera和行動網頁瀏覽器](https://zh.wikipedia.org/wiki/Opera電腦瀏覽器 "wikilink")[Opera
Mini而聞名](../Page/Opera_Mini.md "wikilink")。作為後者剝離其網頁瀏覽器業務的一部分，它從母公司Otello
Corporation被剝離。Opera瀏覽器在全球多個平台上擁有超過3.5億使用者，並表示願景是「在任何設備上提供最佳的互聯網體驗」\[1\]。

該公司還參與透過參與W3C推廣Web標準，總部設在挪威奧斯陸，在中國大陸、台灣、美國、瑞典、波蘭、日本、韓國、澳大利亞、俄羅斯、烏克蘭、冰島和新加坡設有辦事處\[2\]。

當Otello公司在2016年將其網頁瀏覽器和消費者業務以及Opera品牌出售給中國投資者時，該公司更換了所有權，並於當年11月完成了交易\[3\]。

## 歷史

Opera軟件於1995年由Jon Stephenson von
Tetzchner和GeirIvarsøy在挪威獨立創立。該公司的創建旨在繼續最初的挪威電信公司Telenor的研究項目\[4\]
。

1997年，推出Opera瀏覽器，為試用軟件，必須在試用期結束後購買\[5\]。

1998年，為了利用互聯網連接的手持設備的新興市場，啟動了一個將Opera瀏覽器移植到更多平台的項目\[6\]。

2000年，發布Opera
4.0，包括一個新的跨平台核心，協助創建版本Opera適用於多種作業系統和平台，Opera成為了廣告贊助商，向使用者顯示了沒有授權的廣告，這是通常被批評為獲得市場份額的障礙。在較新的版本中，使用者可以選擇通用圖形橫幅或Google提供的基於正在查看的頁面的基於文本的定向廣告\[7\]。

2004年2月，進行了IPO，並於2004年3月11日首次發布，向一家「國際公司」支付了1275萬美元給Opera的訴訟。據推測，宣布和解的聲明中提到的「國際公司」是微軟，該公司之前曾攔阻Opera使用者正常瀏覽MSN.com\[8\]。

2005年1月12日，宣布向高等教育機構免費提供授權 -
與以前的無限制授權1,000美元的費用相比有所變化。選擇免費授權的學校包括麻省理工學院（MIT），哈佛大學，牛津大學，佐治亞理工學院和杜克大學\[9\]。

2005年8月，推出Opera Mini，一款基於Java
ME的手機網頁瀏覽器，最初不銷售給終端使用者，而是銷售給行動網絡運營商，以預裝手機或向使用者提供，廣告被完全刪除，主要財務支持來自Google的收入（Opera的預設搜尋引擎）\[10\]。

2007年，向歐盟委員會提起微軟投訴，聲稱將Internet Explorer與Microsoft
Windows捆綁在一起對消費者和其他網頁瀏覽器公司都有害。投訴導致創建BrowserChoice.eu\[11\]。

2012年，與Bharti Airtel簽署協議，向Airtel行動客戶提供Opera Mini\[12\]。

在2013年，決定不再使用桌面瀏覽器的內部渲染引擎。從用於計算機的Opera瀏覽器的版本15開始，將使用Blink渲染引擎，這是Webkit開發者與Google共同開發的一個分支\[13\]。

2015年3月在巴塞羅那舉行的世界行動通訊大會上，Opera獲得全球行動大獎，最佳行動產品，Opera Web
Pass通行證或新興市場服務，以及贊助網絡通行證\[14\]。

2015年4月，決定集中在波蘭開發Opera Desktop瀏覽器，Opera TV AS成立，將電視相關業務與其他所有資產分離，成為Opera
Software AS的一部分，兩家公司都成為Opera Software ASA（Otello Corporation
ASA）的全資子公司\[15\]。

2015年9月，宣布重新製作一個新的立體「O」標誌，品牌識別從「Opera Software」變成了「Opera」\[16\]。

2016年2月10日，一群中國投資者以12億美元（每股8.31美元）的價格收購Opera Software
ASA，儘管該交易據報導並未達到監管部門的批准\[17\]。

2016年7月18日，宣布已將其瀏覽器，隱私和性能應用以及Opera品牌出售給由崑崙萬維科技和奇虎360牽頭的中國投資者聯盟Golden
Brick Capital Private Equity Fund I Limited
Partnership（金磚資本管理有限公司），金額達6億美元
\[18\]。

2016年11月4日，Golden Brick Capital Private Equity Fund I
L.P.（金磚資本）完成收購，從Opera瀏覽器和品牌撤出後，Opera Software
ASA更名為Otello Corporation ASA\[19\]。

2017年1月，推出Opera Neon，這是探索瀏覽器設計替代品的全新概念瀏覽器。瀏覽器建立在Blink排版引擎之上，可用於Microsoft
Windows和macOS\[20\]。

2018年2月，宣布服務和應用程序將被關閉後，Opera
Max應用程序被出售給三星電子。它被重新命名為三星Max，但只會與符合條件的三星Galaxy系列智能產品兼容\[21\]。

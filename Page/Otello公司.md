**Otello公司**（；全稱：**Otello Corporation ASA**，前稱：**Opera Software ASA**），是[挪威](../Page/挪威.md "wikilink")的一家互聯網公司，為運營商、發行商和廣告商開發廣告和行動解決方案。Opera Software ASA是Opera產品和業務的控股公司，在將其網頁瀏覽器和消費者業務以及Opera品牌（[Opera Software AS](https://zh.wikipedia.org/wiki/Opera_Software_AS "wikilink")）出售給中國投資者財團後，更名為Otello Corporation。雖然最初計劃完全出售給財團，但由於監管機構的批准，這筆交易沒有-{zh-cn:经过; zh-tw:通過;}-，於是決定只出售其面向消費者的網頁瀏覽器業務。

## 收購

2010年1月20日，Opera軟體收購行動廣告公司AdMarvel\[1\]。

2010年4月30日，Opera軟體收購澳洲電子郵件供應商[FastMail.FM](https://zh.wikipedia.org/wiki/FastMail.FM "wikilink")\[2\]。2013年9月，FastMail再次成為一家獨立的公司\[3\]。

2011年9月19日，Opera軟體收購Android第三方應用程式商店Handster\[4\]，用於加強[Opera Mobile Store面向消費者](https://zh.wikipedia.org/wiki/Opera_Mobile_Store "wikilink")、行動運營商和手機廠商提供的服務\[5\]。

2013年2月15日，Opera軟體以1.55億美元收購Skyfire，主要針對該公司的影片優化技術，如Rocket Optimizer平台，以補充自身的內容優化技術\[6\]。

2014年6月4日，Opera軟體收購AdColony以加強其行動平台影音廣告服務，Opera軟體將支付7500萬美元現金以及可能高達2.75億美元的支出\[7\]。

2015年3月，Opera軟體收購加拿大[VPN公司SurfEasy](https://zh.wikipedia.org/wiki/VPN "wikilink")，隨後將SurfEasy VPN整合到Opera中，提供使用者免費和無限流量的服務\[8\]。

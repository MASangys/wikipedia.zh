**PC
Tools**成立於2003年\[1\]，前身叫做**WinGuides.com**\[2\]，其總部位於[盧森堡市](https://zh.wikipedia.org/wiki/盧森堡市 "wikilink")，並在[澳洲](https://zh.wikipedia.org/wiki/澳洲 "wikilink")、[美國](https://zh.wikipedia.org/wiki/美國 "wikilink")、[英國](https://zh.wikipedia.org/wiki/英國 "wikilink")、[愛爾蘭島和](https://zh.wikipedia.org/wiki/愛爾蘭島 "wikilink")[烏克蘭設有其辦公室](https://zh.wikipedia.org/wiki/烏克蘭 "wikilink")。此公司主要開發[微軟](https://zh.wikipedia.org/wiki/微軟 "wikilink")[Windows和](https://zh.wikipedia.org/wiki/Microsoft_Windows "wikilink")[Mac
OS X的安全和最佳化軟體](https://zh.wikipedia.org/wiki/OS_X "wikilink")。PC
Tools已經被[賽門鐵克買下](https://zh.wikipedia.org/wiki/賽門鐵克 "wikilink")，並已停售以PC
Tools為名的產品。

## 產品

PC Tools提供反惡意程式軟體和系統工具軟體供全世界的消費者使用。截至2006年11月29日，PC
Tools的軟體被下載了超過一億二千五百萬次。.\[3\]

### PC Tools瀏覽器保護器（PC Tools Browser Defender）

**PC Tools瀏覽器保護器**（**PC Tools Browser Defender**\[19\],簡稱為**Browser
Defender**）為適用於[Microsoft
Windows系統的](https://zh.wikipedia.org/wiki/Microsoft_Windows "wikilink")[Internet
Explorer和Mozilla](../Page/Internet_Explorer.md "wikilink")
[Firefox的瀏覽器](../Page/Firefox.md "wikilink")[工具列](https://zh.wikipedia.org/wiki/工具列 "wikilink")。此軟體可保障瀏覽網路的安全。

### PC Tools i防毒（PC Tools iAntiVirus）

i防毒於2012年以諾頓為名重新上市。（諾頓為[賽門鐵克旗下的品牌](https://zh.wikipedia.org/wiki/賽門鐵克 "wikilink")）

**PC Tools i防毒**（**PC Tools
iAntiVirus**，簡稱為**iAntiVirus**）為用於以[英特爾](https://zh.wikipedia.org/wiki/英特爾 "wikilink")（Intel）為基礎的蘋果[麥金塔（Mac）電腦的免費](https://zh.wikipedia.org/wiki/麥金塔電腦 "wikilink")[防毒軟體](https://zh.wikipedia.org/wiki/防毒軟體 "wikilink")，適用於[Mac
OS X 10.5 (Leopard)和](../Page/Mac_OS_X_Leopard.md "wikilink")[Mac OS X
10.6 (Snow
Leopard)](../Page/Mac_OS_X_Snow_Leopard.md "wikilink")。此軟體於2008年6月釋出，使用特徵碼和啟發是技術偵測[惡意軟體](https://zh.wikipedia.org/wiki/惡意軟體 "wikilink")、間諜軟體和[exploit](https://zh.wikipedia.org/wiki/exploit "wikilink")（弱點入侵）。i防毒被批評它只能偵測麥金塔病毒，而不像其他替代軟體一樣，可偵測[Microsoft
Windows](https://zh.wikipedia.org/wiki/Microsoft_Windows "wikilink")、[Linux和Mac病毒](../Page/Linux.md "wikilink")。然而，它也因為其系統資源占用率低及其掃描速度快而受到佳評。\[20\]\[21\]

### PC Tools網路安全套裝（PC Tools Internet Security\[22\]）

**PC Tools網路安全套裝**是間諜軟體醫生（Spyware
Doctor）、防火牆和反垃圾郵件軟體的結合，將此三個獨立軟體的功能整合為一。本軟體為[付費軟體](https://zh.wikipedia.org/wiki/商業軟體 "wikilink")，可用於[Windows
8](https://zh.wikipedia.org/wiki/Windows_8 "wikilink")（32/64位元）、[Windows
7](https://zh.wikipedia.org/wiki/Windows_7 "wikilink")（32/64位元）、[Windows
Vista](../Page/Windows_Vista.md "wikilink")（32/64位元）和[Windows
XP](../Page/Windows_XP.md "wikilink")（32位元）。賽門鐵克已在2013年5月18日起停止供應此產品。

### PC Tools登陸檔技師（PC Tools Registry Mechanic\[23\]）

PC Tools登陸檔技師用於掃描登陸檔錯誤，為PC Tools的第一款軟體\[24\]。目前最新版本為11，於2011年10月31日釋出。

### PC Tools間諜軟體醫生（PC Tools Spyware Doctor\[25\]）

**PC
Tools間諜軟體醫生**是一款反惡意程式軟體，可以用其Spider技術偵測電腦遭入侵的徵兆並識別出惡意程式\[26\]。目前最新版本為**9**，於2011年10月31日釋出。賽門鐵克已經於2013年5月18日起停止供應此產品。

#### PC Tools間諜軟體醫生含防毒（PC Tools Spyware Doctor with AntiVirus\[27\]）

和間諜軟體醫生一樣，**PC
Tools間諜軟體醫生含防毒**也是一款反惡意程式軟體，擁有和間諜軟體醫生相同的功能，但外加上傳統的防毒引擎\[28\]。目前最新版本為**9**，於2011年10月31日釋出。賽門鐵克已經於2013年5月18日起停止供應此產品

### ThreatExpert

**ThreatExpert\[29\]**是一個進階自動化威脅分析系統（advanced automated threat analysis
system,字首簡寫為*ATAS*），可全自動分析並報告可執行檔是否有可疑行為.\[30\] {{-}}

## 被賽門鐵克收購

2008年8月18日，[賽門鐵克宣布與PC](https://zh.wikipedia.org/wiki/賽門鐵克 "wikilink")
Tools簽約，賽門鐵克收購PC
Tools並維持其獨立運作\[31\]。收購金額當時並未公布。賽門鐵克在2008年10月6日以262000000美元收購了PC
Tools\[32\]。

賽門鐵克在2013年5月18日星期三停止了所有PC Tools安全性產品的供應\[33\]。自2013年12月4日起所有PC
Tools的軟體都已停止更新及販售。

## 評價

### 間諜軟體醫生

間諜軟體醫生獲得PC World雜誌第2007年10月號的*Best Buy*獎：「PC
Tools間諜軟體醫生5.0是偵測和移除我們廣告和間諜軟體的贏家。」\[34\]間諜軟體醫生也獲得了數個PC
Magazine的Editor’s
Choice獎\[35\]，包括間諜軟體醫生含防毒5.5（2008年）\[36\]。此產品還獲得了其他世界各地的獎項。

並非所有的評價都是正面的。早期版本的間諜軟體醫生5.0（由Scratch改版而成），得到一些負面評價\[37\]。CNET的Download.com給間諜軟體醫生三顆星的評價[reviews](http://www.download.com/Spyware-Doctor/3000-8022_4-10293212.html?cdlPid=10795134)並評論說：「在我們的測試中，間諜軟體醫生反覆地把一些無害的cookie標示為潛在威脅，比起其它我們所測試的反間諜軟體都還來的多。我們也無從得到每個威脅被間諜軟體醫生標示的原因或詳細資訊。」

## 批評

華盛頓居民曾告賽門鐵克旗下的PC Tools登陸檔技師、效能工具箱（Performance
Toolkits）、和[諾頓電腦醫生](https://zh.wikipedia.org/wiki/諾頓電腦醫生 "wikilink")（Norton
Utilities）進行假的掃描，以欺騙消費者付費購買該產品\[38\]。該訴訟主訴諾頓公司蓄意執行假掃描，且其掃描結果為假\[39\]。結果此訴訟賽門鐵克勝訴。

此產品已停售。目前賽門鐵克提供原顧客諾頓產品的折扣。\[40\]

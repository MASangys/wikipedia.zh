**Pro Tools**是用在[Mac OS X和](../Page/Mac_OS_X.md "wikilink")[Microsoft
Windows作業系統的一個](../Page/Microsoft_Windows.md "wikilink")[數位音訊工作站平臺](../Page/數位音訊工作站.md "wikilink")，由[Digidesign發展和製造](../Page/Digidesign.md "wikilink")，是[Avid
Technology的一部分](../Page/Avid_Technology.md "wikilink")，用于音樂录音和编辑、[電影配樂製作以及電影和電視](../Page/電影配樂.md "wikilink")[後製中](../Page/後製.md "wikilink")。它廣泛用於整個音訊行業的專業人士和[音樂製作人](../Page/音樂製作人.md "wikilink")。Pro
Tools 有三個類型的系統：HD、LE 和 M-Powered。HD 是高端包 (Hi-End
Package)，並且為硬體和軟體的結合。該硬體包括外部[A/D轉換器和用板載](../Page/類比數位轉換器.md "wikilink")[DSP的內部](../Page/DSP.md "wikilink")[PCI或](../Page/PCI.md "wikilink")[PCIe音效卡](../Page/PCI_Express.md "wikilink")。Pro
Tools 9（取代了LE和M-Powered系统）以及之后的版本已可以兼容所有声卡，但配套的HD版本依然存在。

## 概述

基本上，Pro Tools 類似于一個多軌磁帶錄音機和混音機，伴隨的附加功能只能在數位介質中執行。高端的版本支援多達 192kHz 取樣速率和
16 及 24
位的位元深度，打開[WAV](../Page/WAV.md "wikilink")、[AIFF](../Page/AIFF.md "wikilink")、[MP3](../Page/MP3.md "wikilink")、[SDII音訊檔和](../Page/SDII.md "wikilink")[QuickTime影片檔](../Page/QuickTime.md "wikilink")。它以[時間代碼](../Page/時間代碼.md "wikilink")、[節奏地圖](../Page/節奏地圖.md "wikilink")、自動化和[環繞聲的功能為特色](../Page/環繞聲.md "wikilink")。

## 介面

大多數的 Pro Tools 基本功能可以由編輯或組合的視窗控制。編輯視窗顯示音訊和
[MIDI](../Page/MIDI.md "wikilink") 的曲目，並提供記錄或導入資訊的圖形表示形式。在這裡，可以在一種
[非線性](../Page/非線性編輯.md "wikilink")、[非破壞性的個別樣本級別中編輯音訊](../Page/非破壞性編輯.md "wikilink")，MIDI
資訊也可以進行操作。混音機視窗顯示每個 Fader 頻道，並允許調整的一個頻道的音量和
Pan，以及常用的地方以插入外掛程式的效果，路由音訊至不同的輸出和輸入。

Pro Tools中的虛擬樂器及處理中的效果達到通過插頭外掛程式的使用，也由 [DSP](../Page/DSP.md "wikilink")
晶片作為 TDM 外掛程式，或主機作為 [RTAS (Real Time
AudioSuite)](../Page/RTAS_\(Real_Time_AudioSuite\).md "wikilink") 外掛程式。

## 系統

### Pro Tools HD 系統

Pro Tools的HD系统由HD Core核心运算卡负责安装和运行软件以及DSP运算，I/O接口负责数据流的输入和输出。HD
Accel加速卡则可以增强HD的运算性能。Pro Tools HD 9已可以兼容所有支持Core
Audio和ASIO的音频接口。

### Pro Tools LE 系統、Pro Tools M Powered 系統

Pro Tools 9及后续版本的推出使得这两个系统已毫无意义，不过以前的系统可以升级成Pro Tools 9。

### Pro Tools HDX 系统

该系统赠送一套HDX卡，可以提供比原来的HD卡快15倍的DSP加速，亦可同时处理比原先多四倍的声音以及四倍的延时补偿。每台机器可安装三个HDX卡。HDX系统推出了一种新的名叫AAX的插件格式，有望替代原先的TDM和RTAS。

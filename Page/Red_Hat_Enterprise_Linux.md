**Red Hat Enterprise Linux**（**RHEL**）是一個由[Red Hat開發的商業市場導向的](https://zh.wikipedia.org/wiki/Red_Hat "wikilink")[Linux发行版](../Page/Linux发行版.md "wikilink")。红帽公司从 Red Hat Enterprise Linux 5 开始对企业版 LINUX 的每个版本提供 10 年的支持\[1\]。Red Hat Enterprise Linux 常被簡稱爲 RHEL，但它並非官方名稱。

Red Hat Enterprise Linux 大約 3 年发布一个新版本。

[RHEL可以使用](https://zh.wikipedia.org/wiki/RHEL "wikilink") [Fedora EPEL](https://zh.wikipedia.org/wiki/Fedora#发行 "wikilink") 来补足软件。

## 基於

最初，Red Hat Enterprise Linux 基於 [Red Hat Linux](../Page/Red_Hat_Linux.md "wikilink")，但使用較為保守的發布週期。後來版本都是基於 [Fedora](../Page/Fedora.md "wikilink")。大約每六個版本的 Fedora 會有一個新版本的 Red Hat Enterprise Linux 發佈，因此：

  - Red Hat Linux 6.2 → Red Hat Linux 6.2E
  - Red Hat Linux 7.2 → Red Hat Enterprise Linux 2.1
  - Red Hat Linux 9 → Red Hat Enterprise Linux 3
  - Fedora Core 3 → Red Hat Enterprise Linux 4
  - Fedora Core 6 → Red Hat Enterprise Linux 5
  - Fedora 12,13 → Red Hat Enterprise Linux 6\[2\]
  - Fedora 19,20 → Red Hat Enterprise Linux 7\[3\]
  - Fedora 28 → Red Hat Enterprise Linux 8

## 衍生版本

衍生版本有[CentOS](../Page/CentOS.md "wikilink")、[Fedora](../Page/Fedora.md "wikilink")、[Scientific Linux及](../Page/Scientific_Linux.md "wikilink")[Oracle Linux等](../Page/Oracle_Linux.md "wikilink")。

各版本比較：

|                  | 免費下載   | 免費使用   | 技術支援 (商業) |
| ---------------- | ------ | ------ | --------- |
| RHEL             | 否      | 否      | 付费        |
| CentOS           | 是      | 是      | 不提供       |
| Fedora           | 是      | 是      | 不提供       |
| Scientific Linux | 是      | 是      | 不提供       |
| Oracle Linux     | 要求簡單登記 | 是\[4\] | 付费        |

**SQL Server Management Studio**（SSMS）是[微软](../Page/微软.md "wikilink")的一個與其[數據庫產品](https://zh.wikipedia.org/wiki/數據庫 "wikilink")[Microsoft SQL Server配合使用的產品](https://zh.wikipedia.org/wiki/Microsoft_SQL_Server "wikilink")，以便於設定、配置及管理其元件。原為SQL Server 2000的管理工具，名為「SQL Server Enterprise Manager」。該工具包括[腳本編輯器和圖形工具](https://zh.wikipedia.org/wiki/腳本編輯器 "wikilink")，與服務器的對象和功能一起使用\[1\]。

SSMS最主要的功能都集中在Object Explorer：透過這個程式，用戶可以瀏覽、選擇、以及對伺服器內的對象作出各種動作\[2\]。

2015年6月，微軟宣布打算於未來發佈獨立於SQL Server數據庫引擎的未來SSMS版本\[3\]。

**SQL Server Management
Studio**（SSMS）是[微软的一個與其](../Page/微软.md "wikilink")[數據庫產品](https://zh.wikipedia.org/wiki/數據庫 "wikilink")[Microsoft
SQL
Server配合使用的產品](https://zh.wikipedia.org/wiki/Microsoft_SQL_Server "wikilink")，以便於設定、配置及管理其元件。這個工具包括[脚本编辑器和图形化界面工具](https://zh.wikipedia.org/wiki/脚本编辑器 "wikilink")，与服务器的元件和功能一起使用\[1\]。

原為SQL Server 2000的管理工具，名為「SQL Server Enterprise Manager」，SQL Server
2005開始改用現時的名稱。

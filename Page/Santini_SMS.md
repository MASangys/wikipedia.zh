**Santini SMS** 是一個義大利[自行車服品牌](../Page/自行車.md "wikilink")，全名為「Santini
Maglificio Sportivo」。由[Pietro
Santini在](https://zh.wikipedia.org/wiki/Pietro_Santini "wikilink")1965年於[義大利](https://zh.wikipedia.org/wiki/義大利 "wikilink")[拉廖成立](../Page/拉廖_\(科莫省\).md "wikilink")。\[1\]
創辦人少年喜歡騎車，後來接手姊姊們的紡織廠\[2\]，轉型成為製作及設計自行車服飾，並堅持產品線留在義大利本土生產。

## 自行車服製造商

d'Italia)的冠軍衫\]\] Santini SMS
自1993年製作[環義自行車賽](https://zh.wikipedia.org/wiki/環義自由車賽 "wikilink")[冠軍衫](https://zh.wikipedia.org/wiki/冠軍衫 "wikilink")——[粉紅衫](https://zh.wikipedia.org/wiki/粉紅衫 "wikilink")，1994年起成為[國際自行車總會及](https://zh.wikipedia.org/wiki/國際自由車總會 "wikilink")[世界錦標賽的自行車服](../Page/世界錦標賽.md "wikilink")[贊助商](https://zh.wikipedia.org/wiki/贊助商 "wikilink")，製作UCI世界錦標賽各項目的冠軍衫——彩虹衫。Santini也提供歐洲錦標賽的車衣、世界盃女子公路賽、場地單車賽、登山賽、越野賽等的領騎衫。\[3\]
該品牌也出品  的車衣，以及環澳自行車賽和多菲納區自行車賽的領騎衫等。Santini和UCI的合約始自1994年，迄今已逾30年\[4\]。

## 歷史

| 年份        | 事件                                                                                       |
| --------- | ---------------------------------------------------------------------------------------- |
| 1965      | 專注於自行車衣的製造。                                                                              |
| 1976      | 開始採用萊卡材質製作自行車吊帶車褲。在此之前車褲皆使用黑色羊毛製成。                                                       |
| 1993      | 與環義自行車賽(Giro d’Italia)的長期合作，製作環義冠軍衫的[粉紅衫](https://zh.wikipedia.org/wiki/粉紅衫 "wikilink")。 |
| 1994      | 成為UCI世界錦標賽彩虹衫與UCI世界盃冠軍衫，官方贊助單位。                                                          |
| 1999/2000 | 與澳洲國家自行車隊的長期合作關係。                                                                        |
| 2017      | 與環西自行車賽合作\[5\]                                                                           |

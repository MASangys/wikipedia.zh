**Socket 478**是英特爾[Pentium 4](https://zh.wikipedia.org/wiki/Pentium_4 "wikilink")，[Celeron及部份](https://zh.wikipedia.org/wiki/Celeron "wikilink")[Celeron D處理器使用的插座](https://zh.wikipedia.org/wiki/Celeron#Celeron_D "wikilink")。[LGA775於](https://zh.wikipedia.org/wiki/LGA775 "wikilink")2004年7月推出後逐漸取代Socket 478。

早期的Socket 478主機板，無法支援FSB 533MHz的Pentium 4處理器。到了中期情况改善，開始全數支持FSB 800MHz的Pentium 4處理器。

## 使用Socket 478的處理器

  - *Willamette* Pentium 4 1.4-2.0GHz (FSB 400MHz, L2 Cache 256KB)
  - *Northwood* Pentium 4 1.6GHz-2.8GHz (FSB 400MHz, L2 Cache 512KB), 2.26GHz-3.06GHz (FSB 533MHz, L2 Cache 512KB), 2.4GHz-3.4GHz (FSB 800MHz, L2 Cache 512KB)
  - *Prescott* Pentium 4 2.4/2.8GHz (FSB 533MHz, L2 Cache 1MB), 2.8GHz-3.4GHz (FSB 800MHz, L2 Cache 1MB)
  - *Gallatin* Pentium 4 Extreme Edition 3.2GHz-3.4GHz (FSB 800MHz, L2 Cache 512KB, L3 Cache 2MB)
  - *Willamette* Celeron 1.7GHz-1.8GHz (FSB 400MHz, L2 Cache 128KB)
  - *Northwood* Celeron 2.0GHz-2.8GHz (FSB 400MHz, L2 Cache 128KB)
  - *Prescott* Celeron D 310-345 (FSB 400MHz, L2 Cache 256KB)

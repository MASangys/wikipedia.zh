**Surface Studio**是[微软](../Page/微软.md "wikilink")推出的运行[64位](https://zh.wikipedia.org/wiki/64位 "wikilink")[Windows 10的](../Page/Windows_10.md "wikilink")[一体机产品](https://zh.wikipedia.org/wiki/一体机 "wikilink")，于2016年10月26日正式发布\[1\]，是[Microsoft Surface系列的新产品](../Page/Microsoft_Surface.md "wikilink")。它拥有一块支持[多点触摸和数字墨水的](https://zh.wikipedia.org/wiki/多点触摸 "wikilink")28[英寸PixelSense显示屏](https://zh.wikipedia.org/wiki/英寸 "wikilink")，[分辨率](../Page/分辨率.md "wikilink")为4500x3000，可以通过放低屏幕变成专业数字画板\[2\]。Surface Studio一共有三种版本，售价从2999美元到4199美元不等。

2018年10月3日，微软发布 Surface Studio 2，对其处理器、显卡、触摸屏及硬盘等配置进行了升级。\[3\]

## 型号

### Surface Studio 初代

Surface Studio 基于[Intel第六代](https://zh.wikipedia.org/wiki/Intel "wikilink")[SkyLake处理器](https://zh.wikipedia.org/wiki/SkyLake "wikilink")，使用[混合固态硬盘](https://zh.wikipedia.org/wiki/混合固态硬盘 "wikilink")。同时，作为一体机设备，Surface Studio各型号均配备了[NVIDIA独立显卡](https://zh.wikipedia.org/wiki/NVIDIA "wikilink")。

| CPU                                                                  | GPU                 | RAM  | ROM | 售价     |
| -------------------------------------------------------------------- | ------------------- | ---- | --- | ------ |
| [Intel Core](https://zh.wikipedia.org/wiki/Intel_Core "wikilink") i5 | Nvidia GTX 965M 2GB | 8GB  | 1TB | 2999美元 |
| Intel Core i7                                                        | Nvidia GTX 965M 2GB | 16GB | 1TB | 3499美元 |
| Intel Core i7                                                        | Nvidia GTX 980M 4GB | 32GB | 2TB | 4199美元 |

### Surface Studio 2

Surface Studio 2 使用 Intel Core i7-7820HQ 处理器，升级为纯[固态硬盘](../Page/固态硬盘.md "wikilink")。

| CPU           | GPU                 | RAM  | ROM     |
| ------------- | ------------------- | ---- | ------- |
| Intel Core i7 | Nvidia GTX 1060 6GB | 16GB | 1TB SSD |
| Intel Core i7 | Nvidia GTX 1070 8GB | 32GB | 1TB SSD |
| Intel Core i7 | Nvidia GTX 1070 8GB | 32GB | 2TB SSD |

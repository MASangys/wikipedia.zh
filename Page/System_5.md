**System Software 5** (简称 System 5)
是[苹果電腦](https://zh.wikipedia.org/wiki/苹果電腦 "wikilink")（今
[蘋果公司](https://zh.wikipedia.org/wiki/蘋果公司 "wikilink")）的[操作系统](../Page/操作系统.md "wikilink")。System
Software
5增加了[MultiFinder来实现多任务处理](https://zh.wikipedia.org/wiki/MultiFinder "wikilink")。System
Software 5只在部分国家在很短的时间内出售， 这些国家包括美国、加拿大和一些欧洲国家。

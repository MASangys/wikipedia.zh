**System Software 5** (简称 System 5)
是[苹果電腦](../Page/苹果電腦.md "wikilink")（今
[蘋果公司](../Page/蘋果公司.md "wikilink")）的[操作系统](../Page/操作系统.md "wikilink")。System
Software
5增加了[MultiFinder来实现多任务处理](../Page/MultiFinder.md "wikilink")。System
Software 5只在部分国家在很短的时间内出售， 这些国家包括美国、加拿大和一些欧洲国家。

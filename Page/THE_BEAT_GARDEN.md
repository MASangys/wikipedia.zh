**THE BEAT GARDEN**是[日本](../Page/日本.md "wikilink")音樂組合，四人男子團體。2016年7月27日以單曲『Never End』正式出道。

## 概要

2016年7月正式出道的四人組合。

## 成員

官網成員介紹。\[1\]

| 名字         | 負責項目     |
| ---------- | -------- |
| **U**      | VOCAL、隊長 |
| **REI**    | VOCAL    |
| **MASATO** | VOCAL    |
| **SATORU** | DJ       |

## 略歴

### 2012年

8月，U、REI、MASATO3人組成「PEEK A BOO\!\!\!」。

### 2014年

8月，改名為「THE BEAT GARDEN」。

### 2015年

2月，發行首張專輯「WILL」。
12月，支援成員DJ SATORU正式加入。

### 2016年

7月27日，以「Never End」正式出道。

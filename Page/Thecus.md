**Thecus**(**樺賦科技**)成立於2004年，經營Thecus®
中英文自有品牌，專注於製造[網路儲存設備](https://zh.wikipedia.org/wiki/網路儲存設備 "wikilink")(
Thecus NAS server )與網路監控設備( VisoGuard NVR
)，提供滿足安全儲存、資源共享、操作簡易、人機介面等需求的網路儲存(
Storage server )解決方案。總部位於新北市汐止區新台五路一段遠東世界中心內，並在美國、荷蘭設有分公司。

<span style="font-size: 9pt">**U2獲獎列表**</span>

此條目是[愛爾蘭](https://zh.wikipedia.org/wiki/愛爾蘭 "wikilink")[搖滾團體](https://zh.wikipedia.org/wiki/搖滾 "wikilink")[U2的獲獎紀錄列表](../Page/U2樂團.md "wikilink")。U2，1976年成立於[愛爾蘭的樂團](https://zh.wikipedia.org/wiki/愛爾蘭 "wikilink")，四名團員自1980年代崛起後至今仍活躍於樂壇，專輯作品全球銷售量超過1億7000萬，獲得22座[葛萊美獎](../Page/葛萊美獎.md "wikilink")，是紀錄保持人。

最初成立的四位青年團員，一直到1980年代中期才成為國際性的巨星，憑藉著1987年專輯《[約書亞樹](../Page/約書亞樹.md "wikilink")》迅速竄起，大多數獎項也是自此之後開始獲得。

## 其他獎項

| 年度           | 獎項                                                                     |
| ------------ | ---------------------------------------------------------------------- |
| **朱諾獎**      |                                                                        |
| 1989年        | 最佳國際藝人                                                                 |
| 1993年        | 最佳國際藝人                                                                 |
| 2002年        | 最佳製作人獎 ([丹尼爾·雷諾伊斯](https://zh.wikipedia.org/wiki/丹尼爾·雷諾伊斯 "wikilink")) |
| **新音樂快遞音樂獎** |                                                                        |
| 2001年        | 最佳搖滾演出                                                                 |
| 天生好手         |                                                                        |
| 2002年        | 最佳現場演出                                                                 |

1989年和1993年，U2曾獲得[朱諾獎](../Page/朱諾獎.md "wikilink")最佳國際藝人獎，之後製作人[丹尼爾·雷諾伊斯也因為製作歌曲](https://zh.wikipedia.org/wiki/丹尼爾·雷諾伊斯 "wikilink")〈Beautiful Day〉、〈Elevation〉而獲得最佳製作人獎\[18\]。1992年，U2以歌曲〈Mysterious Ways〉獲得[告示牌音樂獎最佳專輯歌曲](https://zh.wikipedia.org/wiki/告示牌音樂獎 "wikilink")\[19\]。2001年，U2獲得[新音樂快遞音樂獎](../Page/新音樂快遞音樂獎.md "wikilink")天生好手獎以及最佳搖滾演出獎\[20\]，隔年再度獲得最佳現場演出獎\[21\]。2003年，以電影《[紐約黑幫](https://zh.wikipedia.org/wiki/紐約黑幫 "wikilink")》的配樂歌曲〈The Hands That Built America〉獲得[金球獎](../Page/金球獎.md "wikilink")、[拉斯維加斯影評人協會獎的最佳歌曲獎](https://zh.wikipedia.org/wiki/拉斯維加斯影評人協會 "wikilink")\[22\]\[23\]，隔年獲得[全美民選獎最受歡迎團體獎](https://zh.wikipedia.org/wiki/全美民選獎 "wikilink")\[24\]。2005年入選為[搖滾名人堂的一員](https://zh.wikipedia.org/wiki/搖滾名人堂 "wikilink")\[25\]。

**Univers**（, [法语](../Page/法语.md "wikilink")意思是通用"universe"）是1957年[瑞士设计师](https://zh.wikipedia.org/wiki/瑞士 "wikilink")[阿德里安·弗鲁提格](../Page/阿德里安·弗鲁提格.md "wikilink")设计的一种西文[无衬线](https://zh.wikipedia.org/wiki/无衬线 "wikilink")[字体](../Page/字体.md "wikilink")。

它和[Helvetica](../Page/Helvetica.md "wikilink")一起被称为“[瑞士风格](https://zh.wikipedia.org/wiki/瑞士 "wikilink")”字体，最初是作为[法国](https://zh.wikipedia.org/wiki/法国 "wikilink")[德贝尔尼和佩尼奥铸造所的](https://zh.wikipedia.org/wiki/德贝尔尼和佩尼奥铸造所 "wikilink")[照相排版](../Page/照相排版.md "wikilink")[铅字发表的](https://zh.wikipedia.org/wiki/铅字 "wikilink")。

该字体家族沿用设计师Frutiger使用的“号”（number）系统来区别不同的字重和变体，而不是分别起不同名字。尽管如此，现在Univers家族字体名字由号码和字母-{zh:后缀; zh-hans:后缀; zh-hant:後綴}-两部分组成。Frutiger在保持统一风格的基础上制作了大量不同的宽度、不同字重的字体，使得该字体家族逐渐庞大。 目前Univers字体家族有44种字体，其中16种按照字重、宽度、位置组合编号，20种斜体，8种字体支持[中欧字符集](https://zh.wikipedia.org/wiki/中欧 "wikilink")，8种支持[西里尔字母](../Page/西里尔字母.md "wikilink")。

## Linotype Univers

在1997年Frutiger与[Linotype公司合作复刻了整个Univers家族](https://zh.wikipedia.org/wiki/Linotype "wikilink")，这就是现在的*Linotype Univers*体，包括63种字体。通过这次复刻，增加了一些“极端”的字重，如*Ultra Light*（超细）和*Extended Heavy*（特粗），还增加了一些[等宽字体](../Page/等宽字体.md "wikilink")。下述的编号系统拓展到三位数来适应这个家族众多的变体。

## Frutiger编号

为减少命名、称呼的混乱，Adrian Frutiger设计了这个编号系统，首先适用于Univers，之后也用在[Frutiger](../Page/Frutiger.md "wikilink")，[Avenir](../Page/Avenir.md "wikilink")，[Neue Helvetica等其他字体上](https://zh.wikipedia.org/wiki/Neue_Helvetica "wikilink")。

编号由两位数组成，第一位是字重，第二位代表宽度和位置。

|       | Suffix         |
| ----- | -------------- |
| 编号    | 1              |
| 字重    | \-             |
| 宽度和位置 | Ultra Extended |
|       |                |

由于各个字体厂家对这个编号理解不同，导致事实变得更加混乱。而雪上加霜的是，Univers字体中[伪斜体](../Page/伪斜体.md "wikilink")系列没有自己的编号，而使用和其相对应的正体的号码，而意大利[斜体却有](https://zh.wikipedia.org/wiki/斜体 "wikilink")。

## Linotype编

Linotype Univers字体家族采用3位数编码。第一位代表字重，第二位字宽，第三位是位置。\[1\]

|    | Suffix  |
| -- | ------- |
| 号码 | 0       |
| 字重 | \-      |
| 字宽 | \-      |
| 位置 | Roman正体 |
|    |         |

## 用途

Univers以其清晰的线条，远距离的可读性和优雅的风格一直受到欢迎。 [PowerBook_Univers_keycaps.jpg](https://zh.wikipedia.org/wiki/File:PowerBook_Univers_keycaps.jpg "fig:PowerBook_Univers_keycaps.jpg")的键盘采用Univers字体\]\] Univers该字体虽然没有[Helvetica](../Page/Helvetica.md "wikilink")有名，但仍在20世纪六七十年代被广泛应用，如[瑞士航空](https://zh.wikipedia.org/wiki/瑞士 "wikilink")（这里指Swiss International Air Lines。现在的Swissair使用[Futura体](https://zh.wikipedia.org/wiki/Futura "wikilink")）、[德意志银行](../Page/德意志银行.md "wikilink")、[通用电气](../Page/通用电气.md "wikilink")（1986年至2004年，之后改用[GE Inspira体](https://zh.wikipedia.org/wiki/GE_Inspira "wikilink")）\[2\]、[匯豐銀行](https://zh.wikipedia.org/wiki/匯豐銀行 "wikilink")、日本的[三洋电机等都采用这个字体为企业标准字体](https://zh.wikipedia.org/wiki/三洋电机 "wikilink")。

[苹果公司原先使用该字体](https://zh.wikipedia.org/wiki/苹果公司 "wikilink")，而且它的condensed oblique变体被用于很多苹果电脑的键盘上，直至2007年8月新iMac的超薄键盘才开始改用[VAG Rounded体](https://zh.wikipedia.org/wiki/VAG_Rounded "wikilink")（笔记本电脑自2001年就已经改用了）。

Univers也被[惠普](../Page/惠普.md "wikilink")的[LaserJet](https://zh.wikipedia.org/wiki/LaserJet "wikilink") 1100打印机捆绑，在视窗系统安装LaserJet 1100后将同时安装这些字体，他们在字体列表中显示的时候，字体名字旁边还带有打印机图标，指出所选字体是由打印机处理的打印机字体，而不是系统字体。

[香港](../Page/香港.md "wikilink")[明報](../Page/明報.md "wikilink")、[香港](../Page/香港.md "wikilink")[星島日報](../Page/星島日報.md "wikilink")同時使用此字體與Helvetica字體印刷報紙。

另外方正大黑、方正细黑、方正综艺字体的英文部分也是使用Univers。

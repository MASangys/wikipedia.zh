**Web SQL 数据库**是用于将数据存储在[数据库](../Page/数据库.md "wikilink")中的网页[API](../Page/应用程序接口.md "wikilink") ，这些数据库可以使用[SQL](../Page/SQL.md "wikilink")的变体进行查询。\[1\]\[2\]

[Google Chrome](../Page/Google_Chrome.md "wikilink")，\[3\] [Opera](https://zh.wikipedia.org/wiki/Opera "wikilink")\[4\]和均支持该API。

[W3C](../Page/万维网联盟.md "wikilink") Web 应用工作组 (Web Applications Working Group) 已于2010年11月停止研究该规范，理由是凡是使用Web SQL 数据库的瀏覽器對應的[數據庫管理系統均為](../Page/数据库管理系统.md "wikilink")[SQLite](../Page/SQLite.md "wikilink")，無法在其他[數據庫管理系統上實現](../Page/数据库管理系统.md "wikilink")，因此Web SQL 数据库无法继续成为W3C的推薦規範。\[5\]

[Mozilla公司](../Page/Mozilla公司.md "wikilink")也是该标准的反對者之一，而同时[Mozilla公司](../Page/Mozilla公司.md "wikilink")又是[IndexedDB的主要支持者](../Page/Indexed_Database_API.md "wikilink")。\[6\]\[7\]

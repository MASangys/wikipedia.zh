**Windows 2.0**是一套基于[MS-DOS](../Page/MS-DOS.md "wikilink")[操作系统](../Page/操作系统.md "wikilink")，[微軟于](https://zh.wikipedia.org/wiki/微軟 "wikilink")1987年發行，類似[Mac OS](https://zh.wikipedia.org/wiki/Mac_OS "wikilink")[图形用户界面](../Page/图形用户界面.md "wikilink")的[Windows版本](https://zh.wikipedia.org/wiki/Windows "wikilink")。这个[用户界面](../Page/用户界面.md "wikilink")較[Windows 1.0有著更多的功能](../Page/Windows_1.0.md "wikilink")，且已近似[Windows 3.0](../Page/Windows_3.x.md "wikilink")。它已有一些的改进，而紧接着它就是更受欢迎的[3.0版本但此版本的支援週期長達](../Page/Windows_3.x.md "wikilink")14年，由1987年支援到2001年最後一天為止。

Windows 2.0是16位基于Microsoft Windows GUI的操作环境，于1987年12月9日发布\[1\]，它是Windows 1.0的后续版本。

## 特点

Windows 2.0允许应用程序窗口彼此重叠，而其前身[Windows 1.0只能显示平铺窗口](../Page/Windows_1.0.md "wikilink")。Windows 2.0还引入了更复杂的键盘快捷键\[2\]，以及“最小化”和“最大化”的术语，与Windows 1.0中的“图标化”和“缩放”相对。 此处介绍的基本窗口设置将持续到Windows 3.1。 与Windows 1.x一样，Windows 2.x应用程序不能在Windows 3.1或更高版本上运行，因为它们不是为保护模式设计的。Windows 2.0也是第一个集成控制面板的Windows版本。

Windows 2.0中的新功能还包括16色VGA图形。这也是不需要硬盘的Windows的最新版本。随着速度，可靠性和可用性的提高，计算机现在已开始成为某些工人日常生活的一部分。桌面图标和键盘快捷键的使用有助于加快工作速度。 Windows 2.x EGA，VGA和Tandy驱动程序在[Windows 3.0中为希望在](../Page/Windows_3.x.md "wikilink")8086机器上使用彩色图形的用户提供了一种解决方法（该功能通常不支持该功能）。扩展内存支持和PS/2鼠标支持也首次出现。

[IBM](../Page/IBM.md "wikilink")将Windows OS / 2的GUI的GUI许可为Presentation Manager，并且两家公司表示它和Windows 2.0几乎相同。

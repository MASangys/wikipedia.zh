**Windows 3.x家族**是[微软](../Page/微软.md "wikilink")于1990年到1994年间所發行的[Windows系列操作平台](https://zh.wikipedia.org/wiki/Windows "wikilink")。其中的3.0版是第一个在世界上获得成功的Windows版本。使得微软的[操作系统](../Page/操作系统.md "wikilink")可以和[苹果电脑公司的](https://zh.wikipedia.org/wiki/苹果电脑 "wikilink")[麦金塔电脑以及在](https://zh.wikipedia.org/wiki/麦金塔电脑 "wikilink")[圖形使用者界面的](https://zh.wikipedia.org/wiki/圖形使用者界面 "wikilink")[Commodore的](../Page/康懋达国际.md "wikilink")[Amiga](../Page/Amiga.md "wikilink")竞争。

Windows 3.x需從[MS-DOS](../Page/MS-DOS.md "wikilink")操作系统執行。MS-DOS的文件管理程序被基于图标的程序管理程序（icon-based program Manager）以及基于列表的文件管理程序（list-based File Manager，Winfile）取代，由此简化了程序的启动。[控制面板](../Page/控制面板.md "wikilink")作为系统设置的中心，包括了诸如界面颜色主题的有限控制功能。一些捆绑的应用程序也包括在Windows内，例如“-{zh-cn:写字板;zh-tw:小作家}-”、“计算器”等。

## 沿革

3.0版发行于1990年5月22日，这个系统既包含了对[用户界面](../Page/用户界面.md "wikilink")的重要改善，也包含了对Intel [80286和](https://zh.wikipedia.org/wiki/80286 "wikilink")[80386对内存管理技术的改进](https://zh.wikipedia.org/wiki/80386 "wikilink")。为命令行式操作系统编写的[MS-DOS](../Page/MS-DOS.md "wikilink")下的程序可以在窗口中运行，使得[程序可以在](https://zh.wikipedia.org/wiki/程序 "wikilink")[多任务基础上可以使用](https://zh.wikipedia.org/wiki/多任务 "wikilink")，虽然这个版本只是为家庭用户设计的，很多游戏和娱乐程序仍然要求[DOS](../Page/DOS.md "wikilink")存取，Windows 3.0是最後一版支援[Intel 8088及](../Page/Intel_8088.md "wikilink")[8086處理器的Windows操作系统](https://zh.wikipedia.org/wiki/8086 "wikilink")。[任务管理器](../Page/任务管理器.md "wikilink")和图标都起源于Windows 3.0。

Windows 3.1发行于1992年4月6日，添加了对声音输入输出的基本[多媒体](../Page/多媒体.md "wikilink")的支持和一个CD音频播放器，以及对[桌面出版](../Page/桌面出版.md "wikilink")很有用的[TrueType](../Page/TrueType.md "wikilink")字体，并开始有了[虚拟内存](../Page/虚拟内存.md "wikilink")和[屏幕保护程序](https://zh.wikipedia.org/wiki/屏幕保护程序 "wikilink")。TrueType取代了曾在Windows 3.0版非常流行的[Adobe Type Manager](https://zh.wikipedia.org/wiki/Adobe_Type_Manager "wikilink")（ATM）的市場地位，Windows 3.1是第一版沒有提供真實模式的Windows操作系统。这是微软 Windows 开始涉足簡體中文市场的第一个版本，其简体中文版发布于1993年，多基于对英文版的简体中文化，提供了两种常用的输入法——[拼音和GB](../Page/拼音输入法.md "wikilink")2313区位用于录入汉字。

1992年4月6日发布的Windows 3.1引入了TrueType字体系统（和一组高度清晰的字体），这使Windows首次有效地成为可行的桌面发布平台。 Windows 3.0通过Adobe的Adobe Type Manager（ATM）字体系统提供了类似的功能。

Windows 3.1旨在与旧版Windows平台具有向后兼容性。 与Windows 3.0一样，版本3.1具有文件管理器和程序管理器，但是与所有以前的版本不同，Windows 3.1无法在实模式下运行。 它包括[踩地雷](../Page/踩地雷.md "wikilink")来替代[Reversi](../Page/黑白棋.md "wikilink")（尽管在某些副本中仍包含Reversi）。

Windows 3.1放弃了实模式支持，并且至少需要一台286 PC和1 MB RAM才能运行。 这样做的结果是提高了容易崩溃的Windows 3.0的系统稳定性。 删除了一些较旧的功能，例如[CGA图形支持](https://zh.wikipedia.org/wiki/CGA "wikilink")（尽管Windows 3.0的CGA驱动程序仍可在3.1上运行）以及与实模式Windows 2.x应用程序的兼容性。

添加了Truetype字体支持，可为Windows应用程序提供可缩放的字体，而不必诉诸使用诸如[Adobe Type Manager之类的第三方字体技术](https://zh.wikipedia.org/wiki/Adobe_Type_Manager "wikilink")。 Windows 3.1包括以下字体：Arial，Courier New和Times New Roman，分别为常规，[粗体](../Page/粗體.md "wikilink")，[斜体和粗斜体版本](https://zh.wikipedia.org/wiki/斜体 "wikilink")，以及Symbol（可伸缩符号的集合）。 TrueType字体可以缩放到任意大小并旋转，具体取决于调用的应用程序。\[1\]

### Windows 3.1（中欧和东欧）

发布了一个名为Windows 3.1的[中欧和东欧专用版本](https://zh.wikipedia.org/wiki/中欧 "wikilink")，该版本允许使用[西里尔字母](../Page/西里尔字母.md "wikilink")，并使用带有中欧和[东欧](../Page/东欧.md "wikilink")语言特征的变音符号的字体。微软引入了自己的[代码页](../Page/代码页.md "wikilink")（[Windows-1250](https://zh.wikipedia.org/wiki/Windows-1252 "wikilink")），并支持其违反许多国家/地区的ISO标准的使用（例如，官方的波兰语代码页为ISO-8859-2，微软忽略了该代码页，但当代[Internet Explorer版本支持该代码页](../Page/Internet_Explorer.md "wikilink") ）。 同样，微软还发布了支持[日语](../Page/日语.md "wikilink")的Windows 3.1J，在日本市场上市的第一年（1993年）就发行了146万本。

### 模块化视窗

模块化Windows是Windows 3.1的特殊版本，旨在在Tandy视频信息系统上运行。

#### Windows 3.11

Windows 3.11于1993年11月8日发布。它没有在Windows 3.1上增加许多功能改进；它没有在Windows 3.1中进行任何改进。 它主要包含错误修复程序，但由于这些修复程序被认为是一项重大改进，为操作系统的普及做出了贡献。 微软用Windows 3.11取代了所有零售版的Windows 3.1，并向当前拥有Windows 3.1的所有人提供了免费升级\[2\]。

#### Windows 3.2

1993年11月22日，微软为中国市场发布了Windows的简体中文版本。 一年后，发布了一个更新，将其标识为Windows 3.2。 因此，Windows 3.2是Windows 3.1中文版的更新版本。 该更新仅限于该语言版本，因为它仅修复了与复杂的中文输入系统有关的问题。

Windows 3.2通常由计算机制造商出售，带有十磁盘版本的MS-DOS，该版本在基本输出和某些翻译的实用程序中也具有简体中文字符。

Windows 3.2是微软专为[中国大陆](../Page/中国大陆.md "wikilink")市场开发的视窗版本，于1994年发布，仅提供简体中文版\[3\]。相比Windows 3.1，该版本在图标和界面中文化程度上更加成熟，并且它新加入了[智能ABC和](https://zh.wikipedia.org/wiki/智能ABC "wikilink")[郑码](../Page/郑码.md "wikilink")两种输入法用于录入汉字。这个版本是当年学校、企事业机关普遍使用的版本，中国大陆的软件开发商也纷纷基于此平台开发出第一代视窗下的软件，例如[WPS视窗版1.0和](../Page/WPS_Office.md "wikilink")[科利华电脑家庭教师](https://zh.wikipedia.org/wiki/科利华电脑家庭教师 "wikilink")。

## 工作组的Windows

工作组视窗系统是一个扩展，允许用户无需集中式身份验证服务器即可共享其资源并请求其他资源。 它通过[NetBIOS](../Page/NetBIOS.md "wikilink")使用[SMB协议](https://zh.wikipedia.org/wiki/SMB "wikilink")。

#### Windows for Workgroups 3.1

1992年10月发布的Windows for Workgroups 3.1（最初代号为Winball，以后称为Sparta）是Windows 3.1的扩展版本，具有本机网络支持。 它通过基于NetBIOS的NBF和/或IPX网络传输协议提供了SMB文件共享支持，并引入了[傷心小棧](../Page/傷心小棧.md "wikilink")[紙牌游戏和VSHARE](../Page/卡片遊戲.md "wikilink").386（SHARE.EXE的[VxD](../Page/VxD.md "wikilink")版本）（终止和驻留程序）。

#### Windows for Workgroups 3.11

Windows for Workgroups 3.11（最初代号为"雪球"）于1993年8月11日发布，并于1993年11月发布。它支持32位文件访问，完整的32位网络重定向器以及在它们之间共享的VCACHE.386文件缓存。 WFW 3.11放弃了标准模式支持，并且需要386机器才能运行。

需要[Winsock](../Page/Winsock.md "wikilink")程序包才能在Windows 3.x中支持[TCP / IP网络](https://zh.wikipedia.org/wiki/TCP/IP协议族 "wikilink")。通常使用第三方程序包，但在1994年8月，微软发布了一个附加程序包（代号为Wolverine），该程序包在Windows中为Workgroups 3.11提供了TCP / IP支持。 Wolverine是一个32位堆栈（可通过WinSock Thunk从16位Windows应用程序访问），使其具有优于大多数可用的第三方TCP / IP Windows[堆栈](../Page/堆栈.md "wikilink")的性能。但是，它仅与Windows for Workgroups 3.11兼容，并且缺少对[拨号的支持](https://zh.wikipedia.org/wiki/拨号连接 "wikilink")。 Wolverine堆栈是TCP / IP堆栈的早期版本，后来随[Windows 95一起提供](../Page/Windows_95.md "wikilink")，并且为16到32位兼容性层提供了早期测试平台，这对于Windows 95的成功至关重要。

在1994年发布MS-DOS 6.22之后，WFW 3.11凭借其改进的功能和更高的稳定性在很大程度上取代了Windows 3.1，用于在新的个人电脑上进行安装。

## 更新

#### Windows 3.0a

1990年12月，Microsoft发布了Windows 3.0a。 此版本包含改进的功能，可移动大于64KB的数据（原始版本一次只能操作一个RAM段）。 它还通过减少与网络，打印和低内存状况相关的不可恢复的应用程序错误（UAE）来提高稳定性。\[5\] 该版本在“帮助/关于Windows系统”对话框中显示为“ Windows 3.00a”。

## 销售市场特征

Windows 3.0是与PC兼容的制造商预先安装在硬盘上的第一个版本。 Zenith数据系统公司以前在磁盘上附带了所有装有Windows 1.0或更高版本2.x的计算机，但在Windows 3.0的开发初期就承诺将其预先安装。 的确，由于Zenith在教育市场上与Apple的直接竞争，因此Zenith部门已迫使微软难以开发图形用户界面。 但是，Zenith PC必须运行Windows的专有OEM版本，因为它们使用的硬盘具有1024字节扇区而不是正常的512字节扇区，并且不能使用标准的SWAPFILE.EXE。

Windows 3.0不能像以前的版本那样作为运行时版本使用。 由于Windows的市场渗透率低，Windows 2.x的有限使用版本通常与其他应用程序捆绑在一起（例如Ami Pro）。

Windows 3.0的标准零售和OEM发行是在高密度1.2 MB和1.44 MB软盘上。 还提供了720 KB的版本，可以从Microsoft订购360 KB的版本。 完全安装后，Windows 3.0使用了5 MB的硬盘空间。

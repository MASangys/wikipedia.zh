**Windows
傳真和掃描**是一個[微軟所開發](https://zh.wikipedia.org/wiki/微軟 "wikilink")，內建於[Microsoft
Windows中用來](https://zh.wikipedia.org/wiki/Microsoft_Windows "wikilink")[傳真以及](../Page/傳真.md "wikilink")[掃描文件的元件](../Page/影像掃描器.md "wikilink")。它僅出現在部分[Windows
Vista版本的](../Page/Windows_Vista.md "wikilink")[作業系統中](https://zh.wikipedia.org/wiki/作業系統 "wikilink")。[Windows
XP中相同功能的程式則是](../Page/Windows_XP.md "wikilink")「傳真服務」，且並非內建，使用者可能需要透過Windows
XP的安裝光碟來安裝。\[1\]使用者能夠在擁有適當所需裝置的情況下在電腦上利用這個程式收發[傳真](../Page/傳真.md "wikilink")、使用[電子郵件或傳真來發送掃描的文件](https://zh.wikipedia.org/wiki/電子郵件 "wikilink")、或使用如同電子郵件的方式轉寄傳真。Windows
傳真和掃描另外還支援單鍵掃描，讓使用者能夠只需按一個按鍵就透過連接到電腦的掃描器、網路或[藍芽連接的掃描器擷取文件](https://zh.wikipedia.org/wiki/藍芽 "wikilink")。雖然這個程式能掃描多數類型的文件，但是Windows傳真和掃描仍然比較適合用來掃描、檢視或儲存文字型式的文件。而[Windows相片圖庫則比較適合用來管理掃描的照片或圖片](https://zh.wikipedia.org/wiki/Windows相片圖庫 "wikilink")，並將它們整理於「圖片」資料夾。

不過，Windows 傳真和掃描不會內建在[Windows
Vista的](../Page/Windows_Vista.md "wikilink")[家用入門版以及](https://zh.wikipedia.org/wiki/Windows_Vista版本列表#Windows_Vista家用入門版（Home_Basic） "wikilink")[家用進階版中](https://zh.wikipedia.org/wiki/Windows_Vista版本列表#Windows_Vista家用進階版（Home_Premium） "wikilink")（它在SP1更新後被移除掉了）。Windows傳真和掃描只內建於Windows
Vista的[旗艦版以及](https://zh.wikipedia.org/wiki/Windows_Vista版本列表#Windows_Vista旗艦版（Ultimate） "wikilink")[Windows
Server 2008中](../Page/Windows_Server_2008.md "wikilink")。而在其它版本的[Windows
Vista中](../Page/Windows_Vista.md "wikilink")，Windows傳真和掃描則成為了一個選用的元件。

## 傳真

Windows 傳真和掃描的傳真檢視功能實質上就是原本適用於[Windows
XP中](../Page/Windows_XP.md "wikilink")「傳真服務」的功能提升。該程式主要的新功能包括：傳真帳戶的設定、重新設計過的介面（類似[Outlook
Express](../Page/Outlook_Express.md "wikilink")）、能夠將傳真存為草稿以及一個預覽窗格。同時，它也讓使用者能夠在一個檔案上按右鍵，並直接用傳真的方式發送。要收發傳真，使用者可以建立一個本機的傳真帳戶，使用連接到電腦上的[數據機或者是建立共享的傳真帳戶利用](https://zh.wikipedia.org/wiki/數據機 "wikilink")[網路傳真](https://zh.wikipedia.org/wiki/網路 "wikilink")[伺服器](https://zh.wikipedia.org/wiki/伺服器 "wikilink")。

## 掃描

Windows 傳真和掃描的掃描檢視功能有一個類似[Outlook
Express的](../Page/Outlook_Express.md "wikilink")[使用者介面來管理掃描的文件](https://zh.wikipedia.org/wiki/使用者介面 "wikilink")。它讓使用者能夠執行新增資料夾、重新命名資料夾等基本動作。它的預覽窗格能夠讓使用者快速的瀏覽掃描過的文件。使用者能夠針對特定的[掃描器設定掃描時的選項](../Page/影像掃描器.md "wikilink")，以便之後使用。使用者也可以在設定完郵件帳戶以後透過該程式發送掃描的文件到電子信箱中。「從掃描器傳真」的功能讓使用者能夠直接以傳真的形式發送剛掃描的文件。

除了基本的功能之外，Windows 傳真和掃描比之前的程式有了更大的生產力以及機動性，它可以說是類似「統合通訊」（Unified
Messaging）的軟體。它能夠整合多種媒體，包括傳真、電子郵件、數位文件等。

**Windows娘**（，英文：Windows tan／Windows Girl）是對[微軟操作系統](https://zh.wikipedia.org/wiki/微軟 "wikilink")[萌擬人化](../Page/萌擬人化.md "wikilink")卡通形象代言人的稱呼。

## 背景

當微軟 [Windows 7](https://zh.wikipedia.org/wiki/Windows_7 "wikilink") 操作系統於 2009 年發佈时，微軟日本為了推廣宣傳 Windows 7 Ultimate 操作系統而首次以官方名義推出了 Windows 操作系統的萌擬人化卡通形象代言人：Win7 娘[窗邊奈奈美](../Page/窗邊奈奈美.md "wikilink")（日文：窓辺ななみ）。之後微軟日本相繼推出了 Win7 娘的官方主題和宣傳影像等。

由於微軟日本官方的這一次大膽嘗試，加上[水樹奈奈](../Page/水樹奈奈.md "wikilink")親自爲 Win7 娘配音，所以當時在日本獲得了極大的成功。從而使 Window 7 操作系統在以 Mac 爲主流的日本銷售相當火爆。此後，微軟日本繼續加強二次元宣傳攻勢，於 2012 年 Windows 8 操作系統發佈前夕，推出了 Win8 娘，兩位萌擬人化卡通代言人：長頭髮的（日文：窓辺あい）和短頭髮的（日文：窓辺ゆう）。

## 主要人物

### DOS 娘

DOS 娘（[MS-DOS](../Page/MS-DOS.md "wikilink") 操作系統）是 Windows 娘的起源，形象設定上比較簡陋。身背大鍵盤，全身以黑與白爲主要色調，頭飾爲雙馬尾，總體爲蘿莉形象。只要是在 Windows 操作系統中都有她的身影。召喚方法是：開始\>運行：cmd ( 或 command )。註：這位雙馬尾蘿莉形象的 DOS 娘，設定爲只能看到黑白色的世界。

### 95 娘

95 娘（[Windows 95](../Page/Windows_95.md "wikilink") 操作系統）作為 Windows 操作系統中相對古老的系統，通常被描繪成一個很傳統的日本女性，形象通常比較淑女。棕色的頭髮、身穿和服，頭飾爲象徵着Windows 的四色徽標。她的著裝設定原形是來自日文版的 Windows 95 中的默認桌面壁紙「hana256.bmp」，穿衣打扮看起來很像日本明治時期到大正時期的女學生的形象，這個時期同時也是日本近代時期。因為當時 Windows 95 在日本國內跟 Mac OS 有過激烈的操作系統市場份額競爭，因此 95 娘的形象也多被設定爲一個久經沙場的勇者，精通日本刀等兵器。不過她最經常從事的活動是喝茶、做飯和做家務。

### 98 娘

98 娘（[Windows 98](../Page/Windows_98.md "wikilink") 操作系統）是雙胞胎姊妹，姐姐為 98 娘、妹妹為 98SE 娘，形象設定皆爲蘿莉型。98 娘姊妹均有一頭紫紅色頭髮，頭飾爲“98”數字組合。

### Me 娘

Me 娘（[Windows Me](../Page/Windows_Me.md "wikilink") 操作系統）形象設定爲綠色的頭髮，髮型有一個很鮮明的特點是附帶一縷多出來的頭髮。通常情況下穿著是女仆裝，胸前有一個「！」的符號，用過 Me 或者98 操作系統的用戶應該知道這是系統當機後重啟恢複桌面前的符號，這點也突出了 Me 操作系統當機之頻繁。

### 2000 娘

2000 娘（[Windows 2000](../Page/Windows_2000.md "wikilink") 操作系統）雖然跟 Me 娘大致是同時期的 Windows 娘，但是跟 Windows Me 操作系統相比，Windows 2000 操作系統明顯要穩定許多，因此在形象設定上 2000 娘也是一個非常聰明、專業且看上去很自信的少女。藍色頭髮，戴著眼鏡，頭飾則爲女仆式，並且頭上有一對貓耳一樣的機械裝置。藍色的衣服上還有象征著 Windows 徽標的四種顔色，而衣服和頭髮的藍色均來自于 Windows 2000 默認的藍色桌面。另外，2000 娘也常常也被設定爲像姐姐一樣的形象。由于形象設定跟 Me 娘形成了太鮮明的對比，2000 娘也常常被設定爲 Me 娘的監護人或者保護者。

### XP 娘

XP 娘（[Windows XP](../Page/Windows_XP.md "wikilink") 操作系統）。XP 娘也和 98 娘一樣是姐妹檔，不過大多數出現的都是 Pro 版的 XP 娘，Home 版的 XP 娘較為少見。XP 娘也是知名度較高，且廣受歡迎的Windows 娘。

### Vista 娘

Vista 娘（[Windows Vista](../Page/Windows_Vista.md "wikilink") 操作系統）。Windows 到了 Vista 時期，分爲多種版本：Home Basic、Home Preium、Professional 及 Utimate 等版本。當然 Vista 娘的角色設定也有許多種，但普遍來說，較爲常見的形象都是披肩長髮，頭髮呈現綠色，身著連衣裙的設定。

#### Win7 娘的效應

雖然在此之前即有同人性質的 Windows 娘在日本誕生，但官方行爲尚屬首次。而且此次微軟日本不僅爲 Win7 娘繪制了桌面背景，還讓她爲系統事件進行語音通報。由于這次 Win7 娘是微軟日本的一次大膽嘗試，其自然成爲了大家關注的話題，而日本著名聲優水樹奈奈的親身配音也極大程度地增加了 Win7 娘在日本的影響力，從而使 Windows 在 Mac 橫行的日本銷售相當火爆。此後微軟日本延續了二次元宣傳攻勢，微軟台灣亦加以效仿，製作出了“台灣之光”[藍澤光](../Page/藍澤光.md "wikilink")。其後因大受歡迎而相繼推出藍澤光的三個姐姐：藍澤優、藍澤葵以及藍澤玲。並推出以藍澤家為主推出了的聖誕特別企劃類似於戀愛冒險遊戲的《兩個人的聖誕夜》。

### Win8 娘

Win8 娘是微軟日本官方於 2012 年最新版的 Windows 8 操作系統發佈前夕，推出的萌擬人化 Windows 8 操作系統卡通形象代言人。Win8 娘也是雙胞胎姐妹，分別是長頭髮的窗边愛（）和短頭髮的窗邊優（）。兩人的出生日期同為 1996 年 11 月 18 日，兩人身高也都是 152 厘米，15 歲的高中 1 年級生。目前微軟日本已經推出了官方 Win8 娘主題。\[1\]

**σ迁移反应**（Sigmatropic reaction）是反应物一个[σ键沿着](https://zh.wikipedia.org/wiki/σ键 "wikilink")[共轭体系](../Page/共轭体系.md "wikilink")从一个位置转移到另一个位置的一类[周环反应](https://zh.wikipedia.org/wiki/周环反应 "wikilink")。\[1\]通常反应是[分子内的](https://zh.wikipedia.org/wiki/分子内 "wikilink")，同时伴随有π键的转移，但底物总的π键和σ键数保持不变。一般情况下σ迁移反应不需[催化剂](../Page/催化剂.md "wikilink")，但少数反应会受到[路易斯酸的](https://zh.wikipedia.org/wiki/路易斯酸 "wikilink")[催化](../Page/催化.md "wikilink")。

σ迁移反应符合[分子轨道对称守恒原理](../Page/分子轨道对称守恒原理.md "wikilink")，是[协同反应的一种](https://zh.wikipedia.org/wiki/协同反应 "wikilink")，也就是说原有σ键的断裂，新σ键的生成，以及[π键的转移都是经过环状](https://zh.wikipedia.org/wiki/π键 "wikilink")[过渡态](../Page/过渡态.md "wikilink")[协同一步完成的](https://zh.wikipedia.org/wiki/协同反应 "wikilink")。以[Cope重排反应为例](https://zh.wikipedia.org/wiki/Cope重排反应 "wikilink")：

## 命名

σ迁移反应的命名方法，是先将迁移前的σ键的两个原子均定为1号，从其两端分别开始编号，把新生成的σ键所连接的两个原子的编号*i*、*j*放在方括号内，记为\([i,j]\sigma\,\)迁移。习惯上\([i]\le[j]\,\)。因此，上面图示的反应是一个\([3,3]\sigma\,\)迁移反应。也可以将迁移的取代基原子写在前面，写成诸如\(H[1,5]\sigma\,\)迁移反应的形式。

迁移后，如果新形成的σ键在π体系的同侧，称之为[同面迁移](https://zh.wikipedia.org/wiki/同面 "wikilink")；反之，则称为[异面迁移](https://zh.wikipedia.org/wiki/异面 "wikilink")。[图示](http://www.chemgapedia.de/vsengine/vlu/vsc/en/ch/2/vlu/pericyclische_reaktionen/pericyclisch_sigmatrop.vlu/Page/vsc/en/ch/2/oc/reaktionen/formale_systematik/pericyclische_reaktionen/sigmatrop/stereochemie_1_n.vscml.html)它们也可以写在标记中，同面以s表示，异面以a表示。下面的\(H[1,5]\sigma\,\)迁移反应对于氢原子和戊二烯基来说都是同面的，因此是一个\(H[1s,5s]\sigma\,\)迁移反应。

[SigmatropicHydrideShifts.png](https://zh.wikipedia.org/wiki/File:SigmatropicHydrideShifts.png "fig:SigmatropicHydrideShifts.png")

在\(C[i,j]\sigma\,\)迁移反应中，如果和迁移键相连的碳原子为[手性](../Page/手性.md "wikilink")碳，而且迁移后，该碳原子仍在键断裂处形成新键，称为[构型保持](https://zh.wikipedia.org/wiki/构型保持 "wikilink")；如果新键在原键断裂处的相反位置形成，则称为[构型翻转](https://zh.wikipedia.org/wiki/构型翻转 "wikilink")。[同芳香性可以在标记中用homo表示](https://zh.wikipedia.org/wiki/同芳香性 "wikilink")。

## 机理

以\([1,j]\sigma\,\)为例。根据[前线轨道理论](../Page/前线轨道理论.md "wikilink")，假定反应中发生迁移的σ键首先发生[均裂](https://zh.wikipedia.org/wiki/均裂 "wikilink")，生成一个氢原子（或碳[自由基](../Page/自由基.md "wikilink")）和一个奇数碳的[共轭体系](../Page/共轭体系.md "wikilink")自由基，而反应的实质是氢原子（或碳自由基）在奇数碳的共轭体系自由基上发生的迁移。一般认为，反应中真正涉及到的是奇数碳共轭体系中含有单电子的[前线轨道](https://zh.wikipedia.org/wiki/前线轨道 "wikilink")，它既是[最高占有分子轨道](https://zh.wikipedia.org/wiki/最高占有分子轨道 "wikilink")（HOMO），也是[最低未占有分子轨道](https://zh.wikipedia.org/wiki/最低未占有分子轨道 "wikilink")（LUMO），而且反应的[立体选择性也完全取决于该前线轨道的对称性](https://zh.wikipedia.org/wiki/立体选择性 "wikilink")。基态时，它是一个[非键轨道](https://zh.wikipedia.org/wiki/非键轨道 "wikilink")，具有以下所示的分子轨道示意图，其中的数字与上面命名一节的数字含义吻合。

[Sigmatropic1.png](https://zh.wikipedia.org/wiki/File:Sigmatropic1.png "fig:Sigmatropic1.png")

可以看出，该前线轨道中，所有偶数碳上的[电子云](../Page/电子云.md "wikilink")密度都为零，而奇数碳上的电子云密度数值相等，波相交替变化。奇数碳又分为两种，一种为4n+1类型，另一种为4n-1类型；前者前线轨道具有镜面对称性，而后者具有镜面反对称性。激发态时，电子发生跃迁，因此这两种类型的镜面对称性发生转换，前者具镜面反对称性，后者具镜面对称性。

由于σ迁移反应是符合[分子轨道对称守恒原理](../Page/分子轨道对称守恒原理.md "wikilink")的反应，因此形成新的σ键时，必须发生的是同位相的重叠。以\(H[1,j]\sigma\,\)迁移为例，若它属于4n+1类型，则为了满足对称性的要求，基态时，[单占轨道对镜面对称](https://zh.wikipedia.org/wiki/单占轨道 "wikilink")，只有同面迁移是允许的；激发态时，单占轨道对镜面反对称，只有异面迁移是允许的。对于4n-1类型则正好相反。如果迁移的原子是手性碳原子，那么在进行\([1,4n-1]\sigma\,\)类型的迁移反应时，4n-1与1号原子的波相相反，因此只有用断键轨道的另一瓣（符号相反）与奇数碳共轭体系发生重叠。自然，新键在断键位置的相反方向形成，手性碳发生构型翻转。

\([i,j]\sigma\,\)迁移反应也可以用类似的方法来分析。迁移的σ键发生均裂后，形成两个奇数碳共轭体系自由基，σ迁移反应中起决定作用的是它们两个的单占[前线轨道](https://zh.wikipedia.org/wiki/前线轨道 "wikilink")。σ迁移反应具有以下选择性规则：\[2\]

| 参加环型过渡态的π电子数\((1+j)\,\)或\((i+j)\,\) | 4n+2                 | 4n                   |
| ----------------------------------- | -------------------- | -------------------- |
| 反应分类                                |                      |                      |
| \(H[1,j]\sigma\,\)迁移                | \(C[1,j]\sigma\,\)迁移 | \(C[i,j]\sigma\,\)迁移 |
| 构型保持                                | 构型翻转                 |                      |
| 同面迁移                                | 同面迁移                 | 异面迁移                 |
| 允许                                  | 禁阻                   | 禁阻                   |
| 异面迁移                                | 异面迁移                 | 同面迁移                 |
| 禁阻                                  | 允许                   | 允许                   |
| 注：“允许”指“对称性允许”，“禁阻”指“对称性禁阻”。        |                      |                      |

## 例子

最常见的σ迁移反应有\[3,3\]σ迁移反应和\[\[2,3-σ迁移反应|\[2,3\]σ迁移反应\]\]两种，前者包括[Cope重排反应](https://zh.wikipedia.org/wiki/Cope重排反应 "wikilink")、[Claisen重排反应](https://zh.wikipedia.org/wiki/Claisen重排反应 "wikilink")、[Carroll重排反应](../Page/Carroll重排反应.md "wikilink")以及[Fischer吲哚合成](https://zh.wikipedia.org/wiki/Fischer吲哚合成 "wikilink")，后者包括[Gassman吲哚合成](../Page/Gassman吲哚合成.md "wikilink")以及一大类含有[杂原子](../Page/杂原子.md "wikilink")迁移的反应。按迁移原子来分，则σ迁移反应常见的两类是“甲基转位”和“负氢转位”反应。

下面图中是[维生素D3生物合成的最后几步反应](https://zh.wikipedia.org/wiki/维生素D3 "wikilink")。首先是[7-脱氢胆固醇](../Page/7-脱氢胆固醇.md "wikilink")受光照发生逆[电环化反应](../Page/电环化反应.md "wikilink")，生成含有[1,3,5-己三烯结构的](https://zh.wikipedia.org/wiki/1,3,5-己三烯 "wikilink")[前维生素D3](../Page/前维生素D3.md "wikilink")。而后前维生素D3在加热情况下，发生一个\(H[1,7]\sigma\,\)迁移反应，得到维生素D3。 [VitaminD3Biosynthesis.png](https://zh.wikipedia.org/wiki/File:VitaminD3Biosynthesis.png "fig:VitaminD3Biosynthesis.png")

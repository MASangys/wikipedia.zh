**三分型配列**（Tripartite
aligement），又稱**三分法配列**、**作宾格配列**（Ergative–accusative
aligement），是一类[配列方式](../Page/配列.md "wikilink")，即在[句法或](../Page/句法.md "wikilink")[形态上](../Page/词法学.md "wikilink")，将[不及物动词的单一变元](../Page/不及物動詞.md "wikilink")（**S**）、及物动词的受事论元（**P**）以及[及物动词的](../Page/及物動詞.md "wikilink")[施事论元](../Page/施事.md "wikilink")（**A**）分别对待。\[1\]\[2\]\[3\]呈现这种配列的语言称为**三分型语言**、**三分法语言**或者**作宾格语言**；在[格变化上](../Page/格_\(语法\).md "wikilink")，上述三种论元就分别被标注为三种格。三分型语言十分罕见。\[4\]

## 编码性质

如果一门语言有形态上的格变化，那么谓语中心词的论元一般会如下标记：

  - 不及物动词的单一核心变元（**S**）标记为[作格](../Page/作格.md "wikilink")；
  - 及物动词的受事论元（**P**）标记为[宾格](../Page/宾格.md "wikilink")；
  - 及物动词的施事论元（**A**）标记为[作格](../Page/作格.md "wikilink")。

比如语里面，不及物动词句的[不及物格变元没有任何后缀](https://zh.wikipedia.org/wiki/不及物格 "wikilink")，而动词则有一个[第三人稱](https://zh.wikipedia.org/wiki/第三人稱 "wikilink")[后缀](https://zh.wikipedia.org/wiki/后缀 "wikilink")
*hi-*
与之形成[一致](../Page/一致_\(語法\).md "wikilink")；而及物动词句的[作格论元](../Page/作格.md "wikilink")[後綴为](../Page/後綴.md "wikilink")*-n(i)m*，[宾格论元後綴为](../Page/宾格.md "wikilink")*-ne*，同时动词有一个第三人称后缀
*pée-* 与它们形成一致：\[5\]

|     |                |         |
| --- | -------------- | ------- |
| 1a. | Hi-páayn-a     | háama.  |
|     | 3SG.INT-到达-ASP | 男人.INTR |
|     | **不及物格**       | **S**   |
|     | 男人到了。 \[6\]    |         |

|     |              |                |              |
| --- | ------------ | -------------- | ------------ |
| 1b. | Háama-**nm** | pée-'wi-**ye** | wewúkiye-ne. |
|     | 男人-ERG       | 3SG.TRA-射击-ASP | 加拿大马鹿-ACC    |
|     | **A 作格**     | **第三人称及物**     | **P 宾格**     |
|     | 男人射死了鹿。\[7\] |                |              |

[阿伊努语的](../Page/阿伊努語.md "wikilink")[代名词](../Page/代詞.md "wikilink")[词缀也呈现三分型配列](https://zh.wikipedia.org/wiki/词缀 "wikilink")，如[第一人称代词的](https://zh.wikipedia.org/wiki/第一人稱 "wikilink")[作格作](../Page/作格.md "wikilink")
ku-，[不及物格作](https://zh.wikipedia.org/wiki/不及物格 "wikilink")
-An，[宾格作](../Page/宾格.md "wikilink")
-en-。像一样，阿伊努语同时具有[主宾型语言常见的](https://zh.wikipedia.org/wiki/主宾型语言 "wikilink")[被动语态和](../Page/被動語態.md "wikilink")[作通型语言常见的](https://zh.wikipedia.org/wiki/作通型语言 "wikilink")[反被动语态](../Page/反被動語態.md "wikilink")，这也是三分型语言的特征。\[9\]

## 分布

除上述语言外，[汉特语的](../Page/漢特語.md "wikilink")[瓦赫方言](https://zh.wikipedia.org/wiki/瓦赫河_\(鄂畢河\) "wikilink")、\[10\]、（[单数代名词](https://zh.wikipedia.org/wiki/單數_\(語法\) "wikilink")）、（过去时）等。\[11\]一些[人工語言](../Page/人工語言.md "wikilink")，尤其是[工化語言](../Page/工化語言.md "wikilink")，会使用三分型配列，以[納美語最为显著](../Page/納美語.md "wikilink")。

三分型语言总的来说非常稀少\[12\]，因为从动机来看格只需分辨及物动词句的 A 和 P 即可，故大多数语言中 S 不需要自己独立的格，可以跟
A 或 P 任意一方作相同标记，否则格会过剩。\[13\]

**三国杀标准版**是由游卡桌游早期发行的《三国杀推广版》发展而来，是[三国杀](../Page/三国杀.md "wikilink")游戏的核心版本\[1\]，适合2～10人游玩。可以配合各种[三国杀·扩充包游玩以豐富游戏內容](https://zh.wikipedia.org/wiki/三国杀#扩充包 "wikilink")。2011年游卡桌游推出了新包装的标准版\[2\]，改良了游戏说明、卡牌描述，并更改了游戏牌的背面图案。2012年通过代理商疯桌游和ZiKo Games分别在台湾发售了繁体中文标准版\[3\]，在美国发行了《LTK·Legends of The Three Kingdoms（标准版）》。2013年推出了“三国杀2013新标准版”，其中更换了3名武将的技能与插画，并新增了2个新武将。\[4\]

## 游戏模式

  - 在標準版中，主要的游戏模式为标准身份局模式。
  - 详细规则参见：[三国杀游戏模式·标准模式](https://zh.wikipedia.org/wiki/三国杀游戏模式#标准模式 "wikilink")。

## 基本规则

  - 在標準版中，出现了主公技、锁定技两种特殊技能类型。
  - 技能类型说明参见：[三国杀游戏规则·技能類型](https://zh.wikipedia.org/wiki/三国杀游戏规则#技能類型 "wikilink")

## 游戏卡牌

  - 在標準版中，包含四种卡牌：身份牌、体力牌、武将牌和游戏牌。

### 身份牌

  - 標準版中共有10张身份牌。

|        |        |        |        |
| ------ | ------ | ------ | ------ |
| **名稱** | **顔色** | **圖案** | **數量** |
| 主公     | 红      | 头盔     | 1      |
| 忠臣     | 黄      | 花朵     | 3      |
| 反賊     | 绿      | 火焰     | 4      |
| 內-{奸}- | 蓝      | 眼睛     | 2      |

每张身份牌对游戏有一定的影响。 主公是最重要的角色，任务是消灭所有反贼和内奸。 忠臣的任务则是保护主公，并攻击反贼和内奸。 反贼的任务是把主公及其手下的忠臣杀死，获得游戏的胜利。 内奸是徘徊在主公与反贼边缘的人，任务是除掉游戏里的其他人，成为最后的胜利者。

### 体力牌

  - 標準版中共有10张体力牌。

|         |        |
| ------- | ------ |

### 武將牌

  - 標準版中共有27张武将牌，武将角色以[三国](../Page/三国.md "wikilink")人物为背景，分属魏、蜀、吳、群雄四种势力。
  - 详细武将卡牌分类规则参见：[三国杀游戏卡牌·武將牌](https://zh.wikipedia.org/wiki/三国杀游戏卡牌#武將牌 "wikilink")，完整游戏武将列表参见：[三国杀武将列表](../Page/三国杀武将列表.md "wikilink")。

**三星Galaxy Note Edge(也稱三星Galaxy Note 4 Edge)**是[韩国](https://zh.wikipedia.org/wiki/韩国 "wikilink")[三星电子制造的最新](https://zh.wikipedia.org/wiki/三星电子 "wikilink")[大屏幕旗舰级](../Page/平板手机.md "wikilink")[智能手机](../Page/智能手机.md "wikilink")，於2014年9月3日在[德国](../Page/德国.md "wikilink")举行的[柏林国际广播展上发布](https://zh.wikipedia.org/wiki/柏林国际广播展 "wikilink")。与同期发布的[Galaxy Note 4相比](../Page/三星Galaxy_Note_4.md "wikilink")，Note Edge的最大卖点是右侧附有一块可独立运作的曲面屏幕，该屏幕运用三星最新的柔性[AMOLED](../Page/AMOLED.md "wikilink")工艺，与主屏幕是无缝一体化。\[1\]它的顯示器在設備右側彎曲顯示，可用作邊欄顯示應用程序快捷方式，通知和其他信息。

## 開發和發布

在2013年消費電子展上，三星展示了“Youm”概念 - 用於融合柔性顯示屏的智能手機原型。一個原型機的屏幕沿著手機的右邊緣彎曲，另一個原型機的屏幕在手機底部彎曲。三星解釋說，可以使用額外的“strip”來顯示附加信息，例如通知或新聞摘要。\[2\]\[3\]\[4\]“Youm”概念將作為Galaxy Note Edge的一部分出現，Galaxy Note Edge 將於2014年9月4日與Galaxy Note 4一同推出。\[5\]三星策略師Justin Denison解釋說，該公司喜歡在其產品中冒風險，繼續說“我們不是一家做一次性事業的公司,我們喜歡做大事，並承擔責任。\[6\]

## 產品規格

### 硬件和設計

Galaxy Note Edge在設計上與Galaxy Note 4不同是按照Galaxy Note 3設計，配有金屬框架和塑料皮革後蓋。該器件具有[Exynos 5 Octa 5433](https://zh.wikipedia.org/wiki/Exynos "wikilink")（韓國版）或[Qualcomm Snapdragon 805](https://zh.wikipedia.org/wiki/Snapdragon "wikilink")（國際版），3 GB RAM以及32或64 GB可擴展存儲。與其他Galaxy Note系列設備一樣，它包含一個S Pen [手寫筆](https://zh.wikipedia.org/wiki/手寫筆 "wikilink")，可用於筆輸入，繪圖和手寫。與其他最近的三星旗艦設備類似，它還包括心率傳感器和指紋掃描儀。Galaxy Note Edge採用5.6英寸 “[Quad HD +](https://zh.wikipedia.org/wiki/显示分辨率列表 "wikilink")[Super AMOLED](https://zh.wikipedia.org/wiki/Super_AMOLED "wikilink")”顯示屏，其中包含一個160像素寬的色譜柱，可在曲線上環繞設備的側面。\[7\]\[8\]\[9\]\[10\]該設備包括有一個1600萬像素的後置攝像頭，具有背照式傳感器，[光學防抖和](https://zh.wikipedia.org/wiki/光学防抖 "wikilink")4K視頻錄製功能，以及一個3.7萬像素的前置攝像頭。在2017年，現在**Edge**仍舊可以在二手市場上獲得，並且是最後一款具有可更換電池的手機之一。

### 軟件

Galaxy Note Edge附帶[Android 4](https://zh.wikipedia.org/wiki/Android_4.4 "wikilink").4.4“KitKat”和三星的[TouchWiz](../Page/TouchWiz.md "wikilink")界面和軟件套件，與Note 4類似。屏幕的弧形邊緣用作各種用途的[側邊欄](https://zh.wikipedia.org/wiki/側邊欄 "wikilink")：可用於顯示不同的面板，包括常用應用程序的快捷方式，通知，[新聞](../Page/新闻.md "wikilink")，[股票](../Page/股票.md "wikilink")，[體育](../Page/体育.md "wikilink")，[社交網絡](../Page/社交網路服務.md "wikilink")，音樂和視頻播放器的播放控制，相機控制，數據使用和小遊戲的顯示。工具也可通過面板使用，包括標尺，秒錶，計時器，錄音筆和手電筒按鈕。一個[軟件開發工具包可用於開發人員編碼面板](../Page/软件开发工具包.md "wikilink"); 其他面板可以通過[Galaxy Apps獲得](https://zh.wikipedia.org/wiki/Galaxy_Apps "wikilink")。熄屏顯示“時鐘”模式允許邊沿屏幕在預定時間段內在不使用時顯示[Always on Display](https://zh.wikipedia.org/wiki/Always_on_Display "wikilink")。由於AMOLED顯示器的性質（根本不打開像素而呈現黑色），此模式不會顯著消耗電池電量，但根據軟件限制，它一次不能超過12小時。

**三氟甲磺酸盐**是一种热力学非常稳定的有机盐，其[钠](../Page/钠.md "wikilink")、[硼](../Page/硼.md "wikilink")、[银的无水形式盐的熔点超过](https://zh.wikipedia.org/wiki/银 "wikilink")350°C。这些盐可直接通过三氟甲磺酸和金属氢氧化物或金属碳酸盐在水中反应制备。另外，它们还可通过金属盐酸盐和纯三氟甲磺酸或[三氟甲磺酸银反应制备](../Page/三氟甲磺酸银.md "wikilink")，或通过[三氟甲磺酸钡与金属硫酸盐在水中反应制备](https://zh.wikipedia.org/wiki/三氟甲磺酸钡 "wikilink")：
\[1\]

  -
    MCl<sub>n</sub> + n HOTf → M(OTf)<sub>n</sub> + n HCl
    MCl<sub>n</sub> + n AgOTf → M(OTf)<sub>n</sub> + n AgCl ↓
    M(SO<sub>4</sub>)<sub>n</sub> + n Ba(OTf)<sub>2</sub> →
    M(OTf)<sub>2n</sub> + n BaSO<sub>4</sub> ↓

## 用途

三氟甲磺酸盐在[有机化学中可用作](../Page/有机化学.md "wikilink")[路易斯酸](https://zh.wikipedia.org/wiki/路易斯酸 "wikilink")，这是因为它们在水中的稳定性比起一些传统的催化剂（如[氯化铝](../Page/氯化铝.md "wikilink")）更佳，其中尤其有用。这里的镧族元素盐形式为Ln(OTf)<sub>3</sub>，其中Ln为La、Ce、Pr、Nd、Sm、Eu、Gd、Tb、Dy、Ho、Er、Tm、Yb、Lu、Y元素。另一种流行的催化剂为[三氟甲磺酸钪](https://zh.wikipedia.org/wiki/三氟甲磺酸钪 "wikilink")，它常用于[羟醛反应和](../Page/羟醛反应.md "wikilink")[狄尔斯－阿尔德反应](https://zh.wikipedia.org/wiki/狄尔斯－阿尔德反应 "wikilink")，如在[Mukaiyama羟醛反应反应中](https://zh.wikipedia.org/wiki/Mukaiyama羟醛反应 "wikilink")，[苯甲醛和](../Page/苯甲醛.md "wikilink")[环己酮的](../Page/环己酮.md "wikilink")[烯醇硅醚在上述催化剂下发生反应得到](../Page/烯醇硅醚.md "wikilink")81%的[收率](https://zh.wikipedia.org/wiki/收率 "wikilink")\[2\]。下列反应为通过[钪盐催化的反应](../Page/钪.md "wikilink")：

[數學中](https://zh.wikipedia.org/wiki/數學 "wikilink")，**三維球面**（英文常寫作**3-sphere**）是[球面](../Page/球面.md "wikilink")在高維空間中的類比客體。它由四維[歐幾里得空間中與一固定中心點等距離的所有點所組成](https://zh.wikipedia.org/wiki/歐幾里得空間 "wikilink")。尋常的球面（或者說[二維球面](https://zh.wikipedia.org/wiki/二維球面 "wikilink")）是一個二維[表面](https://zh.wikipedia.org/wiki/表面 "wikilink")，而三維球面是一個具有三個[維度](../Page/維度.md "wikilink")的幾何客體，這樣的幾何客體都可以歸類為[三維流形](https://zh.wikipedia.org/wiki/三維流形 "wikilink")（3-manifold）。

三維球面也稱作[超球面](../Page/超球面.md "wikilink")（hypersphere），雖然這個辭彙可以更廣義地代表任何*n*維球面，而*n* ≥ 3。

## 定義

以[座標表示](https://zh.wikipedia.org/wiki/座標 "wikilink")，三維球面具有中心（*C*<sub>0</sub>, *C*<sub>1</sub>, *C*<sub>2</sub>, *C*<sub>3</sub>）及半徑*r* 乃在**R**<sup>4</sup>符合條件

\[\sum_{i=0}^3(x_i - C_i)^2 = ( x_0 - C_0 )^2 + ( x_1 - C_1 )^2 + ( x_2 - C_2 )^2+ ( x_3 - C_3 )^2 = r^2 \,\] 的所有點的[集合](https://zh.wikipedia.org/wiki/集合 "wikilink")： （*x*<sub>0</sub>, *x*<sub>1</sub>, *x*<sub>2</sub>, *x*<sub>3</sub>）。

三維球面球心在原點，而半徑是1的稱為**單位三維球面**(unit 3-sphere)，常寫作*S*<sup>3</sup>：

\[S^3 = \left\{(x_0,x_1,x_2,x_3)\in\mathbb{R}^4 : x_0^2 + x_1^2 + x_2^2 + x_3^2 = 1\right\}\]。

方便性上，常將**R**<sup>4</sup>另外以[複數](https://zh.wikipedia.org/wiki/複數 "wikilink")**C**<sup>2</sup>或[四元數](https://zh.wikipedia.org/wiki/四元數 "wikilink")(quaternions)**H**等價表示。單位三維球面則可寫為

\[S^3 = \left\{(z_1,z_2)\in\mathbb{C}^2 : |z_1|^2 + |z_2|^2 = 1\right\}\] 或

\[S^3 = \left\{q\in\mathbb{H} : |q| = 1\right\}\]。

最後一個表示法常是最有用的。其將三維球面描述為所有**單位四元數**（[絕對值為](https://zh.wikipedia.org/wiki/絕對值 "wikilink")1的四元數）的集合。正如同所有單位複數的集合在[複數幾何是重要的](https://zh.wikipedia.org/wiki/複數幾何 "wikilink")，所有單位四元數的集合在四元數幾何中也是重要的。

[平面几何中](https://zh.wikipedia.org/wiki/平面几何 "wikilink")，一点关于给定[三角形的](../Page/三角形.md "wikilink")**三线坐标**描述了它到三角形三条边的相对[距离](../Page/距离.md "wikilink")。三线坐标是[齐次坐标的一个例子](../Page/齐次坐标.md "wikilink")，经常简称为**三线**。

## 例子

[内心有三线](https://zh.wikipedia.org/wiki/内心 "wikilink")1:1:1，这就是说，从三角形*ABC*的内心到边*BC*、*CA*、*AB*的有向距离和实际距离[有序三元组](https://zh.wikipedia.org/wiki/有序组 "wikilink")（*r*,
*r*,
*r*）成比例，这里*r*是三角形*ABC*内切圆的半径。注意到记号*x*:*y*:*z*用比例[冒号区分三线和实际有向距离](../Page/冒号.md "wikilink")。实际距离有序三元组（*kx*,
*ky*, *kz*），能从比例*x* : *y* : *z*得到，利用[面积关系不难算得](../Page/面积.md "wikilink")

  -
    \(k = \frac{2\sigma}{ax + by + cz},\)

这里*a*, *b*, *c*分别是边长*BC*、*CA*、*AB*，σ = *ABC*的面积。（“逗号记法”应该避免使用。因为记号（*x*,
*y*, *z*）意味着是一个有序三元组，不允许（*x*, *y*, *z*） =（2*x*, 2*y*,
2*z*）之类运算；然而“比号记法”允许*x* : *y* : *z* = 2*x*
: 2*y* : 2*z*。）

设*A*、*B*和*C*不仅表示三角形的顶点，也是在相应顶点的角。一些熟知点的三线如下：

[Trilinear_coordinates.svg](https://zh.wikipedia.org/wiki/File:Trilinear_coordinates.svg "fig:Trilinear_coordinates.svg")

:\* *A* = 1 : 0 : 0

:\* *B* = 0 : 1 : 0

:\* *C* = 0 : 0 : 1

:\* [内心](https://zh.wikipedia.org/wiki/内心 "wikilink") = 1 : 1 : 1

:\* *A*-[旁心](https://zh.wikipedia.org/wiki/旁心 "wikilink") = −1 : 1 : 1

:\* *B*-旁心 = 1 : −1 : 1

:\* *C*-旁心 = 1 : 1 : −1

:\* [外心](https://zh.wikipedia.org/wiki/外心 "wikilink") = cos *A* : cos
*B* : cos *C*

:\* [垂心](https://zh.wikipedia.org/wiki/垂心 "wikilink") = sec *A* : sec
*B* : sec *C*

:\* [九点圆圆心](https://zh.wikipedia.org/wiki/九点圆圆心 "wikilink") = cos（*B* −
*C*）: cos（*C* − *A*）: cos（*A* − *B*）

:\* [重心](https://zh.wikipedia.org/wiki/重心 "wikilink") = *bc* : *ca* :
*ab* = 1/*a* : 1/*b* : 1/*c* = csc *A* : csc *B* : csc *C*

:\* [类似重心](https://zh.wikipedia.org/wiki/类似重心 "wikilink") = *a* : *b* :
*c* = sin *A* : sin *B* : sin *C*

注意到，内心一般不是[重心](https://zh.wikipedia.org/wiki/重心 "wikilink")，重心有[重心坐标](../Page/重心坐标.md "wikilink")1:1:1（它们和实际有向面积*BGC*、*CGA*、*AGB*成比例，这里G
= 重心）。

## 公式

利用三线坐标可将许多代数方法运用于三角形几何。比如，三点

  -
    *P = p* : *q* : *r*
    *U = u* : *v* : *w*
    *X = x* : *y* : *z*

是[共线的](https://zh.wikipedia.org/wiki/共线点 "wikilink")，[当且仅当](../Page/当且仅当.md "wikilink")[行列式](../Page/行列式.md "wikilink")：

\[D = \begin{bmatrix}p&q&r\\
u&v&w\\x&y&z\end{bmatrix}\]

等于0。这性质的对偶是三条直线

  -
    *pα + qβ + rγ = 0*
    *uα + vβ + wγ = 0*
    *xα + yβ + zγ = 0*

[交于一点](https://zh.wikipedia.org/wiki/共点直线 "wikilink")（若[无穷远点](https://zh.wikipedia.org/wiki/无穷远点 "wikilink")，即[平行](../Page/平行.md "wikilink")）当且仅当*D
= 0*。

另外可算得三角形*PUX*的面积= *KD*，这里*K = abc/8σ*<sup>*2*</sup>，如果*PUX*和*ABC*
[定向相同](https://zh.wikipedia.org/wiki/定向_\(几何\) "wikilink")，定向相反则*K
= - abc/8σ*<sup>*2*</sup>。

许多[三次曲线用三线容易表示](https://zh.wikipedia.org/wiki/三次曲线 "wikilink")。比如，中枢自等共轭三次曲线*Z（U,P）*，作为点*X*的轨迹使得*X*的*P*-等共轭点位于直线*UX*上，由行列式方程

\[\begin{bmatrix}x&y&z\\
qryz&rpzx&pqxy\\u&v&w\end{bmatrix} = 0\]

确定。一些有名的三次曲线*Z（U,P）*：

  -
    Thomson三次曲线：*Z（X（2）,X (1))*，这里''X (2) =
    *[重心](https://zh.wikipedia.org/wiki/重心 "wikilink")，*X (1) =
    ''[内心](https://zh.wikipedia.org/wiki/内心 "wikilink")
    Feuerbach三次曲线：*Z（X（5）,X (1))*，这里''X (5) =
    ''[费尔巴哈点](https://zh.wikipedia.org/wiki/费尔巴哈点 "wikilink")
    Darboux三次曲线：*Z（X（20）,X (1))*，这里''X (20) = ''
    Neuberg三次曲线：*Z（X（30）,X (1))*，这里''X (30) =
    ''[欧拉无穷远点](https://zh.wikipedia.org/wiki/欧拉无穷远点 "wikilink")

## 坐标变换

一点具有三线*α* : *β* : *γ*，则重心坐标为*aα* : *bβ* : *cγ*，这里*a*, *b*,
*c*是三角形三条边长。相反地，重心坐标为*α* : *β* : *γ*的点有三线*α/a* :
*β/b* : *γ/c*。

三线坐标和2维[笛卡尔坐标之间存在转换公式](https://zh.wikipedia.org/wiki/笛卡尔坐标 "wikilink")。给定一个参考三角形*ABC*，将顶点*B*的位置表示成一个笛卡尔坐标的有序组，将其代数地写成一个以顶点*C*为起点的[向量](../Page/向量.md "wikilink")<u>*a*</u>。类似地定义顶点*A*为<u>b</u>。然后任何点*P*关于参考三角形*ABC*能定义一个2维笛卡尔坐标系，写成向量<u>*p*</u>
= α<u>*a*</u> + β<u>*b*</u>。如果点*P*有三线坐标*x:y:z*，那么变换公式是：

  -
    \(x:y:z =  \frac{\beta}{a} : \frac{\alpha}{b} : \frac{1 - \alpha - \beta}{c} ,\)

反过来，

  -
    \(\alpha = \frac{by}{ax + by + cz},\, \beta = \frac{ax}{ax + by + cz}.\)

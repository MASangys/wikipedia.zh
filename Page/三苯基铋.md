**三苯基铋**是一种[金属有机化合物](https://zh.wikipedia.org/wiki/金属有机化合物 "wikilink")，[化学式](../Page/化学式.md "wikilink")为Bi(C<sub>6</sub>H<sub>5</sub>)<sub>3</sub>，或简写为BiPh<sub>3</sub>。

## 制备

  - 三苯基铋可以由[氯化铋](../Page/氯化铋.md "wikilink")(III)和[格氏试剂](../Page/格氏试剂.md "wikilink")反应得到：\[1\]\[2\]

\[\mathsf{C_6H_5Br + Mg \ \xrightarrow{}\ C_6H_5MgBr }\]

\[\mathsf{BiCl_3 + 3C_6H_5MgBr \ \xrightarrow{}\ Bi(C_6H_5)_3 + 3MgBrCl }\]

  - 铋钠合金和[氯苯](../Page/氯苯.md "wikilink")作用，可以得到三苯基铋：

\[\mathsf{Na_3Bi + 3C_6H_5Cl \ \xrightarrow{}\ Bi(C_6H_5)_3 + 3NaCl }\]

  - 通过氯化铋和[二苯基汞](../Page/二苯基汞.md "wikilink")的反应也可以制备三苯基铋：

\[\mathsf{2BiCl_3 + 3Hg(C_6H_5)_2 \ \xrightarrow{}\ 2Bi(C_6H_5)_3 + 3HgCl_2 }\]

## 化学性质

  - 三苯基铋在空气中稳定，可以和浓[盐酸](../Page/盐酸.md "wikilink")反应：\[3\]

\[\mathsf{(C_6H_5)_3Bi + 3 HCl \ \xrightarrow {}\ 3C_6H_6 + BiCl_3 }\]

  - 和[硝酸](../Page/硝酸.md "wikilink")反应产生(C<sub>6</sub>H<sub>4</sub>NO<sub>2</sub>)<sub>3</sub>Bi(NO<sub>3</sub>)<sub>2</sub>，产物为淡黄色晶体。
  - 也可以和[卤素](../Page/卤素.md "wikilink")反应：

\[\mathsf{(C_6H_5)_3Bi + Cl_2 \ \xrightarrow {}\ Bi(C_6H_5)_3Cl_2 }\]

\[\mathsf{(C_6H_5)_3Bi + Br_2 \ \xrightarrow {}\ Bi(C_6H_5)_3Br_2 }\]

  -

      -
        但碘只能产生(C<sub>6</sub>H<sub>5</sub>)BiI<sub>2</sub>和(C<sub>6</sub>H<sub>5</sub>)<sub>2</sub>BiI。

## 用途

三苯基铋可用作固化催化剂。\[4\]\[5\]

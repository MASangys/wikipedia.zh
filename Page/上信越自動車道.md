**上信越自動車道**（）是起於[群馬縣](../Page/群馬縣.md "wikilink")[藤岡市](https://zh.wikipedia.org/wiki/藤岡市 "wikilink")，經[長野縣](https://zh.wikipedia.org/wiki/長野縣 "wikilink")[長野市](../Page/長野市.md "wikilink")至[新潟縣](https://zh.wikipedia.org/wiki/新潟縣 "wikilink")[上越市](../Page/上越市.md "wikilink")的[高速公路](../Page/日本高速公路.md "wikilink")（[高速自動車國道](https://zh.wikipedia.org/wiki/高速自動車國道 "wikilink")），現由[東日本高速公路](../Page/東日本高速公路.md "wikilink")負責管理、營業，略稱為**上信越道**（、JOSHIN-ETSU
EXPWY）。高速公路路線編號被編為「**E18**」。

## 概要

[缩略图](https://zh.wikipedia.org/wiki/File:TrafficJam_near_Nakago_InterChange_20100501.jpg "fig:缩略图")
[國土開發幹線自動車道](../Page/國土開發幹線自動車道.md "wikilink")的路線名稱為[關越自動車道](../Page/關越自動車道.md "wikilink")上越線，上信越自動車道為藤岡JCT以西區間的道路名稱。

本高速道路為[東京至](../Page/東京都區部.md "wikilink")[富山](../Page/富山市.md "wikilink")、[金澤等](../Page/金澤市.md "wikilink")[北陸地方](../Page/北陸地方.md "wikilink")連結的高速道路中路徑最短，連結[關東地方](../Page/關東地方.md "wikilink")與長野縣[北信地方](https://zh.wikipedia.org/wiki/北信地方 "wikilink")、新潟縣[上越地方](../Page/上越地方.md "wikilink")，擔當關東北陸間的物流的[動脈](https://zh.wikipedia.org/wiki/動脈 "wikilink")。

## 設施

  - 交流道（IC）編號欄背景色<span style="color:#BFB">■</span>區間是已通車區間，設施欄背景色<span style="color:#CCC">■</span>區間是未通車區間與未啟用設施，未通車區間名稱全部是暫稱。
  - (數字)為其他路線的編號，<數字>為預定編號。
  - [智能IC](https://zh.wikipedia.org/wiki/智能IC "wikilink")（SIC）背景色為<span style="color:#eda5ff">■</span>。
  - 路線名無特別註名者為[市町道](https://zh.wikipedia.org/wiki/市町村道 "wikilink")。
  - BS（巴士站）：○為使用中、◆為停用中，未註記者即未設置。
  - 停车区：停車場，SA：服務區，IC：交流道，JCT：系統交流道。

## 歷史

  - 1966年7月1日 :
    [東京都](../Page/東京都.md "wikilink")-[直江津市](https://zh.wikipedia.org/wiki/直江津市 "wikilink")（現上越市）預定路線決定。
  - 1971年6月8日 :
    [更埴市](https://zh.wikipedia.org/wiki/更埴市 "wikilink")（現千曲市）-須坂市基本計劃決定。
  - 1972年6月30日 : 藤岡市-長野市基本計劃決定。
  - 1973年10月19日 : 更埴市（現千曲市）-須坂市整備計劃、施行命令。
  - 1973年11月1日 : 長野市-上越市基本計劃決定。
  - 1979年3月2日 : 藤岡市-佐久市整備計劃、施行命令。
  - 1980年7月17日 : 藤岡JCT-藤岡IC間與關越自動車道通車。
  - 1982年1月20日 : 須坂市-中野市整備計劃決定。
  - 1985年5月16日 : 須坂市-中野市施行命令。
  - 1986年1月21日 :
    佐久市-千曲市、中野市-[中鄉村](https://zh.wikipedia.org/wiki/中鄉村 "wikilink")（現上越市）整備計劃決定。
  - 1987年12月8日 : 佐久市-千曲市施行命令。
  - 1988年1月31日 : 中鄉村（現越市）-上越市整備計劃決定。
  - 1988年9月22日 : 中野市-中鄉村（現上越市）施行命令。
  - 1992年12月10日 : 佐久市-千曲市4車道拓寬施行命令。
  - 1993年3月25日 : 更埴JCT-須坂長野東IC間通車，與長野自動車道連接。長野IC-須坂長野東IC間暫定為2車道。
  - 1993年3月27日 :
    藤岡IC-佐久IC間開車。藤岡JCT-藤岡間IC由關越自動車道成為上信越自動車道。碓冰輕井澤IC-佐久平停车区間暫定為2車道。
  - 1995年11月7日 : 佐久IC-小諸IC間通車。
  - 1995年11月30日 : 須坂長野東IC-信州中野IC間通車、長野IC-須坂長野東IC間拓寬為4車道。
  - 1996年11月14日 : 小諸IC-更埴JCT間通車。
  - 1997年10月16日 : 信州中野IC-中鄉IC間通車。
  - 1998年12月25日 : 中野市-信濃町4車道拓寬整備計劃、施行命令。
  - 1999年10月30日 : 中鄉IC-上越JCT間通車，**全線通車**，上越JCT與北陸自動車道連接。
  - 2000年4月28日 : 藤岡停车区（藤岡Highway Oasis、關越自動車道方向）開設。
  - 2002年7月26日 : 佐久IC-小諸IC間拓寬為4車道。
  - 2002年12月18日 : 小諸IC-上田菅平IC間拓寬為4車道。
  - 2003年12月20日 : 碓冰輕井澤IC-佐久IC間拓寬為4車道。
  - 2004年7月17日 : 上田菅平IC-更埴JCT間拓寬為4車道。
  - 2004年12月18日 : 佐久平停车区（佐久平Highway
    Oasis）智能IC[社會實驗開始](https://zh.wikipedia.org/wiki/社會實驗 "wikilink")（至2007年3月31日（當初實驗期間至2005年3月21日））。
  - 2005年1月11日 : 新井停车区 智能IC社會實驗開始（至2006年3月31日（當初實驗期間至2005年3月27日））。
  - 2005年4月24日 : 小布施停车区（小布施Highway
    Oasis）智能IC社會實驗開始（至2006年3月31日（當初實驗期間至2005年8月31日））。
  - 2005年4月29日 : 信州中野IC-豐田飯山IC間拓寬為4車道。
  - 2006年10月1日 : 小布施SIC、新井SIC開始使用。
  - 2007年4月1日 : 佐久平SIC開始使用。
  - 2009年4月27日 : 中野市-上越市4車道拓寬整備計劃變更。
  - 2009年10月9日 :
    [前原誠司](../Page/前原誠司.md "wikilink")[國土交通大臣](../Page/國土交通大臣.md "wikilink")表明凍結信濃町IC-上越JCT間的4車道拓寬事業。

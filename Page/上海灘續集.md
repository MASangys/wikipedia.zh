《**上海灘續集**》（）是[香港](../Page/香港.md "wikilink")[電視廣播有限公司](../Page/電視廣播有限公司.md "wikilink")拍攝製作的民初[劇集](https://zh.wikipedia.org/wiki/劇集 "wikilink")，全劇共20集，監製[招振強](../Page/招振強.md "wikilink")，此劇為《上海灘系列》第二輯。

此劇于2013年4月27日在[TVB星河頻道](../Page/TVB星河頻道.md "wikilink")，2014年在[TVB經典台](../Page/TVB經典台.md "wikilink")重播，2018年5月21日在[翡翠台](../Page/翡翠台.md "wikilink")重播。

## 故事大綱

自剷除所有幫會勢力後，許文強 ( [周潤發飾](https://zh.wikipedia.org/wiki/周潤發 "wikilink") ) 、丁力 ( [呂良偉](../Page/呂良偉.md "wikilink")飾 ) 雙雙成為上海大亨，控制整個上海。及後，許文強慘遭暗殺，丁力誓要替文強報仇，並派其助手祥貴 ( [廖啟智](../Page/廖啟智.md "wikilink")飾 ) 追查兇手，頓時牽起連串黑幫廝殺。

丁力的婚姻由馮程程 ( [趙雅芝](../Page/趙雅芝.md "wikilink")飾 ) 進入修道院，當上修女而結束。在偶然機會下，丁力結識了明艷照人的朱燕燕 ( [黃淑儀](../Page/黃淑儀.md "wikilink")飾 )，為之傾倒，正當兩人感情有進一步發展時，一位風度翩翩的大商家狄雲志 ( [謝賢](../Page/謝賢.md "wikilink")飾 ) 卻在此時出現，燕燕被深深吸引，一場刻骨銘心的三角戀愛，就告展開。

另一方面，雲志背後，原來有日本人勢力所支持，財雄勢大，幾乎可以隻手遮天。他主動和丁力合作，令丁力在名與利俱有極大收穫。誰料雲志最主要的目的，是為日本人控制上海鋪路。丁力與雲志合作不久，已摸清其底細，同時亦查出文強被殺，也是由他主使。一方面力欲替文強報仇，但又不想放棄彼此良好合作關係。當丁力得悉燕燕深愛著雲志時，在愛情、道義及事業上衡量後，為公為私，決定全力對付雲志，令上海再添腥風血雨。

在[微积分中](https://zh.wikipedia.org/wiki/微积分 "wikilink")，一个[函数](../Page/函数.md "wikilink")\(f = f\)的**不定积分**，也称为**原函数**或**反导数**，是一个[导数等于](../Page/导数.md "wikilink")\(f= f\)的函数\(f = F\)，即\(f= F'\)。不定积分和[定积分间的关系由](https://zh.wikipedia.org/wiki/定积分 "wikilink")[微积分基本定理确定](../Page/微积分基本定理.md "wikilink")。

\[\int f(x)dx = F(x)+C\]

其中\(F\)是\(f\)的不定积分。这样，许多函数的[定积分的计算就可以简便地通过求不定积分来进行](https://zh.wikipedia.org/wiki/定积分 "wikilink")。

## 例子

函数\(f= K(x)=\frac{2^x}{\ln2}\)是函数\(f= k(x)=2^x\!\)的一个原函数，但实际上\(f= k\!\)的原函数有无穷多个。与\(f= K\!\)相差一个常数的函数都是\(f= k\!\)的原函数，因为常数函数的导数为零，例如：\(f= \frac{2^x}{\ln2}+\ln2,\ \frac{2^x}{\ln2}-e^\sqrt{\pi}\)。函数族\(f= \left\{\frac{2^x}{\ln2}+C\;| \;C\in\mathbb{R} \right\}\)
称为函数\(f= k\!\)的**原函数族**，也就是\(f= k(x)=2^x\!\)的所有可能的原函数的[集合](https://zh.wikipedia.org/wiki/集合 "wikilink")，其中\(f= C\!\)叫做**积分常数**。从图像上来看，这是\(f= K(x)=\frac{2^x}{\ln2}\!\)垂直平移后得到的一组函数，几何上的解释就是它们在\(f= x\!\)轴同一点上的斜率都是一样的。

## 性质

**原函数存在定理**：若函数f(x)在区间上连续，则f(x)在区间上存在原函数。

### 微积分基本定理

不定积分的一个重要应用是计算定积分，[微积分基本定理建立了两者间的关系](../Page/微积分基本定理.md "wikilink")。

**微积分基本定理**：如果\(f= f\)是闭区间\(f= [a,b]\)上的[连续函数](https://zh.wikipedia.org/wiki/连续 "wikilink")，\(f=F\)是\(f=f\)在\(f=[a,b]\)上的一个原函数，那么

\[\int_{a}^{b} f(x)\,\mathrm{d}x = F(b) - F(a)\]

证明：取区间\(f= [a,b]\)的一个分割：\(f= a = x_0 < x_1 < x_2 < \cdots < x_n = b\)，又设\(f= \Delta x_{i} = x_{i+1} - x_{i}\)，根据[拉格朗日中值定理有](../Page/拉格朗日中值定理.md "wikilink")

\[F(x_{i+1}) - F(x_{i}) = F^\prime(\xi_{i}) \cdot \Delta x_{i}\]

所以
\[\begin{align}
F(b)-F(a) & = \sum_{i=0}^{n-1} (F(x_{i+1}) - F(x_{i})) \\
 & = \sum_{i=0}^{n-1} F^\prime(\xi_{i}) \cdot \Delta x_{i} \\
 & = \sum_{i=0}^{n-1} f(\xi_{i}) \cdot \Delta x_{i} \\
\end{align}\]

\(f= f\)在闭区间\(f= [a,b]\)上连续，故[黎曼可积](../Page/黎曼积分.md "wikilink")，于是
\[\lim_{n \to \infty} \sum_{i=0}^{n-1} f(\xi_{i}) \cdot \Delta x_{i} = \int_a^b f(x)\,\mathrm{d}x\]

于是：

\[\int_a^b f(x)\,\mathrm{d}x = F(b) - F(a).\]

因此，\(f= f\)的**原函数族**中的每个函数都可以叫做\(f= f\)的不定积分，简写作\(f= \int f(x)\, \mathrm{d}x.\)，因为在计算定积分时，积分常数在相减时消掉了。如果\(f= F\)定义在几个不同的[区间上](https://zh.wikipedia.org/wiki/区间 "wikilink")，那么每个区间上的积分常数可以互不相同。例如

\[F(x)=\begin{cases}-\frac{1}{x}+C_1\qquad x<0\\-\frac{1}{x}+C_2\qquad x>0\end{cases}\]

就是\(f= f(x)=\frac{1}{x^2}\)的不定积分的一般形式。它的定义区间是\(f= (-\infty,0)\cup(0,\infty)\)。

### 积分上限函数

什么样的函数具有原函数是微积分理论中的基本问题。首先，每个[连续函数都有原函数](https://zh.wikipedia.org/wiki/连续 "wikilink")，并且由上面可知，原函数的个数是无限个。其次，对于一个有原函数的函数，它的原函数族中在某点取某个特殊值的只有一个。特别来说，对某个点\(a\)，\(f\)恰有一个在\(a\)上取值为零的原函数，它可以表示为如下的**积分上限函数**：

\[F(x)=\int^x_af(t)\,\mathrm{d}t\]

下面给出积分上限函数是原函数的证明：

  -
    设函数\(\Phi(x)=\int^x_af(t)\,\mathrm{d}t\)，下证：

\(\Phi^\prime(x)=\cfrac{\mathrm{d}\int^x_af(t)\mathrm{d}t}{\mathrm{d}x}=f(x)\)。

证明：

\[\Phi(x+\Delta x)-\Phi(x)=\int_a^{x+ \Delta x}f(t)\,\mathrm{d}t-\int_a^{x} f(t)\,\mathrm{d}t\]

\[=\int_x^{x+\Delta x}f(t)\,\mathrm{d}t\]

\[=f(\xi)\cdot\Delta x\]，其中\(x<\xi<x+\Delta x\\)，当\(\Delta x\to0\)时，\(\xi\)趋向于\(x\)。

  -
    所以有\(\Phi^\prime(x)=\lim_{\Delta x\to0}\frac{\Phi(x+\Delta x)-\Phi(x)}{\Delta x}=\lim_{\Delta x\to0}f(\xi)=f(x)\)。

由此可以推出前面的结论：\(f\)的原函数中在点\(a\)上取值为\(A\)的只有一个，就是\(x \mapsto \int_a^x f(t)\,\mathrm{d}t + A\)。

这也可以看作是[微积分基本定理另一个表达形式](../Page/微积分基本定理.md "wikilink")。

不连续的函数也可以有原函数，例如考虑函数\(f\\)：

  -
    当\(x\ne0\)时\(f(x) = 2x\sin\frac{1}{x}-\cos\frac{1}{x}\)，\(\displaystyle f(0)=0\)

这个函数在0上不连续，但可以验证函数：\(F(x)=x^2\sin\frac{1}{x}\)（\(x\ne0\)时），\(\displaystyle F(0)=0\)是\(f\)的原函数。

许多看似很“简单”的函数的原函数是无法用[初等函数](../Page/初等函数.md "wikilink")（[指数函数](../Page/指数函数.md "wikilink")、[对数函数](https://zh.wikipedia.org/wiki/对数函数 "wikilink")、[代数函数](https://zh.wikipedia.org/wiki/代数函数 "wikilink")、[三角函数](../Page/三角函数.md "wikilink")、[反三角函数以及它们的有限次加減乘除開根號组合](../Page/反三角函数.md "wikilink")）来表达的（但它们一样存在！），比如说如下几个不定积分：

\[\int e^{-x^2}\,\mathrm{d}x,\qquad \int \frac{\sin (x)}{x}\,\mathrm{d}x,\qquad \int\frac{1}{\ln x}\,\mathrm{d}x\]。
它们的积分分别为：

\[\int e^{-x^2}\,\mathrm{d}x = \frac{\sqrt{\pi}}{2} \operatorname{erf}(x) + C\]

\[\int \frac{\sin (x)}{x}\,\mathrm{d}x = \operatorname{Si}(x) + C\]

\[\int\frac{1}{\ln x}\,\mathrm{d}x = \operatorname{Li}(x) + C\]
其中erf函数为[误差函数](../Page/误差函数.md "wikilink")，

Si函数为[三角积分](../Page/三角积分.md "wikilink")，

Li函数为[对数积分](../Page/对数积分.md "wikilink")。

关于什么时候原函数可以用初等函数表达, 参见[刘维尔定理](../Page/刘维尔定理_\(微分代数\).md "wikilink").

## 积分技巧

求初等函数的不定积分比求它们的导数要困难得多。如上面所看到的，有些初等函数的原函数无法用初等函数来表达。以下是求不定积分的一些技巧。

  - **积分的线性性质**使得我们可以把较为复杂的函数分成几个较为简单的函数的和来计算
  - [换元积分法可以把被积函数转换成比较容易积分的形式](../Page/换元积分法.md "wikilink")，但对换元函数有一定要求。
  - [分部积分法](https://zh.wikipedia.org/wiki/分部积分法 "wikilink")，用于函数乘积的积分。
  - 对于实值[分式函数的积分](https://zh.wikipedia.org/wiki/分式 "wikilink")，可以先将函数展开成若干一次分式函数以及二次分式函数的幂的和，再进行积分。
  - [Risch算法](https://zh.wikipedia.org/wiki/Risch算法 "wikilink")。
  - 对于常见的不定积分，可以查看**[积分表](../Page/积分表.md "wikilink")**
  - 当函数的不定积分不能用初等函数表达时，可以采用其他办法计算函数的定积分，比如[数值积分](https://zh.wikipedia.org/wiki/数值积分 "wikilink")。

## 不连续函数的积分

微积分基本定理要求\(f\)为连续函数，但是，对于不连续的函数，我们仍然可以考虑求不定积分。对于什么函数有原函数，现在仍存在着未解决的问题。如今已知的结论有：

  - 一些很不“规则”的函数，尽管在“非常多”的点上并不连续，但仍有原函数。
  - 在某些情况下，一些不“规则”的函数的不定积分可以通过**[黎曼积分](../Page/黎曼积分.md "wikilink")**求得。当然更多的不“规则”的函数不是*黎曼可积*的。

## 不定积分公式表

1.  \(\int a\,\mathrm{d}x = ax + C\)，其\(C\,\)为常数；
2.  \(\int x^{a}\,\mathrm{d}x = \frac{1}{a+1}x^{a+1} + C\)，其\(a\,\)是常数\(a \ne -1\);
3.  \(\int \frac{1}{x}\,\mathrm{d}x = \ln \left|x\right| + C\);
4.  \(\int a^{x}\,\mathrm{d}x = \frac{a^{x}}{\ln a} + C\)，其\(a > 0 \,\)，\(a \ne 1\);
5.  \(\int \sin x\,\mathrm{d}x = -\cos x + C\);
6.  \(\int \cos x\,\mathrm{d}x = \sin x + C\);
7.  \(\int \tan x\,\mathrm{d}x = -\ln \left|\cos x\right| + C\)
8.  \(\int \cot x\,\mathrm{d}x = \ln \left|\sin x\right| + C\)
9.  \(\int \sec x\,\mathrm{d}x = {\rm Re} {\rm{Arth}} \tan \frac{x}{2} + C = \ln \left|\sec x + \tan x\right| + C = \frac{1}{2} \ln \left|\frac{1+\sin x}{1-\sin x} \right| + C\)
10. \(\int \csc x\,\mathrm{d}x = {\rm Re} {\rm{Ln}}\tan \frac{x}{2} + C = \ln \left|\csc x - \cot x\right| + C = \frac{1}{2} \ln \left|\frac{1-\cos x}{1+\cos x} \right| + C\)
11. \(\int \sec^{2}x\,\mathrm{d}x = \tan x + C\);
12. \(\int \csc^{2}x\,\mathrm{d}x = -\cot x + C\);
13. \(\int \frac{1}{\sqrt{1-x^2}}\,\mathrm{d}x = \arcsin x + C\);
14. \(\int \frac{1}{\sqrt{a^2-x^2}}\,\mathrm{d}x = \arcsin \frac{x}{a} + C\);
15. \(\int \frac{1}{1+x^2}\,\mathrm{d}x = \arctan x + C\);
16. \(\int \frac{1}{a^2+x^2}\,\mathrm{d}x = \frac{1}{a}\arctan\frac{x}{a} + C\);
17. \(\int \operatorname{sinh}\,x\,\mathrm{d}x = \operatorname{cosh}\,x\,+ C\);
18. \(\int \operatorname{cosh}\,x\,\mathrm{d}x = \operatorname{sinh}\,x\,+ C\);
19. \(\int \frac 1{\sqrt{x^2+a^2}}\mathrm{d}x = \operatorname{ln}(x+\sqrt{x^2+a^2}) + C\);
20. \(\int \frac 1{\sqrt{x^2-a^2}}\mathrm{d}x = \operatorname{ln}|x+\sqrt{x^2-a^2}| + C\);

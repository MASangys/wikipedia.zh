**不尋常數**（）是指一[整數](https://zh.wikipedia.org/wiki/整數 "wikilink")*n*的最大[質因數](../Page/質因數.md "wikilink")大於[\(\sqrt{n}\)](../Page/平方根.md "wikilink")，所有質數均為不尋常數。

*k*-[光滑數](../Page/光滑數.md "wikilink")是指其最大質因數小於或等於*k*，因此若整數*n*不是\(\sqrt{n}\)光滑數，此整數就是不尋常數。

若用*u*(*n*)表示小於等於*n*的整數中的不尋常數個數，*u*(*n*)和*n*有以下的關係：

|         |              |                    |
| ------- | ------------ | ------------------ |
| ***n*** | ***u*(*n*)** | ***u*(*n*) / *n*** |
| 10      | 6            | 0.6                |
| 100     | 67           | 0.67               |
| 1000    | 715          | 0.715              |
| 10000   | 7319         | 0.7319             |
| 100000  | 70128        | 0.70128            |

數學家Richard Schroeppel在1972年證明了若任意選擇整數，選到不尋常數的漸進[機率為](https://zh.wikipedia.org/wiki/機率 "wikilink")[ln(2)](../Page/2的自然对数.md "wikilink")，也就是說：

\[\lim_{n \rightarrow \infty} \frac{u(n)}{n} = \ln(2) = 0.693147 \dots\, .\]

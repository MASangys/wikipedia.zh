**不连续点**又称**间断点**，通常是在單變數實值函數的環境下討論。令\(E\subseteq \mathbb{R},~f:E\to\mathbb{R}\)，且若\(c\in\mathbb{R}\)(不一定要在\(E\)中)，若\(f\)在\(c\)不連續，則稱\(f\)在那裡有個不連續點、\(c\)為一個\(f\)的不連續點。

## 分类

根据不同不连续点的性质，通常把不连续点分为两类：

1.  第一类不连续点：
    1.  可去不连续点：不连续点两侧函数的极限存在且相等 。
    2.  跳跃不连续点：不连续点两侧函数的[极限存在](https://zh.wikipedia.org/wiki/极限 "wikilink")，但不[相等](../Page/相等.md "wikilink")；
2.  第二类不连续点：

<!-- end list -->

  -
    不属于第一类不连续点的任何一种不连续点都属于第二类不连续点。

## 例子

[Discontinuity_removable.eps.png](https://zh.wikipedia.org/wiki/File:Discontinuity_removable.eps.png "fig:Discontinuity_removable.eps.png") 1. 考虑以下函数：

\[f(x)=\begin{cases}x^2 & \mbox{ for } x< 1 \\ 0 & \mbox { for } x=1 \\ 2-x&  \mbox{ for }  x>1\end{cases}\] 点\(x_0=1\)是可去不连续点。

[Discontinuity_jump.eps.png](https://zh.wikipedia.org/wiki/File:Discontinuity_jump.eps.png "fig:Discontinuity_jump.eps.png") 2. 考虑以下函数：

\[f(x)=\begin{cases}x^2 & \mbox{ for } x< 1 \\ 0 & \mbox { for } x=1 \\ 2-(x-1)^2& \mbox{ for } x>1\end{cases}\] 点\(x_0=1\)是跳跃不连续点。

[Discontinuity_essential.eps.png](https://zh.wikipedia.org/wiki/File:Discontinuity_essential.eps.png "fig:Discontinuity_essential.eps.png") 3. 考虑以下函数：

\[f(x)=\begin{cases}\sin\frac{5}{x-1} & \mbox{ for } x< 1 \\ 0 & \mbox { for } x=1 \\ \frac{0.1}{x-1}& \mbox{ for } x>1\end{cases}\] 点\(x_0=1\)是第二类不连续点，又称本性不连续点。

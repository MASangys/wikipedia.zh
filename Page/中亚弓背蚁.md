**中亚弓背蚁**（學名：**）是[蟻科](https://zh.wikipedia.org/wiki/蟻科 "wikilink")[弓背蟻屬](../Page/弓背蟻屬.md "wikilink")的一個物種。

## 物種描述

该种的[蚁后及](https://zh.wikipedia.org/wiki/蚁后 "wikilink")[工蚁体色有较大变化](https://zh.wikipedia.org/wiki/工蚁 "wikilink")\[1\]。

| 蚁种 | 体长     | 颜色          |
| -- | ------ | ----------- |
| 蚁后 | 12mm   | 腿黄，头黑，胸黑，腹黄 |
| 工蚁 | 6-12mm | 通身黄         |
| 兵蚁 | 无      |             |
| 雄蚁 | 8mm    |             |

中亚弓背蚁各蚁种的特征\[2\]

## 生物學

该种为典型荒漠种，在[蒿属](../Page/蒿属.md "wikilink")[禾本科](../Page/禾本科.md "wikilink")草原的石头下面筑巢\[3\]\[4\]。

## 分布

该物种分布于[中国](https://zh.wikipedia.org/wiki/中国 "wikilink")（[新疆](https://zh.wikipedia.org/wiki/新疆 "wikilink")、[四川](https://zh.wikipedia.org/wiki/四川 "wikilink")）、[蒙古](https://zh.wikipedia.org/wiki/蒙古 "wikilink")\[5\]、[哈萨克斯坦](../Page/哈萨克斯坦.md "wikilink")、[伊朗](https://zh.wikipedia.org/wiki/伊朗 "wikilink")、[阿富汗](../Page/阿富汗.md "wikilink")、[外高加索地区](https://zh.wikipedia.org/wiki/外高加索 "wikilink")、[中亚](../Page/中亚.md "wikilink")\[6\]。

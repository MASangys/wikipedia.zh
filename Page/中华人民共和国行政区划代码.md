**中华人民-{}-共和国行政区划代码**是[中华人民共和国用于](https://zh.wikipedia.org/wiki/中华人民共和国 "wikilink")[行政区划](../Page/行政区划.md "wikilink")上的识別码，同时是一项由国家标准机构——[中国国家标准化管理委员会](../Page/中国国家标准化管理委员会.md "wikilink")通过并公开发布的[国家标准](https://zh.wikipedia.org/wiki/国家标准 "wikilink")，国标号为GB/T 2260，并加上后缀发布年份，如GB/T 2260-2007\[1\]。

此国家标准于1980年首次发布，并在1982年、1984年、1986年、1988年、1991年、1995年、1999年、2002年及2007年发布更新版本，现时最新版本是GB/T 2260-2007第1号修改单\[2\]。

## 基本应用

此标准规定了县级或以上行政区划的数字代码（由6位数字组成）和字母代码（省级行政区由2位大写字母组成，地级或县级行政区由3位大写字母组成），适用于对行政区划进行标识、信息处理和交换等\[3\]。然而，在一般应用上，数字代码更为普及，例如有以下范畴：

  - [公安部用于编配](../Page/中华人民共和国公安部.md "wikilink")[公民身份号码](../Page/中华人民共和国公民身份号码.md "wikilink")（头6位数字的地址码）；
  - [民政部用于行政区划的标识](../Page/中华人民共和国民政部.md "wikilink")；
  - [国家统计局用于开展统计调查上的行政区划划分](https://zh.wikipedia.org/wiki/中华人民共和国国家统计局 "wikilink")。

数字代码采用三层六位层次码结构，代码从左至右的含义是：

  - 第一层（第一、二位）表示[省级行政区](https://zh.wikipedia.org/wiki/省级行政区 "wikilink")（[省](https://zh.wikipedia.org/wiki/省 "wikilink")、[自治区](../Page/自治区.md "wikilink")、[直辖市](https://zh.wikipedia.org/wiki/直辖市 "wikilink")、[特别行政区](../Page/特别行政区.md "wikilink")），其中第一位代表[大区](../Page/大行政区.md "wikilink")。
  - 第二层（第三、四位）表示[地级行政区](../Page/地级行政区.md "wikilink")（[地级市](../Page/地级市.md "wikilink")、[地区](https://zh.wikipedia.org/wiki/地区_\(行政区划\) "wikilink")、[自治州](../Page/自治州.md "wikilink")、[盟](https://zh.wikipedia.org/wiki/盟 "wikilink")），直辖市的市辖区、县的汇总码，及[省（自治区）直辖县级行政区的汇总码](../Page/省直辖县级行政区.md "wikilink")。
  - 第三层（第五、六位）表示[县级行政区](../Page/县级行政区.md "wikilink")（[县](../Page/县.md "wikilink")、[自治县](../Page/自治县.md "wikilink")、[市辖区](../Page/市辖区.md "wikilink")、[县级市](../Page/县级市.md "wikilink")、[旗](../Page/旗_\(行政区划\).md "wikilink")、[自治旗](../Page/自治旗.md "wikilink")、[林区](https://zh.wikipedia.org/wiki/林区 "wikilink")、[特区](../Page/特区_\(中国县级行政区\).md "wikilink")）。

## 代码的排序、更改与撤销

一般来说，区划代码是按照政区设立的日期，依数字的顺序排列的。但早期部分地级市因应市辖区的性质，分为城区与郊区两个类型，而区划代码排列的开始位置亦有所不同。城区代码由“02”开始排列（“01”是地级市市辖区的汇总码），郊区则由“11”开始排列。例如[武汉市](../Page/武汉市.md "wikilink")[江汉区](../Page/江汉区.md "wikilink")等6个城区由“420102”排起，[洪山区](../Page/洪山区.md "wikilink")等7个郊区则由“420111”排起。这种排序方式近年已不再使用，但现存此情况的地级市在新设市辖区时，尚未有准则应在城区还是郊区的代码后排下去，造成代码疑问。

单纯的政区更名并不会更改代码。管辖范围有变化，或政区合并且同时沿用原有政区之一名称，一般会沿用原有政区的代码，例如[上海市](https://zh.wikipedia.org/wiki/上海市 "wikilink")[黄浦区](../Page/黄浦区.md "wikilink")与[卢湾区](../Page/卢湾区.md "wikilink")合并成新的黄浦区，就沿用了原黄浦区的代码，但不同类型的政区合并沿用原有政区之一名称，代码可能沿用，也可能不沿用。大部分政区类型改变，例如撤县设区、撤县设市等，代码则需要更改。上级行政区发生变化，例如所属政区撤地区设市，代码也需要更改。上级行政区发生变化，下级行政区间的代码虽改变，但除非下级行政区类型改变，排序一般不变。为保证代码的唯一性，因行政区划变更而撤销的代码一般不会再赋予其他政区使用。

## 代码的发布与维护

虽然理论上行政区划代码应经由国家标准化管理委员会通过后才公开发布并生效，但实际上行政区划经常有变更，而标准化管理委员会并不是恒常地发布最新的代码，而且审核县级或以上行政区划的设立、命名和变更等是由民政部区划地名司负责。因此，当有县级或以上的区划变更时，民政部会按照上述规则赋予变更后行政区新的区划代码。民政部公布的代码几乎与标准化管理委员会公布的一模一样，但亦有例外。例如2017年第2号修改单中[海南省](../Page/海南省.md "wikilink")[儋州市](../Page/儋州市.md "wikilink")与[三沙市的代码分別是](https://zh.wikipedia.org/wiki/三沙市 "wikilink")“460300”、“460400”，正好与民政部公布的互换了排序\[4\]。

## 县级以下行政区划代码编码规则

在《中华人民共和国行政区划代码》（GB/T 2260）的基础上，国家标准化管理委员会亦发布《县级以下行政区划代码编制规则》作为前者扩充与延拓的配套标准，与GB/T 2260配合使用，国标号为GB/T 10114，于1988年12月首次发布，2003年作第一次修订 \[5\]。

此标准用作编制县级以下的行政区划（即[乡级行政区](https://zh.wikipedia.org/wiki/乡级行政区 "wikilink")）代码，包括[街道办事处](../Page/街道办事处.md "wikilink")、[镇](../Page/行政建制镇.md "wikilink")、[乡](https://zh.wikipedia.org/wiki/乡 "wikilink")、[民族乡](../Page/民族乡.md "wikilink")、[苏木](../Page/苏木_\(行政区划\).md "wikilink")、[民族苏木](https://zh.wikipedia.org/wiki/民族苏木 "wikilink")。县级以下的行政区划代码由九位数字构成，代码从左至右的含义是：

<center>

|                             |                             |                             |                             |                             |                             |                             |                             |                             |
| --------------------------- | --------------------------- | --------------------------- | --------------------------- | --------------------------- | --------------------------- | --------------------------- | --------------------------- | --------------------------- |
| <font size="5">**1**</font> | <font size="5">**1**</font> | <font size="5">**0**</font> | <font size="5">**1**</font> | <font size="5">**0**</font> | <font size="5">**1**</font> | <font size="5">**0**</font> | <font size="5">**0**</font> | <font size="5">**1**</font> |
| 省级                          | 地级                          | 县级                          | 乡级                          |                             |                             |                             |                             |                             |

</center>

  - 第一段（第一至六位）即所属行政区采用GB/T 2260编制的六位数字代码。

<!-- end list -->

  - 第二段（第七至九位）表示乡级行政区：
      - 001−099表示街道办事处；
      - 100−199表示镇；
      - 200−399表示乡、民族乡、苏木、民族苏木。

编制县级以下的行政区划代码时，当只表示县级或以上行政区划时，第七至九位可补上“000”，以保证代码长度一致。与GB/T 2260相似，政区更名不需更改代码，政区类型或隶属关系变更，则需重新赋码。为保证代码的唯一性，因行政区划变更而撤销的代码不得再赋予其他政区使用。

值得注意的是，国家标准化管理委员会公布的GB/T 10114只列出了编制规则，并没有如GB/T 2260般列出行政区的具体代码。

## 村级区划代码

在国家标准GB/T 2260与GB/T 10114中未有涵盖[村级行政区的区划代码编制](https://zh.wikipedia.org/wiki/村级行政区 "wikilink")，因此在理论上暂时未有村级区划代码的国家标准。然而在实际应用上，部分政府部门亦有编制村级的区划代码。例如[国家统计局](../Page/国家统计局.md "wikilink")根据国家标准GB/T 2260和GB/T 10114编制的《统计用区划代码和城乡划分代码》，其中《统计用区划代码》由十二位数字组成\[6\]：

<center>

|                             |                             |                             |                             |                             |                             |                             |                             |                             |                             |                             |                             |
| --------------------------- | --------------------------- | --------------------------- | --------------------------- | --------------------------- | --------------------------- | --------------------------- | --------------------------- | --------------------------- | --------------------------- | --------------------------- | --------------------------- |
| <font size="5">**1**</font> | <font size="5">**1**</font> | <font size="5">**0**</font> | <font size="5">**1**</font> | <font size="5">**0**</font> | <font size="5">**1**</font> | <font size="5">**0**</font> | <font size="5">**0**</font> | <font size="5">**1**</font> | <font size="5">**0**</font> | <font size="5">**0**</font> | <font size="5">**1**</font> |
| 省级                          | 地级                          | 县级                          | 乡级                          | 村级                          |                             |                             |                             |                             |                             |                             |                             |

</center>

其中头九位数字是根据国家标准编制的行政区划代码，第十至第十二位数字为村级代码。

  - 001−199表示[居民委员会](../Page/居民委员会.md "wikilink")（社区）；
  - 200−399表示[村民委员会](../Page/村民委员会.md "wikilink")（行政村）、[牧民委员会](https://zh.wikipedia.org/wiki/牧民委员会 "wikilink")（[嘎查](https://zh.wikipedia.org/wiki/嘎查 "wikilink")）。

## 非正式行政区划的代码编制

理论上，虽然根据上述方法编制的行政区划代码可以涵盖全国所有行政区划，但实际上由于种种原因，各地存在各种“非正式”的行政区划。所谓“非正式”，即是民政部未予承认的行政区划，例如各地方政府普遍设立的[经济开发区](https://zh.wikipedia.org/wiki/经济开发区 "wikilink")、[高新区](https://zh.wikipedia.org/wiki/高新区 "wikilink")、[城市新区](https://zh.wikipedia.org/wiki/城市新区 "wikilink")，以及部分因历史原因形成的[政企合一单位](https://zh.wikipedia.org/wiki/政企合一 "wikilink")、[农垦单位](https://zh.wikipedia.org/wiki/农垦 "wikilink")、[林业](../Page/林业.md "wikilink")单位、[兵团单位等](https://zh.wikipedia.org/wiki/兵团 "wikilink")。这些非正式行政区划不会被列入国家标准的行政区划代码中，然而个別部门，例如国家统计局，仍会因应实际需要而赋予这些单位独立的代码。例如2017年版的《统计用区划代码》中列明“为满足统计调查工作组织和数据汇总的需要，国家统计局对一些符合条件的开发区编制了统计汇总识别码”\[7\]。

### 县级或副县级

非正式县级或副县级开发区在《统计用区划代码》的县级码段为71−80，唯应同时满足以下条件：

1.  国家和省人民政府正式批准成立的开发区；
2.  开发区的管理等同于县级人民政府，行使县级人民政府的管理职能，即管理开发区内的社会公共事务；
3.  开发区至少管理一个乡级单位；
4.  开发区管委会成立并运作两年及以上。

较典型的例子有：

  - 江苏省[苏州市](../Page/苏州市.md "wikilink")[苏州工业园区](../Page/苏州工业园区.md "wikilink")，统计局赋码320571；
  - 湖南省[怀化市](../Page/怀化市.md "wikilink")[洪江管理区](../Page/洪江管理区.md "wikilink")，统计局赋码431271；
  - 甘肃省[兰州市](../Page/兰州市.md "wikilink")[兰州新区](../Page/兰州新区.md "wikilink")，统计局赋码620171。

还有一些较特殊的非正式县级或副县级行政区，例如：

  - 黑龙江省[大兴安岭地区](../Page/大兴安岭地区.md "wikilink")[加格达奇区](../Page/加格达奇区.md "wikilink")、[松岭区](../Page/松岭区.md "wikilink")、[新林区](../Page/新林区.md "wikilink")、[呼中区](../Page/呼中区.md "wikilink")，统计局分別赋码232761−232764；
  - 青海省[海西蒙古族藏族自治州](../Page/海西蒙古族藏族自治州.md "wikilink")[大柴旦行政委员会](../Page/大柴旦行政委员会.md "wikilink")，统计局赋码632857。

### 乡镇级

民政部未确认的开发区、工矿区、农场等“类似乡级单位”，乡级代码为400−599。

较典型的例子有：

  - 河北省[宁晋县](../Page/宁晋县.md "wikilink")[大曹庄管理区](https://zh.wikipedia.org/wiki/大曹庄管理区 "wikilink")，统计局赋码130528401；
  - 安徽省[安庆市](../Page/安庆市.md "wikilink")[皖河农场](https://zh.wikipedia.org/wiki/皖河农场 "wikilink")，统计局赋码340803500；
  - 湖北省[沙洋监狱管理局](https://zh.wikipedia.org/wiki/沙洋监狱管理局 "wikilink")（实为副厅级单位），统计局赋码420822450；
  - [新疆生产建设兵团](../Page/新疆生产建设兵团.md "wikilink")的各个团场，统计局一般赋予乡级代码501−599。

另外，“198”是用来赋给虚拟乡镇级单位的代码，例如部分已由市辖区直管社区的城市，如安徽省[铜陵市](../Page/铜陵市.md "wikilink")[铜官区](../Page/铜官区.md "wikilink")，统计局赋予虚拟乡镇代码340705198代表直管社区的汇总。

### 村级

民政部未确认的园区、工矿区、农场等“类似村级单位”，村级代码为400−599（498、598除外）。

  - 400−499表示类似居民委员会（不含498）；
  - 500−599表示类似村民委员会（不含598）；
  - 498表示虚拟居民委员会；
  - 598表示虚拟村民委员会。

另外，“398”是用来赋给乡级单位直接管辖村民小组单位的代码。

## 部门编码的差异

在绝大部分的情况下，县级或以上的行政区划代码在不同部门的系统上都是统一的。出现差异的情况往往是县级以下的行政区划的编码，以及对非正式行政区划的编码。例如[国家统计局](../Page/国家统计局.md "wikilink")赋予黑龙江省[大兴安岭地区](../Page/大兴安岭地区.md "wikilink")[加格达奇区](../Page/加格达奇区.md "wikilink")“232701”这个代码，然而当[漠河县在](https://zh.wikipedia.org/wiki/漠河县 "wikilink")2018年撤县设市时，民政部则将“232701”这个代码赋给了[漠河市](../Page/漠河市.md "wikilink")，国家统计局为避免引起混乱，给加格达奇区另赋代码“232761”\[8\]。

民政部与统计局在乡级行政区划的代码编制有较多差异，例如以内蒙古自治区[包头市](../Page/包头市.md "wikilink")[达尔罕茂明安联合旗](../Page/达尔罕茂明安联合旗.md "wikilink")为例，可以看出两个部门的编码差异。全旗12个乡、镇、苏木中，只有2个镇是两个部门都赋给了同一代码。

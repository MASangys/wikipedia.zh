## 中国国产军用飞机命名系统

[中国使用一套类似于](https://zh.wikipedia.org/wiki/中国 "wikilink")[美国](../Page/美国.md "wikilink")的飞机型号命名方法：代表飞机基本任务的首字，后跟序列号。

首字列表：

|      |      |                                                          |                        |
| ---- | ---- | -------------------------------------------------------- | ---------------------- |
| 首字代码 | 代表汉字 | 含义                                                       | 对应英文                   |
| BA   | 靶    | [靶机](https://zh.wikipedia.org/wiki/靶机 "wikilink")        | Drone                  |
| CJ   | 初教   | [初级教练机](https://zh.wikipedia.org/wiki/初级教练机 "wikilink")  | Primary Trainer        |
| H    | 轰    | [轰炸机](../Page/轰炸机.md "wikilink")                         | Bomber                 |
| J    | 歼    | [歼击机](https://zh.wikipedia.org/wiki/歼击机 "wikilink")（戰鬥機） | Fighter                |
| JL   | 教练   | [教练机](https://zh.wikipedia.org/wiki/教练机 "wikilink")      | Trainer                |
| KJ   | 空警   | [空中预警机](../Page/空中预警机.md "wikilink")                     | Airborne Warning Plane |
| Q    | 强    | [强击机](https://zh.wikipedia.org/wiki/强击机 "wikilink")（攻擊機） | Attacker               |
| SH   | 水轰   | [水上轰炸机](https://zh.wikipedia.org/wiki/水上轰炸机 "wikilink")  | Maritime Bomber        |
| WZ   | 无侦   | [无人侦察机](https://zh.wikipedia.org/wiki/无人侦察机 "wikilink")  | UAV                    |
| X    | 滑翔   | [滑翔机](https://zh.wikipedia.org/wiki/滑翔机 "wikilink")      | Glider                 |
| Y    | 运    | [运输机](https://zh.wikipedia.org/wiki/运输机 "wikilink")      | Transporter            |
| Z    | 直    | [直升机](https://zh.wikipedia.org/wiki/直升机 "wikilink")      | VTOL/Helicopter        |

初教机与教练机共享相同的序列号。

出口版的飞机使用西方化的型号前缀，如"F", "A", "B"分别代替"J", "Q", "H"。

飞机的改型往往有第二前缀字：

|      |      |                                                         |                    |
| ---- | ---- | ------------------------------------------------------- | ------------------ |
| 首字代码 | 代表汉字 | 含义                                                      | 对应英文               |
| D    | 电    | [电子作战机](https://zh.wikipedia.org/wiki/电子作战机 "wikilink") | Electronic Warfare |
| J    | 教    | [教练机](https://zh.wikipedia.org/wiki/教练机 "wikilink")     | Trainer            |
| U    | 油    | [加油机](https://zh.wikipedia.org/wiki/加油机 "wikilink")     | Tanker             |
| Z    | 侦    | [侦察机](../Page/侦察机.md "wikilink")                        | Reconnaissance     |

如：

|    |    |                            |
| -- | -- | -------------------------- |
| HD | 轰电 | Bomber, modified for EW    |
| HJ | 轰教 | Bomber Trainer             |
| HU | 轰油 | Bomber, modified as tanker |
| HZ | 轰侦 | Reconnaissance Bomber      |
| JJ | 歼教 | Fighter Trainer            |
| JZ | 歼侦 | Reconnaissance Fighter     |
| QD | 强电 | Attacker, modified for EW  |

中国飞机的序列号都是从5开始，这是因为在五十年代中国航空工业起步时表示要从“无”到有之意。另一种说法是5代表20世纪50年代世界先进水平，6代表20世纪60年代世界先进水平，以此类推。

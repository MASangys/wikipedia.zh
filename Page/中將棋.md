**中將棋**最早出現在[室町時代](https://zh.wikipedia.org/wiki/室町時代 "wikilink")，是從[大將棋](../Page/大將棋.md "wikilink")的基礎上改進的[日本將棋變體](../Page/日本將棋變體.md "wikilink")。

## 規則

### 基本規則

  - 12×12格的將棋盤，接近自己的四行橫列為自陣，反之，接近對手的四行橫列為敵陣。
  - 棋子共21種，若包含升級棋則28種，各方有46顆棋子。
  - 不同於[本將棋](https://zh.wikipedia.org/wiki/本將棋 "wikilink")，無打入與持駒概念。
  - 進入敵陣時若不即時升級，則必須等到在敵陣吃子時方可升級，或是重新進入敵陣後方可升級。
  - 若一方擁有太子(醉象的升級)，當其玉將被吃時還不算戰敗。
  - 首先殺死敵方的玉將及太子(如果有的話)的一方獲勝。

### 棋子的走法

棋子依照其走法及吃法的特性可以歸類出四種移動能力，分別為步行、躍行、奔行和獅子式的特殊移動。下面圖表說明了四種移動能力和其圖示。

|          |
| -------- |
| 棋子的移動和圖示 |
| ○        |
| ☆        |
| │        |
| ─        |
| ＼        |
| ／        |
| **\!**   |

<ul>
<li>獅子可以在一回合內移動最多兩次，每一次可以向任意方向步行一格。在這兩次行動中，它可以自由變換步行的方向和選擇是否吃子。 因此在一回合內，獅子可以步行至一個空格後再吃掉周圍的棋子或是吃掉鄰格的一個棋子後再步行至周圍的空格，或甚至連續吃兩個棋子。當獅子吃掉鄰格(下圖的"<strong>!</strong>"格)的一個棋子後再回到原格，這種看似在原地吃子的走法稱為"居食"。當獅子步行至一個鄰近的空格又移回原格時，稱為"待步"，看似沒有移動，但可以當做跳過一回合。此外獅子還可以直接躍行至離原格兩步遠的第二格(下圖的"☆"格)，但此走法只充允許在一回合內走一次。</li>
<li>獅子不能升級</li>

<ul>
<li>獅子還有一些特殊的規則，玩家在兌換或吃對方獅子時必須注意並遵守(此規則對由麒麟升級來的獅子亦適用)</li>
</ul>
<ol>
<li>在任何情況下，獅子都可以吃掉在它鄰格(右圖的"<strong>!</strong>"格)的敵方獅子</li>
<li>在任何情況下，獅子都可以吃掉在非鄰格(右圖的"☆"格)沒有受保護的敵方獅子</li>
<li>當敵方的獅子在非鄰格(右圖的"☆"格)並且有受保護時，此情況稱做"獅足"，一般來說不能交換獅足獅。若己方獅子要吃敵方的獅足獅的話，必須先吃鄰格(右圖的"<strong>!</strong>"格)非步兵或仲人的敵方棋子(如果是步兵或仲人，則仍不能吃敵方獅子)，再吃掉它。</li>
<li>如果己方的獅子被敵方用非獅子的棋子吃掉時，己方不能立即用非獅子(假設己方因升級擁有兩個獅子)的棋子吃掉敵方的獅子，此規則稱為先手獅，然而如果己方有兩個獅子，己方允許立刻用獅子(用其他棋子依舊是禁止的)吃掉敵方的獅子</li>

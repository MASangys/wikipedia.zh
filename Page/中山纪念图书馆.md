**中山纪念图书馆**，又称**中山市图书馆**、**中山图书馆**，是一所位于[中华人民共和国](https://zh.wikipedia.org/wiki/中华人民共和国 "wikilink")[广东省](../Page/广东省.md "wikilink")[中山市](../Page/中山市.md "wikilink")的市级[公共图书馆](../Page/公共图书馆.md "wikilink")，为纪念[孙中山而于](https://zh.wikipedia.org/wiki/孙中山 "wikilink")1935年成立。隶属于中山市文化广电新闻出版局，是[文化和旅游部公布的](../Page/中华人民共和国文化和旅游部.md "wikilink")[国家一级图书馆](../Page/国家一级图书馆.md "wikilink")\[1\]。馆舍坐落在中山市[东区兴中道](../Page/东区街道_\(中山市\).md "wikilink")，于2019年启用，建筑面积57660平方米。馆舍的标志上有[中华人民共和国名誉主席](../Page/中华人民共和国名誉主席.md "wikilink")、孙中山夫人[宋庆龄](../Page/宋庆龄.md "wikilink")的题字。因为[中山市的行政区划比较特殊](https://zh.wikipedia.org/wiki/中山市#行政区划 "wikilink")（作为[地级市](../Page/地级市.md "wikilink")下不设[县级行政区](../Page/县级行政区.md "wikilink")，为罕见的[直筒子地级市](../Page/直筒子市.md "wikilink")），故中山纪念图书馆是中山市唯一一所具规模的综合性公共图书馆\[2\]。

## 历史

[缩略图](https://zh.wikipedia.org/wiki/File:Sun_Yat-sen_Memorial_Library-3.jpg "fig:缩略图") [清朝](../Page/清朝.md "wikilink")[光绪](../Page/光绪.md "wikilink")十四年（1888年），“寿香楼”在丰山书院成立，当时这是一个只供文人进出的地方。民国八年（1919年），[香山县通俗图书馆在铁城的关帝庙成立](https://zh.wikipedia.org/wiki/香山县 "wikilink")，但先后因阴暗潮湿、地方狭窄的原因多次搬迁，图书散失。当时的人们认为：作为孙中山的故乡，香山应该有一所像模像样、具规模的图书馆。\[3\]

中山纪念图书馆的历史可追溯到1935年。1925年3月12日，孙中山在北京逝世，享年59岁\[4\]。之后为了纪念孙中山，香山县改名为中山县。当时政府倡议创建图书馆纪念孙中山，并筹集捐款，同时成立了设计委员会。1935年11月1日馆舍兴建，次年12月，馆舍一、二两层在西山上落成（这是当时当地最宏伟的建筑），开放使用，时名**中山纪念图书馆**\[5\]，馆舍面积800平方米\[6\]。并入通俗图书馆图书之后，县里各界捐钱协助运营，购置设备和图书2万余册\[7\]。1937年2月1日，中山纪念图书馆正式揭幕，时任县长[杨子毅主持了揭幕仪式](https://zh.wikipedia.org/wiki/杨子毅 "wikilink")\[8\]；2月10日正式开放\[9\]。当时计划于翌年兴建第三层，但[第二次中日战争中](https://zh.wikipedia.org/wiki/第二次中日战争 "wikilink")，图书馆遭破坏，续建三楼的计划自然也被搁置。中国抗战胜利后，1947年7月11日\[10\]，该馆重新开放。在县内各界人士捐款赞助之下，馆舍第三层终于在1947年10月竣工，馆名改为**中山图书馆**\[11\]，全馆面积达1179平方米\[12\]。

[解放军进驻后](../Page/中国人民解放军.md "wikilink")，[中华人民共和国政府于](https://zh.wikipedia.org/wiki/中华人民共和国政府 "wikilink")1950年2月接管了中山图书馆，短暂更名为**石岐人民图书馆**。1954年，该馆开始在中山各地设立流通图书馆，到20世纪60年代已有近两百个\[13\]。1955年8月，名称改为**石岐中山图书馆**。20世纪60年代，又改名为**中山图书馆**\[14\]。文化大革命期间，图书馆又一次遭到破坏。文革结束后，人们的思想获得解放，图书馆使用者增加，馆舍不能满足人民的文化需要。于是，政府拨款在原馆舍后新建了一座两层的少儿图书室（面积168平方米\[15\]）\[16\]。1981年2月1日，在[中华人民共和国名誉主席](../Page/中华人民共和国名誉主席.md "wikilink")、孙中山夫人[宋庆龄](../Page/宋庆龄.md "wikilink")的提议之下，馆名复为**中山纪念图书馆**\[17\]。同时，宋庆龄还带病\[18\]为该馆题写了馆名\[19\]。之后，中山市政府决定投资建设一座现代化图书馆馆舍。在[泰国](../Page/泰国.md "wikilink")泰华报人公益基金会主席陈世贤夫妇的赞助（捐款数百万[港元](../Page/港元.md "wikilink")），以及中山市政府的出资之下，1988年7月，位于[东区兴中道](../Page/东区街道_\(中山市\).md "wikilink")（又称兴中路）的图书馆新馆开工。\[20\]1991年，新馆舍竣工投用，呈“日”字型，馆内设置园林，环境安静\[21\]，而原馆舍改为少年儿童阅览中心\[22\]（现为中山市[佛教](../Page/佛教.md "wikilink")协会的办公地点\[23\]）。1994年，被[文化部评为国家三级图书馆](../Page/中华人民共和国文化部.md "wikilink")。同年，[中山市少年儿童图书馆成立](https://zh.wikipedia.org/wiki/中山市少年儿童图书馆 "wikilink")，隶属于中山纪念图书馆，这是广东省第二家独立建制的少儿图书馆。1997年10月，馆舍三楼建起了综合艺术厅和方成展览厅。1999年，被评为国家二级图书馆。\[24\]

2001年，中山市少年儿童图书馆撤销，并入中山纪念图书馆，其馆舍另作他用。2005年，升级为[国家一级图书馆](../Page/国家一级图书馆.md "wikilink")，并保持至今。\[25\]2009年，成为“中国图书馆学会青少年阅读推广委员会”主任馆，并以此为契机打造了一批青少年阅读推广相关的活动。在新馆建设被列入中山市“十二五”规划重点建设项目后，2012年10月，原馆舍进行改造建设，图书馆临时搬迁至[东区华凯分馆](../Page/东区街道_\(中山市\).md "wikilink")。\[26\]2014年，中山纪念图书馆新馆馆舍奠基。2019年11月4日，一座以[中山陵](../Page/中山陵.md "wikilink")祭堂中孙中山雕像为原型，由法国Susse艺术工坊铸造的铜像在新馆前完成安装\[27\]。11月12日，中山纪念图书馆新馆开馆，启动试运行\[28\]。

## 馆舍

[缩略图](https://zh.wikipedia.org/wiki/File:Sun_Yat-sen_Memorial_Library-1.jpg "fig:缩略图") 中山纪念图书馆馆舍位于中山市[东区兴中道上](../Page/东区街道_\(中山市\).md "wikilink")，[中山市人民政府西面](https://zh.wikipedia.org/wiki/中山市人民政府 "wikilink")。建筑面积为57660平方米，高41.2米，地下两层、地上七层。\[29\]设计藏书容量为320万册，由、[华东建筑设计研究院有限公司联合设计](https://zh.wikipedia.org/wiki/华东建筑设计研究院 "wikilink")，设计时遵循“人文、科技、绿色、便民”的理念\[30\]。馆内主要划分为信息资源交互平台、阅读空间共享平台两大平台，设有“三中心”、“四书房”、“五大厅”、“六大区”、“七大室”、“八主题馆”\[31\]：

| 两平台  | 信息资源交互平台 | 阅读空间共享平台    |
| ---- | -------- | ----------- |
| 三中心  | 文献资源中心   | 知识媒体中心      |
| 四书房  | 市民大书房    | 中山书房（自助图书馆） |
| 五大厅  | 城市会客厅    | 综艺厅         |
| 六大区  | 自助服务区    | 户外休闲阅读区     |
| 七大室  | 视障阅览室    | 数字阅读体验室     |
| 八主题馆 | 音乐图书馆    | 生活专题文献馆     |

馆内设有2000个阅览座位。空间比较开阔，部分区域有落地玻璃墙。此外，馆内的热回收、[太阳能热水](https://zh.wikipedia.org/wiki/太阳能 "wikilink")、屋顶绿化、雨水回收、蓄水草皮等设施，则使得该馆更加绿色节能\[32\]。馆内的大空间场所采用了全空气[空调系统](https://zh.wikipedia.org/wiki/空调 "wikilink")，其他区域有多种送风方式相结合\[33\]。馆内采用了多种多样的[幕墙](../Page/幕墙.md "wikilink")\[34\]。 [缩略图](https://zh.wikipedia.org/wiki/File:Sun_Yat-sen_Memorial_Library-2.jpg "fig:缩略图") 馆内中庭的大型壁画《香山星座》绘有26位中山的名人，总面积达187平方米，是国内最大的室内[马赛克壁画](https://zh.wikipedia.org/wiki/马赛克 "wikilink")。馆内四楼孙中山雕像是仿制中山陵孙中山大理石坐像（由法国雕塑家[保羅·蘭多斯基创作](https://zh.wikipedia.org/wiki/保羅·蘭多斯基 "wikilink")）的，2016年孙中山诞辰150周年时，蘭多斯基后人决定授权法国Susse艺术工坊铸造孙中山铜像，最终由中山收藏家黄定中购回。此外馆内还有一尊孙中山雕像，有室外大型壁画《中山寻踪》、《和风拂林》系列彩色玻璃、《未来之门》系列壁画等美术作品，它们均由[中央美术学院](../Page/中央美术学院.md "wikilink")唐晖教授、韩晓冬博士及其团队所创作。\[35\]\[36\]

## 馆务

[缩略图](https://zh.wikipedia.org/wiki/File:Sun_Yat-sen_Memorial_Library-7.jpg "fig:缩略图")

### 馆务概况

中山纪念图书馆隶属中山市文化广电新闻出版局，以“读者第一、服务至上”为宗旨，以“创新务实，高效管理，优质服务，读者满意”为办馆方针\[37\]。馆内下设信息技术部、办公室、采编部、借阅部、辅导教育部这五个部门\[38\]。中山纪念图书馆纸质文献总量约300万册件\[39\]，除了普通书籍以外，孙中山遗著及演讲录音，以及中山市的地方文献，都是该馆的重点收藏对象\[40\]。

### 馆内活动

馆内举办了包括“童心故事会”、“妈妈故事会”、“香山人·行世界”、“绿色暑假”、“全民读书月”、“晒书会”、“图书漂流”、“十佳书香家庭评选”在内的各种活动。另外还定期举办“香山讲坛”，这是中山市的品牌文化活动之一。\[41\]

### 信息和科技服务

早在1995年，中山纪念图书馆的采编和流通部门就实现了计算机自动化管理，2000年还更换了管理系统，报纸和[期刊](../Page/期刊.md "wikilink")也加入了计算机管理的行列。1997年，馆内开设了[多媒体](../Page/多媒体.md "wikilink")阅览室。1998年，图书馆建立了官方[网站](https://zh.wikipedia.org/wiki/网站 "wikilink")。\[42\]2016年以来，中山纪念图书馆还打造了一个“全媒体数字化服务平台”，提供全媒体在线服务，旗下包括数字化发布、整合服务、宣传推广、业务支撑四大平台\[43\]。

2019年启用的新馆在一楼大厅中设有“[瀑布](../Page/瀑布.md "wikilink")流”，电子屏幕上有图书封面图片如瀑布一样落下，点击封面即可显示该书简介和含有有声书籍的二维码。而在馆内二楼的“悦听空间”中，天花板上悬挂有可调整长度的耳机，按播放键即可聆听有声书籍。大厅里设有自主办证机器和自助借还机，后者还支持[人脸识别](../Page/人脸识别.md "wikilink")。馆内于负一层设置的图书自动分拣系统则减轻了工作人员的负担。\[44\]

### 分馆概况

中山纪念图书馆为中山市市级公共图书馆，以它为总馆，遍布全市24个镇区的24家镇区图书馆为分馆，26家街区自助图书馆和270个村居图书室为基层服务点，一套图书馆总分馆三级服务体系和通借通还服务模式构建起来\[45\]\[46\]。

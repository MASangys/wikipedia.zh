美国[中央情报局](../Page/中央情报局.md "wikilink")的**西藏项目**（）是指[美国](../Page/美国.md "wikilink")根据1951年到1956年间向[第十四世达赖喇嘛](../Page/第十四世达赖喇嘛.md "wikilink")做出的承诺，采取的一系列政治謀略、散布[宣传](../Page/宣传.md "wikilink")，辅助军事支援和情报收集行动\[1\]。

此任务是向中情局下达的，其也与包括[国务院和](https://zh.wikipedia.org/wiki/美國國務院 "wikilink")[国防部在内的其他](https://zh.wikipedia.org/wiki/美國國防部 "wikilink")[美国政府机构协同完成](https://zh.wikipedia.org/wiki/美国政府 "wikilink")\[2\]。起初的计划是替几个相互独立的西藏抵抗组织增強力量，后来设计了在[尼泊尔](../Page/尼泊尔.md "wikilink")边境約2000人的辅助军事组织。截至1964年2月，中情局所有西藏计划的年度开销超过170万美元\[3\]。

由於政治環境改變，项目在1960年代晚期逐渐停止，[尼克松1972年访华时结束](../Page/1972年尼克松访华.md "wikilink")\[4\]。

## 总览

中情局西藏计划通过政治行动和宣传削弱[中华人民共和国政府对](https://zh.wikipedia.org/wiki/中华人民共和国政府 "wikilink")[西藏的影响和能力](https://zh.wikipedia.org/wiki/西藏 "wikilink")\[5\]。[美国国家安全委员会](../Page/美国国家安全委员会.md "wikilink")[303委员会批准并在后来支持了此项目](https://zh.wikipedia.org/wiki/303委员会 "wikilink")\[6\]。项目包括几个秘密行动，[代号如下](https://zh.wikipedia.org/wiki/代号 "wikilink")：\[7\]\[8\]

  - **ST CIRCUS** —— 在[塞班岛](../Page/塞班岛.md "wikilink")和[科羅拉多州](../Page/科羅拉多州.md "wikilink")的[赫尔营陆军基地](https://zh.wikipedia.org/wiki/赫尔营陆军基地 "wikilink")，训练西藏游击队员
  - **ST BARNUM** —— 向西藏空运中情局特工、军事补给和支援装备
  - **ST BAILEY** —— 秘密宣传项目

1950年代早期，中情局从特殊行动小组抽调队伍，训练引导西藏抵抗武装反抗[中国人民解放军](../Page/中国人民解放军.md "wikilink")。西藏游击队员在美国[落磯山脈](https://zh.wikipedia.org/wiki/落磯山脈 "wikilink")、[科羅拉多州](../Page/科羅拉多州.md "wikilink")赫尔营军事基地接受训练\[9\]\[10\]\[11\] ，后来让他们从[尼泊尔](../Page/尼泊尔.md "wikilink")和[印度](../Page/印度.md "wikilink")反抗解放军。这些[準军事组织在达赖喇嘛出走後负责護送他秘密前往印度](https://zh.wikipedia.org/wiki/準军事组织 "wikilink")\[12\]。

[Tibetan_Supply_drop.jpg](https://zh.wikipedia.org/wiki/File:Tibetan_Supply_drop.jpg "fig:Tibetan_Supply_drop.jpg")向[四水六崗](../Page/四水六崗.md "wikilink")空投物資\]\] 1955年，西藏本地的一些领袖秘密策划了1956年的反抗，围困了几个中国政府组织，杀死了一些中国政府雇员和其他[汉族](../Page/汉族.md "wikilink")人\[13\]。1957年5月，一个拥有武装的反抗组织[四水六崗](../Page/四水六崗.md "wikilink")成立，幾個月後收到中情局的资助\[14\]。

根据老游击队员Palden Wangyal的陈述，这些破坏中国政府西藏设施的反抗组织直接由美国资助：

由2000名反抗士兵组成的[四水六岗由](https://zh.wikipedia.org/wiki/四水六岗 "wikilink")[恩珠倉·貢布扎西領導](https://zh.wikipedia.org/wiki/恩珠倉·貢布扎西 "wikilink")，部份指揮官曾在中情局受訓\[15\]。这些队员擅长从尼泊尔高耸山脉的基地伏击中国目标\[16\]。

1958年，康区騷乱时，阿塔和Lhotse两名游擊队员试图会见达赖喇嘛尋求支持。[噶厦内务大臣](https://zh.wikipedia.org/wiki/噶厦 "wikilink")[帕拉·土登为登](../Page/帕拉·土登为登.md "wikilink")认为不合时宜，阻止了会见。[茨仁夏加](../Page/茨仁夏加.md "wikilink")说：“帕拉从來没有告诉达赖喇嘛和噶厦那两个人的到来。他也没告诉达赖喇嘛美国愿意提供援助。”\[17\]

四水六岗继续攻击中国政府官员、破坏通信线缆并且攻击中国军队\[18\]。

1959年[藏历新年](../Page/藏历新年.md "wikilink")，[拉萨发生](https://zh.wikipedia.org/wiki/拉萨 "wikilink")[骚乱](https://zh.wikipedia.org/wiki/1959年西藏骚乱 "wikilink")，中国人民解放军[西藏军区部队对骚乱動武鎮壓](https://zh.wikipedia.org/wiki/西藏军区 "wikilink")，达赖喇嘛遂流亡印度\[19\]。

中情局特工約翰·肯尼思·諾斯（）表示，中情局支持四水六崗衛教军最大的情報收穫，來自1961年10月25日對解放軍車隊的襲擊，衛教军40余人騎馬襲擊解放军西藏军区边防5团盛永琛副团长带領的一个加强班，全殲解放軍11人，獲得1,500件檔案，其中包括[大躍進失敗的確切證據以及解放軍戰鬥序列](https://zh.wikipedia.org/wiki/大躍進 "wikilink")。其他情報收穫包括1962年獲得中國導彈計畫與發展原子彈的情報，以及在1964年埋下探測器，成為美國證實中國在[羅布泊進行原子彈試爆的最早來源](https://zh.wikipedia.org/wiki/羅布泊 "wikilink")。\[20\]\[21\]

中情局在1960年代晚期逐渐终止。在1972年尼克松访华，调整对华政策时结束。1500名中情局情报的游击队员各获得了10000卢比，在印度购买土地或做生意，不再与中国解放军战斗。白宫因为担心严重破坏中美关系，停止培训西藏游击队员\[22\]。

## 费用

此表格列出了1968年一年中情局西藏项目的开销，作为例子：

| 项目                                    | 费用               |
| ------------------------------------- | ---------------- |
| 培训下级军官                                | US$45,000\[23\]  |
| 纽约和日内瓦的游说机构                           | US$75,000\[24\]  |
| 杂项                                    | US$125,000\[25\] |
| [达赖喇嘛](../Page/达赖喇嘛.md "wikilink")的补助 | US$180,000\[26\] |
| 操作费用                                  | US$225,000\[27\] |
| 科罗拉多策反培训费用                            | US$400,000\[28\] |
| 2100名西藏游击队员的工资                        | US$500,000\[29\] |

从1950年代晚期到1974年，达赖喇嘛每年获得中情局的18万美元资助。这些资金是给他个人的，但是将多数用于西藏流亡政府的活动，资助外交官员游说国际支持\[30\]。

尽管达赖喇嘛的抗辩逐渐丧失效力，他在纽约的办公室并没有放弃向一些联合国代表游说。达赖喇嘛也由美国前驻联合国代表支援\[31\]。

## 批评

14世达赖喇嘛在他1991年的自传《[流亡中的自在](https://zh.wikipedia.org/wiki/流亡中的自在 "wikilink")》中，批评中情局“支援西藏独立运动并非因为关心西藏独立，而是颠覆全世界的共产政府行动的一部分”\[32\]。1999年，达赖喇嘛说中情局西藏计划对西藏有害，因为目的主要是服务美国的利益。“一旦[美国对华政策改变](../Page/中美关系.md "wikilink")，帮助就停止了\[33\]。」2009年，[嘉乐顿珠](../Page/嘉乐顿珠.md "wikilink")接受《[華爾街日報](https://zh.wikipedia.org/wiki/華爾街日報 "wikilink")》采访时称：「我不能說中情局的幫助很有用。他們的幫助確實激怒了中國人，導致了報復。 我為此感到非常難過。」「美国人只能给中国找点麻烦而已，并没有对西藏的长久政策”\[34\]，而在其回忆录《噶伦堡的面条商人：我为西藏奋斗的背后不为人知的故事》的最后表示后悔与美国中情局的合作。\[35\]

**中華商場**是[台灣](https://zh.wikipedia.org/wiki/台灣 "wikilink")[台北市](https://zh.wikipedia.org/wiki/台北市 "wikilink")1992年已拆除的大型[商場](../Page/商場.md "wikilink")，原址位於[中華路一段中央](https://zh.wikipedia.org/wiki/中華路_\(台北市\) "wikilink")，北起[忠孝西路口](https://zh.wikipedia.org/wiki/忠孝西路 "wikilink")，南至[愛國西路口](https://zh.wikipedia.org/wiki/愛國西路 "wikilink")，由一連八座三層樓連棟式的樓座所組成，於1961年落成啟用。

中華商場曾為[大台北地區規模最大的公有綜合商場](https://zh.wikipedia.org/wiki/大台北地區 "wikilink")，緊鄰[西門町](../Page/西門町.md "wikilink")，興盛一時；後來為了因應[市區重新規劃與](https://zh.wikipedia.org/wiki/都市更新 "wikilink")[捷運的施工進程等需求而於](https://zh.wikipedia.org/wiki/台北捷運 "wikilink")1992年10月20日之後拆除，原址則配合[中華路改造計畫而成為](https://zh.wikipedia.org/wiki/中華路改造計畫 "wikilink")[林蔭大道的一部分](https://zh.wikipedia.org/wiki/林蔭大道 "wikilink")。

## 所在地歷史概述

中華商場的位置與[清治時期所興建](https://zh.wikipedia.org/wiki/台灣清治時期 "wikilink")[台北府城的](https://zh.wikipedia.org/wiki/台北府城 "wikilink")[西城牆位置大致重疊](https://zh.wikipedia.org/wiki/台北城牆 "wikilink")，城牆的高度則約略為商場二樓的高度。到了[日治時期](../Page/台灣日治時期.md "wikilink")，西城牆連同[西門於](https://zh.wikipedia.org/wiki/台北府城西門 "wikilink")1900年代初全面拆除，城牆原址隨後配合已填平的西[護城河稍作拓寬](https://zh.wikipedia.org/wiki/護城河 "wikilink")，改築成為西三線路（即現今中華路一段）。由於西三線路在興築之前已有[鐵路](../Page/鐵路.md "wikilink")（即[縱貫線](../Page/縱貫線_\(北段\).md "wikilink")）沿路行經，所以總體路幅比當時的其他三條三線路來得寬廣。

## 商場的起建與落成

[中華商場05.jpg](https://zh.wikipedia.org/wiki/File:中華商場05.jpg "fig:中華商場05.jpg") 1949年[國共內戰時](https://zh.wikipedia.org/wiki/國共內戰 "wikilink")，大批難民隨[中華民國政府](../Page/中華民國政府.md "wikilink")大規模撤退來到台灣，許多來自[中國大陸各地的居民因為一時無法在台北市內找到適當的棲身之處](https://zh.wikipedia.org/wiki/中國大陸 "wikilink")；中央政府為求有效安置，便委託[台北市警民協會](https://zh.wikipedia.org/wiki/台北市警民協會 "wikilink")，沿著行經中華路的鐵路東側，利用原有三線路中緊鄰鐵路側車道的空間，搭建三列臨時性質的簡易竹造[棚屋](https://zh.wikipedia.org/wiki/棚屋 "wikilink")，低價出租供居民暫住並擺攤維持生計。之後隨著湧入該處寄居的人越來越多，在毫無限制的持續擴建下形成大型[違章建築群](https://zh.wikipedia.org/wiki/違章建築 "wikilink")，不但生活環境惡劣，也衍生出不少[衛生與](https://zh.wikipedia.org/wiki/衛生 "wikilink")[治安問題](https://zh.wikipedia.org/wiki/治安 "wikilink")，被批評為「都市之[瘤](https://zh.wikipedia.org/wiki/瘤 "wikilink")」。

1960年，中央政府為了整頓市容，決定於該處原地改建新式商場，同年7月1日起開始進行違建的拆除與商場的興建工作。當時迫於經費不足的限制，因此以居民一次負擔的方式籌措改建經費。負責規劃興建施工的是[陸根記營造](https://zh.wikipedia.org/wiki/陸根記營造 "wikilink")，工程進展相當快速；經過八個多月的施工，1961年4月22日，單一建築體呈長條狀、以連棟方式形成一長串排列的八座三層樓[水泥](../Page/水泥.md "wikilink")建築正式落成，命名為「中華商場」，由[台灣警備總司令部司令](https://zh.wikipedia.org/wiki/台灣警備總司令部 "wikilink")[黃-{杰}-命名並題字](../Page/黃杰_\(將軍\).md "wikilink")。

八座三層商場建築由北（忠孝西路口）而南（愛國西路口），以「[八德](https://zh.wikipedia.org/wiki/四維八德#八德 "wikilink")」為名，分別為[忠](https://zh.wikipedia.org/wiki/忠 "wikilink")、[孝](../Page/孝.md "wikilink")、[仁](../Page/仁.md "wikilink")、[愛](../Page/愛.md "wikilink")、[信](https://zh.wikipedia.org/wiki/信 "wikilink")、[義](https://zh.wikipedia.org/wiki/義 "wikilink")、[和](https://zh.wikipedia.org/wiki/和 "wikilink")、[平棟](https://zh.wikipedia.org/wiki/平 "wikilink")，總長1,171[公尺](https://zh.wikipedia.org/wiki/公尺 "wikilink")，興建經費總計為[新台幣](https://zh.wikipedia.org/wiki/新台幣 "wikilink")47,335,845元。

## 商場的黃金時代

[中華商場03.jpg](https://zh.wikipedia.org/wiki/File:中華商場03.jpg "fig:中華商場03.jpg") 中華商場落成後，總共可容納1,644個租戶，平均每戶只分得約2[坪](../Page/坪.md "wikilink")大的空間。八座商場建築中，每一座商場內的租戶所經營的行業各有其特色：販售[家用電器](https://zh.wikipedia.org/wiki/家用電器 "wikilink")、[音響](../Page/音響.md "wikilink")與[電子零組件的商家多位於忠](https://zh.wikipedia.org/wiki/電子 "wikilink")、孝兩棟，1980年代隨著[個人電腦產業興起](https://zh.wikipedia.org/wiki/個人電腦 "wikilink")，販售相關[軟體](https://zh.wikipedia.org/wiki/軟體 "wikilink")、[硬體與週邊設備的商家也開始出現](https://zh.wikipedia.org/wiki/硬體 "wikilink")；[玉器](https://zh.wikipedia.org/wiki/玉器 "wikilink")、[琺瑯及台灣民俗藝品曾經是](https://zh.wikipedia.org/wiki/琺瑯 "wikilink")1970年代台灣中南部及[日本](../Page/日本.md "wikilink")遊客採購的重點，此類商家與販售[古董](https://zh.wikipedia.org/wiki/古董 "wikilink")、[古玩及](https://zh.wikipedia.org/wiki/古玩 "wikilink")[字畫的商家皆分布在仁](https://zh.wikipedia.org/wiki/字畫 "wikilink")、愛兩棟，[郵票](https://zh.wikipedia.org/wiki/郵票 "wikilink")、古幣蒐集與交換專門店及[相命](https://zh.wikipedia.org/wiki/相命 "wikilink")、[堪輿舖子則零星穿插其間](https://zh.wikipedia.org/wiki/堪輿 "wikilink")；信、義、和三棟有許多中國各地的[小吃](../Page/小吃.md "wikilink")，著名的有「徐州啥鍋」、「真北平飯館」、「點心世界」等；成衣、[制服](../Page/制服.md "wikilink")、[牛仔褲](https://zh.wikipedia.org/wiki/牛仔褲 "wikilink")、軍用衣料、老人茶館、[旗幟](../Page/旗幟.md "wikilink")徽章等相關商家則分別位於和、平兩棟。早期中華商場內商家與顧客漫天叫價、討價還價（甚至殺價）的交易風格盛極一時，懂得行情的顧客較有機會買到價格實惠的物品。當年在中華商場知名的飯館有「-{致}-美樓」、「清真館」、「老陸餡餅」、「老夏水餃」、炸麻花、山西小吃店，還有後來的「徐州啥鍋」、「上大人酒釀湯圓」、水煎包，還有晚上9點以後在六棟長沙街口的炸豬排及各式小菜。

1960年代與1970年代，是中華商場的全盛時期。由於西邊即為日治時代以來的休閒娛樂街區——眾多[電影院](../Page/電影院.md "wikilink")匯集的西門町，因此吸引了不少人潮；而在商場落成後數年間，附近也有一些初具規模的[百貨公司](../Page/百貨公司.md "wikilink")接連開幕（台灣首間大型百貨公司「[第一百貨](https://zh.wikipedia.org/wiki/第一百貨 "wikilink")」即位於中華路上），加上[城中市場](https://zh.wikipedia.org/wiki/城中市場 "wikilink")、衡陽路與博愛路等傳統商圈，配合周圍方便的交通（多條[公車路線行經](https://zh.wikipedia.org/wiki/公車 "wikilink")），共同形成當時大台北最繁華的[中心商業區](https://zh.wikipedia.org/wiki/中心商業區 "wikilink")。

除此之外，西門圓環及其以北商場建築棟與棟間的重要路口，後來亦設置行人天橋，不但藉此得以安全跨越[鐵路平交道](https://zh.wikipedia.org/wiki/鐵路平交道 "wikilink")，連絡中華路東、西兩側，還可以直通商場二樓。由於商場位置離[台北車站並不算遠](https://zh.wikipedia.org/wiki/台北車站 "wikilink")；且在鐵路尚未地下化的時代，北上的列車進入台北車站之前，必定會經過商場西側的鐵道，因此中華商場經常成為中南部遊客旅途中的重要印象。而商場的屋頂，也曾經成為各家企業競相設置大型[霓虹燈](../Page/霓虹燈.md "wikilink")廣告的場所，入夜之後，五光十色的霓虹燈更襯托出街景的亮麗。

## 從沒落到化為瓦礫

[中華商場06.jpg](https://zh.wikipedia.org/wiki/File:中華商場06.jpg "fig:中華商場06.jpg")

| 拆除時間        | 拆除建物     |
| ----------- | -------- |
| 1992年10月20日 | 忠棟       |
| 1992年10月23日 | 孝棟、仁棟    |
| 1992年10月27日 | 愛棟、信棟    |
| 1992年10月30日 | 義棟、和棟、平棟 |
|             |          |

到了1980年代，隨著台北市的商業重心逐漸東移，加上原有消費型態的快速改變，年久失修，已經顯得陳舊的中華商場吸引力與日俱減，影響力也大不如前；而整個西區商圈幾乎也都逐漸面臨老舊所帶來的瓶頸。沿商場西側而行的鐵路，雖然已經於1989年地下化，但商場西側長久累積下來的髒亂，卻已經難以改善；且對都市景觀造成負面影響。

此外，[台北捷運的興建計畫中](https://zh.wikipedia.org/wiki/台北捷運 "wikilink")，[南港線](https://zh.wikipedia.org/wiki/台北捷運南港線 "wikilink")－[板橋線行經中華路](https://zh.wikipedia.org/wiki/台北捷運板橋線 "wikilink")，並設置[西門站](https://zh.wikipedia.org/wiki/西門站 "wikilink")，須取得工程所需用地。因此隨著中華路改造計畫的提出，市政府決定拆除商場，並提出後續方案：包括未來優先進駐興建的[西門地下街](../Page/西門地下街.md "wikilink")及費用補償等。但由於此時商場內租戶以年紀較大者居多，有些租戶經濟能力較薄弱，以及情感上因素使然，因而引發普遍反彈，拆除前夕甚至頻傳激烈抗爭。1992年10月20日，市政府正式下令拆除商場，8座商場建築全部被拆除殆盡。

原本在商場內作生意的商家，有一些搬遷到附近或較遠的地方繼續營業（例如販售音響的商家移到中華路東側和武昌街；販售電腦週邊產品的商家大部分移到[光華商場](../Page/光華商場.md "wikilink")，也有一些移到[西寧電子商場](https://zh.wikipedia.org/wiki/西寧電子商場 "wikilink")；小吃店移至商場拆除當時尚未整建的原[理教公所遺址](https://zh.wikipedia.org/wiki/理教公所 "wikilink")…等），但更多的商家卻從此消失無蹤；當時安置計畫未盡完善的結果，也導致許多租戶倉皇搬離而不知所措。至於商場原址，大部分於商場拆除後隨即作為捷運工程用地。直到地下部分工程告一段落並展開地面恢復工作後，即配合改造中華路為林蔭大道的計畫，原地面改築為車道與安全島（2003年完工），捷運西門站與西門地下街則設置於原商場愛棟、信棟、義棟原址地下。

**中華民國國民身分證**，是[中華民國國民](../Page/中華民國國民.md "wikilink")的身分證明，目前是在[臺澎金馬設有](https://zh.wikipedia.org/wiki/臺澎金馬 "wikilink")[戶籍人民領有](https://zh.wikipedia.org/wiki/戶籍 "wikilink")。《[姓名條例施行細則](https://zh.wikipedia.org/wiki/s:姓名條例施行細則 "wikilink")》第二條：「國內有戶籍國民本名之證明為國民身分證，未滿十四歲者，得用[戶口名簿](../Page/戶口名簿.md "wikilink")或[戶籍謄本](../Page/戶籍謄本.md "wikilink")代替之。」\[1\]

## 歷史

### 起源

[民國時期的國民身分證.jpg](https://zh.wikipedia.org/wiki/File:民國時期的國民身分證.jpg "fig:民國時期的國民身分證.jpg") 1931年12月，[國民政府](../Page/國民政府.md "wikilink")頒佈《戶籍法》，規定要開展[戶籍登記和](https://zh.wikipedia.org/wiki/戶籍 "wikilink")[戶口調查](../Page/人口普查.md "wikilink")，用以掌握各地人口分佈情況。

1939年，國民政府頒佈《國民兵役證施行辦法》，規定適齡[服兵役男子必須辦理](https://zh.wikipedia.org/wiki/服兵役 "wikilink")「國民兵役證」，這應該是中華民國實行身分證制度的雛形；1941年，為進一步加強對壯丁的管理、便於戶口調查，國民政府頒佈《國民兵身分證暫行條例》，規定凡年滿18歲至45歲的役齡男子（必要時可前後各延伸5年，即13歲至50歲），必須領取國民兵身分證。  1946年，國民政府修正公布《戶籍法》，明確提出「已辦戶籍登記之地方，得製發國民身分證，或經內政部核准以戶籍謄本代之」\[2\]；同年6月，[行政院](../Page/行政院.md "wikilink")頒佈了《戶籍法施行細則》，對國民身分證的有關事項做出了具體規定；同年10月，依據[內政部公布的](https://zh.wikipedia.org/wiki/中華民國內政部 "wikilink")《國民身分證實施暨公務員首先領證辦法》，中央及省級公務員率先領取國民身分證\[3\]。

#### 第一代身分證

[ROC_National_ID_Card_in_Guangzhou_City_19470930.jpg](https://zh.wikipedia.org/wiki/File:ROC_National_ID_Card_in_Guangzhou_City_19470930.jpg "fig:ROC_National_ID_Card_in_Guangzhou_City_19470930.jpg")發出的國民身分證\]\] 1947年5月，國民政府開始製作雙頁折疊式白色第一代身分證，發給18歲以上國民。\[4\]\[5\]

早期[照片並不普及](https://zh.wikipedia.org/wiki/照片 "wikilink")，部分省市政府所發放的國民身分證並不張貼照片，不過一定都有記錄左右手[指紋](../Page/指紋.md "wikilink")；規定指紋分「箕」「斗」兩類，分別用「╳」和「○」表示。記錄指紋時由左到右自大拇指起數至小拇指，十個手指的指紋均要標識，如「左○○○○╳，右○○╳○╳」\[6\]。

另外又以地名簡稱編號來辨識。例如[臺灣省政府](../Page/臺灣省政府.md "wikilink")發行的一張國民身分證「市城文字第003號之5」，「市」字代表[臺北市](../Page/臺北市.md "wikilink")，「城」代表是[城中區](../Page/城中區_\(台北市\).md "wikilink")，「文」為[文武里](../Page/建國里_\(台北市\).md "wikilink")，「003之5」則是持證者所屬戶號。

### 臺灣

在[臺灣](../Page/臺灣.md "wikilink")，[日治時期](https://zh.wikipedia.org/wiki/臺灣日治時期 "wikilink")[臺灣總督府](../Page/臺灣總督府.md "wikilink")為方便管理[臺灣人](../Page/臺灣人.md "wikilink")，將人頭編碼，並配發證件。\[7\]

[戰後](../Page/臺灣戰後時期.md "wikilink")，[中華民國政府](../Page/中華民國政府.md "wikilink")接收臺灣，類似[良民證](../Page/良民證.md "wikilink")的臺灣身分證取消。

1949年，[臺灣省政府](../Page/臺灣省政府.md "wikilink")避免[共產黨滲透臺灣](https://zh.wikipedia.org/wiki/共產黨 "wikilink")，宣佈[臺灣省戒嚴令](../Page/臺灣省戒嚴令.md "wikilink")後，立刻進行戶口普查，並正式換發身分證。因為臺灣日治時期遺留戶政資料完整，換發迅速。

除此，1949年12月24日，大批[外省人](../Page/外省人.md "wikilink")開始以各地良民證與居留證，換發標明「外省」字樣的身分證。來臺人員換發臺灣身分證較臺籍人士困難，不僅須申報表和一張保證書，也須找兩家店號作保。外省籍人士若無身分證，遭中華民國[憲](../Page/中華民國憲兵.md "wikilink")、[軍](../Page/中華民國國軍.md "wikilink")、[警盤問](https://zh.wikipedia.org/wiki/中華民國警察 "wikilink")，可能被迫入伍服[兵役](https://zh.wikipedia.org/wiki/兵役 "wikilink")，成為補充兵員。\[8\]

#### 第二代身分證

1954年5月，整理戶籍時第一次全面換發國民身分證給14歲以上國民，形成藍色雙頁折疊式第二代身分證。\[9\]

#### 第三代身分證

[ROC_National_ID_Card_of_Chiang_Kai-shek_19650417_face.jpg](https://zh.wikipedia.org/wiki/File:ROC_National_ID_Card_of_Chiang_Kai-shek_19650417_face.jpg "fig:ROC_National_ID_Card_of_Chiang_Kai-shek_19650417_face.jpg") 1965年4月至1966年5月，第二次全面換證，變成第三代身分證：男為[淺綠色](https://zh.wikipedia.org/wiki/綠 "wikilink")，女為[淺紅色](https://zh.wikipedia.org/wiki/紅 "wikilink")，改成單頁正面膠封。\[10\]1965年4月17日[臺灣省陽明山管理局製發的編號第一號身分證](https://zh.wikipedia.org/wiki/臺灣省陽明山管理局 "wikilink")，擁有者就是當時的[中華民國總統](../Page/中華民國總統.md "wikilink")[蔣中正](../Page/蔣中正.md "wikilink")，字號為「Y10000001」，包括英文字母共九碼，當時尚無檢查號碼。\[11\]1969年（民國58年）電腦作業，加入第十碼，以及出生登記就編身分證統一編號；第十碼就是現在最右方的檢查號碼。\[12\]

#### 第四代身分證

1975年10月至1976年，第三次全面換證，變成第四代身分證。男為[淺藍色](https://zh.wikipedia.org/wiki/藍 "wikilink")，女為[奶油色](https://zh.wikipedia.org/wiki/奶油 "wikilink")，正面套印內政部大印，廢除口號。\[13\]

#### 第五代身分證

1986年，第四次全面換證，變成第五代身分證。男為[土黃色](https://zh.wikipedia.org/wiki/黃 "wikilink")，女為[桃紅色](https://zh.wikipedia.org/wiki/紅 "wikilink")，廢除出生別、血型、教育程度欄位。\[14\]配合臺灣地區戶政電腦化於1995年起實施，身分證同時由手寫資料改為[電腦列印](https://zh.wikipedia.org/wiki/電腦 "wikilink")。

##### 重號情形

在1995年之前，因為手寫資料的緣故，常有重號的情況發生；當[公務員](../Page/公務員.md "wikilink")疏於核對身分時，讓身分證重號的另一人發生困擾，甚至生活陷入困境。即使今日內政部已在積極處理此類問題，但類似的事件還是時有所聞。

##### 捺指紋違憲性

1997年全文修正公布的《戶籍法》第八條：「人民年滿十四歲者，應請領國民身分證；未滿十四歲者，得申請發給。」\[15\]「依前項請領國民身分證，應捺[指紋](../Page/指紋.md "wikilink")並錄存。但未滿十四歲請領者，不予捺指紋，俟年滿十四歲時，應補捺指紋並錄存。」\[16\]「請領國民身分證，不依前項規定捺指紋者，不予發給。」\[17\]但此條文被爭議為侵犯[人權時](https://zh.wikipedia.org/wiki/人權 "wikilink")，[司法院大法官](../Page/司法院大法官.md "wikilink")於2005年6月10日以《[司法院釋字第599號解釋](https://zh.wikisource.org/zh-hant/%E5%8F%B8%E6%B3%95%E9%99%A2%E9%87%8B%E5%AD%97%E7%AC%AC599%E8%99%9F%E8%A7%A3%E9%87%8B)》暫時處分六個月，以暫停捺指紋條款；司法院大法官並於同年9月28日做成《[司法院釋字第603號解釋](https://zh.wikisource.org/wiki/%E5%8F%B8%E6%B3%95%E9%99%A2%E9%87%8B%E5%AD%97%E7%AC%AC603%E8%99%9F%E8%A7%A3%E9%87%8B)》，認定捺指紋條款[違憲](https://zh.wikipedia.org/wiki/違憲 "wikilink")、失效。

#### 第六代身分證

[ROC_ID_2005_specimen_front.jpg](https://zh.wikipedia.org/wiki/File:ROC_ID_2005_specimen_front.jpg "fig:ROC_ID_2005_specimen_front.jpg") [ROC_ID_2005_specimen_back.jpg](https://zh.wikipedia.org/wiki/File:ROC_ID_2005_specimen_back.jpg "fig:ROC_ID_2005_specimen_back.jpg") 第五代身分證因發行多年，常遭[偽造](../Page/偽造文書.md "wikilink")。所以政府在2000年初起便開始研議更換新身分證，原本身分證設計初稿印有象徵[華人的](https://zh.wikipedia.org/wiki/華人 "wikilink")[龍圖案](https://zh.wikipedia.org/wiki/龍 "wikilink")，但由於「龍」在[宗教](../Page/宗教.md "wikilink")上有敏感性，故被一些信奉[亞伯拉罕諸教](../Page/亞伯拉罕諸教.md "wikilink")等人士所反對；經過數年設計，正式版本才於2004年定稿，且於2005年12月21日起至2006年12月31日全面換發。

第六代身分證與以往不同的是，以淺紫、淺咖啡漸層隔色，不再以顏色區分性別。\[18\]第一張新式身分證發給代言人[孫越](https://zh.wikipedia.org/wiki/孫越_\(台灣演員\) "wikilink")。但對於照片的規定嚴格許多；新式身分證照片有強制露出[耳朵的規定](https://zh.wikipedia.org/wiki/耳朵 "wikilink")，曾被部分耳部缺損（等）人士抗議，但已經獲得改善。新式身分證每張均有特定的電腦編號，換（補）發後不再沿用。

第六代身分證樣本照片為「陳筱玲」，內政部宣稱是[電腦合成影像](https://zh.wikipedia.org/wiki/電腦合成影像 "wikilink")。2009年經[新聞報導確認](https://zh.wikipedia.org/wiki/新聞 "wikilink")，照片雖經過電腦處理，但其實是運用一位[公務員](../Page/公務員.md "wikilink")游淑妃的相片[修圖完成](https://zh.wikipedia.org/wiki/修圖 "wikilink")\[19\]。若持證者為現役[中華民國國軍](../Page/中華民國國軍.md "wikilink")[軍人](https://zh.wikipedia.org/wiki/軍人 "wikilink")，則役別欄將會輸入「現役」；退伍後，役別欄依[兵種之不同有](https://zh.wikipedia.org/wiki/兵種 "wikilink")「常官備役」、「預官」、「常士備役」、「預士」、「常兵備役」、「補充兵」、「國民兵」、「替代備役」八種，但除役後換證不會顯示持證者曾任職之役別，此與舊式身分證不同之處\[20\]。

2008年5月全文修正的《[戶籍法](http://law.moj.gov.tw/LawClass/LawAll.aspx?PCode=D0030006)》第五十六條：「國民身分證應隨身攜帶，非依法律不得扣留。」\[21\]沒帶身分證，《戶籍法》罰則章雖不處罰，但[警察若因而顯然無法依據](https://zh.wikipedia.org/wiki/中華民國警察 "wikilink")《[警察職權行使法](http://law.moj.gov.tw/LawClass/LawAll.aspx?PCode=D0080145)》查證可疑人民身分時，得以將該人民帶往勤務處所查證，最多三小時。\[22\]再者，在臺設籍居民沒帶國民身分證，就不能領取總統副總統或者公職人員選舉票。\[23\]\[24\]《戶籍法》第五十七條：「有戶籍國民年滿十四歲者，應申請初領國民身分證；未滿十四歲者，得申請發給。」\[25\]「國民身分證、戶口名簿，滅失或遺失者，應申請補領。」\[26\]

為防止[身份假冒](https://zh.wikipedia.org/wiki/身份盗窃 "wikilink")，向政府機關、[公用事業](https://zh.wikipedia.org/wiki/公用事業 "wikilink")（如[各電信公司](../Page/臺灣電信業者列表.md "wikilink")）等申辦各項業務時，除身分證外，還需要另一有照片及身分證號的證件如[全民健康保險](../Page/全民健康保險.md "wikilink")卡、[駕駛執照等](../Page/中華民國汽車駕駛執照.md "wikilink")。此外，身分證上的新發/換發/補發日期可判斷身分證是否已掛失。

#### 第七代身分證

內政部規劃第七代晶片身分證，預計2020年10月開始換發，並且結合自然人憑證等功能。\[27\]\[28\]\[29\]\[30\]

2019年11月22日，[台灣人權促進會](../Page/台灣人權促進會.md "wikilink")、[台北律師公會](https://zh.wikipedia.org/wiki/台北律師公會 "wikilink")、[台灣同志諮詢熱線](https://zh.wikipedia.org/wiki/台灣同志諮詢熱線 "wikilink")、[台灣伴侶權益推動聯盟](../Page/台灣伴侶權益推動聯盟.md "wikilink")、[婚姻平權大平台](https://zh.wikipedia.org/wiki/婚姻平權大平台 "wikilink")、[婦女新知基金會](../Page/婦女新知基金會.md "wikilink")、[台灣守護民主平台](https://zh.wikipedia.org/wiki/台灣守護民主平台 "wikilink")、[經濟民主連合](../Page/經濟民主連合.md "wikilink")、[國會調查兵團](https://zh.wikipedia.org/wiki/國會調查兵團 "wikilink")、[開放文化基金會](https://zh.wikipedia.org/wiki/開放文化基金會 "wikilink")、[中華民國軟體自由協會等](https://zh.wikipedia.org/wiki/中華民國軟體自由協會 "wikilink")30個團體發布聯合聲明《我們拒絕晶片身分證！》，抨擊行政院罔顧正當程序、強行推動晶片身分證，呼籲行政院暫停晶片身分證決標印製案\[31\]。

## 編號規則

目前的中華民國身分證字號一共有十碼，包括起首一個大寫的[英文字母](../Page/英文字母.md "wikilink")與接續的九個[阿拉伯數字](https://zh.wikipedia.org/wiki/阿拉伯數字 "wikilink")。其中首碼英文代碼是以初次登記的[戶籍地來區分的](https://zh.wikipedia.org/wiki/戶籍 "wikilink")，而首位數字則是拿來區分性別，[男性](../Page/男性.md "wikilink")為1、[女性](../Page/女性.md "wikilink")為2，與[ISO 5218以及類似的](https://zh.wikipedia.org/wiki/ISO_5218 "wikilink")《[中華民國國家標準](../Page/中華民國國家標準.md "wikilink")》CNS 8381《資訊交換-人類性别表示法》的用法相同。

## 驗證規則

身分證號碼的驗證公式如下\[32\]：

假設有一身分證號碼為M140051653，而M轉換為數值是21，將每一碼拆開後分別編號：

|         |         |         |         |         |         |         |         |         |            |            |
| ------- | ------- | ------- | ------- | ------- | ------- | ------- | ------- | ------- | ---------- | ---------- |
| \(n_1\) | \(n_2\) | \(n_3\) | \(n_4\) | \(n_5\) | \(n_6\) | \(n_7\) | \(n_8\) | \(n_9\) | \(n_{10}\) | \(n_{11}\) |
| 2       | 1       | 1       | 4       | 0       | 0       | 5       | 1       | 6       | 5          | 3          |
|         |         |         |         |         |         |         |         |         |            |            |

然後再把每一個數字依序乘上1、9、8、7、6、5、4、3、2、1、1，最後再相加：
\(n_1\times1+n_2\times 9+n_3\times 8+n_4\times7+n_5\times6+n_6\times5+n_7\times4+n_8\times3+n_9\times2+n_{10}\times1+n_{11}\times1\)

套入公式，結果為：

\(2 \times 1+1 \times 9+1 \times 8 +4 \times 7+0 \times 6 +0 \times 5+5 \times 4 +1 \times 3+6 \times 2 +5 \times 1+3 \times 1\)
\(=2+9+8+28+0+0+20+3+12+5+3\)
\(=90\)

然後再除以10：

\(\frac{90}{10}\)

如果整除，則為有效之身分證號碼：

\(90 \equiv 0 \pmod{10}\)

另外，身分證號「A123456789」因可符合驗證規則，故常遭冒用。持該身分證號者為出生於台北市內湖區的新北市民謝條根，他經常出入法院只為解決冒用所引發的糾紛，雖可以透過更換身分證號避免，但因換號後連帶需要更換許多文件，該人尚無意願換號。\[33\]\[34\]

## 軼事

  - 中華民國的[現役軍人以往是以](../Page/中華民國國軍.md "wikilink")「軍人補給證」做為軍人用身分證件，1996年改用「中華民國軍人身分證」。
  - 中華民國身分證上的樣本照片姓名為「陳筱玲」，由於樣貌秀麗引發[輿論關切](https://zh.wikipedia.org/wiki/輿論 "wikilink")，內政部稱此樣本照片為[合成照片](https://zh.wikipedia.org/wiki/合成照片 "wikilink")。事實上，「陳筱玲」的照片其實是利用一名曾任[模特兒](../Page/模特兒.md "wikilink")的公務員「游淑妃」的大頭照[修圖完成的](https://zh.wikipedia.org/wiki/修圖 "wikilink")。\[35\]
  - 第一位在陽明山管理局轄區內領第三代中華民國身分證的是[蔣中正](../Page/蔣中正.md "wikilink")，身分證字號是Y10000001，不過當時沒有檢查碼，只有九碼。\[36\]

**Ullmann反应**（乌尔曼反应），又称“Ullmann联芳烃合成”

经典的Ullmann反应：[芳香](../Page/芳香性.md "wikilink")[卤化物](../Page/卤化物.md "wikilink")与[铜](../Page/铜.md "wikilink")共热发生[偶联反应](../Page/偶联反应.md "wikilink")，得到[联芳烃](https://zh.wikipedia.org/wiki/联芳烃 "wikilink")，\[1\]
例如[碘苯](../Page/碘苯.md "wikilink")与铜共热得到[联苯](../Page/联苯.md "wikilink")。

[Ullmann_reaction.svg](https://zh.wikipedia.org/wiki/File:Ullmann_reaction.svg "fig:Ullmann_reaction.svg")

反应以德国化学家 Fritz Ullmann 的名字命名。\[2\]

这个反应的应用范围广泛，常用来合成许多对称与不对称的联芳烃衍生物。最常用的芳香[碘化物](../Page/碘化物.md "wikilink")，但[溴化物和](https://zh.wikipedia.org/wiki/溴化物 "wikilink")[氯化物甚至](https://zh.wikipedia.org/wiki/氯化物 "wikilink")[硫氰酸酯都可以应用](https://zh.wikipedia.org/wiki/硫氰酸酯 "wikilink")。除铜之外，镍也能使芳香卤化物偶联，\[3\]
例如[双(1,5-环辛二烯)镍(0)及](https://zh.wikipedia.org/wiki/双\(1,5-环辛二烯\)镍\(0\) "wikilink")[四(三苯基膦)镍(0)](https://zh.wikipedia.org/wiki/四\(三苯基膦\)镍\(0\) "wikilink")。

反应物环上的取代基对反应的影响很特殊。[硝基可以活化反应](https://zh.wikipedia.org/wiki/硝基 "wikilink")，但只有間位的硝基才有活化作用，位于鄰位和对位的硝基则无活化作用。\[4\]
R 和 OR
基团在所有位置都有活化作用。相反，OH、NH<sub>2</sub>、NHR、NHCOR、COOH、SO<sub>2</sub>NH<sub>2</sub>
等基团的存在会阻止反应进行，降低产率。

[Ullmann联苯醚合成](https://zh.wikipedia.org/wiki/Ullmann联苯醚合成 "wikilink")（[Ullmann缩合反应](https://zh.wikipedia.org/wiki/Ullmann缩合反应 "wikilink")）是
Ullmann
反应的一种变体。这一类由铜催化的[亲核芳香取代反应也被称为](https://zh.wikipedia.org/wiki/亲核芳香取代反应 "wikilink")
Ullmann-型反应。

## 反应机理

\[\rm \ Ar\!-\!I \  +Cu  \ \rightarrow  \ Ar\!-\!CuI\]
（[氧化加成](../Page/氧化加成.md "wikilink")）

\[\rm \ Ar\!-\!CuI \  + \  Cu \  \rightarrow  \ Ar\!-\!Cu  \ + \  CuI\]
（生成活性的[有机铜](https://zh.wikipedia.org/wiki/有机铜化合物 "wikilink")(I)中间体）

\[\rm \ Ar\!-\!Cu  \ +  \ I\!-\!Ar  \ \rightarrow  \ Ar_2CuI\] （氧化加成）

\[\rm \ Ar_2CuI  \ \rightarrow  \ Ar\!-\!Ar  \ + \  CuI\]
（[还原消除](https://zh.wikipedia.org/wiki/还原消除 "wikilink")）

## 例子

220°C沙浴的条件下，[邻硝基氯苯在铜](https://zh.wikipedia.org/wiki/邻硝基氯苯 "wikilink")-青铜合金催化下偶联为2,2'-二硝基联苯。\[5\]

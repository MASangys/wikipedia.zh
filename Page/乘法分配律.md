**乘法分配律**，也叫做**乘法分配性質**、**十字相乘法**或**十字交乘法**，公式是：\((a+b)(c+d)=ac+ad+bc+bd\)。

## 驗證

### 基本驗證

和平方可直接利用[因式分解驗證](../Page/因式分解.md "wikilink")。公式如下：

\[(a+b)(c+d)\] \(=a(c+d)+b(c+d)\)

\(=ac+ad+bc+bd\)

### 簡單驗證

和平方亦可以表格形式驗證：

|   x)   |  \(a\)  | \(+b\)  |
| :----: | :-----: | :-----: |
| \(c\)  | \(ac\)  | \(+bc\) |
| \(+d\) | \(+ad\) | \(+bd\) |
|        |         |         |

\((a+b)(c+d)=ac+ad+bc+bd\)

### 幾何驗證

[pvs.svg](https://zh.wikipedia.org/wiki/File:pvs.svg "fig:pvs.svg")
和平方可透過圖表來驗證。右圖中，是一個\((a+b)(c+d)\)的長方形。可在右圖中分割為四部分：

  - \(ac\)
  - \(bc\)
  - \(ad\)
  - \(bd\)

將四部分加在一起：

\[ac+ad+bc+bd\] \(=a(c+d)+b(c+d)\)

\(=(a+b)(c+d)\)

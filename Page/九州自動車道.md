**九州自動車道**（）是[日本](../Page/日本.md "wikilink")[福岡縣](https://zh.wikipedia.org/wiki/福岡縣 "wikilink")[北九州市](../Page/北九州市.md "wikilink")[門司區](../Page/門司區.md "wikilink")[門司交流道至](https://zh.wikipedia.org/wiki/門司交流道 "wikilink")[鹿兒島縣](https://zh.wikipedia.org/wiki/鹿兒島縣 "wikilink")[鹿兒島市](https://zh.wikipedia.org/wiki/鹿兒島市 "wikilink")[鹿兒島交流道](https://zh.wikipedia.org/wiki/鹿兒島交流道 "wikilink")，總長346.3公里的[高速公路](../Page/日本高速公路.md "wikilink")（[高速自動車國道](https://zh.wikipedia.org/wiki/高速自動車國道 "wikilink")），簡稱**九州道**（，）。[國土開發幹線自動車道的法定路線名為](../Page/國土開發幹線自動車道.md "wikilink")[九州縱貫自動車道鹿兒島線](https://zh.wikipedia.org/wiki/九州縱貫自動車道 "wikilink")（[門司交流道](https://zh.wikipedia.org/wiki/門司交流道 "wikilink")
-
[蝦野系統交流道間與九州縱貫自動車道宮崎線重複](https://zh.wikipedia.org/wiki/蝦野系統交流道 "wikilink")）。

此外，門司交流道 -
[福岡交流道間被指定為](https://zh.wikipedia.org/wiki/福岡交流道_\(福岡縣\) "wikilink")
 [亞洲公路1號線](../Page/亞洲公路1號線.md "wikilink")。高速公路路線編號被編為 。

## 交流道

  - 設施欄背景色<span style="color:#CCC">■</span>區間為未提供使用之設施，未通車區間名稱全部是暫稱。
  - [智能IC](https://zh.wikipedia.org/wiki/智能IC "wikilink")（SIC）背景色為<span style="color:#eda5ff">■</span>。
  - 路線名無特別註記者為[市町道](https://zh.wikipedia.org/wiki/市町村道 "wikilink")。
  - BS（巴士站）：○為使用中、◆為停用中，未註記者即未設置。
  - 停车区：停車場，SA：服務區，IC：交流道，JCT：系統交流道，TB：設於主線之[收費站](https://zh.wikipedia.org/wiki/公路收費站 "wikilink")(料金所)。

## 歷史

  - 1971年6月30日 : 植木IC-熊本IC間通車。
  - 1972年10月21日 : 南關IC-植木IC間通車。
  - 1973年11月16日 : 鳥栖IC-鳥栖JCT-南關IC間通車，與長崎自動車道（當時為九州道鳥栖IC-鳥栖JCT間）連接。
  - 1973年12月13日 : 加治木IC-薩摩吉田IC間通車。
  - 1975年3月13日 : 古賀IC-鳥栖JCT間通車。
  - 1976年3月4日 : 蝦野IC-蝦野JCT間通車（當時為宮崎道蝦野IC-蝦野JCT間），同日與宮崎自動車道連接。
  - 1976年11月26日 : 熊本IC-御船IC間通車。
  - 1976年11月29日 : 溝邊鹿兒島空港IC-加治木IC間通車。
  - 1977年7月21日 : 若宮IC-古賀IC間通車。
  - 1977年11月15日 : 薩摩吉田IC-鹿兒島北IC間通車。
  - 1978年12月15日 : 御船IC-松橋IC間通車。
  - 1979年3月8日 :
    八幡IC-若宮IC間通車。與[山口縣山口IC起點之](https://zh.wikipedia.org/wiki/山口縣 "wikilink")[一般收費道路](../Page/收費道路.md "wikilink")[北九州道路、北九州直方道路經由熊本縣松橋IC與](https://zh.wikipedia.org/wiki/北九州高速4号線 "wikilink")[高速道路連結](https://zh.wikipedia.org/wiki/高速道路 "wikilink")。
  - 1980年3月12日 : 松橋IC-八代IC間通車。
  - 1980年3月22日 : 栗野IC-溝邊鹿兒島空港IC間通車。
  - 1981年10月1日 : 蝦野JCT-栗野IC間通車。
  - 1984年3月27日 : 門司IC-小倉東IC間通車，與關門橋直通。
  - 1987年2月5日 : 於鳥栖JCT與大分自動車道連接。
  - 1988年3月29日 : 鹿兒島北IC-鹿兒島IC間通車。
  - 1988年3月31日 : 小倉東IC-八幡IC間通車。
  - 1988年10月19日 : 於鹿兒島IC與南九州自動車道連接。
  - 1989年12月7日 : 八代IC-人吉臨時出入口間通車。
  - 1992年3月25日 : 於加治木IC與隼人道路連接。
  - 1993年3月31日 : 新門司IC通車。
  - 1995年7月27日 :
    人吉臨時出入口-蝦野IC間通車，**全線通車**。由此自[青森](../Page/青森市.md "wikilink")-鹿兒島間（約2170km）的高速道路全部連結完成。人吉主線收費站停用，人吉臨時出入口封閉。
  - 1997年7月14日 : 除肥後隧道外，八代IC-人吉IC間拓寬為4車道。
  - 1997年11月28日 : 鳥栖JCT-久留米IC間拓寬為6車道。
  - 1998年3月23日 : 廣川IC通車。
  - 1998年3月31日 : 筑紫野IC通車。
  - 1998年4月20日 : 八代JCT開設，與南九州西回自動車道連接。
  - 1999年3月24日 : 益城熊本空港IC通車。
  - 1999年4月14日 : 肥後隧道前後區間拓寬為4車道。
  - 2001年3月24日 : 鳥栖JCT九州道下行方向至長崎道長崎方面直通連絡橋「」通車。
  - 2001年4月26日 : 蝦野停车区-蝦野IC間拓寬為4車道。
  - 2001年12月19日 : 加治木JCT通車，與加治木IC間連接部分廢除，並與東九州自動車道連接。
  - 2004年12月12日 : 人吉IC-蝦野IC間全線4車道通車，**全線4車道以上通車**。
  - 2004年12月18日 : 須惠停车区智能IC社會實驗開始。
  - 2006年2月26日 : 北九州JCT通車，與東九州自動車道連接。
  - 2006年10月1日 : 須惠SIC開始啟用。
  - 2009年3月29日 : 三山柳川IC通車。
  - 2011年2月19日 : 鞍手IC通車。

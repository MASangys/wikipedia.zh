**九氢合铼(VII)酸钠**是一种过渡金属氢负离子配合物，化学式为Na<sub>2</sub>\[ReH<sub>9</sub>\]。\[1\]\[2\]

## 制备

金属钠还原[高铼酸钠](../Page/高铼酸钠.md "wikilink")或[高铼酸铵](../Page/高铼酸铵.md "wikilink")的[乙醇](../Page/乙醇.md "wikilink")溶液可以得到。\[3\]

\[\mathsf{NH_4ReO_4 + 18Na + 13C_2H_5OH \ \xrightarrow{}\ Na_2[ReH_9]\downarrow + 13C_2H_5ONa + 3NaOH + NH_3\cdot H_2O }\]

## 物理性质

Na<sub>2</sub>\[ReH<sub>9</sub>\]可溶于[水](../Page/水.md "wikilink")和[甲醇](../Page/甲醇.md "wikilink")，微溶于乙醇。\[4\]

## 化学性质

  - Na<sub>2</sub>\[ReH<sub>9</sub>\]受热分解：

\[\mathsf{2Na_2[ReH_9] \ \xrightarrow{245^oC}\ 4NaH + 2Re + 7H_2 }\]

  - 和热水反应：

\[\mathsf{Na_2[ReH_9] + 5H_2O \ \xrightarrow{100^oC}\ NaReO_4 + NaOH + 9H_2 }\]

  - Na<sub>2</sub>\[ReH<sub>9</sub>\]在碱性溶液中稳定，和稀酸反应放出[氢气](https://zh.wikipedia.org/wiki/氢气 "wikilink")：\[5\]

\[\mathsf{2Na_2[ReH_9] + 4HCl \ \xrightarrow{}\ 4NaCl + 2Re\downarrow + 11H_2\uparrow }\]

  - 加热情况下可以被热而浓的氧化剂氧化为[高铼酸钠](../Page/高铼酸钠.md "wikilink")：

\[\mathsf{4Na_2[ReH_9] + 22HNO_3 \ \xrightarrow{100^oC}\ 4NaReO_4 + 9NH_4NO_3 + 4NaNO_3 + 11H_2O }\]

\[\mathsf{Na_2[ReH_9] + 9H_2O_2 \ \xrightarrow{100^oC}\ NaReO_4 + NaOH + 13H_2O }\]

  - 可以被空气氧化：

\[\mathsf{4Na_2[ReH_9] + 11O_2 \ \xrightarrow{\tau}\ 4Re + 8NaOH + 14H_2O }\]

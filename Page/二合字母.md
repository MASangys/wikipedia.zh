**二合字母**（digraph、或稱**二重音字**、**二重字**），在[字母系統](../Page/字母系統.md "wikilink")上是由2個字母組合成的語音，而組成的語音與個別字母的發音不同。且能由二合字母表記出新的[語音](../Page/語音.md "wikilink")（[音位](../Page/音位.md "wikilink")）。在一些發音變化的場合（無聲化等），比如將2個相同或不同的字母排列在一起時整體單詞的發音會有所變化，相對的僅為一個字母時發音上並不會改變，而且語音較強時發音上並不會拉長等。在[歐洲語言上會有這些現象](https://zh.wikipedia.org/wiki/歐洲 "wikilink")，尤其[西歐的語言中經常會見到](https://zh.wikipedia.org/wiki/西歐 "wikilink")，但在[斯拉夫語族裡則相當罕見](https://zh.wikipedia.org/wiki/斯拉夫語族 "wikilink")。

## 愛爾蘭語

隨著二十世紀的演變，[羅馬體取代](https://zh.wikipedia.org/wiki/羅馬體 "wikilink")成為主流。使用羅馬體時，附加在一些子音標示[子音弱化](https://zh.wikipedia.org/wiki/子音弱化 "wikilink") (lenition) 的[上點](../Page/點_\(附加符號\).md "wikilink") (◌̇ )不用，改為後增 h，成為二合字母，對照如下：

| 新正字法 | 舊正字法 | 寬音（）音位                                                 | 窄音（[顎音化](../Page/顎音化.md "wikilink")）音位 | 範例           |
| ---- | ---- | ------------------------------------------------------ | -------------------------------------- | ------------ |
| bh   | ḃ    | w                                                      | vʲ                                     | **bh**ean    |
| mh   | ṁ    | **mh**ór                                               |                                        |              |
| ch   | ċ    | x                                                      | ç                                      | **ch**óir    |
| dh   | ḋ    | ɣ                                                      | j                                      | **dh**uit    |
| gh   | ġ    | **gh**ealach                                           |                                        |              |
| fh   | ḟ    | ∅（[零輔音](https://zh.wikipedia.org/wiki/零輔音 "wikilink")） | **Fh**rainc                            |              |
| ph   | ṗ    | fˠ                                                     | fʲ                                     | **ph**ríosún |
| sh   | ṡ    | h                                                      | h 或 ç                                  | **sh**iopa   |
| th   | ṫ    | h                                                      | h 或 ç                                  | **th**all    |

  - 註：以上音位僅為通則，仍有特例。

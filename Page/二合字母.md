[Lldigraph.png](https://zh.wikipedia.org/wiki/File:Lldigraph.png "fig:Lldigraph.png")裡，二合字母[Ll一度併為一個](../Page/Ll.md "wikilink")[合字](../Page/合字.md "wikilink")。\]\]

**二合字母**（digraph、或稱**二重音字**、**二重字**），在[字母系統上是由](../Page/字母系統.md "wikilink")2個字母組合成的語音，而組成的語音與個別字母的發音不同。且能由二合字母表記出新的[語音](../Page/語音.md "wikilink")（[音位](../Page/音位.md "wikilink")）。在一些發音變化的場合（無聲化等），比如將2個相同或不同的字母排列在一起時整體單詞的發音會有所變化，相對的僅為一個字母時發音上並不會改變，而且語音較強時發音上並不會拉長等。在[歐洲語言上會有這些現象](../Page/歐洲.md "wikilink")，尤其[西歐的語言中經常會見到](../Page/西歐.md "wikilink")，但在[斯拉夫語系裡則相當罕見](../Page/斯拉夫語族.md "wikilink")。

## 愛爾蘭語

隨著二十世紀的演變，[羅馬體取代](../Page/羅馬體.md "wikilink")成為主流。使用羅馬體時，附加在一些子音標示[子音弱化](../Page/子音弱化.md "wikilink")
(lenition) 的[上點](../Page/點_\(附加符號\).md "wikilink") (◌̇ )不用，改為後增
h，成為二合字母，對照如下：

| 新正字法 | 舊正字法 | 寬音（）音位                              | 窄音（[顎音化](../Page/顎音化.md "wikilink")）音位 | 範例           |
| ---- | ---- | ----------------------------------- | -------------------------------------- | ------------ |
| bh   | ḃ    | w                                   | vʲ                                     | **bh**ean    |
| mh   | ṁ    | **mh**ór                            |                                        |              |
| ch   | ċ    | x                                   | ç                                      | **ch**óir    |
| dh   | ḋ    | ɣ                                   | j                                      | **dh**uit    |
| gh   | ġ    | **gh**ealach                        |                                        |              |
| fh   | ḟ    | ∅（[零輔音](../Page/零輔音.md "wikilink")） | **Fh**rainc                            |              |
| ph   | ṗ    | fˠ                                  | fʲ                                     | **ph**ríosún |
| sh   | ṡ    | h                                   | h 或 ç                                  | **sh**iopa   |
| th   | ṫ    | h                                   | h 或 ç                                  | **th**all    |

  - 註：以上音位僅為通則，仍有特例。

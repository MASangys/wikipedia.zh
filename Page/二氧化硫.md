**二氧化硫**，（）[化学式](../Page/化学式.md "wikilink")是SO<sub>2</sub>。是最常见的硫[氧化物](../Page/氧化物.md "wikilink")。无色气体，有强烈刺激性气味。[大气主要污染物之一](https://zh.wikipedia.org/wiki/大气 "wikilink")。[火山爆发时会喷出该气体](https://zh.wikipedia.org/wiki/火山爆发 "wikilink")，在许多工业过程中也会产生二氧化硫。由于[煤](../Page/煤.md "wikilink")和[石油](../Page/石油.md "wikilink")通常都含有硫化合物，因此[燃烧](../Page/燃烧.md "wikilink")时会生成二氧化硫。當二氧化硫溶於水中，會形成[亞硫酸](https://zh.wikipedia.org/wiki/亞硫酸 "wikilink")（[酸雨](../Page/酸雨.md "wikilink")的主要成分）。若把SO<sub>2</sub>进一步氧化，通常在催化剂如[二氧化氮](../Page/二氧化氮.md "wikilink")的存在下，便会生成[硫酸](../Page/硫酸.md "wikilink")。\[1\]这就是对使用这些燃料作为能源的环境效果的担心的原因之一。

## 结构

SO<sub>2</sub>是一个V型的分子，其[对称点群为C](https://zh.wikipedia.org/wiki/对称点群 "wikilink")<sub>2v</sub>。硫原子的[氧化态为](https://zh.wikipedia.org/wiki/氧化态 "wikilink")+4，[形式电荷](../Page/形式电荷.md "wikilink")为0，被5个[电子对包围着](https://zh.wikipedia.org/wiki/电子对 "wikilink")，因此可以描述为[超价分子](../Page/超价分子.md "wikilink")。从[分子轨道理论](../Page/分子轨道理论.md "wikilink")的观点来看，可以认为这些价电子大部分都参与形成S-O键。

SO<sub>2</sub>中的S-O键长（143.1 pm）要比[一氧化硫](../Page/一氧化硫.md "wikilink")中的S-O键长（148.1 pm）短，而O<sub>3</sub>中的O-O键长（127.8 pm）则比[氧气](../Page/氧气.md "wikilink")O<sub>2</sub>中的O-O键长（120.7 pm）长。SO<sub>2</sub>的平均键能（548 kJ mol<sup>−1</sup>）要大于SO的平均键能（524 kJ mol<sup>−1</sup>），而O<sub>3</sub>的平均键能（297 kJ mol<sup>−1</sup>）则小于O<sub>2</sub>的平均键能（490 kJ mol<sup>−1</sup>）。这些证据使化学家得出结论：二氧化硫中的S-O键的[键级至少为](https://zh.wikipedia.org/wiki/键级 "wikilink")2，与臭氧中的O-O键不同，臭氧中的O-O键的键级为1.5\[2\]

## 化学性质

### 酸性氧化物

SO<sub>2</sub>是[酸性氧化物](../Page/酸性氧化物.md "wikilink")，具有酸性氧化物的通性。可以與水作用得到二氧化硫水溶液，即“[亞硫酸](https://zh.wikipedia.org/wiki/亞硫酸 "wikilink")”（中强酸），但真正的亚硫酸分子从未在溶液中观测到。\[3\]

\[\rm SO_2 + H_2O \longleftrightarrow H_2SO_3\]

与[碱](../Page/碱.md "wikilink")反应形成[亚硫酸盐](../Page/亚硫酸盐.md "wikilink")和[亚硫酸氢盐](https://zh.wikipedia.org/wiki/亚硫酸氢盐 "wikilink")。以与[氢氧化钠](../Page/氢氧化钠.md "wikilink")的反应为例，产物是[亚硫酸钠](../Page/亚硫酸钠.md "wikilink")还是[亚硫酸氢钠](../Page/亚硫酸氢钠.md "wikilink")，取决于二者的用量关系。

\[\rm SO_2 + 2NaOH \longrightarrow Na_2SO_3 + H_2O\] 或

\[\rm SO_2 + NaOH \longrightarrow NaHSO_3\]

这也是二氧化硫能使[澄清石灰水变浑浊的原因](https://zh.wikipedia.org/wiki/澄清石灰水 "wikilink")：

  -
    \(\rm SO_2 + Ca(OH)_2 \longrightarrow CaSO_3 \downarrow + H_2O\)

与[碱性氧化物反应生成盐](https://zh.wikipedia.org/wiki/碱性氧化物 "wikilink")。

\[\rm SO_2 + CaO \longrightarrow CaSO_3\]

### 氧化还原反应

SO<sub>2</sub>中的硫元素的化合价为+4价，为中间价态，既可升高，也可下降。所以SO<sub>2</sub>既有[氧化性](https://zh.wikipedia.org/wiki/氧化性 "wikilink")，又有[还原性](https://zh.wikipedia.org/wiki/还原性 "wikilink")，但以还原性为主。

SO<sub>2</sub>的还原性较强，可被多种氧化剂（如 [O<sub>2</sub>](../Page/氧气.md "wikilink")、[Cl<sub>2</sub>](https://zh.wikipedia.org/wiki/氯气 "wikilink")、[Br<sub>2</sub>](../Page/溴.md "wikilink")、[HNO<sub>3</sub>](../Page/硝酸.md "wikilink")、[KMnO<sub>4</sub>等](../Page/高锰酸钾.md "wikilink")）氧化。

\[\rm SO_2 + Cl_2 \longrightarrow SO_2Cl_2\]

\[\rm 2SO_2 + O_2 \longleftrightarrow 2SO_3\]（该反应为[可逆反应](https://zh.wikipedia.org/wiki/可逆反应 "wikilink")，条件为加热和催化剂：[V<sub>2</sub>O<sub>5</sub>](../Page/五氧化二钒.md "wikilink") / [Pt](../Page/铂.md "wikilink") / [Cr<sub>2</sub>O<sub>3</sub>](../Page/三氧化二铬.md "wikilink")）

SO<sub>2</sub>也有一定的氧化性，如：

\[\rm SO_2 + 2H_2S \longrightarrow 3S + 2H_2O\] 工业上可以用此反应制造高纯度[硫磺](https://zh.wikipedia.org/wiki/硫磺 "wikilink")。

## 用途

### 防腐剂

由于二氧化硫的[抗菌性质](https://zh.wikipedia.org/wiki/抗菌 "wikilink")，它有时用作[乾果](../Page/果乾.md "wikilink")、醃漬蔬菜、與經加工處理的肉製品（如香腸及漢堡肉）等不同種類的食物中。用来保持水果的外表，或防止食物[腐烂](https://zh.wikipedia.org/wiki/腐烂 "wikilink")。二氧化硫的存在，可以使水果有一种特殊的化学[味道](https://zh.wikipedia.org/wiki/味道 "wikilink")、及保持新鮮的外觀。

### 酿酒

二氧化硫是酿酒时非常有用的化合物，它的[E编码](../Page/E编码.md "wikilink")为E220。\[4\]它甚至在所谓的“无硫的”酒中也存在，浓度可达每升10毫克。\[5\]它作为抗生素和抗氧化剂，防止酒遭到细菌的损坏和氧化。它也帮助把挥发性酸度保持在想要的程度。\[6\]酒的标签上之所以有“含有亚硫酸盐”等字句，就是因为二氧化硫。根据美国和欧盟的法律，如果酒的SO<sub>2</sub>浓度低于10ppm，则不需要标示“含有亚硫酸盐”。酒中允许的SO<sub>2</sub>浓度的上限在美国为350ppm，而在欧盟，红酒为160ppm，白酒为210ppm。如果SO<sub>2</sub>的浓度很低，那么便很难探测到，但当浓度大于50ppm时，用鼻子就能闻出SO<sub>2</sub>的气味，用舌头也能品尝出来。

SO<sub>2</sub>还是酿酒厂卫生的很重要的要素。酿酒厂和设备必须保持十分清洁，且因为漂白剂不能用于酿酒厂中，SO<sub>2</sub>、水和柠檬酸的混合物通常用来清洁水管、水槽和其它设备，以保持清洁和没有细菌。

### 还原性漂白剂

二氧化硫还是一个很好的[还原剂](https://zh.wikipedia.org/wiki/还原剂 "wikilink")。在水的存在下，二氧化硫可以使物质褪色。特别地，它是[纸张和](https://zh.wikipedia.org/wiki/纸张 "wikilink")[衣物的有用的漂白剂](https://zh.wikipedia.org/wiki/衣物 "wikilink")。这个漂白作用通常不能持续很久。空气中的[氧气](../Page/氧气.md "wikilink")把被还原的染料重新氧化，使颜色恢复。

可以下列化學方程式表示： 　H<sub>2</sub>SO<sub>3</sub> + 染料 → H<sub>2</sub>SO<sub>4</sub> + (染料 - O )

因為空氣提供氧氣給予染料，染料被馬上氧化，顯示原來的顏色，這就是漂白作用通常不能持续很久的原因。

可以下列化學方程式表示： 2 (染料 - O ) + O<sub>2</sub> → 2染料

中学实验室中用碱性品红溶液检测二氧化硫的存在。二氧化硫可以使品红试液褪色，从而说明二氧化硫使有机物漂白的性质；而褪色后的溶液经过加热，又恢复为红色，从而说明了二氧化硫漂白的原理是与有机物生成了“加合物”，而此类加合物不稳定，加热时便分解，又放出二氧化硫。一个相关的化学鉴定方法称为[希夫法](https://zh.wikipedia.org/wiki/希夫法 "wikilink")(Schiff法) ，是用[亚硫酸氢钠](../Page/亚硫酸氢钠.md "wikilink")与[品红或](../Page/品红_\(化合物\).md "wikilink")[副品红发生加成](https://zh.wikipedia.org/wiki/副品红 "wikilink")，再用二氧化硫脱色。如果得到的溶液（[希夫试剂](https://zh.wikipedia.org/wiki/希夫试剂 "wikilink")）与待检试液作用生成粉红色或紫色，则可以证明待检试液中[醛](../Page/醛.md "wikilink")类的存在。目前该反应的机理一般认为是下图所示的机理：\[7\]\[8\]\[9\]

### 硫酸的前体

二氧化硫还用来制备硫酸，首先转化成[三氧化硫](../Page/三氧化硫.md "wikilink")，然后再转化成[发烟硫酸](https://zh.wikipedia.org/wiki/发烟硫酸 "wikilink")，最后转化成[硫酸](../Page/硫酸.md "wikilink")。这个过程中的二氧化硫是含硫矿物与氧气反应产生的。把二氧化硫转化成硫酸的过程，称为[接触法](../Page/接触法.md "wikilink")。

### 制冷剂

由于二氧化硫容易液化，且[汽化热](../Page/汽化热.md "wikilink")很大，因此适合作为[制冷剂](../Page/制冷剂.md "wikilink")。在[氟利昂的发展之前](https://zh.wikipedia.org/wiki/卤代烷 "wikilink")，二氧化硫就曾经用作家用冰箱的制冷剂。

### 试剂和溶剂

液态二氧化硫是万用的惰性溶剂，广泛用于溶解强氧化性盐。它会发生[自偶电离](../Page/自偶电离.md "wikilink")生成SO<sup>2+</sup>和SO<sub>3</sub><sup>2−</sup>。

\[\rm 2SO_2 \rightarrow SO^{2+} + SO_3^{2-}\]

它有时也用作[有机合成](../Page/有机合成.md "wikilink")中[磺酰基的来源](https://zh.wikipedia.org/wiki/磺酰 "wikilink")，把芳基[重氮盐](../Page/重氮盐.md "wikilink")用二氧化硫处理，便可获得对应的芳基磺酰氯。\[10\]

### 脱氯

在城市的污水处理中，二氧化硫用来处理排放前的氯化污水。二氧化硫与氯气反应，氯气被还原，生成Cl<sup>−</sup>。\[11\]

主要由于美国环境保护机构的[酸雨计划](https://zh.wikipedia.org/wiki/酸雨计划 "wikilink")，美国在1983年和2002年期间的二氧化硫排放量减少了33%。这是由于[烟气脱硫](https://zh.wikipedia.org/wiki/烟气脱硫 "wikilink")，一种可以让SO<sub>2</sub>不从[发电厂排放出去的技术](https://zh.wikipedia.org/wiki/发电厂 "wikilink")。特别地，[氧化钙与二氧化硫反应](https://zh.wikipedia.org/wiki/氧化钙 "wikilink")，生成[亚硫酸钙](../Page/亚硫酸钙.md "wikilink")：

\[\rm CaO + SO_2 \longleftrightarrow CaSO_3\] 然后CaSO<sub>3</sub>再被空气氧化成CaSO<sub>4</sub>（[石膏](../Page/石膏.md "wikilink")）。大部分在欧洲出售的石膏都是来自烟气脱硫。

到2006年为止，中国是世界上二氧化硫排放量最大的国家，2005年的排放量估计为25.49百万吨。自从2000年以来，排放量增加了27%，差不多与美国在1980年的排放量相等\[13\]。

2003年，一座伊拉克的硫厂发生了灾难，大量二氧化硫被排放到大气中。

|                  |                   |
| ---------------- | ----------------- |
| 22 g/100ml（0℃）   | 15 g/100ml（10℃）   |
| 11 g/100ml（20℃）  | 9.4 g/100 ml（25℃） |
| 8 g/100ml（30℃）   | 6.5 g/100ml（40℃）  |
| 5 g/100ml（50℃）   | 4 g/100ml（60℃）    |
| 3.5 g/100ml（70℃） | 3.4 g/100ml（80℃）  |
| 3.5 g/100ml（90℃） | 3.7 g/100ml（100℃） |

  - 以上的值是[分压为](https://zh.wikipedia.org/wiki/分压 "wikilink")101.3 kPa的SO<sub>2</sub>在水中的溶解度。根据[亨利定律](../Page/亨利定律.md "wikilink")，气体在液体中的[溶解度取决于气体的分压](https://zh.wikipedia.org/wiki/溶解度 "wikilink")。
  - 这里的溶解度是指在“纯水”中的溶解度，也就是说，水中所含的SO<sub>2</sub>与气相中的二氧化硫平衡。这个“纯水”是[酸性的](https://zh.wikipedia.org/wiki/酸性 "wikilink")。SO<sub>2</sub>在中性（或[碱性](https://zh.wikipedia.org/wiki/碱性 "wikilink")）水中的溶解度一般要更大，因为SO<sub>2</sub>将转化为[亚硫酸氢根和一些](https://zh.wikipedia.org/wiki/亚硫酸氢盐 "wikilink")[亚硫酸根离子](../Page/亚硫酸盐.md "wikilink")。

## 对健康的威胁

二氧化硫具有酸性，可与空气中的其他物质反应，生成微小的亚硫酸盐和硫酸盐颗粒。当这些颗粒被吸入时，它们将聚集于肺部，是呼吸系统症状和疾病、呼吸困难，以及过早死亡的一个原因\[14\]。如果与水混合，再与皮肤接触，便有可能发生冻伤。与眼睛接触时，会造成红肿和疼痛\[15\]。

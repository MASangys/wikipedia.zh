**二氧化鉛**或**過氧化鉛**，化學式[Pb](https://zh.wikipedia.org/wiki/鉛 "wikilink")[O](../Page/氧.md "wikilink")，常溫時為棕色結晶或粉末，几乎不溶于水\[1\]，有强氧化性。

## 制备

二氧化鉛可由[四氧化三鉛](../Page/四氧化三鉛.md "wikilink")與[硝酸](../Page/硝酸.md "wikilink")作用而得。反應式：

  -
    Pb<sub>3</sub>O<sub>4</sub> + 4HNO<sub>3</sub> → PbO<sub>2</sub> + 2Pb(NO<sub>3</sub>)<sub>2</sub> + 2H<sub>2</sub>O

## 化学性质

### 不稳定性

二氧化铅受热分解：

  -

      -
        PbO<sub>2</sub> → Pb<sub>12</sub>O<sub>19</sub> → Pb<sub>12</sub>O<sub>17</sub> → Pb<sub>3</sub>O<sub>4</sub> → PbO

### 强氧化性

由于[6s2惰性电子对效应](https://zh.wikipedia.org/wiki/6s2惰性电子对效应 "wikilink")，二氧化铅具有很强的[氧化性](https://zh.wikipedia.org/wiki/氧化性 "wikilink")，在常温即可将[盐酸](../Page/盐酸.md "wikilink")氧化成[氯气](https://zh.wikipedia.org/wiki/氯气 "wikilink")：

  -
    PbO<sub>2</sub> + 4 HCl → PbCl<sub>2</sub> + 2 H<sub>2</sub>O + Cl<sub>2</sub>\[2\]

也可以将浓[硫酸](../Page/硫酸.md "wikilink")氧化，放出[氧气](../Page/氧气.md "wikilink")：

  -
    2 PbO<sub>2</sub> + 2 H<sub>2</sub>SO<sub>4</sub> → 2 PbSO<sub>4</sub> + 2 H<sub>2</sub>O + O<sub>2</sub>\[3\]

二氧化铅与[碳](../Page/碳.md "wikilink")共热，即可剧烈反应，产生火光和金属[铅](../Page/铅.md "wikilink")：

  -
    2 PbO<sub>2</sub> + C → 2 PbO + CO<sub>2</sub>
    PbO<sub>2</sub> + C → Pb + CO<sub>2</sub>

此外，在酸性条件下，铅单质与二氧化铅可以发生[归中反应](../Page/归中反应.md "wikilink")，这也是[铅酸蓄电池](../Page/铅酸蓄电池.md "wikilink")放电时的工作原理

### 和碱反应

与强碱溶液反应，得铅酸盐：

  -
    PbO<sub>2</sub> + 2 NaOH + 2 H<sub>2</sub>O → Na<sub>2</sub>\[Pb(OH)<sub>6</sub>\]\[4\]

和熔融的强碱反应，可以得到M<sub>4</sub>\[PbO<sub>4</sub>\]。

## 用途

二氧化鉛可製造[鉛蓄電池的](https://zh.wikipedia.org/wiki/鉛蓄電池 "wikilink")[極板](https://zh.wikipedia.org/wiki/極板 "wikilink")。加熱產生[氧](../Page/氧.md "wikilink")氣。与[硫](../Page/硫.md "wikilink")、[赤磷等可燃物混合研磨则发火](https://zh.wikipedia.org/wiki/赤磷 "wikilink")，可用作[火柴](../Page/火柴.md "wikilink")的原料。

二氧化铅也常用于实验室制备其他的高价化合物\[5\]：

  -
    2 MnSO<sub>4</sub> + 5 PbO<sub>2</sub> + 6 HNO<sub>3</sub> → 2 HMnO<sub>4</sub> + 2 PbSO<sub>4</sub> + 3 Pb(NO<sub>3</sub>)<sub>2</sub> + 2 H<sub>2</sub>O

  -
    2 Cr(OH)<sub>3</sub> + 10 KOH + 3 PbO<sub>2</sub> → 2 K<sub>2</sub>CrO<sub>4</sub> + 3 K<sub>2</sub>PbO<sub>2</sub> + 8 H<sub>2</sub>O

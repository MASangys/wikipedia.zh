**二氯二氧化钼**是一种[无机化合物](../Page/无机化合物.md "wikilink")，化学式为MoO<sub>2</sub>Cl<sub>2</sub>。它是浅黄色可溶于水的潮解性固体，可用于制备其它钼化合物。它的加合物MoO<sub>2</sub>Cl<sub>2</sub>(ether)<sub>2</sub>可溶于有机溶剂。二氯二氧化钼可用作一些氧化反应的催化剂。\[1\]

## 制备

二氯二氧化钼可由[三氧化钼](../Page/三氧化钼.md "wikilink")和浓[盐酸](../Page/盐酸.md "wikilink")反应得到：\[2\]

  -
    MoO<sub>3</sub> + 2 HCl → MoO<sub>2</sub>Cl<sub>2</sub> + H<sub>2</sub>O

MoO<sub>2</sub>Cl<sub>2</sub>也能以[MoOCl<sub>4</sub>为原料制备](https://zh.wikipedia.org/wiki/四氯一氧化钼 "wikilink")：\[3\]

  -
    MoOCl<sub>4</sub> + O(Si(CH<sub>3</sub>)<sub>3</sub>)<sub>2</sub> → MoO<sub>2</sub>Cl<sub>2</sub> + 2 ClSi(CH<sub>3</sub>)<sub>3</sub>

二氧化钼于500\~700 °C和氯气反应\[4\]，或者[五氯化钼](../Page/五氯化钼.md "wikilink")在空气中加热也能得到产物：

  -
    MoO<sub>2</sub> + Cl<sub>2</sub> → MoO<sub>2</sub>Cl<sub>2</sub>
    2 MoCl<sub>5</sub> + 2 O<sub>2</sub> → 2 MoO<sub>2</sub>Cl<sub>2</sub> + 3 Cl<sub>2</sub>

MoO<sub>3</sub>和CCl<sub>4</sub>在713\~753 K反应，可以得到二氯二氧化钼，气态产物包括[二氧化碳](../Page/二氧化碳.md "wikilink")、[氯气和](https://zh.wikipedia.org/wiki/氯气 "wikilink")[光气](../Page/光气.md "wikilink")。\[5\]

## 化学性质

MoO<sub>2</sub>Cl<sub>2</sub>可以形成一系列的[加合物](https://zh.wikipedia.org/wiki/加合物 "wikilink")，例如MoO<sub>2</sub>Cl<sub>2</sub>(THF)<sub>2</sub>。它和大分子[苯胺](../Page/苯胺.md "wikilink")反应，生成二亚胺配合物MoCl<sub>2</sub>(NAr)<sub>2</sub>([L](../Page/乙二醇二甲醚.md "wikilink"))，这种配合物是Schrock型卡宾Mo(OR)<sub>2</sub>(NAr)(CH-t-Bu)的前驱体。\[6\]

MoO<sub>2</sub>Cl<sub>2</sub>可以和干燥的[氟化氢](../Page/氟化氢.md "wikilink")或[三氧化硫](../Page/三氧化硫.md "wikilink")[磺酰氯溶液反应](https://zh.wikipedia.org/wiki/磺酰氯 "wikilink")：

  -
    MoO<sub>2</sub>Cl<sub>2</sub> + 2 HF → MoO<sub>2</sub>F<sub>2</sub> + 2 HCl
    MoO<sub>2</sub>Cl<sub>2</sub> + 3 SO<sub>3</sub> → MoO(SO<sub>4</sub>)<sub>2</sub> + SO<sub>2</sub>Cl<sub>2</sub>

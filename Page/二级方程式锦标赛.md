**国际汽车联合会二级方程式锦标赛**\[1\]是一项2017年推出的[开轮式赛车赛事](https://zh.wikipedia.org/wiki/开轮式赛车 "wikilink")，由[GP2系列赛重塑而成](https://zh.wikipedia.org/wiki/GP2系列赛 "wikilink")。

为了让车队能够负担得起参赛并使其成为一级方程式赛车的完美训练场，二级方程式已经强制要求所有车队使用相同的底盘、发动机和轮胎供应商，以便体现车手的真实能力。二级方程式主要在欧洲比赛，但也会在其他国际赛道举办比赛，例如[巴林](../Page/巴林.md "wikilink")的[巴林国际赛道](../Page/巴林国际赛道.md "wikilink")和[阿拉伯联合酋长国](../Page/阿拉伯联合酋长国.md "wikilink")的[亚斯码头赛道](../Page/亚斯码头赛道.md "wikilink")。

## 比赛周末

周五，车手们有45分钟\[2\]的自由练习赛以及30分钟的排位赛。排位赛决定周六180千米的比赛的发车顺位。

在周六的比赛（常规赛）中，每名车手必须至少完成一次进站，并且必须至少使用一套每种规格的乾地胎。

周日的比赛（冲刺赛）长度为120千米。发车顺位由周六常规赛的结果决定，其中排名前8位的车手倒序发车，也就是说周六排名第8的车手将在周日杆位发车，而周六的胜者将会从第8位发车。

这些规则的例外是在摩纳哥和布达佩斯，那里的常规赛分别是140千米和160千米，摩纳哥的冲刺赛是100千米。\[3\]

### 积分系统

常规赛的积分系统与一级方程式的类似：

| 常规赛积分系统 |
| ------- |
| 第1名     |
| 25      |

冲刺赛中前八名完赛获得的积分如下：

| 冲刺赛积分系统 |
| ------- |
| 第1名     |
| 15      |

常规赛的杆位将获得4分，每场比赛的最快圈速也会获得2分。因此，一名车手在单轮中最多可以获得48分。

  - 做出最快圈速的车手需要完成比赛90%的圈数，并且必须最终排名前十才有资格获得最快圈速积分。

当出现平局的情况时，国际汽车联合会二级方程式锦标赛将根据第一名完赛的次数来决定冠军。如果仍然平局，则将根据第二名完赛的次数决定冠军，以此类推。在赛季中或赛季结束时，如果其他名次也出现平局则依然按照此系统决定排名。

## 历史

### 2017年

2017赛季共有11轮，其中10轮是[2017年世界一级方程式锦标赛的垫场赛](https://zh.wikipedia.org/wiki/2017年世界一级方程式锦标赛 "wikilink")，还有在赫雷斯举行的额外的一轮。\[4\]它于2017年4月15日在巴林开赛，并于11月26日在阿布扎比结束。

### 2018年

2018赛季共有12轮，全部作为[2018年世界一级方程式锦标赛](../Page/2018年世界一级方程式锦标赛.md "wikilink")的垫场赛举行。它于2018年4月7日在巴林开赛，并于11月25日在阿布扎比结束。

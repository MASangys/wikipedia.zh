在[概率论和](../Page/概率论.md "wikilink")[统计学中](../Page/统计学.md "wikilink")，**二项分布**（）是*n*个[独立的是](../Page/统计独立性.md "wikilink")/非试验中成功的次数的[离散概率分布](../Page/离散概率分布.md "wikilink")，其中每次试验的成功[概率为](../Page/概率.md "wikilink")*p*。这样的单次成功/失败试验又称为[伯努利试验](../Page/伯努利试验.md "wikilink")。实际上，当*n*
=
1时，二项分布就是[伯努利分布](../Page/伯努利分布.md "wikilink")。二项分布是[显著性差异的](../Page/显著性差异.md "wikilink")[二项试验的基础](../Page/二项试验.md "wikilink")。

## 详述

### 概率质量函数

一般地，如果随机变量\(\mathit{X}\)服从参数为\(\mathit{n}\)和\(\mathit{p}\)的二项分布，我们记\(X \sim b(n,p)\)或\(X \sim B(n,p)\)。n次试验中正好得到*k*次成功的概率由[概率质量函数给出](../Page/概率质量函数.md "wikilink")：

\[f(k;n,p) = \Pr(X = k) = {n\choose k}p^k(1-p)^{n-k}\]

对于*k* = 0, 1, 2, ..., *n*，其中\({n\choose k}=\frac{n!}{k!(n-k)!}\)

是[二项式系数](../Page/二项式系数.md "wikilink")（这就是二项分布的名称的由来），又记为*C*(*n*,
*k*)，<sub>*n*</sub>*C*<sub>*k*</sub>，或<sup>*n*</sup>*C*<sub>*k*</sub>。该公式可以用以下方法理解：我们希望有*k*次成功(*p*<sup>*k*</sup>)和*n*
− *k*次失败(1 − *p*)<sup>*n* −
*k*</sup>。然而，*k*次成功可以在*n*次试验的任何地方出现，而把*k*次成功分布在*n*次试验中共有C(*n*,
*k*)个不同的方法。

在制造二项分布概率的参考表格时，通常表格中只填上*n*/2个值。这是因为*k* \> *n*/2时的概率可以从它的补集计算出：

\[f(k;n,p)=f(n-k;n,1-p) \,\]

因此，我们要看另外一个*k*和另外一个*p*（二项分布一般不是对称的）。然而，它的表现不是任意的。总存在一个整数*M*，满足:

\[(n+1)p-1 < M \leq (n+1)p \,\]

作为*k*的函数，表达式*ƒ*(*k ;* *n*, *p*)当*k* \< *M*时单调递增，*k* \> *M*时单调递减，只有当(*n*
+ 1)*p*是整数时例外。在这时，有两个值使*ƒ*达到最大：(*n* + 1)*p*和(*n* + 1)*p* −
1。*M*是伯努利试验的最可能的结果，称为[众数](../Page/众数.md "wikilink")。注意它发生的概率可以很小。

### 累积分布函数

[累积分布函数可以表示为](../Page/累积分布函数.md "wikilink")：

\[F(x;n,p) = \Pr(X \le x) = \sum_{i=0}^{\lfloor x \rfloor} {n\choose i}p^i(1-p)^{n-i}\]

其中\(\scriptstyle \lfloor x\rfloor\,\)是小于或等于*x*的[最大整数](../Page/最大整数.md "wikilink")。

它也可以用[正则化不完全贝塔函数来表示](../Page/正则化不完全贝塔函数.md "wikilink")：

\[\begin{align}
F(k;n,p) & = \Pr(X \le k) = I_{1-p}(n-k, k+1) \\
& = (n-k) {n \choose k} \int_0^{1-p} t^{n-k-1} (1-t)^k \, dt
\end{align}\]

## 期望和方差

如果*X* \~ *B*(*n*,
*p*)（也就是说，*X*是服从二项分布的随机变量），那么*X*的[期望值为](../Page/期望值.md "wikilink")

  -
    <math>

`   \operatorname{E}[X] = np`
` `</math>

[方差为](../Page/方差.md "wikilink")

  -
    <math>

`   \operatorname{Var}[X] = np(1 - p).`
` `</math>

这个事实很容易证明。首先假设有一个伯努利试验。试验有两个可能的结果：1和0，前者发生的概率为*p*，后者的概率为1−*p*。该试验的期望值等于。该试验的方差也可以类似地计算：.

一般的二项分布是*n*次独立的伯努利试验的和。它的期望值和方差分别等于每次单独试验的期望值和方差的和：

  -
    <math>

`   \mu_n = \sum_{k=1}^n \mu = np, \qquad`
`   \sigma^2_n = \sum_{k=1}^n \sigma^2 = np(1 - p).`
` `</math>

## 众数和中位数

通常二项分布*B*(*n*, *p*)的[众数等于](../Page/众数.md "wikilink")⌊(*n* + 1)*p*⌋，其中
\(\lfloor\cdot\rfloor\) 是[取整函数](../Page/取整函数.md "wikilink")。然而，当(*n* +
1)*p*是整数且*p*不等于0或1时，分布有两个众数：(*n* + 1)*p*和(*n* + 1)*p* −
1。当*p*等于0或1时，众数相应地等于0或 *n*。这些情况可以综述如下：

  -
    <math>

`   \text{mode} = `
`     \begin{cases}`
`       \lfloor (n+1)\,p\rfloor & \text{若 }(n+1)p\text{ 是 0 或 非 整 数 }, \\`
`       (n+1)\,p\ \text{ 和 }\ (n+1)\,p - 1 &\text{若 }(n+1)p\in\{1,\dots,n\}, \\`
`       n & \text{若 }(n+1)p = n + 1.`
`     \end{cases}`
` `</math>

一般地，没有一个单一的公式可以求出二项分布的[中位数](../Page/中位数.md "wikilink")，甚至中位数可能是不唯一的。然而有几个特殊的结果：

  - 如果*np*是整数，那么平均数、中位数和众数相等，都等于*np*。\[1\]\[2\]
  - 任何中位数*m*都位于区间⌊*np*⌋ ≤ *m* ≤ ⌈*np*⌉内。\[3\]
  - 中位数*m*不能离平均数太远：}。\[4\]
  - 如果，或，或|*m* − *np*| ≤ min{*p*, 1 − *p*}（除了*p* =
    ½、*n*是奇数的情况以外），那么中位数是唯一的，且等于*m*
    = [round](../Page/数值修约规则.md "wikilink")(*np*)。\[5\]\[6\]
  - 如果*p* = 1/2，且*n*是奇数，那么区间½(*n* − 1) ≤ *m* ≤ ½(*n* +
    1)中的任何数*m*都是二项分布的中位数。如果*p* =
    1/2且*n*是偶数，那么*m* = *n*/2是唯一的中位数。

## 两个二项分布的协方差

如果有两个服从二项分布的随机变量*X*和*Y*，我们可以求它们的协方差。利用[协方差的定义](../Page/协方差.md "wikilink")，当*n*
= 1时我们有

  -
    \(\operatorname{Cov}(X, Y) = \operatorname{E}(XY) - \mu_X \mu_Y.\)

第一项仅当*X*和*Y*都等于1时非零，而*μ*<sub>*X*</sub>和*μ*<sub>*Y*</sub>分别为*X* = 1和*Y* =
1的概率。定义*p*<sub>*B*</sub>为*X*和*Y*都等于1的概率，便得到

  -
    \(\operatorname{Cov}(X, Y) = p_B - p_X p_Y, \,\)

对于*n*次独立的试验，我们便有

  -
    \(\operatorname{Cov}(X, Y)_n = n ( p_B - p_X p_Y ). \,\)

如果*X*和*Y*是相同的变量，便化为上面的方差公式。

## 与其他分布的关系

### 二项分布的和

如果*X* \~ B(*n*, *p*)和*Y* \~ B(*m*, *p*)，且*X*和*Y*相互独立，那么*X* +
*Y*也服从二项分布；它的分布为

\[X+Y \sim B(n+m, p).\,\]

### 伯努利分布

[伯努利分布是二项分布在](../Page/伯努利分布.md "wikilink")*n* = 1时的特殊情况。*X* \~ B(1,
*p*)与*X* \~
Bern(*p*)的意思是相同的。相反，任何二项分布B(*n*,*p*)都是*n*次独立[伯努利试验的和](../Page/伯努利试验.md "wikilink")，每次试验成功的概率为*p*。

### 泊松二项分布

二项分布是[泊松二项分布的一个特殊情况](../Page/泊松二项分布.md "wikilink")。[泊松二项分布是](../Page/泊松二项分布.md "wikilink")*n*次独立、不相同的[伯努利试验](../Page/伯努利试验.md "wikilink")(*p<sub>i</sub>*)的和。如果*X*服从泊松二项分布，且*p<sub>1</sub>*
= … = *p<sub>n</sub>* =*p*，那么*X* \~ B(*n*, *p*)。

### 正态近似

[Binomial_Distribution.svg](https://zh.wikipedia.org/wiki/File:Binomial_Distribution.svg "fig:Binomial_Distribution.svg")以及[正态近似](../Page/正态近似.md "wikilink")\]\]

如果*n*足够大，那么分布的偏度就比较小。在这种情况下，如果使用适当的[连续性校正](../Page/连续性校正.md "wikilink")，那么B(*n*,
*p*)的一个很好的近似是[正态分布](../Page/正态分布.md "wikilink")：

\[\mathcal{N}(np,\, np(1-p))\]

*n*越大（至少20），近似越好，当*p*不接近0或1时更好。\[7\]不同的[经验法则可以用来决定](../Page/经验法则.md "wikilink")*n*是否足够大，以及*p*是否距离0或1足够远：

  - 一个规则是*x=np*和*n*(1 − *p*)都必须大于 5。

### 泊松近似

当试验的次数趋于无穷大，而乘积*np*固定时，二项分布收敛于[泊松分布](../Page/泊松分布.md "wikilink")。因此参数为*λ*
= *np*的泊松分布可以作为二项分布B(*n*, *p*)的近似，如果*n*足够大，而*p*足够小。\[8\]

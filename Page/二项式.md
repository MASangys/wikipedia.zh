在[初等代数中](https://zh.wikipedia.org/wiki/初等代数 "wikilink")，**二项式**是只有两项的[多项式](https://zh.wikipedia.org/wiki/多项式 "wikilink")，即两个[单项式的和](https://zh.wikipedia.org/wiki/单项式 "wikilink")。二项式是仅次于单项式的最简单多项式。

## 例子

  - \(a + b \quad\)
  - \(x+3 \quad\)
  - \({x \over 2} + {x^2 \over 2}\)
  - \(v t - {1 \over 2} g t^2\)

## 运算法则

二项式与因子 *c* 的[乘法](../Page/乘法.md "wikilink")可以根据[分配律](../Page/分配律.md "wikilink")计算：

\[c (a + b) = c a + c b \\]

两个二项式 *a + b* 与 *c + d* 的乘法可以通过两次分配律得到：

\[(a + b)(c + d) = (a + b) c + (a + b) d \\]

\[= a c + b c + a d + b d \quad\].

二项式 *a + b* 的[平方](../Page/平方.md "wikilink")为

\[(a + b)^2 = a^2 + 2 a b + b^2 \quad\] 二项式 *a - b* 的平方为

\[(a - b)^2 = a^2 - 2 a b + b^2. \quad\]

二项式 \(a^2 - b^2\) 可以[因式分解](../Page/因式分解.md "wikilink")为另外两个二项式的乘积：

\[a^2 - b^2 = (a + b)(a - b). \quad\]

如果二项式的形式为

\[a x + b \quad\] 其中 *a* 与 *b* 是[常数](https://zh.wikipedia.org/wiki/常数 "wikilink")，*x* 是[变量](https://zh.wikipedia.org/wiki/变量 "wikilink")，那么这个二项式是[线性的](../Page/線性關係.md "wikilink")。

[复数是形式为](https://zh.wikipedia.org/wiki/复数 "wikilink")

\[a + i b \quad\] 的二项式，其中 *i* 是 -1 的[平方根](../Page/平方根.md "wikilink")。

两个线性二项式 *a x + b* and *c x + d* 的乘积为：

\[(a x + b)(c x + d) = a c x^2 + (a d + b c) x + b d \!\,\]

表示为

\[(a + b)^n \quad\] 的二项式 *a + b* 的 *n<sup>th</sup>* [次幂可以用](https://zh.wikipedia.org/wiki/幂 "wikilink")[二项式定理](../Page/二项式定理.md "wikilink")或者等价的[杨辉三角形](../Page/杨辉三角形.md "wikilink")展开。

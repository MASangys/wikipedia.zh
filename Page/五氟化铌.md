**五氟化铌**，也称**氟化铌(V)**，化学式为NbF<sub>5</sub>。它是一种无色的[单斜晶体](https://zh.wikipedia.org/wiki/单斜 "wikilink")，通常用作[铌](../Page/铌.md "wikilink")化学的起始原料。

## 制备

五氟化铌是从铌矿石中提炼金属铌的中间产物。它也可由金属铌在250至300°C发生[氟化反应制得](https://zh.wikipedia.org/wiki/氟化 "wikilink")，氟化剂可使用氟气、无水[氟化氢](../Page/氟化氢.md "wikilink")或[二氟化锡](https://zh.wikipedia.org/wiki/二氟化锡 "wikilink")。将五氟化铌蒸汽在[派热克斯玻璃或](https://zh.wikipedia.org/wiki/派热克斯玻璃 "wikilink")[石英玻璃管中浓缩](https://zh.wikipedia.org/wiki/石英玻璃 "wikilink")，然后在120°C下真空[升华](../Page/升华.md "wikilink")即可收集到无色晶体。如果与无水氟化氢反应，则会生成五氟化铌与三氟化铌的混合物。

\[\rm \ 2Nb + 5F_2 \xrightarrow{\Delta} 2NbF_5\]

\[\rm \ 2Nb + 5SnF_2 \xrightarrow{\Delta} 2NbF_5 + 5Sn\]

如果将金属铌溶于浓[硝酸](../Page/硝酸.md "wikilink")和[氢氟酸](../Page/氢氟酸.md "wikilink")的混合酸（或浓氢氟酸），蒸干溶液并灼烧残渣可得到[一氟二氧化铌](https://zh.wikipedia.org/wiki/一氟二氧化铌 "wikilink")（NbO<sub>2</sub>F），因此不常用该法制备五氟化铌。

五氟化铌也可由氟气、[氟化锌](../Page/氟化锌.md "wikilink")、无水氟化氢与[五氯化铌](../Page/五氯化铌.md "wikilink")反应制得：

\[\rm \ 2NbCl_5 + 5F_2 \xrightarrow{\Delta} 2NbF_5 + 5Cl_2\]

氟配合物热分解法也经常被使用：

\[\rm \ (BrF_2)NbF_6 \xrightarrow{\Delta} BrF_3 + NbF_5\]

\[\rm \ K_2NbF_7 + 2AlF_3 \xrightarrow{1073K} KAlF_4 + NbF_5\]

## 性质

五氟化铌吸水性很强，在空气中就会[潮解](../Page/潮解.md "wikilink")，但速率较慢不冒烟。因此不可能在水溶液中制得五氟化铌。五氟化铌气态时为[三角双锥形分子](https://zh.wikipedia.org/wiki/三角双锥 "wikilink")，液态时为双聚体，固态时为四聚体。固态五氟化铌是由四个\[NbF<sub>6</sub>\]八面体组成的，铌不仅与五个氟原子成键，还互相通过[金属—金属键形成四边形的骨架](https://zh.wikipedia.org/wiki/金属—金属键 "wikilink")。液态五氟化铌还会发生[自耦电离](https://zh.wikipedia.org/wiki/自耦电离 "wikilink")：

\[\rm \ 2NbF_5 \rightleftharpoons NbF_4^+ + NbF_6^-\]（小于1%）

五氟化铌是强[路易斯酸](https://zh.wikipedia.org/wiki/路易斯酸 "wikilink")，可溶于水和酸。溶解的过程还伴随着[水解](../Page/水解.md "wikilink")，产生[氟氧化铌和](https://zh.wikipedia.org/wiki/氟氧化铌 "wikilink")[氢氟酸](../Page/氢氟酸.md "wikilink")。与碱溶液反应产生组成不定的水合[五氧化二铌](../Page/五氧化二铌.md "wikilink")沉淀。\[1\]

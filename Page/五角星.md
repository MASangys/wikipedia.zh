[Pentagram_green.svg](https://zh.wikipedia.org/wiki/File:Pentagram_green.svg "fig:Pentagram_green.svg") **五角星**，又稱**五芒星**，是指一種有五隻尖角、並以五條等長直線畫成的星星圖形。英文「五角星」（**pentagram**）一詞出於[希臘語](https://zh.wikipedia.org/wiki/希臘語 "wikilink")「」（*pentagrammos*）或「」（*pentegrammos*）的[名詞](../Page/名詞.md "wikilink")形式「」（*pentagrammon*），原意大概是「五條直線的」或「五條線」；而中文「五角星」的意義則顯而易見，指有五隻角的星形。然而，中文「五角星」不一定指本文所談論的「標準」五角星，有時亦泛指所有有五隻角的星形物。

本文以下所讨论的五角星是指「標準」五角星，即正五角星。

## 几何

[Pentagram2.png](https://zh.wikipedia.org/wiki/File:Pentagram2.png "fig:Pentagram2.png")

五角星是边数最少的[星形多邊形](https://zh.wikipedia.org/wiki/星形多邊形 "wikilink")。最简单画它的方法是先画一个正[五边形](../Page/五边形.md "wikilink")，把各角和其對角用直线相连，并擦去原来的五边形。也可以延长原五边形的各边直到它们相交，从而得到一个大的五角星。

[黄金分割比](https://zh.wikipedia.org/wiki/黄金分割比 "wikilink")（golden ratio），\(\varphi = \frac{1 + \sqrt{5}}{2}= 1.618 \cdots\)，满足：

\[\varphi=1+2\sin\frac{\pi}{10}=1+2\sin 18^\circ\,\]

\[\varphi=\frac{1}{2\sin\frac{\pi}{10}}=\frac{1}{2\sin 18^\circ}\,\]

\[\varphi=2\cos\frac{\pi}{5}=2\cos 36^\circ\,\]

在正五边形和五角星中扮演一个重要的角色。每条边都被分割成几段小的线段，如果用一对线段中较长线段的长度除以较短线段的长度将得到\(\varphi\)。

|               |    |    |    |   |    |
| ------------- | -- | -- | -- | - | -- |
| \[\varphi =\] | 红色 | ＝  | 蓝色 | ＝ | 绿色 |
| 蓝色            | 绿色 | 紫色 |    |   |    |

大五边形的一条边与一条蓝色线段等长，同样的小五角星的一条对角线与一条绿色线段等长。

如果线段有长度（线条），那图中就有两个[相似的正五边形](https://zh.wikipedia.org/wiki/相似 "wikilink")。“线段”怎样，线条就怎样，并且它们的交点也可能变化。例如，在[摩洛哥的国旗上](https://zh.wikipedia.org/wiki/摩洛哥 "wikilink")，绿色五角星有一个黑色的外轮廓，它不“穿过”相交的线条。这跟[埃塞俄比亚](../Page/埃塞俄比亚.md "wikilink")国旗上五角星中的完全黄色和没有外轮廓，是不一样的。

### 一些相关几何值

\[\sin \frac{\pi}{10} = \sin 18^\circ = \frac{\sqrt 5 - 1}{4}=\frac{\varphi-1}{2}=\frac{1}{2\varphi}\]

\[\cos \frac{\pi}{10} = \cos 18^\circ = \frac{\sqrt{2(5 + \sqrt 5)}}{4}\]

\[\tan \frac{\pi}{10} = \tan 18^\circ = \frac{\sqrt{5(5 - 2 \sqrt 5)}}{5}\]

\[\cot \frac{\pi}{10} = \cot 18^\circ = \sqrt{5 + 2 \sqrt 5}\]

\[\sin \frac{\pi}{5} = \sin 36^\circ = \frac{\sqrt{2(5 - \sqrt 5)} }{4}\]

\[\cos \frac{\pi}{5} = \cos 36^\circ = \frac{\sqrt 5+1}{4} =\frac{\varphi}{2}\]

\[\tan \frac{\pi}{5} = \tan 36^\circ =  \sqrt{5 - 2\sqrt 5}\]

\[\cot \frac{\pi}{5} = \cot 36^\circ = \frac{ \sqrt{5(5 + 2\sqrt 5)}}{5}\]

因此，在一个有一个或两个36°角的[等腰三角形里](https://zh.wikipedia.org/wiki/等腰三角形 "wikilink")，两边中较长边的长度，是较短边的\(\phi\)倍，无论是[锐角三角形或者](https://zh.wikipedia.org/wiki/锐角三角形 "wikilink")[钝角三角形](https://zh.wikipedia.org/wiki/钝角三角形 "wikilink")。

## 國旗上的五角星

[Flag_of_Morocco.svg](https://zh.wikipedia.org/wiki/File:Flag_of_Morocco.svg "fig:Flag_of_Morocco.svg")國旗\]\] [Flag_of_Ethiopia.svg](https://zh.wikipedia.org/wiki/File:Flag_of_Ethiopia.svg "fig:Flag_of_Ethiopia.svg")國旗\]\] 有许多國家的國旗設計都包含五角星：如[智利](../Page/智利.md "wikilink")、[賴比瑞亞](https://zh.wikipedia.org/wiki/賴比瑞亞 "wikilink")、[埃塞俄比亞](https://zh.wikipedia.org/wiki/埃塞俄比亞 "wikilink")、[摩洛哥](https://zh.wikipedia.org/wiki/摩洛哥 "wikilink")、[越南](https://zh.wikipedia.org/wiki/越南 "wikilink")、[朝鲜民主主义人民共和国](../Page/朝鲜民主主义人民共和国.md "wikilink")、[中华人民共和国](https://zh.wikipedia.org/wiki/中华人民共和国 "wikilink")、[美国](../Page/美国.md "wikilink")等等。很多時候，五角星的中間被填滿，移除了五角形和五個等腰三角形之間的分界。Ivan Sache 指出，摩洛哥國旗上的五角星代表神和摩洛哥國之間的聯繫。\[1\]

## 其他方面

[Japanese_Crest_Abe_no_Seimei_Hann.svg](https://zh.wikipedia.org/wiki/File:Japanese_Crest_Abe_no_Seimei_Hann.svg "fig:Japanese_Crest_Abe_no_Seimei_Hann.svg")

  - 五角星被很多国家的军队作为军官（尤其是高级军官）的[军衔](../Page/军衔.md "wikilink")标志使用。
  - 一直以來，五角星都與对[金星](../Page/金星.md "wikilink")和[維納斯的崇拜有密切關係](https://zh.wikipedia.org/wiki/維納斯 "wikilink")。造成這關係的各個可能原因中，最可信的是史前[天文學家的觀察](https://zh.wikipedia.org/wiki/天文學家 "wikilink")。由[地球](../Page/地球.md "wikilink")望去，圍著太陽的金星軌道每八年重覆一次。它自成的五個交叉點恰好畫出一個近乎完美的五角星。
  - 在[古希臘](https://zh.wikipedia.org/wiki/古希臘 "wikilink")，五角星是大地女神科爾(Kore)的象徵。
  - 在[巴比倫](https://zh.wikipedia.org/wiki/巴比倫 "wikilink")，五角星被作為女神[伊絲塔](../Page/伊絲塔.md "wikilink")(Ishtar)的孿生姐姐冥界女神的符號。
  - 在[古埃及](../Page/古埃及.md "wikilink")，五角星被作為冥界子宮的符號。
  - 凱爾特人將五角星作為冥界女神摩根的象徵。
  - 五角星是[魔術](../Page/魔術.md "wikilink")的代表符號，是非[猶太教徒的符號](https://zh.wikipedia.org/wiki/猶太教 "wikilink")。初期[基督教](../Page/基督教.md "wikilink")會亦有用五角星代表[耶穌的五個傷口](https://zh.wikipedia.org/wiki/耶穌 "wikilink")，但現在則多代表異教徒和[撒但崇拜者](https://zh.wikipedia.org/wiki/撒但崇拜 "wikilink")。
  - 五角星在[魔法](../Page/魔法.md "wikilink")中是一種特別的圖形，用正的五角星作[魔法陣是](https://zh.wikipedia.org/wiki/魔法陣 "wikilink")[白魔法](https://zh.wikipedia.org/wiki/白魔法 "wikilink")，用倒的五角星則是象徵[黑魔法](https://zh.wikipedia.org/wiki/黑魔法 "wikilink")。
  - 而在東方[陰陽五行裡面](https://zh.wikipedia.org/wiki/陰陽五行 "wikilink")，五行「相剋」的連線剛好是五角星。在日本，[安倍晴明](../Page/安倍晴明.md "wikilink")使用桔梗紋變形的五芒星，被稱為桔梗印，現在是[晴明神社](../Page/晴明神社.md "wikilink")的[神紋](../Page/家紋.md "wikilink")。
  - 从最直观的方面看，五角星是最类似人形的多边形。
  - 倒五芒星可以代表魔鬼（[撒但](../Page/撒但.md "wikilink")）。

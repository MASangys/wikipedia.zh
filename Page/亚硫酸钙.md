**亚硫酸钙**属于[亚硫酸盐](../Page/亚硫酸盐.md "wikilink")与钙盐，化学式<chem>CaSO3.xH2O</chem>。亚硫酸钙有两种已知结晶水合物：半水合亚硫酸钙<chem>CaSO3.1/2H2O</chem>与四水合亚硫酸钙<chem>CaSO3.4H2O</chem>\[1\]，均为白色固体。亚硫酸钙是的产物之一。

## 用途

### 乾壁

在制备[乾壁的主要成分](https://zh.wikipedia.org/wiki/乾壁 "wikilink")[石膏](../Page/石膏.md "wikilink")的过程中，亚硫酸钙是一种中间产物。一间典型的全新美国房屋含有7公吨石膏\[2\]。

### 食品添加剂

亚硫酸钙可用作防腐剂，[E编码](../Page/E编码.md "wikilink")为E226\[3\]。与其他[亚硫酸盐](../Page/亚硫酸盐.md "wikilink")[抗氧化剂](../Page/抗氧化剂.md "wikilink")类似，亚硫酸钙广泛用于保存葡萄酒、苹果酒、水果汁、罐头水果、蔬菜等\[4\]。在溶液中，亚硫酸盐是强还原剂，作为起到保存食品的作用。因为部分人群对亚硫酸盐[过敏](../Page/过敏.md "wikilink")，所以有必要对其进行标注。\[5\]\[6\]

### 木浆生产

化学木浆的生产过程是溶解掉木片中的[木质素](../Page/木质素.md "wikilink")，分离出原本与前者紧密结合的纤维素。在中，用来分离木质素的盐可以是亚硫酸钙。人们曾经用过亚硫酸钙，但如今镁和钠的亚硫酸盐与亚硫酸氢盐已广泛取代之。另一种制浆法是[硫酸盐制浆法](../Page/硫酸盐制浆法.md "wikilink")，使用的是氢氧化物与[硫化物](../Page/硫化物.md "wikilink")。\[7\]

### 酸抑制剂

煤中的[二硫化铁在空气中煅烧会生成酸性气体](https://zh.wikipedia.org/wiki/二硫化铁 "wikilink")，溶于水形成酸，污染环境：

  - \(\rm FeS_2 + 3.5O_2 + H_2O \rightarrow Fe^{2+} + 2SO_4^{2-} + 2H^+\)
  - \(\rm 2Fe^{2+} + 0.5O_2 + 2H^+ \rightarrow 2Fe^{3+} + H_2O\)
  - \(\rm FeS_2 + 14Fe^{3+} + 8H_2O \rightarrow 15Fe^{2+} + 16H^+ + 2SO_4^{2-}\)

亚硫酸盐具有还原性，可以除去氧气从而抑制酸的形成。亚硫酸的钠盐、钾盐和铵盐都相对较贵且生产规模较小，而已经大规模生产的亚硫酸钙是该过程中较理想的酸抑制剂。\[8\]

## 制备

通过（FGD）可大量制备亚硫酸钙。[烟道气](../Page/烟道气.md "wikilink")指的是煤等化石燃料燃烧后得到的一类副产物，通常包括SO<sub>2</sub>。人们为了防止[酸雨](../Page/酸雨.md "wikilink")而限制二氧化硫排放，在其他气体从[烟囱排出之前](https://zh.wikipedia.org/wiki/烟囱 "wikilink")，需要除去二氧化硫。比较经济的做法是用[熟石灰](../Page/氢氧化钙.md "wikilink")（Ca(OH)<sub>2</sub>）或[石灰石](../Page/石灰岩.md "wikilink")（CaCO<sub>3</sub>）处理。

理想状况下，用石灰石除去二氧化硫的反应如下：

  -
    SO<sub>2</sub> + CaCO<sub>3</sub> → CaSO<sub>3</sub> + CO<sub>2</sub>

理想状况下，用熟石灰除去二氧化硫的反应如下：\[9\]\[10\]

  -
    SO<sub>2</sub> + Ca(OH)<sub>2</sub> → CaSO<sub>3</sub> + H<sub>2</sub>O

如果体系中存在[氧气](../Page/氧气.md "wikilink")，亚硫酸根就会被氧化为[硫酸根](../Page/硫酸根.md "wikilink")，最后可以得到二水合硫酸钙晶体，即[石膏](../Page/石膏.md "wikilink")\[11\]。

实验室制法则是令[氯化鈣](../Page/氯化鈣.md "wikilink")水溶液与[亚硫酸钠](../Page/亚硫酸钠.md "wikilink")水溶液反应：

  -
    CaCl<sub>2</sub> + Na<sub>2</sub>SO<sub>3</sub> → CaSO<sub>3</sub> + 2 NaCl

## 结构

亚硫酸钙的结构已由[X射线晶体学](../Page/X射线晶体学.md "wikilink")检验得出\[12\]。

半水合亚硫酸钙晶体中，钙的[配位数](../Page/配位数.md "wikilink")是6，每个钙离子连有五个亚硫酸根的氧原子及一个水分子中的氧原子，六个氧原子组成[三角反棱柱](../Page/三角反棱柱.md "wikilink")的形状。半水合亚硫酸钙晶体热力学较稳定的原因可能是Ca－O键在整个晶体中延展形成网状结构。\[13\]

四水合亚硫酸钙只在低温下稳定，在室温下会迅速转变为半水合物，因此获得四水合物[单晶并分析结构较为困难](https://zh.wikipedia.org/wiki/单晶 "wikilink")。不过，使用[琼脂制造溶胶环境即可减慢反应物分子的速度](https://zh.wikipedia.org/wiki/琼脂 "wikilink")，便于结晶，从而制得晶体。\[14\]四水合亚硫酸钙晶体中钙的配位数同样为6，两个氧原子来自亚硫酸根，四个来自水分子。其热力学不稳定的原因可能是其中的亚硫酸根离子处于无序状态。\[15\]

**亚铁氰化钾**，俗称**黄血盐**，是化学式为K<sub>4</sub>\[Fe(CN)<sub>6</sub>\]·3H<sub>2</sub>O的[配位化合物](https://zh.wikipedia.org/wiki/配位化合物 "wikilink")。室温下为柠檬黄色[单斜晶体](https://zh.wikipedia.org/wiki/单斜 "wikilink")，于沸点分解。它不溶于[乙醇](../Page/乙醇.md "wikilink")，但在水中的溶解度高达300g/L。

亚铁氰化钾水溶液与酸反应放出极毒的[氰化氢](../Page/氰化氢.md "wikilink")（HCN）气体，但亚铁氰化钾自身毒性很低。\[1\]

亚铁氰化钾加热分解得到[氰化钾](../Page/氰化钾.md "wikilink")，与[三价铁离子](../Page/三价铁离子.md "wikilink")反应生成颜料[普鲁士蓝](../Page/普鲁士蓝.md "wikilink")。

需要指出的是，亚铁氰化钾属于欧盟批准使用的食品添加剂（抗结块剂之列），在95/2/EC指令\[2\]中规定最高允许用量为20mg/kg（以无水亚铁氰化钾计）。

## 化学性质

### 热不稳定性

亚铁氰化钾对热不稳定，受热分解\[3\]：

  -
    3 K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] —<sup><u>灼烧</u></sup>→ 12 KCN + Fe<sub>3</sub>C + 2 (CN)<sub>2</sub>↑ + N<sub>2</sub>↑ + C
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] —<sup><u>△</u></sup>→ 4 KCN + FeC<sub>2</sub> + N<sub>2</sub>↑

除了以上分解方式之外，亚铁氰酸钾根据分解的条件不同，还会产生Fe<sub>3</sub>C、Fe、C等純物質。在氧气中則分解产生氧化态产物，如Fe<sub>3</sub>O<sub>4</sub>、KFeO<sub>2</sub>等。\[4\]

### 还原性

亚铁氰化钾可以被氯气氧化\[5\]：

  -
    2 K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + Cl<sub>2</sub> → 2 K<sub>3</sub>\[Fe(CN)<sub>6</sub>\] + 2 KCl
    2 K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + 2 H<sub>2</sub>O + 13 Cl<sub>2</sub>（过量）→ 2 HCN + 10 CNCl + 8 KCl + 2 FeCl<sub>3</sub> + 2 HClO

亚铁氰化钾也可以被过氧化氢氧化\[6\]：

  -
    2 K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + H<sub>2</sub>O<sub>2</sub> → 2 K<sub>3</sub>\[Fe(CN)<sub>6</sub>\] + 2 KOH
    2 K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + H<sub>2</sub>O<sub>2</sub> + 2 HCl → 2 K<sub>3</sub>\[Fe(CN)<sub>6</sub>\] + 2 KCl + 2 H<sub>2</sub>O

此外，它还可以被其他氧化剂，如[高锰酸钾](../Page/高锰酸钾.md "wikilink")，氧化为[铁氰酸钾](https://zh.wikipedia.org/wiki/铁氰酸钾 "wikilink")。

### 和酸反应

亚铁氰化钾和盐酸反应可以得到[亚铁氰酸](https://zh.wikipedia.org/wiki/亚铁氰酸 "wikilink")\[7\]\[8\]：

  -
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\](浓) + 4 HCl → H<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + 4 KCl

和硫酸反应的产物则取决于硫酸的浓度、具体反应条件等：

  -
    2 K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + 6 H<sub>2</sub>SO<sub>4</sub>(稀、沸) → 6 KHSO<sub>4</sub> + K<sub>2</sub>Fe\[Fe(CN)<sub>6</sub>\]↓(白) + 6 HCN
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + 3 H<sub>2</sub>SO<sub>4</sub>(3:1稀) —<sup><u>共沸</u></sup>→ 6 HCN + FeSO<sub>4</sub> + 2 K<sub>2</sub>SO<sub>4</sub>
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + 6 H<sub>2</sub>SO<sub>4</sub>(浓) + 6 H<sub>2</sub>O —<sup><u>共沸</u></sup>→ FeSO<sub>4</sub> + 2 K<sub>2</sub>SO<sub>4</sub> + 3 (NH<sub>4</sub>)<sub>2</sub>SO<sub>4</sub> + 6 CO↑
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + 11 H<sub>2</sub>SO<sub>4</sub>(浓) + 6 H<sub>2</sub>O —<sup><u>共沸</u></sup>→ 6 NH<sub>4</sub>HSO<sub>4</sub> + 4 KHSO<sub>4</sub> + FeSO<sub>4</sub> + 6 CO↑
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + 4 H<sub>2</sub>SO<sub>4</sub>(浓、缓慢加入) → 4 KHSO<sub>4</sub> + H<sub>4</sub>\[Fe(CN)<sub>6</sub>\]

和氮的含氧酸反应：

  -
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + HNO<sub>2</sub> + CH<sub>3</sub>COOH → K<sub>3</sub>\[Fe(CN)<sub>6</sub>\] + CH<sub>3</sub>COOK + NO↑ + H<sub>2</sub>O
    3 K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + 4 HNO<sub>3</sub>(浓) → K<sub>3</sub>\[Fe(CN)<sub>6</sub>\] + 3 KNO<sub>3</sub> + NO↑ + 2 H<sub>2</sub>O

### 和盐溶液反应

亚铁氰化钾和金属盐溶液大多发生[复分解反应](../Page/复分解反应.md "wikilink")，金属离子取代钾离子（部分或完全取代），通常产生沉淀。个别情况的反应比较特殊。其方程式如下\[9\]：

#### 和碱土金属盐的反应

  -
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\](冷、浓) + CaCl<sub>2</sub> → K<sub>2</sub>Ca\[Fe(CN)<sub>6</sub>\] + 2KCl
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + 2CaCl<sub>2</sub> → Ca<sub>2</sub>\[Fe(CN)<sub>6</sub>\]↓ + 4KCl
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + BaCl<sub>2</sub> + 3H<sub>2</sub>O —<U><sup>煮沸</sup></U>→ K<sub>2</sub>Ba\[Fe(CN)<sub>6</sub>\]·3H<sub>2</sub>O + 2KCl
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + MgCl<sub>2</sub> + 7H<sub>2</sub>O —<U><sup>煮沸</sup></U>→ K<sub>2</sub>Mg\[Fe(CN)<sub>6</sub>\]·7H<sub>2</sub>O↓ + 2KCl

#### 和其他主族金属盐的反应

  -
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + 2Pb(NO<sub>3</sub>)<sub>2</sub> → Pb<sub>2</sub>\[Fe(CN)<sub>6</sub>\]↓(白色) + 4KNO<sub>3</sub>

#### 和铁盐的反应

  -
    3K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + 4FeCl<sub>3</sub> → Fe<sub>4</sub>\[Fe(CN)<sub>6</sub>\]<sub>3</sub>↓（普鲁士蓝）+ 12KCl
    6K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + 6FeCl<sub>3</sub>(冷) → K<sub>6</sub>Fe<sub>6</sub>\[Fe(CN)<sub>6</sub>\]<sub>6</sub>(aq) + 18KCl
    6K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + 8FeCl<sub>3</sub> —<sup><U>△</U></sup>→ Fe<sub>8</sub>\[Fe(CN)<sub>6</sub>\]<sub>6</sub>↓ + 24KCl
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + 2FeCl<sub>2</sub> → Fe<sub>2</sub>\[Fe(CN)<sub>6</sub>\]↓(白色) + 4KCl
      -
        6Fe<sub>2</sub>\[Fe(CN)<sub>6</sub>\] + 6H<sub>2</sub>O + 3O<sub>2</sub> → 4Fe(OH)<sub>3</sub> + 2Fe<sub>4</sub>\[Fe(CN)<sub>6</sub>\]<sub>3</sub>
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + Fe<sub>4</sub>\[Fe(CN)<sub>6</sub>\]<sub>3</sub> → 4 KFe\[Fe(CN)<sub>6</sub>\]
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + Fe(OH)(NO<sub>3</sub>)<sub>2</sub> → Fe(OH)<sub>3</sub> + 2 FeK\[Fe(CN)<sub>6</sub>\] + 6 KNO<sub>3</sub>

#### 和其他过渡金属盐的反应

  -
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + 2CoCl<sub>2</sub> → Co<sub>2</sub>\[Fe(CN)<sub>6</sub>\]↓(绿色) + 4KCl
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + 2CdCl<sub>2</sub> → Cd<sub>2</sub>\[Fe(CN)<sub>6</sub>\]↓(白色) + 4KCl
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\]<sub>(S)</sub> + HgCl<sub>2(S)</sub> —<sup><U>△</U></sup>→ (CN)<sub>2</sub>↑ + 4KCN + FeCl<sub>2</sub> + Hg
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + 2AgI → Ag<sub>2</sub>I↓ + K<sub>3</sub>\[Fe(CN)<sub>6</sub>\] + KI
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + Th(NO<sub>3</sub>)<sub>4</sub> → 4KNO<sub>3</sub> + Th\[Fe(CN)<sub>6</sub>\]↓（白色）
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + TiCl<sub>4</sub> → 4KCl + Ti\[Fe(CN)<sub>6</sub>\]↓（淡红棕色）
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + 2Cu(NO<sub>3</sub>)<sub>2</sub> → Cu<sub>2</sub>\[Fe(CN)<sub>6</sub>\]↓(红褐色) + 4KNO<sub>3</sub>
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + 4AgNO<sub>3</sub> → 4KNO<sub>3</sub> + Ag<sub>4</sub>\[Fe(CN)<sub>6</sub>\]↓（白色）
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + 2 UO<sub>2</sub>Cl<sub>2</sub> → (UO<sub>2</sub>)<sub>2</sub>\[Fe(CN)<sub>6</sub>\]↓(黑棕色) + 4 KCl

### 其他反应

亚铁氰化钾可以和碳酸钾在高温下反应：

  -
    K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + K<sub>2</sub>CO<sub>3</sub> → 5 KCN + KOCN + Fe + CO<sub>2</sub>↑

<!-- end list -->

  -
    2 K<sub>4</sub>\[Fe(CN)<sub>6</sub>\] + 2 K<sub>2</sub>CO<sub>3</sub> + 15 S → 12 KSCN + 2 FeS + SO<sub>2</sub>↑ + 2CO<sub>2</sub>↑

亚铁氰化钾和钼酸铵的盐酸溶液反应，可以得到棕色的亚铁氰酸钼沉淀。

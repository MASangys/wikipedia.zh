**亚铬酸亚铁**是[离子化合物](https://zh.wikipedia.org/wiki/离子化合物 "wikilink")，[化学式](../Page/化学式.md "wikilink")为FeCr<sub>2</sub>O<sub>4</sub>。

## 制备

  - 自然界中存在于[铬铁矿](../Page/铬铁矿.md "wikilink")，虽然有许多杂质。

<!-- end list -->

  - 高温灼烧[三氧化二铬](../Page/三氧化二铬.md "wikilink")与[氧化铁](../Page/氧化铁.md "wikilink")：

\[\mathsf{FeO + Cr_2O_3 \ \xrightarrow{1600^oC}\ Fe(CrO_2)_2 }\]

## 物理性质

黑褐色[立方晶体](../Page/立方晶系.md "wikilink")，[空间群](../Page/空间群.md "wikilink")***F
d3m***，其中*a* = 0.8374 nm，Z = 8。

## 化学性质

  - 与浓[盐酸](../Page/盐酸.md "wikilink")反应：

\[\mathsf{Fe(CrO_2)_2 + 8HCl \ \xrightarrow{}\ 2CrCl_3 + FeCl_2 + 4H_2O }\]

  - 与热的浓[硝酸](../Page/硝酸.md "wikilink")反应：

\[\mathsf{Fe(CrO_2)_2 + 10HNO_3 \ \xrightarrow{}\ 2Cr(NO_3)_3 + Fe(NO_3)_3 + NO_2\uparrow + 5H_2O }\]

  - 在不活泼气体保护下，缓慢溶于碱中：

\[\mathsf{Fe(CrO_2)_2 + 6NaOH + 4H_2O \ \xrightarrow{\tau, N_2}\ 2Na_3[Cr(OH)_6] + Fe(OH)_2\downarrow }\]

  - 在空气中反应更快：

\[\mathsf{4Fe(CrO_2)_2 + O_2 + 24NaOH + 14H_2O \ \xrightarrow{}\ 8Na_3[Cr(OH)_6] + 4FeO(OH)\downarrow }\]

  - 被[碳](../Page/碳.md "wikilink")（[焦炭](https://zh.wikipedia.org/wiki/焦炭 "wikilink")）还原为铬和铁：

\[\mathsf{Fe(CrO_2)_2 + 4C \ \xrightarrow{1100-1200^oC}\ [Fe + 2Cr] + 4CO }\]

## 用途

作为[铬](../Page/铬.md "wikilink")的来源。\[1\]

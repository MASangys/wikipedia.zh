[数学上](../Page/数学.md "wikilink")，尤其是在[四则运算和](https://zh.wikipedia.org/wiki/四则运算 "wikilink")[初等代数中](https://zh.wikipedia.org/wiki/初等代数 "wikilink")，给定一个两边各一个[分式或](https://zh.wikipedia.org/wiki/分式 "wikilink")[比的](https://zh.wikipedia.org/wiki/比 "wikilink")[等式](https://zh.wikipedia.org/wiki/等式 "wikilink")，就可以用**交叉相乘**来[化简](https://zh.wikipedia.org/wiki/化简 "wikilink")[等式或求出](https://zh.wikipedia.org/wiki/等式 "wikilink")[变量](https://zh.wikipedia.org/wiki/变量 "wikilink")。
给定一个这样的[等式](https://zh.wikipedia.org/wiki/等式 "wikilink")：

\[\frac a b = \frac c d\]
(当和都不等于[零时](../Page/0.md "wikilink"))，可以交叉相乘来得到：\[1\]

\[ad = bc \qquad\]或\(\qquad a = \frac {bc} d.\)
[欧几里得几何中](../Page/欧几里得几何.md "wikilink")，相同的运算可以通过[相似三角形得到](../Page/相似三角形.md "wikilink")。

## 过程

实践中，*交叉相乘*的方法就是将两边的分子各乘以另一边的分母。\[2\]

\[\frac a b \nwarrow \frac c d \quad \frac a b \nearrow \frac c d.\]
这种方法的[数学证明是由下列](https://zh.wikipedia.org/wiki/数学证明 "wikilink")[数学过程推导而来的](../Page/数学.md "wikilink")。我们从这个简单的[等式开始](https://zh.wikipedia.org/wiki/等式 "wikilink")：

\[\frac a b = \frac c d\] (和都不等于[零](../Page/0.md "wikilink"))
我们可以两边同乘以[相等的](../Page/相等.md "wikilink")[数而两边仍然相等](../Page/数.md "wikilink")，所以如果我们在这个[等式两边同乘以](https://zh.wikipedia.org/wiki/等式 "wikilink")，我们就得到了:
\(\frac a b \times bd = \frac c d \times bd.\)
我们可以把[等式左边的两个](https://zh.wikipedia.org/wiki/等式 "wikilink")和右边的两个约去，剩下

\[ad = bc\] 我们在这里也可以两边同除以\[\frac {ad} {bd} = \frac {cb} {db}.\]来得到：

\[a = \frac {bc} d\] 我们也可以两边同乘以和 (都等于1)，得到：

\[\frac a b \times \frac d d = \frac c d \times \frac b b\] 所以：

\[\frac {ad} {bd} = \frac {cb} {db}.\] 两边同除以得到：

\[ad = cb.\]
这些步骤中的单独的每一步都基于[等式性质](https://zh.wikipedia.org/wiki/等式 "wikilink")，交叉相乘是一条[捷径](https://zh.wikipedia.org/wiki/捷径 "wikilink")，也是一个易于理解的，可以教给学生的过程。

## 使用方法

这是一个用来[化简](https://zh.wikipedia.org/wiki/化简 "wikilink")[等式或求出](https://zh.wikipedia.org/wiki/等式 "wikilink")[变量](https://zh.wikipedia.org/wiki/变量 "wikilink")[数值的](https://zh.wikipedia.org/wiki/数值 "wikilink")[数学方法](../Page/数学.md "wikilink")。如果我们遇到一个这样的方程：

\[\frac x b = \frac c d\] 我们可以用交叉相乘解出

\[x = \frac {bc} d.\]
举个例子，如果我们要求出一辆[车在](../Page/汽车.md "wikilink")7[小时内能开多远](https://zh.wikipedia.org/wiki/小时 "wikilink")，我们如果知道它是[匀速的](https://zh.wikipedia.org/wiki/匀速 "wikilink")，且它已在之前的3[小时内开了](https://zh.wikipedia.org/wiki/小时 "wikilink")90[英里](../Page/英里.md "wikilink")，将这个问题转为[比例我们得到](../Page/比例.md "wikilink")：

\[\frac x {7\ \mathrm{hours}} = \frac {90\ \mathrm{miles}} {3\ \mathrm{hours}}.\]
交叉相乘得到：

\[x = \frac {90\ \mathrm{miles}\times7\ \mathrm{hours}} {3\ \mathrm{hours}}.\]
所以

\[x = 210\ \mathrm{miles}\] 注意即使是这样的方程

\[a = \frac {x} {d}\] 将部分视为1，也可以视为下列方程来用*交叉相乘*来解决

\[\frac a 1 = \frac x d.\]
任何含有[分式的](https://zh.wikipedia.org/wiki/分式 "wikilink")[等式也都可以用两边同时乘以分母的](https://zh.wikipedia.org/wiki/等式 "wikilink")[最小公倍数来](https://zh.wikipedia.org/wiki/最小公倍数 "wikilink")[化简](https://zh.wikipedia.org/wiki/化简 "wikilink")

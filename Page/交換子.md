在[抽象代数中](../Page/抽象代数.md "wikilink")，一个群的**交換子**（commutator）或**换位子**是一个[二元運算子](../Page/二元運算子.md "wikilink")。设*g*及*h*
是 群*G*中的元素，他們的**交換子**是*g*<sup>−1</sup> *h*<sup>−1</sup> *gh*，常記為\[ *g*,
*h* \]。只有当*g*和*h*符合交换律（即gh =
hg）时他们的交换子才是这个[群的](../Page/群.md "wikilink")[单位元](../Page/单位元.md "wikilink")。

一个群*G*的全部交换子生成的子群叫做群*G*的[导群](../Page/导群.md "wikilink")，记作*D(G)*。

## 群論

[群](../Page/群.md "wikilink")中两个元素和的**交换子**为元素

  -

    *g*<sup>−1</sup>*h*<sup>−1</sup>*gh*}}

它等于群的幺元当且仅当和可交换（即 *hg*}}）。

## 環論

[环或](../Page/环.md "wikilink")[结合代数上两个元素](../Page/结合代数.md "wikilink")*a*和*b*的**交换子**定义为：

  -
    \([a, b] = ab - ba.\)

## 量子力學

[量子力学中](../Page/量子力学.md "wikilink")，经常用到**对易关系**（**commutation
relation**），即

\[[\hat{A}, \hat{B}] = \hat{A}\hat{B} - \hat{B}\hat{A}\]；

其中；\(\hat{A}\)、\(\hat{B}\)均为量子力学的[算符](../Page/算符.md "wikilink")，\([\hat{A}, \hat{B}]\)是其对易算符，也称**交换子**。

如果上式等于零，则称\(\hat{A}\)、\(\hat{B}\)是**对易**的，即意味着\(\hat{A}\)和\(\hat{B}\)两个算符的运算顺序可以调换。反之则称**非对易**的，运算顺序不可以调换。

量子力學中，**交換子**有以下特性:

\[[\hat{A},\hat{B}]=-[\hat{B},\hat{A}]\]

\[[\hat{A},\hat{B}+\hat{C}]=[\hat{A},\hat{B}]+[\hat{A},\hat{C}],\quad[\hat{A}+\hat{B},\hat{C}]=[\hat{A},\hat{C}]+[\hat{B},\hat{C}]\]

\[[\hat{A},\hat{B}\hat{C}]=[\hat{A},\hat{B}]\hat{C}+\hat{B}[\hat{A},\hat{C}],\quad[\hat{A}\hat{B},\hat{C}]=[\hat{A},\hat{C}]\hat{B}+\hat{A}[\hat{B},\hat{C}]\]

\[[\hat{A},\hat{A}^n]=0,\quad n=1,2,3...\]

\[[k\hat{A},\hat{B}]=[\hat{A},k\hat{B}]=k[\hat{A},\hat{B}]\]

\[[\hat{A}, [\hat{B}, \hat{C}]]+[\hat{C}, [\hat{A}, \hat{B}]]+[\hat{B}, [\hat{C}, \hat{A}]] = 0\]

量子力学中的各个力学量之间，常用的对易关系有：

以下，\(\hat{x}\)是[坐标算符](../Page/坐标算符.md "wikilink")、\(\hat{p}\)是[动量算符](../Page/动量算符.md "wikilink")、\(\hat{L}\)是[角动量算符](../Page/角动量算符.md "wikilink")（包括轨道角动量、自旋角动量等），而\(\delta_{ij}\)是[克罗内克δ](../Page/克罗内克δ.md "wikilink")、\(\epsilon_{ijk}\)是[列維-奇維塔符號](../Page/列維-奇維塔符號.md "wikilink")。其中i、j、k均可以指代x、y、z三个方向中的任意一个。

| 对易关系                                                        | 更具体的形式                                                                                                                                    |
| ----------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| \([\hat{x}_i, \hat{x}_j] = 0\)                              | \([\hat{x}, \hat{x}] = 0\)、\([\hat{x}, \hat{y}] = 0\)                                                                                     |
| \([\hat{p}_i, \hat{p}_j] = 0\)                              | \([\hat{p}_x, \hat{p}_x] = 0\)、\([\hat{p}_x, \hat{p}_y] = 0\)                                                                             |
| \([\hat{x}_i, \hat{p}_j] = i\hbar \delta_{ij}\)             | \([\hat{x}, \hat{p}_x] = i\hbar\)、\([\hat{x}, \hat{p}_y] = 0\)、\([\hat{y}, \hat{p}_x] = 0\)、\([\hat{y}, \hat{p}_y] = i\hbar\)             |
| \([\hat{L}_i, \hat{L}_j] = i\hbar \epsilon_{ijk}\hat{L}_k\) | \([\hat{L}_x, \hat{L}_y] = i\hbar \hat{L}_z\)、\([\hat{L}_y, \hat{L}_z] = i\hbar \hat{L}_x\)、\([\hat{L}_z, \hat{L}_x] = i\hbar \hat{L}_y\) |

### 正則對易關係

[物理學中](../Page/物理學.md "wikilink")，**正則對易關係**是[正則共軛的量之間的關係](../Page/正則坐標.md "wikilink")，這樣的量從定義可以發現：一個量是其共軛量的[傅立葉變換的結果](../Page/傅立葉變換.md "wikilink")。舉例來說：

\[[x,p] = i\hbar\]

上面的*x*與*p*分別為一維空間中的一點粒子的[位置與](../Page/位置.md "wikilink")[動量](../Page/動量.md "wikilink")，而\([x,p]=xp-px\)為所謂\(x\)與\(p\)的[交換算符](../Page/交換算符.md "wikilink")，\(i\)是[虛數單位](../Page/虛數單位.md "wikilink")，\(\hbar\)為[約化普朗克常數](../Page/約化普朗克常數.md "wikilink")，等於\(h/2\pi\)。此一關係常歸功於[馬克斯·玻恩](../Page/馬克斯·玻恩.md "wikilink")，並且此式子暗示了以海森堡為名的[不確定性原理](../Page/不確定性原理.md "wikilink")。

### 與古典力學的關係

相對於[量子力學](../Page/量子力學.md "wikilink")，[古典物理中所有](../Page/古典物理.md "wikilink")[可觀測量都可對易](../Page/可觀測量.md "wikilink")（交換），而[交換算符會是零](../Page/交換算符.md "wikilink")；然而仍然有類似的關係存在：需將交換子換成[泊松括號](../Page/泊松括號.md "wikilink")，且常數\(i\hbar\)換成\(1\)：

\[\{x,p\} = 1 \,\!\]

這樣的觀察導致了[保羅·狄拉克提出假設](../Page/保羅·狄拉克.md "wikilink")：一般來說，古典的觀測量\(f,g\)其量子對應項\(\hat f,\hat g\)應滿足

\[[\hat f,\hat g]= i\hbar\widehat{\{f,g\}} \,\]。

於1927年，[赫尔曼·外尔](../Page/赫尔曼·外尔.md "wikilink")（Hermann
Weyl）指出了量子算符與[相空間中古典分布之間的對應關係並不成立](../Page/相空間.md "wikilink")。不過他倒是提出了一個機制，稱作[魏爾量子化](../Page/魏爾量子化.md "wikilink")（Weyl
quantization），為了一種稱作[形變量子化](../Page/形變量子化.md "wikilink")（deformation
quantization）的量子化方法提供了數學途徑。

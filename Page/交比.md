[数学上](../Page/数学.md "wikilink")，[複平面上四点的](../Page/複平面.md "wikilink")**交比**是

\[(z_1,z_2;z_3,z_4) = \frac{(z_1-z_3)(z_2-z_4)}{(z_1-z_4)(z_2-z_3)}\]。

这个定义可以[连续延拓至整个](../Page/连续函数.md "wikilink")[黎曼球面](../Page/黎曼球面.md "wikilink")，即複平面加上[无穷远点](../Page/无穷远点.md "wikilink")。

一般来说，交比可以定义在[射影直线](../Page/射影直线.md "wikilink")（黎曼球面就是[複射影直線](../Page/複射影直線.md "wikilink")）。在任何[仿射坐标卡中](../Page/仿射坐标卡.md "wikilink")，交比由上式给出。交比是[射影几何的不变量](../Page/射影几何.md "wikilink")，就是说[射影变换保持交比不变](../Page/射影变换.md "wikilink")。
从前人们注意到如果四条直线穿过一点*P*，第五条直线*L*不穿过*P*，分别与四条直线交于四点，那么在*L*上按序取四点的有向长度，所算出的交比是独立于*L*。它是这四直线系的不变量。

四个複数的交比为[实数当且唯当四点](../Page/实数.md "wikilink")[共线或](../Page/共线.md "wikilink")[共圆](../Page/共圆.md "wikilink")。

## 对称

各著作对交比有不同定义，不过各定义只相异于某些坐标的[置换](../Page/置换.md "wikilink")。一般来说，根据点*z*<sub>*i*</sub>所给出的各种次序，交比可以取六个不同的值。因为四个坐标有24种排列，有些置换保持交比不变。实际上，任意两对坐标对换保持交比：

\[(z_1,z_2;z_3,z_4) = (z_2,z_1;z_4,z_3) = (z_3,z_4;z_1,z_2) = (z_4,z_3;z_2,z_1)\,\]。

运用这些对称，交比就有6个可能值，由点的次序决定：

|                                                      |  |                                                      |
| ---------------------------------------------------- |  | ---------------------------------------------------- |
| \((z_1, z_2; z_3, z_4) = \lambda\,\)                 |  | \((z_1, z_2; z_4, z_3) = {1\over\lambda}\)           |
| \((z_1, z_3; z_4, z_2) = {1\over{1-\lambda}}\)       |  | \((z_1, z_3; z_2, z_4) = 1-\lambda\,\)               |
| \((z_1, z_4; z_3, z_2) = {\lambda\over{\lambda-1}}\) |  | \((z_1, z_4; z_2, z_3) = {{\lambda-1}\over\lambda}\) |

从[群论来说](../Page/群论.md "wikilink")，[对称群](../Page/对称群.md "wikilink")*S*<sub>4</sub>以置换坐标来作用于交比上，这群作用的[核为](../Page/核_\(代数\).md "wikilink")[克莱因四元群](../Page/克莱因四元群.md "wikilink")（这是保持交比的群）。那么有效对称群是其[商群](../Page/商群.md "wikilink")，同构于*S*<sub>3</sub>。

对某些λ值会有更强的对称，交比的可能值就少于六个。这些λ值对应于*S*<sub>3</sub>对黎曼球面的作用的[不动点](../Page/不动点.md "wikilink")（由以上六个函数给出）；等价地，就是在置换群内有非平凡[稳定子群的点](../Page/稳定子群.md "wikilink")。

第一个这样的集合是{0, 1,
∞}。但若四点{*z*<sub>*i*</sub>}相异，交比不可能取这些值。这些值是当有一对坐标彼此趋近时的极限值：

\[(z,z_2;z,z_4) = (z_1,z;z_3,z) = 0\,\]

\[(z,z;z_3,z_4) = (z_1,z_2;z,z) = 1\,\]

\[(z,z_2;z_3,z) = (z_1,z;z,z_4) = \infty\,\]

第二个这样的集点是{−1, 1/2,
2}。这情况古典上称为「谐和交比」。最对称的交比是当\(\lambda = e^{\pm i\pi/3}\)。这时交比只可能是这两个值。

## 从变换出发

交比为黎曼球面的射影变换所保持，也称为[莫比乌斯变换](../Page/莫比乌斯变换.md "wikilink")：

\[f(z) = \frac{az+b}{cz+d}\;,\quad ad-bc \ne 0\]。

所谓它们保持交比就是指

\[(f(z_1), f(z_2); f(z_3), f(z_4)) = (z_1, z_2; z_3, z_4)\,\]。

作用于黎曼球面上的麦比乌斯变换群有一性质：任意3点集要映射到另外的3点集，都存在唯一的麦比乌斯变换。（这个群作用有3重传递性。）所以给出黎曼球面上4点，有唯一变换把其中3点映射到点0，1，和∞。第四点映射到的点，与原来四点的交比有关。

要看到这点，注意到

\[(z,1;0,\infty) = \lim_{w\to\infty} \frac{z(1-w)}{z-w} = z\,\]。 所以给出四点
\((z_1, z_2; z_3, z_4)\)可以找到唯一变换*f*作映射

\[z_2 \to 1,\; z_3 \to 0,\; z_4 \to \infty\]。
点\(z_1\)就被映射到\((z_1, z_2; z_3, z_4) = f(z_1)\)。换个角度看，若把交比看为\(z_1\)的函数，交比是唯一的变换把点\((z_2, z_3, z_4)\)映射到\((1,0,\infty)\)。

## 高等观点

若四点走近，这理论便有了微分学的一面，从而引领至[施瓦茨导数理论](../Page/施瓦茨导数.md "wikilink")，还有更一般的[射影联络理论](../Page/射影联络.md "wikilink")。这些理论被应用在[共形场论](../Page/共形场论.md "wikilink")。

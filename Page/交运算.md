在数学中，在一个集合上的**交**(meet)有两种定义：关于在这个集合上的[偏序的唯一](../Page/偏序关系.md "wikilink")[下确界](../Page/下确界.md "wikilink")(最大下界)，假定下确界存在的话；
或者是满足[幂等律的](../Page/幂等.md "wikilink")[交换](../Page/交换律.md "wikilink")[结合](../Page/结合律.md "wikilink")[二元运算](../Page/二元运算.md "wikilink")。在任何一个情况下，这个集合与交运算一起是[半格](../Page/半格.md "wikilink")。这两个定义产生等价的结果，除了在[偏序方式中有可能直接定义更一般的元素的集合的交](../Page/偏序关系.md "wikilink")。最常见到交运算的领域是[格](../Page/格_\(数学\).md "wikilink")。

通常把 \(x\) 和 \(y\) 的交指示为 \(x \land y\)。

## 偏序定义

设 *A* 是带有[偏序](../Page/偏序关系.md "wikilink") \(\leq\) 的一个集合，并设 \(x\) 和
\(y\) 是 *A* 中的两个元素。*A* 的一个元素 \(z\) 是 \(x\) 和 \(y\)
的交(或最大下界或下确界)，如果满足了下列两个条件:

  -
    **1.** \(z \leq x\) 且 \(z \leq y\) (就是说，\(z\) 是 \(x\) 和 \(y\) 的下界)；
    **2.** 对于 *A* 中的任何 \(w\)，使得 \(w \leq x\) 且 \(w \leq y\)，有着
    \(w \leq z\) (就是说，\(z\) 大于任何其他 \(x\) 和 \(y\) 的下界)。

如果有 \(x\) 和 \(y\) 的交，则它的确是唯一的，因为如果 \(z\) 和 \(z'\) 都是 \(x\) 和 \(y\)
的最大下界，则 \(z \leq z' \leq z\)，因而 \(z = z'\,\)。如果交确实存在，它被指示为
\(x \land y\)。在 *A*
中的某对元素可能缺乏一个交，要么因为它们根本没有下界，要么因为它们的下界中没有一个大于所有其他的。如果所有的元素对都有交，则交实际上是在
*A* 上的二元运算，并且容易看出这个运算满足下列三个条件: 对有 *A* 中任何元素 \(x\), \(y\) 和 \(z\)

  -
    **a.** \(x \land y = y \land x\) ([交换律](../Page/交换律.md "wikilink"))，
    **b.** \(x \land (y \land z) =(x \land y) \land z\)
    ([结合律](../Page/结合律.md "wikilink"))，
    **c.** \(x \land x = x\) ([幂等律](../Page/幂等律.md "wikilink"))。

## 泛代数定义

通过定义，在集合 *A*上的 [二元运算](../Page/二元运算.md "wikilink") \(\land\)
是**交**，如果它满足上面的三个条件 **a**, **b** 和 **c**。有序对
(*A*,\(\land\)) 就是[交半格](../Page/半格.md "wikilink")。此外，我们可以定义在 *A*
上[二元关系](../Page/二元关系.md "wikilink") \(\leq\)，通过声称 \(x \leq y\)
当且仅当 \(x \land y = x\)。实际上，这个关系是在 *A*
上的[偏序](../Page/偏序关系.md "wikilink")。对于 *A* 中任何元素 \(x\),
\(y\) 和 \(z\) 有

\[x \leq x\]，因为 \(x \land x = x\)，通过公理 **c**；

  -
    如果 \(x \leq y\) 且 \(y \leq x\) 则
    \(x  = x \land y = y \land x = y\)，通过公理 **a**；
    如果 \(x \leq y\) 且 \(y \leq z\) 则 \(x \leq z\)，因为
    \(x \land z = (x \land y) \land z = x \land (y \land z) = x \land y = x\)，通过公理
    **b** 。

## 两个定义的等价性

如果 (*A*,\(\leq\)) 是[偏序集合](../Page/偏序关系.md "wikilink")，使得对于每对 *A*
中的元素都有交，则确实有 \(x \land y = x\) 当且仅当
\(x \leq y\)，因为在后者情况下 \(x\) 确实是 \(x\) 和 \(y\)
的下界，并且明显的 \(x\) 是最大下界当且仅当它是下界。所以用泛代数方式的交定义的偏序一致于最初的偏序。

反过来，如果 (*A*,\(\land\)) 是交半格，并且偏序 \(\leq\) 按泛代数方式定义，对于*A* 中某些元素 \(x\) 和
\(y\) 有 \(z = x \land y\)，则 \(z\) 是 \(x\) 和 \(y\) 关于 \(\leq\) 的最大下界，因为
\(z \land x = x \land z = (x \land x) \land y = z \;\Rightarrow\; z \leq x\)，类似的
\(z \leq y\)，并且如果 \(w\) 是 \(x\) 和 \(y\) 的另一个下界，则
\(w \land x = w \land y = w\)，从而
\(w \land z = w \land (x \land y) = (w \land x) \land y = w \land y = w\)。所以这个用最初的交定义的偏序定义的交一致于最初的交。

换句话说，两种方式生成本质上等价的概念，集合配备了二元关系和二元运算二者，使得每个结构都由另一个确定，而且分别满足偏序或交的条件。

## 一般子集的交

如果 (*A*,\(\land\))
是交半格，则交可以被扩为任何非空有限集合的良好定义的交，通过在[迭代二元运算中描述的描述的技术](../Page/迭代二元运算.md "wikilink")。可供选择的，如果交定义或定义自一个偏序，*A*
的某个子集确实有关于它的[下确界](../Page/下确界.md "wikilink")。对于非空有限子集，这两种方式产生同样的结果，因为都可以做为交的定义。在
*A* 的每个子集都有交的情况下，(*A*,\(\leq\))
是[完全格](../Page/完全格.md "wikilink")；详情参见[完全性
(序理论)](../Page/完全性_\(序理论\).md "wikilink")。

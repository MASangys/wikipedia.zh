**亨德森-哈塞尔巴尔赫方程**（）是[化学中关于](https://zh.wikipedia.org/wiki/化学 "wikilink")[酸碱平衡的一个](https://zh.wikipedia.org/wiki/酸碱平衡 "wikilink")[方程](../Page/方程.md "wikilink")。该方程使用p*K*<sub>a</sub>（即[酸解离常数](https://zh.wikipedia.org/wiki/酸解离常数 "wikilink")）描述[pH值的变化](https://zh.wikipedia.org/wiki/pH值 "wikilink")。它可以用来估算[缓冲体系的pH](../Page/缓冲溶液.md "wikilink")。方程的创始人是美国化学家和丹麦科学家。

若有弱酸HA水解如下：

\[\mbox{HA} + \mbox{H}_{2}\mbox{O} \rightleftharpoons \mbox{A}^- + \mbox{H}_{3}\mbox{O}^+\]
则方程写作：

\[\textrm{pH} = \textrm{p}K_\textrm{a}+ \log\frac{[\textrm{A}^-]}{[\textrm{HA}]}\]
又写作：

\[\textrm{pH} = \textrm{p}K_\textrm{a}+\log\left ( \frac{[\mathrm{base}]}{[\mathrm{acid}]} \right )\]
其中A<sup>−</sup>是HA失去质子后的形式，即其共轭碱。中括号表示物质浓度。

若有碱水解如下：

\(\mathrm{B} + \mathrm{H}^{+} \rightleftharpoons \mathrm{BH}^{+}\)

则方程可根据写作：

\(\mathrm{p}K_\mathrm{b} = - \log_{10} (K_\mathrm{b}) = - \log_{10} \left ( \frac{[\mathrm{O}\mathrm{H}^-][\mathrm{HA}]}{[\mathrm{A}^-]} \right )\)，\(K_\mathrm{b}\)表示碱的水解常数。

类比可得：

\(\mathrm{pOH} = \mathrm{p}K_\mathrm{b}+ \log_{10}  \left ( \frac{[\mathrm{BH}^+]}{[\mathrm{B}]} \right )\)，
\([\mathrm{BH}^+]\)表示B的共轭酸。

采用摄氏25度下的性质，可以推导出以下表示pH的公式：

\(\mathrm{pH} = \mathrm{p}K_\mathrm{a} + \log_{10} \left(\frac{[\mathrm{B}]}{[\mathrm{BH}^{+}]}\right)\)

## 历史

1908年，劳伦斯·约瑟夫·亨德森在研究[碳酸的缓冲能力时提出](../Page/碳酸.md "wikilink")**亨德森方程**。1916年，卡尔·阿尔伯特·哈塞尔巴尔赫将其写为对数形式，并用于研究[血液中碳酸引起的](../Page/血液.md "wikilink")[代谢性酸中毒](https://zh.wikipedia.org/wiki/代谢性酸中毒 "wikilink")。[1](http://www.acid-base.com/history.php)

亨德森方程的形式为：

\[K = \frac{[\textrm{H}^+][\textrm{HCO}_3^-]} {[\textrm{CO}_2]}\]

## 原理

一元弱酸HA解离常数为*K*<sub>a</sub>，则有

\[K_\textrm{a} = \frac{[\textrm{H}^+][\textrm{A}^-]} {[\textrm{HA}]}\]
两侧取[对数](https://zh.wikipedia.org/wiki/对数 "wikilink")：

\[\log_{10}K_\textrm{a} = \log_{10}[\textrm{H}^+] + \log_{10} \left ( \frac{[\textrm{A}^-]}{[\textrm{HA}]} \right )\]
再把关于H<sup>+</sup>和*K*<sub>a</sub>的项转化为负对数，因为pH和p*K*<sub>a</sub>较常用：

\[-\textrm{p}K_\textrm{a} = -\textrm{pH} + \log_{10} \left ( \frac{[\textrm{A}^-]}{[\textrm{HA}]} \right )\]
移项即得方程。

由此可见，如果体系中只有这一种平衡，且计算时各种物质均取[活度](https://zh.wikipedia.org/wiki/活度 "wikilink")(有效濃度)的话，该方程是无条件成立的。

然而实际体系中总有[水的自偶电离](../Page/水.md "wikilink")，而且估算时常常会用[浓度甚至](https://zh.wikipedia.org/wiki/浓度 "wikilink")[分析浓度代替活度](https://zh.wikipedia.org/wiki/分析浓度 "wikilink")。故这种估算在溶液极稀、水的电离不能忽略的情况下（低于1mM）不成立，在[离子强度过高的情况下不成立](https://zh.wikipedia.org/wiki/离子强度 "wikilink")，在非HA-A<sup>−</sup>缓冲体系如强酸、强碱甚至过于偏离p*K*<sub>a</sub>的条件下也会有较大误差。

## 推论

在H-H方程成立的前提下，可以得出如下推论：

  - 体系中，只要弱酸及其共轭碱的浓度相等，pH即确定为p*K*<sub>a</sub>，而与浓度无关。
  - 缓冲体系，pH比p*K*<sub>a</sub>大1时，碱浓度是酸浓度的10倍。反之亦然。

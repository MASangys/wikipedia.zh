**亨泽尔引理**是[数学中](../Page/数学.md "wikilink")[模算术的一個结论](../Page/模算术.md "wikilink")。亨泽尔引理说明，如果一个[模](../Page/同余.md "wikilink")（是给定的[质数](../Page/质数.md "wikilink")）的[多项式](../Page/多项式.md "wikilink")[方程有一个](../Page/方程.md "wikilink")[单根](../Page/多项式.md "wikilink")，则可以通过这个根求出该方程在模的更高次方时的根。在[完备](../Page/完备空间.md "wikilink")[交换环](../Page/交换环.md "wikilink")（包括[p进数](../Page/p进数.md "wikilink")）中，亨泽尔引理被看作是类似于[牛顿法的渐进求根方法](../Page/牛顿法.md "wikilink")。由于p进数分析在某些方面比[实分析更加简单](../Page/实分析.md "wikilink")，亨泽尔引理可以加强为多项式方程有根的判定方法。

## 定理内容

設\(f(x)\)為[整係數](../Page/整数.md "wikilink")[多項式](../Page/多項式.md "wikilink")，\(k\)為不少於2的整數，\(p\)為[質數](../Page/質數.md "wikilink")。若整數\(r\)是下面同餘式的根：

  -
    \(f(r) \equiv 0 \pmod{p^{k-1}}.\)

對於

  -
    \(f(r+tp^{k-1}) \equiv 0 \pmod{p^k}\) （I）

，則有：

  - 若\(f'(r) \not\equiv 0 \pmod{p}\)，則存在唯一的整數\(0\le t \le p-1\)使得（I）成立。

<!-- end list -->

  -
    \(t f'(r) \equiv -( f(r) /p^{k-1} ) \pmod{p} .\,\)

<!-- end list -->

  - 若\(f'(r) \equiv 0 \pmod{p}\) 且 \(f(r) \equiv 0 \pmod{p^k}\)
    ，則（I）對任意整數t成立。
  - 若\(f'(r) \equiv 0 \pmod{p}\) 但
    \(f(r) \not\equiv 0 \pmod{p^k}\)，則（I）無整數解。

## 證明

Hensel引理可用[泰勒公式證明](../Page/泰勒公式.md "wikilink")。

  -
    \(f(r+tp^{k-1}) = f(r) + t p^{k-1} f'(r) + \frac{1}{2} t^2 p^{2(k-1)} f''(r) + \frac{1}{6} t^3 p^{3(k-1)} f'''(r) + ...\)

因此可見，由第三項開始，都必能被\(p^k\)整除。因此：

  -
    \(f(r+tp^{k-1}) \equiv f(r) + t p^{k-1} f'(r) \pmod{p^k}\)

## 推廣

若\(K\)為完備局域。設 \(\mathcal{O}_K\)為\(K\)的整數環，設\(f(x)\)為係數在
\(\mathcal{O}_K\)的多項式，若存在 \(\alpha_0 \in \mathcal{O}_K\)使得

  -
    \(|f(\alpha_0)| < |f'(\alpha_0)|^2\)

則\(f(x)\)有根\(\alpha \in K\)。

且：

1.  \(\alpha_{i+1} = \alpha_i - \frac{ f(\alpha_i) }{f'(\alpha_i) }\)
    趨近\(\alpha\)
2.  \(| \alpha - \alpha_0 | \le | \frac{ f(\alpha_i) }{f'(\alpha_i) } | < 1\)

這個引理其中一個重要應用就是在域為[p進數的情形](../Page/p進數.md "wikilink")。

《**亮相**》》（[英文](../Page/英语.md "wikilink")：；前名：《**眼光獨到**》) 是美國[全國廣播公司](https://zh.wikipedia.org/wiki/全國廣播公司 "wikilink")（NBC）原創（見[亮相](../Page/亮相.md "wikilink")），授權[香港](../Page/香港.md "wikilink")[電視廣播有限公司](../Page/電視廣播有限公司.md "wikilink")製作香港版的遊戲節目。本節目為[2008年節目巡禮之一](https://zh.wikipedia.org/wiki/無綫盛世節目2008 "wikilink")。第1輯由2008年5月5日至6月20日逢星期一至五晚上22:30-23:00及2008年6月28日至7月12日一連三個星期六晚上20:30-21:30在[翡翠台](../Page/翡翠台.md "wikilink")及[高清翡翠台](../Page/高清翡翠台.md "wikilink")播放，此3集邀請多位藝人及當屆的香港小姐成為陌生人。第1、2輯旁白為[陳欣](https://zh.wikipedia.org/wiki/陳欣 "wikilink")。第1輯，由[葛民輝](../Page/葛民輝.md "wikilink")主持。助手為[趙嘉迪](https://zh.wikipedia.org/wiki/趙嘉迪 "wikilink")。第2輯由2008年8月25日至9月19日逢星期一至五晚上22:30-23:00在[翡翠台](../Page/翡翠台.md "wikilink")及[高清翡翠台](../Page/高清翡翠台.md "wikilink")播放。因[葛民輝](../Page/葛民輝.md "wikilink")身在[北京](https://zh.wikipedia.org/wiki/北京 "wikilink")，故邀得[森美](https://zh.wikipedia.org/wiki/森美 "wikilink")、[阮小儀](../Page/阮小儀.md "wikilink")擔任第1-15集之主持，第16-20集則分別邀得[方力申](../Page/方力申.md "wikilink")、[楊思琦](../Page/楊思琦.md "wikilink")、[丘凱敏](../Page/丘凱敏.md "wikilink")擔任主持。助手為[孫慧雪](../Page/孫慧雪.md "wikilink")。

## 第1輯

### 遊戲玩法

第一輯節目會邀請12位不同身份人士出現，並大玩「疑似身份」，然後讓參賽者逐一去估他們的真正身份，參賽者每估中一個，便會有不同數目獎金，由10,000到200,000港元不等，相比起美國原裝版本最高獎金50萬美元（折合390萬港元）低得多。不過在遊戲過程中估錯一個，獎金會全數歸零。

|        |         |          |          |          |          |         |
| ------ | ------- | -------- | -------- | -------- | -------- | ------- |
| 猜中身份數量 | 1       | 2        | 3        | 4        | 5        | 6       |
| 獎金（港元） | $10,000 | $20,000  | $30,000  | $40,000  | $50,000  | $60,000 |
| 猜中身份數量 | 7       | 8        | 9        | 10       | 11/12    |         |
| 獎金（港元） | $80,000 | $100,000 | $120,000 | $150,000 | $200,000 |         |

此外，有別於美國原裝版本邀請一般公眾作為參賽者，香港版本以香港藝人參賽者。

與美國原裝版本一樣，參賽者有三大法寶：「[免死金牌](https://zh.wikipedia.org/wiki/免死金牌 "wikilink")」、「三大可能」以及「顧問教路」。

「顧問教路」中的顧問為[麥玲玲](../Page/麥玲玲.md "wikilink")、[Rick Chin](https://zh.wikipedia.org/wiki/Rick_Chin "wikilink")、[鄧達智](../Page/鄧達智.md "wikilink")、[朱蘊晶](https://zh.wikipedia.org/wiki/朱蘊晶 "wikilink")、[李家仁](../Page/李家仁.md "wikilink")、[龐愛蘭](../Page/龐愛蘭.md "wikilink")、[謝偉俊](../Page/謝偉俊.md "wikilink")等。

## 第2輯

### 遊戲玩法

第二輯節目與第一輯一樣會邀請12位不同身份人士出現，並大玩「疑似身份」，然後讓參賽者逐一去估他們的真正身份，參賽者每估中一個，便會有不同數目獎金，由10,000到200,000港元不等，相比起美國原裝版本最高獎金50萬美元（折合390萬港元）低得多。

|        |         |          |          |          |          |         |
| ------ | ------- | -------- | -------- | -------- | -------- | ------- |
| 猜中身份數量 | 1       | 2        | 3        | 4        | 5        | 6       |
| 獎金（港元） | $10,000 | $20,000  | $30,000  | $40,000  | $50,000  | $60,000 |
| 猜中身份數量 | 7       | 8        | 9        | 10       | 11/12    |         |
| 獎金（港元） | $80,000 | $100,000 | $120,000 | $150,000 | $200,000 |         |

此外，有別於美國原裝版本邀請一般公眾作為參賽者，香港版本第1-15集以香港藝人參賽者，第16-20集則由香港運動員為參賽者。

但是，不同的是，第二輯只有兩大法寶，只保留「[免死金牌](https://zh.wikipedia.org/wiki/免死金牌 "wikilink")」及「三大可能」。而「顧問教路」在此輯並沒有出現。

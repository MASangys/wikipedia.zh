**代謝產物** ()，又稱**代謝物**是[代謝的中間或最後產物](https://zh.wikipedia.org/wiki/代謝 "wikilink")，這個詞彙是通常指的是。他們有諸如作為燃料、結構、訊號、刺激、抑止酵素（通常作為酵素的輔因子）、防衛或作用在其他有機分子上（如：[色素](https://zh.wikipedia.org/wiki/色素 "wikilink")、[氣味分子或](https://zh.wikipedia.org/wiki/氣味分子 "wikilink")[費洛蒙](https://zh.wikipedia.org/wiki/費洛蒙 "wikilink")）。在一般成長，發育以及繁殖的階段，就有[初級代謝產物的參與](https://zh.wikipedia.org/wiki/初級代謝產物 "wikilink")。工業中被大量製造的[乙烯](../Page/乙烯.md "wikilink")，就是初級代謝產物中的一種。而[次級代謝產物則不會接參與那些過程](https://zh.wikipedia.org/wiki/次級代謝產物 "wikilink")，但它們在生理功能上具有重要的意義。它們囊括了[抗生素和色素](https://zh.wikipedia.org/wiki/抗生素 "wikilink")，像是[樹脂和](https://zh.wikipedia.org/wiki/樹脂 "wikilink")[松烯](../Page/萜烯.md "wikilink")。[放線菌素就是一種次級代謝產物](https://zh.wikipedia.org/wiki/放線菌素 "wikilink")，它們是從一級代謝產物的[色胺酸轉變而來的](https://zh.wikipedia.org/wiki/色胺酸 "wikilink")，但不是所有的抗生素都會把一級代謝產物當前體來使用。在代謝途徑中，[醣類中的](https://zh.wikipedia.org/wiki/醣類 "wikilink")[葡萄糖](../Page/葡萄糖.md "wikilink")和[果糖](../Page/果糖.md "wikilink")就屬於[代謝產物](../Page/代謝產物.md "wikilink")。

分子生物工業中生產的初級代謝產的例子\[1\]

| 類別   | 例子               |
| ---- | ---------------- |
| 酒精   | 乙烯               |
| 胺基酸  | 谷胺酸 、 天門冬胺酸      |
| 核苷酸  | 5'-磷酸鳥苷          |
| 抗氧化劑 | 異抗壞血酸            |
| 有機酸  | 乙酸 、 乳酸          |
| 多元醇  | 甘油               |
| 維生素  | 維生素B<sub>2</sub> |

來自一處的[酶](../Page/酶.md "wikilink")[化学反应](../Page/化学反应.md "wikilink")的输出成為另一些化学反应的输入，這樣環環相扣而成的[代谢物组](../Page/代谢物组.md "wikilink")（Metabolome）會形成一個龐大的代謝網絡。

在生物體本生的或來自藥物的合成物，形成代謝產物後，會成為降解與消除反應的自然的生物化学过程的一部分\[2\]。化合物本身的降解速率，決定它存在的長短及影響強弱。分析醫藥產品的代謝過程，藥物代謝，在藥物尋找的重要方法，並增進對藥物副作用的暸解。

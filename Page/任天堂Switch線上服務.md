是[任天堂](../Page/任天堂.md "wikilink")為[任天堂Switch](../Page/任天堂Switch.md "wikilink")推出的一項網路服務。從2017年3月3日主机發售日開始免費试运营，在2018年9月19日開始正式收費运营。\[1\]目前公布的收费方案有三种，以日本地区为例分別為1個月300日圓、3個月800日圓與12個月2400日圓。

## 功能

最初免費試營運任天堂Switch線上服務期間，可让用户進行網路連線遊戲和語音聊天。這種線上服務類似於[微軟推出的](https://zh.wikipedia.org/wiki/微軟 "wikilink")[Xbox Live](../Page/Xbox_Live.md "wikilink")。\[2\]\[3\]

在服務正式收费运营後，多人連線遊玩將被納入付費服務，此外還會追加透過智慧型手機App與好友相約連線遊玩或語音聊天的功能，任天堂經典老遊戲選集「經典遊戲選輯（Classic Game Selection）」的下载，以及提供付費會員限定的折扣等。而好友的登錄與管理、畫面截圖的社群網路服務分享、以及家長管理監控功能等服務則開放给所有用户，不論是否付费皆可使用。\[4\]

| 功能                                                        | 加入在线服务者 | 未加入者 |
| --------------------------------------------------------- | ------- | ---- |
| 联机游戏                                                      | 可       | 不可   |
| 使用《Nintendo Switch Online》App                             | 可       | 不可   |
| Nintendo Switch用游戏软件“经典游戏精选<small>（暂定名称）</small>”         | 可       | 不可   |
| 加入在线服务者限定折扣                                               | 可       | 不可   |
| 好友的登陆和管理                                                  | 可       | 可    |
| 在社交网络服务分享游戏截图                                             | 可       | 可    |
| 使用《任天堂家长守护Switch<small>（Nintendo みまもり Switch）</small>》App | 可       | 可    |

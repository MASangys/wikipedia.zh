[Iran_(orthographic_projection).svg](https://zh.wikipedia.org/wiki/File:Iran_\(orthographic_projection\).svg "fig:Iran_(orthographic_projection).svg") **伊朗標準時間**（**IRST**）或**伊朗時間**（**IT**）為[伊朗所採用的時間標準](https://zh.wikipedia.org/wiki/伊朗 "wikilink")。伊朗採用協調世界時[UTC+03:30](https://zh.wikipedia.org/wiki/UTC+03:30 "wikilink")。伊朗標準時間為藉由伊朗所在的[經線東經](https://zh.wikipedia.org/wiki/經線 "wikilink")52.5度而定義出。

時任伊朗總統[马哈茂德·艾哈迈迪内贾德](../Page/马哈茂德·艾哈迈迪内贾德.md "wikilink")下令，伊朗在2005年至2008年期間不採用[夏令時間](https://zh.wikipedia.org/wiki/夏令時間 "wikilink")（IRDT，伊朗夏令時）\[1\]\[2\]。之後伊朗夏令時間於2008年3月21日重新採用。

## 夏令時間

伊朗與其他國家不同的是，夏令時間並無根據其規則來實施轉換，如9月第三個星期一等。通常大部分國家的夏令時間每年皆選在週日施行，而伊朗則否\[3\]。一般而言，以春分後一天開始，以秋分後一天結束。

| 年    | 開始實行          | 結束實行          |
| ---- | ------------- | ------------- |
| 2014 | 3月22日星期六00:00 | 9月22日星期一00:00 |
| 2015 | 3月22日星期日00:00 | 9月22日星期二00:00 |
| 2016 | 3月21日星期一00:00 | 9月21日星期三00:00 |
| 2017 | 3月22日星期三00:00 | 9月22日星期五00:00 |
| 2018 | 3月22日星期四00:00 | 9月22日星期六00:00 |

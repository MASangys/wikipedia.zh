**伊瓦加語**(Iwaidja、/Iwaidja/在[音位](../Page/音位.md "wikilink")上拼讀為/Iwaja/)為一[澳大利亚](../Page/澳大利亚.md "wikilink")最北之[原住民語言](../Page/澳大利亚原住民语言.md "wikilink")，有大約150人還在使用著。歷史上曾在[科堡半岛上通行著](https://zh.wikipedia.org/wiki/科堡半岛 "wikilink")，目前則在在[克魯克島上流通](https://zh.wikipedia.org/wiki/克魯克島 "wikilink")。不過還有小孩子的學講人口。

## 語義結構

|                        |        |            |      |
| ---------------------- | ------ | ---------- | ---- |
| a                      | \-pana | \-maɽjarwu | \-n  |
| *我-對-他*                | 未來式    | *是…的父親*    | 名詞後綴 |
| "我未來的小孩" (字義"我將是他的父親") |        |            |      |

|                            |         |        |
| -------------------------- | ------- | ------ |
| ɹi                         | \-maka  | \-ntuŋ |
| *他-對-她*                    | *是…的丈夫* | 過去式    |
| "他的 (前/後) 妻子" (字義"他是她的前夫") |         |        |

**优麒麟**（）是[Ubuntu](../Page/Ubuntu.md "wikilink")的官方衍生版本之一 \[1\]。

## 背景

早在2010年，Ubuntu
官方就多次要推出Ubuntu中文定制版，并且将Ubuntu的[簡體中文名确定为友帮拓](https://zh.wikipedia.org/wiki/簡體中文 "wikilink")，但一直影响力较小。\[2\]

2013年3月21日，[工信部软件与集成电路促进中心和](https://zh.wikipedia.org/wiki/工信部 "wikilink")[中国人民解放军国防科技大学](../Page/中国人民解放军国防科技大学.md "wikilink")与Ubuntu的支持公司[Canonical在](https://zh.wikipedia.org/wiki/Canonical公司 "wikilink")[北京联合创立](https://zh.wikipedia.org/wiki/北京 "wikilink")“CCN开源软件创新联合实验室”，Ubuntu
Kylin正式创立，第一个版本随Ubuntu
13.04同时推出\[3\]。\[4\]Ubuntu的创始人[马克·沙特尔沃思](../Page/马克·沙特尔沃思.md "wikilink")称将其之为
Ubuntu 中文定制版的“接班人”，将提供一系列适合中国用户的软件应用。\[5\]

### KylinOS

虽然Ubuntu
Kylin与[KylinOS的名字相近](https://zh.wikipedia.org/wiki/KylinOS "wikilink")，但并没有直接关系\[6\]。
Ubuntu Kylin 本质上是 Ubuntu 的官方中国定制版本，尽管有政府的介入，但是它并不能取代军用操作系统。

## Ubuntu Kylin 13.04

根据官方 WIKI 的信息\[18\]\[19\]，Ubuntu Kylin 13.04 主要特性如下。

  - 採用版本3.8的Linux核心。
  - 採用Ubuntu Kylin主题样式。

### 特色应用

  - 中国音乐搜索（Unity Music Scope for China）：在线中文音乐搜索。
  - 中国农历软件（Chinese
    Calendar）：[农历](https://zh.wikipedia.org/wiki/农历 "wikilink")，包含[公历](https://zh.wikipedia.org/wiki/公历 "wikilink")。
  - 中国天气软件（China Weather
    Indicator）：提供来自中国气象局的天气预报信息，包含[PM<sub>2.5</sub>數據](../Page/懸浮粒子.md "wikilink")。
  - Ubuntu Kylin 軟件中心（Ubuntu Kylin Software Center）：Ubuntu
    Kylin特有的軟體商店，可以下載並安裝新的應用程式，或是管理已下載的應用。
  - 小企鹅输入法（Fcitx Input
    Method）：默认使用[fcitx输入法](https://zh.wikipedia.org/wiki/fcitx "wikilink")，支持[云拼音输入](../Page/雲端運算.md "wikilink")。
  - WPS办公套件Ubuntu Kylin版（WPS for Ubuntu Kylin）：提供中国本土办公软件[WPS
    Office](../Page/WPS_Office.md "wikilink")，兼容多种文档格式。

## Ubuntu Kylin 14.04 LTS

2014年4月18日，新版 Ubuntu Kylin 随 Ubuntu 及其他官方衍生版一并发布，是 Ubuntu Kylin
的首个长期支持版。

  - 採用版本 3.13 的 Linux 核心。
  - 採用版本 7.2 的 Unity 桌面。（新版本提供了下一代 Ubuntu 用户界面 Unity 8 的预览选项。）

### Ubuntu Kylin 应用

  - [搜狗输入法](http://pinyin.sogou.com/linux) (Sogou Pinyin for
    Linux)：搜狗再次與國內 Linux
    團隊合作重新開發的拼音輸入法，支持智能糾錯、長詞聯想、模糊音及皮膚等中國用戶習慣的功能。

允许字词语句自动补全，通过与具有内置互联网搜索提示功能的搜狗输入法合作。

  - [优客助手](https://launchpad.net/youker-assistant)：协助Windows用户换到Ubuntu操作系统的工具。
  - 智能搜索（Smart Scopes）：本地化智能搜索。图片搜索来源于搜狗在线；视频搜索来源于优酷视频；音乐搜索来源于百度。
  - [中国传统农历](https://launchpad.net/chinese-calendar)（Chinese
    Calendar）：[农历](https://zh.wikipedia.org/wiki/农历 "wikilink")，包含[公历](https://zh.wikipedia.org/wiki/公历 "wikilink")。
  - [金山WPS办公软件](https://zh.wikipedia.org/wiki/WPS_for_linux "wikilink")：中国较受欢迎的办公套件。
  - [莲花即时通信](https://launchpad.net/lotus)：由国科大开发。可以用它和亲朋好友聊天，也可以在办公环境下与同事联络。
  - 金山快盘云存储：提供个人云存储空间，通过网络或是兼容设备读取文件和内容。

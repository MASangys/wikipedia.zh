**伪满建国忠灵庙旧址**，是日本帝国主义侵华重要罪证遗址之一，位於[吉林省](../Page/吉林省.md "wikilink")[长春市](../Page/长春市.md "wikilink")[朝陽區](https://zh.wikipedia.org/wiki/朝阳区_\(长春市\) "wikilink")。建国忠灵庙是日本傀儡政權[滿洲國政府設立的](https://zh.wikipedia.org/wiki/滿洲國 "wikilink")[廟宇](https://zh.wikipedia.org/wiki/廟宇 "wikilink")，由逾14万名青少年無償參與修建，在1940年9月18日落成。該庙是[建国神庙的](https://zh.wikipedia.org/wiki/建国神庙 "wikilink")[摄庙](https://zh.wikipedia.org/wiki/摄庙 "wikilink")，佔地45.6万平方米，用以祭祀戰死的[日本](../Page/日本.md "wikilink")和滿洲國軍人，旨在強化日本人的民族意識、同時削弱當地人的反抗意識，被喻為「[东北地区最大的](https://zh.wikipedia.org/wiki/东北地区 "wikilink")[靖国神社](../Page/靖国神社.md "wikilink")」。1987年，伪满建国忠灵庙旧址被列入第四批[吉林省文物保護單位](https://zh.wikipedia.org/wiki/吉林省文物保護單位 "wikilink")；旧址現位於[中國人民解放軍空軍航空大學內](https://zh.wikipedia.org/wiki/中國人民解放軍空軍航空大學 "wikilink")，用作倉庫。2019年10月16日，国务院公布其为第八批全国重点文物保护单位。 \[1\]

## 歷史

### 建造原因及過程

1932年，[大日本帝國攻佔](https://zh.wikipedia.org/wiki/大日本帝國 "wikilink")[中國東北地區](https://zh.wikipedia.org/wiki/中國東北地區 "wikilink")，宣佈在當地成立[满洲国](../Page/满洲国.md "wikilink")，首都為[长春](https://zh.wikipedia.org/wiki/长春 "wikilink")；1934年，清遜帝[溥仪](../Page/溥仪.md "wikilink")被日本扶植成為[满洲国皇帝](https://zh.wikipedia.org/wiki/满洲国皇帝 "wikilink")，长春改名为[新京](../Page/新京.md "wikilink")\[2\]。

日本[关东军](../Page/关东军.md "wikilink")在1935年首先建議設立「招魂社」，及後「满洲国招魂社建设准备委员会」成立\[3\]。有關决定案在1936年通过，委员会更名為「护国庙建设委员会」，開始建設「护国庙」，即後來的建国忠灵庙；满洲国軍政部最高顧問[佐佐木擔任委员长](https://zh.wikipedia.org/wiki/佐佐木 "wikilink")，[滿洲國國務院](../Page/滿洲國國務院.md "wikilink")總務廳官員[相贺兼介擔任幹事](https://zh.wikipedia.org/wiki/相贺兼介 "wikilink")，满洲国国务院建筑局负责设计及监理\[4\]。日本强行在满洲国推崇[神道教](https://zh.wikipedia.org/wiki/神道教 "wikilink")，企图把神道教奉為中國東北的[國教](../Page/國教.md "wikilink")，又急欲控制當地人民的思想，在關東軍的指揮下加緊步伐興建忠灵庙\[5\]\[6\]。對日本人來說，修建此庙後可以通過祭祀來強化他們的[民族意識](https://zh.wikipedia.org/wiki/民族意識 "wikilink")，並增強民族自豪感；對滿洲國統治下的國民來說，宗教思想可用作[同化](../Page/同化.md "wikilink")，減低他們反抗日本的意識\[7\]。

由於新京（长春）附近一带是[九一八事變](../Page/九一八事變.md "wikilink")後首個激战後攻陷的城市，因此护国庙選擇設於新京\[8\]。护国庙在1936年動工，由[东条英机等官員举行奠基仪式](https://zh.wikipedia.org/wiki/东条英机 "wikilink")，不久後因滿洲國国务院总务厅指示而改为「建国庙」；廟宇在1940年9月18日完工，但由於关东军在[七七事变](../Page/七七事变.md "wikilink")後把絕大部分财力用於侵华战争，因此中门、园林等的一些設施最終沒有落成，当局又因缺乏[黄金而以黄漆代替](https://zh.wikipedia.org/wiki/黄金 "wikilink")[金箔](../Page/金箔.md "wikilink")\[9\]。工程总投资了160万[滿洲國圓](../Page/滿洲國圓.md "wikilink")，共動用了17万人次，逾14萬名青少年無償參與，時序如下\[10\]\[11\]\[12\]\[13\]：

| 年份                 | 工程項目                                       |
| ------------------ | ------------------------------------------ |
| 1936年              | 展開第一期工程，建设本殿和庙务所。                          |
| 改變建築角度，因而被迫扩大佔地面积。 |                                            |
| 1937年              | 展開第二期工程，建设祭殿、灵殿、禁域空间、水池。                   |
| 1938年              | 展開第三期工程，建设配殿、角楼、神门、回廊、中门、洗手所、前门、侧门、中门、西门等。 |

### 建国忠灵庙的使用

建国忠灵庙由[满洲国祭祀府管理](https://zh.wikipedia.org/wiki/满洲国祭祀府 "wikilink")，用作供奉為满洲国而戰死的日本和满洲国官员、军人、警察，因而被後世中国學者評論為「[靖国神社](../Page/靖国神社.md "wikilink")的满洲版」、「東北地區最大的靖國神社」\[14\]\[15\]\[16\]。廟內共供奉了24,141名被譽為「日滿英靈」的陣亡者，包括[武藤信義](../Page/武藤信義.md "wikilink")以下的19,877名日本人，以及[朱家訓以下的](https://zh.wikipedia.org/wiki/朱家訓 "wikilink")4,264名[滿洲國軍軍人](https://zh.wikipedia.org/wiki/滿洲國軍 "wikilink")\[17\]。日本人提出，[建國神廟供奉的](https://zh.wikipedia.org/wiki/建國神廟 "wikilink")[天照大神是](https://zh.wikipedia.org/wiki/天照大神 "wikilink")「满洲帝国建国之元神」，建国忠灵庙里的祭祀對象則是满洲国成立時出现的护国之神，地位低於天照大神；「护国之神」要护卫天照大神，因此建国忠灵庙成為建國神廟的攝廟\[18\]\[19\]。

根據統計，满洲国政府在建国忠灵庙举行了约20次大祭、近10次中祭，以及数次小祭；其中，1940年9月18日（即廟宇落成當天，亦是九一八事变紀念日）舉行的开庙祭是规模最大的一次，歷時五天，溥仪亦有出席，廟外舉行了默祷和遥拜儀式\[20\]。满洲国政府曾頒布[敕令](https://zh.wikipedia.org/wiki/敕令 "wikilink")，规定建国忠灵庙舉行的祭祀分为大祭、中祭和小祭三类，各項祭祀如下\[21\]\[22\]：

满洲国政府頒佈了《對於建國神廟及其攝廟不敬罪處罰法》，禁止對建国忠灵庙的不敬行為，違者可被判囚七年以下\[23\]。為了麻醉人民的思想，满洲国統治下的青少年学生被要求每天行[鞠躬礼](https://zh.wikipedia.org/wiki/鞠躬 "wikilink")，遥拜滿洲國皇帝溥儀、建国忠灵庙、以至[日本天皇](https://zh.wikipedia.org/wiki/日本天皇 "wikilink")\[24\]。满洲国政府又以建国忠灵庙的各項祭祀來向官吏灌输「尽忠报国」思想，而满洲国最高学府[建国大学的校歌亦有](https://zh.wikipedia.org/wiki/建国大学_\(满洲国\) "wikilink")「建国忠灵庙，是我们的归宿」這句歌词\[25\]。

### 隨滿洲國滅亡而棄用

由於关东军多次战败、滿洲國軍又受到抗日力量袭击，建国忠灵庙祭祀的陣亡軍人數量急剧上升，由1942年春的28,917升至1944年秋的40,850人；同時，滿洲國政府因戰況不利和經濟問題而不能组织大规模的祭拜，因此後来的祭拜规模日益減少，參加人數也大減\[26\]。1945年，[苏联](../Page/苏联.md "wikilink")向日本宣战，进攻中國東北的关东军，满洲国將近灭亡；為了毀灭证據，日本人燒燬建国神庙，但建国忠灵庙只是宣傳工具，不影响满洲国的国体，因此未被毀壞\[27\]。

## 結構

建国忠灵庙是一座建筑群，佔地42.98平方米，建筑方向依据日本[伊势神宫](../Page/伊势神宫.md "wikilink")的方位\[28\]。該廟入口朝西北方，從而順應关东军「庙之方位将来之帝宫」的主張；在祭祀活动中，參加者向东南方参拜，朝向伊势神宫供奉的天照大神\[29\]。建筑群採用中国传统的内向型院落布局，把主要建筑排列在中轴线上，令建筑群具備内向性和均衡性；建筑轴线上呈高差变化，相同於《[清式营造则例](https://zh.wikipedia.org/wiki/清式营造则例 "wikilink")》提出的四合寺观平面布置方法\[30\]。

建国忠灵庙前门的後方有一道长约700米、宽56米的曲线参道，参道两侧有洗手所和焚香炉\[31\]。庙內的建筑空间從外而内有外庭、内庭和禁域，三個空间的地位向内递增\[32\]。外庭长160米、宽130米，佔地2.08平方米，能容纳约15,000人，供日本军人、学生和社会团体参拜；經過「神门」可由外庭进入内庭，内庭是边长83米、佔地6,831平方米的正方形廊院式空间，为建国忠灵庙建筑群的核心部分，可容纳3,000人；禁域以灵殿为中心，长68米、宽34米，被认为是建国忠灵庙最神圣的空间，只允許神职人员進入\[33\]。

## 保護

滿洲國滅亡、日本戰敗投降後，中國爆發[国共内战](../Page/国共内战.md "wikilink")，长春发生多次激烈的战事，並多次易手，但[中國國民黨](../Page/中國國民黨.md "wikilink")与[中国共产党](../Page/中国共产党.md "wikilink")均有保护建国忠灵庙旧址\[34\]。内战结束後，建国忠灵庙旧址一带被划入军事院校控制区，旧址划归[中国人民解放军空军航空大学](../Page/中国人民解放军空军航空大学.md "wikilink")（當時名為解放军空军第一预科总队）；因此，建国忠灵庙旧址受到保护，沒有在[大跃进](../Page/大跃进.md "wikilink")和[文化大革命](../Page/文化大革命.md "wikilink")時受到嚴重的破壞，直至1980年代仍是完好\[35\]。雖然如此，建国忠灵庙旧址至今出現了不少损毀，例如本殿墙壁和天花的装饰被火燒燬，外墙在文化大革命被喷漆；旧址年久失修，而且民眾缺乏保护意识，亦造成其他的損坏\[36\]。

1987年10月，[吉林省人民政府](../Page/吉林省人民政府.md "wikilink")公布第四批[吉林省文物保護單位名单](https://zh.wikipedia.org/wiki/吉林省文物保護單位 "wikilink")，把建國忠靈廟舊址以「伪满建国忠灵庙遗址」的名義列為其一，歸類為「日本帝国主义侵华重要罪证遗址」；省政府並指示組織有關部門為包括建国忠灵庙在內的文物保护单位制定保护措施，例如划定保护范围、作出标志说明\[37\]。目前，建国忠灵庙旧址被用作仓库，旧址正门前的空間則成为居民休憩的广场\[38\]。建国忠灵庙旧址的文物保護工作受其归属权问题制肘，因為旧址雖然是文物保護單位，但仍是军队财产，文物管理单位與旧址擁有者存在分歧；加上管理单位没有专业人員和资金，暫時未有修复计划\[39\]。

**佐保明梨**（；）是[日本偶像藝人](https://zh.wikipedia.org/wiki/日本偶像 "wikilink")，為女子偶像團體成員。原[hello\! Project Egg](https://zh.wikipedia.org/wiki/hello!_Project_Egg "wikilink")、[共生植樹隊](https://zh.wikipedia.org/wiki/共生植樹隊 "wikilink")、[守護甜心EGG\!](../Page/守護甜心EGG!.md "wikilink")以及[あぁ\!的成員](https://zh.wikipedia.org/wiki/あぁ! "wikilink")。

## 經歷

### 2004年

加入Hello\! Project Egg。

### 2005年

7月23日-7月24日 - 在「代代木第一体育館」舉行的『Hello\! Project 2005 夏の歌謡ショー～'05 セレクション\!コレクション\!～』東京公演中，第一次出演，在演唱會中伴舞。

### 2007年

  - 5月13日- 在「渋谷C.C.レモンホール」舉行的「第1回Hello！Project新人公演」，與[光井愛佳](../Page/光井愛佳.md "wikilink")、純純、琳琳、[有原栞菜](../Page/有原栞菜.md "wikilink")（[℃-ute](../Page/℃-ute.md "wikilink")）、THE ポッシボー、其他Hello\! Project Egg成員，共25人一起演出。
  - 8月24日　在「Cutie Circuit 2007 ～MAGICAL CUTIE感謝祭～」的運動會中演出。
  - 8月26日 - 在「横浜BLITZ」舉行的「第2回Hello\! Project新人公演」與THE ポッシボー、其他Hello\! Project Egg成員一起演出。
  - 10月7日、10月8日 - 與古峰桃香一起參加「モーニング娘。“熱っちぃ地球を冷ますんだっ。”文化祭2007 in 横浜のエコバスツアー」。
  - 11月23日 - 在「品川ステラボール」舉行的「第3回Hello\! Project新人公演」中，與其他Hello\! Project Egg成員一起演出。
  - 12月10日 - 加入共生植樹隊，代替湯徳歩美。
  - 12月24日 - 在東京王子飯店鳳凰の間舉行的「五木宏聖誕晚餐會2007」中，與古川小夏、[前田憂佳](../Page/前田憂佳.md "wikilink")、[福田花音](../Page/福田花音.md "wikilink")、[小川紗季](../Page/小川紗季.md "wikilink")一起演出。

### 2008年

  - 3月8日 - 在「サンストリート亀戸」舉行的「ハロプロエッグデリバリーステーション！03」，與澤田由梨、[前田憂佳](../Page/前田憂佳.md "wikilink")、[福田花音](../Page/福田花音.md "wikilink")、古峰桃香等人一起演出。
  - 3月29日 - 在「横浜BLITZ」舉行的「2008 Hello！Project新人公演 3月 ～キラメキの横浜～」中，與其他Hello\! Project Egg成員一起演出。
  - 4月20日 - 在「Berryz工房&℃-ute仲良しバトルコンサートツアー2008春～Berryz仮面vsキューティーレンジャー～」中，與古川小夏、森咲樹、西念未彩、古峰桃香、岡井明日菜、[小川紗季](../Page/小川紗季.md "wikilink")、和田彩花、関根梓、新井愛瞳等人一起演出。
  - 6月22日 -在「赤坂BLITZ」舉行的「2008ハロー！プロジェクト新人公演6月～赤坂HOP！～」，與其他Hello\! Project Egg成員一起演出。
  - 8月9日 - 在「08夏 ナイスガールプロジェクト！エモーショナルライブ～ミラクル セクシー ダイナマイト～」中，與古川小夏、森咲樹、古峰桃香、和田彩花、前田憂佳、福田花音、小川紗季等人一起演出。
  - 8月11日 - 在「パシフィックヘブン」舉行的「ファンクラブ Hello\! Project 10周年記念·ハロプロエッグ ファンクラブ限定イベント」中，與田中杏里、前田憂佳、[福田花音](../Page/福田花音.md "wikilink")、前田彩里、新井愛瞳等人一起演出。
  - 9月13日 - 9月15日 - 預定在「全電通ホール」舉行的つんく♂THEATER第6弾「あぁ 女子合唱部～栄光のかけら2008～」中，與キャナァーリ倶楽部、古川小夏、森咲樹、[前田憂佳](../Page/前田憂佳.md "wikilink")、[福田花音](../Page/福田花音.md "wikilink")、小川紗季等人一起演出。
  - 9月23日 - 預定在「東京メルパルクホール」舉行的「2008 ハロー\!プロジェクト新人公演 9月 ～芝公園STEP～」中，與其他Hello\! Project Egg成員一起演出。

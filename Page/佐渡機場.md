**佐渡機場**（，）是位於[新潟縣](https://zh.wikipedia.org/wiki/新潟縣 "wikilink")[佐渡市](../Page/佐渡市.md "wikilink")（[佐渡島](https://zh.wikipedia.org/wiki/佐渡島 "wikilink")）。根據日本空港整備法被分成第三種機場。

## 航空管制

|        |           |            |
| ------ | --------- | ---------- |
| 種類     | 頻率        | 使用時間(JST)  |
| REMOTE | 118.15MHz | 8:45-17:15 |

## 航空公司與航點

目前無定期航班，2014年3月31日之前曾開行往來[新潟機場](../Page/新潟機場.md "wikilink")的定期航班。

## 如何前往

  - 在[兩津港乘](https://zh.wikipedia.org/wiki/兩津港 "wikilink")[出租車](../Page/出租車.md "wikilink")要約10分鐘。

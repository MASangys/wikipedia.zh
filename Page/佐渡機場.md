**佐渡機場**（，）是位於[新潟縣](../Page/新潟縣.md "wikilink")[佐渡市](../Page/佐渡市.md "wikilink")（[佐渡島](../Page/佐渡島.md "wikilink")）。根據日本空港整備法被分成第三種機場。

## 航空管制

|        |           |            |
| ------ | --------- | ---------- |
| 種類     | 頻率        | 使用時間(JST)  |
| REMOTE | 118.15MHz | 8:45-17:15 |

## 航空公司與航點

目前無定期航班，2014年3月31日之前曾開行往來[新潟機場的定期航班](../Page/新潟機場.md "wikilink")。

## 如何前往

  - 在[兩津港乘](../Page/兩津港.md "wikilink")[出租車要約](../Page/出租車.md "wikilink")10分鐘。

[右](https://zh.wikipedia.org/wiki/File:MorphSyntAlignSP-A.png "fig:右")
**作通型配列**（），也称为**作通格配列**、**施通格配列**、**作格配列**、**施格配列**或**能格配列**，是一類[配列方式](../Page/配列.md "wikilink")，即在[句法或](../Page/句法.md "wikilink")[形态上](../Page/词法学.md "wikilink")，将[不及物动词的单一变元和及物动词的](../Page/不及物動詞.md "wikilink")[受事](https://zh.wikipedia.org/wiki/受事 "wikilink")[论元同等对待](../Page/论元.md "wikilink")（**S=P**），而区别对待[及物动词的](../Page/及物動詞.md "wikilink")[施事论元](../Page/施事.md "wikilink")（**A**）\[1\]。作通型配列可以在形态（编码性质）和句法（表现性质）两方面呈现\[2\]\[3\]，一方面在[综合语中常藉由](../Page/综合语.md "wikilink")[格标记来表示](../Page/格_\(语法\).md "wikilink")；另一方面，一些语言的[句法也会呈现作通型配列](../Page/句法.md "wikilink")，但是十分罕见。\[4\]\[5\]此外，句法上作通型语言通常不是中心词前置就是中心词后置，且罕见[SVO语序](https://zh.wikipedia.org/wiki/主動賓語序 "wikilink")。\[6\]呈现这种配列的语言称为**作通格语言**或者**作格语言**、**施格语言**、**能格语言**。[巴斯克語](../Page/巴斯克語.md "wikilink")、[格鲁吉亚语](../Page/格鲁吉亚语.md "wikilink")、[玛雅语](https://zh.wikipedia.org/wiki/玛雅语 "wikilink")、[藏语](https://zh.wikipedia.org/wiki/藏语 "wikilink")，和一些[印欧语如](../Page/印欧语系.md "wikilink")[库尔德语](https://zh.wikipedia.org/wiki/库尔德语 "wikilink")、[印地语等](https://zh.wikipedia.org/wiki/印地语 "wikilink")，甚至[闪语族的](../Page/闪米特语族.md "wikilink")[阿拉姆语里面都存在这种配列](https://zh.wikipedia.org/wiki/阿拉姆语 "wikilink")。有些假说里面[原始印欧语也被划为作通型语言](https://zh.wikipedia.org/wiki/原始印欧语 "wikilink")，但仍十分有争议。\[7\]

## 编码性质

### 格标记

如果一门语言有形态上的格变化，那么谓语中心词的论元一般会如下标记：

  - 不及物动词的单一核心变元（**S**）和及物动词的受事论元（**P**）一同标记为[通格](../Page/通格.md "wikilink")（**S=P**）；
  - 动词的施事论元（标记为**A**）则区别对待，将它标记为[作格](../Page/作格.md "wikilink")。\[8\]

### 动词一致

如果没有形态上的格变化，作通型配列还能通过其他形态手段呈现，如动词的显在[一致](../Page/一致_\(語法\).md "wikilink")。例如，[阿布哈兹语和大部分](https://zh.wikipedia.org/wiki/阿布哈兹语 "wikilink")[玛雅语没有形态上的作格标记](https://zh.wikipedia.org/wiki/玛雅语 "wikilink")，但是它们可以通过动词的一致结构来呈现作格。在作通一致的语言里，通格往往不被标记（有些例外，如尼亚斯语和\[9\]），如[巴斯克語](../Page/巴斯克語.md "wikilink")\[10\]：

|     |         |        |            |
| --- | ------- | ------ | ---------- |
| 1a. | Martin  | \-Ø    | etorri da. |
|     | 马丁      | \-ABS  | 到达.PERF    |
|     | **S**   | **通格** |            |
|     | 马丁已经到了。 |        |            |

|     |          |        |       |        |           |
| --- | -------- | ------ | ----- | ------ | --------- |
| 1b. | Martin   | \-ek   | Diego | \-Ø    | ikusi du. |
|     | 马丁       | \-ERG  | 迪亚戈   | \-ABS  | 见.PERF    |
|     | **A**    | **作格** | **P** | **通格** |           |
|     | 马丁见过迪亚戈。 |        |       |        |           |

上例的 *-Ø* 是一个，代表不被标记的通格。作格标记在[辅音后作](../Page/辅音.md "wikilink")
*-ek*，而在[元音后作](../Page/元音.md "wikilink")
*-k*。另外，巴斯克语里面大多数名词短语都需要由一个[限定词限定](https://zh.wikipedia.org/wiki/限定词 "wikilink")，一般[单数为](https://zh.wikipedia.org/wiki/單數_\(語法\) "wikilink")
*-a*，[复数为](../Page/複數_\(語法\).md "wikilink")
*-ak*；[数只能标记在限定词上](../Page/数_\(语法\).md "wikilink")。于是便有如下变位（以“人”*gizon*
为例）\[11\]：

|     gizon      | 通格             | 作格       |
| :------------: | -------------- | -------- |
|       单数       | gizon-a        | gizon-ak |
| 人-DET.SING.ABS | 人-DET.SING.ERG |          |
|       复数       | gizon-ak       | gizon-ek |
|  人-DET.PL.ABS  | 人-DET.PL.ERG   |          |

## 表现性质

作通型配列也可以通过[句法呈现](../Page/句法.md "wikilink")。但是作通型句法配列非常罕见，大多数作通格语言都是形态上而非句法上呈作通型配列。与[词法学一样](../Page/词法学.md "wikilink")，句法配列可以放在一个连续统一体上，因此某些句法操作可能会生成[主宾型配列](../Page/主宾型配列.md "wikilink")，而其他的则可以生成作通型配列；其程度取决于有多少句法操作将论元
S 和 P 同等对待。

理论上，作通型配列可以在[语序](https://zh.wikipedia.org/wiki/语序 "wikilink")（尤其是论元与动词的前后关系）、[句法枢纽](https://zh.wikipedia.org/wiki/句法枢纽 "wikilink")、[关系子句](../Page/關係子句.md "wikilink")、[主從複合](../Page/主从复句.md "wikilink")、[转换指称等方面呈现](https://zh.wikipedia.org/wiki/转换指称 "wikilink")。

### 等位省并

一些作通格语言的配列可以通过等位省并体现。如澳洲原住民的[迪尔巴尔语](../Page/迪爾巴爾語.md "wikilink")，省略後句同定论元时，只有论元
S 和 P 可以省略，论元 A 不可以省略。\[12\]

|       |                    |        |                       |                      |        |                      |        |        |
| ----- | ------------------ | ------ | --------------------- | -------------------- | ------ | -------------------- | ------ | ------ |
| 2a.   | ŋuma<sub>i</sub>   | \-Ø    | banaga-n<sup>y</sup>u | yabu                 | \-ŋgu  | ____<sub>i</sub> |        | bura-n |
|       | 父亲                 | \-ABS  | 来-PST                 | 母亲                   | \-ERG  |                      |        | 见-PST  |
|       | **S**              | **通格** |                       | **A**                | **作格** | **P**                | **通格** |        |
|       | 父亲回来了，然后母亲见到了（父亲）。 |        |                       |                      |        |                      |        |        |
| 2b.\* | ŋuma<sub>i</sub>   | \-Ø    | banaga-n<sup>y</sup>u | ____<sub>i</sub> |        | yabu                 | \-Ø    | bura-n |
|       | 父亲                 | \-ABS  | 来-PST                 |                      |        | 母亲                   | \-ABS  | 见-PST  |
|       | **S**              | **通格** |                       | **A**                | **作格** | **P**                | **通格** |        |
|       | 父亲回来了，然后（父亲）见到了母亲。 |        |                       |                      |        |                      |        |        |

反之如[英语](../Page/英语.md "wikilink")（[主宾型](../Page/主宾型配列.md "wikilink")），省略後句同定论元时，只有论元
S 和 A 可以省略，论元 P 不可以省略。

|       |                    |                       |     |                      |       |                       |
| ----- | ------------------ | --------------------- | --- | -------------------- | ----- | --------------------- |
| 3a.   | Father<sub>i</sub> | returned              | and | ____<sub>i</sub> | saw   | mother.               |
|       | 父亲                 | 回来-PST                |     |                      | 见.PST | 母亲                    |
|       | **S 主格**           |                       |     | **A 主格**             |       | **P 宾格**              |
|       | 父亲回来了，然后母亲见到了（父亲）。 |                       |     |                      |       |                       |
| 3b.\* | Father<sub>i</sub> | banaga-n<sup>y</sup>u | and | mother               | saw   | ____<sub>i</sub>. |
|       | 父亲                 | 回来-PST                |     |                      | 见.PST |                       |
|       | **S 主格**           |                       |     | **A 主格**             |       | **P 宾格**              |
|       | 父亲回来了，然后（父亲）见到了母亲。 |                       |     |                      |       |                       |

但实际上，缺乏形态手段、主要靠语序表达句法功能的语言大部分都属于[主宾型语言](https://zh.wikipedia.org/wiki/主宾型语言 "wikilink")，作通型语言极其罕见，迪尔巴尔语算是一个特例。\[13\]\[14\]如果观察主语的一些普遍特征，会发现有些语言无论在形态学还是句法学上都是完美的主宾格语言，但完美的作通格语言却未曾被发现。\[15\]

## 部分作格性

大部分作通型语言都具有[部分作格性](../Page/部份作格性.md "wikilink")（split
ergativity），即在某些状況下呈现作通型配列的特性，但在某些状況下，如特定的[人称](../Page/人称.md "wikilink")、[时](../Page/时态.md "wikilink")、[体等](../Page/体_\(语法\).md "wikilink")，却呈现主宾型配列等其他配列的特性。[巴斯克語是少有的](../Page/巴斯克語.md "wikilink")、在格标记和动词一致上呈现完整作通型配列的语言，不过它在句法配列上则是[主宾型的](../Page/主宾型配列.md "wikilink")。\[16\]

## 任意作格性

很多作通型语言也具有任意作格性（optional
ergativity），即作格标记不会在所有情况下强制出现。但任意作格性并非真正任意，作格显在标记与否受语义和语用影响。任意作格性不像部分作格性那样有规律。\[17\]

任意作格性会受以下因素影响\[18\]：

  - [有生性](../Page/有生性.md "wikilink")，有生的主语更倾向于标记作格
  - 动词语义，具有及物性的动词更倾向于标记作格
  - 语法结构，尤其是
    \[[时](../Page/时态.md "wikilink")-[体](../Page/体_\(语法\).md "wikilink")-[态](https://zh.wikipedia.org/wiki/态_\(语法\) "wikilink")\]

一些[澳大利亚原住民语言](../Page/澳大利亚原住民语言.md "wikilink")、[藏语等拥有任意作格性](https://zh.wikipedia.org/wiki/藏语 "wikilink")。\[19\]

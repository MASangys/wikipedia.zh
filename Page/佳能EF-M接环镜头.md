[Canon_EF-M_lineup.jpg](https://zh.wikipedia.org/wiki/File:Canon_EF-M_lineup.jpg "fig:Canon_EF-M_lineup.jpg")
**EF-M接口**是佳能公司随[佳能 EOS
M系列](../Page/佳能_EOS_M.md "wikilink")[无反相机而推出的新接口](https://zh.wikipedia.org/wiki/无反相机 "wikilink")，全电子触点设计，可以通过转接环支持[EF与](https://zh.wikipedia.org/wiki/佳能EF接环镜头 "wikilink")[EF-S镜头](https://zh.wikipedia.org/wiki/佳能EF-S接环镜头 "wikilink")。

## 镜头列表

[Canon_EF-M_f2.0_22mm.jpg](https://zh.wikipedia.org/wiki/File:Canon_EF-M_f2.0_22mm.jpg "fig:Canon_EF-M_f2.0_22mm.jpg")
目前在EF-M接口上，佳能原厂提供了7枚镜头\[1\]，腾龙提供1枚镜头\[2\]。

关于字母标示：

  - 标有“**STM**”的镜头含有步進馬達
  - 标有“**IS**”（全称“**i**mage **s**tabilize”）的镜头具有光学防手震特性

| 焦距       | 型式                                     | 发布时间         | 重量   | 备注                       |
| -------- | -------------------------------------- | ------------ | ---- | ------------------------ |
| 11-22mm  | 佳能 EF-M 11-22mm f/4-5.6 IS STM         | 2013/06\[3\] | 220g |                          |
| 15-45mm  | 佳能 EF-M 15-45mm F3.5-6.3 IS STM        | 2015/10\[4\] | 130g | 随EOS M3/M10一同上市的轻便标准变焦镜头 |
| 18-55mm  | 佳能 EF-M 18-55mm f/3.5-5.6 IS STM       | 2012/10      | 210g | 第一批发售镜头                  |
| 18-150mm | 佳能 EF-M 18-150mm f/3.5-6.3 IS STM\[5\] | 2016/09      | 210g |                          |
| 18-200mm | 腾龙 18-200mm F3.5-6.3 Di III VC         | 2014/06      |      | 腾龙公司发布的兼容镜头              |
| 55-200mm | 佳能 EF-M 55-200mm f/4.5-6.3 IS STM      | 2014/06      | 260g |                          |
| 22mm     | 佳能 EF-M 22mm f/2 STM                   | 2012/10      | 105g | 随初代EOS M发售               |
| 28mm     | 佳能 EF-M 28mm f/3.5 STM\[6\]            | 2016/05      | 130g | 內建環形補光燈                  |
| 32mm     | 佳能 EF-M 32mm F1.4 STM\[7\]             | 2018/09      | 235g |                          |

其中，佳能 EF-M 22mm F2 STM
属于[饼干镜头](../Page/饼干镜头.md "wikilink")，且素质表现优异，受到许多用户喜爱\[8\]。

**來母**（****）是[中古漢語的一個](../Page/中古漢語.md "wikilink")[聲母](../Page/聲母.md "wikilink")，屬[次濁音](../Page/次濁.md "wikilink")，在[三十六字母中稱作](../Page/三十六字母.md "wikilink")[半舌音](../Page/半舌音.md "wikilink")。該聲母可與所有四個[等配合](../Page/等_\(音韻\).md "wikilink")，但二等字很少，三等沒有[重紐](../Page/重紐.md "wikilink")。

其例字如下：

|                       |                     |                       |                       |
| --------------------- | ------------------- | --------------------- | --------------------- |
| 一等                    | 二等                  | 三等                    | 四等                    |
| 羅la、老laux、亂luanh、落lak | 斕lren、冷lrangx、犖lruk | 劉liu、吕liox、淚lyih、立lip | 靈leng、禮lex、練lenh、歷lek |

## 擬音

由於在大多數漢語方言中，來母字均爲舌尖邊音。東亞普遍缺乏[邊音和](../Page/邊音.md "wikilink")[閃音的對立](../Page/閃音.md "wikilink")，日、朝語的來母字對應閃音，而梵漢對音中來母可對應l或者r。目前學者一致將來母的中古音擬爲舌尖邊音。

## 現代方言、語言中的讀音

在[北京音系和多數北方官話中](../Page/北京音系.md "wikilink")，來母字絕大多數聲母均爲l
，且聲母爲l的絕大多數來自于來母。粵語標準語、吳語中來母也均爲，且與n聲母分開。在部分長江流域的[官話等方言](../Page/官話.md "wikilink")，及[粵語新近的](../Page/粵語.md "wikilink")[懶音現象中](../Page/懶音.md "wikilink")，來母與來自[泥母](../Page/泥母.md "wikilink")、[娘母的n聲母字全部或部分相混](../Page/娘母.md "wikilink")，有時都變成，有時都變成，有時都成爲鼻化的邊音，還有些[洪音相混而](../Page/洪音.md "wikilink")[細音分開的](../Page/細音.md "wikilink")。

[朝鮮語中](../Page/朝鮮語.md "wikilink")，來母對應聲母的閃音ㄹ。但在[大韓民國標準語中按照](../Page/大韓民國標準語.md "wikilink")[頭音法則](../Page/頭音法則.md "wikilink")，只是不在詞首時穩定，詞首的閃音會不穩定。一二等字在詞首時變成鼻音ㄴ，而三、四等字經常和泥母四等、娘母三等字一樣脫落聲母成爲零聲母ㅇ。如二等字“冷”(lrangx)本爲랭(raeng)，但在“冷藏庫”（冰箱）一詞中成爲냉(naeng)。而“李”(lix)本爲리(ri)，在作姓時因在詞首而成爲이(i)，但在英文翻譯中仍按來源寫作Lee。

[日語中不論漢音吳音](../Page/日語.md "wikilink")，均爲ら行字（閃音）。由於日語固有詞不允許閃音在詞首，ら行字開頭的詞一般都是漢字詞、外來詞或象聲詞。

[越南語的](../Page/越南語.md "wikilink")[漢越語部分中](../Page/漢越語.md "wikilink")，來母字均爲l聲母，而[古漢越語中](../Page/古漢越語.md "wikilink")，來母字對應r聲母（通音或顫音）。

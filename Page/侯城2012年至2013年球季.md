**侯城2012年至2013年球季**（）是[赫尔城足球俱乐部在](https://zh.wikipedia.org/wiki/赫尔城足球俱乐部 "wikilink")[英格蘭足球聯賽第](https://zh.wikipedia.org/wiki/英格蘭足球聯賽 "wikilink")97個球季，是於[2010年從](https://zh.wikipedia.org/wiki/2009年至2010年英格蘭超級聯賽 "wikilink")[英超降班後第](https://zh.wikipedia.org/wiki/英超 "wikilink")3個處於第二級聯賽的球季，上季以第8位結束球季。

## 記事

  - 2013年5月4日鹽聯賽煞科日主場對篤定冠軍的[卡迪夫城](../Page/卡迪夫城足球俱乐部.md "wikilink")，雖然完場前被對手憑[十二碼扳平](https://zh.wikipedia.org/wiki/十二碼 "wikilink")2-2，但競爭對手[-{zh-hans:沃特福德;
    zh-hk:屈福特;}-以](../Page/沃特福德足球俱乐部.md "wikilink")1-2負於[-{zh-hans:利兹联;
    zh-hk:列斯聯;}-](https://zh.wikipedia.org/wiki/列斯聯足球會 "wikilink")，以兩分壓倒對方取得亞軍席位和最後一張直接升班入場劵\[1\]。

## 球衣

本季球衣繼續由[愛迪達](../Page/愛迪達.md "wikilink")製造及供應，主場球衣維持傳統的琥珀色和黑色，但改變一貫的直條款式而採用黑肩琥珀衫黑褲琥珀襪\[2\]
；作客球衣為全黑款式\[3\]；保留去季的淺藍衫白褲淺藍襪作客球衣為第三球衣。主要[贊助](https://zh.wikipedia.org/wiki/贊助 "wikilink")[廣告](../Page/廣告.md "wikilink")是「Cash
Converters」。  |  |  |  |}

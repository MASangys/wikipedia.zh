**侯马晋国遗址**，位于[中国](https://zh.wikipedia.org/wiki/中国 "wikilink")[山西](https://zh.wikipedia.org/wiki/山西 "wikilink")[侯马市](../Page/侯马市.md "wikilink")[汾河](../Page/汾河.md "wikilink")、[浍河交汇处](https://zh.wikipedia.org/wiki/浍河 "wikilink")\[1\]。

遗址由山西省文物管理委员会在1954年调查发现，1956年、1957年发掘古城，后又多次发掘。被认是[晋国晚期](https://zh.wikipedia.org/wiki/晋国 "wikilink")[都城新田](https://zh.wikipedia.org/wiki/都城 "wikilink")（新绛）的遗址\[2\]。1961年，[中国国务院公布为](../Page/中华人民共和国国务院.md "wikilink")[第一批全国重点文物保护单位](../Page/第一批全国重点文物保护单位.md "wikilink")。

## 概述

[侯马晋国古城平面图.svg](https://zh.wikipedia.org/wiki/File:侯马晋国古城平面图.svg "fig:侯马晋国古城平面图.svg")
侯马晋国遗址包括晋国古城遗址、铸铜遗址、祭祀遗址、盟誓遗址、宗庙建筑遗址和墓地\[3\]。

| 遗址名称   | 现存文物                                                               |
| ------ | ------------------------------------------------------------------ |
| 古城遗址   | 宫殿台基；牛村、平望、台神、马庄、呈王、北坞等六座东周古城                                      |
| 铸铜遗址   | 十处祭祀遗址，以马、牛、羊祭祀为主                                                  |
| 盟书遗址   | 盟书五千余件，记载[晋国贵族集团之间斗争](https://zh.wikipedia.org/wiki/晋国 "wikilink") |
| 宗庙建筑遗址 | 一百余处大型夯土建筑基址                                                       |
| 墓地     | 上马、下平望、东高三处古墓群；柳泉墓地为晋公室墓区                                          |

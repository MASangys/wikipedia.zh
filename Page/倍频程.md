**倍频程**（），又称为**倍频带**，是振动与噪音分析中将整个[频谱](../Page/频谱.md "wikilink")划分为若干[频带的方法](https://zh.wikipedia.org/wiki/频带 "wikilink")，每个频带的上限频率是下限频率的两倍。类似地，1/3倍频程（one-third octave band）指每个频带的上限频率与下限频率之比是2的[立方根](../Page/立方根.md "wikilink")。

## 倍频程

### 计算

假设每个频带的中心频率为\(f_c\)，该频带的上限频率与下限频率可由下式计算：

\[f_c = \sqrt{2} f_{min} = \frac{f_{max}}{\sqrt{2}}\],

其中\(f_{min}\)为下限频率，\(f_{max}\)为上限频率。

## 1/3倍频程

### 以2为基数

``` matlab
%% Calculate Third Octave Bands (base 2) in Matlab
fcentre  = 10^3 * (2 .^ ([-18:13]/3))
fd = 2^(1/6);
fupper = fcentre * fd
flower = fcentre / fd
```

### 为10为基数

``` matlab
%% Calculate Third Octave Bands (base 10) in Matlab
fcentre = 10.^(0.1.*[12:43])
fd = 10^0.05;
fupper = fcentre * fd
flower = fcentre / fd
```

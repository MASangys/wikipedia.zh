*偽亂數二進位數列*（*Pseudo Randomness Binary
Sequence*）簡稱*PRBS*，是一種特別的[二進位數列](../Page/位元流.md "wikilink")\(a_0,\ldots, a_{N-1}\)，若二進位數列的位元數為*N*，其中為1的數字有*m*'個，則其其[自相关函数](../Page/自相关函数.md "wikilink")：

\[C(v)=\sum_{j=0}^{N-1} a_ja_{j+v}\]

只有以下二個值：

\[C(v)=
\begin{cases}
m, \mbox{ if } v\equiv 0\;\; (\mbox{mod}N)\\
\\
mc, \mbox{ otherwise }
\end{cases}\]

其中

\[c=\frac{m-1}{N-1}\]

稱為偽亂數二進位數列的占空比，類似連續時間信號的[占空比](../Page/占空比.md "wikilink")。

偽亂數二進位數列稱為[偽亂數](https://zh.wikipedia.org/wiki/偽亂數 "wikilink")，雖然它是決定性的，不過其\(a_j\)的數值和前後元素的數值無關，看似隨機的，因此稱為偽亂數。

偽亂數二進位數列可以延伸到無限長，方式是在\(N\)個元素都出現過之後，再從\(a_0,\ldots, a_{N-1}\)再出現一次……，這點和真正的由[放射性衰減或](../Page/放射性.md "wikilink")[白雜訊產生的數列不同](../Page/白雜訊.md "wikilink")，後者在本質上就是無限長的。偽亂數二進位數列比[最大長度數列更普遍](../Page/M-sequence.md "wikilink")，後者是特別的N位元偽亂數二進位數列，是由線性移位暫存器所產生的。最大長度數列的占空比恆為50%，長度為k位元的暫存器，其數列長度為\(N = 2^k-1\)。偽亂數二進位數列可以用在[電信](https://zh.wikipedia.org/wiki/電信 "wikilink")、[密碼學及](https://zh.wikipedia.org/wiki/密碼學 "wikilink")[模擬等應用](https://zh.wikipedia.org/wiki/模擬 "wikilink")。

## 實際的實現

[PRBS_15_generator.png](https://zh.wikipedia.org/wiki/File:PRBS_15_generator.png "fig:PRBS_15_generator.png")後，做為新的位元\]\]

偽亂數二進位數列可以用[线性反馈移位寄存器產生](../Page/线性反馈移位寄存器.md "wikilink")\[1\]。

一些常見的的數列產生多項式為

PRBS7 = \(x^{7} + x^{6} + 1\)

PRBS15 = \(x^{15} + x^{14} + 1\)

PRBS23 = \(x^{23} + x^{18} + 1\)

PRBS31 = \(x^{31} + x^{28} + 1\)

以下是一個用 PRBS-7 產生偽亂數二進位數列的C語言程式

``` c
#include <stdio.h>
#include <stdint.h>
#include <stdlib.h>

int main(int argc, char* argv[]) {
    uint8_t start = 0x02;
    uint8_t a = start;
    int i;
    for(i = 1;; i++) {
        int newbit = (((a >> 6) ^ (a >> 5)) & 1);
        a = ((a << 1) | newbit) & 0x7f;
        printf("%x\n", a);
        if (a == start) {
            printf("repetition period is %d\n", i);
            break;
        }
    }
}
```

此例中，PRBS-7的週期為127位元。

**偽關係**（，又稱為**虛假關係**），指在兩個沒有[因果關係的事件](https://zh.wikipedia.org/wiki/因果關係 "wikilink")，可能基於其他未見的**[干擾因素](https://zh.wikipedia.org/wiki/干擾因素 "wikilink")**（；或稱**潛在變數**，），顯示出[統計學上的相關](https://zh.wikipedia.org/wiki/統計學 "wikilink")，让人很容易猜想「兩個事件有所聯繫」，然而這種聯繫並不能通過更加精細的檢驗。

另外，伪相关现象出现的另一种情形是在变數個數很多但抽样样本量不大时，即使这些变量互相独立，也有很大可能得出變數之間高相關性的偽關係。

另一個造成偽關係的原因是對[對撞因子進行控制](https://zh.wikipedia.org/wiki/對撞因子 "wikilink")，這包括選擇偏誤、[倖存者偏差](../Page/倖存者偏差.md "wikilink")、使用了不當的迴歸模型等情況。

## 相關係數與偽關係

兩個變量可能會受第三者影響，從而在其間得出誤導性的[相關係數](https://zh.wikipedia.org/wiki/相關係數 "wikilink")。當變量
A 和 B 有相關時，有幾種可能：

1.  A → B
2.  A ← B
3.  A ← C → B（C 是[干擾變量](https://zh.wikipedia.org/wiki/干擾變量 "wikilink")）
4.  C → A → B（C是[工具變量](https://zh.wikipedia.org/wiki/工具變量 "wikilink")）
5.  A → C → B（C 是[中介變量](https://zh.wikipedia.org/wiki/中介變量 "wikilink")）
6.  A → C ← B（C 是[對撞變量](https://zh.wikipedia.org/wiki/對撞變量 "wikilink")）

前兩種情況是實際的關係，無法區分因果。第三種情況是偽關係，如果能控制住C變數，A和B就不再相關。然而在第四種及第五種情況中，在控制住 C
之後，也可能造成 A 和 B 的關係消失，而誤以為是偽關係。而第六種情況中，控制了C反而會產生原本不存在的關係。

## 示例

  - 例一
    統計研究發現，冰淇淋銷量最高的時候，就是公共泳池的溺水事故發生得最多的時候。

然而，有可能熱浪造成冰淇淋銷量和公共泳池的溺水事故增多。若視冰淇淋的銷量或遇溺事故為對方的成因，可能就被偽關係誤導了。

  - 例二
    荷蘭的統計數字顯示，在一連串的春季中，鸛鳥巢的數目與人類嬰兒出生數目之間呈現正相關。

兩者之間未必有因果關係。事實上，它們都和數据觀測之前9個月的天氣相關\[1\]。

  - 例三
    [跨性別科學家](../Page/跨性別.md "wikilink")[本·巴雷斯死於胰臟癌](../Page/本·巴雷斯.md "wikilink")，這搞不好是因為他為了[變性而長期施打男性賀爾蒙所致](../Page/變性.md "wikilink")。

兩者間未必有因果關係，事實上，注射賀爾蒙不影響癌症發生的機率。\[2\]

  - 例四
    高度[民主](../Page/民主.md "wikilink")、注重[法治的國家大多富裕繁榮](../Page/法治.md "wikilink")，可見制度對經濟有決定性的影響。

然而，有可能是其他的因素同時導致了民主、法治和富裕，像是根植文化的[工作倫理或](https://zh.wikipedia.org/wiki/工作倫理 "wikilink")[民族性](https://zh.wikipedia.org/wiki/民族性 "wikilink")。

  - 例五
    「[夏以](../Page/夏朝.md "wikilink")[妹喜](https://zh.wikipedia.org/wiki/妹喜 "wikilink")，[殷以](../Page/商朝.md "wikilink")[妲己](../Page/妲己.md "wikilink")，[周以](../Page/周朝.md "wikilink")[褒姒](../Page/褒姒.md "wikilink")，三代所由亡也」（[晉](https://zh.wikipedia.org/wiki/晉 "wikilink")‧[杜預](https://zh.wikipedia.org/wiki/杜預 "wikilink")《[左傳](https://zh.wikipedia.org/wiki/左傳 "wikilink")》注）

然而，有可能朝代滅亡和寵幸[美女是因為別的因素](../Page/美女.md "wikilink")，如君王本身的性格傾向等，所造成的。若將美女的出現與朝代的滅亡視為對方的成因，可能就被偽關係誤導了。

## 實驗

偽關係一詞常用於[統計學上](https://zh.wikipedia.org/wiki/統計學 "wikilink")，特別是在[實驗的測定結果上](https://zh.wikipedia.org/wiki/實驗 "wikilink")。實驗一般用以檢測如「X
→ Y」的因果關係。然而兩者的共同因素(W → X & Y)可造成非因果關係的相關。另外，如「X → W →
Y」形的中介變數若沒有察覺出來的話，兩者非直接的關係便會看似是直接的。因此，除非能排除偽關係的可能性，否則實驗得出的相關並不代表因果關係。

實際上，下列三個條件都要成立，才可以得出X導致Y的結論：

  - X發生在Y之前
  - 若X不發生則Y也不發生
  - 若X發生則Y一定發生

如果上面三項中任何一項不符合，就可以確認出偽關係。

至於間接的因果關係，則不需要上列的第三項條件。例如，手槍決鬥中，兩個男人面對面，向對方開火。若其中一人擊中對方，而對方死亡，則可以推斷出他導致對方死亡。但是，若醫生把受傷的男人救回（不符合第三項條件），這沒有把因果關係終止，而只終止了直接的因果關係。由於開火X而導致身體受傷W，從而導致死亡Y。因為醫生的救治，終止了從W至Y的關係。

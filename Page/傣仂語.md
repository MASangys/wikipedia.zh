**傣語**（，kam tai lue；或**傣泐語**）屬於[侗台语族](https://zh.wikipedia.org/wiki/侗台语族 "wikilink")[台語支](https://zh.wikipedia.org/wiki/台語支 "wikilink")，泰族支系泐人的語言，是[泰語的一种方言](https://zh.wikipedia.org/wiki/泰語 "wikilink")，称为泰语西双版纳话，或泰语泐方言。有自己的文字——[老傣文](../Page/老傣文.md "wikilink")和素可泰文。使用人口分佈于[中國](../Page/中國.md "wikilink")（[西雙版納州和](../Page/西双版纳傣族自治州.md "wikilink")[江城县](https://zh.wikipedia.org/wiki/江城县 "wikilink")）30多萬，[緬甸](https://zh.wikipedia.org/wiki/緬甸 "wikilink")30萬，[老撾](https://zh.wikipedia.org/wiki/老撾 "wikilink")13萬，[泰國](https://zh.wikipedia.org/wiki/泰國 "wikilink")100萬，[越南](https://zh.wikipedia.org/wiki/越南 "wikilink")5千多。操該語言的族群自稱[泐](https://zh.wikipedia.org/wiki/泐 "wikilink")，中國官方認定為[傣族](../Page/傣族.md "wikilink")的一支，泰國認定為[泰族](../Page/泰族.md "wikilink")的一個支系——泰泐，越南認定為[盧族](../Page/傣仂.md "wikilink")；泰语泐方言和泰國北部的阮方言有很大的一致性。泰语泐方言是西雙版納傣族自治州的官方語言之一，又被中国稱爲**西雙版納傣語**。

## 文字

[老傣文](../Page/老傣文.md "wikilink")是一种[元音附标文字](../Page/元音附标文字.md "wikilink")，一般认为创制于13世纪，源自[缅文](../Page/缅文.md "wikilink")。创制之初仅用于宗教活动，被称为“经书文字”。这种文字结构复杂，难于认读。1954年在其基礎上修訂出[西双版纳新傣文](https://zh.wikipedia.org/wiki/西双版纳新傣文 "wikilink")。目前新老傣文並用。

素可泰文是西双版纳古代使用的文字，后来在缅甸扩张下，形态近似于缅文的经书文在缅甸势力范围的地区（兰纳、景栋和西双版纳）成为主流，素可泰文在西双版纳式微。

## 語音

傣仂語以[景洪市](../Page/景洪市.md "wikilink")口音為標準音。有21個[聲母](https://zh.wikipedia.org/wiki/聲母 "wikilink")，91個[韻母](https://zh.wikipedia.org/wiki/韻母 "wikilink")，和6個[聲調](../Page/聲調.md "wikilink")，[促聲只能是其中三個聲調](https://zh.wikipedia.org/wiki/促聲 "wikilink")，舒声則可以是所有六個聲調。本条目採用[新傣仂文](https://zh.wikipedia.org/wiki/新傣仂文 "wikilink")。

### 聲調

傣仂語有六個聲調，分高音和低音兩組，每組三個聲調。高音和低音用發音相同而字形不同的聲母來區別。

|     |              |                       |          |
| --- | ------------ | --------------------- | -------- |
|     | 無標記          | \-（但陽入和元音是/aː/的長陰入不標） | \-       |
| 高音组 | 高平55（陰平、短陰入） | 高升35（陰去、長陰入）          | 低升13（陰上） |
| 低音组 | 降51（陽平）      | 中平33（陽去、陽入）           | 低平11（陽上） |

[舒声韵可以是全部六个声调](https://zh.wikipedia.org/wiki/舒声 "wikilink")。[促聲只有三個聲調](https://zh.wikipedia.org/wiki/促聲 "wikilink")：短陰入、長陰入、陽入。韻母是、、的或者韻尾是的音节不可能是长陰入调；韻母是、、的音节不可能是短陰入调。

在和其他[侗台语比较时](https://zh.wikipedia.org/wiki/侗台语族 "wikilink")，通常按如下顺序排列声调：陰平、陽平、陰上、陽上、陰去、陽去、短陰入、陽入、長陰入，依次为第1至9调。

傣仂語的[現代漢語借詞一般借自](https://zh.wikipedia.org/wiki/現代漢語 "wikilink")[西南官話](https://zh.wikipedia.org/wiki/西南官話 "wikilink")。

|        |         |         |         |         |
| ------ | ------- | ------- | ------- | ------- |
| 西南官話聲調 | 陰平      | 陽平      | 上聲      | 去聲      |
| 傣语聲調   | 陽去 （33） | 陽上 （11） | 陽平 （51） | 陰去 （35） |

漢語入聲字在西南官話派入陽平，在傣仂語的現代漢語借詞中一般是陽上調（11），也可以是陰上調（13）。

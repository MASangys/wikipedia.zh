**傣黯語**，又称**黑傣语**，屬於[侗台语族](https://zh.wikipedia.org/wiki/侗台语族 "wikilink")[台語支](https://zh.wikipedia.org/wiki/台語支 "wikilink")，和[泰語有親屬關係](https://zh.wikipedia.org/wiki/泰語 "wikilink")。母語為傣黯語的总人口超过76万，近70万在越南，是[越南泰族的一支](https://zh.wikipedia.org/wiki/越南泰族 "wikilink")，称为[黑泰](https://zh.wikipedia.org/wiki/黑泰 "wikilink")，自称傣黯。中國境内人数不多，主要分布在[金平县](../Page/金平苗族瑶族傣族自治县.md "wikilink")，被中國官方归入傣族，称为黑傣，自称傣罗迷。

## 語音和文字

### 聲調

六個聲調，分高調和低調兩組，每組三個聲調。高調和低調用發音相同而字形不同的聲母來區別。每組内的三個聲調，通過在字母上不加標記、加標記“1”、加標記“2”來表示。

|      |          |           |            |
| ---- | -------- | --------- | ---------- |
|      | 無標記（調值）  | 上標"1"（調值） | 上標"2" （調值） |
| 低調字母 | 第1聲 （22） | 第2聲（45）   | 第3聲（21）    |
| 高調字母 | 第4聲（55）  | 第5聲（44）   | 第6聲 （31）   |

[入聲](../Page/入聲.md "wikilink")只能是第二聲和第五聲。

### 輔音

[<File:TaiDamConsonant.jpg>](https://zh.wikipedia.org/wiki/File:TaiDamConsonant.jpg "fig:File:TaiDamConsonant.jpg")

全部字母都可以做聲母。其中高調字母-m、-n、-ŋ、-ʔ、-p、-t、-k、-v（w）、-j、-f也用於做[韻尾](https://zh.wikipedia.org/wiki/韻尾 "wikilink")，-ʔ、-p、-t、-k作韻尾的時候是[入聲](../Page/入聲.md "wikilink")。

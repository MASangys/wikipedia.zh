**傳統黑人大學**（Historically black colleges and
universities，簡寫HBCUs）是指[美國](../Page/美國.md "wikilink")1964年前專為[黑人而設的](../Page/非裔美國人.md "wikilink")[高等教育機構](../Page/高等教育.md "wikilink")。\[1\]

美國現有105所此類大學，包括[公立及](../Page/公立學校.md "wikilink")[私立](../Page/私立學校.md "wikilink")、二年制及四年制、[醫學院及](../Page/醫學院.md "wikilink")[社區學院](../Page/社區學院.md "wikilink")。\[2\]除了六所以外（其中一所已關閉），它們全處於美國前[蓄奴州](../Page/蓄奴州.md "wikilink")。這些大學有不少存在百多年後，因競爭、[經濟大蕭條或其他經濟問題於](../Page/經濟大蕭條.md "wikilink")20世紀關閉。

## 歷史

美國在[南北戰爭前](../Page/南北戰爭.md "wikilink")，少數北部和[西部的大學](../Page/美國西部.md "wikilink")（如[歐柏林大學](../Page/歐柏林大學.md "wikilink")）已開始收個別的黑人學生，而北部更有三所黑人大學：1837年建立的[切尼賓州大學](../Page/切尼賓州大學.md "wikilink")（Cheyney
University of
Pennsylvania）、1854年建立的[林肯大學](../Page/林肯大學_\(賓夕凡尼亞州\).md "wikilink")（Lincoln
University）和1856年建立的[威爾伯福斯大學](../Page/威爾伯福斯大學.md "wikilink")（Wilberforce
University）。

南北戰爭期間的1863年，[摩利爾法案](../Page/土地撥贈法案.md "wikilink")（Morrill
Act）通過，規定美國每個州都需要有[贈地大學](../Page/贈地大學.md "wikilink")。南北戰爭後，法案擴展到[南部](../Page/美國南部.md "wikilink")，但該地17個州的贈地大學卻不肯收黑人學生。居住在南部的黑人雖然獲得自由，但經濟及教育處境卻沒甚麼改善。1890年，一條針對這些前[聯盟國成員的第二摩利爾法案由國會通過](../Page/美利堅聯盟國.md "wikilink")：這條法案要求受惠的州必須證明種族並不是入學的標準，否則該州必須另外成立有色種族專有的學校。大部分的傳統黑人學校都是因此建立的，而它們到現在仍獲聯邦政府資助它們的研究、進修教育及外展服務。

1965年，[高等教育法案](../Page/1965年高等教育法案.md "wikilink")（Higher Education Act
of 1965）成立了一個管理聯邦政府怎樣給此類大學資金的方案，包括對私人捐款的分配。\[3\]

美國現在雖然有其他某些教育機構亦有相當多的黑人學生，但因為它們是在[斯韋特訴佩因特案](../Page/斯韋特訴佩因特案.md "wikilink")（Sweatt
v.
Painter）及[布朗訴托皮卡教育局案和](../Page/布朗訴托皮卡教育局案.md "wikilink")[1965年高等教育法案後才建立或開始收黑人學生](../Page/1965年高等教育法案.md "wikilink")，它們並不可算是傳統黑人大學，只是主要黑人大學（predominantly
black）。

2001年開始，幾間傳統黑人大學的圖書館開始談論它們應如何合作並將資源聚集一起；到了2003年，談論終於有成果，而傳統黑人大學圖書館聯盟（HBCU
Library Alliance）——「一個支持資訊專業人士合作和為鞏固傳統黑人學校及它們成員而提供不同資源的組職」\[4\]——亦正式成立。

## 現狀

|                                                               |
| ------------------------------------------------------------- |
| 2004年，美國教育局發表了一項有關傳統黑人大學的研究，其中發現於2001年，有13%黑人大學生在此類大學就讀。\[5\] |

|                                                                                                                                                                              |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2007年，[瑟古德·馬歇爾大學基金會發表了一項有關](../Page/瑟古德·馬歇爾大學基金會.md "wikilink")[財富](../Page/財富_\(雜誌\).md "wikilink")400公司及政府機構對少數族裔的招聘習慣之研究，發現它們聘用的大學畢業生當中，有13%是由傳統黑人大學畢業，其餘的均是從主流大學畢業。\[6\] |

|                                         |
| --------------------------------------- |
| 2009年的「經濟刺激法案」中，有多於$13億是分配給傳統黑人大學。\[7\] |

美國現存的105所此類大學當中，有27所能頒發博士學位，能頒發碩士學位的有52所，學士學位的有83所，而副學士學位的則有38所。\[8\]

在學生的種族分布方面，在美國所有大學之中，黑人大學生從此類大學畢業的比率持續下降，從1976年的35%下降到2001年的21.5%\[9\]，但奇怪的是，這些大學的收生人數卻有增無減，從1976年的18萬上升到2001年的22萬，而其中大部分的增長是由於黑人女學生的入學人數，由8萬8千人上升到約38萬人\[10\]。

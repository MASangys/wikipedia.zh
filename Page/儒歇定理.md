在[数学](../Page/数学.md "wikilink")，特别是[复分析中](https://zh.wikipedia.org/wiki/复分析 "wikilink")，**儒歇定理**（）告诉我们如果[复值](https://zh.wikipedia.org/wiki/复数 "wikilink")[函数](../Page/函数.md "wikilink")
*f* 与 *g* 在一条闭[曲线](../Page/曲线.md "wikilink") *C*
内部及边界上[全纯](../Page/全纯函数.md "wikilink")，在 *C* 上满足
|*g*(*z*)| \< |*f*(*z*)|，则 *f* 与 *f* + *g* 在 *C*
内部[零点](../Page/零点.md "wikilink")个数相同，这里零点按[重数计算](https://zh.wikipedia.org/wiki/重数 "wikilink")。该定理假设曲线
*C* 是简单的，即没有自交点。

儒歇定理通常用于简化局部零点问题。给定一个解析函数，我们将其写为两部分，一部分比较简单且比增长要快于（从而控制）另一部分。例如，多项式
\(z^5 + 3z^3 + 7\) 在圆盘 \(|z| < 2\) 内恰有五个零点，因为对任何 \(|z| = 2\) 有
\(|3z^3 + 7| < 32 = |z^5|\)，并且控制部分 \(z^5\) 在圆盘内有五个零点。

这个定理以[法国数学家](https://zh.wikipedia.org/wiki/法国 "wikilink")[欧仁·儒歇](https://zh.wikipedia.org/wiki/欧仁·儒歇 "wikilink")（，1832-1910）命名，该定理1862年发表于《[综合理工大学校期刊](https://zh.wikipedia.org/wiki/巴黎综合理工大学校 "wikilink")》39期（**
39 (1862)）。

## 几何解释

可以给出一个非正式的解释，说明儒歇定理为何成立。

首先我们将定理稍微改写一下。令 *h*(*z*) = *f*(*z*) + *g*(*z*)，注意到 *f*，*g* 全纯（解析）意味着 *h*
也全纯。和上面的假设一样，则儒歇定理说

  -
    如果 |*f*(*z*)| \> |*h*(*z*) − *f*(*z*)|，则 *f*(*z*) 与 *h*(*z*) 在 *C*
    的内部有同样多零点。

注意到条件 |*f*(*z*)| \> |*h*(*z*) − *f*(*z*)| 意味着对任何 *z*，*f*(*z*) 与原点的距离大于
*h*(*z*) − *f*(*z*)
的长度。如图所示，这说明对任何蓝曲线上的点，连接该点与原点的线段大于相应的绿线段。非正式地，我们可以说红曲线
*h*(*z*) 总是比原点更接近蓝曲线 *f*(*z*)。

但上一段说明了由于 *f*(*z*) 恰好绕零点一圈，故 *h*(*z*)
同样如此，即两条曲线在零点的指标相同。由[辐角原理](../Page/辐角原理.md "wikilink")（），这意味着
*f*(*z*) 与 *h*(*z*) 的零点个数相同。

一个流行的、非正式的表述将如上讨论总结为：如果一个人用皮带牵着一条狗绕一棵树（视为一个点）不停转，且皮带的长度小于树的半径，则这个人与狗绕过这棵树的次数相等。（事实上，可以看到儒歇定理的[逆不成立](https://zh.wikipedia.org/wiki/逆命题 "wikilink")，只要皮带的长度小于树的周长。）

## 应用

考虑多项式 \(z^2 + 2az + b^2\)（这里
\(a > b > 0\)）。由[二次方程求根公式](../Page/一元二次方程.md "wikilink")，我们有两个零点
\(a \pm \sqrt{a^2 - b^2}\)。因为

  -
    对所有 \(|z| = b\)，有 \(|z^2 + b^2| \le 2b^2 < 2a|z|,\)

儒歇定理说此多项式在圆盘 \(|z| < b\) 内部恰有一个零点。因为 \(a + \sqrt{a^2 - b^2}\)
显然在圆盘外部，我们得出这个多项式有一个零点在 \(a - \sqrt{a^2 - b^2}\)
处。这种讨论在应用柯西[留数定理](../Page/留数定理.md "wikilink")处理局部留数时很有效。

儒歇定理也能用来给出[代数基本定理](../Page/代数基本定理.md "wikilink")一个简短证明。设
\(p(z) = a_0 + a_1z + a_2 z^2 + ... + a_n z^n\)，总可以选取实数 \(R\) 足够大，使得对所有
\(|z| = R,\)

\[|a_0 + a_1z + ... + a_{n-1}z^{n-1}| \le \sum_{j=1}^n |a_j| R^{n-1} < |a_n|R^n = |a_n z^n|.\]
因为 \(a_n z^n\) 在圆盘 \(|z| < R\) 内部有 \(n\) 个零点，由儒歇定理 \(p\) 在这个圆盘内部也有同样多零点。

这个证明与其它证明比较，一个优点是不仅证明了多项式至少有一个零点而且指出零点个数和多项式的次数相等（和通常一样计算重数）。

儒歇定理的另一个运用是用来证明解析函数的[开映射定理](https://zh.wikipedia.org/wiki/开映射定理_\(复分析\) "wikilink")（），具体证明参见该条目。

## 儒歇定理的证明

由假设，在 *C* 上 |*g*(*z*)| \< |*f*(*z*)|，蕴含

\[\left|\frac{f(z)+g(z)}{f(z)}-1\right| < 1\]，对所有 *z*∈*C*。

从而函数 *F*(*z*) = \[*f*(*z*)+*g*(*z*)\]/*f*(*z*) 将曲线 *C* 变为以 1 为圆心半径为 1
的圆盘内部一条曲线 *F*(*C*)。从而 *F*(*C*)
关于原点的[卷绕数](../Page/卷绕数.md "wikilink")是零。另一方面，由[辐角原理](../Page/辐角原理.md "wikilink")，这个卷绕数等于

  -
    \(0 = {1\over 2\pi i}\oint_C {F'(z) \over F(z)}\,dz=N_F(C)-P_F(C).\)

这里 *N*<sub>*F*</sub>(*C*) 是 *F* 在*C* 内部的零点个数，*P*<sub>*F*</sub>(*C*) 是
*C* 内部[极点个数](https://zh.wikipedia.org/wiki/极点 "wikilink")。故
*N*<sub>*F*</sub> = *P*<sub>*F*</sub>。但是 *F* 是两个全纯函数 *f*+*g* 与 *f* 在 *C*
内部的商，所以 *F* 的零点是 *f*+*g* 的零点，极点是 *f* 的零点。从而

\[0=N_F(C) - P_F(C) = N_{f+g}(C) - N_f(C),\]

命题得证。

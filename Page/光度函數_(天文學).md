**光度函數**在[天文學](../Page/天文學.md "wikilink")上為[恆星或](https://zh.wikipedia.org/wiki/恆星 "wikilink")[星系](../Page/星系.md "wikilink")的[光度](../Page/光度.md "wikilink")分佈\[1\]，是用來研究數量龐大的族群或類別的性質，像是[星團中的](https://zh.wikipedia.org/wiki/星團 "wikilink")[恆星或是](https://zh.wikipedia.org/wiki/恆星 "wikilink")[星系團中的](https://zh.wikipedia.org/wiki/星系團 "wikilink")[星系](../Page/星系.md "wikilink")。

## 謝克光度函數

**謝克光度函數**給出了以光度為變數的星系空間密度函數，其型式是：

\[n(x) \  \mathrm{d}x = \phi^* x^a \mathrm{e}^{-x} \mathrm{d}x,\] 此處\(x = L/L^*\)，\(L^*\)是星系的特徵光度，決定函數中冪律的適用範圍。\(\,\!\phi^*\)是歸一化常數，單位是數量密度。星系的光度函數在不同的族群和環境中可能會有不同的參數，它不是一個通用的函數。一個從場星系的測量中的估計是　\(a=-1.25,\ \phi^* = 1.2 \times 10^{-3} h^3 \mathrm{Mpc}^{-3}\)　。\[2\]

為了方便，謝克光度函數常被改寫成以[星等](../Page/星等.md "wikilink")為變數，並有以下形式；

\[n(M) \ \mathrm{d}M = 0.4 \ \ln 10 \ \phi^*  [ 10^{ 0.4 ( M^* - M ) } ]^{ \alpha + 1}  \exp [ -10^{ 0.4 ( M^* - M ) } ] \ \mathrm{d}M .\] 注意到因為星等是對數形式，冪律有著對數斜率\(\alpha + 1\)。這就是為什麼\(\alpha = -1\)的謝克光度函數是平的。

## 白矮星光度函數

**白矮星光度函數** (**white dwarf luminosity function，WDLF**) 給定[白矮星](../Page/白矮星.md "wikilink")的數量和光度。當在測量這類恆星的形成和冷卻的速率時，它提供有價值的資訊，像是白矮星冷卻的[物理和年齡](https://zh.wikipedia.org/wiki/物理 "wikilink")，以及[星系的歷史](https://zh.wikipedia.org/wiki/銀河 "wikilink")\[3\]\[4\]。

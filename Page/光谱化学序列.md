**光谱化学序列**由（紫外－可见）[光谱得到](https://zh.wikipedia.org/wiki/光谱 "wikilink")，用以表示[配体和金属离子对分裂能](https://zh.wikipedia.org/wiki/配体 "wikilink") Δ 的影响大小，主要应用在[晶体场理论](../Page/晶体场理论.md "wikilink")和[配位场理论](../Page/配位场理论.md "wikilink")中。分裂能是1个电子从较低能量*d*轨道跃迁到较高能量*d*轨道所需的能量。

光谱化学序列可以解释一些化合物的颜色差异。常见配体在光谱化学序列中的顺序列在下面，分裂能从小到大：

I<sup>−</sup> \< Br<sup>−</sup> \< S<sup>2−</sup> \< SCN<sup>−</sup> \< Cl<sup>−</sup> \< NO<sub>3</sub><sup>−</sup> \< N<sub>3</sub><sup>−</sup> \< F<sup>−</sup> \< OH<sup>−</sup> \< C<sub>2</sub>O<sub>4</sub><sup>2−</sup> ≈ **H<sub>2</sub>O** \< NCS<sup>−</sup> \< CH<sub>3</sub>CN \< py （[吡啶](../Page/吡啶.md "wikilink")） \< NH<sub>3</sub> \< en （[乙二胺](../Page/乙二胺.md "wikilink")） \< bipy （[2,2'-联吡啶](../Page/2,2'-联吡啶.md "wikilink")） \< phen （1,10-[邻菲啰啉](https://zh.wikipedia.org/wiki/邻菲啰啉 "wikilink")） \< NO<sub>2</sub><sup>−</sup> \< PPh<sub>3</sub> \< CN<sup>−</sup> ≈ CO

**H<sub>2</sub>O** 左边的配体称为**弱场**配体，右边的称为**强场**配体。

金属离子相同时,分裂能从小到大：

\[\rm Mn^{2+} < Ni^{2+} < Co^{2+} < Fe^{2+} < V^{2+} < Fe^{3+} < Co^{3+} < Mn^{4+} < Mo^{3+} < Rh^{3+} < Ru^{3+} < Pd^{4+} < Ir^{3+} < Pt^{4+}\]

[廣義相對論](../Page/廣義相對論.md "wikilink")中，**克爾度規**（）或稱**克爾真空**（），描述的一[旋轉](https://zh.wikipedia.org/wiki/自轉 "wikilink")、[球對稱之質量龐大物體](https://zh.wikipedia.org/wiki/球對稱 "wikilink")（例如：[黑洞](../Page/黑洞.md "wikilink")）週遭真空區域的時空幾何。其為，故又稱**克爾解**；廣義相對論的主導方程式——愛因斯坦場方程式是[非線性的](https://zh.wikipedia.org/wiki/非線性 "wikilink")，找出其精確解是相當困難的任務。

克爾[度規是](https://zh.wikipedia.org/wiki/度規 "wikilink")[史瓦西度規](../Page/史瓦西度規.md "wikilink")（1915年）的推廣，後者用以描述靜態不旋轉、球對稱且不帶電荷的龐大物體週遭真空區域的時空幾何。在有帶電荷的情形，史瓦西度規轉成[萊斯納-諾德斯特洛姆度規](https://zh.wikipedia.org/wiki/萊斯納-諾德斯特洛姆度規 "wikilink")（1916年–1918年）。和曾使用弱场近似方法得到过旋转轴对称球状物体度规的近似解。直到1963年方由[羅伊·克爾](../Page/羅伊·克爾.md "wikilink")提出精確解。\[1\]，但他并没有给出推导过程。1973年Schiffer等人给出了克尔度规的推导\[2\]。

克爾度規的帶電荷版本為[克爾-紐曼度規](https://zh.wikipedia.org/wiki/克爾-紐曼度規 "wikilink")（1965年），以上四個相關的解可整理為如下表格：

|                |                                                                       |                                                             |
| -------------- | --------------------------------------------------------------------- | ----------------------------------------------------------- |
|                | 不旋轉 (*J* = 0)                                                         | 旋轉 (*J* ≠ 0)                                                |
| 不帶電荷 (*Q* = 0) | [史瓦西度規](../Page/史瓦西度規.md "wikilink")                                  | **克爾度規**                                                    |
| 帶電荷 (*Q* ≠ 0)  | [萊斯納-諾德斯特洛姆度規](https://zh.wikipedia.org/wiki/萊斯納-諾德斯特洛姆度規 "wikilink") | [克爾-紐曼度規](https://zh.wikipedia.org/wiki/克爾-紐曼度規 "wikilink") |

其中*Q*代表物體所帶電荷，而*J*代表物體的自轉[角動量](https://zh.wikipedia.org/wiki/角動量 "wikilink")。

## 克尔度规的数学表示

若以[波以耳-林德奎斯特座標寫出克爾真空解](https://zh.wikipedia.org/wiki/波以耳-林德奎斯特座標 "wikilink")，則為：

\[\mathrm{d}s^2 = -\left(1-\frac{2Mr}{\Sigma}\right)\mathrm{d}t^2 -\frac{4aMr\sin^2\theta}{\Sigma}\mathrm{d}t\mathrm{d}\phi +\frac{\Sigma}{\Delta}\mathrm{d}r^2\]

\[+\ \Sigma \mathrm{d}\theta^2 + \left(\Delta+\frac{2Mr(r^2+a^2)} {\Sigma}\right) \sin^2\theta \mathrm{d}\phi^2\] 其中

\[\Sigma=r^2+a^2\cos^2\theta\],

\[\Delta=r^2-2Mr+a^2\],

  - *M*為旋轉物體質量；
  - *a*為自轉參數（spin parameter）或稱特定角動量（specific angular momentum），描述此物體的旋轉，與角動量*J*有關，關係式為*a* = *J*/*M*；
  - 所有的物理量採用[幾何單位](https://zh.wikipedia.org/wiki/幾何單位 "wikilink")：*c*=*G*=1。

當自轉參數*a*值為零，則表示物體無旋轉，克爾度規退化成[史瓦西度規](../Page/史瓦西度規.md "wikilink")。*a*=*M*的例子對應到最大旋轉程度的質量物體。

注意到：

  - 一般而言，波以耳-林德奎斯特徑向座標 *r* 並無簡單而直接、如同[徑向座標般的詮釋](../Page/极坐标系.md "wikilink")。
  - 「最大」旋轉程度指的是一[黑洞](../Page/黑洞.md "wikilink")可以存在的最大*a*值，而非旋轉質量物體可以具有的最大*a*值。

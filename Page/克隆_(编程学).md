在[计算机科学中](../Page/计算机科学.md "wikilink")，“克隆”指对一个[对象制作一个完全相同的副本](https://zh.wikipedia.org/wiki/对象_\(计算机科学\) "wikilink")，其常常在[原型程式設計中的](../Page/原型程式設計.md "wikilink")[编程范型或者](../Page/编程范型.md "wikilink")[面向对象程序设计](../Page/面向对象程序设计.md "wikilink")（OOP）中用到。

## 浅拷贝

在大多数编程语言中（但除了：[Ruby等一些语言](../Page/Ruby.md "wikilink")），例如`double`、`float`、`int`、`long`等一些[原始型別仅仅将它们的值存储于计算机内存中的某处](../Page/原始型別.md "wikilink")（通常为[调用栈中](https://zh.wikipedia.org/wiki/调用栈 "wikilink")）。给它进行一次简单赋值，你可以将变量的内容复制到另外一个变量中：

在Java或C++中进行基本类型的复制：

``` java
int original = 42;
int copy = 0;

copy = original;
```

许多OOP编程语言（包括[Java](../Page/Java.md "wikilink")、[D語言](../Page/D語言.md "wikilink")、[ECMAScript和](../Page/ECMAScript.md "wikilink")[C♯](../Page/C♯.md "wikilink")）利用对象引用。对象引用与其他语言中的指针类似，其使得对象可以被[指针所传递而完整的对象不被复制](../Page/指標_\(電腦科學\).md "wikilink")。

以下是在Java语言中的范例，当使用简单赋值“拷贝”一个对象时：

``` java
Object original = new Object();
Object copy = null;

copy = original; // 不复制对象仅仅复制其引用
```

这段代码并没有复制对象，变量“original”与“copy”实际上指向相同的对象。在C++中，以下是相同效果代码：

``` c
Object* original = new Object();
Object* copy = NULL;
copy = original;
```

这段代码指明复制对象的是一个“指针”，而非对象本身。

## 克隆过程

制造与对象相同的一个副本而非仅仅制造它的引用的过程被称为克隆。在大多数编程语言中，编程语言和库可以便利一些克隆的过程。在Java语言中，`Object`类包含了，此方法仅仅复制对象并返回一个指向被复制对象的引用。由于此方法位于`Object`类中，程序员得以在所有定义于Java的类中使用克隆方法（虽然为了正常使用，其需要根据其应用于不同的层级进行改写）。

在Java语言中进行对象的克隆：

``` java
Object originalObj = new Object();
Object copyObj = null;

copyObj = originalObj.clone(); // 复制对象并分配一个指向“copyObj”的引用
```

一般而言，C++中的对象像基本类型一样运行，所以复制一个C++对象可以使用“`=`”（分配）赋值运算符。C++语言中所有的类中存在一个默认的赋值运算符，但其效果可能被[运算符重载所影响](../Page/运算符重载.md "wikilink")。使用此方法时可能存在危险（参见）。防止对象切片的方法的方法之一是使用与Java所有的类中的`clone()`方法来实现，并使用指针（请注意C++中不存在自带的`clone()`方法）。

C++中对象克隆的示例：

``` cpp
Object originalObj;
Object copyObj(originalObj); // 创建名为copyObj的originalObj拷贝
```

C++中使用指针进行对象克隆的示例（防止对象切片请参见\[1\]）：

``` cpp
Object* originalObj = new Object;
Object* copyObj = nullptr;

copyObj = new Object(*originalObj); // 创建originalObj的拷贝并将其地址分配给copyObj
```

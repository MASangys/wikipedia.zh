**免税储蓄账户**（，簡稱TFSA，，簡稱CELI）是[加拿大](../Page/加拿大.md "wikilink")稅務局為個人設立的一個稅務優惠賬戶。在免稅儲蓄賬戶發生的投資收入，包括[資本利得和](https://zh.wikipedia.org/wiki/資本利得 "wikilink")[股息](../Page/股息.md "wikilink")等，都無需交稅。但是存入免稅儲蓄賬戶的金額不能用來扣減個人所得稅，這與[註冊退休儲存計劃有所不同](https://zh.wikipedia.org/wiki/註冊退休儲存計劃 "wikilink")。

雖然這個賬戶被命名為儲蓄賬戶，但是並不一定要是現金[存款](https://zh.wikipedia.org/wiki/存款 "wikilink")。與註冊退休儲存計劃一樣，免税储蓄账户可以包括現金存款、各種投資（例如[共同基金](../Page/共同基金.md "wikilink")、某些[股票](../Page/股票.md "wikilink")、[債券或](https://zh.wikipedia.org/wiki/債券 "wikilink")[擔保投資證等](https://zh.wikipedia.org/wiki/擔保投資證 "wikilink")。\[1\]

## 歷史

該賬戶由當時的聯邦[財政部長](https://zh.wikipedia.org/wiki/加拿大財政部長 "wikilink")[Jim Flaherty於](https://zh.wikipedia.org/wiki/Jim_Flaherty "wikilink")2008年在[2008年聯邦預算中提出](https://zh.wikipedia.org/wiki/2008年加拿大聯邦預算 "wikilink")。並於2009年1月1日起實施。\[2\]

這個舉措受到[C.D. Howe Institute的支持](https://zh.wikipedia.org/wiki/C.D._Howe_Institute "wikilink")：「這個稅務政策對加拿大人來說是個好消息。Flaherty先生和他的政府值得為這個新的計劃受到讚揚。」\[3\] 此外，[Canadian Federation of Independent Business](https://zh.wikipedia.org/wiki/Canadian_Federation_of_Independent_Business "wikilink")、\[4\][加拿大銀行家協會](https://zh.wikipedia.org/wiki/加拿大銀行家協會 "wikilink")、\[5\][滿地可銀行](../Page/滿地可銀行.md "wikilink")經濟學家Doug Porter、\[6\][加拿大商會](https://zh.wikipedia.org/wiki/加拿大商會 "wikilink")、\[7\]以及[加拿大納稅人聯盟](https://zh.wikipedia.org/wiki/加拿大納稅人聯盟 "wikilink")\[8\]也都支持這項稅務政策。

## 特徵

在加拿大年滿18歲、擁有社保編號的居民可以申請在銀行開設免稅賬戶進行儲蓄或投資。在該賬戶獲得的收入無需交稅。持有人可以在任何時間把錢提取出來。

### 供款額度

該賬戶有供款額度的限制，每年根據聯邦稅務局的指引累加。2013年之前為每年5,000加幣，2013年開始，根據通漲，額度被調整至5,500加幣。2015年，當時執政的保守黨將當年的額度提高至10,000加幣。2016年自由黨上台執政後，將當年的額度重新降低至5,500加幣。

截止到2016年，累計的存款額度為$46,500加幣。

| 年份        | TFSA年度限額     | 累計限額    |
| --------- | ------------ | ------- |
| 2009-2012 | $5,000       | $20,000 |
| 2013-2014 | $5,500       | $31,000 |
| 2015      | $10,000\[9\] | $41,000 |
| 2016      | $5,500       | $46,500 |
|           |              |         |

### 合資格投資

TFSA賬戶可以參與任何符合RRSP的投資，包括在合資格[交易所進行的公開股票交易](https://zh.wikipedia.org/wiki/交易所 "wikilink")、合資格的[私人公司](../Page/私人公司.md "wikilink")股票交易、某些[債務](https://zh.wikipedia.org/wiki/債務 "wikilink")，[分期付款收益](https://zh.wikipedia.org/wiki/分期付款收益 "wikilink")、共同基金、[房地產投資信託基金](https://zh.wikipedia.org/wiki/房地產 "wikilink")，[年金合同](https://zh.wikipedia.org/wiki/年金合同 "wikilink")，[權證](https://zh.wikipedia.org/wiki/權證 "wikilink")，[期權](../Page/期權.md "wikilink")，[註冊投資](https://zh.wikipedia.org/wiki/註冊投資 "wikilink")，特許單位，合作單位以及[存託憑證](https://zh.wikipedia.org/wiki/存託憑證 "wikilink")。\[10\]

### 債權人保護

當TFSA賬戶所有人破產或一項財務判決引致賬戶所有人受到不利自己的法律訴訟時，TFSA內的資產不受保護，不能免遭債權人對其的主張。這與RRSP不同。

### 超額供款

在一個自然年度的提款只能在下一個年度1月1日起增加可供款額度。一個年度的超額供款可能由於持有人誤以為提款後會立即增加當年可供款額度，而在當年再次存入該賬戶所致。一個年度的超額供款會導致稅務局的罰款。

2012年，[加拿大稅務局向](https://zh.wikipedia.org/wiki/加拿大稅務局 "wikilink")900萬TFSA賬戶中的74,000超額供款人寄發罰款通知。\[11\]

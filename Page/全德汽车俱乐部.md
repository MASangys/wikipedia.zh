**全德汽车俱乐部**（，**ADAC**），是[德国最大的交通](../Page/德国.md "wikilink")[协会](https://zh.wikipedia.org/wiki/协会 "wikilink")，其总部位于[慕尼黑](../Page/慕尼黑.md "wikilink")，是個非營利的法人組織。

直接或者通过附属公司，ADAC向[德国](../Page/德国.md "wikilink")[汽车](../Page/汽车.md "wikilink")、[摩托车和小型船只驾驶者提供服务](https://zh.wikipedia.org/wiki/摩托车 "wikilink")，同时印刷出版[地图和](../Page/地图.md "wikilink")[交通图](https://zh.wikipedia.org/wiki/交通图 "wikilink")。ADAC在[德国境内设立了多个交通安全中心](../Page/德国.md "wikilink")，其最早开始和最著名的服务是[交通事故救援](https://zh.wikipedia.org/wiki/交通事故 "wikilink")。协会所有交通救援车辆均被漆为黄色，[德国人称其为](https://zh.wikipedia.org/wiki/德国人 "wikilink")“黄色天使”。

此外，ADAC有一个[空中救援的部门](https://zh.wikipedia.org/wiki/空中救援 "wikilink")，为德国两个最大的[直升机救援机构之一](https://zh.wikipedia.org/wiki/直升机 "wikilink")，2009年共执行了45,700次飞行救援行动。

## 历史

[ADAC_Motorrad.JPG](https://zh.wikipedia.org/wiki/File:ADAC_Motorrad.JPG "fig:ADAC_Motorrad.JPG")
1903年5月24日，德国摩托车驾驶者联盟（）在[斯图加特诗尔铂酒店成立](https://zh.wikipedia.org/wiki/斯图加特 "wikilink")，随后于1911年改为现名。由于最后一任[德意志皇帝及](../Page/德意志皇帝.md "wikilink")[普鲁士国王](https://zh.wikipedia.org/wiki/普鲁士国王 "wikilink")[威廉二世的支持](https://zh.wikipedia.org/wiki/威廉二世_\(德國\) "wikilink")，[普鲁士鹰被选定为ADAC的最初的标志](https://zh.wikipedia.org/wiki/普鲁士鹰 "wikilink")。2003年，ADAC庆祝了其100岁的生日。

[纳粹时期](https://zh.wikipedia.org/wiki/纳粹 "wikilink")，所有全德汽车俱乐部、俱乐部均被合并到位于[国家社会主义汽车团](../Page/国家社会主义.md "wikilink")（）附近的的德意志自动车协会（）。

二战之后1946年，ADAC最先在[拜仁州被重新建立](https://zh.wikipedia.org/wiki/拜仁州 "wikilink")，然后1948年开始陆续在剩余的[西方占领区重新建立](https://zh.wikipedia.org/wiki/西方占领区 "wikilink")。

### 无私车运动

1972年，德国汽车事故死亡人数达到一个高峰，为惊人的20,000人每年（现在大约为5,000人每年）。1974年[石油危机中](https://zh.wikipedia.org/wiki/石油危机 "wikilink")，当时的ADAC主席弗朗茨·斯达德勒在德国第一个无车星期日运动三个月后，提出了无私家车运动。但遭到了协会内亲汽车成员的反对和抗议。

但这一运动间接的促进了德国汽车[环保运动的发展](https://zh.wikipedia.org/wiki/环保 "wikilink")。

  - 1938–1945: Fritz Junghans, 德意志自动车协会主席

  - 1948–1953: Hans Meyer-Seebohm
  - 1953–1964: Werner Endress
  - 1964–1972: Hans Bretz
  - 1972–1989: Franz Stadler
  - 1989–2001: Otto Flimm
  - 2001年至今: Peter Meyer

## 协会概况

截至2010年6月，ADAC拥有1700万会员，是[欧洲第一](../Page/欧洲.md "wikilink")，世界第三大的汽车协会，仅次于[美国汽车联合会](https://zh.wikipedia.org/wiki/美国汽车联合会 "wikilink")（）和[日本自动车联盟](https://zh.wikipedia.org/wiki/日本自动车联盟 "wikilink")（）\[1\]。

ADAC从下至上的组织结构如下：

  - 1850个基层协会
  - 18个地区协会
  - 管理委员会
  - 主席团
  - 全体代表大会

ADAC横向分为四个分公司，它们是：ADAC会员及经济服务事务公司、ADAC空中救援公司、黄色天使基金会和ADAC体育基金会。

2009年ADAC总共获得634.3百万[欧元的会费收入](../Page/欧元.md "wikilink")，其支出情况如下\[2\]：

| 项目   | 百万欧元      | 百分比  |
| ---- | --------- | ---- |
| 救援事务 | 308.6     | 49 % |
| 会员服务 | 143.2     | 22 % |
| 宣传事务 | 112.3     | 18 % |
| 行政费用 | 50.1      | 8 %  |
| 体育赛事 | 20.1      | 3 %  |
| *合计* | **634.3** |      |
|      |           |      |

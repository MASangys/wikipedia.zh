[缩略图](https://zh.wikipedia.org/wiki/File:Global_Terrorism_Index_2016_by_country.svg "fig:缩略图") **全球恐怖主義指數**（）為依據世界各國[恐怖活動的程度來排定其排名](https://zh.wikipedia.org/wiki/恐怖活動 "wikilink")。該指數結合了許多影響超過10年時間的[恐怖主義行動](https://zh.wikipedia.org/wiki/恐怖主義 "wikilink")，藉由研究者和決策者提供的一系列數據來加以分析，之後建立指數地圖顯出國家恐怖主義之趨勢。該指數為提供數據，[馬里蘭大學學院市分校](../Page/馬里蘭大學學院市分校.md "wikilink")收集後整理出，之後由出版。在全球恐怖主義資料庫中已記載超過125,000件恐怖活動\[1\]。

2014年全球恐怖主義指數為繼2012年全球恐怖主義指數以來之另種版本\[2\]。該指數以交互性地圖來顯示出國家恐怖主義分佈。

## 計算方法

### 全球恐怖主義指數評分方式

全球[恐怖主義指數評分方式以獨特的評分方式來考慮任何一年因恐怖事件而造成的影響](https://zh.wikipedia.org/wiki/恐怖主義 "wikilink")。各國家年度總分包含4種因素\[8\]：

  - 任一年[恐怖事件的數量](https://zh.wikipedia.org/wiki/恐怖事件 "wikilink")。
  - 任一年因恐怖事件造成死亡的總人數。
  - 任一年因恐怖事件造成受傷的總人數。
  - 任一年因恐怖事件造成財產損失的總額。

此外，每種因素加權值不盡相同，主要反映出該國家在五年間的恐怖主義行為，隨著時間流逝而讓心理造成揮之不去的影響。下表中的加權值為經由[全球和平指數](../Page/全球和平指數.md "wikilink")專案委員會（GPI Expert Panel）的諮詢決定出來：

| 因素     | 加權值  |
| ------ | ---- |
| 恐怖事件數量 | 1    |
| 死亡總人數  | 3    |
| 受傷總人數  | 0.5  |
| 財產損失總額 | 0\~3 |

其中，[死亡](../Page/死亡.md "wikilink")為4個因素當中加權值為最大。

財產損失測定共分成4種類型：下表所列出的嚴重程度，因恐怖事件造成財產損失少於100萬美元屬於「輕微」；財產損失介於100萬美元到10億美元之間屬於「嚴重」；財產損失超過10億美元屬於「非常嚴重」。根據世界恐怖主義資料庫顯示，絕大部分的嚴重程度都屬於「不明」。此外具有「災難性」的恐怖行動為極罕見。

| 加權值 | 嚴重程度                     |
| --- | ------------------------ |
| 0   | 不明                       |
| 1   | 輕微（財產損失少於100萬美元）         |
| 2   | 嚴重（財產損失介於100萬美元到10億美元之間） |
| 3   | 非常嚴重（財產損失超過10億美元）        |
|     |                          |

**《全美超級模特兒新秀大賽》第九季**，是[美國真人秀](https://zh.wikipedia.org/wiki/美國 "wikilink")《[全美超級模特兒新秀大賽](../Page/全美超級模特兒新秀大賽.md "wikilink")》的第九季。在2007年9月19日於The
CW首播。
本季獲勝者可獲得成為Elite模特公司旗下的模特兒、一份CoverGirl價值十萬美元的合同，以及登上《Seventeen》雜誌的封面和內頁。

## 每集內容

### 第1集：巡航的女孩們 The Girls Go Cruisin'

《全美超級模特兒新秀大賽》的第九季終於來了！而本季一共有33位準超模參加比賽，而同樣本季選拔並會在一艘前往加勒比海的郵輪內進行。

女郎們在登上了郵輪後的首項測試，她們要穿著救生衣走台步，而當中有大多數人須多加練習，但同樣並有部分人表現出色。

翌日當女郎們在進食早餐之際，泰雅突然穿著森巴舞蹈服與女郎們見面。隨後泰雅會聯同Mr.J和Miss
J逐一向女郎們面試，當中有在[耶魯大學讀書的Victoria](https://zh.wikipedia.org/wiki/耶魯大學 "wikilink")、曾參加過由泰雅舉辦的青年夏令營的Saleisha、患有[亞斯伯格症的Heather](https://zh.wikipedia.org/wiki/亞斯伯格症 "wikilink")、職業為脫衣舞女郎的Lisa等。

其後郵輪抵達了[聖約翰斯](https://zh.wikipedia.org/wiki/聖約翰_\(安提瓜和巴布達\) "wikilink")，而女郎們並在該處被帶到一個海灘與Mr.J、Miss
J以及[上季冠軍Jaslene會面](../Page/全美超級模特兒新秀大賽_\(第八季\).md "wikilink")，她們被告知須在該海灘拍攝泳衣照。而在拍攝完畢後，Mr.J和Miss
J告知將會有20位參賽者能夠繼續乘坐郵輪進行選拔。

在第一次的淘汰選拔過後，泰雅再一次出現在女郎們面前，並要求女郎們說服她為何想繼續留下參賽的理由。

隨後泰雅聯同Mr.J和Miss
J透過硬照以及其對參賽者的理解挑選出13位成功入圍的參賽者，其後泰雅會在郵輪的甲板上進行最後選拔，她會逐一呼出13位成功入圍的參賽者名字。

特別嘉賓：Gregorio Cuevas、Patrick Dukes
硬照攝影師：Jim Wade

### 第2集：環保的模特們 The Models Go Green

13位準超模們抵達了洛杉磯，而在她們前往模特大屋前，Mr.J告知本季會十分注重環保，而同樣本季參賽者們所坐的轎車並會以生物柴油作為燃料。

隨後女郎們乘坐轎車抵達了模特大屋，而大屋內的設備並注入了環保元素，同時並有游泳池、給予女郎們練習台步的迷你T台等，讓女郎們十分欣喜。

翌日女郎們抵達了攝影棚拍攝硬照，而本季首個硬照主題為「吸煙後遺症」，而參賽者們的硬照並會分開兩次拍攝：第一次拍攝為手持著香煙的美人；而第二次拍攝為吸煙者在出現後遺症後的容貌和症狀，而其容貌並會透過硬照上的鏡子倒影呈現。

| Ambreal          | 肺癌      |
| Bianca           | 嚴重的口腔問題 |
| Chantal          | 氣管切開術   |
| Ebony            | 肺萎縮     |
| Heather＆Saleisha | 二手煙     |
| Janet            | 灼傷      |

| Victoria         | 胎死腹中    |
|                  |         |

拍攝期間，Ebony因為害怕被其他女孩針對，繼而影響到拍攝表現；Mila對於其吸煙後遺症的化妝容貌一直感到滑稽；Heather和Saleisha兩人須一同拍攝硬照，然而在拍攝兩人並沒有任何彼此互動。

接著Miss
J抵達了模特大屋教授有關衣著打扮的技巧，並告知女郎們會前往[老海軍旗艦店篩選服飾](../Page/老海軍.md "wikilink")。而在旗艦店內的Benny
Ninja並告知女郎們須在限時內挑選合適的服飾，而挑戰獲勝者並會在首輪評審時宣佈，而獲勝者的獎品為[老海軍的](../Page/老海軍.md "wikilink")1000美元的現金劵以及成為[老海軍的廣告模特](../Page/老海軍.md "wikilink")。

在首輪評審中，Bianca被指其硬照有些做作；Mila被指其在參與拍攝過程中並不嚴肅；而Heather在面對評審時相當害羞，然而評審們卻欣賞其在硬照中展現的靜態美。

評審過後，泰雅告知女郎們為了讓她們成為一個好榜樣，本季將全面禁煙。隨後泰雅開始宣佈入圍的參賽者，Heather的靜態美讓其能夠成為首名入圍；其後Ebony因其在硬照上僵硬的姿勢而聯同沒有認真了解其硬照信息的Mila成為了倒數兩人，而最後評審們認為Ebony更有模特兒潛質而決定將她留下。

特別嘉賓：Benny Ninja
硬照攝影師：Mike Rosenthal

小挑戰勝出者：Saleisha
首名入圍：Heather
倒數兩人：Ebony＆Mila
淘汰：Mila

### 第3集：攀石的女孩們 The Girls Go Rock Climbing

本週女郎們須接受Miss J的台步訓練，在訓練中她們會穿上拘束衣以及高跟鞋走台步。

而在翌日晚上，女郎們抵達了一個會場接受走秀挑戰，她們會穿上Colleen Quen帶有行動限制的設計走秀，而同樣曾在過往比賽時主持時裝秀的Roy
Campbell會主持女郎們的時裝秀。而走秀挑戰的勝出者將會能夠前往巴黎成為Colleen Quen時裝秀的其中一位模特兒。

而本週硬照拍攝女郎們來到一個攀石場館內，並被告知須穿上高端時尚的服飾拍攝攀石照。

擁有攀石經驗的Jenah在本週拍攝中游刃有餘，而她並給予女郎們一些攀石的指導；Heather的姿勢和表情並獲得Mr.J和女郎們的欣賞；而Kimberly在拍攝時的表現卻是平淡無奇。

評審時間，評審們非常欣賞Heather和Jenah的硬照，而Jenah更是首名入圍；而Bianca和Kimberly的硬照發揮不力，故此兩人成為了倒數兩人，而最後Kimberly那張沒有展現出個人魅力的硬照讓她出局。

特別嘉賓：Roy Campbell、Colleen Quen
硬照攝影師：Matthew Jordan Smith
客座評審：Roy Campbell

小挑戰勝出者：Saleisha
首名入圍：Jenah
倒數兩人：Bianca＆Kimberly
淘汰：Kimberly

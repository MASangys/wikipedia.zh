**《全美超級模特新秀大賽》第八季**，是[美國真人秀](https://zh.wikipedia.org/wiki/美國 "wikilink")《[全美超級模特兒新秀大賽](../Page/全美超級模特兒新秀大賽.md "wikilink")》的第八季。在2007年2月28日於The
CW首播。

獲勝者獎品和[第七季一樣](https://zh.wikipedia.org/wiki/全美超級模特兒新秀大賽第七季 "wikilink")，包括：Elite模特兒經紀合約、登上《[Seventeen](https://zh.wikipedia.org/wiki/17_\(雜誌\) "wikilink")》雜誌封面及拉頁時裝照、十萬美元的CoverGirl化妝品合約。\[1\]

## 每集內容

### 第1-2集：不願閉嘴的女孩 The Girl Who Won't Stop Talking

33名準超模來到洛杉磯開始她們的超模之旅。她們首先要在模特兒訓練營來過軍訓，拍過硬照，然後會逐一會見三位評判：泰雅、Miss J和Mr.
J。當中有曾參選[第七季選拔的Jaslene](https://zh.wikipedia.org/wiki/全美超級模特兒新秀大賽第七季 "wikilink")、曾為時尚攝影師的Sarah、來自俄羅斯，其配偶年齡為40歲的Natasha、泰雅的超級粉絲Kathleen等。而本季並有兩名加大碼女孩來到這裡參加面試，她們分別為Diana和Whitney。

見過三位評判後，12名模特兒首先出局。餘下的女孩立即到派對進行小小的拍攝，評判按照片後立即開始宣佈入圍的13位參賽者。而同樣泰雅並決定本季讓兩名加大碼女孩一同入圍比賽，為歷屆首次。

隨後13名準超模搬到好萊塢山的大屋正式展開未來超模之旅，之後她們收到泰雅的信，她們到達二手時裝店Goodwill，原來她們要選一些衣服進行行天橋的比試，誰的衣服讓人競投得最高價就勝出比試。最後Jael成功勝出了第一個挑戰，而獎勵則為能夠用自己名義把善款捐給慈善機構。Renee對此看得望不在乎，並說其他女孩能勝全部的獎品，她要的是勝出比賽。回到家中Renee不分不滿，更和Jael說這次比賽根本不公平。

接著，模特兒們就要開始拍攝。而本季第一場的硬照主題各參賽者須演繹有關「政治爭議」的話題，由評審尼祖負責拍攝，而同樣每位參賽者的硬照主題並會與另外一位參賽者的硬照來作對比。

| ---- | ---------------- | --------------- |
| 皮草   | Brittany         | Kathleen        |
| 槍械   | Diana            | Renee           |
| 素食主義 | Dionne           | Cassandra（肉食主義） |
| 死刑   | Jaslene          | Sarah（終身監禁）     |
| 墮胎   | Natasha          | Jael（尊重生命）      |
| 同性婚姻 | Samantha＆Whitney | Felicia（傳統婚姻）   |
|      |                  |                 |

來到了第一次評審的環節，泰雅認為她們的表現尚未合格，只有數名女孩能夠突圍而出，而當中Jaslene其硬照中的兇狠表情更深受評審們讚賞，故Jaslene為第一位入圍。

Kathleen因為未能充分了解其硬照主題，而聯同硬照發揮不足兼且過於懼怯的Jael成為倒數兩人。結果Kathleen成為了第一位被淘汰的選手。

### 第3集：參加學校派對的女孩們 The Girls Go To Prom

經歷了第一場評審後，剩下的12位參賽者在翌日來到某高中接受Miss
J的台步訓練，她們首先換上台步訓練服裝，並且會以跑道上的白線作為精準度的練習。當女孩們逐一練習完台步後，Miss
J更要求參賽者們以三人一組並且呈現當年泰雅與其他模特兒並肩走秀時況。

當女孩們完成Miss J的教導後，鈴聲響起代表著她們已經下課了，隨後來到高中禮堂內遇到曾主持第六季旋轉時裝秀的Roy
Campbell，這次她們仍會在校園挑戰伸展台的時裝表演。而在伸展台上，部分女孩的表現不盡人意，如Natasha未能配合與其同組走秀的組員，Sarah在走秀時走光，Samantha在走秀時不夠專注等。然而最後在走秀挑戰獲勝的是Brittany。而Jaslene回到模特大屋後對Miss
J和Roy給予其的評價十分介懷，終於忍不住向媽媽哭訴。另外Natasha言語不通，讓女孩們難以明白她的意思。

而本週的硬照拍攝並會延續走秀挑戰賽的校園主題，而每位參賽者須扮演各種類型的高中生。

| Diana     | 學生會會長   |
| Dionne    | 壞女孩     |
| Felicia   | 運動員     |
| Jael      | 書呆子     |
| Jaslene   | 怪咖      |
| Natasha   | 教師的寵物   |

|           |         |

當眾多女孩們在自己的拍攝時，詮釋的相當好，但Natasha、Renee和Samantha對本次的硬照拍攝陷入苦戰：Samantha因其本身文靜的性格，當她嘗試詮釋其角色時卻總是未能抓到訣竅；而Natasha則在如何詮釋中迷失了，並且無法擺出所要的姿勢；而Renee卻是不滿意自己的角色以及抱怨他人，因而影響了其拍攝表現。

在評審期間，泰雅認為Samantha雖然擁有迷人的地方但總是缺乏自信心，當所有注意到Samantha無明顯的人格特質時，又因為她有那潛力但必須花更多時間去展現。而Natasha的照片實在糟透了，Mr.J更指出她比第三季的Ann還難指導，但Natasha聽後反而稱讚Ann是不錯的模特兒，評審們並因此欣賞她把批評轉化為正面。

Jaslene出色的硬照表現令她連續第二次第一名入圍下一輪的比賽。而本週的倒數二人為Natasha和Samantha，而泰雅指出Samantha有潛質卻缺乏明顯的個性，並且讓人有一種看起來只有15歲的青澀感覺；而Natasha被指擁有迷人外表，卻未能將自身優勢運用在拍攝硬照上。然而最後Natasha獲得留下的機會繼續角逐冠軍。

淘汰：Samantha

### 第5集：改善了態度的女孩 The Girl Who Changes Her Attitude

《Vogue》的擺姿勢教練Benny
Ninja在本集會教導參賽者們如何擺有藝術的姿勢。女郎們在互相抗衡擺設的比賽中，學會如何在誇大與肢體動作中擺好姿勢。

其後在翌日的小挑戰中，她們只有2分鐘展示她們如貓似的敏捷與美妙的姿勢潛行，還要躲開鐳射光線，到達盡頭，而挑戰獲勝者可獲得價值四萬美元的手鍊。結果Renee未能按時完成挑戰而被取消獲勝資格；而Felicia在挑戰中感到非常有自信，因為她認為此次小比賽可以讓身為有舞蹈背景的她有大大優勢，雖然Benny
Ninja有注意到Felicia以雍容美麗的姿態完成挑戰，但最終還是由Whitney獲勝。

Renee受到其他女郎的杯葛，致電給家中的丈夫訴苦。她開始發現自己以前的待人接物太差，決定要好好改變一下自己的態度。

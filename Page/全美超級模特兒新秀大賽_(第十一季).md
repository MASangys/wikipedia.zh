**《全美超級模特兒新秀大賽》第十一季**，是[美國真人秀](../Page/美國.md "wikilink")《[全美超級模特兒新秀大賽](../Page/全美超級模特兒新秀大賽.md "wikilink")》的第十一季，是第五季於「CW電視網路」播映，首播兩小時版本日期為2008年9月3日。[台灣於](../Page/台灣.md "wikilink")2009年2月21日在Channel\[V\]播出。而香港則於2009年10月11日在明珠台首播。

冠軍得到的獎品跟過往四季相似，為：「精英模特兒公司」、「封面女郎化妝品公司」十萬美元合約及十七雜誌的封面連六頁跨頁時裝照。本季的宣傳口號為「感受愛」（Feel
The Love）。

與第十季不相同，上季於[紐約市拍攝](../Page/紐約市.md "wikilink")，而本季則在[亞馬遜](../Page/亞馬遜.md "wikilink")。另外本季的出國地點為[荷蘭首都](../Page/荷蘭.md "wikilink")[阿姆斯特丹](../Page/阿姆斯特丹.md "wikilink")。本季有十四名參賽者，跟第三、第四及第十季相同。

本季首次有跨性別參賽者並打入十四強，名叫依莎，而這個例外被一名外國同性戀組織的會長尼奧‧僑裡安奴（Neil
Giuliano）評為「前所未有的機會」。

當女郎們來到了荷蘭便分為三隊，通過一系列任務去尋找新超模屋位置。綺蓮娜和森瑪花最快到達新居並得到額外的五十張底片作為獎勵。進入新居後，綺蓮娜、安娜莉及瑪祖兒三個一起洗泡泡浴引起的吵鬧聲卻引起其他三位想要早點休息的參賽者不爽。美琪表示：「如果森瑪花及仙娜如果被淘汰，她會想死，因為她跟其他人不合」。
隨後，她們去到紅燈區進行第二場分組小挑戰，任務為於櫥窗內擺出姿勢來推銷不同設計師的裙子。雖然瑪祖兒被批為最好的參賽者，但因為團隊合作的關係，美琪及森瑪花贏得時尚周的幾個時裝走秀機會。

本週拍攝主題要求女郎們穿上十七世紀女裝長袍，並在小船上擺出姿勢拍攝相片。美琪、安娜莉和瑪祖兒得到正面評論，森瑪花則以額外的底片拍出更好的照片。而仙娜則再次擺出被評為放蕩的姿勢，綺蓮娜的集中力令她曾無視傑先生的存在。

評審席上，森瑪花的衣著受到批評但照片卻受到讚賞，瑪祖兒的羞怯性格與戴弗妮的截然不同。安娜莉自然地運用及融合滑冰造型於攝影中，獲得Mr.
J極高的評價。因此安娜莉和美琪得到全場一致的讚賞。仙娜在同時控制表情及姿勢時遇上麻煩及不穩定性令她與照片被批評僵硬及擁有不啟發別人外表的綺蓮娜包尾，但最後評審認為綺蓮娜的照片看似更好，因而再一次留下來，仙娜則被攆回家。

餘下的四名女郎以不常用的東西作道具，跟寶麗娜上模特兒課。隨後，她們要為一段簡短的無聲廣告試鏡，並必須跟男模馬克．凡德魯接吻作為小挑戰任務。瑪祖兒被選為冠軍，獎品為一萬美元的「G-Star」購物券，並揀選安娜莉一起分享這份大獎。該晚，女郎們放鬆心情並邀請前一集面試時的船家到超模屋一聚。

### 全美超級模特兒新秀是……

|                      |
| -------------------- |
| **第十三集**             |
| 標題：全美超級模特兒新秀是…\[11\] |

在瑪祖兒的離開後, 三強誕生了。隨後,安娜麗、森瑪花及美琪要為Cover Girl拍攝廣告及硬照,
而廣告則是於船上與男模進行拍攝。美琪拍廣告的表現雖然進步神速,但仍遭受到J先生的批評。與此同時,
於上次拍廣告有卓越表現的安娜麗卻慘遭滑鐵盧, 出現了忘詞的情況。拍攝廣告完畢後, 三強便到一座大樓中拍攝Cover Girl硬照。

評審中, 美琪廣告的進步表現受到評審們的讚賞,而且Cover Girl硬照更被指是三人之中最佳的。雖然安娜麗拍出絕美的Cover
Girl硬照,但被評審指該硬照更像電影女主角的宣傳照,而在廣告忘詞亦受到斥責。最後,安娜麗與表現平庸森瑪花雙雙落入包尾二人,
但最後安娜麗過往優秀的表現亦不能助她晉級,於三強關頭被淘汰,是為本季季軍。

在安娜麗被淘汰後, 躋身最後二強的美琪和森瑪花參與最後一場走秀, 美琪憑著高挑的身材及其優秀的走秀技巧,於評審中獲得讚賞,
其優秀的相集亦得到不同評審的加許。而森瑪花於走秀亦顯示出其高雅的一面,而她的相集亦不比美琪遜色。最後,超模冠軍得主是美琪,
她是全美超模第二高的冠軍, 亦是第二位沒有進入過包尾二人記錄的冠軍。

## 爭議

在第九季冠軍誕生後，網路上不斷流傳第九季絲麗莎的冠軍寶座早被待定，原因是泰雅知道絲麗莎本已是專業模特兒，並有多次雜誌、電視劇等演出經驗，違反比賽規則；而更重要的是她曾在賽前於第六季的一場走秀表演及「泰雅脫口秀」中出現過，以致絲麗莎成為冠軍惹起爭議。

而這季亦有相似事情發生，事原是參賽者依莎被認出曾於第十季的第一場照片拍攝中充當背景模特兒，而節目亦有指明其中一位背景模特兒名為依莎；有了第九季絲麗莎事件的前科，網友懷疑泰雅幫助依莎成為冠軍，再次惹起「做馬」爭議。

另外，一名曾參與前評判珍妮絲主持的「JD超模公司」的模特兒於節目透露曾入選「全美超級模特兒新秀大賽」，可惜後來被發現為變性人，以致被取消資格；但外國媒體報導依莎親口承認自己是一名變性人。\[13\]，但卻成功打入十四強，以致本季再添疑雲。

而本季第九名選手Clark意外走紅，成為當集首名拿照片下集悲情淘汰的代表性人物。往後幾季悲情淘汰的選手更被網友以＂克拉克的召喚＂代稱，成為新一代話題女王。

在[電腦領域裏](https://zh.wikipedia.org/wiki/電腦 "wikilink")，一個**octet**是指八個位元（bit）為一組的單位，中文稱作八位元組。

在[法國和](https://zh.wikipedia.org/wiki/法國 "wikilink")[羅馬尼亞](../Page/羅馬尼亞.md "wikilink")，
*octet*
這個字通常是指一個[位元組](https://zh.wikipedia.org/wiki/位元組 "wikilink")（byte）的意思；當我們稱一百萬位元組（megabyte，MB），在這些地區會稱作
*megaoctet*。 '' bit'' 和 *byte* 在法語裏是異義同音字。

*Octet*
除了下面提到的唯一例外之外，都是指一個具有八個位元的實體。因此在電腦網路標準中，在*byte*容易引起混淆的地方都僅使用*Octet*。

## 標準

Octets 有兩種不同的前綴，一種為 2
的多次方，另一種是[國際單位制](https://zh.wikipedia.org/wiki/國際單位制 "wikilink")（SI，International
System of Units）。2
的多次方的格式為[國際電工委員會](https://zh.wikipedia.org/wiki/國際電工委員會 "wikilink")（International
Electrotechnical Commission）在 1998 年所制定。

  - kibi 指 « **ki**lo **bi**nary » ;
  - mebi 指 « **me**ga **bi**nary » ;
  - gibi 指 « **gi**ga **bi**nary » ;
  - tebi 指 « **te**ra **bi**nary » ;

|                       |                          |            |
| --------------------- | ------------------------ | ---------- |
| 1 **kibioctet** (Kio) | \= 2<sup>10</sup> octets | \=  octets |
| 1 **mebioctet** (Mio) | \= 2<sup>20</sup> octets | \=  Kio    |
| 1 **gibioctet** (Gio) | \= 2<sup>30</sup> octets | \=  Mio    |
| 1 **tébioctet** (Tio) | \= 2<sup>40</sup> octets | \=  Gio    |
| 1 **pebioctet** (Pio) | \= 2<sup>50</sup> octets | \=  Tio    |
| 1 **exbioctet** (Eio) | \= 2<sup>60</sup> octets | \=  Pio    |
| 1 **zebioctet** (Zio) | \= 2<sup>70</sup> octets | \=  Eio    |
| 1 **yobioctet** (Yio) | \= 2<sup>80</sup> octets | \=  Zio    |

[SI的前綴為](https://zh.wikipedia.org/wiki/SI "wikilink")
kilo、mega、giga、tera等，與所有的 SI 單位的表現法相同，都是基於 10 的次方：

|                      |                           |            |
| -------------------- | ------------------------- | ---------- |
| 1 **kilooctet** (Ko) | \= 10<sup>3</sup> octets  | \=  octets |
| 1 **megaoctet** (Mo) | \= 10<sup>6</sup> octets  | \=  ko     |
| 1 **gigaoctet** (Go) | \= 10<sup>9</sup> octets  | \=  Mo     |
| 1 **teraoctet** (To) | \= 10<sup>12</sup> octets | \=  Go     |
| 1 **petaoctet** (Po) | \= 10<sup>15</sup> octets | \=  To     |

## 源由

**octet**
這個詞是從[拉丁文和](https://zh.wikipedia.org/wiki/拉丁文 "wikilink")[希臘文的數字](https://zh.wikipedia.org/wiki/希臘文 "wikilink")
[octo](https://zh.wikipedia.org/wiki/octo- "wikilink") 衍生而來的，意指八。

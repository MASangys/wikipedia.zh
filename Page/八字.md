**八字**源自古代的[中國](../Page/中國.md "wikilink")，是一種利用[天干](../Page/天干.md "wikilink")和[地支](../Page/地支.md "wikilink")來準確記錄年、月、日、時的方式，由「**年干，年支**」、「**月干，月支**」、「**日干，日支**」、「**時干，時支**」，共八個干支所組成（共八個字），年、月、日、時的干支組合稱為「柱」，形成「**年柱**」、「**月柱**」、「**日柱**」、「**時柱**」，故八字又稱為「四柱」或「四柱八字」。

## 歷史

這種以干支記錄時間的方式從[夏朝](../Page/夏朝.md "wikilink")開始就已經開始使用，但直至[宋朝](../Page/宋朝.md "wikilink")的[徐子平才將人們出生時的干支八字用來論斷人事禍福](https://zh.wikipedia.org/wiki/徐子平 "wikilink")，因此[命理學中的八字命理又稱](https://zh.wikipedia.org/wiki/命理學 "wikilink")「[子平八字學](https://zh.wikipedia.org/wiki/子平八字 "wikilink")」，以紀念徐子平對八字命理學的貢獻。

## 換算方法

學習八字，都要熟悉一套把時間換成八字的方式。首先，每一年以[立春](../Page/立春.md "wikilink")為新一年的開始，[立春](../Page/立春.md "wikilink")至[惊蛰](../Page/惊蛰.md "wikilink")為[寅月](https://zh.wikipedia.org/wiki/寅 "wikilink")（近似農曆[正月](https://zh.wikipedia.org/wiki/正月 "wikilink")），以「[節](https://zh.wikipedia.org/wiki/節氣 "wikilink")」那一分鐘（並非[子時開始](https://zh.wikipedia.org/wiki/子_\(地支\) "wikilink")）為月之始，「[氣](https://zh.wikipedia.org/wiki/節氣 "wikilink")」的最後一分鐘為月之終；然後依次序推算出月柱、日柱、時柱的[天干](../Page/天干.md "wikilink")與[地支](../Page/地支.md "wikilink")。

因八字只爲體現地球與太陽的相對角度位置\[1\]，所以時間換算成八字的公式需參考很多因素，包括[平太陽時](https://zh.wikipedia.org/wiki/平太阳 "wikilink")、[恒星時](https://zh.wikipedia.org/wiki/恒星時 "wikilink")、[均時差](../Page/均時差.md "wikilink")、[時區](https://zh.wikipedia.org/wiki/時區 "wikilink")、[夏令時](https://zh.wikipedia.org/wiki/夏令時 "wikilink")，甚至包括經度，以及該經度與太陽的日照相對夾角\[2\]等。這樣的換算法固然精確，卻也較為煩瑣，多半借助[電腦輔助計算](https://zh.wikipedia.org/wiki/電腦 "wikilink")，一般是以[萬年曆搭配簡易公式換算](https://zh.wikipedia.org/wiki/萬年曆 "wikilink")。

## 八字運用

以下二例，分別以西元、農曆和八字三種表示法參照比較：

  - **<span style="color: Green">例一</span>**

|    |                 |
| -- | --------------- |
| 西元 | 2006年1月28日23:50 |
| 農曆 | 丙戌年正月初一日子時      |
| 八字 | 乙酉年己丑月戊午日壬子時    |

※八字以「[立春](../Page/立春.md "wikilink")」[節氣為新舊年的交界](https://zh.wikipedia.org/wiki/節氣 "wikilink")，丙戌年正月初七辰時（07:27）立春。

  - **<span style="color: Green">例二</span>**

|    |                  |
| -- | ---------------- |
| 西元 | 2006年10月10日10:10 |
| 農曆 | 丙戌年八月十九日巳時       |
| 八字 | 丙戌年戊戌月壬申日乙巳時     |

**八重瀨町**（，），位於[琉球列島](https://zh.wikipedia.org/wiki/琉球列島 "wikilink")[沖繩群島](../Page/沖繩群島.md "wikilink")[沖繩島](../Page/沖繩島.md "wikilink")南部。為2006年1月1日**東風平町**與**具志頭村**[合併而成](../Page/市町村合併.md "wikilink")。

町內區域也為[二次大戰末](https://zh.wikipedia.org/wiki/二次大戰 "wikilink")[沖繩島戰役](../Page/沖繩島戰役.md "wikilink")末期，戰事最激烈的地區。

## 歷史

|                                                       |          |
| ----------------------------------------------------- | -------- |
| 時間                                                    | **行政劃分** |
| [琉球王國](https://zh.wikipedia.org/wiki/琉球王國 "wikilink") | 東風平間切    |
| 1900年代                                                | 東風平村     |
| 1910年代                                                |          |
| 1920年代                                                |          |
| 1930年代                                                |          |
| 1940年代                                                |          |
| 1950年代                                                |          |
| 1960年代                                                |          |
| 1970年代                                                | 東風平町     |
| 1980年代                                                |          |
| 1990年代                                                |          |
| 2000年代                                                | 八重瀨町     |

  - 1896年4月1日：實施[郡區制](https://zh.wikipedia.org/wiki/郡區制 "wikilink")，為島尻郡轄下東風平間切及具志頭間切。
  - 1908年4月1日：實施[島嶼町村制](../Page/島嶼町村制.md "wikilink")，改設為東風平村及具志頭村。
  - 1979年10月1日：東風平村改制為東風平町。
  - 2002年：東風平町、具志頭村、南風原町、大里村成立合併協議會，但因為南風原町最後反對而取消而使協議會解散。解散後大里村改與玉城村、知念村、佐敷町成立合併協議會，並於2006年合併為[南城市](../Page/南城市.md "wikilink")。
  - 2004年秋天：東風平町與具志頭村重新成立合併協議會，商討合併事宜。
  - 2006年1月1日：東風平町和具志頭村正式合併，以區域內的八重瀨岳為名，合併後新名稱定為**八重瀨町**。

**公務人員特種考試外交領事人員考試**，簡稱**外交特考**或**外特**，是[中華民國](../Page/中華民國.md "wikilink")[國家考試的一種](https://zh.wikipedia.org/wiki/國家考試 "wikilink")，為[公務人員特種考試三等考試項目之一](../Page/公務人員特種考試.md "wikilink")。

《[中華民國憲法](../Page/中華民國憲法.md "wikilink")》第一百四十一條：「中華民國之[外交](../Page/外交.md "wikilink")，應本獨立自主之精神、平等互惠之原則，敦睦邦交，尊重[條約及](https://zh.wikipedia.org/wiki/條約 "wikilink")[聯合國憲章](https://zh.wikipedia.org/wiki/聯合國憲章 "wikilink")，以保護僑民權益，促進國際合作，提倡國際正義，確保世界和平。」基於國家永續發展及任務傳承之使命，[中華民國外交部積極培養外交專業人才](../Page/中華民國外交部.md "wikilink")。中華民國外交部每年透過外交特考招聘不定人數的新科外交領事人員，是代表國家的[外交官](../Page/外交官.md "wikilink")。

外交領事人員在訓練期間就有比照[委任五等五級的津貼可領](https://zh.wikipedia.org/wiki/委任 "wikilink")，[月薪約](https://zh.wikipedia.org/wiki/月薪 "wikilink")[新臺幣](../Page/新臺幣.md "wikilink")39,600元；訓練期滿後調升為[薦任第六職等](https://zh.wikipedia.org/wiki/薦任 "wikilink")，本俸月薪新臺幣46,730元；如果派赴駐外，月薪依駐在國[物價水準而定](https://zh.wikipedia.org/wiki/物價 "wikilink")，還有專業加給，月領新臺幣廿、卅萬元的大有人在，堪稱是最高薪的國家公務人員。且外交官生活長年派駐在外、周遊列國、[社會地位高](../Page/社會地位.md "wikilink")，故外交特考為中華民國相當熱門的國家考試之一。

## 應試資格

依據[考選部公布](https://zh.wikipedia.org/wiki/考選部 "wikilink")，外交特考應試資格為：

1.  十八歲以上，四十五歲以下。
2.  [專科以上學校畢業](../Page/專科學校.md "wikilink")。
3.  [公務人員普通考試或](https://zh.wikipedia.org/wiki/公務人員普通考試 "wikilink")[公務人員特種考試及格滿三年](../Page/公務人員特種考試.md "wikilink")。
4.  [高等檢定考試及格](https://zh.wikipedia.org/wiki/高等檢定考試 "wikilink")。

## 應試科目

外交特考共考七科，列舉如下：

| 科目          | 備註                             |
| ----------- | ------------------------------ |
| 國文          | 包含作文、公文與測驗                     |
| 外國文         | 如為英文組考生考科為英文，其它外語組考生為其報考之外語及英文 |
| 外語個別口試      | 就應考人報考組別之外國語應試                 |
| 中華民國憲法與比較政治 |                                |
| 近代外交史與國際關係  |                                |
| 國際經濟學       | 含國際經貿組織                        |
| 國際公法        |                                |

## 外語文

外交官需長期派駐國外，代表中華民國與他國互動，語言能力為其必要工具；故外交特考的外語文部分是加重計分，此一特色也吸引不少外語背景的人才投入外交特考，雖曾有人詬病外交人員會因此疏忽其他領域方面的能力，但到目前為止外交特考依然是以語言加重計分為標準，這也顯示外語能力對外交人才的重要性。

外交特考招考組別當中，英文組最為穩定，每年都有一定人數的缺額；其他組別（如俄文組、韓文組、日文組、阿拉伯文組等）每年招考名額不一。

## 外交官生涯

通過外交特考後的第一年，會外派至美、英、日等國受為期一年的語言訓練。結束語言訓練後，會先在國內分發至外交部各單位服務，為期三年。國內服務滿三年後，即需外派國外兩任，一任三年。之後即國內、國外輪調。

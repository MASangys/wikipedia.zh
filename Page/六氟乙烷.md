**六氟乙烷**，是一種[鹵代烴](https://zh.wikipedia.org/wiki/鹵代烴 "wikilink")（化學式：[C](../Page/碳.md "wikilink")<sub>2</sub>[F](../Page/氟.md "wikilink")<sub>6</sub>），亦是[烃类化合物](../Page/烃.md "wikilink")[乙烷所对应的全氟化合物](../Page/乙烷.md "wikilink")，是一種不易燃的氣體，难溶於水，微溶於[醇](../Page/醇.md "wikilink")。

## 物理性質

六氟乙烷在[固相時擁有兩種](https://zh.wikipedia.org/wiki/固相 "wikilink")[晶型](https://zh.wikipedia.org/wiki/晶型 "wikilink")，不同[相變溫度已於科學文獻中列明](../Page/相變.md "wikilink")，最新的文献将相變溫度定於103
K(-170 °C)。在低於103
K的情況下，它會有一個稍微無序的結構，在相变点以上，會形成[体心立方晶體結構](https://zh.wikipedia.org/wiki/等軸晶系 "wikilink")。\[1\]

六氟乙烷密度表：

| 性質，溫度       | 密度 (kg.m<sup>-3</sup>) |
| ----------- | ---------------------- |
| 液態，-78.2 °C | 1608                   |
| 氣態，-78.2 °C | 8.86                   |
| 氣態，15 °C    | 5.84                   |
| 氣態，20.1 °C  | 5.716                  |
| 氣態，24 °C    | 5.734                  |
|             |                        |

六氟乙烷[蒸氣相对空气的密度為](https://zh.wikipedia.org/wiki/蒸氣 "wikilink")4.823，21°C时[比重為](https://zh.wikipedia.org/wiki/比重 "wikilink")4.773（空氣是1），[比容為](https://zh.wikipedia.org/wiki/比容 "wikilink")0.1748
m³/kg。

## 用途

六氟乙烷可用於一種具多功能的蝕刻技術，常見於半導體之生產。它可用於金屬[矽化物及金屬](../Page/矽化物.md "wikilink")[氧化物並相對其金屬基質的選擇性蝕刻](../Page/氧化物.md "wikilink")。叧外，它亦用於蝕刻[矽上的](https://zh.wikipedia.org/wiki/矽 "wikilink")[二氧化硅](../Page/二氧化硅.md "wikilink")。

六氟乙烷可連同[三氟甲烷一起用於](../Page/三氟甲烷.md "wikilink")[製冷劑R](https://zh.wikipedia.org/wiki/工作介質 "wikilink")508A（六氟乙烷占61%）及R508B（六氟乙烷占54%）。

## 環境影響

六氟乙烷是一種造成[溫室效應的氣體](https://zh.wikipedia.org/wiki/溫室效應 "wikilink")。它非常穩定，可以長時間停留在大氣層中，是一種非常強大的[溫室氣體](https://zh.wikipedia.org/wiki/溫室氣體 "wikilink")。它在大氣中的壽命約為10,000年，[全球暖化潛勢是](../Page/全球暖化潛勢.md "wikilink")9,200（[二氧化碳的係數是](../Page/二氧化碳.md "wikilink")1）。它在大氣層的濃度是
3 pptv（在1750年時是 0
pptv\[2\]）。然而，六氟乙烷在紅外線光譜下有強大的吸收勢。它的[輻射驅動力是](https://zh.wikipedia.org/wiki/輻射驅動力 "wikilink")0.001
W/m²。它的[臭氧破壞潛勢是](../Page/臭氧破壞潛勢.md "wikilink")0。

六氟乙烷已列入《[IPCC國家溫室氣體清單](../Page/政府間氣候變化專門委員會.md "wikilink")》。

六氟乙烷的工業排放主要在以[霍爾電解法生產](https://zh.wikipedia.org/wiki/霍爾電解法 "wikilink")[鋁的過程中產生](https://zh.wikipedia.org/wiki/鋁 "wikilink")。

## 安全

六氟乙烷的相對[密度比較高](../Page/密度.md "wikilink")，可以填滿地面空間範圍，高浓度时會導致[窒息](../Page/窒息.md "wikilink")。其他健康影響與[四氟甲烷相似](https://zh.wikipedia.org/wiki/四氟甲烷#安全 "wikilink")。

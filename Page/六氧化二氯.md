**六氧化二氯**是一种[无机化合物](../Page/无机化合物.md "wikilink")，[化学式](../Page/化学式.md "wikilink")为。这种[氯氧化物是](https://zh.wikipedia.org/wiki/氯氧化物 "wikilink")[氯酸](../Page/氯酸.md "wikilink")和[高氯酸](../Page/高氯酸.md "wikilink")的混合[酸酐](../Page/酸酐.md "wikilink")。

## 制法

它可以由[二氧化氯](../Page/二氧化氯.md "wikilink")与过量的[臭氧](../Page/臭氧.md "wikilink")反应制得：

  -
    \(\rm \ 2ClO_2 + 2O_3 \rightarrow 2ClO_3 + 2O_2 \rightarrow Cl_2O_6 + 2O_2\)

也可用氯气与臭氧在光照下直接反应，或者光解[二氧化氯](../Page/二氧化氯.md "wikilink")以及热分解[高氯酸氯](../Page/高氯酸氯.md "wikilink")制得。

纯净的六氧化二氯由[高氯酸氙分解获得](https://zh.wikipedia.org/wiki/高氯酸氙 "wikilink")\[1\]：

\[\rm \ Xe(ClO_4)_2 \xrightarrow{293K} Xe + Cl_2O_6 + O_2\]

## 结构

它在常温下是深红色发烟液体，固态时为[离子化合物](https://zh.wikipedia.org/wiki/离子化合物 "wikilink")[高氯酸](../Page/高氯酸.md "wikilink")[氯酰](https://zh.wikipedia.org/wiki/氯酰 "wikilink")（）。最早报道它以单体ClO<sub>3</sub>存在于气相中，\[2\]但后来证实，即使进行加热蒸发依然以含有氧桥键的双聚体存在，直到更高温度时分解成[高氯酸氯](../Page/高氯酸氯.md "wikilink")（Cl<sub>2</sub>O<sub>4</sub>）和氧气。\[3\]

## 性质

Cl<sub>2</sub>O<sub>6</sub>具有[反磁性](https://zh.wikipedia.org/wiki/反磁性 "wikilink")，它是一种很强的氧化剂。尽管它在室温下稳定存在，但它能与[有机化合物](../Page/有机化合物.md "wikilink")剧烈反应甚至爆炸。\[4\] 它还能氧化不活泼的金属[金](../Page/金.md "wikilink")，生成四[高氯酸](../Page/高氯酸.md "wikilink")根合金(III)酸氯酰（）。\[5\]许多使用Cl<sub>2</sub>O<sub>6</sub>的反应证明了它是离子晶体（），下面列举一些：\[6\]

  -
    \(\rm \ NO_2F + Cl_2O_6 \rightarrow NO_2ClO_4 + ClO_2F\)
    \(\rm \ NO + Cl_2O_6 \rightarrow NOClO_4 + ClO_2\)
    \(\rm \ 2V_2O_5 + 12Cl_2O_6 \rightarrow VO(ClO_4)_3 + 12ClO_2 + 3O_2\)
    \(\rm \ SnCl_4 + 6Cl_2O_6 \rightarrow [ClO_2]_2[Sn(ClO_4)_6] + 4ClO_2 + 2Cl_2\)

然而，它也能作为ClO<sub>3</sub>[自由基](../Page/自由基.md "wikilink")的供体参与反应：

  -
    \(\rm \ 2AsF_5 + Cl_2O_6 \rightarrow 2ClO_3AsF_5\)

低温下，六氧化二氯溶于浓[硫酸](../Page/硫酸.md "wikilink")或无水[磷酸](../Page/磷酸.md "wikilink")呈[凝聚态](https://zh.wikipedia.org/wiki/凝聚态 "wikilink")。用不同频率的[电磁波](../Page/电磁波.md "wikilink")照射可使其发生不同的[歧化反应](https://zh.wikipedia.org/wiki/歧化反应 "wikilink")。常温下，它与氟气反应生成[氯酰氟](../Page/氯酰氟.md "wikilink")、[七氧化二氯](../Page/七氧化二氯.md "wikilink")、氧气和少量氯气，与无水氟化氢反应生成氯酰氟和[高氯酸](../Page/高氯酸.md "wikilink")。它也可以用于制备无水[高氯酸盐](../Page/高氯酸盐.md "wikilink")。\[7\]

**六氰合钴(III)酸钾**，又称**高钴氰酸钾**，是一种无机化合物，化学式为K<sub>3</sub>\[Co(CN)<sub>6</sub>\]。

## 制备

六氰合钴(III)酸钾可由[氯化钴和](https://zh.wikipedia.org/wiki/氯化钴 "wikilink")[氰化钾](../Page/氰化钾.md "wikilink")反应得到，反应过程中Co(II)被氧化至Co(III)：\[1\]

  -
    CoCl<sub>2</sub> + 5 KCN → K<sub>3</sub>\[Co(CN)<sub>5</sub>\] + 2
    KCl

有氧存在时，

  -
    2 K<sub>3</sub>\[Co(CN)<sub>5</sub>\] + O<sub>2</sub> →
    K<sub>6</sub>\[(CN)<sub>5</sub>CoOOCo(CN)<sub>5</sub>\]
    K<sub>6</sub>\[(CN)<sub>5</sub>CoOOCo(CN)<sub>5</sub>\] + 2 KCN → 2
    K<sub>3</sub>\[Co(CN)<sub>6</sub>\] + K<sub>2</sub>O<sub>2</sub>

惰性气氛中，

  -
    2 K<sub>3</sub>\[Co(CN)<sub>5</sub>\] + 2 KCN + 2 H<sub>2</sub>O
    —<sup><u>煮沸</u></sup>→ 2 K<sub>3</sub>\[Co(CN)<sub>6</sub>\] +
    H<sub>2</sub> + 2 KOH

## 物理性质

六氰合钴(III)酸钾极易溶于水，不溶于乙醇。有反磁性。\[2\]

## 化学性质

### 和酸反应

六氰合钴(III)酸钾的水溶液在光照下会逐渐分解\[3\]：

  -
    K<sub>3</sub>\[Co(CN)<sub>6</sub>\] + H<sub>2</sub>O →
    K<sub>3</sub>\[Co(CN)<sub>5</sub>OH\] + HCN

六氰合钴(III)酸钾和浓[硫酸](../Page/硫酸.md "wikilink")反应，放出[一氧化碳](../Page/一氧化碳.md "wikilink")\[4\]：

  -
    2 K<sub>3</sub>\[Co(CN)<sub>6</sub>\] + 11
    H<sub>2</sub>SO<sub>4</sub>(浓) + 13 H<sub>2</sub>O → 3
    K<sub>2</sub>SO<sub>4</sub> + 6
    (NH<sub>4</sub>)<sub>2</sub>SO<sub>4</sub> + 2 CoSO<sub>4</sub> + 11
    CO↑ + CO2↑

和稀硫酸作用则放出[氰化氢](../Page/氰化氢.md "wikilink")：

  -
    2 K<sub>3</sub>\[Co(CN)<sub>6</sub>\] + 6
    H<sub>2</sub>SO<sub>4</sub>(稀) + 2 H<sub>2</sub>O → 3
    K<sub>2</sub>SO<sub>4</sub> + 2 CoSO<sub>4</sub> +
    NH<sub>4</sub>HSO<sub>4</sub> + 11 HCN↑ + CO<sub>2</sub>↑

和过量[硝酸](../Page/硝酸.md "wikilink")反应，产生[六氰合钴(III)酸](../Page/六氰合钴\(III\)酸.md "wikilink")：

  -
    K<sub>3</sub>\[Co(CN)<sub>6</sub>\] + 3 HNO<sub>3</sub>
    —<sup><u>Δ</u></sup>→ H<sub>3</sub>\[Co(CN)<sub>6</sub>\] + 3
    KNO<sub>3</sub>

和硝酸的反应，改变条件则按下方式进行：

  -
    3 K<sub>3</sub>\[Co(CN)<sub>6</sub>\](浓) + 9 HNO<sub>3</sub> +
    H<sub>2</sub>O —<sup><u>Δ</u></sup>→
    KH<sub>2</sub>Co<sub>3</sub>(CN)<sub>11</sub>·H<sub>2</sub>O(红色) + 8
    KNO<sub>3</sub> + 6 HCN + HOCN + NO<sub>2</sub>↑

### 和盐溶液反应

六氰合钴(III)酸钾可以和一些金属的盐溶液反应，产生沉淀：\[5\]

  -
    K<sub>3</sub>\[Co(CN)<sub>6</sub>\] + 3 ZnSO<sub>4</sub> →
    Zn<sub>3</sub>\[Co(CN)<sub>6</sub>\]<sub>2</sub>↓(白色) + 3
    K<sub>2</sub>SO<sub>4</sub>
    K<sub>3</sub>\[Co(CN)<sub>6</sub>\] + 3 FeSO<sub>4</sub> →
    Fe<sub>3</sub>\[Co(CN)<sub>6</sub>\]<sub>2</sub>↓(黄色) + 3
    K<sub>2</sub>SO<sub>4</sub>
    K<sub>3</sub>\[Co(CN)<sub>6</sub>\] + 3 CuSO<sub>4</sub> →
    Cu<sub>3</sub>\[Co(CN)<sub>6</sub>\]<sub>2</sub>↓ + 3
    K<sub>2</sub>SO<sub>4</sub>
    K<sub>3</sub>\[Co(CN)<sub>6</sub>\] + 3 AgNO<sub>3</sub> →
    Ag<sub>3</sub>\[Co(CN)<sub>6</sub>\]↓ + 3 KNO<sub>3</sub>

**共格**（，[缩写](https://zh.wikipedia.org/wiki/行间标注缩写列表 "wikilink")： ****），又称**伴随格**、**随同格**，是一种[语法格](https://zh.wikipedia.org/wiki/语法格 "wikilink")，表示伴随\[1\]。

[印欧语系](../Page/印欧语系.md "wikilink")当中常用[工具格](../Page/工具格.md "wikilink")表示共格语义；而斯拉夫语族则常用前置词来表示\[2\]，[汉语](../Page/汉语.md "wikilink")的“和”“与”连动结构、英语的 with [介词等实质上也可以表示共格语义](https://zh.wikipedia.org/wiki/介词 "wikilink")\[3\]。

## 核心语义

共格会为某一事件中的两个[参与者编码一种伴随关系](../Page/语义角色.md "wikilink")（Accompaniment），其中这两个参与者分别为伴随者（Accompanier）和伴随对象（Accompanee），而且往往还会有联系词（Relator）去显著标记伴随者\[4\]；这个联系词可以属于多种[词汇范畴](https://zh.wikipedia.org/wiki/词类 "wikilink")，但最常见的还是[介词或](https://zh.wikipedia.org/wiki/介词 "wikilink")[词缀形态](https://zh.wikipedia.org/wiki/词缀 "wikilink")\[5\]。如下面的[意大利语](../Page/意大利语.md "wikilink")例句：

|     |                      |         |           |         |                  |
| --- | -------------------- | ------- | --------- | ------- | ---------------- |
| 1\. | Il professore        | entra   | nell'aula | con     | i suoi studenti. |
|     | 教授                   | 进入.**** | 演讲厅       | 一同      | 和学生.****         |
|     | **伴随者**              |         |           | **联系词** | **伴随对象**         |
|     | 教授和学生（一同）进入演讲厅。\[6\] |         |           |         |                  |

有些语言的共格还会对[有生性](../Page/有生性.md "wikilink")有所要求，即要求伴随者和伴随对象皆为有生或皆为无生，或者伴随者有生而伴随对象无生；但大多数语言都没有这样的限制。\[7\]

## 区分

共格与其他一些格也很相似，如工具格或。

### 共格与工具格

[工具格](../Page/工具格.md "wikilink")会关联三个[参与者](../Page/语义角色.md "wikilink")，即[施事](../Page/施事.md "wikilink")、工具和受事\[8\]。施事会利用工具改变受事的状态\[9\]，且有些语言的工具格和共格在[形态或](../Page/词法学.md "wikilink")[句法](../Page/句法.md "wikilink")上一致，所以两种格难以区分。如英语里面，介词 with 既可以表示共格语义，也可以表示工具格语义。

俄语会将伴随对象降为工具格，并在前加介词  (s)，这个介词不会有工具格变化；而工具同样为工具格，它前面不会有介词。\[11\]

### 共格与附格

有时候共格也会与混淆，因为早期常用“附格”来指代“伴随者-伴随对象”这一语义关系，至今仍存在这种用法。\[12\]附格的语义关系是“X 以及与 X 相关的集合”，在这里“伴随者-伴随对象”的语义关系并不明显，如[匈牙利语](../Page/匈牙利语.md "wikilink")。\[13\]

## 表达手段

### 句法手段

但共格语义也常用[介词等句法手段来表达](https://zh.wikipedia.org/wiki/介词 "wikilink")，英语和法语即属于此类。\[18\]如法语使用介词 avec：

|     |                |      |       |
| --- | -------------- | ---- | ----- |
| 5\. | avec           | sa   | mère. |
|     | ****           | **** | 妈妈    |
|     | **共格**         |      |       |
|     | 和他/她的妈妈。\[19\] |      |       |

共格语义也可以通过[助动词来表达](../Page/助動詞.md "wikilink")，如[拉脱维亚语](../Page/拉脱维亚语.md "wikilink")。\[20\]此外，[连动结构也可以表达共格语义](../Page/連動結構.md "wikilink")，典型如[汉语](../Page/汉语.md "wikilink")。\[21\]

[Entry.jpg](https://zh.wikipedia.org/wiki/File:Entry.jpg "fig:Entry.jpg")（MER）的[整流罩在进入火星大气时情形的艺术想象图](https://zh.wikipedia.org/wiki/整流罩 "wikilink")\]\] **进入大气（层）**，是指自然物体或人造物体从[外層空间进入](https://zh.wikipedia.org/wiki/外層空间 "wikilink")[行星](../Page/行星.md "wikilink")[大气层](../Page/大气层.md "wikilink")的运动过程。如果人造物体（如[人造卫星](https://zh.wikipedia.org/wiki/人造卫星 "wikilink")、[飞船](https://zh.wikipedia.org/wiki/飞船 "wikilink")、[火箭](../Page/火箭.md "wikilink")[导弹](../Page/导弹.md "wikilink")、[空天飞机等](https://zh.wikipedia.org/wiki/空天飞机 "wikilink")）离开[地球大气层](../Page/地球大气层.md "wikilink")，再从[外太空重新进入地球大气层的运动](https://zh.wikipedia.org/wiki/外太空 "wikilink")，称为“**再入**”（reentry）大气层。

例如，[流星進入地球大氣墜落的](https://zh.wikipedia.org/wiki/流星 "wikilink")[隕石](../Page/隕石.md "wikilink")不算再入。

## 原理

人造或者自然物体从[宇宙空间进入](https://zh.wikipedia.org/wiki/宇宙空间 "wikilink")[天体的](https://zh.wikipedia.org/wiki/天体 "wikilink")[大气层](../Page/大气层.md "wikilink")的过程被称作进入大气层（Atmospheric entry），在[地球](../Page/地球.md "wikilink")的场合指的是从宇宙空间一侧越过[海拔](../Page/海拔.md "wikilink")为100km的[卡门线的过程](https://zh.wikipedia.org/wiki/卡门线 "wikilink")。从地面发射后离开大气层的人造航天载具重新进入大气层的过程被称作**再入大气层**（Atmospheric reentry）或**再入**（reentry）。

返回大气层根据其目的和过程被分为以下类型：

| 目的（原因）                                                                                                             | 过程是否可控       | 是否破坏性 |
| ------------------------------------------------------------------------------------------------------------------ | ------------ | ----- |
| 把[航天器](../Page/航天器.md "wikilink")安全降落到行星表面                                                                         | 是            | 否     |
| [洲际弹道导弹的](https://zh.wikipedia.org/wiki/洲际弹道导弹 "wikilink")[弹道飞行后半程](https://zh.wikipedia.org/wiki/弹道飞行 "wikilink") | 是            | 是     |
| 人为消灭航天器或[太空垃圾](../Page/太空垃圾.md "wikilink")                                                                         | 否，或任由其轨道自然衰减 | 是     |
| 因[太空碰撞等意外而导致的返回](https://zh.wikipedia.org/wiki/太空碰撞 "wikilink")                                                    | 否            | 是     |

返回式航天器的设计以安全可控地回到地面为目的。由于在目前的技术条件下返回大气层时航天器的速度极高，因此非破坏性返回的过程一般需要有特殊的措施来保护航天器避免受到[气动力加热和震动](https://zh.wikipedia.org/wiki/气动力加热 "wikilink")、冲击等损害。由于[载人航天一定有](https://zh.wikipedia.org/wiki/载人航天 "wikilink")[航天员返回地面的过程](https://zh.wikipedia.org/wiki/航天员 "wikilink")，因此这一过程也成为载人航天中风险较高的环节之一。

## 歷史

這種雙層隔熱板概念在1920年由[羅伯特·戈達德提出](https://zh.wikipedia.org/wiki/羅伯特·戈達德 "wikilink")，他說："[流星進入大氣層的速度高達每秒](https://zh.wikipedia.org/wiki/流星 "wikilink")30英里，但內部依然寒冷。因此，假如再返回物的表面覆蓋一層抗高溫（不易變質及難熔解）的物質後再用一層不太會導熱的耐高溫物質，這樣物體表面就不會受到太多的侵蝕"（節錄）

而第一次實際應用到此系統是在洲际彈道導彈的再入速度增加所導致的气动加熱。早期的彈道導彈，如[V2火箭](https://zh.wikipedia.org/wiki/V2火箭 "wikilink")，並沒有此問題。而中程彈道導彈，如蘇聯的[R-5](https://zh.wikipedia.org/wiki/R-5 "wikilink")（有1200km）的射程，就需要陶瓷複合材料來保護。首個[洲際彈道導彈](https://zh.wikipedia.org/wiki/洲際彈道導彈 "wikilink")（ICBM，射程達8000至12000km），則已正式進入了現代保護材料的時代。在美國，這技術是由H. Julian Allen再Ames Research率先開發。而蘇聯的Yuri A. Dunaev也曾在列寧格勒物理技術研究所開發類似的技術。

## 飛行器的形狀

### 鈍形飛行器

美國國家航空諮詢委員會的、在1951年發現了鈍形（高阻）[防热大底具有最佳效果](https://zh.wikipedia.org/wiki/防热盾 "wikilink")。因為返回式航天器的气动加熱與[阻力係數成反比](https://zh.wikipedia.org/wiki/阻力係數 "wikilink")，即阻力愈大，熱負荷愈低。钝形再入舱使得气体不能快速离开，成为气垫层隔开了冲击波与加热振动层，使得大部分热空气不再直接接触再入舱，热能保持在冲击波气体中并在大气层中扩散。

艾倫和埃格斯的發現，最初被視為軍事秘密，但於1958年出版。鈍形理論的設計成為可行的隔熱板，都體現在水星、雙子星和阿波羅太空艙，使宇航員返回火熱的地球大氣層時仍生存。蘇聯的[R-7洲際彈道導彈於](https://zh.wikipedia.org/wiki/R-7 "wikilink")1957年使用尖鼻的彈頭成功首次試射，但擊中目標區10公里以外，因而改為鈍鼻的彈頭。蘇聯的隔熱層由多層玻璃纖維與石棉textolite組成。

## 激波层气体物理

防热大底设计的经验规则是：[激波层气体峰值温度](https://zh.wikipedia.org/wiki/激波层 "wikilink")（[开尔文](../Page/开尔文.md "wikilink")为单位）等于进入速度（单位m/s）。例如，宇宙飞船以7.8 km/s进入大气层，激波层气体峰值温度7800 K。因为[动能](../Page/动能.md "wikilink")增加与速度的平方成正比，而气体的[比热容随温度而大幅度增加](https://zh.wikipedia.org/wiki/比热容 "wikilink")（这不同于固体在通常条件下可以假设比热容不变）。

### 理想气体模型

几乎所有的航宇工程师在本科时学过[理想气体模型](https://zh.wikipedia.org/wiki/理想气体 "wikilink")。大部分的理想气体方程与对应的表与图在NACA Report 1135中给出。\[3\] NACA Report 1135的摘要经常出现在热动力学课本的附录被航宇工程师熟悉。

理想气体理论非常精巧并在设计飞机时非常有用，但它假定气体是化学惰性的。从飞机设计角度，大气温度低于550 K时可以假定气体为惰性的。气温550 K时理想气体理论开始出现问题而气温超过2000 K将不再适用，这时防热大底设计者必须使用*真实气体模型*.

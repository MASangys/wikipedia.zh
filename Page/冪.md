**幂運算**（），又稱**指數運算**，是[數學](https://zh.wikipedia.org/wiki/數學 "wikilink")[運算](https://zh.wikipedia.org/wiki/運算 "wikilink")，[表達式為](https://zh.wikipedia.org/wiki/表達式 "wikilink")\(b^n\)，讀作“\(b\)的\(n\)次方”或「\(b\)的\(n\)次幂」。其中，\(b\)稱為[底數](https://zh.wikipedia.org/wiki/底數 "wikilink")，而\(n\)稱為[指數](https://zh.wikipedia.org/wiki/指數 "wikilink")，通常指數寫成[上標](https://zh.wikipedia.org/wiki/上標 "wikilink")，放在底數的[右邊](https://zh.wikipedia.org/wiki/右邊 "wikilink")。當不能用上標時，例如在[編程語言或](https://zh.wikipedia.org/wiki/編程語言 "wikilink")[電子郵件中](https://zh.wikipedia.org/wiki/電子郵件 "wikilink")，\(b^n\)通常寫成**b^n**或**b\*\*n**；也可視為[超運算](https://zh.wikipedia.org/wiki/超運算 "wikilink")，記為**b\[3\]n**；亦可以用[高德納箭號表示法](../Page/高德納箭號表示法.md "wikilink")，寫成**b↑n**。

若*n*為正整數,
可以把\(b^n\)看作[乘方的结果](https://zh.wikipedia.org/wiki/乘方 "wikilink"),
等同於\(b\)自乘\(n\)次。

\[b^n = \underbrace{b \times \cdots \times b}_n\]

當指數為1時，通常不寫出來，因為運算出的值和底數的[數值一樣](https://zh.wikipedia.org/wiki/數值 "wikilink")；指數為2時，可以讀作“\(b\)的[平方](../Page/平方数.md "wikilink")”；指數為
3 時，可以讀作“\(b\)的[立方](https://zh.wikipedia.org/wiki/立方 "wikilink")”。

起始值1（乘法的[單位元](../Page/單位元.md "wikilink")）乘上底數（\(b\)）自乘指數（\(n\)）這麼多次。這樣[定義了後](https://zh.wikipedia.org/wiki/定義 "wikilink")，很易想到如何一般化指數
0 和負數的情況：除 0 外所有數的零次方都是 1
；指數是負數時就等於重複[除以底數](../Page/除法.md "wikilink")（或底數的[倒數自乘指數這麼多次](https://zh.wikipedia.org/wiki/倒數 "wikilink")），即：

\[b^{0} = 1 \qquad\]

\[b^{-n} = {1 \over \underbrace{b\times\cdots\times b}_n} = \frac{1}{b^n} = \left(\frac{1}{b}\right)^{n} \qquad (b \ne 0)\]。

以[分數為指數的冪定義為](../Page/分數.md "wikilink")\(b^{\frac{m}{n}} = \sqrt[n]{b^{m}}\)，即\(b\)的\(m\)次方再开\(n\)次[方根](../Page/方根.md "wikilink")

目前沒有[數學家給予正式的定義](https://zh.wikipedia.org/wiki/數學家 "wikilink")。在部分[數學領域中](https://zh.wikipedia.org/wiki/數學領域 "wikilink")，如[組合數學](https://zh.wikipedia.org/wiki/組合數學 "wikilink")，常用的[慣例是定義為](https://zh.wikipedia.org/wiki/慣例 "wikilink")
1 ，也有人[主張定義為](https://zh.wikipedia.org/wiki/主張 "wikilink") 1 。

因為在[十进制](../Page/十进制.md "wikilink")，[十的次方很易計算](https://zh.wikipedia.org/wiki/十的次方 "wikilink")，只需在後面加零即可，所以[科学记数法借此](../Page/科学记数法.md "wikilink")[簡化](https://zh.wikipedia.org/wiki/簡化 "wikilink")[記錄的數字](https://zh.wikipedia.org/wiki/記錄 "wikilink")；[二的幂在](https://zh.wikipedia.org/wiki/二的幂 "wikilink")[計算機科學相當重要](https://zh.wikipedia.org/wiki/計算機科學 "wikilink")。

當*n*是[複數及](https://zh.wikipedia.org/wiki/複數 "wikilink")*b*是正實數時,

\[b^n = \exp(n \ln(b))\]

exp是[指數函數而](https://zh.wikipedia.org/wiki/指數函數 "wikilink")
ln是[自然對數](../Page/自然對數.md "wikilink")。

## 重要的[恆等式](https://zh.wikipedia.org/wiki/恆等式 "wikilink")

### 运算法则

  - 同底数幂相乘，底数不变，指数相加：

\[a^m \times a^n = a^{m + n}\]

  - 同底数幂相除，底数不变，指数相减：

\[a^m \div a^n = a^{m - n}\]

  - 同指数幂相除，指数不变，底数相除：

\[\frac{a^n}{b^n} = \left(\frac{a}{b}\right)^n\]

### 其他等式

  - \(a^\frac{m}{n} = \sqrt[n]{a^m}\)
  - \(x^{-m} = \frac{1}{x^m} \qquad (x \ne 0)\)
  - \(x^0 = 1 \qquad (x \ne 0)\)
  - \(x^1 = x\,\!\)
  - \(x^{-1} = \frac{1}{x} \qquad (x \ne 0)\)

## 运算律

加法和乘法存在[交换律](https://zh.wikipedia.org/wiki/交换律 "wikilink")，比如：\(2+3=5=3+2\)，\(2 \times 3 =6 =3 \times 2\)，但是幂的运算不存在交换律，\(2^3 = 8\)，但是\(3^2 = 9\)。

同样，加法和乘法存在[结合律](../Page/结合律.md "wikilink")，比如：\((2+3)+4=9=2+(3+4)\)，\((2 \times 3) \times 4=24=2 \times (3 \times 4)\)，儘管\((2^3)^4 = 8^4 = 4096\)，但是\(2^{(3^4)} = 2^{81} = 2,417,851,639,229,258,349,412,352\)。

## 整数指数幂

整数指数幂的运算只需要[初等代数的知识](https://zh.wikipedia.org/wiki/初等代数 "wikilink")。

### 正整数指数幂

表达式\(a^2 = a\cdot a\)被称作\(a\)的[平方](../Page/平方.md "wikilink")，因为边长为*\(a\)*的正方形面积是\(a^2\)。

表达式\(a^3 = a\cdot a\cdot a\)被称作*\(a\)*的[立方](https://zh.wikipedia.org/wiki/立方 "wikilink")，因为邊长为*\(a\)*的正方体体积是\(a^3\)。

所以\(3^2\)读作*3的平方*，\(2^3\)读作*2的立方*。

指数表示的是底数反复相乘多少次。比如\(3^5 = 3\times 3\times 3\times 3\times 3 = 243\)，指数是5，底数是3，表示3反复相乘5次。

或者，整数指数幂可以[递归地定义成](../Page/递归.md "wikilink")：

\[a^n=
\begin{cases}
1 & (n= 0) \\
a \cdot a^{n-1} & (n> 0) \\
\left(\frac{1}{a}\right)^{-n} & (n< 0)
\end{cases}\]

### 指数是1或者0

注意\(3^1\)表示仅仅1个3的乘积，就等于3。

注意\(3^5 = 3\times 3^4\)，\(3^4 = 3\times 3^3\)，\(3^3 = 3\times 3^2\)，\(3^2=3\times 3^1\)，

继续，得到\(3^1 = 3\times 3^0 = 3\)，所以\(3^0 = 1\)

另一个得到此结论的方法是：通过运算法则\(\frac{x^n}{x^m} = x^{n - m}\)

当\(m=n\)时，\(1 = \frac{x^n}{x^n} = x^{n - n} = x^0\)

  - 任何数的**1**次方是它本身。

### 负数指数

我们定义任何不为**0**的数的*-1*次方等于它的倒数。

\[a^{-1} = \frac{1}{a}.\]

对于非零**\(a\)**定义\(a^{-n} = \frac{1}{a^n}\)。因为当\(a=0\)时分母是**0**而没有意义。

这个定义是因为\(a^m\cdot a^n = a^{m+n}\)，当\(m=-n\)时

\[a^{-n} \, a^{n} = a^{-n\,+\,n} = a^0 = 1,\]
因为\(a^0\)已经定义了，所以\(a^{-n} = \frac{1}{a^{n}}\)。

或者还可以像定义***\(a\)***的*0*次方一样定义：

通过运算法则\(\frac{x^m}{x^n} = x^{m - n}\)

当\(m=0\)时，可以约去分子得\(x^{-n} = x^{0-n} = \frac{x^0}{x^n}\)

负数指数\(a^{-n}\)还可以表示成**1**连续除以\(n\)个\(a\)。比如：

\[3^{-4} = \frac{\frac{\frac{\frac{1}{3}}{3}}{3}}{3} = \frac{1}{81} = \frac{1}{3^{4}}\].

#### 10的幂

在[十进制的计数系统中](../Page/十进制.md "wikilink")，10的幂写成1后面跟着很多个0。例如：\(10^3 = 1000,\ 10^{-3} = 0.001\)

因此10的幂用来表示非常大或者非常小的数字。如：299,792,458（真空中[光速](../Page/光速.md "wikilink")，单位是[米每秒](https://zh.wikipedia.org/wiki/米每秒 "wikilink")），可以写成
\(2.99792458\times 10^8\)，[近似值](https://zh.wikipedia.org/wiki/近似值 "wikilink")
\(2.998\times 10^8\) 或 \(3\times 10^8\).

[国际单位制词头也使用](../Page/国际单位制词头.md "wikilink")10的幂来描述特别大或者特别小的数字，比如：词头“千”就是
\(10^3\)，词头“毫”就是 \(10^{-3}\)

#### 2的幂

#### 1的幂

1的任何次幂都为1

#### 0的幂

0的正数幂都等于0。

0的负数幂没有定义。

任何非0之数的0次方都是1；而0的0次方是懸而未決的，某些領域下常用的慣例是約定為1。\[1\]但某些教科書表示0的0次方為無意義。\[2\]也有人主張定義為1。

#### 负1的幂

\-1的奇数幂等于-1

\-1的偶数幂等于1

### 指数非常大时的幂

一个大于1的数的幂趋于[无穷大](https://zh.wikipedia.org/wiki/无穷大 "wikilink")，一个小于-1的数的幂趋于负无穷大

  -
    当\(a > 1\)，\(n \to \infty\)，\(a^n \to \infty\)
    当\(a < -1\)，\(n \to \infty\)，\(a^n \to -\infty\) 或 \(\infty\) , (視乎n
    是奇數或偶數)

一个绝对值小于1的数的幂趋于0

  -
    当\(|a| < 1\)，\(n \to \infty\)，\(a^n \to 0\)

1的幂永远都是1

  -
    当\(a = 1\)，\(n \to \infty\)，\(a^n \to 1\)

如果数*a*趋于1而它的幂趋于无穷，那么极限并不一定是上面几个。一个很重要的例子是：

  -
    当\(n \to \infty,  \left(1+\frac{1}{n}\right)^n \to e\)

参见[e的幂](https://zh.wikipedia.org/wiki/#e的幂 "wikilink")

其他指数的极限参见[幂的极限](https://zh.wikipedia.org/wiki/#幂的极限 "wikilink")

## 正实数的实数幂

一个正实数的[实数幂可以通过两种方法实现](../Page/实数.md "wikilink")。

  - [有理数幂可以通过N次](../Page/有理数.md "wikilink")[方根定义](../Page/方根.md "wikilink")，任何非0实数次幂都可以这样定义
  - [自然对数可以被用来通过指数函数定义实数幂](https://zh.wikipedia.org/wiki/自然对数 "wikilink")

### N次方根

[Root_graphs.svg](https://zh.wikipedia.org/wiki/File:Root_graphs.svg "fig:Root_graphs.svg")

一个[数](../Page/数.md "wikilink")\(a\)的\(n\)次方根是\(x\)，*\(x\)*使\(x^n=a\)。

如果*\(a\)*是一个正实数，*\(n\)*是正整数，那么方程\(x^n=a\)只有一个正实数[根](https://zh.wikipedia.org/wiki/根_\(数学\) "wikilink")。
这个根被称为*\(a\)*的*\(n\)*次方根，记作：\(\sqrt[n]{a}\)，其中\(\sqrt{\ }\)叫做根号。或者，*\(a\)*的*\(n\)*次方根也可以写成\(a^{\frac{1}{n}}\).
例如\(4^{\frac{1}{2}} = 2,\ 8^{\frac{1}{3}} = 2\)

当指数是\(\frac{1}{2}\)时根号上的2可以省略，如：\(\sqrt{4}=4^{\frac{1}{2}}= \sqrt[2]{4} = 2\)

### 有理数幂

有理数指数通常可以理解成

\[a^{\frac{m}{n}} = (a^m)^{\frac{1}{n}} = \sqrt[n]{a^m}\]

### e的幂

这个重要的数学常数[*e*](../Page/E_\(数学常数\).md "wikilink")，有时叫做[欧拉数](https://zh.wikipedia.org/wiki/欧拉数 "wikilink")，近似2.718，是[自然对数的底](https://zh.wikipedia.org/wiki/自然对数 "wikilink")。它提供了定义非整数指数幂的一个方法。
它是从以下极限定义的：

\[e =\lim_{n \to \infty} \left(1+\frac{1}{n} \right)^n\]

**[指数函数](../Page/指数函数.md "wikilink")**的定义是：

\[e^x =\lim_{n \to \infty} \left(1+\frac{x}{n} \right)^n\]

可以很简单地证明*e*的正整数*k*次方\(e^k\)是：

\[e^k = \left[\lim_{n \to \infty} \left(1+\frac{1}{n} \right) ^n\right]^k\]

\[= \lim_{n \to \infty} \left[\left(1+\frac{1}{n} \right) ^n\right]^k\]

\[= \lim_{n \to \infty} \left(1+\frac k {n\cdot k} \right)^{n \cdot k}\]

\[= \lim_{n \cdot k \to \infty} \left(1+\frac k {n\cdot k} \right)^{n \cdot k}\]

\[= \lim_{m \to \infty} \left(1+\frac k m \right)^m\]

### 实数指数幂

[Expo02.svg](https://zh.wikipedia.org/wiki/File:Expo02.svg "fig:Expo02.svg")
因为所有[实数可以近似地表示为有理数](../Page/实数.md "wikilink")，任意实数指数*x*可以定义成\[3\]：

\[b^x = \lim_{r \to x} b^r,\]

例如：

\[x \approx 1.732\] 于是

\[5^x \approx 5^{1.732} =5^{\frac{433}{250}}=\sqrt[250]{5^{433}} \approx 16.241\]

实数指数幂通常使用对数来定义，而不是近似有理数。

[自然对数](https://zh.wikipedia.org/wiki/自然对数 "wikilink")\(\ln{x}\)是指数函数\(e^x\)的[反函数](https://zh.wikipedia.org/wiki/反函数 "wikilink")。
它的定义是：对于任意\(b>0\)，满足

\[b = e^{\ln b}\] 根据对数和指数运算的规则：

\[b^x = (e^{\ln b})^x = e^{x \cdot\ln b}\] 这就是实数指数幂的定义：

\[b^x = e^{x\cdot\ln b}\,\]

实数指数幂\(b^x\)的这个定义和上面使用有理数指数和连续性的定义相吻合。对于复数，这种定义更加常用。

## 负实数的实数幂

如果\(a\)是负数且\(n\)是[偶数](https://zh.wikipedia.org/wiki/偶数 "wikilink")，那么\(x^n = a\)是正數。
如果*\(a\)*是负数且*\(n\)*是[奇数](https://zh.wikipedia.org/wiki/奇数 "wikilink")，那么\(x^n = a\)是负数。

使用对数和有理数指数都不能将\(a^k\)（其中*\(a\)*是负实数，\(k\)实数）定义成实数。在一些特殊情况下，给出一个定义是可行的：负指数的整数指数幂是实数，有理数指数幂对于\(a^\frac{m}{n}\)（\(n\)是奇数）可以使用*\(n\)*次方根来计算，但是因为没有实数\(x\)使\(x^2 = -1\)，对于\(a^\frac{m}{n}\)（*\(n\)*是偶数）时必须使用[虚数单位](https://zh.wikipedia.org/wiki/虚数单位 "wikilink")\(i\)。

使用对数的方法不能定义\(a\leq 0\)时的\(a^k\)为实数。实际上，\(e^x\)对于任何实数*\(x\)*都是正的，所以\(\ln(a)\)对于负数没有意义。

使用有理数指数幂来逼近的方法也不能用于负数\(a\)因为它依赖于[连续性](https://zh.wikipedia.org/wiki/连续性 "wikilink")。函数\(f(r) = a^r\)对于任何正的有理数\(a\)是连续的，但是对于负数*\(a\)*，函数\(f\)在有些有理数\(r\)上甚至不是连续的。

例如：当\(a=-1\)，它的奇数次根等于-1。所以如果*\(n\)*是正奇数整数，\(-1^{\frac m n}=-1\)当\(m\)是奇数，\(-1^{\frac m n}=1\)当*\(m\)*是偶数。虽然有理数\(q\)使\(-1^q=1\)的[集合是](https://zh.wikipedia.org/wiki/集合 "wikilink")[稠密集](../Page/稠密集.md "wikilink")，但是有理数\(q\)使\(-1^q=-1\)的[集合也是](https://zh.wikipedia.org/wiki/集合 "wikilink")。所以函数\(-1^q\)在有理数域不是连续的。

[复数运算的几何意义和](https://zh.wikipedia.org/wiki/复数 "wikilink")[*e*的幂可以帮助我们理解](https://zh.wikipedia.org/wiki/#e的幂 "wikilink")\(e^{ix}\)（*\(x\)*是实数），即[純虛數指數函數](https://zh.wikipedia.org/wiki/純虛數指數函數 "wikilink")。想象一个[直角三角形](../Page/直角三角形.md "wikilink")\((0,1,1+\frac{ix}{n})\)（括号内是复数平面内三角形的三个顶点），对于足够大的\(n\)，这个三角形可以看作一个[扇形](../Page/扇形.md "wikilink")，这个扇形的中心角就等于\(\frac{x}{n}\)[弧度](https://zh.wikipedia.org/wiki/弧度 "wikilink")。对于所有\(k\)，三角形\((0,(1+\frac{ix}{n})^k, (1+\frac{ix}{n})^{k+1})\)互为[相似三角形](../Page/相似三角形.md "wikilink")。所以当*\(n\)*足够大时\((1+\frac{ix}{n})^n\)的极限是复数平面上的[单位圆上](../Page/单位圆.md "wikilink")\(x\)弧度的点。这个点的[极坐标是](https://zh.wikipedia.org/wiki/极坐标 "wikilink")\((r,\theta)=(1,x)\)，[直角坐标是](https://zh.wikipedia.org/wiki/直角坐标 "wikilink")\((\cos x,\sin x)\)。所以\(e^{ix} = \cos x + i \sin x\)，而這個函數可以稱為[純虛數指數函數](https://zh.wikipedia.org/wiki/純虛數指數函數 "wikilink")。这就是[欧拉公式](../Page/欧拉公式.md "wikilink")，它通过[复数的意义将](https://zh.wikipedia.org/wiki/复数 "wikilink")[代数学和](https://zh.wikipedia.org/wiki/代数学 "wikilink")[三角学联系起来了](https://zh.wikipedia.org/wiki/三角学 "wikilink")。

等式\(e^z = 1\)的解是一个整数乘以\(2i\pi\)<ref>This definition of a principal root

更一般地，如果\(e^b = a\)，那么\(e^z = a\)的每一个解都可以通过将\(2i\pi\)的整数倍加上\(b\)得到：

### 三角函数

根据[欧拉公式](../Page/欧拉公式.md "wikilink")，[三角函数余弦和正弦是](../Page/三角函数.md "wikilink")：

\[\cos z = \frac{ e^{i\cdot z} + e^{-i\cdot z}} {2} \qquad \sin z = \frac{e^{i\cdot z} - e^{-i\cdot z}}{2\cdot i}\]

历史上，在复数发明之前，余弦和正弦是用几何的方法定义的。上面的公式将复杂的三角函数的求和公式转换成了简单的指数方程

\[e^{i\cdot (x+y)}=e^{i\cdot x}\cdot e^{i\cdot y}.\,\]
使用了复数指数幂之后，很多三角学问题都能够使用代数方法解决。

### e的复数指数幂

\(e^{x+iy}\)可以分解成\(e^x\cdot e^{iy}\)。其中\(e^x\)是\(e^{x+iy}\)的[模](../Page/模.md "wikilink")，\(e^{iy}\)决定了\(e^{x+iy}\)的方向

### 正实数的复数幂

如果\(a\)是一个正实数，\(z\)是任何复数，\(a^z\)定义成\(e^{z\cdot \ln(a)}\)，其中\(x=\ln (a)\)是方程\(e^x = a\)的唯一解。所以处理实数的方法同样可以用来处理复数。

例如：

\[2^i = e^{i\cdot \ln(2)} = \cos{\ln 2}+i\cdot \sin{\ln 2 } = 0.7692+0.63896i\]

\[e^i = 0.54030+0.84147i\]

\[10^i = -0.66820+0.74398i\]

\[(e^{2\pi})^i = 535.49^i = 1\]

## 在[函數中](https://zh.wikipedia.org/wiki/函數 "wikilink")

當函數名後有上標的數（即函數的指數），一般指要重複它的運算。例如\(f^3 (x )\)即\(f ( f ( f ( x ) ) )\)。特別地，\(f^{-1} (x )\)指\(f (x )\)的[反函數](../Page/反函數.md "wikilink")。

但[三角函数的情況有所不同](../Page/三角函数.md "wikilink")，一個正指數應用於函數的名字時，指答案要進行乘方運算，而指數為-1時则表示其反函數。例如：\((\sin x)^{-1}\)表示\(\csc x\)。因此在三角函數時，使用\(\sin^{-1} x\)來表示\(\sin x\)的反函數\(\arcsin x\)。

## 计算自然数（正整数）*\(n\)*的\(a^n\)的算法

最快的方式计算\(a^n\)，当\(n\)是正整数的时候。它利用了测试一个数是奇数在计算机上是非常容易的，和通过简单的移所有位向右来除以2的事实。

[偽代碼](https://zh.wikipedia.org/wiki/偽代碼 "wikilink")：

``` text
   1. 1 → y, n → k, a → f
   2.若k不為0,執行3至6
     3.若k為奇數, y * f → y
     4. k [[位操作#移位|右移]]1位（即k / 2 → k ,小數點無條件捨去）
     5. f * f → f
     6.回到2
   7.傳回y
```

在[C](https://zh.wikipedia.org/wiki/C語言 "wikilink")/[C++语言中](https://zh.wikipedia.org/wiki/C++语言 "wikilink")，你可以写如下算法：

``` c
   double power (double a, unsigned int n)
   {
        double y = 1;
        double f = a;
        unsigned int k = n;
        while (k != 0) {
           if (k % 2 == 1) y *= f;
           k >>= 1;
           f *= f;
        }
        return y;
   }
```

此算法的[時間複雜度為](https://zh.wikipedia.org/wiki/時間複雜度 "wikilink")\(\Omicron(\log n)\!\)，比普通算法快（a自乘100次，[時間複雜度為](https://zh.wikipedia.org/wiki/時間複雜度 "wikilink")\(\Omicron(n)\!\)），在\(n\)較大的時候更為顯著。

例如計算\(a^{100}\)，普通算法需要算100次，上述算法則只需要算7次。若要計算\(a^{n} (n < 0)\)可先以上述算法計算\(a^{|n|}\)，再作倒數。

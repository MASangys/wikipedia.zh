**凯氏定氮法**（，全称**凯耶达尔定氮法**，简称**凯氮法**）是[分析化学](../Page/分析化学.md "wikilink")中一种常用的确定[有机化合物](../Page/有机化合物.md "wikilink")中[氮](../Page/氮.md "wikilink")含量的检测方法。这种方法是由[凯耶达尔](../Page/凯耶达尔.md "wikilink")在1883年发明的。\[1\]

## 方法

将有机化合物与[硫酸](../Page/硫酸.md "wikilink")共热使其中的氮转化为[硫酸铵](../Page/硫酸铵.md "wikilink")。在这一步中，经常会向混合物中加入[硫酸钾](../Page/硫酸钾.md "wikilink")来提高中间产物的[沸点](../Page/沸点.md "wikilink")（从337[℃到](../Page/摄氏温标.md "wikilink")373℃）。样本的[分解过程的终点很好判断](https://zh.wikipedia.org/wiki/分解 "wikilink")，因为这时混合物会变得无色且透明（开始时很暗）。

在得到的溶液中加入少量[氢氧化钠](../Page/氢氧化钠.md "wikilink")，然后[蒸馏](../Page/蒸馏.md "wikilink")。这一步会将铵盐转化成[氨](../Page/氨.md "wikilink")。而总氨量（由样本的含氮量直接决定）会由[反滴定法确定](../Page/滴定.md "wikilink")：[冷凝管](../Page/冷凝管.md "wikilink")的末端会浸在[硼酸](../Page/硼酸.md "wikilink")溶液中。氨会和酸反应，而过量的酸则会在[甲基橙](../Page/甲基橙.md "wikilink")的指示下用[碳酸钠](../Page/碳酸钠.md "wikilink")[滴定](../Page/滴定.md "wikilink")。滴定所得的结果乘以特定的[转换因子就可以得到结果](https://zh.wikipedia.org/wiki/#转换因子 "wikilink")。

通过凯氏定氮法测得的含氮量一般被称作[总凯氮量](https://zh.wikipedia.org/wiki/总凯氮量 "wikilink")。总凯氮量有时并不能真正地反映样本中的蛋白质含量，因为所测定的部分含氮量可能不是由蛋白质转化来的。

<!-- end list -->

  -
    铵盐的分解： (NH<sub>4</sub>)<sub>2</sub>SO<sub>4</sub>(aq) + 2NaOH → Na<sub>2</sub>SO<sub>4</sub>(aq) + 2H<sub>2</sub>O(l) + 2NH<sub>3</sub>(g)

<!-- end list -->

  -
    氨的固定： B(OH)<sub>3</sub> + H<sub>2</sub>O + NH<sub>3</sub> → NH<sub>4</sub><sup>+</sup> + B(OH)<sub>4</sub><sup>–</sup>

<!-- end list -->

  -
    反滴定： B(OH)<sub>3</sub> + H<sub>2</sub>O + Na<sub>2</sub>CO<sub>3</sub> → NaHCO<sub>3</sub>(aq) + NaB(OH)<sub>4</sub>(aq)

如今，凯氏定氮法已经可以自动化进行，一些[催化剂](../Page/催化剂.md "wikilink")（如[氧化汞](../Page/氧化汞.md "wikilink")或[硫酸铜](../Page/硫酸铜.md "wikilink")）也被使用来提高反应速率。

## 转换因子

转换因子是通过凯氏定氮法测得的氮换算为蛋白质的系数。理想状态下的蛋白质中的氮含量约为16%，所以测量值乘以6.25即为蛋白质量。下表为总结出的转换因子数值。

| 样本          | 转换因子 |
| ----------- | ---- |
| 乳制品         | 6.38 |
| 面粉          | 5.70 |
| 玉米、高粱       | 6.24 |
| 花生          | 5.46 |
| 大米          | 5.95 |
| 大豆          | 5.71 |
| 肉           | 6.25 |
| 大麦、小米、燕麦、裸麦 | 5.83 |
| 芝麻、向日葵      | 5.30 |

## 应用

凯氏定氮法的普遍适用性、精确性和可重复性已经得到了国际的广泛认可。它已经被确定为检测食品中蛋白质含量的标准方法。但是，这种方法并不能给出真实的蛋白质含量，因为所测定的氮可能不仅仅是由蛋白质转化来的。这可以从[2007年美国宠物食品污染事件和](https://zh.wikipedia.org/wiki/2007年美国宠物食品污染事件 "wikilink")[2008年中国毒奶粉事件等食品安全事件中被体现](https://zh.wikipedia.org/wiki/2008年中国毒奶粉事件 "wikilink")：[三聚氰胺](../Page/三聚氰胺.md "wikilink")，一种含氮量较高的物质，被添加到了食品中以伪造较高的含氮量。另外，由于不同的[氨基酸](../Page/氨基酸.md "wikilink")序列，这种方法也需要许多不同的校正因子。而且，凯氏定氮法还需要使用浓硫酸和较长时间的加热（一般来讲，大于1小时）。这也造成了在粗略测量蛋白质含量时，有时更方便些。\[2\]

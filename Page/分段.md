在數學中，**分段**定義的[函數稱為](https://zh.wikipedia.org/wiki/函數 "wikilink")**分段函數**，是由多個子函數而[定義的](https://zh.wikipedia.org/wiki/定義 "wikilink")，施加到主函數的域的一定的時間間隔的每個子函數（子域）。分段實際上是一種表達函數的方式，而不是函數本身的一個特徵，但是具有額外的限定，可以描述函數的本質。例如，分段[多項式](../Page/多項式.md "wikilink")函數是在其每個子域上是多項式的函數，但是每個子域上可能是不同的。 字分段也用來描述適用於每件分段定義的函數的任何屬性，但不一定保持為函數的整個域。一個函數是分段[微分](../Page/微分.md "wikilink")的或分段[連續微分的](https://zh.wikipedia.org/wiki/連續 "wikilink")，如果每個子塊在整個子域內是可區分的，即使整個函數在塊之間的[點上可能是不可區分的](https://zh.wikipedia.org/wiki/點 "wikilink")。在中，[導數的概念可以被分段函數的子導數的概念取代](https://zh.wikipedia.org/wiki/導數 "wikilink")。儘管分段定義中的“塊”不一定是間隔，但是除非是間隔，否則函數不被稱為**分段[線性](https://zh.wikipedia.org/wiki/線性 "wikilink")**、**分段連續**或**分段可微**。

## 符號和解釋

[Absolute_value.svg](https://zh.wikipedia.org/wiki/File:Absolute_value.svg "fig:Absolute_value.svg") 分段函數使用通用函數表示法來定義，其中函數的主體是函數和相關子域的[數組](https://zh.wikipedia.org/wiki/數組 "wikilink")。至關重要的是，在大多數情況下，只有有限數量的子域，每個子域必須是一個區間，以便將整個函數稱為**分段**。例如，考慮[絕對值函數的分段定義](https://zh.wikipedia.org/wiki/絕對值 "wikilink")：

\[|x| = \begin{cases}
  -x, & \mbox{if } x < 0 \\
  x,  & \mbox{if } x \ge 0
\end{cases}\] 對於\(x\)小於零的所有值，使用第一個函數（\(-x\)），它將取消輸入值的符號，使負數為正數。對於x大於或等於零的所有值，使用第二個函數（\(x\)），它對輸入值本身進行簡單計算。 考慮分段函數\(f(x)\)在\(x\)的特定值處計算：

| \(x\)           | \(f(x)\)        | 使用的函數  |
| --------------- | --------------- | ------ |
| \-3             | 3               | \(-x\) |
| \-0.1           | 0.1             | \(-x\) |
| 0               | 0               | \(x\)  |
| \(\frac{1}{2}\) | \(\frac{1}{2}\) | \(x\)  |
| 5               | 5               | \(x\)  |

## 連續性

[Upper_semi.svg](https://zh.wikipedia.org/wiki/File:Upper_semi.svg "fig:Upper_semi.svg")組成的分段函數\]\] 如果滿足以下條件，則分段函數在給定的時間間隔內是連續的：

  - 它在整個間隔中被定義。
  - 其構成函數在該間隔上是連續的。
  - 在該區間內的子域的每個端點處不存在不連續性。

## 應用

在應用的[數學分析中](https://zh.wikipedia.org/wiki/數學分析 "wikilink")，已經發現分段函數與[人類視覺系統的許多模型一致](https://zh.wikipedia.org/wiki/视知觉 "wikilink")，其中在第一階段將圖像感知為包括由邊緣分隔的平滑區域。特別是，剪切片已經被用作表示系統來提供2D和3D中該模型類的稀疏近似。

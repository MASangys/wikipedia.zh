**切薩羅求和**（）是由義大利的數學家[恩納斯托·切薩羅](../Page/恩納斯托·切薩羅.md "wikilink")（Ernesto
Cesàro）發明，是計算[無窮級數和的方式](../Page/無窮級數.md "wikilink")。若一[級數](../Page/級數.md "wikilink")[收斂至α](../Page/收斂.md "wikilink")，則其切薩羅和存在，其值為
α，而[發散級數也可以用切薩羅求和的方式](../Page/發散級數.md "wikilink")，計算出切薩羅和。

## 定義

令 {*a*<sub>n</sub>} 為一 [數列](../Page/數列.md "wikilink")，且令

\[s_k = a_1 + \cdots + a_k\] 為數列前 *k*
項的[部份和](../Page/部份和.md "wikilink")：

\[\sum_{n=1}^\infty a_n\]. 若以下的條件成立，則此數列 {*a*<sub>n</sub>}
的切薩羅和存在，且其值為α。

\[\lim_{n\to\infty} \frac{s_1 + \cdots + s_n}{n} = \alpha\].

## 格蘭迪級數的例子

令 *a*<sub>n</sub> = (-1)<sup>n+1</sup>, *n* ≥ 1。因此{*a*<sub>n</sub>}
為以下的數列：

\[1, -1, 1, -1, \ldots\]。

其部份和組成的數列 {*s*<sub>n</sub>} 為

\[1, 0, 1, 0, \ldots\]； 此數列為[格蘭迪級數](../Page/格蘭迪級數.md "wikilink")，不會收斂。

而數列 {(*s*<sub>1</sub> + ... + *s*<sub>n</sub>)/*n*} 的各項分別為

\[\frac{1}{1}, \,\frac{1}{2}, \,\frac{2}{3}, \,\frac{2}{4}, \,\frac{3}{5}, \,\frac{3}{6}, \,\frac{4}{7}, \,\frac{4}{8}, \,\ldots\]；
當*n*趨近於[無限大](../Page/無限大.md "wikilink")，切薩羅和為如下極限：

\[\lim_{n\to\infty} \frac{s_1 + \cdots + s_n}{n} = 1/2\]。

因此，數列 {*a*<sub>n</sub>} 的切薩羅和為 1/2。

## 推廣

切薩羅在1890年發展了更廣泛的切薩羅和，表示為(C, *n*)，其中*n*為非負整數。 (C, 0) 是一般定義下的和，而(C,
1)就是上述的切薩羅和。

*n*\>1時的(C, *n*) 如下所述： 對於級數Σ*a*<sub>*n*</sub>, 定義

\[A_n^{-1}=a_n; A_n^\alpha=\sum_{k=0}^n A_k^{\alpha-1}\] 且定義
*E*<sub>*n*</sub><sup>α</sup> 為數列 1 + 0 + 0 + 0 + · · · 的
*A*<sub>*n*</sub><sup>α</sup>。 則 Σ*a*<sub>*n*</sub> 的 (C, α) 和則為

\[\lim_{n\to\infty}\frac{A_n^\alpha}{E_n^\alpha}\] 若以上數值存在。\[1\]

**初始化**（）在[计算机](https://zh.wikipedia.org/wiki/计算机 "wikilink")[编程领域中指为](https://zh.wikipedia.org/wiki/编程 "wikilink")[数据对象或](https://zh.wikipedia.org/wiki/对象_\(计算机科学\) "wikilink")[变量赋初值的做法](https://zh.wikipedia.org/wiki/变量 "wikilink")，如何初始化则取决于所用的[程序语言以及所要初始化的对象的存储类型等属性](https://zh.wikipedia.org/wiki/程序语言 "wikilink")。用于进行初始化的程序结构则称为初始化器或初始化列表。初始化和变量[声明是明显有区别的](../Page/前向声明.md "wikilink")，而且初始化也先于变量声明进行，但两者在实践中仍常被混淆。

## 实现

初始化可以在[编译时通过静态嵌入值来实现](https://zh.wikipedia.org/wiki/编译 "wikilink")，也可以在[运行时进行分配](https://zh.wikipedia.org/wiki/运行时 "wikilink")。进行初始化的[代码部分称为](../Page/代码.md "wikilink")“初始化代码”，但这部分代码可能还包含其他的一次性使用[函数](../Page/函数.md "wikilink")（例如打开文件）。将存储单元设为16进制0的做法也称作“清除”，这一般在[机器代码级用](https://zh.wikipedia.org/wiki/机器代码 "wikilink")[异或](https://zh.wikipedia.org/wiki/异或 "wikilink")[指令的方式实现](https://zh.wikipedia.org/wiki/指令 "wikilink")（指令的两个操作数必须为相同的变量），因为这不需要额外的[内存访问](https://zh.wikipedia.org/wiki/内存 "wikilink")。

## C系编程语言

### 初始化器

在[C](https://zh.wikipedia.org/wiki/C语言 "wikilink")/C99/[C++中](../Page/C++.md "wikilink")，初始化器是声明器的可选部分，它由一个'='以及其后的一个[表达式](https://zh.wikipedia.org/wiki/表达式 "wikilink")（或含有多个以','隔开的带圆括号表达式的单一列表）所组成。后面这一列表有时也被称为“初始化（器）列表”（initializer
list）或”初始化列表”（initialization list），不过initializer
list也是C++中用于初始化[类或](https://zh.wikipedia.org/wiki/类 "wikilink")[结构中的元素的](https://zh.wikipedia.org/wiki/结构 "wikilink")[保留字](../Page/保留字.md "wikilink")。

一个包含初始化的声明则通常被称为定义。根据C++标准规定，定义是声明的一种；具体来说，声明只需为函数或数据对象指定类型即可，而定义则还必须为其赋初值。所以“声明和定义”的提法严格来说是不准确的，但这种说法仍被广为采用。\[1\]

### 初始化列表

一个类/结构的[构造函数可以在定义中于构造函数体前包含一个初始化列表](https://zh.wikipedia.org/wiki/构造函数 "wikilink")，用以给类/结构的元素赋初值。例如如下程序段：

``` c
  struct int_complex {
    int re, im;
    int_complex() : re(0), im(0) { }
  };
```

这里的" : re(0), im(0)"就是初始化列表。

有时“初始化列表”这个术语也用来指[数组或结构初始化器中的表达式表](../Page/数组.md "wikilink")。

在[C++0x标准中也以模板的形式提供了一个更强力的初始化表的概念](https://zh.wikipedia.org/wiki/C++0x "wikilink")，即为std:initializer_list。

### 默认初始化

在程序里，数据初始化也可能在没有由明确的语法来实现的情况下进行。例如说，如果一个[静态变量在没有初始化](../Page/静态变量.md "wikilink")（即未指定初值）的情况下被声明，则：

1.  对于[原生数据类型](https://zh.wikipedia.org/wiki/原始类型 "wikilink")：以对应类型的零值进行初始化；
2.  对于类中的静态对象：由其默认的构造函数来进行初始化。

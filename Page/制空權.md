**制空權**是[空戰](../Page/空戰.md "wikilink")[軍事理論](../Page/軍事理論.md "wikilink")，指空中戰力將敵方的空中戰力進行壓制乃至完全摧毀，於期望的空域中取得優勢，使陸、海、空三大軍力能夠無礙進攻的力量。自兩次世界大戰起，兩方軍隊勢力衝突時，最有效也最有利的就是取得制空權，以利我方陸軍／海軍執行任務，並防止對方空軍干擾，也能對我方部隊進行支援。[北大西洋公約組織以及](../Page/北大西洋公約組織.md "wikilink")[五角大廈則將制空權分為完全將敵方之空中武力徹底排除](../Page/五角大廈.md "wikilink")、摧毀之「空中霸權(Air
supremacy)」；以及暫時排除敵方之空中武力，便已取得作戰優勢之「空中優勢(Air superiority)」兩種類別。

制空權最早由義大利[軍事家](../Page/軍事家.md "wikilink")[杜黑提出](../Page/杜黑.md "wikilink")。在杜黑的[專著](../Page/著作.md "wikilink")《制空權（）》中，現代戰爭依靠全體國家[工業基礎的支援](../Page/工業.md "wikilink")，一旦摧毀[基礎建設就足以使敵軍失去發動戰爭的能力](../Page/基礎建設.md "wikilink")。他認為只要敵軍缺乏[槍械](../Page/槍械.md "wikilink")、[子彈](../Page/子彈.md "wikilink")、戰車或飛機等等的軍事裝備，便會因無法繼續作戰而造成社會動亂進一步迫使敵人[投降](../Page/投降.md "wikilink")。如要徹底摧毀敵軍的生產力，就必須對其發動戰爭的軍用設備進行大規模的[戰略轟炸](../Page/戰略轟炸.md "wikilink")\[1\]。

## 程度

| 友軍                     | 敵軍                     |
| ---------------------- | ---------------------- |
| 空中霸權（Air supremacy）    | 空中無能（Air incapability） |
| 空中優勢（Air superiority）  | 空中拒止（Air denial）       |
| 空中對等爭奪（Air parity）     | 空中對等爭奪（Air parity）     |
| 空中拒止（Air denial）       | 空中優勢（Air superiority）  |
| 空中無能（Air incapability） | 空中霸權（Air supremacy）    |

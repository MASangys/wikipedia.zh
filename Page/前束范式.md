在[谓词演算中](../Page/谓词演算.md "wikilink")，一个[公式是](../Page/公式.md "wikilink")**前束[范式](../Page/范式.md "wikilink")**的，如果它可以被写为[量词在前](../Page/量词.md "wikilink")，随后是被称为**母体**的无量词部分的字符串。所有[经典逻辑公式都](../Page/经典逻辑.md "wikilink")[逻辑等价于某个前束范式公式](../Page/逻辑等价.md "wikilink")。

可以用公式在如下重写规则下的逻辑等价来证实：

\[\forall x ( P(x) ) \land Q \equiv \forall x ( P(x) \land Q )\]

\[\forall x ( P(x) ) \lor Q \equiv \forall x ( P(x) \lor Q )\]

\[\exists x ( P(x) ) \land Q \equiv \exists x ( P(x) \land Q )\]

\[\exists x ( P(x) ) \lor Q \equiv \exists x ( P(x) \lor Q )\]

進一步推論可得：(可透過改寫 \(P \rightarrow Q\) 為 \(\lnot P \lor Q\) 推論得出)

\[\forall x ( P(x) \rightarrow Q ) \equiv \exists x P(x) \rightarrow Q\]

\[\forall x ( P \rightarrow Q(x) ) \equiv P \rightarrow \forall x Q(x)\]
它们的[存在对偶](../Page/德·摩根定律.md "wikilink")：

\[\exists x ( P(x) \rightarrow Q ) \equiv \forall x P(x) \rightarrow Q\]

\[\exists x ( P \rightarrow Q(x) ) \equiv P \rightarrow \exists x Q(x)\]
这里的 \(x\) 在 \(Q\) 中是非自由的，并注意通过这些规则的持续应用所有量词都可以移动到公式的前面。

某些证明演算只处理公式写为前束范式的理论。对于开发[算术层次和](../Page/算术层次.md "wikilink")[分析层次这个观念是基本的](../Page/分析层次.md "wikilink")。

前束范式是[哥德尔证明他的](../Page/哥德尔.md "wikilink")[哥德尔完备定理的主要工具](../Page/哥德尔完备定理.md "wikilink")。

**副主教**是一個在施行[主教制的](../Page/主教制.md "wikilink")[教會內之行政職位](../Page/教會.md "wikilink")，沒有治理權。

## 聖公會

在[聖公宗教會內](../Page/聖公宗.md "wikilink")，副主教是[教區主教的全職助理](../Page/教區主教.md "wikilink")，在[三級聖品制中](../Page/三級聖品制.md "wikilink")，副主教是[主教一級](../Page/主教.md "wikilink")。有一些副主教是教區內其中的分區[主教](../Page/主教.md "wikilink")。例如：[加拿大聖公會](../Page/加拿大聖公會.md "wikilink")[多倫多教區的](../Page/多倫多教區.md "wikilink")[余天錫主教就是](../Page/余天錫（聖公宗）.md "wikilink")[約克-士加堡分區的副主教](../Page/約克-士加堡分區.md "wikilink")；前[中華聖公會](../Page/中華聖公會.md "wikilink")[華南教區的](../Page/華南教區.md "wikilink")[莫壽增會督就是廣東主教](../Page/莫壽增.md "wikilink")（副會督）。

## 天主教

在[天主教會裡](../Page/天主教會.md "wikilink")，副主教是[教區](../Page/教區.md "wikilink")[正權主教的全職助理](../Page/正權主教.md "wikilink")\[1\]，其拉丁文原名「」（簡稱），意為「總代理」或「總代表」。在[三級聖品制中](../Page/三級聖品制.md "wikilink")，副主教是屬[鐸品而非](../Page/三級聖品制.md "wikilink")[主教品](../Page/三級聖品制.md "wikilink")，此乃最初中文的誤譯所致。

正權主教必須要在他的[教區內委任最少一名副主教](../Page/教區.md "wikilink")。副主教在教區內會代行主教的行政權力來處理教區日常事務，扮演着除正權主教外，另一位去處理教區日常事務的主要角色。副主教一定要由已[晉鐸品或以上品級之神職人員擔任](../Page/三級聖品制.md "wikilink")。在[東儀天主教會內](../Page/東儀天主教會.md "wikilink")，相等的職位叫做「protosyncellus」。

與副主教類似的身份，還有一種叫主教代表（episcopal
vicar）。一位教區主教也可以為教區任命一位或多位主教代表。他們與副主教享有同樣的教區首長權力，但是，它被限制於教區的特定區域，限制於特定的活動類型，限制於特定禮儀的[信友](../Page/平信徒.md "wikilink")，或限制於人們的某些群體。
(canon 476 of the Code of Canon Law) 在東儀天主教會中，他們被稱為「Syncelli」 (canon 191
of the Code of Canons of the Eastern Churches)。

另外根據《[天主教法典](../Page/天主教法典.md "wikilink")》規定，當一個教區有[助理主教或](../Page/助理主教.md "wikilink")[輔理主教時](../Page/輔理主教.md "wikilink")，他便要同時兼任副主教（[輔理主教可以以主教代表代替](../Page/輔理主教.md "wikilink")）。

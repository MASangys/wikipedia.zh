據2016年加拿大人口普查，[加拿大](../Page/加拿大.md "wikilink")總共有人口35,151,728，佔世界總人口的0.5%\[1\]\[2\] 。加拿大的人口增長原因非常多樣，包括領土擴張及移民。加拿大是一個對移民非常開放的國家，這也是加拿大人口增加的主要原因\[3\]。儘管加拿大的人口密度很低，但人口卻高度集中在南部地區，一些地區如安大略省的南部的人口密度比歐洲一些地區更高。

## 可见少数族裔比例增加

根据2011年人口统计，加拿大全国有超过626万人将自己定义为可见少数族裔，占全国人口的19.1％。\[6\]

| 序号 | 人口普查年份 | 可见少数族裔比例 |
| -- | ------ | -------- |
| 1  | 1986年  | 4.7%     |
| 2  | 1991年  | 9.4%     |
| 3  | 1996年  | 11.2%    |
| 4  | 2001年  | 13.4%    |
| 5  | 2006年  | 16.2%    |
| 6  | 2011年  | 19.1%    |
|    |        |          |

这种变化直接反应了[多元文化政策在加拿大人口结构上的巨大影响](https://zh.wikipedia.org/wiki/多元文化政策 "wikilink")。

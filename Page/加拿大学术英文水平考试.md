**加拿大学术英文水平考试**（，简称：），是一项针对申请[加拿大](../Page/加拿大.md "wikilink")大学入学资格的留学生的考试\[1\]\[2\]，主要对受试者的学术英语沟通能力进行考察。自2016年开始，[不列颠哥伦比亚大学子公司Paragon收购CAEL并开始对CAEL进行运营](https://zh.wikipedia.org/wiki/不列颠哥伦比亚大学 "wikilink")。该考试的内容一般是根据加拿大大学中常见的情景改编而成的，所用的英语也是[加拿大英语](../Page/加拿大英语.md "wikilink")。CAEL成绩的有效期是两年\[3\]\[4\]。

## 考试形式

CAEL分为四个部分：[口语](../Page/口语.md "wikilink")、[阅读](https://zh.wikipedia.org/wiki/阅读 "wikilink")、[听力和](https://zh.wikipedia.org/wiki/听力 "wikilink")[写作](https://zh.wikipedia.org/wiki/写作 "wikilink")。目前，CAEL以全电脑化的形式进行考试，总持续时间约为3.5小时\[5\]。

## 考试分值

CAEL CE的分数区间为10到90分，总分由四个子项加权平均后再用四舍五入的方法近似到最接近的10倍数得出，一共分为低入门级到精通级7个等级。

| 综合成绩  | 分数说明 |
| ----- | ---- |
| 80－90 | 精通   |
| 70    | 熟练   |
| 60    | 高级   |
| 50    | 高中级  |
| 40    | 中级   |
| 30    | 高入门级 |
| 10－20 | 低入门级 |

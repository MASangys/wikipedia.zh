在[數學裡](https://zh.wikipedia.org/wiki/數學 "wikilink")，一個具有[加法](../Page/加法.md "wikilink")[運算的](https://zh.wikipedia.org/wiki/運算 "wikilink")[集合中的](https://zh.wikipedia.org/wiki/集合 "wikilink")**加法單位元**，是指不論它加上任何一個在此集合內的元素*x*都會等於*x*的[元素](https://zh.wikipedia.org/wiki/元素 "wikilink")。

## 基本例子

  - [初等數學中所熟悉的加法單位元為](https://zh.wikipedia.org/wiki/初等數學 "wikilink")[0](../Page/0.md "wikilink")。

如：

  -
    5 + 0 = 5 = 0 + 5.

<!-- end list -->

  - 在[自然數](https://zh.wikipedia.org/wiki/自然數 "wikilink")**N**和其所有的[父集](https://zh.wikipedia.org/wiki/父集 "wikilink")([整數](https://zh.wikipedia.org/wiki/整數 "wikilink")**Z**、[有理數](https://zh.wikipedia.org/wiki/有理數 "wikilink")**Q**、[實數](https://zh.wikipedia.org/wiki/實數 "wikilink")**R**、[複數](https://zh.wikipedia.org/wiki/複數 "wikilink")**C**)內，其加法單位元皆為0。所以對於任何一個[數](https://zh.wikipedia.org/wiki/數 "wikilink")*n*，

<!-- end list -->

  -
    *n* + 0 = *n* = 0 + *n*.

## 形式定義

令*N*是一個在[加法](../Page/加法.md "wikilink")[運算下封閉的](https://zh.wikipedia.org/wiki/運算 "wikilink")[集合](https://zh.wikipedia.org/wiki/集合 "wikilink")。*N*的加法單位元即為任一個能使所有在*N*內的元素*n*有下列公式的元素*e*：

  -
    *e* + *n* = *n* = *n* + *e*.

## 更多例子

  - 在一個[群](../Page/群.md "wikilink")裡，加法單位元即是這個群的[單位元](../Page/單位元.md "wikilink")，通常標記做0，並且是唯一的(見下面證明)。

<!-- end list -->

  - 一個[環或一個](https://zh.wikipedia.org/wiki/環 "wikilink")[體也會是一個在](https://zh.wikipedia.org/wiki/域 "wikilink")[加法](../Page/加法.md "wikilink")[運算下的](https://zh.wikipedia.org/wiki/運算 "wikilink")[群](../Page/群.md "wikilink")，因此它們也會有一個唯一的加法單位元0。它被定義必須和[乘法單位元](https://zh.wikipedia.org/wiki/乘法單位元 "wikilink")[1](../Page/1.md "wikilink")不同，若環(或體)有兩個以上的元素時。如果加法單位元和乘法單位元是同一個的話，這個環則會是[當然的](https://zh.wikipedia.org/wiki/當然 "wikilink")(見下面證明)。

<!-- end list -->

  - 在一個於群*G*上的*m*乘*n*階[矩陣所組成的群M](https://zh.wikipedia.org/wiki/矩陣 "wikilink")<sub>*m*×*n*</sub>(*G*)裡，其加法單位元標記為**0**，且會是個其元素都是在*G*內的[單位元](../Page/單位元.md "wikilink")0的*m*乘*n*[矩陣](https://zh.wikipedia.org/wiki/矩陣 "wikilink")。例如，在一個於[整數上的](https://zh.wikipedia.org/wiki/整數 "wikilink")2階方陣M<sub>2</sub>(**Z**)裡，其加法單位元為

\[0 = \begin{pmatrix}0 & 0 \\ 0 & 0\end{pmatrix}\].

## 證明

### 加法單位元在一個群裡是唯一的

令(*G*,+)是一個[群](../Page/群.md "wikilink")，且設0和0'是在*G*內的兩個加法單位元，則對於所有在*G*內的*g*而言，

  -
    0 + *g* = *g* = *g* + 0 且 0' + *g* = *g* = *g* + 0'.

由上可得

  -
    (0') = (0') + 0 = 0' + (0) = (0)

故可證明 0 = 0'。

### 加法和乘法單位元在一個非平凡環裡是不同的

令*R*是一個[環](https://zh.wikipedia.org/wiki/環 "wikilink")，且假設加法單位元0和[乘法單位元](https://zh.wikipedia.org/wiki/乘法單位元 "wikilink")1會相等，即0=1。設*r*為於*R*內的任一[元素](https://zh.wikipedia.org/wiki/元素 "wikilink")，則

  -
    *r* = *r* × 1 = *r* × 0 = 0

其表示*R*必須是[平凡的](../Page/平凡_\(數學\).md "wikilink")，亦即*R*={0}。再依照[換質位法](https://zh.wikipedia.org/wiki/換質位法 "wikilink")，即可得出若*R*不是平凡的，則0不會等於1的結論。

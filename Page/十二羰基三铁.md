**十二羰基三铁** ()，分子式为Fe<sub>3</sub>(CO)<sub>12</sub>。它是第一种人工合成的金属-羰基簇合物\[1\]。相比[五羰基铁的化学性质更活泼](https://zh.wikipedia.org/wiki/五羰基铁 "wikilink")，更易于分解为单质铁。

## 性质

十二羰基三铁为墨绿色晶体，真空下发生升华同时伴随明显的分解。溶于非极性溶剂形成深绿色溶液，在空气气氛下加热溶液会使十二羰基三铁分解形成铁镜同时溶液发生自燃。十二羰基铁的固体在空气中会缓慢分解，因此通常将其保存在低温惰性的气氛中。

## 合成方法

一种不常用的合成方法是可以通过五羰基铁的热分解得到：
3 Fe(CO)<sub>5</sub> → Fe<sub>3</sub>(CO)<sub>12</sub> + 3 CO
这种方法的产率低。但是光解五羰基铁得到的不是十二羰基三铁，而是[九羰基二铁](../Page/九羰基二铁.md "wikilink")。

有效合成十二羰基三铁的方法是以五羰基铁为原料的\[2\]： 首先五羰基铁与有机碱反应：
3 Fe(CO)<sub>5</sub> + (C<sub>2</sub>H<sub>5</sub>)<sub>3</sub>N + H<sub>2</sub>O → \[(C<sub>2</sub>H<sub>5</sub>)<sub>3</sub>NH\]\[HFe<sub>3</sub>(CO)<sub>11</sub>\] + 3 CO + CO<sub>2</sub>

得到的金属羰基氢化物用酸氧化得到产物：
\[(C<sub>2</sub>H<sub>5</sub>)<sub>3</sub>NH\]\[HFe<sub>3</sub>(CO)<sub>11</sub>\] + HCl + CO → Fe<sub>3</sub>(CO)<sub>12</sub> + H<sub>2</sub> + \[(C<sub>2</sub>H<sub>5</sub>)<sub>3</sub>NH\]Cl

最初合成十二羰基三铁的方式是由德国无机化学家[瓦尔特·希贝尔](https://zh.wikipedia.org/wiki/瓦尔特·希贝尔 "wikilink")(Walter Hieber)发现的\[3\]。利用[二氧化锰](../Page/二氧化锰.md "wikilink")氧化-2价的羰基铁氢化物H<sub>2</sub>Fe(CO)<sub>4</sub>得到零价的十二羰基三铁：
3 \[HFe(CO)<sub>4</sub>\]<sup>-</sup> + 3 MnO<sub>2</sub> → Fe<sub>3</sub>(CO)<sub>12</sub> + 3 OH<sup>-</sup> + 3 MnO

## 反应

和大部分金属羰基配合物一样，十二羰基三铁可发生[取代反应](../Page/取代反应.md "wikilink").比如 Fe<sub>3</sub>(CO)<sub>11</sub>{P(C<sub>6</sub>H<sub>5</sub>)<sub>3</sub>}就可以通过十二羰基三铁与三苯基膦发生取代反应制得 。

## 安全性

十二羰基三铁因其是一种易挥发性、可以释放出一氧化碳的零价铁化合物而具有危险性。固体的十二羰基三铁，尤其是细碎的固体粉末和反应后残余的十二羰基三铁可以发生自燃。

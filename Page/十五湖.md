**十五湖**，或稱**十五胡**、**十五糊**，是流傳於[香港](../Page/香港.md "wikilink")的[紙骨牌](../Page/紙骨牌.md "wikilink")遊戲\[1\]，屬與[亂戳](../Page/亂戳.md "wikilink")類似的[吃墩遊戲](https://zh.wikipedia.org/wiki/吃墩遊戲 "wikilink")。

香港當地有歌詞：「拎冧六、長衫六、高腳七、一隻大頭六。」\[2\]

## 牌規

### 流程

  - 三、四人遊戲。三人時，每人摸二十八張；四人時，每人摸二十一張。如果手牌如果有稱為至尊的1-2與2-4牌組合，可將至尊放在桌上作為牌值。
  - 頭家先引牌。每回合玩家選擇出單張、對子、三條、四條、或以下的特殊牌型。

| 牌型名    | 組合         |
| ------ | ---------- |
| **天九** | 天牌、與總點數為9。 |
| **地八** | 地牌、與總點數為8。 |
| **人七** | 人牌、與總點數為7。 |
| **鵝五** | 鵝牌、與總點數為5。 |

  - 引牌後，兩位玩家需出大過上家的同門、同數量的牌。大牌者下回合作引牌。無可贏得上家牌，放棄不出牌。
  - 四門的大小如下。
      - 文子：6-6〉1-1〉4-4〉1-3〉5-5〉3-3〉2-2〉5-6〉4-6〉1-6〉1-5牌。
      - 第一組武子：4-5〉2-6〉3-4〉1-4〉1-2牌。
      - 第二組武子：3-6〉3-5〉2-5〉2-3牌。
      - 組合：天九〉地八〉人七〉鵝五。
  - 組合可一次出同組、或不同組的數組牌，下家要大過需出同樣組數、能大過各組的組合。
  - 先出到時十五張牌，可宣布獲勝，稱為食胡，也可繼續行牌。

### 牌值

  - 最後一輪以四條贏墩，其他玩家須像[打天九的賀四大](https://zh.wikipedia.org/wiki/打天九 "wikilink")，另外給籌碼。
  - 每先出完一張牌就得1胡。
  - 每有一組至尊就得1胡。

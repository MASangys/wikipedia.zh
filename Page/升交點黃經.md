**升交點黃經**\[1\]（符號是☊ 或
Ω）是用來具體描述天體在空間中[軌道的](../Page/軌道.md "wikilink")[軌道要素之一](../Page/軌道要素.md "wikilink")。它是由參考方向（''[經度原點](../Page/經度原點.md "wikilink")
''）起始，在參考平面上量度至昇交點的角度\[2\]。通常被做為經度原點和參考平面的有：

  - 在[地心座標](../Page/地心座標.md "wikilink")，以[地球的赤道做為參考平面](../Page/地球.md "wikilink")，[春分點做為經度原點](../Page/春分點.md "wikilink")。在這種狀態下，經度會被稱為**升交點赤經**，或縮寫為**RAAN**(right
    ascension of ascending
    node)。角度是從春分點向東量度（或是說由赤道北方看是[逆時針方向](../Page/順時針方向.md "wikilink")）至升交點\[3\]\[4\]。
  - 在[日心軌道](../Page/日心軌道.md "wikilink")，[黃道是參考平面](../Page/黃道.md "wikilink")，[春分點做為經度原點](../Page/春分點.md "wikilink")。角度是從春分點向東量度（或是說由黃道北方看是[逆時針方向](../Page/順時針方向.md "wikilink")）至升交點\[5\]。
  - 在[太陽系外的軌道](../Page/太陽系.md "wikilink")，則在天球上對觀測者的基線方向上做一垂直的平面
    (天球上的切面，稱為[天空平面](../Page/天空平面.md "wikilink"))，這個參考平面的北方，就是指向[天球北極的方向](../Page/天球.md "wikilink")，以[垂直投影的方法將原來的經線投影在參考平面上就可以得到](../Page/正交投影.md "wikilink")。角度的測量以北方為起點向東測量
    (對觀測者而言是逆時針方向)至交點的位置\[6\]<sup>, pp. 40, 72, 137; </sup>\[7\]<sup>,
    chap. 17.</sup>。

在[聯星情況中](../Page/聯星.md "wikilink")，只有目視觀測是不可能知道哪個點是昇交點和哪個點是降交點。在這種情況下所紀錄的軌道參數只是交點的經度，Ω，其數值介於0至180度之間\[8\]<sup>,
chap. 17;</sup>\[9\]<sup>, p. 72.</sup>。

## 由狀態向量計算

在[太空動力學](../Page/太空動力學.md "wikilink")，昇交點經度可以從[軌道狀態向量計算如下](../Page/軌道狀態向量.md "wikilink")：

\[\Omega =\arccos { {n_x} \over { \mathbf{\left |n \right |}}}\ \ (n_y\ge 0);\]

\[\Omega =2\pi - \arccos { {n_x} \over { \mathbf{\left |n \right |}}}\ \ (n_y<0).\]
此處，***n***=（*n*<sub>x</sub>, *n*<sub>y</sub>,
*n*<sub>z</sub>）是指向[升交點的向量](../Page/升交點.md "wikilink")。參考平面是假設的*xy*-平面，經度原點的位置在+*x*-軸的方向。

對[無傾斜軌道](../Page/無傾斜軌道.md "wikilink")（軌道傾斜角為0度）就無須定義Ω。計算時依慣例設為0，也就是說指向昇交點的向量***n***在參考平面上的方向就是+*x*-軸。

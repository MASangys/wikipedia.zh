**半正矢公式**是一种根据两点的[经度和](../Page/经度.md "wikilink")[纬度来确定](../Page/纬度.md "wikilink")[大圆上两点之间距离的计算方法](../Page/大圆距离.md "wikilink")，在[導航有着重要地位](https://zh.wikipedia.org/wiki/導航 "wikilink")。它是[球面三角學中](../Page/球面三角學.md "wikilink")“半正矢定理”公式的特例，该定理涉及了球面三角形的边和角。

尽管第一份英文版的[半正矢表由詹姆斯](https://zh.wikipedia.org/wiki/数学用表 "wikilink")·安德鲁在1805年印刷出版，\[1\]但[弗洛里安·卡喬里相信](../Page/弗洛里安·卡喬里.md "wikilink")[José
de Mendoza y
Ríos在](https://zh.wikipedia.org/wiki/José_de_Mendoza_y_Ríos "wikilink")1801年就使用过类似的术语。\[2\]\[3\] “半正矢”这个名字由[詹姆斯·英曼在](https://zh.wikipedia.org/wiki/詹姆斯·英曼 "wikilink")1835年创造。\[4\]\[5\]

这个公式正如其名，用半正矢函数表达，通过<span class="texhtml">haversin(*θ*) =
sin<sup>2</sup>(<span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em;  text-align:center;"><span style="display:block; line-height:1em; padding:0 0.1em;">*θ*</span><span style="display:none;">/</span><span style="display:block; line-height:1em; padding:0 0.1em; border-top:1px solid;">2</span></span>)</span>的关系做替代而来。该公式可以用半正矢函数的任意倍数表达，如[正矢函数](../Page/正矢.md "wikilink")(半正矢函数的两倍). 在计算机出现之前，为了计算简便，人们会利用对数来计算乘积和利用半正矢函数计算距离，所以在十九和二十世纪初的导航和三角测量书中包含了半正矢值表和对数表。现在，将该公式用半正矢函数表达也很方便，因为它能避免sin<sup>2</sup>的系数。

## 半正矢公式

对于任何球面上的两点，圆心角的半正矢值可以通过如下公式计算：

\[hav(\frac {d}{r})=hav(\varphi_2-\varphi_1)+\cos(\varphi_1)\cos(\varphi_2)hav(\lambda_2-\lambda_1)\]\(\)

  - hav是半正矢函数的缩写：

\[hav(\theta)=\sin^2(\frac {\theta}{2})=\frac {1-\cos(\theta)}{2}\]\(\)

  - d是两点之间的距离(沿大圆，见[球面距离](../Page/大圆距离.md "wikilink")),
  - r是球的半径，
  - φ1φ2：点1的纬度和点2的纬度,以弧度制度量
  - λ1λ2：点1的经度和点2的经度,以弧度制度量

左边的等号d / r是圆心角,以弧度来度量。

可以通过应用反半正矢函数(如果可以查到值)或通过使用反正弦函数来解出d：

\[d=r hav^{-1}(h)=2r\arcsin(\sqrt{h})\]\(\)

<span class="texhtml">*h*</span> =
<span class="texhtml">hav(<span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em;  text-align:center;"><span style="display:block; line-height:1em; padding:0 0.1em;">*d*</span><span style="display:none;">/</span><span style="display:block; line-height:1em; padding:0 0.1em; border-top:1px solid;">*r*</span></span>)</span>，代入可得：

\[d=2r\arcsin\Bigl(\sqrt{hav(\varphi_2-\varphi_1)+\cos(\varphi_1)\cos(\varphi_2)hav(\lambda_2-\lambda_1)}\Bigr)\]\(\)

    \(\)

在使用这个公式时，必须确保 <span class="texhtml">*h*</span>
不超过1（<span class="texhtml">*d*</span> 只当h的值在0到1间才有意义）。
<span class="texhtml">*h*</span>
仅在球面上两点连线过球心时才为1，当测量精度有限时，会产生较大的计算误差。 <span class="texhtml">*d*</span> 如果大到接近圆周的一半长度<span class="texhtml">π*R*</span>
时（这个情况并不常见），一个小错误通常是一个大问题在。虽然还有其他的[大圆距离公式](../Page/大圆距离.md "wikilink")，能够避免这个问题。上述公式有时候用[反正切函数表达](../Page/反正切.md "wikilink")，但是h接近1时这个问题仍然存在。

但是仍有对策来避免上述问题。类似的公式可以使用余弦函数表达（有时亦称为球面余弦定理，但是并非平面[余弦定理](https://zh.wikipedia.org/wiki/余弦定理 "wikilink")）而不是半正矢，但如果这两点十分靠近（例如在地球上的一公里外）你可能得到的结果为
<span class="texhtml">cos(<span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em;  text-align:center;"><span style="display:block; line-height:1em; padding:0 0.1em;">*d*</span><span style="display:none;">/</span><span style="display:block; line-height:1em; padding:0 0.1em; border-top:1px solid;">*R*</span></span>)</span>
=0.99999999，导致一系列的误差。而由于半正矢式使用正弦函数，避免了这一问题。所以两个公式应该互补使用。

因为[地球是一个不完美的球体](../Page/地球.md "wikilink")，故当其应用于地球时，无论哪一个公式只是做一个近似测算，"[地球半径](../Page/地球半径.md "wikilink")"
<span class="texhtml">*R*</span> 在极点地区是6356.752 公里，在赤道地区为6378.137
公里。另外，半径曲率在极点处(≈6399.594 公里)比在赤道处(≈6335.439
公里)大1%，所以半正矢公式或者球面余弦定理是不能保证0.5%以内的误差。更准确的方法，应该是使用考虑地球[离心率的Vincenty的公式或其他有关地理距离的论文所给出方法](https://zh.wikipedia.org/wiki/离心率 "wikilink")。

## 半正矢定理

[右](https://zh.wikipedia.org/wiki/File:Law-of-haversines.svg "fig:右")
给出一个单位球，一个在表面的球面三角形 三个过三点(u,v,w)的大圆所围出来的区域。如图这个球面三角形的三边分别是
<span class="texhtml">*a*</span>（<span class="texhtml">**u**</span> 至
<span class="texhtml">**v**</span>），<span class="texhtml">*b*</span>（<span class="texhtml">**u**</span>
到 <span class="texhtml">**w**</span>）和
<span class="texhtml">*c*</span>（<span class="texhtml">**v**</span> 至
<span class="texhtml">**w**</span>）并且角C对边c那么有如下关系

\(hav(c)=hav(a-b)+\sin(a)\sin(b)hav(C)\)

  -
    \(\)\[6\]

由于这是一个单位球，弧长 <span class="texhtml">*a*</span>、
<span class="texhtml">*b*</span> 和 <span class="texhtml">*c*</span>
与两端点同球心构成的圆心角（[弧度制](https://zh.wikipedia.org/wiki/弧度 "wikilink")）相等。

为了从该定理中得到半正矢公式。只要一个认为特<span class="texhtml">**u**</span>是[北极](../Page/北极点.md "wikilink")，而<span class="texhtml">**v**</span>和<span class="texhtml">**w**</span>两点，其距离
<span class="texhtml">*d*</span> 是要计算的。在这种情况下，
<span class="texhtml">*a*</span> 和 <span class="texhtml">*b*</span> 是
<span class="texhtml"><span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em;  text-align:center;"><span style="display:block; line-height:1em; padding:0 0.1em;">π</span><span style="display:none;">/</span><span style="display:block; line-height:1em; padding:0 0.1em; border-top:1px solid;">2</span></span>
− *φ*<sub>1,2</sub></span> (即，90° − 纬度),
<span class="texhtml">*C*</span> 是 <span class="texhtml">Δ*λ*</span> 和
<span class="texhtml">*c*</span>
是求的 <span class="texhtml"><span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em;  text-align:center;"><span style="display:block; line-height:1em; padding:0 0.1em;">*d*</span><span style="display:none;">/</span><span style="display:block; line-height:1em; padding:0 0.1em; border-top:1px solid;">*R*</span></span></span>.
结合 <span class="texhtml">sin(<span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em;  text-align:center;"><span style="display:block; line-height:1em; padding:0 0.1em;">π</span><span style="display:none;">/</span><span style="display:block; line-height:1em; padding:0 0.1em; border-top:1px solid;">2</span></span>
− *φ*) = cos(*φ*)</span>，半正矢公式即可推导得出。

或者可以从球面余弦定理中得出

\[\cos(c)=\cos(a)\cos(b)+\sin(a)\sin(b)\cos(C)\]\(\)

当 <span class="texhtml">*c*</span>
很小时，这个公式精度不高。所以我们可以用<span class="texhtml">cos(*θ*
) = 1 − 2 hav(*θ* )</span> 来替换
，并引用[三角恒等式](https://zh.wikipedia.org/wiki/三角恒等式 "wikilink")<span class="texhtml">cos(*a*
− *b* ) = cos(*a* )cos(*b* ) + sin(*a* )sin(*b* )</span>，半正矢定理就能推出。

[tetrahedron.png](https://zh.wikipedia.org/wiki/File:tetrahedron.png "fig:tetrahedron.png")\]\] [几何学](../Page/几何学.md "wikilink")上，**单纯形**或者***n*-单纯形**是和三角形类似的*n*维几何体。精确的讲，单纯形是某个n维以上的[欧几里得空间](../Page/欧几里得空间.md "wikilink")中的（*n*+1）个[仿射无关](../Page/仿射变换.md "wikilink")（也就是没有*m-1*维[平面包含](../Page/平面_\(数学\).md "wikilink")*m*+1个点；这样的点集被称为处于[一般位置](https://zh.wikipedia.org/wiki/一般位置 "wikilink")）的[点](../Page/点.md "wikilink")的集合的[凸包](../Page/凸包.md "wikilink")。

例如，0-单纯形就是[点](../Page/点.md "wikilink")，1-单纯形就是[线段](../Page/线段.md "wikilink")，2-单纯形就是[三角形](../Page/三角形.md "wikilink")，3-单纯形就是[四面体](https://zh.wikipedia.org/wiki/四面体 "wikilink")，而4-单纯形是一个[五胞体](../Page/正五胞体.md "wikilink")（每种情况都包含内部）。

**正单纯形**是同时也是[正多胞形的单纯形](https://zh.wikipedia.org/wiki/正多胞形 "wikilink")。正*n*-单纯形可以从正(*n* − 1)-单纯形通过将一个新顶点用同样的边长连接到所有旧顶点构造。

## 基础

任何*n+1*点集的非空子集的凸包定义了一个n-单纯形，称为该n-单纯形的**面**。面本身也是单纯形。（*n+1*点）的*m+1*子集的凸包是一个m-单纯形，称为n-单纯形的***m*-面**。 0-面（也即，一个点构成的面）称为**顶点**，而1-面称为**边**，(*n* − 1)-面称为**面片**，而*n*-面就是*n*-单纯形本身。一般来讲，*m*-面的个数等于[二项式系数](https://zh.wikipedia.org/wiki/二项式系数 "wikilink") *C*(*n* + 1, *m* + 1)。因此，*n*-单纯形的*m*-面的个数可以在[杨辉三角形](../Page/杨辉三角形.md "wikilink")的第(*n*+1)行和第(*m*+1)列找到。*面*和*面片*在描述[单纯复形](../Page/单纯复形.md "wikilink")中的单纯形的类型时可能有不同的含义。参看[单纯复形\#定义](https://zh.wikipedia.org/wiki/单纯复形#定义 "wikilink")。

**正单纯形**族是三族[正多胞体的第一组](https://zh.wikipedia.org/wiki/正多胞体 "wikilink")，[Coxeter将之记为](https://zh.wikipedia.org/wiki/Coxeter "wikilink")*α<sub>n</sub>*，其它两类为[正轴体](https://zh.wikipedia.org/wiki/正轴体 "wikilink")，记为*β<sub>n</sub>*，和[超立方体](https://zh.wikipedia.org/wiki/超立方体 "wikilink")，记为*γ<sub>n</sub>*。第四组，[超立方体的无穷分割被记为](https://zh.wikipedia.org/wiki/超立方蜂巢 "wikilink")*δ<sub>n</sub>*。

## 标准单纯形

[2D-simplex.svg](https://zh.wikipedia.org/wiki/File:2D-simplex.svg "fig:2D-simplex.svg") **标准*n*-单纯形**(或称**单位*n*-单纯形**)是**R**<sup>*n*+1</sup>的子集：

\[\Delta^n = \left\{(t_0,\cdots,t_n)\in\mathbb{R}^{n+1}\mid\sum_{i=0}^n{t_i} = 1 \mbox{ and } \forall i,\ t_i \ge 0 \right\}\] 单纯形Δ<sup>*n*</sup>位于（该超平面可以通过将上面*t*<sub>*i*</sub> ≥ 0的条件去掉而得到）。标准单纯形显然是正单纯形。

标准*n*-单纯形的顶点为

  -
    *e*<sub>0</sub> = (1, 0, 0, …, 0),
    *e*<sub>1</sub> = (0, 1, 0, …, 0),
    \(\vdots\)
    *e*<sub>*n*</sub> = (0, 0, 0, …, 1).

存在从标准*n*-单纯形到顶点为(*v*<sub>0</sub>, …, *v*<sub>*n*</sub>)的任意*n*-单纯形的标准映射

\[(t_0,\cdots,t_n) \mapsto \Sigma_i t_i v_i\] 系数*t*<sub>*i*</sub>称为点在*n*-单纯形中的[重心坐标](../Page/重心坐标.md "wikilink")。这样的一般单纯形常常被称为**仿射*n*-单纯形**，以强调该标准映射是[仿射变换](../Page/仿射变换.md "wikilink")。它有时也称为**定向放射*n*-单纯形**以强调标准映射可以是[保定向或者是反定向的](https://zh.wikipedia.org/wiki/定向_\(数学\) "wikilink")。

## 几何属性

在*n*维空间中的顶点为(*v*<sub>0</sub>, ..., *v*<sub>*n*</sub>)的*n*-单纯形的定向[体积](../Page/体积.md "wikilink")是

\[{1\over n!}\det
 \begin{pmatrix}
  v_1-v_0 & v_2-v_0& \dots & v_{n-1}-v_0 & v_n-v_0
 \end{pmatrix}\]

其中*n* × *n*[行列式](../Page/行列式.md "wikilink")的每列是代表两个顶点的[向量](../Page/向量.md "wikilink")之差。去掉1/*n*\!的公式是*n*-[平行多面体的体积](https://zh.wikipedia.org/wiki/平行多面体 "wikilink")。理解1/*n*\!因子的一个方法如下:在单位*n*维盒子的取任意一点，将其坐标分量和0一起排序，然后将相邻数取差值，得到的数组构成原点和与其最近的*n*个顶点构成的*n*-单纯形中的一点的坐标；取差值的变换是一个保体积的变换，而排序则将点的个数减少到1/*n*\!。

标准*n*-单纯形下的[体积](../Page/体积.md "wikilink")（也即，**R**<sup>n+1</sup>中的原点和单纯形之间的体积）是

\[{1 \over (n+1)!}\]

单位边长的正*n*-单纯形的[体积](../Page/体积.md "wikilink")是

\[{\frac{\sqrt{n+1}}{n!\sqrt{2^n}}}\]

这个结果可以导出如下：将上一个公式乘以*x*<sup>*n+1*</sup>，得到作为顶点离原点距离（所有顶点和原点等距）的函数的*n*-单纯形下的体积；对*x*微分，取导数在\(x=1/\sqrt{2}\)  的值（因为这个位置*n*-单纯形边长为1），这个导数需要除以\(\sqrt{n+1}\)，因为增量\((dx,\dots, dx)\)（垂直于*n*-单纯形的法向）的长度为\(\sqrt{n+1}\)。

### "直角"的单纯形

这里的直角表示存在一个顶点，其所有相邻超平面两两垂直。这样的单纯形是直角三角形的一个推广，对于它们存在着一个n维的[毕达哥拉斯定理](https://zh.wikipedia.org/wiki/毕达哥拉斯定理 "wikilink"):

所有和直角相邻的n维超面的体积平方之和等于对面的n维体积的平方。

\[\sum_{k=1}^{n} |A_{k}|^2 = |A_{0}|^2\] 其中 \(A_{1} \ldots A_{n}\)是两两垂直但不垂直于\(A_{0}\)的超面，而\(A_0\)是直角的对面。

对于2-单纯形，这个定理就是[毕达哥拉斯定理](https://zh.wikipedia.org/wiki/毕达哥拉斯定理 "wikilink")，而对于3-单纯形这个是适用于带立方角四面体的[德古阿定理](../Page/德古阿定理.md "wikilink")。

## 拓扑

[拓扑上](https://zh.wikipedia.org/wiki/拓扑 "wikilink")，*n*-单纯形是[拓扑等价于](https://zh.wikipedia.org/wiki/拓扑等价 "wikilink")[*n*-球體的](https://zh.wikipedia.org/wiki/n维球面 "wikilink")。每个*n*-单纯形是*n*维[带边界流形](https://zh.wikipedia.org/wiki/带边界流形 "wikilink")。

[代数拓扑](../Page/代数拓扑.md "wikilink")上，单纯形是用于构建一类称为[单纯复形](../Page/单纯复形.md "wikilink")的常用[拓扑空间](../Page/拓扑空间.md "wikilink")的基本元素。这些空间可以通过将单纯形用[组合方式粘合在一起来构造](https://zh.wikipedia.org/wiki/组合论 "wikilink")。单纯复形用于定义特定的一类[同调](../Page/同调.md "wikilink")，称为[单纯同调](https://zh.wikipedia.org/wiki/单纯同调 "wikilink")。

嵌入到**R**<sup>n</sup>的[开子集中的](https://zh.wikipedia.org/wiki/开子集 "wikilink")*k*-单纯形的有限集称为**仿射*k*-链**。在链中的单纯形不必唯一；它们可以[重复出现](https://zh.wikipedia.org/wiki/重次 "wikilink")。通常不采用集合的记法来标识仿射链，而是采用加号将它们连起来。若有些单纯形有相反的[定向](https://zh.wikipedia.org/wiki/定向 "wikilink")，它们可以用减号。如果有些单纯形出现多次，可以放一个整数在前面表示出现次数。这样，仿射链可以用整系数的线性组合表示。

注意*n*-单纯形的每个面是一个仿射*n-1*-单纯形，因此*n*-单纯形的[边界](../Page/边界.md "wikilink")可以用一个仿射*n-1*-链来表示。如果定义一个正定向的单纯形

\[\sigma=[v_0,v_1,v_2,...,v_n]\]

其中 \(v_j\) 代表顶点，则其边界 \(\partial\sigma\) 是如下链

\[\partial\sigma = \sum_{j=0}^n
(-1)^j [v_0,...,v_{j-1},v_{j+1},...,v_n]\].

更一般的，单纯形（以及链）可以通过光滑可微映射[嵌入到](https://zh.wikipedia.org/wiki/嵌入 "wikilink")[流形](../Page/流形.md "wikilink")中：\(f\colon\mathbb{R}^n\rightarrow M\)。这个情况下，加法表示和边界算子都和嵌入可交换。也即

\[f(\sum\nolimits_i a_i \sigma_i) = \sum\nolimits_i a_i f(\sigma_i)\]

其中 \(a_i\) 是标识定向和重次的整数。对于边界算子 \(\partial\)，有

\[\partial f(\phi) = f (\partial \phi)\]

其中 φ 为链。边界算子和映射可交换，是因为，链基本就是一个集合，而集合操作和[映射是可交换的](../Page/函数.md "wikilink")（按照映射的定义）。

到[拓扑空间](../Page/拓扑空间.md "wikilink")*X*中的连续映射 \(f:\sigma\rightarrow X\) 常常被称为**奇异*n*-单纯形**。因为f可以有奇点。

## 随机采样

(也称**单纯形采点**) 有两种在单位*K-1*-单纯形中产生有效产生均匀分布的采样方法。

第一种方法基于从*K-1*维单位单纯形采样等价于从参数*α* = (*α*<sub>1</sub>, ..., *α*<sub>*K*</sub>)都等于1的[狄利克雷分布](../Page/狄利克雷分布.md "wikilink")中采样的事实。确切的流程为：

  - 产生*K*个服从单位[指数分布](../Page/指数分布.md "wikilink")的随机数*x*<sub>1</sub>, ..., *x*<sub>*K*</sub>.
      - 这可以通过产生*K*个在[开区间](https://zh.wikipedia.org/wiki/开区间 "wikilink")(0,1)中[均匀分布的随机数](https://zh.wikipedia.org/wiki/均匀分布 "wikilink")*y*<sub>*i*</sub>然后取 *x*<sub>*i*</sub>=-ln(*y*<sub>*i*</sub>).
  - 令*S*为*x*<sub>*i*</sub>之和。
  - 单位单纯形中的点的*K*个坐标*t*<sub>1</sub>, ..., *t*<sub>*K*</sub>由*t*<sub>*i*</sub>=*x*<sub>*i*</sub>/*S*给出。

第二个方法是基于单位区间上的均匀分布的[顺序统计量](https://zh.wikipedia.org/wiki/顺序统计量 "wikilink")（参看[Devroye](https://zh.wikipedia.org/wiki/#References "wikilink"), p.568）。算法如下：

  - 令 *p*<sub>0</sub> = 0 而 *p*<sub>*K*</sub>=1.
  - 产生*K*-1个开区间(0,1)上[均匀分布的随机数](https://zh.wikipedia.org/wiki/均匀分布 "wikilink")*p*<sub>*i*</sub>。
  - 将*K*+1点*p*<sub>0</sub>, ..., *p*<sub>*K*</sub>排序。
  - 单位单纯形中的点的*K*个坐标*t*<sub>1</sub>, ..., *t*<sub>*K*</sub>由*t*<sub>*i*</sub>=*p*<sub>*i*</sub>-*p*<sub>*i*-1</sub>给出。

### 随机漫游

有时需要在单纯形中进行均匀[随机漫游而不是取一点](https://zh.wikipedia.org/wiki/随机漫游 "wikilink")。这样的随机漫游在[蒙特卡罗法中经常用到](https://zh.wikipedia.org/wiki/蒙特卡罗法 "wikilink")，譬如单纯形域中的[马尔可夫链蒙特卡罗](https://zh.wikipedia.org/wiki/马尔可夫链蒙特卡罗 "wikilink")。

可以从单位化*K*个[指数分布](../Page/指数分布.md "wikilink")随机向量来得到单纯形中的均匀分布来衍生出漫游的有效算法。首先定义一个单变量函数在正实直线上"漫游"，其采样点的静态分布为单位指数分布。该函数利用[Metropolis-Hastings算法从旧点得到新点](https://zh.wikipedia.org/wiki/Metropolis-Hastings算法 "wikilink")。这个函数伪代码如下，其中*h*是相对步长：

<code>

    next_point <- function(x_old)
    {
        repeat {
            x_new <- x_old * exp( Random_Normal(0,h) )
            metropolis_ratio <- exp(-x_new) / exp(-x_old)
            hastings_ratio <- ( x_new / x_old )
            acceptance_probability <- min( 1 , metropolis_ratio * hastings_ratio )
            if ( acceptance_probability > Random_Uniform(0,1) ) break
        }
        return(x_new)
    }

</code> 然后在单纯形中随机漫游:

  - 取服从单位指数分布的随机变量*x*<sub>*i*</sub>, *i*= 1, 2, ..., *K*.
  - 对于每个 *i*= 1, 2, ..., *K*
      - *x*<sub>*i*</sub> ← next_point(*x*<sub>*i*</sub>)
  - 令*S*为*x*<sub>*i*</sub>之和
  - 令*t*<sub>*i*</sub> = *x*<sub>*i*</sub>/*S*（对于所有 *i*= 1, 2, ..., *K*）

*t*<sub>*i*</sub>限制在单纯形中，并会以均匀的静态分布密度来反复遍历整个区域。注意不要每一步都单位化*x*<sub>*i*</sub>；那样会得到非均匀的静态分布。实际上，应该把*x*<sub>*i*</sub> 视为"隐"参数，而*t*<sub>*i*</sub>才给出单纯形中的坐标。

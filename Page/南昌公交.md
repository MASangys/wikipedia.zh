**南昌公交**是指由江西南昌公共交通运输集团有限责任公司（原[南昌公交总公司](https://zh.wikipedia.org/wiki/南昌公交总公司 "wikilink")）一家公司主要负责经营南昌市下辖市辖区和县范围内的公共汽车系统\[1\]\[2\]

[SNY02686.jpg](https://zh.wikipedia.org/wiki/File:SNY02686.jpg "fig:SNY02686.jpg")

## 历史

2002年7月南昌公交总公司开始对公交番号进行大量修改，3、4、6、10、14、15、19、21、22、23、29、30、31、32等公交车因驶往郊区而集体加200\[3\]\[4\]。

## 线路

### 线路规则

南昌公交的运营公司对线路的路别设有不同的定义用于区分线路类型\[5\]，但事实上有一定程度的使用混乱，比如大站快线目前均以线路名称前加K，比如204的快线为K204，并非是设计中的7开头。

| 路别开头 | 范围       | 线路类型       |
| ---- | -------- | ---------- |
| 0字头  | 0-99路    | 市区线路       |
| 1字头  | 100-199路 | 远郊线路       |
| 2字头  | 200-299路 | 近郊线路       |
| 3字头  | 300-399路 | 夜间线路       |
| 5字头  | 500-599路 | 区间线路       |
| 6字头  | 600-699路 | 高铁线路       |
| 7字头  | 700-799路 | 大站快线       |
| 8字头  | 800-899路 | 社区巴士、微公交线路 |

## 票价

南昌公共汽车分单一票价和多级票价两种，同时一条线路如果同时有普通车和空调车，也会有两种计价方式，但是现在同一线基本不会有空调车和普通车混合运营。\[6\]

南昌公交的票价

| 票价   | 对应线路类型             |
| ---- | ------------------ |
| 1元   | 市区行驶的线路，以0字头和2字头为主 |
| 2元   | 来往郊区的线路，2字头为主      |
| 阶梯售票 | 开往远郊线路             |
| 3元以上 | 开往远郊线路             |

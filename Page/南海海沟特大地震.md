**南海海沟特大地震**（），是预测将会发生在[菲律宾海板块](../Page/菲律宾海板块.md "wikilink")与[阿穆尔板块](../Page/阿穆尔板块.md "wikilink")\[1\]板块边界[俯冲带的](../Page/隱沒帶.md "wikilink")，震源地沿[南海海沟的特大地震或超级地震](https://zh.wikipedia.org/wiki/本州南海槽 "wikilink")\[2\]\[3\]。

此外，2011年8月日本[内阁府设立的](https://zh.wikipedia.org/wiki/内阁府 "wikilink")「南海海沟特大地震模型研讨会」，将预测中沿南海海沟发生的最大级别地震，也称为「南海海沟特大地震」，本条目也将基于此进行解说。

## 南海海沟的地震特征与「地震画像」

[缩略图](https://zh.wikipedia.org/wiki/File:RuptureAreasNankaiMegathrust_2013.png "fig:缩略图")

南海海沟地震，大约以90 - 150年（若以[中世纪](../Page/中世纪.md "wikilink")前的记录来看则是大于200年）的间隔发生，每次将间隔数小时至数年（或是同时）发生震源地位于东海、东南海、南海的三场地震（[联动型地震](https://zh.wikipedia.org/wiki/联系地震 "wikilink")）的说法已成为定论。另一方面，对庆长地震的震源位于南海海沟的说法也出现了反对意见，存在认为南海海沟的地震是间隔200年发生的见解\[4\]。

根据最新的通过地震计获得观测记录的昭和的地震，以及通过地质调查及文献资料推算出的较早的地震，可以推测出今后也将以同样的间隔发生地震。无论历史还是预测都是[里氏](https://zh.wikipedia.org/wiki/里氏震级 "wikilink")8级以上的特大地震，因摇晃及[海啸](../Page/海啸.md "wikilink")造成巨大灾害。

此外，之后的研究发现，每发生一次地震，[震源](../Page/震源.md "wikilink")地都在一点点发生变化。例如，即使是同样南海道海面的地震，相对于1854年安政南海地震的震源地南海道海面全域，1946年昭和南海地震的推测震源地为西侧四分之一处。\[5\]

## 地震的发生概率

|       |            |             |
| ----- | ---------- | ----------- |
| 区域    | 类型         | 截至2018年1月1日 |
| 规模（M） | 30年内发生的可能性 |             |
| 南海海沟  | 板块间地震      | M8-9级       |

发生概率等的评价（地震调查委员会）

作为接下来可能发生的地震，我们着眼相较从前范围更广的M8 - 9级地震。通过高知县室津港因历代南海地震（宝永·安政·昭和）造成的隆起量，和基于与地震发生间隔的关系制作的时间预测模型\[6\]\[7\]，推测出下一次的M8级地震将发生在[昭和南海地震之后的](../Page/1946年南海道大地震.md "wikilink")88.2年，并以此计算出了上表的概率。虽然接下来也有发生最大级别（超越M9）的地震的可能性，但发生频率（包括历史上的）比起以100 - 200年间隔发生的地震要「低一位数以上」\[8\]。

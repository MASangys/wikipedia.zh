**卡巴尔达语**属于[高加索诸语言](../Page/高加索诸语言.md "wikilink")之[北高加索語系](../Page/北高加索語系.md "wikilink")下之[西北高加索語系](https://zh.wikipedia.org/wiki/西北高加索語系 "wikilink")，主要用在[俄罗斯](../Page/俄罗斯.md "wikilink")[卡巴尔达-巴尔卡尔共和国和](https://zh.wikipedia.org/wiki/卡巴尔达-巴尔卡尔共和国 "wikilink")[卡拉恰伊-切尔克斯共和国](../Page/卡拉恰伊-切尔克斯共和国.md "wikilink")，和[土耳其](../Page/土耳其.md "wikilink")及[中东](../Page/中东.md "wikilink")。它有47-48个辅音，[擦音就多达](https://zh.wikipedia.org/wiki/擦音 "wikilink")22-23个。

| \[+high, -back\]  | 在[邊音](https://zh.wikipedia.org/wiki/邊音 "wikilink")、顎化軟顎音（palatalized palatovelar）與 /j/ 之後 | \[i\]     | \[e\]    |

## 文法

### 格

卡巴爾達語有4個格：作格、主格、工具格，及狀格(adberbial)。如下所示：

| [格](../Page/格_\(语法\).md "wikilink") | [後綴](../Page/後綴.md "wikilink") | 範例                      |
| ----------------------------------- | ------------------------------ | ----------------------- |
| 主格                                  | \-р *-r*                       | дзэ**р** *dza**r***     |
| 作格                                  | \-м *-m*                       | дзэ**м** *dza**m***     |
| 工具格                                 | \-kӀэ *-cç’a*                  | дзэ**kӀэ** *dza**cça*** |
| 狀格                                  | \-уэ *-wa*                     | дзэ**уэ** *dza**wa***   |

### 連接詞

下列為一些[動詞的位置](https://zh.wikipedia.org/wiki/動詞 "wikilink")[連接詞](https://zh.wikipedia.org/wiki/連接詞 "wikilink")，表示出其詞根位之變化：

|         |                |                |                 |
| :-----: | :------------: | :------------: | :-------------: |
|         |     **站**      |     **坐**      |      **躺**      |
| 身體位置/姿勢 |  щыт (*ʃət*)   |  щыс (*ʃəs*)   |  щылъ (*ʃəɬ*)   |
|  在…之上   |  тет (*tajt*)  |  тес (*tajs*)  |  телъ (*tajɬ*)  |
|  在…之下   | чIэт (*cç’at*) | щIэс (*cç’as*) | щIэлъ (*cç’aɬ*) |
|  在…之中   |  хэт (*xat*)   |  хэс (*xas*)   |  хэлъ (*xaɬ*)   |
|  在…區域內  |  дэт (*dat*)   |  дэс (*das*)   |  дэлъ (*daɬ*)   |
|  在…之後   |  Iут (*wət*)   |  Iyc (*wəs*)   |  Iулъ (*wəɬ*)   |

範例：

щыт - 站 (作為一種姿勢);

Iут - 站 (在…之後);

чIэт - 站 (在…之下)、等。

*' 卡比耶語*'（ 或
，發音:）是一種通行於[多哥北部](../Page/多哥.md "wikilink")、[貝寧](../Page/貝寧.md "wikilink")、和[迦納的](../Page/迦納.md "wikilink")[尼日爾
-
剛果語系](../Page/尼日尔-刚果语系.md "wikilink")[古爾下語支的](../Page/:en:Gur_languages.md "wikilink")[語言](../Page/語言.md "wikilink")，其使用者約有一百萬人。其他與埃維語相關、同屬古爾下語支的語言有[喇嘛語](../Page/:en:Lama.md "wikilink")、[沙拉語](../Page/:en:Chala.md "wikilink")、[帕納語和](../Page/:en:Pana.md "wikilink")[格森語等](../Page/:en:Kasem.md "wikilink")。在20世紀之間卡比耶人也廣泛移居至[多哥中部](../Page/多哥.md "wikilink")，南部地區以及到邻近的[加納和](../Page/加納.md "wikilink")[貝寧](../Page/貝寧.md "wikilink")。卡比耶語是[多哥国家的](../Page/多哥.md "wikilink")[官方语言之一](../Page/官方语言.md "wikilink")，在多哥全國[媒體和在正規](../Page/媒體.md "wikilink")[教育部門使用](../Page/教育.md "wikilink")。根據1999年[洛美](../Page/洛美.md "wikilink")[統計局](../Page/統計.md "wikilink")，大概有23%的[多哥民眾會使用卡比耶語](../Page/多哥.md "wikilink")\[1\]
。

## 語法

卡比耶語是[主動賓語序語言](../Page/主動賓語序.md "wikilink")，有十個[名詞類別](../Page/名詞類別.md "wikilink")：

| [類別](../Page/名詞類別.md "wikilink") | 例如                    | 翻譯        |
| -------------------------------- | --------------------- | --------- |
| **1**                            | \[hàl-ʋ́ nɔ̀ɔ́yʋ̀\]   | *某個婦人*    |
| **2**                            | \[hàl-áà nàbɛ̀yɛ̀\]   | *某些婦人*    |
| **3**                            | \[hàk-ʋ́ nàkʋ́yʋ̀\]   | '' 某個鋤頭'' |
| **4**                            | \[hàk-ɩ́ŋ̀ nɩ̀ɩ́yɛ̀\] | *某些鋤頭*    |
| **5**                            | \[sùmɑ̀-ɑ́ nàkɛ́yɛ̀\] | *某個鳥*     |
| **6**                            | \[sùmá-sɩ̀ nàsɩ́yɛ̀\] | *某些鳥*     |
| **7**                            | \[hɩ́-ɖɛ̀ nàɖɩ́yɛ̀\]  | *某個名稱*    |
| **8**                            | \[hɩ̀-lá nàáyɛ̀\]     | '' 某些名稱'' |
| **9**                            | \[há-tʋ̀ nàtʋ́yʋ̀\]   | '' 某個葉子'' |
| **10**                           | \[càlɩ́-m̀ nàbʋ́yʋ̀\] | '' 某些葉子'' |

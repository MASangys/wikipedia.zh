*' 卡比耶語*'（ 或 ，發音:）是一種通行於[多哥](../Page/多哥.md "wikilink")北部、[貝寧](https://zh.wikipedia.org/wiki/貝寧 "wikilink")、和[迦納的](https://zh.wikipedia.org/wiki/迦納 "wikilink")[尼日爾 - 剛果語系](../Page/尼日尔-刚果语系.md "wikilink")的[語言](../Page/語言.md "wikilink")，其使用者約有一百萬人。其他與埃維語相關、同屬古爾下語支的語言有[喇嘛](../Page/喇嘛.md "wikilink")、、和等。在20世紀之間卡比耶人也廣泛移居至[多哥](../Page/多哥.md "wikilink")中部，南部地區以及到邻近的[加納和](https://zh.wikipedia.org/wiki/加納 "wikilink")[貝寧](https://zh.wikipedia.org/wiki/貝寧 "wikilink")。卡比耶語是[多哥](../Page/多哥.md "wikilink")国家的[官方语言](../Page/官方语言.md "wikilink")之一，在多哥全國[媒體和在正規](https://zh.wikipedia.org/wiki/媒體 "wikilink")[教育](../Page/教育.md "wikilink")部門使用。根據1999年[洛美](../Page/洛美.md "wikilink")[統計局](https://zh.wikipedia.org/wiki/統計 "wikilink")，大概有23%的[多哥](../Page/多哥.md "wikilink")民眾會使用卡比耶語\[1\] 。

## 語法

卡比耶語是[主動賓語序語言](https://zh.wikipedia.org/wiki/主動賓語序 "wikilink")，有十個[名詞類別](../Page/名詞類別.md "wikilink")：

| [類別](../Page/名詞類別.md "wikilink") | 例如                    | 翻譯        |
| -------------------------------- | --------------------- | --------- |
| **1**                            | \[hàl-ʋ́ nɔ̀ɔ́yʋ̀\]   | *某個婦人*    |
| **2**                            | \[hàl-áà nàbɛ̀yɛ̀\]   | *某些婦人*    |
| **3**                            | \[hàk-ʋ́ nàkʋ́yʋ̀\]   | '' 某個鋤頭'' |
| **4**                            | \[hàk-ɩ́ŋ̀ nɩ̀ɩ́yɛ̀\] | *某些鋤頭*    |
| **5**                            | \[sùmɑ̀-ɑ́ nàkɛ́yɛ̀\] | *某個鳥*     |
| **6**                            | \[sùmá-sɩ̀ nàsɩ́yɛ̀\] | *某些鳥*     |
| **7**                            | \[hɩ́-ɖɛ̀ nàɖɩ́yɛ̀\]  | *某個名稱*    |
| **8**                            | \[hɩ̀-lá nàáyɛ̀\]     | '' 某些名稱'' |
| **9**                            | \[há-tʋ̀ nàtʋ́yʋ̀\]   | '' 某個葉子'' |
| **10**                           | \[càlɩ́-m̀ nàbʋ́yʋ̀\] | '' 某些葉子'' |

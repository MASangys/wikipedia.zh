**卡累利阿语**是非常近于芬兰语的一种[语言](../Page/语言.md "wikilink")，或被认为是[芬兰语的一种方言](../Page/芬兰语.md "wikilink")。主要使用者位于[俄罗斯联邦境内的](../Page/俄罗斯联邦.md "wikilink")[卡累利阿共和国](../Page/卡累利阿共和国.md "wikilink")。它属于[乌拉尔语系](../Page/乌拉尔语系.md "wikilink")[芬兰-乌戈尔语族](../Page/芬兰-乌戈尔语族.md "wikilink")[芬兰语支](../Page/芬兰语支.md "wikilink")。卡累利阿语无标准语，尽管卡累利阿共和国官方试图制订标准语。其书写形式按照方言口语书写。

## 音系

### 元音

#### 單元音

卡累利阿語若不考慮[元音長度](../Page/元音長度.md "wikilink")，共有8個[元音](../Page/元音.md "wikilink")：

|                                | [前](../Page/前元音.md "wikilink") | [後](../Page/後元音.md "wikilink") |
| ------------------------------ | ------------------------------ | ------------------------------ |
| <small>不圓唇</small>             | <small>圓唇</small>              |                                |
| [閉](../Page/閉元音.md "wikilink") | i                              | y                              |
| [中](../Page/中元音.md "wikilink") | e                              | ö                              |
| [開](../Page/開元音.md "wikilink") | ä                              |                                |

只有閉元音、與可能為長元音。\[1\]的長中元音與長開元音則已經被雙元音化：\*ee, \*öö, \*oo → /ie/, /yö/,
/uo/（也同樣在[芬蘭語發生](../Page/芬蘭語.md "wikilink")）；\*aa, \*ää → /oa/, /eä/ 或
/ua/, /iä/（也同樣在芬蘭語的發生）。

#### 雙元音

北卡累利阿語\[2\]與[奧洛涅茨卡累利阿語](../Page/奧洛涅茨.md "wikilink")\[3\]有21組[雙元音](../Page/雙元音.md "wikilink")：

|      | 前-和諧 | 中性   | 後-和諧 |
| ---- | ---- | ---- | ---- |
| 前-中性 | 前+前  | 中性+前 | 中性+後 |
| 開→閉  | äi   | äy   |      |
| 中→閉  | öi   | öy   | ey   |
| 閉    | yi   |      | iy   |
| 閉→中  |      | yö   |      |
| 閉→開  |      | yä   | iä   |

#### 三元音

除了雙元音以外，北方卡累利阿語尚有一系列的[三元音](../Page/三元音.md "wikilink")：\[4\]

|       | 前-和諧 | 中性   | 後-和諧 |
| ----- | ---- | ---- | ---- |
| 前-中性  | 前+前  | 中性+前 | 中性+後 |
| 閉-中-閉 |      | yöy  | iey  |
| 閉-開-閉 | yäi  | yäy  | iäy  |

奧洛涅茨卡累利阿語則只有*ieu*、*iey*、*iäy*、*uau*、*uou*與*yöy*.\[5\]

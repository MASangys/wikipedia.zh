**卤化银**：卤素（[氟](../Page/氟.md "wikilink")、[氯](../Page/氯.md "wikilink")、[溴](../Page/溴.md "wikilink")、[碘](../Page/碘.md "wikilink")等）与银元素形成的盐（[氟化银](../Page/氟化银.md "wikilink")、[氯化银](../Page/氯化银.md "wikilink")、[溴化银](https://zh.wikipedia.org/wiki/溴化银 "wikilink")、[碘化银](../Page/碘化银.md "wikilink")等）因为具有相似化学性质而被统称为卤化银。

  - [氟化银](../Page/氟化银.md "wikilink")（白色粉末，易溶于水）
  - [氯化银](../Page/氯化银.md "wikilink")（白色沉淀）
  - [溴化银](https://zh.wikipedia.org/wiki/溴化银 "wikilink")（浅黄色沉淀）
  - [碘化银](../Page/碘化银.md "wikilink")（黄色沉淀）

由于其高感光性，氯化银和溴化银大量用于摄影技术中。

## 性质

除[碘化银](../Page/碘化银.md "wikilink")为[硫化锌](../Page/硫化锌.md "wikilink")型结构外，其他卤化银都为[氯化钠型结构](https://zh.wikipedia.org/wiki/氯化钠 "wikilink")。\[1\]

从氟化银到碘化银，卤化银的溶解度逐渐降低，氯化银、溴化银和碘化银都是难溶于水的。它们的溶解度数据列在下表中：\[2\]

|      |                                |
| ---- | ------------------------------ |
| 化合物  | 溶解度 (g / 100 g H<sub>2</sub>O) |
| AgF  | 172                            |
| AgCl | 0.00019                        |
| AgBr | 0.000014                       |
| AgI  | 0.000003                       |

**卤化银的溶解度数据**

  - 氯化银可以溶于[浓盐酸](https://zh.wikipedia.org/wiki/浓盐酸 "wikilink")、[硫代硫酸钠](../Page/硫代硫酸钠.md "wikilink")、[氰化物](../Page/氰化物.md "wikilink")和[氨水](../Page/氨水.md "wikilink")中（通过生成相应的银[配合物](../Page/配合物.md "wikilink")）；
  - 溴化银可以溶于[氢溴酸](../Page/氢溴酸.md "wikilink")、硫代硫酸钠、氰化物和氨水中，但溶于氨水的反应比氯化银要慢得多；
  - 碘化银可以溶于[氢碘酸](../Page/氢碘酸.md "wikilink")、硫代硫酸钠和氰化物中。与氨反应时，则生成白色的AgI·2NH<sub>3</sub>。

  - 除了一价的[氟化银](../Page/氟化银.md "wikilink")外，还有[一氟化二银](../Page/一氟化二银.md "wikilink")和[二氟化银](../Page/二氟化银.md "wikilink")存在。

  -

**印度象棋** 是18到19世紀期間流行於印度的象棋變體，一般認為是從[古印度象棋](../Page/古印度象棋.md "wikilink")演變而來，但相較之下和現代的國際象棋更為相似。印度象棋在1960年代之後逐漸退流行而被國際象棋取代。

## 規則

印度象棋的兵種可以在國際象棋中找到完全的對應，基本的走法也是互相一致呼應的，擺設位置也幾乎是一樣的(除了王和-{zh-tw:后}-，詳見下文)。然而兵種的名稱在不同地區之間有相當大的差異，尤其是車和象這兩個棋子的名稱在某些地方甚至完全相反。而為了避免混淆，此頁面一律使用國際象棋中的稱呼。

| 印度象棋的兵種   | 所對應的國際象棋的兵種  | 走法                  |
| --------- | ------------ | ------------------- |
| 王         | 王            | 直向或斜向一格             |
| 大臣/王妃     | \-{zh-tw:后}- | 直向或斜向任意格            |
| 象/車/船/大將  | 車            | 直向任意格               |
| 馬         | 馬            | 跳躍至2x3的對角格          |
| 駱駝/象/車/大臣 | 象            | 斜向任意格               |
| 兵         | 兵            | 走時直向前進一格，吃子時則斜向前進一格 |


\===與國際象棋的差異===
\*開局時雙方的王和-{zh-tw:后}-是互相錯行而非在同一行相對的。

  - 開局時首3步稱為**自由佈局**，在這個階段雙方不允許吃子或將軍。
  - 位於起始位置的兵不可以走兩格，故也沒有吃過路兵。兵無論如何都只走一格。
  - 無王和車的易位規則。然而在對局中王**在被將之前可以有一次機會**用馬的方式跳躍，但不可以這樣吃子。這種走法叫做王的**馬步易位**。
  - 升當兵走到底列時，只能升變成為**已被吃掉的棋子**並必須升變成**所在的那一行對應的初始兵種**，例如在a行的兵必須升變成車。然而在王那一行的兵升變成-{zh-tw:后}-，不可變王。
  - 如果兵在底列而沒有被吃掉的棋子供其升變的話，將待在原處一直到有符合的棋子被吃掉，方能立即升變。當兵要升變成象時，必須注意到**白色格象與黑色格象的區別**，意思是比在白格的兵只能在白格象被吃掉的狀況下升變成白格的象。因此絶對不可能透過升變出現一方有兩個同色格象的情況。
  - 當兵升變成馬時可以在該回合立即移動這個馬，相當於走兩次。只有升變成馬時有此特權。
  - 當國王不被將但無子可動時(即在國際象棋視為逼和的情況)，可以隨心所欲的移除敵方王以外的任何一個棋子，好讓自己可以移動。
  - 當一方被吃到只剩下王和兵時，是和局，這種情況叫做**無助和**。
  - 在下賭注的對局或計分的比賽中，用不同兵種將死有著不同的價值。用兵將死的價值最高，其次是用馬和象，再來是車，最沒價值的是-{zh-tw:后}-。

## 策略


印度象棋的遊戲策略在許多方面如同國際象棋，但許多獨特規則又使他獨樹一格。

在開局階段，受到自由佈局、兵只能走一格的限制，印度象棋的出子與進攻比起國際象棋還稍緩慢一些，而馬步易位使得王更靈活更有機動性。在中局和殘局時，由於不同行的兵升能升變的兵種不同，使得在某些行的兵更具有價值，衍生出利用棄子迫使對方的兵換行而只能升變較弱的棋子的戰術。同樣的，也可以利用棄子讓某個等待升變的兵得以升變。無助和的規則也讓印度象棋的殘局形成自己的特色。當一方除了王和兵以外只剩下一個棋子時，該子稱為**無懼子**，由於無懼子一旦被吃掉就會立刻造成無助和，因此可以不留後顧地瘋狂攻擊或棄子，削弱敵方強勢的局面。擁有優勢子力的一方若是想獲勝，在進入殘局時，需要特別小心避免這種尷尬的局面。

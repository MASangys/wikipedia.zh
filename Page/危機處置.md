**危機**
(古希臘語：krisis)意謂身處在一個決定性的緊要關頭或轉折點，而危機的英文([crisis](../Page/:en:crisis.md "wikilink"))意涵同時是危險(Danger)與機會(Opportunity)([chinese
word for
"crisis"](../Page/:en:chinese_word_for_"crisis".md "wikilink"))。具體來說，危機是「個人在面對一個既無法逃脫，且利用現有的問題解決資源又無法解決危急事件時所經驗的一段短暫心理不平衡(disequilibrium)時期」\[1\]
\[2\]\[3\]\[4\]。

**危機處置** ([crisis
intervention](../Page/:en:crisis_intervention.md "wikilink"))不是長期性的介入方式，而是有時限性(time-limited)的，其主要在協助個人或團體安然渡過混亂事件的助人過程，使其能夠縮小負面的影響程度。除了幫助個案回復到危機發生前的功能狀況，並加大成長的可能性\[5\]\[6\]\[7\]。

## 危機的類型與實例

| 類 型                                                                         | 實 例                                                      |
| --------------------------------------------------------------------------- | -------------------------------------------------------- |
| 情境性危機(situational crises)                                                   | 自然災害、意外事故、心臟病發作、愛人死亡、公司併購、失業、綁架、劫機、強暴、槍殺、自殺、離婚、家暴、兒童性虐待等 |
| 發展性危機(developmental crises)                                                 | 入學、進入青春期、結婚、生子、大學畢業、退休、二度就業等                             |
| 存在價值性危機([existential crises](../Page/:en:existential_crises.md "wikilink")) | 近四十歲但未在某專業領域或機構組織占有一席之地者、中年危機等                           |
| 環境性危機(environmental crises)                                                 | 工作歧視、違犯社會常模、空氣污染等                                        |

(※資料來源：\[8\]\[9\])

## 危機處置vs.長期心理諮商及治療

|      | 危機處置                                     | 長期心理諮商及[心理治療](../Page/心理治療.md "wikilink") |
| ---- | ---------------------------------------- | ----------------------------------------- |
| 診斷   | 快速三向度(triage)的危機評估。                      | 完整的診斷性衡鑑。                                 |
| 計畫   | 針對個案特定問題，規劃出解決的處置方案，並且主要聚焦在緩和危機症狀的立即性需求。 | 針對個案性格問題，規劃出綜合性處遇計畫，並且主要聚焦在個案的長期性需求。      |
| 方法   | 運用「時限短期諮商」的理論與技術，以立即控制與阻遏危機個案的創傷。        | 短期、中期、長期治療理論與技術之系統性運用。                    |
| 結果評估 | 以個案能夠重新回到危機發生前的平衡狀態，為其確認處置成效的指標。         | 以個案整體功能的治療結果，為其確認處遇成效的指標。                 |

(※資料來源：\[12\])

《即刻槍戰》(英文：GunsRush)，是[慧邦科技](https://zh.wikipedia.org/wiki/慧邦科技 "wikilink")（原 神來也）旗下的一款網頁第三人稱射擊遊戲，標榜「免下載，立刻玩」、「五分鐘，儲好儲滿，想樂最輕鬆」。

為臺灣首款使用Unity遊戲引擎自製的3D射擊遊戲，行銷地區包括[臺灣](../Page/臺灣.md "wikilink")、[香港](../Page/香港.md "wikilink")、[澳門](../Page/澳門.md "wikilink")、[新加坡](../Page/新加坡.md "wikilink")、[馬來西亞](https://zh.wikipedia.org/wiki/馬來西亞 "wikilink")、[美國等](../Page/美国.md "wikilink")，總部設於[臺北](../Page/臺北市.md "wikilink")。[香港](../Page/香港.md "wikilink")及[澳門](../Page/澳門.md "wikilink")由[alta公司代理營運](../Page/AltaVista.md "wikilink")。

## 遊戲模式

| 模式名稱 | 模式簡介                                                                                                           |
| ---- | -------------------------------------------------------------------------------------------------------------- |
| 殲滅模式 | 殲滅敵隊，一方達到50分時提早遊戲結束。                                                                                           |
| 佔領模式 | 依序佔領敵方據點，佔領主據點或先達到100分時遊戲結束。                                                                                   |
| 爆破模式 | 攻方炸毀爆破目標點；守方保護爆破目標點。                                                                                           |
| 亂鬥模式 | 個人賽模式，可攻擊自己以外的所有玩家。時間到或一位玩家達到35分時提早遊戲結束                                                                        |
| 生存模式 | 消滅人類或感染體。                                                                                                      |
| 感染模式 | 人類需在時間內存活，感染體則需在時間內感染所有人類。                                                                                     |
| 奪寶模式 | 奪取寶石後放置在一定地點，時間到後則該方獲勝。([2018/5/3改版](https://www.gunsrush.com/gunsrush/?op=announce&id=1010)後<u>**已無法遊玩**</u>) |
| 收集模式 | 收集場上金幣，時間到後金幣最多者獲勝。([2018/5/3改版](https://www.gunsrush.com/gunsrush/?op=announce&id=1010)後**<u>已無法遊玩</u>**)     |
| 擂台模式 | 單人對戰，可自行組隊對戰其他玩家。於一分鐘內擊敗敵隊玩家可繼續挑戰下位玩家，直到敵方玩家全被殲滅時該方獲勝。                                                         |
| 團隊競技 | 可自行組隊四人對戰其他玩家，分為團隊殲滅、團隊爆破、團隊佔領。                                                                                |
| 鬥陣競技 | 限定可自行組隊四人對戰其他玩家，此模式可自由選擇開放轉蛋槍枝。(每星期六開放)                                                                        |
| 槍技鬥陣 | 限定可自行組隊四人對戰其他玩家，此模式可自由選擇開放商城槍枝。(每兩週星期日開放)                                                                      |
| 狩獵模式 | 擊殺所有關卡BOSS與菁英小怪。                                                                                               |
| 勢力戰  | 所有玩家隨機分配到兩大陣營，遊戲中擊殺敵方BOSS、玩家、NPC。時間到或是一方先達到250分時遊戲結束。                                                          |
| 軍團戰  | 各軍團可在期間互相切磋，積分最高的軍團可控制領地。                                                                                      |
|      |                                                                                                                |

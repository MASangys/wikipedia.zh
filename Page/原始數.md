**原始數**（）是指一個[自然數](https://zh.wikipedia.org/wiki/自然數 "wikilink")*n*，可以用其十進制下的各位數組合出其他[質數](https://zh.wikipedia.org/wiki/質數 "wikilink")，而且其質數的數量比其他較小數字所能產生的質數更多。數學家Mike Keith是第一個提出原始數概念的人。

以[13](../Page/13.md "wikilink")為例，所有的1位數最多都只能產生一個質數，[10](../Page/10.md "wikilink")可以組合出[0](../Page/0.md "wikilink"),[1](../Page/1.md "wikilink"),10，都不是質數，[11](../Page/11.md "wikilink")可以組合出,1,11，其中只有11是質數，[12](../Page/12.md "wikilink")可以組合出1,[2](../Page/2.md "wikilink"),12,21，其中只有2是質數，而13可以組合出1,[3](../Page/3.md "wikilink"),13,[31](https://zh.wikipedia.org/wiki/31 "wikilink")，其中可組合出3,13,31等3個質數，比用其他較小數字時所能產生的質數要多，因此13是原始數。

頭幾個原始數是：

  -
    1, 2, 13, [37](../Page/37.md "wikilink"), [107](https://zh.wikipedia.org/wiki/107 "wikilink"), [113](../Page/113.md "wikilink"), [137](https://zh.wikipedia.org/wiki/137 "wikilink"), 1013, 1037, 1079, 1237, 1367, ...

其可以產生的質數個數為：

  -
    0, 1, 3, 4, 5, 7, 11, 14, 19, 21, 26, 29, ...

在*n*位數的原始數選擇一個，所能產生的最多質數的個數為：

  -
    1, 4, 11, 31, 106, ...

依上述方式，在*n*位數的質數中可以產生的最小質數為：

  -
    2, 37, 137, 1379, 13679, ...

原始數不一定要是質數，第一個是[合數的原始數是](https://zh.wikipedia.org/wiki/合數 "wikilink")1037 = 17×61，原始質數（Primeval prime）是指同時是原始數及質數的數：

  -
    2, 13, 37, 107, 113, 137, 1013, 1237, 1367, 10079, ...

以下列出前6個原始數及其可以產生的質數：

| 原始數 | 產生質數                         | 質數個數 |
| --- | ---------------------------- | ---- |
| 1   | 無                            | 0    |
| 2   | 2                            | 1    |
| 13  | 3, 13, 31                    | 3    |
| 37  | 3, 7, 37, 73                 | 4    |
| 107 | 7, 17, 71, 107, 701          | 5    |
| 113 | 3, 11, 13, 31, 113, 131, 311 | 7    |

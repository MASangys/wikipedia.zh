**反應堆級鈈**是指在[反應堆中經過若乾年燃燒后由](https://zh.wikipedia.org/wiki/反應堆 "wikilink")[鈾-238經](https://zh.wikipedia.org/wiki/鈾-238 "wikilink")[中子俘獲和兩次連續的](https://zh.wikipedia.org/wiki/中子俘獲 "wikilink")[β衰變生成的](https://zh.wikipedia.org/wiki/β衰變 "wikilink")[鈈-239](https://zh.wikipedia.org/wiki/鈈-239 "wikilink")，再被[嬗變成其它鈈的](https://zh.wikipedia.org/wiki/嬗變 "wikilink")[同位素](../Page/同位素.md "wikilink")之後形成的混合物。

\[\mathrm{^{238}_{\ 92}U \ + \ ^{1}_{0}n \ \longrightarrow \ ^{239}_{\ 92}U \ \xrightarrow [23,5\ min]{\beta^-} \ ^{239}_{\ 93}Np \ \xrightarrow [2,3565\ d]{\beta^-} \ ^{239}_{\ 94}Pu}\]

目前的[核電站一般使用的](https://zh.wikipedia.org/wiki/核電站 "wikilink")[熱中子堆只能在有限程度上使用反應堆級鈈](https://zh.wikipedia.org/wiki/熱中子堆 "wikilink")，比如[混合氧化物核燃料](../Page/混合氧化物核燃料.md "wikilink")。這種燃料產生的[乏燃料中的](https://zh.wikipedia.org/wiki/乏燃料 "wikilink")[次錒系元素和含有偶數個中子的鈈同位素含量較高](https://zh.wikipedia.org/wiki/次錒系元素 "wikilink")，因此使用一次後就只能作為[核廢料處理](https://zh.wikipedia.org/wiki/核廢料 "wikilink")。[快中子反應堆現在還沒有商業化](https://zh.wikipedia.org/wiki/快中子反應堆 "wikilink")，但在理論上可以直接使用反應堆級鈈。

根據[美國能源部](../Page/美國能源部.md "wikilink")的定義，反應堆級鈈主要由[鈈-240的含量來決定](https://zh.wikipedia.org/wiki/鈈-240 "wikilink")。傳統上認爲，鈈-240含量越高，用於[裂變](https://zh.wikipedia.org/wiki/裂變 "wikilink")[核武器](../Page/核武器.md "wikilink")的可能性就越小。但是近年來，這種看法越來越受到苛責。雖然含大量鈈-240的鈈不能用作裂變武器，可是隨著技朮進步，只要使用[聚變增強](https://zh.wikipedia.org/wiki/聚變增強 "wikilink")，反應堆級鈈便可以用於[熱核武器](https://zh.wikipedia.org/wiki/熱核武器 "wikilink")。\[1\]比如在迄今爲止[當量最大的](../Page/核武器当量.md "wikilink")[沙皇炸彈](../Page/沙皇炸彈.md "wikilink")裏面，[包覆融合芯](https://zh.wikipedia.org/wiki/包覆融合芯 "wikilink")（tamper）採用的材料是[鉛](https://zh.wikipedia.org/wiki/鉛 "wikilink")。如果使用[貧化鈾或者反應堆級鈈](https://zh.wikipedia.org/wiki/貧化鈾 "wikilink")，該彈的當量會進一步提高。\[2\]\[3\]

## 按同位素組成分類

|       | \<1976年 | \>1976年 |
| ----- | ------- | ------- |
| \<7%  | 武器級     |         |
| 7-19% | 反應堆級    | 燃料級     |
| \>19% | 反應堆級    |         |

[Sasahara.svg](https://zh.wikipedia.org/wiki/File:Sasahara.svg "fig:Sasahara.svg")中各種[核素占縂嬗變率的百分比](https://zh.wikipedia.org/wiki/核素 "wikilink")。\]\]

1976年美國能源部對鈈的定義發生了改變。在1976年前祇有兩種分類：武器級鈈（鈈-240含量少於7%）和反應堆級鈈（鈈-240含量超過7%）。1976年之後，又增加了一個分類：燃料級鈈。武器級鈈的定義仍然不變。燃料級鈈的鈈-240含量在7-19%之間，而反應堆級鈈含有超過19%的鈈-240。根據這個定義，目前[輕水堆或者](https://zh.wikipedia.org/wiki/輕水堆 "wikilink")[沸水堆](https://zh.wikipedia.org/wiki/沸水堆 "wikilink")[乏燃料再處理中得到的鈈均屬於反應堆級鈈](https://zh.wikipedia.org/wiki/乏燃料 "wikilink")，而不是燃料級鈈。厤史上，鈈的同位素組成分類對於控制[核擴散曾非常重要](https://zh.wikipedia.org/wiki/核擴散 "wikilink")。

## 使用"反應堆級"鈈的核試驗

1962年，美國在[内華達試驗場進行了一次使用反應堆級鈈的地下](https://zh.wikipedia.org/wiki/内華達試驗場 "wikilink")[核試驗](https://zh.wikipedia.org/wiki/核試驗 "wikilink")，當量低於二萬噸[TNT](https://zh.wikipedia.org/wiki/TNT "wikilink")。\[4\]1977年7月，[卡特政府將部分關於該核試驗的文件解密](../Page/吉米·卡特.md "wikilink")，並以此作為禁止在[美國進行乏燃料](https://zh.wikipedia.org/wiki/美國 "wikilink")[再處理的依據](https://zh.wikipedia.org/wiki/再處理 "wikilink")。

据信這次核試驗的鈈來自[英國](https://zh.wikipedia.org/wiki/英國 "wikilink")。其同位素組成未知，外界僅知道這種燃料被稱為“反應堆級”。\[5\]這些鈈有可能來自英國軍方的[鎂諾克斯反應堆](https://zh.wikipedia.org/wiki/鎂諾克斯反應堆 "wikilink")。据猜測，核試驗中的鈈-239的含量至少為85%，大大超出現今一般反應堆乏燃料中的鈈-239含量。\[6\]

## 在反應堆中再次使用

快中子反應堆可以使用鈈做燃料，不論其同位素組成如何。1960年代，美國曾計劃提煉鈈，原因是有人猜測到隨著核電的發展，[鈾](../Page/鈾.md "wikilink")的價格會攀升。爲了充分利用鈾，就必須[快中子增殖反應堆](https://zh.wikipedia.org/wiki/快中子增殖反應堆 "wikilink")。結果鈾的供應增加，電的需求量下降，快中子堆的商業化被推遲了。

當今的熱中子堆只能在有限程度上使用反應堆級鈈，比如將它製成[鎂諾克斯燃料](https://zh.wikipedia.org/wiki/鎂諾克斯 "wikilink")。鈈的同位素中，只有[質量數](../Page/質量數.md "wikilink")為奇數的才能在吸收[熱中子後裂變](https://zh.wikipedia.org/wiki/熱中子 "wikilink")，質量數為偶數的鈈同位素會在反應堆中積累下來。鈈-240和鈾-238一樣是一種[可增殖材料](https://zh.wikipedia.org/wiki/可增殖材料 "wikilink")。鈈-240俘獲一個中子後變成鈈-241。但是鈈-242的[中子俘獲截面低](https://zh.wikipedia.org/wiki/中子俘獲截面 "wikilink")，還必須俘獲三個中子才能成爲可裂變核素。這些特質無疑都影響了鈈的廣範應用。

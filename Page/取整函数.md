在[数学和](../Page/数学.md "wikilink")[计算机科学中](../Page/计算机科学.md "wikilink")，**取整函数**是一类将[实数映射到相近的](../Page/实数.md "wikilink")[整数的](../Page/整数.md "wikilink")[函数](../Page/函数.md "wikilink")。\[1\]

常用的取整函数有两个，分别是**下取整函数**和**上取整函数**。

**下取整函数**即為**取底符號**，在数学中一般记作\(\lfloor x \rfloor\)或者\(E(x)\)，在计算机科学中一般记作floor(*x*)，表示不超过*x*的整数中最大的一个。

  -
    \(\lfloor x \rfloor=\max\, \{n\in\mathbb{Z}\mid n\le x\}.\)

举例来说，\(\lfloor 3.633 \rfloor = 3\)，\(\lfloor 56 \rfloor = 56\)，\(\lfloor -2 \rfloor = -2\)，\(\lfloor -2.263 \rfloor = -3\)。对于非负的实数，其下取整函数的值一般叫做它的**整数部分**或**取整部分**。而\(x -\lfloor x\rfloor\)叫做*x*的[小数部分](../Page/小数.md "wikilink")。每个[分数都可以表示成其整数部分与一个](../Page/分数.md "wikilink")[真分数的和](../Page/真分数.md "wikilink")，而实数的整数部分和小数部分是与此概念相应的拓延。

下取整函数的符号也会用[方括号表示](../Page/方括号.md "wikilink")（\([x]\)），称作**[高斯符号](../Page/高斯符号.md "wikilink")**。

**上取整函数**即為**取頂符號**在数学中一般记作\(\lceil x \rceil\)，在计算机科学中一般记作ceil(*x*)，表示不小于*x*的整数中最小的一个。

  -
    \(\lceil x \rceil=\min\{n\in\mathbb{Z}\mid x\le n\}.\)

举例来说，\(\lceil 3.633 \rceil = 4\)，\(\lceil 56 \rceil = 56\)，\(\lceil -2 \rceil = -2\)，\(\lceil -2.263 \rceil = -2\)。

计算机中的上取整函数和下取整函数的命名来自于[英文的](../Page/英文.md "wikilink")*ceiling*（天花板）和*floor*（地板），相关的记法由[肯尼斯·艾佛森于](../Page/肯尼斯·艾佛森.md "wikilink")1962年引入。\[2\]

## 性质

对于下取整函数，有如下性质。

  - 按定义：

\[\lfloor x\rfloor \le x < \lfloor x \rfloor + 1\] 当且仅当*x*为整数时取等号。

  - 设x和n为正实数，则：

\[\left\lfloor \frac{n}{x} \right\rfloor \geq \frac{n}{x} - \frac{n-1}{x}\]

  - 下取整函数为[等幂运算](../Page/等冪.md "wikilink")：\(\lfloor\lfloor x\rfloor\rfloor=\lfloor x\rfloor\).
  - 对任意的整数*k*和任意实数*x*，

\[\lfloor {k+x} \rfloor = k + \lfloor x\rfloor.\]

  - 一般的[數值修約規則可以表述为将](../Page/數值修約規則.md "wikilink")*x*映射到floor(*x* +
    0.5)；
  - 下取整函数不是[连续函数](../Page/连续.md "wikilink")，但是[上半连续的](../Page/半連續性.md "wikilink")。作为一个分段的[常数函数](../Page/常数函数.md "wikilink")，在其导数有定义的地方，下取整函数导数为零。
  - 设*x*为一个实数，*n*为整数，则由定义，*n* ≤ *x*当且仅当*n* ≤ floor(*x*)。

<!-- end list -->

  - 用下取整函数可以写出若干个[素数公式](../Page/素数公式.md "wikilink")，但没有什么实际价值。
  - 对于非整数的*x*，下取整函数有如下的-{zh-cn:富裡葉;zh-tw:傅立葉;}-展开：

\[\lfloor x\rfloor = x - \frac{1}{2} + \frac{1}{\pi} \sum_{k=1}^\infty \frac{\sin(2 \pi k x)}{k}.\]

  - 对于[互素的正整数](../Page/互素.md "wikilink")*m*和*n*，有：

\[\sum_{i=1}^{n-1} \lfloor im / n \rfloor = (m - 1) (n - 1) / 2\]

  - 根据[Beatty定理](../Page/Beatty定理.md "wikilink")，每个正[无理数都可以通过下取整函数制造出一个整数集的](../Page/无理数.md "wikilink")[分划](../Page/分划.md "wikilink")。
  - 最后，对于每个正整数*k*，其在 [p 进制下的表示有](../Page/进位制.md "wikilink")
    \(\lfloor \log_{p}(k) \rfloor + 1\) 个[数位](../Page/数位.md "wikilink")。

对于上取整函数：

  - 显然有：

\[\lceil x \rceil = - \lfloor - x \rfloor\]

  - 以及：

\[x \leq \lceil x \rceil < x + 1\]

  - 对于整数*k*有：

<!-- end list -->

  -
    \(\lfloor k / 2 \rfloor + \lceil k / 2 \rceil = k\).

## 其它等式

  - \([x+1]=[x]+1\)
  - 设*x*为一个实数，*n*为整数，则

\[\sum_{k = 0}^{n - 1} E(x+\frac{k}{n}) = E(nx)\]

\[E(\frac{1}{n}E(nx))=E(x)\]

  - 对于两个[相反数的下取整函数](../Page/相反数.md "wikilink")，有：

<!-- end list -->

  -
    如果x为整数，则\(E(x) + E(-x) = 0\)
    否则\(E(x) + E(-x) = -1\)

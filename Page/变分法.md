**变分法**是处理[泛函](../Page/泛函.md "wikilink")的[数学](../Page/数学.md "wikilink")领域，和处理函数的普通[微积分相对](https://zh.wikipedia.org/wiki/微积分 "wikilink")。譬如，这样的[泛函](../Page/泛函.md "wikilink")可以通过未知函数的积分和它的导数来构造。变分法最终寻求的是[极值](../Page/极值.md "wikilink")函数：它们使得泛函取得极大或极小值。有些[曲线](../Page/曲线.md "wikilink")上的经典问题采用这种形式表达：一个例子是[最速降线](https://zh.wikipedia.org/wiki/最速降线 "wikilink")，在重力作用下一个粒子沿着该路径可以在最短时间从点A到达不直接在它底下的一点B。在所有从A到B的曲线中必须极小化代表下降时间的表达式。

变分法的关键定理是[欧拉－拉格朗日方程](https://zh.wikipedia.org/wiki/欧拉－拉格朗日方程 "wikilink")。它对应于泛函的[临界点](https://zh.wikipedia.org/wiki/临界点 "wikilink")。在寻找函数的极大和极小值时，在一个解附近的微小变化的分析给出一阶的一个近似。它不能分辨是找到了最大值或者最小值（或者都不是）。

变分法在[理论物理中非常重要](https://zh.wikipedia.org/wiki/理论物理 "wikilink")：在[拉格朗日力学](../Page/拉格朗日力学.md "wikilink")中，以及在[最小作用量原理](../Page/最小作用量原理.md "wikilink")在[量子力学](../Page/量子力学.md "wikilink")的应用中。变分法提供了[有限元方法的数学基础](https://zh.wikipedia.org/wiki/有限元方法 "wikilink")，它是求解[边界值问题的强力工具](https://zh.wikipedia.org/wiki/边界值问题 "wikilink")。它们也在材料学中研究材料平衡中大量使用。而在纯数学中的例子有，[黎曼在](https://zh.wikipedia.org/wiki/黎曼 "wikilink")[调和函数](../Page/调和函数.md "wikilink")中使用[狄利克雷原理](../Page/狄利克雷原理.md "wikilink")。

同样的材料可以出现在不同的标题中，例如[希尔伯特空间](../Page/希尔伯特空间.md "wikilink")技术，[莫尔斯理论](../Page/莫尔斯理论.md "wikilink")，或者[辛几何](../Page/辛几何.md "wikilink")。**变分**一词用于所有极值泛函问题。[微分几何](../Page/微分几何.md "wikilink")中的[测地线](../Page/测地线.md "wikilink")的研究是很显然的变分性质的领域。[极小曲面](../Page/极小曲面.md "wikilink")（[肥皂泡](../Page/肥皂泡.md "wikilink")）上也有很多研究工作，称为[普拉托问题](../Page/普拉托问题.md "wikilink")。

## 历史

变分法可能是从[约翰·伯努利](https://zh.wikipedia.org/wiki/约翰·伯努利 "wikilink")（1696）提出[最速曲线](../Page/最速降線問題.md "wikilink")（brachistochrone curve）问题开始出现的。\[1\]它立即引起了[雅各布·伯努利](../Page/雅各布·伯努利.md "wikilink")和[洛必达](https://zh.wikipedia.org/wiki/洛必达 "wikilink")（Marquis de l'Hôpital）的注意。但[欧拉首先详尽的阐述了这个问题](https://zh.wikipedia.org/wiki/莱昂哈德·欧拉 "wikilink")。他的贡献始于1733年，他的《变分原理》（Elementa Calculi Variationum）寄予了这门科学这个名字。[欧拉对这个理论的贡献非常大](https://zh.wikipedia.org/wiki/莱昂哈德·欧拉 "wikilink")。

[勒让德](../Page/阿德里安-马里·勒让德.md "wikilink")（1786）确定了一种方法，但在对极大和极小的区别不完全令人满意。[牛顿和](../Page/艾萨克·牛顿.md "wikilink")[莱布尼茨也是在早期关注这一学科](../Page/戈特弗里德·莱布尼茨.md "wikilink")，对于这两者的区别Vincenzo Brunacci（1810）、[高斯](https://zh.wikipedia.org/wiki/卡尔·弗里德里希·高斯 "wikilink")（1829）、[泊松](../Page/西莫恩·德尼·泊松.md "wikilink")（1831）、Mikhail Ostrogradsky（1834）、和[雅可比](https://zh.wikipedia.org/wiki/卡尔·雅可比 "wikilink")（1837）都曾做出过贡献。Sarrus（1842）的由[柯西](https://zh.wikipedia.org/wiki/奥古斯丁·路易·柯西 "wikilink")（1844）浓缩和修改的是一个重要的具有一般性的成就。Strauch（1849）、Jellett（1850）、Otto Hesse（1857）、Alfred Clebsch（1858）、和Carll（1885）写了一些其他有价值的论文和研究报告，但可能那个世纪最重要的成果是Weierstrass所取得的。他关于这个理论的著名教材是划时代的，并且他可能是第一个将变分法置于一个稳固而不容置疑的基础上的。1900年[希尔伯特发表的](../Page/大卫·希尔伯特.md "wikilink")23个问题中的第20和23个问题促进了其更深远的发展。

在20世纪[希尔伯特](../Page/大卫·希尔伯特.md "wikilink")、[埃米·诺特](../Page/埃米·诺特.md "wikilink")、Leonida Tonelli、[昂利·勒貝格](../Page/昂利·勒貝格.md "wikilink")和[雅克·阿达马](../Page/雅克·阿达马.md "wikilink")等人做出重要贡献。Marston Morse将变分法应用在[莫尔斯理论](../Page/莫尔斯理论.md "wikilink")中。Lev Pontryagin、Ralph Rockafellar和Clarke广义变分法[最优控制理论发展了新的数学工具](https://zh.wikipedia.org/wiki/最优控制 "wikilink")。

## 欧拉-拉格朗日方程

在理想情形下，一函數的极大值及极小值會出現在其[導數為](https://zh.wikipedia.org/wiki/導數 "wikilink")\(0\)的地方。同樣地，求解變分問題時也可以先求解相關的[欧拉-拉格朗日方程](https://zh.wikipedia.org/wiki/欧拉-拉格朗日方程 "wikilink")。以下以尋找連接平面上兩點\((x_1, y_1)\)和\((x_2, y_2)\)最短曲線的例子，說明求解的過程。曲線的長度為

\[A[f] = \int_{x_1}^{x_2} \sqrt{1 + [ f'(x) ]^2} \, dx\]

其中

\[f'(x) = \frac{df}{dx}, \,\] \(f(x_1)=y_1, \,\) \(f(x_2)=y_2 \,\)。

函數\(f\)至少需為一階可微的函數。若\(f_0\)是一個[局部最小值](../Page/极值.md "wikilink")，而\(f_1\)是一個在端點\(x_1\)及\(x_2\)取值为零并且至少有一階導數的函數，則可得到以下的式子

\[A[f_0] \le A[f_0 + \epsilon f_1]\]

其中\(\epsilon\)為任意接近\(0\)的數字。

因此\(A[f_0 + \epsilon f_1]\)對\(\epsilon\)的導數（**A的一階導數**）在\(\epsilon=0\)時必為\(0\)：

\(\frac{d}{d\epsilon} \int_{x_1}^{x_2} \left. \sqrt{1 + [ f_0'(x) + \epsilon f_1'(x)]^2} dx \right|_{\epsilon =0} = \int_{x_1}^{x_2} \left. \frac{(f_0'(x) + \epsilon f_1'(x)) f_1'(x)}{\sqrt{1 + [ f_0'(x) + \epsilon f_1'(x)]^2}}\right|_{\epsilon =0} dx = \int_{x_1}^{x_2} \frac{ f_0'(x)f_1'(x) } {\sqrt{1 + [ f_0'(x) ]^2}}\,dx = 0\)

此條件可視為在可微分函數的空間中，\(A[f_0]\)在各方向的導數均為\(0\)。若假設\(f_0\)二階可微（或至少[弱微分](../Page/弱微分.md "wikilink")存在），則利用[分部積分法](../Page/分部積分法.md "wikilink")可得

\[\int_{x_1}^{x_2}  f_1 (x) \frac{d}{dx}\left[ \frac{ f_0'(x) } {\sqrt{1 + [ f_0'(x) ]^2}} \right] \, dx =0,\]

其中\(f_1\)為在兩端點皆為0的任意二階可微函數。這是[變分法基本引理](../Page/變分法基本引理.md "wikilink")的一個特例：

\[I =\int_{x_1}^{x_2} f_1 (x) H (x) dx =0\]，

其中\(f_1\)為在兩端點皆為\(0\)的任意可微函數。

若存在\(x = \hat x\)使\(H (x) > 0\)，則在\(\hat x\)周圍有一區間的H也是正值。可以選擇\(f_1\)在此區間外為\(0\)，在此區間內為非負值，因此\(I >0\)，和前提不合。若存在\(x = \hat x\)使\(H (x) < 0\)，也可證得類似的結果。因此可得到以下的結論：

\[\frac{d}{dx}\left[ \frac{ f_0'(x) } {\sqrt{1 + [ f_0'(x) ]^2}} \right] =0\]，

由結論可推得下式：

\[\frac{d^2 f_0}{dx^2}=0\]，

因此兩點間最短曲線為一直線。

在一般情形下，則需考慮以下的計算式

\[A[f] = \int_{x_1}^{x_2}  L(x,f,f') dx\]，

其中*f*需有二階連續的導函數。在這種情形下，拉格朗日量L在極值\(f_0\)处滿足[欧拉-拉格朗日方程](https://zh.wikipedia.org/wiki/欧拉-拉格朗日方程 "wikilink")

\[-\frac{d}{dx} \frac{\part L}{\part f'} + \frac{\part L}{\part f}=0\]，

不過在此處，欧拉-拉格朗日方程只是有極值的[必要條件](https://zh.wikipedia.org/wiki/必要條件 "wikilink")，並不是充分條件。

## 費馬原理

[費馬原理](../Page/費馬原理.md "wikilink")指出：光會沿着兩端點之間所需[光程最短的路徑前進](https://zh.wikipedia.org/wiki/光程 "wikilink")。假設\(y=f (x)\)為光的路徑，則光程可以下式表示：

\[A[f] = \int_{x=x_0}^{x_1} n(x,f(x)) \sqrt{1 + f'(x)^2} dx\]，

其中[折射率](../Page/折射率.md "wikilink")\(n(x,y)\)依材料特性而定。

若選擇\(f (x) = f_0 (x) + \epsilon f_1 (x)\)，則\(A\)的一階導數（\(A\)對\(\epsilon\)的微分）為：

\[\delta A[f_0,f_1] = \int_{x=x_0}^{x_1} \left[ \frac{ n(x,f_0)f_0'(x)f_1'(x)}{\sqrt{1 + f_0'(x)^2}} + n_y(x,f_0) f_1\sqrt{1 + f_0'(x)^2}  \right] dx\]，

將括號中的第一項用分部積分處理，可得歐拉-拉格朗日方程

\[-\frac{d}{dx} \left[\frac{ n(x,f_0)f_0'}{\sqrt{1 + f_0'^2}} \right] + n_y(x,f_0) \sqrt{1 + f_0'(x)^2} =0\]。

光線的路徑可由上述的積分式而得。

### 斯乃爾定律

當光進入或離開透鏡面時，折射率會有不連續的變化。考慮

\[n(x,y) = n_- \quad \hbox{if} \quad x<0\]，

\[n(x,y) = n_+ \quad \hbox{if} \quad x>0\]，

其中\(n_-\)和\(n_+\)是常數。在*x*\<0或*x*\>0的區域，歐拉-拉格朗日方程均和以上描述的相同。因為折射率在二個區域均為定值，在二個區域光都以直線前進。而在*x*=0的位置，*f*必須連續，不過''f' ''可以不連續。在上述二個區域用分部積分的方式解歐拉-拉格朗日方程，則其變分量為

\[\delta A[f_0,f_1] = f_1 (0)\left[ n_-\frac{f_0'(0_-)}{\sqrt{1 + f_0'(0_-)^2}} -n_+\frac{f_0'(0_+)}{\sqrt{1 + f_0'(0_+)^2}} \right]\]。

和\(n_-\)相乘的係數是入射角的正弦值，和\(n_+\)相乘的係數則是折射角的正弦值。若依照[斯涅爾定律](https://zh.wikipedia.org/wiki/斯涅爾定律 "wikilink")，上述二項的乘積相等，因此上述的變分量為0。因此斯涅爾定律所得的路徑也就是要求光程一階變分量為0的路徑。

### 費馬原理在三維下的形式

費馬原理可以用向量的形式表示：令\(X=(x_1,x_2,x_3)\)，而*t*為其參數，\(X (t)\)是曲線*C*參數化的表示，而令\(\dot X (t)\)為其法線向量。因此在曲線上的光程長為

\[A[C] = \int_{t=t_0}^{t_1} n (X) \sqrt{ \dot X \cdot \dot X} dt\]。

上述積分和t無關，因此也和*C*的參數表示方式無關。使曲線最短的歐拉-拉格朗日方程有以下的對稱形式

\[\frac{d}{dt} P = \sqrt{ \dot X \cdot \dot X} \nabla n\]，

其中

\[P = \frac{n (X) \dot X}{\sqrt{\dot X \cdot \dot X} }\]。

依P的定義可得下式

\[P \cdot P = n (X)^2\]。

因此上述積分可改為下式

\[A[C] = \int_{t=t_0}^{t_1} P \cdot \dot X \, dt\]。

依照上式，若可以找到一個函數ψ，其梯度为*P*，則以上的積分*A*就可以由在積分端點上ψ的差求得。以上求解曲線使積分量不變的問題就和ψ的level surface有關。為了要找到滿足此條件的函數ψ，需要對控制光線傳動的波動方程式進行進一步的研究。

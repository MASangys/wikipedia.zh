**变格**（）是指[名词](https://zh.wikipedia.org/wiki/名词 "wikilink")、[代词](https://zh.wikipedia.org/wiki/代词 "wikilink")、[冠词](../Page/冠词.md "wikilink")、[形容词在句子中为了实现特定的语法功能而产生的](../Page/形容词.md "wikilink")[词形变化](https://zh.wikipedia.org/wiki/词形变化 "wikilink")。

词通过变格能够反映：

  - [格的变化](../Page/格_\(语法\).md "wikilink")
  - [数的变化](../Page/数_\(语法\).md "wikilink")
  - 性的变化

## 英語

英語的變格系統幾近消失，但在人稱代詞、所有格和第三人稱現在式單數等上仍有體現。

### 人稱代詞

| 人稱（性） | 主格   | 斜格   | 屬格限定詞  | 屬格代詞       | 反身代詞       |
| ----- | ---- | ---- | ------ | ---------- | ---------- |
| 單數    |      |      |        |            |            |
| 第一人稱  | I    | me   | my     | mine       | myself     |
| 第二人稱  | you  | your | yours  | yourself   |            |
| 第三人稱  | 陽性   | he   | him    | his        | himself    |
| 陰性    | she  | her  | hers   | herself    |            |
| 中性    | it   | its  | itself |            |            |
| 複數    |      |      |        |            |            |
| 第一人稱  | we   | us   | our    | ours       | ourselves  |
| 第二人稱  | you  | your | yours  | yourselves |            |
| 第三人稱  | they | them | their  | theirs     | themselves |

### 名詞所有格

|                 | 單數    | 複數    |
| --------------- | ----- | ----- |
| **普通（主格、目的格等）** | boy   | boys  |
| **所有格**         | boy's | boys' |

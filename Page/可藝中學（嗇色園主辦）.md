**嗇色園主辦可藝中學**（）是[香港的一所集](../Page/香港.md "wikilink")[儒](https://zh.wikipedia.org/wiki/儒 "wikilink")、[釋](https://zh.wikipedia.org/wiki/釋 "wikilink")、[道文化於一身的](../Page/道.md "wikilink")[中學](https://zh.wikipedia.org/wiki/中學 "wikilink")，辦學團體為[嗇色園](../Page/嗇色園.md "wikilink")。學校創建於1991年，位於香港[新界](../Page/新界.md "wikilink")[屯門](../Page/屯門.md "wikilink")[愛明里](https://zh.wikipedia.org/wiki/愛明里 "wikilink")8號\[1\]。

## 歷史

於一九九一年創立，原名為嗇色園主辦可藝職業先修中學，校舍於1992年落成，於1997年改名為嗇色園主辦可藝中學，該校和屯門區兩間官立中學設有跨校選修科。

## 辦學宗旨

[Northeast_of_Ho_Ngai_College_(Sponsored_by_Sik_Sik_Yuen).jpg](https://zh.wikipedia.org/wiki/File:Northeast_of_Ho_Ngai_College_\(Sponsored_by_Sik_Sik_Yuen\).jpg "fig:Northeast_of_Ho_Ngai_College_(Sponsored_by_Sik_Sik_Yuen).jpg")
學校秉承嗇色園「普濟勸善」的精神，致力提供優質的教育服務。積極加強與家長的聯系，並透過群體學習，啟導學生發展多元才能，成為才德兼備、勇於承擔、健康積極的好市民。

  - 與時並進 專業教學

教師除接受專業師資培訓外，所有英語教師及普通話教師均已達語文基準試要求。為配合學校特設的「英語校本課程」，聘請了外籍英語教師支援教學。所有老師均掌握資訊科技教學，更約有25%的師教達精鍊程度。

## 開設科目

創新思維‧革新教學

學校於一九九一年創辦至今，一直與時並進。新高中課程的推行，為同學提供一個寬廣、均衡和講求深度的課程。學生除了修讀四個必修科目(中、英、數及通識教育)外，還可按個人興趣和能力，從不同學習領域中選讀二或三個科目（包括應用學習課程），打破了傳統文理的界限，避免學生過早分流。

### 中一至中三級

中國語文、英國語文、數學、綜合人文、中史、科學、家政、電腦認知、設計與科技、音樂、視覺藝術、普通話

### 中四至中六級 

#### 必修科目

中國語文、英國語文、數學及通識教育

#### 選修科目

|      |                                                            |
| ---- | ---------------------------------------------------------- |
| 選修課程 | 生物、物理、化學、經濟、旅遊與款待、企業、會計與財務概論、設計與應用科技、資訊及通訊科技、科技與生活、視覺藝術、體育 |
| 聯校課程 | 中國文學、中國歷史、健康管理與社會關懷                                        |

該校與區內中學（屯門官立中學、南屯門官立中學及廠商會蔡章閣中學）建立一個「跨校合作網絡」聯校課程，共同開放部分選修科目讓四所學校的學生跨校選讀。

## 課外活動

課外活動多姿多采，形式多樣化，是可藝的特色。學會數目達三十個之多，補足了正規課程以外的知識和技能訓練，讓學生發揮個人的天賦潛能，學以致用。在德、智、體、群、美五育得到健全的發展。

## 學生會

該校學生會自2002年創立。學會的宗旨乃培養領導人才，籌辦課外活動，增加學生的歸屬感。

## 校園設施

該校設施完備，除一般學校的基本設施外，更設有遠程教室、講學室、攀石牆、健身室、人體運動實驗室、天文台、射擊場等，務求學生能有多元發展。

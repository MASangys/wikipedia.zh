**可见少数族裔**（英语：Visible Minority）是指可通过肤色判断的少数族群或个人，这词主要在[加拿大](../Page/加拿大.md "wikilink")使用。在非欧裔移民大量进入加拿大之前，加拿大社会主要按语言（英语和法语）和宗教（罗马天主教和新教）区分族裔。1960年代大量非欧洲裔移民进入加拿大后，加拿大创造了这词用来指代非欧裔移民的少数族群。将**可见少数族裔**定义为“除[加拿大原住民](../Page/加拿大原住民.md "wikilink")外的非[高加索人种或肤色不为白色的人](https://zh.wikipedia.org/wiki/高加索人种 "wikilink")”\[1\]。这个词与在美国政治领域使用的[有色人种](https://zh.wikipedia.org/wiki/有色人种 "wikilink")（英语：People in Color）相近，但是美国使用的有色人种包括[印地安人](https://zh.wikipedia.org/wiki/印地安人 "wikilink")。但随着[加拿大人口](../Page/加拿大人口.md "wikilink")构成的变化，在特定的[加拿大](../Page/加拿大.md "wikilink")地区——主要是[卑诗省和](https://zh.wikipedia.org/wiki/卑诗省 "wikilink")[安大略省](../Page/安大略省.md "wikilink")华裔和印度裔聚集地区，包括华裔在内的亚裔人口，显著超过了[高加索人口](https://zh.wikipedia.org/wiki/高加索人种 "wikilink")，导致这个词内涵的已经有了实质性的变化。但是一部适用于整个[加拿大](../Page/加拿大.md "wikilink")的法律，就[加拿大](../Page/加拿大.md "wikilink")总体而言，可见少数族裔的概念依然成立，该法案继续生效。

## 人口比例

根据[2016年人口普查](../Page/2016年加拿大人口普查.md "wikilink")，加拿大全国有約784万人将自己定义为可见少数族裔，占全国人口的22.3％，比2011年增加了158萬，也是首次可见少数族裔超過全國人口五分之一（20%）。1986年到2016年的這三十年間，可見少數族裔平均每年增長0.6%，平均每五年增長3%，按照這個進度估算，2062年左右比例將會超過50%，屆時可見少數族裔將不再是可见「少數」族裔。\[2\]\[3\]

| 序号 | 人口普查年份 | 可见少数族裔比例 |
| -- | ------ | -------- |
| 1  | 1986年  | 4.7%     |
| 2  | 1991年  | 9.4%     |
| 3  | 1996年  | 11.2%    |
| 4  | 2001年  | 13.4%    |
| 5  | 2006年  | 16.2%    |
| 6  | 2011年  | 19.1%    |
| 7  | 2016年  | 22.3％    |
|    |        |          |

这种变化直接反应了[多元文化政策在](https://zh.wikipedia.org/wiki/多元文化政策 "wikilink")[加拿大人口](../Page/加拿大人口.md "wikilink")结构上的巨大影响。

## 列治文为例的演变

根据[加拿大](../Page/加拿大.md "wikilink")[卑诗省](https://zh.wikipedia.org/wiki/卑诗省 "wikilink")[列治文市政府网站资料](https://zh.wikipedia.org/wiki/列治文市 "wikilink")，本来属于传统定义的**可见少数族裔**占[列治文](../Page/列治文.md "wikilink")人口超过70%。而在加拿大其他地区的非**可见少数族裔**——[高加索人口低于](https://zh.wikipedia.org/wiki/高加索人种 "wikilink")30%，因此在[列治文市](https://zh.wikipedia.org/wiki/列治文市 "wikilink")，[高加索人变成了事实上的英语字面意义上的](https://zh.wikipedia.org/wiki/高加索人种 "wikilink")**可见少数族裔**——“一眼就能认出的少数族裔”。 \[4\]

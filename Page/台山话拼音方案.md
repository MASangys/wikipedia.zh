**台山话拼音方案**是部分[台山话辞书](https://zh.wikipedia.org/wiki/台山话 "wikilink")（如《台山方音字典》\[1\]和《台山音字典》\[2\]）里使用的拼音方案，用于标注[台山](../Page/台山市.md "wikilink")（台城）方言语音。

## 方案内容

台山话拼音方案使用了26个[拉丁字母中的](../Page/拉丁字母.md "wikilink")24个，字母r与w不用于方案。

### 声母

台城方言共有21个声母。

|      |      |       |       |      |
| ---- | ---- | ----- | ----- | ---- |
| b  爸 | p  怕 | m  么  | f  科  | v  华 |
| d  早 | t  草 | n  脑  | lh  嫂 | l  老 |
| g  高 | k  卡 | ng  我 | h  何  |      |
| z  助 | c  初 | s  舒  |       |      |
| j  知 | q  持 | x  诗  |       | y  也 |

  - z，c，s与i相拼时，写作j，q，x。如ji知，qi持，xi诗。
  - lh是两个字母表示的声母，普通话、广州话和英语中均无此音。

  - o行韵母中的的oi on od，实际发音为uoi uon uod。

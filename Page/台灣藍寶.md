**Gem silica**，國際名稱為**台灣藍玉髓**，台灣人俗稱**台灣藍寶**，主要生產於台灣東部[海岸山脈](../Page/海岸山脈.md "wikilink")的天藍色玉髓。

## 簡介

數億萬年以來的地殼變動，台灣東部海岸及河邊也因此分佈許多此類受海水或河水沖洗的藍色漂亮石頭，台灣原住民的文化史中，原住民有拿來當裝飾品或器物，[日治時代時逢第二次世界大戰中期](https://zh.wikipedia.org/wiki/台灣日據時期 "wikilink")[日本](../Page/日本.md "wikilink")人需要天然礦產作為戰略物質，之後在花東海岸山脈發現有銅礦並開採，卻不識貨地將藍玉髓的原礦挖出後丟棄之，因此許多原礦的藍玉髓就這樣被丟棄在花東山區上，許多居民也會揀去收藏，不久被日本商人或寶石玩家一次旅途中發現這樣漂亮石頭，便以米與當地原住民交換原礦帶回日本，由於它是台灣發現的藍色寶石，所以日本人將它命名為**太魯閣石**（タロコ石）。台灣藍寶目前在業界是商業名稱。名稱上的台灣是讓大家瞭解它是哪種原料。而不是定位成台灣所屬的原料。

後來陸續在[美國](https://zh.wikipedia.org/wiki/美國 "wikilink")、[秘魯](https://zh.wikipedia.org/wiki/秘魯 "wikilink")、[墨西哥](../Page/墨西哥.md "wikilink")、[智利](../Page/智利.md "wikilink")、[印尼](https://zh.wikipedia.org/wiki/印尼 "wikilink")、[納米比亞等發現此類的寶石](https://zh.wikipedia.org/wiki/納米比亞 "wikilink")，目前全世界最好的品質產地開採地都有好壞。不限於台灣。日治時代之後就有進口原料在台東都蘭一帶買賣。

台灣最主要的開採地區是台東都蘭山，但目前花東地區主要的礦區中，上層礦脈早已枯竭，甚至於遍尋不著，而下層礦脈則因機具及技術跟環境保育等法令限制問題難以克服，可是還是有合法的礦主申請開採。之前2017年在台北的國立臺灣博物館有擺設一百多公斤都蘭開採出來的原礦。本土台灣藍寶市面上還是買的到。

其實台灣藍寶並不是指剛玉家族的藍寶石，而是屬於石英家族的玉髓類，它主要的成份是二氧化矽、矽孔雀石石英，因含有少量銅的成份，而形成了海洋、天空藍、藍中帶綠等色，國際寶石界給予的正確名稱為台灣藍玉髓。

台灣藍寶屬於台灣所有玉髓半寶石中最珍貴一種，因為礦產已經枯竭，2012年一顆一公斤質地好特優的原石價錢約為新台幣100萬元，之後受到中國大陸收藏家瘋狂掃購，而2016年一公斤質地好特優的原石價錢已經漲到新台幣1000萬。2007年以後，出現其他產地的藍玉髓流入華人市場，尤其以印尼、祕魯、美國進口的藍玉髓數量最多，不過質地好的印尼與美國亞利桑那州的藍玉髓價格也不斐。

市面上也會用其他非玉髓類的藍色礦物冒充之，像是異極礦與矽孔雀石，這兩種礦物與玉髓最簡易的分辨是不透光，尤其矽孔雀石，美國加州產量最出名，早期印地安人常拿來用作煉銅，今日在美國則是被拿來作為雕件或配件等藝品，但華人世界，中國廣東商人竟魚目混珠把這種礦物叫做美國藍寶當寶石來賣，實際上它不是玉髓類，它不透光且質地軟、硬度低，真正的美國藍寶還是藍玉髓，質地好漂亮的，價格也是跟台灣藍寶一樣昂貴。另外還有市面上幾種人工製作的假藍寶，市面俗稱的蘇聯藍寶、亞馬遜藍寶、菲律賓藍寶等，特徵長久泡水會變質與變色，若用儀器檢測，便能查出是一種便宜人工製作物，像是前兩者是人工合成物，後者是白玉髓染色。

另外近幾年，土耳其發現新的藍玉髓出土，國際官方稱土耳其藍玉髓，其顏色是淡藍、酒精藍色，而非傳統藍寶的海水藍或天空藍色，目前土耳其這種藍玉髓顏色，全世界僅有土耳其有，礦量也不多，因此非常珍貴，在歐洲珠寶市場非常熱銷，而根據土耳其礦場2018年4月回報的消息是，因為大多用機器開挖且大多礦物藏於淺層帶，因此估計這幾年就會竭礦，市場分析將來價格將會狂飆。

## 台灣產地

主要範圍產在台灣東部海岸山脈從花蓮八里灣山系到台東都蘭的成廣澳山系之間，而其中產地最好的地點為都蘭山，都蘭山數百萬年前是一座火山，在火山爆發後等造山運動，山脈礦物富含矽、鎂、鐵等化學成分的岩漿流入岩層縫隙，經過冰河時期的自然冷卻，加上地層的擠壓產生結晶與原有礦石共生下，累積百萬年而產生。

## 物理性質

台灣藍寶屬[六方晶系](../Page/六方晶系.md "wikilink")，外觀具有藍色之半透明特徵，其物理特性如下表：

| 寶石名稱 | 台灣藍寶（Gemsilica）        |
| ---- | ---------------------- |
| 化學成分 | SiO2，可含多種致色元素          |
| 結晶習性 | 隱晶質集合體，呈塊狀、粒狀或超細纖維狀集合體 |
| 摩氏硬度 | 7　左右                   |
| 相對密度 | 2.58                   |
| 折射率  | 約1.539                 |
| 光澤   | 油脂至玻璃                  |
| 顏色   | 藍、藍綠色，較其他仿品顏色深，色調純正    |
| 透明度  | 微透明至半透明                |
| 放大檢查 | 有些含Fe碧玉                |
| 特殊性質 | 遇酸無反應                  |
|      |                        |
| 產地   | 台灣、美國、智利、秘魯、印尼         |

## 保養

台灣藍寶要注意不要被太陽光照射，溫度也不能超過100℃以上，不然會使藍玉髓的藍色褪色。不過越透明的藍玉髓（[二氧化矽成分越純](https://zh.wikipedia.org/wiki/二氧化矽 "wikilink")，[矽孔雀石含量越低](https://zh.wikipedia.org/wiki/矽孔雀石 "wikilink")）褪色或變色的機率越低。建議平常沒有配戴時，可放置在乾淨的的瓶裝水裡，可保持水頭的淨亮度和透澤度，也可避免隨著氣溫氣候條件下的不必要的變色。

## 玉髓類

玉髓類玉石因質地細膩緻密，玉髓質高，而顯得瑩潤溫婉，透度通常在透明至半透明之間，呈玻璃光澤並略帶油潤感，玉髓或紋彩高雅，不論加工琢製成飾品或雕刻創作，都能有淋漓盡致的發揮。

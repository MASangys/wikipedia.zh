**史瓦帝語**（*siSwati*，）屬於[班圖語支](https://zh.wikipedia.org/wiki/班圖語支 "wikilink")，在非洲南部的使用者約300萬，主要為[史瓦帝尼和](https://zh.wikipedia.org/wiki/史瓦帝尼 "wikilink")[南非的](https://zh.wikipedia.org/wiki/南非 "wikilink")[史瓦帝人](../Page/史瓦濟人.md "wikilink")。史瓦帝語和[英语同為史瓦帝尼的](../Page/英语.md "wikilink")[官方语言](../Page/官方语言.md "wikilink")，也是南非的11種官方語言之一，在史瓦帝尼和南非[普馬蘭加省](https://zh.wikipedia.org/wiki/普馬蘭加省 "wikilink")（特別是原[卡恩格瓦尼地區](https://zh.wikipedia.org/wiki/卡恩格瓦尼 "wikilink")）的學校中教授。

史瓦帝語的母語使用者大多稱本語言為Swati，但英語、[西班牙语等語言中卻稱其為](../Page/西班牙语.md "wikilink")*Swa**z**i*。史瓦帝語與特克拉（Tekela）語支中的其他語言最為接近，但與尊達（Zunda）語支的[祖鲁语](../Page/祖鲁语.md "wikilink")、、和[科萨语等語言也較接近](../Page/科萨语.md "wikilink")。

## 方言

史瓦帝尼境內的史瓦帝語（*eSwatini*）可分為4種方言，對應該國4個行政區：[霍霍](../Page/霍霍.md "wikilink")、[盧邦博](https://zh.wikipedia.org/wiki/盧邦博 "wikilink")、[曼齊尼和](https://zh.wikipedia.org/wiki/曼齊尼 "wikilink")[希塞盧韋尼](https://zh.wikipedia.org/wiki/希塞盧韋尼 "wikilink")。

史瓦帝語至少有2個變種：史瓦帝尼北部、中部和西南部主要說標準史瓦帝語，其餘地區說另一變種。

在史瓦帝尼極南部，尤其是[恩赫蘭加諾和](../Page/恩赫蘭加諾.md "wikilink")[赫拉蒂庫盧](https://zh.wikipedia.org/wiki/赫拉蒂庫盧 "wikilink")，當地的史瓦帝語受[祖鲁语影響極大](../Page/祖鲁语.md "wikilink")，包括說這種方言的人在內，也不被認可它是「標準」的史瓦帝語，只承認是該國境內另一種方言。而南非的[普馬蘭加省和](https://zh.wikipedia.org/wiki/普馬蘭加省 "wikilink")[索韦托也有數量可觀的人說史瓦帝語](../Page/索韦托.md "wikilink")，但在史瓦帝尼境內的該語言使用者看來，這也只是非標準的史瓦帝語。

與史瓦帝尼南部的方言不同，普馬蘭加省的方言受祖魯語影響較小，因此更接近標準史瓦帝語。但該方言與標準史瓦帝語的[語調和](../Page/語調.md "wikilink")[聲調差異較大](../Page/聲調.md "wikilink")，在斯威士蘭人看來，普馬蘭加省的語調（非正式說法稱為「重音」）聽來不和諧。該方言中呈現出受南非境內接近史瓦帝語的其他語言影響的痕迹。

標準史瓦帝語的特徵是緩慢、用力的皇家風格的吐字，被戲稱為有「甜美」感。

## 語法

### 名詞

史瓦帝語的名詞（**）由前綴（**）和詞幹（**）兩部分組成。使用前綴可將名詞歸為名詞組，與[班圖語支的其他語言相比](https://zh.wikipedia.org/wiki/班圖語支 "wikilink")，這些名詞組可用數字連續編號。

下表給出史瓦帝語中名詞組的概況，按單數、複數的組合排列。

| 組     | 單數                 | 複數                |
| ----- | ------------------ | ----------------- |
| 1/2   | um(u)-<sup>1</sup> | ba-, be-          |
| 1a/2a | Ø-                 | bo-               |
| 3/4   | um(u)-<sup>1</sup> | imi-              |
| 5/6   | li-                | ema-              |
| 7/8   | s(i)-<sup>2</sup>  | t(i)-<sup>2</sup> |
| 9/10  | iN-<sup>3</sup>    | tiN-<sup>3</sup>  |
| 11/10 | lu-, lw-           |                   |
| 14    | bu-, b-, tj-       |                   |
| 15    | ku-                |                   |

<sup>1</sup>在單音節詞幹前，**umu-**取代**um-**，例如**<u>umu</u>ntfu**（人，單數）。

<sup>2</sup>在元音開頭的詞幹前，**s-**和**t-**分別取代**si-**和**ti-**，例如**<u>s</u>andla**/**<u>t</u>andla**（手，前者單數，後者複數）。

<sup>3</sup>前綴**iN-**和**tiN-**中的佔位符**N**表示字母**m**、**n**或沒有字母。

## 例文

以[世界人权宣言第一條為例](../Page/世界人权宣言.md "wikilink")，以下是史瓦帝語文本：

*Bonkhe bantfu batalwa bakhululekile balingana ngalokufananako
ngesitfunti nangemalungelo. Baphiwe ingcondvo nekucondza kanye nanembeza
ngakoke bafanele batiphatse nekutsi baphatse nalabanye ngemoya
webuzalwane.*\[3\]\[4\]

宣言中文本：

人人生而自由，在尊嚴和權利上一律平等。他們賦有理性和良心，並應以兄弟關係的精神相對待。\[5\]

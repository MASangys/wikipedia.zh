**斯隆数字化巡天**（，縮寫為）是使用位於[新墨西哥州](../Page/新墨西哥州.md "wikilink")[阿帕契點天文台](../Page/阿帕契點天文台.md "wikilink")的2.5米口径望遠鏡进行的红移巡天项目。该项目开始于2000年，以[阿爾弗雷德·史隆的名字命名](https://zh.wikipedia.org/wiki/艾爾弗雷德·P·斯隆 "wikilink")，计划观测25%的天空，獲取超过一百萬個天體的多色测光资料和[光谱数据](https://zh.wikipedia.org/wiki/光谱 "wikilink")。斯隆数字化巡天的星系样本以[紅移](../Page/紅移.md "wikilink")0.1為中值，对于紅星系的紅移值達到0.4，對于[類星體紅移值则達到](https://zh.wikipedia.org/wiki/類星體 "wikilink")5，並且希望探测到紅移值大于6的類星體。

2006年，斯隆数字化巡天进入了名为的新阶段，进一步探索銀河系的結構和组成，而斯隆超新星巡天计划搜寻[Ⅰa型超新星爆发](https://zh.wikipedia.org/wiki/Ⅰa型超新星爆发 "wikilink")，以测量[宇宙学](../Page/宇宙学.md "wikilink")尺度上的距离。 2008年10月31日，发布了最后一次数据。

斯隆数字化巡天第三期工程已经于2008年7月启动，将持续至2014年。

## 觀測

斯隆数字化巡天使用口径为2.5米的宽视场望遠鏡，测光系统配以分别位于u、g、r、i、z波段的五个滤镜对天体进行拍摄。这些照片经过处理之后生成天体的列表，包含被观测天体的各种参数，比如它们是点状的还是延展的，如果是后者，则该天体有可能是一个星系，以及它们在[CCD上的亮度](https://zh.wikipedia.org/wiki/CCD "wikilink")，这与其在不同波段的[星等](../Page/星等.md "wikilink")有关。

另外，天文学家们还选出一些目標来进行光谱观测。目标的位置用钻孔的方式记录在铝板上，小孔的后面接有光纤，将目标天体的光引入摄谱仪。望远镜每次可以同时拍摄640个天体的光谱，每晚大约需要6到9块铝板对天体进行定位，

## 史隆數位巡天的貢獻

斯隆数字化巡天涵蓋了南銀極周围7,500平方度的星空，记录到近二百萬個天體的数据，包括80多万个星系和10多万个类星体的光譜的数据。這些天體的位置和距離資料为人们研究[宇宙的大尺度结构开辟了道路](https://zh.wikipedia.org/wiki/大尺度结构 "wikilink")。下表是SDSS有光譜的星種數量：

| 種類            | 個數      |
| ------------- | ------- |
| 星系            | 790,220 |
| 類星體 (z \<2.3) | 89,458  |
| 類星體 (z ≥2.3)  | 12,892  |
| M型和更晚型的恆星     | 64,895  |
| 其他恆星          | 126,351 |
| 天空光譜          | 59,843  |
| 待確認天體         | 19,861  |

## 史隆擴大瞭解和探索銀河系計畫

史隆理解和探索銀河的扩充計畫（缩写为）获得了[银河系](../Page/银河系.md "wikilink")内24萬顆[恆星的光譜](https://zh.wikipedia.org/wiki/恆星 "wikilink")，它们的徑向速度在每秒10公里上下，这使天文学家们得以研究銀河系的結構以及各组成部分是如何形成的。

## 史隆超新星巡天

史隆超新星巡天计划快速扫描300平方度的天空，搜寻光度快速变化的天体，寻找[Ia超新星](../Page/Ia超新星.md "wikilink")[爆发](../Page/超新星.md "wikilink")。在2005年，该计划找到了130个爆发的Ia超新星，在2006年找到197顆，总数已经超过了300个，这项计划进行到2007年底。

## 資料的檢索

斯隆数字化巡天将全部图片和光谱数据发布在国际互联网上，并且提供了简单易用的接口。用户只要输入坐标就可以获得斯隆数字化巡天拍摄在该天区拍摄的全部图像。同时还提供了针对从学生到专业天文学家的详尽指南。数据也可以通过[美国宇航局的](https://zh.wikipedia.org/wiki/美国宇航局 "wikilink")[世界风软件获取](https://zh.wikipedia.org/wiki/世界风 "wikilink")。

## 成效

斯隆数字化巡天的数据已經在各种天文出版物中广泛引用，涉及的研究领域包括类星体、星系分布、银河系内恒星的性质、[暗物质](../Page/暗物质.md "wikilink")、[暗能量](../Page/暗能量.md "wikilink")等等。斯隆数字化巡天的网站上提供了这些出版物的[完整列表](https://web.archive.org/web/20080217101608/http://www.sdss.org/publications/index.html)。

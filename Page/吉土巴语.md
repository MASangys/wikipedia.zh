**吉土巴语**（Kituba，或譯**基圖巴語**）是一种广泛用于[中部非洲的](../Page/中部非洲.md "wikilink")[通用语](../Page/通用语.md "wikilink")。这一语言基于[刚果语](../Page/刚果语.md "wikilink")，是与[班图语支关系密切的语言](../Page/班图语支.md "wikilink")
（其中语言无法互相理解）。它是[刚果共和国和](../Page/刚果共和国.md "wikilink")[刚果民主共和国的官方语言](../Page/刚果民主共和国.md "wikilink")。

有时吉土巴语被称作一种[克里奧爾語](https://zh.wikipedia.org/wiki/克里奧爾語 "wikilink")，但是这种说法不完全正确，因为它缺乏典型克里奧爾語演化过程中上层语言和基础语言的差异。

## 名称

吉土巴语在所覆盖人群中有不同的名字。在[刚果共和国它被称作](../Page/刚果共和国.md "wikilink")*Munukutuba*或*Kituba*。前者是一个语法表述错误，直译就是"I
to speak"。后一个的含义是"speech"。*Kituba*是刚果共和国法律条文使用的叫法。

在[刚果民主共和国他被称作](../Page/刚果民主共和国.md "wikilink")*Kikongo ya leta*
（也就是“Kikongo of the state
administration”），不过经常使用的是较短的说法*Kikongo*,
特别是[刚果族人生活区域以外](https://zh.wikipedia.org/wiki/刚果族 "wikilink")。刚果民主共和国的法律将Kikongo作为一种法定语言。实际上这指的是Kikongo
ya leta （也就是吉土巴语），因为这些法律的一个译文本身就是用吉土巴语写的，而且也不存在Kikongo语言译本。

还有其它曾经使用过的名字如*Kibulamatadi*，*Kikwango*，*Ikeleve*和*Kizabave*，但是很长时间都不再使用了。语言学术圈的叫法是*Kikongo-Kituba*。

## 地理分布

[LanguageMap-Kikongo-Kituba.png](https://zh.wikipedia.org/wiki/File:LanguageMap-Kikongo-Kituba.png "fig:LanguageMap-Kikongo-Kituba.png")
说吉土巴语的人大部分住在刚果民主共和国。在下列省份它是主要通用语：[中刚果省](https://zh.wikipedia.org/wiki/中刚果省 "wikilink"),
[宽果省和](https://zh.wikipedia.org/wiki/宽果省 "wikilink")[奎卢省](https://zh.wikipedia.org/wiki/奎卢省 "wikilink")，但是在[金沙萨](https://zh.wikipedia.org/wiki/金沙萨 "wikilink")，[马伊恩东贝省和](https://zh.wikipedia.org/wiki/马伊恩东贝省 "wikilink")[开赛省只有不多的人说这一语言](https://zh.wikipedia.org/wiki/开赛省 "wikilink")。

吉土巴语是刚果共和国使用最广的语言。在这个国家南半部居民使用这一语言[奎卢](https://zh.wikipedia.org/wiki/奎卢 "wikilink")
（特别是在[黑角](../Page/黑角.md "wikilink")）,
[尼阿利](https://zh.wikipedia.org/wiki/尼阿利 "wikilink"),
[布恩扎](https://zh.wikipedia.org/wiki/布恩扎 "wikilink"),
[勒库穆](https://zh.wikipedia.org/wiki/勒库穆 "wikilink"), [普尔
（刚果共和国）和首都](https://zh.wikipedia.org/wiki/普尔_（刚果共和国） "wikilink")[布拉柴维尔](../Page/布拉柴维尔.md "wikilink")。在北部[林加拉语使用更加广泛](../Page/林加拉语.md "wikilink")。

吉土巴语在安哥拉的情况不祥。可能有些刚果族人可以听懂，特别是那些从两个刚果逃离的难民或者移民。

## 官方状态

吉土巴语是刚果-布拉柴维尔和刚果-金沙萨的官方语言。实际上“官方语言“的意思就是这个语言用于行政管理和教育。

作为一种官方语言，它也用于公共和大众传媒。刚果-金沙萨和刚果-布拉柴维尔的国家公共电台和电视台使用吉士巴作为主要语言。

## 历史

吉土巴语最早起源于[刚果河下游](../Page/刚果河.md "wikilink"),
这一区域有[刚果人居住](https://zh.wikipedia.org/wiki/刚果人 "wikilink")。

关于吉土巴语的起源有几种不同的理论。一种理论认为，[刚果王国时期这种语言就发展起来](../Page/刚果王国.md "wikilink")，作为一种简单的跨方言贸易语言，随后殖民统治着将它作为行政管理用语。另一种理论认为一种叫做Kifyoti的简化贸易语言在葡萄牙海岸贸易期间发展起来，之后随着基督教传教士向北传播到达宽果河和开赛河流域，在这里继续变化
（由此产生名字Kikwango）。然而另外的理论强调19世纪末期修建[马塔迪](../Page/马塔迪.md "wikilink")-[金沙萨铁路使用了大量来自](https://zh.wikipedia.org/wiki/金沙萨 "wikilink")[西非](../Page/西非.md "wikilink")，下刚果和[班顿杜省区域附近的强制劳动力](../Page/班顿杜省.md "wikilink")。这些工人有迥异的语言背景，从而催生了一种语法简化的语言。

不管它的起源如何，吉土巴语已经具有了在1885-1960年间殖民时期建造的大型城市中的稳固地位。吉土巴语是大型的刚果人居住城市的主要语言如[莫安达](https://zh.wikipedia.org/wiki/莫安达 "wikilink"),
[Boma](https://zh.wikipedia.org/wiki/Boma "wikilink"),
[马塔迪](../Page/马塔迪.md "wikilink"),
[黑角](../Page/黑角.md "wikilink"),
[卢博莫](https://zh.wikipedia.org/wiki/卢博莫 "wikilink"),
[恩卡伊](https://zh.wikipedia.org/wiki/恩卡伊 "wikilink")，[布拉柴维尔和很多](../Page/布拉柴维尔.md "wikilink")[班顿杜](https://zh.wikipedia.org/wiki/班顿杜 "wikilink"),
[基奎特](../Page/基奎特.md "wikilink"),
和[伊莱博的大型非刚果人城市](https://zh.wikipedia.org/wiki/伊莱博 "wikilink").

## 发音

### 元音

吉土巴语有5个元音：/a/，/e/，/i/，/o/和/u/。其发音十分类似西班牙语和意大利语的元音。元音不发短音，不分重音。元音发音如下：

  - /a/ 发音类似汉语拼音 "a"
  - /e/ 发音类似英语中bad中的"e"
  - /i/ 发音类似拼音"ai"
  - /o/ 发音类似拼音"o" 或者“ao”
  - /u/ 发音类似拼音"u"

## 语法

### 代词

吉土巴语语有主格和宾格代词。如果宾格代词被强调，则用主格形式代替宾格。

|        |        |            |
| ------ | :----: | :--------: |
| **人称** | **单数** |   **复数**   |
| 主格     |   宾格   |     主格     |
| **第一** |   mu   | munu, mono |
| **第二** |  nge   |    nge     |
| **第三** |   yá   |   yandi    |

### 名词

吉土巴语保留了很多某些修改过的刚果部族使用的名词类。第9和11类单数形式实际上被合并到零前缀类，复数形式则使用通用的复数前缀*ba-*。

|        |        |
| :----: | :----: |
| **单数** | **复数** |
|   原型   |   前缀   |
|   0    |   \-   |
|   1    |  mu-   |
|   3    |  mu-   |
|   5    |  di-   |
|   7    |  ki-   |
|   9    | n-/m-  |
|   11   |  lu-   |
|   12   |  ka-   |
|   14   |   bu   |
|   15   |  ku-   |

### 动词

吉土巴语动词具有时态和形态变化。大多数动词都有长格式和短格式。长格式一般用语比较正式的场合和书面通信，短格式反之一般用语口头交流。

不规则动词*kuvanda*或*kuvuanda*（被动态）的变化形式列在下面表格中。它是吉土巴语中唯一的不规则动词。

| 时态            | 长格式                 | 短格式            | 举例                             | 翻译         |
| ------------- | ------------------- | -------------- | ------------------------------ | ---------- |
| **现在时和立即将来时** | kele                | ke             | *Yau kele nkosi.*              | 这是只狮子。     |
| **将来时**       | kele/ata kuv（u）anda | ke/ta v（u）anda | *Mu ta vuanda tata.*           | 我将要成为当爹的了。 |
| **现在进行时**     | kele kuv（u）andaka   | ke v（u）andaka  | *Nge ke vuandaka zoba.*        | 你被愚弄。      |
| **将来进行时**     | ata kuv（u）andaka    | ta v（u）andaka  | *Beno ta vuandaka ya kukuela.* | 你将来会嫁人。    |
| **过去时**       | v（u）andaka          |                | *Yandi vuanda kuna.*           | 他曾经在那。     |
| **过去进行时**     | v（u）andaka          |                | *Beto vuandaka banduku.*       | 我们曾经是朋友。   |
| **过去完成时**     | mene kuv（u）anda     | me v（u）anda    | *Yandi me vuanda na Matadi.*   | 他曾在Matadi. |
| **过去完成进行时**   | mene kuv（u）andaka   | me v（u）andaka  | *Yandi me vuandaka mulongi.*   | 她已经是个教师了。  |

所有其他动词的变形都借助助动词。下面是动词'kusala''（做）的变化形式。

| 时态            | 详细格式            | 简易格式      | 举例                     | 翻译         |
| ------------- | --------------- | --------- | ---------------------- | ---------- |
| **现在时和立即将来时** | kele kusala     | ke sala   | *Yandi ke sala.*       | 他工作/他将会工作。 |
| **现在进行时**     | kele kusalaka   | ke salaka | *Yandi ke salaka.*     | 他在工作。      |
| **过去时**       | salaka          | salaka    | *Yandi salaka.*        | 他工作了。      |
| **立即过去时**     | mene sala       | me sala   | *Yandi me sala.*       | 他工作了。      |
| '''立即过去进行时    | mene salaka     | me salaka | *Yandi me salaka.*     | 他曾经在做工。    |
| **过去进行时**     | vuandaka kusala | va sala   | Yandi vuandaka kusala. | *他曾经工作。*   |
| **叙事态**       | sala            | sala      |                        |            |
| **将来时**       | ata sala        | ta sala   | *Yandi ta sala.*       | 他将会工作。     |
| **将来进行时**     | ata salaka      | ta salaka | *Yandi ta salaka.*     | 他将会做工。     |

## 词汇

大多数吉土巴语的词汇来自[刚果语](../Page/刚果语.md "wikilink")。其他班图语支的语言也对它有较大影响，包括Kiyaka,
Kimbala, Kisongo, Kiyansi,
[林加拉语和](../Page/林加拉语.md "wikilink")[斯瓦希里语](../Page/斯瓦希里语.md "wikilink")。另外，很多词汇也来源于[法语](../Page/法语.md "wikilink"),
[葡萄牙语和](https://zh.wikipedia.org/wiki/葡萄牙语 "wikilink")[英语](../Page/英语.md "wikilink")。包括：

  - sandúku（[斯瓦希里语](../Page/斯瓦希里语.md "wikilink"). sanduku）
  - matáta（斯瓦希里语. matata）
  - letá（[法语](../Page/法语.md "wikilink") *l'état*，状况）
  - kamiyó（法语 *camion*，军用卡车）
  - sodá/solodá（法语 *soldat*，士兵）
  - masínu（法语 *machine*，机器）
  - mísa（[葡语](https://zh.wikipedia.org/wiki/葡萄牙語 "wikilink") *missa*，弥撒）
  - kilápi（葡语 *lápis*，石头）
  - katekisimu（[英语](../Page/英语.md "wikilink") *catechism*，基督教教理问答）
  - bóyi（英语 *houseboy*，男仆）
  - sapatu（葡语. *sapato*，鞋子）
  - mesa（葡语/[西语](https://zh.wikipedia.org/wiki/西班牙語 "wikilink")
    *mesa*，桌子）
  - dikopa（西语 *copa*，杯子）
  - simisi（法语 *chemise*，衬衫;披肩,睡衣）

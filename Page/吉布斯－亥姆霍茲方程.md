**吉布斯─亥姆霍茲方程**，是對計算系統的[吉布斯自由能變化的有用](../Page/吉布斯.md "wikilink")[熱力學公式](../Page/熱力學.md "wikilink")。為一溫度函數。此方程式以[约西亚·吉布斯與](../Page/约西亚·吉布斯.md "wikilink")[赫尔曼·冯·亥姆霍兹來命名](../Page/赫尔曼·冯·亥姆霍兹.md "wikilink")：

\[\left( \frac{\partial ( \frac{G} {T} ) } {\partial T} \right)_{p\,} = - \frac {H} {T^2}\]

其中:

\[H\,\] [焓](../Page/焓.md "wikilink")

\[T\,\] [絕對溫度](../Page/絕對溫度.md "wikilink")

\[G\,\] [吉布斯能](../Page/吉布斯能.md "wikilink")

在定壓\(P\,\)下，達成平衡的關鍵為：在微小變化中 G/T 與 T 的比值.

對於化學反應，方程式又可寫成:

\[\left( \frac{\partial ( \frac{\Delta G} {T} ) } {\partial T} \right)_{p\,} = - \frac {\Delta H} {T^2}\]

其中 \(\Delta G\,\) 為吉布斯能變化量及\(\Delta H\,\) 為焓的變化量(為一溫度獨立變數)。

也可重新寫成:

\[\frac{\Delta G,T_2}{T_2} - \frac{\Delta G^\circ,T_1}{T_1} = \Delta H^\circ(P)*(\frac{1}{T_2} - \frac{1}{T_1})\]

上述方程能夠藉由於25°C 及 1 bar的情況下所定的**標準吉布斯能**，快速的算出在任何溫度下，化學反應所造成的吉布斯能變化

透過:

\[\ \Delta G^\circ = -RT \ln K\]

能讓方程式連結吉布斯能與[平衡常數](../Page/平衡常數.md "wikilink"),或[凡何夫方程式](../Page/凡何夫方程式.md "wikilink")。

## 證明

在封閉系統下，吉布斯能為:

\[dG =  - SdT + VdP \,\]

在定壓\(P\,\) (dP = 0)下可簡化成:

\[dG_{p\,} =  - SdT \,\]

或

\[\left(\frac{\partial G}{\partial T}\right)_{p\,} =  - S \,\]

在[除法定則的幫忙下](../Page/除法定則.md "wikilink")， G/T 的比值可視 T 而定:

\[\left( \frac{\partial ( \frac{G} {T} ) } {\partial T} \right)_{p\,}= \frac{\left( \frac{\partial G}{\partial T} \right)_{p\,}}{T} - \frac{G}{T^2} = \frac{T\left ( \frac{\partial G}{\partial T} \right)_{p\,}- G}{T^2} = \frac{-ST - G}{T^2} = \frac{-H}{T^2}\]

經歷一段時間後，發現可寫成:

\[\left( \frac{\partial ( \frac{G} {T} ) } {\partial\left(\frac{1}{T}\right)}\right)_{p\,}= H\]

**吉茲字母**（；；**），又名**埃塞俄比亚字母**、**斐德子母音合成字母**（**Fidel alphabet**），是一種[元音附標文字書寫系統](https://zh.wikipedia.org/wiki/元音附標文字 "wikilink")，本來用來書寫一種名為[吉茲語](../Page/吉茲語.md "wikilink")的[閃米特語族](https://zh.wikipedia.org/wiki/閃米特語族 "wikilink")。過往由於對[埃塞俄比亞的語言狀況缺乏認知](https://zh.wikipedia.org/wiki/埃塞俄比亞 "wikilink")，所以多數人都以為當地人只說一種語言，統稱為[埃塞俄比亞語](https://zh.wikipedia.org/wiki/埃塞俄比亞語 "wikilink")。但其實，在埃塞俄比亞有9種不同的語言在通行，當中除了[吉茲語](../Page/吉茲語.md "wikilink")、[阿姆哈拉語及](https://zh.wikipedia.org/wiki/阿姆哈拉語 "wikilink")[阿拉伯語以外](https://zh.wikipedia.org/wiki/阿拉伯語 "wikilink")，還包括[提格利尼亞語](../Page/提格利尼亞語.md "wikilink")、[提格雷語](https://zh.wikipedia.org/wiki/提格雷語 "wikilink")、[古拉格語](https://zh.wikipedia.org/wiki/古拉格語 "wikilink")、[哈勒爾語等](https://zh.wikipedia.org/wiki/哈勒爾語 "wikilink")[閃米特語族](https://zh.wikipedia.org/wiki/閃米特語族 "wikilink")，這些[閃米特語族都是採用吉茲字母來書寫的](https://zh.wikipedia.org/wiki/閃米特語族 "wikilink")。其中最廣泛使用的，是[埃塞俄比亞的](https://zh.wikipedia.org/wiki/埃塞俄比亞 "wikilink")[阿姆哈拉語及在埃塞俄比亞與](https://zh.wikipedia.org/wiki/阿姆哈拉語 "wikilink")[厄立特里亞都有使用的](https://zh.wikipedia.org/wiki/厄立特里亞 "wikilink")[提格里尼亞語](https://zh.wikipedia.org/wiki/提格里尼亞語 "wikilink")。此外，亦有其他同樣在[非洲之角](../Page/非洲之角.md "wikilink")通行的其他語言，例如：[奧羅莫語](../Page/奧羅莫語.md "wikilink")都曾經使用過吉茲字母來拼寫，但現時已改用[拉丁字母](../Page/拉丁字母.md "wikilink")了。

現時大多數通行的電腦作業系統都沒有連同支援吉茲字母的字型付運。要正確顯示吉茲字母，需要安裝支援吉茲字母的字型。

## 起源

對於吉茲字母源於[南阿拉伯字母](../Page/南阿拉伯字母.md "wikilink")（以下簡寫作「南阿字母」）的說法普遍都沒有異議，只是怎樣從南阿字母演變成為吉茲字母，中間的過程則有爭議。

吉茲字母的基本輔音字母有26個。這26個基本字母其實可以歸納為24個能夠與南阿字母相對應的字母。連同與其對應的[腓尼基字母](../Page/腓尼基字母.md "wikilink")，這24個字母詳列如下：

## 歷史

早期的吉兹字母是由26个辅音组成的纯[辅音音素文字](../Page/辅音音素文字.md "wikilink")。后来在基督教文字的影响下加入了元音，分别是u、i、a、e、ə、o，以一种糅合的方式加到辅音字母中去，同时元音ə被默认为辅音自带的。这样吉兹字母就成为了一种[元音附标文字](../Page/元音附标文字.md "wikilink")（见下表）。

## 數字

吉茲字母採用一套與[希伯來語數字系統](https://zh.wikipedia.org/wiki/希伯來語數字 "wikilink")、[阿拉伯語輔音數字系統或](https://zh.wikipedia.org/wiki/阿拉伯語輔音數字 "wikilink")[希腊数字](../Page/希腊数字.md "wikilink")系統近似的記數方法。不過與這些系統不同的是：吉茲字母的數字是一套獨立的字母，而不是採用原有的字母變體：

|          | 1 | 2 | 3 | 4 | 5 | 6 | 7 | 8 | 9 |
| -------- | - | - | - | - | - | - | - | - | - |
| × 1      | ፩ | ፪ | ፫ | ፬ | ፭ | ፮ | ፯ | ፰ | ፱ |
| × 10     | ፲ | ፳ | ፴ | ፵ | ፶ | ፷ | ፸ | ፹ | ፺ |
| × 100    | ፻ |   |   |   |   |   |   |   |   |
| × 10.000 | ፼ |   |   |   |   |   |   |   |   |

## Unicode

吉茲字母（或「埃塞俄比亞字母」）從[Unicode的](https://zh.wikipedia.org/wiki/Unicode "wikilink")3.0版本開始編入，編碼區段是U+1200至U+137F（即十進位的4608～4991），包括[吉茲語](../Page/吉茲語.md "wikilink")、[阿姆哈拉语及](https://zh.wikipedia.org/wiki/阿姆哈拉语 "wikilink")[提格利尼亞語](../Page/提格利尼亞語.md "wikilink")所需的基本字母、標點及數碼。為應付吉茲字母書寫其他語言之用，[Unicode在](https://zh.wikipedia.org/wiki/Unicode "wikilink")4.1版本對字母集合擴充，增加了位於U+1380到U+139F（即十進碼4992～5023）的補充區段。

### 組字的考慮

如同[佉盧文字型的實現問題一樣](https://zh.wikipedia.org/wiki/佉盧文 "wikilink")，由於吉茲字母的部份字型有不規則的變化，所以在設計字型時，不能單單靠`GPOS`命令來放置標號，而需要用`GSUB`來作字型的替換。

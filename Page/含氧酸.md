**含氧酸**是由含氧酸根与其化合价相同的氢原子化合成的化合物。

## 命名方法

在含氧酸中，由同一元素生成的各种不同的酸，可按其中所含氧原子数多少，顺序冠以**高**、（**正**）、**亚**或**次**。其中最常见的酸定名为（**正**）某酸，比其氧原子数多的冠以**高**字，比其少的冠以**亚**字，更少的冠以**次**字。

如：

|                  |                                  |
| ---------------- | -------------------------------- |
| HClO<sub>4</sub> | [高氯酸](../Page/高氯酸.md "wikilink") |
| HClO<sub>3</sub> | [氯酸](../Page/氯酸.md "wikilink")   |
| HClO<sub>2</sub> | [亚氯酸](../Page/亚氯酸.md "wikilink") |
| HClO             | [次氯酸](../Page/次氯酸.md "wikilink") |

### 原酸

含氧酸根的化合价与其中氧原子数相同的酸，稱为原酸。命名为**原**某酸。

例如：

|                              |                                                     |
| ---------------------------- | --------------------------------------------------- |
| H<sub>6</sub>SO<sub>6</sub>  | [原硫酸](https://zh.wikipedia.org/wiki/原硫酸 "wikilink") |
| H<sub>5</sub>PO<sub>5</sub>  | [原磷酸](https://zh.wikipedia.org/wiki/原磷酸 "wikilink") |
| H<sub>4</sub>CO<sub>4</sub>  | [原碳酸](../Page/原碳酸.md "wikilink")                    |
| H<sub>6</sub>TeO<sub>6</sub> | [原碲酸](https://zh.wikipedia.org/wiki/原碲酸 "wikilink") |
| H<sub>4</sub>SiO<sub>4</sub> | [原硅酸](https://zh.wikipedia.org/wiki/原硅酸 "wikilink") |

### 偏酸

正酸缩去一水分子而成的酸，称为偏酸。

例如：

| 正酸                             | 偏酸                          |
| ------------------------------ | --------------------------- |
| 名称                             | 化学式                         |
| [磷酸](../Page/磷酸.md "wikilink") | H<sub>3</sub>PO<sub>4</sub> |
| [硼酸](../Page/硼酸.md "wikilink") | H<sub>3</sub>BO<sub>3</sub> |

### 焦酸（重酸）

由两个简单含氧酸缩去一分子水。通常命名为焦酸，亦称重酸。

如：

  - 2H<sub>2</sub>SO<sub>4</sub> − H<sub>2</sub>O → H<sub>2</sub>S<sub>2</sub>O<sub>7</sub>（焦硫酸）
  - 2H<sub>2</sub>CrO<sub>4</sub> − H<sub>2</sub>O → H<sub>2</sub>Cr<sub>2</sub>O<sub>7</sub>（重铬酸）

### 过酸

[过酸是含有](https://zh.wikipedia.org/wiki/过酸 "wikilink")-O-O-[过氧链的酸](https://zh.wikipedia.org/wiki/过氧化物 "wikilink")，例如[过氧乙酸](../Page/过氧乙酸.md "wikilink")、[间氯过氧苯甲酸](../Page/间氯过氧苯甲酸.md "wikilink")等。

**吳語懶音**是[吴語語音在內部規律的控制下發生流變的一種現象](https://zh.wikipedia.org/wiki/吴語 "wikilink")。這種流變常常造成同音字增加而導致辨義不清，因此常被大眾稱為[懶音](https://zh.wikipedia.org/wiki/懶音 "wikilink")。厂[吳語各地方言不同程度上都存在有](https://zh.wikipedia.org/wiki/吳語 "wikilink")“懶音”現象，[上海話懶音現象較為普遍](https://zh.wikipedia.org/wiki/上海話 "wikilink")。

## 聲母

### 聲母簡化

如[上海話](https://zh.wikipedia.org/wiki/上海話 "wikilink")，原普遍分尖團，現在幾乎消失，[蘇州話中亦開始出現此現象](https://zh.wikipedia.org/wiki/蘇州話 "wikilink")；

### 聲母脫落

疑母脫落现象十分普遍，上海話自稱代詞“我”ngu，許多青年發音時疑母脫落發音成wu。其它漢字如“環”等；

### 聲母相混

另外，清濁相混現象也時有發生，如“期”、“帆”等；

### 聲母誤讀

如[上海有人會把](https://zh.wikipedia.org/wiki/上海 "wikilink")“如”讀成“lu”，廣韻如屬日聲魚韻，而對應[普通話為聲母](https://zh.wikipedia.org/wiki/普通話 "wikilink")“r”
吳語日母亦按中古聲母作gn\[ɲ\]及所謂之\[文讀\]“z”，所以“如”聲母應為gn\[ɲ\]。

## 韻母

### 打党不分

主要就是以“打”(an) 为代表的这类字，和以“党”(aon)
为代表的这类字\*，它们的韵母是不一样的，但很多人已经不再区分。具体发音区别基本上是这样的：发an的时候，嘴巴张得比较大，发音比较靠前，有些地方，比如[无锡甚至接近普通话an的发音](https://zh.wikipedia.org/wiki/无锡 "wikilink")；而aon发音比较靠后，特别是[宁波](https://zh.wikipedia.org/wiki/宁波 "wikilink")，接近[英国英语long里面的ong的发音](https://zh.wikipedia.org/wiki/英国英语 "wikilink")，而其他地区多接近[美国英语的这个发音](../Page/美国英语.md "wikilink")。
最近打党这两类字的区别现在许多地方已经模糊了。比较常用的字里面的区别是
an/aon：张/赃、蚌/帮、场/尝、长/上常、厂/唱。基本上还是aon的字比较多。

### 袜麦不分

跟an和aon的区分类似，传统吴语（以苏州话为代表）区分“麦”、“袜”这两组入声。
吴拼以aeh和ah分别表示这两组音，对于这两组读音已经合并的地区，可以就用ah。以[苏州话为例](../Page/苏州话.md "wikilink")，“袜”读maeh，而“麦”读mah。ah里头的a读音近似单韵母a里头的a，而aeh里面的ae读音近似汉语拼音-an-里面的a。

### 晏安不分

适用于[苏南](../Page/苏南.md "wikilink")、[上海和](https://zh.wikipedia.org/wiki/上海 "wikilink")[浙江部分地区](https://zh.wikipedia.org/wiki/浙江 "wikilink")。易错字正确发音如下：

|        |         |          |
| ------ | ------- | -------- |
| **漢字** | **吳語音** | **例字**   |
| 晏      | ae      | 攀盼删      |
| 安      | oe      | 参惨堪砍刊侃陕栓 |

### 司书不分

在一些吴语中，比如中派和新派[上海话](../Page/上海话.md "wikilink")，读音上“斯”=“书”，发生了“书”并入“斯”的过程。在普通话里，这两组字读音是有区别的，在很多[吴语中也是](../Page/吴语.md "wikilink")。有些吴语则是把“书”并入了“虚”中，即“书”=“虚”，如[江阴话](https://zh.wikipedia.org/wiki/江阴话 "wikilink")。这里要讲的所谓“斯书”之分就是指以“斯”、“书”为代表的这两类字的区别。是以[苏州话和老派上海话为蓝本的](../Page/苏州话.md "wikilink")，区分这两组字的方法，即立足于这两组字韵母的不同。
“斯”这类字的发音，跟普通话一样，韵母是吴语拼音的y，没有韵母一样。
“书”的韵母用吴语拼音表示就是yu，是y的圆唇音。先发“斯”，然后慢慢把嘴唇收拢，就是“书”的音。

## 字音

受[普通話的嚴重影響](https://zh.wikipedia.org/wiki/普通話 "wikilink")，中青年特別是青少年不能掌握一些與普通話發音差別較大的吳語字音。如“杏”、“岳”、“鶴”、“獵”、“夕”等。例如：

  - **“夕”**：应读“席”，不读“希”或“西”；[廣韻邪聲清韻入調](../Page/廣韻.md "wikilink")，祥易切，音zek\[zeʔ\]，[陽入調](https://zh.wikipedia.org/wiki/陽入 "wikilink")。上海音zi\[ziɪʔ\]。
  - **“硕”**：应读“石”，不读“朔”soh； 廣韻常聲清韻入語，常隻切，音zek\[zeʔ\]，陽入語。上海音zaok\[zɑʔ\]。
  - **“鹤”**：上海话里读ngoh，不读“学”ghoh；“鹤”从[廣韻匣聲唐韻入聲](../Page/廣韻.md "wikilink")，下各切，音ghak\[ɦɑʔ\]。但[吳語中此字變異](https://zh.wikipedia.org/wiki/吳語 "wikilink")，發類疑母。“学”从廣韻匣聲江韻[入聲](../Page/入聲.md "wikilink")，胡覺切，音ghok\[ɦoʔ\]。
  - **“环寰鬟鹮溃”**等：声母是g，不是w。同“鹤”一样，這些字音在吳語中存有變異現象，由gh 分化成g

有學者批評歸究于吳地漸與廣韻時段脫落，各地政府促倡導方言保護。而部分研究顯示今吳閩語保留較完整中古聲母。閩保留知徹澄母與其他方言不同，吳地保留全清[濁音讀及返無](https://zh.wikipedia.org/wiki/濁音 "wikilink")[捲舌音](../Page/捲舌音.md "wikilink")。

**和平方**是數學公式的一種，它屬於[乘法公式](../Page/乘法公式.md "wikilink")及[因式分解](../Page/因式分解.md "wikilink")，現時經常使用。和平方是指兩個數目的總和的[平方](../Page/平方.md "wikilink")，公式是：

\[(a+b)^2=a^2+2ab+b^2\,\!\]

## 驗證

### 基本驗證

和平方可直接利用[因式分解](../Page/因式分解.md "wikilink")驗證。公式如下：

\[(a+b)^2\]

\[=(a+b)(a+b)\]

\[=a(a+b)+b(a+b)\]

\[=a^2+ab+ab+b^2\]

\[=a^2+2ab+b^2\]

### 簡單驗證

和平方亦可以表格形式驗證：

|   x)   |  \(a\)  |  \(+b\)  |
| :----: | :-----: | :------: |
| \(a\)  | \(a^2\) | \(+ab\)  |
| \(+b\) | \(+ab\) | \(+b^2\) |
|        |         |          |

\((a+b)^2=a^2+2ab+b^2\)

### 幾何驗證

[pfs.svg](https://zh.wikipedia.org/wiki/File:pfs.svg "fig:pfs.svg") 和平方可透過圖表來驗證。右圖中，是一個\((a+b)^2\)的正方形。可在右圖中分割為四部分：

  - \(a^2\)
  - \(ab\)
  - \(ab\)
  - \(b^2\)

將四部分加在一起：

\[a^2+ab+ab+b^2\]

\[=a^2+2ab+b^2\]

## 三數和平方

三數和平方，指三個（或可多個）數目的總和的[平方](../Page/平方.md "wikilink")，得來的公式是：

\[(a+b+c)^2=a^2+b^2+c^2+2ab+2bc+2ac\,\!\]

### 驗證

驗證方法與兩數和平方差不多，可透過多项式乘法驗證：

\[(a+b+c)^2\]

\[=(a+b+c)(a+b+c)\]

\[=a(a+b+c)+b(a+b+c)+c(a+b+c)\]

\[=a^2+ab+ac+ab+b^2+bc+ac+bc+c^2\]

\[=a^2+b^2+c^2+2ab+2bc+2ac\]

三數和平方亦可以表格形式驗證：

|   x)   |  \(a\)  |  \(+b\)  |  \(+c\)  |
| :----: | :-----: | :------: | :------: |
| \(a\)  | \(a^2\) | \(+ab\)  | \(+ac\)  |
| \(+b\) | \(+ab\) | \(+b^2\) | \(+bc\)  |
| \(+c\) | \(+ac\) | \(+bc\)  | \(+c^2\) |

\((a+b+c)^2=a^2+b^2+c^2+2ab+2bc+2ac\)

[PFC_Math.png](https://zh.wikipedia.org/wiki/File:PFC_Math.png "fig:PFC_Math.png") 透過幾何驗證也同樣，根據右圖將所有部分加在一起：

\[a^2+b^2+c^2+ab+ab+bc+bc+ac+ac\]

\[=a^2+b^2+c^2+2ab+2bc+2ac\]

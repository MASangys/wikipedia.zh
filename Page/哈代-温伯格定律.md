**哈代-温伯格定律**，也称**遗传平衡定律**或**哈代-温伯格平衡定律**，分别在1908年和1909年由[英国](../Page/英国.md "wikilink")[数学家](../Page/数学家.md "wikilink")[G·H·哈代](../Page/G·H·哈代.md "wikilink")（Godfrey
Harold
Hardy）和[德国](../Page/德国.md "wikilink")[医生](../Page/医生.md "wikilink")（Wilhelm
Weinberg）独立证明。在[群体遗传学中](../Page/群体遗传学.md "wikilink")，哈代-温伯格定律主要用于描述[群体中](../Page/种群.md "wikilink")[等位基因频率以及](../Page/等位基因频率.md "wikilink")[基因型频率之间的关系](../Page/基因型频率.md "wikilink")。主要内容为：

实际上，总会存在一个或多个干扰因素。因此，哈代-温伯格定律在自然界中是不太可能的。基因的平衡是一种理想状态，并用于测量遗传改变的基准。

最简单的例子是位于单一位点的两个[等位基因](../Page/等位基因.md "wikilink")：显性等位基因记为**A**而隐性等位基因记为**a**，它们的频率分别记为*p*和*q*。频率(**A**) = *p*；频率(**a**) = *q*；*p* + *q* = 1。如果群体处于平衡状态，则我们可以得到

  -
    群体中[纯合子](../Page/纯合子.md "wikilink")**AA**的频率(**AA**) = *p*<sup>2</sup>
    群体中[纯合子](../Page/纯合子.md "wikilink")**aa**的频率(**aa**) = *q*<sup>2</sup>
    群体中[杂合子](../Page/杂合子.md "wikilink")**Aa**的频率(**Aa**) = 2*pq*

## 推导

对于[二倍体物种](../Page/二倍体.md "wikilink")，考虑[种群中两个独立的等位基因A和a](../Page/种群.md "wikilink")，它们的频率分别是*p*和*q*。使用[旁氏表推导出形成新基因型的不同方式](../Page/旁氏表.md "wikilink")，其中每一格的值为行与列概率的乘积。

|         | 雌性        |
| ------- | --------- |
| A (*p*) | a (*q*)   |
| 雄性      | A (*p*)   |
| a (*q*) | Aa (*pq*) |

表1：哈代-温伯格平衡的旁氏表

最终得到后代的三个可能基因型频率：

  - \(f(\mathbf{AA}) =  p^2\,\)
  - \(f(\mathbf{Aa}) = 2pq\,\)
  - \(f(\mathbf{aa}) = q^2\,\)

这些频率称为哈代-温伯格平衡。无限大小的群体经过一个世代的随机交配，基因型频率就能达到平衡。在没有其它因素的影响下，这种平衡状态将一直保持。

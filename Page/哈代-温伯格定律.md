实际上，总会存在一个或多个干扰因素。因此，哈代-温伯格定律在自然界中是不太可能的。基因的平衡是一种理想状态，并用于测量遗传改变的基准。

最简单的例子是位于单一位点的两个[等位基因](https://zh.wikipedia.org/wiki/等位基因 "wikilink")：显性等位基因记为**A**而隐性等位基因记为**a**，它们的频率分别记为*p*和*q*。频率(**A**) = *p*；频率(**a**) = *q*；*p* + *q* = 1。如果群体处于平衡状态，则我们可以得到

  -
    群体中[纯合子](https://zh.wikipedia.org/wiki/纯合子 "wikilink")**AA**的频率(**AA**) = *p*<sup>2</sup>
    群体中[纯合子](https://zh.wikipedia.org/wiki/纯合子 "wikilink")**aa**的频率(**aa**) = *q*<sup>2</sup>
    群体中[杂合子](https://zh.wikipedia.org/wiki/杂合子 "wikilink")**Aa**的频率(**Aa**) = 2*pq*

## 推导

对于[二倍体物种](https://zh.wikipedia.org/wiki/二倍体 "wikilink")，考虑[种群中两个独立的等位基因A和a](https://zh.wikipedia.org/wiki/种群 "wikilink")，它们的频率分别是*p*和*q*。使用[旁氏表](../Page/旁氏表.md "wikilink")推导出形成新基因型的不同方式，其中每一格的值为行与列概率的乘积。

|         | 雌性        |
| ------- | --------- |
| A (*p*) | a (*q*)   |
| 雄性      | A (*p*)   |
| a (*q*) | Aa (*pq*) |

表1：哈代-温伯格平衡的旁氏表

最终得到后代的三个可能基因型频率：

  - \(f(\mathbf{AA}) =  p^2\,\)
  - \(f(\mathbf{Aa}) = 2pq\,\)
  - \(f(\mathbf{aa}) = q^2\,\)

这些频率称为哈代-温伯格平衡。无限大小的群体经过一个世代的随机交配，基因型频率就能达到平衡。在没有其它因素的影响下，这种平衡状态将一直保持。

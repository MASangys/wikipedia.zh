**哈利托諾夫定理**（Kharitonov's theorem）是[控制理论](../Page/控制理论.md "wikilink")中判斷[动力系统](../Page/动力系统.md "wikilink")的定理，此定理是用在無法得到系統參數的確切值，因此無法判斷穩定性（例如判斷所有根的實部都是負值）的情形下，哈利托諾夫定理用在系統係數只確定在一定範圍內的情形下，提供了針對區間多項式（interval polynomial）的穩定性判斷方式，而[勞斯–赫爾維茨穩定性判據是針對一般的](https://zh.wikipedia.org/wiki/勞斯–赫爾維茨穩定性判據 "wikilink")[多項式](../Page/多項式.md "wikilink")。

## 定義

區間多項式是指以下的多項式族

\[p(s)= a_0 + a_1 s^1 + a_2 s^2 + ... + a_n s^n\] 其係數\(a_i \in R\)是在以下區間內的任意值

\[l_i \le a_i \le u_i.\] 一般會假設最高位係數不能為0：\(0 \notin [l_n, u_n]\).

## 定理

區間多項式穩定（也就是其中所有多項式都穩定）若且唯若以下四個「哈利托諾夫多項式」都穩定

\[k_1(s) = l_0 + l_1 s^1 + u_2 s^2 +  u_3 s^3 + l_4 s^4 + l_5 s^5 + \cdots\]

\[k_2(s) = u_0 + u_1 s^1 + l_2 s^2 +  l_3 s^3 + u_4 s^4 + u_5 s^5 + \cdots\]

\[k_3(s) = l_0 + u_1 s^1 + u_2 s^2 +  l_3 s^3 + l_4 s^4 + u_5 s^5 + \cdots\]

\[k_4(s) = u_0 + l_1 s^1 + l_2 s^2 +  u_3 s^3 + u_4 s^4 + l_5 s^5 + \cdots\] are stable.

哈利托諾夫定理結果令人驚訝的是只要確認四個多項式，就可以判斷其中所有的多項式是否都穩定。因此可以用[勞斯–赫爾維茨穩定性判據或是其他方式判斷](https://zh.wikipedia.org/wiki/勞斯–赫爾維茨穩定性判據 "wikilink")。相對於一般多項式的穩定性判斷，哈利托諾夫定理只要花四倍時間，就可以判斷區間多項式內的所有多項式是否穩定。

哈利托諾夫定理可用在[鲁棒控制](../Page/鲁棒控制.md "wikilink")中，即使在因為[测量误差](https://zh.wikipedia.org/wiki/测量误差 "wikilink")、運作條件的變化、設備磨損等造成零件特性的變化時，系統仍然可以正常運作。

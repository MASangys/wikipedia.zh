**哈巴罗夫斯克SKA能源足球俱乐部**（简称SKA哈巴罗夫斯克），亦简称**伯力能源**,是一家位于[哈巴罗夫斯克市的俄罗斯职业足球俱乐部](https://zh.wikipedia.org/wiki/哈巴罗夫斯克市 "wikilink")。他们于2016–17赛季取得[俄羅斯足球甲級聯賽第四名的成绩](https://zh.wikipedia.org/wiki/俄羅斯足球甲級聯賽 "wikilink")，将在2017-18赛季于队史上首次在[俄羅斯足球超級聯賽](../Page/俄羅斯足球超級聯賽.md "wikilink")中出场。

## 历史

该俱乐部在历史上数次更名，如下：

|             |                  |
| ----------- | ---------------- |
| 1953年以前     | DKA              |
| 1954年       | ODO              |
| 1955年-1956年 | DO               |
| 1957年       | OSK              |
| 1957年-1959年 | SKVO             |
| 1960年-1999年 | SKA              |
| 1999年-2016年 | SKA-能源           |
| 2016年至今     | SKA-哈巴罗夫斯克 \[1\] |

该俱乐部自1957年以来一直参加苏联的锦标赛。在1963年的苏联杯决赛中，该队晋级四分之一决赛。直到2017年冲超前，他们从未参加过苏联顶级联赛或俄罗斯超级联赛。

该队在苏联甲级联赛的最佳成绩是1980年的第6位。而他们在俄罗斯甲级联赛中最好的成绩是在2012-13赛季的第4位，后因在附加赛中0-3输给当届俄超第13名罗斯托夫队而错过了在顶级联赛中竞争的机会。

在2016-17赛季结束的时候，该队再次获得第四名的好成绩，并在2017年5月28日的附加赛第二轮（主客场常规时间成绩均为0-0）上，以点球大战5-3的成绩击败了奥伦堡，成功升级。

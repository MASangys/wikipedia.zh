**哈德斯菲爾德2012年至2013年球季**（）是[哈德斯菲尔德在](../Page/哈德斯菲尔德足球俱乐部.md "wikilink")[英格蘭足球聯賽第](https://zh.wikipedia.org/wiki/英格蘭足球聯賽 "wikilink")93個球季，亦是自2001年降班[乙組〈III〉後首度回升第二級的](https://zh.wikipedia.org/wiki/英格蘭足球聯賽 "wikilink")[英格蘭足球冠軍聯賽](../Page/英格蘭足球冠軍聯賽.md "wikilink")。

## 記事

  - 2013年1月24日於球隊12戰不勝，聯賽排名跌至18位，帶領球隊升班的領隊[西蒙·加利臣](../Page/西蒙·加利臣.md "wikilink")遭撤除職務，是本季第11名下台的[英冠領隊](https://zh.wikipedia.org/wiki/英冠 "wikilink")\[1\]。
  - 2013年2月14日聘任前[-{zh-hans:考文垂;
    zh-hk:高雲地利;}-領隊](../Page/考文垂足球俱乐部.md "wikilink")[马克·罗宾斯](../Page/马克·罗宾斯.md "wikilink")（Mark
    Robins）接掌球隊，助理領隊史提夫·泰萊（Steve Taylor）一同加盟\[2\]。

## 球衣

本季球衣繼續由[茵寶](../Page/茵寶.md "wikilink")（Umbro）製造及供應。主場球衣維持傳統的藍白直間衫白褲黑襪，淺藍間條具有威化餅暗紋；而作客為白肩深藍衫深藍褲深藍襪；第三球球衣為胸前具有黑白間"V"形圖形的紅衫黑褲白襪。而主要[贊助商分別是主場的](https://zh.wikipedia.org/wiki/贊助 "wikilink")「Rekorderlig
Cider 」和作客的「Radian B」。  |  |  |  |  |  |  |  |  |  |}

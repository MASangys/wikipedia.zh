**商品及服務稅**，是[增值稅的一種](../Page/增值稅.md "wikilink")，也是[銷售稅的一種](../Page/銷售稅.md "wikilink")，指政府為各種[商品及](../Page/商品.md "wikilink")[服務所徵收的](../Page/服務.md "wikilink")[稅項](../Page/稅.md "wikilink")。不同於一般的銷售稅（[零售稅](../Page/零售稅.md "wikilink")），商品及服務稅並非只由消費者承擔，生產商及分銷商也需要繳交。

## 機制

商品及服務稅有著增值稅的特點，就是每一個商業過程都涉及繳稅。下表將會簡單說明這種稅項的運作，假設稅率為10%：

|          |      |           |      |             |
| -------- | ---- | --------- | ---- | ----------- |
| 角色       | 原本標價 | 售價之商品及服務稅 | 最終售價 | 應付稅項金額      |
| 原材料供應商   | $50  | $5        | $55  | $5          |
| 生產商      | $150 | $15       | $165 | $15-$5=$10  |
| 零售商      | $250 | $25       | $275 | $25-$15=$10 |
| 商品及服務稅總額 |      |           |      | $25         |

  - 原材料供應商向生產商售出原本標價$50的原材料，需要額外收取$5的商品及服務稅。原材料供應商收到的額外$5將會全數付給稅務部門。
  - 生產商向零售商售出原本標價$150的貨品，需要額外收取$15的商品及服務稅。生產商收到的額外$15中，其中$5是補償由原材料供應商代為徵收的商品及服務稅，其餘$10將會付給稅務部門。
  - 零售商向消費者售出原本標價$250的貨品，需要額外收取$25的商品及服務稅。零售商收到的額外$25中，其中$15是補償由生產商代為徵收的商品及服務稅，其餘$10將會付給稅務部門。
  - 稅務部門分別從原材料供應商、生產商及零售商收取$5、$10及$10，總數為$25，即是零售商貨品原本標價$250的10%。

## 各地推行狀況

目前[加拿大](../Page/加拿大.md "wikilink")、[澳洲](../Page/澳洲.md "wikilink")、[新西蘭及](../Page/新西蘭.md "wikilink")[新加坡等國家徵收商品及服務稅](../Page/新加坡.md "wikilink")，[香港於](../Page/香港.md "wikilink")2006年中開始研究徵收。

###

加拿大的GST於1991年1月1日由當時[加拿大總理](../Page/加拿大總理.md "wikilink")[馬丁·布賴恩·馬爾羅尼推行](../Page/馬丁·布賴恩·馬爾羅尼.md "wikilink")。稅項備受爭議，更成為他領導的[加拿大進步保守黨於](../Page/加拿大進步保守黨.md "wikilink")1993年大選中落敗的導火線之一。1993年大选获胜的[加拿大自由党在当时的竞选纲领中承诺取消该税项](../Page/加拿大自由党.md "wikilink")，但是执政期间並没有兌現承諾。

加拿大的GST由2006年7月1日起由7%降至6%，而多項生活必需品也獲得豁免，包括[食物](../Page/食物.md "wikilink")、[房租](../Page/房租.md "wikilink")、[醫療服務及](../Page/醫療.md "wikilink")[金融服務等](../Page/金融.md "wikilink")。2008年1月1日起，GST再降至5%。

###

澳洲的GST於2000年7月1日由當時[澳洲總理](../Page/澳洲總理.md "wikilink")[約翰·霍華德推行](../Page/約翰·霍華德.md "wikilink")，取代在此之前政府所推行的[銷售稅稅制](../Page/銷售稅.md "wikilink")，並計劃取消當地[印花稅及](../Page/印花稅.md "wikilink")[地稅等稅項](../Page/地稅.md "wikilink")。

澳洲的商品及服務稅率為10%，適用於絕大部份商品及服務，包括交通費用。所以，在推行初時，曾導致[悉尼跨海大橋大塞車](../Page/悉尼跨海大橋.md "wikilink")，因為不少人都忘記了要路費從未含稅的兩澳元增加至包含GST的2.2元，而需要在繳費窗即時找20分硬幣。

###

紐西蘭的GST始於1986年10月1日推行，初時稅率為10%，後於1989年6月30日提升至12.5%，再於2010年10月1日起提升至15%。與大部份徵收GST的國家不同，紐西蘭GST的徵收範圍涵蓋大部份生活必需品(包括食物)，只有住宅租金、慈善捐款及金融服務可獲豁免。另外，政府亦規定商品的標價必須包含GST，但以批發為主的商家則不在此限。

###

新加坡的GST於1994年推行，初時稅率為3%，後於2003年提升至4%，再於2004年提升至5%，2007年7月1日提高至7%并维持至今。新加坡政府於1994年及2003年分別給予11.04億[新加坡元及](../Page/新加坡元.md "wikilink")8.25億新加坡元補助低收入家庭。現在新加坡平均每年的商品及服務稅收入約為40億新加坡元。

另一方面，由於得到GST的補貼，新加坡政府得以把當地的利得稅下調5%。

###

早在主權移交前，香港政府曾多次希望在香港推行商品及服務稅，當中以[夏鼎基擔任財政司期間幾乎正式實行](../Page/夏鼎基.md "wikilink")。由於當時香港的通貨膨漲非常熾烈，政府希望透過商品及服務稅來壓抑通漲。不過由於當時政府亦正在推行[代議政制](../Page/代議政制.md "wikilink")，為免打擊殖民地政府在市民的管治威信，商品及服務稅最終終止實行。[香港主權移交後](../Page/香港主權移交.md "wikilink")，經歷了數年財赤，香港政府再有開徵商品及服務稅的構思。不過，因為當時還未有具體方案解決當前的經濟問題，所以其討論又再押後。2006年7月18日，香港政府終於公布改革[香港稅制的諮詢文件](../Page/香港稅制.md "wikilink")，研究通過開徵商品及服務稅以擴闊稅基的可行性，但遭到社會大力反對。

根據諮詢文件，香港政府提出開徵5%的商品及服務稅，庫房收入增加300億港元。其中72億港元會用於寬免措施，包括領取[香港綜合社會保障援助人士的現金紓緩津貼](../Page/香港綜合社會保障援助.md "wikilink")、低收入家庭每年2000港元的津貼，以及所有家庭每年3500港元的[水費及](../Page/水費.md "wikilink")[差餉扣減](../Page/差餉.md "wikilink")。政府亦會取消[酒店房租稅及](../Page/酒店房租稅.md "wikilink")[名義資本額註冊費](../Page/名義資本額註冊費.md "wikilink")，並調低[汽車首次登記稅及](../Page/汽車首次登記稅.md "wikilink")[汽油](../Page/汽油.md "wikilink")、[柴油之](../Page/柴油.md "wikilink")[油稅](../Page/油稅.md "wikilink")，收入將減少19億港元。另一方面，收取商品及服務稅的行政費用預計為5億港元。因此，政府的淨收入將會增加204億港元。然而，政府承諾稅項實施後首5年，將會將全數淨收入回饋市民，考慮方式包括寬免[薪俸稅及](../Page/薪俸稅.md "wikilink")[利得稅](../Page/利得稅.md "wikilink")。

目前香港多個政黨均持保留態度，[香港民主黨](../Page/香港民主黨.md "wikilink")、[香港自由黨及部份獨立](../Page/香港自由黨.md "wikilink")[香港立法會議員已表明反對](../Page/香港立法會.md "wikilink")。很多市民也不支持商品及服務稅。

2006年12月5日，財政司司長[唐英年表示](../Page/唐英年.md "wikilink")，因市民對此稅項未有共識，政府在餘下的諮詢期內，不再考慮推行商品及服務稅。

## 批評

對商品及服務稅最主要的批評是加劇[貧富懸殊](../Page/貧富懸殊.md "wikilink")。因商品及服務稅屬[累退稅](../Page/累退稅.md "wikilink")，低收入人士的消費多屬生活必需品，他們很難因為徵收商品及服務稅而減少消費；反之，高收入人士往往會因商品及服務稅而減少不必要消費。因此，商品及服務稅加重對低收入人士的稅務負擔，劫貧濟富。

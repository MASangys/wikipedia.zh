**啟翔苑**（）是[香港房屋委員會](../Page/香港房屋委員會.md "wikilink")的[居者有其屋計劃](../Page/居者有其屋計劃.md "wikilink")屋苑之一，位於[香港](../Page/香港.md "wikilink")[九龍](../Page/九龍.md "wikilink")[大磡舊址東部](../Page/大磡村.md "wikilink")，鄰近港鐵[屯馬線](https://zh.wikipedia.org/wiki/屯馬線 "wikilink")[鑽石山站](https://zh.wikipedia.org/wiki/鑽石山站 "wikilink")。屋苑連同鄰近的公共屋邨由房屋署總建築師（五）統一負責總體設計，並由[劉榮廣伍振民建築師事務所負責詳細設計](https://zh.wikipedia.org/wiki/劉榮廣伍振民建築師事務所 "wikilink")。

## 樓宇

屋苑共設兩座分別高39層（啟怡閣）及33層（啟悅閣）樓宇（包括地下），提供920伙單位。所有樓宇於2019年7月8日正式命名\[1\]。

| 樓宇名稱（座號） | 樓宇類型                                         | 落成年份  | 層數 |
| -------- | -------------------------------------------- | ----- | -- |
| 啟怡閣（A座）  | [非標準設計大廈](../Page/非標準設計大廈.md "wikilink")（Y型） | 2022年 | 39 |
| 啟悅閣（B座）  | 33                                           |       |    |
|          |                                              |       |    |

項目上蓋承建商暫時未定。升降機方面，啟怡閣設有四台升降機，均不直通全部樓層；而啟悅閣則設三台升降機，其中一台直達全部樓層（共設7台升降機）\[2\]。

## 設施

屋苑地庫設有停車場，並有行人隧道接駁港鐵[鑽石山站及新蒲崗一帶](https://zh.wikipedia.org/wiki/鑽石山站 "wikilink")。地面則設有兒童遊樂場及園景設施\[3\]。旁邊公共屋邨亦設有十多萬平方呎的商場。

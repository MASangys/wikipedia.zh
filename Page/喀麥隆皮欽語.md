**喀麥隆皮欽語**（）或稱**喀麥隆皮欽英語**（\[1\]），通稱Camtok（出自 Cameroon
talk）是一系列[英語為基礎的](https://zh.wikipedia.org/wiki/英語 "wikilink")[皮欽語及](https://zh.wikipedia.org/wiki/皮欽語 "wikilink")[克里奧爾語的統稱](https://zh.wikipedia.org/wiki/克里奧爾語 "wikilink")，由於其部分種類變體已經[克里奧爾化](https://zh.wikipedia.org/wiki/克里奧爾化 "wikilink")，又稱作**喀麥隆克里奧爾語**（）。

## 概況

喀麥隆皮欽語在喀麥隆使用，擁有諸多變體，該國約5%人口使用某種變體作為[母語](../Page/母語.md "wikilink")，但在跨區交流中，該語言使用頻率很高。其中許多使用者覺得自己使用的是[英語](https://zh.wikipedia.org/wiki/英語 "wikilink")，與別國英語並無差異。
其穩定的變體主要由如下幾類：

  - [林貝Kamtok](https://zh.wikipedia.org/wiki/林貝 "wikilink")，[林貝及周邊區域](https://zh.wikipedia.org/wiki/林貝 "wikilink")
  - [昆巴Kamtok](../Page/昆巴.md "wikilink")，[昆巴及周邊區域](../Page/昆巴.md "wikilink")
    \[2\]
  - 草原Kamtok，主要使用在草原地區
  - 教會Kamtok，用於部分天主教區
  - 法式Kamtok，喀麥隆法語區人士同非法語者溝通時使用
  - 波洛洛Kamtok，在交易波洛洛牛時會使用的變體，交易者大多來自尼日利亞和喀麥隆

該語言已經在當地活躍了超過200年，曾在[德國統治時期被禁止](https://zh.wikipedia.org/wiki/德屬喀麥隆 "wikilink")，也曾一段時間在校園和政治領域廣泛使用，目前為喀麥隆的一種口語溝通語言。

## 發音

Camtak的發音很有融入了許多當地語言的特點，元音輔音與英語相比，均發生了音變。例如：英語中的 th 音，通常由t\\d代替，them 被讀作
dem；英語中的-er、-ow可能被讀作 a、o等，如 neva （never）, na (now)。\[3\]

## 詞句

### 問候

Camtok通常也會使用Hi、hello進行問候，此外也會說 How na （英語How are you）,回答使用Ah fine 或者 Ah
dey 表示自己很好。\[6\]

### 詞彙

Camtok中大部分[名詞](../Page/名詞.md "wikilink")、[動詞等詞彙來自英語](https://zh.wikipedia.org/wiki/動詞 "wikilink")，如
pipo ()、 kam ()、 dat kain ting ()等。

亦有部分來自當地語言，如tutu (勤奮)等。

由於法語在該國的重要性，近年也有大量法語詞彙進入Camtok，例如：vinyet （车窗税，）、fonsion
publik（公共部門，）等。\[7\]

#### 名詞

通常，名詞不會通過加 -s表達複數，而是使用 dem 作為複數標記。\[8\] 如：The boy **dem** di cam.
（英語：The boys are coming.）

#### 代詞

Camtok[代詞具有三個](../Page/代詞.md "wikilink")[人稱](https://zh.wikipedia.org/wiki/人稱 "wikilink")，區分[單數和複數](https://zh.wikipedia.org/wiki/單數和複數 "wikilink")，但通常不會[變格](https://zh.wikipedia.org/wiki/格 "wikilink")（即不會像歐洲語言那樣將代詞變化為主格、賓格、所屬格等）。謹部分人稱有不嚴格的區分，如
ma 通常表示“我的”（如： ma beli）， mi 和 a 用於其他場合 mi mos go 或者 a mos go （英語： I must
go）。再入，dem 也有 dei 和 dam 的變化形式。

| [人稱](https://zh.wikipedia.org/wiki/人稱 "wikilink")     | [單數](https://zh.wikipedia.org/wiki/單數 "wikilink") | [複數](https://zh.wikipedia.org/wiki/複數 "wikilink") |
| ----------------------------------------------------- | ------------------------------------------------- | ------------------------------------------------- |
| [第一人稱](https://zh.wikipedia.org/wiki/第一人稱 "wikilink") | mi, a, ma                                         | wi                                                |
| [第二人稱](https://zh.wikipedia.org/wiki/第二人稱 "wikilink") | yu                                                | (w)una                                            |
| [第三人稱](https://zh.wikipedia.org/wiki/第三人稱 "wikilink") | i, -am                                            | dem, -am                                          |
|                                                       |                                                   |                                                   |

#### 動詞

Camtok[動詞通常不會通過](https://zh.wikipedia.org/wiki/動詞 "wikilink")[曲折变化來表示語法上的](https://zh.wikipedia.org/wiki/變位 "wikilink")[時態](https://zh.wikipedia.org/wiki/時態 "wikilink")、[語態等](https://zh.wikipedia.org/wiki/語態 "wikilink")，而通常通過添加如下標記或詞彙表示\[9\]

| Camtok         | [英語](https://zh.wikipedia.org/wiki/英語 "wikilink") | 意義    |
| -------------- | ------------------------------------------------- | ----- |
| *no*           | *not*                                             | (表否定) |
| *neba*, *neva* | *never*                                           | 從不    |
| *bin* *bi*     |                                                   | (表過去) |
| *go*           |                                                   | (表將來) |
| *don* *dong*   |                                                   | (表完成) |
| *di*           |                                                   | (表進行) |
| *fit*          | *can, able to*                                    | 能夠    |
| *lak*          | *must, like*                                      | (表偏好) |
| *mos*          | *must, ought to*                                  | 必須    |
| *wan*          | *want to*                                         | 想     |
| *look*         | *look, watch*                                     | 看     |

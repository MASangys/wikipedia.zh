**喉輪**（[梵語](https://zh.wikipedia.org/wiki/梵語 "wikilink")：विशुद्ध，Viśuddha）是[印度教](../Page/印度教.md "wikilink")傳統的第五個主要[脈輪](https://zh.wikipedia.org/wiki/脈輪 "wikilink")。

## 關聯

喉輪位於[喉部](https://zh.wikipedia.org/wiki/咽喉 "wikilink")，有十六片對應[梵語](https://zh.wikipedia.org/wiki/梵語 "wikilink")[元音](../Page/元音.md "wikilink")的[花瓣](https://zh.wikipedia.org/wiki/花瓣_\(脈輪\) "wikilink")。

喉輪與更高的辨別對錯能力有關，也和創造力以及自我表達有關。它也被稱為「毒與蜜」的中心，與[明點](https://zh.wikipedia.org/wiki/明點 "wikilink")（[Bindu](https://zh.wikipedia.org/wiki/:en:Bindu "wikilink")）關係密切，據說長生不老的秘密就在喉輪裡。當喉輪關閉，我們就會腐朽、死亡。當喉輪打開，負面的經驗就會轉化為智慧與學習。

據說喉輪包含了智慧的力量（[jnana](https://zh.wikipedia.org/wiki/:en:Jnana "wikilink") shakti）。

喉輪的種子音是**हं **。喉輪的花瓣對應以下十六個元音：

|   |   |   |   |   |   |    |    |
| - | - | - | - | - | - | -- | -- |
| अ | आ | इ | ई | उ | ऊ | ऋ  | ॠ  |
| ऌ | ॡ | ए | ऐ | ओ | औ | अः | अं |

（注意：以上所列的某些元音嚴格來說並不符合語法上對梵語元音的定義， 特別是ॡ 、अः 和अं 。詳情請見[梵語語音體系](https://zh.wikipedia.org/wiki/梵語#語音體系 "wikilink")。）

## 象徵

喉輪與以下有關：

  - 神祇：Sadashiva、Shakini
  - 元素：[以太](https://zh.wikipedia.org/wiki/以太_\(傳統元素\) "wikilink")（Akasha）
  - 動物：[象](../Page/象.md "wikilink")
  - 身體部位：[甲狀腺](https://zh.wikipedia.org/wiki/甲狀腺 "wikilink")（下述）、[耳](../Page/耳.md "wikilink")

## 修練

在亢達里尼瑜伽中，喉輪可以透過修練來開啟並平衡，方式包括體位法（例如肩立式）、呼吸法、收頷收束法（喉鎖）和逆舌身印。

## 其他關聯

喉輪經常被認為和人體[內分泌系統](../Page/內分泌系統.md "wikilink")中的[甲狀腺有關](https://zh.wikipedia.org/wiki/甲狀腺 "wikilink")。甲狀腺位於喉部，會製造出生長與成熟過程中必要的激素。

西方神秘學家對於喉輪與卡巴拉的關聯各有不同的說法。有些人認為它和隱藏的原質（sephira）[Da'at有關](https://zh.wikipedia.org/wiki/Da'at "wikilink")，在其中，「智慧」與「了解」在超凡的領域裡被「知識」層面平衡；知識是一個實際的想法，當它被表現出來時，就會引發[創造的行為](../Page/創世神話.md "wikilink")（不過，斯瓦密·薩特亞南達（Swami Satyananda）則認為創造的行為與明點有關）。

其他人則認為喉輪與原質[Chesed及](https://zh.wikipedia.org/wiki/Chesed "wikilink")[Geburah](https://zh.wikipedia.org/wiki/Geburah "wikilink")——也就是仁慈與力量——有關，這兩個原質和道德的關係很密切，而Chesed表現的擴張概念與Geburah表現的限制概念對個體的創造都是必要的。這一點在道德觀方面表現於瑜伽中的[持戒](https://zh.wikipedia.org/wiki/持戒 "wikilink")（yama）與[精進](https://zh.wikipedia.org/wiki/精進 "wikilink")（niyama）。

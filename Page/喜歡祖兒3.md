## 派台歌曲成績

本專輯的派台歌為《明日恩典》、《呼天不應》和《刻不容緩》，當中《明日恩典》為四台冠軍歌，詳細派台成績如下:

| 歌曲   | 903 專業推介 | RTHK 中文歌曲龍虎榜 | 997 勁爆流行榜 | TVB 勁歌金榜 | 備註                                  |
| ---- | -------- | ------------ | --------- | -------- | ----------------------------------- |
| 明日恩典 | **1**    | **1**        | **(1)**   | **1**    | 四台冠軍歌、四洲咖啡糖廣告歌                      |
| 刻不容緩 | 16       | 14           | \-        | 5        | 與[李克勤](../Page/李克勤.md "wikilink")合唱 |
| 呼天不應 | 15       | 3            | 2         | 6        | 幸福樂喉適廣告歌                            |
|      |          |              |           |          |                                     |

  - **(1)** 代表兩週冠軍
  - \* 代表上榜中

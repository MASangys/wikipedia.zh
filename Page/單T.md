為賽馬術語，是賽馬博彩其中一個投注種類，在香港、新加坡、英國、美國、澳洲、日本、韓國等地方均有接受投注，而香港及新加坡粵語同樣稱為「單T」，投注方法是一場賽馬選擇前三名的馬匹。和三重彩不同的是，單T不需要順序選出排名。

## 投注金額

以香港賽馬會為例

單T每-{注}-港幣10元，同時可購買每-{注}-或每票多於港幣10元-{只}-適用於投-{注}-種類的「複式」及「馬-{膽}-」投-{注}-。當每票投注總額達港幣$100元或以上，可以[靈活玩](../Page/靈活玩.md "wikilink")最低金額港幣$1元作投注。

## 投注方法

|                |                                                                                                                          |
| -------------- | ------------------------------------------------------------------------------------------------------------------------ |
| **投-{注}-種類**   | **投-{注}-方法**                                                                                                             |
| 「單式」投-{注}-     | 從自選任何馬匹編號中選出3匹不同馬匹編號。                                                                                                    |
| 「複式」投-{注}-     | 假設您的心水馬為6號、7號、8號、 9號，複式互串會將所選馬匹組成所有可能的組合，即分別為6+7+8、6+7+9、6+8+9 及 7+8+9，總共四注,如此類推。                                        |
| 「馬-{膽}-」投-{注}- | 當您對一匹馬有很大信心能跑入前三名，您可選擇以該匹馬作「馬膽」，其他選擇則作為「配腳」。 假設重心馬為6號，以6號胆拖7號、8號、9號，即分別為6+7+8、6+7+9 及 6+8+9，總共三注，只要其中一個馬匹組合同時跑入前三名，便為中彩。 |

## 中獎方法

選中一場賽事中的第1名、第2名及第3名馬，毋須順序，便獲派彩，派彩將按照該馬匹組合的最後單T賠率計算。 例如最後單T賠率為88倍，以每注投注金額$10計算，派彩為例10 x 88倍=$880。

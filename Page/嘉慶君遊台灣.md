**嘉慶君遊台灣**是[台灣的](../Page/台灣.md "wikilink")[民間故事](../Page/民間故事.md "wikilink")。大意是在[清朝](../Page/清朝.md "wikilink")[乾隆年間](../Page/乾隆.md "wikilink")，尚未內定為[儲君的](../Page/儲君.md "wikilink")[嘉慶帝曾經](../Page/嘉慶帝.md "wikilink")[微服出巡遊歷台灣](../Page/微服出巡.md "wikilink")。迄今仍有許多穿鑿附會的說法\[1\]，但並非史實，嘉慶帝一生不曾到過台灣。明清宗室住過臺灣的是[朱術桂](../Page/朱術桂.md "wikilink")、瀘溪王、魯王世子[朱弘桓](../Page/朱弘桓.md "wikilink")、巴東王等諸明朝宗室\[2\]\[3\]。

## 廟宇傳說

臺灣有不少廟宇有跟「嘉慶君遊臺灣」相關的傳說。

| 廟名                                   | 位置       | 傳說內容                                                                                                                                                                                                                                                                                                           |
| ------------------------------------ | -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 草屯慶安宮福德祠                             | 臺灣南投縣草屯鎮 | 傳說嘉慶君遊臺灣時經過[北投街](../Page/北投街.md "wikilink")，夜宿土地公廟，因[土地公指點免於遭人謀害](../Page/土地公.md "wikilink")（也有說法是託夢警告將有[臺灣原住民要](../Page/臺灣原住民.md "wikilink")[出草](../Page/出草.md "wikilink")），事後乃賜[御匾封該廟為](../Page/御匾.md "wikilink")「慶安宮」，「慶安」二字即是指「保佑嘉慶君平安」。另外也有傳說嘉慶君暫宿時因廟中空間有限而將土地公神像移置一旁，隔日卻忘了放回原位，結果一行人走不出村莊，直到將神像放回原位才能離開。 |
| [崁頂福安宮](../Page/崁頂福安宮.md "wikilink") | 臺灣臺南市白河區 | 傳說嘉慶君遊臺灣時經過[蔴荳保時遇到土匪](../Page/蔴荳堡.md "wikilink")，幸有白鬍老翁相救。日後嘉慶君登基後詔封該廟土地公王侯官職，並賜「三寶」（官帽、日月乾坤、文旦）。                                                                                                                                                                                                              |
|                                      |          |                                                                                                                                                                                                                                                                                                                |

**四呼**是近现代[汉语音韵学](../Page/汉语音韵学.md "wikilink")的概念，用以指称近现代汉语的[介音系统](https://zh.wikipedia.org/wiki/介音 "wikilink")。四呼指**开口呼**、**齐齿呼**、**合口呼**和**撮口呼**。其本质是汉语[韵母](../Page/韵母.md "wikilink")的四种类型。举[现代标准汉语为例](https://zh.wikipedia.org/wiki/现代标准汉语 "wikilink")，韵母不是 ，又没有  [介音的](https://zh.wikipedia.org/wiki/介音 "wikilink")，称为开口呼；或有介音的韵母称为齐齿呼；或有介音的韵母称为合口呼；或有介音的韵母称为撮口呼。（实际上不严格如此，因为考虑了语音系统和历史传承的因素。例如漢語拼音中ong属合口呼，iong属撮口呼，是把它们分别分析为、的变体。而注音符號則依照傳統漢語音韻，忠實反映四呼之分類。）

## 起源

四呼，这是比较后起的术语（大约起于明末清初），和“四等”不能混为一谈。所谓四呼，是指开口呼（开）、合口呼（合）、齐齿呼（齐）和撮口呼（撮）。在宋代的韵图中，只有开口合口两呼。“四呼”是把开口呼分为两类（开口和齐齿），合口呼分为两类（合口和撮口）。照传统的说法，开口一二等是开口呼，开口三四等是齐齿呼，合口一二等是合口呼，合口三四等是撮口呼。这个说法基本上是可以承认的，只是在二等字上，宋代的等呼和明代的四呼稍有出入。

产生“四呼”的说法的原因是语音演变：四等韵已经为三等所同化，二者合而为一；二等韵或者为一等所同化，或者为三等所同化，也往往不再能独立成为一类，于是成了一个简单化的局面。原来两呼四等共有八类，现在四呼只有四类了。

## 与中古汉语等呼的关系

近现代汉语的四呼是由[中古汉语](../Page/中古汉语.md "wikilink")的[等呼](../Page/等呼.md "wikilink")演变而成的。大致来说，中古的开口一二等演变成开口呼；合口一二等演变成合口呼；开口三四等演变成齐齿呼；合口三四等演变成撮口呼。

| ----- | ------- | ---------- |
| 一等    | 二等      | 三等         |
| **通** |         |            |
| **江** |         | 合口、齐齿、\*撮口 |
| **止** |         |            |
| **遇** |         |            |
| **蟹** | 开口      | 开口、齐齿      |
| **臻** | 开口      | 开口         |
| **山** | 开口      | 开口、齐齿      |
| **效** | 开口      | 开口、齐齿      |
| **果** | 开口、合口   |            |
| **假** |         | 开口、齐齿      |
| **宕** | 开口、\*合口 |            |
| **梗** |         | 开口         |
| **曾** | 开口      |            |
| **流** | 开口      |            |
| **深** |         |            |
| **咸** | 开口      | 开口、齐齿      |

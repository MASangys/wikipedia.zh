**四正丁基硒氰酸铵**是一种[化合物](../Page/化合物.md "wikilink")，化学式为\[(*n*-C<sub>4</sub>H<sub>9</sub>)<sub>4</sub>N\]SeCN。

## 制备

该化合物可由四正丁基溴化铵和[硒氰酸钾](../Page/硒氰酸钾.md "wikilink")在乙醇中反应，过滤除去溴化钾沉淀，然后溶液中加入[乙醚](../Page/乙醚.md "wikilink")沉淀出产物得到。\[1\]

## 配合物

四正丁基硒氰酸铵可以和一些金属盐溶液（如Y(NO<sub>3</sub>)<sub>3</sub>、K<sub>3</sub>\[RhCl<sub>6</sub>\]等）反应，得到硒氰酸根[配合物](../Page/配合物.md "wikilink")。其配合物见下表：\[2\]

| 配合物                                               | 颜色  | 熔点（℃）      |
| ------------------------------------------------- | --- | ---------- |
| R<sub>4</sub>\[Mn(NCSe)<sub>6</sub>\]             | 白色  | 85         |
| R<sub>2</sub>\[Fe(NCSe)<sub>4</sub>\]             | 浅褐色 | 80         |
| R<sub>3</sub>\[Fe(NCSe)<sub>6</sub>\]             | 红棕色 | 78         |
| R<sub>4</sub>\[Ni(NCSe)<sub>6</sub>\]             | 暗绿色 | 121（熔化并分解） |
| R<sub>2</sub>\[Zn(NCSe)<sub>4</sub>\]             | 白色  | 60         |
| R<sub>3</sub>\[Y(NCSe)<sub>6</sub>\]              | 白色  | 159        |
| R<sub>3</sub>\[Rh(SeCN)<sub>6</sub>\]             | 暗红色 | 软化后熔化      |
| R<sub>2</sub>\[Pd(SeCN)<sub>6</sub>\]             | 红棕色 | 软化后熔化      |
| R<sub>2</sub>\[Cd<sub>2</sub>(NCSe)<sub>6</sub>\] | 白色  | 119（熔化并分解） |
| R<sub>2</sub>\[Pt(SeCN)<sub>4</sub>\]             | 红棕色 | 132        |

四正丁基硒氰酸铵配合物

注：表中R为四正丁基铵阳离子。

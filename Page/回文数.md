**回文数**或**-{zh-cn:迴文数; zh-tw:回文數;}-**是指一个像14641这样“对称”的[数](https://zh.wikipedia.org/wiki/数_\(数学\) "wikilink")，即：将这个数的[数字按相反的顺序重新排列后](https://zh.wikipedia.org/wiki/数字 "wikilink")，所得到的数和原来的数一样。这裡，“[回文](../Page/回文.md "wikilink")”是指像“妈妈爱我，我爱妈妈”这样的，正读反读都相同的单词或句子。

回文数在[休闲数学领域备受关注](https://zh.wikipedia.org/wiki/休闲数学 "wikilink")。一个典型的问题就是，寻找那些具有某种特性，并且符合回文特征的数。例如：

  - [回文素数](../Page/回文素数.md "wikilink")：2, 3, 5, 7, 11, 101, 131, 151,…
  - 回文[完全平方](../Page/完全平方.md "wikilink")数：0, 1, 4, 9, 121, 484, 676, 10201, 12321,…

[巴克敏斯特·福乐在其著作](../Page/巴克敏斯特·富勒.md "wikilink")《[协同学](https://zh.wikipedia.org/wiki/协同学 "wikilink")》（**）中把回文数也叫做**沙拉扎数**（Scheherazade Numbers），沙拉扎是《[一千零一夜](../Page/一千零一夜.md "wikilink")》中那位讲故事的王妃、即宰相的女儿的名字。

直观地，在任意的[進位制下都存在着](https://zh.wikipedia.org/wiki/進位制 "wikilink")[无穷多个回文数](https://zh.wikipedia.org/wiki/无限集合 "wikilink")。可以这样说明：在任意的基下，一个像101, 1001, 10001,… （即由一个1后接*n*个0再后接一个1）这样的数可组成一个无穷多项的[序列](../Page/序列.md "wikilink")，其各项全部都是回文数，因此这个基下的回文数有无穷多个（其中包括但不限于该序列中的无穷多个项）。

## 正式定义

虽然通常是在[十进制](../Page/十进制.md "wikilink")系统下来考虑回文数，但**回文性**的性质可推广用于任何[记数系统](../Page/记数系统.md "wikilink")中的[自然数](../Page/自然数.md "wikilink")。考虑以 \(b\ (b\ge2)\) 为[基的数](https://zh.wikipedia.org/wiki/基_\(数学\) "wikilink") \(n\ (n >0)\)，在基 \(b\) 下，\(n\) 可按标准方式表示为 \(k+1\) 个[数字](https://zh.wikipedia.org/wiki/数字 "wikilink") \(a_i\)，即：

\[n=\sum_{i=0}^ka_ib^i\] 其中，如惯例，对所有 \(i\) 都要求 \(0 \leq a_i \leq b\)，且 \(a_k \ne 0\)。 则 \(n\) 称为回文数，当且仅当对所有 \(i\) 都有\(a_i = a_{k-i}\)。[零在任何基下均写作](https://zh.wikipedia.org/wiki/零 "wikilink") 0 并由定义认为它也是回文数。

另一种等价的定义如下：在任意固定的基 \(b\) 下，数\(n\)称为回文的当且仅当：

  - \(n\)是单个数字，或
  - \(n\)为两个相同数字，或
  - \(n\)由三个或更多数字组成，其首位和末位数字相同，且从\(n\)中去掉该首位和末尾数字后的数也是回文的。

## 十进制回文数

[10](../Page/十进制.md "wikilink")[基数下](https://zh.wikipedia.org/wiki/数字系统 "wikilink")，所有单个[数字](https://zh.wikipedia.org/wiki/数字 "wikilink"){[0](../Page/0.md "wikilink")、[1](../Page/1.md "wikilink")、[2](../Page/2.md "wikilink")、[3](../Page/3.md "wikilink")、[4](../Page/4.md "wikilink")、[5](../Page/5.md "wikilink")、[6](../Page/6.md "wikilink")、[7](../Page/7.md "wikilink")、[8](../Page/8.md "wikilink")、[9](../Page/9.md "wikilink")}都是回文数。

两位数的回文数有9个：

  -
    {11, 22, 33, 44, 55, 66, 77, 88, 99}

  -
    {101, 111, 121, 131, 141, 151, 161, 171, 181, 191, ..., 909, 919, 929, 939, 949, 959, 969, 979, 989, 999}

    {1001, 1111, 1221, 1331, 1441, 1551, 1661, 1771, 1881, 1991, ..., 9009, 9119, 9229, 9339, 9449, 9559, 9669, 9779, 9889, 9999}

## 其它的基数下的回文数

也可在[十进制](../Page/十进制.md "wikilink")以外的其它[数系中考虑回文数](https://zh.wikipedia.org/wiki/数字系统 "wikilink")。例如，在[二进制中的回文数有](https://zh.wikipedia.org/wiki/二进制 "wikilink")：

  -
    0, 1, 11, 101, 111, 1001, 1111, 10001, 10101, 11011, 11111, 100001,…

以上这些数在十进制中即：0, 1, 3, 5, 7, 9, 15, 17, 21, 27, 31, 33,…。[梅森素数](../Page/梅森素数.md "wikilink")构成了二进制回文素数的一个子集。

通常在一个基数下的回文数在另一个基数下就不再是回文数。例如：16461<sub><small>10</small></sub> = 404D<sub><small>16</small></sub>。（下标的数字表示的是[基数](https://zh.wikipedia.org/wiki/基_\(数学\) "wikilink")，即*n*<sub><small>16</small></sub>表示以[十六进制](../Page/十六进制.md "wikilink")写出的*n*）。然而，有些数字在几个基数中都是回文数（称为“协回文的”，*copalindromic*），例如105<sub><small>10</small></sub>在五个不同的基数下都是回文数：1221<sub><small>4</small></sub> = 151<sub><small>8</small></sub> = 77<sub><small>14</small></sub> = 55<sub><small>20</small></sub> = 33<sub><small>34</small></sub>；十进制数1991在十六进制中为7C7，也是回文的。

在以18为基时，7的一些幂是回文的：

  - 7<sup>3</sup> = 111
  - 7<sup>4</sup> = 777
  - 7<sup>6</sup> = 12321
  - 7<sup>9</sup> = 1367631

对任意数*n*，在所有*b* ≥ *n* + 1的基数*b*下都是回文的（因为这时*n*是一个单位数）；在基为*n*−1时同样也是回文数（因为这时*n*就成了11<sub><small>*n*−1</small></sub>）。如果對於2 ≤ *b* ≤ n − 2，某数在基*b*下都是非回文数，则称其是一个**严格非回文数**（Strictly non-palindromic number）。例如6在[二進制是](https://zh.wikipedia.org/wiki/二進制 "wikilink")110，[三進制是](https://zh.wikipedia.org/wiki/三進制 "wikilink")20，[四進制是](https://zh.wikipedia.org/wiki/四進制 "wikilink")12，都不是回文數，因此它是嚴格非回文數。這樣的數其中一個特質是6以上的數都是[質數](https://zh.wikipedia.org/wiki/質數 "wikilink")。首幾項：1, 2, 3, 4, 6, 11, 19, 47, 53, 79, 103, ... 。

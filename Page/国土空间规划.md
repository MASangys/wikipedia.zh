**国土空间规划**是[中华人民共和国的](https://zh.wikipedia.org/wiki/中华人民共和国 "wikilink")体系，是国家空间发展的指南、可持续发展的空间蓝图，是各类开发保护建设活动的基本依据。\[1\]国土空间规划融合了原有的主体功能区规划、土地利用规划、城乡规划等空间规划，其目的是实现“多规合一”。

## 背景

在[自然资源部成立](../Page/中华人民共和国自然资源部.md "wikilink")、国土空间规划相关政策出台之前，中国内地原有规划种类主要有以下（部分列举）：

| 国民经济和社会发展规划                                           | [国家发展和改革委员会](../Page/中华人民共和国国家发展和改革委员会.md "wikilink")                |                                                                  |
| 主体功能区规划                                               | 国家发展和改革委员会\[2\]                                                      | 《关于加快推进生态文明建设的意见》\[3\]                                           |
| 土地利用规划                                                | 土地行政主管部门（[国土资源部](../Page/中华人民共和国国土资源部.md "wikilink")）                | 《[土地管理法](https://zh.wikipedia.org/wiki/中华人民共和国土地管理法 "wikilink")》 |
| [城乡规划](https://zh.wikipedia.org/wiki/城乡规划 "wikilink") | 城乡规划主管部门（[住房和城乡建设部](../Page/中华人民共和国住房和城乡建设部.md "wikilink")）          | 《[城乡规划法](../Page/中华人民共和国城乡规划法.md "wikilink")》                    |
| 环境保护规划                                                | 环境保护主管部门（[环保部](https://zh.wikipedia.org/wiki/中华人民共和国环保部 "wikilink")） | 《[环境保护法](https://zh.wikipedia.org/wiki/中华人民共和国环境保护法 "wikilink")》 |

除了种类繁多以外，由于各个规划、各个部门的职能之间存在重叠、空白，难以协调，原有的各种规划之间存在矛盾冲突等问题。尽管《城乡规划法》和《土地管理法》都有条文要求其规划要与其他规划衔接，但是并未规定衔接的方式，实际操作中也难以落实。\[4\]

同年12月12-13日，中央城镇化工作会议提到“建立空间规划体系，推进规划体制改革，加快规划立法工作。”\[5\]

2014年3月16日公布的《》提出：“加强城市规划与经济社会发展、主体功能区建设、国土资源利用、生态环境保护、基础设施建设等规划的相互衔接。推动有条件地区的经济社会发展总体规划、城市规划、土地利用规划等‘多规合一’。”\[6\]同年，国家发展改革委、国土资源部、环境保护部、住房城乡建设部联合下发《关于开展市县“多规合一”试点工作的通知》，全国共28个市县纳入“多规合一”试点。\[7\]

2015年9月21日，中共中央、国务院印发《生态文明体制改革总体方案》，其中明确了“构建以空间治理和空间结构优化为主要内容，全国统一、相互衔接、分级管理的空间规划体系，着力解决空间性规划重叠冲突、部门职责交叉重复、地方规划朝令夕改等问题。”“空间规划分为国家、省、市县（设区的市空间规划范围为市辖区）三级。”\[8\]

2018年3月17日[第十三届全国人民代表大会第一次会议通过](https://zh.wikipedia.org/wiki/第十三届全国人民代表大会#第一次会议 "wikilink")《》，批准《》。方案规定：“组建[自然资源部](../Page/中华人民共和国自然资源部.md "wikilink")。将[国土资源部的职责](../Page/中华人民共和国国土资源部.md "wikilink")，[国家发展和改革委员会的组织编制主体功能区规划职责](../Page/中华人民共和国国家发展和改革委员会.md "wikilink")，[住房和城乡建设部的](../Page/中华人民共和国住房和城乡建设部.md "wikilink")[城乡规划管理职责](https://zh.wikipedia.org/wiki/城乡规划 "wikilink")，[水利部的水资源调查和确权登记管理职责](../Page/中华人民共和国水利部.md "wikilink")，[农业部的草原资源调查和确权登记管理职责](../Page/中华人民共和国农业部.md "wikilink")，[国家林业局](../Page/国家林业局.md "wikilink")的森林、湿地等资源调查和确权登记管理职责，[国家海洋局](../Page/国家海洋局.md "wikilink")的职责，国家测绘地理信息局的职责整合，组建自然资源部，作为国务院组成部门。自然资源部对外保留国家海洋局牌子。不再保留[国土资源部](../Page/中华人民共和国国土资源部.md "wikilink")、[国家海洋局](../Page/国家海洋局.md "wikilink")、[国家测绘地理信息局](../Page/国家测绘地理信息局.md "wikilink")。”\[9\]\[10\]\[11\]2018年4月10日，自然资源部正式挂牌\[12\]。自然资源部的成立为各项规划的整合扫清了管理上的障碍。

2019年5月9日，中共中央、国务院印发《》（中发〔2019〕18号），确立了了国土空间规划的改革目标。\[13\]同年5月28日，自然资源部发布《自然资源部关于全面开展国土空间规划工作的通知》（自然资发〔2019〕87号）。通知提到，“各地不再新编和报批主体功能区规划、土地利用总体规划、[城镇体系规划](../Page/城镇体系规划.md "wikilink")、[城市（镇）总体规划](../Page/中华人民共和国城市总体规划.md "wikilink")、海洋功能区划等。已批准的规划期至2020年后的省级国土规划、城镇体系规划、主体功能区规划，城市（镇）总体规划，以及原省级空间规划试点和市县‘多规合一’试点等，要按照新的规划编制要求，将既有规划成果融入新编制的同级国土空间规划中。”\[14\]

## 主要内容

### “五级三类四体系”

2019年5月27日，[国务院新闻办公室举行](https://zh.wikipedia.org/wiki/中华人民共和国国务院新闻办公室 "wikilink")《中共中央 国务院关于建立国土空间规划体系并监督实施的若干意见》发布会。自然资源部总规划师[庄少勤在回答记者提问时介绍](https://zh.wikipedia.org/wiki/庄少勤 "wikilink")，国土空间规划层级和内容可归纳为“五级三类四体系”。其中：

  - “五级”对应[中国的行政管理体系的五个层级](../Page/中华人民共和国行政区划.md "wikilink")：国家级、[省级](../Page/中国一级行政区.md "wikilink")、[市级](../Page/中华人民共和国城市建制.md "wikilink")、[县级](../Page/县级行政区.md "wikilink")、[乡镇级](https://zh.wikipedia.org/wiki/乡级行政区 "wikilink")。其中国家级规划侧重战略性，省级规划侧重协调性，市县级和乡镇级规划侧重实施性。
  - “三类”分为总体规划、详细规划、相关的专项规划三种规划类型。
      - 总体规划强调的是规划的综合性，是对一定区域，如行政区全域范围涉及的国土空间保护、开发、利用、修复做全局性的安排。
      - 详细规划强调实施性，一般是在市县以下组织编制，是对具体地块用途和开发强度等作出的实施性安排。详细规划是开展国土空间开发保护活动，包括实施国土空间用途管制、核发城乡建设项目规划许可，进行各项建设的法定依据。城镇开发边界外的村庄规划作为详细规划。
      - 相关的专项规划强调的是专门性，一般是由自然资源部门或者相关部门来组织编制，可在国家级、省级和市县级层面进行编制，特别是对特定的区域或者流域，为体现特定功能对空间开发保护利用作出的专门性安排。
  - “四体系”包括规划流程中的规划编制审批体系、规划实施监督体系，以及支撑规划运行的法规政策体系和[技术标准体系](../Page/中华人民共和国国家标准.md "wikilink")。\[15\]

### “双评价”

国土空间规划的“双评价”即[资源环境承载能力和国土空间开发适宜性评价](https://zh.wikipedia.org/wiki/环境承载力 "wikilink")，是编制国土空间规划的基本依据。根据自然资源部制定的《资源环境承载能力和国土空间开发适宜性评价技术指南（征求意见稿）》，“资源环境承载能力”是指“一定国土空间内自然资源、环境容量和生态服务功能对人类活动的综合支撑水平”，而“国土空间开发适宜性”是指“国土空间对生态保护、农业生产、城镇建设等不同开发保护利用方式的适宜程度”。

### “三区三线”

“三区三线”中的“三区”是指生态、农业、城镇功能空间，“三线”是“三条控制线”，即生态保护红线、永久基本农田、城镇开发边界空间管控边界。\[17\]其中“三区”突出主导功能划分，“三线”侧重边界的刚性管控。\[18\]

三条控制线的定义分别如下：

  - 生态保护红线：在生态空间范围内具有特殊重要生态功能、必须强制性严格保护的区域。
  - 永久基本农田：为保障国家粮食安全和重要农产品供给，实施永久特殊保护的耕地。依据耕地现状分布，根据耕地质量、粮食作物种植情况、土壤污染状况，在严守耕地红线基础上，按照一定比例，将达到质量要求的耕地依法划入。已经划定的永久基本农田中存在划定不实、违法占用、严重污染等问题的要全面梳理整改，确保永久基本农田面积不减、质量提升、布局稳定。
  - 城镇开发边界：在一定时期内因城镇发展需要，可以集中进行城镇开发建设、以城镇功能为主的区域边界，涉及城市、建制镇以及各类开发区等。\[19\]

“三区”和“三线”的对应关系如下：

| “三区”（三类空间）  | “三线”（三条控制线）及其他 |
| ----------- | -------------- |
| 生态空间        | 生态保护红线         |
| 自然保护地一般控制区  |                |
| 生态保护红线内其他区域 |                |
| 其他生态空间      |                |
| 农业空间        | 永久基本农田         |
| 其他农业空间      |                |
| 城镇空间        | 城镇开发边界         |
| 其他城镇空间      |                |

“三区三线”需要根据“双评价”的结果布局、划定。\[20\]

### 审查要点

根据《自然资源部关于全面开展国土空间规划工作的通知》，省级国土空间规划审查要点包括：

1.  国土空间开发保护目标；
2.  国土空间开发强度、建设用地规模，生态保护红线控制面积、自然岸线保有率，耕地保有量及[永久基本农田保护面积](https://zh.wikipedia.org/wiki/永久基本农田 "wikilink")，用水总量和强度控制等指标的分解下达；
3.  主体功能区划分，、生态保护红线、永久基本农田的协调落实情况；
4.  [城镇体系](../Page/城镇体系.md "wikilink")布局，[城市群](https://zh.wikipedia.org/wiki/城市群 "wikilink")、[都市圈](../Page/都市圈.md "wikilink")等区域协调重点地区的空间结构；
5.  生态屏障、[生态廊道和](https://zh.wikipedia.org/wiki/生态廊道 "wikilink")[生态系统](../Page/生态系统.md "wikilink")保护格局，重大基础设施网络布局，城乡公共服务设施配置要求；
6.  体现地方特色的自然保护地体系和历史文化保护体系；
7.  乡村空间布局，促进乡村振兴的原则和要求；
8.  保障规划实施的政策措施；
9.  对市县级规划的指导和约束要求等。\[21\]

国务院审批的市级国土空间总体规划审查要点，除对省级国土空间规划审查要点的深化细化外，还包括：

1.  市域国土空间规划分区和用途管制规则；
2.  重大交通枢纽、重要线性工程网络、城市安全与综合防灾体系、地下空间、[邻避设施等设施布局](https://zh.wikipedia.org/wiki/邻避 "wikilink")，城镇政策性住房和教育、卫生、养老、文化体育等城乡公共服务设施布局原则和标准；
3.  城镇开发边界内，城市结构性绿地、水体等[开敞空间的控制范围和均衡分布要求](https://zh.wikipedia.org/wiki/开敞空间 "wikilink")，各类历史文化遗存的保护范围和要求，通风廊道的格局和控制要求；城镇开发强度分区及[容积率](../Page/容积率.md "wikilink")、密度等控制指标，高度、风貌等空间形态控制要求；
4.  [中心城区城市功能布局和用地结构等](https://zh.wikipedia.org/wiki/中心城区 "wikilink")。\[22\]

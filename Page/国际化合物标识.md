## 层

国际化合物标识分为六个层，每个层之间以“/”分隔，并在开头以一个小写字母标明层的性质：

1.  主层：以“1”表示
2.  [电荷层](../Page/电荷.md "wikilink")：以“q”表示
3.  [立体化学层](../Page/立体化学.md "wikilink")：以“t”，“m”，“s”表示
4.  [异构体层](../Page/异构.md "wikilink")（Isotopic layer）：以“i”表示
5.  固定氢原子（Fixed-H layer）：以“f”表示
6.  再连接层（Reconnected Layer）：以“r”表示

其中主层是必不可少的，其他的层是可以省略的。

### 子层

每个层都可以分成若干子层（sub-layer），比如主层可以分成一下三个子层：

1.  [分子式子层](../Page/分子式.md "wikilink")（没有前缀）
2.  [原子连接子层](../Page/原子.md "wikilink")（以字母c开始）
3.  氢原子层（以字母h开始）

以维生素C的InChI为例

| 化学式层     | 原子连接层                          | 氢原子层            | 立体化学层         |

### 軟件和服務

  - [NCI/CADD Chemical Identifier
    Resolver](http://cactus.nci.nih.gov/chemical/structure) Generates
    and resolves InChI/InChIKeys and many other chemical identifiers
  - [ChemSpider InChI
    resolver](http://www.chemspider.com/inchi%2Dresolver/)
  - [Search Google for
    molecules](https://web.archive.org/web/20060117153546/http://wwmm.ch.cam.ac.uk/gridsphere/gridsphere?cid=inchigoogle)
    (generates InChI from interactive chemical and searches Google for
    any pages with embedded InChIs). Requires Javascript enabled on
    browser
  - [ChemSketch](http://www.acdlabs.com/download/chemsk.html), free
    chemical structure drawing package that includes input and output in
    InCHI format
  - [PubChem online molecule
    editor](http://pubchem.ncbi.nlm.nih.gov/edit/index.html) that
    supports [SMILES](../Page/SMILES.md "wikilink")/SMARTS and InChI
  - \[<http://www.chemspider.com/Services.aspx>? ChemSpider Services\]
    that allows generation of InChI and conversion of InChI to structure
    (also SMILES and generation of other properties)
  - [MarvinSketch](https://web.archive.org/web/20070404073952/http://www.chemaxon.com/demosite/marvin/index.html)
    from [ChemAxon](../Page/ChemAxon.md "wikilink"), implementation to
    draw structures (or open other file formats) and output to InChI
    file format
  - [BKchem](http://bkchem.zirael.org/inchi_en.html) implements its own
    InChI parser and uses the IUPAC implementation to generate InChI
    strings
  - [CompoundSearch](http://www.compoundsearch.com) implements an InChI
    and InChI Key search of spectral libraries
  - [JNI-InChI](http://jni-inchi.sourceforge.net/) Java library that
    wraps the InChI library
  - the [Chemistry Development
    Kit](../Page/Chemistry_Development_Kit.md "wikilink") uses JNI-InChI
    to generate InChIs, can convert InChIs into structures, and generate
    tautomers based on the InChI algorithms
  - [Bioclipse](../Page/Bioclipse.md "wikilink") generates InChI and
    InChIKeys for drawn structures or opened files

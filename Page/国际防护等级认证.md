|条目中关于ISO 20653的内容}}  **國際防護等級認證**（**International Protection Marking**, IEC 60529）亦稱作**异物防护等级**（Ingress Protection Rating）或**IP代码**（IP Code）\[1\] \[2\]。有时候也被叫做“防水等级”“防尘等级”等，\[3\]定义了机械和电子设备能提供针对固态异物侵入（包括身体部位如手指，灰尘，砂砾等），液态渗入，意外接触有何等程度的防护能力。\[4\]

这个标准着重给用户提供更详尽的产品信息，而不只是模糊的营销词汇，比如“防水”等。不过，该标准并未公开发布。

代码内的数字表示符合下表中的情况。而当不符合任一种情况时，该位数字被字母"X"替代。

例如，一个标称**IP22**的[插座可以有效防止手指的插入而不会损害](https://zh.wikipedia.org/wiki/插座 "wikilink")，并且也不会在垂直或者近似于垂直的情况下由于滴水而变得危险。对于这类室内插座最低的设计标准就是IP22或者IP2X。标称**IP67**的手機可以完全防尘，並可放入最深1m的水里，同時亦擁有抵挡高壓水柱的能力。

### 固态微粒防护

第一位数字用来表示产品外壳用来保护内部组件（比如电子与机械零件）和防御外部物体进入的能力。

| 等级 | 可防護下列直徑之球體完全進入箱體內 | 描述                                           |
| -- | ----------------- | -------------------------------------------- |
| 0  | —                 | 等同於直接暴露，无法保护接触与外物入侵。                         |
| 1  | \>50mm            | 可避免身體上任意大表面接觸，但无法阻擋刻意以身體某部位接触，例如手掌、手背。       |
| 2  | \>12.5mm          | 可阻擋手指大小或相似大小之物体。                             |
| 3  | \>2.5mm           | 能阻止螺丝起子等工具進入。                                |
| 4  | \>1mm             | 隔絕多数电线伸入与細小尖端的工具、或螞蟻等爬入。                     |
| 5  | 防尘                | 并不完全防御灰尘进入，但必须有足够的数量才能对设备的正常运作造成影响，并且完全防止接触。 |
| 6  | 完全防尘              | 灰尘无法进入，完全防止接触。                               |

### 额外字母

标准中可加入额外字母来标明可人为环境下与有害物品接触：

| 等级 | 防止有害物体接触    |
| -- | ----------- |
| A  | 手掌          |
| B  | 手指          |
| C  | 直徑 2.5mm 工具 |
| D  | 直徑 1mm 电线   |

还可以加入额外的字母来标示设备的防护信息：

| 字母 | 含义           |
| -- | ------------ |
| H  | 高电压设备        |
| M  | 设备在水中测试时保持移动 |
| S  | 设备在水中测试时静止   |
| W  | 天气条件         |

## 示例

如果一部手机拥有 **IP67**的防护能力。按照上表，在不破坏外壳的情况下，6表示该手机能完全保护内部电路不受灰尘入侵；7表示该手机在浸入不超过1m深的水下时，能给予内部零件不低于30分钟的防护时间（并不表示完全没有水进入内部）。

## IPX代码

代码各位中的X字母用来表示应避免该项中的情况。选择该代码有各种因素，比如营销方面的考虑。因此，比如一只IPX7的消费产品表示该设备针对有限的浸水能提供防护能力，但有意不给出该设备针对异物入侵或灰尘方面的防护信息。

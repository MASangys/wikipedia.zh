**图瓦卢语**是[图瓦卢使用的一种](https://zh.wikipedia.org/wiki/图瓦卢 "wikilink")[波利尼西亚语言](https://zh.wikipedia.org/wiki/波利尼西亚 "wikilink")。它与[萨摩亚语](https://zh.wikipedia.org/wiki/萨摩亚语 "wikilink")、[毛利语](../Page/毛利语.md "wikilink")、[夏威夷语](../Page/夏威夷语.md "wikilink")、[塔希提语是远亲](https://zh.wikipedia.org/wiki/塔希提语 "wikilink")，而与[密克罗尼西亚与](https://zh.wikipedia.org/wiki/密克罗尼西亚 "wikilink")[美拉尼西亚](../Page/美拉尼西亚.md "wikilink")中部和北部的[波利尼西亚语相近](https://zh.wikipedia.org/wiki/波利尼西亚语 "wikilink")。图瓦卢语从19世纪晚期和20世纪早期基督教传教士的语言——萨摩亚语借入了大量词汇。图瓦卢语总共约有13000名使用者。图瓦卢语有5个元音（i, e, a, o, u），10或11个辅音（p, t, k, m, n, g, f, v, s, h, l），由不同方言决定。该语言有4个冠词，分南北两大方言群。

## 構詞

| 標記       | 功用/語義                        |
| -------- | ---------------------------- |
| *e*      | 現在示標記                        |
| *ka*     | 未來式標記                        |
| *kai*    | '曾經'                         |
| *ke*     | 1\. '將 (祈使式) 2. '和', '以致...' |
| *ke na*  | 祈使式 (禮貌)                     |
| *ko*     | 現在完成式標記                      |
| *koi*    | '仍然' (連續動作)                  |
| *ko too* | '也'                          |
| *o*      | '及', '至' (兩個動詞間的連接詞)         |
| *ma*     | '唯恐, (某事會...)'               |
| *mana*   | '唯恐, (某事會發生)'                |
| *moi*    | '要是...就好了'                   |
| *ne*     | 過去式標記                        |
| (無標記)    | 祈使命令式                        |

## 範例

E fā'nau mai a tino katoa i te saolotoga kae e 'pau telotou tūlaga fakaaloalogina mo telotou aiā. Ne tuku atu ki a lātou a te mafaufau mo te loto lagona, tēlā lā, e 'tau o gā'lue fakatasi lātou e pēlā me ne taina.

漢譯：人皆生而自由；在尊嚴及權利上均各平等。人各賦有理性良知，誠應和睦相處，情同手足。

以下**國立中興大學**校友包含「台中校區」以及「台北校區」兩校區的校友。
台中校區為**國立中興大學校本部**，台北校區為**國立中興大學法商學院**。
台北校區於西元2000年改制為國立臺北大學，依據歷史脈絡，共同隸屬於中興大學與台北大學校友。

## 興大簡史

| 年代               | width="10%" align ="center"|簡要大事紀                                       |
| ---------------- | ----------------------------------------------------------------------- |
| 民國五　十年(1961)七月一日 | 省立農學院（臺中校區）與省立法商學院（臺北校區）,以及新設的理工學院(臺中校區)三大學院整併為「臺灣省立中興大學」。              |
| 民國五十八年(1969)八月一日 | 成立文學院。                                                                  |
| 民國六　十年(1971)七月一日 | 改隸中央，改制為「[國立中興大學](../Page/國立中興大學.md "wikilink")」。                       |
| 民國七十七年(1988)八月一日 | 理工學院改制為理學院及工學院。                                                         |
| 民國八十三年(1994)八月一日 | 整合原屬理學院的動物學系、植物學系及分子生物學研究所，成立生命科學院。                                     |
| 民國八十五年(1996)一月五日 | 成立國立中興大學校友聯絡中心。                                                         |
| 民國八十八年(1999)八月一日 | 成立獸醫學院。                                                                 |
| 民國八十九年(2000)二月一日 | 臺北校區的法商學院改制為「[國立臺北大學](../Page/國立臺北大學.md "wikilink")」；臺中校區設立「社會科學暨管理學院」。 |
| 民國一　百年(2011)八月一日 | 國立中興大學「社會科學暨管理學」改制為「管理學院」及「法政學院」。                                       |

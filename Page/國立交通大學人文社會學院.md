**國立交通大學人文社會學院**(College of Humanities and Social Sciences, National Chiao Tung University)是[國立交通大學](../Page/國立交通大學.md "wikilink")的其中一個學院，旨在培育具創造性、判性之人文社會領域人才，目標為強化亞太地區的相關學術研究，成為與他國交流、合作的平台。\[1\]

## 歷史

[交大自](../Page/國立交通大學.md "wikilink")1958年在[臺灣](../Page/臺灣.md "wikilink")[新竹復校後](../Page/新竹縣.md "wikilink")，一直以[電子](https://zh.wikipedia.org/wiki/電子 "wikilink")、資通訊和[光電等](https://zh.wikipedia.org/wiki/光電 "wikilink")[理工](../Page/理工.md "wikilink")科系馳名，成立涵蓋音樂、藝術、人文社會、教育、傳播等學科的學院除了平衡了學校長期的學術輸出不均衡，也使得大眾對於學校的印象漸漸被改寫。人文社會學院於1995年成立，目前設有1學系、8碩士班、3博士班、2研究中心如下:

| 名稱            | 設立時間 | 備註                                  |
| ------------- | ---- | ----------------------------------- |
| 外國語文學系        | 1994 |                                     |
| 外國文學與語言學碩士班   | 2000 |                                     |
| 傳播研究所碩士班      | 1991 | 分為**傳播管理與互動科技**、**傳播與科技**兩組         |
| 應用藝術研究所碩、博士班  | 1992 | 分為**工業設計**、**視覺傳達設計**、**傳播藝術與科技**三組 |
| 音樂研究所碩士班      | 2000 | 分為**演奏**、**音樂學**、**創作**三組           |
| 建築研究所碩士班      | 2000 | 分為**學士後建築組**、**建築設計**、**數位設計**三組    |
| 教育研究所碩、博士班    | 2001 | 分為**教育心理諮商**、**科學教育**、**數位學習**三組    |
| 社會與文化研究所碩、博士班 | 2002 |                                     |
| 英語教學研究所碩士班    | 2004 |                                     |
| 語言教學與研究中心     | 1998 |                                     |
| 師資培育中心        | 1996 |                                     |

學院另外有與[台聯大系統其他三間學校合作設立](https://zh.wikipedia.org/wiki/台灣聯合大學系統 "wikilink")「亞際文化研究國際碩士學程」

## 榮譽

  - 跨領域設計科學研究團隊（TDIS）於2016年提案參加「中東盃太陽能十項全能綠建築競賽（Solar Decathlon Middle East, SDME 2018）」，其設計藍圖獲評審團青睞，隨後代表[臺灣](../Page/臺灣.md "wikilink")在打造創意行動基地（Creative Action Base, CAB）競賽中晉級，將與其他14間頂尖大學團隊爭奪第一名。\[2\]

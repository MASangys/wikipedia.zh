**國立員林高級中學**（），簡稱**員林高中**、**員中**、**員高**，舊稱**省員中**，為一所位於[臺灣](../Page/臺灣.md "wikilink")[彰化縣](../Page/彰化縣.md "wikilink")[員林市的國立普通型高級中等學校](../Page/員林市.md "wikilink")。

## 學校象徵

### 校徽

十年樹木　百年之樹人

### 校訓

勤奮樸實

### 校史

| **時間**     | **事件**                                                                                                                          |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------- |
| 1937年4月1日  | 該校創立，校名為『**臺中州立員林家政女學校**』。                                                                                                      |
| 1944年      | 改制為「**員林農業實踐學校**」。                                                                                                              |
| 1945年      | [中華民國接管臺灣](../Page/臺灣戰後時期.md "wikilink")，該校由政府接管並派張火樹先生擔任校長。                                                                    |
| 1946年4月12日 | 改制為『**臺中縣立員林初級中學**』，校長為李嘉嵩先生。                                                                                                   |
| 1949年8月    | 改制為『**臺灣省立員林中學**』，增設高中部，校長為張致祥先生。                                                                                               |
| 1951年8月    | 由[洪樵榕先生接掌員林高中](../Page/洪樵榕.md "wikilink")，六年任內樹立員林高中卓著的聲譽。                                                                      |
| 1957年7月    | 洪校長當選南投縣縣長去職，由廖五湖先生接任。                                                                                                          |
| 1965年7月    | 廖校長調職，由羅旭升先生繼任。                                                                                                                 |
| 1968年      | 政府實施九年國民教育，初中部停止招收新生。                                                                                                           |
| 1969年2月    | 羅校長調職，由熊惠民先生接任。                                                                                                                 |
| 1970年6月    | 初中部結束，同年8月1日更名為『**臺灣省立員林高級中學**』。                                                                                                |
| 1971年2月    | 熊校長調職，由王成荃先生接任。                                                                                                                 |
| 1982年      | 王校長退休，由潘星照先生接任。                                                                                                                 |
| 1989年2月1日  | 潘校長退休，由郎致運先生接任。                                                                                                                 |
| 1991年2月1日  | 郎校長退休，廳特派曾任[台中師院教授](../Page/台中師院.md "wikilink")、教務長、訓導長之[簡蒼調教授接任](../Page/簡蒼調.md "wikilink")。                                   |
| 1997年8月    | 簡校長調[台中高農](../Page/台中高農.md "wikilink")，由曾任省立[楊梅高中校長](../Page/楊梅高中.md "wikilink")、[員林家商教務主任孔建國先生接任](../Page/員林家商.md "wikilink")。 |
| 2000年2月    | 改制國立，更名為『**國立員林高級中學**』。                                                                                                         |
| 2002年      | 增設美術資優班。                                                                                                                        |
| 2004年      | 成立語言資優班。總班級數為五十一班。                                                                                                              |
| 2004年8月    | 孔建國校長調彰化女中，由教務主任黃千在代理校長。                                                                                                        |
| 2005年2月    | [北斗家商教務主任粘淑真經校長遴選後接任校長](../Page/北斗家商.md "wikilink")。                                                                            |
| 2006年8月    | 獲行政院國科會高瞻計畫補助。                                                                                                                  |
| 2007年4月1日  | 舉辦慶祝創校七十週年校慶各項活動。                                                                                                               |
| 2008年4月9日  | 高三教室裝設冷氣招標案,於4月9日順利發包,4月30日前全面完工驗收啟用。                                                                                           |
| 2008年4月29日 | 日本愛知縣、崎阜縣觀光協會官員來訪。                                                                                                              |
| 2017年4月8日  | 舉辦慶祝創校八十週年校慶各項活動。                                                                                                               |

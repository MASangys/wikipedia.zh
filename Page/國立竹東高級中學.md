**國立竹東高級中學**，簡稱**竹東高中**，是一所位於[臺灣](../Page/臺灣.md "wikilink")[新竹縣](../Page/新竹縣.md "wikilink")[竹東鎮的國立高中](../Page/竹東鎮.md "wikilink")。學校環境優美，充滿客家風情，校門口相當宏偉壯麗有著兩道長長的階梯，故學生常以樓梯、階梯作為竹東高中之代表。

## 校史

| 時間        | 事件                                                                                                                                         |
| --------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| 1946年5月6日 | 成立**新竹市立新竹初級中學竹東分校**。                                                                                                                      |
| 1948年     | 獨立，改名為**新竹市立竹東初級中學**。                                                                                                                      |
| 1950年     | 政府重劃地方行政區域，隸屬為[新竹縣](../Page/新竹縣.md "wikilink")，改為**新竹縣立竹東初級中學**。                                                                           |
| 1951年8月   | 因應學校發展，遷建於[竹東鎮](../Page/竹東鎮.md "wikilink")[鳳陽山麓](../Page/鳳陽山.md "wikilink")，並且試辦高中。                                                        |
| 1953年     | 增設高中部，成為完全中學，並改名為**臺灣省立竹東高級中學**，並設立北埔分校（今[北埔國中](../Page/新竹縣立北埔國民中學.md "wikilink")）。                                                        |
| 2000年2月1日 | 因應[精省](../Page/精省.md "wikilink")，臺灣的省立學校一律改隸屬[中華民國行政院](../Page/中華民國行政院.md "wikilink")[教育部](../Page/中華民國教育部.md "wikilink")，更名為**國立竹東高級中學**。 |

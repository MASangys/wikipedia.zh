**國語傳習所**是1896年到1898年，[台灣日治時期](../Page/台灣日治時期.md "wikilink")的[台灣總督府於](https://zh.wikipedia.org/wiki/台灣總督府 "wikilink")[台灣實施的基礎教育學制與場所](https://zh.wikipedia.org/wiki/台灣 "wikilink")。此學制或場所的制定或設立，確定[日本](../Page/日本.md "wikilink")統治者對台灣在「放任驅逐」與「同化」兩教育政策中，選擇了後者。國語傳習所也成為台灣首個西式教育的學校，更成為後來[公學校](../Page/公學校.md "wikilink")的前身。因當時台灣已屬日本領土，國語傳習所一詞中的「國語」即為[日本語](https://zh.wikipedia.org/wiki/日本語 "wikilink")。

## 背景

1895年[日本](../Page/日本.md "wikilink")接收台灣後，積極就台灣教育制度研擬未來發展方向。在設立[芝山巖學務部學堂](../Page/芝山巖學務部學堂.md "wikilink")受阻與[六氏先生](../Page/六氏先生.md "wikilink")事件後，台灣總督府並沒有放棄普及日語的施政重點。根據台灣總督府第一任學務部長[伊澤修二](../Page/伊澤修二.md "wikilink")發表的《台灣教育意見書》表示，日本統治台灣之後，最重要工作正是普及日語，因此，所有對台灣總督府的教育制度研擬，也繼續以此為目標。

1896年6月22日，臺灣總督府依據1896年3月31日公布的「臺灣總督府直轄諸學校官制」（敕令第94號）發布「國語傳習所規則」（府令第15號）\[1\]\[2\]，確定在全台各地教導日語的學習場所。其學制基本上仿照西式教育，教授國語，唱遊，數理。

1896年9月總督府公布《[臺灣總督府國語學校](../Page/臺灣總督府國語學校.md "wikilink")規則》。

傳習所設立之初，台人參加者並不踴躍。不過總督府以免學費為號召，加上台灣人民生活的日語需求越來越多，所以參與的台籍學童越來越多。1898年，台灣總督府將實施頗有成效的國語傳習所改制為[公學校](../Page/公學校.md "wikilink")。

## 課程

國語傳習所分成甲乙兩科，其差異如下：

| 科別 | 招生年齡                 | 學習期限 | 科目                                                    |
| -- | -------------------- | ---- | ----------------------------------------------------- |
| 甲  | 15歲以上，30歲以下（已略有普通知識） | 半年   | 日語、初步的讀書作文                                            |
| 乙  | 8歲以上，15歲以下           | 四年   | 日語、讀書、作文、習字、算數。另選設漢文、地理、歷史、唱歌、體操等科中的一科或數科。女性可再加設裁縫一科。 |
|    |                      |      |                                                       |

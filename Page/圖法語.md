**圖法語**(Tofa、Tofalar、Karagas)，是[突厥語族西伯利亞語支](https://zh.wikipedia.org/wiki/突厥語族 "wikilink")，流通於俄羅斯西伯利亞[伊爾庫茨克州的](https://zh.wikipedia.org/wiki/伊爾庫茨克州 "wikilink")[圖法拉爾人](https://zh.wikipedia.org/wiki/圖法拉爾人 "wikilink")，是一種瀕危語言，在2010年只有93人持此語言。

圖法語與[圖瓦語關係密切](https://zh.wikipedia.org/wiki/圖瓦語 "wikilink")，一般認為是圖瓦語方言。

## 詞法和句法

### 代詞

圖法語有六個人稱代詞：

| 單數        | 複數       |
| --------- | -------- |
| 圖法語(音譯)   | 英語       |
| мен (men) | 我        |
| сен (sen) | 你 *(單數)* |
| оң (oŋ)   | 他/她/它    |

Personal 代詞

圖法語也有這些代詞 бо "這個", тээ "那個", кум "誰"，及 чү "什麼"。

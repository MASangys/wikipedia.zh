| BC1200      | 爱琴海沿岸[特洛伊战争](../Page/特洛伊战争.md "wikilink")发生，也就是木马屠城记。                                                                                                                                                                                                                                                                                              |
| BC1300－600  | 入侵这块土地并建立文明的有[弗里吉亚](../Page/弗里吉亚.md "wikilink")人（Phrygia）、[呂底亞](../Page/呂底亞.md "wikilink")人（Lydia），这个时期为希腊化时间（Hellenic）。                                                                                                                                                                                                                           |
| BC550       | 波斯的[大流士大帝入侵统治](https://zh.wikipedia.org/wiki/大流士大帝 "wikilink")。                                                                                                                                                                                                                                                                                    |
| BC334       | [马其顿王国的](https://zh.wikipedia.org/wiki/马其顿王国 "wikilink")[亚历山大大帝](../Page/亚历山大大帝.md "wikilink")统治。                                                                                                                                                                                                                                                  |
| BC323－129   | [亚历山大大帝](../Page/亚历山大大帝.md "wikilink")去世后，他的帝国分裂成好几个国家，[安那托利亞](../Page/安那托利亞.md "wikilink")地方也四分五裂。                                                                                                                                                                                                                                                |
| BC129       | [罗马帝国在](https://zh.wikipedia.org/wiki/罗马帝国 "wikilink")[安那托利亚建立一个省](https://zh.wikipedia.org/wiki/安那托利亚 "wikilink")，成为罗马帝国的一部分，首都为[以弗所](../Page/以弗所.md "wikilink")（Ephesus）。[丝路以这里为连接欧洲的桥梁](https://zh.wikipedia.org/wiki/丝路 "wikilink")。                                                                                                           |
| 公元330       | 东、西罗马帝国开始分裂，君士坦丁大帝迁都从罗马迁至君士坦丁堡（今天的伊斯坦堡）。                                                                                                                                                                                                                                                                                                           |
| 公元395       | [东罗马帝国被称为](https://zh.wikipedia.org/wiki/东罗马帝国 "wikilink")[拜占庭帝国](../Page/拜占庭帝国.md "wikilink")，东西罗马帝国彻底分裂。                                                                                                                                                                                                                                         |
| 公元527－565   | [查士丁尼一世](../Page/查士丁尼一世.md "wikilink")在位，是拜占庭帝国文治武功达最盛时期，建圣索菲亚大教堂。                                                                                                                                                                                                                                                                                 |
| 公元570－622   | [穆罕默德](../Page/穆罕默德.md "wikilink")诞生，受天启写[古兰经](../Page/古兰经.md "wikilink")。                                                                                                                                                                                                                                                                         |
| 公元1071－1243 | 突厥塞尔柱人（Selcuk）人在[科尼亞](../Page/科尼亞_\(土耳其\).md "wikilink")（Konya）建罗姆苏丹国，版图包括土耳其、伊朗、伊拉克。塞尔柱人以建筑及工艺闻名，并有著名的[梅夫拉維教團](https://zh.wikipedia.org/wiki/梅夫拉維教團 "wikilink")（旋转舞创始者）。此帝国后被蒙古人消灭。                                                                                                                                                               |
| 公元1288      | 另一支突厥集团在首领奥斯曼的领导下，于[布尔萨](../Page/布尔萨.md "wikilink")(Bursa)附近建立鄂图曼帝国，势力日益强大。                                                                                                                                                                                                                                                                        |
| 公元1453      | [穆罕默德二世](../Page/穆罕默德二世_\(奥斯曼帝国\).md "wikilink")(Mehmet II)率军占领君士坦丁堡并改名为伊斯坦堡，逐渐发展成当时欧洲的最大城市。                                                                                                                                                                                                                                                       |
| 公元1520－1566 | 奥斯曼帝国在[素檀](https://zh.wikipedia.org/wiki/素檀 "wikilink")(Sultan)[苏萊曼一世](../Page/苏莱曼一世_\(奥斯曼帝国\).md "wikilink")(Suleman I)的领导下，文治武功达最盛期，地跨欧、亚、非、地中海、红海、印度洋皆在其管辖区。                                                                                                                                                                                    |
| 公元1529年     | 土耳其人打到[奥地利](https://zh.wikipedia.org/wiki/奥地利 "wikilink")[维也纳城门下](https://zh.wikipedia.org/wiki/维也纳 "wikilink")，但攻城不克而返。                                                                                                                                                                                                                           |
| 公元1914      | 第一次世界大战爆发，奥斯曼帝国加入[同盟国](https://zh.wikipedia.org/wiki/同盟国 "wikilink")，对英、法宣战。                                                                                                                                                                                                                                                                       |
| 公元1918      | 同盟国战败，战胜国欲瓜分土耳其。                                                                                                                                                                                                                                                                                                                                   |
| 公元1923      | [穆斯塔法·凯末尔率领土耳其人抵抗外侮](https://zh.wikipedia.org/wiki/穆斯塔法·凯末尔 "wikilink")，宣布土耳其共和国正式成立。时代杂志曾刊出在这一千年（1000－1999）中最重要的二十件历史大事，其中关于土耳其鄂图曼帝国的就有两项。                                                                                                                                                                                                        |
| 公元1932      | [土耳其国父](https://zh.wikipedia.org/wiki/土耳其国父 "wikilink")，穆斯塔法·凯末尔建立[土耳其语言研究中心](https://zh.wikipedia.org/wiki/土耳其语言研究中心 "wikilink")，其目的用于简化土耳其文字，减少[阿拉伯语和](https://zh.wikipedia.org/wiki/阿拉伯语 "wikilink")[波斯语](../Page/波斯语.md "wikilink")的使用，并大量借鉴[英语](../Page/英语.md "wikilink")和[法语](../Page/法语.md "wikilink")，从而创造出现如今的土耳其语。现约有75-80%的土耳其文字是那时创造的。 |

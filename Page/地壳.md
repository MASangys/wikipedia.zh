在地理上，**地殼**（）是指一个[星球最外層的實心薄殼](https://zh.wikipedia.org/wiki/星球 "wikilink")，可以用化學方法将它与[地幔](../Page/地幔.md "wikilink")區别。[地球](../Page/地球.md "wikilink")，[月球](../Page/月球.md "wikilink")，[水星](../Page/水星.md "wikilink")，[金星](../Page/金星.md "wikilink")，[火星](../Page/火星.md "wikilink")以及其它星球的地殼大部分都是由火成岩形成的，星球的地殼比起它们的地幔有更多的[不相容成分](https://zh.wikipedia.org/wiki/不相容成分 "wikilink")。

## 地球地壳

[Earth-crust-cutaway-chinese.png](https://zh.wikipedia.org/wiki/File:Earth-crust-cutaway-chinese.png "fig:Earth-crust-cutaway-chinese.png") **地球地壳**是指[地球](../Page/地球.md "wikilink")[地表至](https://zh.wikipedia.org/wiki/地表 "wikilink")[莫霍界面之间一个主要由](https://zh.wikipedia.org/wiki/莫霍界面 "wikilink")[火成岩](../Page/火成岩.md "wikilink")，[变质岩](../Page/变质岩.md "wikilink")和[沉积岩](../Page/沉积岩.md "wikilink")构成的薄壳，是[岩石圈](../Page/岩石圈.md "wikilink")组成的一部分，平均厚度17公里，地壳下面的是[地幔](../Page/地幔.md "wikilink")，上地幔大部分由[橄榄石](../Page/橄榄石.md "wikilink")（一種比普通岩石密度大很多的岩石）构成。地壳和地幔之间的分界线被称为[莫氏不连续面](https://zh.wikipedia.org/wiki/莫氏不连续面 "wikilink")。地壳的[质量](../Page/质量.md "wikilink")只占全地球0.2%，按结构分为[大陆地壳和](https://zh.wikipedia.org/wiki/大陆地壳 "wikilink")[海洋地壳两种](https://zh.wikipedia.org/wiki/海洋地壳 "wikilink")。大陆地壳有硅酸铝层（[花岡岩質](https://zh.wikipedia.org/wiki/花岡岩 "wikilink")）和硅酸镁层（[玄武岩](../Page/玄武岩.md "wikilink")質）双层结构，而海洋地壳只有硅酸镁层（玄武岩質）单层结构，大陆地壳平均厚度有33公里，海洋地壳平均厚度只有10公里。

地壳的温度随着其不断加深而逐渐升高，从200°C（392°F）到400°C（752°F）不等。

### 大陆地壳的组成

一般说来大陆地壳的组成成分是[火成岩](../Page/火成岩.md "wikilink")和[安山岩](../Page/安山岩.md "wikilink")。以下的成分表和接下来的结论大部分基于Rudnick and Gao（2003）的汇总。\[1\]大陆地壳比由[玄武岩](../Page/玄武岩.md "wikilink")构成的海洋地壳拥有更多的不相容成分，而海洋地壳又比它下面的地幔更富有不相容成分。尽管大陆地壳的硅酸盐含量仅仅占地球总含量的0.6%，但它却含有全球20%至70%的不相容成分。 [Bankuai.png](https://zh.wikipedia.org/wiki/File:Bankuai.png "fig:Bankuai.png")\]\]

| 氧化物                         | 含量（百分比） |
| --------------------------- | ------- |
| SiO<sub>2</sub>             | 60.6    |
| Al<sub>2</sub>O<sub>3</sub> | 15.9    |
| CaO                         | 6.4     |
| MgO                         | 4.7     |
| Na<sub>2</sub>O             | 3.1     |
| FeO<sub>x</sub>             | 6.7     |
| K<sub>2</sub>O              | 1.8     |
| TiO<sub>2</sub>             | 0.7     |

所有的其他成分除了水在地壳中只有很少的含量，它们的总量少于地壳总质量的1%。上地壳的平均密度介于2.69 g/cm<sup>3</sup>至2.74 g/cm<sup>3</sup>，下地壳的平均密度介于3.0 g/cm<sup>3</sup>至3.25 g/cm<sup>3</sup>\[2\]。

## 月球地壳

大碰撞说认为一个特别大的陨石在地球正在形成的时候与地球发生了碰撞，而这次撞击产生的碎片喷进入太空并最终成为了月球。当月球形成之后，它的最外层融化形成了一个“[月球岩漿海](../Page/月球岩漿海.md "wikilink")”。在这个“海洋”中，大量的[斜长石](../Page/斜长石.md "wikilink")结晶不断地向着月球表面浮动，并逐渐积累形成了月壳。上月壳大概有88%是由斜长石构成的（上月壳的底部大概有90％为斜长岩）：下月壳含有较多的铁镁矿物如辉石和橄榄石，但即使这样，下月壳仍有大概78％由斜长石构成。月壳下面的地幔也含有大量的[橄榄石](../Page/橄榄石.md "wikilink")。

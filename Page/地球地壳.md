**地球地壳**是指[地球](../Page/地球.md "wikilink")[地表至](https://zh.wikipedia.org/wiki/地表 "wikilink")[莫霍界面之间一个主要由](https://zh.wikipedia.org/wiki/莫霍界面 "wikilink")[火成岩](../Page/火成岩.md "wikilink")，[变质岩](../Page/变质岩.md "wikilink")和[沉积岩](../Page/沉积岩.md "wikilink")构成的薄壳，是[岩石圈](../Page/岩石圈.md "wikilink")组成的一部分，平均厚度17公里，地壳下面的是[地幔](../Page/地幔.md "wikilink")，上地幔大部分由[橄榄石](../Page/橄榄石.md "wikilink")（一種比普通岩石密度大很多的岩石）构成。地壳和地幔之间的分界线被称为[莫氏不连续面](https://zh.wikipedia.org/wiki/莫氏不连续面 "wikilink")。地壳的[质量](../Page/质量.md "wikilink")只占全地球0.2%，按结构分为[大陆地壳和](https://zh.wikipedia.org/wiki/大陆地壳 "wikilink")[海洋地壳两种](https://zh.wikipedia.org/wiki/海洋地壳 "wikilink")。大陆地壳有硅酸铝层（[花岡岩質](https://zh.wikipedia.org/wiki/花岡岩 "wikilink")）和硅酸镁层（[玄武岩](../Page/玄武岩.md "wikilink")質）双层结构，而海洋地壳只有硅酸镁层（玄武岩質）单层结构，大陆地壳平均厚度有33公里，海洋地壳平均厚度只有10公里。

地壳的温度随着其不断加深而逐渐升高，从200°C 到400°C 不等。

## 大陆地壳的组成

一般说来大陆地壳的组成成分是[火成岩](../Page/火成岩.md "wikilink")和[安山岩](../Page/安山岩.md "wikilink")。以下的成分表和接下来的结论大部分基于Rudnick and Gao（2003）的汇总。\[1\]大陆地壳比由[玄武岩](../Page/玄武岩.md "wikilink")构成的海洋地壳拥有更多的不相容成分，而海洋地壳又比它下面的地幔更富有不相容成分。尽管大陆地壳的硅酸盐含量仅仅占地球总含量的0.6%，但它却含有全球20%至70%的不相容成分。 [Bankuai.png](https://zh.wikipedia.org/wiki/File:Bankuai.png "fig:Bankuai.png")\]\]

| 氧化物                         | 含量（百分比） |
| --------------------------- | ------- |
| SiO<sub>2</sub>             | 60.6    |
| Al<sub>2</sub>O<sub>3</sub> | 15.9    |
| CaO                         | 6.4     |
| MgO                         | 4.7     |
| Na<sub>2</sub>O             | 3.1     |
| Fe as FeO                   | 6.7     |
| K<sub>2</sub>O              | 1.8     |
| TiO<sub>2</sub>             | 0.7     |
| P<sub>2</sub>O<sub>5</sub>  | 0.1     |
|                             |         |

所有的其他成分除了水在地壳中只有很少的含量，它们的总量少于地壳总质量的1%。上地壳的平均密度介于2.69 g/cm<sup>3</sup>至2.74 g/cm<sup>3</sup>，下地壳的平均密度介于3.0 g/cm<sup>3</sup>至3.25 g/cm<sup>3</sup>\[2\]。

[Geocaching.svg](https://zh.wikipedia.org/wiki/File:Geocaching.svg "fig:Geocaching.svg") **地理藏寶**（）是一种戶外活動，玩家會使用[全球定位系統](https://zh.wikipedia.org/wiki/全球定位系統 "wikilink")（GPS）接受器或其他[導航技術在世界上的任何地方來藏匿及找尋](https://zh.wikipedia.org/wiki/導航 "wikilink")**地理藏寶盒**。一般的地理藏寶盒是一個小型防水容器（通常是塑膠密封盒或彈藥盒），裡面放置日誌。較大的容器也可以存放交易的物品，通常是價值不高的玩具或小玩意。地理藏寶經常被描述為一個「高科技遊戲的捉迷藏」，和[定向越野](https://zh.wikipedia.org/wiki/定向越野 "wikilink")、[尋寶遊戲](../Page/尋寶遊戲.md "wikilink")和[路標遊戲](https://zh.wikipedia.org/wiki/路標遊戲 "wikilink")（waymarking）有許多相同的特點。 目前地理藏寶盒存在於超過100多個國家，遍及[南極洲在內的](https://zh.wikipedia.org/wiki/南極洲 "wikilink")[七大洲](https://zh.wikipedia.org/wiki/七大洲 "wikilink")。至2010年4月10日，世界上有超過1,031,429個使用中的地理藏寶盒。

## 歷史

地謎藏寶和已有160年歷史的地理郵筒尋寶遊戲（letterboxing）相似，都是使用故事中的線索和[地標](../Page/地標.md "wikilink")的指引。在2000年5月2日，GPS系統中的「選擇性可用干擾」功能被除去。GPS系統的精確性提升至讓藏寶盒一類體積較小的物品可以被明確地定位的程度，因而發展出地謎尋寶。首個有記錄的地謎藏寶盒的位置由來自[俄勒岡州](https://zh.wikipedia.org/wiki/俄勒岡州 "wikilink")[比弗河](https://zh.wikipedia.org/wiki/比弗河 "wikilink")（Beavercreek）的大衛·爾默（Dave Ulmer），在2000年5月3日以GPS記錄。其坐標公佈在[Usenet](../Page/Usenet.md "wikilink")的[新聞組](https://zh.wikipedia.org/wiki/新聞組 "wikilink")。至2000年5月6日，它被兩次發現，其中一次被記錄起來。據戴夫爾默说，藏匿的東西是一個大部分被埋在地下的黑色塑料桶，放有[軟體](https://zh.wikipedia.org/wiki/軟體 "wikilink")、錄像帶、書籍、食品、錢以及一把[彈弓](https://zh.wikipedia.org/wiki/彈弓 "wikilink")。

俄勒岡州公共廣播節目俄勒岡野地指南，2010年2月刊報導地謎尋寶事件，同時拜訪了原來的藏匿地點。金屬紀念碑如今豎立在這個地方，但實際的存放地點是在附近的灌木叢裡。

## 地謎藏寶

傳統的地謎藏寶，藏匿藏寶盒者會把一個裝著日誌、筆以及交換物品的防水盒子藏起來，並記錄坐標。然後將這些坐標及其它相關細節的資料，張貼在網站上。其它地謎獵人在從網站上取得坐標後，会利用手持全球定位系統找尋寶藏。地謎獵人會把戰績記錄在日誌本及網站上。他們可以用類似或更高價值的物品自由換取地謎藏寶盒中的東西(除了日誌、鉛筆、紀念章）。

典型的地謎「寶藏」貨幣價值不高，但對發現者有個人的價值存在。除了日誌，地謎藏寶盒內容物通常是不尋常的硬幣 或貨幣，小玩具，裝飾鈕扣，CD或書籍。這些物品也常常從一個地謎地點移至另一個地謎地點，稱為搭便車 ，如旅行蟲或地理硬幣，它們的流向可在網站上記錄及追蹤。先放旅行蟲或地理硬幣的獵人通常會為這些可追蹤物品定下特定的目標。目標的例子可以是要把它們放在離家遙遠的特定藏寶處或是移至另一個特定國家，或是和其它搭便車物品比賽跑得更快更遠。價值較高的寶藏偶爾作為首次發現（稱為「FTF」），或難以到達地點的獎勵。危險或非法物品，武器和[情色書刊一般而言不能使用](https://zh.wikipedia.org/wiki/情色書刊 "wikilink")，且違反大部分地謎藏寶網站的規則。

藏寶盒的尺寸最小可和小指指尖一樣，只放得下日誌的薄紙，大的可以是五加侖的桶子或是更大。 在鄉村村地區最常用的藏寶盒尺寸是便當盒大小的塑料容器或多餘的軍用彈藥罐。彈藥罐被認為是藏寶盒的黃金標準，因為它們非常堅固，防水，防火、能避免被動物破壞，且相對便宜，又有足夠的空間来放置要交換的物品。而小尺寸的藏寶盒在都市區比較常見，因為它們比較容易藏匿。 如果藏寶盒被破壞或被盜，稱為被盜走或掠奪。前者指對藏寶盒不熟悉尋寶遊戲的人為麻瓜，借自哈利波特，因為當哈利波特日漸走紅同時也是地謎藏寶遊戲開始發展的時候。把食物，或是任何有氣味的東西放在藏寶盒絕對是不被建議的。因為動物可以通過氣味找到藏寶盒，並會想辦法打開藏寶盒或拿走。這會破壞藏寶盒且盒主必須再更換新的。

## 術語解釋

寶藏的簽到簿或是官網上常見以下縮寫，認識這些縮寫能幫助您瞭解 Geocaching 玩家們的語言。

  - DNF = Did Not Find / 未找到寶藏
  - FTF = First to Find / 首先找到寶藏
  - STF = Second to Find / 第二個找到寶藏者
  - TTF = Third to Find / 第三個找到寶藏者
  - TFTC = Thanks for the Cache / 感謝這個寶藏，或是感謝寶主
  - TNLN = Took Nothing，Left Nothing / 沒有從寶藏中拿走東西，也沒有留下東西
  - CO = Cache Owner / 寶主

在[线性代数中](../Page/线性代数.md "wikilink")，*n*
维[向量空间的](../Page/向量空间.md "wikilink")[基是](https://zh.wikipedia.org/wiki/基_\(线性代数\) "wikilink")
*n* 个向量 α<sub>1</sub>, ..., α<sub>n</sub>
的序列，带有所有这个空间中的向量可以唯一的表达为[基向量的](https://zh.wikipedia.org/wiki/基向量 "wikilink")[线性组合的性质](https://zh.wikipedia.org/wiki/线性组合 "wikilink")。因为经常需要处理一个向量空间的多于一个的基，在线性代数中能够轻易的变换向量的逐坐标表达，和变换关于一个基的[线性映射到关于另一个基的等价表达是根本重要的](../Page/线性映射.md "wikilink")。这种变换叫做**基变更**。

尽管下面采用了术语向量空间，符号 R
意味着[实数](../Page/实数.md "wikilink")[域](https://zh.wikipedia.org/wiki/域_\(数学\) "wikilink")，这里讨论的结果成立只要
R
是[交换环](../Page/交换环.md "wikilink")，而这里的向量空间可替代为[自由](https://zh.wikipedia.org/wiki/自由模 "wikilink")
[R-模](../Page/模.md "wikilink")''。

## 预备概念

R<sup>n</sup> 的**平常基**是 \(\{ \vec{e}_1, \cdots, \vec{e}_n \}\)，这里的
\(\vec{e}_j = (0, \cdots, 1, 0, \cdots, 0)\) 是 R<sup>n</sup> 的元素，带有 1 在第
*j* 个位置上其他地方都是 0。

如果 *T* : R<sup>n</sup> → R<sup>m</sup>
是[线性变换](https://zh.wikipedia.org/wiki/线性变换 "wikilink")，*T* 的 m
× n **[矩阵](../Page/矩阵.md "wikilink")**是对于 \(j = 1,\cdots,n\) 其第 *j* 纵列是
\(T(\vec{e}_j) \,\) 的矩阵 **t**。在这种情况下我们有
\(T(\vec{x}) = \mathbf{t}\vec{x}\) 对于所有 R<sup>n</sup> 中的 **x**，这里我们把
**x**
当作列向量，在右侧的乘法是[矩阵乘法](https://zh.wikipedia.org/wiki/矩阵乘法 "wikilink")。在线性代数中一个基本事实是从
R<sup>n</sup> 到 R<sup>m</sup> 的所有线性变换的向量空间
Hom(R<sup>n</sup>, R<sup>m</sup>) 自然的[同构在](../Page/同构.md "wikilink") R
上的 m × n 矩阵的空间 R<sup>m × n</sup>；就是说线性变换 *T* : R<sup>n</sup> →
R<sup>m</sup> 对于所有目的和用途都等价于它的矩阵 **t**。

我们还利用下列简单的观察。

**定理**：设 *V* 和 *W* 是向量空间，设 \(\{ \alpha_1, \cdots, \alpha_n \}\) 是 *V*
的基，并设 \(\{ \gamma_1, \cdots, \gamma_n \}\) 是任何 *W* 中的 *n*
个向量。则存在一个唯一的线性变换 *T* : *V* → *W* ，对于
\(j = 1,\cdots,n\) 有 \(T(\alpha_j) = \gamma_j \,\)。

这个唯一的 *T* 定义自
\(T(x_1 \alpha_1 + \cdots + x_n \alpha_n) = x_1 \gamma_1 + \cdots + x_n \gamma_n\)。当然，如果
\(\{ \gamma_1, \cdots, \gamma_n \}\)碰巧是 *W* 的基，则 *T*
是[双射又是线性的](../Page/双射.md "wikilink")；换句话说，*T*
是[同构](../Page/同构.md "wikilink")。如果在这种情况下我们还有 *W* = *V*，则 *T*
被称为是[自同构](../Page/自同构.md "wikilink")。

现在设 *V* 在 R 上的向量空间并假设 \(\{ \alpha_1, \cdots, \alpha_n \}\) 是 *V*
的基。通过定义，如果 ξ 是 *V* 中的向量，则
\(\xi = x_1 \alpha_1 + \cdots + x_n \alpha_n\) 是 \(x_1, \cdots, x_n\) 在
R 中唯一[标量选择](../Page/标量.md "wikilink")，被叫做 ξ **相对于有序基**
\(\{ \alpha_1, \cdots, \alpha_n \}\) 的**坐标**。 R<sup>n</sup> 中的向量
\(\vec{x} = (x_1, \cdots, x_n)\) 被叫做 ξ (相对于这个基)的**坐标元组**。唯一的线性映射 φ :
R<sup>n</sup> → *V*，对于 \(j = 1,\cdots n\) 有
\(\phi (\vec{e}_j) = \alpha_j \,\)，它被称为对 *V* 和基
\(\{ \alpha_1, \cdots, \alpha_n \}\) 的**坐标同构**。所以
\(\phi (\vec{x}) = \xi \,\) [当且仅当](../Page/当且仅当.md "wikilink")
\(\xi = x_1 \alpha_1 + \cdots + x_n \alpha_n\)。

## 坐标变更

我们实现检查在 *V* 中的向量 ξ 的坐标在选择了另一个基的时候怎样变更的问题。假设
\(\{ \alpha_1, \cdots, \alpha_n \}\) 和
\(\{ \alpha'_1, \cdots, \alpha'_n \}\) 是 *V* 的两个基。设 φ<sub>1</sub> 和
φ<sub>2</sub> 是从 R<sup>n</sup> 到 *V* 的对应的坐标同构就是说
\(\phi_1(\vec{e}_j) = \alpha_j \,\) 而
\(\phi_2(\vec{e}_j) = \alpha'_j \,\) 对于 \(j = 1, \cdots, n\)。如果
\(\vec{x} = (x_1, \cdots, x_n)\) 是 ξ 关于第一个基的坐标 *n*-元组，因此
\(\xi = \phi_1(\vec{x}) \,\)，则 ξ 关于第二个基的坐标元组是
\(\phi_2^{-1}(\xi) = \phi_2^{-1}(\phi_1\vec{x}))\)。现在映射
\(\phi_2^{-1} \circ \phi_1\) 是在 R<sup>n</sup> 上的自同构，因此有一个矩阵 **p**。此外，
**p** 的第 *j* 纵列是
\(\phi_2^{-1} \circ \phi_1(\vec{e}_j) = \phi_2^{-1}(\alpha_j)\)，就是说，\(\alpha_j \,\)
关于第二个基 \(\{ \alpha'_1, \cdots, \alpha'_n \}\) 的坐标 *n*-元组。所以
\(\vec{y} = \phi_2^{-1}(\phi_1(\vec{x})) = \mathbf{p}\vec{x}\) 是 ξ 关于基
\(\{\alpha'_1,\cdots, \alpha'_n \}\) 的坐标 *n*-元组。

## 线性变换的矩阵

现在假设 *T* : *V* → *W* 是线性变换，{α<sub>1</sub>, ..., α<sub>n</sub>} 是 *V*
的一个基而 {β<sub>1</sub>, ..., β<sub>m</sub>} 是 *W* 的一个基。设 φ 和 ψ 分别是
*V* 和 *W* 的相对于给定基的坐标同构。则映射 *T*<sub>1</sub> = ψ<sup>-1</sup> o *T* o φ 是从
R<sup>n</sup> 到 R<sup>m</sup> 的线性变换，并因此有一个矩阵 **t**；它的第 *j* 纵列是
ψ<sup>-1</sup>(T(α<sub>j</sub>)) 对于 *j* = 1, ..., *n*。这个矩阵叫做*T* 关于有序基
{α<sub>1</sub>, ..., α<sub>n</sub>} 和 {β<sub>1</sub>, ...,
β<sub>m</sub>} 的矩阵。如果 η = *T*(ξ) 并且 **y** 和 **x** 是 η 和 ξ 的坐标元组，则 **y**
= ψ<sup>-1</sup>(T(φ(**x**))) = **tx**。反过来，如果 ξ 在 *V* 中，而 **x** =
φ<sup>-1</sup>(ξ) 是 ξ 关于 {α<sub>1</sub>, ..., α<sub>n</sub>} 的坐标元组，我们设置
**y** = **tx** 和 η = ψ(**y**)，则 η = ψ(*T*<sub>1</sub>(**x**)) =
*T*(ξ)。就是说，如果 ξ 在 *V* 中而 η 在 *W* 中并且 **x** 和 **y** 是它们的坐标元组，则
**y** = **tx** [当且仅当](../Page/当且仅当.md "wikilink") η = *T*(ξ)。

**定理**：假设 *U*, *V* 和 *W* 是有限维的向量空间并为每个选择了有序基。如果 *T* : *U* → *V* 和 *S* :
*V* → *W* 是有矩阵 **s** 和 **t** 的线性变换，则线性变换 *S* o *T* : *U* → *W*
(关于给定基)的矩阵是 **st**。

### 基的变更

现在我们要问 *T* : *V* → *W* 的矩阵在变更在 *V* 和 *W* 的基的时候发生了什么。设 {α<sub>1</sub>,
..., α<sub>n</sub>} 和 {β<sub>1</sub>, ..., β<sub>m</sub>} 分别是 *V* 和 *W*
的有序基，并假设给予了第二对基 {α'<sub>1</sub>, ..., α'<sub>n</sub>} 和
{β'<sub>1</sub>, ..., β'<sub>m</sub>}。设 φ<sub>1</sub> 和 φ<sub>2</sub>
是从在 R<sup>n</sup> 中的平常基到 *V* 的第一个和第二个基的坐标同构，并设 ψ<sub>1</sub> 和
ψ<sub>2</sub> 是从在 R<sup>m</sup> 中的平常基到 *W* 的第一个和第二个基的同构。

令 \(T_1 = \psi_1^{-1} \circ T \circ \phi_1\)，并令
\(T_2 = \psi_2^{-1} \circ T \circ \phi_2\)（两者都从 \(\mathbb{R}^n\) 映至
\(\mathbb{R}^m\)）。令 \(\mathbf{t}_1\) 与 \(\mathbf{t}_2\) 为相应的矩阵。令
\(\mathbf{p}, \mathbf{q}\) 分别为对应到基变更自同构
\(\phi_2^{-1} \circ \phi_1 : \mathbb{R}^m \to \mathbb{R}^m\) 与
\(\psi_2^{-1} \circ \psi_1: \mathbb{R}^n \to \mathbb{R}^n\) 的矩阵。

由於我们有
\(T_2 = \psi_2^{-1} \circ T \circ \phi_2 = (\psi_2^{-1} \circ \psi_1) \circ T_1 \circ (\phi_1^{-1} \circ \phi_2)\)，又因为线性映射的合成对应到矩阵乘法，遂得到

  -
    \(\mathbf{t}_2 = \mathbf{q} \mathbf{t}_1 \mathbf{p}^{-1}\)

## 自同态的矩阵

线性变换的矩阵的一个重要情形是自同态的矩阵，亦即从一个向量空间 \(V\) 至其自身的线性映射，换言之就是 *W* = *V*
的情形。我们可以自然地取基 {β<sub>1</sub>, ..., β<sub>n</sub>} =
{α<sub>1</sub>, ..., α<sub>n</sub>} 与 {β'<sub>1</sub>, ...,
β'<sub>m</sub>} = {α'<sub>1</sub>, ..., α'<sub>n</sub>}。此时线性映射 *T*
的矩阵必为方阵。

### 基的变更

套用同样的基变更，使得 **q** = **p**，而基变更公式遂写成

  -
    **t**<sub>2</sub> = **p** **t**<sub>1</sub> **p**<sup>-1</sup>.

在此情形下，[可逆矩阵](https://zh.wikipedia.org/wiki/可逆矩阵 "wikilink") **p**
被称为向量空间 *V* 的**基变更矩阵**，而上述等式言明 **t**<sub>1</sub> 与
**t**<sub>2</sub>
是[相似矩阵](https://zh.wikipedia.org/wiki/相似矩阵 "wikilink")。

## 双线性形式的矩阵

於[域](https://zh.wikipedia.org/wiki/域 "wikilink") *R* 的向量空间 *V*
上的[双线性形式是一个映射](https://zh.wikipedia.org/wiki/双线性形式 "wikilink")
*V* × *V* → *R*，使得它对两个参数都是线性的，也就是说

\[v \mapsto B(v, w)\]

\[v \mapsto B(w, v)\] 对任何固定的 *w*
都是线性的。此定义可推广至於[交换环的](../Page/交换环.md "wikilink")[模](../Page/模.md "wikilink")，此时须将线性映射换为模同态。

对应於基 \(\alpha_1,\dots, \alpha_n\) 的 Gram 矩阵 *G* 定义为

\[G_{i,j} = B(\alpha_i,\alpha_j) \,\].

若 *v*, *w* 以此基表成

  -
    \(v = \sum_i x_i \alpha_i\)
    \(w = \sum_i y_i \alpha_i\)

则该双线性形式由下式给出

\[B(v,w) = x^\top G y \,\] .

若 *B*
是[对称双线性形式](../Page/对称双线性形式.md "wikilink")，则对应的矩阵会是[对称矩阵](https://zh.wikipedia.org/wiki/对称矩阵 "wikilink")。

### 基的变更

若矩阵 *P* 表示从 \(\alpha_1,\dots, \alpha_n\) 至
\(\alpha'_1,\dots, \alpha'_n\) 的基变更，则两组基的 Gram 矩阵依下式变换：

\[G' = P^\top G P \,\]

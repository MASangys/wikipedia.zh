在[生物學與](https://zh.wikipedia.org/wiki/生物學 "wikilink")[进化计算中](https://zh.wikipedia.org/wiki/进化计算 "wikilink")，**基因座**（），也称为“基因位点”或“位点”，是指染色體上的固定位置，例如某個[基因](../Page/基因.md "wikilink")的所在。而基因座上的DNA序列可能有許多不同的變化，各種變化形式稱為[等位基因](https://zh.wikipedia.org/wiki/等位基因 "wikilink")（allele）。基因座在基因組中的排列位置稱為[基因图谱](https://zh.wikipedia.org/wiki/基因图谱 "wikilink")（genetic map），[基因作圖](https://zh.wikipedia.org/wiki/基因作圖 "wikilink")（Gene mapping）則是測定基因座與特定[性狀關係的過程](https://zh.wikipedia.org/wiki/性狀 "wikilink")。

[二倍體與](https://zh.wikipedia.org/wiki/二倍體 "wikilink")[多倍體細胞的某些染色體上](https://zh.wikipedia.org/wiki/多倍體 "wikilink")，在同一基因座上有相同的等位基因，這類細胞稱為[纯合子](https://zh.wikipedia.org/wiki/纯合子 "wikilink")/[同型合子](https://zh.wikipedia.org/wiki/同型合子 "wikilink")（homozygous）。若是相同基因座上含有不同的等位基因，則稱作[杂合子](https://zh.wikipedia.org/wiki/異型合子 "wikilink")/[異型合子](https://zh.wikipedia.org/wiki/異型合子 "wikilink")（heterozygous）。

## 命名

舉例而言，一個典型的基因座可能寫成「6p21.3」，解釋如下

  - 6 = [6號染色體](https://zh.wikipedia.org/wiki/6號染色體 "wikilink")
  - p = 短臂
  - 21 = 2區, 1帶
  - 3 = 子帶3

區、帶和子帶是從[著絲粒向外朝](https://zh.wikipedia.org/wiki/著絲粒 "wikilink")[端粒](../Page/端粒.md "wikilink")開始編號。基因座詳細命名規則如下：

|        |                                                                                                                                                                                       |
| ------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **符號** | **解釋**                                                                                                                                                                                |
| *6*    | 染色體編號。                                                                                                                                                                                |
| *p*    | 染色體上的短臂（*p*）或長臂（*q*）位置。*p*來自法文*petit* (小)；*q*則是選定*p*的下一個字母，也有來自法文*queue* (尾巴)一說。                                                                                                      |
| *21.3* | 染色體臂上的所在位置，21表示2区（region）1帶（band），3表示子帶（sub-band），染色體帶經過適當染色之後，可於[顯微鏡](../Page/顯微鏡.md "wikilink")底下觀察。每個帶皆有編號，最靠近[著絲粒](https://zh.wikipedia.org/wiki/著絲粒 "wikilink")（centromere）的為1號。 |

一個範圍之內的基因座的寫法類似，例如[OCA1基因的基因座可寫成](https://zh.wikipedia.org/wiki/OCA1 "wikilink")「11q1.4-q2.1」，表示此基因位於11號染色體q臂中的1號帶4號子帶，到2號帶1號子帶之間。

染色體的末端寫成「*pter*」或「*qter*」，例如「*2qter*」表示2號染色體的[端粒](../Page/端粒.md "wikilink")。

**堅尼地城站**（）是[港鐵](../Page/港鐵.md "wikilink")[港島綫](../Page/港島綫.md "wikilink")西端終點[鐵路車站](https://zh.wikipedia.org/wiki/鐵路車站 "wikilink")，位於[香港島](../Page/香港島.md "wikilink")[中西區](../Page/中西區_\(香港\).md "wikilink")[堅尼地城](../Page/堅尼地城.md "wikilink")[士美菲路](../Page/士美菲路.md "wikilink")和[科士街](../Page/科士街.md "wikilink")之間的地底，於2014年12月28日隨[西港島綫通車正式啟用並取代](../Page/港島綫西延.md "wikilink")[上環站](../Page/上環站.md "wikilink")成為港島綫的西面終點站。

## 車站設計

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Kennedy Town Station Concourse 201412.jpg | caption1 = 大堂 | image6 = Kennedy Town Station Control Room.JPG | caption6 = 車站控制室 }} 堅尼地城站上方為公園及公共交通交匯處，出入口設計與上方的兩個公園及交通交匯處互相結合，並使用啡黃色掛石作外牆裝飾，頂棚用料為玻璃，可以讓天然光滲透進車站入口處內；而地底部分則分為兩層，地庫一層為車站大堂，地庫二層為兩個側式月台，月台設計並與[上環站](../Page/上環站.md "wikilink")相似，用作分隔出站及入站客流之用，以避免乘客坐反方向誤入[掉頭隧道之內](https://zh.wikipedia.org/wiki/機走線 "wikilink")。

港鐵解釋堅尼地城站鄰近海旁，故此車站大堂及月台以湖水藍色為車站主色。\[1\]

### 大堂

堅尼地城站內提供6間商店供乘客購物或進食，包括兩間7-11便利店、La Boheme Bakery、意大利雪糕店Heyyo、uji bedding床上用品店及銀行櫃員機等服務。車站亦自助服務設施、以至男、女、及傷殘洗手間。另外，大堂近C出入口設有1部「[會員服務站](../Page/會員服務站.md "wikilink")」，供乘客享用港鐵友禮會的會員優惠\[2\]。

### 月台

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Kennedy Town Station 2018 01 part1.jpg | caption1 = 2號月台（落客專用） }} 堅尼地城站港島綫月台為兩個[側式月台](../Page/側式月台.md "wikilink")，全部月台均設有[月台幕門](../Page/月台幕門.md "wikilink")。1號及2號月台在乘客眼中看似兩個位於不同層數的單向式月台，但事實上它們是對向式月台設計，只是兩個月台之間被牆壁完全阻隔，令乘客無從得知兩月台位於同一層，與[南昌站](../Page/南昌站_\(香港\).md "wikilink")（[西鐵綫](../Page/西鐵綫.md "wikilink")）、[美孚站](../Page/美孚站.md "wikilink")（西鐵綫）、[荃灣西站](../Page/荃灣西站.md "wikilink")、[坑口站](../Page/坑口站_\(香港\).md "wikilink")、[上環站](../Page/上環站.md "wikilink")、[何文田站](../Page/何文田站.md "wikilink")（[觀塘綫](../Page/觀塘綫.md "wikilink")）及[青衣站](../Page/青衣站.md "wikilink")的設計類似。每當上下班繁忙時間之後，部份港島綫的港鐵列車在此站清客後，便會暫停服務，先使用調頭隧道折返1號月台，再返回[柴灣車廠](https://zh.wikipedia.org/wiki/柴灣車廠 "wikilink")。

## 利用狀況

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = MTR KET (14).JPG | caption1 = 2號月台頭端可看到列車經由調頭隧道駛入同層1號月台 }} 堅尼地城站鄰近只有[住宅](../Page/住宅.md "wikilink")、[辦公室](../Page/辦公室.md "wikilink")及社區綜合大樓，鄰近並無大型的購物商場和[百貨公司](../Page/百貨公司.md "wikilink")。因此，本站的服務對象主要是居住於鄰近的堅尼地城居民和上班人士。所以，除了上下班的繁忙時間外，整體的出入閘人流會很少。在每天的晚上、星期六、星期日和公眾假期，人流會更少。所以，堅尼地城站是[港鐵](../Page/港鐵.md "wikilink")[港島綫](../Page/港島綫.md "wikilink")17個車站中其中一個人流較少的車站。不過，有少部分[南區西部的居民及到](https://zh.wikipedia.org/wiki/南區 "wikilink")[瑪麗醫院](../Page/瑪麗醫院.md "wikilink")探病的民眾，都會使用本站轉乘小巴往其他地方，對本站的人流仍有一定支持。

## 歷史

堅尼地城站最早在1970年的《集體運輸計劃總報告書》出現。該站本應在1980年代與港島綫一起興建。但是當時地鐵公司（即現時的[香港鐵路有限公司](https://zh.wikipedia.org/wiki/香港鐵路有限公司 "wikilink")）以[上環站](../Page/上環站.md "wikilink")以西的路段客量不足和存在技術困難為由，而沒有展開工程。在1980年代或之前的方案中，這個車站位於[加多近街近](https://zh.wikipedia.org/wiki/加多近街 "wikilink")[堅尼地城巴士總站的地底](https://zh.wikipedia.org/wiki/堅尼地城巴士總站 "wikilink")，同時計劃配合將[港澳客輪碼頭](../Page/港澳客輪碼頭.md "wikilink")搬遷至車站旁邊。

根據2005年6月的建議，堅尼地城站將會是港島綫延伸段──西港島綫的車站，而不是在較早前的建議中，一個[南區地鐵支綫](../Page/南區_\(香港\).md "wikilink")（當時也被稱為西港島綫，即現時[南港島綫](../Page/南港島綫.md "wikilink")的前身）的車站。根據地鐵公司在2006年8月發表的西港島綫的修訂建議方案\[4\]，堅尼地城站位於科士街臨時遊樂場地底，即[爹核士街](../Page/爹核士街.md "wikilink")及[士美菲路](../Page/士美菲路.md "wikilink")一帶。而車站地面出口，即舊[堅尼地城游泳池](../Page/堅尼地城游泳池.md "wikilink")的位置，設有[公共運輸交匯處](../Page/公共運輸交匯處.md "wikilink")，方便鐵路乘客轉乘其他公共交通工具\[5\]。

堅尼地城站於2013年4月10日平頂，成為[西港島綫首個平頂的車站](../Page/港島綫西延.md "wikilink")，標誌著該站的[土木工程](../Page/土木工程.md "wikilink")完成，隨即正式展開軌道鋪設工程、[機電工程及其他後期裝修工程](https://zh.wikipedia.org/wiki/機電工程 "wikilink")。\[6\]本站於2014年9月30日竣工，並已於同年12月28日正式通車\[7\]。

港鐵在2014年12月13及14日（星期六、日）分別開放[香港大學站](../Page/香港大學站.md "wikilink")及堅尼地城站予公眾參觀，由中西區區議會派發入場券。12月14日有逾二千名西區居民參加本站開放日及嘉年華會。\[8\]

### 站名書法題字爭議

{{ multiple image | align = right | direction = vertical | width = 220 | image1 =Kennedy Town Station 2014 part1.JPG| caption1 =堅尼地城站的書法站名牆，「-{堅}-」字的[行書与其簡體](https://zh.wikipedia.org/wiki/行書 "wikilink")「-{坚}-」極為相似 }} 根據堅尼地城站的照片，牆上以大型的書法體展示站名，與港島線其他車站一致。惟書法體的「堅尼地城」四字在引起[網民爭議](https://zh.wikipedia.org/wiki/網民 "wikilink")，多位網民質疑「-{堅}-」字採用了[簡體字](https://zh.wikipedia.org/wiki/簡體字 "wikilink")「-{坚}-」。有網民指出，「-{堅}-」在[行書中亦有寫為](https://zh.wikipedia.org/wiki/行書 "wikilink")「-{坚}-」字的習慣\[9\]。而港鐵發言人澄清，西港島線站名並無採用簡體字，3座新鐵路站內的書法站名均是出自同一位前港鐵建築師[區傑棠](../Page/區傑棠.md "wikilink")之手筆，字型設計融入了作者的個人書法藝術風格。為了讓鐵路站增添特色，港島線最初啟用時就採用書法字在月台顯示車站名稱，令車站添加中國傳統文化氣息，與車站主色調配合，形成獨有的設計風格；由於西港島綫是港島線的延線，故車站內展示的站名沿用港島線的書法字體及風格，保持連貫性。\[10\]

### 塗鴉惡搞港鐵 數小時內被清理

2018年10月23日早上，網民在facebook「西環變幻時」群組貼文，發現車站B出口的石屎牆上被塗鴉上諷刺港鐵的標誌和字句。港鐵標誌上看起來像「屎」字。而標誌的旁邊畫上「Most Troublesome Railway」的字句。塗鴉很快已被黑色膠袋遮蓋。到下午牆壁已重新上油。\[11\]

#### 2019年配合警方行動暫停本站列車服務

2019年7月28日晚上10時18分，港鐵宣佈因應警方要求，[港島綫](../Page/港島綫.md "wikilink")列車服務只維持[柴灣站](../Page/柴灣站.md "wikilink")至[上環站](../Page/上環站.md "wikilink")，[西營盤站](../Page/西營盤站.md "wikilink")、[香港大學站](../Page/香港大學站.md "wikilink")和堅尼地城站列車服務暫停。直至晚上11時34分全綫服務回復正常。\[12\]

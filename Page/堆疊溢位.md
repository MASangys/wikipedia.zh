**-{zh-hans:堆栈溢出;
zh-hant:堆疊溢位}-**（）在[電腦科學中是指使用過多的](https://zh.wikipedia.org/wiki/電腦科學 "wikilink")[記憶體时導致](https://zh.wikipedia.org/wiki/記憶體 "wikilink")[呼叫堆疊產生的溢位](../Page/呼叫堆疊.md "wikilink")\[1\]。堆疊溢位的產生是由於過多的函數呼叫，導致[呼叫堆疊無法容納這些呼叫的](../Page/呼叫堆疊.md "wikilink")[返回地址](https://zh.wikipedia.org/wiki/返回地址 "wikilink")，一般在[遞迴中產生](https://zh.wikipedia.org/wiki/遞迴_\(電腦科學\) "wikilink")。堆疊溢位很可能由[無限](https://zh.wikipedia.org/wiki/無限 "wikilink")[遞迴](https://zh.wikipedia.org/wiki/遞迴_\(電腦科學\) "wikilink")（Infinite
recursion）產生，但也可能僅僅是過多的堆疊層級。

堆疊溢位在[核心設計中尤其危險](https://zh.wikipedia.org/wiki/內核 "wikilink")，因此很多入門核心設計教程建議使用者不要嘗試使用[遞迴程式](https://zh.wikipedia.org/wiki/遞迴_\(電腦科學\) "wikilink")；而是基於安全和效能考量，改用[迴圈處理問題](../Page/迴圈.md "wikilink")。\[2\]\[3\]\[4\]

在[POSIX相容平台上](https://zh.wikipedia.org/wiki/POSIX "wikilink")，堆疊溢位通常會造成[作業系統產生](https://zh.wikipedia.org/wiki/作業系統 "wikilink")[SIGSEGV](../Page/SIGSEGV.md "wikilink")[訊號](https://zh.wikipedia.org/wiki/信号_\(计算机科学\) "wikilink")。

## 遞迴溢位

### 無限遞迴

無限遞迴是堆疊溢位的最常見原因，如以下的[C/C++語言程式會產生堆疊溢位](https://zh.wikipedia.org/wiki/C/C++ "wikilink")：

``` C
int foo()
{
    return foo();  //這裡出現自我呼叫，重複自我呼叫
}
```

然而有些語言（如[Scheme](../Page/Scheme.md "wikilink")）支援[尾端遞迴優化](https://zh.wikipedia.org/wiki/尾端遞迴 "wikilink")，在這些語言中只有一般遞迴會產生堆疊溢位，而尾端遞迴不會：

``` Scheme
(define (foo) (foo))
(define (foo) (+ (foo) 1))
```

这段代码中，前者（第一句）进入[無窮迴圈](../Page/無窮迴圈.md "wikilink")（Infinite
loop），但不会产生堆疊溢位；后者（第二句）则会产生堆疊溢位。

### 防止堆疊溢位

多數無限遞迴出現原因，都是基於程式本身沒有[錯誤檢測機制](https://zh.wikipedia.org/wiki/異常處理 "wikilink")：

``` C
int factorial( const int *const n ){
    if ( *n == 0 )
        return 1;
    else
        return *n * factorial( *n-- );  //這裡出現自我呼叫
};
```

如果在這裡的`n`是負數則會出現無限遞迴。其實，這段程式可以簡單地加以修改，把`n`的型別由整數改為非負整數即可解決：

``` C
unsigned int factorial( const unsigned int *const n ){
    if ( *n == 0 )
        return 1;
    else
        return *n * factorial( *n-- );
};
```

或者使用[迴圈處理](../Page/迴圈.md "wikilink")：

``` C
unsigned int factorial( const unsigned int *const n ){
    unsigned int i, result;
    for ( i = *n, result = 1; i > 0 ; --i )
        result *= i;
  //自我呼叫部份改為迴圈處理
    return result;
};
```

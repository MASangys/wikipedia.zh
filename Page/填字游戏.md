**填字游戏**是一种常见的紙上[益智遊戏](https://zh.wikipedia.org/wiki/益智遊戏 "wikilink")。

游戏一般给出一个[矩形的](../Page/矩形.md "wikilink")[表格](https://zh.wikipedia.org/wiki/表格 "wikilink")。这个表格被分割为若干个大小相同的[方格](https://zh.wikipedia.org/wiki/方格 "wikilink")，方格的颜色有[白色与](../Page/白色.md "wikilink")[黑色两种](../Page/黑色.md "wikilink")。白色的方格组成一些交叉的行与列，行列的长度不等。

玩家根据[题目所提供的有关信息](https://zh.wikipedia.org/wiki/题目 "wikilink")，将答案填入这些[行与](https://zh.wikipedia.org/wiki/行 "wikilink")[列之中](https://zh.wikipedia.org/wiki/列 "wikilink")，每个白色方格中只能填入一个字。一般地说，题目给出的每一条信息就是对应的一行或一列的解题线索。

在行与列交叉的地方，玩家必须保证在交叉的方格中填入的字同时满足题目中对行与列的要求。

## 例子

下列就是一个英式填字游戏的例子：

|               |  |              |  |              |
| ------------- |  | ------------ |  | ------------ |
| <sup>1A</sup> |  | <sup>B</sup> |  |              |
|               |  |              |  |              |
| <sup>2</sup>  |  |              |  | <sup>C</sup> |
|               |  |              |  |              |
|               |  | <sup>3</sup> |  |              |

  - 横行

1\. 羊声 (3)
2\. 不是气体或液体 (5)
3\. 幽默 (3)

  - 直列

A. 交通工具 (3)
B. 应许 (5)
C. "Dorothy"的缩写(3)

这一题的解法就是：

|                                        |                                        |                                        |                                        |                                        |
| -------------------------------------- | -------------------------------------- | -------------------------------------- | -------------------------------------- | -------------------------------------- |
| <sup><small>1A</small></sup>B          | <sup><font color=white>9</font></sup>A | <sup>B</sup>A                          | .                                      | .                                      |
| <sup><font color=white>9</font></sup>U | .                                      | <sup><font color=white>9</font></sup>L | .                                      | .                                      |
| <sup>2</sup>S                          | <sup><font color=white>9</font></sup>O | <sup><font color=white>9</font></sup>L | <sup><font color=white>9</font></sup>I | <sup>C</sup>D                          |
| .                                      | .                                      | <sup><font color=white>9</font></sup>O | .                                      | <sup><font color=white>9</font></sup>O |
| .                                      | .                                      | <sup>3</sup>W                          | <sup><font color=white>9</font></sup>I | <sup><font color=white>9</font></sup>T |

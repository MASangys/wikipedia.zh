**填字游戏**是一种常见的紙上[益智遊戏](https://zh.wikipedia.org/wiki/益智遊戏 "wikilink")。

游戏一般给出一个[矩形](../Page/矩形.md "wikilink")的[表格](https://zh.wikipedia.org/wiki/表格 "wikilink")。这个表格被分割为若干个大小相同的方格，方格的颜色有[白色](../Page/白色.md "wikilink")与[黑色](../Page/黑色.md "wikilink")两种。白色的方格组成一些交叉的行与列，行列的长度不等。

玩家根据题目所提供的有关信息，将答案填入这些[行](../Page/行.md "wikilink")与[列之中](https://zh.wikipedia.org/wiki/列 "wikilink")，每个白色方格中只能填入一个字。 [Hots_CrosswordColor300.png](https://zh.wikipedia.org/wiki/File:Hots_CrosswordColor300.png "fig:Hots_CrosswordColor300.png")

## 例子

下列就是一个英式填字游戏的例子：

|               |  |              |  |              |
| ------------- |  | ------------ |  | ------------ |
| <sup>1A</sup> |  | <sup>B</sup> |  |              |
|               |  |              |  |              |
| <sup>2</sup>  |  |              |  | <sup>C</sup> |
|               |  |              |  |              |
|               |  | <sup>3</sup> |  |              |

  - 横行

1\. 羊声 (3)
2\. 不是气体或液体 (5)
3\. 幽默 (3)

  - 直列

A. 交通工具 (3)
B. 应许 (5)
C. "Dorothy"的缩写(3)

这一题的解法就是：

|                                        |                                        |                                        |                                        |                                        |
| -------------------------------------- | -------------------------------------- | -------------------------------------- | -------------------------------------- | -------------------------------------- |
| <sup><small>1A</small></sup>B          | <sup><font color=white>9</font></sup>A | <sup>B</sup>A                          | .                                      | .                                      |
| <sup><font color=white>9</font></sup>U | .                                      | <sup><font color=white>9</font></sup>L | .                                      | .                                      |
| <sup>2</sup>S                          | <sup><font color=white>9</font></sup>O | <sup><font color=white>9</font></sup>L | <sup><font color=white>9</font></sup>I | <sup>C</sup>D                          |
| .                                      | .                                      | <sup><font color=white>9</font></sup>O | .                                      | <sup><font color=white>9</font></sup>O |
| .                                      | .                                      | <sup>3</sup>W                          | <sup><font color=white>9</font></sup>I | <sup><font color=white>9</font></sup>T |

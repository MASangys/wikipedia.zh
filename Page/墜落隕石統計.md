**墜落隕石統計**是行星科學家最常用來逼近墜落在地球上的[隕石](../Page/隕石.md "wikilink")真實通量的方法。**[墜落隕石](../Page/墜落隕石.md "wikilink")**是被目睹下墜後不久的時間，就被發現並被蒐集起來的隕石。雖然發現隕石的數量是墜落隕石的30倍，然而其類型與物質分布並不足以反映墜落到地面的是些甚麼。這些原因包括：

1.  有些類型的隕石比其他的種類容易發現。
2.  有些類型的隕石受到[風化侵蝕得比其他的快](https://zh.wikipedia.org/wiki/風化 "wikilink")。
3.  有些隕石，特別是[鐵隕石](../Page/鐵隕石.md "wikilink")，因為可以利用或是毫無用處，而在過去曾經被人們收集起來，而未留下科學上的紀錄。
4.  很多隕石墜落像是許多石頭的流星雨，但在事件發生許久之後，可能很難分辨哪些是墜落下來的石頭。
5.  很多隕石被人們拿來販售……，有價值、罕見的類型很快地就會被科學家知道，但那些低價值的可能永遠不會被談論

有許多人試圖校正發現隕石的這些效應，尤其是罕見隕石類型的頻率估計和一些統計上的分析。例如，已經有超過100顆的[月球隕石](../Page/月球隕石.md "wikilink")被發現，但尚未曾觀測與找到墜落隕石。然而，對數量豐富的類型，墜落隕石的統計數量通常可以對應。這些統計的資料截止於2012年6月9日。

## 物質的統計

對多數的墜落隕石，甚至包括那些發生在很久以前，但尚未完整研究其科學上的特徵，只知道是石質、石鐵或鐵隕石。這是依據文獻資料列出每個類型的數量和百分比\[1\]\[2\]\[3\]：

| 物質   | 數量   | %      |
| ---- | ---- | ------ |
| 鐵隕石  | 49   | 4.4%   |
| 石鐵隕石 | 11   | 1.0%   |
| 石隕石  | 1042 | 94.6%  |
| 種數   | 1102 | 100.0% |

## 主要類型的統計

傳統的隕石分類（參見[隕石分類](../Page/隕石分類.md "wikilink")）是分成鐵質、石鐵質、石質的分成[球粒隕石](../Page/球粒隕石.md "wikilink")和[無粒隕石](../Page/無粒隕石.md "wikilink")兩類。對一些被研究得較少的墜落石隕石，不知道是否屬於球粒隕石，因此這一類的隕石數量會低於上面所顯示的4%，這些數量在下表。其中已經做了一些小更正（例如，鐵隕石的數量可以減少0.2%），但此處並未修正。

| 種類   | 數量   | %      |
| ---- | ---- | ------ |
| 鐵隕石  | 49   | 4.6%   |
| 石鐵隕石 | 11   | 1.0%   |
| 無球隕石 | 86   | 8.2%   |
| 球粒隕石 | 915  | 86.2%  |
| 總數   | 1062 | 100.0% |
|      |      |        |

## 隕石群體的統計

或許最有用的統計數字是依據群體的分類，這是隕石分類的基本方法。大約有5%的隕石還沒有被充分的定性，因此未能分入這些群體。再一次，一些數量的調整改變了各項所佔的百分比，但結果並不會有太大的變化。注意到有些墜落隕石的群體只有少數的數量；所以屬於這些群體的墜落數量有很大的不確定性。

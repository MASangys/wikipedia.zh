**音效卡**是[多媒體](../Page/多媒體.md "wikilink")[電腦中用來處理](../Page/電腦.md "wikilink")[聲音的接口卡](../Page/聲音.md "wikilink")。音效卡可以把來自[話筒](../Page/麦克风.md "wikilink")、[收音機](../Page/收音機.md "wikilink")、[錄音機](../Page/錄音機.md "wikilink")、雷射唱機（[鐳射影碟](../Page/鐳射影碟.md "wikilink")）等設備的語音、[音樂等](../Page/音樂.md "wikilink")[聲音變成](../Page/聲音.md "wikilink")[數位訊號交給](../Page/數位訊號.md "wikilink")[電腦處理](../Page/電腦.md "wikilink")，並以文件形式保存，還可以把[數位訊號還原成為真實的](../Page/數位訊號.md "wikilink")[聲音輸出](../Page/聲音.md "wikilink")。音效卡尾部的接口從[機箱後側伸出](../Page/機箱.md "wikilink")，上面有連接[麥克風](../Page/麥克風.md "wikilink")、[音箱](../Page/音箱.md "wikilink")、[遊戲杆和](../Page/遊戲杆.md "wikilink")[MIDI設備的](../Page/MIDI.md "wikilink")[接口](../Page/接口.md "wikilink")。

但自從各大主機板廠商推廣All-In-One的主機板以來，主機板皆內建[AC97或](../Page/AC97.md "wikilink")[HD
Audio聲卡](../Page/HD_Audio.md "wikilink")，迫使中低階的音效卡市場快速萎縮，僅剩高階、音響級與專業用途音效卡仍存在於市場。

## 接口颜色

声卡上的接口都具有不同的颜色，每个颜色都具特殊的意义。 有些声卡接口也附有符号代表不同用途。

| 颜色 | [潘通颜色代号](../Page/潘通.md "wikilink")                            | 功能    | 接口                                      | 符号            |
| -- | ------------------------------------------------------------- | ----- | --------------------------------------- | ------------- |
|    | [粉色](../Page/粉色.md "wikilink")                                | 701C  | 模拟[麦克风输入](../Page/麦克风.md "wikilink")    | 3.5 mm单声道/立体声 |
|    | 淡蓝色                                                           | 284C  | 模拟音源输入                                  | 3.5 mm单声道/立体声 |
|    | 青柠色                                                           | 577C  | 主立体声信号的模拟线路音频输出（扬声器或耳机）                 | 3.5 mm单声道/立体声 |
|    | [橙色](../Page/橙色.md "wikilink")                                | 157C  | 模拟中置扬声器/重低音音响线路音频输出                     | 3.5 mm单声道/立体声 |
|    | [黑色](../Page/黑色.md "wikilink")                                |       | 模拟环绕扬声器/后置立体声线路音频输出                     | 3.5 mm单声道/立体声 |
|    | [銀色](../Page/銀色.md "wikilink")/[灰色](../Page/灰色.md "wikilink") | 422C  | 模拟环绕声/侧声道线路音频输出                         | 3.5 mm单声道/立体声 |
|    | [褐色](../Page/褐色.md "wikilink")                                | 4645C | 模拟特殊音道平移模拟线路音频输出                        | 3.5 mm单声道/立体声 |
|    | [金色](../Page/金色.md "wikilink")                                |       | 游戏端口/[MIDI](../Page/MIDI.md "wikilink") | 15 pin D      |

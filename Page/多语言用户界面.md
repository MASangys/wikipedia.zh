**多语言用户界面**（Multilingual User Interface，**MUI**）是[微软](../Page/微软.md "wikilink")的一种技术，允许在一个软件上安装多种语言，用户可以自主选择一种语言用于用户界面显示。MUI从[Windows 2000开始使用](../Page/Windows_2000.md "wikilink")。它是一项国际专利“Multilingual User Interface for an Operating System”。\[1\]发明人是：Bjorn C. Rettig、Edward S. Miller、Gregory Wilson、Shan Xu。

MUI不同于本地化版本，它只需要安装一份软件并处理不同语言可能导致的冲突。\[2\]

## Windows 2000 与 Windows XP

MUI产品只能用于大客户版本，不适用于零售版本。是在英文版之上提供本地化用户体验。

## Windows Vista 与 Windows 7

[Windows Vista发展了MUI技术](../Page/Windows_Vista.md "wikilink")，支持语言中立、语言独立的单个二进制文件用于多语言皮肤，语言相关资源存放在不同的二进制文件中。操作系统的二进制代码与语言资源是分开的。

MUI包适用于[Windows Vista企业版与](../Page/Windows_Vista.md "wikilink")[Ultimate Extra版](../Page/Windows_Ultimate_Extras.md "wikilink")。也适用于 [Windows 7企业版与旗舰版](https://zh.wikipedia.org/wiki/Windows_7 "wikilink")。

从Windows Vista开始，MUI API可用于应用程序开发。

## Windows 8/8.1/RT 与 Windows 10

从8/RT开始，所有版本的Windows可以下载案子所有的语音包。\[3\]

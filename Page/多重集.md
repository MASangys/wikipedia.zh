**多重集**或**多重集合**是[数学中的一个概念](../Page/数学.md "wikilink")，是[集合概念的推广](../Page/集合.md "wikilink")。在一个集合中，相同的[元素只能出现一次](../Page/元素.md "wikilink")，因此只能显示出有或无的属性。在多重集之中，同一个元素可以出现多次。正式的多重集的概念大约出现在1970年代。\[1\]

## 简介

多重集的[势的计算和一般集合的计算方法一样](../Page/势.md "wikilink")，出现多次的元素则需要按出现的次数计算，不能只算一次。一个元素在多重集里出现的次数称为这个元素在多重集里面的**重数**（或**重次**、**重复度**）。举例来说，\(\left\{ 1, 2, 3\right\}\)是一个集合，而\(\left\{ 1, 1, 1, 2, 2, 3\right\}\)不是一个集合，而是一个多重集。其中元素1的重数是3，2的重数是2，3的重数是1。\(\left\{ 1, 1, 1, 2, 2, 3\right\}\)的元素个数是6。有时为了和一般的集合相区别，多重集合会用方括号而不是花括号标记，比如\(\left\{ 1, 1, 1, 2, 2, 3\right\}\)会被记为\(\left[ 1, 1, 1, 2, 2, 3\right]\)。和[多元组或](../Page/多元组.md "wikilink")[数组的概念不同](../Page/数组.md "wikilink")，多重集中的元素是没有顺序分别的，也就是说\(\left[ 1, 1, 1, 2, 2, 3\right]\)和\(\left[ 1,  1, 2, 1, 2, 3\right]\)是同一个多重集。

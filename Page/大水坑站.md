**大水坑站**（）位於[香港](../Page/香港.md "wikilink")[沙田區](../Page/沙田區.md "wikilink")[馬鞍山的](../Page/馬鞍山_\(香港市鎮\).md "wikilink")[港鐵](../Page/港鐵.md "wikilink")[馬鞍山綫](../Page/馬鞍山綫.md "wikilink") 車站，已於2004年12月21日啟用。車站取名自鄰近的大水坑村。

## 結構

大水坑站位於馬鞍山路中央地面，佔地約3,760平方米\[1\]，共分為3層。大堂位於地面，月台設於大堂之上，而[行人隧道則位於大堂之下](https://zh.wikipedia.org/wiki/行人隧道 "wikilink")。大堂與月台層之間及設有[升降機和](https://zh.wikipedia.org/wiki/升降機 "wikilink")[扶手電梯](../Page/電動扶梯.md "wikilink")，方便乘客上落。

### 大堂

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Tai Shui Hang Station 2019 10 part4.jpg | caption1 = 大堂 }} 大水坑站大堂內設有7間商店，包括[便利店](../Page/便利商店.md "wikilink")、[微波爐便當售賣店及物業顧問等](https://zh.wikipedia.org/wiki/微波爐便當 "wikilink")，以及[自動櫃員機](../Page/自動櫃員機.md "wikilink")\[2\]。

### 月台

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Tai Shui Hang Station 2017 11 part1.jpg | caption1 = 1號月台﹙往大圍／啟德方向﹚ | image2 = Tai Shui Hang Station 2017 11 part2.jpg | caption2 = 2號月台﹙往烏溪沙方向﹚ | image3 = Tai Shui Hang Station 2018 06 part2.jpg | caption3 = 月台佈置 }} 車站設有兩個月台，採用[島式月台](../Page/島式月台.md "wikilink")方式排列，月台北面設有預留空間以便馬鞍山綫列車增至8卡行走，預留空間會於沙中線興建時於預留結構上擴建，現時擴建工程已經完成。此站是馬鞍山綫往烏溪沙方向唯一一個月台是彎曲的車站。而此站北面設有[渡线](https://zh.wikipedia.org/wiki/渡线 "wikilink")，方便列車調動或信號故障時才使用。

### 出入口

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Tai Shui Hang Station 2019 10 part3.jpg | caption1 = 大堂往來A及B出入口行人隧道 | image2 = MTR TSH (8).JPG | caption2 = 行人隧道出入口 }} 大水坑站的東面是[富安花園](../Page/富安花園.md "wikilink")及大水坑村，西面為[錦泰苑](../Page/錦泰苑.md "wikilink")和[曉峯灣畔](../Page/曉峯灣畔.md "wikilink")。車站只有南面設有[閘機](../Page/驗票閘門.md "wikilink")，通往連接行人隧道的兩個出入口。由於車站位處[馬鞍山路](../Page/馬鞍山路.md "wikilink")的中央分隔帶，因此乘客必須利用行人隧道進出車站。此行人隧道24小時開放予行人使用橫過馬鞍山路。

嚴格來說，大水坑站只得一個出入口，而所謂的A、B出入口只是行人隧道的公眾部分（東、西兩邊的出入口）。

## 鄰接車站

大水坑站是馬鞍山綫的中途站，乘客可在此乘坐往烏溪沙或大圍列車，往返[九龍](../Page/九龍.md "wikilink")及[新界](../Page/新界.md "wikilink")等地。

[Tai_Shui_Hang_Station_2018_06_part1.jpg](https://zh.wikipedia.org/wiki/File:Tai_Shui_Hang_Station_2018_06_part1.jpg "fig:Tai_Shui_Hang_Station_2018_06_part1.jpg")

## 利用情況

大水坑站位於沙田新市鎮(馬鞍山部分)的南端，進出站內的多為[錦泰苑](../Page/錦泰苑.md "wikilink")、富安花園、大水坑村、曉峯灣畔的住客，也有附近學校的學生使用車站乘搭馬鞍山綫。乘客主要利用馬鞍山綫到[馬鞍山市中心](https://zh.wikipedia.org/wiki/馬鞍山市中心 "wikilink")，或經[大圍站](../Page/大圍站.md "wikilink")轉乘[東鐵綫](../Page/東鐵綫.md "wikilink")到市區。

### 命名及啟用

大水坑站於2004年12月21日隨馬鞍山鐵路（現今馬鞍山綫）通車啟用。由於車站位於[富安花園](../Page/富安花園.md "wikilink")的對面，因此原來建議的站名為**富安站**（）\[3\]\[4\]。

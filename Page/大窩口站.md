**大窩口站**（）是[港鐵](../Page/港鐵.md "wikilink")[荃灣綫](../Page/荃灣綫.md "wikilink")的一個地底車站，它亦是[新界區第一個建於地底的車站](https://zh.wikipedia.org/wiki/新界區 "wikilink")。根據[十八區區議會區域劃分](https://zh.wikipedia.org/wiki/香港十八區 "wikilink")，行政上屬於[荃灣區](../Page/荃灣區.md "wikilink")\[1\]，[德士古道](../Page/德士古道.md "wikilink")至昌榮路一段的[青山公路](../Page/青山公路.md "wikilink")葵涌段是荃灣區及葵青區的分界線，車站建於青山公路以北，故屬於荃灣區。然而，站名由來的[大窩口邨](../Page/大窩口邨.md "wikilink")則位於[葵青區](../Page/葵青區.md "wikilink")。車站位於大窩口[青山公路](../Page/青山公路.md "wikilink")—葵涌段旁的國瑞路公園地底，由日本青木建設及飛島建設聯營公司負責興建，於1982年5月10日啟用。車站以深綠色和淺綠色為主要色調。

### 月台

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Tai Wo Hau Station 2019 08 part1.jpg | caption1 = 1號月台﹙往荃灣方向﹚ }} 車站設有2個月台，並以[島式月台](../Page/島式月台.md "wikilink")方式排列，各月台均設有[月台幕門](../Page/月台幕門.md "wikilink")。全站只有2條連接大堂及月台的[扶手電梯](../Page/電動扶梯.md "wikilink")，是港鐵系統中最少扶手電梯的車站之一。車站至[葵興站](../Page/葵興站.md "wikilink")之間設有一條側綫。每當此站或[荃灣站](../Page/荃灣站.md "wikilink")出現故障時，往[荃灣方向的上行綫列車將會以葵興站作為臨時的終點站](../Page/荃灣站.md "wikilink")。屆時，前往此站及荃灣站的乘客將須要轉乘其他交通工具前往。

## 利用狀況

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Tai Wo Hau Station 2017 07 part4.jpg | caption1 = 連接月台與大堂的升降機 }} 由於荃灣綫興建時，為減低對鄰近[三棟屋村](../Page/三棟屋村.md "wikilink")地基的影響，因此[荃灣站](../Page/荃灣站.md "wikilink")位置向東北遷移，引致乘客均流向荃灣站乘車，此站是荃灣綫較少人使用的車站之一。

此車站鄰近並無任何公共運輸總站，鄰近[青山公路](../Page/青山公路.md "wikilink")途經巴士及小巴多逾數十條路線（多數巴士及小巴路線亦途經荃灣站，故乘客多不會使用此站轉乘港鐵），加上設施配套不足，啟用以來只有兩個出入口，如[石圍角邨](../Page/石圍角邨.md "wikilink")與[悅來酒店](../Page/悅來酒店.md "wikilink")等人流均流向荃灣站乘車（悅來酒店及[悅來坊](../Page/悅來坊.md "wikilink")商場設置行人天橋橫跨青山公路，往大窩口站因遷就寶血會思源學校而需從路面步行逾5分鐘，往荃灣站可直接途經商場行人天橋），大窩口邨與其附近私人屋苑、城門谷康樂設施以及[葵涌邨升降機相繼落成而逐漸增加人次使用](https://zh.wikipedia.org/wiki/葵涌邨 "wikilink")，但出口不足仍是使用率偏低的最大因素。

## 歷史

[Mass_Transportation_Study_1967_zh.svg](https://zh.wikipedia.org/wiki/File:Mass_Transportation_Study_1967_zh.svg "fig:Mass_Transportation_Study_1967_zh.svg")

### 選址

1967年，顧問公司Freeman, Fox, and Partners於研究報告中建議[荃灣](../Page/荃灣站.md "wikilink")、大窩口兩站設於[沙咀道](../Page/沙咀道.md "wikilink")地底\[5\]。但因後來[荃灣車廠決定於青山公路](https://zh.wikipedia.org/wiki/荃灣車廠 "wikilink")－荃灣段以北興建而將走綫北移，兩站的位置因而改為現時位置。

### 啟用

大窩口站於1982年5月10日隨著荃灣綫首段工程通車而啟用。車站在荃灣綫計劃初期，大窩口站主要服務三棟屋村至大窩口邨一帶居民，後因1979年動工之荃灣綫定線影響三棟屋村大部份地基，政府把舊有三棟屋搬遷，並成為現時由康樂及文化事務署管理的[三棟屋博物館](../Page/三棟屋博物館.md "wikilink")。

### 月台吊板

有別於一般前地鐵地底車站的月台佈局，初期大窩口站月台頂部的乘車資訊是以綠底白字顯示，與此站色系一致，跟其他地底車站的白底黑字不同，直至90年代兩站月台統一安裝白底黑字吊板為止。該吊板已在2004年安裝[月台幕門](../Page/月台幕門.md "wikilink")時被拆除。

### 加建出入站升降機

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Tai Wo Hau Station 2019 03 part7.jpg | caption1 = 地面升降機 }} 港鐵與[路政署](../Page/路政署.md "wikilink")合作，為接駁港鐵車站的行人天橋首個「一梯兩用」升降機\[6\]，乘客可以乘搭升降機直接往返行人天橋、地面及大窩口站大堂，升降機已於2009年初投入服務，但礙於空間所限，每次只乘載不足十人，車站須時常作出廣播讓給有需要人士優先使用，故大部份乘客寧捨等候而步上77級樓梯回大窩口區。曾有大窩口邨[區議員多次要求加建扶手電梯直達行人天橋](https://zh.wikipedia.org/wiki/區議員 "wikilink")，但鑑於屬跨區事宜而屢遇阻滯。

## 未來發展

### 更換升降機

往來大堂及月台升降機將於2017年1月13日暫停使用，以更換為較新型號，並預計於2018年1月底重投服務\[7\]。

## 意外

### 1993年荃灣綫載客列車脫鈎事件

1993年4月27日，當時未與九廣鐵路合併前的地鐵公司，曾發生全球首宗脫卡意外。事發時，一列[荃灣綫](../Page/荃灣綫.md "wikilink")載客列車由[荃灣站](../Page/荃灣站.md "wikilink")駛往大窩口站途中，一個車卡的連接器鬆脫，導致最後3卡車廂與前面車卡分離，失去動力後停在隧道內。但車長未察覺列車脫卡繼續行駛，駛至大窩口站時，經月台職員通知才揭發事故。幸好當時沒有乘客站在車卡與車卡中間，所以沒有人命傷亡。\[8\]\[9\]

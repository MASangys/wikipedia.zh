**大西洋桥**（）是[巴拿马](../Page/巴拿马.md "wikilink")的一座[公路](../Page/公路.md "wikilink")[桥](https://zh.wikipedia.org/wiki/桥 "wikilink")，位于以北约\[1\]，跨越[巴拿马运河](../Page/巴拿马运河.md "wikilink")在[大西洋](../Page/大西洋.md "wikilink")一侧的[入海口](https://zh.wikipedia.org/wiki/入海口 "wikilink")，连接[科隆省](https://zh.wikipedia.org/wiki/科隆省_\(巴拿马\) "wikilink")[首府](https://zh.wikipedia.org/wiki/首府 "wikilink")[科隆市与城市西郊](../Page/科隆_\(巴拿马\).md "wikilink")\[2\]。该桥成为继[美洲大桥](../Page/美洲大桥.md "wikilink")和[世纪大桥之后](../Page/世纪大桥_\(巴拿马\).md "wikilink")，第三座跨越巴拿马运河的桥梁，而前两座桥梁均位于运河的[太平洋一侧](https://zh.wikipedia.org/wiki/太平洋 "wikilink")，靠近[首都](../Page/首都.md "wikilink")[巴拿马城](https://zh.wikipedia.org/wiki/巴拿马城 "wikilink")\[3\]。

## 概况

大西洋桥工程由桥梁部分及其两端连接线组成，总长约，按照美国AASHTO标准设计\[4\]。其中桥梁部分全长，采用双向四车道标准\[5\]，主桥为双塔双索面五跨连续[混凝土](../Page/混凝土.md "wikilink")梁[斜拉桥](https://zh.wikipedia.org/wiki/斜拉桥 "wikilink")，跨径布置为：79+181+530+181+79米，边跨设置1个辅助墩和1个过渡墩。全桥共设128对斜拉索，采用[钢](../Page/钢.md "wikilink")绞线制成，索面按[扇形](../Page/扇形.md "wikilink")布置，每一索面由32对斜拉索组成\[6\]。大桥主跨，是目前世界上主跨径最长的混凝土斜拉桥。标准宽度，设计行车[速度](../Page/速度.md "wikilink")90[千米/小时](https://zh.wikipedia.org/wiki/千米/小时 "wikilink")\[7\]。两座桥塔高，桥下净空高度\[8\]。东、西引桥分别长和\[9\]。

## 历史

2011年6月3日，宣布，中交公路规划设计院有限公司代表[中国交通建设集团](https://zh.wikipedia.org/wiki/中国交通建设集团 "wikilink")（中交集团）与[美国](../Page/美国.md "wikilink")组成的联合体中标大西洋桥施工图设计项目，中标金额4,662,352[美元](../Page/美元.md "wikilink")\[10\]。为全部6家竞标单位中的最低价\[11\]。这是中国企业通过国际公开竞标获得的第一个海外大型桥梁设计项目\[12\]。其他5家竞标单位的报价分别为\[13\]：

| 公司                                                        | 报价          |
| --------------------------------------------------------- | ----------- |
| TYPSA Principia                                           | 4,995,150美元 |
| URS Holdings（URS控股）                                       | 6,150,021美元 |
| Puente de Colón JV（科隆大桥合资公司）                              | 7,212,000美元 |
| ARUP（[奥雅纳](https://zh.wikipedia.org/wiki/奥雅纳 "wikilink")） | 8,542,017美元 |
| T. Y. Lin International（）                                 | 9,669,231美元 |
|                                                           |             |

2012年8月，3家财团获得了大桥施工工程的[投标资格](https://zh.wikipedia.org/wiki/投标 "wikilink")：[阿驰奥纳](../Page/阿驰奥纳.md "wikilink")-特拉德科联合体（[西班牙](../Page/西班牙.md "wikilink")-[墨西哥](../Page/墨西哥.md "wikilink")）、-联合体（[巴西](../Page/巴西.md "wikilink")-[韩国](https://zh.wikipedia.org/wiki/韩国 "wikilink")） 、（[法国](https://zh.wikipedia.org/wiki/法国 "wikilink")）\[14\]。

2012年10月，巴拿马运河管理局与法国万喜建筑签署了大桥的施工[合同](https://zh.wikipedia.org/wiki/合同 "wikilink")，总价365,979,472.39美元，而另外2家竞标者的报价分别为\[15\]：

| 公司             | 报价               |
| -------------- | ---------------- |
| 阿驰奥纳-特拉德科联合体   | 386,619,316.04美元 |
| 奥德布雷赫特-现代建设联合体 | 429,633,298.00美元 |
|                |                  |

大桥在被正式命名以前，常用“三桥”或“大西洋侧大桥”来指代\[16\]\[17\]。大桥于2013年1月开始施工，原计划工期3年半，2016年中完工\[18\]，但截至2016年12月，预计完工日期已推迟到2018年中\[19\]。2019年8月2日，大桥正式举行竣工通车典礼\[20\]。

《**大頭綠衣鬥殭屍**》（）是[香港](../Page/香港.md "wikilink")[電視廣播有限公司](../Page/電視廣播有限公司.md "wikilink")製作的殭屍題材電視劇，全劇共20集，監製[李添勝](../Page/李添勝.md "wikilink")。

此劇曾於1994年逢[星期一](../Page/星期一.md "wikilink")至五11:45-12:45在[翡翠台](../Page/翡翠台.md "wikilink")重播，2003年2月25日至3月26日逢星期一至五10:40-11:40在[翡翠台](../Page/翡翠台.md "wikilink")再次重播，2007年8月23日至9月5日逢星期一至五03:15-05:00在[翡翠台](../Page/翡翠台.md "wikilink")再次重播，每日播映兩集，2014年6月24日至7月24日逢星期一至五03:40-04:35在[翡翠台](../Page/翡翠台.md "wikilink")再次重播，2013年8月17日在[TVB經典台](../Page/TVB經典台.md "wikilink")重播，2018年逢星期一至五18:00-18:55在[TVB星河頻道](../Page/TVB星河頻道.md "wikilink")“好戲珍藏館”時段重播。

## 故事大綱

一代捉鬼大師張可以 ( [楚原](../Page/楚原.md "wikilink")飾 ) 因避戰亂，帶著兩個徒弟張北平 ( [錢小豪](../Page/錢小豪.md "wikilink")飾 ) 及張南京 ( [黎耀祥](../Page/黎耀祥.md "wikilink")飾 )，從京城來到香港，由於生活艱苦，三師徒被迫居於妓院柴房。北平平後來轉職大頭綠衣 ( 警察 )，剛巧殭屍大鬧香港，北平請以重出江湖對付殭屍。北平戀上女鬼飄雪 ( [鄭秀文](../Page/鄭秀文.md "wikilink")分飾 )，雪溫婉嫻淑，可惜人鬼殊途，此段情終告消逝。雪另外有一孿生姊姊飄紅( [鄭秀文](../Page/鄭秀文.md "wikilink")分飾 )，為人豪爽不羈，亦一直暗戀北平，北平曾一度弄錯人鬼身份，弄出笑話連篇，最後，紅為救北平甘願犧牲性命，到底紅、北平之間的愛情，註定要終結還是有所突破？

另外殭屍的禍根，生前是被香港首富屠萬富 ( [江毅](../Page/江毅.md "wikilink")飾) 所害，可惜天理循環，報應郤報在萬富女兒屠天麗 ( [關寶慧](../Page/關寶慧.md "wikilink")飾 ) 身上。天麗在生時迷戀北平，後假扮自殺卻弄假成真，死後卻執迷不悟，受鬼官教梭，報夢父親萬富，將自己煉成殭屍之王，要吸盡100人的陽氣，萬富卻成為第100人被天麗吸取陽氣而死的人。

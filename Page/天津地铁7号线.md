**天津地铁7号线**（）是[天津地铁](../Page/天津地铁.md "wikilink")的在建线路之一。线路大致呈南北走向，代表色是<small> <font color={{天津轨道交通颜色|7}}>█</font> </small>**棕色**</font></span>。

截止2019年7月，天津地铁7号线一期工程（赛达路站至喜峰道站）在建，计划建设期4.5年\[1\]\[2\] 。

## 概要

7号线一期工程途径西青区、河西区、和平区、南开区、红桥区、河北区六个行政区，南起[西青区](../Page/西青区.md "wikilink")赛达路站，站后设出入线与车辆段相接，之后线路沿友谊南路延长线、卫津南路、卫津路、南门外大街敷设，过黑牛城道、宾水西道、吴家窑大街、鞍山西道、福安大街、南马路、北马路，之后进入大胡同地区，过大胡同下穿海河，进入中山路敷设，过金纬路、律纬路后进入南口路敷设，沿南口路至普济河道南侧，北至[河北区](../Page/河北区.md "wikilink")喜峰道站。

一期工程线路长26.525km，共设车站21 座，均为地下站，在西青区大寺设置车辆段一处，占地约32公顷。设两座主变电所，新建六里台主变电所1座，改建10号线解放南路主变电所1座，与中心城区其他轨道交通线路共用地铁3号线华苑车辆段内线网控制中心。

## 历史

  - 2016年3月，天津地铁7号线一期环境影响评价进行了公示\[3\]；
  - 2018年10月，7号线一期福安大街站开始房屋征收工作\[4\]。
  - 2019年5月，天津地铁7号线一期工程[PPP项目中标结果公布](../Page/政府和社会资本合作.md "wikilink")\[5\]。
  - 2019年7月，天津地铁7号线一期工程正式开工\[6\] 。

## 车站

7号线一期工程共设21座车站，均为地下站。

### 换乘站

  - **[海光寺站](https://zh.wikipedia.org/wiki/海光寺站 "wikilink")**：换乘1号线。1号线建设时未做预留，预计为通道换乘。
  - **[鼓楼站](../Page/鼓楼站_\(天津市\).md "wikilink")**：换乘2号线。2号线建设时未做预留，换乘方式未知。
  - **[天塔站](https://zh.wikipedia.org/wiki/天塔站 "wikilink")**：换乘3号线。3号线建设时未做预留，换乘方式未知。
  - **[中山路站](https://zh.wikipedia.org/wiki/中山路站 "wikilink")**：换乘3号线。3号线建设时未做预留，且两站台相距较远，预计为长通道换乘。
  - **[东北角站](https://zh.wikipedia.org/wiki/东北角站 "wikilink")**：换乘4号线。4号线北段规划暂未确定，换乘方式未知。
  - **[肿瘤医院站](https://zh.wikipedia.org/wiki/肿瘤医院站 "wikilink")**：换乘5号线、6号线。5、6号线建设时未做预留，预计为通道换乘。

  - **[六里台站](https://zh.wikipedia.org/wiki/六里台站 "wikilink")**：换乘8号线。8号线规划暂未确定，换乘方式未知。
  - **[丽江道站](https://zh.wikipedia.org/wiki/丽江道站 "wikilink")**：换乘10号线。10号线建设时预留站厅换乘通道，预计为通道换乘。

## 车辆

7号线采用A型车，为天津市区轨道交通中第一条采用A型列车的线路\[7\]。列车最高运行速度为80km/h，初期平均旅行速度为 32km/h，近期及远期平均旅行速度为35km/h。初期、近期、远期均采用6辆编组，列车总长140米，编组形式为

|                                                                                     |             |  |
| :---------------------------------------------------------------------------------: | :---------: | :-: |
|                                                                                     | 天津地铁7号线车辆编组 |  |
|                                        M7}}|                                        |             |  |
| style="font-size: 85%;" colspan=3 padding=1em 0 0 0 align=center|+Tc-MP-M\*M-MP-Tc+ |             |  |

其中，Tc为带司机室的拖车，MP为带受电弓的动车，M为不带受电弓的动车，+为全自动车钩，-为半永久车钩，\*为半自动车钩。

## 未来发展

### 预留延伸

7号线喜峰道站將预留延伸條件，未來可繼續往北延伸，到达小淀地区，为7号线二期工程，截止2018年9月，二期工程暂未有具体规划。

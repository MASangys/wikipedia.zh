**契約**（或**合約**，-{zh-cn:台湾称**契约**; zh-tw:中國稱**合同**;zh-hk:台灣稱**契約**}-）\[1\]，是雙方當事人基於[意思表示](../Page/意思表示.md "wikilink")合致而成立的[法律行為](https://zh.wikipedia.org/wiki/法律行為 "wikilink")，為[私法自治的主要表現](https://zh.wikipedia.org/wiki/私法 "wikilink")。一般而言，契約是指[私法上的法律行為](https://zh.wikipedia.org/wiki/私法 "wikilink")，可分為債權契約（例如[買賣](https://zh.wikipedia.org/wiki/買賣 "wikilink")）、物權契約（例如[所有權移轉登記](https://zh.wikipedia.org/wiki/所有權 "wikilink")）及身分契約（例如[結婚](https://zh.wikipedia.org/wiki/結婚 "wikilink")）等，不過在[公法上也可能存在契約關係](https://zh.wikipedia.org/wiki/公法 "wikilink")（例如 [行政契約](http://www.lawtw.com/article.php?template=article_content&area=free_browse&parent_path=,1,561,&job_id=27028&article_category_id=205&article_id=12392)）。在[民法](../Page/民法.md "wikilink")上，狹義的契約（即債權契約）為債之發生的原因之一，而一般僅稱契約時所指稱者也多屬債權契約。契約行為並不等於「契約書」，一份契約書中可能包含不只一個契約行為；契約行為也不以做成書面為必要，契約原則上為諾成且不要物的法律行為，只有在例外情形，基於特殊考量（例如公益）時[法律](../Page/法律.md "wikilink")會明文要求。

契約是以雙方當事人互相對立合致的意思表示所構成的，其中包括要約及承諾兩個基本的意思表示。**要約**是[表意人所發出](../Page/意思表示.md "wikilink")，欲得到相對人[承諾而發生一定私法上效力的意思表示](https://zh.wikipedia.org/wiki/承諾 "wikilink")。**承諾**則是針對要約所為的肯定答覆，承諾的內容必須和該要約的內容完全一致，否則即為新要約而非承諾。應與要約區分的是**要約之引誘**，其並非意思表示，而是[觀念通知](https://zh.wikipedia.org/wiki/觀念通知 "wikilink")，為[準法律行為之一種](https://zh.wikipedia.org/wiki/準法律行為 "wikilink")，不生要約拘束力。

## 大陆法系

**合同法**主要是[私法的一部分](https://zh.wikipedia.org/wiki/私法 "wikilink")，也是[民事法的一部分](https://zh.wikipedia.org/wiki/民事法 "wikilink")。按照传统[民法的观点](https://zh.wikipedia.org/wiki/大陆法系 "wikilink")，合同法可以归类于[债法里面的合同之债](https://zh.wikipedia.org/wiki/债法 "wikilink")，与[侵权行为](../Page/侵权行为.md "wikilink")、[无因管理](../Page/无因管理.md "wikilink")和[不当得利共同构成债法的四大组成部分](https://zh.wikipedia.org/wiki/不当得利 "wikilink")。

### 中華人民共和國

l(1)《-{合同}-法》第25条规定：“承诺生效时-{合同}-成立。”也就是一般说，受要约人的承诺到达要约人时，承诺生效，-{合同}-也就成立。

l(2)《-{合同}-法》第32条和第33条规定：“当事人采用-{合同}-书面形式订立合同的，自双方当事人签字或者盖章时-{合同}-成立。”

l(3)“当事人采用信件、数据电文形式订立-{合同}-的，可以在-{合同}-成立之前要求签订确认书。签订确认书时-{合同}-成立。”

### 契約的分類

契約可依各種標準分為許多類型，下表為常見的法律分類類型\[2\]。

| 分類方式                                                         | 類型舉例                                                                                                                                                                                                                    |
| ------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1\.                                                          | **有名契約**又稱**典型式契約**。民法中所明文規定、設有特別規定並賦予一定名稱的契約。                                                                                                                                                                          |
| **無名契約**又稱**非典型式契約**。                                        |                                                                                                                                                                                                                         |
| 2\.                                                          | **雙務契約**為雙方當事人所負之債務具有互為[對價](https://zh.wikipedia.org/wiki/對價 "wikilink")(等價)關係的契約。                                                                                                                                      |
| **單務契約**，又稱**片務契約**，為僅由單方當事人負擔債務的契約，此類契約並不产生同時履行抗辯權與危險負擔之問題。 | [贈與](https://zh.wikipedia.org/wiki/贈與 "wikilink")、[保證](https://zh.wikipedia.org/wiki/保證 "wikilink")、[使用借貸](https://zh.wikipedia.org/wiki/使用借貸 "wikilink")、[消費借貸](https://zh.wikipedia.org/wiki/消費借貸 "wikilink")、無償的委任等。 |
| 3\.                                                          | **諾成契約**又稱**不要物契約**，僅須契約當事人意思表示合致，即可成立而不須目的物交付之契約。                                                                                                                                                                      |
| **要物契約**，又稱**踐成契約**，為須意思表示合致及現物之交付才能成立的契約。                   | 使用借貸、消費借貸等。                                                                                                                                                                                                             |
| 4\.                                                          | **有償契約**為當事人之一方為給付，並約定他方須爲對價給付之契約。                                                                                                                                                                                      |
| **無償契約**為僅一方當事人負擔債務之契約，而負擔債務之當事人不自他方當事人取得對價給付之契約。            | 如贈與、使用借貸等。                                                                                                                                                                                                              |
| 5\.                                                          | **要式契約**為須具備一定方式才能夠成立的契約。                                                                                                                                                                                               |
| **非要式契約**又稱**不要式契約**，為僅須當事人意思表示之合致，而不須具備任何方式即能成立的契約形式。       | 近代民法隨契約自由之原則發展，多數契約皆屬非要式契約。如婚約\[3\]。                                                                                                                                                                                    |
| 6\.\[4\]                                                     | **要因契約**為以給付為成立原因(取得財產上利益為目的\[5\])的契約。                                                                                                                                                                                  |
| **不要因契約**為不受原因欠缺使效力受影響的契約。\[6\]                              | 物權契約、票據行爲、債務拘束、債務承認\[7\]等。                                                                                                                                                                                              |
| 7\.                                                          | **主契約**為不須以其他契約的存在爲前提，而能獨立存在的契約。                                                                                                                                                                                        |
| **從契約**為須以其他契約存在前提的契約。                                       | 利息契約、定金契約、違約金契約、保證契約等。                                                                                                                                                                                                  |
| 8\.                                                          | **本約**為履行預約而訂立的契約，本約中的基本要素及權利義務關係皆須明確規定。                                                                                                                                                                                |
| **預約**為約定將來訂立特定契約的契約。                                        | 婚約。                                                                                                                                                                                                                     |
| 9\.                                                          | **一時性契約**為經一次之給付即可實現債之關係的契約。                                                                                                                                                                                            |
| **繼續性契約**為須當事人持續的履行才能實現契約關係的契約。                              | 勞務、僱傭、租賃、終身定期金等。                                                                                                                                                                                                        |
| 10\.                                                         | **債權契約**為發生債之動的關係為目的的契約。\[8\]                                                                                                                                                                                           |
| **物權契約**為發生物之靜的關係為目的的契約。\[9\]                                |                                                                                                                                                                                                                         |
| **身分契約**為融合人格之靜的關係為目的的契約。\[10\]                              |                                                                                                                                                                                                                         |
| 11\.                                                         | **[附合契約](https://zh.wikipedia.org/wiki/附合契約 "wikilink")**為契約一方當事人僅能選擇接受既有契約內容與否的選擇，而無法議約內容之契約。\[11\]                                                                                                                    |
| **非附合契約**為契約雙方當事人均能協議內容的的契約。\[12\]                           |                                                                                                                                                                                                                         |

## 普通法系

在[普通法系中](https://zh.wikipedia.org/wiki/普通法系 "wikilink")，合同法是由[普通法原則](https://zh.wikipedia.org/wiki/普通法 "wikilink")（）、[衡平法](../Page/衡平法.md "wikilink")原則（）、以及在地成文法條文所構成的。

### 香港

如同其他普通法地區，香港除施行[普通法原則與](https://zh.wikipedia.org/wiki/普通法 "wikilink")[衡平法](../Page/衡平法.md "wikilink")原則外，也制定了一系列法例規管——《服務提供（隱含條款）條例》（；Cap.457）\[13\]、《-{zh-hant:不合情理合約條例;zh-hans:不合情理合约条例}-》（；Cap.458）\[14\]等等法例。香港法例（例如：《-{zh-hant:不合情理合約條例;zh-hans:不合情理合约条例}-》）的制定是實際上也是在普通法原則及衡平法原則的基礎上制定的。

### 美國

在美国，《[统一商法典](https://zh.wikipedia.org/wiki/统一商法典 "wikilink")》在买卖合同领域已经总结并修订了传统的判例法。\[15\]

### 英國

在英国，合同法除了被成文法所影響，也同時受限於欧盟的指令與规则。\[16\]

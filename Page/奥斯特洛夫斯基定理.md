**奥斯特洛夫斯基定理**是一个关于[有理数](../Page/有理数.md "wikilink")[域](https://zh.wikipedia.org/wiki/域 "wikilink")[绝对赋值](../Page/绝对赋值.md "wikilink")的定理。于1916年由[亚历山大·奥斯特洛夫斯基](../Page/亚历山大·奥斯特洛夫斯基.md "wikilink")证明。该定理说明，任何非平凡的[有理数](../Page/有理数.md "wikilink")**Q**的[绝对赋值](../Page/绝对赋值.md "wikilink")要么等价于通常[实数](../Page/实数.md "wikilink")域的绝对赋值，要么等价于[p进数的绝对赋值](https://zh.wikipedia.org/wiki/p进数 "wikilink")。

## 定义

定义两个绝对赋值\(|\cdot|\) 和\(|\cdot|_{\ast}\) 是等价的，如果存在一个实数c\>0，使得：

\[\forall x \in \mathbb{K} , \; \;|x|_{\ast} = |x|^{c} .\] 这是比两[绝对赋值](../Page/绝对赋值.md "wikilink")结构的[拓扑同构的更严格的定义](https://zh.wikipedia.org/wiki/拓扑同构 "wikilink")。

任何域的平凡绝对赋值被定义为：

\[|x|_{0} := \begin{cases} 0, & \mbox{if }  x = 0  \\ 1,  & \mbox{if } x \ne 0. \end{cases}\]

有理数\(\mathbb{Q}\)的实绝对赋值是正规实绝对赋值，定义为：

\[|x|_{\infty} := \begin{cases} x, & \mbox{if }  x \geqslant 0  \\ -x,  & \mbox{if } x < 0. \end{cases}\] 有时下标被写成下标1。

任何非零的有理数可以唯一写成\(x=p^{n}\dfrac{a}{b}\)。其中整数、和两两互质。是整数。的进赋值为：

\[|x|_{p} := \begin{cases} 0, & \mbox{if }  x = 0  \\ p^{-n},  & \mbox{if }  x \ne 0. \end{cases}\]

## 另一个奥斯特洛夫斯基定理

另一个奥斯特洛夫斯基定理指出，任何[阿基米德的](https://zh.wikipedia.org/wiki/阿基米德度量 "wikilink")[绝对赋值](../Page/绝对赋值.md "wikilink")[完备域](https://zh.wikipedia.org/wiki/完备域 "wikilink")（从[代数结构](../Page/代数结构.md "wikilink")和[拓扑结构方面](https://zh.wikipedia.org/wiki/拓扑 "wikilink")）同构于[实数域或](https://zh.wikipedia.org/wiki/实数域 "wikilink")[复数域](https://zh.wikipedia.org/wiki/复数域 "wikilink")。这有时也称为奥斯特洛夫斯基定理。

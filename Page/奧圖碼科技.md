**奧圖碼科技**（Optoma）為[投影機](https://zh.wikipedia.org/wiki/投影機 "wikilink")[品牌](../Page/品牌.md "wikilink")事業公司，屬[中強光電](../Page/中強光電.md "wikilink")[子公司](../Page/子公司.md "wikilink")\[1\]。Optoma產品線包括專屬[會議](../Page/會議.md "wikilink")/教學型、高畫質[家庭劇院](../Page/家庭劇院.md "wikilink")型、[電玩](https://zh.wikipedia.org/wiki/電玩 "wikilink")/電視娛樂型及掌上型等四個系列投影機。秉持著「專注、專業、專精」的經營理念，Optoma銷售及[售後服務據點子公司](https://zh.wikipedia.org/wiki/售後服務 "wikilink")，分佈於[美國](https://zh.wikipedia.org/wiki/美國 "wikilink")、[加拿大](../Page/加拿大.md "wikilink")、[英國](https://zh.wikipedia.org/wiki/英國 "wikilink")、[德國](https://zh.wikipedia.org/wiki/德國 "wikilink")、[法國](https://zh.wikipedia.org/wiki/法國 "wikilink")、[挪威](../Page/挪威.md "wikilink")、[西班牙](../Page/西班牙.md "wikilink")、[香港](../Page/香港.md "wikilink")及[中國大陸等地](https://zh.wikipedia.org/wiki/中國大陸 "wikilink")\[2\]\[3\]。中強光電於台灣新竹及中國崑山設立專門的研發/製造總部；奧圖碼科技全球分公司遍及歐洲、北美、中國及亞太地區\[4\]。

## 公司歷史

| 年份         | 事件                                                                                                                                                                                         |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 2002       | 奧圖碼科技正式成立。Optoma成立[美國暨](https://zh.wikipedia.org/wiki/美國 "wikilink")[歐洲分公司](https://zh.wikipedia.org/wiki/歐洲 "wikilink")。Optoma在[上海成立中國營運總部](https://zh.wikipedia.org/wiki/上海 "wikilink")。 |
| 2003       | 奧圖碼推出全球首款65寸數位光處理（DLP）投影電視機 ，並於2003年榮獲消費電子展創新獎。                                                                                                                                            |
| 2005       | 奧圖碼發布全球首款多功能DVD投影機 ── MovieTime™  DV10。                                                                                                                                                    |
| 2006       | 奧圖碼推出全球首款1080P家庭劇院投影機（HD81）。                                                                                                                                                               |
| 2008       | 奧圖碼推出全球首款Pico投影機（PK101）。                                                                                                                                                                   |
| 2013\~2015 | 奧圖碼分別在2013年和2015年發布微型攜帶式LED光源投影機系列（ML750/ML1000），以及首款雷射螢光體技術照明投影機（ZU650）。                                                                                                                  |
| 2014       | 琉璃奧圖碼科技正式淡出琉璃工房經營，復名「奧圖碼科技」，併購美國[音響](../Page/音響.md "wikilink")品牌NuForce跨足音響領域。                                                                                                             |
| 2016       | 奧圖碼在2016年拉斯維加斯影音設備展發布全球首款4K UHD超短焦雷射投影機。                                                                                                                                                   |
| 2017       | 奧圖碼供應首款家庭劇院系列4K UHD雷射投影機（UHZ65）。                                                                                                                                                           |
| 2018       | 奧圖碼在2018年消費電子產品展發布全球首款4K UHD投影機（UHD51A）── 可與Amazon Alexa語音助理搭配使用。 奧圖碼運用德州儀器的數位光處理（DLP）投影技術 成為家庭、商業、教育，以及專業顯示器解決方案之設計商與製造商。                                                                 |
| 2019       | 奧圖碼發布多功能QUAD LED顯示器（FHDQ130）、ProScene 4K UHD雷射投影機（ZK1050），以及Dolby Digital 2.0智慧4K UHD雷射劇院（P1）。                                                                                             |

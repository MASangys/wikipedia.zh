**奧塞羅名人戰**（）是[日本](../Page/日本.md "wikilink")每年舉辦的[黑白棋](../Page/黑白棋.md "wikilink")比賽，是在日本黑白棋界很著名的[比賽](https://zh.wikipedia.org/wiki/比賽 "wikilink")。

## 賽程

名人戰分為無差別、女子和小學生三種類別。\[1\]

無差別組別要對8局（予備戰、本戰6場及決勝等）。

女子組別為予選5局、順位決定戰1局。予選之後，前4名參加決勝戰。

小学生組別則為予選6局。予選之後，前8名參加決勝戰。

限時：無差別及女子組別為每人20分鐘；小學生組別為每人10分鐘，決賽每人15分鐘。

## 獎賞

各組別優勝者除了會獲得名人頭銜外，還會獲得日本奧塞羅聯盟授與段位認定及獎金等獎勵。 　

| 無差別の部     | 女子の部 | 小学生の部              |
| --------- | ---- | ------------------ |
| 名次        | 授與段數 | 獎勵                 |
| 優勝        | 七段   | 会長賞、株式會社MegaHouse賞 |
| 準優勝       | 六段   | 賞金                 |
| 第3名       | 六段   | 賞金                 |
| 第4～9名     | 五段   | 賞金                 |
| 6局勝(含預備戰) | 四段   |                    |
| 5局勝(含預備戰) | 三段   |                    |
| 4局勝(含預備戰) | 二段   |                    |
|           |      |                    |

無差別組別5勝以上、女子組別4勝以上及小學生組別前8名可獲得王座戰出場資格。

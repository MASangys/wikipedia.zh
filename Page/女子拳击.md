**女子拳擊**，是由女性參與的[拳擊運動競賽](https://zh.wikipedia.org/wiki/拳擊 "wikilink")。相較於自古便有的男子拳擊運動，女子拳擊發展較晚。

## 歷史

1720年代，[英國](https://zh.wikipedia.org/wiki/英國 "wikilink")[倫敦即有發展女子拳擊運動的傳言](https://zh.wikipedia.org/wiki/倫敦 "wikilink")。1876年，在[美國舉行的一場女子拳擊比賽](https://zh.wikipedia.org/wiki/美國 "wikilink")，是最早的女子拳擊比賽記錄。

1904年的[聖路易奧運會中](https://zh.wikipedia.org/wiki/聖路易奧運會 "wikilink")，女子拳擊被列入表演賽。，不過往後的數十年裡，由於許多國家發佈禁令，女子拳擊運動的發展因而停頓下來，直到1950年代才逐漸恢復。

1975年，[美國](https://zh.wikipedia.org/wiki/美國 "wikilink")[內華達州率先給予女子拳擊運動員資格認定](https://zh.wikipedia.org/wiki/內華達州 "wikilink")。1993年，美國拳擊協會正式認定女子拳擊運動員，開啟女子拳擊的時代。

1990年，[女子國際拳擊聯盟](https://zh.wikipedia.org/wiki/女子國際拳擊聯盟 "wikilink")（WIBF）創立。2001年，[國際拳擊協會在](https://zh.wikipedia.org/wiki/國際拳擊協會 "wikilink")[美國創辦首屆](https://zh.wikipedia.org/wiki/美國 "wikilink")[女子世界拳擊錦標賽](https://zh.wikipedia.org/wiki/女子世界拳擊錦標賽 "wikilink")。2009年8月14日，国际奥委会主席[罗格在](https://zh.wikipedia.org/wiki/罗格 "wikilink")[柏林宣布](../Page/柏林.md "wikilink")，女子拳击项目将成为2012年伦敦奥运会正式比赛项目。

## 女子拳擊的分級

和男子拳擊一樣，女子拳擊比賽也按照參賽者的體重來分組。現時女子拳擊比賽可分為以下13個級別：

| 階級名稱               | 體重（公斤） |
| ------------------ | ------ |
| 重量級（Heavy）         | 80－86  |
| 輕重量級（Light Heavy）  | 75－80  |
| 中量級（Middle）        | 70－75  |
| 輕中量級（Light Middle） | 66－70  |
| 沉量級（Welter）        | 63－66  |
| 輕沉量級（Light Welter） | 60－63  |
| 輕量級（Light）         | 57－60  |
| 羽量級（Feather）       | 54－57  |
| 雛量級（Bantam）        | 52－54  |
| 輕雛量級（Light Bantam） | 50－52  |
| 蠅量級（Fly）           | 48－50  |
| 輕蠅量級（Light fly）    | 46－48  |
| 別針級（Pin）           | 46     |

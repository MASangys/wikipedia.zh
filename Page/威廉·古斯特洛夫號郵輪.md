**威廉·古斯塔夫號**（）是一艘[納粹德國](../Page/納粹德國.md "wikilink")[郵輪](https://zh.wikipedia.org/wiki/郵輪 "wikilink")，由[布羅姆與沃斯船塢](../Page/布洛姆－福斯.md "wikilink")（Blohm + Voss）所建造，她的名字是來自[納粹黨](https://zh.wikipedia.org/wiki/納粹黨 "wikilink")[瑞士分部的領袖](https://zh.wikipedia.org/wiki/瑞士 "wikilink")，於1937年5月5日下水。在[二戰末期](https://zh.wikipedia.org/wiki/二戰 "wikilink")，此船用來載運遭[蘇聯紅軍圍困在](https://zh.wikipedia.org/wiki/蘇聯紅軍 "wikilink")[東普魯士](../Page/東普魯士.md "wikilink")的[德國人](https://zh.wikipedia.org/wiki/德國人 "wikilink")（包括平民及少數官兵），最後在1945年1月30日於[波羅的海被蘇軍](https://zh.wikipedia.org/wiki/波羅的海 "wikilink")[潛艇](../Page/潛艇.md "wikilink")發射三枚[魚雷](../Page/魚雷.md "wikilink")擊沉。遇難人數估計由5,300至9,931人不等\[1\]\[2\]，超過[泰坦尼克号的傷亡人數六倍](https://zh.wikipedia.org/wiki/泰坦尼克號 "wikilink")，成為歷史上遇難人數最多的[海難](https://zh.wikipedia.org/wiki/海難列表 "wikilink")。湊巧的是，當天正是威廉·古斯特洛夫50歲冥壽，也是[希特勒上台的](https://zh.wikipedia.org/wiki/希特勒 "wikilink")12週年。

## 歷史

這艘船原是作為納粹德國“[力量来自欢乐](../Page/力量来自欢乐.md "wikilink")”组织（，缩写KdF，一个專責為工人提供文娛康樂活動的工會組织）的郵輪，價值約為二千五百萬[帝國馬克](https://zh.wikipedia.org/wiki/帝國馬克 "wikilink")（RM）。直至1939年為止，威廉·古斯特洛夫號是KdF的郵船隊的[旗艦](https://zh.wikipedia.org/wiki/旗艦 "wikilink")。

1939年9月，納粹德國開始對[歐洲各國發動侵略](https://zh.wikipedia.org/wiki/歐洲 "wikilink")，威廉·古斯特洛夫號被軍隊徵用作[醫療船至](https://zh.wikipedia.org/wiki/醫療船 "wikilink")1940年11月。之後這艘船被用作[運兵船](https://zh.wikipedia.org/wiki/運兵船 "wikilink")，負責運載[U-潛艇的見習乘員到東普魯士](https://zh.wikipedia.org/wiki/U-潛艇 "wikilink")[但澤灣](https://zh.wikipedia.org/wiki/但澤灣 "wikilink")（，現[波蘭](https://zh.wikipedia.org/wiki/波蘭 "wikilink")[格但斯克灣](https://zh.wikipedia.org/wiki/格但斯克灣 "wikilink")（Bay of Gdańsk））畔的[哥德哈芬港](https://zh.wikipedia.org/wiki/哥德哈芬 "wikilink")（，現波蘭[格丁尼亞](https://zh.wikipedia.org/wiki/格丁尼亞 "wikilink")（））接受訓練。

## 沉沒

### 最後一程

威廉·古斯特洛夫號最後的任務是由格丁尼亞撤離德國平民、傷兵及海員經波羅的海返回德國[基爾](https://zh.wikipedia.org/wiki/基爾 "wikilink")（Kiel）港，船上已登記的海員及乘客總數是6,050人，但這數字並未計算沒有紀錄的數千名[難民](https://zh.wikipedia.org/wiki/難民 "wikilink")。於1980及1990年代對威廉·古斯特洛夫號沉沒事件作出過深入的研究，結論出當時船上搭載著173名船員（海軍武裝輔助隊）、918名第二潛艇訓練師（2.Unterseeboot-Lehrdivision）的士官及士兵、373名女性海軍醫護人員、162名受重傷的士兵及8,956名難民（當中只有4,424名難民有紀錄），合共10,582人。\[3\]這艘船本來的設計只能搭載2,000人以下，但是在充分利用所有船上的公共空間及遊樂設施後，就可以搭載上述人數作短程航行。也就是因為空間有限，船上不能帶太多救生設備，數量只夠少過半數人使用。

1945年1月30日的早上，威廉·古斯特洛夫號離開格丁尼亞港，隨行的船隻有滿載難民的（Hansa）和兩艘[魚雷艇](https://zh.wikipedia.org/wiki/魚雷艇 "wikilink")。漢薩號和其中一艘魚雷艇發生故障後不能繼續航行，只剩下另外一艘魚雷艇獅子號（Löwe）\[4\]負責護送威廉·古斯特洛夫號。船上有四位[船長](../Page/船長.md "wikilink")，其中三位是民航船長，一位是海軍船長，他們在決定航線上意見出現分歧。曾當潛艇乘員的威廉·扎恩海軍中尉（Lieutenant Commander Wilhelm Zahn）建議把船駛在近岸的淺水中並關掉所有燈火，以避免受蘇軍潛艇伏擊; 經驗較豐富的總船長弗德里希·彼德遜（Friedrich Petersen）則持不同看法，決定把船駛出深水區。當他從[無線電得知德國](https://zh.wikipedia.org/wiki/無線電 "wikilink")[掃雷艦](../Page/掃雷艦.md "wikilink")正迎面駛來時，彼德遜決定開啟船上紅綠兩色的[導航燈以避免相撞](https://zh.wikipedia.org/wiki/導航燈 "wikilink")。但這個錯誤的決定使威廉·古斯特洛夫號的位置在夜間暴露出來。

由領導的蘇軍[S级潛艇](https://zh.wikipedia.org/wiki/S级潛艇 "wikilink")在德軍忽略下冒險駛到但澤灣找尋建功的好機會，果然讓他在格羅森多夫（、，也就是今日的[弗瓦迪斯瓦沃沃](../Page/弗瓦迪斯瓦沃沃.md "wikilink")）及黎巴（、，今名[韋巴](https://zh.wikipedia.org/wiki/韋巴 "wikilink")）之間離岸30[公里](../Page/公里.md "wikilink")的海面上發現了威廉·古斯特洛夫號和獅子號的行蹤，在當地時間晚上九時（[UTC](https://zh.wikipedia.org/wiki/協調世界時 "wikilink")+1），希特勒在[電台上發表執政](https://zh.wikipedia.org/wiki/電台 "wikilink")12週年的演講完結不久後，馬連尼斯高下令向他以為是軍艦的威廉·古斯特洛夫號發射四枚魚雷，四枚魚雷分別以[俄文寫上](https://zh.wikipedia.org/wiki/俄文 "wikilink")：

  - 「為了祖國」（）
  - 「為了[史太林](https://zh.wikipedia.org/wiki/史太林 "wikilink")」（）
  - 「為了蘇聯人民」（）
  - 「為了[列寧格勒](https://zh.wikipedia.org/wiki/列寧格勒 "wikilink")」（）

除了寫上「為了[史太林](https://zh.wikipedia.org/wiki/史太林 "wikilink")」的一枚魚雷卡在發射管外，其餘三枚全部命中目標。\[5\]第一枚擊中左舷的船頭，第二枚鑽進位於船身中間的游泳池，第三枚擊中尾部的[發動機房](https://zh.wikipedia.org/wiki/發動機 "wikilink")，爆炸點附近的乘客即時喪命，包括擠在游泳池的女性醫護人員，而船上其他乘客都聽到響亮的爆炸聲。隨之而來的，是船上乘客的驚惶失措。有些難民紛紛湧向[救生艇及爭奪](https://zh.wikipedia.org/wiki/救生艇 "wikilink")[救生衣](https://zh.wikipedia.org/wiki/救生衣 "wikilink")，在混亂中好些救生設備掉下海去。有些人被湧入的海水溺死，有些是因為發生恐慌時在樓梯裡或[甲板上遭踩死或壓死](https://zh.wikipedia.org/wiki/甲板 "wikilink")，也有些乘客跳進冰冷的海水裡凍斃。一月份的波羅的海水溫平均是[攝氏](https://zh.wikipedia.org/wiki/攝氏 "wikilink")4度，但當天是特別嚴寒的一天，氣溫只有攝氏-10至-18度，海面上有浮冰。有報告指出一些兒童及嬰孩在混亂中與家人失散，也有一些兒童因穿上不合身的成人救生衣而導致在水中頭下腳上，最後溺斃。

被魚雷擊中50分鐘後，威廉·古斯特洛夫號終於沉入45米深的海底。之後，德軍方面派出艦艇進行搜救，以下是獲艦艇救起的生還者數字：

|                   |          |
| ----------------- | -------- |
| **搜救艦艇**          | **獲救人數** |
| 魚雷艇 T-36          | 564      |
| 魚雷艇獅子號（Löwe）      | 472      |
| 掃雷艦 M387          | 98       |
| 掃雷艦 M375          | 43       |
| 掃雷艦 M341          | 37       |
| 汽船哥丁根號（Gottingen） | 28       |
| 魚雷回收艇 TF19        | 7        |
| 貨輪哥特蘭號（Gotland）   | 2        |
| 巡邏艇 1703          | 1        |
| **總數**            | **1252** |

以上是海恩茨·熊恩研究得出的數字，從而推算出是次海難死亡的人數大約9,330人，成為世界航海史上最嚴重的海難。\[6\]

艾雲·卡比斯（Irwin Kappes）在《The Greatest Marine Disaster in History... and why you probably never heard of it》一書中提到威廉·古斯特洛夫號當時載著超過6,000名乘客。事發15分鐘後，魚雷艇獅子號趕到並盡量把生還者救起。本身載著1,500名[難民的郵輪希柏上將號](https://zh.wikipedia.org/wiki/難民 "wikilink")（Admiral Hipper），船長漢尼希斯特（Henigst）接獲瞭望台報告受[魚雷](../Page/魚雷.md "wikilink")襲擊後，仍然選擇繼續冒險救起海面上的生還者。卡比斯並確實地指出海難中的死亡人數應為5,348，但他並沒有在書中提出有關資料來源。\[7\]

申恩的研究中對死難者數目作出不同方法的估算。[探索頻道](../Page/探索頻道.md "wikilink")（Discovery Channel）節目《未解的歷史》（Unsolved History）中採用了電腦軟件maritime EXODUS根據目擊者有關船上載重密度的報告、模擬逃生路線以及沉船後隨時間過去的生還率作出分析，估算出死難人數達9,400人（以原本10,600位乘客計算）。\[8\]\[9\]\[10\]

### 爭議

在[二戰中](https://zh.wikipedia.org/wiki/二戰 "wikilink")，[同盟國和](https://zh.wikipedia.org/wiki/同盟國 "wikilink")[軸心國都有不少搭載平民的船隻被擊沉](https://zh.wikipedia.org/wiki/軸心國 "wikilink")\[11\]，當中以威廉·古斯特洛夫號最為慘烈，成為歷史上死難者最多的海難。1999年[諾貝爾文學獎得主](https://zh.wikipedia.org/wiki/諾貝爾文學獎 "wikilink")[君特·格拉斯](../Page/君特·格拉斯.md "wikilink")（Günter Grass）於2003年4月8日[紐約時報的訪問中提到](https://zh.wikipedia.org/wiki/紐約時報 "wikilink")，其中一個令他寫作《[蟹行](../Page/蟹行.md "wikilink")》（"Krebsgang"）的原因是他不滿一些德國人的言論，說威廉·古斯特洛夫號事件是一樁戰爭罪行。而他覺得並不如此，他認為這樁事件只是戰爭引致的悲慘結果。\[12\]

## 殘骸

威廉·古斯特洛夫號的沉沒位置為[北緯](https://zh.wikipedia.org/wiki/北緯 "wikilink")55.07度，[東經](https://zh.wikipedia.org/wiki/東經 "wikilink")17.41度的海底，於波蘭弗瓦迪斯瓦沃沃（）以西，韋巴（）以東，離岸30公里。沉船附近之海域被劃為戰爭紀念遺址，於波蘭航海圖上以「第73號障礙物」標示。\[13\]

2006年，船上的一個[鐘被](https://zh.wikipedia.org/wiki/鐘 "wikilink")[打撈](../Page/打撈.md "wikilink")起來，用作波蘭一家海鮮餐廳的裝飾，之後被借到[柏林](../Page/柏林.md "wikilink")的「Forced Path」展覽中展出。\[14\]

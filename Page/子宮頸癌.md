**子宮頸癌**又稱**宮頸癌**（），為發生在[子宮頸](../Page/子宮頸.md "wikilink")的[癌症](../Page/癌症.md "wikilink")\[1\] ，源自於不正常[細胞的生長](https://zh.wikipedia.org/wiki/細胞 "wikilink")，甚至能侵襲或轉移至身體其他部位\[2\]。早期通常並不會有症狀，而晚期時可能有不正常的、 \[3\]。

超過90%的子宮頸癌患者曾被[人類乳突病毒](../Page/人類乳突病毒.md "wikilink")（Human papillomavirus，HPV）感染\[4\] \[5\]；但相對而言，大部份感染的人並不會發展成癌症\[6\]\[7\]。其他風險因素包括[抽煙](../Page/吸菸.md "wikilink")、[免疫不全](../Page/免疫抑制.md "wikilink")、常使用[避孕藥](https://zh.wikipedia.org/wiki/複合口服避孕藥 "wikilink")、不[安全性行為](../Page/安全性行為.md "wikilink")、多重[性伴侶](../Page/性伴侶.md "wikilink")等，但皆不如人類乳突病毒感染重要\[8\]\[9\]。典型的子宮頸癌由為期10至20年的發展而來\[10\]。子宮頸癌可分為幾種型態，90%為[鱗狀細胞癌](https://zh.wikipedia.org/wiki/鱗狀細胞癌 "wikilink")，10%為，而其他型態僅佔少數\[11\]。診斷通常藉由，再進一步[切片檢查](../Page/活體組織切片.md "wikilink")。[醫學影像](../Page/醫學影像.md "wikilink")則用於察看是否有癌症轉移\[12\]。

人類乳突病毒有分為許多子型，其中兩型的感染特別容易造成子宮頸癌的產生。[人類乳突病毒疫苗可保護受施打者不受兩型](https://zh.wikipedia.org/wiki/人類乳突病毒疫苗 "wikilink")[病毒](../Page/病毒.md "wikilink")的侵犯，因而可預防65％至75％的子宮頸癌\[13\]\[14\]。然而因為尚有部分的罹癌風險，預防指引仍建議持續規則接受[子宮頸抹片檢查](https://zh.wikipedia.org/wiki/巴氏塗片檢查 "wikilink")\[15\]。其他預防方法包括不從事性行為及使用保險套\[16\]。子宮頸癌篩檢透過抹片檢查、或使用[醋酸來辨別癌前病變](https://zh.wikipedia.org/wiki/醋酸 "wikilink")，若針對癌前病變進一步治療可預防其發展成癌症\[17\]。子宮頸癌的治療包含合併[手術](https://zh.wikipedia.org/wiki/手術 "wikilink")、[化學治療](https://zh.wikipedia.org/wiki/化學治療 "wikilink")、[放射線治療](https://zh.wikipedia.org/wiki/放射線治療 "wikilink")\[18\]。在美國，[五年整體存活率為](https://zh.wikipedia.org/wiki/五年整體存活率 "wikilink")68%\[19\]，影響預後的關鍵主要在於早期發現\[20\]。

在全世界，子宮頸癌是第四常見的癌症，也是女性第四常見癌症死因\[21\]。在2012年，預估有528,000例的子宮頸癌，並造成26.6萬人死亡\[22\]，分別佔癌症發生率與死因的8%\[23\]。大約70%的子宮頸癌發生在[開發中國家](https://zh.wikipedia.org/wiki/開發中國家 "wikilink")\[24\]，而在低收入國家則為最常見癌症死因\[25\]。在[已開發國家](../Page/已開發國家.md "wikilink")，因為子宮頸抹片篩檢的普及，大大降低子宮頸癌的發生率\[26\]。醫學研究中，著名的[細胞株](https://zh.wikipedia.org/wiki/細胞培養 "wikilink")─[Hela細胞](https://zh.wikipedia.org/wiki/海拉細胞 "wikilink")，便是從名為這位女性身上的子宮頸癌細胞所培養出來\[27\]\[28\]。

## 病理

子宮頸癌是由於人類乳突病毒長期慢性感染子宮頸上皮細胞，導致[上皮細胞](https://zh.wikipedia.org/wiki/上皮細胞 "wikilink")[化生不良](https://zh.wikipedia.org/wiki/化生不良 "wikilink")（Epithelial cell dysplasia），進而癌化。其中第十六型人類乳突病毒慢性感染易造成[鱗狀細胞癌](https://zh.wikipedia.org/wiki/鱗狀細胞癌 "wikilink")；第十八型人類乳突病毒慢性感染易造成[腺癌](https://zh.wikipedia.org/wiki/腺癌 "wikilink")。[流行病學上](https://zh.wikipedia.org/wiki/流行病學 "wikilink")，第十六型人類乳突病毒感染的[盛行率高於第十八型人類乳突病毒感染](https://zh.wikipedia.org/wiki/盛行率 "wikilink")，是故臨床上子宮頸癌以[鱗狀細胞癌表現為主](https://zh.wikipedia.org/wiki/鱗狀細胞癌 "wikilink")，[腺癌則相對少見](https://zh.wikipedia.org/wiki/腺癌 "wikilink")。

以陰道內診可發現病灶處出現如草莓表面般的punctation pattern，嚴重時這些punctation pattern甚至會相連成Mosaic pattern的血管網。子宮頸切片[顯微鏡](../Page/顯微鏡.md "wikilink")下可發現細胞[化生不良](https://zh.wikipedia.org/wiki/化生不良 "wikilink")（dysplasia，依化生不良的上皮細胞厚度決定CIN classification）。子宮頸抹片在[顯微鏡](../Page/顯微鏡.md "wikilink")下可觀察到[癌細胞常見的](https://zh.wikipedia.org/wiki/癌細胞 "wikilink")[高核質比](https://zh.wikipedia.org/wiki/高核質比 "wikilink")（High N/C ratio）及[病毒](../Page/病毒.md "wikilink")感染常見的[細胞挖空現象](https://zh.wikipedia.org/wiki/細胞挖空現象 "wikilink")（koilocytosis）。

[宮頸上皮內瘤樣病變](https://zh.wikipedia.org/wiki/宮頸上皮內瘤樣病變 "wikilink")（Cervical intraepithelial neoplasia, CIN）分為三個級別。

## 危險因子與致病機轉

一般認為，人類乳突病毒感染、多重[性伴侶](../Page/性伴侶.md "wikilink")、太年輕發生性經驗、[抽菸](https://zh.wikipedia.org/wiki/抽菸 "wikilink")、長期使用[避孕藥](https://zh.wikipedia.org/wiki/避孕藥 "wikilink")，是子宮頸癌的危險因子。

### 人類乳突病毒感染

人類乳突病毒感染是最根本、最重要原因。常見的高危險株包含第16、18、31、33型等，而以第16型和第18型最為重要。人類乳突病毒的E6與E7基因被認為和致病最有關係，他們抑制了[p53以及](https://zh.wikipedia.org/wiki/p53 "wikilink")[Rb兩個腫瘤抑制基因](https://zh.wikipedia.org/wiki/Rb "wikilink")。

人類乳突病毒會從受傷的子宮頸上皮、以及子宮頸上皮的「鱗狀柱狀上皮過渡區」（即transformation zone或稱squamous-columnar junction）感染細胞。雖然絕大部分的感染屬急性感染，惟一旦演變成慢性感染便容易癌化。多重性伴侶者會增加感染頻率，使得慢性感染的可能性增加。

### 太年輕發生性經驗

女性的[子宮頸](../Page/子宮頸.md "wikilink")口「鱗狀柱狀上皮過渡區」在年輕時分佈在靠外側處，隨著年齡、[生產數增加](https://zh.wikipedia.org/wiki/生產 "wikilink")，這個過渡區會往子宮內部移動。過渡區分佈越靠外側，則受[人類乳突病毒](../Page/人類乳突病毒.md "wikilink")感染的機會就越大。所以越早有[性行為的女性](https://zh.wikipedia.org/wiki/性行為 "wikilink")，其往後的歲月裡發生子宮頸癌的機率也越高。

## 流行病學

子宮頸癌的發生率、年齡等流行病學的資料因國家民族而異。在子宮頸抹片篩檢普及的國家，癌前病變（鱗狀上皮內病變）以及早期子宮頸癌的診斷較多，是以[發生率高而](https://zh.wikipedia.org/wiki/發生率 "wikilink")[死亡率](../Page/死亡率.md "wikilink")低。

## 症狀

早期的子宮頸癌可能完全沒有症狀，可能的表現有陰道出血、接觸出血甚至腫塊等。後期的子宮頸癌的症狀則因侵入或擴散的器官而異。

## 診斷

子宮頸癌以子宮頸抹片作為篩檢工具，子宮頸抹片異常再進一步做其他的檢查或是追蹤。

子宮頸切片通常先以子宮頸鏡檢查，目視或是靠[醋酸辨識病變處](https://zh.wikipedia.org/wiki/醋酸 "wikilink")，再予以切片，也常配合子宮內頸刮搔。

子宮頸的[癌前病變有一些分類系統](https://zh.wikipedia.org/wiki/癌前病變 "wikilink")：

  - 異生：輕微（mild dysplasia）、中等（moderate dysplasia）、嚴重（severe dysplasia）
  - 子宮頸上皮內內贅瘤：1、2、3（cervical intraepithelial neoplasia, CIN, CIN1, CIN2, CIN3）
  - 鱗狀上皮內病變：高度、低度（squamous intraepithelial lesion, SIL; high grade, HSIL; low grade, LSIL）

通常低度鱗狀上皮內病變以追蹤為主。高度鱗狀上皮內病變由於與子宮頸癌的相關度高，通常需要手術切除。LEEP或是圓錐形切除法是常用的方式，但仍須綜合病人的臨床狀況而決定。

## 分期

分期子宮頸癌的分期通常是用FIGO系統：

| 分期      | 描述                                     |
| ------- | -------------------------------------- |
| **第一期** | **癌細胞局限在子子宮頸**                         |
| 第一期A1   | 顯微鏡下觀察之侵犯小於等於0.3公分                     |
| 第一期A2   | 顯微鏡下觀察之侵犯大於0.3公分，但小於0.5公分橫徑散佈小於0.7公分   |
| 第一期B1   | 癌細胞侵犯大大於0.5公分，或橫徑散佈大於0.7公分，但腫瘤大小則小於4公分 |
| 第一期B2   | 同上但腫瘤大小則大於4公分                          |
| **第二期** | **癌細胞侵犯超過子宮頸，但未達到骨盆腔壁**                |
| 第二期A    | 有陰道壁之侵犯，但未達下三分之一之陰道壁                   |
| 第二期B    | 有明顯子宮旁之侵犯。                             |
| **第三期** | **癌細胞侵犯達到骨盆腔壁，或下三分之一之陰道壁之侵犯，或有水腎情形者**  |
| 第三期A    | 癌細胞侵犯未達到骨盆腔壁                           |
| 第三期B    | 遠處器官之侵犯                                |
| **第四期** | **癌細胞侵犯超出骨盆腔，或有膀胱或直腸之侵犯**              |
| 第四期A    | 子宮頸附近器官之侵犯                             |
| 第四期B    | 遠處器官之侵犯                                |

特別值得注意的是此分期和子宮頸上皮內內贅瘤的分級經常被大眾或媒體混淆，實則截然不同。

## 疫苗

|              | 嘉喜（Gradasil MSD） | 保蓓（Cervirax GSK） |
| ------------ | ---------------- | ---------------- |
| **預防的HPV類型** | 6, 11, 16, 18    | 16, 18           |
| **預防疾病**     | 癌症、濕疣            | 癌症               |
| **佐劑**       | 鋁硫酸鹽             | 氧化砷              |

一般來說，子宮頸癌疫苗可以有效預防接近100%的第16、18型人類乳突病毒感染。但是子宮頸癌大約6\~7成是由第16、18型人類乳突病毒感染導致，此[疫苗](../Page/疫苗.md "wikilink")無法有效預防剩下3\~4成子宮頸癌'之病因。所以即便有施打過子宮頸癌疫苗，建議有過性行為之婦女，仍然需要每年進行子宮頸抹片篩檢。

## 治療

  - IA1子宮切除術
  - IA2子宮切除術加上淋巴結廓清

上述二者也可考慮LEEP或是圓錐形切除法

  - IB1或IIA\<4毫米：根除性子宮切除術，加上淋巴結廓清或者放射線治療
  - IB2或IIA\>4毫米：合併手術、化學治療、放射線治療
  - IIB以上 化學治療和放射線治療

在[计算机科学中](../Page/计算机科学.md "wikilink")，**子程式**（，），是一個大型程式中的某部份程式碼，由一个或多个语句块组成。它負責完成某項特定-{zh-cn:任务;
zh-tw:工作;}-，而且相較於其他程式碼，具備相對的獨立性。

一般会有输入参数并有-{zh-cn:返回值;
zh-tw:傳回值;}-，提供对过程的封装和细节的隐藏。这些代码通常被整合为[软件库](https://zh.wikipedia.org/wiki/软件库 "wikilink")。

函数在[面向过程的语言中已经出现](https://zh.wikipedia.org/wiki/面向过程 "wikilink")。是结构（）和[-{zh-cn:类;
zh-tw:類別;}-](https://zh.wikipedia.org/wiki/类 "wikilink")（）的前身。本身就是对具有相关性语句的归类和对某过程的抽象。

## 分類

子程式是一個概括性的術語，子程式是所有高階程式所稱。它經常被使用在[組合語言層級上](https://zh.wikipedia.org/wiki/組合語言 "wikilink")。子程式的主體（）是一個程式碼區塊，當它被呼叫時就會進入執行。

### 函數

**函数**（）是一種子程式，利用函数名稱，可以接收回傳值。例如：

``` c
c = max(a,b);
```

### 过程

**过程**（）是一種子程式，它能夠接受不同的引數，來執行某些特別的動作。例如：

``` c
printf("Hello World\n");
```

函数與过程在某些程式語言中是被分開的，但是在[C語言中](https://zh.wikipedia.org/wiki/C語言 "wikilink")，所有程序都是[函数](../Page/函数.md "wikilink")，這兩者被認為是相同的。這有可能造成一些副作用，特別是在回傳值是`void`[型別時](https://zh.wikipedia.org/wiki/型別 "wikilink")。

### 方法

在[物件導向程式設計語言中](https://zh.wikipedia.org/wiki/物件導向程式設計 "wikilink")，[類別或](../Page/类_\(计算机科学\).md "wikilink")[物件中的子程式](https://zh.wikipedia.org/wiki/对象_\(计算机科学\) "wikilink")，被稱為**方法**（）。

[HKJC_Double_Trio.jpg](https://zh.wikipedia.org/wiki/File:HKJC_Double_Trio.jpg "fig:HKJC_Double_Trio.jpg") **孖T**（）是[香港賽馬](../Page/香港賽馬.md "wikilink")其中一個設有多寶彩金 (Jackpot) 的投注種類，由[香港賽馬會](../Page/香港賽馬會.md "wikilink")接受投注，在指定的連續兩場賽事均選中第一名、第二名及第三名馬匹，毋須順序。

## 投注金額

若每票投注總額少於港幣$100，每注最低投注金額為港幣$10。若每票投注總額達港幣$100或以上，可以[靈活玩](../Page/靈活玩.md "wikilink")方式投注，而每注最低投注金額為港幣$1。

## 投注方法

與單T相若, 投注者可於指定2場場次中選擇單式，複式及馬膽方式投注。

|                |                                                                                                                          |
| -------------- | ------------------------------------------------------------------------------------------------------------------------ |
| **投-{注}-種類**   | **投-{注}-方法**                                                                                                             |
| 「單式」投-{注}-     | 從自選任何馬匹編號中選出3個不同馬匹編號。                                                                                                    |
| 「複式」投-{注}-     | 假設您的心水馬為6號、7號、8號、 9號，複式互串會將所選馬匹組成所有可能的組合，即分別為6+7+8、6+7+9、6+8+9 及 7+8+9，總共四注,如此類推。                                        |
| 「馬-{膽}-」投-{注}- | 當您對一匹馬有很大信心能跑入前三名，您可選擇以該匹馬作「馬膽」，其他選擇則作為「配腳」。 假設重心馬為6號，以6號胆拖7號、8號、9號，即分別為6+7+8、6+7+9 及 6+8+9，總共三注，只要其中一個馬匹組合同時跑入前三名，便為中彩。 |

## 注碼

孖T中獎需選中連續兩場賽事，總注碼為兩場賽事的注數互串，例如: 第一關選單式6號、7號、8號，第二關為6號、7號、8號、 9號, 即是第一關1注x第二關4注，總共4注。\[1\] \[2\]

## 中獎方法

在指定的連續兩場賽事均選中第一名、第二名及第三名馬匹，毋須順序。

## 賽事編排及多寶彩金

日馬賽事共編排三口孖T，夜馬賽事共編排兩口孖T，而每口孖T多寶的最低派彩金額為二百萬\*。

\*以每注$10一注獨得計，若無人選中該口孖T的勝出組合，彩金將會作為多寶撥入將來的賽事。

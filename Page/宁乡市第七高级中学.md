**宁乡市第七高级中学**，坐落在[湖南省](../Page/湖南省.md "wikilink")[长沙市](https://zh.wikipedia.org/wiki/长沙市 "wikilink")[宁乡市](../Page/宁乡市.md "wikilink")[流沙河镇](../Page/流沙河镇.md "wikilink")，毗邻[罘罳峰](../Page/罘罳峰.md "wikilink")，是下宁乡高级中学之一。\[1\]

## 历史

| 年份          | 名称        | 备注                                                                                                                                                                                                                                                  |
| ----------- | --------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1253年       | 林山寺       | 一个高僧乘舟来到草冲，建立了林山寺。这是宁乡七中的前身。\[2\]                                                                                                                                                                                                                   |
| 1917年       | 林山寺       | 林山寺附近的书生刘仲师，租借了林山寺的一间房子开办了私塾。\[3\]                                                                                                                                                                                                                  |
| 1922年到1925年 | 林山书院      | [王凌波](../Page/王凌波.md "wikilink")、张铸陶、何立前等将林山寺改造为林山书院，教授课程[国文](https://zh.wikipedia.org/wiki/中文 "wikilink")、[算术](../Page/算术.md "wikilink")、[天文](https://zh.wikipedia.org/wiki/天文 "wikilink")、[地理](https://zh.wikipedia.org/wiki/地理 "wikilink")。\[4\] |
| 1926年       | 林山学校      | 宁乡县第六高等小学从[上流寺迁移到林山书院](../Page/上流寺_\(宁乡市\).md "wikilink")，并且改名“林山学校”。两个学校合并之后，林山学校被宁乡县改称为“第五高小”。\[5\]                                                                                                                                               |
| 1956年       | 林山完小      | “林山学校”再次改名“林山完小”，又增加初中班。\[6\]                                                                                                                                                                                                                       |
| 1958年       | 宁乡县第七初级中学 | “林山完小”改为“宁乡县第七初级中学”，简称为“宁乡七中”，再次增设高中班。\[7\]                                                                                                                                                                                                         |
| 1982年       | 宁乡七中      | “宁乡七中”迁至[流沙河镇](../Page/流沙河镇.md "wikilink")“[五七干校](../Page/五七干校.md "wikilink")”。\[8\]                                                                                                                                                                |

[Ante_Kovacic.gif](https://zh.wikipedia.org/wiki/File:Ante_Kovacic.gif "fig:Ante_Kovacic.gif") [Family_tree_of_Ante_Kovacic.png](https://zh.wikipedia.org/wiki/File:Family_tree_of_Ante_Kovacic.png "fig:Family_tree_of_Ante_Kovacic.png") **安特·科瓦西奇** ( **Ante Kovačić** ，1854年 – 1889年 ) ，是[克罗地亚](../Page/克罗地亚.md "wikilink")的作家。他也是律师。他死于精神病院。\[1\]\[2\]

  - *Fiškal* ( 律师 )
  - *Smrt babe Čengićkinje*
  - *Baruničina ljubav* ( 男爵夫人的爱 )
  - *Među žabari*
  - *U registraturi* ( 登记处 )

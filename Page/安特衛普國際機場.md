**安特衛普國際機場**（，）是[比利时](../Page/比利时.md "wikilink")[安特卫普](../Page/安特卫普.md "wikilink")的[机场](https://zh.wikipedia.org/wiki/机场 "wikilink")，位于安特卫普东南3千米处。

## 历史

1923年在当地建成了一座[比利时空军](../Page/比利时空军.md "wikilink")的[軍用機場](../Page/軍用機場.md "wikilink")。1929年机场开始建造第一座候机厅，这座候机厅1930年启用。

[第二次世界大战](../Page/第二次世界大战.md "wikilink")德国占领时期机场被大力扩大并被[納粹德國空軍](../Page/納粹德國空軍.md "wikilink")使用。

以下德国空军主动部队（不包括训练和后勤部队）曾经从1940年到1944年驻扎在这里：

| 从       | 到       | 部队\[1\] |
| ------- | ------- | ------- |
| 1940年5月 | 1940年5月 | 第51歼击机队 |
| 1940年6月 | 1941年3月 | 第3战斗机队  |
| 1940年6月 | 1940年6月 | 第4战斗机队  |
| 1944年9月 | 1944年9月 | 第4歼击机队  |

德军从当地被驱逐后[英國皇家空軍](../Page/英國皇家空軍.md "wikilink")使用该机场。

1946年民用航空重新开始。1960年代初机场的使用率下降，机场几乎被关闭。但是关闭的危险被打消后候机厅和跑道甚至被扩大。

2014年10月27日安特卫普机场运营公司获得25年的[特许经营权](https://zh.wikipedia.org/wiki/特许经营 "wikilink")\[2\]。

**客家白話字**（）是一種用於書寫[客家话](../Page/客家话.md "wikilink")的[羅馬拼音系統](https://zh.wikipedia.org/wiki/羅馬拼音 "wikilink")，首度使用于1905年出版的《客英辭典》，1924年於[汕頭出版的客家話](https://zh.wikipedia.org/wiki/汕頭 "wikilink")《新約聖經》亦採用客家白話字書寫。1923年，臺灣苗栗客家籍的鍾天枝牧師曾被派往廣東學習客家白話字，並攜帶客家話聖詩返回台灣，但最終教會沒有使用\[1\]。白話字在粵東客家人當中的影響微乎其微，只有傳教士及部分客家人基督徒使用，並被逐漸遺忘。

## 方案

客家白話字的兩本主要著作基礎音系不同。根據作者（或譯者）生平和書中呈現的音韻特點來看，《客英辭典》基礎音系是粵東長樂縣（今[五華縣](https://zh.wikipedia.org/wiki/五華縣 "wikilink")）客家話，而《新約聖經》基礎音系是揭陽县五經富一帶（今[揭西縣](https://zh.wikipedia.org/wiki/揭西縣 "wikilink")[五經富鎮](https://zh.wikipedia.org/wiki/五經富鎮 "wikilink")）的客家話\[2\]。

### 聲調

客家白話字有6個聲調\[7\]。陰平與陰入都不用調號表示，而陰入有塞音韻尾，因此不會混淆。值得注意的是，客家白話字的調號與閩南白話字相同\[8\]。

|   調類    |      調號       |  範例   |
| :-----: | :-----------: | :---: |
|   陰平    |       無       | fu 夫  |
|   陽平    | <big>◌̂</big> | fû 乎  |
|   上聲    | <big>◌́</big> | fú 府  |
|   去聲    | <big>◌̀</big> | fù 富  |
| 陰入\[9\] |       無       | fuk 福 |
|   陽入    | <big>◌̍</big> | fk 服  |
|         |               |       |

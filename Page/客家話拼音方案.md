**客家話拼音方案**是[广东省教育部門於](../Page/广东省.md "wikilink")1960年9月公佈的[廣東拼音方案之一](../Page/廣東拼音方案.md "wikilink")，以[廣東省](https://zh.wikipedia.org/wiki/廣東省 "wikilink")[梅州市区](../Page/梅州市.md "wikilink")（含[梅江區](https://zh.wikipedia.org/wiki/梅江區 "wikilink")、[梅縣區](https://zh.wikipedia.org/wiki/梅縣區 "wikilink")）的[梅城口音為標準](https://zh.wikipedia.org/wiki/梅州话 "wikilink")，以[拉丁字母拼寫語音](../Page/拉丁字母.md "wikilink")，以右上方標數字表示[聲調](../Page/聲調.md "wikilink")。

## 方案內容

### 字母

拼音採用[拉丁字母](../Page/拉丁字母.md "wikilink")，其中r、w不用來拼寫梅縣客家話。ê 是字母 e 的變體，用以代表
。

### 聲母

聲母總數為19個。不以輔音為首的音節，稱為零聲母。列表如下：

|      |      |       |      |      |
| ---- | ---- | ----- | ---- | ---- |
| b  波 | p  婆 | m  摸  | f  火 | v  窩 |
| d  多 | t  拖 | n  挪  | l  羅 |      |
| g  哥 | k  科 | ng  我 | h  河 |      |
| j  擠 | q  妻 | x  西  |      |      |
| z  資 | c  雌 | s  思  |      |      |

資 zi<sup>1</sup>、雌 ci<sup>1</sup>、思 si<sup>1</sup> 等字的韻母用i表示。

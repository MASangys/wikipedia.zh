**密卡蘇奇語**(英語：Mikasuki
language，Mikasuki這詞英語又作Miccosukee，這語言有時亦稱作Hitchiti-Mikasuki
language)是一種由大約五百名居於佛羅里達州南部的人所使用的語言。它被[密卡蘇奇部族和許多](https://zh.wikipedia.org/wiki/密卡蘇奇部族 "wikilink")[塞米諾爾人所使用](https://zh.wikipedia.org/wiki/塞米諾爾人 "wikilink")。已絕滅的[希奇提語](https://zh.wikipedia.org/wiki/希奇提語 "wikilink")(Hitchiti
language)和密卡蘇奇語可互通。

這語言有三個聲調，分別為高、低和落三種，同時母音長短具辨義作用，像例如*eche* (意即「口」)和*eeche*
(意即「鹿」)之對比，及*ete* (意即「目」)和*eete* (意即「火」)間的對比，即其例也。

## 語法

名詞藉後綴來表達其語法意涵，以下為其例：

|        |        |                     |         |
| ------ | ------ | ------------------- | ------- |
| **後綴** | **功能** | **範例**              | **涵義**  |
|        |        | embaache            | 電池      |
| ot     | 主詞標明   | embaachot hampeepom | 電池變壞了   |
| on     | 受詞標明   | embaachon aklomle   | 我需要一個電池 |
| ee     | 問句標明   | embachee cheméèło?  | 你有電池嗎？  |
|        |        |                     |         |

獨立形的代詞存在，像例如aane意即「我」、chehne意即「你」、pohne意即「我們」等，但這些詞很少使用。一般都以動詞後綴來標明人稱。

聲調中的高聲一般以[尖音符標明](../Page/尖音符.md "wikilink")；低聲一般以[重音符標明](../Page/重音符.md "wikilink")。落聲一般以[尖音符加](../Page/尖音符.md "wikilink")[重音符的方式標明](../Page/重音符.md "wikilink")。在印刷上，在長音時[尖音符印在前一個母音](../Page/尖音符.md "wikilink")，[重音符則印在後一個母音上](../Page/重音符.md "wikilink")；而在其他狀況下，[尖音符一樣印在母音字母上](../Page/尖音符.md "wikilink")，但[重音符印在子音字母上](../Page/重音符.md "wikilink")。以下為其標註範例：

|       |       |        |
| ----- | ----- | ------ |
| 高聲    | 低聲    | 落聲     |
| á, áa | à, àa | áǹ, áà |
|       |       |        |

據了解，在較小心的言談中，一個\[ə\]的音會做為kl、kw和kn等輔音叢集的增音。

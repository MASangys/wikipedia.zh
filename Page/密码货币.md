**-{zh-hans:密码货币;zh-hk:加密貨幣;zh-tw:加密貨幣;}-**\[1\]（英文：Cryptocurrency，又译**密碼學货币**、**-{zh-hans:加密货币;zh-hk:加密貨幣;zh-tw:密碼貨幣;}-**）是一種使用[密碼學原理來確保交易安全及控制交易單位創造的](https://zh.wikipedia.org/wiki/密碼學 "wikilink")[交易媒介](https://zh.wikipedia.org/wiki/交易媒介 "wikilink")。\[2\]

跟平常使用的紙幣需要防偽設計一樣，加密货币的防偽是利用數字貨幣和虛擬貨幣使用密碼學及數字雜湊而成並與智慧合約的綁定之下的新型通證。[比特幣在](../Page/比特币.md "wikilink")2009年成為第一個[去中心化的加密货币](https://zh.wikipedia.org/wiki/去中心化 "wikilink")，這之後加密货币一詞多指此類設計。\[3\] 自此之後數種類似的加密货币被創造，它們通常被稱作altcoins。\[4\]\[5\]\[6\]加密货币基于去中心化的共识机制\[7\]，與依赖中心化监管体系的银行金融系統相對。\[8\]

去中心化的性質源自於使用[分散式賬本](../Page/分散式賬本.md "wikilink")的[區塊鏈](https://zh.wikipedia.org/wiki/區塊鏈 "wikilink")（Blockchain）技術。\[9\]

## 定义

根据 Jan Lansky 所述，加密货币是满足六个条件的系统： \[10\]

1.  该系统无需中心机构，其状态通过分布式的共识机制得以维持。
2.  该系统能对加密货币及其所有权进行记录。
3.  该系统定义能否产生新的加密货币。如果可以，则系统需定义新币的来源，并定义如何确定这些新币的所有者。
4.  只能通过[密码学](../Page/密码学.md "wikilink")的方式来证明加密货币的所有权。
5.  该系统允许通过交易来改变加密货币的所有权。交易仅可从能证明加密货币当前所有权的实体发布。
6.  如果同一时间产生了两个改变相同加密货币所有权的指令，该系统最多只能执行其中一个。

2018年3月，加密货币（英语：cryptocurrency）一词被加入[韦氏词典](../Page/韦氏词典.md "wikilink")中。 \[11\]

## 架构

去中心化的加密货币是由整个加密货币系统共同生成的，加密货币的生成速率在系统创建时就已定义并公开。在中心化的银行和经济体系中（例如[联邦储备系统](../Page/联邦储备系统.md "wikilink")），企业董事会或政府通过印刷[法定货币或要求增加数字银行分类账的方式来控制货币的供应](https://zh.wikipedia.org/wiki/法定貨幣 "wikilink")。而对于去中心化的加密货币，公司或政府无法制造新的加密货币，且迄今为止还没有为持有资产价值的其他公司、银行或企业实体提供背书。分布式加密货币所依托的技术体系由名为[中本聪](../Page/中本聪.md "wikilink")的团体或个人所构建。\[12\]

，已有超过1,800种加密货币标准。\[13\]在加密货币系统中，分类账的安全性、完整性和分类账的余额由许多被称作矿工的互不信任的实体所维护：他们使用计算机帮助验证交易、为交易添加[时间戳](https://zh.wikipedia.org/wiki/時間戳 "wikilink")，并根据特定的时间戳机制将交易添加至分类账中。\[14\]

在大多数加密货币的设计中，加密货币的货币产量将会逐渐减少，使得最终能够流通的货币的总额不超过设定的上限。\[15\]相比金融机构所持有的传统货币或是手持的[现金](https://zh.wikipedia.org/wiki/現金 "wikilink")，加密货币更难以被执法机构搜查或扣押。\[16\]这种困难源自加密货币所使用的密码学技术。

## 比特币與其他加密货币

[比特币](../Page/比特币.md "wikilink")是第一種以加密货币型態存在的數位貨幣，而比特币以外的同型態货币，又稱為山寨币、競爭幣（），可經由獨立創建、硬分叉、透過ICO發行的代幣等方式建立，一部分参考比特币、以太币等思想、原理、源代码产生的，与比特币相似的虚拟货币，目前有數千種以上，可公開交易的密码货币在流通。

新的加密貨幣可能在任何時間被任何人建立出來，具備更多的功能、更完善的隱私保護、更快的區塊生成時間、支援[智能合約](https://zh.wikipedia.org/wiki/智能合約 "wikilink")、不同的演算法與共識機制等。

由于比特币本身并没有权威的发行机构和国家政权来维持其权威性、唯一性，比特币与其模仿者之间只能平等地相处，即使其是最早、最知名、人們最熟悉的虚拟货币，具有最大的使用者网络社区以及很强的网络效应，大部分時間也是市值最高的密码货币，但是并不具有绝对排它的地位。\[17\]\[18\]\[19\]

## 加密货币與其他貨幣模式的比較

人類歷史上，自有貨幣體制以來，主要出現了3種不同貨幣型態：[贵金属](https://zh.wikipedia.org/wiki/贵金属 "wikilink")、[法定貨币](https://zh.wikipedia.org/wiki/法定貨币 "wikilink")、**加密货币**

|           |                        |                 |                |
| :-------: | :--------------------: | :-------------: | :------------: |
|  **比較**   |        **加密货币**        |     **贵金属**     |    **法定貨币**    |
|    示例     |          比特币           |       黃金        |       美元       |
|   呈現方式    |        電子錢包、私鑰         |      金條、金塊      |     現金、存款      |
|   发行方法    |        算法、工作證明         |      開採、提煉      |   发行债券，印鈔鑄幣    |
|   存放方式    |       電子檔案（區塊鏈）        |   存放于金庫、自己持有    |  自己持有、在金融机构存款  |
|  機構存放成本   |           免费           |   每年約0.15%到1%   |     存放款利差      |
|   交易成本    |        依交易狀況而定         |       較高昂       |  較低，幅度变化视国家而定  |
|   防偽方法    |       密码学及區塊鏈技術        |  物理上難以複製或人工合成   |   防偽設計、法律規範    |
|   實體運送    |    無須運送，有網路及私鑰即可使用     |    昂貴且有安全問題     | 比贵金属方便，但仍有安全問題 |
|   信用媒介    |        了解型信任机制         | 物理的稀少性与实用性、历史因素 | 國家或聯盟主權、黄金储备量  |
|   记录保存    |        區塊鏈自动記入         |      需另行處理      |     需另行處理      |
|   支付结算    |          分散式           |       不流通       |      中心化       |
|   稀有性來源   |       通常設有最高發行量        |     物理與技術限制     |      央行控制      |
|    鉴定     |          區塊鏈           |     物理化學化验      |  信赖来源、各種驗鈔方式   |
|    凍結     |     只要私鑰在，無法被扣押、凍結     |      會被扣押       |      會被凍結      |
|    隱私     |   原則匿名但公開，但部分幣種交易保密    |    存放時實名但保密     |    存款時實名但保密    |
| 一年内最大变化幅度 | 相當的大，一般不少于±1000%\[20\] |      ±10%       |    视国家状况而定     |

## 国家数字货币

中国、瑞典、英国和新加坡等的[中央银行](../Page/中央银行.md "wikilink")已经研究法定数字货币用例。\[26\]

  - e-Dinar 是[突尼斯共和国政府用区块链技术发行的数字货币](https://zh.wikipedia.org/wiki/突尼斯共和国 "wikilink")。也是第一个国家数字货币。\[27\]
  - eCFA 是[塞内加尔共和国政府用区块链技术发行的数字货币](https://zh.wikipedia.org/wiki/塞内加尔共和国 "wikilink")。\[28\]
  - 委內瑞拉的[石油幣](https://zh.wikipedia.org/wiki/石油幣 "wikilink")

然而，有評論指出，国家数字货币，只是將國家的法幣替換成上「數位、加密」的型態而已，政府仍可恣意操縱之，並不符合比特幣去中心化的理念。\[29\]

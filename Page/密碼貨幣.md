**-{zh-hans:密码货币;zh-hk:加密貨幣;zh-tw:加密貨幣;}-**\[1\]（英文：Cryptocurrency，又译**密碼學货币**、**-{zh-hans:加密货币;zh-hk:加密貨幣;zh-tw:密碼貨幣;}-**）是一種使用[密碼學原理來確保交易安全及控制交易單位創造的](https://zh.wikipedia.org/wiki/密碼學 "wikilink")[交易媒介](https://zh.wikipedia.org/wiki/交易媒介 "wikilink")。\[2\]

跟平常使用的紙幣需要防偽設計一樣，加密货币的防偽是利用數字貨幣和虛擬貨幣使用密碼學及數字雜湊而成並與智慧合約的綁定之下的新型通證。[比特幣在](../Page/比特币.md "wikilink")2009年成為第一個[去中心化的加密货币](https://zh.wikipedia.org/wiki/去中心化 "wikilink")，這之後加密货币一詞多指此類設計。\[3\] 自此之後數種類似的加密货币被創造，它們通常被稱作altcoins。\[4\]\[5\]\[6\]加密货币基于去中心化的共识机制\[7\]，與依赖中心化监管体系的银行金融系統相對。\[8\]

去中心化的性質源自於使用的[區塊鏈](https://zh.wikipedia.org/wiki/區塊鏈 "wikilink")（Blockchain）技術。\[9\]

根據一項預測，加密貨幣到2018年的市值預計將高達1-2萬億美元。\[10\]

## 比特币與其他加密货币

[比特币是第一種以加密货币型態存在的數位貨幣](../Page/比特币.md "wikilink")，而比特币以外的同型態货币，又稱為山寨币、競爭幣（），可經由獨立創建、硬分叉、透過ICO發行的代幣等方式建立，一部分参考比特币、以太币等思想、原理、源代码产生的，与比特币相似的虚拟货币，目前有數千種以上，可公開交易的密码货币在流通。

新的加密貨幣可能在任何時間被任何人建立出來，具備更多的功能、更完善的隱私保護、更快的區塊生成時間、支援[智能合約](https://zh.wikipedia.org/wiki/智能合約 "wikilink")、不同的演算法與共識機制等。

由于比特币本身并没有权威的发行机构和国家政权来维持其权威性、唯一性，比特币与其模仿者之间只能平等地相处，即使其是最早、最知名、人們最熟悉的虚拟货币，具有最大的使用者网络社区以及很强的网络效应，大部分時間也是市值最高的密码货币，但是并不具有绝对排它的地位。\[11\]\[12\]\[13\]

## 加密货币的特徵

|        |                   |
| :----: | :---------------: |
| **特徵** |     **加密货币**      |
|   舉例   |   比特币、萊特幣、以太币等    |
|   呈現   |      電子錢包、私鑰      |
|  发行方式  |      算法、工作證明      |
|  存放方式  |       電子檔案        |
|  存放成本  |        免费         |
|  交易成本  |      依交易狀況而定      |
|   防偽   |     密码学及區塊链技術     |
|   運送   |  無須運送，有網路及私鑰即可使用  |
|  信用媒介  |      區塊鏈共識机制      |
| 交易记录保存 |      區塊链自动记录      |
|  支付结算  |      去中心化分散式      |
| 稀有性來源  |   依協定，通常設有最高發行量   |
|  鉴定真偽  |      區塊链共識机制      |
| 凍結可能性  |  只要私鑰在，無法被扣押、凍結   |
|   隱私   | 原則匿名但公開，但部分幣種交易保密 |
| 相對法幣變動 |  變化劇烈，24小時內可達數十%  |

## 国家数字货币

中国、瑞典、英国和新加坡等的[中央银行已经研究法定数字货币用例](../Page/中央银行.md "wikilink")。\[19\]

  - e-Dinar
    是[突尼斯共和国政府用区块链技术发行的数字货币](https://zh.wikipedia.org/wiki/突尼斯共和国 "wikilink")。也是第一个国家数字货币。\[20\]
  - eCFA
    是[塞内加尔共和国政府用区块链技术发行的数字货币](https://zh.wikipedia.org/wiki/塞内加尔共和国 "wikilink")。\[21\]

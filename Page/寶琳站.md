**寶琳站**（）是[香港](../Page/香港.md "wikilink")[港鐵](../Page/港鐵.md "wikilink")[將軍澳綫](../Page/將軍澳綫.md "wikilink")在新界的終點站之一，位於[香港](../Page/香港.md "wikilink")[新界](../Page/新界.md "wikilink")[西貢區](../Page/西貢區.md "wikilink")[將軍澳](../Page/將軍澳.md "wikilink")[寶琳貿業路旁](https://zh.wikipedia.org/wiki/寶琳 "wikilink")，[新都城中心一至三期中間](https://zh.wikipedia.org/wiki/新都城中心 "wikilink")，2002年8月18日啟用，承建商是[前田建設](../Page/前田建設.md "wikilink")工業株式會社。寶琳站雖然鄰近[寶林邨](../Page/寶林邨.md "wikilink")，但寶琳站名則取自附近的街道[寶琳北路](https://zh.wikipedia.org/wiki/寶琳北路 "wikilink")，二者的「林」和「琳」寫法不同。

## 車站設計

寶琳站為半開放設計，月台、大堂同位於地面，設有空調。此站是少數可直接由月台層走出車站的地面車站之一，在將軍澳綫車站中，此乃獨一無二。

鐵路路軌由地面在隔音牆內慢慢下降至地底，在寶琳站南端一路沿著[景林邨](../Page/景林邨.md "wikilink")旁建有一個綠化[公園](https://zh.wikipedia.org/wiki/公園 "wikilink")，一方面覆蓋路軌，另一方面可以為區內提供綠化空間。

### 大堂

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Po Lam Station 2017 part4.jpg | caption1 = 大堂 | image2 = MTR TKL Opening Commemoration Plaque.JPG | caption2 = 將軍澳綫通車紀念碑記位於此站 }}

大堂非收費區內，及A、B出入口通道旁，均設有各類型的商店，如[便利店](../Page/便利商店.md "wikilink")、[時裝店](https://zh.wikipedia.org/wiki/時裝店 "wikilink")、[麵包糕餅店](https://zh.wikipedia.org/wiki/麵包糕餅店 "wikilink")、診所等，以及不少自助服務設施供乘客使用，包括[自動櫃員機](../Page/自動櫃員機.md "wikilink")、[自動售賣機和自動照相機](../Page/自動販賣機.md "wikilink")\[1\]。客務中心位於大堂中央，近出入口行人電動扶梯旁則設有[香港郵政](../Page/香港郵政.md "wikilink")[郵箱](../Page/郵箱.md "wikilink")，方便乘客投寄[郵件](../Page/郵件.md "wikilink")\[2\]\[3\]。而B出入口旁亦設有「[會員服務站](../Page/會員服務站.md "wikilink")」\[4\]。

### 月台

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Po Lam Station 2017 part1.jpg | caption1 = 1號月台﹙往北角方向﹚（北面） | image2 = Po Lam Station 2017 07 part2.jpg | caption2 = 1號月台﹙往北角方向﹚（南面） }} 與很多港鐵站不同，月台與大堂位處同一層。寶琳站只有一個[側式月台](../Page/側式月台.md "wikilink")，並設有[月台幕門](../Page/月台幕門.md "wikilink")。[列車只能在同一個方向到達和離開](https://zh.wikipedia.org/wiki/列車 "wikilink")，因此列車只會停留一段短時間，以免妨礙下班列車進站；正因如此設計，不時列車會為了保持[安全距離而在未到站時減速](../Page/閉塞_\(鐵路\).md "wikilink")，甚至停下，直至前方列車離開。另外，在車站1號月台後方，設有一條不對外開放的路軌供列車停泊，但列車不能從該路軌駛進1號月台上客；而列車進入1號月台後，將無法駛進該段路軌。

[Po_Lam_Station_2017_07_part3.jpg](https://zh.wikipedia.org/wiki/File:Po_Lam_Station_2017_07_part3.jpg "fig:Po_Lam_Station_2017_07_part3.jpg")

### 出入口

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Po Lam Station Exit A Shops 201405.jpg | caption1 = A出入口及商店 | image2 = Po Lam Station Exit B Podium 201405.jpg | caption2 = B3出入口及平台 | image3 = Po Lam Station 2018 05 part1.jpg | caption3 = C出入口 }}

寶琳站主要出入口設於貿業路，有五條[行人天橋連接港鐵站及公共運輸交匯處](https://zh.wikipedia.org/wiki/行人天橋 "wikilink")，可以直接由港鐵站前往[新都城](../Page/新都城.md "wikilink")各期和叠翠軒，亦可經行人天橋到達茵怡花園、寶林邨及欣明苑。車站外設有升降機，可以經C出口前往，連接行人天橋。

## 事件

### 港鐵站長被示威者毆打

2019年9月4日晚上約9時正，大批示威者無視車站張貼的法庭[禁制令到寶琳站集結要求站長交待就](https://zh.wikipedia.org/wiki/禁制令 "wikilink")8月31日晚上[將軍澳綫](../Page/將軍澳綫.md "wikilink")各站提早關閉的原因。期間，兩名寶琳站站長因保護一名受襲女乘客而被示威者毆打後需送院治理；至晚上約11時正，有示威者用雨傘破壞多部閘機，更將汽水倒進閘機零件內；至凌晨約零時，一名[調景嶺站](../Page/調景嶺站.md "wikilink")男站長李健宏放工後身穿便服乘搭港鐵抵達寶琳站後被示威者包圍毆打至倒地後需送院治理，最終兩名施襲示威者分別為22歲何梓聰和23歲鍾展鴻被防暴警察制服後拘捕。港鐵其後宣布，寶琳站關閉，將軍澳綫列車改以[坑口站為終點站](../Page/坑口站_\(香港\).md "wikilink")\[5\]\[6\]。

## 未來發展

### 東九龍綫

據《[鐵路發展策略2014](https://zh.wikipedia.org/wiki/鐵路發展策略2014 "wikilink")》，[香港特區政府建議興建](https://zh.wikipedia.org/wiki/香港特區政府 "wikilink")[東九龍綫](../Page/東九龍綫_\(2014年方案\).md "wikilink")，連接[將軍澳綫](../Page/將軍澳綫.md "wikilink")寶琳站及[觀塘綫](../Page/觀塘綫.md "wikilink")[鑽石山站](https://zh.wikipedia.org/wiki/鑽石山站 "wikilink")，走線途經[觀塘](../Page/觀塘.md "wikilink")半山北部，中途設置：[寶達站](https://zh.wikipedia.org/wiki/寶達站 "wikilink")、[秀茂坪站](https://zh.wikipedia.org/wiki/秀茂坪站 "wikilink")、[順天站及](https://zh.wikipedia.org/wiki/順天站 "wikilink")[彩雲站](https://zh.wikipedia.org/wiki/彩雲站 "wikilink")。報告指出東九龍綫可充當[將軍澳](../Page/將軍澳.md "wikilink")地區和[九龍](../Page/九龍.md "wikilink")之間行程的替代路線，如將軍澳綫的列車服務受阻，受影響的乘客可取道這條替代路線以轉乘觀塘綫\[7\]。

**导纳**（）是[电导](../Page/电导.md "wikilink")和[电纳的统称](https://zh.wikipedia.org/wiki/电纳 "wikilink")，在电力电子学中导纳定义为[阻抗](../Page/阻抗.md "wikilink")*(impedance)*的倒数，[符号](../Page/符号.md "wikilink")Y，单位是[西门子](https://zh.wikipedia.org/wiki/西門子_\(單位\) "wikilink")，简称西(S)。

## 把阻抗转换成导纳

阻抗Z由实数部分和虚数部分组成：

\[Z = R + jX \,\]

其中：

  -
    *R*是[电阻](../Page/电阻.md "wikilink")，单位为[欧姆](https://zh.wikipedia.org/wiki/欧姆 "wikilink")；
    *X*是[电抗](../Page/电抗.md "wikilink")，单位为[欧姆](https://zh.wikipedia.org/wiki/欧姆 "wikilink")。

\[Y = Z^{-1}= \frac{1}{R+jX} = \left( \frac{1}{R+jX} \right) \cdot \left( \frac{R-jX}{R-jX} \right) = \left( \frac{R}{R^2+X^2} \right) + j\left(\frac{-X}{R^2+X^2}\right)\]

导纳也是一个复数，由[实数](../Page/实数.md "wikilink")部分（电导*G*）和[虚数](../Page/虚数.md "wikilink")部分（电纳*B*）组成：

\[Y = G + j B \,\]

\[Y = G + jB = \left( \frac{R}{R^2+X^2} \right) + j \left( \frac{-X}{R^2+X^2} \right)\]

则*G*和*B*由以下两式给出：

\[G = \Re(Y) = \left( \frac{R}{R^2+X^2} \right)\]

\[B = \Im(Y) = \left( \frac{-X}{R^2+X^2}\right)\]

导纳的大小和相位为：

\[\left | Y \right | = \sqrt {G^2 + B^2} = \frac {1} {\sqrt {R^2 + X^2} }  \,\]

\[\angle Y = \arctan \left( {\frac{B}{G}} \right)= \arctan \left( {\frac{-X}{R}} \right)\]

其中：

  -
    *G*是[电导](../Page/电导.md "wikilink")，单位为[西门子](https://zh.wikipedia.org/wiki/西門子_\(單位\) "wikilink")；
    *B*是[电纳](https://zh.wikipedia.org/wiki/电纳 "wikilink")，单位为西门子。

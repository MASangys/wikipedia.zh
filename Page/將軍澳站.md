**將軍澳站**（）是[香港](../Page/香港.md "wikilink")[港鐵](../Page/港鐵.md "wikilink")[將軍澳綫](../Page/將軍澳綫.md "wikilink")的一個地下車站，位於[新界](../Page/新界.md "wikilink")[西貢區](../Page/西貢區.md "wikilink")[將軍澳市中心](../Page/將軍澳市中心.md "wikilink")[寶邑路](../Page/寶邑路.md "wikilink")一帶，於2002年8月18日正式啟用。

## 車站設計

將軍澳站西面將軍澳綫管道下層設有觀塘綫越位隧道\[1\]。車站由國際建築事務所[Aedas設計](https://zh.wikipedia.org/wiki/Aedas "wikilink")。

### 大堂

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Tseung Kwan O Station 2019 04 part2.jpg | caption1 = 大堂，近A出入口 | image2 = Tseung Kwan O Station 2019 04 part1.jpg | caption2 = 大堂，近B出入口 | image3 = TKO Station art.jpg | caption3 =車站藝術 《層層叠叠的傳統》 }} 大堂內設有不同類型的商店供乘客購物，例如有[便利店](../Page/便利商店.md "wikilink")、糖果店、[麵包糕餅店](https://zh.wikipedia.org/wiki/麵包糕餅店 "wikilink")、[時裝店等](https://zh.wikipedia.org/wiki/時裝店 "wikilink")，所有商店主要分佈在非收費區內，以及不少自助服務設施供乘客享用，包括[自動櫃員機](../Page/自動櫃員機.md "wikilink")、[自動售賣機和自動照相機](../Page/自動販賣機.md "wikilink")\[2\]。客務中心位於大堂中央C出入口前，旁邊設有[香港郵政](../Page/香港郵政.md "wikilink")[郵箱](../Page/郵箱.md "wikilink")方便乘客在車站內投寄郵件\[3\]\[4\]。而B2出入口旁亦設有「[會員服務站](../Page/會員服務站.md "wikilink")」\[5\]。

另外，將軍澳站大堂近C出口设有[艺术墙](../Page/港鐵文化.md "wikilink")《層層叠叠的傳統》。觀衆可領略繪畫過程前歷史的變遷。

### 出入口

將軍澳站现时设有5出入口供乘客进出。

另外，C出口在將軍澳站啟用初期為預留出入口，左右兩旁曾為[自動售票機的位置](https://zh.wikipedia.org/wiki/自動售票機 "wikilink")（現已改建為商鋪）。該位置連接[PopCorn](../Page/PopCorn.md "wikilink")商場一期、天晉、[香港九龍東皇冠假日酒店](../Page/香港九龍東皇冠假日酒店.md "wikilink")及[香港九龍東智選假日酒店](../Page/香港九龍東智選假日酒店.md "wikilink")，方便乘客不用繞經A或B出入口的行人路前往。站內店鋪亦已配合改建，C出入口已於2012年3月31日隨PopCorn商場開業而啟用\[6\]。

## 鄰接車站

## 利用狀況

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Tsang Kwan O Station Concourse 2010.jpg | caption1 = 2010年的大堂，當時尚未更換願景顯示屏 | image2 = Tsang Kwan O Station Exit C 201009.jpg | caption2 = C出入口預留位置（2010年9月） | image3 = Tseung Kwan O Station 2017 08 part5.jpg | caption3 = 願景顯示屏（班次顯示器） }} 將軍澳站啟用初期，因鄰近的住宅（[寶盈花園](../Page/寶盈花園.md "wikilink")、[將軍澳中心](../Page/將軍澳中心.md "wikilink")、[將軍澳廣場](../Page/將軍澳廣場.md "wikilink")、[君傲灣](../Page/君傲灣.md "wikilink")和[天晉](../Page/天晉.md "wikilink")等）仍未完全入伙，距離[尚德邨](../Page/尚德邨.md "wikilink")較遠，因此客量偏低，加上車站附近有不少建築地盤，亦有不少工程車輛出入，路面空氣環境惡劣，部分居民仍選擇以巴士代步。

自[將軍澳市中心](../Page/將軍澳市中心.md "wikilink")周邊住宅陸續入伙及商場啟用、通往尚德邨的一段唐俊街行人路加設上蓋及城巴開辦往返[將軍澳電視廣播城的穿梭](https://zh.wikipedia.org/wiki/將軍澳電視廣播城 "wikilink")-{巴士}-後，客量已大幅增加。

本站與[調景嶺站](../Page/調景嶺站.md "wikilink")相差僅約800米，僅次於[太子站](../Page/太子站.md "wikilink")與[旺角站](../Page/旺角站.md "wikilink")之間，對部分居於兩站之間（將軍澳中心及唐明苑）的乘客來説，他們也樂於步行往來調景嶺站，以便節省車費及轉少一次車。

由於將軍澳綫設有[寶琳站](../Page/寶琳站.md "wikilink")及[康城站兩個上行尾站](../Page/康城站_\(香港\).md "wikilink")，而前往以上兩個總站的列車均會停靠在本站的一號月台，因此從康城站前往寶琳站及[坑口站的乘客抵達本站後](../Page/坑口站_\(香港\).md "wikilink")，如未看清楚就跑上對面的列車，或會誤乘前往康城站的列車因而返回原處而誤了行程，有網民更以「不能逃出康城」創作了網上笑話。

## 車站被惡意破壞

兩名被告分別是報稱將軍澳咖啡店侍應的盧智聰(20歲)及報稱無業的林俊(23歲)，被控一項刑事毀壞罪，於2019年10月8日凌晨在當時已關閉的港鐵將軍澳站內，無合法辯解而損壞屬於香港鐵路有限公司的財物，包括將軍澳站內的玻璃幕牆，涉嫌將大型雜物掉落路軌，對公眾安全構成莫大危險。\[7\]

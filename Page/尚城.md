**尚城**（），為[香港](../Page/香港.md "wikilink")[新界](../Page/新界.md "wikilink")[元朗區](../Page/元朗區.md "wikilink")[洪水橋](../Page/洪水橋.md "wikilink")的大型[私人屋苑](https://zh.wikipedia.org/wiki/私人屋苑 "wikilink")。位於元朗[青山公路](../Page/青山公路.md "wikilink")[洪水橋](../Page/洪水橋.md "wikilink")段600號，發展商為[長江實業](../Page/長江實業.md "wikilink")。尚城在2011年分7座落成，合共734個單位。其中有697個分層單位和37幢連花園洋房，單位於2012年3月入伙。保安管理方面則由港基物業管理有限公司負責。\[1\]在樓盤推出初期，據[長江實業](../Page/長江實業.md "wikilink")執行董事[趙國雄](../Page/趙國雄.md "wikilink")指出，尚城的開價屬「平通街價」。而且該樓宇的銷售重點在於透過薄利多銷以讓更多市民可以置業。\[2\]物業距離[深圳灣口岸](../Page/深圳灣口岸.md "wikilink")10多分鐘車程。\[3\]

## 屋苑設施

尚城有一個佔地65,000平方呎的園林會所「Club CASA」\[4\]，當中的設施分作六個主題\[5\]：

|          |                                                        |
| -------- | ------------------------------------------------------ |
| **設施主題** | **設施**                                                 |
| 動感地帶     | 健身室、拳擊室、舞蹈室、美容按摩空間、蒸氣室、桑拿室                             |
| 品味天地     | 電影院、品酒室、遊戲室、桌球室、環宇飛鏢廊、環球體育搜影                           |
| 派對生活     | 翡冷翠宴會廳、芊色燒烤園                                           |
| 水上樂園     | 動感水世界、日光按摩池                                            |
| 兒童王國     | 舞蹈室、藏書室、創作室、IT小王國、四季演奏廳                                |
| 寵物世界     | 寶貝沖沖樂、狗明星酒店、Doga瑜伽天地、寵兒美容室、寵物健身地帶、寵物水世界、開心寵物園、Wo Wo訓練場 |

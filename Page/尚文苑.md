**尚文苑**（）是[香港房屋委員會](../Page/香港房屋委員會.md "wikilink")的[居屋屋苑之一](https://zh.wikipedia.org/wiki/居屋 "wikilink")，位於[香港](../Page/香港.md "wikilink")[新界](../Page/新界.md "wikilink")[葵青區](../Page/葵青區.md "wikilink")[大窩口](../Page/大窩口.md "wikilink")[德士古道](../Page/德士古道.md "wikilink")69號，鄰近[荃灣市中心](https://zh.wikipedia.org/wiki/荃灣市中心 "wikilink")、[大窩口邨](../Page/大窩口邨.md "wikilink")及[大窩口站](../Page/大窩口站.md "wikilink")，前身為房屋署職員宿舍。屋苑將於2019年12月完工，並預計於2020年6月交付\[1\]，由房屋署總建築師（5）設計，並由關黃建築師有限公司負責詳細設計，安保工程有限公司負責承建。

## 樓宇

| 樓宇名稱（座號） | 樓宇類型                                         | 落成年份  | 層數 |
| -------- | -------------------------------------------- | ----- | -- |
| 尚文苑      | [非標準設計大廈](../Page/非標準設計大廈.md "wikilink")（長型） | 2019年 | 40 |
|          |                                              |       |    |

  - 屋苑只設一座樓宇，連同基座高40層（不計地下），提供494個單位，而每層設有13伙。由於樓宇其中一邊鄰近德士古道，1及2室將採用撇角式設計（但佈局與一般單位相同），而1至5室一律會裝有隔音窗戶（樣式與[景泰苑](../Page/景泰苑.md "wikilink")、[彩興苑](../Page/彩興苑.md "wikilink")相似），以減少交通噪音的影響。此屋苑不設可間三房單位，且以B型單位（開放式單位）為主\[2\]。升降機則採用由[通力](../Page/通力.md "wikilink")製產品，共設三台，其中一台直達全部樓層。屋苑已在2018年8月命名。\[3\]
  - 此項目原先擬作[綠表置居計劃](../Page/綠表置居計劃.md "wikilink")首個推出的屋苑\[4\]，但最後恢復為居屋出售。所有單位將撥入居屋2019年中發售，於2019年5月30日至6月12日間接受申請，並於8月15日攪珠，12月9日起揀樓，2020年中交付。

**屈臣氏集團（香港）有限公司**（英語：**A.S. Watson Group (Hong Kong)
Ltd.**，簡稱：**屈臣氏集團**，英語：Watson）是[長江和記實業旗下的國際](../Page/長江和記實業.md "wikilink")[零售及食品](../Page/零售.md "wikilink")[製造機構](https://zh.wikipedia.org/wiki/製造業 "wikilink")，截至2013年12月31日屈臣氏集團門市（不包括瑪利娜[Marionnaud](https://zh.wikipedia.org/wiki/Marionnaud "wikilink")2013年收益總額200.93億港元）於遍佈25個地區，共經營超過10581間零售商店，2013年收益總額1491.47億港元。集團涉及的商品包括有保健產品、美容產品、[香水](../Page/香水.md "wikilink")、[化妝品](../Page/化妝品.md "wikilink")、[食品](https://zh.wikipedia.org/wiki/食品 "wikilink")、[飲品](https://zh.wikipedia.org/wiki/飲品 "wikilink")、[電子產品](https://zh.wikipedia.org/wiki/電子產品 "wikilink")、洋酒及機場零售業務。

2015年收益總額1519.03億港元，[稅前息前利潤](https://zh.wikipedia.org/wiki/稅前息前利潤 "wikilink")123.28億港元，12400間零售商店；2016年收益總額1515.02億港元，[稅前息前利潤](https://zh.wikipedia.org/wiki/稅前息前利潤 "wikilink")120.59億港元，13331間零售商店；2017年上半年收益總額735.57億港元，[稅前息前利潤](https://zh.wikipedia.org/wiki/稅前息前利潤 "wikilink")52.32億港元，在24個市場有超過13,500間零售商店；2017年全年收益總額1561.63億港元，[稅前息前利潤](https://zh.wikipedia.org/wiki/稅前息前利潤 "wikilink")120.89億港元，在24個市場有超過14124間零售商店。

屈臣氏最早可以追溯到1820年[皮爾森及](https://zh.wikipedia.org/wiki/皮爾森 "wikilink")[李文斯頓在](https://zh.wikipedia.org/wiki/李文斯頓 "wikilink")[澳門開設的澳門藥房](../Page/澳門.md "wikilink")。1828年，屈臣氏於中國[廣州開設](https://zh.wikipedia.org/wiki/廣州 "wikilink")「廣東大藥房」。1871年更名为**屈臣氏公司**，並成為正式的商業品牌。1981年被[和记黄埔集团收购](https://zh.wikipedia.org/wiki/和记黄埔 "wikilink")。中華民國國父[孫中山曾經領過屈臣氏頒發的獎學金](../Page/孫中山.md "wikilink")。

2014年3月21日[新加坡](../Page/新加坡.md "wikilink")[淡馬錫控股斥資](https://zh.wikipedia.org/wiki/淡馬錫控股 "wikilink")440億港元現金入股屈臣氏集團控股24.95％間接股本權益，完成交易後，屈臣氏控股成為整體零售部門的控股公司，和黃仍持有屈臣氏控股
75.05％股權，仍保留屈臣氏控股與零售部門的控制權。交易反映出屈臣氏集團的估值約為港幣1770億元，佔和黃每股約港幣41.50元。

2017年2月，屈臣氏集團以119億韓元（約合6897萬元人民幣）將其擁有的GS-Watsons公司50%股份出售給[GS零售](https://zh.wikipedia.org/wiki/GS零售 "wikilink")，截止2018年1月，屈臣氏在韓國擁有188家分店門市。

| 1937年 | 抗日戰爭全面爆發，幾個月之後，日本軍隊占領香港並接管「屈臣氏非酒精飲料製造廠」，重新命名為「日本海軍飲料製造廠」；日本的侵略讓屈臣氏損失慘重，全國各地的屈臣氏資產多被日軍侵佔，直到1945年日軍投降後，屈臣氏才陸續收回其經營權，各分店重新在香港恢復營業。                                                                                                                                                                                                |
| 1949年 | 中共建政後，屈臣氏資產被共產黨侵占，被迫退出大陸市場，至1989年才重返經營。                                                                                                                                                                                                                                                                                        |
| 1963年 | 香港和記企業獲得屈臣氏相對多數的股權，當年的重點計劃是開發香港北角的土地，預計屈臣氏的年度收益將超過港幣100萬元。此後的幾年，屈臣氏大力投入土地的開發與建設，由於屈臣氏的資產因此愈來愈多。                                                                                                                                                                                                                                |
| 1972年 | 甚至把百佳超市併購；其後的幾年，屈臣氏陸續購買土地和併購工廠，因而逐漸成為香港財力驚人的企業集團。                                                                                                                                                                                                                                                                              |
| 1973年 | 屈臣氏集團收購Peter Pan玩具公司，並推出雪山雪糕(Mountain Cream)，但雪山雪糕在1999年出售予[聯合利華公司](https://zh.wikipedia.org/wiki/聯合利華 "wikilink")，其後併入[和路雪品牌](../Page/和路雪.md "wikilink")。                                                                                                                                                                     |
| 1981年 | 屈臣氏正式被納入香港[和記黃埔集團下的營利機構](../Page/和記黃埔.md "wikilink")，和記黃埔是全球知名的企業集團，這個集團以香港為基地，業務遍及全球56個國家，年營業額超過100億美元，擁有港口、地產、能源、零售、通路等事業。                                                                                                                                                                                                   |
| 1987年 | 屈臣氏正式在臺灣開店，臺灣的第一家屈臣氏開在[臺北市衡陽路](../Page/臺北市.md "wikilink")，至2015年為止，屈臣氏在臺灣的店數已有513家。                                                                                                                                                                                                                                            |
| 1989年 | 屈臣氏於4月重返中國經營，首間店鋪位於北京。                                                                                                                                                                                                                                                                                                         |
| 2004年 | 8月24日屈臣氏收購德國保健及美容產品連鎖店Dirk Rossmann GmbH百分之四十的股權。Rossmann由德國人Dirk Rossmann於1972年創立，現時在德國、波蘭、匈牙利及捷克擁有超過1100間分店。Rossmann以德國為基地，在當地擁有786間分店，是保健及美容產品零售市場的三大巨頭之一，在業界享負盛名。                                                                                                                                                          |
| 2008年 | [2008年屈臣氏蒸餾水工潮](../Page/2008年屈臣氏蒸餾水工潮.md "wikilink")。同年7月9日屈臣氏於台灣推出「寵i會員卡」。                                                                                                                                                                                                                                                    |
| 2011年 | 集團營收1435.64億港元，2011年集團[除稅及利息前盈利](https://zh.wikipedia.org/wiki/除稅及利息前盈利 "wikilink")(EBIT)93.3億港元、資產總額532.7億港元、負債總額280.56億港元。 在十三個市場經營逾五千二百家店舖，於馬來西亞、香港、泰國、新加坡與菲律賓九個亞洲市場營運超過一千五百家店舖，其他零售及製造匯聚香港著名之零售品牌概念，包括百佳超級市場與其他相關概念式商店、豐澤消費電子及電器產品零售連鎖店、以及屈臣氏酒窖。該分部並包括內地百佳超級市場與屈臣氏酒窖、香港與新加坡國際機場之Nuance-Watson，以及在香港與內地生產與分銷家喻戶曉之瓶裝水、果汁與飲品。 |
| 2013年 | 歐洲保健及美容產品營收609.11億港元，[EBIT](https://zh.wikipedia.org/wiki/除稅及利息前盈利 "wikilink")35.78億港元；中國保健及美容產品營收179.62億港元，EBIT35.67億港元；亞洲保健及美容產品營收197.13億港元，EBIT17.79億港元；歐洲高級香水及化妝品營收179.42億港元，EBIT5.5億港元；其他零售及製造營收347.43億港元，EBIT17.64億港元。                                                                                                   |

**山麗苑**（）是[香港房屋委員會](../Page/香港房屋委員會.md "wikilink")的[居者有其屋計劃](../Page/居者有其屋計劃.md "wikilink")屋苑之一，與鄰近的[皇后山邨](../Page/皇后山邨.md "wikilink")屬於同一個項目。屋苑位於[香港](../Page/香港.md "wikilink")[新界](../Page/新界.md "wikilink")粉嶺上水市地段276號（FSSTL 276），前身為[皇后山軍營](../Page/皇后山軍營.md "wikilink")東南面用地及[皇后山軍營印度廟](https://zh.wikipedia.org/wiki/皇后山軍營印度廟 "wikilink")。屋苑由房屋署總建築師（四）負責總體設計，並由[王歐陽（香港）有限公司負責詳細設計](https://zh.wikipedia.org/wiki/王歐陽（香港）有限公司 "wikilink")，承建商為[有利建築有限公司](https://zh.wikipedia.org/wiki/有利建築 "wikilink")\[1\]。

## 樓宇

屋苑共設六座，提供3,222個單位。除橋山閣、梨山及楹山閣分別僅高23層及36層外，其他樓宇均設37層，不設可間三房單位，而每層單位為16個（除松山閣為12個）\[2\]。景觀方面，向南單位可享流水響、布吉仔一帶山景，而向北單位更可眺望深圳市羅湖區的摩天大廈。所有樓宇已於2018年8月正式命名，詳情如下\[3\]：

| 松山閣（A座）  | [非標準設計大廈](../Page/非標準設計大廈.md "wikilink")（Y型） | 2021年 |
| 榕山閣（B座）  |                                              |       |
| 杏山閣（C座）  |                                              |       |
| 梨山閣（D座）  |                                              |       |
| 楹山閣（E座）  |                                              |       |
| 橋山閣（F座）  |                                              |       |

所有單位原擬撥入居屋2019中發售，並已於同年4月18日簽立地契。但按照房委會在2019年3月公佈的發售安排，山麗苑將押後至往後期數發售。

順帶一提，所有樓宇均將採用[OTIS生產的升降機](../Page/奧的斯電梯公司.md "wikilink")。除橋山閣只有3台升降機（其中一台直通所有樓層）外，其餘樓宇均設4台升降機（兩台可達1-18樓，另外兩台可達19-36或37樓，沒有升降機可直通所有樓層）。

## 設施

屋苑設兒童遊樂場、籃球場、羽毛球場、園景設施，以及位於松山、榕山及杏山閣地庫的停車場。此外，位於屋苑內的一座舊營房（前警犬基地警官食堂）及[皇后山軍營印度廟將獲得保留](https://zh.wikipedia.org/wiki/皇后山軍營印度廟 "wikilink")，用作山麗苑管業處及業主立案法團/委員會辦事處。

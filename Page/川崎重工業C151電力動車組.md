{{ multiple image | align = fight | direction = vertical | width = 250 | image1 = C151AngMoKio.jpg| caption1 = 翻新前 C151 列車外觀 }} {{ multiple image | align = fight | direction = vertical | width = 250 | image1 = Kawasaki_train.jpg| caption1 = 翻新前 C151 列車內裝 | image2 = Kawasaki C151 1006.jpg| caption2 = 翻新後 C151 列車內裝 }} **新加坡地鐵 C151 電力動車組**是[新加坡地鐵](../Page/新加坡地鐵.md "wikilink")目前正在使用的四種車輛系統之一，共有66組列車，每組列車由6個車廂組成，為[SMRT地鐵有限公司所擁有](https://zh.wikipedia.org/wiki/SMRT地鐵有限公司 "wikilink")，於1987年開始使用，在[南北線和](https://zh.wikipedia.org/wiki/新加坡地鐵南北線 "wikilink")[東西線上運行](https://zh.wikipedia.org/wiki/新加坡地鐵東西線 "wikilink")。

## 歷史

為配合新加坡地鐵開建，MRTC公司（今SMRT）於1983年就本型列車的製造合約招標，入標者包括AESA公司-[勝科海事聯營](https://zh.wikipedia.org/wiki/勝科海事 "wikilink")、[猛獅集團](https://zh.wikipedia.org/wiki/猛獅集團 "wikilink")-[AEG](https://zh.wikipedia.org/wiki/AEG "wikilink")-[西門子](https://zh.wikipedia.org/wiki/西門子 "wikilink")-[勃朗-包維利股份公司聯營](https://zh.wikipedia.org/wiki/勃朗-包維利股份公司 "wikilink")、[川崎重工業](../Page/川崎重工業.md "wikilink")為首財團、[阿爾斯通](../Page/阿爾斯通.md "wikilink")-Francorail-SOFRETU公司-[新科工程聯營](https://zh.wikipedia.org/wiki/新科工程 "wikilink")、[龐巴迪運輸](../Page/龐巴迪運輸.md "wikilink")為首財團，以及[都城嘉慕](../Page/都城嘉慕.md "wikilink")-[新科工程聯營](https://zh.wikipedia.org/wiki/新科工程 "wikilink")6家財團。原先，MRTC公司屬意都城嘉慕-新科工程方案（香港地鐵市區線列車第三軌供電版本，動力系統預計判給予英國通用電氣），但因受都城嘉慕製造工程失誤事件影響，最後改為由川崎中標。列車合約於1984年批出，共400卡（150卡共25列行走南北線，246卡共41列行走東西線，另有4卡為解款用列車），總值[新加坡元](../Page/新加坡元.md "wikilink")5億8150萬元。

車輛在1986年至1989年之間共分3批生產，這3批之間並沒有任何規格的差異；車輛分別由[川崎重工業](../Page/川崎重工業.md "wikilink")、[日本車輛製造](../Page/日本車輛製造.md "wikilink")、[東急車輛及](https://zh.wikipedia.org/wiki/東急車輛 "wikilink")[近畿車輛](../Page/近畿車輛.md "wikilink")製造。本型列車車身設計仿傚[香港地鐵](../Page/香港地鐵.md "wikilink")市區線列車，但車門改為日本式佈局，即每側4組、採用較闊窗戶。另外，本型車的機電配套廠商組合與較早批次的香港地鐵市區線列車大致相同，但改為使用與香港地鐵港島線批次列車相似的[三菱電機動力系統](https://zh.wikipedia.org/wiki/三菱電機 "wikilink")（唯走行音較香港地鐵港島線批次列車明顯、尖銳），而該等產品耗電較香港地鐵較早批次列車採用的[英國通用電氣製動力系統少約一半](https://zh.wikipedia.org/wiki/英國通用電氣 "wikilink")。

首套列車模型於1984年運抵新加坡，並向公眾公開展示；而首列列車3001/3002編組則於1986年7月8日運抵，而所有列車均由[東方海皇航運承運](https://zh.wikipedia.org/wiki/東方海皇航運 "wikilink")。

目前車輛已於2006年至2008年末間完成第1次現代化工程，將車箱座位、冷氣系統及車身設計更換。在第1次現代化工程中，4卡解款用列車未有翻新，隨後於2007年交予[新加坡武裝部隊](../Page/新加坡武裝部隊.md "wikilink")作軍事訓練車輛之用，目前存放於波揚蓄水池附近的軍營。

而第2次現代化工程涉及將現有動力系統更換為磁浮電動機連IGBT-VVVF，以及安裝新訊號系統-[SelTrac](../Page/SelTrac.md "wikilink") CBTC的機件。在2015年，首兩列列車的改裝工程完成；本來此工程將於2018年完工，但隨著[陸路交通管理局分別於](https://zh.wikipedia.org/wiki/陸路交通管理局 "wikilink")2016年8月宣佈購入取代本型列車，在完成3025/3026、3055/3056、3065/3066、3109/3110及3131/3132編組列車的改裝工作後，第2次現代化工程正式腰斬， 陸路交通管理局更於2018年7月於社交平台宣佈向[龐巴迪訂購](https://zh.wikipedia.org/wiki/龐巴迪 "wikilink")66列型號代號CR151型的Movia電動列車；而所有列車將於2023年退役。

## 列車編組

列車以6卡為一列（除解款車為4卡一列），以3-1-2-2-1-3形式排列。當中，3-1-2為一組，2-1-3亦為一組。調卡時，都會整組調走。

| 川崎重工業C151電力動車組車卡組合註解（不包括前解款車卡，該等車卡不在此編號系統內） |
| ------------------------------------------- |
| 車廂                                          |
| 3XXX駕駛室拖卡（DT）                               |
| 1XXX動力車廂（M1）                                |
| 2XXX動力車廂（M2）                                |

而在本型列車中，各廠商生產的編組如下：

  - 3001/3002-3019/3020及一列四卡解款車（第一批），3051/3052-3061/3062（第二批），以及3093/3094-3101/3102（第三批）編組由[川崎重工業](../Page/川崎重工業.md "wikilink")生產，共22列。
  - 3021/3022-3029/3030（第一批）, 3063/3064-3071/3072（第二批）, 以及3103/3104-3111/3112（第三批）編組由[近畿車輛](../Page/近畿車輛.md "wikilink")生產，共15列。
  - 3031/3032-3039/3040（第一批）, 3073/3074-3081/3082（第二批）, 以及3113/3114-3121/3122（第三批）編組由[日本車輛製造](../Page/日本車輛製造.md "wikilink")生產，共15列。
  - 3041/3042-3049/3050（第一批）, 3083/3084-3091/3092（第二批）, 以及3123/3124-3131/3132（第三批）編組由[東急車輛生產](https://zh.wikipedia.org/wiki/東急車輛 "wikilink")，共15列。

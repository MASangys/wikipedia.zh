**差分进化算法**（）又称**微分进化算法**，是一种求解[最佳化问题的](https://zh.wikipedia.org/wiki/最佳化问题 "wikilink")[进化算法](https://zh.wikipedia.org/wiki/进化算法 "wikilink")。因為[进化算法對於](https://zh.wikipedia.org/wiki/进化算法 "wikilink")[最佳化问题的要求極少](https://zh.wikipedia.org/wiki/最佳化问题 "wikilink")，所以被視為一種。雖然後設启发式算法適用於多種[最佳化问题](https://zh.wikipedia.org/wiki/最佳化问题 "wikilink")，但是並不保證可以找到[全局最優解](https://zh.wikipedia.org/wiki/全局最優解 "wikilink")。

差分进化算法被使用在多[維度](../Page/維度.md "wikilink")[實數編碼的](https://zh.wikipedia.org/wiki/實數 "wikilink")[最佳化问题](https://zh.wikipedia.org/wiki/最佳化问题 "wikilink")。因為此算法不使用問題的[梯度資訊](../Page/梯度.md "wikilink")，故可解不[可微分的](https://zh.wikipedia.org/wiki/可微分 "wikilink")[最佳化问题](https://zh.wikipedia.org/wiki/最佳化问题 "wikilink")。也因此，差分进化算法可用於[不連續的](https://zh.wikipedia.org/wiki/不連續 "wikilink")，[雜訊的](../Page/雜訊.md "wikilink")，隨著時間改變的[最佳化问题](https://zh.wikipedia.org/wiki/最佳化问题 "wikilink")。

差分进化算法類似[遗传算法](../Page/遗传算法.md "wikilink")，包含变异，交叉操作，淘汰机制。本质上说，它是一种基于实数编码的具有保优思想的贪婪[遗传算法](../Page/遗传算法.md "wikilink")\[1\]。而差分进化算法与[遗传算法不同之處](../Page/遗传算法.md "wikilink")，在於变异的部分是隨選兩個解成員變數的差異，經過伸縮後加入當前解成員的變數上，因此差分进化算法無須使用[機率分佈產生下一代解成員](https://zh.wikipedia.org/wiki/機率分佈 "wikilink")
\[2\]。

算法的原理采用对个体进行方向扰动，以达到对个体的函数值进行下降的目的，同其他进化算法一样，差分进化算法不利用函数的梯度信息，因此对函数的可导性甚至连续性没有要求，适用性很强。同时，算法与[粒子群优化有相通之处](../Page/粒子群优化.md "wikilink")，但因为差分进化算法在一定程度上考虑了多变量间的[相关性](https://zh.wikipedia.org/wiki/相关性 "wikilink")，因此相较于[粒子群优化在变量](../Page/粒子群优化.md "wikilink")[耦合问题上有很大的优势](https://zh.wikipedia.org/wiki/耦合 "wikilink")。由于差分进化算法在连续域优化问题的优势已获得广泛应用，并引发[进化算法研究领域的热潮](https://zh.wikipedia.org/wiki/进化算法 "wikilink")。算法的实现参考实现代码部分\[3\]

## 演算法原理

差分進化演算法之目的為求解[最佳化問題](https://zh.wikipedia.org/wiki/最佳化問題 "wikilink")，使用突變、交叉、選擇計算以演化多個可能的解。首先，產生足量的[隨機變數](https://zh.wikipedia.org/wiki/隨機 "wikilink")，做為初始的可能解。接著，依序進行突變、交叉、選擇計算，做完一輪後，檢查某個終止條件。若終止條件尚未滿足，則回到突變、交叉、選擇計算，否則終止差分進化演算法，輸出最後一輪的最佳解。

### 突變

在[進化計算中](https://zh.wikipedia.org/wiki/進化計算 "wikilink")，突變是用於產生隨機解的計算方法。

### 交叉

在突變之後，差分進化演算法使用交叉計算以增強隨機解的多樣性。

### 選擇

在交叉之後，差分進化演算法對隨機解做選擇，移除演化失敗的解，留下演化成功的解。選擇之後，進行突變計算，直到滿足某個終止條件。

## 实现代码（MATLAB）

``` matlab
tic
F = 0.9;
CR = .1;
n = 2; %问题维数，以简单的球函数为目标函数
NP = 30;
lu = [-10,-10 ;10 ,10]; %求解空间的上下界
LB = repmat(lu(1,:),NP,1);
UB = repmat(lu(2,:),NP,1);
%用于生成随机选择个体的表
tab = 1:NP; tab = tab(ones(1,NP),:)';
dig = 1:NP; D =(dig-1)*NP +(1:NP);
tab (D) = [];
tab = reshape(tab,NP-1,[])';
TAB = tab;
%测试次数
TIMES = 10;
Solve = zeros(1,TIMES);
numOfevol = zeros(1,TIMES);
for time = 1:TIMES
%
Result = []; %记录结果
rand('seed',sum(100*clock));
%
X = LB+rand(NP,n).*(UB-LB);
U = X;
%%
fit = fitness (X); %首次评价
FES = NP;
while FES<n*10000
    %产生随机个体参与变异
    tab = TAB;
    rand1 = floor(rand(NP,1)*(NP-1))+1;
    rand2 = floor(rand(NP,1)*(NP-2))+2;
    rand3 = floor(rand(NP,1)*(NP-3))+3;
    RND1 =(rand1-1)*NP+(1:NP)';
    RND2 =(rand2-1)*NP+(1:NP)';
    RND3 =(rand3-1)*NP+(1:NP)';
    r1 = tab (RND1); tab (RND1)=tab(:,1);
    r2 = tab (RND2); tab (RND2)=tab(:,2);
    r3 = tab (RND3);
    % rand/one/变异模式
    V = X(r1,:) + F.*(X(r2,:)-X(r3,:));
    %越界检验
    BL = V<LB ; V(BL) = 2*LB(BL) - V(BL);
    BLU = V(BL)>UB(BL); BL (BL) = BLU ; V(BL) = UB (BL);
    BU = V>UB;  V (BU) = 2*UB(BU) - V(BU);
    BUL = V(BU)<LB(BU); BU (BU) = BUL ; V(BU) = LB (BU);
    %交叉操作
    J_= mod(floor(rand(NP,1)*n),n)+1;
    J =(J_-1)*NP+(1:NP)';
    C = rand(NP,n)<CR;
    U (J) = V(J);
    U (C) = V(C);
    %评价子代
    fit_ = fitness (U);
    %比较并竞争
    S = fit_<fit;
    X(S,:) = U(S,:);
    fit (S) = fit (S)|;
    %记录函数评价次数
    FES = FES + NP;
    %记录结果（用于绘图，并不是算法必要环节）
    Result = [Result ,min (fit)];
end
Solve (time) = min (fit);
%试验次数
plot(log10(Result),'b');hold on;
end
disp(['求解结果:',num2str(Solve)]);
toc
%附上球函数代码（新建一个M文件即可）
function Y = fitness (X)
Y = sum(X.^2 ,2);
```

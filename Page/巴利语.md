**巴利语**（[IAST](../Page/IAST.md "wikilink")/[ISO
15919](../Page/ISO_15919.md "wikilink"):
、पालि）是[古印度语言](https://zh.wikipedia.org/wiki/古印度 "wikilink")，属於[印欧语系](../Page/印欧语系.md "wikilink")、[印度-伊朗语族](../Page/印度-伊朗语族.md "wikilink")、[印度-雅利安语支的](../Page/印度-雅利安语支.md "wikilink")[中古印度-雅利安語](../Page/中古印度-雅利安語.md "wikilink")，与[梵語十分相近](https://zh.wikipedia.org/wiki/梵語 "wikilink")，是印度西方所用的俗語（[普拉克里特諸語言](https://zh.wikipedia.org/wiki/普拉克里特諸語言 "wikilink")）。巴利語可以用各种文字书写，比如[婆罗米文](../Page/婆罗米文.md "wikilink")、[天城文](https://zh.wikipedia.org/wiki/天城文 "wikilink")，以及由[巴利圣典协会的英国语言学家](https://zh.wikipedia.org/wiki/巴利圣典协会 "wikilink")所采用的適用於各種印度語言的[拉丁字母](../Page/拉丁字母.md "wikilink")[轉寫](https://zh.wikipedia.org/wiki/轉寫 "wikilink")。

## 起源

從字面上來看，巴利（Pāli），是線、行，或是文章、經典的意思，和註釋、評論（[Attha-katha](https://zh.wikipedia.org/wiki/義註 "wikilink")）相對。按[南傳上座部的記載](https://zh.wikipedia.org/wiki/南傳上座部 "wikilink")，稱傳誦巴利三藏的語言，為「摩揭陀語」（Magadhi），或「根本語」（Mula-Bhasa）。後來才叫其「聖典（Pali）語」，即「巴利語」\[1\]。

據[錫蘭](https://zh.wikipedia.org/wiki/錫蘭 "wikilink")[上座部佛教的傳說](../Page/上座部佛教.md "wikilink")，現存的《[巴利語大藏經](https://zh.wikipedia.org/wiki/巴利語大藏經 "wikilink")》就是[阿育王的兒子](../Page/阿育王.md "wikilink")\[2\][摩哂陀](../Page/摩哂陀.md "wikilink")（巴利語：，梵語：）帶到錫蘭去的版本，所以巴利語也就是[摩揭陀語](../Page/摩揭陀語.md "wikilink")（，）。在公元前二六年前後以[僧伽羅文字母記錄下來](../Page/僧伽羅文.md "wikilink")\[3\]。在公元五世紀，[覺音尊者將](../Page/覺音.md "wikilink")[僧伽羅語](https://zh.wikipedia.org/wiki/僧伽羅語 "wikilink")（sīhaḷa-bhāsa）[注釋書轉譯為巴利聖典語](https://zh.wikipedia.org/wiki/義註 "wikilink")\[4\]。

然而，據[季羨林考證](https://zh.wikipedia.org/wiki/季羨林 "wikilink")，根據語言學的特徵，巴利語應是屬於古印度西部的方言，與流行於東部的[摩揭陀語不同](../Page/摩揭陀語.md "wikilink")\[5\]。印度地方現存的三十多件[阿育王敕令中](https://zh.wikipedia.org/wiki/阿育王敕令 "wikilink")，屬於西部群的基爾納爾（Girnar）刻文，最接近巴利語。巴利佛教所屬的上座部，其根據地即是西部印度[優禪尼](https://zh.wikipedia.org/wiki/優禪尼 "wikilink")（Ujjeni）為主的地方，且其附近地區發現的刻文也與巴利語相近。並且，把上座部佛教傳入錫蘭的[摩哂陀長老](../Page/摩哂陀.md "wikilink")，據說生於優禪尼，並在這裡出家\[6\]。

在巴利語和[摩揭陀俗語](../Page/摩揭陀俗語.md "wikilink")(Ardhamāgadhī)之間有很多顯著的相似性，摩揭陀俗語是保存在古代[耆那教經典中的古老形式的摩揭陀語](../Page/耆那教.md "wikilink")。摩揭陀俗語同巴利語在類似要點上不同於阿育王題刻的東部俗語。例如，摩揭陀俗語也不把變更為，并且在名詞變格中至少在很多押韻的地方顯示了結尾而非東部俗語的。這種相似性不是偶然的，因為耆那教的創始者[尼乾陀若提子](../Page/筏馱摩那.md "wikilink")（被教徒尊稱為大雄
Mahāvīra）和[釋迦牟尼佛在相同的地域](https://zh.wikipedia.org/wiki/釋迦牟尼佛 "wikilink")（[摩揭陀國](https://zh.wikipedia.org/wiki/摩揭陀國 "wikilink")）傳教。

長和短元音只對比於[開音節中](https://zh.wikipedia.org/wiki/開音節 "wikilink")；在閉音節中所有元音總是短的。短和長的與是互補分布的：短變體只出現閉音節中，長變體只出現在開音節中。短和長的與因此不是不同的音位。

有叫做的聲音（梵語術語；巴利語為：），用羅馬化字母（[ISO
15919](../Page/ISO_15919.md "wikilink")）或（[IAST](../Page/IAST.md "wikilink")）表示，在多數傳統字母中用凸出點來表示，最初用作標記前面的元音要鼻音化的事實。就是說，,
和表示, 和。但是在很多傳統發音中，被發音得更強，像軟腭鼻音，所以這些聲音轉而被發音為,
和。但是永遠不跟隨在長元音之后發音；在被增加到以長元音結尾的詞幹的時候，,
和要轉換成相應的短元音，比如變成而非，變成而非。

## 構詞法

巴利語是高度詞形變化的語言，其中幾乎所有詞，除了承載基本意義的詞根之外，還包含一個或多個[詞綴](../Page/詞綴.md "wikilink")（通常是后綴），它們某種方式修改意義。名詞要為了[性](https://zh.wikipedia.org/wiki/性_\(語法\) "wikilink")、[數和格而變化詞尾](https://zh.wikipedia.org/wiki/數_\(語法\) "wikilink")；動詞詞形變化要承載關於[人稱](https://zh.wikipedia.org/wiki/人稱 "wikilink")、數、[時態和](https://zh.wikipedia.org/wiki/時態 "wikilink")[語氣的信息](https://zh.wikipedia.org/wiki/語氣 "wikilink")。

### 名詞變格

巴利語名詞為三個文法[性](https://zh.wikipedia.org/wiki/性_\(語法\) "wikilink")（陽性，陰性，中性）和兩個數（單數，複數）而詞尾變化。名詞還在原則上顯示八個[格](https://zh.wikipedia.org/wiki/格_\(語法\) "wikilink")：[主格](https://zh.wikipedia.org/wiki/主格 "wikilink")（），[呼格](../Page/呼格.md "wikilink")，[賓格](https://zh.wikipedia.org/wiki/賓格 "wikilink")（），[工具格](../Page/工具格.md "wikilink")（），[與格](https://zh.wikipedia.org/wiki/與格 "wikilink")（），[離格](https://zh.wikipedia.org/wiki/離格 "wikilink")，[屬格](https://zh.wikipedia.org/wiki/屬格 "wikilink")（），和[方位格](../Page/方位格.md "wikilink")（）；但是在很多實例中，兩個或多個格在形式上是同一的；特別是對於屬格和與格。

## 書寫

### 帶有變音符號的巴利語字母

歷史上，巴利語的最先書寫記錄據信是在[斯里蘭卡基於早先口頭傳承而寫成的](../Page/斯里蘭卡.md "wikilink")。依據斯里蘭卡的大編年史，由於國家鬧大饑荒，佛教僧侶在公元前100年王期間寫下了巴利語圣典。寫下的巴利語的傳播保持了一套通用的字母音值系統，但使用各種各樣的實際文字來表達這些音值。這使很多西方人糊涂，他們意圖假定一種文字不可避免的連結著一組音素。

在[斯里蘭卡](../Page/斯里蘭卡.md "wikilink")，巴利語文本記錄於新哈勒文（Sinhala，[僧伽羅文](../Page/僧伽羅文.md "wikilink")）。其他地區文字，最顯著的有[高棉文](../Page/高棉文.md "wikilink")、[緬甸文](https://zh.wikipedia.org/wiki/緬甸文 "wikilink")、傣文（[传统傣仂文](../Page/老傣文.md "wikilink")）和現代[泰文](https://zh.wikipedia.org/wiki/泰語字母 "wikilink")（自從1893年），[天城文和东南亚孟族人](https://zh.wikipedia.org/wiki/天城文 "wikilink")（Mon）的[孟文](https://zh.wikipedia.org/wiki/孟文 "wikilink")、寮文（寮国Laos）也已經被用來記錄巴利語。

自從十九世紀，巴利語已經用羅馬文字書寫了。[Frans
Velthuis設計了一個可選方案允許使用普通](https://zh.wikipedia.org/wiki/Frans_Velthuis "wikilink")[ASCII碼來打字而不用](../Page/ASCII.md "wikilink")[變音符號](https://zh.wikipedia.org/wiki/變音符號 "wikilink")，但是它比標準Rhys
Davids系統（見下）要更少可讀性。

巴利語字母如下排序（以天城字母的順序為基準，巴利語專用的發音擺在最後）：

  -
    a ( अ )、ā ( आ )、i ( इ )、ī ( ई )、
    u ( उ )、ū ( ऊ )、e ( ए )、o ( ओ )、-ṃ ( अं )、
    k ( क )、kh ( ख )、g ( ग )、gh ( घ )、ṅ ( ङ )、
    c ( च )、ch ( छ )、j ( ज )、jh ( झ )、ñ ( ञ )、
    ṭ ( ट )、ṭh ( ठ )、ḍ ( ड )、ḍh ( ढ )、ṇ ( ण )、
    t ( त )、th ( थ )、d ( द )、dh ( ध )、n ( न )、
    p ( प )、ph ( फ )、b ( ब )、bh ( भ )、m ( म )、
    y ( य )、r ( र )、l ( ल )、v ( व )、
    s ( स )、h ( ह )、ḷ ( ळ )、ḷh ( ऴ )。

儘管是和的連寫，但實際上是一個單一音素，如同kh、gh、ch...等等。

### 計算機上的巴利語轉寫

有多種字體用於巴利語轉寫，但是不推薦老的ASCII字體比如Leedsbit PaliTranslit, Times_Norman,
Times_CSX+, Skt Times, Vri RomanPali
CN/CB等，因為它們彼此不兼容并且在技術上過時了。正好相反，推薦基於[Unicode標準的字體](https://zh.wikipedia.org/wiki/Unicode "wikilink")，因為Unicode是所有字體的前景還有它們易於相互移植。

但是不是所有Unicode字體都包含必須的字符。要正確的顯示用於羅馬化巴利語（或梵語）的所有變音符號，Unicode字體必須包含下列字符范圍：

  - Basic Latin: U+0000 – U+007F
  - Latin-1 Supplement: U+0080 – U+00FF
  - Latin Extended-A: U+0100 – U+017F
  - Latin Extended-B: U+0180 – U+024F
  - Latin Extended Additional: U+1E00 – U+1EFF

[巴利圣典协会](http://www.palitext.com/subpages/PC_Unicode.htm)向Windows和Linux計算機推薦[VU-Times字體](http://zencomp.com/greatwisdom/fonts/)和[Gandhari
Unicode字體](http://depts.washington.edu/ebmp/software.php)。[雪域數碼圖書館](http://www.thdl.org/tools/fonts/diafonts.html)推薦[Times
Ext
Roman字體](https://web.archive.org/web/20070617033953/http://www.bcca.org/services/fonts/)，和到其他一些可用於巴利語的Unicode字體的鏈接。還有一名英國佛教僧侶Bhikkhu
Pesala提供了原為自己設計的一些巴利語Unicode
[字體](https://web.archive.org/web/20120320233551/http://www.aimwell.org/Fonts/fonts.html)，和用於Windows
XP的某些巴利語[鍵盤](https://web.archive.org/web/20120321073228/http://www.aimwell.org/Fonts/Keyboards/keyboards.html)。進一步的，Alanwood的Unicode資源的[字體段落](http://www.alanwood.net/unicode/fonts.html)鏈接了可用於巴利語錄入的一些通用字體，如果它們覆蓋了上述字符范圍的話。

## 巴利語與梵語

儘管巴利語不能被認為是[古典梵語或更早的](https://zh.wikipedia.org/wiki/古典梵語 "wikilink")[吠陀梵語的直接後代](../Page/吠陀梵語.md "wikilink")，這些語言明顯有非常密切關聯，并且巴利語和[梵語的共同特征總是很容易被熟悉二者的印度人識別出來](https://zh.wikipedia.org/wiki/梵語 "wikilink")。實際上，大部分的巴利語和梵語的詞幹都是在形式上同一的，只在詞尾變化的細節上有所區別。

這種聯繫是如此充分的眾所周知，以至於來自梵語的技術術語可以通過一組常規的音韻變換輕易的轉換成巴利語。這種變換模擬出了曾在原巴利語中發生的音韻發展的子集。由於這些變換的流行，不總能得知給定的巴利詞是古俗語詞匯的一部分，還是從梵語借來的變換後的詞。梵語詞有規律的對應於巴利詞的存在性不總是巴利詞源的可靠證據，因為在某些情況下，通過反向構詞可以從俗語詞創造出假造的梵語詞。

下列音位學處理不意圖作為對從古印度祖先語言產生巴利語的歷史變化的詳盡描述，而是在梵語和巴利語之間最常見的音韻等式的總結，并且不聲稱其完備性。

有對上述規則的一些顯著的例外；其中很多是常用的俗語詞而不是從梵語中借用來的。

  - → （比較於）

  - → （adj.）（比較於（n.））

  - → （非*pur**us**a*）

  - →  → （非***va**kkha*）

1919年，挪威数学家[維果·布朗](https://zh.wikipedia.org/wiki/維果·布朗 "wikilink")（[Viggo Brun](https://zh.wikipedia.org/wiki/:en:Viggo_Brun "wikilink")）证明了所有[孪生素数](../Page/孪生素数.md "wikilink")的倒数之和收敛于一个[数学常数](../Page/数学常数.md "wikilink")，称为**布朗常数(Brun's constant)**，记为*B*<sub>2</sub> ：

\[B_2 = \left(\frac{1}{3} + \frac{1}{5}\right)
+ \left(\frac{1}{5} + \frac{1}{7}\right)
+ \left(\frac{1}{11} + \frac{1}{13}\right)
+ \left(\frac{1}{17} + \frac{1}{19}\right)
+ \left(\frac{1}{29} + \frac{1}{31}\right) + \cdots\]

而[所有**素数**的倒数之和则是发散的](https://zh.wikipedia.org/wiki/证明所有素数的倒数之和发散 "wikilink")。假如以上的级数发散，则我们立刻就可以证明[孪生素数猜想](../Page/孪生素数猜想.md "wikilink")。但由于它收敛，我们就不知道是否有[无穷多个孪生素数](https://zh.wikipedia.org/wiki/无穷多 "wikilink")(若孪生素数之[平方根](../Page/平方根.md "wikilink")的倒數和發散，則亦可知其為無限多)。类似地，如果证明了布朗常数是[无理数](https://zh.wikipedia.org/wiki/无理数 "wikilink")，也立刻就可以证明[孪生素数猜想](../Page/孪生素数猜想.md "wikilink")。但如果它是[有理数](../Page/有理数.md "wikilink")，则仍然无法知道孪生素数是不是无限的。

Thomas R. Nicely把孪生素数算到10<sup>14</sup>，估计布朗常数大约为1.902160578。目前最精确的估计是Pascal Sebah和Patrick Demichel在2002年发现的，他们把孪生素数算到了10<sup>16</sup>：

  -
    *B*<sub>2</sub> ≈ 1.902160583104.

我们知道1.9 \< *B*<sub>2</sub>，但不知道是否能大于2。

除此以外，还有一个**四胞胎素数的布朗常数**，它是所有的[四胞胎素数](../Page/四胞胎素数.md "wikilink")的倒数之和，记为*B*<sub>4</sub>：

\[B_4 = \left(\frac{1}{5} + \frac{1}{7} + \frac{1}{11} + \frac{1}{13}\right)
+ \left(\frac{1}{11} + \frac{1}{13} + \frac{1}{17} + \frac{1}{19}\right)
+ \left(\frac{1}{101} + \frac{1}{103} + \frac{1}{107} + \frac{1}{109}\right) + \cdots\]

它的值为

  -
    *B*<sub>4</sub> =0.87058 83800 ± 0.00000 00005。

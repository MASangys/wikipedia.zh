FM聯播網](../Page/Hit_FM聯播網.md "wikilink")
、[中國廣播公司等台灣三大電台協辦](../Page/中國廣播公司.md "wikilink")。主持人為[黃子佼](../Page/黃子佼.md "wikilink")、[黃路梓茵](https://zh.wikipedia.org/wiki/黃路梓茵 "wikilink")\[1\]後台直播兼助理主持人為[林舒嬅](https://zh.wikipedia.org/wiki/林舒嬅 "wikilink")，[中視主頻](https://zh.wikipedia.org/wiki/中視主頻 "wikilink")、[中天娛樂台於](../Page/中天娛樂台.md "wikilink")2017年10月7日首播\[2\]\[3\]。[中天綜合台則於](../Page/中天綜合台.md "wikilink")2017年10月8日同步播出最新集數，即為中視現場直播完成後，再於同日22:00起繼續重播當日節目。

## 節目內容

### 投票規則

拿起手機下載《希望之星》手機應用程式，並完成認證後與電視連動。電視機觀眾即為評審，名為「全民評審」。

每次參賽者開始演唱時，每個人均有增加三千個「星星」為下注籌碼。如果觀眾有另外指定下載App製造商指定的遊戲App，則可額外再獲得六千個「星星」，所以一次表演就有九千個「星星」，在獲得更高額獎金的機會也就更大一些。

特別注意的是，每一次投票的觀眾（可包括現場觀賞參賽者表演之觀眾），均必須以手機的「星星數」為基準，而且必須要準確預測，方可有機會得到對應搖錢樹的評審獎金。

每次均以三千個「星星」為單位進行投票，全民評審與觀眾的投票，可以「讚」與「再加油」的兩者之間均給予投票。當然，該集累計的星星，也無法累計到下一集繼續使用。

評審的搖錢樹獎金表如下表。

|      |                      |             |
| ---- | -------------------- | ----------- |
| 評審階級 | 累積「星星」數              | 獎金          |
| 鐵頭評審 | 0至50,000個「星星」        | 新臺幣5,000元整  |
| 銅牌評審 | 50,001至100,000個「星星」  | 新臺幣10,000元整 |
| 白銀評審 | 100,001至200,000個「星星」 | 新臺幣20,000元整 |
| 黃金評審 | 200,001至500,000個「星星」 | 新臺幣30,000元整 |
| 鑽石評審 | 500,001個以上個「星星」      | 新臺幣50,000元整 |
|      |                      |             |

特別注意的是，投票時段只限定於現場直播節目的時段，如在中天重播節目時段，觀眾是無法參與投票的。

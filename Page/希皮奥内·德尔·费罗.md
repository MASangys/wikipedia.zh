**希皮奥内·德尔·费罗**（Scipione del Ferro，），[意大利](../Page/意大利.md "wikilink")[数学家](../Page/数学家.md "wikilink")，1496年至1526年任[博洛尼亚大学](../Page/博洛尼亚大学.md "wikilink")[代数学和](https://zh.wikipedia.org/wiki/代数学 "wikilink")[几何学](../Page/几何学.md "wikilink")教授，他第一个发现了[一元三次方程的解法](../Page/三次方程.md "wikilink")。

## 生平

费罗出生在意大利北部的[博洛尼亚](../Page/博洛尼亚.md "wikilink")。当时[約翰尼斯·古騰堡刚在](https://zh.wikipedia.org/wiki/約翰尼斯·古騰堡 "wikilink")15世纪50年代獨立发明[活字印刷术](../Page/活字印刷术.md "wikilink")，这使得各类著作能够通过书本得到流传，由于费罗的父亲在[纸业工作](https://zh.wikipedia.org/wiki/纸业 "wikilink")，费罗在年轻的时候就能够接触到各种各样的作品。

费罗毕业于[博洛尼亚大学](../Page/博洛尼亚大学.md "wikilink")，从1496年开始直到他去世，费罗都在博洛尼亚大学教授[代数学和](https://zh.wikipedia.org/wiki/代数学 "wikilink")[几何学](../Page/几何学.md "wikilink")。

## 费罗与一元三次方程

意大利数学家[卢卡·帕西奥利](../Page/卢卡·帕西奥利.md "wikilink")（Luca Pacioli，1445年—1514年或1517年）于1494年在[威尼斯](../Page/威尼斯.md "wikilink")发表了[文艺复兴](../Page/文艺复兴.md "wikilink")时期最伟大的数学著作《Summa de arithmetica, geometrica, proportioni et proportionalita》，他在书中记录了对[一元三次方程解法的艰辛探索](../Page/三次方程.md "wikilink")，并下结论认为在当时的数学，求解一元三次方程是根本不可能的。

帕西奥利曾于1501年至1502年间来到[博洛尼亚大学](../Page/博洛尼亚大学.md "wikilink")任教，期间与同在博洛尼亚大学的费罗讨论过许多数学问题，人们并不知晓他们是否也曾讨论过一元三次方程问题，但是在帕西奥利离开[博洛尼亚](../Page/博洛尼亚.md "wikilink")后不久，费罗就至少解决了一元三次方程在一种情况下（x<sup>3</sup> + mx = n）的解，这在求解一元三次方程的道路上是一个突破性的成功。然而费罗并没有马上发表自己的成果，而是对解法保密，这很大程度上是因为他拒绝公开交流他的思想，他更愿意与他的朋友和学生交流，而不是将它们写下来出版，因此费罗的手稿并没有流传至今。\[1\]尽管如此，他曾有过一本笔记簿，记录了他所有的重要发现，其中包括一元三次方程的解法。在他1526年去世后，这本笔记簿由他的女婿Hannival Nave继承了，Nave也是一个数学家，他替代费罗继续在博洛尼亚大学授课。同时被传授这一解法的还有费罗的学生[菲奥尔](https://zh.wikipedia.org/wiki/菲奥尔 "wikilink")。

一元三次方程解法的进展在费罗去世后充满了戏剧性，先是菲奥尔在得到秘传后吹嘘自己能够解所有的一元三次方程，其实他只会费罗传授他的x<sup>3</sup> + mx = n，而另一位意大利数学家（[尼科洛·塔尔塔利亚](../Page/尼科洛·塔尔塔利亚.md "wikilink")（1499年—1557年12月13日）在1534年宣称自己发现了形如x<sup>3</sup> + mx<sup>2</sup> = n的方程的解，两人相约在[米兰进行公开比赛](https://zh.wikipedia.org/wiki/米兰 "wikilink")。1535年就在比赛前夕，塔塔利亚苦思冥想出来其他多种形式的一元三次方程解，从而轻而易举地赢得了比赛，并在1541年终于完全解决了一元三次方程的求解问题。与费罗相同的是，塔塔利亚同样选择保守解法的秘密。

同样研究一元三次方程的意大利[医生](../Page/医生.md "wikilink")、[哲学家和数学家](https://zh.wikipedia.org/wiki/哲学家 "wikilink")[吉罗拉莫·卡尔达诺](../Page/吉罗拉莫·卡尔达诺.md "wikilink")在允诺不公开的条件下，1539年从塔尔塔利亚那里得到了他的解法，在其基础上也发现了所有一元三次方程的解法。而在1543年，卡尔达诺和他的学生[卢多维科·费拉里](https://zh.wikipedia.org/wiki/卢多维科·费拉里 "wikilink")（Lodovico Ferrari，1522年2月2日—1565年10月5日）曾前往博洛尼亚，从费罗的女婿Nave处得知，其实费罗早于塔塔利亚已经发现了一元三次方程的解法，他便摒弃了给塔塔利亚的承诺，将他拓展的解法在1545年的著作《[大术](https://zh.wikipedia.org/wiki/大术 "wikilink")》（又译《[数学大典](https://zh.wikipedia.org/wiki/数学大典 "wikilink")》，Ars Magns）中发表，他在书中称，是费罗第一个发现了一元三次方程的解法，而他所给出的解法其实就是费罗的解法。由于卡尔达诺最早发表了求解一元三次方程的方法，因而该解法至今仍被称为“[卡尔达诺公式](https://zh.wikipedia.org/wiki/卡尔达诺公式 "wikilink")”。在《大术》中同时发表的还有费拉里的[一元四次方程一般解法](https://zh.wikipedia.org/wiki/一元四次方程 "wikilink")。

## 费罗的一元三次方程解法

一元三次方程形如

\[x^3 + ax^2 + bx + c = 0 \,\] 与费罗同时代的数学家们已经知道，一元三次方程可以用代入法（如y = x + a/3）消去二次项后，简化成四种形式：

\[x^3 + mx = n \,\]

\[x^3 = mx + n \,\]

\[x^3 + n = mx \,\]

\[x^3 + mx + n = 0 \,\] 其中系数m和n都为正数。费罗得出的是其中第一种形式的解法：

|                                                               |                                                 |
| ------------------------------------------------------------- | ----------------------------------------------- |
| \(x^3+m*x-n=0 \,\)                                            | \(x^3+6x-20=0 \,\)                              |
| \(D = \Big( \frac{m}{3} \Big)^3 + \Big( \frac{n}{2} \Big)^2\) | \(D = \big( 2 \big)^3 + \big(10 \big)^2 = 108\) |
| \(\text{ 在 } D > 0 \text{ 的情况下可解}\)                           |                                                 |
| \(v = \sqrt[3]{ \frac{n}{2} + \sqrt{D} }\)                    | \(v = \sqrt[3]{ 10 + \sqrt {108} } = 2.732051\) |
| \(u= \sqrt[3]{ \frac{n}{2} - \sqrt{D} }\)                     | \(u= \sqrt[3]{ 10 - \sqrt{108} } = -0.732051\)  |
| \(\text { 则得到解 } x=u+v\)                                      | \(\quad x=2.732051-0.732051=2\)                 |
| \(\text{将解代入方程：}\)                                            | \(\quad 2^3+6*2-20=0\)                          |

费罗公式只给出了一元三次方程的部分解，[卡尔达诺公式给出了完全解](https://zh.wikipedia.org/wiki/卡尔达诺公式 "wikilink")。

## 其他成就

除了一元三次方程的求解外，费罗还对分数的有理化做出了重要的贡献，他将分母从两个平方根之和扩展到了三个三次方根之和。

**帶式欖球**（亦稱：**非撞式欖球**）是一種[聯盟式橄欖球](../Page/聯盟式橄欖球.md "wikilink")的[衍生運動](https://zh.wikipedia.org/wiki/衍生運動 "wikilink")，以拿走對方帶在腰上的魔術貼以表示[抱截](https://zh.wikipedia.org/wiki/抱截 "wikilink")。當對方拿走其魔術貼時，需要大聲地說「TAG！」及將其拿高，以令被抱截者知道要傳球給後面的人。這個運動並沒有任何年齡、性別的限制。它的[危險度亦比](https://zh.wikipedia.org/wiki/危險 "wikilink")[聯盟式橄欖球](../Page/聯盟式橄欖球.md "wikilink")低，這是因為沒有[抱截的動作](https://zh.wikipedia.org/wiki/抱截 "wikilink")，並不會令他人受傷。這個運動雖然只是一種[衍生運動](https://zh.wikipedia.org/wiki/衍生運動 "wikilink")，但是這個運動亦都有變種，它們分別是[澳式帶式欖球及](https://zh.wikipedia.org/wiki/澳式帶式欖球 "wikilink")[小型帶式欖球](https://zh.wikipedia.org/wiki/小型帶式欖球 "wikilink")。如果持欖球的球員的魔術貼沒有被扯下，並到達[得分區](https://zh.wikipedia.org/wiki/得分區 "wikilink")，該隊並可得一分。比賽時，每隊分別有七人，每次比賽三場，每場約十五分鐘。

## 犯規設定

| 犯規描述                                                              | 懲罰           |
| ----------------------------------------------------------------- | ------------ |
| 向前跌球                                                              | 定位開球         |
| 向前[傳球](https://zh.wikipedia.org/wiki/傳球 "wikilink")               | 定位開球         |
| [越位](https://zh.wikipedia.org/wiki/越位 "wikilink")                 | 定位開球         |
| 開球的[球員不可以自己一開始便帶球進攻](https://zh.wikipedia.org/wiki/球員 "wikilink") | 轉發球          |
| 阻礙及身體碰撞                                                           | 被罰離場５分鐘或定位開球 |
| 將對方的帶子脫下後，將其亂拋                                                    | 定位開球         |
| 故意或無意踢球                                                           | 定位開球         |

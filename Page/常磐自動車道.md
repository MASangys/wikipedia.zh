**常磐自動車道**（）是起於[日本](../Page/日本.md "wikilink")[埼玉縣](../Page/埼玉縣.md "wikilink")[三鄉市](https://zh.wikipedia.org/wiki/三鄉市 "wikilink")[三鄉系統交流道](../Page/三鄉系統交流道.md "wikilink")，經[千葉縣](../Page/千葉縣.md "wikilink")、[茨城縣](../Page/茨城縣.md "wikilink")、[福島縣](https://zh.wikipedia.org/wiki/福島縣 "wikilink")，訖於[宮城縣](../Page/宮城縣.md "wikilink")[亘理町](https://zh.wikipedia.org/wiki/亘理町 "wikilink")[亘理交流道](../Page/亘理交流道.md "wikilink")的[高速公路](../Page/日本高速公路.md "wikilink")（[高速自動車國道](https://zh.wikipedia.org/wiki/高速自動車國道 "wikilink")）。略稱為**常磐道**（，JOBAN EXPWY）。高速公路路線編號與、還有的[仙台港北交流道](../Page/仙台港北交流道.md "wikilink") - [利府系統交流道](../Page/利府系統交流道.md "wikilink")間的路段被共同編為「**E6**」。

該路線與[東北縱貫自動車道](../Page/東北縱貫自動車道.md "wikilink")（[東北自動車道](https://zh.wikipedia.org/wiki/東北自動車道 "wikilink")）連結，以強化[北東國土軸為目的](https://zh.wikipedia.org/wiki/國土軸 "wikilink")。其中東京都練馬區-三鄉市區間與[東北縱貫自動車道](../Page/東北縱貫自動車道.md "wikilink")、[東關東自動車道](../Page/東關東自動車道.md "wikilink")共線，宮城縣黑川郡富谷町-仙台市區間與東北縱貫自動車道共線。

常磐自動車道當初的終點為磐城市，1987年國土開發幹線自動車道建設法改正，延伸至仙台市。\[1\]。包含建設中區間的全區間與[國道6號並行](../Page/國道6號_\(日本\).md "wikilink")。

2011年發生[東日本大地震之後](https://zh.wikipedia.org/wiki/東日本大地震 "wikilink")，因[福島第一核電廠事故影響](https://zh.wikipedia.org/wiki/福島第一核電廠事故 "wikilink")，位於[核電廠方圓](../Page/福島第一核電廠.md "wikilink")20公里警戒區內的廣野交流道至常磐富岡交流道區間封閉；常磐富岡交流道至原町交流道施工中路段也在20公里警戒區範圍而曾停工，直到2015年3月1日才全線通車。富岡交流道至原町交流道間路段裝有六套偵測[輻射劑量的設備](https://zh.wikipedia.org/wiki/輻射 "wikilink")，並且在路旁的電子看板顯示。據估計以時速70公里通過該路段，單程所受到的輻射劑量約0.2微[西弗](../Page/西弗.md "wikilink")，為照一張[胸部X光所受劑量的](https://zh.wikipedia.org/wiki/胸部X光 "wikilink")300分之一\[2\]。

以下記述的常磐自動車道為[東日本高速道路](../Page/東日本高速道路.md "wikilink")株式會社（NEXCO東日本）管理的高速自動車國道常磐自動車道，一般略稱**常磐道**。

  - 路線名稱未特別註明者為[市町村道](https://zh.wikipedia.org/wiki/市町村道 "wikilink")。
  - BS為巴士站：○/●是使用中、◆是停用中，未註記者即未設置。

## 歷史

  - 1966年7月1日 : 國土開發幹線自動車道路線預定。
  - 1981年4月27日 : 柏IC-谷田部IC間通車。
  - 1982年3月30日 : 谷田部IC-千代田石岡IC間通車。
  - 1983年12月22日 : 谷田部IC-千代田石岡IC間拓寬為6車道。
  - 1984年3月27日 : 千代田石岡IC-那珂IC間通車。
  - 1984年12月27日 : 柏IC-谷田部IC間拓寬為6車道。
  - 1985年1月24日 : 三鄉IC/JCT-柏IC間通車，三鄉JCT與首都高速6號三鄉線連接。
  - 1985年2月20日 : 那珂IC-日立南太田IC間通車。
  - 1985年7月3日 : 日立南太田IC-日立北IC間通車。
  - 1987年6月30日 : [第四次全國總合開發計劃](https://zh.wikipedia.org/wiki/第四次全國總合開發計劃 "wikilink")[閣議決定](https://zh.wikipedia.org/wiki/閣議 "wikilink")，**常磐自動車道延伸**為高規格幹線道路的構想。
  - 1987年9月1日 : 國土開發幹線自動車道建設法改正，東京都-仙台市為國幹道預定路線。
  - 1988年3月24日 : 日立北IC-磐城中央IC間通車，為最初的**全線通車**。
  - 1992年3月26日 : 流山IC通車。
  - 1992年11月27日 : 三鄉JCT與東京外環自動車道連接。
  - 1993年10月20日 : 日立中央IC通車，同時高鈴停车区改名日立中央停车区。
  - 1995年8月2日 : 磐城JCT通車，與磐越自動車道連接。
  - 1999年3月25日 : 磐城中央IC-磐城四倉IC間通車。
  - 2000年2月1日 : 千代田石岡IC-友部JCT間拓寬為6車道。
  - 2000年3月18日 : 友部JCT通車，與北關東自動車道連接。
  - 2000年6月5日 : 友部JCT-水戶IC間拓寬為6車道。
  - 2002年3月23日 : 磐城四倉IC-廣野IC間通車。
  - 2003年3月29日 : 筑波JCT通車，與圈央道連接。
  - 2004年4月14日 : 廣野IC-常磐富岡IC間通車。
  - 2005年7月1日 : 友部SA SMART IC[社會實驗開始](https://zh.wikipedia.org/wiki/社會實驗 "wikilink")（至2006年9月30日，當初實驗期間至2005年8月31日）。
  - 2005年10月1日 : [日本道路公團](../Page/日本道路公團.md "wikilink")分割民營化，由東日本高速道路承繼。
  - 2006年9月25日 : 水戶北SMART IC社會實驗開始（至2009年3月31日，當初實驗期間至2007年3月31日）。
  - 2006年10月1日 : 友部SA SMART IC開始使用。
  - 2008年12月19日 : 三鄉收費站SMART IC社會實驗開始。
  - 2009年3月29日 : 東海停车区 SMART IC開始使用。
  - 2009年4月1日 : 三鄉收費站SMART IC與水戶北SMART IC開始使用。
  - 2009年9月12日 : 山元IC-亘理IC間通車。
  - 2011年3月24日：石岡小美玉智能IC啟用。
  - 2012年4月8日：南相馬IC至相馬IC開通。
  - 2015年3月1日：常磐富岡IC至浪江IC開通，全線通車。

[数学中](../Page/数学.md "wikilink")，[複數的](https://zh.wikipedia.org/wiki/複數 "wikilink")**辐角**是指复数在[复平面上对应的](../Page/复平面.md "wikilink")[向量和正向](../Page/向量.md "wikilink")[实数轴所成的有向](../Page/实数.md "wikilink")[角](https://zh.wikipedia.org/wiki/角 "wikilink")。复数的辐角值可以是一切实数，但由于相差\(360^\circ\)（即[弧度](https://zh.wikipedia.org/wiki/弧度 "wikilink")\(2 \pi\)）的辐角在实际应用中没有差别，所以定义复数的**辐角主值**为辐角[模](https://zh.wikipedia.org/wiki/同余 "wikilink")\(360^\circ\)（\(2 \pi\)）后的余数，定义取值范围在\(0^\circ\)到\(360^\circ\)（\(2 \pi\)）之间。复数的辐角是复数的重要性质，在不少理论中都有重要作用。

## 定义

[Complex_number_illustration_multiarg.svg](https://zh.wikipedia.org/wiki/File:Complex_number_illustration_multiarg.svg "fig:Complex_number_illustration_multiarg.svg")
设有非零复数\(z \in \mathbb{C}\setminus \{ 0 \}\)，记作\(z = x + yi\)，其中的\(x\)和\(y\)为实数，那么复数\(z\)的幅角\(\varphi\)指的是使下列等式：

\[z = x + yi = \sqrt{x^2 + y^2}(\cos \varphi + i \sin \varphi)\]
成立的任何实数\(\varphi\)。直观上来说，假设非零复数\(z\)在复平面\(O_{xy}\)中对应的向量是\(\overrightarrow{OP}\)（右图蓝色向量），那么它的幅角是所有能够描述正实数轴到\(\overrightarrow{OP}\)的转角的有向角。其中有向角的正方向规定为逆时针方向。图中可以看出，相差\(2 \pi\)的倍数的角都可以是幅角。这个性质也可以从[三角函数](../Page/三角函数.md "wikilink")\(\cos\)和\(\sin\)是以\(2 \pi\)为周期的[周期函数中推导出来](../Page/周期函数.md "wikilink")。

只有非零复数才有幅角，复数\(0\)的幅角是没有定义的。

## 幅角主值

同一个复数的幅角有[无穷多个](../Page/无穷.md "wikilink")，以集合表示为\(\{ \varphi + 2k \pi \, | \, k \in \mathbb{Z} \}\)，而对于所有\(\varphi_k = \varphi + 2k \pi\)，\(\cos \varphi_k + i \sin \varphi_k\)都相同，所以实际只需要以其中一个幅角为代表，此幅角称为**幅角主值**或**主幅角**，记作\(\operatorname{Arg}(z)\)。一般约定使用[区间](https://zh.wikipedia.org/wiki/区间 "wikilink")\((-\pi, \pi]\)中的值作为幅角主值（也有另一种常见的约定是以区间\([0, 2 \pi)\)中的值作为幅角主值）。如果复数的幅角主值是\(\operatorname{Arg}(z)\)，那么它的所有幅角值就是：

\[\arg(z) = \{ \operatorname{Arg}(z) + 2k \pi \, | \, k \in \mathbb{Z} \}\]

## 幅角的计算

给定一个形如\(z = x + yi\)的非零复数，幅角主值\(\operatorname{Arg}(z)\)是将它[映射到区间](https://zh.wikipedia.org/wiki/映射 "wikilink")\((-\pi, \pi]\)中的[函数](../Page/函数.md "wikilink")。幅角主值函数可以用[反三角函数来描述](../Page/反三角函数.md "wikilink")：

\[\operatorname{Arg}(x + yi) =
\begin{cases}
\arccos \dfrac{x}{\sqrt{x^2 + y^2}} & y > 0 \\
-\arccos \dfrac{x}{\sqrt{x^2 + y^2}} & y < 0 \\
0 & x > 0 \land y = 0 \\
\pi & x < 0 \land y = 0 \\
\end{cases}\]

或者配合[半角公式](https://zh.wikipedia.org/wiki/半角公式 "wikilink")：

\[\operatorname{Arg}(x + yi) =
\begin{cases}
2 \arctan \dfrac{y}{\sqrt{x^2 + y^2} + x} & y \ne 0 \\
0 & x > 0 \land y = 0 \\
\pi & x < 0 \land y = 0 \\
\end{cases}\]

## 性质

复数\(z\)的一个幅角\(\varphi \in \arg(z)\)和[绝对值](../Page/绝对值.md "wikilink")\(|z|\)可以用来组成复数的[极坐标形式](https://zh.wikipedia.org/wiki/极坐标 "wikilink")：

\[z = |z|e^{i\varphi}\]。 在极坐标形式下计算，可以得到复数乘积和商的幅角的规律：

\[\operatorname{Arg}(z_1 z_2) = \operatorname{Arg}(z_1) + \operatorname{Arg}(z_2) \pmod{(-\pi, \pi]}\]

\[\operatorname{Arg} \left( \frac{z_1}{z_2} \right) = \operatorname{Arg}(z_1) - \operatorname{Arg}(z_2) \pmod{(-\pi, \pi]}\]
于是对复数[幂次的幅角也有](https://zh.wikipedia.org/wiki/幂次 "wikilink")：

\[\operatorname{Arg}(z^n) = n\operatorname{Arg}(z) \pmod{(-\pi, \pi]}\]
复数的[共轭的幅角则满足](../Page/共轭复数.md "wikilink")：

\[\operatorname{Arg}(\overline{z}) = -\operatorname{Arg}(z) \pmod{(-\pi, \pi]}\]

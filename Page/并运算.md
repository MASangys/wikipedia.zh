在数学中，集合上的**并**（）可以用两种方式定义：关于这个集合上的[偏序的唯一](https://zh.wikipedia.org/wiki/偏序集合 "wikilink")[上确界](https://zh.wikipedia.org/wiki/上确界 "wikilink")（最小上界），假定这种上确界存在的话；或者是满足[幂等律的](https://zh.wikipedia.org/wiki/幂等律 "wikilink")[交换](https://zh.wikipedia.org/wiki/交换律 "wikilink")[结合](../Page/结合律.md "wikilink")[二元运算](../Page/二元运算.md "wikilink")。在任何一个情况下，这个集合与并运算一起是[并半格](https://zh.wikipedia.org/wiki/并半格 "wikilink")。两个定义生成等价的结果，除了[偏序方式有可能直接的定义更一般的元素的集合的并之外](../Page/偏序关系.md "wikilink")。最常见到并运算的领域是[格](../Page/格_\(数学\).md "wikilink")。

\(x\) 和 \(y\) 的并通常被指示为 \(x \lor y\)。

## 偏序定义

设 *A* 是带有[偏序](../Page/偏序关系.md "wikilink") \(\leq\) 的一个集合，-{zh-hans:并;
zh-hant:並;}-设 \(x\) 和 \(y\) 是 *A* 中的两个元素。*A* 中的一个元素 \(z\) 是 \(x\) 和
\(y\) 的并（或最小上界或上确界），如果满足下列两个条件:

1.  \(x \leq z\) 且 \(y \leq z\) （就是说，\(z\) 是 \(x\) 和 \(y\) 的一个上界）
2.  对于 *A* 中任何 \(w\)，使得 \(x \leq w\) 且 \(y \leq w\)，有着 \(z \leq w\)
    （就是说，\(z\) 小于任何其他 \(x\) 和 \(y\) 的上界）。

如果 \(x\) 和 \(y\) 有并，则实际上它是唯一的，因为如果 \(z\) 和 \(z'\) 都是 \(x\) 和 \(y\)
的最小上界，则 \(z \leq z' \leq z\)，因此确实 \(z = z'\,\)。如果并存在，它被指示为
\(x \lor y\)。*A*
中的某些对元素可能缺乏并，要么因为它们根本就没有上界，要么因为它们的上界没有一个小于所有其他的。如果所有的元素对都有并，则这个交实际上是在
*A* 上的[二元运算](../Page/二元运算.md "wikilink")，并且容易看出这个运算满足下列三个条件: 对于 *A*
中任何元素 \(x\), \(y\) 和 \(z\) 有

1.  [交换律](https://zh.wikipedia.org/wiki/交换律 "wikilink")：\(x \lor y = y \lor x\)
2.  [结合律](../Page/结合律.md "wikilink")：\(x \lor (y \lor z) =(x \lor y) \lor z\)
3.  [幂等律](https://zh.wikipedia.org/wiki/幂等律 "wikilink")：\(x \lor x = x\)

## 泛代数定义

通过定义，在集合 *A* 上的[二元运算](../Page/二元运算.md "wikilink") \(\lor\)
是并，如果它满足上述三个条件 **a**, **b** 和 **c**。有序对
(*A*,\(\lor\))
就是[并半格](https://zh.wikipedia.org/wiki/并半格 "wikilink")。此外，我们可以定义在
*A* 上的[二元关系](../Page/二元关系.md "wikilink") \(\leq\)，通过声称 \(x \leq y\) 当且仅当
\(x \lor y = y\)。事实上，这个关系是在 *A*
上的[偏序](../Page/偏序关系.md "wikilink")。实际上，对于
*A* 中任何元素 \(x\), \(y\) 和 \(z\) 有

\[x \leq x\]，因为 \(x \lor x = x\)，通过公理 **c**；

  -
    如果 \(x \leq y\) 且 \(y \leq x\)，则
    \(y = x \lor y = y \lor x = x\)，通过公理 **a**；
    如果 \(x \leq y\) 且 \(y \leq z\)，则 \(x \leq z\)，因为
    \(x \lor z = x \lor (y \lor z) = (x \lor y) \lor z = y \lor z = z\)，通过公理
    **b**。

## 两个定义的等价性

如果 (*A*,\(\leq\))
是[偏序集合](https://zh.wikipedia.org/wiki/偏序集合 "wikilink")，使得
*A* 中每对元素都有并，则确实 \(x \lor y = y\) 当且仅当 \(x \leq y\)，因为在后者情况下 \(y\) 的确是
\(x\) 和 \(y\) 的上界，并且因为明显的 \(y\) 是最小上界当且仅当它是上界。因此，以泛代数方式的并定义的偏序一致于最初的偏序。

反过来说，如果 (*A*,\(\lor\)) 是并半格，并用泛代数的方式定义偏序 \(\leq\)，对于 *A* 中某些元素 \(x\) 和
\(y\) 有 \(z = x \lor y\)，则 \(z\) 是 \(x\) 和 \(y\) 关于 \(\leq\) 的最小上界，因为
\(x \lor z = (x \lor x) \lor y = z \;\Rightarrow\; x \leq z\)，类似的
\(y \leq z\)，并且如果 \(w\) 是 \(x\) 和 \(y\) 的另一个上界，则
\(x \lor w = y \lor w = w\)，因而
\(z \lor w = (x \lor y) \lor w = x \lor (y \lor w) = x \lor w = w\)。所以最初的并定义的偏序定义的并一致于最初的并。

换句话说，这两种方式生成本质上等价的概念，集合配备了二元关系和二元运算二者，使得每个结构都有另一个确定，而且分别满足关于偏序或并的那些条件。

## 一般子集的并

如果 (*A*,\(\lor\))
是并半格，则并可以被扩展为任何非空有限集合的良好定义的并，通过在[迭代二元运算中描述的技术](https://zh.wikipedia.org/wiki/迭代二元运算 "wikilink")。可作为替代的，并定义或定义自偏序，*A*
的某个子集的确有关于它的[上确界](https://zh.wikipedia.org/wiki/上确界 "wikilink")。对于非空有限子集，这两种方式生成同样的结果，因此任何一个都可以作为并的定义。在
*A* 的每个子集都有并的情况下，实际上 (*A*,\(\leq\))
是[完全格](../Page/完全格.md "wikilink")；详情请参见[完全性
(序理论)](https://zh.wikipedia.org/wiki/完全性_\(序理论\) "wikilink")。

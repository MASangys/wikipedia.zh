《**幹探群英**》（英文：**The Crime
File**），是香港無綫電視1991年拍攝的電視連續劇，由13個單元故事組成，監製[彭濟材](https://zh.wikipedia.org/wiki/彭濟材 "wikilink")。

## 各集單元

| 集數 | 單元主題     |
| -- | -------- |
| 1  | 犂庭掃穴     |
| 2  | 紅星五四     |
| 3  | 火網柔情     |
| 4  | 虎穴追擊（上集） |
| 5  | 虎穴追擊（下集） |
| 6  | 烈血胭脂（上集） |
| 7  | 烈血胭脂（下集） |

| 9  | 偽證情真     |

| 11 | 城市煞星（下集） |
| 12 | 情義邊緣（上集） |
| 13 | 情義邊緣（下集） |

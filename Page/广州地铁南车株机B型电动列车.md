**广州地铁南车株机B型电动列车**是[广州地铁](../Page/广州地铁.md "wikilink")的[电动车组车款之一](https://zh.wikipedia.org/wiki/电动车组 "wikilink")，目前在[3号线运营](../Page/广州地铁3号线.md "wikilink")。

## 简介

该批列车由[南车株洲电力机车进行制造](https://zh.wikipedia.org/wiki/南车株洲电力机车 "wikilink")，部分列车由南车株机旗下的广州南车城市轨道装备有限公司进行组装制造。目的是为了缓解3号线的客流压力以及为3号线北延段通车而购买的。

本列车分两种批次，第一批次内部型号另称**B2型**；第二批次内部型号另称**B4型**。前者在2010年4月20日首次载客，后者在亚运会之后陆续上线载客。B2型共制造25列（150辆），B4型共制造26列（156辆）。

该批所有列车不定期行驶于3号线北延段以及3号线主支线，因此列车所停放的[厦滘车辆段和](https://zh.wikipedia.org/wiki/厦滘车辆段 "wikilink")[嘉禾车辆段亦不固定](https://zh.wikipedia.org/wiki/嘉禾车辆段 "wikilink")。

## 历史

  - 2008年12月8日，三号线北延段列车采购合同正式签订。中标制造商为南车株洲-[西门子联合体](https://zh.wikipedia.org/wiki/西门子 "wikilink")，将为广州地铁提供25列150辆高速B型列车。
  - 2010年1月28日，在中国南车株洲电力机车有限公司地铁交车线上，首列广州地铁三号线北延的地铁列车交付。后于2010年1月30日运抵广州（列车编号为03043-03044）。
  - 2010年4月20日，广州东站至体育西路段开始使用6节编组列车亦代表该列车首次进行载客，之后该批列车从同年4月28日起行走于番禺广场至天河客运站段。
  - 2010年11月14日，国家发展改革委批复增购车辆，其中包含3号线的增购列车。
  - 2011年5月，[南車株洲電力機車投得該款列車的生產權](https://zh.wikipedia.org/wiki/南車株洲電力機車 "wikilink")，並簽訂車輛採購合同（該合同同時包含一二八號綫增購列車），26列，6輛編組共156輛。
  - 2011年7月1日，增购车运抵广州，后于同年年底陆续上线载客。
  - 2015年1月5日，首列搭載株洲南車時代電氣（現株洲中車時代電氣）旋轉電機牽引系統列車（03121-122）正式上线载客。

## 列车细节概要

### 外观

列车外观与初期购买的[B1型列车除涂装外基本大致相同](https://zh.wikipedia.org/wiki/广州地铁南车株机-西门子列车 "wikilink")，车身使用[梯型结构](https://zh.wikipedia.org/wiki/梯型结构 "wikilink")，使用[铝合金车体](https://zh.wikipedia.org/wiki/铝合金 "wikilink")。涂装方面使用类似[飞机形状的车头](https://zh.wikipedia.org/wiki/飞机 "wikilink")，两边颜色以橙色和银色色带为主，另各有一条白色饰带，车顶部分亦涂成亮灰色。

### 内部

[Guangzhou_Metro_L3-CSR_B2_Train_for_inside_trunk_space.JPG](https://zh.wikipedia.org/wiki/File:Guangzhou_Metro_L3-CSR_B2_Train_for_inside_trunk_space.JPG "fig:Guangzhou_Metro_L3-CSR_B2_Train_for_inside_trunk_space.JPG") [Guangzhou_Metro_Line_3_for_B4_train_Wheelchair.JPG](https://zh.wikipedia.org/wiki/File:Guangzhou_Metro_Line_3_for_B4_train_Wheelchair.JPG "fig:Guangzhou_Metro_Line_3_for_B4_train_Wheelchair.JPG") 每个单元车设6张压纹铝合金长座椅，两车连接处部分会设置设备区、3人用的压纹铝合金座椅以及无障碍空间。照明使用[荧光灯光管及不透明的塑料燈罩](https://zh.wikipedia.org/wiki/荧光灯 "wikilink")。車窗左侧配备4:3的显示器，播放地铁电视。车门旁两侧亦设有紧急开门装置，部分设有紧急求助按钮。

B2型所有列車及B4型前10列（03091～03110）均設連續式閃燈路綫圖，均有主綫和支綫兩段的閃燈；B4型其余列车改设貼紙路線圖及LED显示屏，显示下一站、終點站、換乘路線等行車信息。

列车使用不锈钢扶手，部分列车的扶手涂上灰色油漆，扶手偏上的位置用红色衬托，值得一提的是，车厢对两车门之间，取消了竖直的扶手。

值得一提的是，B2型列车以及B4型部分列车设有大件行李架供来往[机场的乘客使用](https://zh.wikipedia.org/wiki/广州白云国际机场 "wikilink")，唯3号线客流日益增大，地铁公司决定拆卸大件行李架以增大列车载客量。

### 其它部分细节

  - B2型的03087-03088号装备有国产化制动系统。
  - B4型的03121-03122采用[株洲南车时代自主研发的](https://zh.wikipedia.org/wiki/株洲南车时代 "wikilink")120km/h速度级牵引系统。
  - B4型的03121-03122乃首列在广州南车城市轨道装备有限公司组装的列车，在广州南车组装的列车照明会使用[暖色荧光灯](https://zh.wikipedia.org/wiki/荧光灯 "wikilink")。

## 列车编组

列車以6节為一列，編組方式為A-B-C-C-B-A（= Mc + Tp + M - M + Tp + Mc =）。當中A-C-B（或B-C-A）為一組，調卡時，都會整組調走，但本系列車尚未有任何調卡情況。

|         |
| ------- |
| B2型、B4型 |
| 車廂編號    |
| 受电弓     |
| 动力单元    |
| 动力单元    |

列車編組符號意義：

  - M—动车
  - Tp—带受电弓的拖车
  - Mc—带驾驶室的动车
  - ＝ —全自动车钩
  - \+ —半永久牵引杆

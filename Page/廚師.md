**廚師**是一種為他人提供專業烹飪服務的人。通常意義上，廚師是指對烹飪已經高度熟練掌握的人，他們熟悉食物製作的各個方面。现代[社会](../Page/社会.md "wikilink")中，多数厨师就职于公开服务的饭馆、饭店等场所。也有少数服务于特定的顾客（例如私人厨师、[家廚](https://zh.wikipedia.org/wiki/家廚 "wikilink")）。現代的厨师一般需要先在烹饪学校学习并通过考试，以保证具备足够的业务标准和食品安全知识。

## 廚師等級

### 西式料理

[Chef_préparant_une_truffe.jpg](https://zh.wikipedia.org/wiki/File:Chef_préparant_une_truffe.jpg "fig:Chef_préparant_une_truffe.jpg")。\]\]

以下是給那些在專業廚房工作人士的頭銜，每種都可被認為是一種獨特的廚師類型。這些頭銜大部分是來自[奧古斯特·埃科菲所著](https://zh.wikipedia.org/wiki/奧古斯特·埃科菲 "wikilink")《》，而另一部分則是由該廚師所擔任的工作來決定的。

#### 主廚

主廚是廚房內負責管理菜單設計、人員調度、食材採購和庫存，以及餐廳風格設計的人。

主廚在英文中有多種寫法，例如：、和。源自法語，是英語中對“主廚”的標準說法；而則偏向俚語；其中相較於另兩個詞又略有不同，更偏向於行政主廚，意味著他可能是多個廚房的主廚。

在美國，根據美國烹飪協會的規定，主廚在美國可以認證為：主廚執照（）、行政主廚執照（）和大師主廚執照（）三級。

#### 副主廚

[Tandoor_chef_2.jpg](https://zh.wikipedia.org/wiki/File:Tandoor_chef_2.jpg "fig:Tandoor_chef_2.jpg")的廚師。\]\]

副主廚是廚房內的第二號管理者，同時也是主廚的直接助手。在主廚休息的時候，副主廚擁有調配廚房人手的權力；同時副主廚還要在主管廚師需要的時候替補或輔助他們的工作。此外，副主廚還要負責廚房的庫存狀況、清潔工作、人員組織和廚師們的上崗培訓工作。一些較小的餐廳可能不設有副主廚這個職位；但是在一些很大的餐廳裡，通常會有多位副主廚。

#### 主管廚師

主管廚師一般是負責廚房內某個特定的烹飪區域。在大廚房里，每個主管廚師可能還會被分配有幾個廚師或助理廚師；而在小廚房里，主管廚師可能就是這個烹飪區的唯一工作人員。如果是在流水線廚房內，根據每個人的工作層次，主管廚師下還可以分為“第一廚師”、“第二廚師”等等。

廚房內的烹飪區一般包括：

| 中文   | 描述                                         |
| ---- | ------------------------------------------ |
| 熱炒廚師 | 負責廚房內所有熱炒的項目及醬汁的調配。這是主管廚師中地位最高的。           |
| 魚案廚師 | 準備魚類食材，負責魚類的宰殺和基礎調味。通常魚案廚師會和熱炒廚師組合在一起。     |
| 烤煮廚師 | 負責肉類食材的蒸煮及燒烤，并負責相應的醬汁調配。                   |
| 燒烤廚師 | 負責所有食物的燒烤。這個工作會和烤煮廚師有所重疊。                  |
| 煎炸廚師 | 負責所有食物的煎炸。這個工作也會和烤煮廚師有所重疊。                 |
| 蔬菜廚師 | 負責熱盤開胃菜的準備，同時也負責湯、菜、麵及澱粉類食物的烹製。            |
| 煮湯廚師 | 負責湯類的製作。在小廚房裡，這個工作一般由蔬菜廚師替代。               |
| 素菜廚師 | 負責蔬菜類的烹飪。在小廚房裡，這個工作一般由蔬菜廚師替代。              |
| 茶飲廚師 | 負責冷盤開胃菜的製作，例如：沙拉、冷盤、醬肉等熟食項目。               |
| 切肉師  | 負責廚房里肉類、禽類（有時也包含魚類）的切塊及預處理。                |
| 點心師  | 烘焙食品如糕點，蛋糕，麵包和甜點。在規模較大的餐廳，點心師會有自己獨立的團隊和廚房。 |
| 應急廚師 | 沒有固定的安排，通常作為餐廳裡最緊急區域的應急人手。                 |

## 厨师的历史

[Tandoor_chef.jpg](https://zh.wikipedia.org/wiki/File:Tandoor_chef.jpg "fig:Tandoor_chef.jpg")厨师在烤炉（）前工作\]\] 厨师的职业应该和人类烹调同步出现。但在历史上多数时候厨师是比较低微的职业。所以虽然饮食艺术在很多国家的传统中占重要地位，真正动手烹调的厨师们却不见于史册。可考的厨师多是在权贵传记和美食家的记载中出现。

### 中国历史

[上古传说帝](https://zh.wikipedia.org/wiki/中國傳說時代 "wikilink")[尧](../Page/尧.md "wikilink")的时候，[彭铿因进献雉羹](../Page/彭祖.md "wikilink")，尧便把[彭城封给他](https://zh.wikipedia.org/wiki/彭城 "wikilink")，所以后世称他为彭祖，[舜](../Page/舜.md "wikilink")的时候，他从师尹寿子，学得真道，遂隐居[武夷山](https://zh.wikipedia.org/wiki/武夷山 "wikilink")，[汉代](https://zh.wikipedia.org/wiki/汉代 "wikilink")[楚辞](../Page/楚辞.md "wikilink")专家[王逸注曰](https://zh.wikipedia.org/wiki/王逸 "wikilink")： “彭铿，彭祖也。好和滋味，善斟雉羹，能事帝尧，帝尧美而飨食之也”。宋代洪兴祖补注曰：“彭祖姓钱名铿，帝颛顼玄孙，善养气， 能调鼎，进雉羹于尧，封于彭城”。 彭铿是彭部族的始祖，以后子孙繁衍，主要是他的“雉羹之道”可祖，便尊称他为彭祖，他的后裔就叫彭祖氏。彭祖的“雉羹之道”逐步发展成为“烹饪之道”，雉羹是中国典籍中 记载最早的名馔，被誉为“天下第一羹”。中国烹饪史略中称彭祖“是我国第一位著名的职业厨师”，而且是“寿命最长的厨师”，至今被尊为厨行的祖师爷。

[中国古代最早有记载的厨师是](https://zh.wikipedia.org/wiki/中国 "wikilink")[商汤](../Page/商汤.md "wikilink")的私人厨师[伊尹](../Page/伊尹.md "wikilink")。

[春秋时](https://zh.wikipedia.org/wiki/春秋 "wikilink")，厨师[易牙](../Page/易牙.md "wikilink")曾蒸了自己的儿子给[主君](https://zh.wikipedia.org/wiki/主公 "wikilink")[齐桓公](../Page/齐桓公.md "wikilink")品尝。[庄子](../Page/庄子.md "wikilink")在《[庄子](https://zh.wikipedia.org/wiki/南華經 "wikilink")·养生主》里讲了“[庖丁解牛](../Page/庖丁解牛.md "wikilink")”的故事，借一位名叫「丁」的屠夫兼厨师，肢解牛的故事，来阐述哲学原理。

[南宋](../Page/南宋.md "wikilink")的宋五嫂所制宋嫂鱼羹非常有名，《摭青杂记》记载她曾得到[宋高宗](../Page/宋高宗.md "wikilink")的召见。

[清朝](../Page/清朝.md "wikilink")的[作家和美食家](https://zh.wikipedia.org/wiki/作家 "wikilink")[袁枚](../Page/袁枚.md "wikilink")在《随员食单》中提到了萧美人、陶方伯夫人等厨师。另外他还写有[中国历史](../Page/中国历史.md "wikilink")的首篇厨师传记《厨者王小余传》\[1\]。

### 西方历史

[Chefs.jpg](https://zh.wikipedia.org/wiki/File:Chefs.jpg "fig:Chefs.jpg") [US_Navy_080822-A-1912B-026_A_food_service_worker_from_Joint_Task_Force_Guantanamo's_Seaside_Galley_prepares_a_special_meal.jpg](https://zh.wikipedia.org/wiki/File:US_Navy_080822-A-1912B-026_A_food_service_worker_from_Joint_Task_Force_Guantanamo's_Seaside_Galley_prepares_a_special_meal.jpg "fig:US_Navy_080822-A-1912B-026_A_food_service_worker_from_Joint_Task_Force_Guantanamo's_Seaside_Galley_prepares_a_special_meal.jpg") 在古代[欧洲](../Page/欧洲.md "wikilink")，高级厨师们多为[贵族所雇用](https://zh.wikipedia.org/wiki/贵族 "wikilink")。饭馆、旅店等公众用餐地方的烹调相对平庸。但到了[法国大革命](../Page/法国大革命.md "wikilink")期间，贵族们很多被镇压或者出逃国外。这些厨师们只得改受雇于饭馆饭店等地，从而提高了平民饮食的精致程度。

[英文裡的厨师称作](https://zh.wikipedia.org/wiki/英文 "wikilink")，这个词直接来自[法文](https://zh.wikipedia.org/wiki/法文 "wikilink")，原意是头儿、主管。

现代厨师戴的白色高帽子是19世纪[法国厨师](https://zh.wikipedia.org/wiki/法国 "wikilink")所创。白色象征厨房的洁净，厨师的等级越高[帽子](../Page/帽子.md "wikilink")也越高。据说Carmene自己的帽子高达18英寸（45厘米）。

## 厨师的级别

### 美国

[美国厨师协会](https://zh.wikipedia.org/wiki/美国厨师协会 "wikilink")(ACF)目前承认13种厨师级别：\[2\]

  - 总厨师长（Executive Chef）：负责厨房的总体[管理](https://zh.wikipedia.org/wiki/管理 "wikilink")，包括设计菜单、人事和[商业](../Page/商业.md "wikilink")方面。
  - 私人厨师长（Personal Executive Chef）
  - 面点厨师长（Executive Pastry Chef）
  - 烹调管理员（Culinary Administrator）
  - 副厨师长 （Sous Chef）：服务于厨师长，烹调之外要监管其他厨师。
  - 司厨长（Chef de Cuisine）：直接负责食品烹调。
  - 大厨/面点大厨 （Master Chef/Master Pastry Chef）
  - 私人厨师（Personal Chef）
  - 操作面点厨师（Working Pastry Chef）
  - 烹调教师（Culinary Educator）：可就职于高等教育机构。
  - 二级烹调教师（Secondary Culinary Educator）：可就职于高中级别的职业学校。
  - 初级厨师（Culinarian）
  - 初级面点厨师（Pastry Culinarian）

### 中国大陆

下列各类又分初级、中级、高级、技师和高级技师\[3\]：

  - 中式烹调师
  - 中式面点师
  - 中式烧腊师
  - 西式烹调师
  - 西式面点师

### 台湾

下列各类分乙级和丙级\[4\]：

  - 中餐烹調技術士
  - 西餐烹調技術士
  - 烘焙技術士 麵包、蛋糕
  - 調酒技術士

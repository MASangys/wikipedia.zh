## 列车特点

[Guangzhou_Metro_L9_LCD_Information_System_above_train_doors.jpg](https://zh.wikipedia.org/wiki/File:Guangzhou_Metro_L9_LCD_Information_System_above_train_doors.jpg "fig:Guangzhou_Metro_L9_LCD_Information_System_above_train_doors.jpg") 广州地铁九号线列车每节车厢车门上方均设有LCD动态屏，以提供列车终点站、到站及换乘信息\[1\]，这是首款使用LCD屏幕来显示运营资讯的广州地铁车辆。

另外，這批列車採用ABB製造的牽引系統，有別於其他中車株洲電力機車製造列車採用的[西門子或](https://zh.wikipedia.org/wiki/西門子 "wikilink")[中車時代電氣製造的牽引系統](https://zh.wikipedia.org/wiki/中車時代電氣 "wikilink")。

## 歷史

  - 2017年4月21日，九號線首列列車正式交付。
  - 2017年12月28日，九號線列車隨9號線開通而正式投入使用。

## 編組

本款列車以6節車廂為一列，每列車由兩個動力單元組成，每個動力單元均由一個不帶[受電弓](https://zh.wikipedia.org/wiki/受電弓 "wikilink")[動車](https://zh.wikipedia.org/wiki/動車組動車 "wikilink")、一個帶受電弓動車和一個带[駕駛室](https://zh.wikipedia.org/wiki/控制車 "wikilink")[拖車組成](https://zh.wikipedia.org/wiki/動車組拖車 "wikilink")，合計四動二拖。以下是列車的編組方式：

|           |
| --------- |
| **九號線列車** |
| 動力單元      |
| 車廂編號      |
| 受电弓       |
| 列车编组      |

列車編組符號意義：

  - Tc — 带駕駛室拖車
  - Mp — 带受電弓動車
  - M — 不带受電弓動車
  - \= — 全自动車鉤
  - \+ — 半自動車鉤
  - \- — 半永久牽引桿

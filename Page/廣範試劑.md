**廣用試劑**，又名通用指示劑或廣範試劑。一般用來指示溶液的酸鹼性，並可以指示酸性和鹼性範圍內的各種[pH值](https://zh.wikipedia.org/wiki/pH值 "wikilink")，其變色之 pH 範圍如下表：　　　

|         |   |   |   |   |   |   |    |
| ------- | - | - | - | - | - | - | -- |
| 所指示的pH值 | 4 | 5 | 6 | 7 | 8 | 9 | 10 |
| 廣範試劑的顏色 | 紅 | 橘 | 黃 | 綠 | 藍 | 靛 | 紫  |

pH值\<4，顏色皆為<span style="color:red">紅</span>色

pH值\>10，顏色皆為<span style="color:purple">紫</span>色

廣範試劑是[pH指示劑的一種](https://zh.wikipedia.org/wiki/pH指示劑 "wikilink")，其適用範圍為pH3\~12，可用以顯示不同溶液之酸鹼度。雖然市面上有多種不同的廣範試劑，但大部分都是從於1933年所獲得專利的配方變化而來的。\[1\] 這個專利細節可以在[化學文摘找到](https://zh.wikipedia.org/wiki/化學文摘 "wikilink")。\[2\] 有關山田的廣範試劑之實驗也刊登在。\[3\]

## 试剂配方

|                                                                                                      |
| ---------------------------------------------------------------------------------------------------- |
| pH值：大約7                                                                                              |
| [蒸氣壓](../Page/蒸氣壓.md "wikilink")：25 mmHg (3.3 kPa)                                                   |
| [蒸氣](https://zh.wikipedia.org/wiki/蒸氣 "wikilink")[密度](../Page/密度.md "wikilink")：1.3 kg/m<sup>3</sup> |

配方成分如下：\[4\]

|                                                       |      |
| ----------------------------------------------------- | ---- |
| [酚酞](../Page/酚酞.md "wikilink")                        | 1.3克 |
| [溴百里酚藍](../Page/溴百里酚藍.md "wikilink")                  | 0.9克 |
| [甲基紅](https://zh.wikipedia.org/wiki/甲基紅 "wikilink")   | 0.4克 |
| [百里酚藍](https://zh.wikipedia.org/wiki/百里酚藍 "wikilink") | 0.2克 |

把以上4種指示劑的质量量度好，然後溶解在1公升的70～80％[酒精中](https://zh.wikipedia.org/wiki/酒精 "wikilink")（或普通工業酒精中）。

配方成分各種藥品所佔的百分率如下\[5\]:

|                                                       |        |
| ----------------------------------------------------- | ------ |
| [水](../Page/水.md "wikilink")                          | 62.1 % |
| [異丙醇](https://zh.wikipedia.org/wiki/異丙醇 "wikilink")   | 37 %   |
| [甲醇](../Page/甲醇.md "wikilink")                        | 0.8 %  |
| [酚酞](../Page/酚酞.md "wikilink")                        | 0.03 % |
| [溴百里酚藍](../Page/溴百里酚藍.md "wikilink")                  | 0.03 % |
| [百里酚藍](https://zh.wikipedia.org/wiki/百里酚藍 "wikilink") | 0.03 % |
| [甲基紅](https://zh.wikipedia.org/wiki/甲基紅 "wikilink")   | 0.02 % |

完全溶解後，再加少量0.1M [氫氧化鈉溶液](https://zh.wikipedia.org/wiki/氫氧化鈉 "wikilink")，使它剛變綠色即可應用。如備製試紙，可用[濾紙](../Page/濾紙.md "wikilink")吸取試液後陰乾備用。

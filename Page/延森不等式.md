**琴生不等式（Jensen's
inequality）**以[丹麥](../Page/丹麥.md "wikilink")[數學家](../Page/數學家.md "wikilink")[約翰·延森](../Page/約翰·延森.md "wikilink")（Johan
Jensen）命名。它給出[積分的](../Page/積分.md "wikilink")[凸函數值和凸函數的積分值間的關係](../Page/凸函數.md "wikilink")。延森不等式有以下推论：过一个[凸函数上任意两点所作](../Page/凸函数.md "wikilink")[割线一定在这两点间的函数图象的下方](../Page/割线.md "wikilink")，即：

\[t f(x_1) + (1-t) f(x_2) \leq f \left (t x_1 + (1-t) x_2 \right ), 0 \leq t \leq 1.\]

## 一般形式

延森不等式可以用[測度論或](../Page/測度論.md "wikilink")[概率論的語言給出](../Page/概率論.md "wikilink")。這兩種方式都表明同一個很一般的結果。

### 測度論的版本

假設\(\mu\)是集合\(\Omega\)的正測度，使得\(\mu(\Omega) = 1\)。若\(g\)是[勒貝格可積的](../Page/勒貝格積分.md "wikilink")[實值函數](../Page/實數.md "wikilink")，而\(\varphi\)是在\(g\)的值域上定義的[凸函數](../Page/凸函數.md "wikilink")，則

\[\varphi\left(\int_{\Omega} g\, d\mu\right) \le \int_\Omega \varphi \circ g\, d\mu\]。

### 概率論的版本

以概率論的名詞，\(\mu\)是個[概率測度](../Page/概率測度.md "wikilink")。函數\(g\)換作實值[隨機變數](../Page/隨機變數.md "wikilink")\(X\)（就純數學而言，兩者沒有分別）。在\(\Omega\)空間上，任何函數相對於概率測度\(\mu\)的積分就成了[期望值](../Page/期望值.md "wikilink")。這不等式就說，若\(\varphi\)是任一凸函數，則

\[\varphi\left(E(X)\right) \leq E(\varphi(X))\,\]。

## 特例

### 機率密度函數的形式

假設\(\Omega\)是實數軸上的可測子集，而\(f(x)\)是非負函數，使得

\[\int_{-\infty}^\infty f(x)\,dx = 1\]。

以概率論的語言，\(f\)是個[機率密度函數](../Page/機率密度函數.md "wikilink")。

延森不等式变成以下關於凸積分的命題：

若\(g\)是任一實值可測函數，\(\phi\)在\(g\)的值域中是凸函數，則

\[\varphi\left(\int_{-\infty}^\infty g(x)f(x)\, dx\right) \le \int_{-\infty}^\infty \varphi(g(x)) f(x)\, dx\]。

若\(g(x)=x\)，則這形式的不等式簡化成一個常用特例：

\[\varphi\left(\int_{-\infty}^\infty x\, f(x)\, dx\right) \le \int_{-\infty}^\infty \varphi(x)\,f(x)\, dx\]。

### 有限形式

若\(\Omega\)是有限集合\(\{x_1,x_2,\ldots,x_n\}\)，而\(\mu\)是\(\Omega\)上的[正規](../Page/正規化常數.md "wikilink")[計數測度](../Page/計數測度.md "wikilink")，則不等式的一般形式可以簡單地用和式表示：

\[\varphi\left(\sum_{i=1}^{n} g(x_i)\lambda_i \right) \le \sum_{i=1}^{n} \varphi(g(x_i))\lambda_i\]，

其中\(\lambda_1 + \lambda_2 + \cdots + \lambda_n = 1, \lambda_i \ge 0\)。

若\(\phi\)是凹函數，只需把不等式符號調轉。

假設\(x_1, x_2, \ldots, x_n\)是正實數，\(g(x)=x\)，\(\lambda_i = 1/n\)及\(\varphi(x) = \log(x)\)。上述和式便成了

\[\log\left(\sum_{i=1}^{n} \frac{x_i}{n} \right) \ge \sum_{i=1}^{n} \frac{\log(x_i)}{n}\]，

兩邊取[自然指數就得出熟悉的](../Page/自然指數.md "wikilink")[-{zh-cn:均值不等式;
zh-hant:平均數不等式}-](../Page/平均數不等式.md "wikilink")：

\[\frac{x_1 + x_2 + \cdots + x_n}{n} \ge \sqrt[n]{x_1 x_2 \cdots x_n}\]。

這不等式也有無限項的離散形式。

### 統計物理學

統計物理學中，若凸函數是指數函數，延森不等式特別重要：

\[e^{\langle X \rangle} \leq \left\langle e^X \right\rangle\]，

其中方括號表示[期望值](../Page/期望值.md "wikilink")，是以[隨機變數](../Page/隨機變數.md "wikilink")*X*的某個[概率分佈算出](../Page/概率分佈.md "wikilink")。這個情形的證明很簡單（參見Chandler,
Sec. 5.5）：在以下等式的第三個指數函數

\[\left\langle e^X \right\rangle
= e^{\langle X \rangle} \left\langle e^{X - \langle X \rangle} \right\rangle\]

套用不等式

\[e^X \geq 1+X \,\]，

即得出所求的不等式。

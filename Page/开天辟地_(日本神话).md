广义上，**开天辟地**是一个在[日本神话中描述天地形成](https://zh.wikipedia.org/wiki/日本神话 "wikilink")，早期诸神诞生的故事，而狭义上仅表示《[日本书纪](https://zh.wikipedia.org/wiki/日本书纪 "wikilink")》中“古天地未剖，陰陽不分，……”所描述的部分。

## 概述

### 《日本书纪》

#### 天地形成

在《日本书纪》中，对于形成前的天地是这样描述的：“古天地未剖，陰陽不分，渾沌如鷄子，溟涬而含牙。”后来，其中清明部分往上升腾就形成天，重浊部分凝结下沉就形成地。清明部分的气容易汇聚到一起，重浊部分的气很难融合在一起，所以最先形成的是天，地是后来才形成的。

#### 根源神的形成

天地初成后，诸神开始出现。

在**本文**中是如此记载的：“于時，天地之中生一物，狀如[葦牙](../Page/芦苇.md "wikilink")，便化爲神。”

  - 先生[国常立尊](https://zh.wikipedia.org/wiki/国常立尊 "wikilink")（くにのとこたち の みこと）
  - 次[国狭槌尊](https://zh.wikipedia.org/wiki/国狭槌尊 "wikilink")（くにのさつち の みこと）
  - 次[丰斟渟尊](https://zh.wikipedia.org/wiki/丰斟渟尊 "wikilink")（とよくんぬ の みこと）

这三位神因是阳刚之道自行转化，所以没有性别。

**第一种异说**中说：“天地初判，一物在於虛中，狀貌難言。其中自有化生之神。”

  - 先生[国常立尊](https://zh.wikipedia.org/wiki/国常立尊 "wikilink")（くにのとこたち の みこと）
      - 又称 国底立尊（くにのそこたち の みこと）
  - 次[国狭槌尊](https://zh.wikipedia.org/wiki/国狭槌尊 "wikilink")（くにのさつち の みこと）
      - 又称 国狭立尊（くにのさたち の みこと）
  - 次[丰国主尊](https://zh.wikipedia.org/wiki/丰国主尊 "wikilink")（とよくんぬ の みこと）
      - 又称 丰组野尊（とよくんの の みこと）
      - 丰香节野尊（とよかんの の みこと）
      - 浮经野丰买尊（うきふの の とよかひのみこと）
      - 丰国野尊（とよくんの の みこと）

      - 葉木国野尊（はこくんの の みこと）

**第二种异说**中说：“古國稚地稚之時，譬猶浮膏而漂蕩。于時國中生物，狀如葦牙之抽出也。因此有化生之神。”

  - 先生[可美苇牙彦舅尊](https://zh.wikipedia.org/wiki/可美苇牙彦舅尊 "wikilink")（うましあしかひひこぢ の みこと）

**第三种异说**中说：“天地混成之時，始有神人焉。”

  - 先生[可美苇牙彦舅尊](https://zh.wikipedia.org/wiki/可美苇牙彦舅尊 "wikilink")（うましあしかひひこぢ の みこと）
  - 次[国底立尊](https://zh.wikipedia.org/wiki/国底立尊 "wikilink")（くにそこたち の みこと）

#### 男女四对神的形成

**本文**中记载：接下来又出现了神。

  - 先生[埿土煑尊](https://zh.wikipedia.org/wiki/埿土煑尊 "wikilink")（うひぢにのみこと）、[沙土煑尊](https://zh.wikipedia.org/wiki/沙土煑尊 "wikilink")（すひぢにのみこと）
      - 又称 埿土根尊（うひぢねのみこと）、沙土根尊（すひぢねのみこと）
  - 次[大户之道尊](https://zh.wikipedia.org/wiki/大户之道尊 "wikilink")（おほとんちのみこと）、[大苫边尊](https://zh.wikipedia.org/wiki/大苫边尊 "wikilink")（おほとんめのみこと）
      - 又称 大户摩彦尊（おほとんひこのみこと）、大户摩姬尊（おほとんひめのみこと）
      - 又称 大富道尊（おほとんちのみこと）、大富边尊（おほとんめのみこと）
      - 大苫邊尊又称 大户之边（おほとんめ）
  - 次[面足尊](https://zh.wikipedia.org/wiki/面足尊 "wikilink")（おもたるのみこと）、[惶根尊](https://zh.wikipedia.org/wiki/惶根尊 "wikilink")（かしこねのみこと）
      - 面足尊又称 青橿城根尊（あをかしきねのみこと）
      - 惶根尊又称 吾屋惶根尊（あやかしこねのみこと）、吾忌橿城尊（あいかしきのみこと）、吾屋橿城尊（あやかしきのみこと）
  - 次[伊奘诺尊](https://zh.wikipedia.org/wiki/伊奘诺尊 "wikilink")（いざなきのみこと）、[伊奘冉尊](../Page/伊奘冉尊.md "wikilink")（いざなみのみこと）

**第一种异说**中说：伊奘诺尊、伊奘冉尊为[青橿城根尊之子](https://zh.wikipedia.org/wiki/青橿城根尊 "wikilink")。

**第二种异说**中说：

  - [国常立尊](https://zh.wikipedia.org/wiki/国常立尊 "wikilink")，生[天鏡尊](https://zh.wikipedia.org/wiki/天鏡尊 "wikilink")（あめかがみのみこと）。
  - 天鏡尊，生[天万尊](https://zh.wikipedia.org/wiki/天万尊 "wikilink")（あまよろづのみこと）。
  - 天万尊，生[沫蕩尊](https://zh.wikipedia.org/wiki/沫蕩尊 "wikilink")（あわなきのみこと）。
  - 沫蕩尊，生[伊奘诺尊](https://zh.wikipedia.org/wiki/伊奘诺尊 "wikilink")（いざなきのみこと）。

总共这八位神，在天地之间，相互参错幻化，出现了男女。

### 《古事记》

#### 別天神

天地初成的时候，在[高天原育出一位神](https://zh.wikipedia.org/wiki/高天原 "wikilink")。

  - 其名曰 [天之御中主神](https://zh.wikipedia.org/wiki/天之御中主神 "wikilink")（あまのみなかぬしのかみ）
  - 次[高御产巢日神](https://zh.wikipedia.org/wiki/高御产巢日神 "wikilink")（たかみむすひのかみ）
  - 次[神产巢日神](https://zh.wikipedia.org/wiki/神产巢日神 "wikilink")（かみむすひのかみ）

这三位神都是[独神](https://zh.wikipedia.org/wiki/独神 "wikilink")，成坐而隐其身于高天原。

后国土稚如浮脂，犹如海中之水母浮在水上，有一如萌芽之物化作一神。

  - 其名 [宇摩志阿斯诃备比古迟神](https://zh.wikipedia.org/wiki/宇摩志阿斯诃备比古迟神 "wikilink")（うましあしかびひこぢのかみ）
  - 次[天之常立神](https://zh.wikipedia.org/wiki/天之常立神 "wikilink")（あまのとこたちのかみ）

这两位神也都是独神，成坐而隐其身于高天原。

以上五位神，就是所说的**[别天神](https://zh.wikipedia.org/wiki/别天神 "wikilink")**。

#### 神世七代

后又生两神。

  - [国之常立神](https://zh.wikipedia.org/wiki/国之常立神 "wikilink")（くにのとこたちのかみ）
  - [丰云野神](https://zh.wikipedia.org/wiki/丰云野神 "wikilink")（とよくんぬのかみ）

这两位神也都是独神，成坐而隐其身于高天原。

后又生五组十神，男女对应。

  - 先生[宇比地迩神](https://zh.wikipedia.org/wiki/宇比地迩神 "wikilink")（うひぢにのかみ）、妹[须比智迩神](https://zh.wikipedia.org/wiki/须比智迩神 "wikilink")（すひぢにのかみ）
  - 次[角杙神](https://zh.wikipedia.org/wiki/角杙神 "wikilink")（つのぐひのかみ）、妹[活杙神](https://zh.wikipedia.org/wiki/活杙神 "wikilink")（いくくひのかみ）
  - 次[意富斗能地神](https://zh.wikipedia.org/wiki/意富斗能地神 "wikilink")（おほとんちのかみ）、妹[大斗乃辨神](https://zh.wikipedia.org/wiki/大斗乃辨神 "wikilink")（おほとんめのかみ）
  - 次[淤母陀流神](https://zh.wikipedia.org/wiki/淤母陀流神 "wikilink")（おもだるのかみ）、妹[阿夜诃志古泥神](https://zh.wikipedia.org/wiki/阿夜诃志古泥神 "wikilink")（あやかしこねのかみ）
  - 次[伊邪那岐神](https://zh.wikipedia.org/wiki/伊邪那岐神 "wikilink")（いざなきのかみ）、妹[伊邪那美神](https://zh.wikipedia.org/wiki/伊邪那美神 "wikilink")（いざなみのかみ）

从国之常立神以后，一直到伊邪那美神以前，共十二位神，并称为**[神世七代](https://zh.wikipedia.org/wiki/神世七代 "wikilink")**。

**微型摄影术**（）是用[微型相机拍摄](https://zh.wikipedia.org/wiki/微型相机 "wikilink")[相片的技术](../Page/相片.md "wikilink")。微型摄影术以胶卷底片的类别界定，凡底片小于35毫米、大于2毫米直径的摄影术，称为微型摄影术，底片小于2毫米的照像机，称为[微点摄影机](../Page/微点.md "wikilink")（），其摄影方法与微型摄影术完全不同，另有一套方法称为微点摄影术（\[1\]）。
微型摄影家一般指使用16毫米，9.5毫米，或17毫米，17.5毫米胶卷的相机\[2\]

微型摄影机素有“间谍摄影机”之称，微型摄影术当然是世界各国[特工收集情报的多种方法之一](https://zh.wikipedia.org/wiki/特工 "wikilink")；时至今日，微型摄影术以其不需电池或电池寿命特长，便于在条件艰难的地方、场合运用，仍然是特工爱好的收集情报的一种方法\[3\]。但微型摄影术并非特工的专利，许多摄影爱好者也喜爱微型摄影术，因为它具有特殊优点，包括极其轻便，可以经常随身携带，而在摄影或冲洗底片时，要求特别严格，富于挑战性，可以培养一丝不苟的习惯，对于35毫米摄影，中幅相机摄影或大幅相机摄影都有帮助。[德国](../Page/德国.md "wikilink")\[4\]、[美国](../Page/美国.md "wikilink")\[5\]、[加拿大和](../Page/加拿大.md "wikilink")[日本等国有专门切磋微型摄影术的摄影协会](../Page/日本.md "wikilink")，不乏业余摄影爱好者参加。此外还有更多的收藏家受集各种微型相机，包括其精巧比美珠宝的镀金微型相机\[6\]。

## 简史

19世纪初叶、英国摄影术发明家塔尔博特（）发明用纸底片拍摄出25毫米乘25毫米大小的影像，他的照相机大小也只有几英寸，他是世界上第一位成功的微型2摄影术的发明人\[7\]。
1839年[德国](../Page/德国.md "wikilink")[慕尼黑大学](../Page/慕尼黑大学.md "wikilink")[物理学教授施泰因海尔](../Page/物理学.md "wikilink")（Steinheil）已经用[达盖尔](https://zh.wikipedia.org/wiki/达盖尔 "wikilink")[银版摄影法拍摄大约](../Page/银版摄影法.md "wikilink")8×11毫米的照片了，不过达盖尔[照相机和达盖尔照相板本身都相当笨重](https://zh.wikipedia.org/wiki/照相机 "wikilink")\[8\]。
1902年瑞士工程师设袋表型微型相机TICKA，使用17.5毫米胶卷，成像16
x22毫米。1921年法国人发明手枪型微型相机。1931年[列宁格勒斯科达公司出产火柴盒形微型相机](https://zh.wikipedia.org/wiki/列宁格勒 "wikilink")。1932年[柏林卡夫坦斯基](../Page/柏林.md "wikilink")（Kaftanski）发明16毫米微型相机
Minifex。1937年[扎普发明](https://zh.wikipedia.org/wiki/扎普 "wikilink")[美乐时微型相机](https://zh.wikipedia.org/wiki/美乐时 "wikilink")\[9\]。[第一次世界大战时开始出现的隐藏在](../Page/第一次世界大战.md "wikilink")[火柴盒](../Page/火柴.md "wikilink")、[皮鞋跟或手杖中的小型摄影机](../Page/皮鞋.md "wikilink")，主要用于[军事和](https://zh.wikipedia.org/wiki/军事 "wikilink")[谍报用途](https://zh.wikipedia.org/wiki/谍报 "wikilink")。时至今日，精密微型相机仍是世界各国[特工的工具之一](https://zh.wikipedia.org/wiki/特工 "wikilink")。\[10\]

## 微型相机优点

  - 大[景深](../Page/景深.md "wikilink")：16毫米微型相机的镜头多数为25毫米，[密诺斯镜头的焦距为](https://zh.wikipedia.org/wiki/密诺斯 "wikilink")15毫米，因为焦距短，微型相机相对于35毫米相机而言，有较大的景深，例如密诺斯镜头在
    f/3.5的[景深相当](../Page/景深.md "wikilink")35毫米照相机50毫米镜头在 f/11的景深。
  - 取景方便：微型相机的取景器大而清晰，没有附加框，比[莱卡](https://zh.wikipedia.org/wiki/莱卡 "wikilink")35毫米相机还方便。
  - 快捷：用[密诺斯或爱迪飒](https://zh.wikipedia.org/wiki/密诺斯 "wikilink")16，
    可以每秒拍摄一张照片；密诺斯微型相机用双卡胶卷，无需倒片，换新的胶卷只需2秒。
  - 相机小，照片大。密诺斯相机底片如小手指甲大小，[爱迪飒](../Page/爱迪飒.md "wikilink")16微型相机的底片，小如姆指甲，可以放大到20×30厘米，其质量与高档35毫米照相机的20×30厘米放大照片不相上下。

## 微型摄影技术

### 取景

取景宜简洁，避免多余物体，以便尽量利用有限的底片面积，不宜事后剪裁。一些使用35毫米照相机或中幅照相机的摄影者，专门用微型相机训练取景简约的习惯。

### 测距

微型相机不同于35毫米单反照相机或数码照相机，既没有测距器也没有自动测距器，全靠摄影者估计距离，并充分利用照相机上的[景深表及](../Page/景深.md "wikilink")“红点超焦距摄影术”；如图四所示，将[密诺斯TLX的距离盘上用小红点标志的](https://zh.wikipedia.org/wiki/密诺斯 "wikilink")[超焦距对准指标](../Page/超焦距.md "wikilink")，由2米至无穷远的景物都清晰。

### 曝光

[Edixa16meter.jpg](https://zh.wikipedia.org/wiki/File:Edixa16meter.jpg "fig:Edixa16meter.jpg")
多数微型相机如Minox B， Minox C, Minox TLX, Edixa 16M 等都带曝光表。 Minox B，Edixa 16
带快门联动双针重叠式测光表，如图所示为Edixa
16测光表白针是测光表针，转动右边光圈转盘使红针与白针重叠，即是正确的曝光时间。Minox
TLX, CLX, EC, ECX 等型号微型相机带全自动曝光表，快门开启的时间由曝光表自动决定。一些不带曝光表的微型照相机，如 Minox
IIIs, Minolta 16 等，可以
用[阳光16口诀推算](https://zh.wikipedia.org/wiki/阳光16 "wikilink")。

[Minox_binocular_attachment.JPG](https://zh.wikipedia.org/wiki/File:Minox_binocular_attachment.JPG "fig:Minox_binocular_attachment.JPG")
微型相机可以安装在[双筒望远镜的一个](../Page/双筒望远镜.md "wikilink")[目镜上](../Page/目镜.md "wikilink")，将相机的镜头对焦在无穷远，摄影者通过望远镜的另一目镜对准拍摄对象，即可拍摄远距离的对象。密诺斯有特制的望远镜夹，前端的夹子夹住双筒望远镜的一个目镜。夹口的大小可以通过螺丝钮控制，可以安装在各种双筒、单筒望远镜上。密诺斯的视野相当于35毫米相机50毫米镜头，安装在用8X望远镜上相当于400毫米镜头\[21\]。

### 显微镜摄影

[Minox_TLX_w_microscope.JPG](https://zh.wikipedia.org/wiki/File:Minox_TLX_w_microscope.JPG "fig:Minox_TLX_w_microscope.JPG")的目镜上\]\]
将微型相机安装在[显微镜的目镜上](https://zh.wikipedia.org/wiki/显微镜 "wikilink")，可通过显微镜拍摄显微物体。先将显微镜光源调校，上下移动显微镜镜筒，使目镜中成像清晰，另用暴光表通过显微镜目镜测量亮度，选择微型相机的快门，再将微型相机安装在显微镜目镜上即可进行拍摄。

### 微型底片的冲洗

[Jobo16reel.jpg](https://zh.wikipedia.org/wiki/File:Jobo16reel.jpg "fig:Jobo16reel.jpg")
[Nikor16.jpg](https://zh.wikipedia.org/wiki/File:Nikor16.jpg "fig:Nikor16.jpg")
[Minoxtank.jpg](https://zh.wikipedia.org/wiki/File:Minoxtank.jpg "fig:Minoxtank.jpg")微型相机日光冲洗罐\]\]

  - **显影罐**:冲洗微型照相机胶卷，可用特制的[显影罐](https://zh.wikipedia.org/wiki/显影罐 "wikilink")，例如冲洗[密诺斯胶卷用](https://zh.wikipedia.org/wiki/密诺斯 "wikilink")**密诺斯日光冲洗罐**，冲洗16毫米胶卷用**万能达日间显影罐**；用这些显影罐不需要[暗室](https://zh.wikipedia.org/wiki/暗室 "wikilink")\[22\]。
  - **螺旋卷盘**:也可以在暗室内将已拍摄的胶卷从胶卷合取出，小心地装入密诺斯8×11毫米螺旋卷盘或16毫米螺旋卷盘，放入显影罐，然后按照程序冲洗。[暗室和用具都必须尽量无灰尘](https://zh.wikipedia.org/wiki/暗室 "wikilink")，因为同样的一粒灰尘在35毫米底片上也许不明显，但是微型底片放大倍数是35毫米底片的2至3倍，任何灰尘都会太显著。

彩色胶卷用C41或E6显影剂冲洗，黑白胶卷宜用[微粒显影剂冲洗](https://zh.wikipedia.org/wiki/微粒显影剂 "wikilink")。暗室必须干净无灰尘，温度必须准确控制，配制显影液、定影液必须用蒸馏水，保障绝对无杂质。

### 放大

所有微型照相机拍摄的底片，都必须放大。放大到10厘米×15厘米可以存入相册，放大到20x25厘米以上，可以镶入镜框。

放大机可用普通35毫米放大机改装，但是必须换下用来放大35毫米底片的焦距为50毫米至60毫米的镜头，换成焦距为25毫的米放大镜头。有些放大机厂出售14×21毫米，12×17毫米，8×11毫米的金属制底片框；也可用黑纸剪裁出所需的底片框。

下表是各不同相幅底片，用25毫米放大机镜头、15毫米镜头，放大到20×25厘米时的放大率、放大机相距。

|   | 微型照相机      | 底片边长    | 放大倍数 | 25毫米放大镜头像距 | 密诺斯放大机像距 |
| - | ---------- | ------- | ---- | ---------- | -------- |
| 1 | *特熙纳35型*   | *14×21* | *15* | *40厘米*     |          |
| 2 | *爱迪飒16*    | *12×17* | *17* | *45厘米*     |          |
| 3 | ''密诺斯 B '' | *8×11*  | *25* | *65厘米*     | *39厘米*   |

由此可见，焦距=25毫米的放大机镜头，适用于**特熙纳35**和**爱迪飒16**拍摄的底片，因为放大机镜头只需离放大机底板40-45厘米，而对于**密诺斯微型照相机**的8×11毫米底片，25毫米放大机镜头必须离放大机底板65厘米，勉强可用；最好用特制配15毫米镜头的密诺斯放大机，放大机镜头只需离放大机底板39厘米。

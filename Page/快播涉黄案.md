**快播涉黄案**，司法称**深圳市快播科技有限公司傳播淫穢物品牟利案**，是指2013年年底，[北京市](../Page/北京市.md "wikilink")公安和版权部门查处[快播](../Page/快播.md "wikilink")公司深圳总部，并于2014年9月，将快播公司5名高管移送[北京市人民检察院以传播淫秽物品牟利罪起诉的案件事件](https://zh.wikipedia.org/wiki/北京市人民检察院列表 "wikilink")。2014年9月，快播公司5名高管被移送北京市人民检察院起诉\[1\]，2016年1月7日案件开审，罪名是传播淫秽物品牟利罪\[2\]。[王欣认为](../Page/王欣_\(深圳\).md "wikilink")，快播只提供技术不提供内容，不构成犯罪\[3\]。

2016年9月13日，[北京市海淀区人民法院](../Page/北京市海淀区人民法院.md "wikilink")一审宣判，快播公司犯传播淫秽物品牟利罪，判处罚金人民币1000万元；而4名被告人则获3年至3年6个月不等的有期徒刑\[4\]。被告人吴铭对判决不服，随后提起上诉。2016年12月16日，[北京市第一中级人民法院](../Page/北京市第一中级人民法院.md "wikilink")二审宣判，维持原判。

## 案件缘由

[快播](../Page/快播.md "wikilink")是一款视频播放器，拥有视频点播、[BT种子文件边播边放](https://zh.wikipedia.org/wiki/BT种子 "wikilink")\[5\]以及[P2P视频分享功能](https://zh.wikipedia.org/wiki/P2p "wikilink")\[6\]，使用人数最高曾达到4亿人\[7\]，因[色情](../Page/色情.md "wikilink")内容泛滥，被网友戏称为“[宅男](../Page/宅男.md "wikilink")必备”\[8\]，快播在“小二广场”网站上储存了300部淫秽视频，可通过会员通道观看\[9\]。2013年底，[北京市](../Page/北京市.md "wikilink")公安和版权部门在执法检查中查扣了快播公司4台[服务器](../Page/服务器.md "wikilink")，仅在1部服务器上就发现了淫秽色情视频达3000多部\[10\]，[北京市公安局](../Page/北京市公安局.md "wikilink")在四台服务器中提取了25175个视频文件进行鉴定，最终认定其中属于淫秽视频的文件为21251个\[11\]。2014年1月起，快播将视频储存方式改为碎片阵列式，通过这种方式逃避监管\[12\]。

2013年11月，中国国内数十家正版视频网站和版权方发起“中国网络视频反盗版联合行动”，对[百度](../Page/百度.md "wikilink")影音及快播的盗版采取了技术反制及法律诉讼。2013年12月27日，[国家版权局](../Page/国家版权局.md "wikilink")认定，百度和快播公司构成盗版事实，分别对二者处以25万元人民币的罚款，并责令其停止侵权行为\[13\]。

2014年4月16日，受中国[净网行动行动影响](../Page/扫黄打非·净网2014.md "wikilink")，快播科技发布公告称，快播将关闭qvod服务器，停止基于快播技术的视频点播和[下载](../Page/下载.md "wikilink")，并表示要清理低俗内容及涉盗版内容；同时启动商业模式转型，转型原创内容，重视版权内容及[微电影发展](https://zh.wikipedia.org/wiki/微电影 "wikilink")\[14\]\[15\]\[16\]。4月22日，因接到[乐视举报盗版侵权](https://zh.wikipedia.org/wiki/乐视 "wikilink")\[17\]，警方进入[深圳市](../Page/深圳市.md "wikilink")南山高新园的快播科技有限公司调查\[18\]。5月15日，公安部门立案侦查，刑拘多人\[19\]。同日，[全国“扫黄打非”工作小组办公室](../Page/全国“扫黄打非”工作小组办公室.md "wikilink")通报称，经调查证实快播公司在提供互联网信息服务中存在传播淫秽色情内容信息行为，且情节严重\[20\]。5月17日快播发表致歉声明\[21\]。5月20日，深圳市市场监督管理局（知识产权局）到深圳市快播科技有限公司送达了拟行政处罚听证通知书，拟对该公司处以2.6亿元罚款，通知书留置送达\[22\]。5月21日，快播仅因侵犯版权被罚款2.6亿元\[23\]。6月26日，广东省通信管理局吊销该公司的增值电信业务经营许可证\[24\]\[25\]。

## 开庭

2014年8月8日，逃往境外110天的快播公司法人兼总经理[王欣在](../Page/王欣_\(深圳\).md "wikilink")[韩国被抓捕归案](https://zh.wikipedia.org/wiki/韩国 "wikilink")\[26\]。后经国际司法合作渠道移交中国警方\[27\]。王欣称自己心存侥幸心理、快播可能影响一代人\[28\]。2014年9月，案件被北京市公安局海淀分局移送到北京市海淀区人民检察院审查起诉\[29\]，2015年2月6日，北京市海淀区人民检察院对深圳快播科技有限公司王欣、吴铭等人涉嫌传播淫秽物品牟利罪一案向北京市海淀区人民法院提起[公诉](https://zh.wikipedia.org/wiki/公诉 "wikilink")\[30\]。2016年1月7日，案件在北京市海淀区人民法院开庭，王欣和其他三名高管否认指控\[31\]。

### 辩论

王欣认为公司和个人不构成犯罪：快播只做技术，不提供视频，也不具备[搜索功能](../Page/搜索引擎.md "wikilink")，被查服务器是缓存服务器，缓存视频是行业的普遍做法，快播无法分辨视频是淫秽视频还是普通视频，快播还成立了“110”系统，用关键词禁止淫秽视频播放，并接受举报屏蔽四千多个色情网站\[32\]\[33\]\[34\]。

控方认为快播公司及王欣是色情内容的受益者，“情节特别严重”，应判王欣十年以上，而王欣认为快播反而是色情内容的受害者，辩护人认为如果判只提供技术的快播有罪，那么将产生[寒蝉效应使互联网公司将人人自危](https://zh.wikipedia.org/wiki/寒蝉效应 "wikilink")\[35\]。对应快播监管不利的指责，辩护人回应称其屏蔽了四千多个黄色网站，而2014年全国扫黄打非8个月才查处了422家\[36\]。面对转型问题，快播用[中国移动有诈骗短信的问题回应](https://zh.wikipedia.org/wiki/中国移动 "wikilink")\[37\]，称自己不做技术也会有其他公司做\[38\]，并列举了[淘宝网](../Page/淘宝网.md "wikilink")、[腾讯](../Page/腾讯.md "wikilink")的[QQ和](../Page/腾讯QQ.md "wikilink")[微信](../Page/微信.md "wikilink")、[百度存在的问题](https://zh.wikipedia.org/wiki/百度云 "wikilink")\[39\]。公诉人指出以[AV女优为关键字](https://zh.wikipedia.org/wiki/AV女优 "wikilink")，“快播+*关键字*”得到的搜索结果远超过与“[暴风影音](../Page/暴风影音.md "wikilink")”、“[迅雷看看](https://zh.wikipedia.org/wiki/迅雷看看 "wikilink")”等一起搜索时的结果，但王欣表示这毫无意义，并暗示用“[QQ](https://zh.wikipedia.org/wiki/QQ "wikilink")+*关键字*”会有更多条搜索结果\[40\]。

### 一审判决

2016年1月8日，法官宣布休庭择日宣判\[41\]。2015年11月1日正式施行的[《刑法修正案（九）》明确了网络服务提供者的义务](../Page/中华人民共和国刑法.md "wikilink")，并设置了相应处罚\[42\]。

2016年9月9日，快播传播淫秽物品牟利案在北京市海淀区人民法院再次开庭，快播公司、王欣、张克东、牛文举均承认有罪\[43\]。2016年9月13日，北京市海淀区人民法院一审宣判：快播公司犯传播淫秽物品牟利罪，判处罚金人民币1000万元；4名被告人分别被判处3年至3年6个月不等的有期徒刑。具体判刑情况如下\[44\]：

| 被告人姓名 | 判刑情况                |
| ----- | ------------------- |
| 王欣    | 有期徒刑3年6个月，并处罚金100万元 |
| 张克东   | 有期徒刑3年3个月，并处罚金50万元  |
| 吴铭    | 有期徒刑3年3个月，并处罚金30万元  |
| 牛文举   | 有期徒刑3年，并处罚金20万元     |

被告人吴铭对判决结果不服，随后提出上诉。

### 二审判决

2016年12月16日，[北京市第一中级人民法院](../Page/北京市第一中级人民法院.md "wikilink")二审公开宣判。法院裁定驳回吴铭的上诉，维持一审判决\[45\]。

## 评论和影响

公安部治安管理局局长[刘绍武表示](https://zh.wikipedia.org/wiki/刘绍武 "wikilink")，“无论采取什么新技术，采取什么隐蔽措施，我们都会及时破解。”\[46\]2016年1月10日，[中国国家互联网信息办公室发言人](https://zh.wikipedia.org/wiki/中国国家互联网信息办公室 "wikilink")[姜军称](https://zh.wikipedia.org/wiki/姜军 "wikilink")：“所有利用网络技术开展服务的网站，都应对其传播的内容承担法律责任。”\[47\]\[48\]这次庭审暴露出部分法律人员[网络知识不足](../Page/互联网.md "wikilink")\[49\]，以及[网络法律不完善](https://zh.wikipedia.org/wiki/网络法律 "wikilink")\[50\]。同时，因为鉴黄人员的不规范操作使得作为证据的[硬盘](../Page/硬盘.md "wikilink")存在篡改的可能\[51\]。这起案件将推动中国网络法律的完善。

这起案件引起社会广泛关注\[52\]\[53\]，也有娱乐性，有超过100万人在同时观看这次庭审的直播\[54\]。快播方精彩的辩论在网络上赢得了大量的支持\[55\]\[56\]，[人民日报](../Page/人民日报.md "wikilink")称“快播的辩词再精彩也不配赢得掌声”\[57\]。庭审上爆出举报快播的是乐视后，快播粉丝炮轰乐视，[乐事](../Page/乐事.md "wikilink")[薯片也因与](https://zh.wikipedia.org/wiki/薯片 "wikilink")“乐视”同音而被网友调侃：“你居然是这种薯片”\[58\]、“居然还有[黄瓜](../Page/黄瓜.md "wikilink")味的，一看就不是什么正经薯片”\[59\]，有关[贴吧在网民针对乐视的](../Page/百度贴吧.md "wikilink")[爆吧](../Page/爆吧.md "wikilink")行动中也受波及\[60\]。

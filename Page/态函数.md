在[熱力學中](../Page/熱力學.md "wikilink")，
**热力学函数**（），或称**热力学参数**、**狀態函數**，是描述[热力学系统的宏观](../Page/热力学系统.md "wikilink")[物理量](../Page/物理量.md "wikilink")。处于平衡态的热力学系统，各宏观物理量具有确定的值，并且这些物理量仅由系统所处的状态所决定，与达到平衡态的过程无关。决定物质状态的物理量被称为状态函数。其中包含了“[热力学势](../Page/热力学势.md "wikilink")”，[热力学势特指下面提到的四个具有](../Page/热力学势.md "wikilink")[能量](../Page/能量.md "wikilink")[量纲的热力学函数](../Page/量纲.md "wikilink")。

熱力學系统的狀態函数一般存在一定的相互依存关系。如[理想氣體的](../Page/理想氣體.md "wikilink")[狀態方程式中](../Page/狀態方程式.md "wikilink")，可以任意选取其中的兩個狀態函數為[独立变量](../Page/独立变量.md "wikilink")，而把其他的統計量看作它们的[函数](../Page/函数.md "wikilink")。热力学函数之间的依存关系具有普适性。

## 简单系统的的热力学函数

简单热力学系统（如量子、古典氣體系統）一般具有以下热力学函数，可以任意选取其中两个作为独立变量：
[量綱](../Page/量綱.md "wikilink")（單位）不是[能量的热力学函数](../Page/能量.md "wikilink")

| 物理量                            | 符号 | 单位            |
| ------------------------------ | -- | ------------- |
| [体积](../Page/体积.md "wikilink") | V  | m<sup>3</sup> |
| [压强](../Page/压强.md "wikilink") | P  | Pa和atm        |
| [温度](../Page/温度.md "wikilink") | T  | K和℃           |
| [熵](../Page/熵.md "wikilink")   | S  | J/(mol·K)     |

[量綱](../Page/量綱.md "wikilink")（單位）是能量的[熱力學勢](../Page/熱力學勢.md "wikilink")

| 物理量                                      | 符号 | 单位 |
| ---------------------------------------- | -- | -- |
| [内能](../Page/内能.md "wikilink")           | U  | J  |
| [焓](../Page/焓.md "wikilink")             | H  | J  |
| [吉布斯能](../Page/吉布斯能.md "wikilink")       | G  | J  |
| [亥姆霍兹自由能](../Page/亥姆霍兹自由能.md "wikilink") | F  | J  |

## 热力学势

上面给出的热力学函数中，后四个具有能量的量纲，单位都为[焦耳](../Page/焦耳.md "wikilink")，这四个量通常称为「热力学势」。

|                                          |               |           |
| ---------------------------------------- | ------------- | --------- |
| [内能](../Page/内能.md "wikilink")           | \(U\)         | 有时也用*E*表示 |
| [亥姆霍兹自由能](../Page/亥姆霍兹自由能.md "wikilink") | \(A=U-TS\)    | 也常用*F*表示  |
| [焓](../Page/焓.md "wikilink")             | \(H=U+PV\)    |           |
| [吉布斯能](../Page/吉布斯能.md "wikilink")       | \(G=U+PV-TS\) |           |

其中，

  -
    *T* =[温度](../Page/温度.md "wikilink")，
    *S* =[熵](../Page/熵.md "wikilink")，
    *P* =[压强](../Page/压强.md "wikilink")，
    *V* =[体积](../Page/体积.md "wikilink")

具有 [廣義力](../Page/廣義力.md "wikilink") 和 [廣義位移](../Page/廣義力.md "wikilink")
\(X_i\) \(x_i\) 熱力學系統，
內能\(U\)的微分式可從[熱力學第一定律得知](../Page/熱力學第一定律.md "wikilink")：

\[dU = T \, dS - \sum_i X_i \, dx_{i} + \sum_j \mu_j \, dN_j\,\]

公式內的U、S和V是[熱力學的狀態函數](../Page/熱力學函數.md "wikilink")，也可用於非平衡、不可逆的過程。

其餘三個熱力學勢可經由 [勒壤得轉換](../Page/勒壤得轉換.md "wikilink") (Legendre
transform)轉換自變數而得到。

|                   |               |         |                      |         |                    |                                     |
| ----------------- | ------------- | ------- | -------------------- | ------- | ------------------ | ----------------------------------- |
| \(\mathrm{d}U\,\) | \(\!\!=\!\!\) |         | \(T\mathrm{d}S\,\)   | \(-\,\) | \(p\mathrm{d}V\,\) | \(+\sum_i \mu_i \,\mathrm{d}N_i\,\) |
| \(\mathrm{d}F\,\) | \(\!\!=\!\!\) | \(-\,\) | \(S\,\mathrm{d}T\,\) | \(-\,\) | \(p\mathrm{d}V\,\) | \(+\sum_i \mu_i \,\mathrm{d}N_i\,\) |
| \(\mathrm{d}H\,\) | \(\!\!=\!\!\) |         | \(T\,\mathrm{d}S\,\) | \(+\,\) | \(V\mathrm{d}p\,\) | \(+\sum_i \mu_i \,\mathrm{d}N_i\,\) |
| \(\mathrm{d}G\,\) | \(\!\!=\!\!\) | \(-\,\) | \(S\,\mathrm{d}T\,\) | \(+\,\) | \(V\mathrm{d}p\,\) | \(+\sum_i \mu_i \,\mathrm{d}N_i\,\) |

通过对以上微分表达式求[偏导](../Page/偏导.md "wikilink")，可以得到*T*，*S*，*P*，*V*四个变量的[偏导数间的](../Page/偏导数.md "wikilink")“[麦氏关系](../Page/麦氏关系.md "wikilink")”

**性别符号**是用来表示[性别的](../Page/性别.md "wikilink")[符号](../Page/符号.md "wikilink")。

## 最初的性别符号

最初的两个性别符号来自于[天文符号](../Page/天文符号.md "wikilink")。1751年，[卡尔·林奈最早使用它们来表示植物的性别](../Page/卡尔·林奈.md "wikilink")。\[1\]

|                                                                  |        |                                                                                                           |
| ---------------------------------------------------------------- | ------ | --------------------------------------------------------------------------------------------------------- |
| [♂](https://zh.wikipedia.org/wiki/File:Mars_symbol.svg "fig:♂")  | U+2642 | 来自[火星符号](../Page/火星符号.md "wikilink")，表示[男性或其他](../Page/男性.md "wikilink")[雄性生物](../Page/雄性.md "wikilink")。 |
| [♀](https://zh.wikipedia.org/wiki/File:Venus_symbol.svg "fig:♀") | U+2640 | 来自[金星符号](../Page/金星符号.md "wikilink")，表示[女性或其他](../Page/女性.md "wikilink")[雌性生物](../Page/雌性.md "wikilink")。 |

## 其他性别符号

除了上述最常见的性别符号外，还有其他一些性别符号，包括：

|                                                                                 |        |                                                                                                                |
| ------------------------------------------------------------------------------- | ------ | -------------------------------------------------------------------------------------------------------------- |
| [☿](https://zh.wikipedia.org/wiki/File:Mercury_symbol.svg "fig:☿")              | U+263F | 来自[水星符号](../Page/水星.md "wikilink")，表示[双性人或其他](../Page/双性人.md "wikilink")[雌雄间性的生物](../Page/雌雄间性.md "wikilink")。 |
| [⚥](https://zh.wikipedia.org/wiki/File:Male_and_female_sign.svg "fig:⚥")        | U+26A5 | 来自雄性符号与雌性符号，表示[双性人或](../Page/双性人.md "wikilink")[跨性別](../Page/跨性別.md "wikilink")。                               |
| [⚧](https://zh.wikipedia.org/wiki/File:Gendersign.svg "fig:⚧")                  | U+26A7 | 另一种跨性别符号，由雄性符号、雌性符号与跨性别符号组合而成。                                                                                 |
| [⚢](https://zh.wikipedia.org/wiki/File:Female_homosexuality_symbol.svg "fig:⚢") | U+26A2 | 由两个雌性符号组成，有时用来表示[女同性恋](../Page/女同性恋.md "wikilink")。                                                            |
| [⚣](https://zh.wikipedia.org/wiki/File:Male_homosexuality_symbol.svg "fig:⚣")   | U+26A3 | 由两个雄性符号组成，有时用来表示[男同性恋](../Page/男同性恋.md "wikilink")。                                                            |
| [⚤](https://zh.wikipedia.org/wiki/File:Heterosexuality_symbol.svg "fig:⚤")      | U+26A4 | 由雄性符号与雌性符号相互连结而成，偶尔用来表示[异性恋](../Page/异性恋.md "wikilink")。                                                       |
| [⚦](https://zh.wikipedia.org/wiki/File:Transgender_symbol_1.svg "fig:⚦")        | U+26A6 | 雄性符号加上一短横，表示[跨性別](../Page/跨性別.md "wikilink")。                                                                  |
| [⚪](https://zh.wikipedia.org/wiki/File:Asexual_symbol.svg "fig:⚪")              | U+26AA | 表示[无性恋](../Page/无性恋.md "wikilink")。                                                                            |
| [⚢](https://zh.wikipedia.org/wiki/File:Female_bisexuality_symbol.svg "fig:⚢")   |        | 兩個雌性符號加一個雄性符號，表示女性[雙性戀](../Page/雙性戀.md "wikilink")。                                                            |
| [⚢](https://zh.wikipedia.org/wiki/File:Male_bisexuality_symbol.svg "fig:⚢")     |        | 兩個雄性符號加一個雌性符號，表示男性[雙性戀](../Page/雙性戀.md "wikilink")。                                                            |

**愛國婦人會**是由於1901年在[日本](../Page/日本.md "wikilink")[東京創立的婦人團體](https://zh.wikipedia.org/wiki/東京 "wikilink")，成員多為[日本皇族](https://zh.wikipedia.org/wiki/日本皇族 "wikilink")、[華族等上流階層的女性](../Page/華族_\(日本\).md "wikilink")，主要是從事社會事業，後來隨著戰爭爆發改擔負後方婦人奉公的任務。其主要工作有達成軍事輔助事業和分擔生活輔助、慰問軍眷或遺族、籌備軍事後援與救濟救護的資金。

## 沿革

愛國婦人會的創立者是[九州](https://zh.wikipedia.org/wiki/九州 "wikilink")[佐賀縣人奧村五百子](https://zh.wikipedia.org/wiki/佐賀縣 "wikilink")，1900年[八國聯軍](../Page/八國聯軍.md "wikilink")時，她在[中國華北見到](https://zh.wikipedia.org/wiki/中國華北 "wikilink")[日軍傷兵的慘況](https://zh.wikipedia.org/wiki/日軍 "wikilink")，並在返回日本後得知陣亡日軍將士家屬缺乏照顧，遂積極籌組婦人團體義務支援照顧工作。1901年該會成立時，請到皇族[閑院宮親王夫人擔任總裁](https://zh.wikipedia.org/wiki/閑院宮 "wikilink")，後來由日本社會支持得以迅速增加支會，而臺灣方面則到1904年2月與6月在臺中、臺南、臺北設立支部，後來在1905年7月17日在臺北成立愛國婦人會臺灣支部（1933年升格為臺灣本部）。

愛國婦人會不管在日本還是臺灣，成員均由上流社會階層人士之夫人組成，主要工作有照顧日本征戰受傷軍人、遺族，以及募款充實軍需。戰爭告一段落後，愛國婦人會亦從事社會工作，如與[日本赤十字社](../Page/日本赤十字社.md "wikilink")合辦巡迴診療等。而該會的經費來源除了創會當時[日本天皇](https://zh.wikipedia.org/wiki/日本天皇 "wikilink")、[日本皇后等人捐款](https://zh.wikipedia.org/wiki/日本皇后 "wikilink")，還有一般入會費、經營事業收入與各界捐款。

1942年，愛國婦人會與[國防婦人會合併為](https://zh.wikipedia.org/wiki/國防婦人會 "wikilink")[大日本婦人會](https://zh.wikipedia.org/wiki/大日本婦人會 "wikilink")；[第二次世界大戰](https://zh.wikipedia.org/wiki/第二次世界大戰 "wikilink")[日本戰敗後](https://zh.wikipedia.org/wiki/日本戰敗 "wikilink")，大日本婦人會遭到解散。

## 勳章與感謝狀

對於捐款達到一定金額者以及有功的幹部，愛國婦人會會依其規則頒予勳章或感謝狀，最初有功勳章由該會總裁親自頒授，較為偏遠的地方才由地方支部長代為頒授。
[Patriotic_Women's_Association_merit_badge_3rd_Class-028.JPG](https://zh.wikipedia.org/wiki/File:Patriotic_Women's_Association_merit_badge_3rd_Class-028.JPG "fig:Patriotic_Women's_Association_merit_badge_3rd_Class-028.JPG")
愛國婦人會的勳章最初分為一等七寶製有功章、二等七寶金色製有功章、三等七寶銀色製有功章三種，1907年於〈有功章謝狀贈與規則〉中增列特別徽章，1917年再增列附加章、特別勳章，1931年再次修正規則且沿用至該會解散。1931年時該會有功章分為最高特別有功章（原特別勳章）與一等至三等有功章四種，而一等到三等的有功章各增有附加章一項。四種有功章的條件如下：

|      |                     |
| ---- | ------------------- |
| 特別勳章 | 捐款千圓以上、介紹會員千人以上     |
| 一等勳章 | 捐款三百圓以上、介紹會員五百人以上   |
| 二等勳章 | 捐款一百五十圓以上、介紹會員三百人以上 |
| 三等勳章 | 捐款五十圓以上、介紹會員一百五十人以上 |
|      |                     |

另外，該會有兩種感謝狀，一種是發給捐款七圓以上者，另外一種是七圓以下者。

《**紅字**》（，），為[韓國](https://zh.wikipedia.org/wiki/韓國 "wikilink")[MBC自](https://zh.wikipedia.org/wiki/文化廣播_\(韓國\) "wikilink")2010年8月9日起播出的[晨間連續劇](../Page/MBC晨間連續劇.md "wikilink")。

## 劇情

因为交错的爱而蒙受痛苦的四个男女！ 一个女孩得到的是负罪感和贫穷，另一个女孩得到了成功与报复。残酷的命运让他们再次相逢， 在四个男女心中分别刻下了各自的红字。 \[1\]

## 演員陣容

### 主要角色

|                                                         |         |                                                                                                                             |
| ------------------------------------------------------- | ------- | --------------------------------------------------------------------------------------------------------------------------- |
| 演員                                                      | 角色      | 介紹                                                                                                                          |
| **[李丞涓](../Page/李丞涓.md "wikilink")**                    | **韓京瑞** | 電視劇編劇。并非出于本意地夺去朋友的恋人，由此遭遇意想不到的不幸。对大学师兄在龙单相思，但同学慧蘭先得到了在容。日后两人偶然重逢后命中注定般地相爱，却遇到丈夫被捕的不幸。就在她艰辛度日之时，遇到了电视剧制片人東州，并得到了作为编剧东山再起的机会。 |
| **[金妍珠](https://zh.wikipedia.org/wiki/金妍珠 "wikilink")** | **車慧蘭** | 著名女演員。作为演员即将获得成功时遭到恋人在容的遗弃。而哥哥也遭遇可怕的事故，让她经受磨难。但她成功地东山再起，如今成为当红演员。她想报复给自己带来创伤的景瑞，于是主动要求出演景瑞创作的电视剧。                           |
| **[金英浩](../Page/金英浩.md "wikilink")**                    | **張在容** | 舞台劇編劇。抛弃大学师妹慧蘭，与京瑞结婚。 其间导致慧蘭的哥哥重伤，被判5年徒刑。出狱后，不顾慧蘭为了复仇而执著地纠缠并对他百般诱惑，始终没有对她动心。                                                |
| **[趙軟祐](../Page/趙軟祐.md "wikilink")**                    | **李東州** | 電視劇導演。处女作电视单剧创下了收视率最高的纪录，作为制片人一帆风顺地获得成功。很有时尚感觉，口才出众，深受女孩青睐。负责执导京瑞创作的电视剧，在京瑞与慧蘭之间摇摆不定。                                       |

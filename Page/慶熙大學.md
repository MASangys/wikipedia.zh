**慶熙大學**（[韓文](https://zh.wikipedia.org/wiki/韓文 "wikilink"):），創立於1949年5月12日，是韓國知名的私立高等学府，現有兩個校區一個大學院，分别位於[首爾特別市](https://zh.wikipedia.org/wiki/首爾特別市 "wikilink")[東大門區](../Page/東大門區.md "wikilink")、[京畿道](../Page/京畿道.md "wikilink")[龍仁市](../Page/龍仁市.md "wikilink")、[京畿道](../Page/京畿道.md "wikilink")[南楊州市](../Page/南楊州市.md "wikilink")，是一所綜合型私立大學。慶熙大學有二十多個系科，并設有慶熙網路大學，具備從幼稚園到研究生院的綜合教育體系，是[韓國三大私立名校之一](https://zh.wikipedia.org/wiki/韓國 "wikilink")。 慶熙大學擁有首爾校區，國際校區（原水原校區）和光陵校區的16所研究生院，26所大學，99個院系。慶熙大學同時具備[醫學](https://zh.wikipedia.org/wiki/醫學 "wikilink")、[牙科](https://zh.wikipedia.org/wiki/牙科 "wikilink")、[韓醫](https://zh.wikipedia.org/wiki/韓醫 "wikilink")、[藥學](https://zh.wikipedia.org/wiki/藥學 "wikilink")、[護理學等](https://zh.wikipedia.org/wiki/護理學 "wikilink")5個醫學領域的綜合醫療體系大學校，其中在經營學、經濟學、醫學、新聞信息學方面的尤为突出。跆拳道作为传统专业，已达到世界一流水平。慶熙大學與世界56個國家298所大學具姊妹交流關係，互相承認學分並行語言交換和國際交流等活動\[1\]。慶熙大學在2006年度在大學綜合評價中被授予最佳大學殊榮，1996年至目前，在學科綜合評價中獲韓國全國首位，2000年在大學國際化系門中評為全國第一名。 該校2016QS世界大学排名中第264位，亚洲大学排名第38位；2017年世界大学排名第266位，亚洲大学排名第37位；2018[QS世界大學排名中排名第](https://zh.wikipedia.org/wiki/QS世界大学排名 "wikilink")256位，亞洲大學排名第33位。\[2\]2019年QS世界大學排名中排名第264位，慶熙大學在2020年QS亞洲大學排名中位列第33位，QS世界大學排名中位列第247位。 韩国第19届总统文在寅于1972年入读庆熙大学法学院。

## 歷史

[Kyung_Hee_Univ._Grand_Auditorium.JPG](https://zh.wikipedia.org/wiki/File:Kyung_Hee_Univ._Grand_Auditorium.JPG "fig:Kyung_Hee_Univ._Grand_Auditorium.JPG") [Kyung_Hee_Univ._Central_Library_and_Museum.JPG](https://zh.wikipedia.org/wiki/File:Kyung_Hee_Univ._Central_Library_and_Museum.JPG "fig:Kyung_Hee_Univ._Central_Library_and_Museum.JPG")

慶熙大學創立於1949年5月12日，1952年12月9日四年制大學正式設立。1954年3月31日大學院設立。1955年2月28日University設立認可。1955年3月1日邀請（조영식）博士就任校长。1960年3月1日由新興大學變更為慶熙大學。1963年1月5日경희초급대학（kyonhi初級大學）設立認可。1965年4月27日與東洋醫科大學合併。1979年設立水原校區，1999年完工和平殿堂，2007年從水原分校改为國際分校。 庆熙大学的前身是1911年创立的新兴武馆学校。

此外，2000年庆熙网络大学依托于韩国庆熙大学而成立，并基于大韩民国精神‘弘益人间’的建国理念和创造文化世界的办学精神，树立了素质教育、科学教育、民主教育、人格教育四个教育理念。庆熙网络大学也是韩国第一所网络大学。http://www.khcu.ac.kr/ch/index.html

## 学校特点

1.位于韩国的首都，韩国排名前十的著名高等学府。

2.在韩国的大学校当中，经营，酒店/观光，韩医，艺术方面最优秀的大学之一。

3.通过与各国各校频繁交流，该大学招收各国多数学生，并给学生提供加强国际化的机会。

4.为了满足中韩两国经济、文化交流与合作的需要，大学专门开设跆拳道、书法、电脑、韩国歌曲、韩国语课程给留学生。该教育学院教学环境优美，教学设施优良，并配有专职教师为学生在韩国期间的学习、升学及发展提供帮助，以利于留学生体验韩国文化，培养世界知识型人才。

5.学校校园环境优美，建筑雄伟极具特色，被誉为韩国最美的大学之一，尤其是每年春季樱花盛开之时更是美不胜收。

6.庆熙大学图书馆创建于1951年，是一所历史悠久、珍藏丰富、在各大学中属于特色鲜明，居于领先地位，具有较大影响的综合性大学图书馆。

7.交通便利，地理位置在地铁回基站一号出口，周边各种商店齐全，适合留学生居住。

8.截止到2018年，庆熙大学约有1,700名留学生在读。

## 專業课程

本科生學制四年，醫學專業六年，研究生學制三年。每年分為春季學期和秋季學期，春季學期是每學年第一學期，始于3月第一週，于6月第2週結束；秋季學期為每學年第二學期，始于9月第一週，于12月最後一週結束。每年5月第3週為Magnolia Festival，每年5月18日為校慶紀念日。

入学时间： 春季：2月报名 秋季：一批4月报名，二批6月报名 申请者需要提前3个月准备好材料。 入学事项： 普高毕业不超过二年。 大本／大专毕业及在读（年龄27周岁以下） 本科申请条件：国内外高中毕业topik能力3级以上，插班生topik能力四级以上 硕士申请条件：在国内外正规大学取得学士学位者，硕士学位（包括预科）共五个学期。

**`申请制度`**

**本科生奖学金**

**Scholarship Type A：**免除1年学费（不包括入学金）

**申请资格：**在招生评估阶段评估成绩处于前3%以上的位置。

**Scholarship Type B：**减免学费50% 一年（不包括入学金）

**申请资格：**在招生评估阶段评估成绩处于前3%-5%以上的位置。

**Scholarship Type C：**免除入学金

**申请资格：**在招生评估阶段评估成绩处于前5%-10%以上的位置。

**Scholarship Type D（仅限水原校区）：**第一学期学费减免30%（不包括入学金）

**申请资格：**获得韩国4年制大学为期一年（4学期）韩国语课程毕业证书。

**Scholarship Type E：**第一学期学费减免40%（不包括入学金）

**申请资格：**在招生评估阶段评估成绩处于前20%以上的位置，并获得庆熙大学国际教育学院为期一年（4学期）韩国语课程毕业证书。

**Scholarship Type F：**2年或4年学费免除（不包括入学金）

**申请资格：**学校奖学金委员会每年进行选择评估，凡具有优秀的学术能力者，如优异的成绩记录、突出的获奖记录和优秀的面试评估成绩等，均有可能获得。

除上述所列奖学金类型外，还有其他类型奖学金可以申请，学校在每期招生简章中有明确说明。一般每个人只能选择接受一种奖学金，不能兼得两类，但取得优异成绩和重大进步的可以获得系列奖学金。Type A和Type B奖学金获得者，如果上一学期完成15学分并取得GPA不低于2.0分，可以在下一学期继续享受优惠。

**研究生可以申请**

（1）庆熙大学校长奖学金：申请者所在研究生院各自的导师提名，获得者可免除全部学费和注册费，但名额有限。

（2）韩国政府奖学金（韩国信息通讯部）：毕业于信息技术专业相关院校的外国留学生可申请，提供所有费用的经济援助，硕士生2年，博士生4年。

**水原国际校区对外国学生还提供**

国际学生入学奖学金：授予在招录中被认为成绩最好的人。

国际学生优秀奖学金：学期平均成绩达到A或取得显著进步者可以申请。

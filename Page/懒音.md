比如英語making是標準音，説話或唱歌卻會變成makin'；粵語“猛”正確發音是maang5，但懶音變成maan5。英語和粵語說懶音時，\[ŋ\]都成了\[n\]。依此定義，可以認為世界上所有的語言都有不同程度“懶音”化現象。

網路流行後，也有為了少打幾個字而將兩個字音合併的例子，比如將“這樣”簡化成“醬”。

## 對“懶音”一詞的質疑

指出“懶音”一詞帶貶義，有道德批判成份，它不完全等同於英語的Relaxed pronunciation。因為在傳統語境裏，「懶」與「勤」相對，是貶義的。如果懶音的形成和個人責任無關的話，這種稱呼便是歧視了。因為這些人未必因為“懶”才將這些字音相混，有可能是從來沒有人教他們如何分辨這些不同的發音部位及教導正確的發音方法。\[1\]\[2\]此外，在[粵語正音運動](../Page/粵語正音運動.md "wikilink")中，[香港人口來源組成等因素常常被忽視](https://zh.wikipedia.org/wiki/香港人口 "wikilink")，故亦有人質疑這種說法帶有[沙文主義色彩](https://zh.wikipedia.org/wiki/沙文主義 "wikilink")，主張代之以“簡讀”等比較中立的名稱。\[3\]

### 北京話

  - [北京話有](https://zh.wikipedia.org/wiki/北京話 "wikilink")“兒懶音”，在語流中把聲母為舌尖擦音的音節說成含混的“兒”。如：

      - 告诉你 → gao su ni → gaor ni
      - 就是他 → jiu shi ta → jiur ta
      - 四十四 → si shi si → sir si

  - 北京话“这一”、“那一”等发生合音，分别变成“zhei”、“nei”等，作“这个”，“那个”的發音。

  - 另外，现代北京话有一些兩個音節合為一個音節的弱化現象，如“们”的韵母脱落成为-m词缀\[4\]，如

      - 我们（women） → wom
      - 你们（nimen） → nim

### 客家語

[客家語各地方言不同程度上都存在有](https://zh.wikipedia.org/wiki/客家語 "wikilink")“懶音”現象。這裡以廣東客家語-粵東片四縣腔為說明：

  - **來去(開始行動)：**主要是去的意思，客語四縣腔發音(loi hi)，loi是饒舌音，hi是氣喉音，一般口語上會將loi hi念成loii。客家話在句尾加上嘆詞、虛詞，顯現客家語的生動、貼切。\[來去(開始行動)\]後方會加上(o)的助詞，loi hi o 也會說成loiio。
  - **無愛mo oi：**念法係\[mɔ ɔi̯\],客語四縣腔常被唸成\[mɔi̯\]。
  - **分佢=分其：**給予他、給他，正確發音是\[pʊn ki\](國際拼音phun ki),客語四縣腔常被唸成\[pi\](國際拼音phi)，就是\[畀\]的古音。

### 吳語

[吳語各地方言不同程度上都存在有](https://zh.wikipedia.org/wiki/吳語 "wikilink")“懶音”現象。

  - **打党不分：**主要就是以“打”(an) 为代表的这类字，和以“党”(aon) 为代表的这类字\*，它们的韵母是不一样的。具体发音区别基本上是这样的：发an的时候，嘴巴张得比较大，发音比较靠前，有些地方，比如[无锡甚至接近普通话an的发音](https://zh.wikipedia.org/wiki/无锡 "wikilink")；而aon发音比较靠后，特别是[宁波](https://zh.wikipedia.org/wiki/宁波 "wikilink")，接近[英国英语long里面的ong的发音](https://zh.wikipedia.org/wiki/英国英语 "wikilink")，而其他地区多接近[美国英语](../Page/美国英语.md "wikilink")的这个发音。 最近打党这两类字的区别现在许多地方已经模糊了。比较常用的字里面的区别是 an/aon：张/赃、蚌/帮、场/尝、常长/上、厂/唱。基本上还是aon的字比较多。
  - **袜麦不分：**跟an和aon的区分类似，传统吴语（以苏州话为代表）区分“麦”、“袜”这两组入声。 吴拼以aeh和ah分别表示这两组音，对于这两组读音已经合并的地区，可以就用ah。以[苏州话](../Page/苏州话.md "wikilink")为例，“袜”读maeh，而“麦”读mah。ah里头的a读音近似单韵母a里头的a，而aeh里面的ae读音近似汉语拼音-an-里面的a。
  - **晏安不分：**适用于[苏南](../Page/苏南.md "wikilink")[上海和](https://zh.wikipedia.org/wiki/上海 "wikilink")[浙江部分地区](https://zh.wikipedia.org/wiki/浙江 "wikilink")。易错字正确发音如下：

|        |         |                 |
| ------ | ------- | --------------- |
| **漢字** | **吳語音** | **例字**          |
| 晏      | ae      | 攀、盼、删           |
| 安      | oe      | 参、惨、堪、砍、刊、侃、陕、栓 |

  - **司书不分：**在一些吴语中，比如中派和新派[上海话](../Page/上海话.md "wikilink")，读音上“斯”=“书”，发生了“书”并入“斯”的过程。在普通话里，这两组字读音是有区别的，在很多[吴语](../Page/吴语.md "wikilink")中也是。有些吴语则是把“书”并入了“虚”中，即“书”=“虚”，如[江阴话](https://zh.wikipedia.org/wiki/江阴话 "wikilink")。这里要讲的所谓“斯书”之分就是指以“斯”、“书”为代表的这两类字的区别。是以[苏州话](../Page/苏州话.md "wikilink")和老派上海话为蓝本的，区分这两组字的方法，即立足于这两组字韵母的不同。 “斯”这类字的发音，跟普通话一样，韵母是吴语拼音的y，没有韵母一样。 “书”的韵母用吴语拼音表示就是yu，是y的圆唇音。先发“斯”，然后慢慢把嘴唇收拢，就是“书”的音。

\-在[台州片当中](https://zh.wikipedia.org/wiki/台州片 "wikilink")，如[温岭方言中表现出懒化的有且不只](https://zh.wikipedia.org/wiki/温岭 "wikilink")：

` 不晓得——覅得`
` 不好过——孬（fao）过`

### 粤语

#### n/l不分

“你諗下囉”这个句子，前2个字（你諗）都原本是n作声母，最后一个字（囉）是l作声母。但现实生活中基本上全部都作l发音，虽然不影响理解。因為中國方言古時曾受韻書統一化的關係，其實差不多所有普通話內，'n' 聲母的字，粵語也是 'n' 聲母的字。例外的是 '弄' ，普通話是 'n' 聲母的字，粵語是 'l' 聲母的字。按傳統韻書音應為'l'，普通話在傳承的過程中變為了'n'。 同樣，'粒' 普通話是 'l' 聲母的字，粵語是 'n' 聲母的字。按傳統韻書音應為'l'，粤语在傳承的過程中變為了'n'。

香港人用**粒**作為量詞時，由於太繞口，所以會發**l**音。只有粤西地区，如肇庆一带的粤语，会将粒以n开头来读。

#### 軟顎鼻音的脱落

至于，如**我**(ngo5)、**咬**、**鈎**等。**我**现在读成“o5”音懶音出現的原因是因為使用鼻音\[ŋ\]發音比較用力，說話的人因為"懶惰"而不發出來，故稱為懶音。但粵語流行曲內最常用生字，反而矯枉過正。很多字之正音其實是[聲門塞音](../Page/聲門塞音.md "wikilink")聲母\[ʔ\]，但都插上 \[ŋ\] 聲母。 例如：**愛**、**惡**、**暗**、**矮**、**壓**。**鴨**、**鴉** 、**屙** 是清塞音聲母；**鵝**、**雅**、**牙**是 \[ŋ\] 聲母字。\[5\]

#### 复合声母中w的脱落

复合声母gw在日常生活中懒音现象尤其明显，而脱落的多数是ong, ok为韵母的字。

以下是一些例子：声母gw-\>g kw-\>k

  - **廣**(gwong2) → **港**(gong2)
  - **果**(gwo2) → **嗰**(go2)
  - **廓**(kwok3) → **確**(kok3)

由於客家話中並沒有gw-及kw-聲母，故此，部分人可能受客家話的影響。就以上例子，「廣」與「港」在客家話裏是同音（gong）；「果」與「嗰」在客家話裏都是同音（go）；「廓」與「確」在客家話裏也是同音（kok）\[6\]。

#### 韻尾的混淆

韻尾-ng 與 韻尾-n 的不分，或 韻尾-k 與 韻尾-t 的不分。

除了受其他方言的影響，主要是由於香港生活節奏急速，連說話都變得急速起來。於是乎一音未完，一音又起，遇到以 -ng 和以 -k 收音時問題便出現了。本來以 -ng 和 -k 收音的字，舌尖一定要平放，不可上翹，才能準確發音。但如果收音未發完便立即連接發出下一個字的讀音，便有可能無意中移動了舌頭所放的位置，不是平放而是錯誤地向上翹。

以下是一些例子：

  - 冷(laang5) 變成 懶(laan5)
  - 猛(maang5) 變成 晚(maan5)

由於 -k 是 -ng 的入聲，同樣的情況會發生在 -k 與 -t 上。發 -k 的音值時，如果錯誤地把舌尖翹起，放在門牙的背面，就會變成 -t 的音值。

以下是一些例子：

  - 策(caak3) 變成 察(caat3)
  - 責(zaak3) 變成 札(zaat3)

有些讀音錯誤的人，知道這樣發音是錯誤的，於是又矯枉過正，是為補償性錯誤發音。

相同的例子如下：

  - 懶(laan5) 變成 冷(laang5)
  - 晚(maan5) 變成 猛(maang5)
  - 察(caat3) 變成 策(caak3)
  - 札(zaat3) 變成 責(zaak3)

#### 声音m变n

这种情况常常出现在[周星驰的电影中](https://zh.wikipedia.org/wiki/周星驰 "wikilink")，[周星驰的这种发音对香港人造成极大的影响](https://zh.wikipedia.org/wiki/周星驰 "wikilink")，導致影迷相繼模仿。

  - **深**(sam1)入-\>**伸**(san1)入
  - **含**(ham4)义-\>**痕**(han4)义

## 英语

[英语](../Page/英语.md "wikilink")中懒音有一部分是因为[互联网](../Page/互联网.md "wikilink")的产生而大量繁衍的。如最常见的“What's up?”演变成“Wazzup?”。又例如：“How about ...?” -\> “How 'bout ...?”

今天英國標準英語裏大量的\[ə\]，其實就是把幾百年來流傳的懶音定為單詞的標準發音。term、girl、turn、work的發音本來正如其拼法，在[中古英語中讀作](https://zh.wikipedia.org/wiki/中古英語 "wikilink")、、、，可是懶音的讀法就成爲、、、，現代英國英語不屬於懶音。

**或非门**（）是数字逻辑中实现[逻辑或非的](../Page/逻辑或非.md "wikilink")[逻辑门](../Page/逻辑门.md "wikilink")，功能见右侧[真值表](../Page/真值表.md "wikilink")。若输入均为低电平（0），则输出为高电平（1）；若输入中至少有一个为高电平（1），则输出为低电平（0）。或非是[逻辑或加](../Page/逻辑或.md "wikilink")[逻辑非得到的结果](../Page/逻辑非.md "wikilink")。或非是一种具有[函数完备性的运算](../Page/函数完备性.md "wikilink")，因此其他任何逻辑函数都能用或非门实现。相比之下，[逻辑或运算器是一种单调的运算器](../Page/逻辑或.md "wikilink")，其只能将低电平变为高电平，但不能将高电平变为低电平。

在绝大多数但不是所有的电路设计中，逻辑非的功能本身就包含在结构中，如[CMOS和](../Page/CMOS.md "wikilink")[TTL等](../Page/晶体管－晶体管逻辑.md "wikilink")。在这样的逻辑系列中，要实现或门，唯一的方法是用2个或更多的逻辑门来实现，如一个或非门加一个反相器，但一个重要的例外是，因为其结构中本身就没有反相逻辑。

## 或非逻辑

[与非门实现的或非门逻辑函数如下](../Page/与非门.md "wikilink")：

\[x \overline{\lor} y = \left[ \left( x \overline{\land} x \right) \overline{\land} \left( y \overline{\land} y \right) \right] \overline{\land} \left[ \left( x \overline{\land} x \right) \overline{\land} \left( y \overline{\land} y \right) \right]\]

或非门具有[函数完备性](../Page/函数完备性.md "wikilink")，和与非门一样可以仅用其实现其他所有的逻辑功能。

|                                        |                                     |            |                                                                                                                                                                                                                               |
| -------------------------------------- | :---------------------------------: | ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| NOT（[非](../Page/逻辑非.md "wikilink")）    |          \(\overline{x}\)           | \(\equiv\) | \(x \overline{\lor} x\)                                                                                                                                                                                                       |
|                                        |                                     |            |                                                                                                                                                                                                                               |
| AND（[与](../Page/逻辑与.md "wikilink")）    |            \(x \land y\)            | \(\equiv\) | \(\left( x \overline{\lor} x \right) \overline{\lor} \left( y \overline{\lor} y \right)\)                                                                                                                                     |
| NAND（[与非](../Page/逻辑与非.md "wikilink")） |      \(x \overline{\land} y\)       | \(\equiv\) | \(\left[ \left( x \overline{\lor} x \right) \overline{\lor} \left( y \overline{\lor} y \right) \right] \overline{\lor} \left[ \left( x \overline{\lor} x \right) \overline{\lor} \left( y \overline{\lor} y \right) \right]\) |
| OR（[或](../Page/逻辑或.md "wikilink")）     |            \(x \lor y\)             | \(\equiv\) | \(\left( x \overline{\lor} y \right) \overline{\lor} \left( x \overline{\lor} y \right)\)                                                                                                                                     |
| NOR（[或非](../Page/逻辑或非.md "wikilink")）  |       \(x \overline{\lor} y\)       | \(\equiv\) | \(x \overline{\lor} y\)                                                                                                                                                                                                       |
| XOR（[异或](../Page/逻辑异或.md "wikilink")）  |      \(x \underline{\lor} y\)       | \(\equiv\) | \(\left( x \overline{\lor} y \right) \overline{\lor} \left[ \left( x \overline{\lor} x \right) \overline{\lor} \left( y \overline{\lor} y \right) \right]\)                                                                   |
| XNOR（[同或](../Page/同或.md "wikilink")）   | \(x \overline{\underline{\lor}} y\) | \(\equiv\) | \(\left[ \left( x \overline{\lor} y \right) \overline{\lor} x \right] \overline{\lor} \left[ \left( x \overline{\lor} y \right) \overline{\lor} y \right]\)                                                                   |
|                                        |                                     |            |                                                                                                                                                                                                                               |
| [蕴涵](../Page/实质条件.md "wikilink")       |         \(x \rightarrow y\)         | \(\equiv\) | \(\left[ \left( x \overline{\lor} x \right) \overline{\lor} y \right] \overline{\lor} \left[ \left( x \overline{\lor} x \right) \overline{\lor} y \right]\)                                                                   |
|                                        |         \(x \leftarrow y\)          | \(\equiv\) | \(\left[ x \overline{\lor} \left( y \overline{\lor} y \right) \right] \overline{\lor} \left[ x \overline{\lor} \left( y \overline{\lor} y \right) \right]\)                                                                   |
| [等价](../Page/逻辑等价.md "wikilink")       |       \(x \leftrightarrow y\)       | \(\equiv\) | \(\left[ \left( x \overline{\lor} y \right) \overline{\lor} x \right] \overline{\lor} \left[ \left( x \overline{\lor} y \right) \overline{\lor} y \right]\)                                                                   |
|                                        |                                     |            |                                                                                                                                                                                                                               |

| [矛盾式](../Page/矛盾式.md "wikilink")       |              \(\perp\)              | \(\equiv\) | \(\left( x \overline{\lor} x \right) \overline{\lor} x\)                                                                                                                                                                      |

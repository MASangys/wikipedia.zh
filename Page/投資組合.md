**投資組合**（**Investment portfolio**），又名**資產投資組合**，所重視的是[資產](../Page/資產.md "wikilink")，例如[股票](../Page/股票.md "wikilink")、[债券](../Page/债券.md "wikilink")、[外幣](https://zh.wikipedia.org/wiki/外幣 "wikilink")、[期权](https://zh.wikipedia.org/wiki/期权 "wikilink")、[貴金屬](../Page/貴金屬.md "wikilink")、[金融衍生工具](../Page/金融衍生工具.md "wikilink")、[房地产](../Page/房地产.md "wikilink")、[土地](https://zh.wikipedia.org/wiki/地皮 "wikilink")、[古董](../Page/文物.md "wikilink")、[上市公司](../Page/上市公司.md "wikilink")地位（俗稱「殼」）、[藝術品](https://zh.wikipedia.org/wiki/藝術品 "wikilink")、及至[紅酒等](../Page/红葡萄酒.md "wikilink")。一個投資組合是一個[投資者手上持手的資產性投資組合的成分](https://zh.wikipedia.org/wiki/投資者 "wikilink")，其中可分類為進取型、保守型等。一個優質的資產投資組合最理想的是具高[流動性](../Page/市场流通性.md "wikilink")、平穩及较高收益、低[投資風險等](https://zh.wikipedia.org/wiki/投資風險 "wikilink")。

資產投資組合的成份不會包括消費品，例如[跑车](../Page/跑车.md "wikilink")、[电视机](../Page/电视机.md "wikilink")、[化妝品](../Page/化妝品.md "wikilink")、[成衣等](https://zh.wikipedia.org/wiki/成衣 "wikilink")，因為它們都並無增值潛力，甚至只有[折舊](../Page/折舊.md "wikilink")。

投资组合风险和收益的计算如下： Portfolio Risk For Example Correlation Coefficient = .4

| Stocks   | σ  | % of Portfolio | Avg Return |
| -------- | -- | -------------- | ---------- |
| ABC Corp | 28 | 60%            | 15%        |
| Big Corp | 42 | 40%            | 21%        |

Standard Deviation = weighted avg = 33.6 (Brealey, 2010 p.165) &(Brealey, 2010 p.170): The expected return on your portfolio is simply a weighted average of the expected returns on the individual stocks: Expected portfolio return=股票1的标准偏差σ1乘以x1股票1的投资组合所占百分比例+投资品2的标准偏差σ2乘以x2投资产品2所占的百分比例=σ1\*r1+σ2\*r2; Standard Deviation Portfolio Variance = 28.1 =x<sup>2</sup><sub>1</sub>σ<sup>2</sup><sub>1</sub>+x<sup>2</sup><sub>2</sub>σ<sup>2</sup><sub>2</sub>+2(x<sub>1</sub>x<sub>2</sub>ρ<sub>12</sub>σ<sub>1</sub>σ<sub>2</sub>)

Real Standard Deviation:

`= (28)(.6) + (42)(.4) + 2(.4)(.6)(28)(42)(.4)`
`      =  28.1 CORRECT`

投资组合收益Return : r = (15%)(.60) + (21%)(.4) = 17.4%

delta是标准偏差Standard deviation,离正态分布远的偏差就大。 delta=variance投资组合方差的开平方，variance方差的算术平方根。方差计算公式如下： Portfolio variance投资组合方差=x<sup>2</sup><sub>1</sub>σ<sup>2</sup><sub>1</sub>+x<sup>2</sup><sub>2</sub>σ<sup>2</sup><sub>2</sub>+2(x<sub>1</sub>x<sub>2</sub>ρ<sub>12</sub>σ<sub>1</sub>σ<sub>2</sub>)

Expected portfolio return=stock1 return \* stock1 percent of portfolio + stock2 return \* stock2 percent of portfolio

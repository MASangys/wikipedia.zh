[Pizza_theorem_1.svg](https://zh.wikipedia.org/wiki/File:Pizza_theorem_1.svg "fig:Pizza_theorem_1.svg") **披薩定理**是平面[几何学](../Page/几何学.md "wikilink")之中的一个定理。它指出，如果以[圆](../Page/圆.md "wikilink")盘中任意一个指定點為中心，切下刀，使相邻的两刀隔的角度相同；然后按顺时针（或逆时针）的顺序给切出的各块交替染上两种颜色，将圆盘分为两个部分。那么有下列結論：

  - 当是大于2的[偶数](https://zh.wikipedia.org/wiki/奇數和偶數 "wikilink")（ \(=\) 4,6,8,10,12,14,..），或有任一刀通過圓心时：两种颜色的部分面积一样大。
  - 若任意一刀都不通過圓心，那么：
      - 当 \(=\) 1,2或除以4余3（ \(=\) 1,2,3,7,11,15,..）的时候，包含圓心的部分面积比较大。
      - 当大于4且除以4余1（ \(=\) 5,9,13,..）的时候，包含圓心的部分面积比较小。

这个定理之所以被称为披萨定理，是因为其中分割圆盘的方式类似于分披萨的过程。这个定理可以说明，当两个人用以上的方法分[披萨的时候](https://zh.wikipedia.org/wiki/披萨 "wikilink")，谁能拿到更多的披萨。 [Pizza_theorem2.jpg](https://zh.wikipedia.org/wiki/File:Pizza_theorem2.jpg "fig:Pizza_theorem2.jpg")

## 历史

披萨定理首先是作为一个数学难题在1967年的《数学杂志》上提出的，问题序号为660。提出此问题的人是L.J.厄普顿\[1\]。

厄普顿首先说明了切两刀时的情况，然后要求证明：切4刀的时候两部分的面积是一样的。1968年，《数学杂志》上登载了迈克尔·哥德堡对问题660的解答。解答中不仅解决了切4刀的问题，还解决了偶数刀时的情况\[2\]。

1994年，拉里·卡特和斯丹·瓦根用割补法对切4刀的问题给出了一个直观的[图解证明](../Page/无字证明.md "wikilink")\[3\]。他们的文章中还提到，堂·科波史密斯曾经利用[圆周率](https://zh.wikipedia.org/wiki/圆周率 "wikilink")的[超越性证明了切奇数刀的时候](../Page/超越數.md "wikilink")，两个部分面积不等，但没有给出更进一步的结果\[4\]。同年，卡特、瓦根和约翰·邓肯在《数学杂志》上提出切3刀和切5刀的问题\[5\]。

1995年，保罗·迪尔曼和里克·马布里在《数学杂志》上登载了3刀和5刀的解答，同时提出任意奇数刀的问题，邀请读者解决\[6\]。2009年，保罗·迪尔曼和里克·马布里最终解决了奇数刀时的情况，至此披萨问题完全解决\[7\]。

\(=\) 4时，2 \(=\) 8，圆盘被分为8份，每切一刀时转过的角度是45度。这个特例是1967年厄普顿首次提出披萨问题时的版本。1968年，哥登堡给出的证明是对于所有大于等于4的更广泛情况\[10\]。

1994年，卡特和瓦根用割补法给出了一个[直观的证明](../Page/无字证明.md "wikilink")（见左图）\[11\]。假设切的四刀将圆盘分成如左图所示的四个橙色部分和四个蓝色部分。则可以将每种颜色的部分继续分割，各分成8个更小的部分。如左图编号之后，其中相同数字对应的两个区块是全等的。这样就证明了，橙色的部分和蓝色的部分所占的面积是一样大的\[12\]。这个证明的妙处是一目了然，而不必再用公式或语言叙述了。

### 一般情况

\(\geqslant\) 4时的一般情况，证明的方法由迈克尔·哥德堡于1968年给出。这时候圆盘被分为2份，每切一刀时转过的角度是\(\scriptstyle \frac{\pi}{N}\)度。思路是用[极坐标的方式给出每一块的面积的](../Page/极坐标系.md "wikilink")[积分](../Page/积分.md "wikilink")表示，转化成一个代数问题\[13\]。

[PizzaTheoremNeven.png](https://zh.wikipedia.org/wiki/File:PizzaTheoremNeven.png "fig:PizzaTheoremNeven.png") 设在极坐标系中，圆上一点与点定义的直线和水平轴的夹角是，而记的长度是，那么

\[a_i = r(\beta + i\alpha)\] 而“第一块”图形（右图中右上角的橙色部分）的面积是：

  -
    \(S_1 = \frac12 \int_{\beta}^{\beta + \alpha} r^2 (\theta) \rm{d} \theta\)

“第二块”图形（右上角偏上的绿色部分）的面积是：\(S_2 =  \frac12 \int_{\beta + \alpha}^{\beta + 2\alpha} r^2 (\theta) \rm{d} \theta\)，“第块”图形的面积是

\(S_i = \frac12 \int_{\beta + (i-1) \alpha}^{\beta + i\alpha}  r^2 (\theta)  \rm{d} \theta\)

</center>

橙色部分的总面积是：

\[S_{r} = S_1 + S_3 +  \cdots + S_{2N-1}\]

\[=  \frac12 \int_{\beta}^{\beta + \alpha}  r^2 (\theta) \rm{d} \theta  +  \frac12 \int_{\beta+ 2\alpha}^{\beta + 3\alpha}  r^2 (\theta) \rm{d} \theta  +  \cdots +  \frac12 \int_{\beta+ (2N-2)\alpha}^{\beta + (2N-1)\alpha}  r^2 (\theta) \rm{d} \theta\]

  -

      -
        \(=  \frac12 \int_{0}^{\alpha}  r^2 (\beta + \theta) \rm{d} \theta  +  \frac12 \int_{0}^{\alpha}  r^2 (\beta + 2\alpha + \theta) \rm{d} \theta  +  \cdots +  \frac12 \int_{0}^{\alpha}  r^2 (\beta+ (2N-2)\alpha + \theta)  \rm{d} \theta\)

注意到对每个固定的，设 \(=\) ，再对应用引理，就有：

\[\sum_{i=0}^{N-1} r^2 (\beta + \theta + 2i\alpha )  = N\] 因此

\[S_{r} =  \frac12 \int_{0}^{\alpha} N \rm{d} \theta  = \frac{N\alpha}{2} = \frac{\pi}{2}\] 也就是说，橙色部分的面积是圆盘面积的一半。所以两种颜色的面积相等\[14\]。

\=== N=5的情况 === [PizzaTheoremN=5.png](https://zh.wikipedia.org/wiki/File:PizzaTheoremN=5.png "fig:PizzaTheoremN=5.png")  5的时候，将圆盘切成10个部分，每切一刀转过的角度是36度。设包含圆心的部分为蓝色，其余部分为白色。证明的思路和}} 3的时候一样.过圆心做平行于某条分割线的切线，然后将它与另一条分割线的交点记为。要注意的是由于交点有若干个，证明中选取的是最接近圆心的一点作为点。}}

注意到带状图形的面积可以表示成两个[弓形](../Page/弓形.md "wikilink")的面积之差。所以证明了以上的等式后，蓝红面积差实际上可以简化成：

\[\Delta S = f \left( 0, PP^* \sin \left( \frac{\pi}{5} \right)  \right) + f\left( OP^* \sin \left( \frac{2\pi}{5} \right), PP^* \sin \left( \frac{2\pi}{5} \right)  \right)\]

\[- f \left( OP^* \sin \left( \frac{\pi}{5} \right), PP^* \sin \left( \frac{2\pi}{5} \right)  \right) - f \left( OP^* \sin \left( \frac{2\pi}{5} \right), PP^* \sin \left( \frac{\pi}{5} \right)   \right)\] 其中的函数是表示圆心距等于，宽度为的带状图形的面积：

\[f (d, h) = s(d) - s(d+h)\] 而函数则是圆心距等于的弓形的面积：

\[s (d) =\cos^{-1}(d) - d\sqrt{1- d^2}.\] 其导数为\(s'(x) =- 2\sqrt{1- x^2}.\) 因此，要证明小于0，可以转化为证明以上关于的式子小于0。迪尔曼和马布里把函数在0点[展开](../Page/泰勒公式.md "wikilink")，得到的一个[无穷级数展开式](../Page/级数.md "wikilink")。经过一系列转化后，将问题变形为证明有关[黄金比值](../Page/黄金分割率.md "wikilink")\(\scriptstyle g = \frac{\sqrt{5} +1}{2} = 2  \cos \left( \frac{\pi}{5} \right)\)的[不等](../Page/不等.md "wikilink")式：

<center>

\(\forall n \ge 3, 1 \le k \le n-1, \quad g^k + g^{n-k} - g^n \le 0\)

</center>

可以证明这个不等式成立，因此小于0，即蓝色面积比较小\[16\]。

### 一般情况

3和5的情况後，希望能将}} 5的证明方法推广到一般的情况中去。证明分为两个部分。第一部分是证明“蓝色部分”面积减去“红色部分”面积的差等于两类相应的带状图形的面积和的差。第二部分则是将这个差表示成一系列的代数和，然后证明： \((-1)^{\frac{N-1}{2}}\Delta S\)小于0. 他们成功地将原命题转化成证明

\[\forall N \ge 3, \,  n \ge 3, \, 1 \le k \le n-1, \quad W(N, n, k) = (-1)^{\frac{N-1}{2}} \sum_{j=1}^{N}(-1)^{j-1} \sin^k \left(\frac{j\pi}{N} \right)
\sin^{(n-k)} \left(\frac{(j-1)\pi}{N} \right) \ge 0\] 并且仅当小于的时候等于0。}}

由于形式十分复杂，他们在寻找了14年后才发现了恰当的代数工具来证明以上不等式。2009年，他们在《[美国数学月刊](https://zh.wikipedia.org/wiki/美國數學月刊 "wikilink")》发表了最后的证明\[17\]。

## 类似问题和结论

### 披萨边定理

[PizzaCrust.png](https://zh.wikipedia.org/wiki/File:PizzaCrust.png "fig:PizzaCrust.png") 迪尔曼和马布里在2009年的证明论文中也探讨了所谓的“披萨边问题”\[18\]。这可看做是披萨定理的应用。披萨边可以看做是披萨减去披萨内馅（也是圆形）后的部分，也就是两个同心圆之间的圆[环](https://zh.wikipedia.org/wiki/环 "wikilink")。因此应用披萨定理，可以探讨按照披萨定理中的切法哪一部分的披萨边比较多。当切的刀数是偶数刀的时候，由于大圆和小圆都被等分为两部分，所以它们的差（圆环部分）也被等分为两部分。切的刀数是奇数刀的时候，还要分“薄边披萨”和“厚边披萨”来讨论。对于“薄边披萨”——两个同心圆的半径差不多，这时候的函数近似于披萨时函数的导数，用类似的方法可以推出和披萨相反的结论。也就是说，分得披萨较多的人，披萨边反而较少；分到披萨较少的人，披萨边会较多\[19\]。

厚边披萨的函数就不能近似于披萨函数的导数了，所以还需要进一步分析\[20\]\[21\]。

### 等边长切法

[EquiPizza.png](https://zh.wikipedia.org/wiki/File:EquiPizza.png "fig:EquiPizza.png") 如果切披萨的时候不是按照固定的角度，而是按照切下固定的边长（\(\frac{ \pi}{N}\)[弧度角](https://zh.wikipedia.org/wiki/弧度 "wikilink")）来分披萨，那么两部分的面积是否相等呢？答案是肯定的。无论切的刀数是偶数还是奇数，两个部分的面积都一样。这时两个人分到的披萨和披萨边都一定是一样多的了\[22\]\[23\]。

### 三维的情况

迪尔曼和马布里在2009年的证明论文中还探讨了披萨问题在三维空间中的推广\[24\]。这时的问题不再是“切披萨”，而是“切布丁”或“切西瓜”了。也就是说，切分的对象从圆形变成了平底的三维旋转体，即横截面是圆形的三维物体。这时候，三维物体可以看成是无数个“同心”圆盘的叠加\[25\]。所以三维空间中的情况仍然可以转化成关于函数性质的讨论，所以在的性质足够明确的时候，可以得到类似于披萨定理的结论。迪尔曼和马布里研究了半抛球体和半椭球体的情况，发现前者的性质和披萨的情况恰好相反，而后者在 \(\geqslant\) 5的时候，两个部分的体积永远相等。也就是说，如果按照分披萨的方法切半个西瓜，那么只要切的刀数大于等于5，无论切的刀数是奇数还是偶数，两个人分到的西瓜总是一样多\[26\]。

### 多人分披萨

如果分披萨的人不止两个，问题就变得更加复杂了。1999年，赫史霍恩等人发现了一个特例下人等分披萨的方法。他们证明了：如果将一个圆盘按照披萨定理中的切法切2刀分成4块，那么可以将其分为面积相等的份，每份4块。具体方法是选择任一块后，再选择它旋转90度、180度以及270度之后位置的3块，一起作为一份\[27\]。

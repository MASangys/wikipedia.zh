**拉廊府**（，）在[泰國南部](https://zh.wikipedia.org/wiki/泰國 "wikilink")，她座落在[安達曼海滂](https://zh.wikipedia.org/wiki/安達曼海 "wikilink")。西面是[緬甸](https://zh.wikipedia.org/wiki/緬甸 "wikilink")[德林達依省的](https://zh.wikipedia.org/wiki/德林達依省 "wikilink")[高当](https://zh.wikipedia.org/wiki/高当 "wikilink")（Kawthaung），東到南分別是泰國的[春蓬府](../Page/春蓬府.md "wikilink")、[素叻府和](https://zh.wikipedia.org/wiki/素叻府 "wikilink")[攀牙府](../Page/攀牙府.md "wikilink")。

## 地理

拉廊府處在[馬來亞半島中部狹長的](https://zh.wikipedia.org/wiki/馬來亞半島 "wikilink")[克拉地峽](https://zh.wikipedia.org/wiki/克拉地峽 "wikilink")，西端為[普吉山脈](https://zh.wikipedia.org/wiki/普吉山脈 "wikilink")。她臨著[安達曼海有漫長的海岸](https://zh.wikipedia.org/wiki/安達曼海 "wikilink")。

在北部的[卡泊縣](https://zh.wikipedia.org/wiki/卡泊縣 "wikilink")（Amphoe Kapoe）有一個面積為303.09平方公里的自然生態圈保護區。她建立於1997年，主要是用以保護紅水杉林。

在本府南部還有另一個納柯運河野生動物保護區（Khlong Nakha Wildlife Sanctuary）。

## 氣候

本府的氣候特徵就是每年長達七、八個月連綿不斷的豪雨，在1955年全年雨量曾達到6699.5毫米。 而泰國中部的平均降雨量僅為1200毫米。

## 人口

本府是泰國人煙最稀少的一府，百分之八十的地面是無人居住的[森林](../Page/森林.md "wikilink")地帶，過去人多從事採[錫礦](https://zh.wikipedia.org/wiki/錫 "wikilink")，現代人口多參與[燒瓷](https://zh.wikipedia.org/wiki/瓷 "wikilink")、[捕魚](https://zh.wikipedia.org/wiki/捕魚 "wikilink")、[橡膠](../Page/橡膠.md "wikilink")和[核桃](../Page/核桃.md "wikilink")的種植。

## 歷史

拉廊原先主要為[馬來人居住](https://zh.wikipedia.org/wiki/馬來人 "wikilink")，後來到[十七至](https://zh.wikipedia.org/wiki/十七世紀 "wikilink")[十八世紀泰國人不斷大規模遷入](https://zh.wikipedia.org/wiki/十八世紀 "wikilink")，馬來人漸漸失勢及遷出。

## 府徽和府花

|                                                                                                                            |  |
| -------------------------------------------------------------------------------------------------------------------------- |  |
| 五世泰王[朱拉隆功大帝曾經駕臨過拉廊府地](https://zh.wikipedia.org/wiki/朱拉隆功大帝 "wikilink")，他在尼威斯吉利山（Niveskiri Hill）上一座城堡中棲息，這座城堡後來就成為拉廊府的徽章標誌。 |  |

本府的府樹是[大花紫薇](../Page/大花紫薇.md "wikilink")（*Lagerstroemia speciosa*），府花是[石斛兰](https://zh.wikipedia.org/wiki/石斛兰 "wikilink")（*Dendrobium formosum*）。

## 行政及管理

本府行政上被劃分為五個縣（ampoe），又進一步被劃分為30個區（tambon）及167個村（muban）。

**停損點轉向操作系統**（[英文](https://zh.wikipedia.org/wiki/英文 "wikilink")：**Stop
And Reverse**，[縮寫](../Page/縮寫.md "wikilink")：**SAR**），1978年在威爾德（[Welles
Wilder
JR.](https://zh.wikipedia.org/wiki/:en:J._Welles_Wilder "wikilink")）的著作《技術交易系統中的新觀念》裡提出，原名為**抛物線指標**（Parabolic）。因停損價位的軌跡類似抛物線，稱為**抛物線交易系統**（Parabolic
Trading System）\[1\]，亦有人以**Parabolic SAR**表示。

SAR在圖形和運用上與移動平均線相似，屬於價格與時間並重之順勢交易的分析系統。\[2\]

[SAR_TWSE.png](https://zh.wikipedia.org/wiki/File:SAR_TWSE.png "fig:SAR_TWSE.png")、60日[移動平均和SAR](../Page/移動平均.md "wikilink")（抛物線指標）。\]\]

## 計算方法

〖基本公式〗

\[{SAR}_t = {SAR}_{t-1} + \alpha ({EP}_{t-1} - {SAR}_{t-1})\]

  -
    【註】
    1.  \(t\)：交易日，\(t-1\)：前一交易日；
    2.  \(\alpha\)：加速因子、調整係數，\(0.02 \le \alpha \le 0.2\)；
    3.  \(SAR\)：抛物線指標；
    4.  \(EP\)：區間極值（最高價EPH或最低價EPL）。

〖漲勢買進〗

\[SAR_0=\] 近期相對最低價\(\begin{smallmatrix}EPL\end{smallmatrix}\)。

\[{SAR}_t = {SAR}_{t-1} + \alpha ({EPL}_{t-1} - {SAR}_{t-1})\]

:\*當股價高點向上穿越SAR時，採漲勢買進公式，計算新SAR值。

〖跌勢賣出〗

\[SAR_0=\] 近期相對最高價\(\begin{smallmatrix}EPH\end{smallmatrix}\)。

\[{SAR}_t = {SAR}_{t-1} - \alpha ({SAR}_{t-1} - {EPH}_{t-1})\]

:\*當股價低點向下跌破SAR時，採跌勢賣出公式，計算新SAR值。

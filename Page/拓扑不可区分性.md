在[拓扑学中](../Page/拓扑学.md "wikilink")，[拓扑空间](../Page/拓扑空间.md "wikilink")*X*內的两点若有完全相同的[鄰域](../Page/鄰域.md "wikilink")，便稱這兩個點為「**拓扑不可区分的**」。亦即，設*x*及*y*為*X*內的兩點，*A*為由所有包含*x*的鄰域所組成的集合，且*B*為由所有包含*y*的鄰域所組成的集合，則*x*及*y*為「拓撲不可區分的」若且唯若*A*
= *B*。

直觀上來說，若*X*的拓撲無法分辦之中的兩點，即可稱這兩點為拓撲不可區分的。

若*X*內的兩點不是拓撲不可區分的，則稱這兩點為「**拓撲可區分的**」。这表示存在只包含两点之中的其中一點的[开集](../Page/开集.md "wikilink")（或等价地说，存在只包含两点之中的其中一點的[闭集](../Page/闭集.md "wikilink")），而这个开集則可以用来使两个点可以區分。[T<sub>0</sub>空间是一個拓撲空間](../Page/T0空间.md "wikilink")，其中任意兩個相區別的點都是拓扑可区分的。这是[分离公理中最弱的一個限制條件](../Page/分离公理.md "wikilink")。

拓扑不可区分性會在拓扑空间*X*上定义出一個[等价关系](../Page/等价关系.md "wikilink")。設*x*和*y*為*X*內的兩個点，若*x*和*y*為拓撲不可區分的，便標記成*x*
≡ *y*；*x*的[等價類則標記為](../Page/等價類.md "wikilink")\[*x*\]。

## 例子

对[T<sub>0</sub>空间](../Page/T0空间.md "wikilink")（特别是[豪斯多夫空间](../Page/豪斯多夫空间.md "wikilink")）而言，拓扑不可区分的概念是沒有意義的，因此若要尋找有趣的例子，必须要在非T<sub>0</sub>空间中才行。另一方面，由於[正则性和](../Page/正则空间.md "wikilink")[正规性並不蕴涵T](../Page/正规空间.md "wikilink")<sub>0</sub>，所以可以找到一些有這些性质的例子。事实上，下面给出的例子就几乎都是[完全正则的](../Page/完全正则空间.md "wikilink")。

  - 在[不可分空間中](../Page/密着拓扑.md "wikilink")，任意两个点都是拓扑不可区分的。
  - 在[伪度量空间中](../Page/偽度量.md "wikilink")，两点是拓扑不可区分的，当且仅当在兩點之间的距离為零。
  - 在[半赋範向量空间中](../Page/賦範向量空間.md "wikilink")，*x* ≡ *y*当且仅当‖*x* − *y*‖ =
    0。
      - 舉例來說，设*L*<sup>2</sup>（R）是一個拓撲空間，由所有从R映射至R的[平方可积的](../Page/可积函数.md "wikilink")[可測函數所組成](../Page/可測函數.md "wikilink")（詳见[*L*<sup>*p*</sup>空间](../Page/Lp空间.md "wikilink")）。则在*L*<sup>2</sup>（R）內，函数*f*及*g*為拓扑不可区分的，当且仅当兩個函數[几乎处处相等](../Page/几乎处处.md "wikilink")。
  - 在[拓扑群中](../Page/拓扑群.md "wikilink")，*x* ≡ *y*，当且仅当*x*<sup>−1</sup>*y*
    ∈
    cl{*e*}，这里的cl{*e*}為[当然子群的](../Page/当然群.md "wikilink")[闭包](../Page/闭包.md "wikilink")；而其等价类則為cl{*e*}的[陪集](../Page/陪集.md "wikilink")（cl{*e*}总會是個[正规子群](../Page/正规子群.md "wikilink")）。
  - [一致空间推广了伪度量空间和拓扑群](../Page/一致空间.md "wikilink")。在一致空间中，*x* ≡
    *y*当且仅当有序对 (*x*,
    *y*)属于所有[周围](../Page/周围_\(拓扑学\).md "wikilink")（）。所有周围的交集正是用*X*上拓扑不可区分性來定義的等价关系。
  - 设*X*有关于函数族\(\{f_\alpha : X \to Y_\alpha\}\)的[初拓撲](../Page/初拓撲.md "wikilink")。*X*中两个点*x*和*y*是拓扑不可区分的，如果\(f_\alpha\)族不区分它们（就是说对所有\(\alpha\)，\(f_\alpha(x) = f_\alpha(y)\)）。
  - 给定集合*X*上的任何等价关系，存在*X*上的拓扑，它的拓扑不可区分概念一致于这个等价关系。你可以简单选取这个等价关系为这个拓扑的[基](../Page/基_\(拓扑学\).md "wikilink")。这叫做*X*上的[划分拓扑](../Page/划分拓扑.md "wikilink")。

## 特殊化预序

在空间*X*上的拓扑不可区分性可以从在*X*上的叫做[特殊化预序的自然](../Page/特殊化预序.md "wikilink")[预序来复原](../Page/预序.md "wikilink")。对于*X*中的点*x*和*y*这个预序定义为

  -
    *x* ≤ *y* [当且仅当](../Page/当且仅当.md "wikilink")*x* ∈ cl{*y*}

这里的cl{*y*}指示{*y*}的[闭包](../Page/闭包.md "wikilink")。等价的说，*x* ≤
*y*如果*x*的[邻域系统](../Page/邻域系统.md "wikilink")，指示为*N*<sub>*x*</sub>，被包含在*y*的邻域系统内：

  -
    *x* ≤ *y*当且仅当*N*<sub>*x*</sub> ⊂
    *N*<sub>*y*</sub>。容易看出在*X*上的这个关系是[自反的和](../Page/自反关系.md "wikilink")[传递的](../Page/传递关系.md "wikilink")，所以定义了预序。但是一般的说，这个预序不是[反对称的](../Page/反对称关系.md "wikilink")。实际上，确定自≤的等价关系完全就是拓扑不可区分性的关系：
    *x* ≡ *y*当且仅当*x* ≤ *y*并且*y* ≤ *x*。

拓扑空间被称为[对称（或R<sub>0</sub>）的](../Page/R0空间.md "wikilink")，如果特殊化预序是[对称的](../Page/对称关系.md "wikilink")（就是说*x*
≤ *y*蕴涵*y* ≤
*x*）。在这种情况下，关系≤和≡是同一的。拓扑不可区分性在这些空间中表现良好并易于理解。注意这类空间包括所有[正则空间和](../Page/正则空间.md "wikilink")[完全正则空间](../Page/完全正则空间.md "wikilink")。

## 性质

### 等价条件

有很多确定两个点是拓扑不可区分的等价方式。设*X*是拓扑空间并设*x*和*y*是*X*的点。把*x*和*y*的闭包分别指示为cl{*x*}和cl{*y*}，并把它们的[邻域系统分别指示为](../Page/邻域系统.md "wikilink")*N*<sub>*x*</sub>和*N*<sub>*y*</sub>。则下列陈述是等价的：

  - *x* ≡ *y*
  - 对于每个*X*中的开集*U*，要么*U*包含*x*和*y*二者要么都不包含
  - *N*<sub>*x*</sub> = *N*<sub>*y*</sub>
  - *x* ∈ cl{*y*}并且*y* ∈ cl{*x*}
  - cl{*x*} = cl{*y*}
  - *x* ∈ <big>∩</big>*N*<sub>*y*</sub>并且*y* ∈
    <big>∩</big>*N*<sub>*x*</sub>
  - <big>∩</big>*N*<sub>*x*</sub> = <big>∩</big>*N*<sub>*y*</sub>
  - *x* ∈ cl{*y*}并且*x* ∈ <big>∩</big>*N*<sub>*y*</sub>
  - *x*属于包含*y*的所有开集和所有闭集
  - [网或](../Page/网_\(数学\).md "wikilink")[滤子会聚于](../Page/滤子_\(数学\).md "wikilink")*x*当且仅当它会聚于*y*

这些条件可以在*X*是[对称空间的情况下简化](../Page/R0空间.md "wikilink")。对于这些空间(特别是[正则空间](../Page/正则空间.md "wikilink"))，下列陈述是等价的：

  - *x* ≡ *y*
  - 对于每个开集*U*，如果*x* ∈ *U*则*y* ∈ *U*
  - *N*<sub>*x*</sub> ⊂ *N*<sub>*y*</sub>
  - *x* ∈ cl{*y*}
  - *x* ∈ <big>∩</big>*N*<sub>*y*</sub>
  - *x*属于包含*y*的所有闭集
  - *x*属于包含*y*的所有开集
  - 所有会聚于*x*的网或滤子会聚于*y*

### 等价类

要讨论*x*的[等价类](../Page/等价类.md "wikilink")，為了方便，首先定义*x*的[上闭集合和下闭集合](../Page/上闭集合.md "wikilink")。它们都是关于上述[特殊化预序而定义的](../Page/特殊化预序.md "wikilink")。

*x*的下部集合就是{*x*}的闭包：

\[\mathop{\darr}x = \{y\in X : y\leq x\} = \textrm{cl}\{x\}\]，而*x*上部集合是在*x*的[邻域系统的](../Page/邻域系统.md "wikilink")[交集](../Page/交集.md "wikilink")：

\[\mathop{\uarr}x = \{y\in X : x\leq y\} = \bigcap \mathcal{N}_x\]。

*x*的等价类接着给出为交集

\[[x] = {\mathop{\darr}x} \cap {\mathop{\uarr}x}\]。

因为↓*x*是包含*x*的所有闭集的交集而↑*x*是包含*x*的所有开集的交集，等价类\[*x*\]是包含*x*的所有开集和闭集的交集。

cl{*x*}和<big>∩</big>*N*<sub>*x*</sub>二者都包含等价类\[*x*\]。一般的说，两个集合都会包含额外的点。但是在[对称空间中](../Page/R0空间.md "wikilink")(特别是在[正则空间中](../Page/正则空间.md "wikilink"))，这三个集合是一致的：

\[[x] = \textrm{cl}\{x\} = \bigcap\mathcal{N}_x\]。一般的说，等价类\[*x*\]会是闭集，当且仅当这个空间是对称的。

### 连续函数

设*f* : *X* →
*Y*是[连续函数](../Page/连续函数_\(拓扑学\).md "wikilink")。则对于任何*X*中的*x*和*y*有

  -
    *x* ≡ *y*蕴涵*f*(*x*) ≡
    *f*(*y*)。逆命题一般为假(T<sub>0</sub>空间有是[平凡的](../Page/平凡拓扑.md "wikilink")[商](../Page/商空间.md "wikilink"))。逆命题在*X*有引发自*f*的[初拓扑的条件下为真](../Page/初拓扑.md "wikilink")。更一般的说，如果*X*有引发自映射族\(f_\alpha : X \to Y_\alpha\)的初拓扑则
    *x* ≡ *y*当且仅当*f*<sub>α</sub>(*x*) ≡
    *f*<sub>α</sub>(*y*)对于所有α。可得出在[乘积空间中两个元素是拓扑不可区分的](../Page/乘积空间.md "wikilink")，当且仅当每个它们的分量都是拓扑不可区分的。

## 柯尔莫果洛夫商空间

因为拓扑不可分别性是在任何拓扑空间*X*上的等价关系，我们可以形成[商空间](../Page/商空间.md "wikilink")*KX* =
*X*/≡。空间*KX*被叫做[柯爾莫哥洛夫商空間或](../Page/柯爾莫果洛夫空間#柯爾莫哥洛夫商空间.md "wikilink")*X*的T<sub>0</sub>同一。事实上，空间*KX*是T<sub>0</sub>（就是说所有点都是拓扑可区分的）。此外，通过商映射的特征性质，任何从*X*到T<sub>0</sub>空间的连续映射*f*
: *X* → *Y*通过商映射*q* : *X* → *KX*而因子化。

尽管商映射*q*一般不是[同胚](../Page/同胚.md "wikilink")（因为它一般不是[单射](../Page/单射.md "wikilink")），它确实引发在*X*的拓扑和*KX*的拓扑之间的[双射](../Page/双射.md "wikilink")。直觉上说，柯尔莫果洛夫商不改变一个空间的拓扑。它只將点集精簡化，直到点都成为拓扑可区分的。

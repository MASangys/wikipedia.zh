**捷運**，全名為**大眾捷運系統**，該詞源於[臺灣對於](../Page/臺灣.md "wikilink")[英語](../Page/英語.md "wikilink")「」（簡稱）的[中文直譯](../Page/中文.md "wikilink")，含有「快速」之意。在多數場合可同義於[地鐵或](../Page/地鐵.md "wikilink")[城市軌道交通](../Page/城市軌道交通.md "wikilink")，為[鐵路運輸的一種形式](../Page/鐵路運輸.md "wikilink")；而隨著各[城市的發展與需求不同](../Page/城市.md "wikilink")，該詞也擴大使用在一些同樣具有高密度運輸特性、但並非運行於[軌道上的](../Page/鐵路軌道.md "wikilink")[交通工具](../Page/交通工具.md "wikilink")，例如「[公車捷運系統](../Page/公車捷運系統.md "wikilink")」。

主要使用「捷運」之名的是服務臺灣各[都會區的城市軌道交通系統](../Page/臺灣都會區.md "wikilink")，[美國](../Page/美國.md "wikilink")[舊金山及](../Page/舊金山灣區捷運系統.md "wikilink")[新加坡等地的地鐵系統及其管理單位亦稱作](../Page/新加坡地鐵.md "wikilink")「」，而[港鐵則採用相近的](../Page/港鐵.md "wikilink")「」（，譯作「集體運輸鐵路」）做為英文名稱。在中国大陆，部分轨道交通系统亦采用捷运一词，指代[胶轮路轨系统或机场内部的运输系统](../Page/胶轮路轨系统.md "wikilink")（[上海地鐵浦江线](../Page/上海轨道交通浦江线.md "wikilink")、[浦东机场捷运](../Page/浦东机场捷运.md "wikilink")）。

## 歷史

目前臺灣社會多以「捷運」一詞做為城市軌道交通系統的通稱，如臺灣鐵道學者[蘇昭旭的](../Page/蘇昭旭.md "wikilink")《世界捷運與輕軌圖鑑》一書\[1\]。

臺灣最初使用「捷運」一詞者為《[大眾捷運法](../Page/s:大眾捷運法.md "wikilink")》，由於其施行後營運之各大眾捷運系統有效改善了都市交通，使得「捷運」一語逐漸流行，並擴張應用在許多不適用該法的交通設施。例如，嘉義地區因應[高鐵通車而設置的](../Page/台灣高速鐵路.md "wikilink")[BRT中文名稱即訂為](../Page/BRT.md "wikilink")「[嘉義公車捷運](../Page/嘉義公車捷運.md "wikilink")」；而[臺灣鐵路管理局亦有名為](../Page/臺灣鐵路管理局.md "wikilink")[捷運化之各項既有路線改善計畫](../Page/臺鐵捷運化.md "wikilink")，期望以都會地區路段增班、增站、提速等方式提高其服務品質。或是[桃園捷運網路](../Page/桃園捷運.md "wikilink")、[臺中捷運網路](../Page/臺中捷運.md "wikilink")，因經費成本考量，網路中以[公車捷運系統](../Page/公車捷運系統.md "wikilink")（BRT）或是[臺鐵捷運化路線作為其網路](../Page/臺鐵捷運化.md "wikilink")。

[地下鐵路](../Page/地下鐵路.md "wikilink")（[地鐵](../Page/地鐵.md "wikilink")）是另一個經常與捷運相提並論的名詞，世界各國許多[城市的類似系統都是採地下化為主的鋪設方式](../Page/城市.md "wikilink")，因此雖然習慣以「地下鐵路」稱呼之，但實際上卻並非全部都在地下運行（事實上，大部分的地鐵系統在離開市中心的人口稠密區後，都會改為地面行駛以節省建造成本）。而在臺灣，並不是所有行走於地下的鐵路運輸系統都一概歸類為「捷運」，例如臺灣鐵路管理局在其都會區路段進行的[地下鐵路工程](../Page/臺鐵地下化.md "wikilink")，習慣上被稱呼為「[鐵路地下化](../Page/鐵路地下化.md "wikilink")」。因此臺灣將城市軌道交通稱為「大眾捷運系統」，即「捷運」，以避免在名稱上不符實情的狀況。

臺灣以外的場合上，马来西亚新建的[双溪毛糯－加影线状况大同小异](../Page/双溪毛糯－加影线.md "wikilink")，即「捷运」大部分并非位于地下。

## 分類

根据臺灣的定义，大眾捷運系統的分類主要依載具運量能量分為高運量、中運量及低運量三種分類方式，其級距分別為二萬、五千及五千以下。

| 捷運種類                             | 運量(人次/小時/單方向) | 系統名稱                           | 車輪型式  | 專用路權 |
| -------------------------------- | ------------- | ------------------------------ | ----- | ---- |
| [高運量](../Page/高運量.md "wikilink") | 20,000以上      | 鐵路捷運(Metro/Subway)             | 鋼輪    | 是    |
| 區域通勤鐵路(Regional Rail)            | 鋼輪            | 是                              |       |      |
| 膠輪捷運(Rubber-tyred Rapid Transit) | 膠輪            | 是                              |       |      |
| [中運量](../Page/中運量.md "wikilink") | 5,000\~20,000 | 輕軌捷運(Light Rail Rapid Transit) | 鋼輪    | 是    |
| 自動導軌運輸(Automated Guided Transit) | 膠輪/鋼輪         | 是                              |       |      |
| 單軌捷運(Monorail Rapid Transit)     | 膠輪            | 是                              |       |      |

**-{zh-cn:接口描述语言;zh-tw:介面描述語言}-**（Interface description language，缩写**IDL**），是用来描述[软件组件](https://zh.wikipedia.org/wiki/软件组件 "wikilink")[介面的一种](https://zh.wikipedia.org/wiki/介面 "wikilink")[计算机语言](https://zh.wikipedia.org/wiki/计算机语言 "wikilink")。IDL通过一种中立的方式来描述接口，使得在不同平台上运行的对象和用不同语言编写的程序可以相互通信交流；比如，一个组件用[C++](../Page/C++.md "wikilink")写成，另一个组件用[Java](../Page/Java.md "wikilink")写成。

IDL通常用于[远程调用软件](https://zh.wikipedia.org/wiki/远程调用 "wikilink")。在这种情况下，一般是由远程客户终端调用不同操作系统上的对象组件，并且这些对象组件可能是由不同计算机语言编写的。IDL建立起了两个不同操作系统间通信的桥梁。

在IDL基础上开发出来的软件系统有[Sun的](https://zh.wikipedia.org/wiki/Sun_Microsystems "wikilink")[ONC RPC](https://zh.wikipedia.org/wiki/ONC_RPC "wikilink")，[The Open Group的](https://zh.wikipedia.org/wiki/The_Open_Group "wikilink")[Distributed Computing Environment](https://zh.wikipedia.org/wiki/Distributed_Computing_Environment "wikilink")，[IBM](../Page/IBM.md "wikilink")的[System Object Model](https://zh.wikipedia.org/wiki/System_Object_Model "wikilink")，[Object Management Group的](https://zh.wikipedia.org/wiki/Object_Management_Group "wikilink")[CORBA](../Page/CORBA.md "wikilink")，和[SOAP](https://zh.wikipedia.org/wiki/Simple_Object_Access_Protocol "wikilink")（用于[Web service](https://zh.wikipedia.org/wiki/Web_service "wikilink")）。

## 典型的-{zh-cn:接口描述语言;zh-tw:介面描述語言}-

  - [IDL specification language](https://zh.wikipedia.org/wiki/IDL_specification_language "wikilink")，the original Interface Description Language.
  - [Microsoft Interface Definition Language](https://zh.wikipedia.org/wiki/Microsoft_Interface_Definition_Language "wikilink")
  - [HIDL](https://zh.wikipedia.org/wiki/HIDL "wikilink")（HAL Interface Definition Language，[硬件抽象层接口描述语言](https://zh.wikipedia.org/wiki/硬件抽象层 "wikilink")），用于[Android](../Page/Android.md "wikilink")操作系统8.0及以上版本
  - [Open Service Interface Definitions](https://zh.wikipedia.org/wiki/Open_Service_Interface_Definitions "wikilink")
  - [Platform-Independent Component Modeling Language](https://zh.wikipedia.org/wiki/Platform-Independent_Component_Modeling_Language "wikilink")
  - [Simple Object Access Protocol](https://zh.wikipedia.org/wiki/Simple_Object_Access_Protocol "wikilink")（SOAP）
  - [WDDX](https://zh.wikipedia.org/wiki/WDDX "wikilink")
  - [XML-RPC](../Page/XML-RPC.md "wikilink")，the predecessor of SOAP

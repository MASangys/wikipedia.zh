插值排序是使用插值公式分散資料並以標簽陣列操控遞迴方式的排序演算法，屬於分布排序類下桶排序變型直方圖排序的遞迴排序方法。

## 演算法

**插值排序**（interpolation sort）或稱為直方圖排序（histogram
sort），是一種使用[插值公式分散資料](../Page/插值.md "wikilink")[分而治之的排序演算法](https://zh.wikipedia.org/wiki/分而治之 "wikilink")。插值排序也是[桶排序](../Page/桶排序.md "wikilink")[演算法的一種變型](https://zh.wikipedia.org/wiki/演算法 "wikilink")。插值排序遞迴方式運用一個記錄桶子長度的陣列對應原數列，透過操作維護長度陣列可以避免[遞歸演算法因記憶體堆疊而使空間複雜度變成O](https://zh.wikipedia.org/wiki/遞歸 "wikilink")(n
^ 2)，藉由長度陣列的分段記錄可以動態的宣告與刪除鏈表的記憶體空間，使得遞歸程序得以控制所需空間複雜度在O(n \*
k)，但是執行複雜度仍可維持為O(n +
k)的高效排序方法。動態分配記憶體的[鏈表可以由堆棧](https://zh.wikipedia.org/wiki/鏈表 "wikilink")、佇列、關聯式陣列、樹狀結構等實作，例如Javascript的陣列物件即適用。資料結構的不同關係著資料存取的速度進而影響到排序所需的時間。當被排序陣列內的數值是均勻分散近似等差級數時，插值排序的線性時間為O(n)。

插值排序演算法：

1.  設置一個長度陣列記錄未完成排序桶子的長度。
2.  主排序程序判斷長度陣列是否清空，未清空執行劃分。
3.  從長度陣列末端取出一個長度執行劃分，如果最大值等於最小值該序列排序完成停止劃分。
4.  設置一個定量的陣列當作空桶子，尋訪序列並且把項目一個一個放到插值對應的桶子。
5.  從不是空的桶子裡把項目逐一放回原來的序列中，並在長度陣列放入桶子鏈表的長度。
6.  返迴主排序處理程序。

## 實作

JavaScript code:

``` javascript
Array.prototype.interpolationSort = function() {
    var divideSize = new Array();
    var end = this.length;
    divideSize[0] = this.length;
    while (divideSize.length > 0) divide(this);

    function divide(A) {
        var size = divideSize.pop();
        var start = end - size;
        var min = A[start];
        var max = A[start];
        for ( i = start + 1; i < end; i++)
            if (A[i] < min) min = A[i];
            else if (A[i] > max) max = A[i];
        if (min == max) end = end - size;
        else {
            var p = 0;
            var bucket = new Array(size);
            for ( i = 0; i < size; i++)
                bucket[i] = new Array();
            for ( i = start; i < end; i++) {
                p = Math.floor(((A[i] - min ) / (max - min ) ) * (size - 1 ));
                bucket[p].push(A[i]);
            }
            for ( i = 0; i < size; i++) {
                if (bucket[i].length > 0) {
                    for ( j = 0; j < bucket[i].length; j++)
                        A[start++] = bucket[i][j];
                    divideSize.push(bucket[i].length);
                }
            }
        }
    }
};
```

## 變種

### 插值標簽排序

插值標簽排序是插值排序的類原地排序方法，通過操做維護N個位元標簽可以達到只需N次的交換即可完成排序；但數字陣列必需是連續的整數，或者陣列是完全平均分散近似於等差級數。插值標簽排序對不重複的連續整數數列排序，只需一個長度與原陣列相等的位元標簽陣列，序列從頭開始計算資料的插值，插值指向陣列的一個新位置兩者相互調換位置，將調換過的位置在標簽陣列相對應位置標記為真，遞增而行至序列結尾排序完成。因數列資料必定不重複以此方法排序可一步到位，調換次數為O(n)，計算時間複雜度為O(n)。

美國著名電腦科學家[高德納教授在演算法數學分析中指出](https://zh.wikipedia.org/wiki/高德納 "wikilink")，原地置換的時間效率本質上牽涉到尋找循環領頭的問題，假如我們允許操做額外的N個位元標簽，隨時指出已經進行了多少置換，可以輕易的達成執行O(n)次的原地排列。

## 類似排序方法

[Flash Sort](http://en.wikipedia.org/wiki/Flashsort)

[ProxmapSort](http://en.wikipedia.org/wiki/Proxmap_sort)

排序演算法若是由桶排序與插入排序混合，當數列出現乖離很大的數值例如數列最大值大於次大值的N倍時，數列分桶處理後分佈情形是除了最大值以外其餘元素都落入同一個桶子，緊接著第二個排序方法使用插入排序，會使得執行複雜度陷入O(n^2)，如此便降低了使用桶排序分散的效能與意義。插值排序遞歸方式第二次以後仍然使用分散方式可以避免上述情形，如欲使得遞歸的插值排序執行複雜度陷入O(n^2)，則必需在整個數列呈現階乘放大的狀況，相對上數列要出現這種特殊分佈情形的機率很少。

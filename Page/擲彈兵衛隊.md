top{{\!}}100px }} **擲彈兵衛隊**（[英语](../Page/英语.md "wikilink")：****）是[英國陸軍的一支步兵團](https://zh.wikipedia.org/wiki/英國陸軍 "wikilink")，於[英國御林軍](../Page/英國御林軍.md "wikilink")編制中，是[衛兵師](../Page/衛兵師.md "wikilink")歷史最悠久的團，早於1656年已經成立。

## 沿革列表

擲彈兵衛隊在300多年的歷史中，命名亦有所不同。\[1\]

| **日期**    | 全稱      |
| --------- | ------- |
| 1656-1665 | 溫特沃斯勳爵團 |
| 1665-1815 | 步兵衛隊第一團 |
| 1815至今    | 擲彈兵衛隊   |

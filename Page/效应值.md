在[統計學中](https://zh.wikipedia.org/wiki/統計學 "wikilink")，**效應值**（effect size，或譯**效果量**）是量化現象強度的數值。\[1\]效應值實際的統計量包括了二個變數間的[相關程度](https://zh.wikipedia.org/wiki/相關 "wikilink")、迴歸模型中的[迴歸係數](../Page/線性回歸.md "wikilink")、不同處理間[平均值的差異](https://zh.wikipedia.org/wiki/平均值 "wikilink")……等等。無論哪種效應值，其[絕對值越大表示效應越強](https://zh.wikipedia.org/wiki/絕對值 "wikilink")，也就是現象越明顯。效應值與[特效检验的概念是互補的](../Page/假設檢定.md "wikilink")。在估算[統計檢定力](https://zh.wikipedia.org/wiki/統計檢定力 "wikilink")、需要的與進行[元分析](../Page/元分析.md "wikilink")時，效應值經常扮演重要角色。

在研究結果中報導效應值被視為洽當的或必須的。\[2\]\[3\]相對於統計學上的[顯著性](https://zh.wikipedia.org/wiki/顯著性差異 "wikilink")，效應值有利於了解研究結果的強度。\[4\]特別是在[社會科學和](https://zh.wikipedia.org/wiki/社會科學 "wikilink")[醫學研究上](https://zh.wikipedia.org/wiki/醫學 "wikilink")，效應值更顯得重要。絕對與相對效應值可以傳遞不同的訊息，又可互相補充訊息。有個[心理學的研究學會鼓勵學者報導效應值](https://zh.wikipedia.org/wiki/心理學 "wikilink")：

在比較平均數的情況下，效應值經常指的就是實驗結束後，實驗組與[控制組之間](https://zh.wikipedia.org/wiki/控制組 "wikilink")「[標準化後的](https://zh.wikipedia.org/wiki/標準化 "wikilink")[平均差異程度](https://zh.wikipedia.org/wiki/平均 "wikilink")」，依照慣例，一些常用的effect size=可解讀為以下情況：

| Effect size | *d\[5\]* | *r\[6\]* |
| ----------- | -------- | -------- |
| 較小          | 0.2      | 0.10     |
| 中等          | 0.5      | 0.30     |
| 較大          | 0.8      | 0.50     |

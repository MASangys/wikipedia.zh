**敕書**今簡作**敕**（[異體字簡稱](https://zh.wikipedia.org/wiki/異體字 "wikilink")：**勅**\[1\]、**勑**\[2\]、**飭**\[3\]）是[君主制國家的一種](https://zh.wikipedia.org/wiki/君主制 "wikilink")[文件](https://zh.wikipedia.org/wiki/文件 "wikilink")。

## [中國](../Page/中國.md "wikilink")

敕書是[中國皇帝行文予朝臣的](https://zh.wikipedia.org/wiki/中國皇帝 "wikilink")[詔書](https://zh.wikipedia.org/wiki/詔書 "wikilink")\[4\]\[5\]（敕書是詔書的一種，敕書不等於詔書）。敕書在[南北朝起才專指皇帝寫給臣僚的詔書](https://zh.wikipedia.org/wiki/南北朝 "wikilink")\[6\]，其實「敕」在[漢朝至](https://zh.wikipedia.org/wiki/漢朝 "wikilink")[晉朝期間泛指長輩或官吏對後輩或](https://zh.wikipedia.org/wiki/晉朝 "wikilink")[幕僚的告誡](https://zh.wikipedia.org/wiki/幕僚 "wikilink")。

[明朝時](../Page/明朝.md "wikilink")，[皇帝有一種專門頒贈給](../Page/皇帝.md "wikilink")[女真人的](https://zh.wikipedia.org/wiki/女真人 "wikilink")「敕書」，是一種來經商的「貿易許可証」或者「通行証」，故成為女真人爭奪的重要物品。

## [英國](https://zh.wikipedia.org/wiki/英國 "wikilink")

「敕書」亦是[英文](https://zh.wikipedia.org/wiki/英文 "wikilink")「Command
paper」的[意譯](https://zh.wikipedia.org/wiki/意譯 "wikilink")。[英國政府](../Page/英國政府.md "wikilink")、[皇家委員會](https://zh.wikipedia.org/wiki/皇家委員會 "wikilink")（Royal
Commissions）等國家機構或公營機構提交予[英國國會的](https://zh.wikipedia.org/wiki/英國國會 "wikilink")[白皮書](https://zh.wikipedia.org/wiki/白皮書 "wikilink")、[綠皮書也算作敕書](https://zh.wikipedia.org/wiki/綠皮書 "wikilink")。雖然到國會報告的人未必是[英國君主](https://zh.wikipedia.org/wiki/英國君主 "wikilink")，但因為報告者的身分是「英國君主的代表」，故這種報告通稱「敕書」。英國的敕書在1870年起使用至今。在不同[年代](https://zh.wikipedia.org/wiki/年代 "wikilink")，敕令編號前的「Command」字[縮寫都不同](../Page/縮寫.md "wikilink")：

|       |             |
| ----- | ----------- |
| C.    | 1870年–1899年 |
| Cd.   | 1900年–1918年 |
| Cmd.  | 1919年–1956年 |
| Cmnd. | 1956年–1986年 |
| Cm.   | 1986年–今     |
|       |             |

敕令由The Stationery Office為The Office of Public Sector Information出版。

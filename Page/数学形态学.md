**数学形态学（Mathematical morphology）**
是一门建立在[格论和](https://zh.wikipedia.org/wiki/格论 "wikilink")[拓扑学基础之上的图像分析学科](../Page/拓扑学.md "wikilink")，是数学形态学[图像处理的基本理论](../Page/图像处理.md "wikilink")。其基本的运算包括：[腐蚀和](https://zh.wikipedia.org/wiki/腐蚀_\(形态学\) "wikilink")[膨胀](https://zh.wikipedia.org/wiki/膨胀_\(形态学\) "wikilink")、[开运算和](https://zh.wikipedia.org/wiki/开运算_\(形态学\) "wikilink")[闭运算](https://zh.wikipedia.org/wiki/闭运算_\(形态学\) "wikilink")、骨架抽取、极限腐蚀、击中击不中变换、形态学梯度、Top-hat变换、颗粒分析、流域变换等。

## 二值形态学

在二值形态学中，一个图案被看做是 \(n\) 维[欧几里得空间](../Page/欧几里得空间.md "wikilink")
\(\mathbb {R}^n\) 或网格
\(\mathbb {Z}^{n}\)的[子集](../Page/子集.md "wikilink")。

### 结构元素

在二值结构学中，**结构元素**为一个二值影像，作为分析影像时使用的「探针」，代表当处理影像上的某点时、要取出周围的哪些点进行运算。\[1\]

以下是几个常用的结构元素(将原图写作A、结构元素写作*B*)：

  - 待处理影像为二维类比影像 \(A\in E=\mathbb{R}^2\)，使用的结构元素*B*为一以原点为圆心、半径为*r*的圆盘。
  - 待处理影像为二维类比影像 \(A\in E=\mathbb{R}^2\)，使用的结构元素*B*为一以原点为中心的3x3方形。
  - 待处理影像为二维类比影像
    \(A\in E=\mathbb{R}^2\)，使用的结构元素*B*为一以原点为中心的十字形，或写作\(B=\{(-1,0),(0,-1),(0,0),(0,1),(1,0)\}\)。

### 基础运算子

二值形态学的基础运算子为具[平移对称性的](../Page/平移.md "wikilink")、与[闵可夫斯基和直接相关的运算子](https://zh.wikipedia.org/wiki/闵可夫斯基和 "wikilink")。基础运算子包含膨胀、腐蚀，以及由前两者组合而成的开运算、闭运算。

#### 膨胀

**膨胀(Dilation)**的定义为「位於某个点的探针(结构元素)是否*有*探测到物件？」一个影像*A*经过结构元素*B*膨胀後的结果可写为：\[2\]

\[A \oplus B = \{ x | B_x \cap A \ne \empty \}\].

其中\(B_x = \{ x+b | b \in B \}\)，代表结构元素平移x後的点集合，*b*是图像*B*的元素的座标。

另外也可写为：

\[A \oplus B = \bigcup_{b\in B} A_{-b}\].

同上，其中\(A_{-b}\)是指二值影像*A*经过平移*-b*後新的点集合。

#### 腐蚀

**腐蚀(Erosion)**的定义为「位於某个点的探针(结构元素)是否*全都有*探测到物件？」一个影像*A*经过结构元素*B*腐蚀後的结果可写为：\[3\]

\[A \ominus B = \{ x | B_x \subseteq A \} = \bigcap_{b\in B} A_{-b}\].

#### 开运算、闭运算

**开运算(Opening)**与**闭运算(Closing)**是使用相同结构函数的腐蚀与膨胀的组合：

开运算为先腐蚀再膨胀，

\[A \circ B  = (A \ominus B) \oplus B\].

闭运算为先膨胀再腐蚀

\[A \bullet B  = (A \oplus B) \ominus B\].

## 基础运算子的性质

  - 所有的运算子具有
  - 所有的运算子都是[递增的](../Page/单调函数.md "wikilink")，例：如果 \(A\subseteq C\)，则
    \(A\oplus B \subseteq C\oplus B\) 且
    \(A\ominus B \subseteq C\ominus B\)
  - 膨胀具有[交换律](../Page/交換律.md "wikilink")，例：\(A\oplus B = B\oplus A\)
  - 膨胀具有[结合律](../Page/结合律.md "wikilink")，例：\((A\oplus B)\oplus C = A\oplus (B\oplus C)\)；另外腐蚀则为
    \((A\ominus B)\ominus C = A\ominus (B\oplus C)\)
  - 如果B包含原点(0,0)，则有
    \(A\ominus B\subseteq A\circ B\subseteq A\subseteq A\bullet B\subseteq A\oplus B\)
  - 膨胀与腐蚀间的关系为：\(A \oplus B = (A^{c} \ominus B^{s})^{c}\)，上标\(^{c}\)代表[补集](../Page/补集.md "wikilink")，上标\(^{s}\)代表对原点的[点对称集合](https://zh.wikipedia.org/wiki/点反演 "wikilink")。
  - 开运算与闭运算间的关系为：\(A \bullet B = (A^{c} \circ B^{s})^{c}\)
  - 膨胀对[联集有](../Page/并集.md "wikilink")[分配律](../Page/分配律.md "wikilink")，例：\(A\oplus (B\cup C) = (A\oplus B)\cup (A\oplus C)\)；腐蚀对[交集有分配律](../Page/交集.md "wikilink")，例：\(A\ominus (B\cap C) = (A\ominus B)\cap (A\ominus C)\)
  - 膨胀与腐蚀为彼此的[广义逆运算](../Page/广义逆阵.md "wikilink")：\(A\subseteq (C\ominus B)\)
    若且为若 \((A\oplus B)\subseteq C\)
  - 开运算与闭运算是[幕等的](../Page/冪等.md "wikilink")：\((A\bullet B)\bullet B = A\bullet B\)

## 历史

数学形态学诞生于1964年，由当时[国立巴黎高等矿业学校的马瑟荣](../Page/国立巴黎高等矿业学校.md "wikilink")（G.
Matheron）和赛拉（J.
Serra）两人共同奠定了其理论基础。1968年4月法国[枫丹白露数学形态学研究中心成立](../Page/枫丹白露.md "wikilink")，巴黎矿业学院为中心提供了研究基地。

20世纪数学形态学的发展过程可大致分为：

  - 60年代的孕育和形成期
  - 70年代的充实和发展期
  - 80年代的成熟和对外开放期
  - 90年代至今的扩展期

**整数**，是[序列](../Page/序列.md "wikilink")\(\{\ldots, -4, -3, -2, -1, 0, 1, 2, 3, \ldots\}\)中所有的[数的统称](../Page/数.md "wikilink")，包括[负整数](../Page/负整数.md "wikilink")、[零](../Page/0.md "wikilink")（0）与[正整数](../Page/正整数.md "wikilink")。和[自然數一樣](../Page/自然數.md "wikilink")，整數也是一個[可數的](../Page/可數.md "wikilink")[無限集合](../Page/無限集合.md "wikilink")。這個[集合在数学上通常表示](../Page/集合_\(数学\).md "wikilink")[粗體](../Page/粗體.md "wikilink")\(Z\)或\(\mathbb{Z}\)，源于[德语单词](../Page/德语.md "wikilink")*Zahlen*（意为“[数](../Page/数.md "wikilink")”）的[首字母](../Page/首字母.md "wikilink")。

在[代數數論中](../Page/代數數論.md "wikilink")，這些屬於[有理數的一般](../Page/有理數.md "wikilink")[整數會被稱為](../Page/整數.md "wikilink")**有理整數**，用以和[高斯整數等的概念加以區分](../Page/高斯整數.md "wikilink")。

## 正整数与负整数

整數是一个[集合](../Page/集合_\(数学\).md "wikilink")，通常可以分为[正整數](../Page/正整數.md "wikilink")、[零](../Page/零.md "wikilink")（0）和[負整數](../Page/負整數.md "wikilink")。**正整數**（符号：**Z**<sup>+</sup>或\(\mathbb{Z}^{+}\)）即[大於](../Page/大於.md "wikilink")0的整數，是[正数与整数的](../Page/正数.md "wikilink")[交集](../Page/交集.md "wikilink")。而**負整數**（符号：\(Z^-\)或\(\mathbb{Z}^{-}\)）即[小於](../Page/小於.md "wikilink")0的整數，是[负数与整数的交集](../Page/负数.md "wikilink")。和整數一样，两者都是[可數的](../Page/可數.md "wikilink")[無限集合](../Page/無限集合.md "wikilink")。除正整數和負整數外，通常将0與正整數统称为**非負整數**（符号：**Z**<sup>+</sup><sub>0</sub>或\(\mathbb{Z}^{+}_{0}\)），而将0與負整數统称为**非正整數**（符号：**Z**<sup>-</sup><sub>0</sub>或\(\mathbb{Z}^{-}_{0}\)）。在[数论中](../Page/数论.md "wikilink")[自然数通常被视为与正整數等同](../Page/自然数.md "wikilink")，即1，2，3等，但在[集合论和](../Page/集合论.md "wikilink")[计算机科学中自然数则通常是指非负整数](../Page/计算机科学.md "wikilink")，即0，1，2等。

## [代数性质](../Page/代数.md "wikilink")

下表给出任何整数\(a, b, c\)的[加法和](../Page/加法.md "wikilink")[乘法的基本性质](../Page/乘法.md "wikilink")。

| 性質                                 | 加法                                             | 乘法                                                                                                                                                                                                                                     |
| ---------------------------------- | ---------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [封闭性](../Page/封闭性.md "wikilink")   | \(a + b\)是整数                                   | \(a \times b\)是整数                                                                                                                                                                                                                      |
| [结合律](../Page/结合律.md "wikilink")   | \(a + (b + c) = (a + b) + c\)                  | \(a \times (b \times c) = (a \times b) \times c\)                                                                                                                                                                                      |
| [交换律](../Page/交换律.md "wikilink")   | \(a + b = b + a\)                              | \(a \times b = b \times a\)                                                                                                                                                                                                            |
| 存在[单位元](../Page/单位元.md "wikilink") | \(a + 0 = a\)                                  | \(a \times 1 = a\)                                                                                                                                                                                                                     |
| 存在[逆元](../Page/逆元.md "wikilink")   | \(a + (-a) = 0\)                               | 在[整数集中](../Page/整数集.md "wikilink")，只有[1或](../Page/1.md "wikilink")[-1对于](../Page/-1.md "wikilink")[乘法存在整数](../Page/乘法.md "wikilink")[逆元](../Page/逆元.md "wikilink")，其余整数\(a\)关于乘法的[逆元为](../Page/逆元.md "wikilink")\(\frac{1}{a}\)，都不为整数。 |
| [分配律](../Page/分配律.md "wikilink")   | \(a \times (b + c) = a \times b + a \times c\) |                                                                                                                                                                                                                                        |

全体**整数**关于[加法和](../Page/加法.md "wikilink")[乘法形成一个环](../Page/乘法.md "wikilink")。[环论中的](../Page/环论.md "wikilink")[整环](../Page/整环.md "wikilink")、[无零因子环和](../Page/无零因子环.md "wikilink")[唯一分解域可以看作是整数的](../Page/唯一分解域.md "wikilink")[抽象化模型](../Page/抽象化.md "wikilink")。

**\(\mathbb{Z}\)**是一个加法[循环群](../Page/循环群.md "wikilink")，因为任何整数都是若干个1或-1的和。1和-1是**\(\mathbb{Z}\)**仅有的两个生成元。每个元素个数为无穷个的循环群都与\((\mathbb{Z},+)\)[同构](../Page/同构.md "wikilink")。

## 有序性质

**Z**是一个[全序集](../Page/全序集.md "wikilink")，没有上界和下界。\(\mathbb{Z}\)的序列如下：

\[\ldots < -2 < -1 < 0 < 1 < 2 < \ldots\]

一个整数大于零则为正，小于零则为负。零既非正也非负。

整数的序列在代数运算下是可以比较的，表示如下：

1.  若\(a < b\)且\(c < d\)，则\(a + c < b + d\)
2.  若\(a < b\)且\(c > 0\)，则\(a * c < b * c\)；若\(c < 0\)，则\(a * c > b * c\).

整数环是一个[欧几里德域](../Page/欧几里德域.md "wikilink")。

## \(\mathbb{Z}\)的基數

\(\mathbb{Z}\)的[基數](../Page/基数_\(数学\).md "wikilink")（或勢）是[ℵ](../Page/阿列夫数.md "wikilink")<sub>0</sub>，與\(\mathbb{N}\)相同。這可以從**\(\mathbb{Z}\)**建立一[雙射函數到](../Page/雙射函數.md "wikilink")**\(\mathbb{N}\)**來證明，亦即該函數要同時滿足[單射及](../Page/單射.md "wikilink")[滿射的條件](../Page/滿射.md "wikilink")，例如：

\[f(x) = \begin{cases} 2x + 1, & \mbox{if } x \ge 0 \\ 2|x|, & \mbox{if } x < 0 \end{cases}\]

當該函數的[定義域僅限於](../Page/定義域.md "wikilink")**\(\mathbb{Z}\)**，則證明**\(\mathbb{Z}\)**與**\(\mathbb{N}\)**可建立一一對應的關係，即兩集[等勢](../Page/等勢.md "wikilink")。

  - [Secure Shell File System](https://zh.wikipedia.org/wiki/Secure_Shell_File_System "wikilink") (SSHFS) - 在一台服务器上只使用一个[Secure Shell登录来本地挂载一个远程目录](../Page/Secure_Shell.md "wikilink")。

  - ，用户空间中的GPL [加密文件系统](../Page/加密文件系统.md "wikilink")

  -
  - Solid File System - （SolFS）跨平台单文件虚拟文件系统，具有加密和压缩功能

  - [EFS](https://zh.wikipedia.org/wiki/EFS "wikilink") – Encrypted File System。面向[Microsoft Windows和](https://zh.wikipedia.org/wiki/Microsoft_Windows "wikilink")[IBM AIX](../Page/IBM_AIX.md "wikilink")。一个[NTFS](../Page/NTFS.md "wikilink")的扩展

  - FSFS - Fast Secure File System

  - [ZFS](../Page/ZFS.md "wikilink")的加密支持

  - TeaSafe。一个基于容器的加密文件系统，支持AES、Serpent和Twofish密码本。

### 文件系统接口

这些不是真的文件文件系统；它们允许从操作系统角度访问文件系统。

  - [FUSE](../Page/FUSE.md "wikilink")（用户空间中的文件系统，类似，但更好维护）
  - LUFS（Linux用户级文件系统 - 似乎在[FUSE](../Page/FUSE.md "wikilink")之后被放弃）
  - VFS虚拟文件系统
  - Callback File System - 该SDK建立自定义的文件系统和插入到Windows操作系统。

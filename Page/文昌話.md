**文昌话**是[汉藏语系](../Page/汉藏语系.md "wikilink")[汉语族](../Page/汉语族.md "wikilink")[闽语支](https://zh.wikipedia.org/wiki/闽语支 "wikilink")[海南话](../Page/海南话.md "wikilink")的一种方言，也是[文昌片的代表方言](https://zh.wikipedia.org/wiki/文昌片 "wikilink")。通行于[海南省](../Page/海南省.md "wikilink")[文昌市](../Page/文昌市.md "wikilink")。

文昌话是海南语中历史最悠久的一种方言。海南话电视广播多以文昌话为播音的标准方言，东南亚的海南移民中，使用文昌话的人数也是最多的。

## 历史

宋代末期，由于战乱，大量[闽南人南下避乱](https://zh.wikipedia.org/wiki/闽南人 "wikilink")。这些闽南人先到达[潮汕平原](https://zh.wikipedia.org/wiki/潮汕平原 "wikilink")，后来跨越[珠江三角洲](../Page/珠江三角洲.md "wikilink")，进入[雷州半岛](../Page/雷州半岛.md "wikilink")和[海南岛](../Page/海南岛.md "wikilink")，将[闽南语传播到这些地区来](https://zh.wikipedia.org/wiki/闽南语 "wikilink")。此后，由于长期受到周边少数民族语言以及[中原官话](../Page/中原官话.md "wikilink")的影响，海南话的音韵体系与闽南语产生了很大差异，以至于无法互通，形成一种独特的语言。后来，文昌话因为语音清晰且影响最大，被当作海南话的标准方言，在海南的电台、电视台和社交场合广泛使用。\[1\]

在文昌话中，平、上、去、入四声都可以区分阴阳，古汉语的8个声调被完整的保留了下来。这一点，不论是在闽语还是其他汉语方言中都是不多见的。

## 音韵体系

文昌话有声母18个、韵母45个、声调8个。

## 连读变调

连读变调，即两个音节连读的时候，前字的声调受后字声调的影响而发生变化。与闽南语一样，海南语文昌话的连读变调规律比较整齐。不过在阴平、阳平、阴上、阳上、阴去和阳入作为前字时，有时候也不变调，一时看不准分化的条件。

文昌话的连读变调规律如下：

|          |           |           |           |           |           |           |                  |                  |
| -------- | --------- | --------- | --------- | --------- | --------- | --------- | ---------------- | ---------------- |
| **前字調類** | **陰平** 24 | **陽平** 22 | **陰上** 21 | **陽上** 42 | **陰去** 11 | **陽去** 51 | **陰入** <u>55</u> | **陽入** <u>33</u> |
| **前字變調** | 陽上 42     | 陰上 21     | 陰入 55     | 陰上 21     | 陰入 55     | 陽曲 51     | 陽入 55            | 陰上 21            |
| **例詞**   | 兄弟        | 和尚        | 廣東        | 重點        | 過年        | 獎金        | 國家               | 日光               |

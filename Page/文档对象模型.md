{{ Otheruses|DOM (消歧义)}}    **文件物件模型**（，缩写****），是[W3C组织推薦的处理](https://zh.wikipedia.org/wiki/W3C "wikilink")[可扩展置标语言的标准](https://zh.wikipedia.org/wiki/可扩展置标语言 "wikilink")-{zh-hant:程式;zh-cn:编程}-接口。

Document Object Model的歷史可以追溯至1990年代後期微軟與[Netscape的](https://zh.wikipedia.org/wiki/Netscape "wikilink")“[瀏覽器大戰](https://zh.wikipedia.org/wiki/瀏覽器大戰 "wikilink")”（browser wars），雙方為了在[JavaScript](../Page/JavaScript.md "wikilink")與[JScript](../Page/JScript.md "wikilink")一決生死，於是大規模的賦予瀏覽器強大的功能。微軟在網頁技術上加入了不少專屬事物，既有[VBScript](../Page/VBScript.md "wikilink")、[ActiveX](../Page/ActiveX.md "wikilink")、以及微軟自家的[DHTML格式等](https://zh.wikipedia.org/wiki/DHTML "wikilink")，使不少網頁使用非微軟平台及瀏覽器無法正常顯示。DOM即是當時醞釀出來的傑作。

## 標準化

W3C對DOM進行標準化的動作，目前已經推行至第四代。

### Level 1

  - [Document Object Model (DOM) Level 1 Specification](http://www.w3.org/TR/REC-DOM-Level-1/)
      - 支援[XML](../Page/XML.md "wikilink") 1.0

### Level 2

  - [Document Object Model (DOM) Level 2 Core Specification](http://www.w3.org/TR/DOM-Level-2-Core/)
      - XML 1.0與命名空間
  - [Document Object Model (DOM) Level 2 HTML Specification](http://www.w3.org/TR/DOM-Level-2-HTML/)
      - HTML 4.0x、XHTML 1.0
  - [Document Object Model (DOM) Level 2 Views Specification](http://www.w3.org/TR/DOM-Level-2-Views/)
  - [Document Object Model (DOM) Level 2 Style Specification](http://www.w3.org/TR/DOM-Level-2-Style/)
      - 支援CSS與CSS Level2
  - [Document Object Model (DOM) Level 2 Events Specification](http://www.w3.org/TR/DOM-Level-2-Events/)
      - 支援HTML 4.01
  - [Document Object Model (DOM) Level 2 Traversal and Range Specification](http://www.w3.org/TR/DOM-Level-2-Traversal-Range/)

### Level 3

  - [Document Object Model (DOM) Level 3 Core Specification](http://www.w3.org/TR/DOM-Level-3-Core/)
  - [Document Object Model (DOM) Level 3 Load and Save Specification](http://www.w3.org/TR/DOM-Level-3-LS/)
  - [Document Object Model (DOM) Level 3 Validation Specification](http://www.w3.org/TR/DOM-Level-3-Val/)

### Level 4

  - [Document Object Model (DOM) Level 4](https://www.w3.org/TR/dom/)

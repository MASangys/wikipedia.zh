**斌道**（，）是緬甸防身空手道的一個門派\[1\]，斌道有時被誤以為是所有緬甸武術的統稱，但實際上它只是一個武術流派，而緬甸所有的武術被統稱。

## 訓練

與大多數相同，所有教導斌道的學校都從基本的姿態和步法開始。這種訓練的初期階段在以前會持續好幾個月，但現今許多教師都不這樣做。在培訓的第二階段，將會教授一系列的壓制技巧。\[2\]

## 技巧

| 式    | 特徵           |
| ---- | ------------ |
| 豬式   | 迅猛攻擊，重視肘擊、膝擊 |
| 牛式   | 衝擊與阻截        |
| 鹿式   | 騰挪身法         |
| 鷹式   | 雙手攻防法        |
| 猴式   | 靈活身法         |
| 鷺式   | 迅速的手部移動、短程跳躍 |
| 豹式   | 繞敵移動、撲擊對手    |
| 蟒式   | 擒拿鎖喉         |
| 蠍式   | 擒拿神經         |
| 虎式   | 爪擊           |
| 眼鏡蛇式 | 攻擊對手上身要害     |
| 蝰蛇式  | 攻擊對手下身要害     |
| 黑豹式  | 組合技          |

**斯普林特约定叫**（Splinter bid）是[桥牌中的一种约定叫](https://zh.wikipedia.org/wiki/桥牌 "wikilink")，也被称为“爆裂叫”。指通过不寻常的跳叫新花色来表示对同伴的花色有很好的支持（一般是4张以上），且在己方所叫花色上有单张或缺门。该叫品是逼叫进局的，并表示了满贯兴趣。\[1\] 斯普林特要求的点力范围与所采用的叫牌体系有关，通常为13-16点。\[2\]

## 示例

在下面的例子中，南家的4表示4张以上黑桃支持，且草花单张或缺门，逼叫。

| 北 | 南 |
| - | - |
| 1 | 4 |

下面的叫牌进程里，南家的3也是斯普林特。表示同意红桃为将牌，且黑桃单缺。

| 北 | 南 |
| - | - |
| 1 | 3 |

对方未参与叫牌的情况下，对于开叫人或应叫人的再叫叫品也可以应用斯普林特。基本原则是：如果某一新花色叫品是逼叫，那么比此叫品高一阶的同一花色叫品是斯普林特。\[3\] 如下例所示：

| 北 | 南 |
| - | - |
| 1 | 1 |
| 4 |   |

南家4是对黑桃的斯普林特。

当持有非均型牌，但点力超出了斯普林特约定叫的上限时（例如17点以上），可以用[雅可比2NT](../Page/雅可比2NT.md "wikilink")回应同伴的1/1开叫。

另外，在竞叫中，如果对方的叫品是[技术性加倍](https://zh.wikipedia.org/wiki/技术性加倍 "wikilink")，斯普林特仍然有效。\[4\]

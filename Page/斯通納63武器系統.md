**Stoner 63**或**M63**是由當時在卡迪拉克·盖奇公司工作[尤金·斯通纳設計](../Page/尤金·M·斯通纳.md "wikilink")、詹姆斯·沙利文（L. James Sullivan）以发射[7.62×51mm NATO枪弹的Stoner](../Page/7.62×51mm_NATO.md "wikilink") 62縮小口径改造而成发射[5.56×45mm NATO枪弹的模組化系列武器系統](../Page/5.56×45mm_NATO.md "wikilink")，由[卡迪拉克·盖奇公司在](https://zh.wikipedia.org/wiki/卡迪拉克·盖奇公司 "wikilink")1963年起生產，在[越南戰爭中是](https://zh.wikipedia.org/wiki/越南戰爭 "wikilink")[美國海軍海豹部隊使用的武器之一](https://zh.wikipedia.org/wiki/美國海軍海豹部隊 "wikilink")，包括著名的**Mk 23 Mod 0**輕機槍。

## 設計及歷史

1960年尤金·斯通納在阿玛莱特公司（斯通納在阿玛莱特设计了[AR-10](../Page/AR-10自動步槍.md "wikilink")）改组时离开，后致力研究“[模块化枪械](https://zh.wikipedia.org/wiki/模块化 "wikilink")”的概念，此概念被[卡迪拉克·盖奇公司看中](https://zh.wikipedia.org/wiki/卡迪拉克·盖奇公司 "wikilink")，并提供给位于加州的研发机构。1962年由[尤金·斯通納設計发射](https://zh.wikipedia.org/wiki/尤金·斯通納 "wikilink")[7.62×51mm NATO枪弹的斯通纳](../Page/7.62×51mm_NATO.md "wikilink")62（Stoner 62）武器系统推出，包括步枪、突击步枪，轻机枪。此时美军已开始采纳小口径的5.56×45毫米枪弹，发射5.56×45毫米枪弹的斯通纳63（Stoner 63）武器系統在1963年推出，一直生產至1971年停产为止。 [Stoner63-patent.jpg](https://zh.wikipedia.org/wiki/File:Stoner63-patent.jpg "fig:Stoner63-patent.jpg") Stoner 63有多種型號，衍生型包括[卡賓槍](https://zh.wikipedia.org/wiki/卡賓槍 "wikilink")、[突擊步槍及可以](https://zh.wikipedia.org/wiki/突擊步槍 "wikilink")[彈鏈](../Page/彈鏈.md "wikilink")從左或右供彈的[輕機槍](https://zh.wikipedia.org/wiki/輕機槍 "wikilink")，有15種部件可換，當中小部份可通用，枪身机匣通用是“模块化”设计构想的体现，氣動系統可根據其武器型號來改變設定位置。采用活塞长行程式导气方式，枪机回转式闭锁，在机枪的导气箍有气体调节器。Stoner 63的槍管可快速更換，亦可在短時間內轉換至不同型號，具有良好的可靠性和通用性，尤其是在潮濕悶熱的越南叢林仍可有效地運作，在[海豹部隊](../Page/海豹部隊.md "wikilink")的使用下更成為高效能作戰武器。

Stoner 63的特製[刺刀](../Page/刺刀.md "wikilink")名為KCB70，由德國Eickhorn-Solingen製造，設計靈感來自[AK-47的刺刀](https://zh.wikipedia.org/wiki/AK-47突击步枪 "wikilink")，具有7寸長鋸齒刀身和鋼絲剪，比美軍的[M7刺刀](../Page/M7刺刀.md "wikilink")更為高級，KCB70後來改進成[KCB77並成為多種突擊步槍的專用刺刀](../Page/KCB77刺刀.md "wikilink")\[1\]。

1963年，尤金·斯通納把Stoner 63送至[美國海軍陸戰隊作武器評估](https://zh.wikipedia.org/wiki/美國海軍陸戰隊 "wikilink")，海軍陸戰隊試驗了六個月後提供了一些建議以改進設計，包括改用[不鏽鋼](../Page/不鏽鋼.md "wikilink")導氣管、兩段式射擊模式選擇鈕、獨立式安全制、拋殼口和改進[彈鏈](../Page/彈鏈.md "wikilink")供彈系統，改良後的版本名為斯通纳63A（Stoner 63A）並於1966年投產\[2\]。 [Vietnam_p05.jpg](https://zh.wikipedia.org/wiki/File:Vietnam_p05.jpg "fig:Vietnam_p05.jpg") 直至1967年，Stoner 63A被海軍陸戰隊第1師第1團第3營L連（Lima Company）作實戰測試，當中步槍手採用步槍型，軍官和某些士兵採用卡賓槍型，機槍手採用自動步槍型，而L連則同時採用輕機槍型及中型機槍型作試驗\[3\]。而美国海军也决定为“海豹”突击队配备斯通纳63。1970年，[美國陸軍將突擊型命名為XM](https://zh.wikipedia.org/wiki/美國陸軍 "wikilink")207並裝備部份美國陸軍特種部隊作試驗，但因其設計複雜且維修較難（相對於[M16](../Page/M16突擊步槍.md "wikilink")），美軍在1971年終止了測試，但[美國海軍](../Page/美國海軍.md "wikilink")的船隻上仍然可見，美國海軍把這批Stoner 63/63A突擊型轻机枪命名為Mk 23 Mod 0。直至1980年代Stoner 63系列被[M249班用機槍全面取代](../Page/M249班用自動武器.md "wikilink")。至1971年停产为止，當時共生產了約4000支Stoner 63和Stoner 63A，除役的斯通纳63武器系统大部份被消毀。因现存的斯通纳63武器系统数量很少，在民用市场上受到收藏家的欢迎。

罗宾逊军备公司（Robinson Armament）後來亦以Stoner 63為基礎開發出[半自動的](https://zh.wikipedia.org/wiki/半自動步槍 "wikilink")[Robinson M96](https://zh.wikipedia.org/wiki/Robinson_M96半自動步槍 "wikilink")。KAC（Knight's Armament Company）也以Stoner 63的設計開發出[斯通納輕機槍](../Page/斯通納輕機槍.md "wikilink")（Stoner LMG），Stoner 63機槍型所使用的[5.56×45 NATO可散式彈鏈在改進後成為了](../Page/5.56×45mm_NATO.md "wikilink")[M27彈鏈](../Page/M27彈鏈.md "wikilink")，亦即是現代[美軍和](https://zh.wikipedia.org/wiki/美軍 "wikilink")[北約國家通用的輕機槍彈鏈](https://zh.wikipedia.org/wiki/北約 "wikilink")。

## 衍生型

**Stoner 63/63A步槍**

  -
    標準突擊步槍型，採用30發彈匣供彈，[機匣右邊拋殼](https://zh.wikipedia.org/wiki/機匣 "wikilink")，拉機抦及導氣管位於槍管頂部，回转設計，A型配有可摺疊式[兩腳架](https://zh.wikipedia.org/wiki/兩腳架 "wikilink")。美國海軍陸戰隊在1967年曾經試用。美国陆军曾命名为XM22(63)/XM22E1(63A)进行测试，最终没有采用。

[National_Firearms_Museum_rifle.jpg](https://zh.wikipedia.org/wiki/File:National_Firearms_Museum_rifle.jpg "fig:National_Firearms_Museum_rifle.jpg") **Stoner 63/63A卡賓槍**

  -
    卡賓槍型類似步槍型，但改用短槍管及摺疊[槍托](../Page/槍托.md "wikilink")，海軍陸戰隊在1967年曾經試用。美国陆军曾命名为XM207进行测试，最终没有采用。

**Stoner 63/63A自動步槍**

  -
    自動步槍型採用閉鎖式槍機設計，配30發彈匣，[機匣頂部供彈](https://zh.wikipedia.org/wiki/機匣 "wikilink")，準星和照門因要避開位于机匣顶部的彈匣改為靠左，只能全自动射击沒有半自動模式，海軍陸戰隊在1967年曾經試用。

**Stoner 63/63A輕機槍**

  -
    輕機槍型採用开膛待机式設計，配100發彈鏈及塑料，[機匣右邊供彈](https://zh.wikipedia.org/wiki/機匣 "wikilink")，左邊拋殼，可快速更換式槍管，導氣管位於槍管下方（与步枪型相反），海軍陸戰隊在1967年曾經試用。

**Stoner 63/63A中型機槍**

  -
    中型機槍型比輕機槍型更大型，但內部設計與輕機槍相同，配有接合器以安將在[M2/M122三腳架](../Page/M2三腳架.md "wikilink")，海軍陸戰隊在1967年曾經試用。

**Stoner 63/63A固定式機槍**

  -
    內部設計與輕機槍相同，移除前後照門、護木、握把，扳機改由24伏特電動開關搖控，原為凱迪拉克Commando裝甲運兵車設計，但從來沒有正式裝備。

**Stoner 63/63A突擊型**

  -
    突擊型衍生自輕機槍型，配100發彈鼓下掛於[機匣底部](https://zh.wikipedia.org/wiki/機匣 "wikilink")，機匣右邊供彈，拉機抦內置於護木底部，為了減輕重量取消了可換式槍管，[越戰時期曾被](https://zh.wikipedia.org/wiki/越戰 "wikilink")[海豹部隊](../Page/海豹部隊.md "wikilink")採用。

**Stoner 63救生步槍**

  -
    救生步槍型在1964年完成設計，開發目的是為飛行員及機組人員提供一種救生自衛武器（類似[柯爾特M608](https://zh.wikipedia.org/wiki/CAR-15#CAR-15救生步槍 "wikilink")），救生步槍型內部類似步槍型，但在尺寸作了修改以符合[美國空軍要求](https://zh.wikipedia.org/wiki/美國空軍 "wikilink")，包括切短的握把、移除護木、改用短槍管、頂置式拉機抦，救生步槍型只制造過一把，沒有推出過63A的改良版本，而這把救生步槍至今仍然倖存\[4\]。

## 美軍命名

除救生步槍型外，大部份獲美軍試用的Stoner 63衍生型皆有其命名，見下表：

| 美國陸軍命名  | 美國海軍命名      | 型號              |
| ------- | ----------- | --------------- |
| XM22    | 沒有          | Stoner 63，步槍型   |
| XM22E1  | 沒有          | Stoner 63A，步槍型  |
| XM23    | 沒有          | Stoner 63，卡賓槍型  |
| XM23E1  | 沒有          | Stoner 63A，卡賓槍型 |
| XM207   | 沒有          | Stoner 63，輕機槍型  |
| XM207E1 | 沒有          | Stoner 63A，輕機槍型 |
| 沒有      | Mk 23 Mod 0 | Stoner 63A，突擊型  |

**新北市立樹林高級中學**（），簡稱**樹林高中**為[臺灣](../Page/臺灣.md "wikilink")[新北市最早的三所市立](../Page/新北市.md "wikilink")[完全中學之一](../Page/完全中學.md "wikilink")（另外兩所為[明德中學](../Page/新北市立明德高級中學.md "wikilink")、[永平中學](../Page/新北市立永平高級中學.md "wikilink")）。

## 沿革

| 年     | 月   | 事跡                                                             |
| ----- | --- | -------------------------------------------------------------- |
| 1944年 |     | 設**鶯歌女子農業實踐學校**                                                |
| 1946年 | 3月  | 改制為**鶯歌初級中學**                                                  |
| 1946年 | 8月  | 因鄉鎮之劃分更名為**樹林初級中學**                                            |
| 1948年 |     | 增設高中部                                                          |
| 1965年 | 8月  | 增設板橋分部                                                         |
| 1966年 | 8月  | 板橋分部獨立為**板橋初級中學**（今日之[板橋國中](../Page/新北市立板橋國民中學.md "wikilink")） |
| 1968年 | 8月  | 改制為**樹林國民中學**                                                  |
| 1975年 |     | 設立星隆分部於柑園                                                      |
| 1979年 | 8月  | 星隆分部獨立為[柑園國民中學](../Page/新北市立柑園國民中學.md "wikilink")              |
| 1983年 | 8月  | 增設國民中學補校                                                       |
| 1995年 | 8月  | 改制為**台北縣立樹林完全中學**                                              |
| 1999年 | 8月  | 增設高中部補校                                                        |
| 2000年 | 8月  | 因納入《高級中學法》之規範而更名為**台北縣立樹林高級中學**                                |
| 2010年 | 12月 | 因臺北縣升格為新北市而更名為**新北市立樹林高級中學**                                   |

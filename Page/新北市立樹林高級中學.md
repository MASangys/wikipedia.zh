[Shu-Lin_Senior_High_School20170102.jpg](https://zh.wikipedia.org/wiki/File:Shu-Lin_Senior_High_School20170102.jpg "fig:Shu-Lin_Senior_High_School20170102.jpg") **新北市立樹林高級中學**（），簡稱**樹林高中**，位於[臺灣](../Page/臺灣.md "wikilink")[新北市](https://zh.wikipedia.org/wiki/新北市 "wikilink")[樹林區](../Page/樹林區.md "wikilink")，為[新北市最早的三所市立](https://zh.wikipedia.org/wiki/新北市 "wikilink")[完全中學](../Page/完全中學.md "wikilink")之一（另外兩所為[明德中學](../Page/新北市立明德高級中學.md "wikilink")、[永平中學](../Page/新北市立永平高級中學.md "wikilink")），也是樹林區目前唯一一所公立[高級中學](../Page/高級中學.md "wikilink")。

## 沿革

| 年     | 月   | 事跡                                                                              |
| ----- | --- | ------------------------------------------------------------------------------- |
| 1944年 |     | 設**鶯歌女子農業實踐學校**                                                                 |
| 1946年 | 3月  | 改制為**臺北縣立鶯歌初級中學**                                                               |
| 1946年 | 8月  | 因鄉鎮之劃分，改名為**臺北縣立樹林初級中學**                                                        |
| 1948年 |     | 增設高中部，改名為**臺北縣立樹林中學**                                                           |
| 1965年 | 8月  | 增設板橋分部                                                                          |
| 1966年 | 8月  | 板橋分部獨立為[臺北縣立板橋初級中學](https://zh.wikipedia.org/wiki/新北市立板橋國民中學 "wikilink")        |
| 1968年 | 8月  | [九年國教實施](https://zh.wikipedia.org/wiki/九年國教 "wikilink")，高中部裁撤，改名為**臺北縣立樹林國民中學** |
| 1975年 |     | 設立星隆分部                                                                          |
| 1979年 | 8月  | 星隆分部獨立為[臺北縣立柑園國民中學](https://zh.wikipedia.org/wiki/新北市立柑園國民中學 "wikilink")        |
| 1983年 | 8月  | 增設國民中學補習學校                                                                      |
| 1995年 | 8月  | 高中部恢復招生，改制為**臺北縣立樹林完全中學**                                                       |
| 1999年 | 8月  | 增設補校高中部                                                                         |
| 2000年 | 8月  | 改名為**臺北縣立樹林高級中學**                                                               |
| 2010年 | 12月 | 臺北縣升格為新北市，改隸為**新北市立樹林高級中學**                                                     |

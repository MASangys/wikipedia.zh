**新北市鶯歌區鶯歌國民小學**，簡稱鶯歌國小，是一所位於[新北市](https://zh.wikipedia.org/wiki/新北市 "wikilink")[鶯歌區](../Page/鶯歌區.md "wikilink")的市立國民小學，該校創立於[日治時期](https://zh.wikipedia.org/wiki/日治時期 "wikilink")[明治](../Page/明治.md "wikilink")44年12月25日\[1\]，並有附設幼兒園。

## 校史

| 日期          | 說明                     |
| ----------- | ---------------------- |
| 明治40年12月25日 | 創立『三角湧公學校』(即現三峽國小)尖山分校 |
| 明治44年4月1日   | 改稱『尖山公學校』              |
| 昭和16年4月1日   | 尖山公學校改稱『鶯歌國民學校』        |
| 民國34年8月25日  | 第一任校長王茶坡到任             |
| 民國57年8月1日   | 改稱為台北縣鶯歌鎮鶯歌國民小學        |
| 民國95年12月25日 | 正式邁入創校一百年              |

《**新父子時代**》是[香港無綫電視](https://zh.wikipedia.org/wiki/香港無綫電視 "wikilink")1994年拍攝的電視連續劇，全劇共20集，監製[張乾文](https://zh.wikipedia.org/wiki/張乾文 "wikilink")。主要演員有[莫少聰](../Page/莫少聰.md "wikilink")、[鄭家麟](https://zh.wikipedia.org/wiki/鄭家麟 "wikilink")、[梁小冰](../Page/梁小冰.md "wikilink")、[張國強](https://zh.wikipedia.org/wiki/張國強_\(香港\) "wikilink")、[何婉盈](../Page/何婉盈.md "wikilink")、[黃日華](../Page/黃日華.md "wikilink")等。主題曲由[莫少聰](../Page/莫少聰.md "wikilink")主唱。

本劇於2011年6月12日在[TVB星河頻道](../Page/TVB星河頻道.md "wikilink")重播。

## 故事大綱

李世民 ( [莫少聰](../Page/莫少聰.md "wikilink")飾 ) 與吳國生 ( [黃日華](../Page/黃日華.md "wikilink")飾 ) 為貨車司機，二人感情深厚。身為單親父親的國生後來意外喪生，臨終將獨生兒子吳向星 ( 鄭家麟飾 ) 交託給世民，世民基於情義及責任，毅然承諾撫養向星。惟向星生性反叛，令世民無所適從，世民妻郭可儀 ( [何婉盈](../Page/何婉盈.md "wikilink")飾 ) 亦因向星關係，與世民的感情更趨破裂，終至離婚收場。而向星眼見世民為了自己而陷入困境，內心歉疚，二人在危難中互相扶持，建立親如父子的感情。

正當兩人感情融洽之際，向星之生母葉芳 ( [梁小冰](../Page/梁小冰.md "wikilink")飾 ) 突然回港，欲取回向星之撫養權，更不惜揭露了向星的身世，世民至此才明白葉芳所受的委屈及多年的痛苦，只好犧牲自己與向星的濃厚感情，勸向星重歸生母葉芳的懷抱。

世民在不知不覺中對葉芳產生了微妙的感情，可是葉芳在美國已嫁作人婦，其夫楊文禮 ( [張國強飾](https://zh.wikipedia.org/wiki/張國強 "wikilink") ) 更從美追至。葉芳處於情義兩難間，不知如何取捨；葉芳、世民與向星三人之間複雜的情感又該如何分解…？

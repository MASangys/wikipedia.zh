**旋转温度**（）是[统计力学](../Page/统计力学.md "wikilink")术语，用以简化[转动配分函数以及描述旋转对](../Page/配分函数.md "wikilink")[分子](../Page/分子.md "wikilink")的[热力学](../Page/热力学.md "wikilink")性能的贡献。它具有[温度](../Page/温度.md "wikilink")的[量纲](../Page/量纲.md "wikilink")并被定义为\[1\]

\(\theta _R=\frac{hc \bar B}{k_{B}}=\frac{\hbar ^{2}}{2k_{B}I}\),

上式中\(\bar B = B/hc\)是旋转常数，\(I\)是分子的[转动惯量](https://zh.wikipedia.org/wiki/转动惯量 "wikilink")，\(h\)是[普朗克常数](../Page/普朗克常数.md "wikilink")，\(c\)是[光速](../Page/光速.md "wikilink"), \(\hbar=h/2\pi\)是[约化普朗克常数](https://zh.wikipedia.org/wiki/约化普朗克常数 "wikilink")，\(k_B\) 是[玻尔兹曼常数](../Page/波茲曼常數.md "wikilink")。

转动温度的物理意义是用于估计在什么温度下，其[热能](../Page/热能.md "wikilink")（\(E=k_B T\)）和分子转动能级间隔相当（E=hcB）。在该温度下受激分子数占据主要地位。一些典型的值在下面的表格中给出。分子中的[元素均是最常见的](https://zh.wikipedia.org/wiki/元素 "wikilink")[同位素](../Page/同位素.md "wikilink")。

| 分子                                   | \(\) (K)\[2\] |
| ------------------------------------ | ------------- |
| H<sub>2</sub>                        | 87.6          |
| N<sub>2</sub>                        | 2.88          |
| O<sub>2</sub>                        | 2.08          |
| F<sub>2</sub>                        | 1.27          |
| HF                                   | 30.2          |
| HCl                                  | 15.2          |
| <sub><font size="3">CO</font>2</sub> | 0.561\[3\]    |
| HBr                                  | 12.2\[4\]     |
| CO                                   | 2.78\[5\]     |

## 概要

与其他的[传真一样](https://zh.wikipedia.org/wiki/传真 "wikilink")，无线电传真一般只能传输黑白图像。无线电传真通常使用[幅度调制](https://zh.wikipedia.org/wiki/幅度调制 "wikilink")（AM）以及[单边带](https://zh.wikipedia.org/wiki/单边带 "wikilink")（SSB）模式（通常为上边带（USB）模式）、通过[短波](../Page/短波.md "wikilink")波段（3-30MHz）传输。需要注意的是，接收机在接收无线电传真信号时，实际频率与标示频率有一定偏差（例如使用USB模式接收时，实际频率应低于播出频率1.9kHz）。在USB模式下，无线电传真的黑白二色基带信号分别调制到1500Hz和2300Hz（下边带（LSB）模式下黑白完全相反）；发送速率一般为120线/分钟（lines per minute，简称LPM），也有机构使用60、100、180、240LPM的速率发射传真信号。无线电传真图像的分辨率由“协调因子”（index of cooperation，简称IOC）决定，通常为IOC576和IOC288两种，每行的像素数为IOC与[π](https://zh.wikipedia.org/wiki/π "wikilink")（[圆周率](https://zh.wikipedia.org/wiki/圆周率 "wikilink")）的乘积，例如IOC576的图像，每行像素为576×π，约等于1810像素/行。

由于无线电传真占用带宽小，传输时间由传输图像大小和传输速度决定，传输一副完整的图像通常需要数分钟到数十分钟。目前无线电传真通常使用“”（APT）进行传输。APT格式允许无人值守服务，通常用于陆地气象传送基站及静止[气象卫星](https://zh.wikipedia.org/wiki/气象卫星 "wikilink")。一个完整的无线电传真数据传输格式通常如下：

| 信号   | 持续时间（秒） | IOC576 | IOC288 | 备注           |
| ---- | ------- | ------ | ------ | ------------ |
| 启动音  | 5\~10   | 300Hz  | 675Hz  | 彩色传真为200Hz   |
| 相位信号 | 10\~30  |        |        | 有白色间隔的黑色连续信号 |
| 图像信号 | 不确定     | 1200线  | 600线   | 120LPM时      |
| 停止音  | 5\~20   | 450Hz  | 450Hz  |              |
| 黑色   | 10      |        |        |              |
|      |         |        |        |              |

虽然现时[互联网](../Page/互联网.md "wikilink")及卫星通讯日益普及，包括无线电传真在内的传真使用率持续减少，但在一些特定场合（尤其是远洋航海、地处偏远等无法上网的情况下），无线电传真还在发挥着重要的作用。目前使用无线电传真传送信号的机构，绝大多数是世界各地的气象或海事机构，例如[中国大陆](../Page/中国大陆.md "wikilink")上海海岸电台（由[中国气象局](../Page/中国气象局.md "wikilink")提供气象图）、[台湾](https://zh.wikipedia.org/wiki/台湾 "wikilink")[中央气象局](https://zh.wikipedia.org/wiki/中央气象局 "wikilink")（[呼号为BMF](../Page/无线电台呼号.md "wikilink")）和[日本](../Page/日本.md "wikilink")[气象厅](../Page/氣象廳_\(日本\).md "wikilink")（呼号为JMH）。日本[共同社是目前全世界唯一一家仍在使用无线电传真播发新闻的通讯社](https://zh.wikipedia.org/wiki/共同社 "wikilink")。

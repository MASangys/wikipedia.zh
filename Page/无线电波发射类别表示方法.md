[世界无线电行政会议](../Page/世界无线电通信大会.md "wikilink")（WARC）于1979年在[瑞士](https://zh.wikipedia.org/wiki/瑞士 "wikilink")[日内瓦](../Page/日内瓦.md "wikilink")召开会议，制定了**无线电波发射类别的表示方法**，并于1982年1月1日正式开始实施。

## 表示方法

### 发射类别的任选特性

以下两种符号作为任选特性，用于完整描述某种发射，使用时分别为第四个（信号的详细说明）和第五个（[复用性质](https://zh.wikipedia.org/wiki/复用 "wikilink")）符号。如不使用第四个或第五个符号，则应在对应位置标上一短划。

| 第四个符号                                                                     |    | 第五个符号                                                 |
| ------------------------------------------------------------------------- | -- | ----------------------------------------------------- |
| 信号的详细说明                                                                   | 符号 | 复用性质                                                  |
| 具有不同数目和/或不同持续时间[码元的双态代码](https://zh.wikipedia.org/wiki/码元 "wikilink")     | A  | 没有复用                                                  |
| 没有纠错的具有相同数目和持续时间码元的双态代码                                                   | B  |                                                       |
| 有纠错的具有相同数目和持续时间码元的双态代码                                                    | C  | 码分复用                                                  |
| 每个状态代表一个信号码元（一个或多个[比特](https://zh.wikipedia.org/wiki/比特 "wikilink")）的四态代码 | D  |                                                       |
| 每个状态代表一个信号码元（一个或多个[比特](https://zh.wikipedia.org/wiki/比特 "wikilink")）的多态代码 | E  |                                                       |
| 每个状态或状态的组合代表一个字符的多态代码                                                     | F  | [频分复用](https://zh.wikipedia.org/wiki/频分复用 "wikilink") |
| 广播音质的声音（单声道）                                                              | G  |                                                       |
| 广播音质的声音（立体声或四声道立体声）                                                       | H  |                                                       |
| 商用音质的声音                                                                   | J  | [时分复用](https://zh.wikipedia.org/wiki/时分复用 "wikilink") |
| 利用频率倒置或频段分割法的商用音质的声音                                                      | K  |                                                       |
| 利用单独频率调制的信号来控制解调后信号电平的商用音质的声音                                             | L  |                                                       |
| 单色                                                                        | M  | 频分复用与时分复用的组合                                          |
| 彩色                                                                        | N  |                                                       |
| 上述的组合                                                                     | W  | 其他复用方式                                                |
| 上述各项未能包括的其他情况                                                             | X  |                                                       |

## 发射类型举例

  - [调幅广播](https://zh.wikipedia.org/wiki/调幅广播 "wikilink")（长波、中波、短波）：A3E（单声道）、D8E（中波立体声）
  - [调频广播](../Page/调频广播.md "wikilink")：F3E（单声道）、F8E（立体声）
  - [模拟电视广播](https://zh.wikipedia.org/wiki/模拟电视 "wikilink")：
      - 图像：C3F
      - 声音：F3E（单声道）、F8E（立体声）
  - [数字电视](../Page/数字电视.md "wikilink")广播：G7W（[DVB-T](../Page/DVB-T.md "wikilink")、[ISDB-T](https://zh.wikipedia.org/wiki/ISDB-T "wikilink")、[DTMB](../Page/DTMB.md "wikilink")）、C7W（[ATSC](../Page/ATSC.md "wikilink")）
  - [无线电传真](../Page/无线电传真.md "wikilink")：J3C、F3C
  - [授时台](https://zh.wikipedia.org/wiki/授时 "wikilink")（[电波钟](https://zh.wikipedia.org/wiki/电波钟 "wikilink")）：A1B
  - [业余无线电](../Page/业余无线电.md "wikilink")：
      - [CW](https://zh.wikipedia.org/wiki/CW "wikilink")：A1A
      - [SSB](https://zh.wikipedia.org/wiki/单边带 "wikilink")：J3E
      - [FM](../Page/FM.md "wikilink")：F3E
      - [AM](https://zh.wikipedia.org/wiki/调幅 "wikilink")：A3E

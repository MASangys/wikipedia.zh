**日光**
是[日間](https://zh.wikipedia.org/wiki/日間 "wikilink")（並且可能包括[曙暮光](https://zh.wikipedia.org/wiki/曙暮光 "wikilink")）在戶外直接或間接照射到的所有[太陽光](https://zh.wikipedia.org/wiki/太陽光 "wikilink")。這裡面包括直射的[太陽光](https://zh.wikipedia.org/wiki/太陽光 "wikilink")、[天空漫射和來自地球的和陸地的反射](../Page/天空漫射.md "wikilink")（經常兩者都有）。被太空中的物體（也就是地球大氣層之外）反射或散射的太陽光不包含在日光內。因此，儘管它是「間接的太陽光」，但[月光不能算是日光](../Page/月光.md "wikilink")，**日間**是每一天中有日光發生的時間。

## 定義

日光只存在於一些特定的地區，就某種程度而言，[太陽總是在那些地區的地平線之上](https://zh.wikipedia.org/wiki/太陽 "wikilink")（事實上，在任何給定的時間，[地球上有略大於](../Page/地球.md "wikilink")50%的地區在日光下。解釋不是正好一半的原因，請參見[此處](https://zh.wikipedia.org/wiki/日#介紹 "wikilink")）。但是，在戶外的[照度](https://zh.wikipedia.org/wiki/照度 "wikilink")，因為[太陽光直接的照射](https://zh.wikipedia.org/wiki/太陽光 "wikilink")，在[正午可以達到](https://zh.wikipedia.org/wiki/正午 "wikilink")120,000勒克司，足以造成[眼睛的](https://zh.wikipedia.org/wiki/眼睛 "wikilink")[刺痛](https://zh.wikipedia.org/wiki/刺痛 "wikilink")，或是當太陽在地平線上又有濃厚的暴風雲而低至5[勒克司](https://zh.wikipedia.org/wiki/勒克司 "wikilink")（在極端的情況下還會低於1勒克司），這會使遠處的[街燈能照射出可見的影子](../Page/街燈.md "wikilink")。它也可能在特殊的情況下，如[日食或很高濃度的煙塵](../Page/日食.md "wikilink")、塵埃\[1\]或火山灰\[2\]而變得黑暗。

## 不同狀況下的日光強度

| 照度                | 例子                                                                                             |
| ----------------- | ---------------------------------------------------------------------------------------------- |
| 120,000 勒克司       | 最亮的太陽光                                                                                         |
| 110,000 勒克司       | 明亮的太陽光                                                                                         |
| 20,000 勒克司        | 在正午清澈的藍天下陰影內的照度                                                                                |
| 1,000 - 2,000 勒克司 | 典型陰天的正午                                                                                        |
| \<200 勒克司         | 在正午最黑暗的暴風雲下                                                                                    |
| 400 勒克司           | 在晴朗天氣下的[日出或](../Page/日出.md "wikilink")[日沒](https://zh.wikipedia.org/wiki/日沒 "wikilink")（室溫的照度） |
| 40 勒克司            | 完全陰天下的日出或日沒                                                                                    |
| \<1 勒克司           | 最黑暗的暴風雲下的日出或日沒                                                                                 |

作為比較的夜間照度級別：

| 照度          | 例子                                  |
| ----------- | ----------------------------------- |
| \<1 勒克司     | [月光](../Page/月光.md "wikilink")\[3\] |
| 0.25勒克司     | 在滿月下的晴朗夜空\[4\]\[5\]                 |
| 0.01勒克司     | 弦月                                  |
| 0.002 勒克司   | 沒有月光，星光清澈且包含氣輝的夜晚\[6\]              |
| 0.0002 勒克司  | 沒有月光，也沒有氣輝的夜晚\[7\]                  |
| 0.00014 勒克司 | 金星最亮的光度\[8\]                        |
| 0.0001 勒克司  | 被雲遮蔽且沒有月光的夜空\[9\]                   |

## 效應

[Terminator_line_January.png](https://zh.wikipedia.org/wiki/File:Terminator_line_January.png "fig:Terminator_line_January.png")
日光被普遍的認為對人類的心理有積極的影響，並且在白天較短的冬季出現[心理健康的問題](https://zh.wikipedia.org/wiki/心理健康 "wikilink")，比實施夏令時的夏季期間為多。[臨床憂鬱症被與有限的日光聯結在一起被稱為](https://zh.wikipedia.org/wiki/臨床憂鬱症 "wikilink")[季節性情緒障礙](https://zh.wikipedia.org/wiki/季節性情緒障礙 "wikilink")。

[採光是對戶內空間的照明](https://zh.wikipedia.org/wiki/採光 "wikilink")，像是通過[窗戶和](https://zh.wikipedia.org/wiki/窗戶 "wikilink")[天窗所允許進入建築物的日光](https://zh.wikipedia.org/wiki/天窗 "wikilink")。這種形式的照明可以節約[能源](../Page/能源.md "wikilink")，也可以避免在人工燈光和[美學下的](https://zh.wikipedia.org/wiki/美學 "wikilink")[過度照明對健康產生的不良影響](https://zh.wikipedia.org/wiki/過度照明 "wikilink")。因為白天有陽光可以減少人工照明能源的使用，所以只需安裝較少的電燈，或是使用調光的電子開關自動回應存在的日光，這種過程稱為[日光採集](https://zh.wikipedia.org/wiki/日光採集 "wikilink")。

在這幾年，已經可以在工作場所以人工重新創建日光照射的效果。不過這是需要極度昂貴的設備和極度的消耗能源，因此幾乎只在無論如何都需要這種強度光的專業領域，像是[影片製作的工作場所才會使用](https://zh.wikipedia.org/wiki/影片製作 "wikilink")。

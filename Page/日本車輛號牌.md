[Tesla_Roadster_with_Japanese_license_plates.jpg](https://zh.wikipedia.org/wiki/File:Tesla_Roadster_with_Japanese_license_plates.jpg "fig:Tesla_Roadster_with_Japanese_license_plates.jpg") **日本車輛號牌**（）是指日本公路車輛的[登記牌照](../Page/車輛號牌.md "wikilink")，依據《》、《[道路運送車輛法施行規則](https://web.archive.org/web/20061111183521/http://law.e-gov.go.jp/htmldata/S26/S26F03901000074.html)》、《[自動車登錄規則](https://web.archive.org/web/20141006102230/http://law.e-gov.go.jp/htmldata/S45/S45F03901000007.html)》和其他相關的法令制定，由[國土交通省](../Page/國土交通省.md "wikilink")下轄的各運輸支局、自動車檢查登錄事務所及檢驗予以認可。

此外，[市區町村可以依據](https://zh.wikipedia.org/wiki/市區町村 "wikilink")《》對管轄區內的[附發動機腳踏車](https://zh.wikipedia.org/wiki/摩托化自行車 "wikilink")、和[迷你車課稅](https://zh.wikipedia.org/wiki/迷你車 "wikilink")，納稅車輛同樣有義務展示課稅的標識號牌。日本車輛號牌的名稱根據不同類型的車輛和運作情況，可以分為：運輸支局發行的「**[汽車註冊號牌](https://zh.wikipedia.org/wiki/#汽車註冊號牌 "wikilink")**」、輕自動車檢查協會發行的「**[車輛號牌](https://zh.wikipedia.org/wiki/#車輛號牌 "wikilink")**」和基礎地方公共團體/市區町村發行的「**[標識](https://zh.wikipedia.org/wiki/#標識 "wikilink")**」三種類型。 __TOC__ {{-}}

依法規定，在日本公路行駛的車輛必須經過、註冊或的程序，然後予以認證發出車輛編號。運行車輛有義務在指定的車身位置展示車輛的編號，展示編號的號牌根據不同類型的車輛和運作情況，正式名稱分為「汽車註冊號牌」和「車輛號牌」兩種。\[1\]

### 汽車註冊號牌

《[道路運送車輛法](https://web.archive.org/web/20110904053631/http://law.e-gov.go.jp/htmldata/S26/S26HO185.html)》將公路運行車輛的種類分為「[汽車](https://zh.wikipedia.org/wiki/汽車 "wikilink")」（）、「附發動機腳踏車」（）和「[慢車](https://zh.wikipedia.org/wiki/慢車 "wikilink")」（）三種類型，其中「汽車」根據車輛的大小和結構，細分為「普通汽車」、「小型汽車」、「[輕型汽車](../Page/輕型車.md "wikilink")」、「大型特殊汽車」和「小型特殊汽車」五種類別；\[2\] 除了「輕型汽車」、「小型特殊汽車」和「二輪的小型汽車」以外的汽車都要經過註冊以後，才能在公路上合法行駛。

註冊車輛使用的號牌稱為「**汽車註冊號牌**」（），是依據《道路運送車輛法》、《道路運送車輛法施行規則》和《自動車登錄規則》制定，裝置於普通汽車、小型汽車和大型特殊汽車，包括：

  - 普通汽車（），總排氣量2,000cc以上的車輛
  - 小型汽車（），長4.7m以下、寬1.7m以下、高2.0m以下，總排氣量2,000cc以下的車輛
  - 大型特殊汽車（），起重機、挖掘機、推土機、壓路機、堆高機、除雪機、拖拉機等各種特殊車輛

### 車輛號牌

豁免註冊車輛使用的號牌稱為「**車輛號牌**」（），是依據《道路運送車輛法》和《道路運送車輛法施行規則》制定，裝置於輕型車輛和摩托車，包括：

  - 輕型摩托車（），俗稱「」，長2.5m以下、寬1.3m以下、高2.0m以下，總排氣量125cc以上、250cc以下的摩托車（包括加裝邊車的摩托車）
  - 小型摩托車（），俗稱「」，總排氣量250cc以上的摩托車（包括加裝邊車的摩托車）
  - 輕型機動三輪車（），長3.4m以下、寬1.48m以下、高2.0m以下，總排氣量660cc以下的機動三輪車
  - 輕型車（），長3.4m以下、寬1.48m以下、高2.0m以下，總排氣量660cc以下的車輛

### 標識

[Japanese_motorcycle_license_plate_Matsuyama_cloud.png](https://zh.wikipedia.org/wiki/File:Japanese_motorcycle_license_plate_Matsuyama_cloud.png "fig:Japanese_motorcycle_license_plate_Matsuyama_cloud.png")[松山市](../Page/松山市.md "wikilink")發出的雲型號牌，取材自[司馬遼太郎的著名作品](https://zh.wikipedia.org/wiki/司馬遼太郎 "wikilink")「[坂上之雲](../Page/坂上之雲.md "wikilink")」；地名標記為「」，代表松山市的觀光景點「[道後溫泉](../Page/道後溫泉.md "wikilink")」\]\] 基礎的地方公共團體依據《[地方稅法](https://web.archive.org/web/20141006115613/http://law.e-gov.go.jp/htmldata/S25/S25HO226.html)》和各市區町村的條例，可以向管轄區內的附發動機腳踏車、小型特殊車輛和迷你車課收，然後予以發出「**課稅標識**」（），納稅車輛同時有義務展示有關課稅的標識編號。

大多數地方的標識號牌樣式是以[總務省](../Page/總務省.md "wikilink")（前[自治省](https://zh.wikipedia.org/wiki/自治省 "wikilink")）的公告為基準，然而公告並未有法律上的約束力，實際上的標識號牌可以由地方公共團體自行設計，通常會在號牌上印製地方上的風光名勝，做為推廣旅遊觀光的用途。\[3\] 在2007年7月，[愛媛縣](https://zh.wikipedia.org/wiki/愛媛縣 "wikilink")[松山市](../Page/松山市.md "wikilink")率先導入「雲型」的標識號牌，採用不規則形狀的號牌設計；\[4\] 隨後，這股潮流也蔓延至其他的市區町村，此類號牌俗稱為「**原創號牌**」（）或「**當地號牌**」（）。\[5\]

使用於附發動機腳踏車、小型特殊車輛和迷你車的標識號牌沒有全國統一規定的格式，通常是尺寸約縱100mm、橫200mm，縱橫比率約1：2的超小型號牌，分爲白色、黃色、粉紅色、綠色和淺藍色5種類型，包括：

  - 第1種附發動機腳踏車（），俗稱「」，總排氣量50cc以下，輸出功率600w以下的機動二輪車

  - 第2種附發動機腳踏車乙型（），俗稱「」，總排氣量90cc以下，輸出功率600w以上、800w以下的機動二輪車

  - 第2種附發動機腳踏車甲型（），俗稱「」，總排氣量125cc以下，輸出功率800w以上、1000w以下的機動二輪車

  - 小型特殊車輛（），長4.7m以下、寬1.7m以下、高2.8m以下，最高速度15km/h以下的特殊車輛（並最高速度35km/h以下的[農耕拖拉機](../Page/全地形車.md "wikilink")）

  - 迷你車（），長2.5m以下、寬1.3m以下、高2.0m以下，總排氣量50cc以下，備有車室的機動車輛

## 樣式

日本車輛號牌的樣式是依據《道路運送車輛法》、《道路運送車輛法施行規則》、《自動車登錄規則》和其他相關的法令來實施，施行規則對號牌的製作材料、尺寸、顏色和文字進行制定，上面記錄有關車輛檢驗和註冊的相關資料。「塗装式號牌」使用的製作材料是[鋁板](https://zh.wikipedia.org/wiki/鋁 "wikilink")，文字和數字是以刻印的方式浮凸在鋁板表面；「字光式號牌」使用的製作材料是[甲基丙烯酸](../Page/甲基丙烯酸.md "wikilink")[樹脂板](https://zh.wikipedia.org/wiki/樹脂 "wikilink")，在號牌背面加裝LED照明裝置，讓其在夜間產生自行發光的效果。

### 顏色

### 地名

「**地名**」標示的是有關車輛的[戶籍](https://zh.wikipedia.org/wiki/戶籍 "wikilink")，使用的是管轄有關地區的或自動車檢查登錄事務所的名稱，而不是車主住所的登錄戶籍；當車主搬遷到另一個運輸支局管轄區的時候，車輛戶籍也要相應地進行轉移。\[17\]\[18\]

### 分類編號

「**分類編號**」（）是位於「地名」右側的一組編號，用來表示註冊車輛的類型。\[21\]\[22\]\[23\]

以下為分類表:\[24\]

| 1962-71 | 1967-99 | 1998- | 描述                        |
| ------- | ------- | ----- | ------------------------- |
| 1       | 11, 1x  | 1xx   | 普通貨車（大於2000cc）            |
| 2       | 22, 2x  | 2xx   | 普通客車                      |
| 3       | 33, 3x  | 3xx   | 普通汽車（大於2000cc）            |
| 4       | 44至49   | 4xx   | 小型貨車（由660cc至2000cc）       |
| 5       | 55, 5x  | 5xx   | 小型汽車（由660cc至2000cc）       |
| 6       | 66, 6x  |       | 三輪貨車（小於360cc）             |
| 7       |         |       | 三輪汽車                      |
|         | 77, 7x  |       | 小型汽車（由660cc至2000cc）（重號部份） |
| 8       | 88, 8x  | 800   | 特殊用途車輛（大於660cc）           |
| 9       | 99      | 900   | 大型特殊車輛                    |
| 0       | 00      | 000   | 建築機械                      |

### 車輛編號

日本車輛號牌的編號根據不同類型的車輛和運作情況，分爲註冊車輛的「**連續編號**」（）和豁免註冊車輛的「**車輛編號**」（）兩種稱呼。每一組編號都有4個位置，四位數字以下的編號會將「・」填入空白處，四位數字的編號會在第二、三位數字之間以「-」連結，從「・・・1」至「99-99」共有9,999組編號。

除了用於[希望編號制度的編號外](https://zh.wikipedia.org/wiki/#希望編號制度 "wikilink")，其餘的編號都是以順序的方式來發行。在一般派發的編號中，後2位數字帶有「42」、「49」的編號分別會讓人聯想到「[死](../Page/四的禁忌.md "wikilink")」及「終生捱苦」的諧音，所以通常並不會發行；派發給駐留軍人的編號沒有「42」、「49」的忌諱，但是後2位數字帶有「[13](../Page/十三恐懼症.md "wikilink")」的編號也不會發行。所以，扣除後2位數字帶有「13」、「42」、「49」的編號後，實際上分派的一般編號是9,799組，派發給駐留軍人的編號是9,899組。

## 希望編號制度

註冊車輛的希望編號制度（）於1999年5月14日開始實施，輕型車於2005年1月4日導入這項制度，是為了讓車主可以。\[30\] 車主可以在車輛註冊的時候，支付較高金額獲得指定的牌號，包括300組在[平常不會發行的號碼](https://zh.wikipedia.org/wiki/#車輛編號 "wikilink")；部分熱門的編號則會被預先抽起，以抽籤的方式決定誰將獲得投注的編號。\[31\]\[32\]

### 皇室座車（宮內廳）

[NISSAN_PRINCE_ROYAL1_Nov_2009.jpg](https://zh.wikipedia.org/wiki/File:NISSAN_PRINCE_ROYAL1_Nov_2009.jpg "fig:NISSAN_PRINCE_ROYAL1_Nov_2009.jpg") [日本天皇](https://zh.wikipedia.org/wiki/日本天皇 "wikilink")、[皇后](../Page/皇后.md "wikilink")及[皇太后](../Page/皇太后.md "wikilink")座車使用專門的標識號牌，編號方式是以「皇」字加上[阿拉伯數字](https://zh.wikipedia.org/wiki/阿拉伯數字 "wikilink")，形狀是直徑100mm的圓形梨地圖案，以銀底金字寫上車輛編號。\[38\]\[39\] 皇室座車出現在官方場合時，會裝置一個金色的[菊花紋章](../Page/菊花紋章.md "wikilink")，表示車內的乘客正在執行公務，這個紋章會在非進行公務時除下，真正的座車號牌是位於[汽車保險桿左上角的位置](https://zh.wikipedia.org/wiki/保險桿 "wikilink")。

## 搬運砂石等大型車輛的標示編號

[搬運砂石等大型車輛的標示編號是標註在總重量](https://zh.wikipedia.org/wiki/砂石車 "wikilink")8噸以上、最大積載量5噸以上的[大型自動卸載車的申報編號](https://zh.wikipedia.org/wiki/礦山自卸車 "wikilink")，依據《[關於預防搬運砂石等大型車輛引發交通意外等的特別措施法案](https://web.archive.org/web/20141006105254/http://law.e-gov.go.jp/htmldata/S42/S42HO131.html)》制定。\[48\] 標示編號是以白底黑字書寫在自卸車貨斗兩側和背面的位置，標示有關車輛申報的地區、營業分類和編號。\[49\] 地名的標示方法是以2位文字為準則，營業的標示方法是把文字書寫在「〇」內，申報編號共有5位數字。\[50\] 營業的分類共有7種，包括：

  - 「-{営}-」，運輸業
  - 「-{石}-」，採石業
  - 「-{砕}-」，碎石業
  - 「-{砂}-」，採砂業
  - 「-{販}-」，砂石買賣業
  - 「-{建}-」，建築業
  - 「-{他}-」，其他（廢物處理業、洋灰製造業等）

[Max_planck.jpg](https://zh.wikipedia.org/wiki/File:Max_planck.jpg "fig:Max_planck.jpg") **普朗克單位制**是一種[計量單位制度](https://zh.wikipedia.org/wiki/計量單位 "wikilink")，由德國物理學家[馬克斯·普朗克最先提出](https://zh.wikipedia.org/wiki/馬克斯·普朗克 "wikilink")，因此命名為普朗克單位制。這種單位制是[自然單位制的一個實例](https://zh.wikipedia.org/wiki/自然單位制 "wikilink")，經過特別設計，使得某些基礎物理常數的值能夠簡化為1，這些基礎物理常數是

  - [萬有引力常數](https://zh.wikipedia.org/wiki/萬有引力常數 "wikilink")\(G\,\!\)，
  - [約化普朗克常數](https://zh.wikipedia.org/wiki/約化普朗克常數 "wikilink")\(\hbar\,\!\)，
  - 在真空裏的[光](../Page/光.md "wikilink")的[光速](../Page/光速.md "wikilink")\(c\,\!\)，
  - [庫侖常數](https://zh.wikipedia.org/wiki/庫侖定律 "wikilink")\(k_e=\frac{1}{4\pi\epsilon_0}\,\!\)，其中\(\epsilon_0\,\!\)是[真空電容率](https://zh.wikipedia.org/wiki/真空電容率 "wikilink")，也就是[電常數](https://zh.wikipedia.org/wiki/電常數 "wikilink")，
  - [波茲曼常數](../Page/波茲曼常數.md "wikilink")\(k_B\,\!\)。

上述每一個常數都至少出現於一個基本物理理論：\(G\,\!\)在[廣義相對論](../Page/廣義相對論.md "wikilink")與[牛頓的](https://zh.wikipedia.org/wiki/牛頓 "wikilink")[萬有引力定律](https://zh.wikipedia.org/wiki/萬有引力定律 "wikilink")、\(\hbar\,\!\)在[量子力學](https://zh.wikipedia.org/wiki/量子力學 "wikilink")、\(c\,\!\)在[狹義相對論](https://zh.wikipedia.org/wiki/狹義相對論 "wikilink")、\(\epsilon_0\,\!\)在[靜電學](../Page/靜電學.md "wikilink")、\(k_B\,\!\)在[統計力學與](https://zh.wikipedia.org/wiki/統計力學 "wikilink")[熱力學](https://zh.wikipedia.org/wiki/熱力學 "wikilink")。实际上，以上的五个常数在許多物理定律的代數表達式中多次出现，因此引入普朗克單位制可以将這些代數表達式简化，普朗克單位制也因此成为了理論物理學一個非常有用的工具。在統一理論方面的研究，特別如[量子重力學中](https://zh.wikipedia.org/wiki/量子重力 "wikilink")，普朗克單位制能夠給研究者一點大概的提示。

## 基本普朗克單位

每一個單位制都有一組基本單位。（在[國際單位制裏](https://zh.wikipedia.org/wiki/國際單位制 "wikilink")，長度的基本單位是公尺）在普朗克單位制裏，長度的基本單位是[普朗克長度](../Page/普朗克長度.md "wikilink")，時間的基本單位是[普朗克時間](../Page/普朗克時間.md "wikilink")，等等。這些單位都是由表1的五個基礎物理常數衍生的。表2展示出這些基本普朗克單位。

使用普朗克單位後，表1的五個基礎物理常數的數值都約化為1，因此表2的普朗克長度，普朗克質量，普朗克時間，普朗克電荷，與普朗克溫度這些計量也都約化為1。這可以無因次地表達為

  -
    因為\(G = c = \hbar = \frac{1}{4 \pi \epsilon_0} = k_B = 1\,\!\)，所以\(l_\text{P} = m_\text{P} = t_\text{P} = q_\text{P} = T_\text{P} = 1\,\!\)。

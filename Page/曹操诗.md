**[曹操](../Page/曹操.md "wikilink")**（155年—220年）是[东汉](../Page/东汉.md "wikilink")（25年—220年）[末年崛起并成为](https://zh.wikipedia.org/wiki/东汉末年 "wikilink")[中国政府实质首脑的](https://zh.wikipedia.org/wiki/中国 "wikilink")[军阀](../Page/军阀.md "wikilink")。他为由其子和继承人[曹丕](../Page/曹丕.md "wikilink")于[三国](../Page/三国.md "wikilink")时期（220年—280年）所建的[曹魏](../Page/曹魏.md "wikilink")（220年—265年）奠定了基础。诗歌是曹操留下的众多文化遗产之一。

## 概述

曹操和儿子曹丕、[曹植](../Page/曹植.md "wikilink")都是有造诣的诗人。曹操也是[徐干等诗人的赞助者](https://zh.wikipedia.org/wiki/徐干 "wikilink")。\[1\]曹操的作品只有一部分留存至今。他的诗文朴实而深刻，帮助重塑了他所处时代及以后的诗风，也最终促成了与[唐诗](../Page/唐诗.md "wikilink")相关的诗风。曹操、曹丕、曹植合称“[三曹](../Page/三曹.md "wikilink")”。三曹的诗与其他诗人的诗最终发展形成了[建安文学](https://zh.wikipedia.org/wiki/建安文学 "wikilink")：[建安为](../Page/建安_\(东汉\).md "wikilink")196年—220年间的年号。曹氏等诗人在汉末及曹魏建立后继续写和发展此诗风，他们就是建安诗人。汉末内战对诗歌的影响导致了建安文学时期一种对人生短暂的阴沉而感伤的基调的发展。

根植于[汉朝诗](../Page/汉朝诗.md "wikilink")民歌的建安文学发展进入一种学者诗形式，即[六朝诗](../Page/六朝诗.md "wikilink")的特征。曹操和其他建安诗人发展了源于如句长不等的诗等民歌或传统民谣的独特的汉朝[赋](../Page/赋.md "wikilink")（或[乐府](https://zh.wikipedia.org/wiki/乐府 "wikilink")）。不规律的诗句被转换成了规律的五言体，很像唐诗的五言绝句，也给后者带来灵感。曹操尤以显然配乐的民谣风格的诗句闻名。\[2\]

曹操也用《[诗经](https://zh.wikipedia.org/wiki/诗经 "wikilink")》旧的四字句格式写诗。[伯顿·沃森](../Page/伯顿·沃森.md "wikilink")描述曹操为：“那个时期唯一成功将活力注入旧的四字韵律的作家，主要因为他抛弃了与之相关的过时的语言且使用了他当时的普通诗歌语言。”\[3\]曹操也以早期对[山水诗体裁的贡献闻名](https://zh.wikipedia.org/wiki/山水诗 "wikilink")，曾作每句四字、共14句的诗《[观沧海](https://zh.wikipedia.org/wiki/观沧海 "wikilink")》（由[叶维廉所译](https://zh.wikipedia.org/wiki/叶维廉 "wikilink")）。\[4\]

## 诗歌

### 《龟虽寿》

曹操最著名的诗篇之一是用旧的四字格式写就的《[龟虽寿](https://zh.wikipedia.org/wiki/龟虽寿 "wikilink")》。这是由四部分诗组成的《[步出夏门行](https://zh.wikipedia.org/wiki/步出夏门行 "wikilink")》的一部分。它作于[建安十二年](../Page/建安_\(东汉\).md "wikilink")（207年）[白狼山之战](../Page/白狼山之战.md "wikilink")期间。

<center>

|            |  |
| ---------- |  |
| 《龜雖壽》      |  |
| 神龜雖壽，猶有竟時。 |  |
| 螣蛇乘霧，終為土灰。 |  |
| 老驥伏櫪，志在千里； |  |
| 烈士暮年，壯心不已。 |  |
| 盈縮之期，不但在天； |  |
| 养怡之福，可得永年。 |  |
| 幸甚至哉！歌以咏志。 |  |

</center>

### 《短歌行》

曹操的另一名篇是在建安十三年（208年）冬[赤壁之战前所作的](https://zh.wikipedia.org/wiki/赤壁之战 "wikilink")《[短歌行](https://zh.wikipedia.org/wiki/短歌行 "wikilink")》。

<center>

|                                                              |  |
| ------------------------------------------------------------ |  |
| 《短歌行》                                                        |  |
| 對酒當歌，人生幾何？                                                   |  |
| 譬如朝露，去日苦多。                                                   |  |
| 慨當以慷，憂思難忘。                                                   |  |
| 何以解憂？唯有杜康。(1)                                                |  |
| 青青子衿，悠悠我心。(2)                                                |  |
| 但為君故，沈吟至今。                                                   |  |
| 呦呦鹿鳴，食野之蘋。                                                   |  |
| 我有嘉賓，鼓瑟吹笙。(3)                                                |  |
| 明明如月，何時可掇？                                                   |  |
| 憂從中來，不可斷絕。                                                   |  |
| 越陌度阡，枉用相存。                                                   |  |
| 契闊談宴，心念舊恩。                                                   |  |
| 月明星稀，烏鵲南飛，                                                   |  |
| 繞樹三匝，何枝可依？                                                   |  |
| 山不厭高，海不厭深。                                                   |  |
| [周公吐哺](https://zh.wikipedia.org/wiki/周公 "wikilink")，天下歸心。(4) |  |

</center>

(1) [杜康](../Page/杜康.md "wikilink")是传说中酒的发明者。
(2) 这是引用了《诗经》中的爱情诗。
(3) 这一全节是引自《诗经》的一首诗。
(4) 当人们在周公吃饭时来访，周公就吐出食物不吃完饭而是去见他们。后世评论者嘉奖他的自我牺牲。

**金曲獎最佳台語專輯獎**為現行流行音樂類[金曲獎演唱類之出版獎項](../Page/金曲獎.md "wikilink")。1990年[第2屆所設立之](https://zh.wikipedia.org/wiki/第2屆金曲獎 "wikilink")「最佳專輯獎」，於1991年（第3屆）及2005年（第16屆）先後分別依音樂類型及演唱語言劃分後，便獨立設獎，並於2007年（第18屆）更名為「最佳台語專輯獎」迄今。2017年[第28屆時](https://zh.wikipedia.org/wiki/第28屆金曲獎 "wikilink")，除了原本按語言劃分的四項專輯獎之外，另增設「[年度專輯獎](../Page/年度專輯獎_\(金曲獎\).md "wikilink")」，受理不拘語言、類型之各類演唱專輯（類似第16屆前的「最佳演唱專輯獎」）。\[1\]
在歷屆獲獎及入圍作品之演唱者中，江蕙共獲提名10次、獲獎5次，居獲獎及入圍次數首位。

## 沿革

1990年[第2屆金曲獎設立](https://zh.wikipedia.org/wiki/第2屆金曲獎 "wikilink")「最佳專輯獎」。自1991年[第3屆起](https://zh.wikipedia.org/wiki/第3屆金曲獎 "wikilink")，專輯獎始依照音樂類型劃分為「演唱專輯」及「[演奏專輯](../Page/演奏類最佳專輯獎_\(金曲獎\).md "wikilink")」兩類。2005年第16屆時，演唱專輯獎再依演唱語言分別設立：[國語](../Page/現代標準漢語.md "wikilink")、[台語](https://zh.wikipedia.org/wiki/台語 "wikilink")、[客語以及](https://zh.wikipedia.org/wiki/客語 "wikilink")[原住民語四項流行音樂演唱專輯獎](../Page/台灣南島語言.md "wikilink")；其中「最佳台語流行音樂演唱專輯獎」自2007年第18屆起，更名為「最佳台語專輯獎」迄今。

| 年份    | 屆次   | 內容                       |
| ----- | ---- | ------------------------ |
| 1990年 | 第2屆  | 設立「最佳專輯獎」                |
| 1991年 | 第3屆  | 依音樂類型分設為「最佳演唱專輯獎」        |
| 1997年 | 第8屆  | 更名為「最佳流行音樂演唱唱片獎」         |
| 2000年 | 第11屆 | 更名為「最佳流行音樂演唱專輯獎」         |
| 2005年 | 第16屆 | 再依演唱語言分設為「最佳台語流行音樂演唱專輯獎」 |
| 2007年 | 第18屆 | 更名為「最佳台語專輯獎」迄今           |

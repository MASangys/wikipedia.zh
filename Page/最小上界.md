在[数学中](../Page/数学.md "wikilink")，最小上界（，亦称**上确界**，记为sup
E）是[序理论的重要概念](../Page/序理论.md "wikilink")，在[格论和](../Page/格_\(数学\).md "wikilink")[数学分析等领域有广泛应用](../Page/数学分析.md "wikilink")。

## 定义

给定[偏序集合](../Page/偏序集合.md "wikilink")(T,≤)，对于S⊆T，S的**上确界**sup(S)定义为S的所有[上界组成的集合的最小元](../Page/上界.md "wikilink")（若有）。即sup(S)满足：

  - ∀s∈S ⇒ s≤sup(S)
  - ∀t∈T，若t满足∀s∈S ⇒ s≤t，则有sup(S)≤t。
  - sup(S)∈T。

上确界也被称为**最小上界**、**lub** 或
**LUB**，在[格论中也被称为](../Page/格_\(数学\).md "wikilink")[并](../Page/并运算.md "wikilink")，在[序理论中S的上确界也被记为](../Page/序理论.md "wikilink")\(\vee\)S。

  - 若S包含[最大元素](../Page/最大元.md "wikilink")，则该元素就是上确界。
  - 若S有上确界，则上确界是唯一的。
  - 上确界的[对偶概念](../Page/对偶性_\(序理论\).md "wikilink")[最大下界叫做](../Page/最大下界.md "wikilink")[下确界或](../Page/下确界.md "wikilink")[交](../Page/交运算.md "wikilink")。
  - 偏序集合的子集可能没有上确界，即使它有上界。
  - 上确界一定不能混淆于[极小](../Page/极小元.md "wikilink")，[上界](../Page/上界.md "wikilink")，[极大元或](../Page/极大元.md "wikilink")[最大元](../Page/最大元.md "wikilink")。

## 数学分析中的上确界

在[数学分析中](../Page/数学分析.md "wikilink")，[实数的集合S的](../Page/实数.md "wikilink")**上确界**或**最小上界**记为
sup(*S*)，并被定义为大于或等于 *S*
中所有成员的最小实数。实数的一个重要性质是它的[完备性](../Page/完备空间.md "wikilink")：实数集合的所有[非空子集是有上界的就是这个实数集合成员的上确界](../Page/非空集合.md "wikilink")。

### 例子

\[\sup \{ 1, 2, 3 \} = 3\,\]

\[\sup \{ x \in \mathbb{R} : 0 < x < 1 \}  =  \sup \{ x \in \mathbb{R} : 0 \leq x  \leq 1 \} = 1\,\]

\[\sup \{ (-1)^n - \frac{1}{n} : n \in \mathbb{N} \} = 1\,\]

\[\sup \{ a + b : a \in A \mbox{ and } b \in B\} = \sup(A) + \sup(B)\,\]

\[\sup \{ x \in \mathbb{Q} : x^2 < 2 \} = \sqrt{2}\,\]

这个[有理数的集合的上确界是个](../Page/有理数.md "wikilink")[无理数](../Page/无理数.md "wikilink")，这意味着有理数是[不完备的](../Page/完备空间.md "wikilink")。

此外，如果我们定义在 *S* 是空集的时候 sup(*S*) = −∞ 和在 *S* 没有上界的时候 sup(*S*) = +∞
，则实数的所有集合都在[扩展的实数轴上有上确界](../Page/扩展的实数轴.md "wikilink")。

\[\sup \mathbb{Z} = \infty\,\]

\[\sup \varnothing = -\infty\,\]

如果上确界属于这个集合，则它是这个集合的[最大元素](../Page/最大元.md "wikilink")。术语[极大元在处理实数或任何其他](../Page/极大元.md "wikilink")[全序集合的时候是同义的](../Page/全序集合.md "wikilink")。

要证明 *a* = sup(*S*)，必须证明 *a* 是 *S* 的上界并且 *S* 的任何其他上界大于 *a*；等价地，也可以证明 *a*
是 *S* 的上界并且小于 *a* 的任何数都不是 *S* 的上界。

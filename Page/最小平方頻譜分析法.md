**最小平方頻譜分析法**（）是一種利用[最小平方法尋找適配於資料點之最佳](https://zh.wikipedia.org/wiki/最小平方法 "wikilink")[正弦曲線](../Page/正弦曲線.md "wikilink")，以估算[頻譜的方法](https://zh.wikipedia.org/wiki/頻譜 "wikilink")。其數學原理與科學界中最常用的[傅立葉分析相似](https://zh.wikipedia.org/wiki/傅立葉分析 "wikilink")\[1\]\[2\]。一般而言，傅立葉分析會將間隔較長之訊號的長周期雜訊放大，而最小平方頻譜分析法則解決了這個問題\[3\]。

最小平方頻譜分析法也稱為**凡尼切克法**（Vaníček method）\[4\]、**隆布法**（Lomb
method）\[5\]\[6\]或**隆布—史卡構法**（Lomb–Scargle
method）\[7\]\[8\]\[9\]，分別取名自對其有所貢獻的、尼可拉斯·隆布（Nicholas R.
Lomb）\[10\]以及傑佛瑞·史卡構（Jeffrey D. Scargle）\[11\]。此外，麥可·科恩伯格（Michael
Korenberg）、史考特·陳（Scott Chen）以及等人也曾開發出與之關係密切的其它方法。

## 歷史背景

長久以來，科學界對於[傅立葉分析](https://zh.wikipedia.org/wiki/傅立葉分析 "wikilink")、[週期圖法與正弦曲線之](https://zh.wikipedia.org/wiki/週期圖法 "wikilink")[最小平方](https://zh.wikipedia.org/wiki/最小平方法 "wikilink")[擬合間的密切關聯早有所知](../Page/曲線擬合.md "wikilink")\[12\]。然而，大多數以上述理論為基礎開發的方法僅適用於取樣間距相等的[訊號](../Page/時間序列.md "wikilink")。1963年，[荷蘭數學和計算機科學研究學會的弗里克](https://zh.wikipedia.org/wiki/荷蘭數學和計算機科學研究學會 "wikilink")·巴寧（Freek
J. M.
Barning）提出類似的方法處理了取樣間距不同的訊號\[13\]：先以週期圖法分析（現又稱為隆布法），再從所得的週期圖中選取特定頻率的正弦曲線並以最小平方法擬合之；這兩段過程間則是透過[匹配追蹤連接](../Page/匹配追蹤.md "wikilink")，並以反擬合後處理（post-backfitting）以避免[過適](../Page/過適.md "wikilink")\[14\]、或使用正交匹配追蹤法\[15\]。

[新不伦瑞克大学的加拿大](../Page/新不伦瑞克大学.md "wikilink")[大地測量學家](https://zh.wikipedia.org/wiki/大地測量學 "wikilink")於1969年也提出了匹配追蹤法，他稱之為「連續頻譜分析」，並稱其所得的結果為「最小平方週期圖」，且同時適用於取樣間距相同或不同的訊號\[16\]。他更於1971年將該方法一般化，使其能夠分析出訊號中單一平均值以外的各種系統成分，包含可預測但大小未知的線性、二次方或指數趨勢\[17\]。

[雪梨大學的尼可拉斯](https://zh.wikipedia.org/wiki/雪梨大學 "wikilink")·隆布於1976年簡化了凡尼切克法，並指出該方法與[週期圖法有相當密切的關連](https://zh.wikipedia.org/wiki/週期圖法 "wikilink")\[18\]。後來，傑佛瑞·史卡構也定義了非等距取樣訊號的週期圖，並對其做了分析\[19\]；他並發現，只要將該週期圖稍作修改，便能夠得到與隆布的最小平方擬合法完全相同的結果。

史卡構曾表示他的論文「並未提出一個新的測量方法，而是對現有最常用的方法——週期圖法——在取樣間距不相等時的可靠性以及效率進行研究」。他在論文中同時引用了最小平方弦波擬合以及週期圖法分析，並指出該兩種方法（在他所提出的修改之下）是完全相等的\[20\]。

媒體對相關研究的發展則有如下的摘要：

1989年，[皇后大學的麥可](../Page/皇后大學_\(加拿大\).md "wikilink")·科恩伯格開發了所謂的「快速正交搜尋法」（fast
orthogonal
search），以更迅速地找到一組頻譜成分分析的近似最佳解\[21\]；該方法與後來出現的正交匹配追蹤法有許多相似之處。1994年，[史丹佛大學的史考特](../Page/史丹佛大學.md "wikilink")·陳與大衛·多諾霍提出了「[基底追蹤法](https://zh.wikipedia.org/wiki/基底 "wikilink")」（basis
pursuit）；該方法藉由將目標轉化為[線性規劃問題](https://zh.wikipedia.org/wiki/線性規劃 "wikilink")，並使各項係數的[L<sub>1</sub>-距離最小化](../Page/曼哈頓距離.md "wikilink")，以更快速地得到解答\[22\]。

## 方法

### 凡尼切克法

凡尼切克法乃是利用標準的[線性回歸或](../Page/線性回歸.md "wikilink")[最小平方法](https://zh.wikipedia.org/wiki/最小平方法 "wikilink")，將一個離散訊號近似為許多不同頻率的弦波之加權總和\[23\]。這些弦波的頻率則是以類似巴寧所提出的方法來決定，並另外藉由減少最小平方擬合後的[殘差以使每次決定的新頻率得到最佳化](https://zh.wikipedia.org/wiki/誤差 "wikilink")（這與[匹配追蹤法加上反擬合前處理所得到的結果相同](../Page/匹配追蹤.md "wikilink")\[24\]）。並且，決定後的弦波數量必須不超過整個訊號的[取樣點數量](../Page/取樣.md "wikilink")（同一頻率的正弦和餘弦波需視為不同的弦波）。

假設\(\phi\)為一個由許多不同頻率的基底弦波加權總和所得的訊號向量，其基底函數以矩陣\(\textbf{A}\)表示，而每個基底所對應的權重則以\(x\)向量表示，也就是：

\[\phi \approx \textbf{A}x\]

會希望權重向量\(x\)使得上式在近似\(\phi\)時的平方誤差和為最小。利用標準的[線性回歸法即可求得](../Page/線性回歸.md "wikilink")\(x\)的閉合形式解為\[25\]：

\[x = (\textbf{A}^{\mathrm{T}}\textbf{A})^{-1}\textbf{A}^{\mathrm{T}}\phi\]

其中，矩陣\(\textbf{A}\)可由任意彼此[獨立](https://zh.wikipedia.org/wiki/線性獨立 "wikilink")（但不需[正交](../Page/正交.md "wikilink")）的基底函數所組成；對頻譜分析而言，這些基底函數一般為某特定頻率範圍內均勻分布的正弦或餘弦波。如果在某過窄的頻寬內選擇了過多的頻率，則這些所選的函數可能不夠獨立、矩陣\(\textbf{A}\)本身較為[病態](https://zh.wikipedia.org/wiki/條件數 "wikilink")，而使得最後求得的頻譜不具意義\[26\]。

當矩陣\(\textbf{A}\)的所有基底函數皆彼此正交（也就是彼此不[相關](https://zh.wikipedia.org/wiki/相關 "wikilink")，且矩陣中的每一行兩兩[內積皆為零](https://zh.wikipedia.org/wiki/內積 "wikilink")）時，\(\textbf{A}^{\mathrm{T}}\textbf{A}\)為一[對角矩陣](../Page/對角矩陣.md "wikilink")；此外，當\(\textbf{A}\)之每一行的能量（即向量之元素平方和）皆相等時，\(\textbf{A}^{\mathrm{T}}\textbf{A}\)便為一[單位矩陣與某常數相乘的積](../Page/單位矩陣.md "wikilink")，而其[反矩陣的計算便相當容易](https://zh.wikipedia.org/wiki/反矩陣 "wikilink")。假設取樣間距為\(t\)，那麼後者的情況即是當取樣間距相同，且基底函數選擇的是頻率範圍為\((0,1/2t)\)內成對的正弦和餘弦波（每個基底的頻率為\(1/Nt\)，並忽略區間中取樣值皆為零的最小和最大頻率），而這情況同時也是[離散傅立葉變換的特例之一](https://zh.wikipedia.org/wiki/離散傅立葉變換 "wikilink")\[27\]。

\[x = \textbf{A}^{\mathrm{T}}\phi\]    
（離散傅立葉變換中\(N\)個等距取樣點和頻率乘上某比例常數的情況）

隆布所提出上述的簡化式可用於大多數的情況，因為一般而言不同弦波彼此之間的相關係數較小（至少當它們的間距夠大時），惟同一頻率下成對的正弦和餘弦波彼此間並非獨立而不適用。本質上，這和傳統[週期圖法的公式無異](https://zh.wikipedia.org/wiki/週期圖法 "wikilink")，但可用於非等距取樣的訊號。其中，\(x\)向量可為訊號的潛在頻譜做有效的估計，但由於基底之間的相關性被忽略，\(\textbf{A}x\)便不再能夠良好地近似原始的訊號，這個方法也因此不再是真正的「最小平方法」，惟學界仍繼續如是稱之。

### 隆布—史卡構法

在標準的週期圖法公式中，一般是直接計算訊號與正弦或餘弦波之間的內積。史卡構則對其稍作修改：首先，先找到一時間延遲項\(\tau\)使得成對的正弦與餘弦波在取樣時間為\(t_j\)時將會彼此正交；同時，他也對可能具有不同能量的兩個基底之間做了標準化，以取得在某頻率下更加準確的能量估計值\[28\]\[29\]。這些修改使得史卡構的週期圖法和隆布的最小平方法即完全相同。時間延遲項\(\tau\)的定義如下：

\[\tan{2 \omega \tau} = \frac{\sum_j \sin 2 \omega t_j}{\sum_j \cos 2 \omega t_j}\]

頻率\(\omega\)的週期圖則可估算如下：

\[P_x(\omega) = \frac{1}{2}
\left(
  \frac { \left[ \sum_j X_j \cos \omega ( t_j - \tau ) \right] ^ 2}
        { \sum_j \cos^2 \omega ( t_j - \tau ) }
+
 \frac {\left[ \sum_j X_j \sin \omega ( t_j - \tau ) \right] ^ 2}
        { \sum_j \sin^2 \omega ( t_j - \tau ) }
\right)\]

史卡構並表示，上式所得的週期圖與等距取樣情況下的結果具有相同的統計分布\[30\]。

在任一單獨的頻率\(\omega\)下，這個方法求得的頻譜能量與最小平方法使用該頻率之弦波擬合後所得的能量相同。擬合後函數的形式如下：

\[\phi(t) = A \sin \omega t + B \cos \omega t\]\[31\]

#### 一般化

標準的隆布—史卡構週期圖法適用於平均值為零的模型（即擬合用弦波的集合），因此一般而言在計算週期圖之前會先減去訊號的平均值並假設其為零。然而，當模型的平均值為非零時，這項假設並不準確。一般化的隆布—史卡構週期圖法便將此假設移除，並在計算頻譜時一併求得其平均值。如此一來，擬合後函數的形式如下：

\[\phi(t) = A \sin \omega t + B \cos \omega t + C\]\[32\]

此一般化的隆布—史卡構週期圖法也被稱為「浮動平均週期圖法」\[33\]。

### 快速正交搜尋法

科恩伯格於1989年提出了如下的方法：先從一的函數集合中選出一稀疏子集（以頻譜分析而言即指弦波）用於擬合，稱為快速正交搜尋法（fast
orthogonal
search）。數學上，快速正交搜尋是在縮小[均方误差](../Page/均方误差.md "wikilink")（MSER）時使用了一種略為修改過的[楚列斯基分解法](../Page/Cholesky分解.md "wikilink")，並以[稀疏矩阵之求逆方法實作](../Page/稀疏矩阵.md "wikilink")\[34\]\[35\]。快速正交搜尋如同其它的最小平方頻譜分析法，也能夠避免離散傅立葉變換最主要的缺點，並且可以求得高準確度的潛在週期性，同時對於處理非等距取樣訊號也相當有效。快速正交搜尋法同時也被應用於非線性系統鑑別（nonlinear
system identification）等其它的問題。

### 基底追蹤法

陳與多諾霍提出的法（basis
pursuit）也是從一過完備的函數集合中選出一稀疏子集（其中可包含弦波或其它函數）用於擬合，惟此法定義能夠使各項係數的[L<sub>1</sub>-距離得到最小化者即為最佳解](../Page/曼哈頓距離.md "wikilink")。如此一來，頻譜分析便可藉此轉化為已知的[線性規劃問題](https://zh.wikipedia.org/wiki/線性規劃 "wikilink")，並可利用現有的快速演算法得到解答\[36\]。

### 卡方法

大衛·帕爾默（David
Palmer）提出的[卡方法則能夠找到任意數量之諧波的最佳擬合函數](../Page/卡方分佈.md "wikilink")，提高了尋找非弦波諧和函數的自由度\[37\]。這是一項基於[快速傅立葉變換](../Page/快速傅里叶变换.md "wikilink")、對取樣間距任意且標準誤差不均的訊號進行分析的快速演算法。實作此方法的原始碼也已被公開\[38\]。由於離散訊號往往不具有相等的取樣間距，此方法先將訊號「柵格化」，即稀疏地在數個取樣點間填入一串[時間序列](../Page/時間序列.md "wikilink")。所有時間軸上互相交會的柵格點之統計權重都將被設為零，亦即取樣點之間的為無窮大。

## 應用

最小平方頻譜分析法最主要的用處是對未完整記錄的訊號進行[頻譜分析](https://zh.wikipedia.org/wiki/谱密度#解释 "wikilink")，且不須對其[竄改或摻入不存在的數據](https://zh.wikipedia.org/wiki/統計誤用#數據的竄改 "wikilink")。

利用最小平方頻譜分析法所得的[频谱中之各](../Page/频谱.md "wikilink")[強度量值即代表了某個頻率或週期對於整個](https://zh.wikipedia.org/wiki/量_\(数学\) "wikilink")[時間序列的](../Page/時間序列.md "wikilink")[變異數之貢獻](https://zh.wikipedia.org/wiki/變異數 "wikilink")\[39\]。一般而言，經由上述方法所定義的頻譜強度可用於推算輸出結果的[顯著水準狀態](../Page/第一型及第二型錯誤.md "wikilink")\[40\]。此外，凡尼切克法求得的頻譜強度也可由[分貝值表示](https://zh.wikipedia.org/wiki/分貝 "wikilink")\[41\]。值得注意的是，凡尼切克頻譜在統計上具有[Β分布](../Page/Β分布.md "wikilink")\[42\]。

凡尼切克的最小平方頻譜之反運算可由下述方法實現：若將正運算的過程表示為一矩陣的乘法，那麼（當它為[非奇异矩陣時](https://zh.wikipedia.org/wiki/非奇异方阵 "wikilink")）求取它的反矩陣或偽反矩陣便可進行反運算；當所選的基底弦波在取樣點時皆彼此獨立且其總數和取樣點數量相等時，反運算所得的結果便會與原始的訊號相同\[43\]。相對地，週期圖法並不具有已知的反運算方法。

## 實作

最小平方頻譜分析法可用一頁以內的[MATLAB程式碼實作完成](../Page/MATLAB.md "wikilink")\[44\]。簡言之：

> 「欲求得最小平方頻譜，我們必須計算\(m\)個頻譜值……可以藉由\(m\)次的最小平方法近似，每次求得一種不同頻率（的頻譜能量）\[45\]。」

也就是說，對一所欲擬合之頻率集合中的每個頻率\(\omega_i\)，將其對應的[正弦](../Page/正弦.md "wikilink")（\(\sin{\omega_i t}\)）和[餘弦函數](../Page/餘弦.md "wikilink")（\(\cos{\omega_i t}\)）於訊號的取樣時間點\(t_j\)取值，計算訊號與弦波之取樣值的[內積並給予適當的標準化](https://zh.wikipedia.org/wiki/內積 "wikilink")；若是使用隆布—史卡構的週期圖法，則在取內積之前先計算每個頻率\(\omega_i\)的時間延遲項\(\tau_i\)使得該頻率下的正弦與餘弦成分能夠彼此正交\[46\]；最後，從[振幅的內積值便可求得該兩項弦波成分的頻譜能量](https://zh.wikipedia.org/wiki/振幅 "wikilink")。若將這項程序套用於取樣間距相等、且擬合弦波頻率選為取樣時間全長倒數之整數倍的訊號時，即相當於[离散傅立葉变换](../Page/离散傅里叶变换.md "wikilink")。

上述即凡尼切克的原始方法，它在處理每個弦波成分時都將其視為彼此獨立，即使在取樣點時這些弦波有可能並不彼此正交。相對地，透過矩陣方程式求解、並將原始訊號的總變異數分割至指定的弦波頻率之間的方法，也能夠在一次考慮所有弦波成分（即不獨立處理）的情況下求得頻譜值\[47\]。這種矩陣求解的最小平方法可直接透過MATLAB內建的[反斜线運算子得到](../Page/反斜线.md "wikilink")\[48\]。

麥克·克雷默（Mike
Craymer）解釋道，相對於獨立處理各成分的作法（以及隆布的週期圖法），一次考慮所有成分的矩陣求解法無法擬合比取樣點數量還多的基底（即正弦與餘弦波），並指出：

相對地，隆布提出的方法便如同標準的[週期圖法](https://zh.wikipedia.org/wiki/週期圖法 "wikilink")，可使用任意多數或任意高密度的頻率成分做為擬合用的基底；也就是說，這個方法可在頻域下以任意的倍率進行過取樣\[49\]。

在傅立葉分析（例如[傅立葉變換或](https://zh.wikipedia.org/wiki/傅立葉變換 "wikilink")[離散傅立葉變換](https://zh.wikipedia.org/wiki/離散傅立葉變換 "wikilink")）中，用於擬合訊號的弦波皆為彼此正交，所以前述的獨立投影內積法和一次計算的矩陣求解最小平方法並沒有差異；也就是說，使用最小平方法將變異數分割至頻率不同且彼此正交的弦波時即不須計算[反矩陣](https://zh.wikipedia.org/wiki/反矩陣 "wikilink")\[50\]。由於此類分析可用[快速傅立葉變換的方法實作](https://zh.wikipedia.org/wiki/快速傅立葉變換 "wikilink")，當訊號完整且取樣間距相等時，便不會使用相對耗時的最小平方頻譜分析法。

在[数学的一个分支](../Page/数学.md "wikilink")[代数中](../Page/代数.md "wikilink")，**有序域**是一个[偏序关系通过](../Page/偏序关系.md "wikilink")[加法和](../Page/加法.md "wikilink")[乘法运算不被改变的](../Page/乘法.md "wikilink")[域](../Page/域_\(數學\).md "wikilink")。有序域最常见的例子是[实数](../Page/实数.md "wikilink")。

## 定义

一个满足下面两个条件的、拥有偏序关系\(\leq\)的域\((K,+,\cdot)\)被定义为有序域：对于任何\(K\)中的元素\(a, b, c\)以下两个条件获得满足：

  - 若\(a \leq b\)，则\(a + c \leq b + c\)。
  - 若\(0 \leq a\)且\(0 \leq b\)，则\(0 \leq a\cdot b\)。

大于0的元素被称为是**正的**，小于0的元素被称为是**负的**。

## 特性

由以上定义可以直接推导出以下特性（\(a, b, c, d\)是\(K\)的元素）：

  - 一个正的元素的负数是负的，一个负的元素的负数是正的：即任何\(K\)中的\(a\)，假如\(a\neq 0\)则\(-a<0<a\)或\(a < 0 < -a\)。
  - 不等式可以相加：\(a \le b\)和\(c \le d\)则\(a + c \le b + d\)。
  - 不等式可以与正元素相乘：\(a \le b\)和\(0 \le c\)则\(ac \le bc\)。
  - 平方数不是负的：\(0 \le a^2\)，尤其\(0 < 1\)。
  - 通过[数学归纳法可以推导出任何一的有限的和是正的](../Page/数学归纳法.md "wikilink")：\(0 < 1 + 1 + \cdots + 1\)。

## 结构

所有有序域都具有[特征数](../Page/特征_\(代数\).md "wikilink")0。这个结论直接出于上述的最后一个特性\(0 < 1 + 1 + \cdots + 1\)。

每个有序域的子域也是有序域。任何含特征数0的域其最小子域与[有理数同構](../Page/有理数.md "wikilink")，且这个子域的排序与\(\mathbb{Q}\)一致。

假如一个有序域中的任何元素都介于两个有理数之间的话，则该域具有[阿基米德性质](../Page/阿基米德性质.md "wikilink")。比如实数是具有阿基米德性质的，而[超实数则不具有](../Page/超实数.md "wikilink")。

有序域\(K\)的排序可用來定義\(K\)的[拓扑空间](../Page/拓扑空间.md "wikilink")，这个拓扑空间可由\(\{x \in K \mid x < a\}\)和\(\{x \in K \mid x > a\}\)作為[準基來生成](../Page/基_\(拓撲學\)#準基.md "wikilink")，稱之為[序拓撲](../Page/序拓撲.md "wikilink")。加法和乘法运算相对于这个拓扑空间是[连续的](../Page/連續函數_\(拓撲學\).md "wikilink")。

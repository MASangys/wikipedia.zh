用於書寫[朝鮮語](../Page/朝鮮語.md "wikilink")的[諺文](../Page/諺文.md "wikilink")有多種羅馬化方案，目前有兩套比較流行。[馬科恩-賴肖爾轉寫系統是](https://zh.wikipedia.org/wiki/馬科恩-賴肖爾轉寫系統 "wikilink")1937年由美國人[喬治·馬科恩與](https://zh.wikipedia.org/wiki/喬治·馬科恩 "wikilink")[埃德溫·賴肖爾合作創制的](https://zh.wikipedia.org/wiki/賴世和 "wikilink")，這套撰寫系統是影響力最大並在歐美廣為使用的韓文羅馬字轉寫系統，2000年韓國公佈的[韓文羅馬字表記法修正案也是建立在這套系統之上的](https://zh.wikipedia.org/wiki/韓文羅馬字表記法修正案 "wikilink")。

## 馬科恩-賴肖爾轉寫系統

馬科恩-賴肖爾轉寫系統根據韓文字母的實際發音選擇相對應的羅馬字母作為轉寫字母。在母音字母方面，對於羅馬字母中沒有的「」「」兩個單母音字母，該系統使用字母上方添加[短音符](../Page/短音符.md "wikilink")的方式來表達。而這也成了日後韓國新羅馬字標記法修正的對象之一．在子音字母的轉寫上，該系統不使用「b，g，d」來轉寫無氣音「」而使用「p，k，t」。對於有氣音的「」則採用在「p，k，t」添加「上標點」的方式來轉寫。

| ㅁ m  | ㅂ p   | ㅃ pp  | ㅅ s  | ㅆ ss | ㅇ ∅ / ng |  |

## 羅馬字表記法修正案

2000年韓國政府公佈了羅馬字標記法，由於該法是基於對馬科恩-賴肖爾轉寫系統部分規則的修正，所以一般以「修正案」或「修正版」稱呼它。

因為韓語沒有不送氣清音和濁音的對立，修正案的對馬-賴系統的重大修改有兩個。第一個是無氣音全部使用濁音字母表示，第二個是取消子音字母用添加符號的表達方法。

|         |          |          |         |         |          |  |
| ------- | -------- | -------- | ------- | ------- | -------- |  |
| **ㄱ g** | ㄲ kk     | ㄴ n      | **ㄷ d** | ㄸ tt    | ㄹ r / l  |  |
| ㅁ m     | **ㅂ b**  | ㅃ pp     | ㅅ s     | ㅆ ss    | ㅇ ∅ / ng |  |
| **ㅈ j** | **ㅉ jj** | **ㅊ ch** | **ㅋ k** | **ㅌ t** | **ㅍ p**  |  |
| ㅎ h     |          |          |         |         |          |  |

style="font-size:smaller;" | ****

  -
    粗體部分顯示了修正案與馬-賴系統相異的轉寫法
    「ㄱ」和「ㅋ」做韻尾的時候，轉寫為「k」。例如 곡 go<span style="color: blue">k</span>
    「ㄷ」、「ㅌ」、「ㅅ」和「ㅈ」做韻尾的時候，轉寫為「t」。例如 꽃 kko<span style="color: blue">t</span>
    「ㅂ」和「ㅍ」做韻尾的時候，轉寫為「p」。例如 탑 ta<span style="color: blue">p</span>

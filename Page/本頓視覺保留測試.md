**本頓視覺保留測試**（ 縮寫： BVRT）是一項針對8歲至成年期的測量[視覺和](https://zh.wikipedia.org/wiki/視覺 "wikilink")[視覺記憶的單獨施用測試](https://zh.wikipedia.org/wiki/視覺記憶 "wikilink")。同時它也可以用來幫助識別其他疾病中可能影響個體記憶的的[學習障礙](../Page/學習障礙.md "wikilink")。被試（被測試的個體）將會看到10個圖案（一次一個），並且需要從記憶中盡可能精確地在每張普通紙上重現每一個圖案。該測試不計時，最後通過畫在紙張的形式，形狀，圖案和排列進行專業評分。

## 發展歷程

是一位心理學家，在的軍事任務中與神經病學家一起工作。他在治療遭受[創傷性腦損傷的軍人的過程中受到啟發](../Page/腦損傷.md "wikilink")，從而發明了本頓視覺保留測試。\[1\]\[2\]本頓博士發明此測試是為了向即時非語言記憶提供更短的評估，以補充流行的數字廣度測試，最終選擇了能夠降低情緒影響和被試主觀影響的版本。\[3\]1946年，該測試正式發布。目前已經更新到了第5版。

## 測試

### 測試方法

[右](https://zh.wikipedia.org/wiki/File:BVRT.JPG "fig:右") 本頓視覺保留測試由用於測量被試的視覺和記憶能力的包含10張圖案（每張8.5×5.5英寸，即21.59×13.97厘米）的3套或多套表格以及一組用於重複測試的替代圖案組成。\[4\]受試者將得到一本有10張空白頁的小冊子，供他們再現圖案。這些圖案用於衡量被試的視覺和記憶，並且可以通過五種不同的方式進行。這五種方法如下：\[5\]

| 方法類型 | 內容                             |
| ---- | ------------------------------ |
| A    | 受試者在再現每個圖案之前可觀察10秒鐘            |
| B    | 受試者在再現每個圖案之前可觀察5秒鐘             |
| C    | 受試者可不限時地觀看每個圖案後才再現它們           |
| D    | 受試者觀看每個圖案10秒鐘，然後等待15秒後才再現它們    |
| M    | 受試者觀看每個圖案10秒鐘，然後從四個顯示屏中選擇正確的圖案 |

除了版本C以外的所有版本，原始圖案在再現開始之前都是隱藏的。

### 測試評分

該測試可以通過兩種方式進行評分。一是正確得分，這是基於全有或全無方法計算的; 如果圖案的複製品與原件相符，則獎勵點數。而另一種是錯誤得分，是根據每個圖案所產生的錯誤的數量和類型來計算的。這些錯誤的主要類型有遺漏、扭曲、固執、旋轉、錯位和大小錯誤等。\[6\]然後將這些得分與手冊中提供的規範性數據進行比較，每組數據代表不同的人口統計特徵，然後得出結論。\[7\]兩種不同的評分方法可以對個人測試進行[定量和](https://zh.wikipedia.org/wiki/定量分析 "wikilink")[定性分析](https://zh.wikipedia.org/wiki/定性分析 "wikilink")。\[8\]

## 用途

本頓測試對許多症狀的腦損傷和疾病很敏感，但通過測試很難診斷特定疾病。事實上，由測試手冊知，不應將大量單一類型的錯誤用作診斷依據，但應該用於進一步的測試。\[9\]例如，大量的持續言語表明[額葉](../Page/額葉.md "wikilink")損傷，而次要圖案的遺漏則表明潛在的[腦損傷](../Page/腦損傷.md "wikilink")，大機率是[右頂葉的損傷](https://zh.wikipedia.org/wiki/顶叶 "wikilink")。整體表現似乎不能區分那些[單側左半球或右半球的損傷](https://zh.wikipedia.org/wiki/半空間忽視 "wikilink")。\[10\]儘管如此，測試者已經證明能夠通過對測試結果的分析來區分感知、記憶和運動障礙。\[11\][癡呆症](../Page/失智症.md "wikilink")，[腦部病變](../Page/腦損傷.md "wikilink")，[丘腦中風和](https://zh.wikipedia.org/wiki/德杰林-鲁塞综合征 "wikilink")[阿爾茨海默病也包括在內](../Page/阿茲海默症.md "wikilink")，這些病已被證明會大大降低被試的BVRT評分。\[12\]該測試的重現和記憶版本對癡呆症特別敏感，並可能有助於識別將來可能患上阿爾茨海默氏症的人。此外，通過本頓測試還可以識別兒童的[學習障礙](../Page/學習障礙.md "wikilink")。\[13\]1983年，本頓測試被納入[神經行為核心組合](https://zh.wikipedia.org/wiki/神經行為核心組合 "wikilink")（NCTB），以測試化學品對神經系統的影響，後來用於評估工作場所的化學品暴露。\[14\]從其在NCTB中的應用來看，本頓測試可顯示出對接觸各種化學品的敏感性，包括[汞暴露](../Page/汞中毒.md "wikilink")，[鉛暴露和](../Page/鉛中毒.md "wikilink")[農藥暴露](https://zh.wikipedia.org/wiki/农药中毒 "wikilink")。\[15\]

## 心理測量學特性

根據本頓測試手冊所說，本頓測試的[重測試可靠性為](https://zh.wikipedia.org/wiki/重測信度 "wikilink")0.85，替代測試可靠性為0.79到0.84。立即和延遲記憶回憶（施用類型分別為A和D）之間的相關性範圍為0.40到0.83，該大小取決於所用測試方法的組合。手冊中記錄了兒童，青少年和成人的標準化結果，但每種施用方法都有自己的標準化數據。\[16\]需要注意的是，測試中的[總誤差會隨著被试年齡的增長而增加](../Page/总变差.md "wikilink")，尤其是當被試的年齡超過70歲。\[17\]

## 批評及其可靠性

本頓試驗受到的批評有該試驗對被試[智商](../Page/智商.md "wikilink")的要求較高，這要求在被試注意力不集中時可能很難達到。\[18\] 此外，研究中[重新測試的可行性的研究數據顯示出相互矛盾的結果](https://zh.wikipedia.org/wiki/可重複性 "wikilink")：一些研究表明，經過一段時間後再次測試，得分沒有差異，而其他研究則表明存在差異。\[19\]儘管有這些問題，本頓測試與其他視覺記憶測試相比仍然具有幾個優勢。\[20\]\[21\]這些優勢包括：比其他測試更清晰的評分系統，\[22\]因存在多個表單集而具有[並行形式的可靠性](https://zh.wikipedia.org/wiki/可靠性（心理測試學） "wikilink")，\[23\]以及較短的施用時間。 因此，BVRT目前仍廣泛用於診斷大量臨床病症。\[24\]

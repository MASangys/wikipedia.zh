**札幌大學**（さっぽろだいがく，Sapporo University，簡稱**札大**）是一所位於[日本](../Page/日本.md "wikilink")[北海道](../Page/北海道.md "wikilink")[札幌市](../Page/札幌市.md "wikilink")的[私立大學](../Page/私立大學.md "wikilink")，該校創立於1967年。

## 沿革

| 年    | 沿革                                                         |
| ---- | ---------------------------------------------------------- |
| 1967 | 札幌大學創校，開設經濟學部經濟學科、外國語學部英語學科及俄語學科                           |
| 1968 | 開設管理學部管理學科及札幌大學女子短期大學部（英文科、國文科）                            |
| 1982 | 女子短期大学部（開設文化學科、管理學科＜管理、秘書専攻＞，英文科和國文科，分別更名為英文學科和國文學科）       |
| 1989 | 開設法學部法學科                                                   |
| 1993 | 開設外國語專攻科（專攻英語）                                             |
| 1997 | 開設文化學部日本語日本文化學科、比較文化學科、管理學部產業情報學科、法學研究科碩士課程；女子短期大學部停止招生並廢止 |
| 2000 | 開設外國語學研究科碩士課程                                              |
| 2001 | 開設經濟學研究科與文化學研究科之碩士課程                                       |
| 2007 | 比較文化学科和日本語・日本文化学科合併後建立文化学科                                 |
| 2007 | 建校40周年                                                     |

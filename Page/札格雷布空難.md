**札格雷布空難**（）指的是1976年9月10日發生在當年[南斯拉夫](https://zh.wikipedia.org/wiki/南斯拉夫 "wikilink")[札格雷布](https://zh.wikipedia.org/wiki/札格雷布 "wikilink")（今日[克羅地亞的首都](https://zh.wikipedia.org/wiki/克羅地亞 "wikilink")）附近的[弗爾博韋茨市上空](https://zh.wikipedia.org/wiki/弗爾博韋茨 "wikilink")，一架[英國航空](../Page/英國航空.md "wikilink")的客機與一架[伊內克斯-亞德里亞航空在當地時間](https://zh.wikipedia.org/wiki/亞德里亞航空 "wikilink")10時14分，於該空域發生相撞，造成兩機上合共176人全部遇難，這亦是英航自1974年成立以來，兩次致命事故之一（另一次發生於1990年，[英國航空149號班機於](https://zh.wikipedia.org/wiki/英國航空149號班機 "wikilink")[科威特](../Page/科威特.md "wikilink")被襲擊，造成一人死亡）。

## 背景

1970年代中期，札格雷布的[航空交通管制](../Page/航空交通管制.md "wikilink")是當時歐洲其中一個最繁忙的管制區，但它同時卻面對著人手不足及設備落後的問題。札格雷布的[甚高頻全向信標是提供信息給眾多來自歐洲北部](https://zh.wikipedia.org/wiki/甚高頻全向信標 "wikilink")、歐洲東南部、[中東等地的航班的導航系統](https://zh.wikipedia.org/wiki/中東 "wikilink")。而當地[領空則劃分為三個區隔](https://zh.wikipedia.org/wiki/領空 "wikilink")：最低空域是24,000呎以下；中間位置是24,000至31,000呎；最高者為31,000呎以上高空。

## 事故班机

**英國航空476號班機**於當地時間早上8時32分，從英國[倫敦希斯路機場](../Page/倫敦希斯路機場.md "wikilink")飞往土耳其的[伊斯坦堡](../Page/伊斯坦堡.md "wikilink")。使用[霍克薛利三叉戟 3B](../Page/霍克薛利三叉戟型.md "wikilink")（機身編號G-AWZT）执飞。本班机的[機長为丹尼士](https://zh.wikipedia.org/wiki/機長 "wikilink")·塔蘭（Dennis Tann）（1932年6月21日出生），擁有10,781飛行小時經驗；他的搭檔為[副機師布萊因](https://zh.wikipedia.org/wiki/副機師 "wikilink")·希姆（Brian Helm）及[飛航工程師](../Page/飛航工程師.md "wikilink")馬丁·費連特（Martin Flint）。機上載著54名乘客及9名機員。476號班機一直穩定地飛行，直至進入札格雷布管制區。 [trident.arp.750pix.jpg](https://zh.wikipedia.org/wiki/File:trident.arp.750pix.jpg "fig:trident.arp.750pix.jpg")

**伊內克斯-亞德里亞航空550號班機**於早上9時48分，從南斯拉夫的[斯普利特](../Page/斯普利特.md "wikilink")（現位於克羅地亞南部）起飛前往聯邦德國的[科隆](../Page/科隆.md "wikilink")。使用[DC-9](https://zh.wikipedia.org/wiki/DC-9 "wikilink")(機身編號YU-AJR)执飞。駕駛艙內，負責駕駛的為正機長荷茲·古坎姆巴克（Jože Krumpak）及副機長道森·艾雲奴斯（Dušan Ivanuš）。機上共有108名乘客及5名機組人員。 [Navy-dc9-N932ML-071129-01adj-16.jpg](https://zh.wikipedia.org/wiki/File:Navy-dc9-N932ML-071129-01adj-16.jpg "fig:Navy-dc9-N932ML-071129-01adj-16.jpg")

## 航空交通管制

在空難發生時，札格雷布的空中監控中心內共有5名管制員在工作（還有一名最高級的督導員）：

  - Julio Dajčić，他是最高級的督導員（生於1932年12月17日）
  - Gradimir Tasić（1949年4月29日出生），高級區域管制員（出事時）
  - Nenad Tepeš（1943年9月2日出生），高級區域管制員
  - Mladen Hochberger（1946年10月13日出生），高級區域管制員
  - Bojan Erjavec（1947年6月15日出生），中級區域管制員
  - Gradimir Pelin（1948年4月18日出生），中級區域管制員

## 意外經過

早上10時04分，英航476號班機與札格雷布的空中監控中心取得聯繫，並與高級區域管制員Gradimir Tasić取得溝通。以下為當時兩者對話：

|          |        |                                                                                                           |
| -------- | ------ | --------------------------------------------------------------------------------------------------------- |
| 10:04:12 | 476號班機 | 札格雷布，這是英航476班機，早安！                                                                                        |
|          | 札格雷布   | 英航476，早安，請繼續前進。                                                                                           |
| 10:04:19 | 476號班機 | 嗯，476班機，02分到達[克拉根福](../Page/克拉根福.md "wikilink")，[空層](../Page/飛航空層.md "wikilink")330（33000呎），估計於14分到達札格雷布。 |
| 10:04:27 | 札格雷布   | 英航476，收到，請於飛過札格雷布後通知我，及維持[空層](../Page/飛航空層.md "wikilink")330，頻道2312。                                      |
| 10:04:38 | 476號班機 | 2312收到。                                                                                                   |

同時，亞德里亞550號班機亦與中級區域管制員Bojan Erjavec取得聯繫，詢問最高飛行高度。當時飛機位於26,000呎高空，而當時28,000至31,000呎並不能開放予550號班機，於是Erjavec指示550號班機爬升至35,000呎高空，機長亦接受。

為了騰出該空域予550號班機爬升，首先必須得到高級區域管制員的許可。Erjavec揚手示意Tasić's，但Tasić當時非常忙碌未予理會，於是Pelin代替Tasić指導550號班機爬升。

Pelin於是走到航班編排表，拿住代表550號班機的標籤，詢問Tasić可否讓550號班機爬升至35,000呎。Tasić於是從Pelin取了550號班機標籤看了一看，問該班機處於甚麼位置，Pelin指著屏幕上550班機的光點，就在[Kostajnica附近](https://zh.wikipedia.org/wiki/Kostajnica "wikilink")（位於札格雷布東南方）。Tasić於是回答：「好，讓他爬升吧！」。Pelin於是將550號班機的標籤貼在[Metlika](https://zh.wikipedia.org/wiki/Metlika "wikilink")（今[斯洛文尼亞南部](https://zh.wikipedia.org/wiki/斯洛文尼亞 "wikilink")）位置，然後告知Tasić「等待他們通過」。Pelin回到自己的位置上確認了自己已經安排了550號班機接著的行程，然後回到Tasić處並留意著被Tasić授權爬升的550號班機，與476號班機互相接近對方。然後Pelin通知Erjavec「好，讓550號班機爬升」。Erjevec於是於10時07分（40秒）通知550號班機爬升至35,000呎高空。

於10時12分（03秒），550號班機呼叫札格雷布的中央區域管制區，告知飛機高度31,000呎。之後，Erjavec將會停止550班機的[異頻雷達收發機控制器](https://zh.wikipedia.org/wiki/異頻雷達收發機 "wikilink")134.45MHz。同時Erjavec於航管中央處理正發佈550號班機一個代碼，於此過渡期，550號班機於航管的雷達屏幕上只是一個光點而已。如果在正常情況下，550號班機於雷達屏幕上將會顯示其航班編號及高度，但因航管員不當的工作程序，令550號班機一直只是一個光點，沒有顯示高度及航班編號。此外，Tasić當時正忙著指揮其他航班，而550號班機也沒有即時跟航管員通話。這有可能是當時的頻率已很繁忙，但550號班機的機員或許有其他不明原因而延遲了跟航管員對話。

於10時14分，550號班機飛越32,500呎高空後，隨即與高級區域管制員取得聯繫：

|          |        |                                                           |
| -------- | ------ | --------------------------------------------------------- |
| 10:14:04 | 550號班機 | Dobar dan（今天好）札格雷布，這是亞德里亞550班機。                           |
| 10:14:07 | 札格雷布   | 亞德里亞550，這是札格雷布，今天好，請繼續前進。                                 |
| 10:14:10 | 550號班機 | 現時[空層](../Page/飛航空層.md "wikilink")325（32,500呎），14分越過札格雷布。 |
| 10:14:14 | 札格雷布   | 現在你的高度是多少？                                                |
| 10:14:17 | 550號班機 | 327（32,700呎）。                                             |

| 10:14:58 | 札格雷布  | 我不是呼叫你……                 |

兩機在札格雷布附近的弗爾博韋茨上空發生相撞，550號班機的左翼末端5米左右把476號班機的駕駛艙削掉，然後立即以機鼻朝下姿勢往下俯衝；476號班機的機身在空中短暫飛行後，跟著[失速](../Page/失速.md "wikilink")下墜。476班機上的63人及550號班機上的113人，無一生還。而亞德里亞航空副機長最後一句對Tasic所溝通的最後一句話是:「我們完了\!再見\!(We're finished, good bye\!)」

一架[德國漢莎航空的](https://zh.wikipedia.org/wiki/德國漢莎航空 "wikilink")[波音737](../Page/波音737.md "wikilink")客機，於29000呎高、距離英航476班機後方僅15英哩處目擊了兩機相撞，甚至親眼目睹兩機相撞後雙雙墜地的全過程。副機長及機長先後多次回報給薩格勒布控制室。

## 審判

當天下午，控制中心內的所有人都被拘捕及被審問。不久，除了Tasić於開審期間被拘留外，所有人都獲釋。

審判於1977年4月11日於札格雷布地方法院展開，所有當日當值的控制員在南斯拉夫的刑事法律下被起訴，並被指「這些人等危害鐵路、航海及航空，及威脅人們及財產之安全。」

Tasić是唯一被判有罪的航管員，並被判處有期徒刑7年。但後來一批航管員為其請願，指Tasić是這次悲劇的[代罪羔羊](https://zh.wikipedia.org/wiki/代罪羔羊 "wikilink")。最後他於1978年11月29日被釋放，其間他在牢獄被囚禁了2年又3個月。

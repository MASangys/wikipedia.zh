## 线路分类

该号段描述仅为杭州公交编号原则。现由于城市急速扩张及公交线路频繁变更，加之部分号段资源紧张和编码体系缺陷，故存在一定混乱，含有大量特例，请以实际线路信息为准。

| 线路编码                | 线路类别                                                                                                         | 是否可用公交IC优惠卡 |
| ------------------- | ------------------------------------------------------------------------------------------------------------ | ----------- |
| **B1-B8**           | [快速公交](https://zh.wikipedia.org/wiki/快速公交 "wikilink")（[BRT](https://zh.wikipedia.org/wiki/BRT "wikilink")）   | 是           |
| **B支1-B支8**         | [快速公交](https://zh.wikipedia.org/wiki/快速公交 "wikilink")（[BRT](https://zh.wikipedia.org/wiki/BRT "wikilink")）支线 | 是           |
| **1-149、170-179**   | 主城区汽车线路                                                                                                      | 是           |
| **150-169**         | [电车线](https://zh.wikipedia.org/wiki/无轨电车 "wikilink")（153、156路目前为汽车线）                                         | 是           |
| **180-199**         | 大站快线                                                                                                         | 是           |
| **200-259**         | [夜间通宵线](https://zh.wikipedia.org/wiki/夜间通宵线 "wikilink")，[晚间线](https://zh.wikipedia.org/wiki/晚间线 "wikilink")  | 是           |
| **260-299**         | 早晚[高峰线](https://zh.wikipedia.org/wiki/高峰线 "wikilink") （部分线路常规化，但高峰线路不仅限于该号段）                                 | 是           |
| **300-449**         | 郊区线路，其中部分为萧山区区内线路                                                                                            | 是           |
| **450-499**         | 余杭区区内线路，主要开行于余杭区西片                                                                                           | 是           |
| **501-599、868**     | 专线、跨市线路                                                                                                      | 否           |
| **600-649**         | 富阳区区内线路，主要开行于富阳城区                                                                                            | 是           |
| **650-759**         | 萧山区区内线路                                                                                                      | 是           |
| **760-799**         | 余杭区区内线路，主要开行于余杭区东片                                                                                           | 是           |
| **801-867、869-899** | 临安区及少量萧山区公交线路                                                                                                | 是           |
| **900-990**         | 专线，临时线路                                                                                                      | 否           |
| **990-999**         | 特服线                                                                                                          | 否           |
| **1000-1999**       | 社区微公交                                                                                                        | 是           |
| **2000-2599**       | 富阳区公交线路，主要为城乡线路或村村通线路                                                                                        | 是           |
| **2700-2999**       | 临安区公交线路，主要为村村通线路                                                                                             | 是           |
| **Y1-Y14**          | 游览线                                                                                                          | 否           |

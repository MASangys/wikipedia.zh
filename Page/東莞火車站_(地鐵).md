## 車站出入口

東莞火車站地鐵設有五個出入口，分別為A，B，C，D，H。不設E，F，G出入口。目前已開通A，B，D，H出入口。

| 編號 | 目的地              |
| -- | ---------------- |
| A  | 東莞火車站石龍廣場出入口（地下） |
| B  | 東莞火車站檢票口（地下）     |
| C  | 東莞火車站茶山廣場        |
| D  | 石龍西湖城市公園         |
| H  | 東莞火車站石龍廣場公交站     |

**柴灣站**（）是[香港](../Page/香港.md "wikilink")[港鐵](../Page/港鐵.md "wikilink")[港島綫](../Page/港島綫.md "wikilink")東端終點[鐵路車站](https://zh.wikipedia.org/wiki/鐵路車站 "wikilink")，位於[香港島](../Page/香港島.md "wikilink")[東區](../Page/東區_\(香港\).md "wikilink")[柴灣](../Page/柴灣.md "wikilink")，1985年5月31日啟用。柴灣站自落成而來，一直是港鐵系統中最南端車站，此地位於2016年已被[南港島綫](../Page/南港島綫.md "wikilink")的[利東站](../Page/利東站.md "wikilink")所取代。

## 車站設計

柴灣站是港島綫東面總站，並且是港島綫唯一的架空車站。它設於[柴灣道](../Page/柴灣道.md "wikilink")[迴旋處旁](https://zh.wikipedia.org/wiki/迴旋處 "wikilink")，連接中西區的[4號幹線東面起點](https://zh.wikipedia.org/wiki/4號幹線 "wikilink")，同時為[東區走廊](../Page/東區走廊.md "wikilink")起點，前往港島各区。

### 大堂

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Chai Wan Station 2017 10 part6.jpg | caption1 = 大堂 | image2 = Chai Wan Station Exterior 2016.jpg | caption2 = 車站外觀 }} 站內提供不同類型的商店供乘客購物或進食，例如有[便利店](../Page/便利商店.md "wikilink")、[麵包糕餅店](https://zh.wikipedia.org/wiki/麵包糕餅店 "wikilink")、[銀行](../Page/銀行.md "wikilink")及化妝品店等\[1\]\[2\]。由於港鐵的收費區是不能進食，因此售賣[食物](../Page/食物.md "wikilink")的商店均不會設在收費區內。

此外，柴灣站也有不少自助服務設施供乘客使用，包括[自動櫃員機](../Page/自動櫃員機.md "wikilink")、自動照相機等\[3\]\[4\]。近B出入口的大堂亦設有[香港郵政](../Page/香港郵政.md "wikilink")[郵箱](../Page/郵箱.md "wikilink")，方便乘客投寄[郵件](../Page/郵件.md "wikilink")\[5\]\[6\]。另外，大堂亦設有「[會員服務站](../Page/會員服務站.md "wikilink")」，供乘客享用港鐵友禮會的會員優惠\[7\]。

### 月台

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Chai Wan Station MTR Track.jpg | caption1 = 港島綫東端、位於此車站以南的架空路軌盡頭，靠近[興華廣場](https://zh.wikipedia.org/wiki/興華廣場 "wikilink") }} 車站設有2個月台，以[島式月台](../Page/島式月台.md "wikilink")排列，兩個月台的列車均前往[堅尼地城方向](../Page/堅尼地城站.md "wikilink")\[8\]，並不設落客專用月台及調頭路轨，列車到站後會在該月台同一時間上落客及調頭，乘客可按指示於1號月台或2號月台乘搭港島綫往堅尼地城方向的列車。

[柴灣車廠雖然在](https://zh.wikipedia.org/wiki/港鐵車廠#柴灣車廠 "wikilink")[杏花邨站](../Page/杏花邨站.md "wikilink")旁邊，然而實際車務上柴灣站為最直接連接車廠的車站，每當上下班繁忙時間之後，部份港島綫的列車在本站清客後，便會暫停服務，返回該車廠。列車入站時，月台會在該列車到站前及列車入站時發出廣播。該列車到站後，目的地顯示屏將會顯示*回廠*，而月台顯示屏左下方將顯示*不載客列車*，使候車的乘客知道該列車在到達車站後，只作落客之用，並不會接載乘客。該列車關門離開車站前，在月台的西端的紅藍訊號燈上將會亮起*DP*（意思為Depot）字樣的白色顯示訊號燈，之後列車離開車站，駛回柴灣車廠。

## 利用狀況

{{ multiple image | align = right | direction = vertical | width = 220 }} 柴灣站位於柴灣市中心一帶，鄰近住宅及工業區，因此有不少區內乘客利用車站往返其他地區。而且車站底部設有一個公共運輸交匯處，有大量乘客在此轉乘其他交通工具往[小西灣](../Page/小西灣.md "wikilink")、[哥連臣角](https://zh.wikipedia.org/wiki/哥連臣角 "wikilink")、[赤柱](../Page/赤柱.md "wikilink")等地；而在[清明節及](https://zh.wikipedia.org/wiki/清明節 "wikilink")[重陽節](../Page/重陽節.md "wikilink")期間，因部份乘客前往哥連臣角掃墓和步行往小西灣，車站人流十分繁忙。

## 歷史

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Chai Wan Station.jpg | caption1 = 安裝月台閘門前月台（2008年） }}

### 啟用

柴灣站由[永保建築國際有限公司興建](https://zh.wikipedia.org/wiki/永保建築國際有限公司 "wikilink")，於1985年5月31日與港島綫第一期同時啟用。原址前身為填海地，後設置為舊小巴站及柴灣公園。\[9\]

### 月台閘門

現時柴灣站所有月台已完成裝上[月台閘門](../Page/月台閘門.md "wikilink")，並已投入運作。該月台閘門高大約1.5米。整個計劃於平日深夜終止服務的時間加建，已於2011年5月完成。此站與[杏花邨站](../Page/杏花邨站.md "wikilink")屬於第一期進行月台閘門加裝工程，原計劃於2010年8月完成，但由於工程進度緩慢，第二度預計於2010年12月完成。

### 翻新月台

與市區線很多非地下車站一樣，柴灣站由2011年起也開始展開翻新工程，當中包括在大堂及月台安裝假天花、改善照明，亦翻新了月台指示吊板：原「往上環」的行車方向資訊則改為張貼在月台閘門之上；舊有吊板上「柴灣」站名標示形式保留，但改用車站顏色底色加大號白字標示，猶如昔日[荔景站](../Page/荔景站.md "wikilink")及[大窩口站](../Page/大窩口站.md "wikilink")月台吊牌標示站名方式，於港鐵系統中屬碩果僅存。

## 未來發展

### 延伸至小西灣（已擱置）

小西灣已發展為一個成熟的社區，計有大型運動場、多所學校及多個大型私人屋苑及大型公共屋邨，市民對公共交通需求極大，巴士及小巴服務早已不勝負荷，早上巴士及小巴排隊人龍，逾一公里，實有迫切需要興建港鐵[小西灣支線去解決市民出行問題](https://zh.wikipedia.org/wiki/小西灣支線 "wikilink")。雖然多位區議員經過多年爭取也無功而還。

然而港鐵港島綫在設計以至開通初期，[小西灣](../Page/小西灣.md "wikilink")未有發展計劃，故無預計向東延伸，但在柴灣站末端已預留一段超越線路軌，可供延長之用。月台也設計成西南走向，因此向東延伸技術上是可行的，政府曾建議地鐵公司研究把港島綫從柴灣延伸至小西灣站，但地鐵公司認為路線迂迴（須興建穿越[歌連臣角](../Page/歌連臣角.md "wikilink")的隧道前往小西灣），且有技術難度，故未被地鐵接納。地鐵亦研究過利用[輕便鐵路](../Page/輕便鐵路.md "wikilink")把小西灣及柴灣連接，但由於沒有足夠地方興建車站上蓋物業，以致造價及融資等各種問題多番推搪而不了了之。2009年10月23日，[運輸及房屋局](../Page/運輸及房屋局.md "wikilink")官員在立法會交通事務委員會的會議上表示\[10\]，受到柴灣站附近的樓宇及其他建築物之限制，延伸港島綫至小西灣的工程難度頗高，相信方案難以落實；但局方和港鐵會繼續關注小西灣對外的交通接駁情況。此外，興建[小西灣支線](https://zh.wikipedia.org/wiki/小西灣支線 "wikilink")（杏花邨站↔小西灣站，採類似[欣澳站](../Page/欣澳站.md "wikilink")的模式（需要額外購入類似[南港島線的](https://zh.wikipedia.org/wiki/南港島線 "wikilink")3卡車[中型鐵路系統](../Page/中型鐵路系統.md "wikilink")穿梭列車）；或採取[將軍澳站](../Page/將軍澳站.md "wikilink")的分拆模式，成本較小，惟這將會削減往返柴灣站的班次），但因須填海興建而擱置。

現時如需利用港鐵往返小西灣，須在此站轉乘[新巴82M線](../Page/新巴82M線.md "wikilink")或小巴47M線，或於[杏花邨站](../Page/杏花邨站.md "wikilink")轉乘[城巴85線](../Page/城巴85線.md "wikilink")或小巴62線。此外，因為大部分港島東的巴士均在[小西灣（藍灣半島）公共運輸交匯處](../Page/小西灣（藍灣半島）公共運輸交匯處.md "wikilink")設為巴士總站，所以小西灣延線最終會落實興建的機會已較低，因此西貢區議員劉偉章和觀塘區議員蔡澤鴻已建議在柴灣站建設架空行人天橋連接工業區和小西灣取代。

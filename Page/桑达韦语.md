**桑达韦语**是[坦桑尼亚](../Page/坦桑尼亚.md "wikilink")[多多马地区](../Page/多多马.md "wikilink")[桑达韦人使用的一种](https://zh.wikipedia.org/wiki/桑达韦人 "wikilink")[声调语言](https://zh.wikipedia.org/wiki/声调语言 "wikilink")。它在年长和年青人之中仍然具有活力，有些地区的桑达韦人甚至只说桑达韦语。Albert
Drexel在1920年代把它划归[科依桑语系](../Page/科依桑语系.md "wikilink")，但依据仅仅是该语言中存在[搭嘴音](../Page/搭嘴音.md "wikilink")。近来的研究表明它与[科爾語系的联系可能更紧密](https://zh.wikipedia.org/wiki/科爾語系 "wikilink")。

桑达韦语有西北和东南两大方言。两者之间的区别在于语速、声调、禁忌词，以及词汇、语法上的细微差别。一些[阿拉瓜人放弃了母语转而使用桑达韦语](https://zh.wikipedia.org/wiki/阿拉瓜人 "wikilink")。

## 语法

### 代词

自由代词

|       | 单数    | 复数    |
| ----- | ----- | ----- |
| 第一人称  | tsi   | sũũ   |
| 第二人称  | hapu  | sĩĩ   |
| 第三人称男 | he-we | he-so |
| 第三人称女 | he-su |       |

代词后缀

|       | 单数   | 复数   |
| ----- | ---- | ---- |
| 第一人称  | \-és | \-wà |
| 第二人称  | \-i  | \-è  |
| 第三人称男 | \-à  | \-ʔà |
| 第三人称女 | \-sà |      |

### 名词

名词由词干和后缀组成，后缀是为了标明阳性、阴性、中性以及单复数。

### 语序

桑达韦语的基本语序是[主宾谓结构](https://zh.wikipedia.org/wiki/主宾谓结构 "wikilink")，但实际语序却十分灵活。

**梅德韦杰夫·施蓬霍伊尔·卡尔尼克地震烈度表**（，缩写**MSK**或**MSK-64**），是一种用于评估地震[烈度的度量](https://zh.wikipedia.org/wiki/烈度 "wikilink")，以量化[地震对某一特定地点所受到的影响](../Page/地震.md "wikilink")。

该烈度表最初由[苏联科学家谢尔盖](../Page/苏联.md "wikilink")·梅德韦杰夫、[东德科学家施蓬霍伊尔和](https://zh.wikipedia.org/wiki/东德 "wikilink")[捷克斯洛伐克科学家卡尔尼克于](https://zh.wikipedia.org/wiki/捷克斯洛伐克 "wikilink")1964年提出。
它基于20世纪60年代初从[麦加利地震烈度和](https://zh.wikipedia.org/wiki/麦加利地震烈度 "wikilink")1953版的梅德韦杰夫烈度度量的应用所获得的经验，因此也称为GEOFIAN烈度度量\[1\]。

## 发展历史

在20世纪70年代中期和80年代初期进行了小幅修改后，MSK烈度表在[欧洲和](../Page/欧洲.md "wikilink")[苏联得到广泛应用](../Page/苏联.md "wikilink")。
在20世纪90年代初，[欧洲地震委员会](https://zh.wikipedia.org/wiki/欧洲地震委员会 "wikilink")（ESC）采用了MSK的许多原则开发[欧洲宏观地震度量](../Page/欧洲地震烈度表.md "wikilink")，该标准现已成为欧洲国家地震烈度评估的事实标准。直到现在，MSK-64烈度表仍被[印度](../Page/印度.md "wikilink")、[以色列](../Page/以色列.md "wikilink")、[俄罗斯以及](../Page/俄罗斯.md "wikilink")[独联体国家所使用](https://zh.wikipedia.org/wiki/独联体 "wikilink")。

## 烈度划分

梅德韦杰夫·施蓬霍伊尔·卡尔尼克地震烈度表与美国使用的[麦加利地震烈度](https://zh.wikipedia.org/wiki/麦加利地震烈度 "wikilink")（MM）量表有些相似。MSK有12个强度等级，用罗马数字表示：

| <big><big>I.无感</big></big>                                        | 没有感觉，只有仪器能观察到。                                                                   |
| :---------------------------------------------------------------- | -------------------------------------------------------------------------------- |
| <big><big>II.几乎无感</big></big>                                     | 在静止的情况下，少部分人有感觉。                                                                 |
| <big><big>III.轻微震颤</big></big>                                    | 大部分室内的人感觉到震动，悬挂的物体轻微摆动。                                                          |
| <big><big>IV.普遍有感</big></big>                                     | 室外的人感觉到震动，少数人惊醒。有中等强度的震荡。家中饰物摇晃，但不对建筑物造成破坏。                                      |
| <big><big>V.强烈摇晃</big></big>                                      | 所有人都有感觉；睡者惊醒，牲畜不宁，摆钟停摆，重心不稳的物件可能倾倒。                                              |
| <big><big>VI.轻微破坏</big></big>                                     | 许多人慌张逃跑，走路摇晃；玻璃破碎，书籍或摆设从书架掉下，家具移动或倾倒，破坏轻微。                                       |
| <big><big>VII.中等破坏</big></big>                                    | 大多数人害怕并试图逃出室内。家具移位，可能倾倒。物体从架子上掉下来。水从容器中溅出。对旧建筑造成严重破坏，砖石烟囱倒塌，会造成小规模滑坡。            |
| <big><big>VIII.严重破坏</big></big>                                   | 烟囱、纪念碑、塔、墙垮塌，重型家具偏移。未周密考虑抵抗外力的建筑物遭受某种损坏，部分严重损毁。水平方向抗力弱的建筑物倒塌。破坏力强。               |
| <big><big>IX.毁坏</big></big>                                       | 居民普遍恐慌。对构造良好的建筑造成重大损害。地下管线破裂，地面破裂。会造成大规模山体滑坡。                                    |
| <span style="color: white;"><big><big>X.严重毁坏</big></big></span>   | 砖石建筑被摧毁，基础设施瘫痪。会造成大规模的山体滑坡。水体可能会被覆盖，导致周围地区泛滥，形成[堰塞湖](../Page/堰塞湖.md "wikilink")。 |
| <span style="color: white;"><big><big>XI.灾难性毁坏</big></big></span> | 只有少数建筑物尚未倒塌。                                                                     |
| <span style="color: white;"><big><big>XII.山河改观</big></big></span> | 天崩地裂，所有建筑物遭受毁灭性破坏。地貌有极大改动，河流变道，会造成[海啸](../Page/海啸.md "wikilink")。                |

[Principales_vias_pecuarias.png](https://zh.wikipedia.org/wiki/File:Principales_vias_pecuarias.png "fig:Principales_vias_pecuarias.png") **梅斯塔荣誉会**（），又译**美斯塔\[1\]**，是西班牙[卡斯蒂利亚国王](../Page/卡斯蒂利亚王国.md "wikilink")[阿方索十世于](../Page/阿方索十世_\(卡斯蒂利亞\).md "wikilink")1273年建立的全国性[游牧](https://zh.wikipedia.org/wiki/游牧 "wikilink")[畜牧业](../Page/畜牧业.md "wikilink")从业者[同業公會](https://zh.wikipedia.org/wiki/同業公會 "wikilink")，荣誉会伴随的家畜迁徙在16世纪前期达到顶峰，后来随着[半岛战争](../Page/半岛战争.md "wikilink")等多重因素影响而逐渐没落，最终于1836年解散。

梅斯塔荣誉会的迁移路线“梅斯塔家畜[小径](https://zh.wikipedia.org/wiki/小径 "wikilink")”已经作为[文化遗产](../Page/文化遗产.md "wikilink")项目，由西班牙文化部于2007年4月27日提交至[联合国教科文组织世界遗产预备名单](https://zh.wikipedia.org/wiki/联合国教科文组织世界遗产 "wikilink")。\[2\]

## 历史

西班牙[卡斯蒂利亚](../Page/卡斯蒂利亚.md "wikilink")地区拥有充裕的土地，在[中世纪](../Page/中世纪.md "wikilink")早期就发展出了羊毛工业。牧羊人在[夏季](../Page/夏季.md "wikilink")把羊群赶到高地，在[冬季](../Page/冬季.md "wikilink")赶到低地。国王[阿方索十世在一次财政危机中意识到对牲畜征税比对人征税更为容易](../Page/阿方索十世_\(卡斯蒂利亞\).md "wikilink")。\[3\]他在1260年左右与牧羊人签下了一系列许可\[4\]，1273年正式建立全国性同业公会——“梅斯塔荣誉会”。作为[纳税的交换条件](https://zh.wikipedia.org/wiki/纳税 "wikilink")，梅斯塔荣誉会的畜牧业者从国王手中获得了一系列司法和经济特权。他们不仅可以免交一些杂税，还有权利组织警卫队保护牲畜，并能与路途中的[地主进行利益谈判](https://zh.wikipedia.org/wiki/地主 "wikilink")。\[5\]\[6\]

自16世纪以来，梅斯塔荣誉会每年举办两次全国性集会，一次在南部于1月至2月举行；另一次在北部，时间大致在9月至10月。\[7\]

梅斯塔牧道按照宽度可分为三种类型：\[8\]

| 名称       | 宽度       |
| -------- | -------- |
| cañadas  | 不超过75米   |
| cordeles | 不超过37.5米 |
| veredas  | 不超过20米   |

## 词源

人们一般认为“mesta”一词来自[拉丁语](../Page/拉丁语.md "wikilink")“mixta”（西班牙语“mezclar”），中文即“混合”，指某一位牧羊人拥有的羊在迷路后混入了其他牧羊人的羊群，这样的状况经常发生。也有认为来自西班牙语动词“amistar”，意指牧羊人之间的友好关系。另有学者推测它为北非[柏柏尔语来源](../Page/柏柏尔语族.md "wikilink")，最初指阿尔及利亚[柏柏尔人](../Page/柏柏尔人.md "wikilink")的冬季营地，后来成为牧羊人周期性会议的代称。\[9\]

## 现况

西班牙1995年3月23日第3/1995号牧道法律明确牧道为不可移动的公共财产并强调其生态保护意义。各段牧道归属相应的[自治区管理](https://zh.wikipedia.org/wiki/西班牙自治区 "wikilink")，任何破坏行为将受到处罚。\[10\]

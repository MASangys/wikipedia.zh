**梵蒂冈先进技术望远镜**（，简称**VATT**）由**爱丽丝·P·列侬望远镜**和**Thomas J. Bannan天体物理设施**组成，它是一种可在[可见光和](https://zh.wikipedia.org/wiki/可见光天文学 "wikilink")[红外波段进行观测的](../Page/红外天文学.md "wikilink")[格里望远镜](https://zh.wikipedia.org/wiki/格里望远镜 "wikilink")，位于[美国](../Page/美国.md "wikilink")[亚利桑那州东南部](https://zh.wikipedia.org/wiki/亚利桑那州 "wikilink")，并在1993年实现了[开光](https://zh.wikipedia.org/wiki/开光 "wikilink") (第一次观测或取得影像的行为) 。

VATT是的一部分，由世界上最早的天文研究机构之一[梵蒂冈天文台与](https://zh.wikipedia.org/wiki/梵蒂冈天文台 "wikilink")[亚利桑那大学](../Page/亚利桑那大学.md "wikilink")合作运营。

## 设计

|         |                                |
| ------- | ------------------------------ |
| 光学系统    | *Aplanatic Gregorian* f/9      |
| 焦距      | 16.48 m                        |
| 主镜      | f/1.0，直径 1.83 m                |
| 次镜      | f/0.9，直径 0.38 m （焦距控制: 0.1 mm） |
| 视野      | 72 mm (15')                    |
| 标度      | 12.52 "/mm                     |
| 画面质量    | 0.1 ' - 6.8 "                  |
| 天文望远镜安装 | Alt-Az +直径                     |

望远镜的核心是由f/1.0蜂窝结构的硅酸盐主镜组成，因其焦距为f/1所以视野的跟踪异常快速，f/1意味着焦距等于直径，而又如此短的焦距，所以可以采用[格里设计](https://zh.wikipedia.org/wiki/格里望远镜 "wikilink")，可以在主焦点之外的点用凹面次镜，使它在视野范围内进行异常清晰的聚焦\[1\]。 不寻常的光学设计和新颖的镜面制造技术意味着主镜和副镜都是有史以来最精确的地面望远镜镜面。 此外，上方的天空是北美洲大陆最清澈，最稳定，最黑暗的天空之一。 即使没有自适应光学系统，通常也可以实现优于一秒弧秒的效果。

## 建设

[popescope.JPG](https://zh.wikipedia.org/wiki/File:popescope.JPG "fig:popescope.JPG") 主镜由[亚利桑那大学](../Page/亚利桑那大学.md "wikilink")的[[斯图尔德天文台](../Page/斯图尔德天文台.md "wikilink")制镜实验室](http://mirrorlab.as.arizona.edu/)制造，该[实验室开创了旋转铸造和应力圈抛光技术](https://zh.wikipedia.org/wiki/实验室 "wikilink")，这些技术被用于包括6.5米光圈MMT和Magellan望远镜，以及两个直径8.4米的[大双筒望远镜](../Page/大双筒望远镜.md "wikilink")镜片。

## 研究

鉴于其卓越的光学质量，该望远镜主要用于成像和[光密度测定工作](../Page/光度测定_\(天文学\).md "wikilink")，它通常在性能上优于设立在他处的大型天文望远镜。VATT的成果显著， 在[仙女座星系](../Page/仙女座星系.md "wikilink")发现了[晕族大质量致密天体](https://zh.wikipedia.org/wiki/晕族大质量致密天体 "wikilink")，验证了 Stromvil 光度过滤系统，找到关于星系的形状和尺寸随着宇宙年龄而改变的证据 ，发现了第一个二进制 '灶神星芯片'[小行星](https://zh.wikipedia.org/wiki/小行星 "wikilink")，以及通过对100个[海王星外天体的可见颜色进行分类来分析其表面特征](https://zh.wikipedia.org/wiki/海王星外天体 "wikilink")，而且其中大部分天体都比21[等星都暗](../Page/视星等.md "wikilink")。

## 资金

[梵蒂冈](../Page/梵蒂冈.md "wikilink")政府提供[梵蒂冈天文台的工作人员和定期研究费用](https://zh.wikipedia.org/wiki/梵蒂冈天文台 "wikilink")，但建立和维持VATT本身的成本来自私人捐助者：支持VATT建设的主要捐助者是Fred和Alice P. Lennon以及 Thomas J. Bannan。梵蒂冈天文台基金会的捐助者继续支持VATT的运营成本。

**森林公园**为[中国大陆](../Page/中国大陆.md "wikilink")、[台湾](https://zh.wikipedia.org/wiki/台湾 "wikilink")、[泰国](../Page/泰国.md "wikilink")、[马来西亚](../Page/马来西亚.md "wikilink")、[新西兰](../Page/新西兰.md "wikilink")、[塞浦路斯等地的](https://zh.wikipedia.org/wiki/塞浦路斯 "wikilink")[休闲](https://zh.wikipedia.org/wiki/休闲 "wikilink")、[旅游](../Page/旅游.md "wikilink")、[环境保护](../Page/环境保护.md "wikilink")和[生态多样性相关的用语](https://zh.wikipedia.org/wiki/生态多样性 "wikilink")。

## 森林公园的定义

在[中华人民共和国](https://zh.wikipedia.org/wiki/中华人民共和国 "wikilink")，森林公园指具有一定规模和质量的森林风景资源与环境条件，可以开展森林旅游，并按法定程序申报批准的森林地域\[1\]。

## 森林公园的分级

[中华人民共和国](https://zh.wikipedia.org/wiki/中华人民共和国 "wikilink")[林业部于](../Page/国家林业局.md "wikilink")1993年12月11日颂布《森林公园管理办法》，《办法》第六条规定森林公园分为以下三级：国家森林公园、省级森林公园和市、县级森林公园\[2\]。

| 类型       | 概念                                                                                          | 代表 |
| -------- | ------------------------------------------------------------------------------------------- | -- |
| 国家森林公园   | 森林景观特别优美，人文景物比较集中，观赏、科学、文化价值高，地理位置特殊，具有一定的区域代表性，旅游服务设施齐全，有较高的知名度，可供人们游览、休息或进行科学、文化、教育活动的场所。 |    |
| 省级森林公园   | 森林景观优美，人文景物相对集中，观赏、科学、文化价值较高，在本行政区域内具有代表性，具备必要的旅游服务设施，有一定的知名度                               |    |
| 市、县级森林公园 | 森林景观有特色，景点景物有一定的观赏、科学、文化价值，在当地知名度较高。                                                        |    |

## 国家森林公园的分类

在[中华人民共和国](https://zh.wikipedia.org/wiki/中华人民共和国 "wikilink")，国家森林公园按主要景观特征大致划分为以下11类\[3\]：

| 类型          | 概念                                                                              | 代表                       |
| ----------- | ------------------------------------------------------------------------------- | ------------------------ |
| 山岳型         | 以名山大川、奇峰异石景观而闻名。我国是多山的国家，故山岳型国家森林公园在我国最为普遍。                                     |                          |
| 湖泊型         | 以大面积水体为特征，多建立在森林环绕的高远平湖或人工水库湖周围，其湖光山色相映成辉。                                      | 光明森林公园（深圳）\[4\]          |
| 火山遗址型       | 多建立在火山的遗址地区。其公园景观具有火山遗川与森林景观相结合而形成的独特魅力。                                        | 蜀山森林公园（合肥）\[5\]          |
| 沙漠与沙地型      | 建立在沙漠、沙地或荒漠的绿洲上，展现了一幅自然界中生命与恶劣环境抗争的生动画卷。                                        | 金沙滩国家森林公园（沈阳）\[6\]       |
| 现代冰川 与冰川遗迹型 | 以现代冰川、冰川遗迹与原始森林镶嵌形成的风景区，海拔较高，景观独特。                                              | 海螺沟冰川森林公园（四川）\[7\]       |
| 海岛型         | 由海岛与森林相结合而形成多山，天然森林环岛而生。                                                        | 玉环大鹿岛（台州）\[8\]           |
| 海滨型         | 由天然林或沿海防护林形成，多为海滩与森林相结合而构成的独特景观。 海滨型国家森林公园在我国从南至北均有分布。                          | 日照海滨国家森林公园（山东）\[9\]      |
| 溶洞型         | 由地下大型溶洞与地上森林景观相结合而形成，多在喀斯特地貌上存在。                                                | 特克斯县科桑溶洞森林公园（新疆）\[10\]   |
| 温泉型         | 是大面积的天然森林景观与温泉相结合而形成的国家森林公园，兼具峰峦密林和溪河幽谷。                                        | 七仙岭温泉国家森林公园（海南）\[11\]    |
| 草原型         | 是在大型草原上建立的国家森林公园。其特点是草原开阔、牛羊肥美、森林茂密、景色优美。                                       |                          |
| 城郊园林型       | 地处城市郊区，以大面积的森林景观和人文景观为特色。这类公园大多是在历史悠久、 人文景观突出的名胜之地建立起来的，一般面积不大，景点密集，有较多人造景观河设施。 | 洛阳的枕头山森林公园、翠屏山森林公园\[12\] |

## 森林公园的功能

[Parke-Ghale-RoodKhaan.JPG](https://zh.wikipedia.org/wiki/File:Parke-Ghale-RoodKhaan.JPG "fig:Parke-Ghale-RoodKhaan.JPG")

### 游玩休憩

森林公园里植被覆盖率高，嗓音低，水质清新，空气里负离子高，有“氧吧”之称\[13\]，为游客们提供了一个放松身心的好场所\[14\]。

### 保护自然

通过对森林公园进行管理，可以遏制居民和游客对自然环境的破坏\[15\]。《森林公园管理办法》第十二条就禁止在森林公园毁林开垦和毁林采石、采砂、采土以及其他毁林行为\[16\]。

### 科研与教育

森林公园里动物、植物、微生物众多，具有很大的研究价值\[17\]。在中国多个森林公园里都传出了新发现珍稀物种的新闻，如承德平泉森林公园内发现国家濒危保护物种[沼兰](https://zh.wikipedia.org/wiki/沼兰 "wikilink")\[18\]，福州森林公园发现国家一级保护动物东方白鹳\[19\]。游客在森林公园里近距离接触大自然，有益于增近对自然的了解，唤醒环境保护的意识\[20\]。

## 森林公园之最

  - 建立最早的森林公园：张家界国家森林公园（1982年）\[21\]
  - 海拔最高的森林公园：太白山国家森林公园（620米-3511米）\[22\]

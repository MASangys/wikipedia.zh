**楚科奇語語法**（）為基於[楚科奇語之](https://zh.wikipedia.org/wiki/楚科奇語 "wikilink")[語法](https://zh.wikipedia.org/wiki/語法 "wikilink")。在語法的分類上歸為[古西伯利亞語言之](../Page/古西伯利亞語言.md "wikilink")[楚科奇－堪察加语系](../Page/楚科奇－堪察加语系.md "wikilink")，且並不同於一般的[分析語或其它](https://zh.wikipedia.org/wiki/分析語 "wikilink")[综合语裡的語式結構及變化](../Page/综合语.md "wikilink")。楚科奇語是一種主要具有[粘著法構詞](https://zh.wikipedia.org/wiki/粘著法構詞 "wikilink")(agglutination)特性之[作通格語言](../Page/作通格語言.md "wikilink")。它還含有非常普遍的[併合](https://zh.wikipedia.org/wiki/併合 "wikilink")(incorporation
(linguistics))結構，亦屬於[元音和谐律的語言](../Page/元音和谐律.md "wikilink")。

## 名詞

名詞被分為[有生性的三個變格](../Page/有生性.md "wikilink")。第一變格，其中包含非人稱，僅在[通格有複數標記](../Page/通格.md "wikilink")。第二變格，其中包含個人名稱以及主要為老舊親屬的詞語，在各種形式上均為制性的複數標記。第三個變格，其包含除了第二變格外的其他人稱，為選擇性的複數標記。

名詞結構上：有兩種語法[數的特徵](../Page/数_\(语法\).md "wikilink")，大約有九個[格](../Page/格_\(语法\).md "wikilink")（[通格](../Page/通格.md "wikilink")、[作格](../Page/作格.md "wikilink")、[工具格](../Page/工具格.md "wikilink")、[方位格](../Page/方位格.md "wikilink")、[离格](../Page/离格.md "wikilink")、[向格](../Page/向格.md "wikilink")(allative
case)、[定向格](https://zh.wikipedia.org/wiki/定向格 "wikilink")(orientative
case)、二個[随伴格](https://zh.wikipedia.org/wiki/随伴格 "wikilink")(comitative
case)，及一個[指定格](https://zh.wikipedia.org/wiki/指定格 "wikilink")(designative
case)）。

## 动词

動詞分為三個[人稱](https://zh.wikipedia.org/wiki/人稱 "wikilink")，兩個[數](../Page/数_\(语法\).md "wikilink")，三種[语气](https://zh.wikipedia.org/wiki/语气 "wikilink")（陳述，祈使及條件），兩種[语态](../Page/语态.md "wikilink")（主動語態及[反被動語態](../Page/反被動語態.md "wikilink")）以及六個[时态](../Page/时态.md "wikilink")：現在式
I（進行式），現在式 II（[靜態](../Page/静态动词.md "wikilink")），過去式
I（[不定過去時](../Page/不定過去時.md "wikilink")），過去式
II（[完成體貌](https://zh.wikipedia.org/wiki/完成體貌 "wikilink")），未來式
I（未來完成式），未來式I I（未來未完成）。有意思的是，過去式 II
是以[所有](https://zh.wikipedia.org/wiki/所有 "wikilink")(Possession
(linguistics))的形式（字面意思為"與...(to be
with)"）所形成，類似於在英語和其他西歐語言的所使用的"有"(have)。

主詞及直接賓語交叉引用在語言鏈裡，且人稱[一致性在](../Page/一致_\(語法\).md "wikilink")[不及物動詞與](../Page/不及物動詞.md "wikilink")[及物動詞上有很大的不同](../Page/及物動詞.md "wikilink")。人稱一致性涉以[前綴與](https://zh.wikipedia.org/wiki/前綴 "wikilink")[後綴的複雜系統來表達](../Page/後綴.md "wikilink")；儘管語言的[粘著](https://zh.wikipedia.org/wiki/粘著 "wikilink")(agglutination)特性，對人稱、數及時態的每個體組合，總是以相當直截了當的語式來表示。除了[有限形式](https://zh.wikipedia.org/wiki/有限形式 "wikilink")(finite
forms)外、也有[不定詞](../Page/不定詞.md "wikilink")、動詞不定式的(目的式/supine
(purposive))、數種[動名詞形式](../Page/動名詞.md "wikilink")、以及現在分詞與過去分詞，而這些大都與[助動詞併用](../Page/助動詞.md "wikilink")；可基於此進一步的來分析句式的結構。

### 及物动词与不及物动词

在楚科奇语中，及物和不及物动词的形态变化完全不同，这个特征和[汉藏语系](../Page/汉藏语系.md "wikilink")[藏缅语族的](https://zh.wikipedia.org/wiki/藏缅语族 "wikilink")[嘉绒语支](https://zh.wikipedia.org/wiki/嘉绒语支 "wikilink")[嘉绒语相似](../Page/嘉绒语.md "wikilink")。

#### 不及物動詞

不及物动词的动词形态比较简单，只有六个形式：

|    |                   |      |             |         |
| -- | ----------------- | ---- | ----------- | ------- |
| 主语 | 动词形态（阳性）          | 汉语翻译 | 动词形态（阴性）    | 汉语翻译    |
| 我  | tə-kətɣəntat-ɣ'ak | 我跑了  | tə-r'e-ɣ'ek | 我做了什么？  |
| 我们 | mət-kətɣəntat-mək | 我们跑了 | mət-r'e-mək | 我们做了什么？ |
| 你  | kətɣəntat-ɣ'e     | 你跑了  | r'e-ɣ'i     | 你做了什么？  |
| 你们 | kətɣəntat-tək     | 你们跑了 | r'e-tək     | 你们做了什么？ |
| 他  | kətɣəntat-ɣ'e     | 他跑了  | r'e-ɣ'i     | 他做了什么？  |
| 他们 | kətɣəntat-ɣ'at    | 他们跑了 | r'e-ɣ'et    | 他们做了什么？ |

*楚科奇语不及物动词变位（过去时）*

以上是*kətɣəntat-ək*“跑步”和*req-ək* “做什么？”的形态变化，第二个动词是人称代词*r'enut*
“什么”的[派生词](https://zh.wikipedia.org/wiki/派生词 "wikilink")。
从以上表格可以看出，不及物动词不分第二和第三人称的单数形式（都使用-ɣ'i /-ɣ'e后缀）。

#### 及物動詞

[及物动词形态变化比较复杂](https://zh.wikipedia.org/wiki/及物动词 "wikilink")，一共有24个不同的形态。以下表格介绍楚科奇语及物动词的形态变化，以
*ԓ'у(l'u)*
"看"为例。横格表示[主语](https://zh.wikipedia.org/wiki/主语 "wikilink")，竖格表示[宾语](https://zh.wikipedia.org/wiki/宾语 "wikilink")：
<span lang=ru>

|     |             |             |             |             |              |
| --- | ----------- | ----------- | ----------- | ----------- | ------------ |
| | 我 | 我们          | 你           | 你们          | 他           | 他们           |
| 我   |             |             | tə-l'u-ɣət  | tə-l'u-tək  | tə-l'u-ɣ'en  |
| 我们  |             |             | mət-l'u-ɣət | mət-l'u-tək | mət-l'u-ɣ'en |
| 你   | ine-l'u-ɣ'i | l'u-tku-ɣ'i |             |             | l'u-ɣ'en     |
| 你们  | ine-l'u-tək | l'u-tku-tək |             |             | l'u-tkə      |
| 他   | ine-l'u-ɣ'i | ne-l'u-mək  | ne-l'u-ɣət  | ne-l'u-tək  | l'u-nin      |
| 他们  | ne-l'u-ɣəm  | ne-l'u-mək  | ne-l'u-ɣət  | ne-l'u-tək  | ne-l'u-ɣ'en  |

*楚科奇语及物动词变位（过去时）*

例如：*Мыт-ԓ'у-г'эн. (mət-l'u-ɣ'en)* “我们看见了他”，*Инэ-ԓ'у-тык. (ine-l'u-tək)*
“你们看见了我” 人称范畴的词缀可以如此分析：

  - ты (tə-) 第一人称单数（主语）
  - мыт (mət)- 第一人称复数（主语）
  - гыт (-ɣət) 第二人称单数（宾语）
  - тык (-tək) 第二人称复数（应该把-tku 和 -tkə 分析成-tək-u 和 tək-ə）
  - нэт (-net) 第三人称复数

### 動詞轉化

楚科奇语是[多式綜合語](https://zh.wikipedia.org/wiki/多式綜合語 "wikilink")（<span lang=en>polysynthetic
language</span>），往往可以把及物动词转化成不及物动词，并把原来的宾语作为这个不及物动词的一个前缀。在这种情况下，宾语和动词词根之间将会发生元音和谐。例如：

  - *tə*-ɣərit-*ənet* qora-t（及物动词+宾语，这个形式和以上表格中的*tə*-l'u-*net*相同）
  - 我守护鹿群。
  - *tə*-qora-ɣəret-*ɣ'ak*（不及物动词，原先的宾语qora被插入第一人称前缀tə-和动词词根之间，动词词根ɣərit受到qora的同化影响，经过了元音和谐；这个形式和以上的*tə*-kətɣəntat-*ɣ'ak*相同）。
  - 我赶了鹿群出去放牧。

ӄоpат(qora-t) “驯鹿”（ӄоpаӈы/qora-ŋə/的复数），гыритык(ɣərit-ək) “守护”

## 数词

5"等)。[括號內為](https://zh.wikipedia.org/wiki/括號 "wikilink")[IPA發音及註記](https://zh.wikipedia.org/wiki/IPA "wikilink")。

## 語序

詞序是相當靈活的，基本上是以[SOV語序為基本的語序](../Page/主賓動語序.md "wikilink")。所有者位置通常先於所有修飾格，即採用[後置介詞法而不是](../Page/介詞.md "wikilink")[前置介詞法](../Page/介詞.md "wikilink")。

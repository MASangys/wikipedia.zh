**樂佩公主**（），是[華特迪士尼影業第](https://zh.wikipedia.org/wiki/華特迪士尼影業 "wikilink")50部[經典動畫長片](https://zh.wikipedia.org/wiki/華特迪士尼經典動畫長片 "wikilink")《[魔髮奇緣](../Page/魔髮奇緣.md "wikilink")》（2010年）中的女主角，之後亦於該片後續的番外篇短片《魔髮奇緣：大喜之日》（，2012年）中和同名电视剧集（，2017年）以及电视电影《魔髮奇緣：幸福前奏》（，2017年）登場。在兩片中均由[美國演員暨歌手](https://zh.wikipedia.org/wiki/美國 "wikilink")[曼蒂·摩爾](https://zh.wikipedia.org/wiki/曼蒂·摩爾 "wikilink")[配音](../Page/配音.md "wikilink")，由其角色動畫總監、迪士尼首席動畫師[葛連·基恩創造](https://zh.wikipedia.org/wiki/葛連·基恩 "wikilink")。她於2011年10月2日被迪士尼正式列為第10位
[迪士尼公主](../Page/迪士尼公主.md "wikilink")，也因此成了第1位以[電腦動畫繪成的迪士尼公主](https://zh.wikipedia.org/wiki/電腦動畫 "wikilink")。

樂佩公主的角色原型是[格林兄弟](../Page/格林兄弟.md "wikilink")[童話](../Page/童話.md "wikilink")《[長髮姑娘](../Page/長髮姑娘.md "wikilink")》中的長髮公主，而在迪士尼版本中，樂佩的身世背景被設定為一位金色魔髮公主，幼時被貪圖其頭髮魔力的女巫——葛索媽媽（）從王宮裡偷走，關在與世隔絕的高塔中撫養，以用頭髮的魔力使自己長保年輕。長大之後成為一位具有藝術天分和矯捷身手的少女，後來在本名尤金·費茲柏特（）的大盜費林·雷德（）幫助下逃出高塔，以前往探索小时候在窗邊看到的美麗星火（實際上是王國為了引回失蹤公主而施放的大量[天燈](../Page/天燈.md "wikilink")）。

對於樂佩公主一角的評價大多相當正面，這些評論中又主要推崇其自由奔放、嚮往獨立而具有現代感的人格。樂佩因為具有純真、活潑的個性，且充滿對未知世界的好奇心，而常被拿來與同為迪士尼公主的[愛麗兒](https://zh.wikipedia.org/wiki/愛麗兒 "wikilink")（1989年《[小美人魚](https://zh.wikipedia.org/wiki/小美人魚 "wikilink")》主角）比較。

## 創造

### 初始構思

|                                                                                         |
| --------------------------------------------------------------------------------------- |
| 「樂佩是個受到遏制的女孩，使她不能變為原本應該要成為的樣子。而我便覺得她的長髮象徵著必須洋溢出來的潛力。每次你看著她，就會想到這女孩的心靈是不容被束縛的，連頭髮都長到了長。」 |
| — 葛連·基恩繪製樂佩的構圖有感\[1\]                                                                   |

當葛連·基恩作為一名動畫師，參與製作[泰山的](https://zh.wikipedia.org/wiki/泰山_\(電影\) "wikilink")1996年時\[2\]，便萌生了將格林童話《長髮姑娘》改編為動畫長片的想法\[3\]。之後，將樂佩創造成動畫角色的想法當作主要目標\[4\]、而著手起步的基恩，逐漸投身該項企畫之中，著迷於「一個人天生被賦予才能，並且要使之激發出來」的概念，接著將這點與自己過去在[華特迪斯尼動畫工作室工作時的經驗連結起來](https://zh.wikipedia.org/wiki/華特迪斯尼動畫工作室 "wikilink")\[5\]。他將自己認為是一位「因為從角色的心境思考，並由此開始發展一個故事」而「與眾不同的導演」\[6\]。在2008年因為一次[心臟病發作後](https://zh.wikipedia.org/wiki/心臟病 "wikilink")，基恩決定退出，請納森·格萊諾（）和[拜倫·霍華德接手工作](../Page/拜倫·霍華德.md "wikilink")，但自己仍然擔任著該企畫的總策劃者、兼樂佩的角色動畫總監\[7\]。

基恩不是第一位想將長髮姑娘的角色給動畫化的人，[華特·迪士尼本人早在](../Page/華特·迪士尼.md "wikilink")1937年[白雪公主推出後](../Page/白雪公主.md "wikilink")，即試圖將長髮姑娘的故事改編為長篇動畫，但後來這樣的構想，卻因為故事本身被認為是「一個難以著手的燙山芋」而遭到放棄。基恩相信這乃由於長髮姑娘大部分的情節都是在高塔內發展的緣故，他後來說道：「這就是為何...為何當看到拜倫和納森讓樂佩走出塔外時，會覺得如此的完美...這為童話原著注入了不一樣的新意」\[8\]。身為導演的格萊諾和霍華德，還進一步將樂佩塑造為一位比原著中更積極向外的女主角，並提到：「我們知道自己正在為當代的觀眾呈現此片，而希望樂佩能成為更貼近真實的角色，也希望她的充沛能量能駕馭起整個故事。因此她不為任何事情等待，她是聰穎的女孩，懷有希望和夢想，而且將會從生命中得到自己所想要的。」\[9\]。

### 配音人選

在最初，製作群方面宣佈的[配音者是百老匯演員](../Page/配音.md "wikilink")[克莉絲汀·肯諾維斯](https://zh.wikipedia.org/wiki/克莉絲汀·肯諾維斯 "wikilink")\[10\]\[11\]\[12\]，之後他們更有一短暫時期相中女演員[瑞絲·薇斯朋來擔當此角](https://zh.wikipedia.org/wiki/瑞絲·薇斯朋 "wikilink")\[13\]，但最終她仍未出任配音工作\[14\]。在物色過數百名人選之後\[15\]，最終選中了錄音藝人、創作歌手兼女演員[曼蒂·摩爾](https://zh.wikipedia.org/wiki/曼蒂·摩爾 "wikilink")。根據副導霍華德所說，因為她「嗓音中帶有如此絕妙的靈魂」，及「純樸、[鄰家女孩般的特質](https://zh.wikipedia.org/wiki/鄰家女孩 "wikilink")，使她具備了迪士尼片中女主角所需的任何條件」\[16\]。

從小便熱愛迪士尼動畫片的曼蒂·摩爾，將這次得到配音工作的參與機會形容為「無上的夢幻」\[17\]。她在初期因為聽說樂佩一角的配音甄試相當競爭激烈，而幾無意願加入，害怕落選之後只會純粹帶來失望\[18\]\[19\]。在下定決心參加後，據說摩爾以近乎「窮追」的方式來爭取這個機會，參加了兩次試音甄試。由於《魔髮奇緣》一片被設定為[歌舞片](../Page/歌舞片.md "wikilink")，因此參選者都必須進行說、唱兩方面的試驗。甄試時，在評審要求以[創作歌手風試唱的情形下](https://zh.wikipedia.org/wiki/唱作人 "wikilink")\[20\]，摩爾表演了自己在2003年專輯《》中的《》，其翻唱自[瓊尼·米歇爾的同名經典歌曲](https://zh.wikipedia.org/wiki/瓊尼·米歇爾 "wikilink")\[21\]\[22\]\[23\]。

當加入《魔髮奇緣》的配音陣容後，摩爾還不知道自己參與的將是迪士尼動畫工作室的第50部經典動畫長片。此後她很感激自己對此一無所知，解釋道：「我覺得很幸運，因為若是在開始配音工作前就知道這件事的話，就將受到更大的壓力。」\[24\]。之後，她也因為發現自己大部份的配音都不會與另兩名主要配音員——[柴克·萊威](../Page/柴克·萊威.md "wikilink")（飾費林·雷德）和（飾葛索媽媽）一起在場進行錄製，而感到驚訝。據說配音過程期間，摩爾僅有在錄製片中的男女對唱曲《》時才與萊威碰到面\[25\]，而摩爾之後上「[艾倫·狄珍妮秀](../Page/艾倫·狄珍妮秀.md "wikilink")」時，也表示自己與墨菲更是到影片上映前一個月的某次記者會才相見\[26\]。摩爾亦形容配音錄製的過程具有相當的挑戰性，因為配音時幾乎無法看到劇中的動畫畫面。她之後接受[英國](https://zh.wikipedia.org/wiki/英國 "wikilink")訪問時提到：「可供我揣摩的就只有幾張草圖而已，但配音過程也因此變得很有趣，因為這樣讓我可以進入更深的想像空間中」。而在被問及如何創造出樂佩的嗓音時，摩爾說自己純粹就「豁出去」\[27\]。她經常被要求將同一段台詞重配至最多四遍，直到導演滿意為止。摩爾表示當她首次看到自己配音的成品搭配動畫播映時，心裡想的是「天啊，我的聲音真的很尖」\[28\]。

### 角色的詳細設計

|                                                                                                                                                                       |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 「我為樂佩繪製了大量的草圖，而想讓她的外觀有些許的不對稱。曾經讀過一本論述女性美的書，它提到美的關鍵在於女性面龐上的奇特性。臉孔上的要素——如鼻子、嘴唇、牙齒等——必須有輕微的偏位，或是一眼略高於另一眼等。甚至是樂佩的牙、她講話的方式，在她牙齒的位置也會有一點點的晃動。透過諸如此類的刻意安排，就讓她看上去顯得更真實而楚楚動人。」 |
| — 基恩談論樂佩的外貌設計及「女性美」概念\[29\]                                                                                                                                           |

葛連·基恩在擔任擔任樂佩的角色動畫總監時，曾在之一的資深動畫師——指導下進行角色設計\[30\]\[31\]。在看過基恩對樂佩的鉛筆稿測試（）後，強司頓建議基恩，在捕捉她的動作時不是僅有呈現她在做的事情，還要融入她內心的想法。強司頓說道：「我感好奇的是，她*在想些什麼*？」\[32\]。基恩後來把強司頓的建議比喻作「讓我無法忘懷的一記當頭棒喝，因此之後當我在繪製人類主題的作品時，總試著進入該角色的中心想法」\[33\]。

至於副導拜倫·霍華德個人則是被1989年迪士尼動畫[小美人魚中的女主角](https://zh.wikipedia.org/wiki/小美人魚 "wikilink")，即同樣由基恩創造出來的愛麗兒所激發。根據霍華德的闡述，「愛麗兒是第一個讓我覺得在眼神裡流露著靈魂的角色...我們也希望讓樂佩的眼裡能找到吸引人的靈魂和深情」。另一方面，希望能「將手繪帶入電腦」\[34\]的基恩，也提到了小美人魚愛麗兒和長髮公主樂佩的相似性，說這兩位女主角同樣「被藩籬隔閡而無法享受夢想，且都懷有不容被壓抑的精神」\[35\]。當設計角色時，基恩受一本介紹「女性美」（）的書影響。在將書中「美的關鍵在於女性面龐上的奇特性」這個概念納入考量後，基恩使樂佩的外貌帶有些微的，在面容上也有數處程度微小的缺陷，例如刻意被設計到略為突出的門牙、鼻子週邊分布的[雀斑等](https://zh.wikipedia.org/wiki/雀斑 "wikilink")\[36\]。基恩也將樂佩的雙眼畫大，以讓她增添些許「不受拘束」的氣質。動畫師們總共創造出了9個版本的樂佩，才決定出最受滿意的設計\[37\]。

#### 性格與靈感來源

[Rapunzel-Disney.jpg](https://zh.wikipedia.org/wiki/File:Rapunzel-Disney.jpg "fig:Rapunzel-Disney.jpg")。\]\]

|                                                                                                                                                                   |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 「她（指樂佩）也具有藝術創意，驅使她必須在牆上作畫、創造作品。然後我想，嗯，我認識的人中誰像她一樣的？然後就想到是小女克萊兒。她從小就一直說：『媽，我想在天花板上畫畫！我有個點子想在上面畫出來！』。後來她去念藝術學校，畢業之後又正好是我需要找人幫樂佩設計畫風的時候。所以後來樂佩在牆上的繪圖，其實都出自我女兒克萊兒之手。」 |
| — 葛連·基恩提及樂佩的藝術天分\[38\]                                                                                                                                            |

《魔髮奇緣》執行製作人[約翰·拉薩特有次向](https://zh.wikipedia.org/wiki/約翰·拉薩特 "wikilink")[紐約時報解釋道](https://zh.wikipedia.org/wiki/紐約時報 "wikilink")：「挑戰在於，你想讓樂佩這個18年來足不出戶的女孩子，能令人覺得像是個聰明伶俐、有教養、健康而又有趣的人類」\[39\]。為了避免創造出來的是個「過度公主氣又孤僻」的女主角，製作人員在設定樂佩的角色性格時，略為參考了一些女性名人，如作為創意靈感的女演員瑞絲·薇斯朋、[娜塔莉·波曼](https://zh.wikipedia.org/wiki/娜塔莉·波曼 "wikilink")，及「[週六夜現場](../Page/週六夜現場.md "wikilink")」演員班底中的[艾米·波勒](../Page/艾米·波勒.md "wikilink")。結果使樂佩融入了些許「難以捉摸和古靈精怪」的氣質，以「重塑人們以往對公主的印象」\[40\]。紐約時報亦提到樂佩這樣的性格是一次顯著的變更，與先前迪士尼動畫的諸位女主角區別開來了。紐約時報評論家巴恩斯（）指出樂佩「相當剛強；她在費林爬到她的塔內時，用[平底鍋把他敲到眼花](https://zh.wikipedia.org/wiki/平底鍋 "wikilink")，而且用起頭髮來像是甩長鞭一樣」\[41\]。而根據曼蒂·摩爾的說法，樂佩「並不是...典型的迪士尼公主...她很獨立自主，可以照顧好自己」\[42\]。

實際上，角色動畫總監葛連·基恩，本身就因為曾將自己的數位家人作為基礎來創造角色而聞名，如小美人魚愛麗兒的外型便是參考他的妻子\[43\]。而本片中，樂佩對於藝術和創作的熱情則是參考自他的女兒克萊兒·基恩（）\[44\]，樂佩在高塔家中的壁畫和天花板上所塗的圖案，也都是克萊兒的作品\[45\]。而基恩忙於《魔髮奇緣》製作期間，克萊兒替他生下的孫女瑪諦斯（）則被當成靈感來創造嬰孩時期的樂佩\[46\]\[47\]。

#### 長髮與製作技術

|                                                                                                                                      |
| ------------------------------------------------------------------------------------------------------------------------------------ |
| 「頭髮...已被證明是全片中最大的挑戰，因為這正是樂佩的招牌賣點，所以必須要在螢幕上看起來栩栩如生。導演說為了創造它，動畫師們繪出了一堆像義大利麵條的管子，『大約有1千根這樣的管子，實際上共代表了10萬根的毛髮，使動畫師們可以從這些管子掌握到大致上的運動姿態』。」 |
| — 愛荷華州《》採訪霍華德與格萊諾之後的報導內容\[48\]                                                                                                       |

樂佩是繼1959年[睡美人女主角奧蘿拉](../Page/睡美人_\(電影\).md "wikilink")（）後，多年來迪士尼的第一位[金髮女主角](../Page/金髮.md "wikilink")\[49\]。利用[電腦生成圖像](https://zh.wikipedia.org/wiki/電腦生成圖像 "wikilink")（）技術繪出她的長髮，在《魔髮奇緣》一片的製作中已被普遍認為是最具挑戰性的環節\[50\]。據《[洛杉磯時報](../Page/洛杉磯時報.md "wikilink")》報導，首席動畫師葛連·基恩已經因為製作過迪士尼系列電影中數個「最偉大的毛髮作品」而揚名，當中包括了小美人魚愛麗兒、《[美女與野獸](https://zh.wikipedia.org/wiki/美女與野獸_\(1991年電影\) "wikilink")》中的野獸、《[風中奇緣](../Page/风中奇缘_\(电影\).md "wikilink")》中的[寶嘉康蒂](https://zh.wikipedia.org/wiki/寶嘉康蒂 "wikilink")\[51\]。對於樂佩的長髮，基恩形容它「提醒人們樂佩是被天生賦予魔力的」\[52\]，霍華德則打趣地戲稱這長髮本身就可以成為一獨立的角色\[53\]。霍華德與格萊諾規定麾下的動畫師以實物當參考，以構思如何呈現樂佩的長髮及它運動的形態，例如將串好的長繩綁在[棒球帽後](https://zh.wikipedia.org/wiki/棒球帽 "wikilink")、戴著它到處走動，甚至招募多年沒剪髮的女性進入團隊，充當真人模特兒\[54\]。

[北卡羅萊納大學畢業的資深](https://zh.wikipedia.org/wiki/北卡羅萊納大學 "wikilink")[軟體工程師和毛髮模擬專家凱莉](https://zh.wikipedia.org/wiki/軟體工程師 "wikilink")·華德（）\[55\]負責製作特殊軟體，來協助動畫師們創造樂佩的長髮\[56\]。華德表示在真實世界中，70英呎的長髮將會重達60[磅](../Page/磅.md "wikilink")（27[公斤](https://zh.wikipedia.org/wiki/公斤 "wikilink")），「重到無法使一個真人能像我們讓樂佩在電影中那樣地擺動自如」\[57\]。為了簡單起見，動畫師保留了頭髮在外觀上的擬真程度，但減少了表面內部包含的髮絲數量以便於控制\[58\]。而為了要替長髮調出顯得獨特、同時卻又不失真實的金黃色澤，動畫師也在這方面遇到了一番挑戰\[59\]。

| —《》劇評家凱利·藍格爾（）對樂佩的評論\[60\]                                                                          |

樂佩公主一角在大部分的評論裡都很受到歡迎。《》的莎拉·維茲卡隆多（）喜歡這個角色，形容她「是位活力十足的女主角，以活水浸潤了嚴謹的公主角色典型」\[61\]。紐澤西《》的史蒂芬·惠迪（）形容該角色是「一位才華洋溢的年輕女子」\[62\]。《洛杉磯時報》的也在文中表示樂佩「是位具有現代感的年輕女子」\[63\]。《[紐約客](../Page/紐約客.md "wikilink")》的布魯斯·迪奥内斯（）形容樂佩「有與生俱來的慧黠和靈敏的關懷力，而又不會顯得像個聒噪的青少女」\[64\]。《[今日美國](../Page/今日美國.md "wikilink")》的克勞蒂亞·普伊格（）則在文中指出，與之前數位年齡相仿的迪士尼公主相比，「樂佩扮演的青少年角色顯得較為可信」\[65\]。

在其他將樂佩與先前諸位迪士尼公主相比的評論中。《[紐約每日新聞](../Page/紐約每日新聞.md "wikilink")》的喬·紐瑪爾（）將樂佩的獨立自主與[美女與野獸中的貝兒相作比較](https://zh.wikipedia.org/wiki/美女與野獸 "wikilink")\[66\]。英國《》雜誌的強納森·克羅克（）指出了樂佩與愛麗兒的相似性，形容前者是「一位意志堅強的女主角，渴望能一睹外頭的世界」\[67\]。紐澳良《》的麥可·史考特（）
在文中表示樂佩的無邪「讓人想起[艾美·亞當斯在](https://zh.wikipedia.org/wiki/艾美·亞當斯 "wikilink")[曼哈頓奇緣中](../Page/曼哈頓奇緣.md "wikilink")，飾演的那位天真爛漫的公主吉賽兒」\[68\]。影評人湯姆·查瑞帝（）
則將樂佩的獨立自主同時與愛麗兒和[花木蘭相比](https://zh.wikipedia.org/wiki/花木蘭_\(1998年電影\) "wikilink")，並提到樂佩「替迪士尼近年慣例下的豪放不拘型女主角中，再添一例」\[69\]。

也有部分影評對樂佩顯示出的好感較少。《[時代週刊](https://zh.wikipedia.org/wiki/時代週刊 "wikilink")》的儘管很喜歡魔髮奇緣這部電影，卻覺得聚焦於費林·雷德的部分太多，而對樂佩的著墨不足。他寫道：「60年來...女孩子是被期待能長大、克服困境、變得更有擔當的角心人物」，但「[皮克斯](https://zh.wikipedia.org/wiki/皮克斯 "wikilink")、[夢工廠等工作室卻破壞了這些以女性為核心的電影](https://zh.wikipedia.org/wiki/夢工廠 "wikilink")」\[70\]。網的線上影評人表示樂佩「雖然很可愛又活潑，但仍不如[白雪公主](../Page/白雪公主.md "wikilink")、愛麗兒或貝兒來得令人印象深刻」\[71\]。

### 褒揚與獲獎

[CNN的史蒂芬妮](https://zh.wikipedia.org/wiki/CNN "wikilink")·郭德堡（）在文章《[勇敢傳說的](../Page/勇敢傳說.md "wikilink")[梅莉達與其他動畫女角](https://zh.wikipedia.org/wiki/梅莉達公主 "wikilink")》裡，將樂佩列在迪士尼幾位最英勇而獨立的女主角中\[72\]。線上雜誌的莎萊雅（）
在文章《排行榜：最柔弱到最自主的迪士尼公主們》中把樂佩排到第四名，將她活潑的性格與上一代迪士尼公主中的愛麗兒和《[阿拉丁](https://zh.wikipedia.org/wiki/阿拉丁_\(電影\) "wikilink")》裡的茉莉公主（）比較。莎萊雅形容樂佩是個「悍女子」，儘管「她的天真有時牴觸了她先進的性格」。又說「她也察覺了所處的困境裡有所不公，並且找到了突破的方法，以智謀擺脫了她的『媽媽』——實為綁架她的人——，而到外面的世界探險」\[73\]。[菲律賓女性網站](https://zh.wikipedia.org/wiki/菲律賓 "wikilink")「」的戴里特（）在文章《30位威猛和有趣的卡通女角》中，提到樂佩與原典中的「長髮姑娘」相去甚遠，「她並非無助、鎖在塔裡枯等未知命運的嬌弱閨女，而是位可以在打鬥中捍衛自己的堅強女子」\[74\]。

樂佩在片中與尤金的愛情對唱曲《》也得到許多好評，獲得[葛萊美獎的](../Page/葛萊美獎.md "wikilink")及[拉斯維加斯影評人協會獎的最佳歌曲獎](https://zh.wikipedia.org/wiki/拉斯维加斯影评人协会 "wikilink")\[75\]\[76\]、亦曾被[奧斯卡最佳原創歌曲獎及](https://zh.wikipedia.org/wiki/奧斯卡最佳原創歌曲獎 "wikilink")[第83屆奧斯卡金像獎提名](https://zh.wikipedia.org/wiki/第83屆奧斯卡金像獎 "wikilink")\[77\]\[78\]\[79\]。

### 迪士尼公主與相關商品

樂佩成為第10位[迪士尼公主後](../Page/迪士尼公主.md "wikilink")\[81\]，針對以少女為主的消費族群推出了玩具、影音錄製品、服裝和[電子遊戲等週邊系列產品](https://zh.wikipedia.org/wiki/電子遊戲 "wikilink")\[82\]。[華特迪士尼公司透過](https://zh.wikipedia.org/wiki/華特迪士尼公司 "wikilink")[加冕將角色排入迪士尼公主行列](https://zh.wikipedia.org/wiki/加冕 "wikilink")，而樂佩的加冕典禮於2011年10月2日在[英國](https://zh.wikipedia.org/wiki/英國 "wikilink")[倫敦的](https://zh.wikipedia.org/wiki/倫敦 "wikilink")[肯辛頓宮舉辦](https://zh.wikipedia.org/wiki/肯辛頓宮 "wikilink")\[83\]\[84\]\[85\]，這也是肯辛頓宮首次舉行「虛擬」公主的加冕儀式\[86\]。然而在許多相關產品中，樂佩是使用傳統動畫繪製的版本呈現\[87\]。隨著加冕典禮舉辦之後，迪士尼公主英文版官網上也增設了她的頁面\[88\]。

\[89\]迄今發行的許多產品中都有樂佩的登場\[90\]，例如在一套名為「」的[冒險遊戲中](https://zh.wikipedia.org/wiki/冒險遊戲 "wikilink")\[91\]，樂佩被設定為玩家可操控角色\[92\]\[93\]。該電玩由發行於2010年11月23日，比電影的上映日還早一天，遊戲內容則依據電影情節設計，適用於[Wii和](https://zh.wikipedia.org/wiki/Nintendo_Wii "wikilink")[DS等](https://zh.wikipedia.org/wiki/Nintendo_DS "wikilink")[任天堂](../Page/任天堂.md "wikilink")[電子遊戲平台](https://zh.wikipedia.org/wiki/電子遊戲 "wikilink")\[94\]\[95\]。電玩中的樂佩依然錄用了曼蒂·摩爾的配音\[96\]，而她本人也對此感到很自豪\[97\]。

此外，樂佩也被製成多個[洋娃娃產品](https://zh.wikipedia.org/wiki/洋娃娃 "wikilink")，在「迪士尼動畫師收藏系列」（）中11位幼兒時期版的迪士尼公主娃娃裡面，也是第一位推出的角色。該系列的洋娃娃均由葛連·基恩和馬克·韓恩（）兩位迪士尼動畫師參與設計\[98\]\[99\]。

### 迪士尼主題樂園及宣傳作品

樂佩在全球數個[華特迪士尼樂園及度假區裡都有出現過](../Page/華特迪士尼樂園及度假區.md "wikilink")\[100\]。而[佛羅里達](https://zh.wikipedia.org/wiki/佛羅里達 "wikilink")[華特迪士尼世界度假區及](../Page/華特迪士尼世界度假區.md "wikilink")[加州迪士尼樂園裡更設立了數座遊樂設施](../Page/迪士尼樂園.md "wikilink")，仿造電影中樂佩活動過的場景\[101\]，當中還包括了位於佛羅里達的迪士尼[神奇王國](../Page/幻想世界.md "wikilink")、以真實比例建成了樂佩居住的高塔\[102\]。

在[安妮·萊柏維茲為宣傳迪士尼樂園及度假區的活動](../Page/安妮·萊柏維茲.md "wikilink")「百萬夢想年」（），而於2007年起\[103\]拍攝的「」寫真作品中，由美國鄉村音樂女歌手[泰勒絲在](https://zh.wikipedia.org/wiki/泰勒絲 "wikilink")2013年宣傳照「冒險之旅將起於此」（）裡化身為真人版的樂佩\[104\]\[105\]。《》對該攝影作品的圖片說明寫道：「在這張標題為『冒險之旅將起於此』、且令人摒息的寫真裡，這位23歲的葛萊美獎得主倚靠在一座滿佈蘚苔的塔窗邊。當她若有所思地望向遠方時，粉色的襯裙從紫色長裙下探出，金色長髮華麗地流洩在空中。」\[106\]泰勒絲之後向娛樂新聞「」（）表示自己很榮幸能被選為宣傳照的女主角\[107\]。

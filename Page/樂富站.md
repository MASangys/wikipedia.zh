**樂富站**（）是[港鐵](../Page/港鐵.md "wikilink")[觀塘綫](../Page/觀塘綫.md "wikilink")的一個[鐵路車站](https://zh.wikipedia.org/wiki/鐵路車站 "wikilink")，位於[黃大仙區](../Page/黃大仙區.md "wikilink")[橫頭磡](../Page/橫頭磡.md "wikilink")[樂富邨](../Page/樂富邨.md "wikilink")，由四大聯營公司（Dragages、Hochtief、[Gammon](../Page/金門建築.md "wikilink")、Sentab）負責興建，於1979年10月1日啟用。

## 結構

樂富站共有兩層，地面為出入口及大堂，而月台則位於地底。車站共有3條扶手電梯連接大堂及月台（其中一條在非繁忙時間暫停使用），並設有1部升降機及1條樓梯來往月台及大堂之間\[1\]。

### 大堂

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Lok Fu Station 2019 03 part5.jpg | caption1 = 大堂 }} 現時，樂富站內提供不同類型的商店供乘客購物或進食，例如有[便利店](https://zh.wikipedia.org/wiki/便利店 "wikilink")、[麵包糕餅店](https://zh.wikipedia.org/wiki/麵包糕餅店 "wikilink")、[銀行](../Page/銀行.md "wikilink")等[零售業](https://zh.wikipedia.org/wiki/零售業 "wikilink")，以及自助服務設施，包括[自動櫃員機](../Page/自動櫃員機.md "wikilink")、[自動售賣機](https://zh.wikipedia.org/wiki/自動售賣機 "wikilink")、自動照相機等\[2\]\[3\]。大堂[售票機旁亦設有](https://zh.wikipedia.org/wiki/售票機 "wikilink")[香港郵政](../Page/香港郵政.md "wikilink")[郵箱](../Page/郵箱.md "wikilink")\[4\]\[5\]，方便乘客在車站內投寄[郵件](../Page/郵件.md "wikilink")。

### 月台

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Lok Fu Station 2019 03 part4.jpg | caption1 = 1號月台（往調景嶺方向） | image2 = Lok Fu Station 2019 03 part3.jpg | caption2 = 2號月台（往黃埔方向） }} 樂富站設有1個[島式月台](../Page/島式月台.md "wikilink")，並設有[月台幕門](../Page/月台幕門.md "wikilink")\[6\]。由於樂富站大堂及扶手電梯集中在月台的東面，因此月台西面沒有一般乘客可見的建構物，並且以牆壁分隔1號及2號月台間，只有兩條連接1號及2號月台的通道\[7\]，並與[石硤尾站](../Page/石硤尾站.md "wikilink")結構相近。在樂富站剛通車時，當時成為[修正早期系統](../Page/修正早期系統.md "wikilink")中，月台為最短的車站（4卡）\[8\]。

在剛通車時，樂富站是修正早期系統中唯一有三條並列的[扶手電梯的車站](../Page/電動扶梯.md "wikilink")。至於升降機則由本屬職員使用的升降機改裝成客用升降機\[9\]，並於2015年11月16日暫停使用，以更換為較新型號，並於2016年5月底重投服務\[10\]。

### 出入口

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Lok Fu Station 2012.JPG | caption1 = A出入口 | image2 = Lok Fu Station 2013 05.JPG | caption2 = B出入口 }} 樂富站設有2個車站出入口，一個位於大堂的尾端的A出入口，在樂富站啟用時，只設有A出入口，連接樂富邨及[公共運輸交匯處](../Page/公共運輸交匯處.md "wikilink")等地方\[11\]。而B出入口則在後期因應樂富中心（現為[樂富廣場](../Page/樂富廣場.md "wikilink")）工程於1991年啟用，連接樂富中心二期。

## 鄰接車站

樂富站是觀塘綫的中途站，乘客可在此乘坐往[黃埔或](../Page/黃埔站_\(香港\).md "wikilink")[調景嶺列車](../Page/調景嶺站.md "wikilink")，往返[九龍](../Page/九龍.md "wikilink")及[新界](../Page/新界.md "wikilink")等地。

## 利用狀況

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Lok Fu Station 2017 12 part1.jpg | caption1 = 此為本站唯一一組扶手電梯 | image2 = Lok Fu Station 2017 12 part2.jpg | caption2 = 此為本站唯一往來月台至大堂的樓梯 | image3 = Lok Fu Station 2019 03 part6.jpg | caption3 = 往來月台至大堂升降機 }} 樂富站啟用初期，由於當時[橫頭磡邨](../Page/橫頭磡邨.md "wikilink")正在重建，故此它的使用量在通車初期一直偏低，直到1983年橫頭磡邨重建完成後客量才上升。

另外，在1998年之前，本站是最接近當時仍在運作的[啟德機場](../Page/啟德機場.md "wikilink")的地鐵站，因此亦有不少乘客使用此站轉乘巴士及小巴往返。本站鄰近[九龍城](../Page/九龍城.md "wikilink")，但須沿[聯合道](../Page/聯合道.md "wikilink")步行15至20分鐘。现在的使用者多數也是附近居民、[香港浸會大學](../Page/香港浸會大學.md "wikilink")學生及教職員，以及在那里上班的[房屋署](../Page/房屋署.md "wikilink")職員。

### 命名及選址

在早期系統時，樂富站曾考慮命名為**老虎岩站**，後因樂富邨建成，最後決定命名為**樂富站**。樂富站原址是橫頭磡邨內的足球場，1976年被徵用作樂富地鐵站地盤。1979年10月1日隨着[前地鐵修正早期系統正式通車](../Page/香港地鐵.md "wikilink")\[13\]。

## 軼事

本站為1988年電影《南北媽打》的拍攝場地之一。

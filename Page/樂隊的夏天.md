是[愛奇藝及米未繼](https://zh.wikipedia.org/wiki/愛奇藝 "wikilink")《[奇葩說](https://zh.wikipedia.org/wiki/奇葩说 "wikilink")》後再次合作，於2018年10月公佈拍攝計劃\[1\]，節目入選「匠人之道權力榜」2019最具期待網綜\[2\]；節目於2019年4月開始錄製，計劃2019年第二季度播出。

節目前後籌備了8個月，他們從中國1000多個樂隊中，先挑選出300個，經細緻篩選、奔赴各地考察樂隊現場演出以及坦誠地雙向溝通後\[3\]，最終31支包括[搖滾](../Page/摇滚乐.md "wikilink")、[龐克](https://zh.wikipedia.org/wiki/龐克 "wikilink")、[爵士等多種音樂類型的樂隊於](../Page/爵士乐.md "wikilink")2019年4月13日正式集結，共同爭奪年度HOT5樂隊席位。同時，《樂隊的夏天》將聚集中國200家[livehouse主理人](https://zh.wikipedia.org/wiki/展演空間 "wikilink")\[4\]，中國17個城市24
家livehouse更於首播當天舉辦「樂隊之夜」播放節目首播，參演樂隊甚或出現於livehouse一同觀看節目。

為力求還原livehouse、音樂節的場景體驗，節目現場設置了A、B兩個舞台\[5\]，觀眾區亦不設座位。節目更把所有的投票權交給樂迷（包括超級樂迷、專業樂迷及大眾樂迷）；超級樂迷來到《樂隊的夏天》既不作導師亦不帶隊，他們會和台下樂迷一起狂歡，並幫助大家更好地了解樂隊。\[6\]\[7\]

## 爭議

| 節目雷同                                                                                                                                                                              |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 《樂隊的夏天》上線幾個小時後，有說這與韓國綜藝《超級樂隊》雷同。馬東表示賽制就有很大差異，《超級樂隊》重在各種樂器和音樂的比拼，而《樂隊的夏天》主要想講的「樂隊裡的隊」和「樂隊裡的人」\[9\]。而且，以製作周期來看兩節目應在差不多同時進行的\[10\]，《超級樂隊》偏向透秀，將新人組成一個樂隊；而《樂隊的夏天》則是已經成團的樂隊互相比拼。\[11\] |

| 剪接及標題                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 首集剪接上相對完整的樂隊表演相對較少，在約2小時節目只有5首左右，其餘的只剪剩半首歌甚至幾個鏡頭快速切換\[12\]，完整表演只放在純享版播出\[13\]。反之，除了開首樂隊互相投票的環節外，節目放了較長篇幅以及為照顧對樂團所知甚少的觀眾而介紹音樂類型的片段，但對部份樂迷來說此剪輯則顯混亂\[14\]；而佔時長亦不少的馬東訪談式銜接，部份內容亦被指與音樂無關及沒有分寸\[15\]。節目於第二集開首特意表示對無法將所有樂隊完整呈現感抱歉，並提到可搜索純享版直接觀看所有完整表演；第三集尾段播放了部份第一輪被淘汰樂隊的歌曲段落，該集開集特別提醒觀眾觀看該部份。 此外，首集多人接連向吳青峰問蘇打綠的事，加上吳青峰在節目間提到蘇打錄正在經歷「很多很重要的時刻」，故請教成團30年的面孔樂隊經過什麼最大的困難。雖然吳青峰在節目中也表明「就算死了也還是蘇打綠」，但節目標題卻是「吳青峰曝蘇打綠成團危機」。此標題及相關宣傳在官方微博引發熱烈反響，吳青峰在節目上線不久亦在微博回應說「沒有什麼危機，我不喜歡那個標題。」\[16\]之後，首集節目陸續被修改標題。 |

**次氯酸甲酯**是最简单的[次氯酸](../Page/次氯酸.md "wikilink")酯，化学式为CH<sub>3</sub>OCl。它有一个同分异构体氯甲醇（ClCH<sub>2</sub>OH），这个化合物不稳定，会迅速脱[氯化氢](../Page/氯化氢.md "wikilink")，生成[甲醇](../Page/甲醇.md "wikilink")。而次氯酸甲酯也较为不稳定，毒性很大，可爆炸性地分解，并生成有毒的[甲醛](../Page/甲醛.md "wikilink")和[氯化氢](../Page/氯化氢.md "wikilink")气体。

## 制备

[次氯酸](../Page/次氯酸.md "wikilink")（HOCl）和[甲醇](../Page/甲醇.md "wikilink")（CH<sub>3</sub>OH）反应，生成不稳定的次氯酸甲酯：

<div style='text-align: center;'>

\(\mathrm{CH_3OH + HOCl \xrightarrow{} CH_3OCl + H_2O}\)

</div>

## 化学性质

次氯酸甲酯具有亲电的Cl<sup>+</sup>和亲核的CH<sub>3</sub>O<sup>-</sup>。

### 分解反应

次氯酸甲酯不稳定，很快分解为[甲醛](../Page/甲醛.md "wikilink")和[氯化氢](../Page/氯化氢.md "wikilink")：

<div style='text-align: center;'>

\(\mathrm{CH_3OCl \xrightarrow{} HCHO + HCl}\)

</div>

### 取代反应

次氯酸甲酯可以和碱反应。如和[氢氧化钠](../Page/氢氧化钠.md "wikilink")反应，生成[甲醇钠](../Page/甲醇钠.md "wikilink")和次氯酸。和其它醇钠反应，生成甲醇钠和对应的烷基[次氯酸](../Page/次氯酸.md "wikilink")酯。\[3\]

<div style='text-align: center;'>

\(\mathrm{
CH_3OCl + NaOH \xrightarrow{} CH_3ONa + HOCl}\)

\(\mathrm{
CH_3OCl + RONa \xrightarrow{} CH_3ONa + ROCl}\)

</div>

和炔钠、[氰化钠](../Page/氰化钠.md "wikilink")、[硫氢化钠](../Page/硫氢化钠.md "wikilink")、硫醇钠、卤化钠反应，生成甲醇钠和相应的氯代物种：\[4\]

<div style='text-align: center;'>

\(\mathrm{CH_3OCl + RC \equiv CNa \xrightarrow{} RC \equiv CCl + CH_3ONa}\)

\(\mathrm{CH_3OCl + NaCN \xrightarrow{} ClCN + CH_3ONa}\)

\(\mathrm{CH_3OCl + NaSH \xrightarrow{} CH_3ONa + ClSH }\)

\(\mathrm{CH_3OCl + RSNa \xrightarrow{} CH_3ONa + RSCl }\)

\(\mathrm{CH_3OCl + NaX \xrightarrow{} CH_3ONa  + XCl}\)

</div>

次氯酸甲酯和[卤代烃反应](https://zh.wikipedia.org/wiki/卤代烃 "wikilink")，生成甲基醚和氯代烃：\[5\]

<div style='text-align: center;'>

\(\mathrm{CH_3OCl + RX \xrightarrow{} CH_3OR + XCl}\)

</div>

和烷基锂的作用类似，生成[甲基锂](../Page/甲基锂.md "wikilink")和氯代烃：\[6\]

<div style='text-align: center;'>

\(\mathrm{CH_3OCl + RLi \xrightarrow{} RCl + CH_3OLi}\)

</div>

和氨、伯胺、仲胺或[磷化氢](../Page/磷化氢.md "wikilink")反应，生成甲醇和少一个氢的取代产物：\[7\]\[8\]

<div style='text-align: center;'>

\(\mathrm{CH_3OCl + NH_3 \xrightarrow{} CH_3OH + NH_2Cl}\)

\(\mathrm{CH_3OCl + RNH_2 \xrightarrow{} CH_3OH + RNHCl}\)

\(\mathrm{CH_3OCl + R\acute{}\;NHR \xrightarrow{} R\acute{}\;N(Cl)R + CH_3OH}\)

\(\mathrm{CH_3OCl + PH_3 \xrightarrow{} CH_3OH + PH_2Cl}\)

</div>

氯甲酸甲酯和[硅烷在](https://zh.wikipedia.org/wiki/硅烷 "wikilink")[三氟化硼](../Page/三氟化硼.md "wikilink")的存在下反应，生成[氯化氢](../Page/氯化氢.md "wikilink")和[甲氧基硅烷](https://zh.wikipedia.org/wiki/甲氧基硅烷 "wikilink")：\[9\]:

<div style='text-align: center;'>

\(\mathrm{CH_3OCl + SiH_4 \xrightarrow{BF_3} HCl + CH_3OSiH_3}\)

</div>

它和[芳烃反应](https://zh.wikipedia.org/wiki/芳烃 "wikilink")，如和[苯](../Page/苯.md "wikilink")反应，生成[氯苯](../Page/氯苯.md "wikilink")和[甲醇](../Page/甲醇.md "wikilink")：

<div style='text-align: center;'>

\(\mathrm{PhH + CH_3OCl \xrightarrow{} PhCl + CH_3OH}\)

</div>

### 加成反应

次氯酸甲酯可以和[烯烃](../Page/烯烃.md "wikilink")、[炔烃](../Page/炔烃.md "wikilink")以及[共轭二烯烃发生加成反应](https://zh.wikipedia.org/wiki/共轭二烯烃 "wikilink")，生成更稳定的氯代醚：

<div style='text-align: center;'>

\(\mathrm{
CH_2=CH_2 + CH_3OCl \xrightarrow{} ClCH_2CH_2OCH_3}\)

\(\mathrm{
HC \equiv CH + CH_3OCl \xrightarrow{} ClCH=CHOCH_3}\)

\(\mathrm{
CH_2=CHCH=CH_2 + CH_3OCl \xrightarrow{} ClCH_2CH=CHCH_2OCH_3 + 4H_2O}\)

</div>

和小环化合物（如[环丙烷](https://zh.wikipedia.org/wiki/环丙烷 "wikilink")、[环丁烷](../Page/环丁烷.md "wikilink")）反应，小环化合物开环加成：

<div style='text-align: center;'>

[Cyclopropane-skeletal.png](https://zh.wikipedia.org/wiki/File:Cyclopropane-skeletal.png "fig:Cyclopropane-skeletal.png") \(\mathrm{
+ CH_3OCl \xrightarrow{} ClCH_2CH_2CH_2OCH_3}\)

</div>

和含杂原子的小环化合物有类似反应，如和[环氧乙烷](../Page/环氧乙烷.md "wikilink")反应，生成爆炸性的2-甲氧基次氯酸乙酯\[10\]；和环丙胺反应生成2-甲氧基氯乙胺\[11\]：

<div style='text-align: center;'>

[30 px](https://zh.wikipedia.org/wiki/File:Ethylene_oxide.svg "fig:30 px") \(\mathrm{
+ CH_3OCl \xrightarrow{} CH_3OCH_2CH_2OCl}\)

[40 px](https://zh.wikipedia.org/wiki/File:Aziridine.svg "fig:40 px") \(\mathrm{
+ CH_3OCl \xrightarrow{} CH_3OCH_2CH_2NHCl}\)

</div>

它可以和活泼羰基化合物（如甲醛）、亚胺化合物发生加成反应：

<div style='text-align: center;'>

\(\mathrm{
HCHO + CH_3OCl \xrightarrow{} CH_3OCH_2OCl}\)

\(\mathrm{
CH_2=NH + CH_3OCl \xrightarrow{} CH_3OCH_2NHCl}\)

</div>

[卡宾](../Page/卡宾.md "wikilink")可以和次氯酸甲酯发生插入反应，生成次氯酸乙酯：\[12\]:

<div style='text-align: center;'>

\(\mathrm{CH_3OCl + CH_3Cl + KOH \xrightarrow{} CH_3CH_2OCl + KCl + H_2O}\)

</div>

**次氯酸钠**（sodium hypochlorite），化学式NaClO，是[钠](../Page/钠.md "wikilink")的[次氯酸盐](https://zh.wikipedia.org/wiki/次氯酸盐 "wikilink")。次氯酸钠与[二氧化碳](../Page/二氧化碳.md "wikilink")反应产生的[次氯酸](../Page/次氯酸.md "wikilink")是[漂白剂有的成分](https://zh.wikipedia.org/wiki/漂白剂 "wikilink")。

## 製備

[氯气溶於冷而且稀的](https://zh.wikipedia.org/wiki/氯气 "wikilink")[氫氧化鈉溶液產生次氯酸鈉](https://zh.wikipedia.org/wiki/氫氧化鈉 "wikilink")、[氯化鈉及](https://zh.wikipedia.org/wiki/氯化鈉 "wikilink")[水](https://zh.wikipedia.org/wiki/水分子 "wikilink")：

  -
    Cl<sub>2</sub> + 2NaOH → NaClO + NaCl + H<sub>2</sub>O

注意氯气溶於水，生成[盐酸](../Page/盐酸.md "wikilink")和[次氯酸](../Page/次氯酸.md "wikilink")。

  -
    Cl<sub>2</sub> + H<sub>2</sub>O → HClO + HCl

## 化学反应

次氯酸钠是强碱弱酸盐，会水解，水溶液呈碱性：

  -
    NaClO<sub>(aq)</sub> + H<sub>2</sub>O<sub>(l)</sub> → HClO<sub>(aq)</sub> + NaOH<sub>(aq)</sub>

次氯酸鈉水溶液[漂白釋出](https://zh.wikipedia.org/wiki/漂白 "wikilink")[氧](../Page/氧.md "wikilink")後，會變成[氯化鈉水溶液](https://zh.wikipedia.org/wiki/氯化鈉 "wikilink")，其反應式如下:

  -
    NaClO + H<sub>2</sub>O → NaOH + HClO
    2HClO → 2HCl + O<sub>2</sub>↑
    NaOH + HCl → NaCl +H<sub>2</sub>O

與[鹽酸反應會放出有毒的氯氣](https://zh.wikipedia.org/wiki/鹽酸 "wikilink")，因此使用相關製品時應注意避免混合：

  -
    NaClO + 2HCl<sub>(aq)</sub> → NaCl + H<sub>2</sub>O + Cl<sub>2</sub>↑

與[過氧化氫反應生成](https://zh.wikipedia.org/wiki/過氧化氫 "wikilink")[氧氣](https://zh.wikipedia.org/wiki/氧氣 "wikilink")：

  -
    NaClO + H<sub>2</sub>O<sub>2</sub> → H<sub>2</sub>O + NaCl + O<sub>2</sub>↑

### 分解

次氯酸钠在較熱的條件下易[歧化成氯化钠及氯酸钠](https://zh.wikipedia.org/wiki/歧化 "wikilink")：

  -
    3NaClO → 2NaCl + NaClO<sub>3(aq)</sub>

在水中，也會緩慢發生以下反應：

  -
    4NaClO + 2H<sub>2</sub>O → Na<sup>+</sup> + 4OH<sup>-</sup> + 2Cl<sub>2</sub>↑ + O<sub>2</sub>↑

次氯酸根离子是极不稳定的[离子](../Page/离子.md "wikilink")，它会在光照下分解：

  -
    2ClO<sup>-</sup> → \>O<sub>2</sub>↑ + 2Cl<sup>-</sup>

以上三種分解均會發生，屬於[競爭反應](https://zh.wikipedia.org/wiki/競爭反應 "wikilink")。為延長保存期，次氯酸鈉應避光、低溫保存。

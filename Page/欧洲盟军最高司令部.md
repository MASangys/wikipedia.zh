**欧洲盟军最高司令部**（，缩写**SHAPE**）在2003年前是[北大西洋公约组织](https://zh.wikipedia.org/wiki/北大西洋公约组织 "wikilink")**欧洲盟军司令部**（Allied
Command Europe，ACE）的总部，2003年后则成为了**盟军作战司令部**（Allied Command
Operations，ACO）的总部，负责统一指挥北约的军事行动。

## 历史

[朝鲜战争爆发后](../Page/朝鲜战争.md "wikilink")，西欧各国对于能否有效抵御[苏联进攻产生了怀疑](../Page/苏联.md "wikilink")，对统一的军事指挥机构的需求也随之诞生。1951年，隶属北约的欧洲盟军最高司令部正式成立，负责北约在欧洲的所有军事行动。首任司令为[美国陆军五星上将](../Page/五星上将.md "wikilink")[德怀特·艾森豪威尔](../Page/德怀特·艾森豪威尔.md "wikilink")，副司令则为[英国陆军元帅](https://zh.wikipedia.org/wiki/英国陆军元帅 "wikilink")[伯纳德·蒙哥马利](https://zh.wikipedia.org/wiki/伯纳德·蒙哥马利 "wikilink")。自1967年来，其地址设于[比利时](../Page/比利时.md "wikilink")[蒙斯以北的](https://zh.wikipedia.org/wiki/蒙斯 "wikilink")小镇，在此之前则坐落于[法国](https://zh.wikipedia.org/wiki/法国 "wikilink")[巴黎](../Page/巴黎.md "wikilink")[凡尔赛镇附近](https://zh.wikipedia.org/wiki/凡爾賽鎮 "wikilink")。\[1\]从1951年设立到2003年，SHAPE一直是北约欧洲盟军司令部的总部，2003年北约大规模改革后则收编（Allied
Command Atlantic，ACA）的部分机构成为盟军作战司令部总部 ，统一指挥北约在全球的军事行动。\[2\]

2003年北约改革时，除欧洲盟军司令部在蒙斯的行动指挥部外，位于[葡萄牙](../Page/葡萄牙.md "wikilink")[里斯本原本隶属大西洋盟军司令部的行动指挥部也被划分给了新的盟军作战司令部](../Page/里斯本.md "wikilink")。（大西洋盟军司令部的剩余部分则在重组后成为了。）尽管欧洲盟军最高司令部在改革后作为盟军作战司令部所管辖的地域已不仅限于欧洲，SHAPE的名字出于法律考虑却被保留，并没有做出相应的更改。\[3\]
盟军作战司令部的指挥官也依旧保留了“欧洲盟军最高司令”（Supreme Allied Commander
Europe，缩写SACEUR）的头衔，并按照传统依旧由一名美军上将担任（此人同时也兼任[美国欧洲司令部司令](../Page/美国欧洲司令部.md "wikilink")）。
[NATO_Allied_Command_Europe_chart_1952.jpg](https://zh.wikipedia.org/wiki/File:NATO_Allied_Command_Europe_chart_1952.jpg "fig:NATO_Allied_Command_Europe_chart_1952.jpg")

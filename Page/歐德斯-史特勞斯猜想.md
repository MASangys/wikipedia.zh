**歐德斯-史特勞斯猜想**（Erdős–Straus
conjecture），簡稱**歐德斯猜想**，是由[匈牙利](../Page/匈牙利.md "wikilink")[犹太](../Page/犹太.md "wikilink")[数学家](../Page/数学家.md "wikilink")[保罗·埃尔德什與](../Page/保罗·埃尔德什.md "wikilink")[德裔](../Page/德裔.md "wikilink")[美國數學家](../Page/美國.md "wikilink")於1948年共同提出的[數論](../Page/數論.md "wikilink")[猜想](../Page/猜想.md "wikilink")，其陳述为：

例如，若*n* = 1801，則存在一組 *x* = 451、*y* = 295364、*z* = 3249004 的解，使得

\[\frac4{1801} = \frac1{451} + \frac1{295364} + \frac1{3249004}\]

在基本式子中，只需考慮 *n* = *p* 為素數的情況，因為若

\[\frac4p = \frac1x + \frac1y +  \frac1z.\]

成立，則對於大於 1 的整數 *m*

\[\frac{4}{pm}=\frac{1}{xm}+\frac{1}{ym}+\frac{1}{zm}\]

也會成立。

計算機已經驗證到 *n* ≤ 10<sup>14</sup> 的情況\[1\]，但此猜想還是有待證明。

### 歐德斯猜想的特别形式

  - \(\frac{4}{p} = \frac{1}{a \times b} + \frac{1}{a \times c} + \frac{1}{a \times b\times c\times p}\)

例如

\(\frac{4}{409} = \frac{1}{63 \times 2} + \frac{1}{63 \times 13} + \frac{1}{63 \times 2\times 13\times 409}\)

**正八面體**是一種[八面體](../Page/八面體.md "wikilink")，由八個[等邊三角形](https://zh.wikipedia.org/wiki/等邊三角形 "wikilink")，分別為上、下各四個[三角形](../Page/三角形.md "wikilink")與一個[正方形](../Page/正方形.md "wikilink")組成的[正方錐體](https://zh.wikipedia.org/wiki/正方錐體 "wikilink")，上下黏合在一起而構成，是五種[正多面體的第三種](../Page/柏拉圖立體.md "wikilink")，有6個頂點和12條邊。**正八面體**也是正三角反棱柱。正八面体是三维的[正轴形](../Page/正轴形.md "wikilink")，施莱夫利符号{3,4}，。

正八面體每四条棱可以成为一个[正方形](../Page/正方形.md "wikilink")，共有三个独立的正方形。

## 性質

  - 頂點數目：6
  - 邊數目：12
  - 面數目：8
  - 當邊長為a時：
      - [表面積](../Page/表面積.md "wikilink")：\(S=2\sqrt{3}a^{2}\)
      - [體積](https://zh.wikipedia.org/wiki/體積 "wikilink")：\(V=\frac{1}{3}\sqrt{2}a^{3}\)
      - 外接球半径：\(r_u=\frac{\sqrt{2}}{2}a\)（外接球即过正八面体各顶点的球）
      - 内切球半径：\(r_i=\frac{\sqrt{6}}{6}a\)（内切球即与正八面体各面相切的球）
      - 中交球半径：\(r_m=\frac{1}{2}a\)（中交球即过正八面体各边中点的球）

### 坐标系

以棱长为√2的正八面体的几何中心作为原点，将正八面体的对角线作为x,y,z轴建立三维直角坐标系（正八面体的3条对角线两两[正交](../Page/正交.md "wikilink")，这也是正八面体被叫做“正轴形”的原因），则我们能将正八面体的顶点坐标记为
( ±1, 0, 0 )
( 0, ±1, 0 )
( 0, 0, ±1 )
正八面体表面方程为： |x|+|y|+|z|=1
更一般的，如果正八面体的对角线平行于坐标轴，中心为(x<sub>0</sub>,y<sub>0</sub>,z<sub>0</sub>)，外接圆半径为r（棱长为(√2)r），则正八面体表面方程为： |x-x<sub>0</sub>|+|y-y<sub>0</sub>|+|z-z<sub>0</sub>|=r
如果中心在原点的正八面体被拉长，成为菱形体，则更一般的八面体方程为
\(\left|\frac{x}{x_m}\right|+\left|\frac{y}{y_m}\right|+\left|\frac{z}{z_m}\right| = 1\)
其内接于椭球体
\(\frac{x^2}{x_m^2}+\frac{y^2}{y_m^2}+\frac{z^2}{z_m^2} = 1\)
表面积S和体积V为：
\(S=4 \, x_m \, y_m \, z_m \times \sqrt{\frac{1}{x_m^2}+\frac{1}{y_m^2}+\frac{1}{z_m^2}}\)
\(V=\frac{4}{3}\,x_m\,y_m\,z_m\)
它的[惯性张量](https://zh.wikipedia.org/wiki/惯性张量 "wikilink")*I*是：

\[I =
\begin{bmatrix}
  \frac{1}{10} m (y_m^2+z_m^2) & 0 & 0 \\
  0 & \frac{1}{10} m (x_m^2+z_m^2) & 0 \\
  0 & 0 & \frac{1}{10} m (x_m^2+y_m^2)
\end{bmatrix}\]
当\(x_m=y_m=z_m=a\,\frac{\sqrt{2}}{2}\)时，菱形体为上述正八面体。

### 对称性和表面涂色

正八面体作为3维的正轴体正多面体，自身拥有较高的对称性，它的所有面都是不可区分的。可是我们也可以想象将正八面体的面“涂上”不同的“颜色”，使它其的不同面拥有不同的“几何意义”，使正八面体拥有不同的对称性。正八面体的[对称群是O](https://zh.wikipedia.org/wiki/对称群 "wikilink")<sub>h</sub>（正八面体群），是三维的。在此对称性下，正八面体的所有面都带有相同对“颜色”，对称性最高，群[阶](https://zh.wikipedia.org/wiki/阶（群论） "wikilink")48。该群的[子群](../Page/子群.md "wikilink")体现了正八面体更低的对称性：T<sub>d</sub>（群阶24），截半正四面体的对称群；D<sub>3d</sub>（群阶12），三角[反棱柱](../Page/反棱柱.md "wikilink")的对称群；D<sub>4h</sub>（群阶16），四角[双棱锥](https://zh.wikipedia.org/wiki/双棱锥 "wikilink")（正四棱柱的对偶）的对称群；D<sub>2h</sub>（群阶8），三维长菱体（三维长方体的对偶）的对称群。

### 其它几何关联

[缩略图](https://zh.wikipedia.org/wiki/File:Compound_of_two_tetrahedra.png "fig:缩略图")两正四面体的交集\]\] 两个互为对偶的正四面体可以组成一个复合正多面体，这两个[正四面体的](https://zh.wikipedia.org/wiki/正四面体 "wikilink")[交集](../Page/交集.md "wikilink")即是正八面体。这个复合多面体，也叫做[星形八面体](https://zh.wikipedia.org/wiki/星形八面体 "wikilink")，是正八面体的第一个也是唯一一个星形展（英Stellation，暂译，目前还没有好的译名）。另一方面，从正四面体各棱中点处截去4个包含原顶点在内的线性大小只有原正四面体一半的正四面体，你也能得到正八面体，也就是说，正八面体是“截半正四面体”。在这里，正四面体与正八面体之间的关系就像[立方体](https://zh.wikipedia.org/wiki/立方体 "wikilink")、正八面体与[截半立方体](https://zh.wikipedia.org/wiki/截半立方体 "wikilink")；[正十二面体](https://zh.wikipedia.org/wiki/正十二面体 "wikilink")、[正二十面体与](https://zh.wikipedia.org/wiki/正二十面体 "wikilink")[截半正十二面体一样](https://zh.wikipedia.org/wiki/截半正十二面体 "wikilink")。
除此以外，我们知道正二十面体还是“扭棱正四面体”，因此，正八面体与其也应该有关系。事实上，我们能够利用[黄金分割从正八面体的棱上得到正二十面体的顶点](https://zh.wikipedia.org/wiki/黄金分割 "wikilink")。具体操作是：用[有向线段代替这个正八面体的各棱](https://zh.wikipedia.org/wiki/有向线段 "wikilink")，使每个面的3条有向线段恰好首尾相接，构成一圈。接着顺着每个有向线段的方向将其以黄金比例分割，分割点即是正二十面体的顶点。如果给定一正二十面体，则有5个不同的正八面体都可用上述操作得到给定正二十面体，这5个正八面体又可构成一复合正多面体，即[五複合正八面體](../Page/五複合正八面體.md "wikilink")。
[缩略图](https://zh.wikipedia.org/wiki/File:Tetrahedral-octahedral_honeycomb.png "fig:缩略图") 正八面体可以和正四面体一起完成三维空间的密铺，这密铺被叫做正八面体—正四面体堆砌(Octahedral-Tetrahedral Honeycomb)（同时它也是[交错](https://zh.wikipedia.org/wiki/交错 "wikilink")[立方体堆砌](../Page/立方体堆砌.md "wikilink")(Alternated Cubic Honeycomb)，亦即半立方体堆砌(Demicubic Honeycomb)），是28个三维半正[堆砌之一](https://zh.wikipedia.org/wiki/密铺 "wikilink")，是除立方体堆砌以外唯一一个完全由正多面体完成的三维堆砌。正八面体也参与了截半立方体堆砌。
正八面体是柏拉图立体中唯一一个在顶点处有偶数个面相交的，也是唯一一个所有对称镜面不穿过任何一面的。
以[约翰逊多面体](../Page/约翰逊多面体.md "wikilink")的角度来看，正八面体是双四棱锥，将其上下两个顶点截取，即得到双四棱台。
正八面体是四[连通的](https://zh.wikipedia.org/wiki/连通 "wikilink")，意味着要想打断正八面体6个顶点之间的连接，至少要撤掉4个顶点。它是4个四连通单纯形面全覆盖多面体之一，意味着它所有顶点的都有相同大小。其余3个多面体是约翰逊多面体[双五棱锥](https://zh.wikipedia.org/wiki/双五角锥 "wikilink")、[变棱双五角锥和一个非半正的有](../Page/變稜雙五角椎.md "wikilink")12个顶点和20个正三角形面的多面体。

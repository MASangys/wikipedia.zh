**正矢**（英文：**Versine**、**Versed sine**），在[三角函数](../Page/三角函数.md "wikilink")之中被定義為\(\textrm{versin} \theta = 1 - \cos \theta \,\)，值域在0～2之間。

## 相關函數

  - **餘矢**（英文：**coversed sine**、**coversine**），寫為\(\operatorname{coversin}(\theta)\)，有時亦縮寫為\(\operatorname{cvs}(\theta)\)。
  - **半正矢**（英文：**haversed sine**、**haversine**），寫為\(\operatorname{haversin}(\theta)\)，因[半正矢公式](../Page/半正矢公式.md "wikilink")出名，且曾用於[導航術](https://zh.wikipedia.org/wiki/導航 "wikilink")。
  - **半餘矢**（英文：**hacoversed sine**、**hacoversine**、**cohaversine**），寫為\(\operatorname{hacoversin}(\theta)\)。

## 微分與積分

|                                                             |                                                                 |
| ----------------------------------------------------------- | --------------------------------------------------------------- |
| \(\frac{d}{dx}\mathrm{versin}(x) = \sin{x}\)                | \(\int\mathrm{versin}(x) \,dx = x - \sin{x} + C\)               |
| \(\frac{d}{dx}\mathrm{coversin}(x) = -\cos{x}\)             | \(\int\mathrm{coversin}(x) \,dx = x + \cos{x} + C\)             |
| \(\frac{d}{dx}\mathrm{haversin}(x) = \frac{\sin{x}}{2}\)    | \(\int\mathrm{haversin}(x) \,dx = \frac{x - \sin{x}}{2} + C\)   |
| \(\frac{d}{dx}\mathrm{hacoversin}(x) = \frac{-\cos{x}}{2}\) | \(\int\mathrm{hacoversin}(x) \,dx = \frac{x + \cos{x}}{2} + C\) |
|                                                             |                                                                 |

**武汉话**（[西南官话](../Page/西南官话.md "wikilink")**武汉话:**）指[武汉市区](https://zh.wikipedia.org/wiki/武汉 "wikilink")（通常指的是[汉口](../Page/汉口.md "wikilink")、[武昌](https://zh.wikipedia.org/wiki/武昌 "wikilink")、[汉阳三镇老城区](https://zh.wikipedia.org/wiki/漢陽區 "wikilink")）的[方言](../Page/方言.md "wikilink")，属于[官话](../Page/官话.md "wikilink")中的[西南官话](../Page/西南官话.md "wikilink")[湖廣片](https://zh.wikipedia.org/wiki/湖廣片 "wikilink")。武汉方言共有阴平、阳平、上声、去声四个调，中古入聲大多歸入陽平調；相比之下，武汉周边属于[江淮官话](../Page/江淮官话.md "wikilink")[黃孝片的](https://zh.wikipedia.org/wiki/黃孝片 "wikilink")[黃陂話有阴平](https://zh.wikipedia.org/wiki/黃陂話 "wikilink")、阳平、上声、阴去、阳去、入声六个调。通常，武汉话被认为是[西南官话](../Page/西南官话.md "wikilink")的东极。 武汉话在[中華人民共和國成立後受到普通話的影響](https://zh.wikipedia.org/wiki/中華人民共和國 "wikilink")，故有語言學家把之前的武漢話稱作「老武漢話」而此後受[普通話影響的武漢話稱為](https://zh.wikipedia.org/wiki/普通話 "wikilink")「新武漢話」。

## 语音概述

### 声调

作为西南官话，現代的武汉话和多数西南官话一样具有4个聲调，分别为[阴平](https://zh.wikipedia.org/wiki/阴平 "wikilink")、[阳平](https://zh.wikipedia.org/wiki/阳平 "wikilink")、[上声](https://zh.wikipedia.org/wiki/上声 "wikilink")、[去声](https://zh.wikipedia.org/wiki/去声 "wikilink")，有[輕聲](../Page/輕聲.md "wikilink")變調。

### 声母

中古的精（除細音）、莊、章組合流，現代武漢話的日母大部分字與泥、來合流。

### 韵母

沒有入聲韻，鼻音韻尾有/n/和/ŋ/。

## 形态

形态方面，武汉话具有自己的特征。首先，武汉话在词汇方面显示出强烈的地域性；其次，武汉话在量词方面的使用方面也与漢語普通話不太一樣；再次，武汉话的体貌系统显示出与[湘方言的](https://zh.wikipedia.org/wiki/湘方言 "wikilink")[语言接触现象](https://zh.wikipedia.org/wiki/语言接触 "wikilink")。

#### 量词

武漢話的量詞顯示出漢語方言裡面南北過渡的漸變特徵。既包含南北方言各自的特徵，也具有極強的區域性色彩。在武漢話裡面，有一批區域性極強的量詞，與[普通話不同](https://zh.wikipedia.org/wiki/普通話 "wikilink")。例如在描述“筆”的時候，武漢話用「桿」 (kan˥)，而不用普通話的「支 」(普通話/ʈ͡ʂʐ̩˥/)：

|       |                 |                  |                  |                 |
| ----- | --------------- | ---------------- | ---------------- | --------------- |
| 武漢話   | 把               | 桿                | 筆                | 我               |
| IPA   | pa<sup>42</sup> | kan<sup>55</sup> | pi<sup>213</sup> | ŋo<sup>42</sup> |
| 逐字翻譯  | 給 give          | 支 CL             | 筆 pen            | 我 1SG           |
| 普通話對照 | 給我一支筆           |                  |                  |                 |

**「量詞＋名詞」的結構**

「量詞＋名詞」結構在漢語方言中都普遍存在，但以普通話為代表的一些方言只能出現在賓語位置上，如「交了個朋友」。但是，武漢話中，「量詞＋名詞」結構也能出現在主語位置上。

|       |                 |                   |                   |                                     |
| ----- | --------------- | ----------------- | ----------------- | ----------------------------------- |
| 武漢話範例 | 個               | 人                 | 蠻                 | 討嫌                                  |
| IPA   | ko<sup>35</sup> | lən<sup>213</sup> | man<sup>213</sup> | tʰau<sup>42</sup>ɕiɛn<sup>213</sup> |
| 逐字翻譯  | 個 CL            | 人 person          | 很 very            | 令人討厭 not nice                       |
| 普通話   | 這個/那個人很討人厭      |                   |                   |                                     |

**「人稱代詞＋量詞＋名詞」的結構**

武漢話中還存在大量「人稱代詞＋量詞＋名詞」的結構，此結構中的量詞有時可以表從屬關係，而這樣的結構在漢語普通話中也是不合乎語法的。

|       |                    |                 |                  |                   |                    |       |
| ----- | ------------------ | --------------- | ---------------- | ----------------- | ------------------ | ----- |
| 武漢話範例 | 你                  | 個               | 伢                | 蠻                 | 精                  | 咧     |
| IPA   | li<sup>42</sup>    | ko<sup>35</sup> | ŋa<sup>213</sup> | man<sup>213</sup> | t͡ɕin<sup>55</sup> | liɛ   |
| 逐字翻譯  | 你 2SG              | 個 CL            | 孩子 kid           | 很 very            | 機靈 smart           | 呢 PTC |
| 普通話   | 你這個孩子很機靈呢/你的孩子很機靈呢 |                 |                  |                   |                    |       |

## 近年发展

近年，普通话极为强烈地影响武汉话，致使武汉方言较有特色的语音、语法、词汇现象大幅被普通话所取代，形成新的武汉话，这在外來人口衆多的地區例如高校密集的地区尤为普遍。新的武汉话在某种程度上可以理解为改变调值的普通话。武汉艺术家[夏雨田用](https://zh.wikipedia.org/wiki/夏雨田 "wikilink")「办事脑子灵，说话喉咙喳」来形容武汉话。由于政府的大力推广，普通话自解放后即开始影响武汉话的语音，目前尤为明显。同时，武汉还有「弯管子普通话」，即把武汉话套上普通话的调值，这种语言通常是调侃而用。湖北省和武汉市的有些电视台开辟了方言节目，比如《经视故事会》、《新闻故事会》和《都市茶座》等。

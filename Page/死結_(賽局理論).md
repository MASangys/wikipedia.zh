在[賽局中](../Page/博弈论.md "wikilink")，**死結** 是指一個遊戲中，雙方得到最大利益的純策略為[支配性策略](https://zh.wikipedia.org/wiki/支配性策略 "wikilink")，且和對方互斥(右邊的[收支矩陣就是一個例子](https://zh.wikipedia.org/wiki/收支矩陣 "wikilink"))。對於列玩家(row)而言，選擇*d*能得到的利益永遠比*c*大，行(column)玩家也是如此，因為他們會使用(*d*, *D*)。

## 定義

|     |      |      |
| --- | ---- | ---- |
|     | *C*  | *D*  |
| *c* | a, b | c, d |
| *d* | e, f | g, h |

任何滿足了以下兩個條件的，就是一個死結：(1) e\>g\>a\>c 和 (2) d\>h\>b\>f。以上條件都需要*d*和*D*是支配性策略。(*d*, *D*)對雙方而言是互斥的，他們都希望對手選擇*c*(*C*)而不是*d*(*D*)。

和囚犯困境一樣，這些遊戲只有唯一一個[納什均衡點](../Page/納什均衡點.md "wikilink")：(*d*, *D*)。

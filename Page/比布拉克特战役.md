**比布拉克特战役**（）是公元前58年，[赫尔维蒂人部落与](https://zh.wikipedia.org/wiki/赫尔维蒂人 "wikilink")[凯撒麾下的六个](https://zh.wikipedia.org/wiki/凯撒 "wikilink")[军团之间发生的一场战斗](../Page/罗马军团.md "wikilink")。这是[高卢战争中第二次重大战役](https://zh.wikipedia.org/wiki/高卢战争 "wikilink")。

同时根据凯撒在战争开始时统计的部落人口总数：

| 部落名    | 统计人口（人） |
| ------ | ------- |
| 赫尔维蒂人  | 263,000 |
| 图林吉人   | 36,000  |
| 拉托布里契人 | 14,000  |
| 劳拉奇人   | 23,000  |
| 波伊人    | 32,000  |
| 共计     | 368,000 |
| 战士     | 92,000  |

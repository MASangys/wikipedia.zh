**毛里求斯国旗**，称为**四色旗**（，），由紅、藍、黃、綠四個平行相等的長方形構成，于1968年3月12日在独立之时采用。

国旗由Gurudutt Moher设计，当时他是一名小学教师。

## 象征意义

紅色象徵為獨立自由而鬥爭，藍色表示[毛里求斯](../Page/毛里求斯.md "wikilink")位于藍色的南[印度洋](../Page/印度洋.md "wikilink")，黃色象徵獨立的光芒照耀島國，綠色表示國家的農業和四季常青。

## 格式

横向四条等宽。

颜色标准：[彩通](../Page/彩通.md "wikilink")家居时尚棉布色卡

  - 红色 18-1664TC 或 185 C
  - 蓝色 19-3939TC 或 295 C
  - 黄色 14-0957TC 或 116 C
  - 绿色 17-6030TC 或 356 C

**氟化钐**是一种[无机化合物](../Page/无机化合物.md "wikilink")，化学式为SmF<sub>3</sub>。

## 制备

氟化钐可由SmCl<sub>3</sub>或Sm<sub>2</sub>(CO<sub>3</sub>)<sub>3</sub>和40%氢氟酸反应得到：\[1\]

  -
    SmCl<sub>3</sub> + 3 HF → SmF<sub>3</sub>↓ + 3 HCl
    Sm<sub>2</sub>(CO<sub>3</sub>)<sub>3</sub> + 6 HF → 2
    SmF<sub>3</sub> + 3 H<sub>2</sub>O + 3 CO<sub>2</sub>↑

[硝酸钐和](https://zh.wikipedia.org/wiki/硝酸钐 "wikilink")[氟硼酸钠在](../Page/氟硼酸钠.md "wikilink")200
°C进行水热反应，也能制得SmCl<sub>3</sub>。\[2\]

## 化学性质

氟化钐在高温下和一些还原剂反应，可以得到[二氟化钐](https://zh.wikipedia.org/wiki/二氟化钐 "wikilink")：\[3\]：

  -
    \(\mathsf{ 4 SmF_3 + C \ \xrightarrow{2000^oC}\ 4 SmF_2 + CF_4 }\)
    \(\mathsf{ 2 SmF_3 + Sm \ \xrightarrow{1800^oC}\ 3 SmF_2 }\)

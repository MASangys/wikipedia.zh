**氢化锌**（化学式：ZnH<sub>2</sub>）是二价[锌](../Page/锌.md "wikilink")生成的[氢化物](../Page/氢化物.md "wikilink")。

## 性质

白色非挥发性固体，贮存时发生分解变灰。\[1\]可以[聚合](https://zh.wikipedia.org/wiki/聚合 "wikilink")。有明显的[共价键](../Page/共价键.md "wikilink")化合物特性。\[2\]

蒸汽中存在直线型的 ZnH<sub>2</sub> 分子，键长 153.5 pm。\[3\]

易被空气氧化，甚至着火。可以被水分解放出[氢气](https://zh.wikipedia.org/wiki/氢气 "wikilink")，遇酸剧烈分解。不溶于[乙醚](../Page/乙醚.md "wikilink")。真空中高于80℃时分解为[锌](../Page/锌.md "wikilink")和[氢](../Page/氢.md "wikilink")。在[醚](../Page/醚.md "wikilink")中[二硼烷使它转变为](https://zh.wikipedia.org/wiki/二硼烷 "wikilink")[硼氢化锌](../Page/硼氢化锌.md "wikilink")（Zn(BH<sub>4</sub>)<sub>2</sub>）。\[4\]

## 制取

1、在[乙醚](../Page/乙醚.md "wikilink")溶液中，由[二甲基锌和](https://zh.wikipedia.org/wiki/二甲基锌 "wikilink")[氢化铝锂](../Page/氢化铝锂.md "wikilink")反应，得到白色沉淀，用乙醚洗涤，加热至50℃，除去乙醚而得。\[5\]

\[\rm \ Zn(CH_3)_2 \ + \ 2LiAlH_4 \ \xrightarrow[0^oC] {Et_2O} \ ZnH_2 \ + \ 2LiAlH_3CH_3\]

2、[碘化锌](../Page/碘化锌.md "wikilink")与[氢化铝锂](../Page/氢化铝锂.md "wikilink")低温在乙醚中反应而得：

\[\rm \ 2ZnI_2 \ + \ LiAlH_4 \ \xrightarrow[-40^oC]{Et_2O} \ 2ZnH_2 \ + \ LiI \ + \ AlI_3\]

3、由碱金属氢化物与卤化锌反应制备：\[6\]\[7\]

\[\rm \ 2LiH \ + \ ZnBr_2 \ \rightarrow \ ZnH_2 \ + \ 2LiBr\]

\[\rm \ 2NaH \ + \ ZnI_2 \ \rightarrow \ ZnH_2 \ + \ 2NaI\]

于1947年由 Finholt 等首次制得。\[8\] 熔融锌中只能溶解极少量氢，因此从300℃起到锌的熔点，锌与氢并不发生直接的反应。\[9\]

## 用途

[有机合成](../Page/有机合成.md "wikilink")中用作[还原剂](https://zh.wikipedia.org/wiki/还原剂 "wikilink")。

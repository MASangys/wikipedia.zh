| Section1 =  | Section2 =  | Section7 =  | Section8 =  }} **氫氧化鈉**，又称**烧碱**和**苛性钠**，[化學式為](https://zh.wikipedia.org/wiki/化學式 "wikilink")，是一種具有高[腐蝕性](../Page/腐蝕性.md "wikilink")的強[鹼](https://zh.wikipedia.org/wiki/鹼 "wikilink")，一般為白色片狀或顆粒，能[溶於](../Page/溶解.md "wikilink")[水](../Page/水.md "wikilink")生成鹼性[溶液](../Page/溶液.md "wikilink")，另也能溶解於[甲醇](../Page/甲醇.md "wikilink")及[乙醇](../Page/乙醇.md "wikilink")。此鹼性物具有[潮解](../Page/潮解.md "wikilink")性，會吸收[空氣裡的](https://zh.wikipedia.org/wiki/空氣 "wikilink")[水蒸氣](https://zh.wikipedia.org/wiki/水蒸氣 "wikilink")，亦會吸取[二氧化硫](../Page/二氧化硫.md "wikilink")等酸性氣體。

氫氧化鈉為常用的化學品之一。其應用廣泛，為很多[工業過程的必需品](https://zh.wikipedia.org/wiki/工業 "wikilink")：常用於製造[木浆](../Page/木浆.md "wikilink")[紙張](https://zh.wikipedia.org/wiki/紙 "wikilink")、[紡織品](https://zh.wikipedia.org/wiki/紡織品 "wikilink")、[肥皂](../Page/肥皂.md "wikilink")及其他[清潔劑等](https://zh.wikipedia.org/wiki/清潔劑 "wikilink")，另也用於家用的[水管疏通剂](https://zh.wikipedia.org/wiki/水管疏通剂 "wikilink")。2004年全球總共製造了六千萬噸的氫氧化鈉，而總消耗量為五千一百萬噸。\[1\]

## 化学性質

### 和酸反应

氢氧化钠於水中會完全解离成钠离子與氢氧根离子，可以和[酸](../Page/酸.md "wikilink")进行酸鹼中和反应：

\[\rm NaOH + HCl \rightarrow NaCl + H_2O\]

\[\rm 2NaOH + H_2SO_4 \rightarrow Na_2SO_4 + 2H_2O\] 利用这一性质，可以制备一些酸的钠盐，如：

\[\rm NaOH + C_6H_5COOH \rightarrow C_6H_5COONa + H_2O\] 酸性很弱的[苯酚](../Page/苯酚.md "wikilink")也能与之反应：

\[\rm NaOH + C_6H_5OH \rightarrow C_6H_5ONa + H_2O\]

### 和酸性氧化物反应

氫氧化鈉在空氣中容易變質，就是因為和空氣中的[二氧化碳](../Page/二氧化碳.md "wikilink")发生了反应：

\[\rm 2NaOH + CO_2 \rightarrow Na_2CO_3 + H_2O\] 在溶液中发生，过量的二氧化碳会将[碳酸钠](../Page/碳酸钠.md "wikilink")转化为[碳酸氫鈉](https://zh.wikipedia.org/wiki/碳酸氫鈉 "wikilink")（俗稱小蘇打）：

\[\rm Na_2CO_3 + CO_2 +H_2O \rightarrow 2NaHCO_3\] 由于玻璃製品中含有[二氧化硅](../Page/二氧化硅.md "wikilink")，氢氧化钠会与之反应生成[硅酸钠](https://zh.wikipedia.org/wiki/硅酸钠 "wikilink")，使得玻璃儀器中的活塞黏著於儀器上，無法再次使用。因此，存放氢氧化钠的细口瓶一般用橡胶塞封口。如果以玻璃容器长时间盛裝热的氫氧化鈉溶液，会造成玻璃容器损坏，甚至破裂的情況。

\[\rm 2NaOH + SiO_2 \rightarrow Na_2SiO_3 + H_2O\] 同样地，氢氧化钠也能和[三氧化铬](../Page/三氧化铬.md "wikilink")、[五氧化二磷](../Page/五氧化二磷.md "wikilink")、[三氧化二砷](../Page/三氧化二砷.md "wikilink")、[二氧化硫](../Page/二氧化硫.md "wikilink")、[二氧化硒](../Page/二氧化硒.md "wikilink")等其它酸性氧化物反应，生成它们的盐：

\[\rm 2NaOH + CrO_3 \rightarrow Na_2CrO_4 + H_2O\]

\[\rm 2NaOH + 2CrO_3 \rightarrow Na_2Cr_2O_7 + H_2O\]

### 和两性氧化物及氢氧化物的反应

氢氧化钠可以和[两性氧化物或氢氧化物反应](https://zh.wikipedia.org/wiki/两性氧化物 "wikilink")，生成盐和水，如：

\[\rm 2NaOH + Al_2O_3 + 3H_2O \rightarrow 2Na[Al(OH)_4]\]

### 和单质的反应

[硼](../Page/硼.md "wikilink")、[硅](../Page/硅.md "wikilink")和两性金属（如[铍](../Page/铍.md "wikilink")、[铝](../Page/铝.md "wikilink")、[锌](../Page/锌.md "wikilink")等）和氢氧化钠反应，放出氢气：

\[\rm Si + 2NaOH + H_2O \rightarrow Na_2SiO_3 + 2H_2 \uparrow\] 英國在1986年有一油罐車誤裝載重量百分率濃度為25%的氫氧化鈉水溶液，氫氧化鈉便與油罐壁上的[鋁產生化學變化](https://zh.wikipedia.org/wiki/鋁 "wikilink")，導致油罐因內部壓力過載而受損，反應方程式如下所示：

\[\rm 2Al + 2NaOH + 2H_2O \rightarrow 2NaAlO_2 + 3H_2 \uparrow\] [氯](../Page/氯.md "wikilink")、[溴](../Page/溴.md "wikilink")、[碘](../Page/碘.md "wikilink")、[硫](../Page/硫.md "wikilink")、[白磷等单质和氢氧化钠发生歧化反应](https://zh.wikipedia.org/wiki/白磷 "wikilink")：

\[\rm Cl_2 + 2NaOH \rightarrow NaCl + NaOCl + H_2O\]

### 和盐的反应

过渡金属的盐类和一些主族金属的盐可以和氢氧化钠反应，生成更难溶的氢氧化物，或转化为可溶性的[羟基配合物再次溶解](https://zh.wikipedia.org/wiki/羟基配合物 "wikilink")。

\[\rm 2NaOH + CoSO_4 \rightarrow Na_2SO_4 + Co(OH)_2\]

\[\rm 2NaOH + CuCl_2 \rightarrow 2NaCl + Cu(OH)_2\]

\[\rm 3NaOH + Cr(NO_3)_3 \rightarrow 3NaNO_3 + Cr(OH)_3\]

\[\rm NaOH + Cr(OH)_3 \rightarrow Na[Cr(OH)_4]\] 对于[汞](../Page/汞.md "wikilink")等氢氧化物不能稳定存在的物质来说，会生产氧化物或氧化物的水合物沉淀：

\[\rm 2NaOH + HgCl_2 \rightarrow 2NaCl + HgO + H_2O\] 氢氧化钠可以将过渡金属的[酰基](../Page/酰基.md "wikilink")离子转化为阴离子酸根（简单离子或多聚离子），如VO<sub>2</sub><sup>+</sup>、UO<sub>2</sub><sup>2+</sup>等：

\[\rm 6NaOH + 2UO_2(NO_3)_2 \rightarrow 4NaNO_3 + Na_2U_2O_7 + 3H_2O\]

### 和其它无机物的反应

[二氧化氯](../Page/二氧化氯.md "wikilink")、[二氧化氮](../Page/二氧化氮.md "wikilink")等可以和氢氧化钠发生反应。

[高锰酸钠](../Page/高锰酸钠.md "wikilink")在氢氧化钠溶液中煮沸，可以还原为[锰酸钠](https://zh.wikipedia.org/wiki/锰酸钠 "wikilink")。

### 和有机物的反应

氢氧化钠可以将[卤代烃](https://zh.wikipedia.org/wiki/卤代烃 "wikilink")、[酯](../Page/酯.md "wikilink")和[酰卤](../Page/酰卤.md "wikilink")水解。

## 工业制法

氢氧化钠為工业中製[氯氣过程的副产物](https://zh.wikipedia.org/wiki/氯氣 "wikilink")，工業上，透過[电解](../Page/电解.md "wikilink")大量饱和食盐水以量產氯氣，然而，當氯離子氧化成氯气逸出的同時，溶液中也只剩下氢氧化钠一种溶质，反应方程式如下所示：

  -
    <chem>2NaCl + 2H_2O -\> 2NaOH + Cl_2 ^ + H_2 ^</chem>

19世紀以前，氫氧化鈉的製備通常會先以[勒布朗制碱法](../Page/勒布朗制碱法.md "wikilink")生成前驅物[碳酸钠](../Page/碳酸钠.md "wikilink")，再通过高温煅烧，使得碳酸钠分解为[氧化钠](../Page/氧化钠.md "wikilink")與[二氧化碳](../Page/二氧化碳.md "wikilink")，最後，將氧化钠溶於水中，便可製得氢氧化钠：

  -
    <chem>Na_2CO_3 -\> Na_2O + CO_2 ^</chem>
    <chem>Na_2O + H_2O -\> 2NaOH</chem>

另外，也可利用[複分解制備氫氧化鈉](https://zh.wikipedia.org/wiki/複分解反應 "wikilink")，碳酸鈉（俗稱為蘇打）與[氫氧化鈣](https://zh.wikipedia.org/wiki/氫氧化鈣 "wikilink")（俗稱為熟石灰）的反應方程式如下所示：\[2\]

  -
    <chem>Na_2CO_3 + Ca(OH)_2 -\> CaCO_3 v + 2NaOH</chem>

## 安全性

[GHS-pictogram-acid.svg](https://zh.wikipedia.org/wiki/File:GHS-pictogram-acid.svg "fig:GHS-pictogram-acid.svg") [Sodium_hydroxide_burn.png](https://zh.wikipedia.org/wiki/File:Sodium_hydroxide_burn.png "fig:Sodium_hydroxide_burn.png")

氢氧化钠固体或其溶液皆能灼伤皮肤，對无防护措施者可造成永久性伤害（如疤痕）。倘若讓氫氧化鈉直接接触眼睛的話，严重者甚至可造成[失明](../Page/失明.md "wikilink")。个人防护措施，諸如橡胶手套、防護衣與护目镜等便能大大降低接触氢氧化钠所带来的危险。

氢氧化钠溶于水中（如稀釋），抑或是與酸反應，都会放出大量的热量，可能导致灼伤或点燃[易燃物](https://zh.wikipedia.org/wiki/易燃物 "wikilink")（如有機溶劑）。除此之外，氢氧化钠能夠腐蚀一些金属（如[铝](../Page/铝.md "wikilink")）生成易燃的[氢](../Page/氢.md "wikilink")气，还能夠轻度腐蚀玻璃製品，應慎選儲存氫氧化鈉的容器材質。\[3\]

    <chem>2Al + 2NaOH + 2H_2O -\> 3H_2 + 2NaAlO_2</chem>
    <chem>2Al + 6NaOH + xH_2O -\> 3H_2 + 2 Na_3AlO_3 + x H_2O</chem>
    <chem>2Al + 2NaOH + 6H_2O -\> 3H_2 + 2NaAl(OH)_4</chem>

**氦合氢离子**，化学式为HeH<sup>+</sup>，是一个带正电的[离子](../Page/离子.md "wikilink")。它首次发现于1925年，通过[质子](https://zh.wikipedia.org/wiki/质子 "wikilink")（或[氢离子](https://zh.wikipedia.org/wiki/氢正离子 "wikilink")）和[氦原子在气相中反应制得](../Page/氦.md "wikilink")。\[1\]它是已知最强的[布朗斯特质子酸](../Page/酸碱质子理论.md "wikilink")，[质子亲和能为](https://zh.wikipedia.org/wiki/质子亲和能 "wikilink")177.8 kJ/mol。\[2\]这种离子也被称为**氦氢分子离子**。有人认为，这种物质可以存在于自然[星际物质中](../Page/星际物质.md "wikilink")。\[3\]这是最简单的异核离子，可以与同核的[氢分子离子H](https://zh.wikipedia.org/wiki/氢分子离子 "wikilink")<sub>2</sub><sup>+</sup>相比较。与H<sub>2</sub><sup>+</sup>不同的是，它有一个永久的[键偶极矩](https://zh.wikipedia.org/wiki/键偶极矩 "wikilink")，使它更容易表现出[光谱特征](https://zh.wikipedia.org/wiki/光谱 "wikilink")。\[4\]英国《自然》杂志于2019年4月17日发表相关学术成果：人类初次在太空中检测到了氦合氢离子。\[5\]

## 性质

HHe<sup>+</sup>不能在[凝聚相中制备](https://zh.wikipedia.org/wiki/凝聚相 "wikilink")，因为这会使它与任何[阴离子](https://zh.wikipedia.org/wiki/阴离子 "wikilink")、分子、原子发生作用。但是，可以用[盖斯定律预测它在水溶液中的酸性](https://zh.wikipedia.org/wiki/盖斯定律 "wikilink")：

|                           |       |                         |                |                  |       |
| ------------------------- | ----- | ----------------------- | -------------- | ---------------: | ----- |
| HHe<sup>+</sup>(*g*)      | →     | H<sup>+</sup>(*g*)      | \+ He(*g*)     |     \+178 kJ/mol | \[6\] |
| HHe<sup>+</sup>(*aq*)     | →     | HHe<sup>+</sup>(*g*)    |                |     \+973 kJ/mol | \[7\] |
| H<sup>+</sup>(*g*)        | →     | H<sup>+</sup>(*aq*)     |                |    – 1530 kJ/mol |       |
| He(*g*)                   | →     | He(*aq*)                |                |      \+19 kJ/mol | \[8\] |
| **HHe<sup>+</sup>(*aq*)** | **→** | **H<sup>+</sup>(*aq*)** | **+ He(*aq*)** | **– 360 kJ/mol** |       |
|                           |       |                         |                |                  |       |

电离过程–360 kJ/mol的[自由能变化相当于](https://zh.wikipedia.org/wiki/自由能 "wikilink")[p*K*<sub>a</sub>为](../Page/酸度系数.md "wikilink")-63。

HeH<sup>+</sup>中[共价键的长度是](../Page/共价键.md "wikilink")0.772[Å](https://zh.wikipedia.org/wiki/埃 "wikilink")。\[9\]

其他氦氢离子已经知道或者在理论上研究。HeH<sub>2</sub><sup>+</sup>，已经被微波光谱观测到，\[10\]科学家计算出它的亲和能为6
kcal/mol，而HeH<sub>3</sub><sup>+</sup>为0.1 kcal/mol。\[11\]

## 中性分子

不同于氦合氢离子，氢和氦构成的中性分子在一般情况下是很不稳定的。但是，它作为一个[准分子在激发态是稳定的](https://zh.wikipedia.org/wiki/准分子 "wikilink")，于20世纪80年代中期首次在光谱中观测到。\[12\]\[13\]\[14\]

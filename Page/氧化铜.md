**氧化銅**（化学式：CuO）是[铜](../Page/铜.md "wikilink")的[氧化物](../Page/氧化物.md "wikilink")，为黑色固体。属于[强电解质](../Page/电解质.md "wikilink")。不溶于[水](../Page/水.md "wikilink")和[乙醇](../Page/乙醇.md "wikilink")，溶于[酸](../Page/酸.md "wikilink")、[氯化铵](../Page/氯化铵.md "wikilink")及[氰化钾](../Page/氰化钾.md "wikilink")溶液，[氨](../Page/氨.md "wikilink")溶液中缓慢溶解。

## 制备

氧化铜可由[氢氧化铜](../Page/氢氧化铜.md "wikilink")的热分解制得：\[1\]

  -
    Cu(OH)<sub>2</sub> —<u><sup>353-363 K</sup></u>→ CuO + H<sub>2</sub>O

## 化学性质

氧化铜在一定温度下，可以被一些具有还原性的气体还原，如[氢气](https://zh.wikipedia.org/wiki/氢气 "wikilink")、[碳](../Page/碳.md "wikilink")、[一氧化碳](../Page/一氧化碳.md "wikilink")、[氨](../Page/氨.md "wikilink")等：

  -
    CuO + CO → Cu + CO<sub>2</sub>
    C + 2 CuO → CO<sub>2</sub> + 2 Cu
    CuO + H<sub>2</sub> → Cu + H<sub>2</sub>O

氧化铜和铝的混合物在灼热时会发生爆炸：

  -
    3 CuO + 2 Al → 2Al<sub>2</sub>O<sub>3</sub> + 3 Cu

氧化铜可溶于酸，形成铜盐，如：

  -
    CuO + H<sub>2</sub>SO<sub>4</sub> → CuSO<sub>4</sub> + H<sub>2</sub>O
    CuO + 2 HClO<sub>4</sub> → Cu(ClO<sub>4</sub>)<sub>2</sub> + H<sub>2</sub>O

**氫氧化鉛**，是一種[無機化合物](https://zh.wikipedia.org/wiki/無機化合物 "wikilink")，其[化學式為Pb](https://zh.wikipedia.org/wiki/化學式 "wikilink")(OH)<sub>2</sub>，是[鉛的](https://zh.wikipedia.org/wiki/鉛 "wikilink")[氫氧化物](https://zh.wikipedia.org/wiki/氫氧化物 "wikilink")，由一個2價的[鉛和](https://zh.wikipedia.org/wiki/鉛 "wikilink")[2](../Page/2.md "wikilink")個[氫氧根離子](https://zh.wikipedia.org/wiki/氫氧根離子 "wikilink")([O](../Page/氧.md "wikilink")[H](https://zh.wikipedia.org/wiki/氫 "wikilink")<sup>[-](https://zh.wikipedia.org/wiki/陰離子 "wikilink")</sup>)組成。**氫氧化鉛**是一種[鉛鏽](https://zh.wikipedia.org/wiki/鉛鏽 "wikilink")(其他還有:[一氧化鉛](../Page/一氧化鉛.md "wikilink")、[二氧化鉛](../Page/二氧化鉛.md "wikilink")、[四氧化三鉛](../Page/四氧化三鉛.md "wikilink"))，氫氧化鉛常溫下為固體，外觀通常是白色的，當鉛放在水裡久了[生鏽生成的](https://zh.wikipedia.org/wiki/生鏽 "wikilink")[鉛鏽含有](https://zh.wikipedia.org/wiki/鉛鏽 "wikilink")**氫氧化鉛**。

## 制備

氫氧化鉛可以由[氫氧化鈉或](https://zh.wikipedia.org/wiki/氫氧化鈉 "wikilink")[氫氧化鉀與](https://zh.wikipedia.org/wiki/氫氧化鉀 "wikilink")[硝酸铅反应制得](https://zh.wikipedia.org/wiki/硝酸铅 "wikilink")\[1\]

  -
    Pb(NO<sub>3</sub>)<sub>2</sub> + 2NaOH → Pb(OH)<sub>2</sub> + 2NaNO<sub>3</sub>

[鉛在](https://zh.wikipedia.org/wiki/鉛 "wikilink")[水](../Page/水.md "wikilink")中(要含[氧](../Page/氧.md "wikilink"))也會緩慢的生成氫氧化鉛，若至於[碳酸水](../Page/碳酸水.md "wikilink")則會加快這個反應\[2\]

  -
    2Pb + O<sub>2</sub> + 2H<sub>2</sub>O → 2Pb(OH)<sub>2</sub>

## 性質

氫氧化鉛是一種白色無定形粉末，兩性，它溶解在水中呈[弱鹼性](https://zh.wikipedia.org/wiki/弱鹼性 "wikilink")。它在脫水溫度[130](https://zh.wikipedia.org/wiki/130 "wikilink") [°C和](https://zh.wikipedia.org/wiki/攝氏溫標 "wikilink")[145](https://zh.wikipedia.org/wiki/145 "wikilink")[°C時會分解為](https://zh.wikipedia.org/wiki/攝氏度 "wikilink")[一氧化鉛](../Page/一氧化鉛.md "wikilink")和[水](../Page/水.md "wikilink")\[3\]。

  -
    Pb(OH)<sub>2</sub><sup>\(\xrightarrow{\ 145 ^o C \ }\)</sup>PbO + H<sub>2</sub>O

氫氧化鉛與[二氧化碳](../Page/二氧化碳.md "wikilink")反應會形成[碳酸鉛](https://zh.wikipedia.org/wiki/碳酸鉛 "wikilink")

  -
    Pb(OH)<sub>2</sub> + CO<sub>2</sub> → PbCO<sub>3</sub> + H<sub>2</sub>O

氫氧化鉛接觸酸會形成對應的鉛鹽(如接觸[硫酸](../Page/硫酸.md "wikilink")形成[硫酸鉛](https://zh.wikipedia.org/wiki/硫酸鉛 "wikilink")、若接觸[檸檬酸](../Page/檸檬酸.md "wikilink")也會形成[檸檬酸鉛](https://zh.wikipedia.org/wiki/檸檬酸鉛 "wikilink")、接觸[硝酸](../Page/硝酸.md "wikilink")則形成[硝酸鉛](https://zh.wikipedia.org/wiki/硝酸鉛 "wikilink"))

  -
    Pb(OH)<sub>2</sub> + H<sub>2</sub>SO<sub>4</sub> → PbSO<sub>4</sub> + 2H<sub>2</sub>O
    3Pb(OH)<sub>2</sub> + 2C<sub>6</sub>H<sub>8</sub>O<sub>7</sub> → Pb<sub>3</sub>(C<sub>6</sub>H<sub>5</sub>O<sub>7</sub>)<sub>2</sub> + 3H<sub>2</sub>O
    Pb(OH)<sub>2</sub> + 2HNO<sub>3</sub> → Pb(NO<sub>3</sub>)<sub>2</sub> + 2H<sub>2</sub>O

當它溶解於[強鹼](https://zh.wikipedia.org/wiki/強鹼 "wikilink")[溶液](../Page/溶液.md "wikilink")的羥基形成一個複雜的離子\[Pb(OH)<sub>4</sub>\]<sup>2−</sup>、Pb<sub>4</sub>(OH)<sub>4</sub><sup>4+</sup>、Pb<sub>3</sub>(OH)<sub>4</sub><sup>2+</sup>、Pb<sub>6</sub>O(OH)<sub>6</sub><sup>4+</sup>.\[4\] 和形成[铅酸盐](https://zh.wikipedia.org/wiki/铅酸盐 "wikilink")\[5\]\[6\]

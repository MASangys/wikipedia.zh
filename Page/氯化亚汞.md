**氯化亚汞**（[化学式](../Page/化学式.md "wikilink")：），俗称**甘汞**，室温下为无味白色至黄色的[四方晶系](../Page/四方晶系.md "wikilink")晶体，是[汞](../Page/汞.md "wikilink")(I)的[氯化物](https://zh.wikipedia.org/wiki/氯化物 "wikilink")。少量的甘汞无毒，常用于制取[饱和甘汞电极](https://zh.wikipedia.org/wiki/饱和甘汞电极 "wikilink")，是[电化学](../Page/电化学.md "wikilink")中常用的[参比电极](https://zh.wikipedia.org/wiki/参比电极 "wikilink")，\[1\]\[2\] 也可在医药上用于泻剂、防腐剂、利尿剂等。25°C时，它在水中的[标准电极电势](../Page/标准电极电势.md "wikilink")*E*<sup>o</sup>为0.2682V：

  -

      -
        Hg<sub>2</sub>Cl<sub>2(c)</sub> + 2 e<sup>−</sup> ⇌ 2 Hg<sub>(l)</sub> + 2 Cl<sup>−</sup> ； *E*<sup>o</sup> = 0.2682V

## 制备

在[氯化汞](../Page/氯化汞.md "wikilink")的溶液中通入[二氧化硫](../Page/二氧化硫.md "wikilink")，或将[汞](../Page/汞.md "wikilink")与氯化汞在一起混合研磨都可以得到氯化亚汞：

  -
    2 HgCl<sub>2</sub> + SO<sub>2</sub> + 2 H<sub>2</sub>O → Hg<sub>2</sub>Cl<sub>2</sub>↓ + 2 HCl + H<sub>2</sub>SO<sub>4</sub>
    HgCl<sub>2</sub> + Hg → Hg<sub>2</sub>Cl<sub>2</sub>

[硝酸亚汞与](https://zh.wikipedia.org/wiki/硝酸亚汞 "wikilink")[氯化钠或](https://zh.wikipedia.org/wiki/氯化钠 "wikilink")[盐酸](../Page/盐酸.md "wikilink")在水中发生[复分解反应](../Page/复分解反应.md "wikilink")，也会生成氯化亚汞沉淀：

  -
    2HCl + Hg<sub>2</sub>(NO<sub>3</sub>)<sub>2</sub> → Hg<sub>2</sub>Cl<sub>2</sub>↓ + 2HNO<sub>3</sub>

酸性溶液中，[氯化亚锡会将二价汞离子](https://zh.wikipedia.org/wiki/氯化亚锡 "wikilink")（Hg<sup>2+</sup>）还原为Hg<sub>2</sub>Cl<sub>2</sub>。如果氯化亚锡过量，生成的氯化亚汞会被进一步还原为黑色的金属汞，使沉淀变黑。这个反应可用于鉴定Hg<sup>2+</sup>或Sn<sup>2+</sup>的存在：

  -
    2 HgCl<sub>2</sub> + SnCl<sub>2</sub> + 2HCl → Hg<sub>2</sub>Cl<sub>2</sub>↓ + H<sub>2</sub>SnCl<sub>6</sub>
    Hg<sub>2</sub>Cl<sub>2</sub> + SnCl<sub>2</sub> + 2 HCl → 2 Hg↓ + H<sub>2</sub>SnCl<sub>6</sub>

## 反应

用[氨水](../Page/氨水.md "wikilink")或[碱](../Page/碱.md "wikilink")溶液处理氯化亚汞，汞发生[歧化为二价汞与单质汞](https://zh.wikipedia.org/wiki/歧化 "wikilink")：

  -
    Hg<sub>2</sub>Cl<sub>2</sub> + 2 NH<sub>3</sub> → Hg + Hg(NH<sub>2</sub>)Cl + NH<sub>4</sub>Cl
    Hg<sub>2</sub>Cl<sub>2</sub> + 2 OH<sup>−</sup> → Hg + HgO + 2 Cl<sup>−</sup> + H<sub>2</sub>O

与氨的反应可用于在[分析化学](../Page/分析化学.md "wikilink")中鉴定亚汞离子，[氯化氨基汞](../Page/氯化氨基汞.md "wikilink")（HgNH<sub>2</sub>Cl）是白色沉淀，常混有细小金属汞颗粒而显黑色。\[3\] 氯化亚汞也可被[硫酸](../Page/硫酸.md "wikilink")和[硝酸](../Page/硝酸.md "wikilink")氧化为二价的[硫酸汞](../Page/硫酸汞.md "wikilink")和[硝酸汞](../Page/硝酸汞.md "wikilink")，相应的还原产物为[二氧化硫](../Page/二氧化硫.md "wikilink")及[氮氧化物](../Page/氮氧化物.md "wikilink")。

[饱和甘汞电极是](https://zh.wikipedia.org/wiki/饱和甘汞电极 "wikilink")[电化学](../Page/电化学.md "wikilink")中常用的[参比电极](https://zh.wikipedia.org/wiki/参比电极 "wikilink")，比[标准氢电极](../Page/标准氢电极.md "wikilink")更易操作，但由于汞对环境的毒害，它已在很大程度上被毒性更小的[银](https://zh.wikipedia.org/wiki/银 "wikilink")/[氯化银](../Page/氯化银.md "wikilink")（Ag/AgCl）电极所取代。

### 分解

氯化亚汞很不稳定，见光便歧化分解为[氯化汞](../Page/氯化汞.md "wikilink")和汞，因此需要保存在棕色瓶中：

  -

      -
        Hg<sub>2</sub>Cl<sub>2</sub> → HgCl<sub>2</sub> + Hg

用[光化剂测定生成的汞](https://zh.wikipedia.org/wiki/光化剂 "wikilink")，该反应可用于计算光束中的[光子](../Page/光子.md "wikilink")数量。[氯化汞](../Page/氯化汞.md "wikilink")与[草酸铵](../Page/草酸铵.md "wikilink")在光照下反应生成氯化亚汞。该法首先由J.M. Eder在1880年发现，也称“Eder反应”。\[4\]

  -
    2HgCl<sub>2</sub> + (NH<sub>4</sub>)<sub>2</sub>C<sub>2</sub>O<sub>4</sub> + 光 → Hg<sub>2</sub>Cl<sub>2(s)</sub> + 2NH<sub>4</sub>Cl + 2CO<sub>2</sub>

## 性质

[锌族元素中](https://zh.wikipedia.org/wiki/锌族元素 "wikilink")，受[惰性电子对效应](../Page/惰性电子对效应.md "wikilink")的影响，只有汞形成正一价Hg<sub>2</sub><sup>2+</sup>的倾向较强。氯化亚汞分子为直线形，Cl-Hg-Hg-Cl，[晶格见下图](https://zh.wikipedia.org/wiki/晶格 "wikilink")。Hg-Hg键长为253pm（单质中Hg-Hg距离为300pm），Hg-Cl键长为243pm。\[5\] 汞为[八面体配位](https://zh.wikipedia.org/wiki/八面体 "wikilink")，与两个紧邻的氯与四个相隔较远的氯配位，后者距离为321pm。

氯化亚汞有毒，但由于它难溶于水，因此其毒性较[氯化汞](../Page/氯化汞.md "wikilink")而言要小得多。它以前曾用作[利尿剂](https://zh.wikipedia.org/wiki/利尿剂 "wikilink")、[泻剂](https://zh.wikipedia.org/wiki/泻剂 "wikilink")、肥皂及美白去斑霜的组分，但这些应用现在已经基本停止。[2,3-二巯基丙磺酸钠可作为汞的解毒剂](https://zh.wikipedia.org/wiki/2,3-二巯基丙磺酸钠 "wikilink")。\[6\]

## 卤化亚汞

其他汞(I)的[卤化物](../Page/卤化物.md "wikilink")中，[氟化亚汞](../Page/氟化亚汞.md "wikilink")是黄色的四方晶体，不稳定，与水反应立即水解为汞、[氧化汞](../Page/氧化汞.md "wikilink")和[氢氟酸](../Page/氢氟酸.md "wikilink")；[溴化亚汞](../Page/溴化亚汞.md "wikilink")是白色至淡黄色的四方晶体，难溶于水，可通过溶液中的复分解反应制备。它较氯化亚汞更不稳定，在[紫外光下发出橙色](https://zh.wikipedia.org/wiki/紫外光 "wikilink")[荧光](../Page/荧光.md "wikilink")；[碘化亚汞](../Page/碘化亚汞.md "wikilink")是黄色至绿色四方晶系晶体，难溶于水，可通过单质化合或溶液中的复分解制备。

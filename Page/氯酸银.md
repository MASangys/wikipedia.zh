**氯酸银**（AgClO<sub>3</sub>）是白色四方晶体。像所有氯酸盐一样，它可溶于水并在热水中溶解度增加\[1\]，是强氧化剂。作为一种简单的金属盐，在碱性无机化学实验中，它是一种常见的化学品。对光敏感，必须存储在深色密闭容器中。

## 制备

  - 用[氧化银](../Page/氧化银.md "wikilink")或[碳酸银](../Page/碳酸银.md "wikilink")与[氯酸](../Page/氯酸.md "wikilink")反应：

\[\mathsf{Ag_2O + 2HClO_3 \ \xrightarrow{}\ 2AgClO_3 + H_2O }\]

\[\mathsf{Ag_2CO_3 + 2HClO_3 \ \xrightarrow{}\ 2AgClO_3 + CO_2\uparrow + H_2O }\]

  - 向[氧化银](../Page/氧化银.md "wikilink")的悬浮液通入[氯气](https://zh.wikipedia.org/wiki/氯气 "wikilink")：

\[\mathsf{Ag_2O + 6Cl_2 + 5H_2O \ \xrightarrow{}\ 2AgClO_3 + 10HCl }\]

  - 用[氯酸钾](../Page/氯酸钾.md "wikilink")与[硝酸银](../Page/硝酸银.md "wikilink")反应：

\[\mathsf{AgNO_3 + KClO_3 \ \xrightarrow{85^oC}\ AgClO_3 + KNO_3 }\]

## 性质

  - 加热分解：

\[\mathsf{2AgClO_3 \ \xrightarrow{270^oC}\ 2AgCl + 3O_2\uparrow }\]

## 用途

作为[氯化](https://zh.wikipedia.org/wiki/卤化 "wikilink")[苯](../Page/苯.md "wikilink")环时的催化剂。\[2\]

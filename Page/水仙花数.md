在[数论中](../Page/数论.md "wikilink")，**[水仙花数](https://zh.wikipedia.org/wiki/水仙花 "wikilink")**（**Narcissistic
number**）\[1\]\[2\]，也被稱為**超完全数字不变数**（**pluperfect digital invariant,
PPDI**）\[3\]、**[自戀數](../Page/自戀.md "wikilink")**、**自幂數**、**[阿姆斯壯數](https://zh.wikipedia.org/wiki/阿姆斯壯 "wikilink")**或**阿姆斯特朗數**（**Armstrong
number**）\[4\] ，用来描述一个**N**位非负整数，其各位数字的**N**次方和等于该数本身。

## 水仙花数的定义

设有[自然数n](../Page/自然数.md "wikilink")，d为该自然数各位数字，即 *n* = *d*<sub>*k*</sub>*d*<sub>*k*-1</sub>...*d*<sub>1</sub> ，则有：

  -
    *n* = *d*<sub>*k*</sub>·10<sup>*k*-1</sup> + *d*<sub>*k*-1</sub>·10<sup>*k*-2</sup> + ... + *d*<sub>2</sub>·10 + *d*<sub>1</sub>,

如果该自然数n满足条件：

  -
    *n* = *d*<sub>*k*</sub><sup>*k*</sup> + *d*<sub>*k*-1</sub><sup>*k*</sup> + ... + *d*<sub>2</sub><sup>*k*</sup> + *d*<sub>1</sub><sup>*k*</sup>.

则这个自然数就被称为超完全数字不变数。 例如153、370、371及407就是三位超完全数字不变数，其各个数之立方和等于该数：

  -
    153 = 1<sup>3</sup> + 5<sup>3</sup> + 3<sup>3</sup>。
    370 = 3<sup>3</sup> + 7<sup>3</sup> + 0<sup>3</sup>。
    371 = 3<sup>3</sup> + 7<sup>3</sup> + 1<sup>3</sup>。
    407 = 4<sup>3</sup> + 0<sup>3</sup> + 7<sup>3</sup>。

若將條件放寬，一個N位数，其各个数之M[次方和等于该数](https://zh.wikipedia.org/wiki/次方 "wikilink")，M和N不一定相等，這樣的數稱為**完全數字不變數**（perfect
digital invariant）\[5\]\[6\]，例如數字4150等於各位數字的5次方。

  -
    4150 = 4<sup>5</sup> + 1<sup>5</sup> + 5<sup>5</sup> + 0<sup>5</sup>,

水仙花数一定是完全數字不變數，但完全數字不變數不一定是水仙花数。 严格意义来说水仙花数指三位数。

**水情预警信号**，是根据[中华人民共和国水利部制订的](../Page/中华人民共和国水利部.md "wikilink")《水情预警信号发布办法（试行）》发布的水文警报，其目的是防御和减轻水旱灾害，规范水情预警发布工作\[1\]。

现行的预警信号于2012年3月7日启用\[2\]。警报分为[洪水](../Page/洪水.md "wikilink")、[枯水等两类](https://zh.wikipedia.org/wiki/枯水 "wikilink")。并按照灾害警报重要性程度标注颜色，由低到高依次为蓝色、黄色、橙色、红色。级别越高表明灾害越严重，并启动相应的应急响应\[3\]。

## 洪水

洪水预警信号共分为四级，依据《水情预警信号发布办法（试行）》发布，反映的洪水量级分别为小洪水、中洪水、大洪水和特大洪水\[4\]。

| 等级 | 信号名称     | 信号标志 | 信号含义                                                                        |
| -- | -------- | ---- | --------------------------------------------------------------------------- |
| Ⅳ  | 洪水蓝色预警信号 |      | 水位（流量）接近警戒水位（流量）；或者洪水要素重现期接近 5 年；或者洪水可能对社会交通、农业生产和人类活动带来影响。                 |
| Ⅲ  | 洪水黄色预警信号 |      | 水位（流量）达到或超过警戒水位（流量）；或者洪水要素重现期达到或超过 5 年；或者洪水已开始对社会交通、农业生产和人类活动带来影响。          |
| Ⅱ  | 洪水橙色预警信号 |      | 水位（流量）达到或超过保证水位（流量）；或者洪水要素重现期达到或超过 20 年；或者洪水已开始并可能持续对社会交通、农业生产和人类活动带来较严重影响。 |
| Ⅰ  | 洪水红色预警信号 |      | 水位（流量）临近或超过历史最高水位（最大流量）；或者洪水要素重现期达到或超过 50 年；或者洪水对社会交通、农业生产和人类活动带来严重影响。      |

## 枯水

枯水预警信号共分为四级，依据《水情预警信号发布办法（试行）》发布，反映的枯水量级分别为轻度枯水、较重枯水、严重枯水和特别严重枯水\[5\]。

| 等级 | 信号名称     | 信号标志 | 信号含义                                                                                    |
| -- | -------- | ---- | --------------------------------------------------------------------------------------- |
| Ⅳ  | 枯水蓝色预警信号 |      | 水位（流量）接近旱警水位（流量）；或者30 天来水量比常年同期偏少 4 成以上；或者枯水可能对农业灌溉、工业用水、生态环境等用水安产生影响。                  |
| Ⅲ  | 枯水黄色预警信号 |      | 水位（流量）降至或低于旱警水位（流量）；或者30 天来水量比常年同期偏少 6 成以上；或者枯水可能或已开始城乡生活、工农业生产、生态环境等用水安全带来较重影响。        |
| Ⅱ  | 枯水橙色预警信号 |      | 水位（流量）降至或低于常年同期最低（小）；或者30 天来水量比常年同期偏少 7.5 成以上；或者枯水可已开始并可能持续对城乡生活、工农业生产、生态环境等用水安全带来严重影响。 |
| Ⅰ  | 枯水红色预警信号 |      | 水位（流量）降至或低于历史最低（小）；或者30 天来水量比常年同期偏少 9 成以上；或者枯水对城乡生活、工农业生产、生态环境等用水安全带来特别严重影响。            |

**永恒之蓝**（\[1\]\[2\]\[3\]）是[美国国家安全局开发的](../Page/美国国家安全局.md "wikilink")[漏洞利用程序](https://zh.wikipedia.org/wiki/漏洞利用 "wikilink")，于2017年4月14日被黑客组织[影子掮客洩漏](https://zh.wikipedia.org/wiki/影子掮客 "wikilink")\[4\]\[5\]。該工具利用445/TCP端口的[檔案分享協議的漏洞進行散播](../Page/伺服器訊息區塊.md "wikilink")。\[6\]

尽管[微软于](../Page/微软.md "wikilink")2017年3月14日已经发布过[Microsoft
Windows](https://zh.wikipedia.org/wiki/Windows "wikilink")[补丁修补了这个](https://zh.wikipedia.org/wiki/补丁_\(计算机\) "wikilink")[漏洞](https://zh.wikipedia.org/wiki/漏洞 "wikilink")，然而在5月12日[WannaCry](../Page/WannaCry.md "wikilink")[勒索軟體利用这个漏洞传播时](../Page/勒索軟體.md "wikilink")，很多用户仍然因为没有安装补丁而受害\[7\]\[8\]。

由于WannaCry的严重性，微软于2017年5月13日为已超过支持周期的[Microsoft Windows
XP](../Page/Windows_XP.md "wikilink")、[Microsoft Windows
8和](https://zh.wikipedia.org/wiki/Windows_8 "wikilink")[Windows Server
2003发布了紧急安全更新](../Page/Windows_Server_2003.md "wikilink")，以阻止其散布造成更大的危害\[9\]\[10\]。

2017年6月27日，有勒索软件冒充是Petya利用永恒之蓝漏洞在欧洲地区大肆传播，该勒索软件被命名为NotPetya。其带有较强的政治目的进行攻击即进行破坏而不是索要赎金，即便受害者支付赎金也不会得到任何密钥，因为NotPetya根本就没有编写密钥上传模块到控制者的服务器。此后真正的Petya作者公布解密密钥并声明与欧洲地区的攻击无关，但解密密钥仅可解密Petya而不支持NotPetya加密的文件。\[11\]

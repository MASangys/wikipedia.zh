**江刺郡**（）是位在日本[岩手縣的](https://zh.wikipedia.org/wiki/岩手縣 "wikilink")[郡](../Page/郡.md "wikilink")，曾屬於[令制國體系下的](../Page/令制國.md "wikilink")[陸奧國](../Page/陸奧國.md "wikilink")（或[陸中國](https://zh.wikipedia.org/wiki/陸中國 "wikilink")）。

## 郡域

[明治十一年](../Page/明治.md "wikilink")（1878年）發展為行政區劃時的郡域為今天[奧州市的一部分](../Page/奧州市.md "wikilink")（[北上川以東除去](https://zh.wikipedia.org/wiki/北上川 "wikilink")[前澤區生母](https://zh.wikipedia.org/wiki/前澤區 "wikilink")）、[北上市的一部分](../Page/北上市.md "wikilink")（口内町、稻瀨町）。

## 歷史

江刺郡在[幕末時期屬於](../Page/幕末.md "wikilink")[陸奧國](../Page/陸奧國.md "wikilink")，全域在**[仙臺藩](https://zh.wikipedia.org/wiki/仙臺藩 "wikilink")**統治下。《[舊高舊領取調帳](https://zh.wikipedia.org/wiki/舊高舊領取調帳 "wikilink")》中記載於[明治初期便已存在的村如以下所列](../Page/明治.md "wikilink")。（41村）

  -
    次丸村、角掛村、淺井村、横瀨村、伊手村、人首村、野手崎村、菅生村、栗生澤村、輕石村、鴨澤村、一關村、歌書村、枛木田村\[1\]、小池村、水押村、上口内村、下口内村、片岡村、高寺村、田谷村、二子町村、田茂山村、羽黒堂村、鶯澤村、黒田助村、黒石村、大田代村、小田代村、原體村、石山村、土谷村、餅田村、增澤村、二關村、三關村、石關村、上門岡村、下門岡村、三照村、倉澤村

### 町村制施行以前的沿革

  - [明治元年九月廿四日](../Page/明治.md "wikilink")（1868年11月8日）——[仙臺藩主](https://zh.wikipedia.org/wiki/仙臺藩 "wikilink")[伊達慶邦向薩長軍投降](https://zh.wikipedia.org/wiki/伊達慶邦 "wikilink")。全領土62萬石遭到沒收。
  - 明治元年十二月初七日（1869年1月19日）
      - 隨著陸奧國的分割，江刺郡從屬於**[陸中國](https://zh.wikipedia.org/wiki/陸中國 "wikilink")**。
      - 成為[信濃](https://zh.wikipedia.org/wiki/信濃國 "wikilink")[松本藩管理地](../Page/松本藩.md "wikilink")，稱作[花卷縣](https://zh.wikipedia.org/wiki/花卷縣 "wikilink")\[2\]。
  - 明治二年八月十八日（1869年9月23日）：花卷縣編入**[江刺縣](../Page/江刺縣.md "wikilink")**。
  - 明治四年十一月初二日（1871年12月13日）：隨著第1次府縣整合，納入**[一關縣](https://zh.wikipedia.org/wiki/一關縣 "wikilink")**的管轄。
  - 明治四年十二月十三日（1872年1月22日）：一關縣（第2次）改稱為**[水澤縣](https://zh.wikipedia.org/wiki/水澤縣 "wikilink")**。
  - 明治八年（1875年）10月17日：隨著成為水澤縣，底下的村也進行了整合。（15村）
      - 片岡村 ← 片岡村、増澤村、餅田村
      - 梁川村 ← 野手崎村、菅生村、栗生澤村
      - 廣瀨村 ← 一關村、輕石村、鴨澤村、歌書村
      - 稻瀨村 ← 上門岡村、下門岡村、石關村、二關村、三關村
      - 福岡村 ← 上口内村、下口内村、小池村、水押村、枛木田村\[3\]
      - 愛宕村 ← 高寺村、田谷村、二子町村
      - 羽田村 ← 羽黒堂村、田茂山村、黒田助村、鶯澤村
      - 田代村 ← 大田代村、小田代村
      - 石原村 ← 石山村、原體村、土谷村
      - 照澤村 ← 三照村、倉澤村
      - 玉里村 ← 次丸村、角掛村
      - 藤里村 ← 淺井村、横瀨村
      - 米里村 ← 人首村
  - 明治九年（1876年）4月18日：隨著第2次府縣整合，納入**[岩手縣](https://zh.wikipedia.org/wiki/岩手縣 "wikilink")**的管轄。
  - 明治十一年（1879年）11月26日：隨著[郡區町村編制法的施行](https://zh.wikipedia.org/wiki/郡區町村編制法 "wikilink")，與[膽澤郡一同在膽澤](https://zh.wikipedia.org/wiki/膽澤郡 "wikilink")、江刺郡役所的管轄下（郡役所設置於膽澤郡鹽竈村）。

### 町村制施行以後的沿革

[Iwate_Esashi-gun_1889.png](https://zh.wikipedia.org/wiki/File:Iwate_Esashi-gun_1889.png "fig:Iwate_Esashi-gun_1889.png")

  - 明治二十二年（1889年）4月1日：隨著[町村制的施行](../Page/町村制.md "wikilink")，發展為下列的町村。（1町12村）除了特別註記以外，現為[奧州市境內](../Page/奧州市.md "wikilink")
      - **[岩谷堂町](../Page/岩谷堂町.md "wikilink")** ← 片岡村
      - **[愛宕村](../Page/愛宕村_\(岩手縣\).md "wikilink")** ← 愛宕村
      - **[羽田村](https://zh.wikipedia.org/wiki/羽田村 "wikilink")** ← 羽田村
      - **[黒石村](https://zh.wikipedia.org/wiki/黒石村 "wikilink")** ← 黒石村
      - **[田原村](../Page/田原村_\(岩手縣\).md "wikilink")** ← 田代村、石原村
      - **[藤里村](../Page/藤里村.md "wikilink")** ← 藤里村
      - **[伊手村](../Page/伊手村.md "wikilink")** ← 伊手村
      - **[米里村](../Page/米里村_\(岩手縣\).md "wikilink")** ← 米里村
      - **[玉里村](../Page/玉里村_\(岩手縣\).md "wikilink")** ← 玉里村
      - **[梁川村](../Page/梁川村_\(岩手縣\).md "wikilink")** ← 梁川村
      - **[福岡村](https://zh.wikipedia.org/wiki/福岡村_\(岩手縣\) "wikilink")**
        ← 福岡村（現：北上市）
      - **[廣瀨村](../Page/廣瀨村_\(岩手縣\).md "wikilink")** ←
        廣瀨村、稻瀨村的一部分［二關、三關］
      - **[稻瀨村](../Page/稻瀨村.md "wikilink")** ← 照澤村、稻瀨村的一部分［上門岡、下門岡、石關］
  - 明治三十年（1897年）4月1日：[郡制施行](https://zh.wikipedia.org/wiki/郡制 "wikilink")。隨著膽澤、江刺郡役所的廢止，於岩谷堂町設置了江刺郡役所。
  - [大正十二年](../Page/大正.md "wikilink")（1923年）4月1日：隨著郡制廢止也廢止了郡會。而為了處理殘留的業務所以郡役所仍留存。
  - 大正十五年（1926年）7月1日：郡役所廢止。
  - [昭和廿九年](../Page/昭和.md "wikilink")（1954年）4月1日（1町9村）
      - 黒石村、羽田村與膽澤郡[水澤町](https://zh.wikipedia.org/wiki/水澤町 "wikilink")、[姉體村](https://zh.wikipedia.org/wiki/姉體村 "wikilink")、[真城村](https://zh.wikipedia.org/wiki/真城村 "wikilink")、[佐倉河村合併](https://zh.wikipedia.org/wiki/佐倉河村 "wikilink")，成為**[水澤市](https://zh.wikipedia.org/wiki/水澤市 "wikilink")**（今奧州市）。
      - 福岡村與[和賀郡](https://zh.wikipedia.org/wiki/和賀郡 "wikilink")[黒澤尻町](https://zh.wikipedia.org/wiki/黒澤尻町 "wikilink")、[飯豐村](https://zh.wikipedia.org/wiki/飯豐村_\(岩手縣\) "wikilink")、[鬼柳村](https://zh.wikipedia.org/wiki/鬼柳村 "wikilink")、[更木村](https://zh.wikipedia.org/wiki/更木村_\(岩手縣\) "wikilink")、[二子村](https://zh.wikipedia.org/wiki/二子村_\(岩手縣\) "wikilink")、膽澤郡[相去村合併成](https://zh.wikipedia.org/wiki/相去村 "wikilink")**[北上市](../Page/北上市.md "wikilink")**。
  - 昭和三十年（1955年）
      - 2月10日：岩谷堂町、伊手村、稻瀨村、愛宕村、玉里村、田原村、廣瀨村、藤里村、梁川村、米里村合併為**[江刺町](../Page/江刺市.md "wikilink")**。（1町）
      - 7月10日：江刺町的一部分（稻瀨裡面舊下門岡村12字、舊上門岡村3字、227世帶、1425人、666町餘）編入北上市。
  - 昭和卅三年（1958年）11月3日：江刺町施行了市制，改制為**[江刺市](../Page/江刺市.md "wikilink")**（今奧州市）。不再隸屬於郡。同日江刺郡撤銷。

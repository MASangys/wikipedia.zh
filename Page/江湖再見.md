## 劇情簡介

周勝龍 ( [劉松仁](../Page/劉松仁.md "wikilink")飾 ) 與刑俊男 ( [李立群](../Page/李立群.md "wikilink")飾 ) 原本在高中時代曾是死黨，勝龍當時也因此和俊男的妹妹刑美麗 ( [湯蘭花](../Page/湯蘭花.md "wikilink")飾 ) 交往；後遭俊男當時的女友朱燕妮 ( [柯素雲](../Page/柯素雲.md "wikilink")飾 ) 挑撥，使得勝龍和俊男發生流血衝突，勝龍因而被判刑入獄，從此兩人反目成仇；孰料美麗與勝龍已未婚懷孕，但勝龍並不知情，在勝龍服刑期間生下了刑過 ( [謝祖武](../Page/謝祖武.md "wikilink")飾 )，刑家為了掩人耳目，假裝把刑過當成刑父正輝 ( [常楓](../Page/常楓.md "wikilink")飾 )、刑母張阿萍 ( [盧碧雲](../Page/盧碧雲.md "wikilink")飾 ) 的第三個孩子，而生母美麗自然成了刑過的姊姊，俊男成了他的哥哥，也計畫不讓刑過知道真相；之後美麗旅居[法國發展](https://zh.wikipedia.org/wiki/法國 "wikilink")[服裝設計事業](https://zh.wikipedia.org/wiki/服裝設計 "wikilink")，藉此逃避在台灣的情傷；而身為外婆的張阿萍，始終看刑過不順眼，把他當成刑家的恥辱 ( 因此取名為「刑過」)，不把他當親孫看，就這樣過了 18 年。

常青 ( [關詠荷](../Page/關詠荷.md "wikilink")飾 ) 與母親林寶寶 ( [馬之秦](../Page/馬之秦.md "wikilink")飾 ) 和父母離異正讀高中的姪女常可喜 ( [張庭](../Page/張庭.md "wikilink")飾 ) 一家人搬到某棟公寓來住，恰巧住在刑家的樓下。在搬家期間，因雞毛蒜皮的小事與刑母張阿萍交惡，刑母從此以後對常家沒好臉色，甚至把林寶寶冠上「老妖精」的封號；但俊男與常青一見鍾情，刑過也與可喜有了曖昧情結，他們大小兩對，經常在張阿萍的極力反對下悄悄的往來；而林寶寶很明理，不喜歡與人交惡，也知道刑家是好人家，並不阻止他們交往。

在當年勝龍與俊男發生衝突時把俊男打傷，以傷害罪被判刑入獄，之後便展開了他的[江湖](../Page/江湖.md "wikilink")生涯。但勝龍本質不壞，是個重情義的江湖人，在[黑道上很受敬重](https://zh.wikipedia.org/wiki/黑道 "wikilink")，18 年來曾入獄五次，這次出獄之後立誓退出江湖，可是自己沒有一技之長，於是又開了地下賭場維生，遊走在黑道邊緣，但他持守心中的那把尺，有所為，有所不為，絕不做傷天害理的事；後又發現前女友阿秀被黑道角頭猴子 ( [顧寶明](../Page/顧寶明.md "wikilink")飾 ) 搶去做了他的女人，勝龍去教訓了猴子一頓，猴子不敢得罪他，為了息事寧人，把跟他走很近的演藝界明星鳳凰 ( [張媛婷飾](https://zh.wikipedia.org/wiki/張媛婷 "wikilink") ) 補償給他，之後勝龍與鳳凰在沒有真正感情的狀態下[同居](https://zh.wikipedia.org/wiki/同居 "wikilink")。勝龍始終念念不忘他的初戀情人刑美麗，偶然間遇到[大學聯考失利的少年刑過](https://zh.wikipedia.org/wiki/大學聯考 "wikilink")，實際上是他的親生兒子，但兩人完全不知情，並且得知刑過是刑美麗的弟弟 ( 實際上是兒子 )，於是積極向刑過打聽初戀情人的下落，也把刑過當成自己的弟弟一樣照顧，還出錢給他[補習重考大學](https://zh.wikipedia.org/wiki/補習 "wikilink")，兩人在性情上感到莫名的契合，成了相差二十歲的忘年之交。

俊男繼承父志，警專畢業做了警察，在市刑大服務。得知周勝龍復出了，又跟刑過往來密切，於是三番兩次以警察的身份去警告周勝龍不許糾纏刑過，後查抄了他的賭場，兩人又展開了誓不兩立的對峙，刑過夾在中間非常煩惱。

刑過在外婆的歧視下長大，性情人格也因此扭曲，變得叛逆固執，一心想進入黑道賺取大筆的錢，以證明自己有社會地位；俊男屢勸刑過不聽，終於被猴子以身家性命威脅利用，陷在幫派裡無法自拔，俊男深深為此所苦，於是緊急喚回旅居法國的美麗回來......

常青立志走[演藝圈](https://zh.wikipedia.org/wiki/演藝圈 "wikilink")，卻始終不得其門而入，她與周勝龍曾在街頭因誤會而相識。在周勝龍賭場被查抄之後，偶然遇到老友崔文正 ( [趙舜](../Page/趙舜.md "wikilink")飾 )，也是金嗓子唱片公司老闆，於是把常青介紹給他，原本不看好常青的崔文正，在勝龍信誓旦旦自願擔任常青唱片專輯的[監製之下](https://zh.wikipedia.org/wiki/監製 "wikilink")，讓常青出第一張唱片就奪得排行榜冠軍，常青於是在演藝界嶄露頭角，漸漸走紅，成了[明日之星](https://zh.wikipedia.org/wiki/明日之星 "wikilink")；之後，身為[公眾人物](../Page/公眾人物.md "wikilink")的常青，與身為警察的男友俊男在生活上步調不一，因此感情也起了變化......

周勝龍積極想找出當年挑撥離間的朱燕妮，以澄清自己的清白，化解周刑二人的仇恨......

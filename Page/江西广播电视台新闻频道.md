**江西广播电视台新闻频道**是[江西广播电视台](../Page/江西广播电视台.md "wikilink")下属一个省级电视频道，于2018年8月16日正式开播\[1\]，前身是红色经典频道，于2008年1月1日开播\[2\]。江西新闻频道以播出的新闻类和资讯类节目，为播出的自办节目和[江西卫视](../Page/江西卫视.md "wikilink")的新闻节目。

## 主要节目

| 类型                                          | 节目名                                                |
| ------------------------------------------- | -------------------------------------------------- |
| 自办节目                                        | 整点新闻，新闻大搜索，问政江西，政策面对面，深度观察，新时代·江西人，创智空间，新闻发布厅，直通现场 |
| 重播及首播[江西卫视](../Page/江西卫视.md "wikilink")新闻节目 | 晨光新视界，杂志天下，社会传真，江西新闻联播，新闻夜航                        |
|                                             |                                                    |

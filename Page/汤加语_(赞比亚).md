這種**東加語**（[英語](https://zh.wikipedia.org/wiki/英語 "wikilink")：Tonga）是一種主要由贊比亞和津巴布韋的東加人講的[班圖語](https://zh.wikipedia.org/wiki/班圖語 "wikilink")。說這種語言的人主要生活在[贊比亞南部和西部省份](https://zh.wikipedia.org/wiki/贊比亞 "wikilink")、[津巴布韋北部和](https://zh.wikipedia.org/wiki/津巴布韋 "wikilink")[莫桑比克](../Page/莫桑比克.md "wikilink")的少數地區。除了那些會雙語的贊比亞和津巴布韋人，Ila人、Iwe人、[托卡人和](https://zh.wikipedia.org/wiki/托卡人 "wikilink")[萊亞人也會說這種語言](https://zh.wikipedia.org/wiki/萊亞人 "wikilink")。 和[本巴語](https://zh.wikipedia.org/wiki/本巴語 "wikilink")、[洛奇語](https://zh.wikipedia.org/wiki/洛奇語 "wikilink")、[尼昂家語一樣](https://zh.wikipedia.org/wiki/尼昂家語 "wikilink")，東加語也是贊比亞[通用語](../Page/通用語.md "wikilink")。這種語言和[馬拉維的東加語並不十分接近](https://zh.wikipedia.org/wiki/東加語_\(馬拉維\) "wikilink")。

說這種語言的人是最古老的班圖移民以及Tumbuka，後者是一個居住在現在的贊比亞。這種東加語有兩種區別明顯的[方言](../Page/方言.md "wikilink")：山谷東加語和高原東加語。前者主要流行於[贊比西河](https://zh.wikipedia.org/wiki/贊比西河 "wikilink")[河谷和Batonga人分佈的一些南部地區](https://zh.wikipedia.org/wiki/河谷 "wikilink")，後者則主要在[蒙澤](https://zh.wikipedia.org/wiki/蒙澤 "wikilink")（Monze）地區Batonga人分佈的北部地區。\[1\]

東加語在歐洲[傳教士到達之前並沒有從口頭語發展成為書面語](https://zh.wikipedia.org/wiki/傳教士 "wikilink")。該語言本身並不標準化，當有必要寫出來的時候，即使同一種方言的使用者也會在同一個單詞的拼寫上出現分歧。\[2\]

## 語言結構

湯加語遵循標准的[班圖語支語言結構](https://zh.wikipedia.org/wiki/班圖語支#語言特徵 "wikilink")，一個單詞可能包含一個動詞、一個直接賓語、一個時態標記甚至一個間接賓語。

| 時態\[3\]      | 時態標記                |
| ------------ | ------------------- |
|              | 主語-（時態標記）-動詞詞根-（結尾） |
| 一般現在時        | \-（動詞詞根）            |
| 現在完成時        | \-a-（動詞詞根）-ide      |
| 現在進行時        | \-la-               |
| 通常的現在時       | \-la-（動詞詞根）-a       |
| 較近的過去（今天之前）  | \-ali--（動詞詞根）ide    |
| 一般過去時        | \-aka-              |
| 最近的過去進行時     | \-ali-ku-（動詞詞根）     |
| 通常的過去進行時     | \-akali-ku-（動詞詞根）   |
| 遙遠的過去        | \-aka-              |
| 較近的未來        | \-la-               |
| 一般將來時        | \-ya-ku-（動詞詞根）-a    |
| 通常的未來        | \-niku-（動詞詞根）-a     |
| 延伸的未來（明天或後天） | \-yaku-(verb)-a     |

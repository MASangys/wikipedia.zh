**法國海軍棋**（Dover Patrol），是[法國陸軍棋](../Page/法國陸軍棋.md "wikilink")作者Harry Gibson於1919年所推出的軍棋類遊戲。

## 規則

  - 兩方將所有己棋豎立、正面朝後的方式秘密布子後，再輪流移動一枚己棋，軍旗只能放在大本營。

<!-- end list -->

  - 一般棋子可縱橫方向移動一格至空格或敵棋處，棋子需維持正面朝後。
      - 棋子可使用特殊移動，但需將棋子公開。
      - 若至敵棋處，將兩棋公開，將勝的棋子改採躺平、正面朝上放回原位，輸的棋子移除遊戲。
          - 若兩子無法互相消滅，則一起放回被攻擊的位置。
          - 若攻擊到軍旗，改將軍旗放在該攻擊棋子上。
          - 若與有軍旗的敵棋發生衝突，並且己方勝利，則將軍旗改放在己方該棋上。

<!-- end list -->

  - 將對方軍旗移回己方大本營或使對方無子可動獲勝。\[1\]\[2\]\[3\]

| 1    | 旗艦    | 1    |                                            |
| 2    | 副旗艦   | 1    |                                            |
| 3    | 戰鬥中隊  | 1    |                                            |
| 4    | 戰列艦   | 2    |                                            |
| 5    | 戰鬥巡洋艦 | 3    |                                            |
| 6    | 輕巡洋艦  | 4    |                                            |
| 7    | 驅除艦   | 4    | 唯一能消滅佈雷艇；唯二能消滅飛行艇。                         |
| 8    | 輔助巡洋艦 | 4    |                                            |
| 9    | 魚雷艇   | 5    | 唯二能消滅潛艇、飛行艇。                               |
| 10   | 巡邏艇   | 5    |                                            |
| MS   | 掃雷艇   | 2    | 唯一能消滅水雷。與掃雷艇、佈雷艇、飛行艇相遇，皆無法消滅。              |
| FB   | 飛行艇   | 1    | 可縱或橫走兩格，不得有子抵擋。與敵相遇時，不能消滅敵方。只會被驅除艦、魚雷艇所消滅。 |
| SS   | 潛艇    | 3    | 只會被魚雷快艇、水雷所消滅。與佈雷艇、飛行艇、潛艇相遇，皆無法消滅。         |
| ML   | 佈雷艇   | 1    | 只能移動至空格，無法攻擊。只會被驅除艦所消滅。                    |
| MI   | 水雷    | 3    | 只會被掃雷艇所消滅。                                 |
| F    | 軍旗    | 1    | 不能移動。                                      |

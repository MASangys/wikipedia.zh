**波蘭總督府**（）是
1939年9月[波蘭戰役之後](../Page/波蘭戰役.md "wikilink")，被[德國軍隊所佔領的](https://zh.wikipedia.org/wiki/德國軍隊 "wikilink")[波蘭領土中](https://zh.wikipedia.org/wiki/波蘭 "wikilink")，未直接與[德國合併的部分](https://zh.wikipedia.org/wiki/德國 "wikilink")\[1\]。1941年6月[巴巴羅薩作戰之後](https://zh.wikipedia.org/wiki/巴巴羅薩作戰 "wikilink")，[希特勒頒布法令](https://zh.wikipedia.org/wiki/希特勒 "wikilink")，將部分[波蘭前東部領土併入波蘭總督府](https://zh.wikipedia.org/wiki/波蘭前東部領土 "wikilink")\[2\]。

因[德國曾在](https://zh.wikipedia.org/wiki/德國 "wikilink")[海牙公約上簽字](https://zh.wikipedia.org/wiki/海牙公約 "wikilink")，因而德軍在波蘭的一系列行為都是違法的\[3\]。波蘭總督府並不是一個[傀儡政權](https://zh.wikipedia.org/wiki/傀儡政權 "wikilink")，其統治者在戰爭期間並沒有任何與波蘭人協作的計劃。納粹政權甚至避免在政府公文中出現任何有關波蘭的字眼。這一地區唯一僅存的帶有波蘭色彩的事物是其貨幣[茲羅提](https://zh.wikipedia.org/wiki/茲羅提 "wikilink")，而這也是用於作為宣傳。德國的最終目標是要讓這一地區成為德意志民族居住地區\[4\]。

## 名稱

這一地區的全名在1940年7月之前是*Generalgouvernement für die besetzten polnischen
Gebiete*，通常譯為波蘭被佔領區總督府。1940年7月31日，其名稱被其總督[漢斯·法朗克簡化為](https://zh.wikipedia.org/wiki/漢斯·法朗克 "wikilink")*Generalgouvernement*\[5\]。也有人俗稱其為
***Restpolen*** （意為剩下的波蘭）。

## 行政區劃

[Generalgouvernement_fur_die_besetzten_polnischen_gebiete.png](https://zh.wikipedia.org/wiki/File:Generalgouvernement_fur_die_besetzten_polnischen_gebiete.png "fig:Generalgouvernement_fur_die_besetzten_polnischen_gebiete.png")
波蘭總督府分被劃分為四個大區（*Distrikten*）。分別是[華沙大區](https://zh.wikipedia.org/wiki/華沙 "wikilink")、[盧布林大區](https://zh.wikipedia.org/wiki/盧布林 "wikilink")、[拉多姆大區和](../Page/拉多姆.md "wikilink")[克拉科夫大區](../Page/克拉科夫.md "wikilink")。1941年6月德國入侵蘇聯之後，之前屬於[烏克蘭蘇維埃社會主義共和國的東](https://zh.wikipedia.org/wiki/烏克蘭蘇維埃社會主義共和國 "wikilink")[加利西亞也被劃入波蘭總督府](https://zh.wikipedia.org/wiki/加利西亞 "wikilink")，成為其第五個區加利西亞大區。

五個區下設有*Stadtkreise*（設於都會）和*Kreishauptmannschaften*
（設於農村）。1941年9月15日發布了一項法令，幾個主要都市的名稱被改回其歷史上的德語名稱或另起德語名稱。但其過去的名稱仍然有效。

波蘭總督府的財政長官[Lutz von
Krosigk曾因財政問題而提議將五個大區合併為三個大區](https://zh.wikipedia.org/wiki/Lutz_von_Krosigk "wikilink")。\[6\]他曾在1943年3月宣布加利西亞大區和克拉科夫大區合併。

## 人口

波蘭總督府區域內最初人口約有1200萬人。不過後來約有860,000波蘭人和猶太人被驅逐出這一區域。加上疾病和飢荒的影響，這一地區從1941年開始出現人口減少。

| 民族   | 平均每日獲得熱量 |
| ---- | -------- |
| 德國人  | 2310     |
| 外國人  | 1790     |
| 烏克蘭人 | 930      |
| 波蘭人  | 654      |
| 猶太人  | 184      |

1941年12月的一次食物配給中不同民族所能獲得的熱量\[7\]

有約100萬波蘭人被驅逐到德國從事強制勞動，其中有許多就死在德國。在1940年，德國在這一地區實施人種劃分，每個種族獲得的權利並不相同，由高到低是：

1.  來自德國的德國人（*Reichdeutsche*）；
2.  來自德國以外的德國人；
3.  德國以外的德國人、與德國人結婚的人及其家庭；
4.  烏克蘭人；
5.  高地人；
6.  波蘭人；
7.  吉普賽人；
8.  猶太人。

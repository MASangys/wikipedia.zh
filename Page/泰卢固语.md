**泰卢固语**（）是[印度](../Page/印度.md "wikilink")[安得拉邦](../Page/安得拉邦.md "wikilink")的[官方语言](https://zh.wikipedia.org/wiki/印度国家语言列表 "wikilink")，属于[达罗毗荼语系](../Page/达罗毗荼语系.md "wikilink")，是印度六大傳統語言之一。泰卢固语是印度安得拉邦[泰卢固人的语言](https://zh.wikipedia.org/wiki/泰卢固人 "wikilink")，印度宪法承认的语言之一。属达罗毗荼语系中部语族。在印度的使用泰盧固語為母語的人數排名第三（7400萬为母语，总数8千万，2001年人口普查），居同语系各语言之首。

## 音系

### 元音

泰卢固语存在 [元音和谐](https://zh.wikipedia.org/wiki/元音和谐 "wikilink")\[1\]

| 元音– అచ్చులు *acchulu* |
| --------------------- |
| ఇ i                   |
| ఎ e                   |
|                       |

只出现在外来语中。 泰卢固语有两个双元音 ఐ \[ai\] and ఔ \[au\].

## 语法

泰卢固语语法被称作 *vyākaranam* (వ్యాకరణం).

| 句子                             | రాము బడికి వెళ్తాడు. |
| ------------------------------ | -------------------- |
| 单词                             | రాము                 |
| [转写](../Page/转写.md "wikilink") | rāmu                 |
| 意思                             | Ramu（人名）             |
| 语法作用                           | 主语                   |
| 翻译                             | Ramu去学校。             |

这句话也可以根据上下文翻译成 'Ramu（要）去学校' ，但这不影响主谓宾顺序。

### 屈折变化

泰卢固语名词有数（单数、复数）、性（阳性、阴性、中性）和格 （主格、对格、造格、与格、夺格和属格）的变化。\[2\]

## 书写系统

[Telugu.svg](https://zh.wikipedia.org/wiki/File:Telugu.svg "fig:Telugu.svg") 与其他印度文字一样，泰卢固文从左向右书写。句末使用竖线 । (“purna viramam”) 或者双竖线 ॥ (“deergha viramam”)。传统的手写里是不分词的。现代标点（逗号，分号等）是印刷术出现之后才引入的。\[3\] 泰卢固文在 [Unicode](https://zh.wikipedia.org/wiki/Unicode "wikilink") 的码位区间是 0C00-0C7F (3072–3199)。\[4\]

### 数字系统

泰卢固文有自己的一套 [数字如下](https://zh.wikipedia.org/wiki/数字 "wikilink")，但不常用。

### 字母

泰卢固文字母表包括60个符号，其中有 16 个元音，3 个元音指示和 41 个辅音。梵文和泰卢固文非常类似且可以一一对应。 \[5\]它们中的一些在发音上只有很少的区别。\[6\]

泰卢固文有一个圆圈 ([随韵](https://zh.wikipedia.org/wiki/随韵 "wikilink")) ( ం ), 半圆 ( [仰月点](https://zh.wikipedia.org/wiki/仰月点 "wikilink")) (ఁ) 和双点（[止韵](https://zh.wikipedia.org/wiki/止韵 "wikilink")） ( ః ) 等附加符号，区分 la 和 La, ra 和 Ra.\[7\]。泰卢固文中还有梵文里所没有的 .CH 和 .JH；而 S, SH, 和 KSH 也是泰米尔文中所没有的。\[8\]

**洛伦茨吸引子**是**洛伦茨振子**（Lorenz
oscillator）的长期行为对应的[分形结构](../Page/分形.md "wikilink")，以[爱德华·诺顿·洛伦茨的姓氏命名](../Page/爱德华·诺顿·洛伦茨.md "wikilink")。洛伦茨振子是能产生[混沌流的三维动力系统](../Page/混沌理论.md "wikilink")，是一種[吸引子](../Page/吸引子.md "wikilink")，以其双纽线形状而著称。映射展示出[动力系统](../Page/动力系统.md "wikilink")（三维-{}-系统的三个变量）的状态是如何以一种复杂且不重复的模式，随时间的推移而演变的。

## 简述

[Lorenzstill-rubel.png](https://zh.wikipedia.org/wiki/File:Lorenzstill-rubel.png "fig:Lorenzstill-rubel.png")结构\]\]
洛伦茨吸引子及其导出的方程组是由[爱德华·诺顿·洛伦茨於](../Page/爱德华·诺顿·洛伦茨.md "wikilink")1963年发表，最初是发表在《大气科学杂志》（*Journal
of the Atmospheric Sciences*）杂志的论文《*Deterministic Nonperiodic
Flow*》中提出的，是由[大气方程中出现的](../Page/地球大气层.md "wikilink")[对流卷方程简化得到的](../Page/对流.md "wikilink")。

这一洛伦茨模型不只对非线性数学有重要性，对於气候和天气预报来说也有着重要的含义。行星和恒星大气可能会表现出多种不同的准周期状态，这些准周期状态虽然是完全确定的，但却容易发生突变，看起来似乎是随机变化的，而模型对此现象有明确的表述。

从技术角度看来，洛伦茨振子具有[非線性](../Page/非線性.md "wikilink")、三维性和[确定性](../Page/确定性系统.md "wikilink")。2001年，沃里克·塔克尔（Warwick
Tucker）证明出在一组确定的参数下，系统会表现出混沌行为，显示出人们今天所知的[奇异吸引子](../Page/奇异吸引子.md "wikilink")。这样的奇异吸引子是[豪斯多夫维数在](../Page/豪斯多夫维数.md "wikilink")2与3之间的[分形](../Page/分形.md "wikilink")。[彼得·格拉斯伯格](../Page/彼得·格拉斯伯格.md "wikilink")（Peter
Grassberger）已於1983年估算出豪斯多夫维数为2.06 ± 0.01，而为2.05 ± 0.01。

此系统也会出现在单模[激光](../Page/激光.md "wikilink")\[1\]和[发电机](../Page/发电机.md "wikilink")\[2\]的简化模型中。除此之外，闭环对流、水轮转动等物理模型也有此系统的应用。

## 洛伦茨方程

[Lorenz_attractor_boxed.svg](https://zh.wikipedia.org/wiki/File:Lorenz_attractor_boxed.svg "fig:Lorenz_attractor_boxed.svg")
洛伦茨方程是基於[纳维－斯托克斯方程](../Page/纳维－斯托克斯方程.md "wikilink")、[热传导方程和](../Page/热传导方程.md "wikilink")[连续性方程简化得出](../Page/连续性方程.md "wikilink")，最初的形式为：

  -
    <math>

\\frac { \\partial \\vec v }{\\partial t} + \\left( \\vec v \\nabla
\\right) \\vec v = -\\frac {\\nabla p}{\\rho} + \\nu \\nabla ^2 \\vec v
+ \\vec g </math>

  -
    <math>

\\frac { \\partial \\rho }{\\partial t} + \\nabla \\left( \\rho \\vec v
\\right) = 0 </math>

  -
    <math>

\\frac { \\partial T }{\\partial t} + \\nabla \\left( T \\vec v \\right)
= \\chi \\nabla ^2 T </math>

  -
    <math>

\\rho = \\rho_0 \\left( 1 - \\gamma \\left( T - T_0 \\right) \\right)
</math>

\(\vec v\)——流速，\(T\)——流体温度，\(T_0\)——上限温度（也可以写成\(T_0 + \Delta T\)），\(\rho\)——密度，\(p\)——压强，\(\vec g\)——重力，\(\gamma\)、\(\chi\)、\(\nu\)——依次为[热膨胀系数](../Page/热膨胀系数.md "wikilink")、[热扩散率和动](../Page/热扩散率.md "wikilink")[黏滯係數](../Page/黏滯係數.md "wikilink")。

简化後的形式称为**洛伦茨方程**，是决定洛伦茨振子状态的方程为一组[常微分方程](../Page/常微分方程.md "wikilink")：

  -
    \(\frac{dx}{dt} = \sigma (y - x)\)

<!-- end list -->

  -
    \(\frac{dy}{dt} = x (\rho - z) - y\)

<!-- end list -->

  -
    \(\frac{dz}{dt} = xy - \beta z\)

含时间参数的形式：

\[\begin{cases}
\frac{\mathrm{d}x(t)}{\mathrm{d}t}=\sigma \bigl( y(t) - x(t) \bigr)\\
\frac{\mathrm{d}y(t)}{\mathrm{d}t}=\rho \, x(t) - y(t) - x(t) \, z(t)\\
\frac{\mathrm{d}z(t)}{\mathrm{d}t} =x(t) \, y(t) - \beta \, z(t) \end{cases}\]

\(\sigma\)称为**[普兰特尔数](../Page/普兰特尔数.md "wikilink")**
，\(\rho\)称为**[瑞利数](../Page/瑞利数.md "wikilink")**。所有的\(\sigma\)，\(\rho\)，\(\beta\)
\> 0，但通常\(\sigma\) = 10，\(\beta\) =
8/3，\(\rho\)不定。若\(\rho<1\)，则吸引子为原点，没有任何其他稳定点。1≤ρ\<13.927时，螺线轨迹接近两点（这相当於存在阻尼振子），两点的位置由下列式子决定：\(~ x = \pm \sqrt{b (\rho-1) }\)、\(~ y = \pm \sqrt{b (\rho-1) }\)、\(~ z = \rho-1\)。系统在\(\rho\)
=
28时表现出混沌特性，但\(\rho\)为其他值时会显示出具纽结的周期轨道。例如，当\(\rho = 99.96\)时，图像变为一个*T*(3,2)[环面纽结](../Page/环面纽结.md "wikilink")。

## 源代码

### GNU Octave

下面是[GNU Octave模拟洛伦茨吸引子的源代码](../Page/GNU_Octave.md "wikilink")：

    ## Lorenz Attractor equations solved by ODE Solve
    ## x' = sigma*(y-x)
    ## y' = x*(rho - z) - y
    ## z' = x*y - beta*z
    function dx = lorenzatt(X)
        rho = 28; sigma = 10; beta = 8/3;
        dx = zeros(3,1);
        dx(1) = sigma*(X(2) - X(1));
        dx(2) = X(1)*(rho - X(3)) - X(2);
        dx(3) = X(1)*X(2) - beta*X(3);
        return
    end

    ## Using LSODE to solve the ODE system.
    clear all
    close all
    lsode_options("absolute tolerance",1e-3)
    lsode_options("relative tolerance",1e-4)
    t = linspace(0,25,1e3); X0 = [0,1,1.05];
    [X,T,MSG]=lsode(@lorenzatt,X0,t);
    T
    MSG
    plot3(X(:,1),X(:,2),X(:,3))
    view(45,45)

### Borland C

``` c
#include <graphics.h>
#include <conio.h>
void main()
{
    double x = 3.051522, y = 1.582542, z = 15.62388, x1, y1, z1;
    double dt = 0.0001;
    int a = 5, b = 15, c = 1;
    int gd=DETECT, gm;
    initgraph(&gd, &gm, "C:\\BORLANDC\\BGI");
    do {
    x1 = x + a*(-x+y)*dt;
    y1 = y + (b*x-y-z*x)*dt;
    z1 = z + (-c*z+x*y)*dt;
    x = x1; y = y1; z = z1;
    putpixel((int)(19.3*(y - x*0.292893) + 320),
         (int)(-11*(z + x*0.292893) + 392), 9);
    } while (!kbhit());
    closegraph();
}
```

### Fortran

``` fortran
program LorenzSystem

real,parameter::sigma=10
real,parameter::r=28
real,parameter::b=2.666666
real,parameter::dt=.01
integer,parameter::n=1000

real x,y,z

open(1,file='result.txt',form='formatted',status='replace',action='write')

x=10.;y=10.;z=10.

do i=1,n,1
    x1=x+sigma*(y-x)*dt
    y1=y+(r*x-x*z-y)*dt
    z1=z+(x*y-b*z)*dt
    x=x1
    y=y1
    z=z1
    write(1,*)x,y,z
enddo

print *,'Done'

 close(1)

end program LorenzSystem
```

**海丰话**是[汉藏语系](../Page/汉藏语系.md "wikilink")[汉语族](../Page/汉语族.md "wikilink")[闽语支](https://zh.wikipedia.org/wiki/闽语支 "wikilink")[闽南语的一种方言](https://zh.wikipedia.org/wiki/闽南语 "wikilink")，通行于[中國](https://zh.wikipedia.org/wiki/中華人民共和國 "wikilink")[广东省](../Page/广东省.md "wikilink")[汕尾市](../Page/汕尾市.md "wikilink")[海丰县](../Page/海丰县.md "wikilink")中部、东南部和沿海地区，包括[海城](https://zh.wikipedia.org/wiki/海城鎮_\(海豐縣\) "wikilink")、[鲘门](https://zh.wikipedia.org/wiki/鲘门鎮 "wikilink")、[赤坑](https://zh.wikipedia.org/wiki/赤坑鎮_\(海豐縣\) "wikilink")、[大湖](https://zh.wikipedia.org/wiki/大湖鎮_\(海豐縣\) "wikilink")、[汕尾](../Page/汕尾市.md "wikilink")、[马宫等地](https://zh.wikipedia.org/wiki/马宫街道 "wikilink")。属闽南语的[泉漳片](../Page/泉漳片.md "wikilink")。海丰话使用者人数约占海丰县全境总人口的80%，当地人称之为**學佬話**、**福佬話**或**鶴佬話**。

海丰话以[海丰县](../Page/海丰县.md "wikilink")的[海城鎮口音为代表口音](https://zh.wikipedia.org/wiki/海城鎮 "wikilink")，与[陆丰话](../Page/陆丰话.md "wikilink")存在较高地相似性，在[广东省](../Page/广东省.md "wikilink")境内，这两种方言往往被合称为**海陆丰话**。

廣東省惠州市[惠東縣及](https://zh.wikipedia.org/wiki/惠東縣 "wikilink")[汕尾市](../Page/汕尾市.md "wikilink")（[海陸豐](../Page/海陸豐.md "wikilink")）都有早期的漳泉移民，他們使用的[閩南語發音為極偏漳州音的](https://zh.wikipedia.org/wiki/閩南語 "wikilink")[漳泉濫](https://zh.wikipedia.org/wiki/漳泉濫 "wikilink")，讀音及用字仍與[漳州地區有着極高的相似度](../Page/漳州市.md "wikilink")，能夠順利溝通。

[臺湾客家語有](https://zh.wikipedia.org/wiki/臺湾客家語 "wikilink")「海陸腔」，其代代相傳的歷史為來自海豐、陸豐地區的客家人及客家話，而不是海陆丰閩南話。

## 方言片区

各地使用的海丰话大同小异，若从语音差别为标准，海丰话又分为以下几个方言片区：

  - 海城片：以城关话（[海城话](https://zh.wikipedia.org/wiki/海城鎮_\(海豐縣\) "wikilink")）为代表，通行-{zh-hans:于;zh-hant:於}-[海丰县](../Page/海丰县.md "wikilink")中心的[城东](https://zh.wikipedia.org/wiki/城東鎮_\(海豐縣\) "wikilink")、[梅陇](https://zh.wikipedia.org/wiki/梅隴鎮_\(海豐縣\) "wikilink")、[联安](https://zh.wikipedia.org/wiki/联安鎮 "wikilink")、[陶河](https://zh.wikipedia.org/wiki/陶河鎮 "wikilink")，以及县城以西的[鮜门](https://zh.wikipedia.org/wiki/鮜门鎮 "wikilink")、[赤石等地](https://zh.wikipedia.org/wiki/赤石鎮 "wikilink")、县城东南的[可塘](https://zh.wikipedia.org/wiki/可塘鎮 "wikilink")、[赤坑等地](https://zh.wikipedia.org/wiki/赤坑鎮_\(海豐縣\) "wikilink")。使用者最多、影响力最大。
  - 汕尾片：以[汕尾话为代表](https://zh.wikipedia.org/wiki/汕尾话 "wikilink")，通行于[汕尾沿海的](../Page/汕尾市.md "wikilink")[马宫](https://zh.wikipedia.org/wiki/馬宮街道 "wikilink")、[田墘](https://zh.wikipedia.org/wiki/田墘街道 "wikilink")、[捷胜](https://zh.wikipedia.org/wiki/捷胜鎮 "wikilink")、[东涌和](https://zh.wikipedia.org/wiki/東涌鎮_\(汕尾市\) "wikilink")[大湖](https://zh.wikipedia.org/wiki/大湖鎮_\(海豐縣\) "wikilink")、[红草的大部分地方](https://zh.wikipedia.org/wiki/红草鎮 "wikilink")。
  - 公平片：通行于[海丰县](../Page/海丰县.md "wikilink")[公平镇内及附近乡村的](https://zh.wikipedia.org/wiki/公平鎮_\(海豐縣\) "wikilink")[福佬话地区](https://zh.wikipedia.org/wiki/福佬话 "wikilink")。

## 连读变调

连读变调，即两个或两个以上音节连读的时候，前字的声调受后字声调的影响而发生变化。

海丰话连读变调有两种情况：一种是前字发生变调，后字一概不变调。另一种是为了強調前字而使前字不变调，但后字读作轻声。

海丰话连读变调时，前字变调的规律如下表：

|          |           |           |           |           |           |           |                  |                  |
| -------- | --------- | --------- | --------- | --------- | --------- | --------- | ---------------- | ---------------- |
| **前字調類** | **陰平** 33 | **陽平** 55 | **陰上** 51 | **陽上** 35 | **陰去** 21 | **陽去** 31 | **陰入** <u>12</u> | **陽入** <u>55</u> |
| **前字變調** | 低平 22     | 陽去 31     | 陰去 21     | 低平 22     | 陽平 55     | 低平 22     | 陽入 <u>55</u>     | 陰入 <u>12</u>     |
| **例詞**   | 刚才        | 雷公        | 古典        | 近来        | 对立        | 树木        | 急忙               | 合作               |
|          |           |           |           |           |           |           |                  |                  |

连读时产生了一个新的调值22，被称为低平，往往被表记为第九调。

## 词汇

### 文白异读

海陆丰话有白读音及文读音二套完全不同语音系统，几乎每个文字均有白读音及文读音，而且其中的差别非常大，如：我（文读ngo2，白读wa2)，香（文读hiang1，白读hiũ1及phang1（音韵上说应为“芳”字），臭（文读ngiu3，白读cao3）、数（文读sou3，白读xiao3）、东（文读dong1，白读dang1）等等。

### 保留大量古汉语成分

海陆丰话保留有较多古汉底层成分，包括语音、语法和词汇。古汉语中不同韵但在普通话中同音的在海陆丰话中也有明显区别，如“冬”“东”在古汉语中不同韵，但在海丰话的文读中则不同。海陆丰话在日常用语中保留了大量的古汉语词汇，如“擤”鼻、“刓”（“挖”的意思）、扁食（馄饨，[臺灣河洛語也這樣說](../Page/臺灣話.md "wikilink")，中原官话部分地区亦保留此用法）、书房（音“珠房”，意指学校）、<茨>非厝（意指“家”， 臺灣河洛語也這樣說）等等

### 倒装词汇

海陆丰话有大量的倒装词汇，举例如下： 风台（台风）、鞋拖（拖鞋）、闹热（热闹）

## 海陆丰话与[潮州话](../Page/潮州话.md "wikilink")的异同

海陸豐雖與潮汕地區接壤，但海陆丰话与潮州话在语调及用语习惯上有不少的区别。对于海丰话應否列入潮汕话，还有很大争议。

海丰话和[潮州话](../Page/潮州话.md "wikilink")之间无论在语音、词汇还是语法上都大体相近，因此交流没有太大障碍。海陆丰話与潮州話的主要差异表现在语音和韻母上。海丰话保留的韵尾比[潮州话](../Page/潮州话.md "wikilink")较为完整些。\[1\]

惠州市惠东县靠近海丰县的部分村镇，使用海丰[學佬話](https://zh.wikipedia.org/wiki/學佬話 "wikilink")。

[台灣](https://zh.wikipedia.org/wiki/台灣 "wikilink")[國立台中教育大學台灣語文學系教授兼主任](https://zh.wikipedia.org/wiki/國立台中教育大學 "wikilink")，曾研究[臺灣話](../Page/臺灣話.md "wikilink")的[洪惟仁](../Page/洪惟仁.md "wikilink")對海陸豐話有「海陸豐境內的漳州話」的提法（洪惟仁，《台灣方言之旅》，前衛出版社，1994年，[台北](https://zh.wikipedia.org/wiki/台北 "wikilink")）。

海陆丰话、潮州话都和臺灣話、[漳州話](https://zh.wikipedia.org/wiki/漳州話 "wikilink")、[泉州话](../Page/泉州话.md "wikilink")有大量相同和相似的語言狀況，许多字音和词汇高度相同，雙方可以通话。

### 与潮汕音区别举例

  - 潮汕音中的u韵，海陆丰音为i的：如
      - 你（潮汕音“lir/lu”，海陆丰音“li”）；
      - 去（潮汕音“khu”，海陆丰音“khi”）；
      - 豬（潮汕音“teu/tu”，海陆丰音“di”）。
      - 魚（潮汕音“hu”，海陆丰音“hi”）。
      - 鞋（潮汕音“oi”，海陆丰音“ei”）。

<!-- end list -->

  - 潮州音中的oi韵，海丰音为ei（海丰话没有oi韵），如
      - 会（潮汕音“oi”，海陆丰音“ei”）；
      - 买（潮汕音“mboi”，海陆丰音“mbei”）；
      - 细（潮汕音“soi”，海陆丰音“sei”）。
      - 第（潮汕音“doi”，海陆丰音“dei”）。
      - 齊（潮汕音“tsoi”，海陆丰音“tsei”）。

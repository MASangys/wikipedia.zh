**海宝**（**Haibao**），意為「四海之寶」，是[中国2010年上海世界博览会的](https://zh.wikipedia.org/wiki/中国2010年上海世界博览会 "wikilink")[吉祥物](https://zh.wikipedia.org/wiki/吉祥物 "wikilink")。“海宝”是在[巫永坚汉字发展成吉祥物创意的基础上](https://zh.wikipedia.org/wiki/巫永坚 "wikilink")，在[邵隆图聚焦的](https://zh.wikipedia.org/wiki/邵隆图 "wikilink")“人”的核心理念下，由上海姑娘方芳执笔进行造型设计的。2007年12月18日正式对外公布。\[1\]

### 身体特征

海宝身体的不同部位也融入了不同的设计理念：\[3\]

| 部位 | 特征                                                                                          |
| -- | ------------------------------------------------------------------------------------------- |
| 头发 | 类似翻卷的[海浪](../Page/海浪.md "wikilink")，表示海宝来自[大海](https://zh.wikipedia.org/wiki/大海 "wikilink") |
| 脸部 | 卡通化的脸部表现友好和自信                                                                               |
| 眼睛 | 大而圆，表示对未来的期待                                                                                |
| 蓝色 | 象征包容与想象力                                                                                    |
| 身体 | 圆润而可爱                                                                                       |
| 拳头 | 翘起的[拇指表示对来宾的欢迎](https://zh.wikipedia.org/wiki/拇指 "wikilink")                                |
| 大脚 | 作为稳重的支撑，以示办好世博会的信心                                                                          |

### 名称来源

上海世博会吉祥物征集办公室主任刘军曾透露，在确定海宝这一名称之前，整个团队一共取了200多个名字，比如有“海朋”、“海海”等，最后才确定了“海宝”这一名称。\[4\]“海宝”意为“四海之宝”，读起来琅琅上口，也与其身上的色彩相吻合。\[5\]

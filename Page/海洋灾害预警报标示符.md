**海洋灾害预警报标示符**，是根据[国家海洋局](../Page/国家海洋局.md "wikilink")制订的《海洋灾害预警报标示符》发布的海洋气象警报，其目的是为规范海洋气象灾害警报发布工作，增强全民防灾减灾意识，提高海洋气象灾害警报使用效率，有效防御和减轻海洋气象灾害，保护国家和人民生命财产安全\[1\]。

现行的警报标识符于2017年6月30日启用。警报分为[海浪](../Page/海浪.md "wikilink")、[海啸](../Page/海啸.md "wikilink")、[风暴潮](../Page/风暴潮.md "wikilink")、[海冰](../Page/海冰.md "wikilink")、[赤潮](../Page/赤潮.md "wikilink")、[海雾](https://zh.wikipedia.org/wiki/雾 "wikilink")、[海水低温](https://zh.wikipedia.org/wiki/海水低温 "wikilink")、[咸潮等八类](https://zh.wikipedia.org/wiki/咸潮 "wikilink")。并按照灾害警报重要性程度标注颜色，由低到高依次为白色、蓝色、黄色、橙色、红色。级别越高表明灾害越严重，并启动相应的应急响应\[2\]。

## 海啸

海啸警报共分为四级，依据2015年《风暴潮、海浪、海啸和海冰灾害应急预案》发布。

[2011年日本东北地方太平洋近海地震发生后](https://zh.wikipedia.org/wiki/2011年日本东北地方太平洋近海地震 "wikilink")，国家海洋局发布了海啸警报消息，并于当日下午向[浙江省](../Page/浙江省.md "wikilink")沿岸、[台湾省沿岸发出海啸蓝色警报](../Page/台湾省_\(中华人民共和国\).md "wikilink")，这是国家海洋局首次也是唯一一次发布海啸警报。该警报其后于当日17时41分解除\[3\]\[4\]。最终，浙江省沈家门验潮站监测到0.55米的海啸波幅\[5\]。

## 海浪

海浪警报共分为五级，依据《风暴潮、海浪、海啸和海冰灾害应急预案》发布\[6\]。

| 等级 | 信号名称   | 信号标志 | 信号含义                                                                            |
| -- | ------ | ---- | ------------------------------------------------------------------------------- |
|    | 海浪警报消息 |      | 国家海洋局发布海浪警报消息时。                                                                 |
| Ⅳ  | 海浪蓝色警报 |      | 受热带气旋或温带天气系统影响，预计未来24小时受影响近岸海域出现2.5m-3.5m（不含）有效波高。                              |
| Ⅲ  | 海浪黄色警报 |      | 受热带气旋或温带天气系统影响，预计未来24小时受影响近岸海域出现3.5m-4.5m（不含）有效波高，或者近海预报海域出现6.0m-9.0m（不含）有效波高。  |
| Ⅱ  | 海浪橙色警报 |      | 受热带气旋或温带天气系统影响，预计未来24小时受影响近岸海域出现4.5m-6.0m（不含）有效波高，或者近海预报海域出现9.0m-14.0m（不含）有效波高。 |
| Ⅰ  | 海浪红色警报 |      | 受热带气旋或温带天气系统影响，预未来24小时受影响近岸海域出现达到或超过6.0m有效波高，或者近海预报海域出现达到或超过14.0m有效波高。          |

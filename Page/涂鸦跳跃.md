《**塗鴉跳躍**》（）是由Lima Sky開發的一款電子遊戲，支持[Windows
Phone](../Page/Windows_Phone.md "wikilink")、[iOS](../Page/iOS.md "wikilink")、[BlackBerry](../Page/黑莓手機.md "wikilink")、[Android](../Page/Android.md "wikilink")、Java
Mobile（[J2ME](../Page/Java_ME.md "wikilink")）、諾基亞[Symbian和](../Page/Symbian.md "wikilink")[Xbox
360等平台](../Page/Xbox_360.md "wikilink")，\[1\]其作者為[克羅埃西亞人Igor和Marko](../Page/克羅埃西亞人.md "wikilink")
Pusenjak。\[2\]遊戲於2009年4月6日在iOS全球發行，之後於2010年3月2日在Android和Blackberry平台發佈。遊戲在Symbian平台發佈的時間為2010年5月1日，在Windows
Phone 7上則於2011年6月1日發佈\[3\]（2013年8月21日在Windows Phone
8上重新發佈）。\[4\]\[5\]2011年9月1日，遊戲在iPad上發佈。遊戲的玩法靈感最初源於的《PapiJump》，其角色靈感則源於伊丽丝·格繪製的插圖。\[6\]

在《塗鴉跳躍》中，玩家操控一隻名為“The Doodler”的四腳動物在一個永無休止的平台上一直跳高。如果The
Doodler墮下，則遊戲結束。跳得越高，遊戲的分數便越高。The
Doodler可由控制遊戲設備的傾斜度而移動的。在黑莓版本，則是由方向鍵控制。\[7\]\[8\]

《塗鴉跳躍》在App
Store上曾連續4個月日銷量達到25,000份。截至2011年12月，遊戲在[iTunes和](../Page/iTunes.md "wikilink")[Google
Play上銷量已達到](../Page/Google_Play.md "wikilink")1000萬，在全部平台上銷量總和達到1500萬。\[9\]\[10\]遊戲發行後收穫了外界的好評，[GameRankings根據](../Page/GameRankings.md "wikilink")6個評論給出遊戲85.00%的好評率。\[11\]

## 道具

| 道具             | 功能                                         | 分數   |
| -------------- | ------------------------------------------ | ---- |
| 火箭             | 使Doddle快速飛高一段時間。期間不能發射。                    | 3307 |
| 飛行帽            | 跟火箭差不多,但飛行得較慢。期間不能發射。（備註: 在戴飛行帽時，會自動擊退怪獸。) | 1736 |
| 彈弓（彈簧）         | 使Doodle彈高。期間可以發射。                          | 352  |
| 彈床             | 跟彈弓的功能一樣，但可跳得更高。                           | 520  |
| 彈弓鞋（彈簧鞋）       | 跟彈弓的功能一樣，但它會留在Doodle的腳上，並可以跳六下。            | 2068 |
| 保護罩            | 使Doodler碰到怪獸都不會掉下。                         | N/A  |
| 超級火箭（只在太空主題中有） | 功能跟火箭一樣，但會飛得更高更快。                          | 6521 |

**測光系統**是[天文學中以一套已知敏感度的分離通帶](../Page/天文學.md "wikilink")（過濾器），測量事件輻射的裝置，而敏感度通常依所使用的光學系統、探測器和濾鏡有關。每一個測光系統都需要提供[主標準星](../Page/測光標準星.md "wikilink")。

最早規範出來的測光系統是約翰遜摩根或[UBV測光系統](../Page/UBV測光系統.md "wikilink")（1953年），現今已經有200套以上這種的系統。

測光系統通常根據濾鏡的通帶寬度來描述其特性：

  - 寬通帶：通帶寬度超過30奈米，最常見的就是約翰遜摩根的[UBV測光系統](../Page/UBV測光系統.md "wikilink")。
  - 中通帶：通帶寬度介於10至30奈米。
  - 窄通帶：通帶寬度小於10奈米。

目前在各型望遠鏡或機構內使用的濾鏡有下列幾種：\[1\].

| 史隆                 | u' = 354nm             | g' = 475nm                                   |
| CFHTLS (Megacam)   | u<sup>\*</sup> = 374nm | g' = 487nm                                   |
| HDF                | 450nm                  | 606nm                                        |
| GOODS (Hubble ACS) | B = 435nm              | V = 606nm                                    |
| UKIDSS (WFCAM)     | Z = 882nm              | Y = 1031nm                                   |
| 2MASS              | J = 1.25μm             | H = 1.65μm                                   |
| Spitzer IRAC       | 3.6μm                  | 4.5μm                                        |
| Spitzer MIPS       | 24μm                   | 70μm                                         |
| VISTA IRC          | J = 1.25μm             | H = 1.65μm                                   |
| Pan-STARRS         | 使用與史隆相同的濾鏡：g,r,i,z,y   | [全景巡天望遠鏡和快速回應系統](../Page/泛星計畫.md "wikilink") |
| Stromvil           | U = 345                | P = 374                                      |

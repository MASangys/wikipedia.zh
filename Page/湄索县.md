**湄索县**
()，一譯**夜速縣**，是屬於[泰國](../Page/泰國.md "wikilink")[達府的一個縣](../Page/達府.md "wikilink")，位於泰緬邊境隔著[梅河](../Page/梅河.md "wikilink")(Moei
River)與[緬甸相望](../Page/緬甸.md "wikilink")。與美索毗鄰的區域分別是（從北以順時針方向）：Mae Ramat,
Mueang Tak以及Phop Phra。湄公河對岸是緬甸的苗瓦迪鎮(Myawaddy)。美索距離Mueang Tak
86公里，可經由高速公路105抵達，高速公路105也就是[亞洲高速公路AH1](../Page/亞洲高速公路AH1.md "wikilink")。

## 往緬甸的通道

[Thai_myanmar_friendship_bridge.jpg](https://zh.wikipedia.org/wiki/File:Thai_myanmar_friendship_bridge.jpg "fig:Thai_myanmar_friendship_bridge.jpg")
湄索位於穿越泰國與緬甸的亞洲高速公路AH1上，因此這裡是泰國通往緬甸最重要的通道(另一個是[三佛塔關口](../Page/三佛塔關口.md "wikilink"))。'''泰緬友誼大橋
'''(Thai-Myanmar Friendship
Bridge)建立於1997年，橫跨湄公河連結兩個國家。橋頭入口處有移民官署的辦公室，辦公時間為上午6點至下午6點。

作為一個邊關城市，湄索有一個國內機場。但是因為油價高漲的關係，某些航空公司(例如：[普吉航空](../Page/普吉航空.md "wikilink"))已經取消了[曼谷到湄索之間的航班](../Page/曼谷.md "wikilink")。

[達府商會每年都會舉辦自行車越野賽](../Page/達府.md "wikilink")，終點在緬甸的苗瓦迪鎮。

## 經濟

[Rim_Moei.jpg](https://zh.wikipedia.org/wiki/File:Rim_Moei.jpg "fig:Rim_Moei.jpg")
當地最主要的經濟活動是與緬甸的邊關貿易、旅遊以及銷售地方性的手工藝品。那些會經過此地的觀光客通常都是穿過邊境到緬甸，或者北上到[媚宏順省](../Page/媚宏順府.md "wikilink")，又或者是往南到翁龐郡(Amphoe
Umphang)的風景名勝。市中心也有些販售珠寶的商店。**媚公河岸市場**(Rim Moei
Marke)是最著名的市場，位於友誼大橋的泰國境內，主要販售舶來品。新、舊穆塞市場(Mussur
Market)座落於AH1高速公路邊，靠近美索和Mueang
Tak郡的邊界，主要販售農產品。這兩個市場相距一公里，分別在AH1的兩側，舊的穆塞市場位在美索郡，新的市場在Mueang
Tak縣。*Mussur*是拉祜族([Lahu people](../Page/Lahu_people.md "wikilink"))的泰文。

## 緬甸難民

[梅道診所位於美索靠近緬甸邊境的郊外](../Page/梅道診所.md "wikilink")，為緬甸的[難民提供實質的醫療服務](../Page/難民.md "wikilink")。梅道診所對於美索來說扮演相當重要的地下或志工經濟，因為大量的緬甸人因為
[內戰逃離戰火](../Page/緬甸內戰.md "wikilink")、或受到軍政府的迫害、或因為失業，因此選擇逃離家園而成為難民。許多國際志工在這裡與[辛西雅醫生一起為這些緬甸難民提供援助](../Page/辛西雅.md "wikilink")。

## 行政

湄索縣轄下有10個次縣(tambon)，86個村(muban)。10次縣分別為：

| No. | Name     | Thai name |  |  |
| --- | -------- | --------- |  |  |
| 1\. | Mae Sot  | แม่สอด    |  |  |
| 2\. | Mae Ku   | แม่กุ     |  |  |
| 3\. | Phawo    | พะวอ      |  |  |
| 4\. | Mae Tao  | แม่ตาว    |  |  |
| 5\. | Mae Kasa | แม่กาษา   |  |  |

泰國目前打算在湄索建立一個新的省府中心\[1\]。特別是這個新的行政單位將涵蓋與多個次縣，如湄索、Mae Pa 以及Tha Sai
Luat。\[2\]

**湖南大众传媒职业技术学院**（）位于[湖南省](../Page/湖南省.md "wikilink")[长沙市](https://zh.wikipedia.org/wiki/长沙市 "wikilink")，为湖南省的一所[专科学校](https://zh.wikipedia.org/wiki/专科学校 "wikilink")。\[1\]

湖南大众传媒职业技术学院是由[湖南省教育厅](../Page/湖南省教育厅.md "wikilink")、[湖南省广播电视局和](https://zh.wikipedia.org/wiki/湖南省广播电视局 "wikilink")[湖南广播电视台](../Page/湖南广播电视台.md "wikilink")共同创办的学校，\[2\]拥有三个校区，分别位于：[星沙](https://zh.wikipedia.org/wiki/星沙 "wikilink")、[雨花亭](https://zh.wikipedia.org/wiki/雨花亭 "wikilink")、[东塘](https://zh.wikipedia.org/wiki/东塘 "wikilink")。\[3\]

## 历史

### 校名更易

| 年份    | 历史变更                                                                                                                                                                     |
| ----- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 1949年 | [中国人民银行](../Page/中国人民银行.md "wikilink")湖南省分行干部培训班。                                                                                                                        |
| 1951年 | “中国人民银行湖南省分行干部培训班”和“长沙私立涵德女子会计学校”以及“昭信学校”，合并为“湖南省银行学校”。                                                                                                                  |
| 1956年 | ”湖南省银行学校“划分为：“长沙银行中等专业学校”和“湖南银行干部学校”。                                                                                                                                    |
| 1957年 | “长沙银行中等专业学校”合并入“安徽合肥银行学校”。                                                                                                                                               |
| 1963年 | [湖南省编制委员会再次恢复](https://zh.wikipedia.org/wiki/湖南省编制委员会 "wikilink")“湖南银行干校”。                                                                                               |
| 1969年 | “湖南银行干校”在[文化大革命](../Page/文化大革命.md "wikilink")中被迫停办。                                                                                                                      |
| 1978年 | [湖南省革命委员会](../Page/湖南省革命委员会.md "wikilink")再次恢复“湖南银行干校”，学校在1979年开学。                                                                                                       |
| 2000年 | [湖南省教育厅](../Page/湖南省教育厅.md "wikilink")接受“湖南银行干校”，并且[湖南省教育厅](../Page/湖南省教育厅.md "wikilink")与[湖南教育电视台合伙开办](https://zh.wikipedia.org/wiki/湖南教育电视台 "wikilink")“湖南大众传媒职业技术学院”。 |

## 学校院系

湖南大众传媒职业技术学院开设了9个教学系、32个专业。

  - 教学系：主持与播音系、网络传媒系、影视艺术系、动画艺术系、应用美术系、电广传媒系、国际传播系、国际传播系、报刊传媒系、经济与管理系
  - 思想政治课教学部
  - 体育卫生课教学部
  - 研究所：现代传播研究所、传媒职业与教育研究所、动漫游戏研究所、现代远程教育研究所、传媒管理研究所

## 学校文化

  - 校训：明德博学、求是致远
  - 教风：敬业、爱岗、爱生\[4\]
  - 学风：勤奋、上进、守纪
  - 工作作风：高效、廉洁、民主、务实\[5\]
  - 期刊报纸：《湖南大众传媒职业技术学院学报》是该校官方刊物，创刊于2001年。

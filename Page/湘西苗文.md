拉丁**湘西苗文**，為[中央民族事務委員會於](../Page/中央民族事務委員會.md "wikilink")1957年為[湘西苗語區](../Page/苗語湘西方言.md "wikilink")，主要是[湖南省](../Page/湖南省.md "wikilink")[湘西苗族自治州](../Page/湘西苗族自治州.md "wikilink")、[貴州省](../Page/貴州省.md "wikilink")[松桃苗族自治縣的](../Page/松桃苗族自治縣.md "wikilink")[苗族群眾創製的](../Page/苗族.md "wikilink")[表音文字](../Page/表音文字.md "wikilink")。以[湘西苗語西部次方言](../Page/苗語湘西方言.md "wikilink")[花垣縣](../Page/花垣縣.md "wikilink")[吉衛鎮鎮政府附近的臘乙村的語音為標準音](../Page/吉衛鎮.md "wikilink")。

### 聲調

苗文有6個[聲調](../Page/聲調.md "wikilink")，用六個子音字母當作聲調字母寫在韻母的後面。聲調[調值與聲調字母對照如下表所示](../Page/調值.md "wikilink")：

|      |      |      |      |      |      |      |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| 聲調字母 | b    | x    | d    | l    | t    | s    |
| 調值   | 35   | 31   | 44   | 33   | 53   | 42   |
| 例字   | jiub | jiux | jiud | jiul | jiut | jius |
| 漢語解釋 | 針    | （雞）籠 | 抱    | 完    | 年    | 吠    |

  - 附注：

<!-- end list -->

1.  依據[古漢語借詞以及其他](../Page/古漢語.md "wikilink")[苗語支語言的比較](../Page/苗語支.md "wikilink")，古苗語的4個聲調因聲母[清濁分化成](../Page/清濁.md "wikilink")8個聲調，後來在[吉衛鎮苗語中](../Page/吉衛鎮.md "wikilink")，[陰上](../Page/上聲.md "wikilink")、[陰入合併](../Page/陰入.md "wikilink")（如古漢語借詞“長”，“翅”在[黔東苗語養蒿話中分屬陰上](../Page/苗語黔東方言.md "wikilink")35、陰入53，吉衛話統一讀44調）；[陽上](../Page/上聲.md "wikilink")、[陽入合併](../Page/入聲.md "wikilink")（如古漢語借詞“馬”、“八”在[黔東苗語養蒿話中分屬陽上](../Page/苗語黔東方言.md "wikilink")11和陽入31，但吉衛話都讀33調）。
2.  現代漢語借詞，陰平歸入44調，陽平歸入33調與少部分31調，上聲歸入53調，去聲歸入35調。

## 書寫範例

`    Wuxcheid nangd nes nex wud daos:"Mous nangd nuxnus nib jib ad leb nex bangd nux?"`

  - 直譯：寨子 的 人 問 他 道：“你 的 野雞 在 哪 一 個 人 打 鳥？”
  - 翻譯：寨子的人問他：“你在哪裡一個人打到野雞的？”

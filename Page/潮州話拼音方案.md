**潮州話拼音方案**（[潮州话](../Page/潮州话.md "wikilink")：（潮州）（汕头）；[泉漳话](../Page/泉漳片.md "wikilink")[閩南拼音](../Page/閩南方言拼音方案.md "wikilink")：，[台羅拼音](https://zh.wikipedia.org/wiki/台灣閩南語羅馬字拼音方案 "wikilink")：）是[廣東省教育部門於](../Page/广东省.md "wikilink")1960年9月公佈的[廣東拼音方案](../Page/廣東拼音方案.md "wikilink")之一，以汕頭市的潮州話語音為基礎，以[拉丁字母](../Page/拉丁字母.md "wikilink")拼寫語音，聲調以上標數字顯示。由於潮州話屬於[閩南語的次方言](https://zh.wikipedia.org/wiki/闽南语 "wikilink")，所以拼寫閩南語的[白話字也可以拼寫潮州話](https://zh.wikipedia.org/wiki/白話字 "wikilink")。方案又按潮州話讀音稱爲，也就是“拼音”。

## 方案內容

### 字母

拼音採用[拉丁字母](../Page/拉丁字母.md "wikilink")，其中f、j、q、v、w、x、y不用。[ê是字母e的變體](https://zh.wikipedia.org/wiki/ê "wikilink")。

### 聲母

聲母總數為十八個。不以輔音為首的音節，稱為零聲母。其餘聲母順序如下：

| 潮州話聲母  |
| ------ |
| 塞音     |
| 清（不送氣） |
| b  波   |
| d  刀   |
|        |
| g  哥   |

其中，b和g可用作韻尾。

  - 詩 - si<sup>1</sup>
  - 死 - si<sup>2</sup>
  - 世 - si<sup>3</sup>
  - 薛 - sih<sup>4</sup>

### 說明

本方案基本上是根據[汉语拼音方案擬訂](https://zh.wikipedia.org/wiki/汉语拼音方案 "wikilink")，原則上不另制新字母，不隨便改變原來的發音。

本方案以[汕頭音為依據](../Page/汕头市.md "wikilink")，其他各縣語音與汕頭音有出入者，則根據本方案增刪聲韻母（及聲調）。

如[潮安縣](https://zh.wikipedia.org/wiki/潮安縣 "wikilink")（現潮州市）音有iêng（焉）、uêng（冤）、iêg（噎）和uêg（越）等韻母；[海豐和](../Page/海丰县.md "wikilink")[陸豐音有韻母ei](../Page/陆丰市.md "wikilink")，在拼注潮安、海豐和陸豐音時，可酌情增加；又如[澄海縣城音缺im](../Page/澄海区.md "wikilink")、am、iam和uam等韻母，而[潮陽](../Page/潮阳区.md "wikilink")、[惠來](../Page/惠来县.md "wikilink")、海豐和陸豐等縣缺韻母e（e韻母併入u和i），拼注上述各地語音時可以刪去。

一些聲母發音有不同的，也可適當加以改動 ，如iou韻，除汕頭、潮安和澄海外，其他各縣都念成iao，拼注其他各縣音時便可改iou為iao。

## 拼音方案的不同詮釋

在一些中國大陸出版的書籍，編者可能會根據具體情況，約略調整拼音方案\[9\]。

例如，編者為了讓潮州話拼音方案與[漢語拼音方案匯合](../Page/汉语拼音.md "wikilink")，會加入上文所沒有的j、q和x。三者讀音分別與上述的z、c和s一樣，但是j、q和x只與i、in和韻頭是i、in的韻母相拼（事实上，可以认为是腭化的z、c和s，音色略有不同）。拼音字母組合也有不同詮釋，例如r可能按其[國際音標](../Page/國際音標.md "wikilink")（）而改為dz，鼻化韻也如斯處理。\[10\]

### 韻母、鼻化韻母

  - e //近似於台羅拼音的 ir //（白話字無此符號）。
  - ê 同白話字、台羅的 e //。
  - ou //近似白話字的 o͘ 和台羅的 oo //。
  - iou //近似白話字和台羅的 io //。
  - (i)ao // 同白話字的 (i)ao、台羅的 (i)au。
  - uê 同白話字的 oe、台羅的 ue //。
  - ua 同白話字的 oa、台羅的 ua //。
  - 標示鼻化的 -n //，同白話字的 -ⁿ、台羅的 -nn。

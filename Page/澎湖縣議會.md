**澎湖縣議會**（），是[中華民國](../Page/中華民國.md "wikilink")[臺灣省](../Page/臺灣省.md "wikilink")[澎湖縣](../Page/澎湖縣.md "wikilink")的最高[立法機關](https://zh.wikipedia.org/wiki/立法機關 "wikilink")，位於[澎湖縣](../Page/澎湖縣.md "wikilink")[馬公市](../Page/馬公市.md "wikilink")中正路76號。\[1\]前身可追溯至[日治時期的](../Page/台灣日治時期.md "wikilink")**澎湖廳協議會**。\[2\]目前為第十九屆縣議會，自2018年12月25日就任，當前議長為[中國國民黨](../Page/中國國民黨.md "wikilink")籍的[劉陳昭玲](../Page/劉陳昭玲.md "wikilink")，副議長為[中國國民黨](../Page/中國國民黨.md "wikilink")籍的[陳雙全](../Page/陳雙全.md "wikilink")。

## 歷史

### 協議會時期

澎湖群島在清[光緒](../Page/光绪.md "wikilink")21年（1895）年，因清廷簽署〈[馬關條約](https://zh.wikipedia.org/wiki/馬關條約 "wikilink")〉轉由[日本帝國統治](../Page/大日本帝国.md "wikilink")\[3\]，最初政府機關皆因襲[清代舊制](https://zh.wikipedia.org/wiki/清代 "wikilink")，未設地方議會。[大正](../Page/大正.md "wikilink")九年（1920年）[臺灣總督府](../Page/臺灣總督府.md "wikilink")更易地方行政官制，新制規定在州、市、街、庄等各級需設置[協議會](../Page/協議會_\(台灣日治時期\).md "wikilink")（地方公共團體），但[澎湖郡隸屬於](https://zh.wikipedia.org/wiki/澎湖郡 "wikilink")[高雄州](../Page/高雄州.md "wikilink")，郡級形同虛設，故澎湖地區僅設街庄層級的協議會，且協議會成員乃官派、並非民選。此外，協議會成員僅有供施政諮詢之用，無有參與決策之權。澎湖郡在大正15年（1926年）升格為澎湖廳，卻直到[昭和](../Page/昭和.md "wikilink")12年（1937年）才正式成立廳級協議會，而每屆十名澎湖廳協議會的會員中，日本籍便佔七位，台灣籍僅有三名。\[4\]

昭和12年至20年（公元1937年-1945年）期間，澎湖廳協議會中姓名可考的臺籍議會員有三屆，其中，第一屆：戴耀閭（[白沙人](../Page/白沙鄉_\(台灣\).md "wikilink")，醫師）、呂哲（白沙人，醫師）、[吳爾聰](https://zh.wikipedia.org/wiki/吳爾聰 "wikilink")（[湖西人](../Page/湖西鄉.md "wikilink")，教員）；第二屆：洪耀澎（[臺中人](https://zh.wikipedia.org/wiki/臺中市 "wikilink")，醫師）、呂哲、李昭生（[西嶼人](https://zh.wikipedia.org/wiki/西嶼鄉 "wikilink")，陸軍雇員）；第三屆：[鄭暻星](https://zh.wikipedia.org/wiki/鄭暻星 "wikilink")（[馬公人](../Page/馬公市.md "wikilink")，商界）、呂哲、李昭生。\[5\]

### 參議會時期

1945年，日本帝國在[第二次世界大戰中宣布投降](../Page/第二次世界大战.md "wikilink")，[國民政府](../Page/國民政府.md "wikilink")派遣[陳儀](../Page/陳儀.md "wikilink")接收台灣。同年（即民國34年）11月8日，台灣行政長官公署民政處組織「各州廳接管委員會」，12月頒布「台灣省各級民意機關成立方案」，要求台灣各鄉鎮限期內成立代表會及參議會等機構，並宣布翌年三月辦理各縣市參議員選舉。但此次選舉並非全面民選，分作「區域選舉」以及「職業選舉」，澎湖縣鄉鎮數未滿七個，區域選舉名額為七人，職業團體依規定選舉名額不得超過十分之一，名額僅三人。\[6\]應屆當選人名單如下：

澎湖縣參議會成立於民國35年（1946年）4月15日，吳爾聰出任首任議長、[郭石頭則為副議長](https://zh.wikipedia.org/wiki/郭石頭 "wikilink")，議會辦公室則使用日治時期高等女學校教室，位於今馬公市光明路上。參議員任期原本僅規定為兩年，但台灣省的各縣市議會尚未成立，[憲法遂延長首屆參議員的任期直到民國](https://zh.wikipedia.org/wiki/憲法 "wikilink")39年，中央奉准台灣省實施地方自治為止，所以澎湖縣參議會僅舉辦過一次選舉。\[8\]\[9\]

### 縣議會時期

#### 四選區

民國39年（1950年），國民政府頒布「[臺灣省各縣市實施地方自治綱要](../Page/臺灣省各縣市實施地方自治綱要.md "wikilink")」，自同年七月辦理第一屆縣市議員選舉，任期規定為兩年，第三屆起修改為三年，第六屆起延長為四年。縣議會名額根據全縣人口分配，若台灣東部或偏遠山區人口稀少處，則酌情增額分配，澎湖縣議會在第一屆至第十一屆被劃分為四個選區，第一屆選區轄區與名額如下：

| 選區    | 應選人數           | 轄區          |
| ----- | -------------- | ----------- |
| 第一選舉區 | 10名            | 馬公鎮、湖西鄉、白沙鄉 |
| 第二選舉區 | 2名             | 西嶼鄉         |
| 第三選舉區 | 2名             | 望安鄉         |
| 第四選舉區 | 1名             | 七美鄉         |
| 合計    | 15名（含婦女保障名額一名） |             |

四選區時期的選舉轄區（以第一屆為例）\[10\]

澎湖縣第一屆議會於民國39年（1950年）10月27日成立，首任議長[鄭大洽](../Page/鄭大洽.md "wikilink")（馬公人）、副議長[陳乾坤](https://zh.wikipedia.org/wiki/陳乾坤 "wikilink")（西嶼人）。議員名額並非定制，爾後隨中央規定每年調整，又屢有增減。\[11\]

#### 六選區

澎湖縣議會自第一屆開辦以來，由於「第一選舉區」涵蓋馬公、湖西和白沙三個鄉鎮，區域過於遼闊，容易造成鄉鎮間民意反應的鴻溝，民間一直都有爭取細部劃分的檢討聲浪，直到民國77年（1988年）第12屆縣議會議員選舉才落實。第12屆縣議員名額和選區劃分如下：

| 選區    | 應選人數 | 轄區  | 備註        |
| ----- | ---- | --- | --------- |
| 第一選舉區 | 10名  | 馬公市 | 含婦女保障名額一名 |
| 第二選舉區 | 3名   | 湖西鄉 |           |
| 第三選舉區 | 2名   | 白沙鄉 |           |
| 第四選舉區 | 2名   | 西嶼鄉 |           |
| 第五選舉區 | 1名   | 望安鄉 |           |
| 第六選舉區 | 1名   | 七美鄉 |           |
| 合計    | 19名  |     |           |

六選區時期選舉轄區（以第12屆為例）\[12\]

澎湖縣議會自第12屆議員選舉（1988年）實施「六選區」制之後，沿用迄今，應選名額則因應規定，每年偶有增減。\[13\]\[14\]

## 本屆議員

澎湖縣議會議員為公民直選，共劃分6個選區，共有19席。第十九屆澎湖縣議員任期由2018年到2022年。各區應選縣議員人數：

  - **第一選區：（11席）[馬公市](../Page/馬公市.md "wikilink")**
  - **第二選區：（3席）[湖西鄉](../Page/湖西鄉.md "wikilink")**
  - **第三選區：（2席）[白沙鄉](../Page/白沙鄉_\(台灣\).md "wikilink")**
  - **第四選區：（1席）[西嶼鄉](https://zh.wikipedia.org/wiki/西嶼鄉 "wikilink")**
  - **第五選區：（1席）[望安鄉](https://zh.wikipedia.org/wiki/望安鄉 "wikilink")**
  - **第六選區：（1席）[七美鄉](https://zh.wikipedia.org/wiki/七美鄉_\(臺灣\) "wikilink")**

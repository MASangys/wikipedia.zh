**澳門十六浦索菲特大酒店**（）位於[澳門](../Page/澳門.md "wikilink")[內港](../Page/內港.md "wikilink")[十六浦發展項目地段](../Page/十六浦.md "wikilink")，由[雅高酒店集團](../Page/雅高酒店集團.md "wikilink")（ACCOR
Hospitality）管理經營，旗下酒店集團除索菲特（Sofitel）品牌外，還包括Novotel、Ibis、Mecure、Swissotel、Sofitel
Legend等，此家澳門索菲特則於2008年8月正式開幕。酒店為[五星級](../Page/五星級.md "wikilink")，設有408間豪華客房，同時設有一個法式及地中海風味餐廳及一個大堂[酒吧](../Page/酒吧.md "wikilink")、室外[游泳池](../Page/游泳池.md "wikilink")、健身中心、桑拿及蒸氣浴室、商務中心、一個300平方米的多功能廳及多個會議室等酒店設施，所有客房均備有法國名牌[L'Occitane浴室用品](../Page/L'Occitane.md "wikilink")；同時還有設有索菲特行政樓層及19所奢華式別墅-濠庭十六浦，浴室用品均採用Hermes。

酒店位處的十六浦位於澳門內港，是舊時[葡萄牙領地內最為古老的城區](../Page/葡萄牙.md "wikilink")，酒店的建築風格及整個發展項目將會反映澳門20世紀初至中期的歷史特色，25個世遺景點盡在咫尺，並與作為中國第31個[世界遺產的](../Page/世界遺產.md "wikilink")[澳門歷史城區環境相融合](../Page/澳門歷史城區.md "wikilink")。\[1\]

## 交通配套

### 自設交通

十六浦設有來往酒店與口岸的接駁專巴服務。接駁專巴服務設有以下2條路線：

|     |                                    |                                  |
| --: | :--------------------------------: | -------------------------------- |
| 十六浦 |                 ↔                  | [關閘口岸](../Page/關閘.md "wikilink") |
|   ↔ | [外港碼頭](../Page/外港碼頭.md "wikilink") |                                  |

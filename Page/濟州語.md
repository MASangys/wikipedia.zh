**濟州語**（），亦作**濟州方言**（），是[濟州地域除了](../Page/濟州特別自治道.md "wikilink")[楸子群岛以外所通行的語言](https://zh.wikipedia.org/wiki/楸子群岛 "wikilink")。有时被认为是[韓語的一種方言](../Page/朝鮮語.md "wikilink")，而且保留了不少現時已在朝鮮半島失去的古舊用詞和用語。另外，濟州語亦有跟朝鮮半島韓語成分不同的-{A|zh-hant:借詞;zh-hans:借词}-，當中與朝鮮半島韓語不同的用詞亦有不少。

## 與朝鮮半島方言的差異

濟州方言與朝鮮半島的韓語方言最明顯的差異，在於濟州方言沒有[敬語](https://zh.wikipedia.org/wiki/敬語 "wikilink")，其語尾變化亦比較簡單。另外，濟州語保存了不少與朝鮮半島韓語方言不同的古語，亦形成了不少獨自的單語。濟州方言地域的語言因此與朝鮮半島方言地域有着頗明顯的差異。

### 古語

濟州方言裡有不少在濟州獨有的用語：這些用語只存在於[古韓語](../Page/古朝鮮語.md "wikilink")，但已在韓語的標準語消失。語言學家估計，這些古語很可能源自[耽羅國的語言](https://zh.wikipedia.org/wiki/耽羅國 "wikilink")。

### 借用語

濟州語有大量的借詞，\[1\]這與朝鮮半島周邊國家有關，例如：在1274年至1294年期間，[耽羅](../Page/耽羅.md "wikilink")是[元朝](../Page/元朝.md "wikilink")的直轄地，有不少蒙古語借詞都是在這時間借入；在[清朝](../Page/清朝.md "wikilink")時，借入了[滿語的借詞](../Page/满语.md "wikilink")；在日本殖民時期，亦引入了不少日語詞彙。而來自漢語的[借詞](../Page/外来语.md "wikilink")，一直都有存在。

但另一方面，由於濟州語保存了不少古語，亦缺少了敬語，無形中為數不少已融入標準韓語的漢語詞亦不存在於濟州語，例如：「食事」、「安寧」等。

## 文法

### 現在式

#### 動詞

| 語尾      | 標準語   | 例                    | 比較           |
| ------- | ----- | -------------------- | ------------ |
| –암/엄수과? | –ㅂ니까? | 적엄수과?(적습니까/적고 있습니까?) | 적다(=書/write) |
| –암/엄수다  | –ㅂ니다  | 적엄수다(적습니다/적고 있습니다)   |              |

| 濟州語       | 標準語       | 意思       |
| --------- | --------- | -------- |
| 어디 감수광?   | 어디 가십니까?  | 您去哪兒？    |
| 서울에 감수다.  | 서울에 갑니다.  | 去首爾。     |
| ○○를 알암수과? | ○○를 아십니까? | 您知道...吗？ |
| 잘 몰람수다.   | 잘 모릅니다.   | 不知道。     |
|           |           |          |

例文

#### 形容詞

| 語尾       | 標準語        | 例           | 比較         |
| -------- | ---------- | ----------- | ---------- |
| –우꽈/–수꽈? | –ㅂ니까/–습니까? | 적수꽈?(적습니까?) | 적다(=少/few) |
| –우다/–수다  | –ㅂ니다/–습니다  | 적수다(적습니다)   |            |

| 濟州語      | 標準語       |
| -------- | --------- |
| 개똥이 잇수꽈? | 개똥이 있습니까? |
| 어수다.     | 없습니다.     |
| 곱닥허우다.   | 이쁩니다.     |
| 고맙수다.    | 고맙습니다.    |
| 반갑수다.    | 반갑습니다.    |

例文

### 過去式

| 結尾      | 標準語      | 例             | 注意              |
| ------- | -------- | ------------- | --------------- |
| –앗/엇수과? | –았/었습니까? | 알앗수과? (알았습니까) | 과/꽈/광/꽝으로 발음한다. |
| –앗/엇수다. | –았/었습니다. | 알앗수다. (알았습니다) |                 |

| 濟州語      | 標準語       | 意思      |
| -------- | --------- | ------- |
| 점심 먹엇수과? | 점심 먹었습니까? | 你吃午飯了嗎？ |
| 예, 먹엇수다. | 예, 먹었습니다. | 是的，吃過了。 |
| 언제 왓수과?  | 언제 왔습니까?  | 什麼時候來了？ |
| 오늘 왓수다.  | 오늘 왔습니다.  | 今天來了。   |

## 第一次偵察兵計畫

[鳳凰號目的是尋找適合微生物生存的環境](https://zh.wikipedia.org/wiki/鳳凰號_\(探測器\) "wikilink")，與研究水的歷史。鳳凰號已於2007年8月4日發射，2008年5月25日降落於火星高緯度的平原。成功完成預定90天的任務後，於2008年11月10日無法繼續連絡鳳凰號而結束任務。原因是冬天降臨，沒有陽光而失去能源。

## 第二次偵察兵計畫

2008年9月15日，[NASA宣佈選擇](https://zh.wikipedia.org/wiki/NASA "wikilink")[MAVEN](../Page/MAVEN.md "wikilink")作為2013年的偵察兵計畫。\[1\] 2007年預計提供的預算不超過4.75億美元\[2\]，但由於任務延期預算可能會變更。

雖然一開始預期會是個「雙重」任務－－會有兩個軌道器，然而NASA目前只計畫資助一項任務。\[3\]

### 最終結果

2007年1月8日，NASA宣佈由MAVEN贏得第二次偵察兵計畫。\[4\]

| 任務名稱         | 敘述                                                                                                                                                                                                                                                                                                    |
| ------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| MAVEN        | 根據眾多證據，火星幾十億年來已失去其大部分的大氣。MAVEN（Mars Atmosphere and Volatile Evolution，火星大氣與揮發物演化）將研究目前大氣的流失速率，以及與[太陽風的關係](https://zh.wikipedia.org/wiki/太陽風 "wikilink")。                                                                                                                                              |
| Great Escape | 這項計劃將要直接透過測量火星上層大氣的結構與動態，來了解火星大氣層演變的基礎過程。除此之外也會測量如[甲烷](../Page/甲烷.md "wikilink")一類潛在的有機大氣構成物。主要調查者是[科羅拉多州](../Page/科羅拉多州.md "wikilink")[博爾德西南研究所](https://zh.wikipedia.org/wiki/博爾德_\(科羅拉多州\) "wikilink")（Southwest Research Institute）的艾倫·斯特恩博士（Dr. Alan Stern）。聖安東尼奧（San Antonio）西南研究所會負責計劃管理。\[5\] |

### 其他候選

| 任務名稱        | 敘述                                                                                                                           |
| ----------- | ---------------------------------------------------------------------------------------------------------------------------- |
| Artemis     | 從繞行火星的母船中發射碟形登陸器登陸火星，每個直徑兩英尺（61公分），最多可能發射四個。每個登陸器將利用降落傘的方式登陸火星表面，並分析火星土壤與大氣。四個登陸器其中的兩個預計將登陸在火星的極地區域。                         |
| ARES        | 以一架無人飛機觀察火星。\[6\]\[7\]                                                                                                       |
| Chronos     | 一個魚雷外型的探測器，利用噴嘴加熱的方式融化火星極地的冰冠。探測器預計將深入地表100碼（91公尺），並且分析這些融化的水以研究火星氣候史。\[8\]                                                  |
| KittyHawk   | 以三台或四台滑翔機觀測，滑翔機翼展約六呎（1.83公尺），並且探測[水手號峽谷](https://zh.wikipedia.org/wiki/水手號峽谷 "wikilink")。這些滑翔機將攜帶紅外線頻譜儀與照相機。                 |
| MOO         | 火星有機物觀察者（Mars Organics Observer）以軌道衛星觀察[火星大氣的甲烷](https://zh.wikipedia.org/wiki/火星大氣層#甲烷 "wikilink")：從哪裡被釋放出來、量有多少、還有釋放的頻率為何。 |
| The Naiades | 本計畫的命名由來為希臘神話中居住在山林水澤中的女神。這個計畫預計將兩台登陸器送往可能含有地下水的區域，並且使用低頻電磁與其他裝備探測火星是否有地下水存在。                                                |
| SCIM        | 屬於樣本取回任務，在接近火星時劃過火星大氣、撈起約1000粒沙塵與幾公升的空氣，並在速度不低於脫離速度下返回。                                                                      |
| Pascal      | 將24台小型氣象觀測站散佈在火星各地。                                                                                                          |
| THOR        | 類似NASA的[深撞號](https://zh.wikipedia.org/wiki/深撞號 "wikilink")：挑選出淺層地表下有水冰、甚至液態水的地點，發射兩顆銅球撞出撞擊坑，再由軌道衛星分析。                        |
| Urey        | 包含一個登陸器與探測車，研究火星岩石的年齡。計畫預計將針對科伯洛斯高原一帶進行研究，並且將會尋找特定的礦物，以幫助科學家研究火星上的撞擊坑與月球上的撞擊坑的不同。                                            |

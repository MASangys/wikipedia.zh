**灭多威**（**Methomyl**），又譯**纳乃得**，是一种[氨基甲酸酯](../Page/氨基甲酸酯.md "wikilink")类[杀虫剂](https://zh.wikipedia.org/wiki/杀虫剂 "wikilink")，原药为白色晶体粉末，商品态为可湿性粉剂、可溶性粉剂或乳油。1966年美国[杜邦](../Page/杜邦.md "wikilink")公司首先以“Lannate/万灵”的商品名将其推广为广谱杀虫剂。主要用于[棉蚜](https://zh.wikipedia.org/wiki/棉蚜 "wikilink")（*Aphis gossypii*）、[斜纹夜蛾](https://zh.wikipedia.org/wiki/斜纹夜蛾 "wikilink")（*Spodoptera litura*）等的防杀。

灭多威被[美国国家环境保护局](../Page/美国国家环境保护局.md "wikilink")（EPA）归为第一类高毒性物质，对人体[皮肤](https://zh.wikipedia.org/wiki/皮肤 "wikilink")、[肺](../Page/肺.md "wikilink")、[消化道](../Page/消化道.md "wikilink")、[肾](https://zh.wikipedia.org/wiki/肾 "wikilink")、[脾和](https://zh.wikipedia.org/wiki/脾 "wikilink")[造血器官有不同程度的毒害作用](https://zh.wikipedia.org/wiki/造血器官 "wikilink")，但在对大鼠和兔的动物实验中未发现致畸、致突变、致癌作用。\[1\]

## 残留限量

中华人民共和国国家标准《食品中百草枯等54种农药最大残留限量》（GB26130-2010）规定灭多威在食品中的最大残留限量为：

|                                                       |                                                          |
| ----------------------------------------------------- | -------------------------------------------------------- |
| [茶叶](https://zh.wikipedia.org/wiki/茶叶 "wikilink")     | 3 [mg/kg](https://zh.wikipedia.org/wiki/百萬分率 "wikilink") |
| [菜薹](https://zh.wikipedia.org/wiki/菜薹 "wikilink")     | 1 mg/kg                                                  |
| [结球甘蓝](https://zh.wikipedia.org/wiki/结球甘蓝 "wikilink") | 2 mg/kg                                                  |

中华民国行政院卫生署《残留农药安全容许量标准》（101年1月20日）规定的最大残留限量为：

|      |           |
| ---- | --------- |
| 小叶菜类 | 3 mg/kg   |
| 小浆果类 | 2 mg/kg   |
| 包叶菜类 | 2 mg/kg   |
| 瓜菜类  | 1 mg/kg   |
| 米类   | 0.5 mg/kg |
| 豆菜类  | 1 mg/kg   |
| 果菜类  | 1 mg/kg   |
| 柑桔类  | 1 mg/kg   |
| 核果类  | 2 mg/kg   |
| 根菜类  | 2 mg/kg   |
| 茶类   | 1 mg/kg   |
| 干豆类  | 0.5 mg/kg |
| 梨果类  | 0.5 mg/kg |
| 番荔枝  | 1 mg/kg   |
| 杂粮类  | 1 mg/kg   |

残留检测可以参照[ASTM](https://zh.wikipedia.org/wiki/ASTM_國際標準組織 "wikilink") D7645 - 10e1使用[液相色谱](https://zh.wikipedia.org/wiki/液相色谱 "wikilink")-[串联质谱](https://zh.wikipedia.org/wiki/串联质谱 "wikilink")（LC/MS/MS）检测。采用中华人民共和国农业行业标准NY/T 761-2008第三部分的检测方法，检出限亦可达到0.01 mg/kg。

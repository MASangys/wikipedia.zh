[汉语](../Page/汉语.md "wikilink") |capital = |government_type = [保护国](https://zh.wikipedia.org/wiki/保护国 "wikilink") |title_leader1 = |leader1 = |religion = |stat_area1 = |stat_pop1 = |stat_year1 = |currency = |today =  |deputy1 = Oyun Ölzey-Ochur oglu Kombu-Dorzhu |deputy2 = Irgit Agbaan-Demchi |deputy3 = Oyun Kombu-Dorzhu oglu Sodunam-Balchyr |deputy4 = |year_deputy1 = 1914-1915 |year_deputy2 = 1915-1916 |year_deputy3 = 1916-1921 |year_deputy4 = |title_deputy = 大臣 }} **烏梁海边疆区**是一個位於今[圖瓦共和國的](https://zh.wikipedia.org/wiki/圖瓦共和國 "wikilink")[俄国](https://zh.wikipedia.org/wiki/俄国 "wikilink")[保護國](https://zh.wikipedia.org/wiki/保護國 "wikilink")，存在于1914年至1921年。

在[辛亥革命](../Page/辛亥革命.md "wikilink")和[外蒙古獨立後](https://zh.wikipedia.org/wiki/外蒙古獨立 "wikilink")，唐努烏梁海的[那顏们于](https://zh.wikipedia.org/wiki/那顏 "wikilink")1911年12月1日成立。1913年秋，两个那颜上书俄国沙皇請求将乌梁海交由俄羅斯保護，其他那颜纷纷效仿。沙皇[尼古拉二世同意请求](https://zh.wikipedia.org/wiki/尼古拉二世 "wikilink")，于1914年4月17日成立了乌梁海边疆区，乌梁海成为俄国的一个保护国。

[二月革命後](https://zh.wikipedia.org/wiki/二月革命 "wikilink")，烏梁海边疆区繼續作為一個保護國留在俄羅斯，在[俄國內戰期间](https://zh.wikipedia.org/wiki/俄國內戰 "wikilink")（1918年－1921年），烏梁海边疆区分別被[白軍和中華民國軍隊佔領](https://zh.wikipedia.org/wiki/白軍 "wikilink")。在[苏俄紅軍支持下](../Page/苏联红军.md "wikilink")，[圖瓦人民革命黨于](https://zh.wikipedia.org/wiki/圖瓦人民革命黨 "wikilink")1921年8月14日成立了圖瓦人民共和國。

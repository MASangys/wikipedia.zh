在[數學裡](../Page/數學.md "wikilink")，有著許多明顯[矛盾的虛假](../Page/矛盾.md "wikilink")[證明存在](../Page/證明.md "wikilink")。即使其證明是有缺陷的，其錯誤－通常是經過設計的－卻常是較難抓摸的。這些[謬誤一般都儘止於好奇而已](../Page/謬誤.md "wikilink")，但可以被使用顯示嚴謹在數學中的重要性。

大多數此類的證明都仰賴著同種錯誤的變形。此一錯誤為採一非[單射的](../Page/單射.md "wikilink")[函數](../Page/函數.md "wikilink")\(f\)，以觀察對某些\(x\)和\(y\)，會有\(f(x)=f(y)\)，來（錯誤地）做出\(x=y\)的結論。[零除數是此類錯誤的一特例](../Page/除以零.md "wikilink")；\(f\)為將\(x\)映射至\(x\times 0\)的函數，而其錯誤的一步是起於將\(x\times 0=y\times 0\)的等式做成\(x=y\)的結論。相似地，下面證明了\(5=4\)的句子也是以函數\(f(x)=x^{2}\)的同一種錯誤造成的。其錯誤的一步始於有某個\(x\)和\(y\)會使得\(x^{2}=y^{2}\)的一正確申論，然後做出了\(x=y\)的一錯誤結論。

## 算术例子

### 證明1是最大的正整數

  - 假設最大的正整數不是\(1\)，而是\(a\)，有\(a > 1\)；

<!-- end list -->

  - \(a > 1 > 0\)，\(a\)為正的，所以由\(a > 1\)得到\(a \times  a > a\)；

<!-- end list -->

  - 但是\(a \times  a\)還是正整數，可是沒有任何正整數比\(a\)大，矛盾；

<!-- end list -->

  - 所以最大的正整數是[1](../Page/1.md "wikilink")。

*[Q.E.D.](../Page/Q.E.D..md "wikilink")*

此一證明是無效的，因為最大的正整數不存在，因此不能如此假設。

### 證明1等於-1

  - 由一等式開始
      -
        \(-1 = -1 \,\)
  - 將兩邊轉成[假分數](../Page/分數.md "wikilink")
      -
        \(\frac{1}{-1} = \frac{-1}{1}\)
  - 將兩邊[開方](../Page/平方根.md "wikilink")
      -
        \(\sqrt{\frac{1}{-1}} = \sqrt{\frac{-1}{1}}\)
  - 其會等於
      -
        \(\frac{\sqrt{1}}{\sqrt{-1}} = \frac{\sqrt{-1}}{\sqrt{1}}\)
  - 兩邊同乘\(\sqrt{-1}\)以來消去分數
      -
        \(\sqrt{1}\sqrt{1} = \sqrt{-1}\sqrt{-1}\)
  - 但任一數的開方之平方會給出原本的數來，故
      -
        \(1 = -1 \,\)

*[Q.E.D.](../Page/Q.E.D..md "wikilink")*

此一證明是無效的，因為負數的開方不是实数，\(\sqrt{\frac{1}{-1}} = \sqrt{\frac{-1}{1}}\)推出\(\frac{\sqrt{1}}{\sqrt{-1}} = \frac{\sqrt{-1}}{\sqrt{1}}\)是错误的（事實上，\(\frac{\sqrt{1}}{\sqrt{-1}} = -i\)，\(\frac{\sqrt{-1}}{\sqrt{1}} = i\)）。

### 證明1等於2

1.令\(a = b \,\)，且\(a \ne 0\)

2.將兩邊乘以a

\[a^{2} = ab \,\] 3.將兩邊減掉\(b^{2} \,\)

\[a^{2} - b^{2} = ab - b^{2} \,\]
4.將兩邊[因式分解](../Page/因式分解.md "wikilink")

\[(a + b)(a - b) = b(a - b) \,\] 5.將兩邊除以\(a - b \,\)

\[a + b = b \,\] 6.因為\(a = b \,\)因此

\[b + b = b \,\] 7.簡化

\[2b = b \,\] 8.將兩邊除以b

\[2 = 1 \,\] *[Q.E.D.](../Page/Q.E.D..md "wikilink")*

這個證明的錯誤點在於第五步，由於\(a - b \,\)等於[零](../Page/零.md "wikilink")，而[除以零是無效的](../Page/除以零.md "wikilink")。

### 證明4等於5

  - 由一等式開始
      -
        \(-20 = -20 \,\)
  - 將等式兩邊以稍微不同但相等的方式表示
      -
        \(25 - 45 = 16 - 36 \,\)
  - 將兩邊做因式分解
      -
        \(5^2 - 5 \times 9 = 4^2 - 4 \times 9\)
  - 將兩邊加上相同的數
      -
        \(5^2 - 5 \times 9 + \frac{81}{4} = 4^2 - 4 \times 9 + \frac{81}{4}\)
  - 將兩邊再做一次因式分解
      -
        \(\left(5 - \frac{9}{2}\right)^2 = \left(4 - \frac{9}{2}\right)^2\)
  - 將兩邊開方
      -
        \(5 - \frac{9}{2} = 4 - \frac{9}{2}\)
  - 消去相同的項
      -
        \(5 = 4 \,\)

*[Q.E.D.](../Page/Q.E.D..md "wikilink")*

那一證明內的錯誤在於\(x^2 = y^2\)不表示\(x = y\)的這一事實。到此之前的算術都是正確的，而事實上，\(-\left(5 - \tfrac{9}{2}\right) = 4 - \tfrac{9}{2}\)。需注意的是，若將4減去\(\tfrac{9}{2}\)，會得到\(-\tfrac{1}{2}\)。若再平方的話，則會得到正的\(\tfrac{1}{4}\)。其下一個邏輯的數學步驟為取兩邊的平方。若這樣做的話，則將會看見\(\tfrac{1}{2}\)會等於\(\tfrac{1}{2}\)。原始的\(-20=-20\)式子事實上是會導致一個正確的等式的（若此一問題是以此一純粹的方式運算的話）。

\=== 證明1+1=0 ===

  - 求\(1+1\)︰

\[\begin{align}
1+1&=1+\sqrt{1} \\
&=1+\sqrt{(-1)(-1)} \\
&=1+\sqrt{-1} \times \sqrt{-1} \\
&=1+i \times i \\
&=1+(-1) \\
&=0\\
\end{align}\]

*[Q.E.D.](../Page/Q.E.D..md "wikilink")*

此證明的錯誤在於\(\sqrt{ab} = \sqrt{a} \times \sqrt{b}\)只有在a與b不皆為負數才成立，\(\sqrt{(-1)(-1)}\)並不等於\(\sqrt{-1} \times \sqrt{-1}\)。

\=== 證明0=1 === 首先，設定一個無窮級數。

\[0 = 0 + 0 + 0 + \cdots\]

因為\(0 = 1 - 1\)，因此：

\[0 = (1 - 1) + (1 - 1) + (1 - 1) + \cdots\]

拆括號之後在於不同的地方加上括號：

\[0 = 1 + (-1 + 1) + (-1 + 1) + (-1 + 1) + \cdots\]

\(-1 + 1 = 0\)，因此：

\[0 = 1 + 0 + 0 + 0 + \cdots\]

\[0 = 1 \,\]

*[Q.E.D.](../Page/Q.E.D..md "wikilink")*

這個證明的錯誤在於，無窮等比級數在公比的絕對值大於等於一的情況下，將括號插入無窮級數求無窮和是沒有意義的，因為這樣的無窮等比級數和發散。因此這類條件不適用於[格蘭迪級數](../Page/格蘭迪級數.md "wikilink")
\(s = 1 - 1 + 1 - 1 + 1 - 1 + \cdots\)。

### 證明0/0等於0

首先，我們知道：

\[0 ^ 4 = 0 \times 0 \times 0 \times 0 = 0\,\]

\[0 ^ 2 = 0 \times 0 = 0\,\] 由於\(a ^ {m - n} = \frac{a ^ m}{a ^ n}\)

因此\(\frac{0 ^ 4}{0 ^ 2} = 0^{4 - 2} = 0 ^ 2 = 0\)

因此\(\frac{0}{0} = 0\)

*[Q.E.D.](../Page/Q.E.D..md "wikilink")*

這個證明的錯誤在於，\(a ^ {m - n} = \frac{a ^ m}{a ^ n}\)成立的前提有\(a \ne 0\)。

### 證明任意兩數都是相等的

設\(u = a \ , \ v = b - c\,\)
設\(a = x - y \ , \ b = (x + y)^2 \ , \ c = 4 (x^2 - x y + y^2)\,\)

由[和立方與](../Page/和立方.md "wikilink")[差立方公式可知](../Page/差立方.md "wikilink")：

  -
    \((u + \sqrt {v})^3 = u^3 + 3 u^2 \sqrt{v} + 3 u v + v \sqrt{v}\)
    \((u - \sqrt {v})^3 = u^3 - 3 u^2 \sqrt{v} + 3 u v - v \sqrt{v}\)

由於\(u = a \ , \ v = b - c\,\)

  -
    \((a + \sqrt {b - c})^3 = a \ (a^2 + 3 b - 3 c) + (3 a^2 + b - c)\sqrt{b - c}\)
    \((a - \sqrt {b - c})^3 = a \ (a^2 + 3 b - 3 c) - (3 a^2 + b - c)\sqrt{b - c}\)

將\(a = x - y \ , \ b = (x + y)^2 \ , \ c = 4 (x^2 - x y + y^2)\,\)代入\(3 a^2 + b - c\,\)，可得：

  -
    \(3 a^2 + b - c = 3 (x^2 - 2 x y + y^2) + x^2 + 2 x y + y^2 - 4 (x^2 - x y + y^2) = 0\,\)

因此：

\[\begin{align}
(a + \sqrt{b - c})^3 &= (a - \sqrt {b - c})^3 \\
a + \sqrt{b - c} &= a - \sqrt {b - c} \\
\sqrt {b - c} &= 0 \\
b - c &= 0 \\
b &= c \\
\end{align}\]

代入\(b = (x + y)^2 \ , \  c = 4 (x^2 - x y + y^2)\,\)，可得：

\[\begin{align}
(x + y)^2 &= 4 (x^2 - x y + y^2) \\
x^2 + 2 x y + y^2 &= 4 (x^2 - x y + y^2) \\
-3 x^2 + 6 x y - 3 y^2 &= 0 \\
(x - y)^2 &= 0 \\
x - y &= 0 \\
x &= y \\
\end{align}\]

*[Q.E.D.](../Page/Q.E.D..md "wikilink")*

这个证明的错误在于：

1、在以上的假设下，可得\(v=b-c=(x+y)^2-4(x^2-xy+y^2)=-3(x-y)^2=-3a^2=-3u^2\)，所以\(u\)和\(v\)并不是独立的；

2、在[复数域中](../Page/复数域.md "wikilink")，由\(x^3=y^3\)得不出\(x=y\)。在此证明中，由\((a+\sqrt{b-c})^3=(a-\sqrt{b-c})^3\)得出\(a+\sqrt{b-c}=a-\sqrt{b-c}\)是错误的。

## 几何例子

### 第一题：证明任何三角形都是正三角形

[Isosceles_Proof_False.PNG](https://zh.wikipedia.org/wiki/File:Isosceles_Proof_False.PNG "fig:Isosceles_Proof_False.PNG")
[Isosceles_Proof_True.PNG](https://zh.wikipedia.org/wiki/File:Isosceles_Proof_True.PNG "fig:Isosceles_Proof_True.PNG")
[Errorprove.png](https://zh.wikipedia.org/wiki/File:Errorprove.png "fig:Errorprove.png")
[Rightprove.png](https://zh.wikipedia.org/wiki/File:Rightprove.png "fig:Rightprove.png")
给定三角形△ABC，证明AB = AC:

1.  作∠A的[角平分线](../Page/角平分线.md "wikilink")。
2.  作BC的垂直平分线，并设BC的中点为D。
3.  设这两条直线的交点为P。
4.  从P向AB和AC作垂线，并设垂足为E和F。
5.  作直线PB和PC。
6.  △EAP ≅ △FAP（AP = AP；∠PAF ≅ ∠PAE由于AP平分∠A；∠AEP ≅ ∠AFP都是直角）。
7.  △PDB ≅ △PDC（∠PDB、∠PDC是直角；PD = PD；BD = CD由于PD平分BC）。
8.  △EPB ≅ △FPC（EP = FP由于△EAP ≅ △FAP；BP = CP由于△PDB ≅ △PDC；∠EPB ≅
    ∠FPC由于它们是对顶角）。
9.  因此，AE ≅ AF，EB ≅ FC，AB = AE + EB = AF + FC = AC。
10. 同理，AB = BC，AC = BC。

*证毕。*

这个证明的错误在于，只有在△ABC為等腰三角形，P才會位于三角形的内部，而且AP与DP会重合。

### 第二题：证明直角等于钝角

给定一个矩形ABCD，证明∠DCB=∠ECB；

1.  在矩形ABCD外作CE=CD。
2.  联结AE。
3.  作BC、AE的[中垂线](../Page/中垂线.md "wikilink")，它们的垂足分别是G、F，两条[直线交于H](../Page/直线.md "wikilink")。
4.  在中垂线上的点到线段两端的距离是相等的，所以HA=HE，HB=HC。
5.  矩形的对边相等，得AB=DC；加上作图要求，得AB=EC。
6.  利用*[S.S.S](../Page/全等三角形.md "wikilink")*得△ABH≅△ECH。于是得∠ABH=∠ECH。
7.  由于HB=HC，则得∠HBC=∠HCB。
8.  等量减等量，得∠ABC=∠ECB。
9.  矩形的四个角都是90°，得∠ABC=∠ECB=90°。

*[Q.E.D.](../Page/Q.E.D..md "wikilink")*

这个证明的错误在于，由于△ABH≅△ECH，则∠BHA=∠CHE，即∠AHE=∠BHC-∠BHA+∠CHE，可以把∠AHE看作是∠BHC的旋转，因AH穿过了矩形ABCD，则EH是不可能穿过矩形ABCD的。

## 微积分例子

### 证明0等于1

我们从计算以下的[不定积分开始](../Page/不定积分.md "wikilink")：

\[\int \frac{1}{x} dx\] 利用[分部积分法](../Page/分部积分法.md "wikilink")，可得：

\[u=\frac{1}{x}\]，\(dv=dx\) 因此：

\[du=-\frac{1}{x^2}dx\]，\(v=x\) 所以，有：

\[\int \frac{1}{x} dx=\frac{x}{x} - \int \left ( - \frac{1}{x^2} \right ) x dx\]

\[\int \frac{1}{x} dx=1 + \int \frac{1}{x} dx\]

\[0 = 1 \,\]

*证毕。*

这个证明的错误在于，忽略了積分完會出現的[積分常數C](../Page/積分常數.md "wikilink")。若繼續計算，會得到\(1 + \int \frac{1}{x} dx=1 + \ln\ |x|+C=\ln\ |x|+C'\)。

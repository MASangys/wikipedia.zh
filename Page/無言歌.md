**無言歌**（德語：*Lieder ohne Worte*）是[浪漫樂派作曲家](https://zh.wikipedia.org/wiki/浪漫樂派 "wikilink")[-{zh-hans:费利克斯·门德尔松; zh-hant:費利克斯·孟德爾頌;}-所作的八冊](https://zh.wikipedia.org/wiki/孟德爾頌 "wikilink")48首[鋼琴獨奏曲](https://zh.wikipedia.org/wiki/鋼琴 "wikilink")。這八冊分別在他生命中的不同時間點，為了讓業餘鋼琴家能夠演奏所作。當時鋼琴越來越普遍，慢慢變成許多家庭的焦點。

這些鋼琴曲目屬於浪漫樂派的一部分，其特點是短且狂熱。

## 八冊

第一冊 *Opus 19* 作於西元1830年到1832年。

1.  E大调，甜蜜的回忆
2.  a小调，追悔
3.  A大调，猎歌
4.  A大调，无标题
5.  升f小调，不安
6.  g小调，第一号威尼斯船歌

第二冊 *Opus 30* 作於西元1835年。 在第一冊和第二冊裡，-{zh-hans:门德尔松; zh-hant:孟德爾頌;}-都為兩冊的第六首樂曲給予同樣的曲名：*Venezianisches Gondellied (Venetian Gondelier's Song)*。

1.  降E大调，瞑想
2.  降D小调，不安宁（也有称担心的）
3.  E大调，安慰
4.  b小调，无标题
5.  D大调，无标题
6.  升f小调，第二号威尼斯船歌

第三冊 *Opus 38* 作於西元1837年。 第六首樂曲的曲名是*Duetto*，因為曲中旋律代表兩位不同的歌唱家。

1.  降E大调，无标题
2.  c小调，失去的幸福
3.  E大调，无标题
4.  A大调，无标题
5.  a小调，无标题
6.  降A大调，热情

第四冊 *Opus 53* 作於西元1841年。 [That_Mesmerizing_Mendelssohn_Tune_1.jpg](https://zh.wikipedia.org/wiki/File:That_Mesmerizing_Mendelssohn_Tune_1.jpg "fig:That_Mesmerizing_Mendelssohn_Tune_1.jpg")的一首[拉格泰姆](../Page/拉格泰姆.md "wikilink")取材于op. 62，no. 6\]\]

1.  降A大调，海滨
2.  降E大调，羊毛似的云霞
3.  g小调，骚动
4.  F大调，灵魂的悲伤
5.  a小调，民歌
6.  A大调，飞翔

第五冊 *Opus 62* 作於西元1844年。 第六首樂曲《春之歌》（A major 'Spring Song')是無言歌中最著名的一首。

1.  G大调，五月的薰风
2.  降B大调，无标题
3.  e小调，葬礼进行曲
4.  G大调，晨歌
5.  a小调，第三号威尼斯船歌
6.  A大调，春之歌

第六冊 *Opus 67* 作於西元1845年。 第六首樂曲，*Spinnerlied (Spinner's Song)*，也被人們改了「蜜蜂的婚禮」(Bee's Wedding) 的別名，因為樂曲好像一大群蜜蜂在叫。

1.  降E大调，快板，无标题
2.  升f小调，失去的幻想
3.  降B大调，宁静的快板，无标题
4.  C大调，纺纱歌（蜜蜂的婚礼）
5.  b小调，牧人的怨述
6.  E大调，摇篮曲

第七冊 *Opus 85* 於-{zh-hans:门德尔松; zh-hant:孟德爾頌;}-死後才公諸於世。

1.  F大调，田园风格
2.  a小调，告别
3.  降E大调，急板，无标题
4.  D大调，挽歌
5.  A大调，无标题
6.  降B大调，无标题

第八冊 *Opus 102* 作於西元1845年。

1.  e小调，无标题
2.  D大调，无标题
3.  C大调，骑行
4.  g小调，无标题
5.  A大调，快乐的农夫
6.  C大调，信念

**熔化热**，亦称**熔解热**\[1\]，是单位质量物质由固态转化为液态时，物体吸收的热量\[2\]。物体熔化时的温度称为[熔点](https://zh.wikipedia.org/wiki/熔点 "wikilink")。

熔化热是一种[潜热](https://zh.wikipedia.org/wiki/潜热 "wikilink")，在[熔化](../Page/熔化.md "wikilink")的过程中，物质不断吸收[热量而](https://zh.wikipedia.org/wiki/热量 "wikilink")[温度](../Page/温度.md "wikilink")不变，因此不能通过温度的变化直接探测到这一热量。每种物质具有不同的熔化热。[晶体](../Page/晶体.md "wikilink")在一定压强下具有固定的熔点，也具有固定的熔化热；非晶体，比如[玻璃](../Page/玻璃.md "wikilink")和[塑料](../Page/塑料.md "wikilink")，不具有固定的熔点，因而也不具有固定的熔化热。\[3\]

同一种物质中，[液态比](https://zh.wikipedia.org/wiki/液态 "wikilink")[固态拥有更高的](https://zh.wikipedia.org/wiki/固态 "wikilink")[内能](../Page/内能.md "wikilink")，因此，在熔化的过程中，固态物质要吸收热量来转变为液态。同样，物质由液态转变为固态时，也要释放相同的能量。\[4\]液体中的物质微粒与固体中的相比，受到更小的[分子间作用力](../Page/分子间作用力.md "wikilink")，因此拥有更高的内能。

熔化热的数值在大多数情况下是大于0的，表示物体在熔化时吸热，在凝固时放热，而[氦](../Page/氦.md "wikilink")是唯一的例外。[氦-3](../Page/氦-3.md "wikilink")在温度为0.3[开尔文](../Page/开尔文.md "wikilink")以下时，熔化热小于0。[氦-4在温度为](https://zh.wikipedia.org/wiki/氦-4 "wikilink")0.8[开尔文](../Page/开尔文.md "wikilink")以下是也轻微地显示出这种效应。这说明，在一定的恒定压强下，这些物质凝固时会吸收热量。

## 与溶解度的关系

熔化热数据也能用来计算固体物质在水中的[溶解度](https://zh.wikipedia.org/wiki/溶解度 "wikilink")。在[理想溶液中](https://zh.wikipedia.org/wiki/理想溶液 "wikilink")，[溶质达到饱和时的](https://zh.wikipedia.org/wiki/溶质 "wikilink")[摩尔分数](../Page/摩尔分数.md "wikilink")\(x_2\)是该溶质熔化热、[熔点](https://zh.wikipedia.org/wiki/熔点 "wikilink")\(T_fus\)和溶液[温度](../Page/温度.md "wikilink")的[函数](../Page/函数.md "wikilink")。

\[\ln x_2  = - \frac {\Delta H^\circ_{\mathit{fus}}}{R} \left(\frac{1}{T}- \frac{1}{T_{\mathit{fus}}}\right)\]

这里的R是[普适气体常数](https://zh.wikipedia.org/wiki/普适气体常数 "wikilink")。

比如，298[K](../Page/开尔文.md "wikilink")（约25[℃](https://zh.wikipedia.org/wiki/摄氏度 "wikilink")）时，[对乙酰氨基酚](../Page/对乙酰氨基酚.md "wikilink")在水中的溶解度为：

\[x_2  = \exp {\left(- \frac {28100 \mbox{ J mol}^{-1}} {8.314 \mbox{ J K}^{-1} \mbox{ mol}^{-1}}\left(\frac{1}{298}- \frac{1}{442}\right)\right)}= 0.0248\]

换算为[克](https://zh.wikipedia.org/wiki/克 "wikilink")/[升](../Page/公升.md "wikilink")：

\(\frac{0.0248*\frac{1000\mathrm{g}}{18.053\mathrm{mol^{-1}}}}{1-0.0248}*151.17 \mbox{ mol}^{-1} = 213.4\)

这样计算得出的理论值与实际值（240 g/L）的[误差](../Page/误差.md "wikilink")为11%。由于溶液并不是理想溶液，若将额外的[热容量的影响考虑在内](https://zh.wikipedia.org/wiki/热容量 "wikilink")，将得到更精确的结果。\[12\]

### 证明

固体在[溶剂](../Page/溶剂.md "wikilink")中溶解，达到[溶解平衡](../Page/溶解平衡.md "wikilink")后，溶液中的溶质与未溶固体的[化学势](../Page/化学势.md "wikilink")是相同的：

\[\mu^\circ_{solid} = \mu^\circ_{solution}\,\]

或

\[\mu^\circ_{solid} = \mu^\circ_{liquid} + RT\ln X_2\,\]

其中\(\mu^\circ_{liquid}\)是该条件下，该固体熔液的化学势。这一步利用了理想溶液的假设和[拉乌尔定律](../Page/拉乌尔定律.md "wikilink")。化简后得到：

\[RT\ln X_2  = - (\mu^\circ_{liquid} -  \mu^\circ_{solid})\,\]

又因为：

\[\Delta G^\circ_{\mathit{fus}}  = - (\mu^\circ_{liquid} -  \mu^\circ_{solid})\,\]

其中\(\Delta G^\circ_{\mathit{fus}}\)是摩尔熔化[自由焓变](https://zh.wikipedia.org/wiki/自由焓 "wikilink")。所以溶质固体和溶质熔液之间的化学势差异遵循以下方程：

\[RT\ln X_2  = - ( \Delta G^\circ_{\mathit{fus}})\,\]

应用[吉布斯－亥姆霍茲方程](../Page/吉布斯－亥姆霍茲方程.md "wikilink")：

\[\left( \frac{\partial ( \frac{\Delta G^\circ_{\mathit{fus}} } {T} ) } {\partial T} \right)_{p\,} = - \frac {\Delta H^\circ_{\mathit{fus}}} {T^2}\]

经过计算得到：

\[\left( \frac{\partial ( \ln X_2 ) } {\partial T} \right)  =  \frac {\Delta H^\circ_{\mathit{fus}}} {RT^2}\]

或：

\[\mathrm{d} (\ln X_2)   =  \frac {\Delta H^\circ_{\mathit{fus}}} {RT^2}*\mathrm{d} T\]

对上面的方程等号两边进行[积分](../Page/积分.md "wikilink")（忽略了摩尔熔化焓随温度的改变）

\[\int^{X_2=x_2}_{X_2 = 1} \mathrm{d} (\ln X_2) = \ln x_2  = \int_{T_{\mathit{fus}}}^T \frac {\Delta H^\circ_{\mathit{fus}}} {RT^2}*\mathrm{d}T\]

可以得到最终结果：

\[\ln x_2  = - \frac {\Delta H^\circ_{\mathit{fus}}} {R}\left(\frac{1}{T}- \frac{1}{T_{\mathit{fus}}}\right)\]

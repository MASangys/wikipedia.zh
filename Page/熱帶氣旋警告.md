**熱帶氣旋警告**（）是指全球各處常常受[熱帶氣旋](../Page/熱帶氣旋.md "wikilink")影響的地區，於風暴侵襲時發出的警告，用以通知當地居民及民防組織採取適當的防禦或撤離措施。這些警告涉及警告範圍內可能遭受的災害，而不是單純重複熱帶氣旋的預測路徑及強度，對於保障人命及財產安全尤為重要。

### [廣東省](https://zh.wikipedia.org/wiki/廣東省 "wikilink")

2015年1月1日起，为更好地服务社会公众，增强预警信号的发布效果，使突发气象灾害预警信号图标更加清晰易懂、指示明确，并与国家有关规定相一致，广东省气象局决定启用新版突发气象灾害预警信号图标（电白除外\[2\]）。\[3\]

4.  颱風減弱為[熱帶性低氣壓或變性為](https://zh.wikipedia.org/wiki/熱帶性低氣壓 "wikilink")[溫帶氣旋](../Page/溫帶氣旋.md "wikilink")時，直接[解除颱風警報](https://zh.wikipedia.org/wiki/解除颱風警報 "wikilink")。

## 菲律賓

[菲律賓大氣地球物理和天文服務管理局](../Page/菲律賓大氣地球物理和天文服務管理局.md "wikilink")（PAGASA）負責菲律賓各省的熱帶氣旋警告發佈，警佈分為5種：

| 級別     | 定義                                 |
| ------ | ---------------------------------- |
| 一號風暴信號 | 受熱帶氣旋影響，未來36小時內該區風力可達30－60公里／小時。   |
| 二號風暴信號 | 受熱帶氣旋影響，未來24小時內該區風力可達60－120公里／小時。  |
| 三號風暴信號 | 受熱帶氣旋影響，未來18小時內該區風力可達121－170公里／小時。 |
| 四號風暴信號 | 受熱帶氣旋影響，未來12小時內該區風力可達171－220公里／小時。 |
| 五號風暴信號 | 受熱帶氣旋影響，未來12小時內該區風力可逾220公里／小時。     |

當中「五號風暴信號」為2015年5月21日新增\[19\]，是因應[2013年颱風海燕摧毀菲律賓中部及造成災難性傷亡而增設](../Page/颱風海燕_\(2013年\).md "wikilink")，亦配合同樣因應海燕掠菲而於2014年增設的「超強颱風」分級。於[2016年颱風海馬吹襲菲律賓時首次發出](../Page/颱風海馬_\(2016年\).md "wikilink")。

## 越南

[越南中央水文氣象預報中心](https://zh.wikipedia.org/wiki/越南中央水文氣象預報中心 "wikilink")（NCHMF）負責越南各地及南中國海的熱帶氣旋警告發佈，警佈分為3種：

| 級別       | 定義                                                                               |
| -------- | -------------------------------------------------------------------------------- |
| 低壓區警告    | 受[低壓區影響](https://zh.wikipedia.org/wiki/低壓區 "wikilink")，可能會影響該地之海上作業或是陸上天氣。       |
| 熱帶性低氣壓警告 | 受[熱帶性低氣壓影響](https://zh.wikipedia.org/wiki/熱帶性低氣壓 "wikilink")，可能會影響該地之海上作業或是陸上天氣。 |
| 颱風警告     | 受[颱風](../Page/颱風.md "wikilink")影響，可能會影響該地之海上作業或是陸上天氣並且造成災害。                      |

## 印度

[印度](../Page/印度.md "wikilink")氣象部會於氣旋接近印度沿海地區時，發出兩種氣旋警報。

| 警報名稱                  | 意義                     |
| --------------------- | ---------------------- |
| 氣旋警戒（Cyclone Alert）   | 氣旋將為沿海地區帶來惡劣天氣前48小時發出。 |
| 氣旋警告（Cyclone Warning） | 氣旋將為沿海地區帶來惡劣天氣前24小時發出。 |

於氣旋接近印度時，印度氣象部會每天發佈四至八次風暴消息。

## 澳洲

[澳洲氣象局](../Page/澳洲氣象局.md "wikilink")會於熱帶氣旋接近澳洲沿海地區時，發出兩種熱帶氣旋警報。

| 警報名稱                             | 意義                               |
| -------------------------------- | -------------------------------- |
| 熱帶氣旋觀察預警（Tropical Cyclone Watch） | 預料沿海地區將於24至48小時內吹烈風（每小時63公里）或以上。 |
| 熱帶氣旋警告（Tropical Cyclone Warning） | 沿海地區正在或於24小時內吹烈風（每小時63公里）或以上。    |

熱帶氣旋觀察預警（注意報告）為每6小時發佈一次，而熱帶氣旋警告為每3小時發佈一次；在某些情況下，如果熱帶氣旋逼近沿海地區，氣象局會每小時發佈最新的風暴消息。

## 美國

[美國](https://zh.wikipedia.org/wiki/美國 "wikilink")[國家颶風中心](../Page/國家颶風中心.md "wikilink")為北[大西洋](../Page/大西洋.md "wikilink")及[加勒比海](../Page/加勒比海.md "wikilink")國家發佈以下五種警報，這些警報也同時適用於[加利福尼亞州](https://zh.wikipedia.org/wiki/加利福尼亞州 "wikilink")、[美國墨西哥灣沿岸地區](../Page/美國墨西哥灣沿岸地區.md "wikilink")、[美國東岸](https://zh.wikipedia.org/wiki/美國東岸 "wikilink")、[墨西哥](../Page/墨西哥.md "wikilink")和[中美洲](../Page/中美洲.md "wikilink")的[太平洋沿岸地區](https://zh.wikipedia.org/wiki/太平洋 "wikilink")。此外，中太平洋颶風中心和位於[關島](../Page/關島.md "wikilink")的天氣預測辦公室會分別為[夏威夷群島](../Page/夏威夷群島.md "wikilink")和部分[密克羅尼西亞群島發出警報](https://zh.wikipedia.org/wiki/密克羅尼西亞群島 "wikilink")。 [加拿大](../Page/加拿大.md "wikilink")亦有幾乎相同的一套警告系統，由[加拿大環境部](../Page/加拿大環境部.md "wikilink")氣象局發出。唯一不同之處是加拿大的觀察預警（注意報告）和警告提前發出時間比美國少12小時。\[20\]

| 警報名稱                           | 意義\[21\]                                                                                                          |
| ------------------------------ | ----------------------------------------------------------------------------------------------------------------- |
| 熱帶風暴觀察預警（Tropical Storm Watch） | 預警範圍內的沿海地區**可能**遭受熱帶風暴狀況影響；此注意報會提前48小時發出。                                                                         |
| 熱帶風暴警告（Tropical Storm Warning） | 警告範圍內的沿海地區**會**遭受熱帶風暴狀況影響，風力將達；此警告會提前36小時發出。                                                                      |
| 颶風觀察預警（Hurricane Watch）        | 預警範圍內的沿海地區**可能**遭受颶風狀況影響；此注意報會提前48小時發出。                                                                           |
| 颶風警告（Hurricane Warning）        | 警告範圍內的沿海地區**會**遭受颶風狀況影響，風力將達以上；此警告會提前36小時發出。                                                                      |
| 極端大風警告（Extreme Wind Warning）   | 警告範圍內的某地區**會**遭受三級或以上颶風狀況影響，風力將達或以上；此警告只會在預計熱帶氣旋的[眼牆](https://zh.wikipedia.org/wiki/眼牆 "wikilink")（最強的風）影響該地區才發出。 |

如果熱帶氣旋的預測強度或路徑無法確定（如接近颶風程度的熱帶風暴，或某地區接近暴風及颶風圈邊界時），部分地區可能會同時發布熱帶風暴警告及颶風觀察預警（注意報告）。

即使風力减弱至颶風程度以下，但如果潮水依然高漲或海面仍有巨浪，颶風觀察預警（注意報告）仍可能繼續生效。

當熱帶氣旋於北大西洋及東太平洋出現時，國家颶風中心會每六小時發佈一次公眾風暴消息。當熱帶氣旋逼近沿海地區時，當地的氣象組織會視乎情況，更頻密地發布最新消息。

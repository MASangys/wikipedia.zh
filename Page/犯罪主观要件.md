**犯罪主观要件**（subjective elements of crime），是一类[刑法学和](https://zh.wikipedia.org/wiki/刑法学 "wikilink")[犯罪学的定义](https://zh.wikipedia.org/wiki/犯罪学 "wikilink")。刑法规定的成立犯罪所必须具备的行为人对自己实施的危害社会的行为及其结果所持的心理态度，包括犯罪的故意、过失，犯罪的[动机](../Page/动机.md "wikilink")、[目的](https://zh.wikipedia.org/wiki/目的 "wikilink")。

其中犯罪的[故意及](https://zh.wikipedia.org/wiki/故意 "wikilink")[过失合称为罪过](https://zh.wikipedia.org/wiki/过失 "wikilink")。[罪过是一切犯罪构成的必备要件](https://zh.wikipedia.org/wiki/罪过 "wikilink")，而犯罪的目的只是某些犯罪构成的必备要件，犯罪动机不是犯罪构成的必备要件，但它是量刑时需要考虑的一个因素。犯罪的主观方面具有以下特征\[1\]。

## 犯罪故意、过失、意外事件

犯罪主观要件主要包括[犯罪故意](https://zh.wikipedia.org/wiki/犯罪故意 "wikilink")、[犯罪过失](https://zh.wikipedia.org/wiki/犯罪过失 "wikilink")、[意外事件](https://zh.wikipedia.org/wiki/意外事件 "wikilink")。犯罪故意是具有社会危害性的特定内容，具体表现为行为人对自己实施的危害行为及其危害结果所持的心理态度\[2\]，根据认识因素和意志因素的不同，主要分为[直接故意和](https://zh.wikipedia.org/wiki/直接故意 "wikilink")[间接故意](https://zh.wikipedia.org/wiki/间接故意 "wikilink")\[3\]。

犯罪的过失，就是指行为人应当预见自己的行为可能发生危害社会的结果，因为疏忽大意而没有预见，或者已经预见而轻信能够避免的一种心理态度\[4\]。

|            | 直接故意               | 间接故意             | 疏忽大意的过失            | 过于自信的过失            | 意外事件               |
| ---------- | ------------------ | ---------------- | ------------------ | ------------------ | ------------------ |
| 认识因素\[5\]  | 明知自己的行为必然或可能发生危害结果 | 明知自己的行为可能发生危害结果  | 没有预见到自己的行为可能发生危害结果 | 已经预见到自己的行为可能发生危害结果 | 没有预见到自己的行为可能发生危害结果 |
| 有无预见危害后果可能 | 有                  | 有                | 有，且行为人有义务预见        | 有，但行为人轻信能够避免       | 无                  |
| 意志因素       | 希望，即积极追求危害结果的发生    | 放任，即对危害结果的发生听之任之 | 反对发生危害结果           | 反对发生危害结果           | 反对发生危害结果           |
| 有无犯罪动机、目的  | 有                  | 无                | 无                  | 无                  | 无                  |
| 有无犯罪未完成形态  | 有                  | 无                | 无                  | 无                  | 无，不构成犯罪            |

## 主观认识错误和期待可能性

刑法学上所说的认识错误，是指行为人对自己的行为性质、后果和有关的事实存在不正确的认识。它可以分为两类：一是行为人在法律上的认识错误；二是行为人在事实上的认识错误\[6\]\[7\]。

| 分类                            | 具体分类                                                                     | 处理                                   |
| ----------------------------- | ------------------------------------------------------------------------ | ------------------------------------ |
| 法律认识错误（按法律的实际规定处理）            | 误认为无罪为有罪                                                                 | 无罪                                   |
| 误认为有罪为无罪                      | 有罪                                                                       |                                      |
| 对自己实施的犯罪行为在罪名、罪数、量刑等方面有不正确的理解 | 按法律规定处理                                                                  |                                      |
| 犯罪构成事实认识错误（按主客观相统一原则处理）       | 对行为性质的认识错误                                                               | 不成立故意犯罪，如果行为人有过失，构成过失犯罪，如无过失，则属于意外事件 |
| 对行为手段的认识错误                    | 将不能犯手段当成能犯手段：犯罪未遂；将能犯手段当成不能犯手段：不成立故意犯罪，如果行为人有过失，构成过失犯罪，如无过失，则属于意外事件\[8\] |                                      |
| 对行为对象的认识错误                    | 所发生的错误认识与原认识属于同一犯罪构成的，仍成立原犯罪的既遂；否则按未遂或过失、意外事件处理\[9\]。                    |                                      |
| 打击错误                          |                                                                          |                                      |
| 对因果关系具体样态的认识错误                | 不影响定罪和量刑                                                                 |                                      |

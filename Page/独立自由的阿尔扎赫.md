《**獨立自由的阿爾札赫**》（）是尚未受到普遍承認的[阿尔扎赫共和国的國歌](../Page/阿尔扎赫共和国.md "wikilink")，於1992年宣布成為國歌。由伏爾丹‧哈克柏楊（Վարդան
Հակոբյանը）作曲，亞爾曼‧納西柏楊（Արմեն Նասիբյան）填詞。

## 歌詞

<div style="float:left;margin:0 0 .5em 1em;" class="toccolours">

<center>

**Ազատ ու անկախ Արցախ** **獨立自由的阿爾札赫**

</center>

|                                  |
| -------------------------------- |
|                                  |
| Ազատ ու անկախ Արցախ,             |
| Քո տուն-ամրոցը կերտեցինք,        |
| Պատմությունը մեր երկրի,          |
| Մեր սուրբ արյամբ մենք սերտեցինք։ |
|                                  |
| Դու բերդ ես անառիկ,              |
| Բարձունք սրբազան, վեհ անուն,     |
| Մասունք աստվածային,              |
| Քեզնով ենք հավերժանում։          |
|                                  |
| Դու մեր լույս հայրենիք,          |
| Երկիր, հայրենյաց դուռ սիրո       |
| Ապրիր դու միշտ խաղաղ,            |
| Մեր հին ու նոր Ղարաբաղ։          |
|                                  |
| Քաջերն ենք մենք հայկազուն,       |
| Մռավ ենք, Քրիս ենք ու Թարթառ,    |
| Մեր վանքերով լեռնապահ՝           |
| Անհաղթելի մի բուռ աշխարհ։        |
|                                  |

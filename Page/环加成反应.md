**环加成反应**（英文：Cycloaddition）是两个或多个[不饱和化合物](https://zh.wikipedia.org/wiki/不饱和化合物 "wikilink")（或同一化合物的不同部分）结合生成环状[化合物](../Page/化合物.md "wikilink")，并伴随有系统总[键级数减少的化学反应](https://zh.wikipedia.org/wiki/键级 "wikilink")。\[1\]它可以是[周环反应或非](https://zh.wikipedia.org/wiki/周环反应 "wikilink")[协同的](https://zh.wikipedia.org/wiki/协同反应 "wikilink")[分步反应](https://zh.wikipedia.org/wiki/分步反应 "wikilink")。逆过程称为**环消除反应**。

环加成反应的两种主要类型是[狄尔斯-阿尔德反应和](https://zh.wikipedia.org/wiki/狄尔斯-阿尔德反应 "wikilink")[1,3-偶极环加成反应](../Page/1,3-偶极环加成反应.md "wikilink")。

根据[前线轨道理论](../Page/前线轨道理论.md "wikilink")，两个分子之间的环加成反应符合以下几点：

1.  环加成反应中起决定作用的是一个分子的[HOMO与另一个分子的](https://zh.wikipedia.org/wiki/HOMO "wikilink")[LUMO](https://zh.wikipedia.org/wiki/LUMO "wikilink")。电子在反应过程中，从一个分子的HOMO流入另一个分子的LUMO。这主要是由于[反键效应的缘故](https://zh.wikipedia.org/wiki/反键效应 "wikilink")，使得两个HOMO相互作用时体系的能量升高，而在LUMO与HOMO作用时体系能量降低，趋于稳定；
2.  在形成新的σ键时，两个起作用的轨道必须发生同[位相的重叠](https://zh.wikipedia.org/wiki/位相 "wikilink")。同位相重叠使能量降低，互相吸引，而异位相重叠使能量升高，产生排斥作用；
3.  相互作用的两个轨道能量必须接近，而且能量越接近，反应就越容易进行。这是因为能差的减小会使新形成的[成键轨道能级降低](https://zh.wikipedia.org/wiki/成键 "wikilink")，从而使体系的能量降低值增大，体系趋于稳定。

## 分类

环加成反应有两种分类方法。一种是根据参加反应的原子数标记，写为\(( i + j + ... )\,\)的形式。*i*和*j*等是参加反应的每个分子或某个分子的不同部分提供的原子数，反应后形成的新环含有的原子数应为*i*+*j*+...。例如，[狄尔斯-阿尔德反应记为](https://zh.wikipedia.org/wiki/狄尔斯-阿尔德反应 "wikilink")(4+2)环加成反应，[1,3-偶极环加成反应](../Page/1,3-偶极环加成反应.md "wikilink")（如[烯烃臭氧化反应的第一步加成](https://zh.wikipedia.org/wiki/烯烃臭氧化反应 "wikilink")）记为(3+2)环加成反应。

第二种是IUPAC鼓励的分类方法，也就是按照参加反应的电子数来标记，写为\([ i + j + ... ]\,\)。在这种命名方法下，[降冰片二烯与活化](https://zh.wikipedia.org/wiki/降冰片二烯 "wikilink")[炔烃](../Page/炔烃.md "wikilink")的反应记为\[2+2+2\]环加成反应，狄尔斯-阿尔德反应记为\[4+2\]环加成反应，而烯烃臭氧化的第一步反应则记为\[2+2\]环加成反应。此外，还可以进一步用下标表明反应的其他性质。a代表[异面](https://zh.wikipedia.org/wiki/异面 "wikilink")，s代表[同面](https://zh.wikipedia.org/wiki/同面 "wikilink")，写在数字的后下方；轨道性质（σ、π、n）写在数字的前下方。这样，狄尔斯-阿尔德反应便可记作\([ _{\pi} 4_{s} + _{\pi} 2_s]\,\)，表明一个反应物给出4个π电子，另一个反应物给出2个π电子，它们发生的是同面-同面加成。

## 反应机理

在加热时发生环加成反应的原料多为(4n+2)体系，受[轨道对称性影响](https://zh.wikipedia.org/wiki/轨道对称性 "wikilink")，反应以[同面](https://zh.wikipedia.org/wiki/同面 "wikilink")-同面加成或[异面](https://zh.wikipedia.org/wiki/异面 "wikilink")-异面加成（少见）进行。同面指π键同一侧的两个轨道瓣进行反应，异面对应异侧。少数4n体系反应物的环加成反应是允许的。它们为同面-异面加成机理，例如在[烯酮](../Page/烯酮.md "wikilink")的二聚反应中，烯酮正交的*p*轨道促使交叉中间体的生成，使得反应可以进行，生成产物[二烯酮](https://zh.wikipedia.org/wiki/二烯酮 "wikilink")。

4n体系底物可以在受光作用下，[HOMO](https://zh.wikipedia.org/wiki/HOMO "wikilink")（π成键轨道）的一个电子被激发到[LUMO](https://zh.wikipedia.org/wiki/LUMO "wikilink")（π\*[反键](https://zh.wikipedia.org/wiki/反键 "wikilink")）上，这样激发态的HOMO与基态的LUMO波相符合，可以发生同面-同面加成。下面[肉桂酸](../Page/肉桂酸.md "wikilink")受光激发二聚为环丁烷衍生物的反应便是一个例子。\[2\]

一般地讲，环加成反应具有以下选择性规则：

| 参加反应的π电子数之和                  | 4n+2 | 4n   |
| ---------------------------- | ---- | ---- |
| 同面-同面                        | Δ    | *h*ν |
| 允许                           | 禁阻   | 禁阻   |
| 同面-异面                        | Δ    | *h*ν |
| 禁阻                           | 允许   | 允许   |
| 注：“允许”指“对称性允许”，“禁阻”指“对称性禁阻”。 |      |      |


以上规则只表明反应按照协同机理进行的[活化能大小](https://zh.wikipedia.org/wiki/活化能 "wikilink")，并不排除反应按照其他[机理进行](../Page/反应机理.md "wikilink")。例如，按反应选择性规则，(2+2)环加成反应在加热时是对称性禁阻的，但二氟二氯乙烯在200°C时形成四氟四氯环丁烷似乎与以上规则相矛盾。实际上，该反应是通过一个双[自由基](../Page/自由基.md "wikilink")的反应中间体进行的，并不经由协同机理。这也是很多(2+2)[环化反应看上去不符合选择性规则的原因](https://zh.wikipedia.org/wiki/环化反应 "wikilink")。

有些具有张力的[环丙烷环系](https://zh.wikipedia.org/wiki/环丙烷 "wikilink")，由于具有显著的π键性质，也可以发生环加成反应。例如下图中的[四环庚烷与](https://zh.wikipedia.org/wiki/四环庚烷 "wikilink")[DMAD的反应](https://zh.wikipedia.org/wiki/DMAD "wikilink")：

## “表环加成”

有些金属催化的环加成反应属于多步的非[协同反应](https://zh.wikipedia.org/wiki/协同反应 "wikilink")，因此被称为“表环加成反应”（Formal cycloaddition），以将它们与一步完成的[周环反应相区别](https://zh.wikipedia.org/wiki/周环反应 "wikilink")。一般来讲，表环加成反应包括的情况有：反应中间体为带有电荷或为[自由基](../Page/自由基.md "wikilink")；产物由多步反应得到。下面便是一个表\[3+3\]环加成反应，反应物是一个环状[α,β-不饱和酮和一个](https://zh.wikipedia.org/wiki/α,β-不饱和酮 "wikilink")[烯胺](https://zh.wikipedia.org/wiki/烯胺 "wikilink")，在[正丁基锂](../Page/正丁基锂.md "wikilink")作用下，发生[烯胺烷基化和](https://zh.wikipedia.org/wiki/烯胺烷基化 "wikilink")[1,2-加成的](https://zh.wikipedia.org/wiki/亲核加成 "wikilink")[连续反应](https://zh.wikipedia.org/wiki/连续反应 "wikilink")，得到类似于环加成的产物。\[3\]

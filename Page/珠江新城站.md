**珠江新城站**是[广州地铁](../Page/广州地铁.md "wikilink")[3号线及](../Page/广州地铁3号线.md "wikilink")[5号线的换乘站](../Page/广州地铁5号线.md "wikilink")\[1\]，车站位於[天河区](../Page/天河区.md "wikilink")华夏路花城大道路口的地底。于2005年12月26日随3号线首通段开通而正式启用。

跟广州地铁2006年之前开通的車站一樣，在通車初期，本站的英文站名以拼音「Zhu jiang xin cheng」表示。

## 車站結構

### 車站樓層

珠江新城站共设有3层，地面为华夏路、花城大道、高德置地廣場、富力盈凱廣場、中國海關、廣州市政務服務中心、以及其它商場或寫字樓等建築；地下一层为车站站厅；地下二层为3号线站台；地下三层为5号线站台。

因本站為兩線統一設計且同步建成的車站，雖未能同步啟用，但整個車站均使用了搪瓷板裝修。車站配色為藍色。惟兩線月台書法字字體有所差別：3號線使用手寫字體；5號線使用葉根友電腦字體。

### 站廳

珠江新城站站廳為一個十字型的共用站廳，5號線站廳為東西向，3号线站厅为南北向，均位于地下一层。

為了方便乘客進出，站廳中部大部分被劃作非付費區並連通各出入口通道；付費區分別位處十字形站廳的四個端頭。车站商店大都分设于站厅东侧非收费区，站厅付費區內設有多組[扶手電梯](https://zh.wikipedia.org/wiki/扶手電梯 "wikilink")、专用电梯（[升降机](../Page/升降机.md "wikilink")）以及樓梯供乘客往返不同线路的月台。

#### 商店及自助服務

現時，珠江新城站內提供不同類型的商店供乘客購物或進食，例如有[便利店](../Page/便利商店.md "wikilink")、凉茶店、[麵包糕餅店](https://zh.wikipedia.org/wiki/麵包糕餅店 "wikilink")、[書店](../Page/書店.md "wikilink")以及[洗衣店等](https://zh.wikipedia.org/wiki/洗衣店 "wikilink")。本站之前设有羊城通客户服务中心以及广州地铁客户服务中心，到后来亦关闭。

此外，珠江新城站也有不少自助服務設施供乘客使用，包括[民生银行以及](https://zh.wikipedia.org/wiki/民生银行 "wikilink")[招商银行](../Page/招商银行.md "wikilink")的[自動櫃員機](../Page/自動櫃員機.md "wikilink")、[自動售賣機](../Page/自動販賣機.md "wikilink")、自動照相機、“好易”自助服务等。

### 月台以及换乘

本站3号线设有一个岛式月台，位于华夏路地底；5号线设有两个侧式月台，位于花城大道地底。兩線月台組成一個十字型。3號線月台在負二層，5號線月台在負三層。另外，本站5号线的侧式月台是对向式侧式月台设计，只是两个月台之间被墙壁完全阻隔，但位處5號線站台頭尾兩端仍可窺見另一方向的列車進出月台。

另外，5号线月台西边设有一条渡线，當5號線東段出現故障時，往文冲站的列車將會通過此渡线折返，以本站為臨時總站。此時往猎德至文冲沿途各站則需轉乘其它交通工具。

本站3号线和5号线采用节点换乘的方式，由于本站5号线有两个侧式月台，因此从3号线月台处换乘5号线时，需注意月台的导向指示进行换乘，到达前往相应的方向的扶手电梯以及楼梯后，搭乘扶手电梯和楼梯即可前往该方向的5号线月台\[2\]。但受此月台佈局的限制，部分來自3號線的換乘乘客可能在下車後需要步行大半段3號線月台，尋找自己需要前往的5號線方向。 {{ multiple image | align = center | direction = vertical | width = 550 | image1 = Zhujiang New Town Station For Line 3 2014 07.JPG | caption1 =

### 出入口

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Exit A1, Zhujiang Newtown Station.jpg | caption1 = A1出入口 | image2 = Zhujiangnewtownstation.JPG | caption2 = B1出入口 | image3 = Zhujiang New Town Station Exit B2.jpg | caption3 = B2出入口 }} 珠江新城站现时共设有6个出入口，其中4個位于华夏路西侧的出入口獨立通往地面，另外2个位於華夏路東側的出入口则分别连接位于华夏路东侧的高德置地广场，富力盈凯广场、[富力中心](../Page/富力中心.md "wikilink")和合景国际金融广场，其中D出入口通道亦連通花城廣場地下空間，並可通往[西塔](../Page/广州国际金融中心.md "wikilink")、[花城广场和](../Page/珠江新城.md "wikilink")[APM线](../Page/珠江新城旅客自动输送系统.md "wikilink")**[花城大道站](https://zh.wikipedia.org/wiki/花城大道站 "wikilink")**。

C和D出入口在車站啟用早期原為獨立通往地面的出入口。後來改為分別接入高德置地广场和富力盈凯广场等物業的地庫後，兩個出入口編號均被取消，原來獨立通往地面的結構也已被廢棄或拆除。因此現時乘客若從華夏路東側进出地铁站就必须要经过写字楼或者商场的内部，使用时间受不同物业本身的开放时间的限制\[3\]。2019年5月起，運營方重新恢復這兩個出入口的編號。

另外，本站A1出入口在2010年5月至11月期间进行了加装扶梯工程，因此在施工过程中本站A1出入口暂时封闭\[4\]。

## 接驳交通

珠江新城站周边设花城大道（华穗路口）、华城路口、市政务服务中心等多个巴士站，方便乘客利用巴士前往珠江新城周边等地。

另外，本站離鄰近的APM線[花城大道站只有約](https://zh.wikipedia.org/wiki/花城大道站 "wikilink")300米的距離，惟兩者並非3、5號線與APM線的正式換乘站。乘客可在本站D出口出站後通過花城匯步行前往花城大道站A出口。

## 利用情況

珠江新城站是**[珠江新城](../Page/珠江新城.md "wikilink")中央商务区**的配套车站，3号线开通之初期，车站周边仍處於初期开发阶段，導致本站的客流一直處於極低的水平\[5\]；后来3号线一期全線通車，以及珠江新城的各个写字楼陆续投入建成使用，同时广州海关、广州市政务服务中心等各个政府机构遷移進入珠江新城后，本站的出入闸客流开始增加。

时至今日，珠江新城的发展已初具规模；隨著亞運前夕5号线的开通运营，和花城广场及附近公共設施或商場陸續開幕，每天均有大量乘客利用本站出入閘或換乘，尤其是節假日更是挤得水泄不通，是广州地铁站中人流最多及最繁忙的換乘站之一，在此站候車經常需輪候兩班甚至三班列車或以上才能上車，车站早晚高峰月台都会聚集大量乘客，兩線換乘扶梯和樓梯口在高峰期都會造成擁堵，經常需要實施客流控制。现时大部份使用該站出入閘的人流都是上班人士、酒店住客、娛樂和購物人士为主。

后来同为[珠江新城](../Page/珠江新城.md "wikilink")中央商务区配套的APM线开通，分流了部分來往花城广场以及少年宫、圖書館和博物馆等游客的客流，同时也分流了部分利用3号线前往本站的上班人士。但本站的客流依然居高不下。

另外，如遇上附近[花城广场](../Page/花城广场.md "wikilink")举办[广州国际灯光节](../Page/广州国际灯光节.md "wikilink")等大型活动时，车站部分出入口将实行单向人流进（出）站等管制措施，同时部分3号线或5号线列车将会不停站通过本站\[6\]\[7\]\[8\]。

## 历史

在1997年的《廣州市城市快速軌道交通線網規劃研究（最終報告）》中，當時的5號線環線的東段（即3号线的體育西路至客村段），以及3号线（即现时的5号线）设有**珠江新城站**，本站被规划为一座换乘车站，位置与现时一致。後來於2000年以及2003年方案中，原5號線環線的東段易名为3号线并南北延长至[番禺区](../Page/番禺区.md "wikilink")以及[新白云国际机场](https://zh.wikipedia.org/wiki/广州白云国际机场 "wikilink")，原3号线亦易名为5号线，本站仍為該兩線的換乘站。

车站随3号线在2001年12月26日开始动工建设，由于车站规划时就明确落实本站为换乘车站，在车站建设时就已经预留了5号线的两个侧式月台。2004年，五號線首期工程動工建設。2005年12月26日，3號線首通段（广州东站-客村）開通試運營，本站啟用。2009年12月28日上午，廣州地鐵五號線開通儀式於珠江新城站地面舉行\[9\]，同日下午2時，全線正式通車，5号线两个侧式月台启用，本站正式成为换乘车站。

## 未來發展

未來珠江新城站將對站廳公共區佈局進行重新佈置，重劃付費區和非付費區，包括把現時互不相通的四個付費區連接，合併成一個位於站廳中部，呈十字形的付費區；同時把非付費區調整為環繞付費區一圈，令車站各出入口在站廳維持相互連通。

付費區合併後，3號線和5號線將可經過站廳換乘。屆時換乘節點的扶梯將會改為單向運行，僅供來自5號線的乘客換乘至3號線。兩線的換乘客亦可下車後就近乘搭扶梯前往站廳，再前往另一條線的月台，從而有效分流現時兩線月台和換乘節點樓扶梯的客流壓力。

連接站廳和兩線月台的升降機因位置佈局原因，站廳佈局重新佈置後將會與主付費區分離，屆時會在升降機出入口處另外劃出付費區並設置雙向閘機，可供乘客繼續自行使用升降機來往站廳和月台\[10\]\[11\]。

**班士利2013年至2014年球季**（）是[巴恩斯利足球俱乐部](../Page/巴恩斯利足球俱乐部.md "wikilink")自1898年加入[英格蘭足球聯賽後第](https://zh.wikipedia.org/wiki/英格蘭足球聯賽 "wikilink")105個球季，亦是連續第8個處於第二級聯賽的球季，[上季再次以第](../Page/班士利2012年至2013年球季.md "wikilink")21位結束球季僅免降班。

## 記事

  - 2013年11月30日於球隊主場0-3負於[伯明翰城後](../Page/伯明翰足球會.md "wikilink")，成績位列榜末，執教未滿一年的領隊[大卫·费列哥夫](../Page/大卫·费列哥夫.md "wikilink")（David Flitcroft）終被辭退\[1\]。
  - 2013年12月17日邀請[丹尼·威爾遜](../Page/丹尼·威爾遜_\(1960年出生足球運動員\).md "wikilink")（Danny Wilson）再作馮婦，於離隊15年後重掌帥印\[2\]。
  - 2014年4月26日作客1-3負於[-{zh-hans:米德尔斯堡; zh-hk:米杜士堡;}-](https://zh.wikipedia.org/wiki/米德尔斯堡足球俱乐部 "wikilink")，提前一輪確定來季降班[英甲](https://zh.wikipedia.org/wiki/英甲 "wikilink")\[3\]。

## 球衣

本季球衣繼續由[-{耐克}-](https://zh.wikipedia.org/wiki/Nike "wikilink")（-{Nike}-）設計及供應，主場球衣一如以往維持傳統的紅衫白褲紅襪；而作客為全藍款式，上衫左方有白直間，而贊助商的[商標直排置於白直間內](https://zh.wikipedia.org/wiki/商標 "wikilink")。而主要[贊助](https://zh.wikipedia.org/wiki/贊助 "wikilink")[廣告](../Page/廣告.md "wikilink")仍然是「*CK Beckett*」。  |  |  |  |}

**电环化反应**是[周环反应的一类](https://zh.wikipedia.org/wiki/周环反应 "wikilink")，反应中[共轭体系](../Page/共轭体系.md "wikilink")两端的原子环合形成新的[σ键](https://zh.wikipedia.org/wiki/σ键 "wikilink")，形成比原来分子少一个[π键的产物](https://zh.wikipedia.org/wiki/π键 "wikilink")。它的逆反应也属于电环化反应，有时为了区分，将前者成环反应称为“电环合反应”。\[1\]电环化反应是[立体选择性的反应](https://zh.wikipedia.org/wiki/立体选择性 "wikilink")，通常使用的反应底物是[环烯烃和对应的](https://zh.wikipedia.org/wiki/环烯烃 "wikilink")[共轭烯烃](https://zh.wikipedia.org/wiki/共轭多烯多烯 "wikilink")。它于1960年前后被发现。

[ElectrocyclicRxn.png](https://zh.wikipedia.org/wiki/File:ElectrocyclicRxn.png "fig:ElectrocyclicRxn.png")

为了使π电子环合成为σ键，烯烃末端碳原子的键必须旋转，而旋转的方向可以是两个键朝同一方向旋转，或两个键朝不同方向旋转，分别称为[顺旋和](https://zh.wikipedia.org/wiki/顺旋 "wikilink")[对旋](https://zh.wikipedia.org/wiki/对旋 "wikilink")。顺旋又可分为顺时针顺旋和反时针顺旋两种，对旋又可分为内向对旋和外向对旋两种，但这个因素一般很少考虑。

根据[分子轨道对称守恒原理](../Page/分子轨道对称守恒原理.md "wikilink")，为了发生电环化反应，共轭烯烃[HOMO两端的两个](https://zh.wikipedia.org/wiki/HOMO "wikilink")*p*轨道必须发生同位相的重叠。由于链形烯烃总π电子数会对HOMO的对称性造成影响，加热或光照也会使[分子轨道能级图上的](https://zh.wikipedia.org/wiki/分子轨道能级图 "wikilink")[电子排布](../Page/电子排布.md "wikilink")发生改变，因此电环化反应存在以下选择性的规则，可用于预测某一反应的产物：

| 链形共轭烯烃的π电子数                  | 4n+2 | 4n   |
| ---------------------------- | ---- | ---- |
| 顺旋                           | Δ    | *h*ν |
| 禁阻                           | 允许   | 允许   |
| 对旋                           | Δ    | *h*ν |
| 允许                           | 禁阻   | 禁阻   |
| 注：“允许”指“对称性允许”，“禁阻”指“对称性禁阻”。 |      |      |

光照时HOMO一个电子被激发到[LUMO上去](https://zh.wikipedia.org/wiki/LUMO "wikilink")，使得反应的选择性颠倒，禁阻变为允许，允许变为禁阻。以上规则只表明反应按照[协同反应机理进行时的](https://zh.wikipedia.org/wiki/协同反应 "wikilink")[活化能高低](https://zh.wikipedia.org/wiki/活化能 "wikilink")，并不排除反应按照其他机理进行。

电环化反应是[可逆反应](https://zh.wikipedia.org/wiki/可逆反应 "wikilink")，正逆反应途径是相同的，因此需要注意基态时，反应平衡朝哪一个方向进行更为有利。

[苯并环丁烷发生的电环化反应是电环化反应中研究较多的一类](https://zh.wikipedia.org/wiki/苯并环丁烷 "wikilink")。以下图为例，苯并环丁烷加热顺旋开环，生成一个具有类[醌](../Page/醌.md "wikilink")结构的不稳定的双烯体。它可以和强[亲双烯体](https://zh.wikipedia.org/wiki/亲双烯体 "wikilink")（如[顺丁烯二酸酐](../Page/顺丁烯二酸酐.md "wikilink")）发生[狄尔斯-阿尔德反应生成](https://zh.wikipedia.org/wiki/狄尔斯-阿尔德反应 "wikilink")[内型的加合物](https://zh.wikipedia.org/wiki/endo "wikilink")。苯并环丁烷的开环反应产率因此与取代基R具有很大的关系，\[2\]例如，在110°C和[甲苯](../Page/甲苯.md "wikilink")作溶剂的条件下，随着R由[甲基](https://zh.wikipedia.org/wiki/甲基 "wikilink")、异丁基甲基变为三甲基硅基甲基，产率也随着上升。

[Electrocyclic_reaction_Matsuya_2006.png](https://zh.wikipedia.org/wiki/File:Electrocyclic_reaction_Matsuya_2006.png "fig:Electrocyclic_reaction_Matsuya_2006.png")

[Nazarov成环反应是二乙烯基酮环化成为](https://zh.wikipedia.org/wiki/Nazarov成环反应 "wikilink")[环戊烯酮的电环化反应](https://zh.wikipedia.org/wiki/环戊烯酮 "wikilink")。很多带正电荷或负电荷的共轭烯烃也可以发生电环化反应，比如，[环丙烷正离子广义上也属于](https://zh.wikipedia.org/wiki/环丙烷 "wikilink")4n+2体系，因此也可以在加热时发生对旋开环，生成烯烃。

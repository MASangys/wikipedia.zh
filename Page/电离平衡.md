**电离平衡**是一种[化学现象](https://zh.wikipedia.org/wiki/化学 "wikilink")，通常发生在具有[极性](https://zh.wikipedia.org/wiki/极性键 "wikilink")[共价键](../Page/共价键.md "wikilink")的[化合物](../Page/化合物.md "wikilink")溶于[水](../Page/水.md "wikilink")中的情况。

[Ionization_balance.png](https://zh.wikipedia.org/wiki/File:Ionization_balance.png "fig:Ionization_balance.png")

## 概念与定义

具有极性共价键的[弱电解质](https://zh.wikipedia.org/wiki/弱电解质 "wikilink")(例如部分[弱酸](https://zh.wikipedia.org/wiki/弱酸 "wikilink")、[弱碱](https://zh.wikipedia.org/wiki/弱碱 "wikilink"))溶于水时，其[分子](../Page/分子.md "wikilink")可以微弱[电离出](https://zh.wikipedia.org/wiki/电离 "wikilink")[离子](../Page/离子.md "wikilink")；同时，[溶液](../Page/溶液.md "wikilink")中的相应离子也可以结合成分子。一般地，自上述反应开始起，弱电解质分子电离出离子的[速率](../Page/速率.md "wikilink")不断降低，而离子重新结合成弱电解质分子的速率不断升高，当两者的反应速率相等时，溶液便达到了电离平衡。此时，溶液中电解质分子的浓度与离子的浓度分别处于稳定状态，不再发生变化。

用简单的语言概括电离平衡的定义，即：在一定条件下，弱电解质的离子化速率等于其分子化速率。

在右侧的示意图中，横轴表示反应时间，纵轴表示反应速率，曲线v1表示弱电解质分子电离出离子的速率，曲线v2表示离子结合成弱电解质分子的速率。如图所示，经过时间*t*<sub>0</sub>，两个反应的速率相等，此时溶液达到电离平衡。

## 平衡方向的改变

在上述反应过程中，弱电解质分子电离与离子结合成为分子，二者共同构成一组[可逆反应](https://zh.wikipedia.org/wiki/可逆反应 "wikilink")。常以弱电解质分子电离出离子的反应方向为正反应方向，以离子重新结合成弱电解质分子的反应方向为逆反应方向。

与化学平衡一样，电离平衡是可以因为条件变化(如浓度、温度、酸碱性等)而移动的。

以下列表归纳了外界条件变化对电离平衡及*c*(H<sup>+</sup>)的影响，这里假设弱电解质AB可以电离成(A<sup>+</sup>)和(B<sup>−</sup>),且电离过程吸热，则

|                      |                                           |
| -------------------- | ----------------------------------------- |
| 电离方程式                | AB＝(可逆反应)＝(A<sup>+</sup>)+(B<sup>−</sup>) |
| 条件改变内容               | *v*(正),*v*(逆)关系                           |
| 增大*c*(AB)            | *v*(正)＞*v*(逆)                             |
| 减小*c*(AB)            | *v*(正)＜*v*(逆)                             |
| 增大*c*(A<sup>+</sup>) | *v*(正)＜*v*(逆)                             |
| 减小*c*(A<sup>+</sup>) | *v*(正)＞*v*(逆)                             |
| 增大*c*(B<sup>−</sup>) | *v*(正)＜*v*(逆)                             |
| 减小*c*(B<sup>−</sup>) | *v*(正)＞*v*(逆)                             |
| 稀释溶液                 | *v*(正)＞*v*(逆)                             |
| 升高温度                 | *v*(正)＞*v*(逆)                             |
| 降低温度                 | *v*(正)＜*v*(逆)                             |

为了便于理解，可以参考醋酸(CH<sub>3</sub>COOH)的电离平衡，将CH<sub>3</sub>COOH代入上表的AB，将H<sup>+</sup>代入A<sup>+</sup>，将CH<sub>3</sub>COO<sup>−</sup>代入B<sup>−</sup>。

另外，电离平衡也遵循[勒夏特列原理](https://zh.wikipedia.org/wiki/勒夏特列原理 "wikilink")。

## 示例

  - 醋酸电离成氢离子和醋酸根离子

<chem>CH3COOH \<=\> H+ + CH3COO-</chem>

  - 氨水电离成铵根离子和氢氧根离子

<chem>NH3.H2O \<=\> NH4+ + OH-</chem>

  - 磷酸的分步电离

<chem>H3PO4 \<=\> H+ + H2PO4-</chem>

<chem>H2PO4- \<=\> H+ + HPO4^{2-}</chem>

<chem>HPO4^{2-} \<=\> H+ + PO4^{3-}</chem>

多元弱酸、弱碱的酸碱性主要是由其第一步电离所决定的，因为每一步的电离常数（*K*<sub>电离</sub>）往往相差10<sup>4</sup> - 10<sup>5</sup>倍。

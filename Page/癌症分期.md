**癌症分期**（）是決定[癌症](../Page/癌症.md "wikilink")發展與擴散程度的方法。當代的作法是將一種癌症以數字分成第一到第五期，第一期指的是能分離出癌細胞，第五期代表癌細胞已經擴散到測量方法的極限。癌症的分期一般會考慮[腫瘤大小](https://zh.wikipedia.org/wiki/腫瘤 "wikilink")、是否已經侵犯到鄰近的[器官](../Page/器官.md "wikilink")、有多少癌細胞擴散到鄰近區域的[淋巴結](../Page/淋巴結.md "wikilink")（如果已經開始擴散到淋巴結的話），以及是否出現在遠端的部位當中（[遠端轉移](../Page/遠端轉移.md "wikilink")）。

## 分期系統

不同種類的癌症，會有不同的分期系統（如[乳癌](../Page/乳癌.md "wikilink")和[肺癌](../Page/肺癌.md "wikilink")）。然而部分癌症沒有分期系統。雖然部分癌症的分期系統仍然有多個版本，一般被認可的癌症分期系統是由所提出，而也有同樣的定義分類。

分期系統會因為疾病的表現，在各疾病之間有所不同。

### 血液

  - [淋巴瘤](../Page/淋巴瘤.md "wikilink")：使用。
  - [霍奇金淋巴瘤](../Page/霍奇金淋巴瘤.md "wikilink")：使用安阿伯分期系統，但分成第一到第四期，每期進一步分成A和B，依據病患是否出現症狀（如發燒）而定。這被稱做「考德茲伍德系統」（）或「改良版安阿伯分期系統」（）\[1\]。

### 固態瘤

對於固態瘤，[TNM分期是最常見的分期系統](https://zh.wikipedia.org/wiki/TNM分期 "wikilink")，但在個別癌症會與不同的應用情況。

  - [乳癌](../Page/乳癌.md "wikilink")：在乳癌分類當中，是基於TNM分期系統\[2\]，但會分成第一期至第四期。
  - [大腸癌](../Page/大腸癌.md "wikilink")：起初採用A、B、C、D四期分類方式（[杜克斯分類法](https://zh.wikipedia.org/wiki/杜克斯分類法 "wikilink")）。最近則同時採用杜克斯分類法與TNM分期系統\[3\]。
  - [腎臟癌](https://zh.wikipedia.org/wiki/腎臟癌 "wikilink")：使用TNM分期系統\[4\]。
  - [喉癌](../Page/喉癌.md "wikilink")：使用TNM分期系統\[5\]。
  - [肝癌](../Page/肝癌.md "wikilink")：使用TNM分期系統\[6\]。
  - [肺癌](../Page/肺癌分期.md "wikilink")：使用TNM分期系統\[7\]。
  - [前列腺癌](../Page/前列腺癌.md "wikilink")：美國以外使用TNM分期系統。美國地區Jewett-Whitmore系統有時會被採用\[8\]。
  - [皮膚癌](../Page/皮膚癌.md "wikilink")：使用TNM分期系統\[9\]。
  - [膀胱癌](../Page/膀胱癌.md "wikilink")：使用TNM分期系統\[10\]。

## 癌症分期與其期程特徵

**癌症分期與其期程特徵**大致上可由下表表示：

|       |                                                                                                                                                                                                                               |
| ----- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 第一期癌症 | [腫瘤較小且尚未](https://zh.wikipedia.org/wiki/腫瘤 "wikilink")[擴散/轉移](https://zh.wikipedia.org/wiki/metastasis "wikilink")（癌細胞可被定位、手術移除，不需借助循環全身的[化學藥物治療](https://zh.wikipedia.org/wiki/化學藥物治療 "wikilink")）                           |
| 第二期癌症 | 腫瘤較大但尚未擴散（癌細胞尚可被定位、手術移除，可能不需借助循環全身的化學藥物治療）                                                                                                                                                                                    |
| 第三期癌症 | 不斷增生的癌細胞撐破腫瘤，癌細胞因而[擴散至腫瘤](https://zh.wikipedia.org/wiki/metastasis "wikilink")<u>周遭</u>的組織及<u>鄰近</u>的[淋巴結](../Page/淋巴結.md "wikilink")甚至更遠處。此時癌細胞流竄於原始病灶附近，必需借助循環全身的[化學治療藥物治療](https://zh.wikipedia.org/wiki/化學治療 "wikilink")。 |
| 第四期癌症 | 癌細胞[擴散至距離原始腫瘤遙遠的器官及組織](https://zh.wikipedia.org/wiki/metastasis "wikilink")。此時癌細胞流竄於全身，必需借助循環全身的[化學治療藥物治療](https://zh.wikipedia.org/wiki/化學治療 "wikilink")。[安寧照護將是治療選項之一](https://zh.wikipedia.org/wiki/安寧照護 "wikilink")。      |

註解：癌細胞將於所到之處繼續失控的繁殖。舉例：若肺癌細胞[擴散到肝臟](https://zh.wikipedia.org/wiki/擴散 "wikilink")，則肝臟將出現不斷增生的肺部細胞（不具肺功能），且此細胞具有入侵正常肝臟細胞的能力，導致肝功能不斷降低；不具入侵能力但失控繁殖的細胞，稱為[良性腫瘤](https://zh.wikipedia.org/wiki/良性腫瘤 "wikilink")。 \[11\]

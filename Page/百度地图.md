**百度地图**是[百度](../Page/百度.md "wikilink")于2005年9月发布的地图服务项目，它是[百度公司与北京图为先科技有限公司联合开发的](https://zh.wikipedia.org/wiki/百度公司 "wikilink")[电子地图服务](../Page/电子地图服务.md "wikilink")。图为先是一个电子地图服务提供公司，先前开发了“图吧”（）。

## 介绍

百度地图目前主要地图数据支持大中华地区，包括中国大陆、香港、澳门，并支持海外国家的地图 但海外的地图数据与[Google地图](../Page/Google地图.md "wikilink")相比 仍然不够丰富，台灣仍不在所提供之圖資範圍內，就台灣都市之資料老舊錯誤。

百度地圖在部分大城市如[北京](https://zh.wikipedia.org/wiki/北京 "wikilink")、[上海](https://zh.wikipedia.org/wiki/上海 "wikilink")、[廣州](https://zh.wikipedia.org/wiki/廣州 "wikilink")、[深圳推出了三维地图功能](https://zh.wikipedia.org/wiki/深圳 "wikilink")，用戶可縮放瀏覽城市的[三維街道圖和建築物仿真構圖](https://zh.wikipedia.org/wiki/三維 "wikilink")。

百度地图已推出卫星地图，缩放可显示全球卫星，目前已支持中国绝大多数大中城市，以及港澳地区。但与[Google地图](../Page/Google地图.md "wikilink")的全球地图仍差距甚远。不过，百度没有出现谷歌的[漂移现象](https://zh.wikipedia.org/wiki/中华人民共和国测绘限制#GCJ-02 "wikilink")。

通过百度地图搜索，用户可以找到指定的城市、街区、街道、建筑物等所在的地理位置，也可以找到离用户最近的所有餐馆、学校、银行、公园等等。

在交通方面，百度地图搜索还为用户提供了路线查询功能，如果用户要去某个地点，百度地图搜索会提示用户如何换乘公交车，如果用户想自己驾车去，百度地图搜索同样会为用户推荐最佳路线。

除了网页版，目前也已经推出适用于[Android](../Page/Android.md "wikilink")、[iOS等移动客户端](https://zh.wikipedia.org/wiki/iOS "wikilink")。

## 坐标系

[Baidu_map_with_BD-09,_WGS-84_and_GCJ-02_markers.png](https://zh.wikipedia.org/wiki/File:Baidu_map_with_BD-09,_WGS-84_and_GCJ-02_markers.png "fig:Baidu_map_with_BD-09,_WGS-84_and_GCJ-02_markers.png")

百度地图服务器使用一种网络[墨卡托投影给地图分块](https://zh.wikipedia.org/wiki/墨卡托投影 "wikilink")。这种投影的背后使用一种**BD-09**经纬坐标系统，是中国[GCJ-02](https://zh.wikipedia.org/wiki/GCJ-02 "wikilink")\[1\]（基于国际事实标准[WGS84](https://zh.wikipedia.org/wiki/WGS84 "wikilink")）的再混淆版本。百度号称这样可以保护用户隐私。\[2\]

BD-09 分为经纬度版本（BD09LL）与[墨卡托米制版本](https://zh.wikipedia.org/wiki/墨卡托投影法 "wikilink")（BD09MC）。如无特别标注，通常指前者\[3\]。

百度地图 Web 服务 API 中提供了“坐标转换接口”，可将 WGS-84、GCJ-02、BD-09、mapbar、51 地图等坐标（经纬度或墨卡托米制）转换为 BD-09 或 GCJ-02 坐标（经纬度或墨卡托米制），但“根据相关法律规定”，不支持从任何坐标系转换到 WGS-84 坐标系\[4\]。 该服务亦有 JavaScript、C\#、C++、Java 等语言的离线版本，但只对合作伙伴提供，未公开下载。

百度API文档称，“真实GPS坐标”（WGS-84）必须经过转换才能使用。\[5\]百度提供HTTP、JavaScript、Android SDK和iOS SDK实现。

百度早期提供的网页版JavaScript API演示，没有提供任何反向（如BD转GCJ）功能。\[6\]反向转换有多种语言的开源实现。\[7\]\[8\]，与[反向GCJ算法的做法类似](../Page/中华人民共和国测绘限制.md "wikilink")。

数学上，BD-09 变换将 GCJ 坐标当作直角坐标转成极坐标后加噪音，在变回直角坐标之后再加入一个度数恒定的偏移量：\[9\]

``` py
from cmath import polar, rect
from math import sin, cos, pi
# 使用复数类型简化表示
coords = complex

# 百度使用 x/实部作经度；y/虚部作纬度。
def gcj_bd(gcj: coords) -> coords:
    r, θ = polar(gcj)
    r += 2e-5 * sin(gcj.imag * pi / 180 * 3000)
    θ += 3e-6 * cos(gcj.real * pi / 180 * 3000)
    return rect(r, θ) + (0.0065 + 0.006j)
```

## 百度全景（街景）

2013年8月，百度推出自己的街景视图——百度全景，成为继腾讯、高德之后，中國国内第三家成功上线街景的公司。

不過街景的用處範圍較小，因此建設速度較慢，截止2018年11月中国大陆仅有腾讯地图与百度地图两大应用保有街景功能，其中百度地图街景覆盖率高。

## 事件

2018年6月12日，有网友反映百度地圖手机客户端的語音播報和提示訊息出現大規模故障，語音提示和地圖顯示訊息不符，導致出現左右不分、前後不分等情況。例如導航上規劃的是直線，然而語音错误提示左轉、右轉或者調頭；又或者危險地帶、甚至是高架橋上，卻错误提示掉頭。据媒体报道，百度地圖的官方微博的評論區也出现了大批网友的相关投訴，其中一則於6月11日發布的，關於「城市圖鑒」下的微博評論，大部分都是网友发表的有关周二百度地圖故障的投訴。然而至6月12日中午，該則微博下已無法查看評論。对此，百度地圖官方微博於早上10時許回應表示，称由於当日凌晨服務進行例行升級時出現故障，部分用戶使用百度地圖導航時，語音播報和提示訊息會出現錯誤。目前相关服務已恢復正常\[40\]。同日上午11时，百度方面回应称，百度地图9.5及以上版本用户使用百度地图导航时，如因发生导错或违章所产生的相关问题，可以于30日内在百度地圖手机客户端上完成申请，获取相应的导航保障赔付\[41\]。

2018年[国庆节假期期间](../Page/中华人民共和国国庆节.md "wikilink")，百度地图显示浙江[绍兴柯岩景区](https://zh.wikipedia.org/wiki/绍兴 "wikilink")“今日不营业，请谨慎前往”。游客误以为景区闭园，导致游客数量比往年国庆少很多。百度的回应是，有人提供了一条闭园信息，百度确认之后发布在百度地图上，但柯岩景区方面表示，他们从未收到百度地图的核实电话。\[42\]

2018年11月下旬，网友爆料百度地图抄袭高德数据\[43\]，百度回应“信息来自交警，已修正”\[44\]。且在此前已有多次有关百度地图抄袭高德地图的爆料出现。

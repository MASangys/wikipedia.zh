在[微波与](https://zh.wikipedia.org/wiki/微波 "wikilink")[射频工程中](https://zh.wikipedia.org/wiki/射频 "wikilink")，**短截线**是只在一端连接的[传输线或](../Page/传输线模型.md "wikilink")[波导](../Page/电磁波导.md "wikilink")。短截线的自由端开路或（在波导的情形）短路。忽略传输线的损耗，短截线的[输入阻抗是纯](../Page/输入阻抗.md "wikilink")[抗性的](../Page/电抗.md "wikilink")；是[容性还是](../Page/电容器.md "wikilink")[感性](../Page/电感.md "wikilink")，取决于短截线的以及是开路还是短路。短截线在无线电频率可能用作[电容](../Page/电容器.md "wikilink")、[电感和](../Page/电感元件.md "wikilink")[谐振电路](../Page/LC电路.md "wikilink")。

短截线通过沿其长度方向的无线电波的[驻波发挥作用](https://zh.wikipedia.org/wiki/驻波 "wikilink")。它们的[电抗特性是由它们的物理长度与无线电波的](../Page/电抗.md "wikilink")[波长之间的关系决定的](../Page/波长.md "wikilink")。因此短截线最常用在波长足够短的[UHF或](https://zh.wikipedia.org/wiki/特高频 "wikilink")[微波电路中](https://zh.wikipedia.org/wiki/微波 "wikilink")，于是短截线也较小。\[1\]
它们经常被用来代替分立电容和电感，因为在UHF和微波频率下，由于寄生电抗，[集总元件表现不佳](https://zh.wikipedia.org/wiki/集总电路 "wikilink")。\[2\]
短截线常用在[天线](../Page/天线.md "wikilink")[阻抗匹配电路](https://zh.wikipedia.org/wiki/阻抗匹配 "wikilink")、选频[滤波器和UHF](../Page/电子滤波器.md "wikilink")[电子振荡器与](../Page/电子振荡器.md "wikilink")[射频放大器的](../Page/放大器.md "wikilink")[谐振电路中](../Page/LC电路.md "wikilink")。

任何类型的[传输线都可以做成短截线](../Page/传输线模型.md "wikilink")：（它们称为）、[同轴电缆](../Page/同轴电缆.md "wikilink")、、[波导管以及](https://zh.wikipedia.org/wiki/波导管 "wikilink")。短截线电路可以用[史密斯图](../Page/史密斯图.md "wikilink")（一个可以确定多长的线可以得到所需电抗的图形工具）设计。

## 短路短截线

无耗短路传输线的[输入阻抗为](https://zh.wikipedia.org/wiki/Reflections_of_signals_on_conducting_lines#Input_impedance "wikilink")，

\[Z_\mathrm{SC} = j Z_0 \tan(\beta l)\,\!\]

其中 j 是[虚数单位](https://zh.wikipedia.org/wiki/虚数单位 "wikilink")，\(Z_0\)
是传输线的，\(\beta = 2\pi/\lambda\,\) 是传输线的，而 \(l\) 是传输线的物理长度。

因此，\(\tan(\beta l)\) 为正还是为负会决定短截线是感性还是容性。

在[角频率为](https://zh.wikipedia.org/wiki/角频率 "wikilink") \(\omega\) 时，充当电容的
*C* 的短截线的长为：

\[l = \frac{1}{\beta} \left[(n+1)\pi - \arctan \left(\frac{1}{\omega C Z_0}\right) \right]\]

相同频率下，充当电感 *L* 的短截线的长为：

\[l = \frac{1}{\beta} \left[ n \pi + \arctan\left(\frac{\omega L}{Z_0}\right) \right]\]

## 开路短截线

无耗开路短截线的输入阻抗为

\[Z_\mathrm{OC} = -j Z_0 \cot (\beta l) \,\!\]

由此得出 \(\cot(\beta l)\) 的正负，就分别对应于短截线的容性或感性。

在角频率为 \(\omega\) 时，充当电感的开路短截线 *L* 的长度为：

\[l = \frac{1}{\beta} \left[(n+1)\pi - \arccot\left(\frac{\omega L}{Z_0}\right) \right]\]

相同频率下，充当电容 *C* 的开路短截线的长为：

\[l = \frac{1}{\beta} \left[n \pi + \arccot\left(\frac{1}{\omega C Z_0}\right) \right]\]

## 谐振短截线

短截线经常被用在[振荡器和](../Page/电子振荡器.md "wikilink")中作为振荡电路。长度为
\(\scriptstyle l\) 的开路短截线，当 \(\scriptstyle \beta l < \pi /2\)
时，在低频下会有容抗。在此频率以上阻抗就会为感性。恰好
\(\scriptstyle \beta l = \pi /2\)
时，短截线表示短路。定性地来说，这与串联谐振电路的行为相同。对于无耗线，相位变化常数与频率成正比，

\[\beta = {\omega \over v}\]

其中 *v* 是传播速度，对无耗线来说是与频率无关的常数。对于此情形，谐振频率为

\[\omega_0 = \frac {\pi v}{2 l}\]

当短截线用作谐振电路时，它们与[集总元件谐振电路不同之处在于它们有多个谐振频率](https://zh.wikipedia.org/wiki/集总电路 "wikilink")；除了基波谐振频率
\(\scriptstyle \omega_0\,\) 以外, 还会再这个频率的倍数 \(\scriptstyle n\omega_0\,\)
发生谐振。如在集总调谐电路中一样，谐振后阻抗不会随着频率持续[单调上升](../Page/单调函数.md "wikilink")。阻抗会上升到
\(\scriptstyle \beta l = \pi\)
这一点变为开路。在这一点后（其实是一个点），阻抗会再次变为容性并开始减小。它会持续下降到
\(\scriptstyle \beta l = 3 \pi /2\,\)，然后再次表现为短路。在这一点短截线的滤波作用彻底消失。短截线的响应随着频率增加持续重复，在谐振和反谐振之间交替。某频率之上滤波器不起作用，并产生多个不需要的这一点，不仅是短截线的特点，还是所有分布元件滤波器的特点。\[3\]

同样，短路短截线在 \(\scriptstyle \pi /2\) 是一个反谐振器，也就是说它的行为同并联谐振电路，但当接近
\(\scriptstyle 3 \pi /2\) 时失效。\[4\]

## 短截线匹配

[Stripline_stub_matching.svg](https://zh.wikipedia.org/wiki/File:Stripline_stub_matching.svg "fig:Stripline_stub_matching.svg")
短截线可以用来将负载阻抗与传输线特性阻抗相匹配。短截线的位置与负载有一段距离。该距离选择在负载阻抗的阻性部分与主线长度的的特性阻抗的阻性部分相等的点。短截线的长度要恰好能抵销表现出的负载的抗性部分。也就是说，短截线是根据主线表现出感性还是容性负载而做成容性还是感性的。这与负载实际的阻抗不同，因为负载阻抗的抗性部分以及阻性部分会受到阻抗变换器作用。匹配短截线可以做成可调的，使得可以在测试中修正。\[5\]

单独的短截线只能在某一特定频率达到完美匹配。对宽带匹配来说，会将若干短截线以一定间隔沿主传输线分布。得到的结构类似于滤波器，也会用到滤波器设计方法。例如，匹配网络会设计成[切比雪夫滤波器](../Page/切比雪夫滤波器.md "wikilink")，但会针对阻抗匹配而非通带传输进行优化。得到的网络的传输函数会与切比雪夫滤波器一样有通带[涟波](https://zh.wikipedia.org/wiki/涟波 "wikilink")，但在通带中纹波绝不会像标准滤波器那样，达到
0dB 的插入损耗。\[6\]

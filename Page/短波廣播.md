[Grundig_Satellit_Professional_400.JPG](https://zh.wikipedia.org/wiki/File:Grundig_Satellit_Professional_400.JPG "fig:Grundig_Satellit_Professional_400.JPG")牌数码显示[晶体管](../Page/晶体管.md "wikilink")收音机含短波(SW)頻道。\]\] [Shortwave_Radio.jpg](https://zh.wikipedia.org/wiki/File:Shortwave_Radio.jpg "fig:Shortwave_Radio.jpg")牌指针式短波廣播收音機。\]\] **短波廣播**（，SW）是利用[短波](../Page/短波.md "wikilink")波段播送的[广播](https://zh.wikipedia.org/wiki/广播 "wikilink")。由於传播距离较远，因而[國際廣播通常都位於短波波段](https://zh.wikipedia.org/wiki/國際廣播 "wikilink")，部分国家的对内广播也会使用短波频段。其具有高度戰略價值，至今仍被專家們普遍認為是大規模全球傳送的唯一最有效途徑，且安全、便宜、快捷。

## 短波特性

[Ionospheric_reflection.png](https://zh.wikipedia.org/wiki/File:Ionospheric_reflection.png "fig:Ionospheric_reflection.png") [QSL-VOA-Thessaloniki-1972.jpg](https://zh.wikipedia.org/wiki/File:QSL-VOA-Thessaloniki-1972.jpg "fig:QSL-VOA-Thessaloniki-1972.jpg")的QSL卡，图案为[希腊](../Page/希腊.md "wikilink")[塞萨洛尼基](../Page/塞萨洛尼基.md "wikilink")短波发射台的天线。\]\]  短波是[無線電的一個波長範圍](https://zh.wikipedia.org/wiki/無線電 "wikilink")，其對應的頻率範圍被稱為[高頻](https://zh.wikipedia.org/wiki/高頻 "wikilink")。短波的波長範圍是10米至100米，高頻的範圍則是3MHz到30MHz。\[1\]短波波段的[電磁波除了能夠利用地波傳播外](https://zh.wikipedia.org/wiki/電磁波 "wikilink")，還可通過[電離層的反射進行遠距離傳輸](https://zh.wikipedia.org/wiki/電離層 "wikilink")，穿透力強，不易干擾，因而國際廣播通常都位於短波波段。\[2\]

## 短波广播使用波段

短波广播通常在3-30MHz的[高频](https://zh.wikipedia.org/wiki/高频 "wikilink")（HF）波段进行，但一般高于[中波波段上限](https://zh.wikipedia.org/wiki/中波 "wikilink")（1611kHz或1710kHz）而低于30MHz的广播都可以视为“短波广播”。根据1997年[世界无线电通信大会](../Page/世界无线电通信大会.md "wikilink")（WRC）的决议，分配给短波广播电台使用的频段如下：

| 波段（m） | 频率范围（MHz）   | 备注                                               |
| ----- | ----------- | ------------------------------------------------ |
| 120   | 2.3-2.495   | 又称“热带波段”                                         |
| 90    | 3.2-3.4     |                                                  |
| 75    | 3.9-4       | 与北美地区[业余无线电](../Page/业余无线电.md "wikilink")80m波段重合 |
| 60    | 4.75-5.06   | 又称“热带波段”                                         |
| 49    | 5.9-6.2     |                                                  |
| 41    | 7.2-7.6     | 与业余无线电40m波段重合                                    |
| 31    | 9.4-9.9     | 目前电台最为集中的频段                                      |
| 25    | 11.6-12.2   |                                                  |
| 22    | 13.57-13.87 |                                                  |
| 19    | 15.1-15.8   |                                                  |
| 16    | 17.48-17.9  |                                                  |
| 15    | 18.9-19.02  | 几乎无电台使用                                          |
| 13    | 21.45-21.85 |                                                  |
| 11    | 25.1-26.1   | 主要为DRM电台使用，模拟电台几乎不使用                             |

需要注意的是，一些广播电台（尤其是部分“[秘密电台](https://zh.wikipedia.org/wiki/地下电台 "wikilink")”和“海盗电台”）可能不在上述频率范围内播音，如韩国对朝鲜广播的[人民之声电台](../Page/人民之声电台.md "wikilink")就使用6600kHz播音，欧美海盗电台大多不会在上述表格内规定的频率进行广播。部分国家的授时台（如中国的[BPM短波授时台](../Page/BPM短波授时台.md "wikilink")）也会在WRC规定的频率范围之外播出。不同时间内，不同频段的接收效果有所不同，例如19m和16m波段在午后至晚间效果较好，49m和41m波段在晚间至早晨效果较好。

## 调制方式

目前绝大多数短波广播电台都采用传统的模拟声音广播，使用[调幅](https://zh.wikipedia.org/wiki/调幅 "wikilink")（AM）方式进行调制，少数电台如[AFN和一些海盗电台使用](https://zh.wikipedia.org/wiki/AFN "wikilink")[单边带模式](https://zh.wikipedia.org/wiki/单边带 "wikilink")。另外，部分电台也开设了[DRM](https://zh.wikipedia.org/wiki/DRM "wikilink")（Digital Radio Mondiale）数字广播。

## 短波廣播技術特性

  - 難以封鎖

[美國之音報導](https://zh.wikipedia.org/wiki/美國之音 "wikilink")，專家們指出，較新技術比傳統廣播更容易被干擾，因為「無線電廣播可以同時以很多不同頻率進行」。美國之音研究人員艾利奧特表示「在所有可用於進行[國際廣播的媒介中](https://zh.wikipedia.org/wiki/國際廣播 "wikilink")，短波是最難阻斷的，這是因為[短波](../Page/短波.md "wikilink")頻率發射傳送的物理特性。」相對下，干擾電視信號，尤其是[衛星](../Page/衛星.md "wikilink")電 視信號比較簡單。「來自距離遠的發射機信號比距離近的發射機信號更容易傳入。電視信號的傳送距離比無線電廣播要短得多，因此干擾阻斷電視信號要容易得 多。」尤其「衛星傳送的電視信號很容易被干擾，只需要幾瓦的功率就可以辦到，而且這些信號沒有短波信號具有的那種抗擾性。」\[3\]

專家指出，干擾一個短波信號所需投注的資源，等同創辦一家短波電台，而且過量干擾波也會破壞極權當局自己的官方廣播\[4\]。

  - 互聯網能否取代短波

[互聯網問世後](https://zh.wikipedia.org/wiki/互聯網 "wikilink")，曾出現放棄短波的討論；[美國之音短波廣播也曾被討論存廢](https://zh.wikipedia.org/wiki/美國之音 "wikilink")，但在[美國國務卿堅持下仍保留短波廣播](https://zh.wikipedia.org/wiki/美國國務卿 "wikilink")。不過，許多專業人士認為，跟網路相比，短波廣播「快捷、安全的優點無可取代」\[5\]。

  - 短波廣播只需發射設備，播音端、接收端都更簡便快速。但透過互聯網獲得信息，在中國大陸等網路封鎖的國家，使用者必須擁有並懂操作電腦、且能成功「[翻牆](https://zh.wikipedia.org/wiki/翻牆 "wikilink")」突破網路封鎖。
  - 從安全性考量，登入網路可能讓政府當局得以追查相關痕跡。但收聽透過空氣傳送的短波廣播，當局難以尋獲聽眾，媒體受眾的安全可以減少風險。
  - 極權當局技術上可以輕易全面切斷互聯網，但要阻斷短波卻不容易。

<!-- end list -->

  - 大規模全球傳送的唯一最有效途徑

[記者無國界組織的亞洲專家布羅賽爾認為](https://zh.wikipedia.org/wiki/記者無國界 "wikilink")，無線電廣播仍然是世界各地很多人們獲取信息的主要來源：「基於世界上很多人不識字或者無法使用網際網路，無線電廣播可以說是世界上最具民主化，最受歡迎的媒體。能夠接觸到全球億萬民眾的唯一途徑就是無線電廣播。」\[6\]

以中國大陸為例，據[中華人民共和國政府官方統計](https://zh.wikipedia.org/wiki/中華人民共和國 "wikilink")，全中國家庭共擁有5億台收音機、4億台電視機。另根據中國電網信息中心數據，中國大陸使用[網路人口約](https://zh.wikipedia.org/wiki/互聯網 "wikilink")4.5億人；成年人中，未使用互聯網者約7.3億人；約2億人外出工作，沒有上網、不會操作電腦者，這些人是短波廣播愛用群\[7\]。

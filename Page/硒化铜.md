**硒化铜**是一种[无机化合物](../Page/无机化合物.md "wikilink")，[化学式](../Page/化学式.md "wikilink")为CuSe。

## 制备

将[四氯化硒用还原剂还原之后](https://zh.wikipedia.org/wiki/四氯化硒 "wikilink")，和[氯化铜](../Page/氯化铜.md "wikilink")反应，可以得到CuSe。\[1\]

  -
    SeCl<sub>4</sub> + 3 H<sub>2</sub>O → H<sub>2</sub>SeO<sub>3</sub> + 4 HCl
    H<sub>2</sub>SeO<sub>3</sub> —<sup><u>\[H\]</u></sup>→ Se （H＝N<sub>2</sub>H<sub>4</sub>·H<sub>2</sub>O等）
    3 Se + 6 OH<sup>−</sup> → 2Se<sup>2−</sup> + SeO<sub>3</sub><sup>2-</sup> + 3H<sub>2</sub>O
    Cu<sup>2+</sup> + Se<sup>2−</sup> → CuSe↓

[硒](../Page/硒.md "wikilink")粉和[亚硫酸钠](../Page/亚硫酸钠.md "wikilink")反应，得到[硒代亚硫酸钠](https://zh.wikipedia.org/wiki/硒代亚硫酸钠 "wikilink")，以此为原料也能得到硒化铜，[微波](https://zh.wikipedia.org/wiki/微波 "wikilink")\[2\]和[超声](https://zh.wikipedia.org/wiki/超声 "wikilink")\[3\]能促进产物的生成：

  -
    Se + SO<sub>3</sub><sup>2-</sup> → SeSO<sub>3</sub><sup>2-</sup>
    Cu<sup>2+</sup> + SeSO<sub>3</sub><sup>2-</sup> → CuSe + SO<sub>3</sub><sup>2-</sup>

反应物按照不同的配比，还能得到Cu<sub>3</sub>Se<sub>2</sub>等铜-硒化合物。

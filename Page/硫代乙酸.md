**硫代乙酸**是一种[有机化合物](../Page/有机化合物.md "wikilink")，[化学式](../Page/化学式.md "wikilink")为CH<sub>3</sub>C(O)SH，它是黄色液体，有强烈的硫醇气味。它用于[有机合成](../Page/有机合成.md "wikilink")中[巯基的引入](https://zh.wikipedia.org/wiki/巯基 "wikilink")。\[1\]

## 制备

硫代乙酸可以由[乙酸酐](../Page/乙酸酐.md "wikilink")和[硫化氢反应得到](https://zh.wikipedia.org/wiki/硫化氢 "wikilink")：\[2\]

  -
    (CH<sub>3</sub>C(O))<sub>2</sub>O + H<sub>2</sub>S → CH<sub>3</sub>C(O)SH + CH<sub>3</sub>CO<sub>2</sub>H

## 化学性质

硫代乙酸离解出的硫代乙酸根可以用于制备硫代乙酸酯。\[3\] 硫代乙酸酯的水解会产生硫代乙酸。由硫代乙酸钠和卤代烃的反应，可以制备硫代乙酸酯。一些反应如下：

  -
    CH<sub>3</sub>C(O)SH + NaOH → CH<sub>3</sub>C(O)SNa + H<sub>2</sub>O
    CH<sub>3</sub>C(O)SNa + RX → CH<sub>3</sub>C(O)SR + NaX (X = Cl, Br, I, etc)
    CH<sub>3</sub>C(O)SR + 2 NaOH → CH<sub>3</sub>CO<sub>2</sub>Na + RSNa + H<sub>2</sub>O
    RSNa + HCl → RSH + NaCl

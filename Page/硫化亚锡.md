**硫化亚锡**是一种无机化合物，化学式为SnS。它出现在罕见的硫锡矿中。

## 制备

硫化亚锡可由单质直接化合得到，为灰色晶体。

  -
    Sn + S —<sup><u>Δ</u></sup>→ SnS

如果将[硫化氢通入Sn](https://zh.wikipedia.org/wiki/硫化氢 "wikilink")<sup>2+</sup>（如[氯化亚锡](https://zh.wikipedia.org/wiki/氯化亚锡 "wikilink")）溶液中，可以得到暗棕色沉淀，为水合物。该水合物很容易脱水，形成无水物。

  -
    SnCl<sub>2</sub> + H<sub>2</sub>S → SnS↓ + 2 HCl

用[锡和](../Page/锡.md "wikilink")[石墨作为电极](../Page/石墨.md "wikilink")，在[硫代硫酸钠溶液中交流电解](../Page/硫代硫酸钠.md "wikilink")，得到黑色的SnS。

将无水[硫氰酸钾熔融至开始出现蓝色](../Page/硫氰酸钾.md "wikilink")（约450℃）之后，缓慢加入[二氧化锡](../Page/二氧化锡.md "wikilink")，反应结束后继续加热一段时间，缓慢冷却，用水处理熔融物，过滤后得到SnS。\[1\]

  -
    SnO<sub>2</sub> + 2 KSCN → SnS + K<sub>2</sub>S + 2 CO↑ +
    N<sub>2</sub>↑

## 化学性质

硫化亚锡可溶于3\~4mol/L的[盐酸和](../Page/盐酸.md "wikilink")2mol/L的[氢氧化钠溶液](../Page/氢氧化钠.md "wikilink")。它和氯化氢加热才能反应。\[2\]

硫化亚锡不溶于碱金属硫化物及[(NH<sub>4</sub>)<sub>2</sub>S](../Page/硫化铵.md "wikilink")，但溶于(NH<sub>4</sub>)<sub>2</sub>S<sub>2</sub>，因为多硫化物将Sn(II)氧化为了Sn(IV)\[3\]：

  -
    SnS + (NH<sub>4</sub>)<sub>2</sub>S<sub>2</sub> →
    (NH<sub>4</sub>)<sub>2</sub>SnS<sub>3</sub>

此外，FeCl<sub>3</sub>和CuCl<sub>2</sub>的热溶液也能将SnS氧化：

  -
    4 FeCl<sub>3</sub> + SnS → SnCl<sub>4</sub> + 4 FeCl<sub>2</sub> + S
    4 CuCl<sub>2</sub> + SnS → SnCl<sub>4</sub> + 4 CuCl + S
    （CuCl可与SnS进一步发生复分解反应）

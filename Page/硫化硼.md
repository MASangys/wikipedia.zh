**硫化硼**是一种无机化合物，化学式为B<sub>2</sub>S<sub>3</sub>。它曾在“高科技”玻璃与制备有机硫化合物方面受到重视。和硫化硅、硫化磷一样，硫化硼可以和水（或者潮湿的空气）反应，放出[硫化氢](https://zh.wikipedia.org/wiki/硫化氢 "wikilink")。因此，硫化硼必须储存在无水环境中。

和[氧化硼一样](https://zh.wikipedia.org/wiki/氧化硼 "wikilink")，硫化硼和其他硫化物（如[P<sub>4</sub>S<sub>10</sub>](../Page/五硫化二磷.md "wikilink")）混合时，形成玻璃。与传统的硼硅玻璃相比，这类玻璃可以吸收低频的[红外线的能量](../Page/红外线.md "wikilink")。

硫化硼可以将[酮转化为相应的](../Page/酮.md "wikilink")[硫酮](../Page/硫酮.md "wikilink")。例如，[二苯甲酮转化为](../Page/二苯基甲酮.md "wikilink")[二苯甲硫酮的过程如下](https://zh.wikipedia.org/wiki/二苯甲硫酮 "wikilink"):

  -

    \+ 3  →  + 3

根据实践，B<sub>2</sub>S<sub>3</sub>有很高的实用价值。\[1\]

## 制备

第一次硫化硼的合成是由[Jöns Jakob
Berzelius在](../Page/永斯·贝采利乌斯.md "wikilink")1824年，通过无定形[硼与](../Page/硼.md "wikilink")[硫蒸汽的直接化合得到](../Page/硫.md "wikilink")。\[2\]

\[\mathsf{2B + 3S \ \xrightarrow{\ }\ B_2S_3 }\] [Friedrich
Wöhler和](../Page/弗里德里希·维勒.md "wikilink")[Henri Etienne Sainte-Claire
Deville则在](https://zh.wikipedia.org/wiki/亨利·爱丁·圣克莱尔·德维尔 "wikilink")1858年用了另一种合成方法来制备硫化硼，他们用硼和[硫化氢的反应来制备](https://zh.wikipedia.org/wiki/硫化氢 "wikilink")。\[3\]\[4\]

\[\mathsf{2B + 3H_2S \ \xrightarrow{\ }\ B_2S_3 + 3H_2 }\]
除了以上方法外，B<sub>2</sub>S<sub>3</sub>还可以通过铁或者锰的硼化物与[硫化氢在](https://zh.wikipedia.org/wiki/硫化氢 "wikilink")300℃反应得到。\[5\]

\[\mathsf{2FeB + 4H_2S \ \xrightarrow{\ }\ B_2S_3 + FeS + 4H_2 }\]

\[\mathsf{2MnB + 4H_2S \ \xrightarrow{\ }\ B_2S_3 + MnS + 4H_2 }\]
二硫化碳在930℃作用于硼单质也可产生硫化硼：

\[\mathsf{4B + 3CS_2 \ \xrightarrow{930^oC}\ 2B_2S_3 + 3C }\]

## 物理性质

硫化硼是一种白色晶体或无定形的粉末，加热变粘稠。微溶于[三氯化磷和](../Page/三氯化磷.md "wikilink")[二氯化硫](../Page/二氯化硫.md "wikilink")\[6\]。

## 化学性质

硫化硼极易水解，也易醇解\[7\]\[8\]：

\[\mathsf{B_2S_3 + 6H_2O \ \xrightarrow{\ }\ 2H_3BO_3 + 3H_2S\uparrow }\]
其水解在潮湿空气中便可发生。

硫化硼也能还原[氧化性的酸](https://zh.wikipedia.org/wiki/氧化性 "wikilink")：

\[\mathsf{B_2S_3 + 9H_2SO_4 \ \xrightarrow{\ }\ 2H_3BO_3 + 12SO_2\uparrow + 6H_2O }\]

\[\mathsf{B_2S_3 + 24HNO_3 \ \xrightarrow{100^oC}\ 2H_3BO_3 + 3H_2SO_4 + 24NO_2\uparrow + 6H_2O }\]

## 应用

硫化硼可以应用于有机合成。也能用作[核反应堆的](../Page/核反应堆.md "wikilink")[控制棒](../Page/控制棒.md "wikilink")\[9\]。

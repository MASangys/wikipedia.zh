**硫氰酸鉀**，分子式为KSCN，又稱**硫氰化鉀**。硫氰酸钾是無色[單斜晶系晶體](../Page/單斜晶系.md "wikilink")，溶於水，並大量吸熱而降溫，也溶於[酒精](../Page/酒精.md "wikilink")、[丙酮](../Page/丙酮.md "wikilink")。它遇
Fe<sup>3+</sup> 生成血紅色的[硫氰酸鐵离子](../Page/硫氰酸鐵.md "wikilink")
\[Fe(SCN)<sub>n</sub>\]<sup>3-n</sup>(n=1–6)，是檢驗 Fe<sup>3+</sup>
離子十分靈敏的方法，可以排除目前已知的一切其它金屬離子的影響。

## 制备

硫氰酸钾可以通过[氰化钾和](../Page/氰化钾.md "wikilink")[硫共熔制备](../Page/硫.md "wikilink")\[1\]：

  -
    <chem>KCN + S -\>\[\\Delta\] KSCN</chem>

或者通过硫氰酸铵和钾盐的复分解反应来制备。\[2\]

## 化学性质

硫氰酸钾在酸性介质中长时间放置会分解\[3\]：

  -
    KSCN + HCl → HSCN + KCl
    HSCN → HCN + S

若酸性介质中存在某些金属，如[锌](../Page/锌.md "wikilink")，也会分解\[4\]：

  -
    KSCN + 3 HCl + Zn → HCN + H<sub>2</sub>S + ZnCl<sub>2</sub> + KCl

於500°C時會發生分解

  -
    KSCN → KCN + S

硫氰酸钾也可以和硫酸氢钾反应\[5\]\[6\]：

  -
    KSCN + KHSO<sub>4</sub> → HSCN + K<sub>2</sub>SO<sub>4</sub>

## 用途

  -
    硫氰酸钾是检验Fe<sup>3+</sup>的灵敏试剂，在酸性溶液中产生血红色的配离子\[Fe(SCN)<sub>*x*</sub>\]<sup>3-*x*</sup>(*x*＝1**–**6)\[7\]。
    可以将(C<sub>2</sub>H<sub>4</sub>O<sub>2</sub>CO)转化为[环硫乙烷](../Page/环硫乙烷.md "wikilink")(C<sub>2</sub>H<sub>4</sub>S)\[8\]。
    可以检验Co<sup>2+</sup>离子。在试液中加入[戊醇](../Page/1-戊醇.md "wikilink")，再滴加1mol·L<sup>-1</sup>KSCN，震荡，若戊醇层出现蓝色，则表示Co<sup>2+</sup>存在。\[9\]

**硫氰酸银**是[硫氰酸](../Page/硫氰酸.md "wikilink")的[盐](https://zh.wikipedia.org/wiki/盐 "wikilink")，化学式为AgSCN\[1\]。遇热分解。不溶于水。不形成结晶水合物。遇浓酸和浓碱分解。络合物可形成溶液。

## 制备

硫氰酸银可由[硝酸银](../Page/硝酸银.md "wikilink")和[硫氰酸钾](../Page/硫氰酸钾.md "wikilink")反应生成。

\[\mathsf{AgNO_3 + KSCN \rightarrow AgSCN\downarrow + KNO_3}\]

## 化学性质

\[\mathsf{2AgSCN \xrightarrow[]{>170^oC} 2Ag + N_2 + 2C + 2S}\]

\[\mathsf{2AgSCN + 3H_2SO_4 +2H_2O \rightarrow 2CSO + 2NH_4HSO_4 + Ag_2SO_4}\]

\[\mathsf{2AgSCN + 2KOH \rightarrow 2KSCN + Ag_2O + H_2O}\]

\[\mathsf{AgSCN + 2NH_3\cdot H_2O \rightarrow [Ag(NH_3)_2]SCN + 2H_2O}\]

\[\mathsf{2AgSCN + Br_2 \rightarrow 2AgBr + (SCN)_2}\]

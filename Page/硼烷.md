[Borane-3D-balls.png](https://zh.wikipedia.org/wiki/File:Borane-3D-balls.png "fig:Borane-3D-balls.png")的[球棍模型](https://zh.wikipedia.org/wiki/球棍模型 "wikilink")\]\] [Diborane-3D-balls.png](https://zh.wikipedia.org/wiki/File:Diborane-3D-balls.png "fig:Diborane-3D-balls.png")二聚生成[乙硼烷](../Page/乙硼烷.md "wikilink")\]\] **硼烷**（Borane）即**硼氢化合物**，因其[物理性质类似于](https://zh.wikipedia.org/wiki/物理性质 "wikilink")[烷烃](../Page/烷烃.md "wikilink")，故称之为硼烷。在已知的20多种中性硼烷中，最简单的是[乙硼烷](../Page/乙硼烷.md "wikilink") B<sub>2</sub>H<sub>6</sub>。甲硼烷只在气态状态中发现。

按照[硼](../Page/硼.md "wikilink")[氢](../Page/氢.md "wikilink")个数比，中性硼烷一般可分为 B<sub>n</sub>H<sub>n+4</sub> 类（少氢硼烷）和 B<sub>n</sub>H<sub>n+6</sub> 类（多氢硼烷）。此外还有大量的硼烷阴离子：

1.  B<sub>n</sub>H<sub>n</sub><sup>2−</sup> 类，如 B<sub>12</sub>H<sub>12</sub><sup>2−</sup>。为完整硼原子多面体闭合结构，成键类似于相应中性硼烷分子。
2.  由中性硼烷通过脱质子或与 H<sup>−</sup> 加成形成的阴离子，如 [BH<sub>4</sub><sup>−</sup>](../Page/硼氢化钠.md "wikilink")。
3.  由连接的氢化硼单元衍生出的阴离子，如 B<sub>18</sub>H<sub>21</sub><sup>−</sup>。

## 通式

母体硼烷（即沒有取代基）有下列的通式，n是硼原子數：

| 類型                 | [通式](https://zh.wikipedia.org/wiki/通式 "wikilink") | 附註                                                                                       |
| ------------------ | ------------------------------------------------- | ---------------------------------------------------------------------------------------- |
| 闭式-／笼式-（*closo-*）  | B<sub>n</sub>H<sub>n</sub><sup>2−</sup>           | 目前尚未發現[電中性的B](https://zh.wikipedia.org/wiki/電中性 "wikilink")<sub>n</sub>H<sub>n+2</sub>硼烷 |
| 巢式-（*nido-*）       | B<sub>n</sub>H<sub>n+4</sub>                      |                                                                                          |
| 网式-（*arachno-*）    | B<sub>n</sub>H<sub>n+6</sub>                      |                                                                                          |
| 敞网式-／叶式-（*hypho-*） | B<sub>n</sub>H<sub>n+8</sub>                      | 只合成了相关的加合物                                                                               |
| 联式-（*conjuncto-*）  |                                                   | 两个或两个以上简单结构单元相连的硼烷                                                                       |

还有一类*hypercloso-*硼烷，母体的通式为B<sub>n</sub>H<sub>n</sub>。现在已合成出一些较为稳定的取代中性*hypercloso-*硼烷，例如B<sub>12</sub>(OCH<sub>2</sub>Ph)<sub>12</sub>（B<sub>12</sub>H<sub>12</sub>的衍生物）。\[1\]

## 命名規則

電中性硼烷的[命名類似有機](../Page/IUPAC命名法.md "wikilink")[烷烴](https://zh.wikipedia.org/wiki/烷烴 "wikilink")，但必須在其後使用括號附註[氫原子的數量](https://zh.wikipedia.org/wiki/氫 "wikilink")：

  - B<sub>5</sub>H<sub>9</sub> 戊硼烷(9)
  - B<sub>6</sub>H<sub>12</sub> 己硼烷(12)

在命名硼烷阴离子时，应在母体后的括号内注明阴离子的电荷。必要时，可将氢原子数于结构类型前注明。

  - B<sub>5</sub>H<sub>8</sub><sup>−</sup> 八氢合戊硼烷根(1−)

对于较为复杂的硼烷，需要用前缀表明硼烷的结构类型：

  - B<sub>5</sub>H<sub>9</sub> 巢式-戊硼烷(9)
  - [B<sub>4</sub>H<sub>10</sub>](../Page/丁硼烷.md "wikilink") 网式-丁硼烷(10)
  - B<sub>6</sub>H<sub>6</sub><sup>2−</sup> 六氢合-闭式-己硼烷根(2−)

一些较为常见的硼烷也有俗名。

## 成键

硼烷中常出现五种类型的[化学键](../Page/化学键.md "wikilink")，其中有包括[氢桥键](https://zh.wikipedia.org/wiki/氢桥键 "wikilink")，[硼桥键和](https://zh.wikipedia.org/wiki/硼桥键 "wikilink")[闭合式硼键的三种](https://zh.wikipedia.org/wiki/闭合式硼键 "wikilink")[缺电子的](https://zh.wikipedia.org/wiki/缺电子 "wikilink")[三中心二电子键和两种一般的化学键](https://zh.wikipedia.org/wiki/三中心二电子键 "wikilink")——硼氢键 B-H、硼硼键 B-B。在[癸硼烷(14)](https://zh.wikipedia.org/wiki/癸硼烷\(14\) "wikilink") B<sub>10</sub>H<sub>14</sub> 中涉及到这五种键型。

例如，乙硼烷中三中心二电子的氢桥键的具体成键情况是：

每个[硼](../Page/硼.md "wikilink")原子均采取 sp<sup>3</sup> [杂化](https://zh.wikipedia.org/wiki/杂化 "wikilink")，4个[杂化轨道中有](https://zh.wikipedia.org/wiki/杂化轨道 "wikilink")3个单电子轨道和一个空轨道。上方[氢](../Page/氢.md "wikilink")原子的有1个电子的 1s 轨道与两个硼原子的共含1个电子的两个 sp<sup>3</sup> 杂化轨道三者互相成键。这种键称为三中心二电子键，同时由于其类似一座桥，故称为氢桥键。

1976年，[威廉·利普斯科姆](../Page/威廉·利普斯科姆.md "wikilink")因為硼烷結構的研究得到諾貝爾化學獎。

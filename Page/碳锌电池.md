[Zncells.png](https://zh.wikipedia.org/wiki/File:Zncells.png "fig:Zncells.png") **碳锌电池**，又称**碳锌干电池**、**碳性電池**、**碳性電芯**、**乾電池**、**酸性鋅碳電池**。碳锌电池有一层由[锌](../Page/锌.md "wikilink")构成的外壳，作为电池的负极。碳锌电池是从液体[Leclanché电池发展而来](https://zh.wikipedia.org/wiki/Leclanché电池 "wikilink")。傳統或一般型以[氯化铵](../Page/氯化铵.md "wikilink")為電解質；“超级”或“高能”（英文標示：Super Heavy Duty）电池则通常是使用[氯化锌](../Page/氯化锌.md "wikilink")為電解質的碳鋅电池，是一般使用的廉价电池的一种改良版。电池的正极主要是由粉末状的[二氧化锰](../Page/二氧化锰.md "wikilink")和[碳](../Page/碳.md "wikilink")构成。电解液是把[氯化锌](../Page/氯化锌.md "wikilink")、[氯化铵](../Page/氯化铵.md "wikilink")、[碳](../Page/碳.md "wikilink")粉和[澱粉溶于水中所形成的糊状物质](https://zh.wikipedia.org/wiki/澱粉 "wikilink")。碳锌电池是最便宜的[原电池](https://zh.wikipedia.org/wiki/原电池 "wikilink")，因此成为很多厂商的首选，因为这些厂商所销售的设备中常常需要配送电池。锌碳电池適合用於[遥控器](https://zh.wikipedia.org/wiki/遥控器 "wikilink")、[闪光灯](../Page/闪光灯.md "wikilink")、[玩具](../Page/玩具.md "wikilink")或[晶体管收音机](../Page/晶体管收音机.md "wikilink")等功率不大的设备可發揮最大效用。

此電池正極的碳棒與二氧化錳中所混合的碳只負責引出電流，並不參與反應，正極實際參與還原反應並提供正電的是[二氧化錳](https://zh.wikipedia.org/wiki/二氧化錳 "wikilink")，因此，又稱為**錳鋅電池**、**鋅錳電池**或鋅－二氧化錳電池 ，也有簡稱錳乾電池 的。

## 原电池

一个碳锌干电池是一个原电池，因为一旦电池放电完毕，不能够被再次充电而只能被丢弃。“电池翻新”曾经也出现在市场上，能够通过对电池加上一个反向[电流](../Page/电流.md "wikilink")来恢复已经部分放电的电池电量。但这些设备往往只能暂时的充电而且容易导致电池泄漏以及爆炸。与其它种类电池相比，碳锌电池之所以更容易泄漏，是因为锌同时作为放电时被消耗的负极和电池容器。锌锰电池发明于19世纪60年代，并在后来通过改进电解液形式成为干电池。

## 化学反应

碳锌电池的容器是一个锌罐。里面有一层由[NH<sub>4</sub>Cl和](https://zh.wikipedia.org/wiki/氯化銨 "wikilink")[ZnCl<sub>2</sub>所构成的糊状液体](https://zh.wikipedia.org/wiki/氯化鋅 "wikilink")，这个糊状液体通过一个纸层与粉末状的碳和二氧化锰隔开。这些粉末填充在一个碳棒的周围。

[Zincbattery.png](https://zh.wikipedia.org/wiki/File:Zincbattery.png "fig:Zincbattery.png")棒（[正极](https://zh.wikipedia.org/wiki/正极 "wikilink")），3：[锌](../Page/锌.md "wikilink")外壳（[负极](https://zh.wikipedia.org/wiki/负极 "wikilink")），4：[二氧化锰](../Page/二氧化锰.md "wikilink")（[正极](https://zh.wikipedia.org/wiki/正极 "wikilink")），5：潮湿的[氯化铵](../Page/氯化铵.md "wikilink")糊状液体（[电解质](../Page/电解质.md "wikilink")），6：金属末端（-）\]\]

在这个干电池里，最外层的锌是电池的负极。在[半反应中](https://zh.wikipedia.org/wiki/半反应 "wikilink")，锌被氧化，如下式所示：

  -
    ([陽極](../Page/陽極.md "wikilink")) Zn(*s*) → Zn<sup>2+</sup>(*aq*) + 2 e<sup>−</sup>

一个由粉末状碳和二氧化锰所围绕的[石墨](../Page/石墨.md "wikilink")棒作为正极。其中，二氧化锰和碳粉混合是为了增加正极的导电性。阴极反应如下图所示：

  -
    ([陰極](https://zh.wikipedia.org/wiki/陰極 "wikilink")) 2MnO<sub>2</sub>(*s*) + 2 H<sup>+</sup>(*aq*) + 2 e<sup>−</sup> → Mn<sub>2</sub>O<sub>3</sub>(*s*) + H<sub>2</sub>O(*l*)

H<sup>+</sup>来自于NH<sub>4</sub><sup>+</sup>(*aq*)：

  -
    NH<sub>4</sub><sup>+</sup>(*aq*) → H<sup>+</sup>(*aq*) + NH<sub>3</sub>(*aq*)

产生的NH<sub>3</sub>可以与Zn<sup>2+</sup>结合：

  -
    Zn<sup>2+</sup>(*aq*) + 2 NH<sub>3</sub>(*aq*) + 2Cl<sup>-</sup>(*aq*) → Zn(NH<sub>3</sub>)<sub>2</sub>Cl<sub>2</sub>(*s*)

在整个半反应中，[锰](../Page/锰.md "wikilink")的[氧化态由](https://zh.wikipedia.org/wiki/氧化态 "wikilink")+4被还原到了+3。

尽管很多副反应也可能发生，但锌碳电池的总反应还是可以由下式表示：

  -
    Zn(*s*) + 2 MnO<sub>2</sub>(*s*) + 2 NH<sub>4</sub><sup>+</sup>(*aq*) → Mn<sub>2</sub>O<sub>3</sub>(*s*) + Zn(NH<sub>3</sub>)<sub>2</sub><sup>2+</sup>(*aq*) + H<sub>2</sub>O(*l*)

碳锌电池的[电动势大约是](https://zh.wikipedia.org/wiki/电动势 "wikilink")1.54[V](../Page/伏特.md "wikilink")。电动势的不确定是由于阴极反应十分复杂，相比来说，阳极反应（锌端）则有一个已知的电势。副反应及活性反应物的消耗直接导致电池的[内阻增加](https://zh.wikipedia.org/wiki/内阻 "wikilink")，电池电动勢降低。

## 洩漏

当一个碳锌电池使用一段时间以后，由于金属锌被氧化成为锌离子，锌外壳会逐渐变薄。因此，[氯化锌](../Page/氯化锌.md "wikilink")溶液常常可以从电池中洩漏出来。洩漏出来的氯化锌往往会使电池表面变粘。一些老的电池没有洩漏保护。一个锌碳电池的使用寿命比较短，[保存期一般为一年半](https://zh.wikipedia.org/wiki/保存期 "wikilink")。

另外，就算电池没有使用，锌外壳也会慢慢的变薄。这是因为电池内的[氯化銨有弱酸性](https://zh.wikipedia.org/wiki/氯化銨 "wikilink")，可以与锌反应。

## 氯化锌电池

高能氯化锌电池是简单锌碳电池的一个改良版，用来提供更长的使用寿命以及更稳定的电压输出。与普通锌碳电池相比，氯化锌电池的电解液不含氯化铵，由纯的氯化锌糊状液体构成。因此，阴极反应也有一些不同：

  -
    MnO<sub>2</sub>(*s*) + H<sub>2</sub>O(*l*) + e<sup>−</sup> → MnO(OH)(*s*) + OH<sup>−</sup>(*aq*)

总反应如下：

  -
    Zn(*s*) + 2 MnO<sub>2</sub>(*s*) + ZnCl<sub>2</sub>(*aq*) + 2 H<sub>2</sub>O(*l*) → 2 MnO(OH)(*s*) + 2 Zn(OH)Cl(*aq*)

## 环保

过去锌锰电池常加入[汞](../Page/汞.md "wikilink")以抑制负极的析氢腐蚀。出于环保考虑，[中華民國](../Page/中華民國.md "wikilink")于2001年和2005年分两阶段对锌锰电池的含汞量作出了限制。现在，[中華民國](../Page/中華民國.md "wikilink")出售碱性锌锰电池含汞量不高于0.0001%，碳锌电池含汞量最高不超过0.025%。不过，鈕扣锌锰电池的含汞量依然没有作出要求。

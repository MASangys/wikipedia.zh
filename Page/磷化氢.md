**磷化氢**，又名**[膦](https://zh.wikipedia.org/wiki/膦 "wikilink")**（拼音：lìn），分子式：，一种无色、剧毒、有鱼腥臭味的气体。

## 製取

实验室中可以用[白磷与浓](https://zh.wikipedia.org/wiki/白磷 "wikilink")[氢氧化钾](../Page/氢氧化钾.md "wikilink")共热制取：

  -
    P<sub>4</sub> + 3KOH(浓) + 3H<sub>2</sub>O → PH<sub>3</sub>↑ + 3KH<sub>2</sub>PO<sub>2</sub>

生成的磷化氢往往因为含有[聯膦](https://zh.wikipedia.org/wiki/聯膦 "wikilink")（P<sub>2</sub>H<sub>4</sub>）而会自燃。磷化氢可用铜、汞、银等较不活泼的金属盐溶液吸收：

  -
    4CuSO<sub>4</sub> + PH<sub>3</sub> + 4H<sub>2</sub>O → 4Cu↓ + H<sub>3</sub>PO<sub>4</sub> + 4H<sub>2</sub>SO<sub>4</sub>
    24CuSO<sub>4</sub> + 11PH<sub>3</sub> + 12H<sub>2</sub>O → 8Cu<sub>3</sub>P↓ + 3H<sub>3</sub>PO<sub>4</sub> + 24H<sub>2</sub>SO<sub>4</sub>

磷化氢还可通过金属磷化物与水作用得到。例如：

  -
    Ca<sub>3</sub>P<sub>2</sub> + 6H<sub>2</sub>O → 3Ca(OH)<sub>2</sub> + 2PH<sub>3</sub>↑

## 性質

磷化氢的密度大于空气，化学性质與[氨](../Page/氨.md "wikilink")相似，也有[孤對電子](https://zh.wikipedia.org/wiki/孤對電子 "wikilink")，所以磷化氫也為一種[路易斯鹼](https://zh.wikipedia.org/wiki/路易斯鹼 "wikilink")，具有配位能力，如低温下和[BF<sub>3</sub>反应生成H](../Page/三氟化硼.md "wikilink")<sub>3</sub>P·BF<sub>3</sub>。與[氫離子結合則生成](https://zh.wikipedia.org/wiki/氫離子 "wikilink")[-{zh-hans:𬭸;zh-hant:鏻;}-離子](../Page/鏻.md "wikilink")(只有与[强酸](../Page/强酸.md "wikilink")如[HI](../Page/氢碘酸.md "wikilink")、[HClO<sub>4</sub>等反应时](../Page/高氯酸.md "wikilink"))。

  -
    PH<sub>3</sub> + H<sup>+</sup> → PH<sub>4</sub><sup>+</sup>

磷化氢还具有强还原性。例如，磷化氢能还原Ag<sup>+</sup>、Cu<sup>2+</sup>等某些金属离子，使Ag<sup>+</sup>还原为Ag,使Cu<sup>2+</sup>还原为[Cu<sub>3</sub>P和Cu](https://zh.wikipedia.org/wiki/磷化亚铜 "wikilink")。

  -
    PH<sub>3</sub> + 6Ag<sup>+</sup> + 3H<sub>2</sub>O → 6Ag + H<sub>3</sub>PO<sub>3</sub> + 6H<sup>+</sup>

此外，磷化氢还能参与有机反应。磷化氢和醛在加热下（100℃）可得到含有[羟甲基的叔膦化合物](https://zh.wikipedia.org/wiki/羟甲基 "wikilink")\[1\]：

  -
    PH<sub>3</sub> + 3 RCHO → \[RCH(OH)\]<sub>3</sub>P

磷化氢是剧毒气体。磷化氢在空气中的最高允许量为0.3[ppm](https://zh.wikipedia.org/wiki/百萬分率 "wikilink")。气体中的磷化氢可用[重铬酸钾](../Page/重铬酸钾.md "wikilink")、[铬酸银](../Page/铬酸银.md "wikilink")、[活性炭](../Page/活性炭.md "wikilink")、[漂白粉的](https://zh.wikipedia.org/wiki/漂白粉 "wikilink")[悬浮液消除其毒性](https://zh.wikipedia.org/wiki/悬浮液 "wikilink")。空气中微量的磷化氢可根据用[硝酸银](../Page/硝酸银.md "wikilink")浸过的[硅胶](../Page/硅胶.md "wikilink")变黑情况指示出来。[磷化铝](../Page/磷化铝.md "wikilink")、[磷化锌](../Page/磷化锌.md "wikilink")和空气中的[水蒸气](../Page/水蒸气.md "wikilink")反应生成磷化氢，因此要密封保存。这一特性使其被用作粮食仓库的烟熏[消毒剂](https://zh.wikipedia.org/wiki/消毒剂 "wikilink")。\[2\]

## 安全

磷化氫氣體可能與空氣形成爆炸性混合物並可以[自燃](../Page/自燃.md "wikilink")。磷化氫燃燒時，它會產生白色煙霧[五氧化二磷](../Page/五氧化二磷.md "wikilink")，吸入後可導致嚴重刺激呼吸道。

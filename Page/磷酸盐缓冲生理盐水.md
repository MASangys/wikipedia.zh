**磷酸盐缓冲生理盐水**（，简称为**PBS**）是[生物学](../Page/生物学.md "wikilink")研究中常用的[缓冲液](https://zh.wikipedia.org/wiki/缓冲液 "wikilink")。这是一种以水为溶剂的含[磷酸钠](../Page/磷酸钠.md "wikilink")的盐溶液，在某些配方中还加入了[氯化钾](../Page/氯化钾.md "wikilink")及[磷酸钾](https://zh.wikipedia.org/wiki/磷酸钾 "wikilink")。该溶液的与离子浓度与人体中的是相匹配的。

## 应用

因其等张且对细胞无毒性，PBS非常常用。用法包括：稀释物质及清洗细胞容器。含[EDTA的PBS亦常用于解开贴壁及成团的细胞](https://zh.wikipedia.org/wiki/EDTA "wikilink")，而不能加入如[锌](../Page/锌.md "wikilink")等的二价[金属](../Page/金属.md "wikilink")，因为这样会导致沉淀。针对不同的应用，常使用。

## 配制

配制PBS的方法有多种。一些配方不含钾离子，而另一些配方含钙或镁\[1\]。通常PBS含下列组分：

| 盐                                             | 浓度       | 浓度    |
| --------------------------------------------- | -------- | ----- |
|                                               | （mmol/L） | （g/L） |
| NaCl                                          | 137      | 8.01  |
| KCl                                           | 2.7      | 0.20  |
| Na<sub>2</sub>HPO<sub>4</sub>·2H<sub>2</sub>O | 10       | 1.78  |
| KH<sub>2</sub>PO<sub>4</sub>                  | 2.0      | 0.27  |
| pH 7.4                                        |          |       |

PBS常用组分

配制PBS溶液的最简单方法是使用PBS片，后者是已制好的配方片剂，当需要使用PBS时只需将其溶解于一定量的蒸馏水里即可，标准容积有：100、200、500和1000 mL\[2\]。

配好的PBS溶液经分装并[高压灭菌](https://zh.wikipedia.org/wiki/高压灭菌 "wikilink")（20分钟，121℃）后即可用于细胞培养。PBS可储存于室温，但如果溶液未经灭菌而希望保存时间长些，则需要将其冷藏于冰箱中，这样会抑制细菌生长。然而浓缩储存液在冷藏时会产生[沉淀](../Page/沉淀.md "wikilink")，在使用之前需要置于室温使沉淀完全溶解。

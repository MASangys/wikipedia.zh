**磷酸钇**是一种[钇的](../Page/钇.md "wikilink")[磷酸盐](https://zh.wikipedia.org/wiki/磷酸盐 "wikilink")，化学式为YPO<sub>4</sub>。\[1\]它在自然界中以[磷钇矿](../Page/磷钇矿.md "wikilink")\[2\]和针磷钇铒矿\[3\]的形式存在。

## 制备

磷酸钇可以由[氯化钇和](../Page/氯化钇.md "wikilink")[磷酸钠反应](../Page/磷酸钠.md "wikilink")，或由[硝酸钇和](../Page/硝酸钇.md "wikilink")[磷酸氢二铵于溶液中反应得到](../Page/磷酸氢二铵.md "wikilink")：\[4\]

\[\mathrm{YCl_3 + (NH_4)_3PO_4 \longrightarrow YPO_4 \downarrow + 3 \ NH_4Cl}\]

\[\mathrm{Y(NO_3)_3 + (NH_4 )_2 HPO_4 \longrightarrow YPO_4 \downarrow \ + 2 \ NH_4NO_3 + HNO3}\]

[氧化钇和磷酸氢二铵的反应也能制备磷酸钇](../Page/氧化钇.md "wikilink")：\[5\]

\[\mathrm{Y_2O_3 + 2 \ (NH_4 )_2HPO_4 \longrightarrow YPO_4 + 2 \ NH_3 + 3 \ H_2 O}\]

氯化钇和[磷酸于](../Page/磷酸.md "wikilink")35\~40°C混合，之后滴加[氨水反应](../Page/氨水.md "wikilink")：\[6\]

\[\mathrm{ YCl_3 + H_3PO_4 + 3NH_3\cdot H_2O \ \xrightarrow{35-40^oC}\ YPO_4 + 3NH_4Cl + 3H_2O }\]

## 性质

磷酸钇属[四方晶系](../Page/四方晶系.md "wikilink")，[晶胞参数为a](https://zh.wikipedia.org/wiki/晶胞参数 "wikilink")=0.68832
nm，c=0.60208
nm。它可以以无水物、一水合物或二水合物的形式存在。二水合物属[单斜晶系](../Page/单斜晶系.md "wikilink")，[空间群为](../Page/空间群.md "wikilink")***B
2/b***，晶胞参数为a=0.648 nm，b=1.512 nm，c=0.628 nm，β=129.4°，Z = 4。\[7\]

磷酸钇和浓碱反应，生成[氢氧化钇](../Page/氢氧化钇.md "wikilink")。\[8\]

## 用途

掺杂Ce<sup>3+</sup>的磷酸钇可用作发光材料\[9\]，Ce<sup>3+</sup>-Tb<sup>3+</sup>等双掺杂材料也有报道。\[10\]

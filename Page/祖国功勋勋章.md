**“祖国功勋”勋章**（）是[俄罗斯联邦的国家级](https://zh.wikipedia.org/wiki/俄罗斯联邦 "wikilink")[勳章](../Page/勳章.md "wikilink")，由1994年3月2日第442号[总统令设立](../Page/俄罗斯联邦总统令.md "wikilink")\[1\]。在1998年[聖安德列勛章](../Page/聖安德列勛章.md "wikilink")设立之前，“祖国功勋”勋章一直是俄罗斯联邦的最高级别军事勋章。1999年1月6日第19号总统令\[2\]和2010年9月7日第1099号总统令\[3\]修改了“祖国功勋”勋章授予法的部分内容。

## 法令规定

  - “祖国功勋”勋章授予为国家做出杰出贡献者，如促进俄罗斯国家制度的发展，取得突出劳动成就，促进[和平](../Page/和平.md "wikilink")、友谊和[合作以及为保卫祖国做出重大贡献的](https://zh.wikipedia.org/wiki/合作 "wikilink")[公民](../Page/公民.md "wikilink")。
  - “祖国功勋”勋章共有四个[等级](https://zh.wikipedia.org/wiki/等级 "wikilink")，一级为最高级别。一级和二级“祖国功勋”勋章有勋章和星章，三级和四级只有勋章。
  - 带剑“祖国功勋”勋章授予在战斗中建立功勋的军人。
  - 通常情况下，只有在曾经荣获过一级[“祖国功勋”奖章和其他俄罗斯联邦国家勋章的情况下](../Page/祖国功勋奖章.md "wikilink")，才能申请授予四级“祖国功勋”勋章。
  - 一些曾获得过“[俄罗斯联邦英雄](../Page/俄罗斯联邦英雄.md "wikilink")”/“[蘇聯英雄](../Page/蘇聯英雄.md "wikilink")”/“[社会主义劳动英雄](../Page/社会主义劳动英雄.md "wikilink")”称号，或[圣乔治勋章勋章](https://zh.wikipedia.org/wiki/圣乔治勋章 "wikilink")/[亚历山大·涅夫斯基勋章](https://zh.wikipedia.org/wiki/亚历山大·涅夫斯基勋章 "wikilink")/[苏沃洛夫勋章](../Page/苏沃洛夫勋章.md "wikilink")/[乌沙科夫勋章](https://zh.wikipedia.org/wiki/乌沙科夫勋章 "wikilink")/[朱可夫勋章](https://zh.wikipedia.org/wiki/朱可夫勋章 "wikilink")/[库图佐夫勋章](../Page/库图佐夫勋章.md "wikilink")/[纳希莫夫勋章](https://zh.wikipedia.org/wiki/纳希莫夫勋章 "wikilink")/[勇气勋章或荣誉称号的公民可以在未曾荣获](https://zh.wikipedia.org/wiki/勇气勋章 "wikilink")“祖国功勋”奖章的情况下被授予“祖国功勋”勋章。
  - 在特殊情况下，[俄罗斯联邦总统可以将](https://zh.wikipedia.org/wiki/俄罗斯联邦总统 "wikilink")“祖国功勋”勋章授予之前从未被授予过俄罗斯联邦国家奖励的公民。\[4\]
  - “祖国功勋”勋章之前曾被授予外国政要和[国家元首](https://zh.wikipedia.org/wiki/国家元首 "wikilink")，如[雅克·希拉克](../Page/雅克·希拉克.md "wikilink")等；2010年的修正法令规定只有俄罗斯联邦公民才能被授予“祖国功勋”勋章\[5\]。

## 勋章样式

| “祖国功勋”勋章 |
| -------- |
|          |
| **星章**   |
| **勋章**   |
| **带剑勋章** |
| **绶带**   |
| **带剑绶带** |
| **完整样式** |
|          |

  - **十字**：十字正面镶有俄罗斯双头鹰国徽的图样。背面正中是圆形框，刻有“利益、荣誉和光荣”（"ПОЛЬЗА, ЧЕСТЬ И СЛАВА"）字样。中央是其铸造年份1994年，下方依次为[月桂枝和编号](https://zh.wikipedia.org/wiki/月桂枝 "wikilink")。一级勋章十字宽度为60[毫米](../Page/毫米.md "wikilink")，配有100毫米宽的绶带；二级和三级的十字宽度为50毫米，二级绶带宽45毫米，二级绶带宽24毫米。四级十字宽40毫米，以耳座和小环固定在[五角形板条上](https://zh.wikipedia.org/wiki/五角形 "wikilink")，板条由24毫米宽的绶带包裹。
  - **带剑勋章**：带剑勋章授予军人，主体与一级勋章相同，上方连接小环的部位为两柄十字交叉的镀金宝剑。每柄剑长28毫米，宽3毫米。\[6\]

《**神奇的汉字**》是[中国大陆](../Page/中国大陆.md "wikilink")[湖南卫视](../Page/湖南卫视.md "wikilink")推出的一档汉字文化竞技类综艺節目，主持人为[何炅](../Page/何炅.md "wikilink")与[杜海涛](../Page/杜海涛.md "wikilink")，汉字专家有王磊、郑贤章、郦波、鵬宇、楊雨等\[1\]。節目中的参赛选手来自各行各业，涵盖各年龄段\[2\]。《神奇的汉字》由湖南衛視刘伟工作室与刘建立工作室共同打造。其於2019年3月27日啟動錄製，並於2019年6月10日起每週一至週四19:30接档《放学后》播出\[3\]。该节目是2019年[国家广播电视总局的重点文化创意扶持题材节目](../Page/國家廣播電視總局.md "wikilink")\[4\]。

## 節目格式

《神奇的汉字》的节目格式如下：

### 第一部分

每期（週一至週三）节目选手将组成红蓝两队（红队队长：[胡一天](../Page/胡一天.md "wikilink")／[黄新淳](https://zh.wikipedia.org/wiki/黄新淳 "wikilink")／[高天鹤](https://zh.wikipedia.org/wiki/高天鹤 "wikilink")／[仝卓](../Page/仝卓.md "wikilink")；蓝队队长：[仝卓](../Page/仝卓.md "wikilink")／[黄明昊](https://zh.wikipedia.org/wiki/黄明昊 "wikilink")／[于小彤](../Page/于小彤.md "wikilink")／[王鶴棣](../Page/王鶴棣.md "wikilink")），並进行汉字挑战。挑战分为三轮，每一轮每队将派出三位选手与队长一起挑战，先答出答案并且正确组词者获得胜利，而对方选手将会在本轮淘汰。三轮过后积分高的队伍将获胜並获得完成心愿的奖励。节目中两位汉字专家，会选出四名汉字精英，进行週四的汉字精英赛\[5\]\[6\]。

#### 題型

  - **汉字組裝：**两队选手依次抢答，每一题在8秒之内必须根据给出的三个部分组合並猜字。

<!-- end list -->

  -
    **例题：**从下列三个部分中识别出一字并组词。（-{负 忄 束}-）**答案**：-{懒}- -{偷懒}-

<!-- end list -->

  - **汉字組裝升級版：**与原版相同，但会给出四个部分，需在內選出三个部分组合並猜字。

<!-- end list -->

  -
    **例题：**从下列四个部分中识别出一字并组词。（-{小 白 厂 子}-）**答案**：-{原}-（去除 子）

<!-- end list -->

  - **以小见大：**两队选手依次抢答，每一题在8秒之内根据圖片中只显示了局部的汉字來猜字。

<!-- end list -->

  - **汉字演变：**两队选手依次抢答，每一题在8秒之内根据圖片中不同時期的汉字來猜字。

<!-- end list -->

  - **成語还原：**给出的成语中，有三个字缺少了一部分。两队选手依次抢答，每一题在8秒之内根据圖片将成语正确还原。

<!-- end list -->

  -
    **例题：**从下列四个部分中将成语正确还原。（-{门 月 <small><sub>丑</sub></small> 艹}-）**答案**：-{闭月羞花}-

<!-- end list -->

  - **成語还原升級版：**与原版相同，但给出的成语中，四个字都会缺少一部分。

<!-- end list -->

  - **水落石出：**两队选手依次抢答，每题在8秒内猜出被遮住的汉字，而被遮住的部分，将会在倒数时慢慢现形，先答出答案并正确组词者获胜。

<!-- end list -->

  - **組词猜字：**两队选手依次抢答，每题在8秒内给出的十字格填写中间的字，使它能与任意方向组合成词。

|   |          |   |
| - | -------- | - |
|   | 抽        |   |
| 失 | **答案**：血 | 液 |
|   | 型        |   |

**組词猜字例题**

  - **組词猜字升級版：**与原版相同，但会给出品字格。
  - **成語聯想：**在八秒內進行搶答，根據九宮格內閃現出現的字猜出成語。

### 第二部分

週四挑战仍分为三轮，規則及題型與第一部分相同，三轮过后积分高的队伍将进入汉字精英榜。

**神木**是指樹齡超過千年的樹。通常體積龐大，故又稱**巨樹**、**巨木**。

## 界定標準

| 類別         | 樹胸圍        | 樹齡               | 其他條件  | 附帶條件 |
| ---------- | ---------- | ---------------- | ----- | ---- |
| 超級神木       | 12公尺以上（紅檜） | 2000年以上          | 早有命名者 | 林班地內 |
| 8公尺以上（扁柏）  |            |                  |       |      |
| 神木         | 10公尺以上     | 1000年以上\~2000年以下 | 早有命名者 | 林班地內 |
| 12公尺以下（紅檜） |            |                  |       |      |

## 各地神木

### 台灣

台灣常見的樹種有[台灣扁柏](../Page/台灣扁柏.md "wikilink")、[紅檜](../Page/紅檜.md "wikilink")、[香杉](../Page/香杉.md "wikilink")、[台灣五葉松與](../Page/台灣五葉松.md "wikilink")[樟樹](../Page/樟樹.md "wikilink")、[柳杉](../Page/柳杉.md "wikilink")，有時亦會發現上百年的[榕樹和](../Page/榕樹.md "wikilink")[白楊](../Page/白楊.md "wikilink")，甚至還發現過[梅樹與](../Page/梅.md "wikilink")[櫻桃樹](../Page/櫻桃.md "wikilink")（且還可以產出）。

  - 北台灣：[拉拉山神木群](../Page/拉拉山神木群.md "wikilink")、[棲蘭神木群](../Page/棲蘭神木群.md "wikilink")、[鎮西堡神木群](../Page/鎮西堡神木群.md "wikilink")、[司馬庫斯神木群](../Page/司馬庫斯神木群.md "wikilink")
  - 中台灣：[大安溪神木](../Page/大安溪神木.md "wikilink")、[鹿林神木](../Page/鹿林神木.md "wikilink")、[塔塔加夫妻神木](../Page/塔塔加夫妻神木.md "wikilink")、[后里神木](../Page/后里.md "wikilink")（唯一之樟樹）、[溪頭神木等](../Page/溪頭.md "wikilink")
  - 南台灣：[阿里山神木群](../Page/阿里山神木群.md "wikilink")

屏東縣大小鬼湖也有發現一片肖楠巨木群。

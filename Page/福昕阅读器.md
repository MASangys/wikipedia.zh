**福昕阅读器**（，原名：**Foxit PDF
Reader**），是一款用來閱讀[PDF格式文件的軟體](https://zh.wikipedia.org/wiki/PDF "wikilink")，這款軟體是由福昕軟件（FoxitSoftware
Inc.）所研發。福昕閱讀器的特點在於軟件內核小，運行速度快，界面符合[Microsoft
Office用戶的習慣](../Page/Microsoft_Office.md "wikilink")，功能與[Adobe
Reader類似](https://zh.wikipedia.org/wiki/Adobe_Reader "wikilink")。福昕閱讀器支持多國語言，具體包含：英語，德語，法語，西班牙語，葡萄牙語，意大利語，俄語，荷蘭語，芬蘭語，丹麥語，挪威語，波蘭語，瑞典語，簡體中文，繁體中文，韓文，日文。

除了開啟速度快之外，福昕阅读器亦有其他特別的功能，例如可設定列印多於一張紙上，輸入符號及註腳等（但此功能於另存新檔時會加入Foxit公司的備註）。

除福昕阅读器軟體是免費提供外，FoxitSoftware 公司的其他PDF工具程式則要付費購買，因此該公司在商業策略上與[Adobe
Systems近似](https://zh.wikipedia.org/wiki/Adobe_Systems "wikilink")，皆是提供免費的PDF閱讀軟體，並銷售其他延伸性需求的PDF相關軟體。

## 系统需求

### Windows平台

Windows 7
Windows 8
Windows 10
Windows Office 2007或更高版本（针对部分PDF文档创建功能）

### macOS平台

macOS 10.12 及以上版本

### Linux平台

Ubuntu Desktop 14.04 \~ 16.10
Red Hat Enterprise Linux Server 6.x（64位）\~ 7.x
SUSE 12 Linux Enterprise Server
OpenSUSE 13.2
OpenSUSE Leap 42.x
针对活动目录权限管理服务，支持以下系统：
Ubuntu Desktop 14.04 \~ 16.10
Red Hat Enterprise Linux Server 6.x（64位）\~ 7.x
OpenSUSE 13.2
OpenSUSE Leap 42.x

### Citrix

通过Citrix Ready验证，支持Citrix XenApp 7.6

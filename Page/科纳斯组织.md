**科纳斯组织**（Khronos Group）是一个由成员资助的，专注于制定开放标准（Open
standard）的行业协会，重点制定免费的[API](https://zh.wikipedia.org/wiki/API "wikilink")，使在各种平台和设备上创作或播放的[多媒体](../Page/多媒体.md "wikilink")可以得到硬件加速。所有Khronos成员都能够对Khronos
API规范的发展作出贡献，在API公开发布前提出自己的意见，在规范制定初期得到草案并进行平台和应用程序测试。

2006年7月31日，Khronos
Group在[SIGGRAPH](../Page/SIGGRAPH.md "wikilink")上宣布得到[OpenGL](../Page/OpenGL.md "wikilink")规范的控制权。\[1\]\[2\]

## 工作组

  - [COLLADA](https://zh.wikipedia.org/wiki/COLLADA "wikilink"), a
    file-format intended to facilitate interchange of 3D assets
  - [EGL](https://zh.wikipedia.org/wiki/EGL_\(API\) "wikilink"), an
    interface between Khronos rendering APIs such as OpenGL ES or OpenVG
    and the underlying native platform window system\[3\]
  - [glTF](https://zh.wikipedia.org/wiki/glTF "wikilink"), a file format
    specification for 3D scenes and models\[4\]
  - [OpenCL](../Page/OpenCL.md "wikilink")，[跨平台通用计算API](https://zh.wikipedia.org/wiki/跨平台 "wikilink")。\[5\]
  - [OpenGL](../Page/OpenGL.md "wikilink")，[跨平台计算机图形API](https://zh.wikipedia.org/wiki/跨平台 "wikilink")。
  - [OpenGL
    ES](../Page/OpenGL_ES.md "wikilink")，OpenGL的衍生产品，用于移动和嵌入式系统，如手机、便携式游戏设备等。
  - [OpenGL SC](../Page/OpenGL_SC.md "wikilink"), a safety critical
    profile of OpenGL ES designed to meet the needs of the
    safety-critical market
  - [OpenKCam](https://zh.wikipedia.org/wiki/OpenKCam "wikilink"),
    Advanced Camera Control API
  - [OpenKODE](https://zh.wikipedia.org/wiki/OpenKODE "wikilink"), an
    API for providing abstracted, portable access to operating system
    resources such as file systems, networks and math libraries
  - [OpenMAX](../Page/OpenMAX.md "wikilink"), a layered set of three
    programming interfaces of various abstraction levels, providing
    access to multimedia functionality
  - [OpenML](https://zh.wikipedia.org/wiki/OpenML "wikilink"), an API
    for capturing, transporting, processing, displaying, and
    synchronizing digital media
  - [OpenSL ES](https://zh.wikipedia.org/wiki/OpenSL_ES "wikilink"), an
    audio API tuned for embedded systems, standardizing access to
    features such as 3D positional audio and MIDI playback
  - [OpenVG](../Page/OpenVG.md "wikilink")，二维[矢量图形](../Page/矢量图形.md "wikilink")加速API。
  - [OpenVX](../Page/OpenVX.md "wikilink"), Hardware acceleration API
    for Computer Vision applications and libraries
  - [OpenWF](https://zh.wikipedia.org/wiki/OpenWF "wikilink"), APIs for
    2D graphics composition and display control
  - [OpenXR](../Page/OpenXR.md "wikilink"), an open and royalty-free
    standard for [virtual
    reality](https://zh.wikipedia.org/wiki/virtual_reality "wikilink")
    and [augmented
    reality](https://zh.wikipedia.org/wiki/augmented_reality "wikilink")
    applications and devices
  - [SPIR](https://zh.wikipedia.org/wiki/Standard_Portable_Intermediate_Representation "wikilink"),
    an intermediate compiler target for OpenCL and Vulkan
  - [StreamInput](https://zh.wikipedia.org/wiki/StreamInput "wikilink"),
    an API for consistently handling input devices
  - [SYCL](../Page/SYCL.md "wikilink"), a single-source
    [C++](../Page/C++.md "wikilink")
    [DSEL](https://zh.wikipedia.org/wiki/DSEL "wikilink") for
    [heterogeneous
    computing](https://zh.wikipedia.org/wiki/heterogeneous_computing "wikilink")
  - [Vulkan](../Page/Vulkan_\(API\).md "wikilink"), a low-overhead
    computer graphics API
  - [WebCL](https://zh.wikipedia.org/wiki/WebCL "wikilink")，将浏览器中的[JavaScript](../Page/JavaScript.md "wikilink")绑定到OpenCL。
  - [WebGL](../Page/WebGL.md "wikilink")，将任意平台上浏览器中的[JavaScript](../Page/JavaScript.md "wikilink")绑定到OpenGL
    ES，支持OpenGL或OpenGL ES图形标准。

## 会员

科纳斯组织将会员分为：\[6\]

  - 创始会员 - 拥有完整的工作组参与权和投票权，以及向科纳斯董事会指定董事的权利。
  - 贡献者会员 - 拥有完整的工作组参与权和投票权。
  - 准会员 - 拥有完整的工作组参与权，但没有投票权。
  - 学术会员 - 拥有完整的工作组参与权，但没有投票权。

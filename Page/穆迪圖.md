**穆迪圖**（）是一個[流體力學中的](https://zh.wikipedia.org/wiki/流體力學 "wikilink")[無因次圖](https://zh.wikipedia.org/wiki/無因次 "wikilink")，表示在一個圓形截面[管路中](https://zh.wikipedia.org/wiki/管路 "wikilink")，完全成形（fully developed）的[流體](https://zh.wikipedia.org/wiki/流體 "wikilink")，其[達西摩擦因子](https://zh.wikipedia.org/wiki/達西摩擦因子 "wikilink")、[雷諾數及相對粗糙度之間的關係](https://zh.wikipedia.org/wiki/雷諾數 "wikilink")，可以用來計算在管路中流體的壓降或流率。

## 說明

[Moody_EN.svg](https://zh.wikipedia.org/wiki/File:Moody_EN.svg "fig:Moody_EN.svg")

穆迪圖標示了在流體不同雷諾數、不同流動形態（[層流或](https://zh.wikipedia.org/wiki/層流 "wikilink")[紊流](https://zh.wikipedia.org/wiki/紊流 "wikilink")）及不同相對粗糙度下的達西摩擦因子，其中相對粗糙度是以的平均高度和管路直徑的比值\(\epsilon \over d\)。

穆迪圖可用來計算管路中的壓降\(\Delta P\)，單位為[Pa](../Page/帕斯卡.md "wikilink")（或是水頭損失\(h_{\mathrm{f}}\)，單位為公尺）或是流率。水頭損失（head loss）可以用[達西–威斯巴哈方程式](../Page/達西–威斯巴哈方程式.md "wikilink")計算：

\[{h_{\mathrm{f}}=f\frac{l}{d}\frac{V^2}{2\,g}};\]

而用下式可以計算壓降：

\[\Delta P = \rho\,g\,h_{\mathrm{f}}\]或其直接表示為\(\Delta P =f \frac{\rho V^2}{2}\frac{l}{d},\)

其中

\[\rho\]為流體密度

\[V\]為管路中平均速度

\[f\]為由穆迪圖中求得的達西摩擦因子

\[l\]為管路長度

\[d\]為管路直徑

穆迪圖可分為[層流及](https://zh.wikipedia.org/wiki/層流 "wikilink")[紊流二種流動形態](https://zh.wikipedia.org/wiki/紊流 "wikilink")。層流時達西摩擦因子的解析解由法國科學家所求得，為\(\frac{64}{Re}\)，此區域中相對粗糙度對摩擦因子沒有顯著影響。紊流時達西摩擦因子及雷諾數的關係較複雜，可以用包括摩擦因子\(f\)的科爾布魯克方程（Colebrook equation）來描述：

\[{1 \over \sqrt{\mathit{f}}}= -2.0 \log_{10} \left(\frac{\frac{\epsilon}{d}}{3.7} + {\frac{2.51}{Re \sqrt{\mathit{f} } } } \right).\]

1944年時繪製達西摩擦因子和雷諾數及相對粗糙度的之間的關係，即為今天所見的穆迪圖\[1\]。

## 不同摩擦因子的穆迪圖

摩擦因子除了達西摩擦因子外，還有一個是，其數值是達西摩擦因子的四分之一。達西摩擦因子較常用在木土工程及機械工程的領域中，而范寧摩擦因子較常用在化學工程的領域中。

可以用下式由范寧摩擦因子計算水頭損失：

\[{h_{\mathrm{f}}=4f\frac{l}{d}\frac{V^2}{2\,g}},\]

也有對應范寧摩擦因子的穆迪圖，其特點是層流區的解析解為\(\frac{16}{Re}\)。

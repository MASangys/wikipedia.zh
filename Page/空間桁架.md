**空間桁架**，屬於輕便剛性結構構造組成一個[幾何樣式](../Page/幾何.md "wikilink")。空間桁架通常運用一個多向間距，是由[張力桿和](../Page/張力桿.md "wikilink")[壓力桿組成的結構](../Page/壓力桿.md "wikilink")，結構處在[三度空間的受力狀態下](../Page/三度空間.md "wikilink")，無論哪一類桁架，能承受來自各個方向的載荷，對抗震大垮距的建築物更能發揮功用。

種類有平面桁架，曲面式空間桁架，平頂式空間桁架等三種，優點重量輕，可做成各種造型。

平面桁架是一個讓所有成員和節點屬於平面，而空間桁架的成員節點可以延伸到立體層面。

## 概述

基本單元常用的如四角錐單元和三角錐單元，由這些基本單元組成完整的空間桁架系統。\[1\]

| 平面桁架系統   | 四角錐系統    | 三角錐系統   |
| -------- | -------- | ------- |
| 兩向正交正放桁架 | 正放四角錐系   | 三角錐系    |
| 兩向正交斜放桁架 | 正放抽空四角錐系 | 抽空三角錐系  |
| 兩向斜交斜放桁架 | 棋盤形四角錐系  | 蜂巢形三角錐系 |
| 三向桁架     | 斜放四角錐系   |         |
| 單向折線形桁架  | 星形四角錐系   |         |
|          |          |         |

## 歷史

  - 1930年代[德國MERO創造出空間桁架系統](../Page/德國.md "wikilink")。
  - 1970年[大阪](../Page/大阪.md "wikilink")[世界博覽會展現出這種特殊的結構魅力](../Page/世界博覽會.md "wikilink")。

## 幾何

最常見的幾何形狀是根據[正多面體](../Page/正多面體.md "wikilink")。最簡單的形式是[鋁或](../Page/鋁.md "wikilink")[鋼管三角形](../Page/鋼管.md "wikilink")。在許多方面，這看起來像是一個水平三角形正反，反復多次，使之更廣泛。更強的純形式是由相互關聯的四面體[金字塔中](../Page/金字塔.md "wikilink")，所有的支柱有單位長度。技術上更是被稱為各向同性向量矩陣或一個單位寬度的八向桁架。更為複雜的變化改變長度的支柱，以曲線的整體結構或可能納入其他[幾何形狀](../Page/幾何.md "wikilink")。

## 材料

管材料（圓形，方形）或是節點，有[鋼](../Page/鋼.md "wikilink")、[鋁](../Page/鋁.md "wikilink")、[不銹鋼](../Page/不銹鋼.md "wikilink")、工程塑膠、車削件或木頭等可供選擇。\[2\]

## 構件、系統

主要概分為兩大構件，平板式與圓球式。主要概分為四大系統，德國MERO；新加坡RAMCO-YKK；日本NIPPON
STEEL；美國STARNRT\[3\]

## 運用

[Eden_Project_geodesic_domes_panorama.jpg](https://zh.wikipedia.org/wiki/File:Eden_Project_geodesic_domes_panorama.jpg "fig:Eden_Project_geodesic_domes_panorama.jpg")
現今[橋樑](../Page/橋樑.md "wikilink")、[體育館](../Page/體育館.md "wikilink")、[機場](../Page/機場.md "wikilink")、[車站等大尺寸及特殊造型建物上廣泛使用](../Page/車站.md "wikilink")。空間桁架，是一個越來越普遍的建築技術，尤其適用於大型屋頂覆蓋，例：[商業和](../Page/商業.md "wikilink")[工業廠房大門](../Page/工業.md "wikilink")[屋簷](../Page/屋簷.md "wikilink")。現今容量更大的可攜式階段和[照明支架](../Page/照明.md "wikilink")，也經常建造空間桁架和八向桁架。管式空間桁架也被廣泛用於生產現代[摩托車和](../Page/摩托車.md "wikilink")[汽車](../Page/汽車.md "wikilink")，但單體車身已較為常見，用在最新的化身[自行車設計](../Page/自行車.md "wikilink")。

## 素質考慮

  - 方便攜帶：空間桁架可被完全拆除，物品正常放於集裝箱由卡車方便運送。此外，這種類型的結構明顯輕於其他任何類型的結構。這樣可以節省運輸和可能帶來的所有元素的結構，現場沒有任何問題。基於這個原因空間桁架適合出口。
  - 消除焊接工作：不需焊接。
  - 支撐結構是在兩個主要方向，給它很大的剛性，這是一個很大的優勢。
  - 可以減輕70%～80%重量，節省大量水泥和模板

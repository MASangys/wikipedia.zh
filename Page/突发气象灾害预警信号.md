现行的预警信号预警信号分为[台风](https://zh.wikipedia.org/wiki/台风 "wikilink")、[暴雨](../Page/暴雨.md "wikilink")、[暴雪](https://zh.wikipedia.org/wiki/雪暴 "wikilink")、[寒潮](../Page/寒潮.md "wikilink")、[大风](https://zh.wikipedia.org/wiki/大风 "wikilink")、[沙尘暴](../Page/沙尘暴.md "wikilink")、[高](https://zh.wikipedia.org/wiki/高 "wikilink")[温](https://zh.wikipedia.org/wiki/温 "wikilink")、[干旱](../Page/干旱.md "wikilink")、[雷电](https://zh.wikipedia.org/wiki/雷电 "wikilink")、[冰雹](../Page/冰雹.md "wikilink")、[霜冻](https://zh.wikipedia.org/wiki/霜冻 "wikilink")、[大雾](https://zh.wikipedia.org/wiki/大雾 "wikilink")、[霾](../Page/霾.md "wikilink")、[道路](../Page/道路.md "wikilink")[结](https://zh.wikipedia.org/wiki/结 "wikilink")[冰](../Page/冰.md "wikilink")等十四类。并按照灾害预警重要性程度标注颜色，由低到高依次为蓝色、黄色、橙色、红色。级别越高表明灾害越严重。此外，各地方气象局也会自行制定不同的气象灾害预警信号体系\[4\]。

## 道路结冰

道路结冰预警信号分三级，分别以黄色、橙色、红色表示\[18\]。

| 等级 | 信号名称       | 信号标志                                                                                                                                                         | 信号含义                                        |
| -- | ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------- |
| Ⅲ  | 道路结冰黄色预警信号 | [Yellow_road_icing_alert_-_China.svg](https://zh.wikipedia.org/wiki/File:Yellow_road_icing_alert_-_China.svg "fig:Yellow_road_icing_alert_-_China.svg") | 当路表温度低于0℃，出现降水，12小时内可能出现对交通有影响的道路结冰。        |
| Ⅱ  | 道路结冰橙色预警信号 | [Orange_road_icing_alert_-_China.svg](https://zh.wikipedia.org/wiki/File:Orange_road_icing_alert_-_China.svg "fig:Orange_road_icing_alert_-_China.svg") | 当路表温度低于0℃，出现降水，6小时内可能出现对交通有较大影响的道路结冰。       |
| Ⅰ  | 道路结冰红色预警信号 | [Red_road_icing_alert_-_China.svg](https://zh.wikipedia.org/wiki/File:Red_road_icing_alert_-_China.svg "fig:Red_road_icing_alert_-_China.svg")          | 当路表温度低于0℃，出现降水，2小时内可能出现或者已经出现对交通有很大影响的道路结冰。 |

## 海上大风

海上大风预警信号分三级，分别以蓝色、黄色、橙色表示。该预警信号的发布以国家标准《海上大风预警等级》（GB/T20958-2011）为标准。

| 等级 | 信号名称       | 信号含义                                              |
| -- | ---------- | ------------------------------------------------- |
| Ⅳ  | 海上大风蓝色预警信号 | 预报海区未来48小时内可能出现7\~8级风力（或阵风9\~11级）时，或者已经出现并将持续时。   |
| Ⅲ  | 海上大风黄色预警信号 | 预报海区未来48小时内可能出现9\~10级风力（或阵风11\~12级）时，或者已经出现并将持续时。 |
| Ⅱ  | 海上大风橙色预警信号 | 预报海区未来48小时内可能出现11级以上风力（或阵风达13级以上）时，或者已经出现并将持续时。   |

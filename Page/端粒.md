**端粒**（）是[真核生物](../Page/真核生物.md "wikilink")[染色體末端的](https://zh.wikipedia.org/wiki/染色體 "wikilink")[DNA重複序列](https://zh.wikipedia.org/wiki/DNA重複序列 "wikilink")，作用是保持染色體的完整性和控制細胞分裂週期。 由於[DNA複製的機制](https://zh.wikipedia.org/wiki/DNA複製 "wikilink")，每次染色體複製後，上的染色體末端必無法被複製。\[1\]因此，真核生物在染色體末端演化出端粒以作為可被重複遺棄的片段。\[2\]一旦端粒消耗殆盡，細胞將會立即啟動[凋亡機制](https://zh.wikipedia.org/wiki/凋亡 "wikilink")。因此，端粒被推測和[細胞老化有明顯的關係](https://zh.wikipedia.org/wiki/細胞老化 "wikilink")。人體的部分細胞，例如[精原母細胞](https://zh.wikipedia.org/wiki/精原母細胞 "wikilink")、[癌症](../Page/癌症.md "wikilink")細胞等，含有[端粒酶](../Page/端粒酶.md "wikilink")，能在DNA末端接上新的端粒片段，其端粒不會隨著分裂次數增加而縮短，因此能無限複製。

## 概念

端粒是存在於[真核細胞線狀染色體末端的一小段DNA](https://zh.wikipedia.org/wiki/真核細胞 "wikilink")-蛋白質複合體，它與[端粒結合蛋白一起構成了特殊的](https://zh.wikipedia.org/wiki/端粒結合蛋白 "wikilink")“帽子”結構，能夠維持染色體的完整和控制細胞分裂週期。

端粒DNA是由簡單的DNA高度重複序列組成的，染色體末端沿著5'到3' 方向的鏈富含 GT。在[酵母](../Page/酵母.md "wikilink")和人中，端粒序列分別為C1－3A/TG1－3和TTAGGG/CCCTAA，並有許多蛋白與端粒DNA結合。端粒DNA主要功能有：第一，保護染色體不被[核酸酶](../Page/核酸酶.md "wikilink")降解；第二，防止染色體相互融合；第三，為[端粒酶](../Page/端粒酶.md "wikilink")提供底物，解決DNA複製的末端隱縮，保證染色體的完全複製。端粒、著絲粒和復制原點是染色體保持完整和穩定的三大要素。同時，端粒又是基因調控的特殊位點, 常可抑制位於端粒附近基因的轉錄活性（稱為端粒的位置效應，TPE）。在大多真核生物中，端粒的延長是由端粒酶催化的，另外，[重組機制也介導端粒的延長](https://zh.wikipedia.org/wiki/重組機制 "wikilink")。

## 发现和历史

1.  20世纪30年代，Muller发现被X射线打断的果蝇[染色体其末端存在一种特殊序列](https://zh.wikipedia.org/wiki/染色体 "wikilink")，该序列与常染色体相较极稳定，故根据希腊文将其命名为「端粒」（Telomere）。
2.  20世紀70年代，Olovnikov提出假設，認為染色體末端序列的丟失能夠導致細胞退出增殖週期，但並沒有直接證據證實這一假說的成立。
3.  1978年，Blackburn和Greider等[克隆](../Page/克隆.md "wikilink")出四膜蟲端粒結構<sup>\[1\]</sup>，證明為[串聯線性核苷酸序列](https://zh.wikipedia.org/wiki/串聯線性核苷酸序列 "wikilink")，組成為5′—GGGGGTT —3′。後來實驗又證明了[脊椎動物的端粒均含有豐富的](https://zh.wikipedia.org/wiki/脊椎動物 "wikilink")[鳥嘌呤重複序列](https://zh.wikipedia.org/wiki/鳥嘌呤 "wikilink")。
4.  1985年，Greider等发现[端粒酶](../Page/端粒酶.md "wikilink")，可用于给端粒[DNA加尾](https://zh.wikipedia.org/wiki/DNA "wikilink")。

## 结构

端粒（telomere）是由许多成串短的重复序列所组成。该重复序列通常一条链上富含G（G-rich），而其互补链上富含C（C-rich）。一个基因组内的所有端粒都是由相同的重复序列组成，但不同物种的端粒的重复序列是不同的。

| 物种                        | 端粒DNA重复序列 |
| ------------------------- | --------- |
| 酵母                        |           |
| Saccharomyces cerevisiae  | G1-3T     |
| Schizosaccharomyces pombe | G2-5TTAC  |
| 原生動物                      |           |
| 四膜虫                       | GGGGTT    |
| Dictyostelium             | G1-8A     |
| 植物                        |           |
| 拟南芥(Arabidopsis)          | TTTAGGG   |
| 哺乳動物                      |           |
| 人                         | TTAGGG    |

例如，原生動物四膜蟲端粒的重複單位為TTGGGG（僅列一條鏈的序列）；哺乳類和其他脊椎動物的端粒為TTAGGG，串聯重複500\~3000次，序列長度在2kb到20kb之間不等。 TG鏈常比AC鏈更長些，形成3′單鏈末端。端粒的功能為穩定染色體末端結構，防止染色體間末端連接，並可補償滯後鏈5′末端在消除RNA引物後造成的空缺。原核生物的染色體是環狀的，其5′最末端岡崎片段的RNA引物被除去後可藉助另半圈DNA鏈向前延伸來填補。但是真核生物線性染色體在復制後，不能像原核生物那樣填補5′末端的空缺，從而會使5′末端序列從此而縮短。真核生物通過形成端粒結構來解決這個問題。複製使端粒5′末端縮短，而[端粒酶](../Page/端粒酶.md "wikilink")（telomerase）可外加重複單位到5′末端上，結果維持端粒一定長度。

### 结合蛋白

端粒除了含有重複DNA序列外，還包含有特殊的「非[核小體](../Page/核小體.md "wikilink")蛋白」，即端粒結合蛋白。根據該蛋白的結合特性分為兩類，一類與端粒重複序列特異性結合，在維持端粒長度方面起到重要作用<sup>\[3\~5\]</sup>，並且對端粒具有保護和調節作用<sup>\[6\]</sup>；另一類與3′末端的單股突出結合，用於合成染色體末端的帽子結構以及調節端粒酶活性<sup>\[7\~9\]</sup>。

## 作用

  - 染色体由于融合、[降解](https://zh.wikipedia.org/wiki/降解 "wikilink")、[重排而形成不稳定结构从而威胁到DNA的正确复制和细胞生存](https://zh.wikipedia.org/wiki/重排 "wikilink")，端粒的存在能够保护染色体免于化学修饰、被核酸降解以及因端粒作用而产生的威胁。

<!-- end list -->

  - 端粒使染色體末端區域形成[異染色質](../Page/異染色質.md "wikilink")，在細胞進行[減數分裂的過程中結合到](https://zh.wikipedia.org/wiki/減數分裂 "wikilink")[核膜](../Page/核膜.md "wikilink")一特定區域，使得染色體尋找[同源染色體](../Page/同源染色體.md "wikilink")啟動和配對的過程更加容易，並且保證了染色體分離的正確性<sup>\[10\~13\]</sup>。

<!-- end list -->

  - 在细胞有丝分裂的过程中，端粒会随着分裂次数的增加逐渐缩短，当端粒缩短到一定程度时便无法继续维持染色体的稳定，细胞最终死亡，故而能够根据端粒的长度预测细胞的寿命。但是在[生殖细胞](../Page/生殖细胞.md "wikilink")中，端粒的长度不随细胞分裂而缩短，推测是由于生殖细胞中富含端粒酶的缘故。

[胚胎發生中](https://zh.wikipedia.org/wiki/胚胎發生 "wikilink")，[桑椹胚](https://zh.wikipedia.org/wiki/桑椹胚 "wikilink")－胚泡转型（the morula to blastocyst transition）伴随着端粒长度的重新设定。实验建立在在[小鼠和牛的胚胎实验](https://zh.wikipedia.org/wiki/小鼠 "wikilink")。这些胚胎不管是来自自然受精还是[体外受精](https://zh.wikipedia.org/wiki/体外受精 "wikilink")，甚至是由端粒已经缩短的体细胞-{zh-cn:克隆;zh-tw:複製;}-发育而成，都有端粒长度的重新设定这一过程。

[端粒酶](../Page/端粒酶.md "wikilink")對這一過程有重要作用，一旦缺少端粒酶，該過程就不能發生。桑椹胚－胚泡轉型是植入前發育的重要階段，能夠導致首次二細胞系分化――[內細胞團](../Page/內細胞團.md "wikilink")（inner cell mass）和[滋養層細胞](https://zh.wikipedia.org/wiki/滋養層細胞 "wikilink")（trophoblast）。端粒的長度重製能夠保證代與代之間的端粒正常，也可能和出生和的老化與腫瘤發生有關。端粒酶，是基本的核蛋白-{zh-cn:[逆轉錄酶](https://zh.wikipedia.org/wiki/逆轉錄酶 "wikilink");zh-tw:[反轉錄酶](https://zh.wikipedia.org/wiki/反轉錄酶 "wikilink");}-，可將端粒DNA加至[真核細胞染色體末端](https://zh.wikipedia.org/wiki/真核細胞 "wikilink")。端粒在不同物種細胞中對於保持染色體穩定性和細胞活性有重要作用，端粒酶能延長縮短的端粒（縮短的端粒其細胞複製能力受限），從而增強體外細胞的增殖能力。端粒酶在正常人體組織中的活性被抑制，在[腫瘤中被重新激活](https://zh.wikipedia.org/wiki/腫瘤 "wikilink")，端粒酶可能參與[惡性轉化](../Page/惡性轉化.md "wikilink")。端粒酶在保持端粒穩定、[基因組](../Page/基因組.md "wikilink")完整、細胞長期的活性和潛在的繼續增殖能力等方面有重要作用。

对这个过程的研究可以防止体细胞克隆中的端粒异常，也有助于通过[移植和](https://zh.wikipedia.org/wiki/移植 "wikilink")[基因治疗的方法解决退型性疾病](https://zh.wikipedia.org/wiki/基因治疗 "wikilink")。

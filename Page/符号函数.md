**符號函數**（**Sign
function**，簡稱**sgn**）是一個[邏輯函數](../Page/邏輯.md "wikilink")，用以判斷[實數的正](../Page/實數.md "wikilink")[負號](../Page/負數.md "wikilink")。為避免和英文讀音相似的[正弦函數](../Page/正弦函數.md "wikilink")（sine）混淆，它亦稱為**Signum
function**。其定義為：

\[\sgn x = \left\{ \begin{matrix}
-1 & : &  x < 0 \\
0 & : &  x = 0 \\
1 & : &  x > 0 \end{matrix} \right.\]

## 性质

用[艾佛森括號定義](../Page/艾佛森括號.md "wikilink")：

\[\sgn x = -[x < 0] + [x > 0]\]

任何[實數都可以表示為其](../Page/實數.md "wikilink")[絕對值和符號函數的積](../Page/絕對值.md "wikilink")：

  -
    \(x=(\sgn x)|x|\)

若x不為零，可以由上式得出符號函數的另一個定義：

  -
    \(\sgn x = {x \over |x|}\)

符號函數是絕對值函數的導數：

\[\frac{d|x|}{dx} = \frac{x}{|x|} = \sgn x\]

除了在0，符號函數可微分，其導數為0。透過一般化微分概念，可以說符號函數的導數是[狄拉克δ函數的兩倍](../Page/狄拉克δ函數.md "wikilink")：

\[{d \ \sgn x \over dx} = 2 \delta (x)\]

它和[單位步階函數的關係](../Page/單位步階函數.md "wikilink")：

  -
    \(\sgn x = 2 H_{1/2}(x) - 1\)

## 推广到复数

符號函數可以推廣到[複數](../Page/複數.md "wikilink")：對於任意\(z \in \mathbb{C} \backslash \{0\}\)，

\[\sgn z = \frac z{|z|}\] 对于任何*z* ∈ \(\mathbb{C}\)，除了*z* =
0以外。复数*z*的符号函数，是[复平面上中心为](../Page/复平面.md "wikilink")[原点的](../Page/原点.md "wikilink")[单位圆上距离](../Page/单位圆.md "wikilink")*z*最近的点。那么，对于*z*
≠ 0，有：

\[\sgn z = \exp(i\arg z)\,,\] 其中arg表示[辐角](../Page/辐角.md "wikilink")。

符号函数在复数范围的另外一个推广是*csgn*函数，定义为：

\[\operatorname{csgn}(z)= \begin{cases}
 1 & \text{if } \Re(z) > 0 \lor (\Re(z) = 0 \land \Im(z) > 0), \\
 -1 & \text{if } \Re(z) < 0 \lor (\Re(z) = 0 \land \Im(z) < 0), \\
 0 & \text{if } \Re(z)=\Im(z)=0.
\end{cases}\] 即是在一四象限及 y 轴正半轴为正一、二三象限及 y 轴负半轴为负一、原点为零。
对于 csgn，我们有（除了*z* = 0以外）：

\[\operatorname{csgn}(z) = \frac{z}{\sqrt{z^2}} = \frac{\sqrt{z^2}}{z}.\]

**第66屆英國電影學院獎**頒獎典禮於2013年2月10日於[倫敦](https://zh.wikipedia.org/wiki/倫敦 "wikilink")[皇家歌劇院舉行](https://zh.wikipedia.org/wiki/皇家歌劇院 "wikilink")，表揚2012年[英國國內和海外傑出的電影](https://zh.wikipedia.org/wiki/英國 "wikilink")。 \[1\]

## 時間表

| 日期          | 事件                       |
| ----------- | ------------------------ |
| 2012年11月14日 | 截止遞交及收取電影人事表格            |
| 2012年11月26日 | 向電影學院可投票成員發放已報名電影的詳請     |
| 2012年12月12日 | 第一輪投票在10:00 GMT開始        |
| 2013年1月4日   | 第一輪投票在18:00 GMT結束        |
| 2013年1月9日   | 公佈提名名單，第二輪投票在18:00 GMT開始 |
| 2013年1月6日   | 第二輪投票在18:00 GMT結束        |
| 2013年2月8日   | 已報名電影須在此日期前全面公映          |
| 2013年2月10日  | 第66屆英國電影學院獎頒獎典禮          |

**墨球**又叫**笔迹球**，是在[Windows
Vista中新增的一项](../Page/Windows_Vista.md "wikilink")[電腦游戏](https://zh.wikipedia.org/wiki/電腦游戏 "wikilink")\[1\]，包含在入门版、家庭基本版、家庭高級版、商業版、企業版和旗艦版中，最初是出现在[Windows
XP Tablet PC
Edition](https://zh.wikipedia.org/wiki/Windows_XP_Tablet_PC_Edition#Tablet_PC_Edition "wikilink")
2005中。\[2\]遊戲玩法為使用[手写板或](https://zh.wikipedia.org/wiki/手写板 "wikilink")[滑鼠](https://zh.wikipedia.org/wiki/滑鼠 "wikilink")，藉由畫線讓球直接進入相應顏色的洞中。在Windows
XP Tablet PC
Edition中，需要灵活地使用[手写板](https://zh.wikipedia.org/wiki/手写板 "wikilink")，而鼠标指针在游戏窗口中是隐藏的，不过按两下[Alt键](https://zh.wikipedia.org/wiki/Alt键 "wikilink")，鼠标指针就会出现在游戏窗口中。在[Windows
Vista中](../Page/Windows_Vista.md "wikilink")，该游戏也可用鼠标操作。\[3\]

在[Windows 7中该游戏被取消](https://zh.wikipedia.org/wiki/Windows_7 "wikilink")。

## 规则

在限定的[时间](../Page/时间.md "wikilink")内将球通过[墨迹的遮挡打入同色的](https://zh.wikipedia.org/wiki/墨迹 "wikilink")[洞口](https://zh.wikipedia.org/wiki/洞口 "wikilink")，墨迹在遮挡一次后会消失；带颜色的砖块只有用相同颜色的球才能打碎。如果时间用完，或有球打进非相同颜色的洞内，游戏立刻结束。

## 记分

| 球的颜色 | 得分   |
| ---- | ---- |
| 灰色   | 0    |
| 红色   | 200  |
| 蓝色   | 400  |
| 绿色   | 800  |
| 金色   | 1600 |

## 提示

  - 如果时间用完，或有球打进非相同颜色的洞内，或者你调整了游戏难度，游戏立刻结束。
  - 灰色是一个特殊的颜色，灰色球可以打进任何颜色的洞中，任何球都可以打进灰色球洞，但不会得分。
  - 球的反射与[光的反射定律类似](https://zh.wikipedia.org/wiki/光的反射定律 "wikilink")。
  - 墨迹在遮挡一次后会消失，所以一笔墨迹只能影响一个球。
  - 随着[关卡的增加](../Page/關卡_\(電子遊戲\).md "wikilink")，难度会增大，这具体表现在球速的增加，球和洞数量的增加，包括游戏盘的布局。

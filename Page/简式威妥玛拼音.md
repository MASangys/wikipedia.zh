**簡式威妥瑪**（）是用於拼寫[官话](../Page/官话.md "wikilink")（[現代標準漢語](../Page/現代標準漢語.md "wikilink")）的[威妥瑪](https://zh.wikipedia.org/wiki/威妥瑪 "wikilink")[羅馬拼音系統的修改版本](https://zh.wikipedia.org/wiki/羅馬拼音 "wikilink")。[瑞典](../Page/瑞典.md "wikilink")[语言学家](https://zh.wikipedia.org/wiki/语言学家 "wikilink")[奥洛夫·安德森贝蒂尔设计并在](https://zh.wikipedia.org/wiki/Olov_Bertil_Anderson "wikilink")1970年第一次发布了这一系统。简式威妥玛使用音调拼读，换言之，它通过修改音节中的字母来表示[音调区别](https://zh.wikipedia.org/wiki/現代標準漢語#音韵 "wikilink")。以此种方式表示音调的[汉字拉丁化](../Page/汉字拉丁化.md "wikilink")系统仅有简式威妥玛和[國語羅馬字](../Page/國語羅馬字.md "wikilink")。其他所有系统都使用[讀音符號或数字来表示音调](https://zh.wikipedia.org/wiki/讀音符號 "wikilink")。

## 拼读惯例

安德森对[威妥瑪拼音](../Page/威妥瑪拼音.md "wikilink")所做的重要改变之一，是将跟随[送气音的](https://zh.wikipedia.org/wiki/送气音 "wikilink")[撇号](../Page/撇号.md "wikilink")以一个 <sup>h</sup> 替换。\[1\]这一曾被用于[理雅各拼音](../Page/理雅各拼音.md "wikilink")中的修改，也被[李約瑟在其所著](https://zh.wikipedia.org/wiki/李約瑟 "wikilink")《[中国的科学与文明](../Page/中国科学技术史_\(李约瑟\).md "wikilink")》中被采用。\[2\]下表揭示了拼读的区别。

|                                      |           |                                        |                                        |
| ------------------------------------ | --------- | -------------------------------------- | -------------------------------------- |
| **[威妥玛](../Page/威妥玛.md "wikilink")** | **简式威妥玛** | **[汉语拼音](../Page/汉语拼音.md "wikilink")** | **[國際音標](../Page/國際音標.md "wikilink")** |
| t'                                   | th        | t                                      | tʰ                                     |
| p'                                   | ph        | p                                      | pʰ                                     |
| k'                                   | kh        | k                                      | kʰ                                     |
| ch'                                  | chh       | q/ch                                   | tɕʰ/tʂʰ                                |

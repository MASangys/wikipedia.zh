**簡譜**（）為[記譜法之一](https://zh.wikipedia.org/wiki/記譜法 "wikilink")，主要以[數字](../Page/數字.md "wikilink")作表達，亦稱為**數字譜**。。其起源於[18世紀的](https://zh.wikipedia.org/wiki/18世紀 "wikilink")[法國](https://zh.wikipedia.org/wiki/法國 "wikilink")\[1\]，後經[德國人改良](https://zh.wikipedia.org/wiki/德國人 "wikilink")，遂成今日之貌。

另外，在德文裡，其名為「**」，意若「數字系統」（）。

## 歷史

1665年[巴黎](../Page/巴黎.md "wikilink")[方濟會修士Jean](https://zh.wikipedia.org/wiki/方濟會 "wikilink")-Jacques Souhaitty的著作《[學習素質和音樂的新方法](https://zh.wikipedia.org/wiki/學習素質和音樂的新方法 "wikilink")》提出把[阿拉伯數字](https://zh.wikipedia.org/wiki/阿拉伯數字 "wikilink")1至7代入唱名「*do*」、「*re*」、「*mi*」、「*fa*」、「*sol*」、「*la*」、「*ti*」，並加點表明高音和低音。

17世紀，[法國天主教神父Jean](https://zh.wikipedia.org/wiki/法國 "wikilink")-Jacques Souhaitty，在指導信眾唱[讚美詩時](https://zh.wikipedia.org/wiki/讚美詩 "wikilink")，感受到[五線譜](../Page/五線譜.md "wikilink")教學的困難，就想要創造一種更便捷的辦法。於是乎他以阿拉伯數字1234567來代替Do Re Mi Fa So La Ti，並在1665年和1679年，分別發表了「學習音樂的新方法」及「用數字譜唱教會歌曲的實驗」兩篇論文，只可惜當時未受到音樂家的重視，反倒是引起衛道人士的非議，新方法於是胎死腹中。

18世紀中葉，瑞士裔法國[思想家](https://zh.wikipedia.org/wiki/思想家 "wikilink")、[哲學家](https://zh.wikipedia.org/wiki/哲學家 "wikilink")、[政治理論家](https://zh.wikipedia.org/wiki/政治理論家 "wikilink")[盧梭曾編寫](../Page/让-雅克·卢梭.md "wikilink")100多首歌曲，獨幕歌劇《鄉村的算命師》，獲得國王青睞，演出400多場。他主張「音樂是語言的昇華」、「人工堆砌的，都是違背自然。」「旋律是音樂的主角，和聲是襯托的配角。」他還協助朋友編輯音樂辭典，於1767年出版。當時的精緻音樂仍是貴族們的休閒享受，盧梭崇尚「全民音樂」，於1742年在[法國科學院發表並講解此一發明](https://zh.wikipedia.org/wiki/法國科學院 "wikilink")，并且大力呼籲使用數字簡譜，但他的「業餘身分」遭到音樂學院保守教授們的諷刺。

多年之後，簡譜在數學老師Pierre Galin（1786年─1821年）和Aimé Paris（1798年─1866年）的整理之下，漸漸受到重視。到了19世紀，音樂教育家E. J Cheve(1804─1864）再加以改進，得到法國政府教育機構的正式認可，稱為Galin-Paris-Cheve記譜法，簡稱Cheve記譜法，俗稱「數字簡譜」或「簡譜」。這時簡譜記譜法則已經完備，成為當時正統的音樂工具。

### 音長

通常只有數字的是[四分音符](https://zh.wikipedia.org/wiki/四分音符 "wikilink")。數字下加一條橫線，就可令四分音符的長度減半，即成為[八分音符](https://zh.wikipedia.org/wiki/八分音符 "wikilink")；兩條橫線可令八分音符的長度減半，即成為[十六分音符](https://zh.wikipedia.org/wiki/十六分音符 "wikilink")，以此類推；簡單來説，下加横綫數目與五綫譜的符尾數目相對應。數字後方的橫線延長音符，每加一條橫線延長一個[四分音符的長度](https://zh.wikipedia.org/wiki/四分音符 "wikilink")。

正如五綫譜的[附點一樣](https://zh.wikipedia.org/wiki/附點 "wikilink")，數字後方加一點會將音符長度增加一半。如果低八度的下加点与音长的下加线同时使用，则点在线的下面。

### 休止符

[休止符用](https://zh.wikipedia.org/wiki/休止符 "wikilink")「0」來表示。比四分休止符長，就只需在每一個四分休止符再多加一個「0」，不需在後方加上橫線。若整個小節均是休止狀態的話，就在每個四分音符加一個「0」或是寫成「| \> 0 \< |」，就像是3拍4可以寫成「| 0 0 0 |」或「| \> 0 \< |」，而4拍4可以寫成「| 0 0 0 0 |」或「| \> 0 \< |」。

### 無音高音符

就像敲擊樂的樂器那樣是沒有音高的，這時，簡譜就需要用「X」或「x」表達。例如[啦啦隊](../Page/啦啦隊.md "wikilink")開心就拍手時，簡譜上就需如此表達：

`4/4                          　  > >`
`開心就拍手：| X  X  `<u>`X X`</u>` X  | `<u>`X X X X`</u>` `<u>`0 X`</u>` X ||`

### 小節線

正如五線譜一樣，[小節與小節之間以](https://zh.wikipedia.org/wiki/小節 "wikilink")[小節線分隔](https://zh.wikipedia.org/wiki/小節線 "wikilink")，樂曲終止以一粗一細的直線表示(但是一般不分粗細)；重複樂段以縱線後兩點表示開始，縱線前兩點表示終止。

### 拍號

[拍號](../Page/拍號.md "wikilink")的表達方式為：「2/4」、「3/4」、「4/4」、「6/8」等等。拍號一般都是置於[調號的後方](https://zh.wikipedia.org/wiki/調號 "wikilink")。

### 變調與調號

表示[調號有兩種方法](https://zh.wikipedia.org/wiki/調號 "wikilink")：可以直接寫「C調」，「C小調」或「Key: C」，「Key: Cm」。亦可寫清楚[主音](../Page/主音.md "wikilink")代表哪一個音，例如「1 = C」，就是C大調（大調以1为主音）。「6 = C」就是[C小調](https://zh.wikipedia.org/wiki/C小調 "wikilink")（小调以6为主音）。「2 = D」就是D[多利安調](https://zh.wikipedia.org/wiki/多利安調 "wikilink")（多利安調以2为主音）。

五線譜有[變音記號](https://zh.wikipedia.org/wiki/變音記號 "wikilink")，簡譜同樣有變音記號。在簡譜中，若需表達升降音，就把符號加在數字1至7的前方，讓該音升高或降低。若要把變音記號加在調號跟[和弦](../Page/和弦.md "wikilink")CDEFGAB七個字母，就須加在它們的後方。

在五綫譜中，C小調曲調表達爲B，E，A三個音各降低[半音](https://zh.wikipedia.org/wiki/半音 "wikilink")。在表達和聲C小調的导音時，會把一個[還原號放在B](https://zh.wikipedia.org/wiki/還原號 "wikilink")♭的音符前面。而在簡譜中用「6 = C」記譜方式表達C小調時，這個导音寫作♯5，因爲5本身不能被還原，這與五線譜中B已被降半音而需還原的情況不同。

### 延音線、圓滑線與其他音樂符號

簡譜[圓滑線](../Page/圓滑線.md "wikilink")跟[連結線都是跟五線譜相同](https://zh.wikipedia.org/wiki/連結線 "wikilink")，为圆弧线。若线太长，允许换为平头线，以保持行高平整，但一定得置在數字上方。

表達[跳音](https://zh.wikipedia.org/wiki/跳音 "wikilink")（staccato）有時很容易會跟高八度的那一點混淆，因此在表達跳音時把那一點寫得粗點及低點，甚至使用[大斷奏的寫法](https://zh.wikipedia.org/wiki/大斷奏 "wikilink")。一种避免混淆的方法是用“×”代替跳音点。

## 使用情況

### 中國大陸

在中國大陸很多人懂簡譜，甚至比懂[五線譜](../Page/五線譜.md "wikilink")的人多\[2\]。有些樂譜出版物甚至同時使用五線譜和簡譜。

簡譜從歐洲傳到[日本](../Page/日本.md "wikilink")，透過留學生[李叔同](../Page/李叔同.md "wikilink")等在1904年引入中國\[3\]，連保守的音樂家都欣然接受，漸漸放棄[工尺譜或口傳](https://zh.wikipedia.org/wiki/工尺譜 "wikilink")，英商[百代唱片公司出版的](https://zh.wikipedia.org/wiki/百代唱片公司 "wikilink")「時代曲」歌單，也以簡譜印製。

### 香港

[香港](../Page/香港.md "wikilink")的中小學音樂教育中幾乎不會教授簡譜，但會大量地教授並使用[五線譜](../Page/五線譜.md "wikilink")，音樂教科書中的曲子也使用五線譜寫成。

### 台灣

[日治時代](../Page/台灣日治時期.md "wikilink")，台灣[公學校](../Page/公學校.md "wikilink")「唱歌」教材（音樂課程的前身）有五線譜，也有簡譜，流行歌壇則受日本影響，全部以簡譜建立「全民音樂」的基礎，培養大量喜歡音樂的群眾。

1945年後，[台灣學校的音樂教育一直排斥數字簡譜](https://zh.wikipedia.org/wiki/台灣 "wikilink")，在教育部頒布的課程標準中，明示「不得使用簡譜」，直到1991年，開放民間出版社編印的「審定本」時，才於國小6年級音樂課本介紹[盧梭和簡譜](https://zh.wikipedia.org/wiki/盧梭 "wikilink")。有學者認為當許多人無法理解、使用五線譜而放棄享受音樂的權利時，簡譜絕對能幫助開闢更多的音樂人口，達到「全民音樂」的最高目標\[4\]。

## \-{zh-cn:计算机;zh-tw:電腦}-简谱排版软件

由于简谱在中国使用较为广泛，市面上存在的大部分简谱编辑排版软件都是中文界面的软件，它们大多提供“[所见即所得](https://zh.wikipedia.org/wiki/所见即所得 "wikilink")”的操作界面\[5\]\[6\]\[7\]\[8\]。这些软件大都只支持简谱排版，不能进行五线谱和简谱的混合排版，但[Sibelius中有一个插件可以实现简谱与五线谱的混合排版](https://zh.wikipedia.org/wiki/Sibelius_\(software\) "wikilink")\[9\]。在[GNU LilyPond中](https://zh.wikipedia.org/wiki/GNU_LilyPond "wikilink")，也可以通过一些技术性的手段，排版出简谱样式的乐谱\[10\]\[11\]\[12\]。

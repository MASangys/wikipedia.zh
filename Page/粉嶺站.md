{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Fanling Station 2013 part2.JPG | caption1 = 車站外觀 }} **粉嶺站**（），位於[香港](../Page/香港.md "wikilink")[新界](../Page/新界.md "wikilink")[北區](../Page/北區_\(香港\).md "wikilink")[粉嶺](../Page/粉嶺.md "wikilink")的[港鐵](../Page/港鐵.md "wikilink")[東鐵綫](../Page/東鐵綫.md "wikilink")一個地面車站，它與[沙田站](../Page/沙田站.md "wikilink")、[旺角東站](../Page/旺角東站.md "wikilink")同為香港三座最早通車、並是香港最早啟用而仍然提供鐵路服務的三個車站之一。

寒

### 大堂

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Fanling Station 2017 03 part2.jpg | caption1 = 南面大堂 }} 粉嶺站大堂位於月台之上，因此大堂中央為非收費區的出入口，收費區與閘囗皆分為南北兩面，大堂北面收費區為舊大堂，設有[洗手間及售票處](../Page/廁所.md "wikilink")（[兩鐵合併](../Page/兩鐵合併.md "wikilink")後改稱「客務中心」）；與[上水站未擴建前的大堂設計相似](../Page/上水站_\(香港\).md "wikilink")。而南面的收費區則為後期加建部份，原不與北面收費區互通，乘客需利用月台往返南北面大堂的收費區；現已改建為完全接通。

大堂內有不同類型的商店，如[便利店](../Page/便利商店.md "wikilink")、[零食](../Page/零食.md "wikilink")店、[麵包糕餅店](https://zh.wikipedia.org/wiki/麵包糕餅店 "wikilink")、[涼茶](../Page/涼茶.md "wikilink")專賣店及外賣[壽司店等](https://zh.wikipedia.org/wiki/壽司 "wikilink")。本站設有不少自助服務設施：包括[自動櫃員機](../Page/自動櫃員機.md "wikilink")、[自動售賣機](../Page/自動販賣機.md "wikilink")\[1\]以及[香港郵政](../Page/香港郵政.md "wikilink")[郵箱](../Page/郵箱.md "wikilink")\[2\]\[3\]。 大堂非付費區設有「[會員服務站](../Page/會員服務站.md "wikilink")」，供乘客享用港鐵友禮會的會員優惠\[4\]。

在[互聯網服務上](https://zh.wikipedia.org/wiki/互聯網 "wikilink")，粉嶺站大堂及月台均設有由[電訊盈科](../Page/電訊盈科.md "wikilink")提供的[Wi-Fi](../Page/Wi-Fi.md "wikilink")熱點\[5\]，乘客可在候車時以Wi-Fi功能的電子器材連接互聯網。

### 月台

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Fanling Station 2018 11 part1.jpg | caption1 = 1號月台﹙往羅湖／落馬洲方向﹚ | image2 = Fanling Station 2018 11 part2.jpg | caption2 = 2號月台﹙往紅磡方向﹚ }} 粉嶺站設有2個[側式月台](../Page/側式月台.md "wikilink")，位於地面，不設[月台幕門](../Page/月台幕門.md "wikilink")。粉嶺站月台中央設有扶手電梯通往大堂，月台兩旁均設有緊急出口。

藝術設計上，本站採用歷史建築元素及區內兩條原居民圍村元素：前方有三個圓形圖案是[粉嶺正圍的圍門](../Page/粉嶺圍.md "wikilink")，而後方「老圍」二字的則是[龍躍頭老圍的圍門](https://zh.wikipedia.org/wiki/老圍_\(龍躍頭\) "wikilink")。粉嶺彭氏和龍躍頭鄧氏皆爲[新界五大族](../Page/新界五大氏族.md "wikilink")，是區內規模最大的兩個圍村族群。

## 接駁交通

粉嶺站設於粉嶺市中心，接駁交通十分完善。車站旁設有[公共運輸交匯處](../Page/公共運輸交匯處.md "wikilink")，內有[專線小巴站及](https://zh.wikipedia.org/wiki/專線小巴 "wikilink")[的士站](https://zh.wikipedia.org/wiki/的士站 "wikilink")。到2013年起，A2出口外設「雙層單車泊架」系統，方便居民利用[單車代步](https://zh.wikipedia.org/wiki/單車 "wikilink")。在[百和路](../Page/百和路.md "wikilink")亦有一[巴士站](../Page/巴士站.md "wikilink")，方便乘客轉乘其他公共交通工具前往各區。

## 利用狀況

{{ original research | 章節 }} 粉嶺站附近有多個大型屋苑及公共屋邨，加上粉嶺南和聯和墟一帶居民皆依賴鐵路出入市區，所以其使用量一直高企。在[清明](../Page/清明.md "wikilink")和[重陽節](../Page/重陽節.md "wikilink")期間，九巴更提供接駁巴士76S前往和合石墳場。本站亦鄰近[蓬瀛仙館](../Page/蓬瀛仙館.md "wikilink")，令其使用量更進一步增加。

與[上水站一樣](https://zh.wikipedia.org/wiki/上水站 "wikilink")，憑一般成人[八達通](../Page/八達通.md "wikilink")卡由港鐵各站（[東鐵綫](../Page/東鐵綫.md "wikilink")新界路段及[馬鞍山綫](../Page/馬鞍山綫.md "wikilink")除外）乘搭東鐵綫普通等往來[羅湖或](../Page/羅湖站_\(香港\).md "wikilink")[落馬洲](../Page/落馬洲站.md "wikilink")，在粉嶺站出閘再入閘繼續餘下旅程，可節省3.9元車費。

## 歷史

[粉嶺車站.jpg](https://zh.wikipedia.org/wiki/File:粉嶺車站.jpg "fig:粉嶺車站.jpg")

### 沙頭角支線轉車站

1910年，[九廣鐵路](../Page/九廣鐵路.md "wikilink")英段通車時已經設有粉嶺站，其後[九廣鐵路局](../Page/九廣鐵路局.md "wikilink")興建連接粉嶺站至沙頭角站長約11.67公里的[沙頭角支線](../Page/沙頭角支線.md "wikilink")。沙頭角支線在1912年4月1日通車，共設5個站，路線以沿現今[沙頭角公路](../Page/沙頭角公路.md "wikilink")，由粉嶺站經[洪嶺站](../Page/洪嶺站.md "wikilink")、禾坑站、石涌凹站至沙頭角站。

兩線採用不同寬度路軌和火車，故兩線不能連接。沙頭角支線的粉嶺站和九廣英段的粉嶺站往九龍一方呈90°度交錯。

1927年，沙頭角公路啟用，因沙頭角支線虧蝕，沙頭角支線遂於1928年4月1日停駛，而粉嶺站則改回中途站。

### 九廣鐵路電氣化

[Fanling_Station.jpg](https://zh.wikipedia.org/wiki/File:Fanling_Station.jpg "fig:Fanling_Station.jpg") 1983年7月15日粉嶺站成為九廣鐵路電氣化最後一期的其中一個車站。

### 擴建及翻新

為應付使用此站的乘客數量大幅增加，九鐵於1998年為車站進行擴建工程。新大堂部份位於A、B出入口南面，於1999年啟用，新大堂增設4道扶手電梯、12部出入閘機、客務中心（非收費區及收費區，目前已停用）及5 間非收費區店舖。

到了2001年，原有80年代建成的大堂亦進行重舖地台工程，感覺較現代化。

### 擴闊月台

2001年，九廣鐵路為車站南行頭尾部份月台進行擴闊工程，令人流更暢通。

### 翻新月台

兩鐵合併後，港鐵為東鐵綫各車站月台進行較低成本的翻新工程，以遮掩東鐵綫車站月台殘舊的觀感。工程包括重新為各東鐵綫車站配主題色，並加入車站代表植物，為月台橫樑及柱子翻上月台配色的油漆，貼上印有車站主題植物的圖案、車站附近建築物、車站名書法字的牆紙，加設直立港鐵式站牌、路線圖、指示牌、告示等等。本站被配色為淺綠色，主題植物為馬蹄蘭，街景為粉嶺老圍。

### 翻新大堂

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Fanling Station Concourse 2008.jpg | caption1 = 翻新前的北面大堂（2008年） }} [港鐵公司於](https://zh.wikipedia.org/wiki/港鐵公司 "wikilink")2008年引入嶄新的「融入大自然」建築概念，作為東鐵綫大型車站改善工程的設計方向。繼[大埔墟](../Page/大埔墟站.md "wikilink")、[旺角東及](../Page/旺角東站.md "wikilink")[沙田站](../Page/沙田站.md "wikilink")大型翻新工程竣工後，東鐵綫上水及粉嶺站亦開始進行翻新工程。港鐵公司於2013年進行粉嶺站大堂翻新。 包括：

  - 公共廁所及客務中心；
  - 增加商店數目。商店總面積由454平方米擴充至549平方米；
  - 優化指示牌和廣告牌；
  - 美化大堂天花、牆身及地板。

改善工程於2015年完成。\[7\]

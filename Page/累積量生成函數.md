隨機變數的**累積量生成函數**κ<sub>*n*</sub>*X*是定義為：對動差生成函數（動差母函數）取自然對數的函數，如果符合定義，將如下所示：

\[g(t)=\log(E (e^{tX}))=\sum_{n=1}^\infty\kappa_n \frac{t^n}{n!}=\mu t + \sigma^2 \frac{ t^2}{2} + \cdots\]。

將累積量生成函數*g*(*t*)對t等於零之處微分

\[\begin{align} \kappa_1 &= \mu = g'(0), \\
                     \kappa_2 &= \sigma^2 = g''(0), \\
                              &{} \  \  \vdots \\
                     \kappa_n &= g^{(n)}(0).
       \end{align}\]

累積量生成函數與機率分佈的動差值有很強的關聯性。假如隨機變數*X*存在期望值μ = *E*(*X*)及變異數σ<sup>2</sup> =
*E*((*X* − μ)<sup>2</sup>)，則累積量生成函數*g*(*t*)的一階與二階微分剛好是上述數值：μ =
κ<sub>1</sub>及σ<sup>2</sup> = κ<sub>2</sub>。第*c*個累積量表達的方式為

\[\kappa_n = \langle X^n\rangle_c. \,\]

使用累積量生成函數優於動差值的情況在於獨立變數*X*和*Y*，

\[\begin{align}
g_{X+Y}(t)& =\log(E(e^{t\cdot (X+Y)})) = \log(E(e^{tX}) \cdot E(e^{tY})) \\
& = \log(E(e^{tX})) + \log(E(e^{tY})) = g_X(t) + g_Y(t).
\end{align}\]

如此一來相加累積量的合可表達成累積量的相加，也就是具有加成性。

一個分佈的累積量κ<sub>*n*</sub>可以使用[Edgeworth
series來近似](../Page/Edgeworth_series.md "wikilink")。

有些作者\[1\]\[2\]偏好定義累積量生成函數為對特徵函數取自然對數，或者有人稱為**第二特徵函數，**\[3\]\[4\]

\[h(t)=\log(E (e^{i t X}))=\sum_{n=1}^\infty \kappa_n \frac{(it)^n}{n!}=\mu it - \sigma^2 \frac{ t^2}{2} + \cdots.\,\]

使用此函數的好處在於，即便可能隨機變數*X*是一大變量仍被完整定義。儘管他的累積量生成函數或者是動差母函數是存在的，但在這種情況下，通常不允許被展開成累積量生成函數或者是動差母函數而表達成線性級數數列的模式。[Cauchy
distribution](../Page/Cauchy_distribution.md "wikilink")（也稱作Lorentzian）和[stable
distribution](../Page/stable_distribution.md "wikilink")（與「Lévy
distribution」有關）是生成函數無法被展開的兩個例子。

## 一些離散隨機變數的累積量

  - 退化的隨機變數*X* = 1的累積量生成函數為*g* (*t*) = 1.第一累積量為κ<sub>1</sub> = *g* '(0)
    = 1，其他的累積量為零，κ<sub>2</sub> = κ<sub>3</sub> = κ<sub>4</sub> = ...
    = 0.

<!-- end list -->

  - 退化的隨機變數*X* = μ.每一個累積量是退化的隨機變數*X* = 1的μ倍。其積量生成函數為*g* '(*t*) = μ.
    第一累積量為κ<sub>1</sub> = *g* '(0) = μ，其他的累積量為零，κ<sub>2</sub>
    = κ<sub>3</sub> = κ<sub>4</sub> = ... = 0.

<!-- end list -->

  - [伯努利分佈](../Page/伯努利分布.md "wikilink")，特殊情形為*p* = 1時是退化的隨機變數*X* = 1.累積量生成函數為*g* '(*t*)
    =
    ((*p*<sup> −1</sup>−1)·e<sup>−*t*</sup> + 1)<sup>−1</sup>。第一累積量為κ<sub>1</sub> = *g* '(0)
    = *p*，κ<sub>2</sub> = *g* ' '(0) = *p*·(1 − *p*) .其累積量可以整理成下面形式

<!-- end list -->

  -

      -
        \(\kappa_{n+1}=p(1-p) \frac{d\kappa_n}{dp}.\,\)

<!-- end list -->

  - [幾何分佈](../Page/幾何分佈.md "wikilink")，累積量生成函數為*g* '(*t*) = ((1 −
    *p*)<sup>−1</sup>·e<sup>−*t*</sup> − 1)<sup>−1</sup>。第一累積量為κ<sub>1</sub> = *g* '(0) = *p*<sup>−1</sup> − 1，κ<sub>2</sub> = *g* ' '(0) = κ<sub>1</sub>·*p*<sup> − 1</sup>.代換*p* = (μ+1)<sup>−1</sup>可得*g* '(*t*)
    = ((μ<sup>−1</sup> +
    1)·e<sup>−*t*</sup> − 1)<sup>−1</sup>及κ<sub>1</sub> = μ.

<!-- end list -->

  - [泊松分布](../Page/泊松分布.md "wikilink")，累積量生成函數為*g* '(*t*) = μ·e<sup>*t*</sup>.所有的累積量均為：κ<sub>1</sub> = κ<sub>2</sub> = κ<sub>3</sub> = ...=μ.

<!-- end list -->

  - [二項分佈](../Page/二項分佈.md "wikilink")，其特殊情形是*n* = 1時為伯努利分佈。每一累積量是*n*倍相對應的伯努利分佈。累積量生成函數為*g* '(*t*) = *n*·((*p*<sup>−1</sup>−1)·e<sup>−*t*</sup> + 1)<sup>−1</sup>。第一累積量為κ<sub>1</sub> = *g* '(0) = *n·p*及κ<sub>2</sub> = *g* ' '(0) = κ<sub>1</sub>·(1−*p*)。代換*p* = μ·*n*<sup>−1</sup>可得*g* '(*t*)
    =
    ((μ<sup>−1</sup> − *n*<sup>−1</sup>)·e<sup>−*t*</sup> + *n*<sup>−1</sup>)<sup>−1</sup>及κ<sub>1</sub> = μ。極限值逼近情形則為*n*<sup>−1</sup> = 0之卜瓦松分佈。

<!-- end list -->

  - [負二項分佈](../Page/負二項分布.md "wikilink")，其特殊情形為*n* = 1時是為幾何分佈。每一累積量是*n*倍相對應的幾何分佈。累積量生成函數為*g* '(*t*) = *n*·((1−*p*)<sup>−1</sup>·e<sup>−*t*</sup>−1)<sup>−1</sup>。第一累積量為κ<sub>1</sub> = *g* '(0) = *n*·(*p*<sup>−1</sup>−1)，及κ<sub>2</sub> = *g* ' '(0) = κ<sub>1</sub>·*p*<sup>−1</sup>.代換*p* = (μ·*n*<sup>−1</sup>+1)<sup>−1</sup>可得*g* '(*t*)
    =
    ((μ<sup>−1</sup>+*n*<sup>−1</sup>)·e<sup>−*t*</sup>−*n*<sup>−1</sup>)<sup>−1</sup>及κ<sub>1</sub> = μ.比較二項分佈與本公式可以知悉負二項分佈名字的由來。極限值逼近情形則為*n*<sup>−1</sup> = 0之卜瓦松分佈。

在[天文學上](../Page/天文學.md "wikilink")，**絕對星等**（）是指把[天體放在指定的距離时](../Page/天體.md "wikilink")（10[秒差距](../Page/秒差距.md "wikilink")）天体所呈现出的[视星等](../Page/视星等.md "wikilink")（Apparent
magnitude, *m*）。此方法可把天體的[光度在不受距離的影響下](../Page/光度.md "wikilink")，作出客觀的比較。

## 恆星與星系的絕對星等M

在测量[恆星與](../Page/恆星.md "wikilink")[星系的絕對星等](../Page/星系.md "wikilink")，標準距離設為10[秒差距](../Page/秒差距.md "wikilink")，约32.6164[光年或](../Page/光年.md "wikilink")300兆[公里](../Page/公里.md "wikilink")，此时该天體的[視差值為](../Page/視差.md "wikilink")0.1"。

在定義其絕對星等時，必須指定要[测量哪一類型的](../Page/测量.md "wikilink")[電磁輻射](../Page/電磁輻射.md "wikilink")。如果按其釋出的[能量計算](../Page/能量.md "wikilink")，其結果會稱為**輻射熱強度**。星等值越低，代表天體越亮。绝对星等和视星等，可以通过视差（距離）换算。

由于距离较远的原因，许多恒星的绝对星等要比其视星等低（亮）很多；而有些恒星由于距离我们较近的原因，其绝对星等会变大（暗）很多。以下是一些恒星绝对星等与视星等的参照表：

|      |                                  |                                  |                                  |                                  |                                |
| ---- | -------------------------------- | -------------------------------- | -------------------------------- | -------------------------------- | ------------------------------ |
| 星名   | [参宿四](../Page/参宿四.md "wikilink") | [参宿七](../Page/参宿七.md "wikilink") | [天津四](../Page/天津四.md "wikilink") | [天狼星](../Page/天狼星.md "wikilink") | [太阳](../Page/太阳.md "wikilink") |
| 绝对星等 | \-5.3                            | \-6.7                            | \-7.2                            | 1.4                              | 4.83                           |
| 视星等  | 0.45                             | 0.18                             | 1.25                             | \-1.46                           | −26.8                          |

恒星的绝对星等的范围通常在-10到+17之间。星系的绝对星等通常更低（亮），例如，[椭圆星系](../Page/椭圆星系.md "wikilink")[M87的绝对星等为](../Page/M87.md "wikilink")-22。

### 换算

如果已知天体的视星等*m*和距离*d*，那么可以根据下式得出天体的绝对星等*M*：

\[M = m + 5 \log_{10}\frac{d_0}{d}\!\,\]

或

\[M = m + 5 (1 + \log_{10}{\pi})\!\,\]

其中，\(d_0\!\,\)为10秒差，即32.616光年，\(\pi\!\,\)是天体的视差，单位是[弧秒](../Page/弧秒.md "wikilink")。

如果已知天体的绝对星等*M*，和距离*d*，那么可以根据下式得出天体的视星等*m*：

\[m = M - 5 \log_{10}\frac{d_0}{d}\!\,\]

或

\[m = M - 5 (1 + \log_{10}{\pi})\!\,\]

#### 例子

  -
    [参宿七的视星等](../Page/参宿七.md "wikilink")+0.18，距离773光年，则其绝对星等为：
      -
        M<sub>参宿七</sub> = 0.18 + 5\*log<sub>10</sub>(32.616/773) = -6.7
    [织女星的视差为](../Page/织女星.md "wikilink")0.133"，视星等+0.03，则其绝对星等为：
      -
        M<sub>织女星</sub> = 0.03 + 5\*(1 + log<sub>10</sub>(0.133)) =
        +0.65
    [南门二的视差](../Page/南门二.md "wikilink")0.750"，绝对星等+4.37，则其视星等为：
      -
        m<sub>南门二</sub> = 4.37 - 5\*(1 + log<sub>10</sub>(0.750)) =
        -0.01

## 行星的绝对星等H

对于[行星](../Page/行星.md "wikilink")，[彗星](../Page/彗星.md "wikilink")，[小行星等非恒星天体来说](../Page/小行星.md "wikilink")，它们的绝对星等定义是完全不同的。恒星的绝对星等定义对其不适用。此时，绝对星等被定义成天体在距离太阳和地球的距离都为一个[天文单位](../Page/天文单位.md "wikilink")（au），且[相位角为](../Page/相位角.md "wikilink")0°时，呈现的视星等。这实际上是不可能的，只是为了计算方便。

### 计算

绝对星等*H*:

  -
    \(H = m_{Sun} - 5 \log_{10}\frac{ \sqrt { a } r}{d_0}\!\,\)

其中\(m_{Sun}\!\,\)是太阳的视星等（-26.73），\(a\!\,\)是天体表面的几何[反照率](../Page/反照率.md "wikilink")（0和1之间），\(r\!\,\)是天体半径，\(d_0\!\,\)是一个天文单位。

### 例子

月亮：

  -
    \(a_{Moon}\!\,\) = 0.12, \(r_{Moon}\!\,\) = 3476/2 km = 1738 km
    \(H_{Moon} = m_{Sun} - 5 \log_{10}\frac{ \sqrt { a_{Moon} } r_{Moon}}{d_0} = +0.25\!\,\)

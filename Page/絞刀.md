**鉸刀**，可以用来放大鑽孔等，或縮小形狀的[工具](../Page/工具.md "wikilink")。有粗加工、精​​加工根據不同的應用，是一種螺旋形，如鑽頭的刀片的直線形狀。雖然適合在廣泛的範圍內的孔的大小，孔也可以是圓錐形。

## 鉸刀類型

## 可調節手用鉸刀

[ReamerAdjustableHand.jpg](https://zh.wikipedia.org/wiki/File:ReamerAdjustableHand.jpg "fig:ReamerAdjustableHand.jpg") 可調節手用鉸刀，是靠調整節兩邊的螺母,使刀片沿本體上的斜底槽移動，以改變鉸刀的直徑尺寸。

## 刀具材料

\[1\]

| 常見的刀具材料 | 應用                                                            |
| ------- | ------------------------------------------------------------- |
| 高速鋼     | 最常用的。 物美價廉。                                                   |
|         | 硬度達67 RC鋒利的刀刃，這意味著更少的切削力。                                     |
|         | 高鈷版本是非常耐熱，因而非常適合擴孔磨料和/或加工硬化材料，如鈦和不銹鋼。                         |
| 碳化鎢     | 價格比高速鋼貴。                                                      |
|         | 硬度可達92 RC。 將拖垮[高速鋼](../Page/高速鋼.md "wikilink")（通常由約10：1）擴孔鋼時。 |
|         | 需要鉸淬硬材料。                                                      |

## [潤滑](../Page/潤滑.md "wikilink")

在擴孔摩擦的過程中引起[摩擦熱](https://zh.wikipedia.org/wiki/摩擦熱 "wikilink")。適當的[潤滑](../Page/潤滑.md "wikilink")[冷卻增加了工具的壽命](https://zh.wikipedia.org/wiki/冷卻 "wikilink")。潤滑的另一個好處包括可以使用更高的切削速度。\[2\] \[3\]

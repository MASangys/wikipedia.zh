**絲滿市**（，）\[1\]為[琉球群島](https://zh.wikipedia.org/wiki/琉球群島 "wikilink")[沖繩島](../Page/沖繩島.md "wikilink")最南端的城市，位於[那霸市](../Page/那霸市.md "wikilink")附近，近年來人口成長快速，城市開發迅速，境內有全長約10公里的通得川從東向西穿越市區。過去以捕[青花魚聞名](https://zh.wikipedia.org/wiki/青花魚 "wikilink")，1982年可讓大型船使用的糸滿漁港完工，使得水產加工相關產業成為糸滿市的產業之一。還有以「琉球玻璃村」為代表的玻璃工藝和漆器。也有農業、畜牧業。每年農曆5月4日有龍舟競賽，農曆8月15日有海神祭的拔河儀式。糸滿市南部的舊[三和村一帶在](../Page/三和村_\(琉球政府\).md "wikilink")[沖繩島戰役](../Page/沖繩島戰役.md "wikilink")末期戰事最激烈的地區，現已被指定為沖繩戰跡國定公園。

## 歷史

|                                                       |          |
| ----------------------------------------------------- | -------- |
| 時間                                                    | **行政劃分** |
| [琉球王國](https://zh.wikipedia.org/wiki/琉球王國 "wikilink") | 兼城間切     |
| 1900年代                                                | 糸滿町      |
| 1910年代                                                |          |
| 1920年代                                                |          |
| 1930年代                                                |          |
| 1940年代                                                | 三和村      |
| 1950年代                                                |          |
| 1960年代                                                | 糸滿町      |
| 1970年代                                                | 糸滿市      |
| 1980年代                                                |          |
| 1990年代                                                |          |
| 2000年代                                                |          |

  - 1896年4月1日：實施[郡區制](https://zh.wikipedia.org/wiki/郡區制 "wikilink")，為[島尻郡](../Page/島尻郡.md "wikilink")轄下兼城間切、高嶺間切、真壁間切、喜屋武間切、摩文仁間切。
  - 1908年4月1日：實施[島嶼町村制](../Page/島嶼町村制.md "wikilink")，改設為兼城村、高嶺村、真壁村、喜屋武村、摩文仁村，並從原兼城間切內分割獨立設置絲滿町。
  - 1946年4月1日：[沖繩島戰役](../Page/沖繩島戰役.md "wikilink")結束後，真壁村、喜屋武村、摩文仁村因戰爭中傷亡過多，使村裡人數過少無法維持運作，因此合併為三和村
  - 1961年10月1日：絲滿町、兼城村、高嶺村、三和村[合併為新成立的](../Page/市町村合併.md "wikilink")**糸滿町**。（原有絲滿町與其餘三村皆廢除）
  - 1971年12月1日：糸滿町改制為**糸滿市**。

**红玉髓，**又称**红石髓**，是一种棕红色的[二氧化硅矿物](../Page/二氧化硅.md "wikilink")，常被用作半宝石。“红玉髓”一词可用作指颜色较浅的**光玉髓**()\[1\]，亦可指而色暗且质地较硬的**褐红玉髓**（）。\[2\]\[3\]

红玉髓属于[玉髓的一种变体](../Page/玉髓.md "wikilink")，因掺杂[氧化铁而显色](../Page/氧化铁.md "wikilink")。红玉髓的颜色多样，色泽最亮的显橙白色，而最暗的接近黑色。红玉髓最常见的产地为[巴西](../Page/巴西.md "wikilink")、[印度](../Page/印度.md "wikilink")、[德国和](../Page/德国.md "wikilink")[俄罗斯](../Page/俄罗斯.md "wikilink")[西伯利亚](../Page/西伯利亚.md "wikilink")。

## 历史

早在[新石器时代早期的](../Page/新石器时代.md "wikilink")[保加利亚](https://zh.wikipedia.org/wiki/保加利亚 "wikilink")，红色的玉髓就被用做宝珠。最早有打磨晶面的红玉髓出自保加利亚的，可上溯至公元前5千年中期。宝珠两侧各有16个打磨面，合计32个打磨面。\[4\]公元前5000-4000年的[梅赫尔格尔遗迹中出土了有钻孔的红玉髓](https://zh.wikipedia.org/wiki/梅赫尔格尔 "wikilink")。\[5\][克里特岛上](https://zh.wikipedia.org/wiki/克里特岛 "wikilink")[克诺索斯的](../Page/克诺索斯.md "wikilink")[米诺斯](../Page/米诺斯文明.md "wikilink")[青铜文明层中发现了公元前](https://zh.wikipedia.org/wiki/青铜文明 "wikilink")1800年的红玉髓，其形制表明曾用作装饰。\[6\]在罗马时代，红玉髓被广泛用于制作；又因为红玉髓不沾[蜡油](https://zh.wikipedia.org/wiki/蜡 "wikilink")，还被用来制作蜡封用的[印章](../Page/印章.md "wikilink")。\[7\]褐红玉髓被用作[亚述的](../Page/亚述.md "wikilink")[滚筒印章](../Page/滚筒印章.md "wikilink")、[古埃及和](../Page/古埃及.md "wikilink")[腓尼基的](../Page/腓尼基.md "wikilink")[圣甲虫](https://zh.wikipedia.org/wiki/圣甲虫 "wikilink")、以及[古希腊和](../Page/古希腊.md "wikilink")[伊特拉斯坎的珠宝](https://zh.wikipedia.org/wiki/伊特拉斯坎 "wikilink")。\[8\],
[犹太大祭司的](../Page/大祭司_\(犹太教\).md "wikilink")上的第一块宝石(被称作odem或sardius)是一块红色宝石，很有可能是褐红玉髓，也有可能是红色的[碧玉](https://zh.wikipedia.org/wiki/碧玉 "wikilink")。\[9\]在[启示录](../Page/启示录.md "wikilink")4:3中,
[使徒约翰描述天上的宝座上坐着的](../Page/約翰_\(使徒\).md "wikilink")"好像碧玉和红宝石"，其中"红宝石"即为
sardius. 同一种“红宝石”还出现在启示录21:20中，是圣城城墙根基上的十二种宝石之一。\[10\]
[缩略图](https://zh.wikipedia.org/wiki/File:Cornelian_necklace_BM_GR1897.4-1.623.jpg "fig:缩略图")出土的项链，由金箔和红玉髓珠制成，艺术风格受到[迈锡尼文明的影响](../Page/迈锡尼文明.md "wikilink")。项链的年代为公元前1400-1200年，现藏于[大英博物馆](../Page/大英博物馆.md "wikilink")。\]\]

[牛津英语词典认为](https://zh.wikipedia.org/wiki/牛津英语词典 "wikilink")"carnelian" 是
"cornelian"的变种，来自拉丁语的<span>caro</span>'',
''carnis(肉)。[老普林尼认为sard一词来自](../Page/老普林尼.md "wikilink")[吕底亚的](https://zh.wikipedia.org/wiki/吕底亚 "wikilink")[萨第斯](https://zh.wikipedia.org/wiki/萨第斯 "wikilink")(Sardis)，褐红玉髓的产地；而其他人认为这一词最终可追溯到[波斯语的سرد](../Page/波斯语.md "wikilink")
(<span>sered</span>)，意为红黄色。

## 光玉髓和褐红玉髓的区别

红玉髓有时指光玉髓、有时指褐红玉髓；在英文中carnelian和sard也时常混用。通常而言，光玉髓和褐红玉髓有如下区别:

|    |                |                 |
| -- | -------------- | --------------- |
|    | 光玉髓(Carnelian) | 褐红玉髓(Sard)      |
| 颜色 | 较浅，色泽呈浅橙色到红棕色  | 较深，色泽呈深红棕色至近黑色。 |
| 硬度 | 较软             | 较坚硬             |
| 断口 | 不均匀，碎裂，呈贝壳状    | 与光玉髓相近，但更暗且呈锯齿状 |

上述性质是连续变化的，因此光玉髓与褐红玉髓之间没有明确的界限。

**细猪笼草**（[学名](https://zh.wikipedia.org/wiki/学名 "wikilink")：**）是[苏门答腊特有的热带](https://zh.wikipedia.org/wiki/苏门答腊 "wikilink")[食虫植物](../Page/食虫植物.md "wikilink")。\[1\]其[种加词](../Page/种加词.md "wikilink")“*tenuis*”来源于[拉丁文](https://zh.wikipedia.org/wiki/拉丁文 "wikilink")“*tenuis*”，意为“薄的、细的、修长的”。1957年，首次在苏门答腊岛西部一座偏远山上采集到了细猪笼草的标本。直到1994年才被正式描述，并仅于2002年重新在野外发现。在2002年之前，仅有细猪笼草的一张照片和一份干燥标本。

## 植物学史

1957年8月24日，[威廉·梅哲首次在](../Page/威廉·梅哲.md "wikilink")[西苏门答腊省](../Page/西苏门答腊省.md "wikilink")[塔拉姆](https://zh.wikipedia.org/wiki/塔拉姆 "wikilink")[天普河](https://zh.wikipedia.org/wiki/天普河 "wikilink")附近的一个砂岩地区采集到了细猪笼草的标本。其生长于海拔约1000米的山脊上。其原生地为“明亮的亚山地森林”。\[2\]

1994年，[约阿希姆·那兹和](../Page/约阿希姆·那兹.md "wikilink")[安德烈亚斯·维斯图巴根据由威廉](../Page/安德烈亚斯·维斯图巴.md "wikilink")·梅哲存放于[莱顿的一副标本](../Page/莱顿.md "wikilink")，及在1957年采集标本时拍摄的一张黑白照片，正式描述了细猪笼草。\[3\]经过多次的失败\[4\]，安德烈亚斯·维斯图巴、约阿希姆·那兹、迈克尔·沙赫及其他人组成的团队在2002年末终于在野外重新发现了细猪笼草\[5\]\[6\]\[7\]。

## 形态特征

细猪笼草为[藤本植物](../Page/藤本植物.md "wikilink")。茎带棱角至菱形，较纤细，直径仅为2至3毫米。[节间距为](https://zh.wikipedia.org/wiki/节间距 "wikilink")5至6.5厘米。\[8\]

细猪笼草的叶片[无柄](https://zh.wikipedia.org/wiki/wikt:无柄 "wikilink")，[革质](https://zh.wikipedia.org/wiki/wikt:革质 "wikilink")。其呈[披针形](https://zh.wikipedia.org/wiki/wikt:披针形 "wikilink")，长5至6厘米，宽1至1.5厘米。[叶尖为](https://zh.wikipedia.org/wiki/叶尖 "wikilink")[急尖](https://zh.wikipedia.org/wiki/wikt:急尖 "wikilink")，[叶基](https://zh.wikipedia.org/wiki/叶基 "wikilink")[渐狭](https://zh.wikipedia.org/wiki/wikt:渐狭 "wikilink")，包住茎周长的三分之二。叶脉不明显。[中脉的两侧各约有](https://zh.wikipedia.org/wiki/中脉 "wikilink")3条[纵脉](https://zh.wikipedia.org/wiki/纵脉 "wikilink")，其发自中脉基部的三分之一，平行延伸于叶片的外半部。[羽状脉斜发](https://zh.wikipedia.org/wiki/羽状脉 "wikilink")，为不规则的网状，但不易辨别。[笼蔓的长度约为叶片的](https://zh.wikipedia.org/wiki/笼蔓 "wikilink")1.5倍，可长达12厘米\[9\]，厚0.5至1毫米。是否出现笼蔓圈取决于周围是否存在支持物。\[10\]

细猪笼草的捕虫笼是猪笼草属中仅次于[阿金特猪笼草](../Page/阿金特猪笼草.md "wikilink")第二小的。[上位笼与笼蔓的连接处形成一个宽约](https://zh.wikipedia.org/wiki/上位捕虫笼 "wikilink")10毫米的弯曲。上位笼整体呈宽漏斗形，唇下缩小。通常高2.5至4.5厘米，宽1.5至2.5厘米。[下位笼和上位笼都无](https://zh.wikipedia.org/wiki/下位捕虫笼 "wikilink")[笼翼及](https://zh.wikipedia.org/wiki/笼翼 "wikilink")[翼须](https://zh.wikipedia.org/wiki/翼须 "wikilink")。取而代之的是一对隆起，间距約3至5毫米。捕虫笼的整个内表面都覆盖着腺体。\[11\]腺体小型，密度为每平方厘米600至800个。[笼口为卵形](https://zh.wikipedia.org/wiki/笼口 "wikilink")，几乎水平。[唇平展](https://zh.wikipedia.org/wiki/猪笼草#形态特征 "wikilink")，可宽达6毫米，具[唇肋](https://zh.wikipedia.org/wiki/唇肋 "wikilink")，间距为1.25毫米。[笼盖为窄椭圆形](https://zh.wikipedia.org/wiki/笼盖 "wikilink")，无附属物，但中脉两侧各有一条明显的侧脉。笼盖一般长1.5至2.5厘米，宽0.5至0.8厘米。笼盖下表面均匀的覆盖着蜜腺。笼盖基部的后方具一根不分叉的[笼蔓尾](https://zh.wikipedia.org/wiki/笼蔓尾 "wikilink")，其可长达1毫米。\[12\]

细猪笼草植株的大部分都无毛，少部分可能覆盖着稀疏的毛被。细猪笼草的捕虫笼为黄绿色，且不同程度具红褐色斑点。唇为黄色至暗褐色。干燥标本为深褐色。\[13\]

## 相关物种

在[马修·杰布和](../Page/马修·杰布.md "wikilink")[马丁·奇克](../Page/马丁·奇克.md "wikilink")1997年的专著中将细猪笼草列为[疑惑猪笼草](../Page/疑惑猪笼草.md "wikilink")的同物异名时是“有一些犹豫的”。无刺猪笼草与[塔蓝山猪笼草的](../Page/塔蓝山猪笼草.md "wikilink")[自然杂交种](https://zh.wikipedia.org/wiki/自然杂交种 "wikilink")也被他们归入疑惑猪笼草。\[14\]但[查尔斯·克拉克并不赞同这个鉴定](../Page/查尔斯·克拉克_\(植物学家\).md "wikilink")，他并不认为这三个类群属于同一个物种。\[15\]

疑惑猪笼草和细猪笼草的区别相对简单。最明显的区别在于细猪笼草的捕虫笼较短，笼盖较宽且打开角度不会超过180°，而疑惑猪笼草的笼盖常常会超过180°。它们捕虫笼的颜色也不同，细猪笼草的捕虫笼外表面常具有红色的斑点，而疑惑猪笼草一般为黄色或橙色。此外，细猪笼草的唇也不如疑惑猪笼草接近圆柱形。\[16\]

细猪笼草与[贾桂琳猪笼草](../Page/贾桂琳猪笼草.md "wikilink")的区别在于细猪笼草的叶片为[线形至](https://zh.wikipedia.org/wiki/wikt:线形 "wikilink")[披针形](https://zh.wikipedia.org/wiki/wikt:披针形 "wikilink")，而贾桂琳猪笼草为卵形至[匙形](https://zh.wikipedia.org/wiki/wikt:匙形 "wikilink")，且细猪笼草笼盖下表面无巨型的蜜腺。贾桂琳猪笼草捕虫笼的笼盖基部还显著的缩小。此外，贾桂琳猪笼草的体型远大于细猪笼草，其捕虫笼也更发达，唇更宽，笼口几乎是水平的。\[17\]

在约阿希姆·那兹和安德烈亚斯·维斯图巴对细猪笼草的描述中，包含了一个“细猪笼草于数个近似物种的显著区别”的表格：\[18\]

|                                            |
| :----------------------------------------: |
| **邦苏猪笼草、疑惑猪笼草、塔蓝山猪笼草与细猪笼草之间形态特征的区别**\[19\] |
|                    形态特征                    |
|                   上位笼的形状                   |
|                    叶片形状                    |
|                  上位笼的长宽比                   |

查尔斯·克拉克写道：“细猪笼草、塔蓝山猪笼草和贾桂琳猪笼草之间的明显相似性是近期适应性辐射的一种暗示，并增加了非火山的[巴里桑山脉是苏门答腊猪笼草属植物形态多样性的主要来源该观点的分量](https://zh.wikipedia.org/wiki/巴里桑山脉 "wikilink")。”\[20\]

2001年，查尔斯·克拉克对来自[苏门答腊岛和](https://zh.wikipedia.org/wiki/苏门答腊岛 "wikilink")[西马来西亚的猪笼草进行了](https://zh.wikipedia.org/wiki/西马来西亚 "wikilink")[分支系统学分析](https://zh.wikipedia.org/wiki/分支系统学 "wikilink")，共利用了70个形态特征。以下为猪笼草进化树的“第1分支”，其相关性达51%。\[21\]

## 自然杂交种

[Neptenuis2.jpg](https://zh.wikipedia.org/wiki/File:Neptenuis2.jpg "fig:Neptenuis2.jpg")

细猪笼草生长于砂岩山脊顶部的[苔藓森林或亚山地森林中](https://zh.wikipedia.org/wiki/苔藓森林 "wikilink")。其仅出现于海拔1000米至1200米的地区。\[22\]\[23\]细猪笼草为[西苏门答腊省](../Page/西苏门答腊省.md "wikilink")[塔拉姆西部的](https://zh.wikipedia.org/wiki/塔拉姆 "wikilink")[天普河地区](https://zh.wikipedia.org/wiki/天普河 "wikilink")[特有的热带](https://zh.wikipedia.org/wiki/特有种 "wikilink")[食虫植物](../Page/食虫植物.md "wikilink")。该地区的海拔大多在1000米以下，仅少数几处超过该高度。\[24\]

细猪笼草的下位笼通常嵌于[苔藓中生长](https://zh.wikipedia.org/wiki/苔藓 "wikilink")。植株下部的茎也可能被苔藓覆盖，使其在野外难见其踪影。

在其原生地，细猪笼草于[宽叶猪笼草](https://zh.wikipedia.org/wiki/宽叶猪笼草 "wikilink")同域分布，并且其分布地靠近于[白环猪笼草](../Page/白环猪笼草.md "wikilink")、[苹果猪笼草](../Page/苹果猪笼草.md "wikilink")、[真穗猪笼草](../Page/真穗猪笼草.md "wikilink")、[小猪笼草](../Page/小猪笼草.md "wikilink")、[长叶猪笼草](../Page/长叶猪笼草.md "wikilink")和[两眼猪笼草](../Page/两眼猪笼草.md "wikilink")的分布地。尽管如此，仍未发现细猪笼草的自然杂交种。\[25\]\[26\]

细猪笼草已被列入《2006年[世界自然保护联盟濒危物种红色名录](https://zh.wikipedia.org/wiki/世界自然保护联盟濒危物种红色名录 "wikilink")》中，保护状况为[濒危](https://zh.wikipedia.org/wiki/濒危物种 "wikilink")。\[27\]其原生地在不久的将来可能会遭受到农业开垦放火烧山的威胁。\[28\]

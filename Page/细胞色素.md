**细胞色素**（[英文](../Page/英文.md "wikilink")：cytochrome）一般是指一类膜结合的[血红素蛋白](../Page/血红素蛋白.md "wikilink")，以[血基質为](../Page/血基質.md "wikilink")[辅基](../Page/辅基.md "wikilink")，参与[电子传递](../Page/电子传递.md "wikilink")。它可以以[单体的形式](../Page/单体.md "wikilink")（如[细胞色素c](../Page/细胞色素c.md "wikilink")）或作为复合物[酶中的一个](../Page/酶.md "wikilink")[亚基来发挥](../Page/蛋白质亚基.md "wikilink")[氧化还原作用](../Page/氧化还原反应.md "wikilink")。细胞色素是各种生物体中都很常见的[蛋白质](../Page/蛋白质.md "wikilink")，广泛存在于[真核生物的](../Page/真核生物.md "wikilink")[线粒体内膜和](../Page/线粒体.md "wikilink")[内质网中](../Page/内质网.md "wikilink")，植物的[叶绿体中](../Page/叶绿体.md "wikilink")，以及[光合成](../Page/光合作用.md "wikilink")[微生物和](../Page/微生物.md "wikilink")[细菌中](../Page/细菌.md "wikilink")。

## 结构和功能

细胞色素中血红素上的[卟啉环以四个](../Page/卟啉环.md "wikilink")[配位键与一个铁原子相连](../Page/配位键.md "wikilink")，形成四配位的[络合物](../Page/络合物.md "wikilink")。由于在血红素这样一个环状体系中，铁原子处于活泼的化学状态，可以在还原态(Fe<sup>2+</sup>)和氧化态(Fe<sup>3+</sup>)之间可逆变化，使得
细胞色素可以进行氧化还原反应。因为细胞色素或以及其他参与氧化还原的相关复合物以有序状态锚定在膜上，这样就可以以正确的次序进行氧化还原反应，以最大化反应的效率。

细胞色素参与了[氧化磷酸化](../Page/氧化磷酸化.md "wikilink")（如细胞色素c）和[光合磷酸化](../Page/光合磷酸化.md "wikilink")（如细胞色素b<sub>563</sub>）的[电子传递链](../Page/电子传递链.md "wikilink")。氧化磷酸化反应所产生的跨膜[质子浓度梯度可为合成](../Page/质子.md "wikilink")[ATP提供能量](../Page/三磷酸腺苷.md "wikilink")，而ATP是生命活动的能量来源。也有部分细胞色素参与非磷酸化的电子传递链，如[细胞色素P450参与](../Page/细胞色素P450.md "wikilink")[脂类物质的](../Page/脂.md "wikilink")[代谢](../Page/代谢.md "wikilink")。

## 分类

细胞色素种类较多，可以用[光谱学](../Page/光谱学.md "wikilink")、血红素基团的精确结构、抑制剂的敏感度以及还原电势的大小来分辨。

根据辅基的不同，可以将细胞色素大致分为三类：

|        |                                                                                   |
| ------ | --------------------------------------------------------------------------------- |
| **分类** | **辅基**                                                                            |
| 细胞色素a  | [血红素a](../Page/血红素a.md "wikilink")（[法尼基修饰的甲酰血红素](../Page/法尼基焦磷酸.md "wikilink")）   |
| 细胞色素b  | [血红素b](../Page/血红素b.md "wikilink")（原血红素）                                          |
| 细胞色素d  | [二氢卟酚](../Page/二氢卟酚.md "wikilink")（Chlorin）的[衍生物](../Page/衍生物.md "wikilink")\[1\] |

细胞色素c不是根据辅基来命名；与其他三类细胞色素[非共价结合血红素不同的是](../Page/共价键.md "wikilink")，细胞色素c通过分子中的[半胱氨酸](../Page/半胱氨酸.md "wikilink")[巯基与血红素](../Page/巯基.md "wikilink")[共价结合](../Page/共价键.md "wikilink")。\[2\]
目前没有细胞色素e，但有[细胞色素f](../Page/细胞色素f.md "wikilink")（细胞色素c中的一种）。\[3\]
此外，[细胞色素d仅存在于细菌中](../Page/细胞色素d.md "wikilink")。

在线粒体和叶绿体中，细胞色素常常结合在电子传递和相关代谢途径的复合物酶中。

|                                                     |                                                                                           |
| --------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| **细胞色素**                                            | **复合物酶**                                                                                  |
| *a*和*a<sub>3</sub>*                                 | [细胞色素c氧化酶](../Page/细胞色素氧化酶.md "wikilink")（“复合物IV”）                                        |
| *b*和[*c<sub>1</sub>*](../Page/细胞色素c1.md "wikilink") | [辅酶Q－细胞色素c还原酶](../Page/辅酶Q－细胞色素c还原酶.md "wikilink")（“复合物III”）                              |
| *b<sub>6</sub>*和*f*                                 | [质体醌一质体蓝素氧化还原酶](../Page/质体醌一质体蓝素氧化还原酶.md "wikilink")（Plastoquinol—plastocyanin reductase） |

[细胞色素P450是一个完全不同的细胞色素家族](../Page/细胞色素P450.md "wikilink")，命名是根据其还原态（血红素上的铁原子被还原，并结合有[一氧化碳](../Page/一氧化碳.md "wikilink")）的特征[Soret带峰](../Page/Soret带峰.md "wikilink")（Soret
peak）吸收波长为450[nm](../Page/nm.md "wikilink")。这类酶主要参与[類固醇的合成和毒物的氧化代谢](../Page/類固醇.md "wikilink")（解毒作用）。

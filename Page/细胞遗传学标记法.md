下表总结了[细胞遗传学中所使用的标记与缩写](../Page/细胞遗传学.md "wikilink")\[1\]:

| 标记      | 描述                                                                                                                                                                                        |
| ------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| ,       | 分割众数（，染色体总数）、性染色体与染色体异常变化                                                                                                                                                                 |
| \-      | 染色体缺失                                                                                                                                                                                     |
| ( )     | 将断点及结构被改变的染色体归类                                                                                                                                                                           |
| \+      | 染色体增加                                                                                                                                                                                     |
| ;       | Separates rearranged chromosomes and breakpoints involving more than one chromosome                                                                                                       |
| /       | Separates cell lines or clones                                                                                                                                                            |
| //      | Separates recipient and donor cell lines in bone marrow transplants                                                                                                                       |
| del     | 缺失                                                                                                                                                                                        |
| der     | [衍生染色体](https://zh.wikipedia.org/wiki/衍生染色体 "wikilink")                                                                                                                                   |
| dic     | [双着丝粒染色体](https://zh.wikipedia.org/wiki/双着丝粒染色体 "wikilink")                                                                                                                               |
| dn      | *de novo* (not inherited) chromosomal abnormality                                                                                                                                         |
| dup     | [重复](https://zh.wikipedia.org/wiki/基因重复 "wikilink") of a portion of a chromosome                                                                                                          |
| fra     | 脆性位点 (usually used with [X染色体易裂症](https://zh.wikipedia.org/wiki/X染色体易裂症 "wikilink"))                                                                                                      |
| h       | [异染色质](https://zh.wikipedia.org/wiki/异染色质 "wikilink") region of chromosome                                                                                                                |
| i       | [等臂染色体](https://zh.wikipedia.org/wiki/等臂染色体 "wikilink")                                                                                                                                   |
| ins     | [插入](https://zh.wikipedia.org/wiki/插入_\(遗传学\) "wikilink")                                                                                                                                 |
| inv     | [倒位](https://zh.wikipedia.org/wiki/染色体倒位 "wikilink")                                                                                                                                      |
| .ish    | Precedes [核型](https://zh.wikipedia.org/wiki/核型 "wikilink") results from [FISH](https://zh.wikipedia.org/wiki/FISH "wikilink") analysis                                                    |
| mar     | 标记染色体                                                                                                                                                                                     |
| mat     | 母系源性染色体重排                                                                                                                                                                                 |
| p       | 染色体短臂                                                                                                                                                                                     |
| pat     | 父系源性染色体重排                                                                                                                                                                                 |
| psu dic | *伪双着丝粒* - only one [centromere](https://zh.wikipedia.org/wiki/centromere "wikilink") in a [Dicentric chromosome](https://zh.wikipedia.org/wiki/Dicentric_chromosome "wikilink") is active |
| q       | 染色体长臂                                                                                                                                                                                     |
| r       | 环状染色体                                                                                                                                                                                     |
| t       | [易位](https://zh.wikipedia.org/wiki/染色体易位 "wikilink")                                                                                                                                      |
| ter     | Terminal end of arm (e.g. 2qter refers to the end of the long arm of chromosome 2)                                                                                                        |
| tri     | [三体性](https://zh.wikipedia.org/wiki/三体性 "wikilink")                                                                                                                                       |
| trp     | 染色体一部分三倍化                                                                                                                                                                                 |
|         |                                                                                                                                                                                           |

《**经分别**》（Suttavibhanga），为《[巴利文大藏经](https://zh.wikipedia.org/wiki/巴利文大藏经 "wikilink")》中[律藏](../Page/律藏.md "wikilink")的组成部分，[南传上座部佛教典籍](../Page/上座部佛教.md "wikilink")。

该经核心爲[戒本](https://zh.wikipedia.org/wiki/戒本 "wikilink")（即[波罗提木叉](https://zh.wikipedia.org/wiki/波罗提木叉 "wikilink")，[解脱戒](https://zh.wikipedia.org/wiki/解脱戒 "wikilink")）。其分为《大分别》和《比丘尼分别》，分别阐述关于[比丘](../Page/比丘.md "wikilink")（227条戒律）、[比丘尼](../Page/比丘尼.md "wikilink")（311条戒律）的戒规。

## 分类

这些戒规分为以下几类：

| 类型         | 内容                                                                                                                                        | 比丘戒 | 比丘尼戒 |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------- | --- | ---- |
| 波罗夷        | 行淫、偷窃、杀人和妄称上人，均属大罪，犯者必须逐出[僧团](https://zh.wikipedia.org/wiki/僧团 "wikilink")                                                                | 4条  | 8条   |
| 僧残（僧伽胝施沙）  | 故意拽精、故意触摸女人、充当男女私通牵线、私自建造房屋、毁谤其他比丘、挑拨离间和不听忠告等，属于重罪，犯者接受六夜[摩那陲灭罪法](https://zh.wikipedia.org/wiki/摩那陲灭罪法 "wikilink")，然后在僧团大众面前认罪忏悔，方能恢复僧尼资格 | 13条 | 17条  |
| 不定         | 独自与妇女一起坐在隐蔽处，犯者视情况定为波罗夷、僧残或舍堕                                                                                                             | 2条  | 无    |
| 舍堕（尼萨耆波逸提） | 衣钵卧具超过应有数量或者不按照规定获得、使用，犯者须当众忏悔，衣钵卧具没收                                                                                                     | 30条 | 30条  |
| 单堕（波逸提）    | 说谎、漫骂、与妇女同屋住宿、比丘和非亲比丘尼结伴而行、比丘和比丘尼一起坐在隐蔽处、饮食过量、贪吃鱼肉乳酪、饮酒、杀害牲畜和饮用有虫之水等，犯者须当众忏悔                                                              | 92条 | 116条 |
| 波胝提舍尼      | 比丘从非亲比丘尼中接受食物、未受邀请而自取食物、或住处危险而不事先向前来送食的施主说明，犯者须忏悔                                                                                         | 4条  | 8条   |
| 众学         | 有关服装、饮食和说法等等日常行为的戒律                                                                                                                       | 75条 | 75条  |
| 灭诤         | 有关平息僧团内部纷争的方法（包括现前止诤、忆念止诤、不痴止诤、自言止诤、觅罪相止诤、多觅罪相止诤、如草覆地止诤）                                                                                  | 7条  | 7条   |
|            |                                                                                                                                           |     |      |

**絕對值**用來表示**一個數至原點的距離之大小**。絕對值的概念也可以定義在[複數](../Page/複數.md "wikilink")、有序環以及[域上](../Page/域.md "wikilink")。

## 實數的絕對值

[Absolute_value.svg](https://zh.wikipedia.org/wiki/File:Absolute_value.svg "fig:Absolute_value.svg")
若實數 \(a \neq 0\)，則在兩個相互對稱的數 \(a\) 和 \(-a\) 中必有且僅有一個數大於 0，這個大於 0 的數就稱為數
\(a\) 和數 \(-a\) 的**絕對值**，記為
\(\left |  a \right | = \left | -a \right |\) ，0 的絕對值為 0 。
一個[數的絕對值](../Page/數.md "wikilink")**永遠非負，沒有負號**，某數的絕對值表示為
\(|\) 某數 \(|\) 。對於所有[實數](../Page/實數.md "wikilink") \(x\) ：若 \(x\)
是[負數](../Page/負數.md "wikilink")， \(|x| = -x\) （即是 \(-x\) 是一個正數）；若
\(x\) 非負， \(|x| = x\) 本身。即：

\[\left| a \right| =
\begin{cases} a & (a>0) \\
0 & (a=0) \\
-a & (a<0) \end{cases}\]

一個數的絕對值可以視為該數在[数線上的点和零的距離](../Page/数線.md "wikilink")。例如 3 同時是 3 和 -3
的絕對值。

絕對值有以下性質：

1.  \(|a| \ge 0\)
2.  若 \(a = 0\)，\(|a| = 0\)
3.  \(|ab| = |a| |b|\)
4.  若 \(b \neq 0\)，\(|\frac{a}{b}| = \frac{|a|}{|b|}\)
5.  \(|a + b| \le |a| + |b|\) ( 見[三角不等式](../Page/三角不等式.md "wikilink") )
6.  \(|a - b| \ge \Big|\big|a\big| - \big|b\big|\Big|\)
7.  若
    \(a>0\)，則\(a = \left | a \right |\)；若\(a<0\)，則\(a = - \left | a \right |\)

## 複數的絕對值

[複數的絕對值定義為](../Page/複數.md "wikilink")：若\(z=a+ib\)，則
\(|z| = \sqrt{a^2 + b^2} = \sqrt{z\bar{z}}\)（見[平方根和](../Page/平方根.md "wikilink")[共軛複數](../Page/共軛複數.md "wikilink")）。它符合以上的六項性質，但以下的三項就**不成立**：

1.  \(\left| a \right| = \sqrt{a^2}\)
2.  \(|a| \le b\) 若且唯若 \(-b \le a \le b\)
3.  \(|a| \ge b\) 若且唯若 \(a \le -b\) ''' 或 ''' \(b \le a\)

但此時有

1.  \(|\bar{z}| = |z|\)
2.  \(\sqrt{z\bar{z}} = |z|\)

最後兩道式子常用於計算涉及複數絕對值的[不等式](../Page/不等式.md "wikilink")。

## 有序環上的絕對值

絕對值的定義可以照搬到有序環上。定義如下：

\[|a| = \begin{cases} a, & \;  a \ge 0  \\ -a,  & a < 0, \end{cases}\]

其中 \(-a\) 是 \(a\) 的加法反元素，而 0 是有序環的加法單位元素。

## 域上的絕對值

在抽象的域上，我們用絕對值的基本性質來推廣定義。一個域 \(F\) 上的**絕對值**是一個函數
\(|\cdot|: F \rightarrow \mathbb{R}\)，滿足以下四條公理:

  -
    {| cellpadding=10

|- |\(|a| \ge 0\) |非負性質 |- |\(|a| = 0 \iff a = \mathbf{0}\) |正定性質 |-
|\(|ab| = |a| \cdot |b| \,\) |積性 |- |\(|a+b|  \le |a| + |b|\) |三角不等式 |}

由以上公理可以導出 \(|1|=1\)。距離函數 \(d(x,y) = |x-y|\) 賦予 \(F\)
[度量空間結構](../Page/度量空間.md "wikilink")。

如果將定義中的三角不等式換作以下較強的形式（有時又叫做**強三角不等式**）

\[\forall a,b \in F \quad |a + b| \le \mathrm{max}\{|a|, |b|\}\]

則稱 \((F,|\cdot|)\)為[超度量域](../Page/超度量域.md "wikilink")，或稱絕對值 \(|\cdot|\)
不滿足**阿基米德性質**；反之則稱 \(|\cdot|\) 滿足阿基米德性質。\[1\]。

超度量域的典型例子是 [p進數域](../Page/p進數.md "wikilink")。一般來說，值群在 \(\mathbb{R}\)
裡的[賦值環對應到超度量域](../Page/賦值環.md "wikilink")，此時賦值與絕對值的關係由
\(|x| := a^{v(x)}\) 給出，其中 \(a \in ]0,1[\)；不同的 \(a\) 給出等價的拓撲結構。

## 微積分

絕對值函數在 x=0 不[可導](../Page/可微函数.md "wikilink")。

\[\frac{d}{dx}|x| = \frac{x}{|x|} = \begin{cases} 1 & x>0 \\  -1 & x<0 \end{cases}\]

\[\int|x|dx=\frac{x|x|}{2}+C\] 其中 ***C***
是[積分常數](../Page/積分常數.md "wikilink")。

## 演算法

[C 語言關於絕對值的](../Page/C語言.md "wikilink")[函數有](../Page/函數.md "wikilink")：
`abs()`, `labs()`, `llabs()`（在 C99 中），`fabs()`、`fabsf()` 與 `fabsl()`
函數計算一個對象的絕對值。當輸入值不是最大負整數時，很容易寫出計算絕對值的[巨集或函數](../Page/巨集.md "wikilink")。

以下巨集可接受整數或浮點數：

``` C
#define abs(i) ((i)>0 ? (i) : (-i))
```

如果以函數計算，需要寫多個函數，[多載來處理不同數據類型](../Page/函数重载.md "wikilink")：

``` C
int abs (int);
float abs (float);
double abs (double);

int abs (int i) {
  if (i>0) {
    return i;
  } else {
    return -i;
  }
}
```

關於[浮點數的絕對值](../Page/浮點數.md "wikilink")[演算法就要用點技巧](../Page/演算法.md "wikilink")，因為要為[無窮大及](../Page/無窮大.md "wikilink")
[NaN](../Page/NaN.md "wikilink")（Not a Number）撰寫特別的程式碼。

在
[Pascal](../Page/Pascal.md "wikilink")、[Fortran](../Page/Fortran.md "wikilink")
和 [Matlab](../Page/Matlab.md "wikilink") 語言裡，取得絕對值的函數是
`abs`，可以計算[整數](../Page/整數.md "wikilink")、[實數](../Page/實數.md "wikilink")，以及複數。

如以[組合語言撰寫](../Page/組合語言.md "wikilink")，應有可能以三行指令在暫存器內完成絕對值的判斷與轉換演算法，以下例子是
x86 結構上的 32 位元暫存器，採英特爾語法。 <code>

``` asm
cdq
xor eax, edx
sub eax, edx
```

</code>

`cdq` 指令將帶號位元（sign bit）的 `eax` 轉成 `edx`。如果 `eax` 是非負值，那 `edx` 變成
0，接下來的兩個指令會沒有影響，`eax` 將不變。如果 `eax` 是負數，那麼 `edx` 會變成
0xFFFFFFFF ，或是 -1。接下來的兩個指令會變成倒轉的[二補數](../Page/二補數.md "wikilink")，並從
`eax`暫存器中取得負數的絕對值。

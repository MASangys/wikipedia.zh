**絕熱指數**（）是指等壓[熱容](https://zh.wikipedia.org/wiki/熱容 "wikilink")（\(C_P\)）和等容（等體積）熱容（\(C_V\)）的比值，也稱為**熱容比**（）、**比熱比**（）或**絕熱膨脹係數**（），常用符號\(\gamma\)或\(\kappa\)表示。後者常在[化學工程領域中使用](https://zh.wikipedia.org/wiki/化學工程 "wikilink")，在[機械工程領域中](https://zh.wikipedia.org/wiki/機械工程 "wikilink")，會使用字母\(k\)表示絕熱指數\[1\]：

\[\gamma = \frac{C_P}{C_v} = \frac{c_P}{c_v}\]

其中，\(C\)是氣體的熱容，\(c\)是氣體[比熱容](../Page/比熱容.md "wikilink")，而下標\(P\)及\(v\)分別表示在等壓條件及等體積條件下的結果。

絕熱指數也可表示為以下的形式

\[\gamma = \frac{C_{P,m}}{C_{v,m}}\]

其中，\(C_{P,m}\)是氣體的等壓莫耳熱容，也就是一莫耳氣體的等壓熱容，\(C_{v,m}\)是氣體的等容莫耳熱容。

絕熱指數也是[理想氣體](../Page/理想氣體.md "wikilink")在[等熵過程](https://zh.wikipedia.org/wiki/等熵過程 "wikilink")（[準靜態](https://zh.wikipedia.org/wiki/準靜態過程 "wikilink")、[可逆的](https://zh.wikipedia.org/wiki/可逆過程 "wikilink")[絕熱過程](https://zh.wikipedia.org/wiki/絕熱過程 "wikilink")）下的多方指數，即以下體積和壓強關係式中體積的次方：

\[pV^\gamma = \text{const}\]

其中 \(p\) 是壓強而 \(V\) 是體積。

## 理想氣體的绝热指數

[理想氣體](../Page/理想氣體.md "wikilink")的熱容不隨溫度變化。[焓](../Page/焓.md "wikilink")及[內能分別為](https://zh.wikipedia.org/wiki/內能 "wikilink")\(H = C_p T\)及\(U = C_V T\)。因此绝热指數也可以視為是[焓](../Page/焓.md "wikilink")及[內能的比值](https://zh.wikipedia.org/wiki/內能 "wikilink")：

\[\gamma = \frac{H}{U}\]

理想氣體的定壓莫耳熱容及定容莫耳熱容及[氣體常數](../Page/氣體常數.md "wikilink")（\(R\)）之間有以下的關係

\[C_{p,m} - C_{V,m} = R \!\]

因此莫耳熱容也可以用绝热指數（\(\gamma\)）及氣體常數表示：

\[C_{p,m} = \frac{\gamma R}{\gamma - 1} \qquad \mbox{,} \qquad C_{V,m} = \frac{R}{\gamma - 1}\]

### 和自由度的關係

|       | \(C_{V,m}\)             | \(C_{p,m}\)              | \(\kappa=\frac{C_{p,m}}{C_{V,m}}\) |
| ----- | ----------------------- | ------------------------ | ---------------------------------- |
| 單原子氣體 | \(\frac{3}{2} \cdot R\) | \(\frac{5}{2} \cdot R\)  | \(\frac{5}{3}=1.\overline{6}\)     |
| 雙原子氣體 | \(\frac{5}{2} \cdot R\) | \(\frac{7}{2} \cdot R\)  | \(\frac{7}{5}=1.4\)                |
| 三原子氣體 | \(\frac{6}{2} \cdot R\) | \(\frac{8}{2} \cdot  R\) | \(\frac{4}{3}=1.\overline{3}\)     |

理想氣體分子的原子數和等容摩爾熱容（\(C_{V,m}\)）、等壓摩爾熱容（\(C_{p,m}\)）及绝热指数\(\kappa\)之間的關係

理想氣體的绝热指數（\(\gamma\)）和分子的[自由度之間](https://zh.wikipedia.org/wiki/自由度_\(物理學\) "wikilink")，有以下的關係：

\[\gamma\ = \frac{f+2}{f}\qquad \mbox{, } \qquad f = \frac{2}{\gamma-1}\]

[單原子氣體的自由度是](https://zh.wikipedia.org/wiki/單原子 "wikilink")3，因此绝热指數為：

\[\gamma\ = \frac{5}{3} \approx 1.67\], [雙原子氣體](https://zh.wikipedia.org/wiki/雙原子分子 "wikilink")，在室溫下的自由度為5（平移自由度3，旋轉自由度2，室溫下不考慮振動自由度），因此绝热指數為：

\[\gamma = \frac{7}{5} = 1.4\].

[空氣主要由雙原子氣體組成](https://zh.wikipedia.org/wiki/空氣 "wikilink")，包括約78%的[氮氣](https://zh.wikipedia.org/wiki/氮氣 "wikilink")（N<sub>2</sub>）及約21%的[氧氣](https://zh.wikipedia.org/wiki/氧氣 "wikilink")（O<sub>2</sub>），室溫下的乾燥空氣可視為理想氣體，因此其绝热指數為：

  -
    \(\gamma = \frac{5 + 2}{5} = \frac{7}{5} = 1.4\).

以上數據和實際量測而得的數據1.403相當接近。

## 熱力學的關係

在一些特定的工程應用中（如計算氣體經過導管或閥的流速），\(C_p - C_v = n R\)（n為莫耳數）的關係不夠準確，因此定體積熱容\(C_v\)需要由實測求得，若依下式計算定體積熱容\(C_v\)，即得求得精確的绝热指數\(\frac{C_p}{C_v}\)：

\[C_p - C_v \ = \ -T \frac{{\left( {\frac{\partial V}{\partial T}} \right)_P^2 }} {\left(\frac{\partial V}{\partial P}\right)_T} \ = \ -T \frac{{ \left( {\frac{\partial P}{\partial T}} \right) }^2} {\frac{\partial P}{\partial V}}\]

其中\(C_p\)的數值可以由查表求得。

上述的定義可由來推導嚴謹的[狀態方程式](https://zh.wikipedia.org/wiki/狀態方程式 "wikilink")（例如Peng-Robinson狀態方程式），這些方程式所求得的值和實測值非常接近，因此定體積熱容或绝热指數可直接用方程式計算，不需查表。也可以利用[有限差分法](../Page/有限差分法.md "wikilink")來計算其數值。

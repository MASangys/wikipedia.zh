**网络信标**（web beacon）也称**网页臭虫**（web bug）、**追踪臭蟲**（tracking bug），是指暗藏在任何[网页元素或](https://zh.wikipedia.org/wiki/网页 "wikilink")[邮件内約](https://zh.wikipedia.org/wiki/邮件 "wikilink")1[像素](../Page/像素.md "wikilink")大小的透明[GIF](../Page/GIF.md "wikilink")或[PNG](../Page/PNG.md "wikilink")图片，常用来收集目标电脑用户的上网习惯等数据，并将这些数据写入[Cookie](../Page/Cookie.md "wikilink")，网络信标常見於[垃圾邮件中](https://zh.wikipedia.org/wiki/垃圾邮件 "wikilink")。\[1\]

## 概述

网络信标（web beacon）也称为网络臭虫（web bug），是一个放置在网页或电子邮件上的文件对象，用于监测用户的行为。

它不像[Cookie](../Page/Cookie.md "wikilink")那样可以被浏览器用户接受或拒绝，网络臭虫只以图形交换格式（GIF）或其他文件对象的形式出现。

它通常只能被检测，如果用户查看网页的源版本会发现一个从不同的Web服务器而不是从网页的其他部分负载的标签。

虽然互联网隐私倡导者反对使用网络臭虫，但是他们大部分承认网络臭虫有积极用途，例如跟踪侵犯版权的网站。 根据Richard M.Smith，网络臭虫（web bug）可以收集以下资料：

1.  获取网络臭虫的计算机的IP地址
2.  网络臭虫所在网页的网址
3.  网络臭虫图象的网址
4.  网络臭虫被访问的时间
5.  获取网络臭虫图象的浏览器的类型
6.  一个提前设定的cookie值

网络信标（web beacon）经常被垃圾邮件发送者用来验证电子邮件地址。当收件人打开一封有网络信标的电子邮件时，返回给发件人的信息就会显示邮件已被打开，这样就可以确认电子邮件地址是有效的。

## Beacon API

信标API（Beacon API）是一种较新的Web技术，它不需要使用不可见图像或类似手段就能达到相同的目的。，它还是一个[万维网联盟](../Page/万维网联盟.md "wikilink")的候选建议。\[2\]其旨在使Web开发人员能在用户离开页面时将信息（如分析或诊断数据）发回Web服务器，以跟踪用户的活动。\[3\]使用Web信标API能够不干扰或影响网站导航的完成此种跟踪，并且对最终用户不可见。\[4\]信标API已于2014年被相继引入到[Mozilla Firefox](https://zh.wikipedia.org/wiki/Mozilla_Firefox "wikilink")\[5\]和[Google Chrome网页浏览器](../Page/Google_Chrome.md "wikilink")。\[6\]

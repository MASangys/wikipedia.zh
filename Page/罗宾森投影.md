[robinson-projection.jpg](https://zh.wikipedia.org/wiki/File:robinson-projection.jpg "fig:robinson-projection.jpg") **罗宾森投影法**（）是1960年代以来广泛使用的一种[地图投影](../Page/地图投影.md "wikilink")方法，这种投影方法可以同时在地图上展示整个世界。这种方法试图寻找一种在平面图上显示整个地球的方便的折衷方案。

## 介绍

罗宾森投影1963年由[亚瑟·H·罗宾森](https://zh.wikipedia.org/wiki/:en:Arthur_H._Robinson "wikilink")（）博士开发。使用了笔直的[纬线](https://zh.wikipedia.org/wiki/纬线 "wikilink")，且每根[经线之间的空间均匀分布](https://zh.wikipedia.org/wiki/经线 "wikilink")，罗宾森投影被列为[伪圆柱投影的一种](https://zh.wikipedia.org/wiki/伪圆柱投影 "wikilink")。中间的[经线也是直线](https://zh.wikipedia.org/wiki/经线 "wikilink")，而其它[经线则为曲线](https://zh.wikipedia.org/wiki/经线 "wikilink")。罗宾森规定了这一投影方法在对经纬度进行特定插值后参考[直角坐标系的数值表进行构建](https://zh.wikipedia.org/wiki/直角坐标系 "wikilink")。中间的位置通过[插值](../Page/插值.md "wikilink")后得到，可以参见下方的表格。这一方法也反映了在研究这一投影方法的过程中，他做了一系列的试验和插值直至确定了最适合的经线形状和纬线的间距。相比之下，多数其它投影方法只是一些数学方程。罗宾森投影也出现了很多方程的表示法，以取代查找表格的方法。

## 历史

罗宾森1946年起是[威斯康星大学麦迪逊分校](../Page/威斯康星大学麦迪逊分校.md "wikilink")地理系的教授，并于1980年退休。由于[Rand McNally出版社不满意现有对整个世界进行直观描述的现有投影方法](https://zh.wikipedia.org/wiki/:en:Rand_McNally "wikilink")，便委托罗宾森教授开发了这一投影方法。Rand McNally出版社从此在很多地图集和书中大量使用这一方法。这是第一个大型私有企业委托完成的重要地图投影方法。\[1\]这一投影方法最初被罗宾森命名为“矫形法（）”（意为“正确的样子”），但这个名称从来没有被使用，并很快就变成了著名的罗宾森投影。它也被称为“极线伪圆柱投影（）”，因为南北两极变成两条直线而不是其它投影方法中的极点。[国家地理学会](https://zh.wikipedia.org/wiki/国家地理学会 "wikilink")1988年在他们的世界地图上采用了这一方法，但在10年后废弃了这一投影改用[Winkel Tripel投影](https://zh.wikipedia.org/wiki/Winkel_Tripel投影 "wikilink")。许多教育机构和教科书跟着国家地理学会采用了这一投影，且大多数仍在采用。

## 优缺点

[Usgs_map_robinson.PNG](https://zh.wikipedia.org/wiki/File:Usgs_map_robinson.PNG "fig:Usgs_map_robinson.PNG") 和其它投影方法一样，罗宾森投影有优点也有缺点。这种投影方法不是[等积方法](https://zh.wikipedia.org/wiki/等积 "wikilink")，也没有做到[共形映射](../Page/共形映射.md "wikilink")。放弃这两点的同时，创造者希望能构建更好的整体视觉效果并可以尽量达到平衡。经度缓缓地弯曲，避免了巨大变化。因而将两极拉长到一条线而不是合成一点。因此两极的变形较厉害，但离两极不远的地方就很快回到了适当的水平。在地图外侧的高纬度区域有较严重的角度变形，这是在任何偽圆柱投影都存在的内在缺陷。但是，这种投影方法在研究出来时，有效地达到了Rand McNally想要美观描述整个世界的目的。

## 规范

这一投影方法由以下表格进行定义：

| 纬度 | PLEN   | PDFE   |
| -- | ------ | ------ |
| 00 | 1.0000 | 0.0000 |
| 05 | 0.9986 | 0.0620 |
| 10 | 0.9954 | 0.1240 |
| 15 | 0.9900 | 0.1860 |
| 20 | 0.9822 | 0.2480 |
| 25 | 0.9730 | 0.3100 |
| 30 | 0.9600 | 0.3720 |
| 35 | 0.9427 | 0.4340 |
| 40 | 0.9216 | 0.4958 |
| 45 | 0.8962 | 0.5571 |
| 50 | 0.8679 | 0.6176 |
| 55 | 0.8350 | 0.6769 |
| 60 | 0.7986 | 0.7346 |
| 65 | 0.7597 | 0.7903 |
| 70 | 0.7186 | 0.8435 |
| 75 | 0.6732 | 0.8936 |
| 80 | 0.6213 | 0.9394 |
| 85 | 0.5722 | 0.9761 |
| 90 | 0.5322 | 1.0000 |

这一表格使用纬度进行索引，并使用了[插值](../Page/插值.md "wikilink")的方法。**PLEN**列是纬线的长度，而**PDFE**列乘于0.5072可以得到纬线到赤道的距离。同一纬度的经度之间距离相等。

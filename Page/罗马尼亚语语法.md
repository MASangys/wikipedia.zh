## 冠词

罗马尼亚语的名词分阳阴中三性、单复两数。

|      | 单数             | 复数        |
| ---- | -------------- | --------- |
| 阳性   | 阴性             | 中性        |
| 定冠词  | *-ul, -le, -l* | *-a, -ua* |
| 不定冠词 | *un*           | *o*       |
|      |                |           |

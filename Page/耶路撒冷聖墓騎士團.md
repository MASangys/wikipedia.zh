**耶路撒冷聖墓騎士團**（），也稱**聖墓騎士團**，為[天主教](../Page/天主教.md "wikilink")，創立時名稱為「聖墓守護者團」（）。聖墓騎士團首先由[第一次十字軍東征領導者與](https://zh.wikipedia.org/wiki/第一次十字軍東征 "wikilink")[耶路撒冷王國](../Page/耶路撒冷王國.md "wikilink")統治者[布永的戈弗雷](../Page/布永的戈弗雷.md "wikilink")[公爵](../Page/公爵.md "wikilink")提倡，被稱做「[聖墓的先驅者](https://zh.wikipedia.org/wiki/聖墓 "wikilink")」\[1\]；1113年，該團由[教宗賈利二世發布](https://zh.wikipedia.org/wiki/教宗賈利二世 "wikilink")[教宗詔書認可](https://zh.wikipedia.org/wiki/教宗詔書 "wikilink")\[2\]。

聖墓騎士團的早期成員有詠禱司鐸。後來該團從[十字軍的部隊中](https://zh.wikipedia.org/wiki/十字軍 "wikilink")，選拔具有勇氣與奉獻精神的武裝騎士入團；這些[騎士發誓會遵守具有](https://zh.wikipedia.org/wiki/騎士 "wikilink")[守貧精神的](https://zh.wikipedia.org/wiki/真福八端 "wikilink")[奧古斯丁會規](https://zh.wikipedia.org/wiki/奧古斯丁會 "wikilink")，並服從耶路撒冷國王的指揮，專門保衛[聖墓教堂與聖地的安全](https://zh.wikipedia.org/wiki/聖墓教堂 "wikilink")。

[耶路撒冷王國](../Page/耶路撒冷王國.md "wikilink")滅亡後，任命新騎士的權力成為位於之最高天主教權力代表——的特權。

1496年，[教宗亞歷山大六世](../Page/教宗亞歷山大六世.md "wikilink")取得了總團長的頭銜，並將之傳承給繼任的教宗。1847年，[教宗庇護九世重建了](https://zh.wikipedia.org/wiki/教宗庇護九世 "wikilink")[天主教耶路撒冷宗主教區](../Page/天主教耶路撒冷宗主教區.md "wikilink")，並重新認可聖墓騎士團。1949年起，該團的總團長由[樞機](../Page/樞機.md "wikilink")擔任，教宗則維持騎士團的主權地位，因而可以享有[聖座](../Page/聖座.md "wikilink")的保護。目前聖墓騎士團的總部位於[羅馬](https://zh.wikipedia.org/wiki/羅馬 "wikilink")，靠近[梵蒂岡](https://zh.wikipedia.org/wiki/梵蒂岡 "wikilink")\[3\]。

## 組織

### 總團長

1496年，[教宗亞歷山大六世](../Page/教宗亞歷山大六世.md "wikilink")任命自己為聖墓騎士團的總團長，之後由教宗兼任總團長至1949年\[4\]。1949年起，由以下樞機擔任總團長：

| ----------- |
| 任期          |
| 1949年－1960年 |
| 1960年－1972年 |
| 1972年－1988年 |
| 1988年－1995年 |
| 1995年－2007年 |
| 2007年－2011年 |
| 2011年－2019年 |
| 2019年至今     |

**联立制**（*Mixed-Member Proportional representation*，MMP）又稱「補償制」（Top-Up system；Compensatory system）、「德國制」，是[单一选区两票制选举制度的一种](https://zh.wikipedia.org/wiki/单一选区两票制 "wikilink")，更接近于[比例代表制](../Page/比例代表制.md "wikilink")，以第二張圈選[政黨的票](https://zh.wikipedia.org/wiki/政黨 "wikilink")，決定每個政黨最終總席次。[德國為採用此制度的主要代表](https://zh.wikipedia.org/wiki/德國 "wikilink")。\[1\]\[2\]\[3\]\[4\]\[5\]\[6\]

## 席次分配

[German_Federal_Election_2017_-_Results_by_Constituency_&_Regional_Seats.svg](https://zh.wikipedia.org/wiki/File:German_Federal_Election_2017_-_Results_by_Constituency_&_Regional_Seats.svg "fig:German_Federal_Election_2017_-_Results_by_Constituency_&_Regional_Seats.svg")選舉結果，顯示了選區代表直接贏得的席位和通過黨派名單獲得的席位。例如，[自由民主黨](https://zh.wikipedia.org/wiki/自由民主黨_\(德國\) "wikilink")（黃色）沒有贏得一個單一選區; 所有80名議員都是在政黨名單上當選的。\]\] 各政党总当选名额，以全国总应选名额依各政党在第二张投政党名单的得票比率分配，扣除该党在選举区已得议席，其差额再由政党比例代表名额中补足。此選舉制度弱點與比例代表制一樣，容易導致无一党得过半数议席，需要组成[联合政府](https://zh.wikipedia.org/wiki/联合政府 "wikilink")。另外，在德国政党得票率要跨越5%门槛或在小选区得3席以上，才能分配議會席次。

假設某次選舉，預計會產生100席，其中60名為分區席次，另外將有40名為比例代表席次。

| 联立制                   | 政黨A            | 政黨B          | 政黨C            | 政黨D            |
| --------------------- | -------------- | ------------ | -------------- | -------------- |
| 政黨票數比例                | 25%            | 5%           | 40%            | 30%            |
| 全國分區當選席次              | 15             | 6            | 19             | 20             |
| 政黨總席次計算（預計總席次×政黨票數比例） |                |              |                |                |
| 政黨總席次                 | 100×25%＝**25** | 100×5%＝**5** | 100×40%＝**40** | 100×30%＝**30** |
| 分配席次計算（政黨總席次－分區當選席次）  |                |              |                |                |
| 比例代表席次                | 25－15＝**10**   | 5＜6**0**     | 40－19＝**21**   | 30－20＝**10**   |
| 總計                    |                |              |                |                |
| 實際總席次                 | 25             | 6            | 40             | 30             |

政黨B在政黨票獲百分之五，它在國會的席次就是100席中的百分之五，就是5席，但政黨B在全國分區議員選舉中卻獲得6席。因為已經超過數額，故B黨將不再獲得任何政黨代表。所以最後，B黨獲得6席。這多出的一席稱為「**[超額席次](https://zh.wikipedia.org/wiki/超額席次 "wikilink")**」（overhang seats）。

政黨ABCD在本次選舉中總共有101席，超過原先預定的100席，所以採用此制度的議席數多半是不固定的。因此聯立制比較並立制，更加偏向[比例代表制](../Page/比例代表制.md "wikilink")。

## 優缺點

此種制度的好處是，某政黨就算沒有地區候選人當選，甚至不提名地區候選人，只要通過政黨門檻就可獲得席次，有利於小型政黨，且不會有並立制「總得票率與席次不對等」之困擾。

但缺點是，因為聯立制的總席次百分比是與政黨票得票率一致，比例代表席次由應得總席次扣除地區當選者得出，不同於[並立制的比例代表席次百分比是與政黨票得票率一致](https://zh.wikipedia.org/wiki/並立制 "wikilink")，若是某政黨於小選區所獲席次愈多，愈不利於政黨名單候選人，最極端的情形則會造成比例代表候選人通通落選，容易造成政黨內部關係緊張。因為可能會有超額席次，使席次分配變得困難，也會因為席次過多造成後續政府組成困難，所以比例代表席次的比例亦不能太少，以降低這種情形的發生率。

不過，現時實行的國家如[德國及](https://zh.wikipedia.org/wiki/德國 "wikilink")[纽西兰](https://zh.wikipedia.org/wiki/纽西兰 "wikilink")，由於設有百分之五的得票門檻，單一政黨雖然難以在得票佔多數下獲過半議席，但小黨仍難以跨越門檻進入國會，仍然有利政府組成。

例如：

  - [2013年德國聯邦議院選舉](https://zh.wikipedia.org/wiki/2013年德國聯邦議院選舉 "wikilink")，[自民黨](https://zh.wikipedia.org/wiki/德國自由民主黨 "wikilink")（FDP）和反[歐元](https://zh.wikipedia.org/wiki/歐元 "wikilink")[另類選擇黨](../Page/德國另類選擇.md "wikilink")（AfD）分別取得4.8%及4.7%的政黨票，但因未能跨越5%政黨票門檻，未能進入國會，自民黨更因此失去上屆聯邦議院的全部93個議席。
  - 不過AfD在2015年起看準總理[難民政策問題和歐盟問題](../Page/歐洲移民危機.md "wikilink")，藉此再提反移民的主張，讓該黨在[2017年德國聯邦議院選舉獲得](https://zh.wikipedia.org/wiki/2017年德國聯邦議院選舉 "wikilink")12.6%的選票而成為二戰後進入議院的極右政黨，[自民黨](https://zh.wikipedia.org/wiki/德國自由民主黨 "wikilink")（FDP）亦在複雜選情之下以10.7%重返議會。

## 計算制度

### 超額席次

前述的政黨B在政黨票獲百分之五，它在國會的席次就是100席中的百分之五，就是5席，但政黨B在全國分區議員選舉中卻獲得6席。因為已經超過數額，故B黨將不再獲得任何政黨代表。

### 變體

聯立制亦可像並立制度存在變體：例如只有一張選票，席次固定。

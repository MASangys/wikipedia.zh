**聖母幼稚園(Our Lady's
Kindergarten）**是一所由[香港](../Page/香港.md "wikilink")[天主教](../Page/天主教.md "wikilink")[母佑會所創辦的非牟利幼稚園](../Page/母佑會.md "wikilink")，於1953年創校，歷史悠久。其直屬中學及小學分別為[聖母書院及](../Page/聖母書院.md "wikilink")[聖母小學](../Page/聖母小學.md "wikilink")。該校位於[九龍](../Page/九龍.md "wikilink")[黃大仙沙田坳道](https://zh.wikipedia.org/wiki/黃大仙 "wikilink")113-115號，現任校長為胡妙然修女。

## 辦學宗旨

校方自稱以仁愛、理智、宗教及慈幼家庭精神，來培育學生對生命、品德、學業、靈性的熱愛。

## 學校歷史

|        |                                                               |
| ------ | ------------------------------------------------------------- |
| **年份** | **大事表**                                                       |
| 1953   | 聖母學校在[黃大仙竹園奠基](https://zh.wikipedia.org/wiki/黃大仙 "wikilink")。 |
| 1954   | 聖母學校開辦幼稚園及小學一、二年級，由 Sr. Anthonietta Pilla (白志潔修女)任校監、校長。      |
| 1959   | 聖母學校在沙田坳道擴建校舍。增設英文中學部。                                        |
| 1961   | 聖母學校改名為[聖母書院](../Page/聖母書院.md "wikilink")，分(中學部)、(小學部)及(幼稚園)。 |
| 1979   | [聖母書院慶祝創校銀禧紀念](../Page/聖母書院.md "wikilink")，舉行開放日。             |
| 1998   | 聖母書院(幼稚園)家長教師會成立，由九位家長及九位老師組成執行委員會。                           |
| 1999   | 聖母書院(幼稚園)改名為聖母幼稚園。                                            |
| 2004   | 聖母幼稚園新翼落成。慶祝創校金禧紀念。                                           |

## 學校設施

校舍環境偌大，佔地約3500平方米，由樓高三層的主校舍及樓高六層的新翼組成，是本港目前少數擁有獨立建築物的幼稚園。該校有課室12間、另設有創作室、遊戲室、視聽室、音樂室、碟趣坊、體能室、語言室、舞蹈室和圖書館。此外，還有一個操場及兩個戶外場地（包括一個後園種植小園地）。

## 班級結構

  - 分上午班和下午班。
  - 各設有幼兒班、低班及高班。

## “没有香港人”

在以“我爱中国”为主题的写作课中，学生写上“我是香港人”，老师即在评核时用红笔写下“没有香港人的！”。网民强烈不满，为何有“北京人”“台北人”但却没有“香港人”，校方解释该写作课要求学生明白国籍是以国家为本\[1\]\[2\]。

[Spike-waves.png](https://zh.wikipedia.org/wiki/File:Spike-waves.png "fig:Spike-waves.png") **腦電圖**、**腦波圖**（）是透过[医学](../Page/医学.md "wikilink")仪器脑电图描记仪，将人体[脑部自身产生的微弱](https://zh.wikipedia.org/wiki/脑部 "wikilink")[生物电于](https://zh.wikipedia.org/wiki/生物电 "wikilink")[头皮处收集](https://zh.wikipedia.org/wiki/头皮 "wikilink")，并放大记录而得到的曲线图。脑电图所测量的是众多锥体细胞兴奋时的突触后电位的同步总和，脑电图测量来自[大脑](../Page/大脑.md "wikilink")中[神经元的](https://zh.wikipedia.org/wiki/神经元 "wikilink")产生的电压波动\[1\]。脑电图用于辅助诊断脑部相关疾病，但因为其易受到干扰，故临床上通常要结合其他手段来使用。

脑电图最常用于诊断[癫痫](../Page/癫痫.md "wikilink")，因为癫痫会导致异常的脑电图读数\[2\]。它也被用于诊断[睡眠障碍](../Page/睡眠障碍.md "wikilink")、[昏迷](../Page/昏迷.md "wikilink")、脑血管疾病等，和[脑死](../Page/脑死.md "wikilink")。脑电波曾被用于诊断[肿瘤](https://zh.wikipedia.org/wiki/肿瘤 "wikilink")、[中风和其他局灶性脑疾病的第一线方法](https://zh.wikipedia.org/wiki/中风 "wikilink")\[3\]，但这种使用已经随着高分辨率解剖成像技术的出现而减少，例如[核磁共振](../Page/核磁共振.md "wikilink")（MRI）和[電腦断层扫描](https://zh.wikipedia.org/wiki/X光成像 "wikilink")（CT）。與CT和MRI不同，腦電圖具有較高的時間分辨率（temporal resolution）。因此，尽管EEG的空间分辨率有限，它仍然是用于研究和诊断的宝贵工具，特别是当指定研究需要毫秒范围的时间分辨率時。

## 腦波種類

[EEG_cap.jpg](https://zh.wikipedia.org/wiki/File:EEG_cap.jpg "fig:EEG_cap.jpg")

|              |                       |                          |
| ------------ | --------------------- | ------------------------ |
| 腦波種類         | 頻率                    | 人體狀態                     |
| Delta（δ）     | 0.1\~3 Hz             | 深度睡眠且沒有做夢時               |
| Theta（θ）     | 4\~7Hz                | 成人情緒受到壓力時，尤其是失望或挫折       |
| Alpha（α）     | 8\~12Hz               | 放鬆、平靜、閉眼但清醒時             |
| Beta（β）      | Low Range             | 12.5 \~ 16 Hz            |
| Middle Range | 16.5 \~ 20 Hz         | 思考、處理接收到外界訊息（聽到或想到）      |
| High Range   | 20.5 \~ 28 Hz         | 激動、焦慮                    |
| Gamma（γ）     | 25 \~ 100 Hz（通常在40Hz） | 提高意識、幸福感、減輕壓力、冥想         |
| Lambda（λ）    | 誘發電位                  | 眼睛受光刺激時100ms後誘發（又稱作P100） |
| P300         | 誘發電位                  | 看到或聽到腦中想像的東西時300ms後誘發    |
|              |                       |                          |

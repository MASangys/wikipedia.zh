在[粒子物理學](../Page/粒子物理學.md "wikilink")中，**膠子**（gluon）是負責在兩個[夸克](../Page/夸克.md "wikilink")之間傳遞[強作用力的](https://zh.wikipedia.org/wiki/強作用力 "wikilink")[基本粒子](../Page/基本粒子.md "wikilink")，類似[光子](../Page/光子.md "wikilink")負責在兩個帶電粒子之間傳遞[電磁力](../Page/電磁力.md "wikilink")一般。\[1\]

用科學術語來說明，膠子是[量子色動力學](../Page/量子色動力學.md "wikilink")用來在兩個夸克之間傳遞[強相互作用的](https://zh.wikipedia.org/wiki/強相互作用 "wikilink")[矢量](https://zh.wikipedia.org/wiki/矢量 "wikilink")[規範玻色子](https://zh.wikipedia.org/wiki/規範玻色子 "wikilink")。膠子本身帶有強相互作用的[色荷](../Page/色荷.md "wikilink")，這與光子不同，光子不帶有[電荷](../Page/電荷.md "wikilink")。因此，膠子不但傳遞強相互作用，它還參與強相互作用，這使得量子色動力學的分析遠比[量子電動力學](../Page/量子電動力學.md "wikilink")困難。\[2\]

## 性質

膠子是[矢量](https://zh.wikipedia.org/wiki/矢量 "wikilink")[規範玻色子](https://zh.wikipedia.org/wiki/規範玻色子 "wikilink")，如同光子，它的[自旋](../Page/自旋.md "wikilink")為1。自旋為1的帶質量粒子可以擁有三種[偏振](../Page/偏振.md "wikilink")態。在[量子場論裏](https://zh.wikipedia.org/wiki/量子場論 "wikilink")，為了滿足[局域規範不變性](https://zh.wikipedia.org/wiki/希格斯機制#局域規範不變性 "wikilink")，規範玻色子的質量必須為零，因此膠子不帶質量（實驗上限為\[3\]），所以，只有兩種偏振態。\[4\]

類似光子，膠子是個[矢量粒子](https://zh.wikipedia.org/wiki/矢量 "wikilink")，所以膠子的[內秉宇稱是負數](../Page/宇稱.md "wikilink")-1。\[5\]

## 膠子的工作機制

在量子色動力學裏，依照所帶有的色荷與反色荷來區分，一共存在有8種不同的膠子。每個夸克都帶有三個不同的色荷：紅色、藍色與綠色。每個反夸克都帶有三個不同的反色荷：反紅色、反藍色與反綠色。每個膠子帶有一個色荷與一個反色荷。要想正確了解它們怎樣組合在一起，就必需更仔細地思考色荷數學。

### 色荷與量子疊加

在量子力學裏，根據[態疊加原理](https://zh.wikipedia.org/wiki/態疊加原理 "wikilink")，假若粒子處於由幾種[量子態](../Page/量子態.md "wikilink")組合而成的[疊加態](https://zh.wikipedia.org/wiki/疊加態 "wikilink")，則粒子處於其中任意量子態的[概率](../Page/概率.md "wikilink")為有限值；假若對於這系統測量某物理量，則可能得到幾種不同的數值。\[6\]例如，設定膠子的色態為紅色-反藍色加上藍色-反紅色：

\[(r\bar{b}+b\bar{r})/\sqrt{2}\]；

其中，\(r\)、\(b\)、\(\bar{r}\)、\(\bar{b}\)分別為紅色、藍色、反紅色、反藍色。

假若測量此膠子的顏色，則它的色荷是紅色-反藍色的機會為50%，是藍色-反紅色的機會為50% 。

### 色單態

在科普界時常會提到，由幾個夸克或反夸克組成的穩定粒子，例如像質子與中子一類的強子，假若能夠在大自然被觀察得到，則總色荷必須是無色的（或白色的）：\[7\]

更精準地說，所有自然發生的粒子都處於色單態，這是為了遵守[夸克禁閉](../Page/夸克禁閉.md "wikilink")。例如，每一種重子的波函數在顏色部分必須是反對稱的色單態：\[8\]

\[|\mathrm{baryon\ color}\rang=(rgb-rbg+grb-gbr+brg-bgr)/\sqrt{6}\]； 其中，\(g\)為綠色。

膠子的色單態為\[9\]

\[|\mathrm{gluon\ color}\rang=(r\bar{r}+b\bar{b}+g\bar{g})/\sqrt{3}\]。

由於色單態的膠子不違反夸克禁閉，它應該能夠以自由粒子的形式存在於大自然，並且在兩個強子之間傳遞強作用力，例如，在質子與中子之間傳遞強作用力，因此，強作用力也會變為遠距作用力，但是強作用力是一種短距作用力，因此色單態的膠子不存在。

### 八種膠子

每個膠子帶有一個單位色荷的顏色與一個單位色荷的反顏色。顏色可以是紅色\(r\)、藍色\(b\)或綠色\(g\)。反顏色可以是反紅色\(\bar{r}\)、反藍色\(\bar{b}\)或反綠色\(\bar{g}\)。所以，膠子可能處於九種不同的色態，分別為\(r\bar{r}\)、\(r\bar{b}\)、\(r\bar{g}\)、\(b\bar{r}\)、\(b\bar{b}\)、\(b\bar{g}\)、\(g\bar{r}\)、\(g\bar{b}\)、\(g\bar{g}\)。實際而言，膠子是處於這九種色態的線性獨立組合，但是由於先前提到的色單態並不存在，所以只有八種色態，分別為\[10\]

|                                  |  |                                             |
| -------------------------------- |  | ------------------------------------------- |
| \((r\bar{b}+b\bar{r})/\sqrt{2}\) |  | \(-i(r\bar{b}-b\bar{r})/\sqrt{2}\)          |
| \((r\bar{g}+g\bar{r})/\sqrt{2}\) |  | \(-i(r\bar{g}-g\bar{r})/\sqrt{2}\)          |
| \((b\bar{g}+g\bar{b})/\sqrt{2}\) |  | \(-i(b\bar{g}-g\bar{b})/\sqrt{2}\)          |
| \((r\bar{r}-b\bar{b})/\sqrt{2}\) |  | \((r\bar{r}+b\bar{b}-2g\bar{g})/\sqrt{6}\)。 |

這些色態共同組成「八重態」，或「色八重態」。這些色態的色波函數等價於[蓋爾曼矩陣](https://zh.wikipedia.org/wiki/蓋爾曼矩陣 "wikilink")，分别为：

  -
    {| border="0" cellpadding="5" cellspacing="0"

|\(\lambda_1 = \begin{pmatrix} 0 & 1 & 0 \\ 1 & 0 & 0 \\ 0 & 0 & 0 \end{pmatrix}\) |\(\lambda_2 = \begin{pmatrix} 0 & -i & 0 \\ i & 0 & 0 \\ 0 & 0 & 0 \end{pmatrix}\) |\(\lambda_3 = \begin{pmatrix} 1 & 0 & 0 \\ 0 & -1 & 0 \\ 0 & 0 & 0 \end{pmatrix}\) |- |\(\lambda_4 = \begin{pmatrix} 0 & 0 & 1 \\ 0 & 0 & 0 \\ 1 & 0 & 0 \end{pmatrix}\) |\(\lambda_5 = \begin{pmatrix} 0 & 0 & -i \\ 0 & 0 & 0 \\ i & 0 & 0 \end{pmatrix}\) |\(\lambda_6 = \begin{pmatrix} 0 & 0 & 0 \\ 0 & 0 & 1 \\ 0 & 1 & 0 \end{pmatrix}\) |- |\(\lambda_7 = \begin{pmatrix} 0 & 0 & 0 \\ 0 & 0 & -i \\ 0 & i & 0 \end{pmatrix}\) |\(\lambda_8 = \frac{1}{\sqrt{3}} \begin{pmatrix} 1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & -2 \end{pmatrix}\) | |}

在\(\lambda_1\)蓋爾曼矩陣裏，第一橫排第二豎排的元素對應於紅色-反藍色，第二橫排第一豎排的元素對應於藍色-反紅色，整個矩陣對應於色態\((r\bar{b}+b\bar{r})/\sqrt{2}\)。

這八個色態彼此線性獨立，並且獨立於色單態；幾個色態的任何線性組合都無法複製其它色態；它也無法製成\(r\bar{r}\)、\(b\bar{b}\)、\(g\bar{g}\)，否則，禁戒的色單態也可被製成。\[11\]

上述八重態並不是唯一選擇，也可挑選其它種八重態，但它們都數學等價，會給出同樣的物理結果，並且運算過程不會更簡易。

### 群論

用[群論來表述](https://zh.wikipedia.org/wiki/群論 "wikilink")，色單態不存在這句話只是量子色動力學具有[SU(3)對稱](../Page/特殊酉群.md "wikilink")（八個線性獨立態），而不具有[U(3)](../Page/酉群.md "wikilink")（九個線性獨立態）對稱的體現。並沒有任何[先天理由選擇這種群而不選擇那種群](https://zh.wikipedia.org/wiki/先天 "wikilink")，但是實驗結果支持SU(3)對稱。\[12\]

## 禁閉

[Quark_confinement.svg](https://zh.wikipedia.org/wiki/File:Quark_confinement.svg "fig:Quark_confinement.svg") 由於膠子本身帶有色荷，膠子也參與強相互作用。膠子-膠子相互作用使得[色場成為像絲弦一般的物體](../Page/量子色動力學.md "wikilink")，稱為「通量管」（flux tube）。\[13\]當通量管被拉長時，會出現[張力](../Page/張力.md "wikilink")，因此將夸克禁閉於強子內部，這機制有效地局限強作用力的範圍半徑至 m以內，大約為[原子核](../Page/原子核.md "wikilink")的尺寸。當超過某特定長度後，假若連結兩個夸克的通量管的長度越長，則能量越高，呈線性增長；當通量管被拉到足夠長之時，在能量方面，從真空製成一個夸克-反夸克對會比一味地增加通量管長度更為有利，這時，繼續拉長通量管可能會導致通量管會斷裂，形成一個夸克-反夸克對。\[14\]

雖然在，單獨膠子無法自由移動，物理學者猜測，可能存在純粹由膠子形成的強子，稱為[膠球](../Page/膠球.md "wikilink")。\[15\]其它種[奇異強子](../Page/奇異強子.md "wikilink")也可能存在，這些奇異強子的重要成分將會是真實膠子，而不是[虛膠子](../Page/虛粒子.md "wikilink")。當不處於正常相之時，即在極端高溫與極端高壓強狀況，會形成[夸克-膠子等離子體](https://zh.wikipedia.org/wiki/夸克-膠子等離子體 "wikilink")，在這夸克-膠子等離子體裏，不會有機會形成強子，因為夸克與膠子都會變成自由粒子。\[16\]

## 實驗觀察

[Feynman_Diagram_Y-3g.svg](https://zh.wikipedia.org/wiki/File:Feynman_Diagram_Y-3g.svg "fig:Feynman_Diagram_Y-3g.svg") 夸克與膠子藉著分裂成更多夸克與膠子來顯現自己。這些分裂出的夸克與膠子又會成為無色的正常粒子。1978年夏季，在國際會議、座談會、專題研討會等等多個學術場合裏，\[17\][德國電子加速器的](https://zh.wikipedia.org/wiki/德國電子加速器 "wikilink")[正負電子對撞機與儲存環](../Page/粒子加速器.md "wikilink")（DORIS）的報告，發現非常狹窄共振Y(9.46)的強子型衰變可以詮釋為由三個膠子製成的 的證據。同一團隊後來發表分析報告確定這詮釋正確無誤，並且展示出膠子的自旋為1。\[18\]<ref>

</ref>\[19\]\[20\]

1979年夏季，在[德國電子加速器的正負電子對撞機](https://zh.wikipedia.org/wiki/德國電子加速器 "wikilink")，、\[21\] 、\[22\] \[23\]（後來，在1980年，\[24\])）又觀察到三重噴流事件，這被詮釋為q膠子[軔致輻射](https://zh.wikipedia.org/wiki/軔致輻射 "wikilink")，現在更為明顯可見。 1980年，\[25\]與\[26\]確定膠子的自旋為1。\[27\]1991年，在[歐洲核子研究組織](../Page/歐洲核子研究組織.md "wikilink")[大型正負電子對撞機儲存環完成的一項後續實驗確定這結果正確無誤](https://zh.wikipedia.org/wiki/大型正負電子對撞機 "wikilink")\[28\]

在[德國電子加速器的](https://zh.wikipedia.org/wiki/德國電子加速器 "wikilink")，膠子的物理性質被特別地研究分析。與，這兩項實驗對於膠子在質子裏的數量分佈與動量分佈做出仔細測量。\[29\]從1996年至2007年，研究膠子對於質子自旋的貢獻。從蒐集的光子製備數據，被用來計算光子內部的膠子密度，當光子呈現強子行為之時。\[30\]

色禁閉可以用無法找到自由夸克來核對，也就是說無法找到非整數的電荷。通常，為了抵銷量子顏色與風味量子數，夸克會成對產生（夸克與反夸克）。可是，在[費米實驗室的](https://zh.wikipedia.org/wiki/費米實驗室 "wikilink")團隊與團隊於2009年報告，探測到[頂夸克](../Page/頂夸克.md "wikilink")單獨產生的證據（雖然這仍舊涉及到成對產生，但是夸克與反夸克的風味不同）。\[31\]至今為止，尚未能找到任何[膠球](../Page/膠球.md "wikilink")存在的證據。

2000年，[歐洲核子研究組織](../Page/歐洲核子研究組織.md "wikilink")的[超級質子同步加速器](../Page/超級質子同步加速器.md "wikilink")聲稱，在重離子對撞時觀察到，\[32\]這意味著觀察到一種新的物質態：[夸克-膠子等離子體](https://zh.wikipedia.org/wiki/夸克-膠子等離子體 "wikilink")。2004年至2010年，在[布魯克黑文國家實驗室的](https://zh.wikipedia.org/wiki/布魯克黑文國家實驗室 "wikilink") （RHIC），四個不同實驗同時期找到夸克-膠子等離子體。\[33\]2010年，在[歐洲核子研究組織](../Page/歐洲核子研究組織.md "wikilink")的[大型強子對撞機](../Page/大型強子對撞機.md "wikilink")，三個實驗[大型離子對撞機實驗](https://zh.wikipedia.org/wiki/大型離子對撞機實驗 "wikilink")、[超環面儀器](../Page/超環面儀器.md "wikilink")與[緊湊緲子線圈](../Page/緊湊緲子線圈.md "wikilink")確定探測到夸克-膠子等離子體。\[34\]

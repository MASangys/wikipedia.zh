**自偶电离**（**Molecular autoionization**）是指[液态的](https://zh.wikipedia.org/wiki/液态 "wikilink")[极性](../Page/极性.md "wikilink")共价[分子化合物电负性强的部分与电负性弱的部分相互作用](https://zh.wikipedia.org/wiki/分子化合物 "wikilink")，自发[电离的过程](https://zh.wikipedia.org/wiki/电离 "wikilink")，电离後的產物是分別是由分子形成的阴离子及阳离子。极性共价分子化合物可以是水，也可以是其他液态极性的分子化合物

只要是液态极性共价分子化合物就可发生自偶电离，极性越强电离程度越大。

## 例子

  -
    BrF<sub>3</sub> + BrF<sub>3</sub> → BrF<sub>4</sub><sup>-</sup> + BrF<sub>2</sub><sup>+</sup>
    NH<sub>3</sub> + NH<sub>3</sub> → NH<sub>4</sub><sup>+</sup> + NH<sub>2</sub><sup>-</sup>
    CH<sub>3</sub>CH<sub>2</sub>OH + CH<sub>3</sub>CH<sub>2</sub>OH → CH<sub>3</sub>CH<sub>2</sub>OH<sub>2</sub><sup>+</sup> + CH<sub>3</sub>CH<sub>2</sub>O<sup>-</sup>
    H<sub>2</sub>O + H<sub>2</sub>O → H<sub>3</sub>O<sup>+</sup> + OH<sup>-</sup>
    SOCl<sub>2</sub> → SOCl<sup>+</sup> + Cl<sup>-</sup>
    2NO<sub>2</sub> → NO<sup>+</sup> + NO<sub>3</sub><sup>-</sup>
    2SO<sub>2</sub> → SO<sup>2+</sup> + SO<sub>3</sub><sup>2-</sup>

至于非极性共价分子在气态时也偶有自偶电离，例如：

  -
    2PCl<sub>5</sub> → PCl<sub>6</sub><sup>-</sup> + PCl<sub>4</sub><sup>+</sup>
    2PBr<sub>5</sub> → PBr<sub>6</sub><sup>-</sup> + PBr<sub>4</sub><sup>+</sup>

**臺灣發電廠列表**為列出當前[中華民國](../Page/中華民國.md "wikilink")[臺灣地區](https://zh.wikipedia.org/wiki/台澎金馬 "wikilink")1945年以後包含已停機或改建中[發電廠在內的所有發電廠清單](../Page/發電廠.md "wikilink")。

## 概況

當前，臺灣地區所有發電廠之總裝置容量截止2016年達42,132.5[MW](https://zh.wikipedia.org/wiki/MW "wikilink")。下列為當前各種發電類型所佔的裝置容量

| 發電類型                                                    | 裝置容量（[MW](https://zh.wikipedia.org/wiki/MW "wikilink")） | 含括項目          |
| ------------------------------------------------------- | ------------------------------------------------------- | ------------- |
| [水力發電](https://zh.wikipedia.org/wiki/水力發電 "wikilink")   | 4,683.3                                                 | 慣常式、抽蓄式       |
|                                                         |                                                         |               |
| [火力發電](https://zh.wikipedia.org/wiki/火力發電 "wikilink")   | 30,065.2                                                | 燃煤、汽電共生、燃油、輕油 |
|                                                         |                                                         |               |
| [核能發電](https://zh.wikipedia.org/wiki/核能發電 "wikilink")   | 5,144.0                                                 | 核分裂（未計入核四廠）   |
|                                                         |                                                         |               |
| [風力發電](https://zh.wikipedia.org/wiki/風力發電 "wikilink")   | 690.3                                                   | 陸地風力、離岸風力     |
|                                                         |                                                         |               |
| [太陽能發電](https://zh.wikipedia.org/wiki/太陽能發電 "wikilink") | 1,206.0                                                 | 太陽光電          |
|                                                         |                                                         |               |
|                                                         |                                                         |               |

[臺灣多數發電廠由](../Page/臺灣.md "wikilink")[國營的](https://zh.wikipedia.org/wiki/國營 "wikilink")[台灣電力公司運作](https://zh.wikipedia.org/wiki/台灣電力公司 "wikilink")，另有[長生電力公司](https://zh.wikipedia.org/wiki/長生電力公司 "wikilink")、[國光電力公司](https://zh.wikipedia.org/wiki/國光電力公司 "wikilink")、[新桃電力公司](https://zh.wikipedia.org/wiki/新桃電力公司 "wikilink")、[星元電力公司](https://zh.wikipedia.org/wiki/星元電力公司 "wikilink")、[星能電力公司](https://zh.wikipedia.org/wiki/星能電力公司 "wikilink")、[麥寮汽電公司](https://zh.wikipedia.org/wiki/麥寮汽電公司 "wikilink")、[和平電力公司](../Page/和平電廠.md "wikilink")、[嘉惠電力公司](https://zh.wikipedia.org/wiki/嘉惠電廠 "wikilink")、[森霸電力公司等民營發電廠營運以](https://zh.wikipedia.org/wiki/森霸電力公司 "wikilink")[煤或](../Page/煤.md "wikilink")[天然氣為燃料的火力發電廠](https://zh.wikipedia.org/wiki/天然氣 "wikilink")\[1\]。

民營水力發電廠則有[嘉南實業股份有限公司於](https://zh.wikipedia.org/wiki/嘉南實業股份有限公司 "wikilink")[臺南市](https://zh.wikipedia.org/wiki/臺南市 "wikilink")[烏山頭水庫設置](../Page/烏山頭水庫.md "wikilink")[烏山頭](https://zh.wikipedia.org/wiki/烏山頭水力發電廠 "wikilink")、[八田](../Page/八田水力發電廠.md "wikilink")、[西口等三座小型水力發電廠](../Page/西口水力發電廠.md "wikilink")，[臺中市](https://zh.wikipedia.org/wiki/臺中市 "wikilink")[臺中農田水利會旗下的](https://zh.wikipedia.org/wiki/臺中農田水利會 "wikilink")[后里圳低落差示範電廠](../Page/后里圳低落差示範電廠.md "wikilink")，目前由台灣電力公司[大甲溪發電廠代營運](../Page/大甲溪發電廠.md "wikilink")，[南投縣](../Page/南投縣.md "wikilink")[名間水力發電廠為](../Page/名間水力發電廠.md "wikilink")[經濟部水利署與](https://zh.wikipedia.org/wiki/經濟部水利署 "wikilink")[名間電力公司所合作之](https://zh.wikipedia.org/wiki/名間電力公司 "wikilink")[BOT開發案的民營水力發電廠](https://zh.wikipedia.org/wiki/BOT "wikilink")。[臺東縣有](../Page/臺東縣.md "wikilink")[聚電企業開發股份有限公司所投資興建之](https://zh.wikipedia.org/wiki/聚電企業開發股份有限公司 "wikilink")[卑南上圳小型發電廠](../Page/卑南上圳小型發電廠.md "wikilink")。

**臺灣蜆**（[学名](https://zh.wikipedia.org/wiki/学名 "wikilink")：）\[1\]\[2\]\[3\]，为[蜆科](https://zh.wikipedia.org/wiki/蜆科 "wikilink")[蜆屬](../Page/蜆屬.md "wikilink")[河蚬](../Page/河蚬.md "wikilink")下的一个亞种。

在台灣，可食用的[雙殼綱貝類都被泛稱為](https://zh.wikipedia.org/wiki/雙殼綱 "wikilink")[蛤蜊](../Page/蛤蜊.md "wikilink")，如[文蛤](https://zh.wikipedia.org/wiki/文蛤 "wikilink")、[花蛤](https://zh.wikipedia.org/wiki/花蛤 "wikilink")、粉蛤、[竹蛤](https://zh.wikipedia.org/wiki/竹蛤 "wikilink")，相當於古代的[蚶](https://zh.wikipedia.org/wiki/蚶 "wikilink")、[車螯和](https://zh.wikipedia.org/wiki/車螯 "wikilink")[花蛤](https://zh.wikipedia.org/wiki/花蛤 "wikilink")。居住在[海水](../Page/海水.md "wikilink")中者稱**蚶仔**（），主要是指[文蛤](https://zh.wikipedia.org/wiki/文蛤 "wikilink")。居住在[淡水](../Page/淡水.md "wikilink")中者**蜊仔**（），或[蜆](https://zh.wikipedia.org/wiki/蜆 "wikilink")。

## 分布

分佈極廣，遍及韓國、日本、中國大陸及台灣，主要棲息於淡水或[半淡鹹水區之河川](https://zh.wikipedia.org/wiki/汽水_\(水域\) "wikilink")、湖泊、水塘、溝渠、水田之泥沙底。天然蜆苗多產於稍具[鹽分之河口地帶](https://zh.wikipedia.org/wiki/鹽 "wikilink")。喜棲息有緩流水、無水草、沙泥底、水深左右的水域環境中。

## 型態

接近正三角形，腹緣圓形，有明顯的生長線，外殼顏色為黃色、黃綠色或黑褐色，內殼為紫色或白色。而水質越乾淨則顏色越金黃。
當中原產於台灣[花蓮縣](../Page/花蓮縣.md "wikilink")[壽豐鄉](../Page/壽豐鄉.md "wikilink")的特稱作「黃金蜆」。
現時市場上亦有生產商以「黃金蜆」製作成碇狀的營養補充劑。

## 生命史

| 時間        | 分期                                                        |
| --------- | --------------------------------------------------------- |
| 精卵釋出      | 水中受精                                                      |
| 2小時30分後   | 開始分裂                                                      |
| 14\~20時候  | 達到囊胚期，開始迴轉運動                                              |
| 20小時左右後   | 發育成擔輪子 (Trochophore)，大小約 150μｍ，具一圈纖毛環及一束細小的頂纖毛，而後孵出行浮游生活。 |
| 孵出之後      | 進入被面子期(Veliger stage) ，此期初具有面盤(Velum) ，而後形成透明的貝殼，面盤即消失。   |
| 持續成長到殼頂鼓出 | 形成稚蜆(Juvenile stage)，則改為底棲生活。                             |
| 2～3個月     | 較小的蜆苗（供應一般養殖使用）                                           |
| 4～5個月     | 較大的蜆苗（供應一般養殖使用）                                           |
|           |                                                           |

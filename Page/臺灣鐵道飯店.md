**臺灣鐵道飯店**（，）是[臺灣](../Page/臺灣.md "wikilink")[日治時期的一座洋式](../Page/台灣日治時期.md "wikilink")[旅館](../Page/旅館.md "wikilink")，當時位於[臺北市](../Page/臺北市_\(州轄市\).md "wikilink")[表町二丁目七番地](../Page/表町_\(臺北市\).md "wikilink")（今[臺北市](../Page/臺北市.md "wikilink")[中正區](../Page/中正區_\(臺北市\).md "wikilink")[臺北車站對面](../Page/臺北車站.md "wikilink")）。建物在二次大戰[臺北大空襲中遭美軍炸毀](../Page/臺北大空襲.md "wikilink")，今已不存在，現址為[新光三越百貨臺北站前店及](../Page/新光人壽保險摩天大樓.md "wikilink")[K
Mall時尚購物中心](../Page/K_Mall.md "wikilink")（原大亞百貨、統一元氣館），約在[忠孝西路](../Page/忠孝西路.md "wikilink")、[館前路](../Page/館前路.md "wikilink")、[許昌街](../Page/許昌街.md "wikilink")、[南陽街所圍街區內](../Page/南陽街.md "wikilink")，大門入口在今館前路上。

在第二代[臺南車站二樓的](../Page/臺南車站.md "wikilink")[臺南鐵道飯店開業之後](../Page/臺南鐵路飯店.md "wikilink")，改稱**臺北鐵道飯店**。

## 簡介

該飯店於1907年夏天開始動工，1908年（[明治](../Page/明治.md "wikilink")41年）10月落成，由[臺灣總督府遞信部直營](../Page/臺灣總督府.md "wikilink")，是台灣第一座、也是當時唯一的一座西式飯店。當時來臺訪問的日本皇族、財政界的大人物都下榻於此，第一位客人是來臺灣主持「臺灣縱貫鐵道全通式」的[閑院宮](../Page/閑院宮.md "wikilink")[載仁親王](../Page/閑院宮載仁親王.md "wikilink")。

臺灣鐵道飯店建築是由[松崎萬長等人所設計](../Page/松崎萬長.md "wikilink")，本體為紅磚砌成，為德式建築。內部有挑高大廳與吊燈等設施。裡面所有的配件，從餐廳的刀叉到房間的瓷製馬桶，都是英國的舶來品。飯店一共有30間房，住宿費依等級從27[日圓到](../Page/日圓.md "wikilink")3日圓不等，在當時是有錢人才能進駐的昂貴價格。

臺灣鐵道飯店在1945年5月31日的[臺北大空襲中被炸毀](../Page/臺北大空襲.md "wikilink")，戰後曾小規模修建復業，改稱「臺灣鐵路飯店」並繼續營業，招攬如臺灣環島十二天遊的休憩飯店，與[臺灣大飯店](../Page/圓山大飯店.md "wikilink")、臺北招待所、臺旅飯店、臺旅食堂、淡水沙崙海濱飯店、陽明山眾樂園飯店、[臺中鐵路飯店](../Page/臺中鐵路飯店.md "wikilink")、[臺南鐵路飯店](../Page/臺南鐵路飯店.md "wikilink")、[日月潭](../Page/日月潭.md "wikilink")[涵碧樓招待所齊名](../Page/涵碧樓.md "wikilink")，並排首名。惟50、60年代間即被「金華飯店」（後被[希爾頓大飯店併購重建](../Page/希爾頓酒店.md "wikilink")）擠落首名。

## 建築

臺灣鐵道飯店佔地有3069坪，不過建築物實際的面積是620坪，有三層樓高。一樓是大廳、吸煙室、讀書室等公共空間，戶外還設有餘興舞臺一座。

## 費用

依據日本鐵道省出版的《観光地と洋式ホテル》一書，1930年代的鐵道飯店費用如下：

| 房間類型     | 房間數 | 費用    |
| -------- | --- | ----- |
| 一人房（無浴室） | 3   | 3－7圓  |
| 一人房（附浴室） | 1   | 16圓   |
| 二人房（無浴室） | 25  | 8－11圓 |
| 二人房（附浴室） | 1   | 27圓   |
|          |     |       |

餐費方面，早餐是1圓50錢，午餐2圓50錢，晚餐3圓，茶菓40錢。

**臺語諺文**是一套利用[諺文來表記](../Page/諺文.md "wikilink")[臺語語音的](../Page/臺語.md "wikilink")[拼音文字系統](../Page/拼音文字.md "wikilink")，最早由[許曹德所創](../Page/許曹德.md "wikilink")，並曾於1987年在[台灣文化促進會中推行](../Page/台灣文化促進會.md "wikilink")，後來[洪惟仁也表示贊同](../Page/洪惟仁.md "wikilink")。\[1\]\[2\]

## 差異

|        | [臺語](../Page/臺灣話.md "wikilink")                                               | [韓語](../Page/韓國語.md "wikilink")      |
| ------ | ----------------------------------------------------------------------------- | ------------------------------------ |
| ㅃ/ㄲ/ㅉ  | 濁音（、、或）                                                                       | 緊音（、、）                               |
| ᄌ/ᄎ/ᄍ  | [齒齦](../Page/齒齦音.md "wikilink")、[齦齶音](../Page/齦齶音.md "wikilink")              | 僅齦齶音                                 |
| ㄸ      | 聲母                                                                            | 聲母                                   |
| ㅎ      | 韻尾表示[聲門塞音](../Page/聲門塞音.md "wikilink")（）                                      | 韻尾                                   |
| ㅅ      | 韻尾表示元音[鼻化](../Page/鼻化.md "wikilink")                                          | 韻尾                                   |
| ㅆ      | 不作聲母，韻尾表示元音[鼻化並接](../Page/鼻化.md "wikilink")[聲門塞音](../Page/聲門塞音.md "wikilink") | 聲母，韻尾                                |
| ᅥ      | 韻母（南部腔為）                                                                      | 韻母                                   |
| ᅩ      | 韻母                                                                            | 韻母                                   |
| ㅐ      | 韻母                                                                            | 韻母                                   |
| 알      | 韻母                                                                            | 韻母ㅏ + 韻尾ㄹ                            |
| ㅡ      | 不發音，表示：음、응                                                                    | 韻母                                   |
| ㅢ      | 不發音，表示韻化輔音的入聲：읨、읭                                                             | 韻母或或                                 |
| i, u介音 | 以元音、發音                                                                        | 以輔音[近音](../Page/近音.md "wikilink")、發音 |

  - 此外臺語諺文只針對臺語優勢腔做設計，並無地方腔的（台羅\[er\]）、（台羅\[ir\]）、（台羅\[ee\]）等設計。

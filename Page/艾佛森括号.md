在[数学中](../Page/数学.md "wikilink")，以[Kenneth E.
Iverson命名的](../Page/Kenneth_E._Iverson.md "wikilink")“**艾佛森括号**”，是一种用方括号记号，如果方括号内的条件满足则为1，不满足则为0.
更确切地讲，

\[[P] = \begin{cases} 1 & \text{If } P \text{ is true;} \\ 0 & \text{Otherwise.} \end{cases}\]
此处  是一个可真可假的[命题](../Page/命题.md "wikilink")。该记号由[Kenneth E.
Iverson在他的编程语言](../Page/Kenneth_E._Iverson.md "wikilink")[APL中引进](../Page/APL.md "wikilink")\[1\]，而特别使用方括号则是由[高德纳倡导的](../Page/高德纳.md "wikilink")，目的是避免含括号的表达式中的歧义。\[2\]

## 用途

艾弗森括号通过自然的映射\(\textbf{false}\mapsto 0;  \textbf{true}\mapsto1\)将[布尔值转化为整数值](../Page/布尔值.md "wikilink")，这就允许计数被表示为和式。例如，计数与小于n且正整数n互质的正整数的个数的[欧拉函数可以表示为](../Page/欧拉函数.md "wikilink")

  -
    \(\phi(n)=\sum_{i=1}^{n}[\gcd(i,n)=1],\qquad\text{for }n\in\mathbb N^+.\)

更一般地，此记号使得将和式和积分式中繁多的条件移入并成为被加（积）项的一个因子成为可能。这将减少累加记号周围的空间，更重要的是这允许运算更加代数化。例如，

  -
    \(\sum_{1\le i \le 10} i^2 = \sum_{i} i^2[1 \le i \le 10].\)

另一个例子是化简带特例的方程，例如公式

\[\sum_{1\le k\le n \atop \gcd(k,n)=1}\!\!k = \frac{1}{2}n\varphi(n)\]

对一切有效，但是右边有  对于  1}}。为了得到一个一切正整数n都成立的恒等式，可以利用艾弗森括号补充等式：

\[\sum_{1\le k\le n \atop \gcd(k,n)=1}\!\!k = \frac{1}{2}n(\varphi(n)+[n=1])\]

## 样例

[克罗内克函数](../Page/克罗内克函数.md "wikilink") : \(\delta_{ij} = [i=j].\)

[符号函数和](../Page/符号函数.md "wikilink")[单位阶跃函数](../Page/单位阶跃函数.md "wikilink")：

  -
    \(\sgn(x) = [x > 0] - [x < 0]\)
    \(H(x) = [x > 0].\)

最值与绝对值：

\[\max(x,y) = x[x>y]+y[x\leq y],\]

\[\min(x,y) = x[x\leq y]+y[x> y],\]

\[|x| = x[x\geq 0]-x[x<0].\]

上下[取整函数](../Page/取整函数.md "wikilink")：

  -
    \(\lfloor x \rfloor = \sum_{n=-\infty}^{\infty}n[n \le x < n+1]\)
    \(\lceil x \rceil = \sum_{n=-\infty}^{\infty}n[n-1 < x \le n].\)

可被表示为

\[\{x\} = x\cdot[x\geq 0].\]

实数的[三分律等价于下面的恒等式](../Page/三分律.md "wikilink")：

\[[a < b] + [a = b] + [a > b] = 1.\]

**蘇州碼子**又称**花碼**、**草碼**、**菁仔碼**、**番仔碼**、**[商碼](https://zh.wikipedia.org/wiki/商碼 "wikilink")**，是一種傳統在[中國民間流行的數字](https://zh.wikipedia.org/wiki/中国數字 "wikilink")，产生于[中国的](https://zh.wikipedia.org/wiki/中国 "wikilink")[苏州](https://zh.wikipedia.org/wiki/苏州 "wikilink")，起源自算籌。因為蘇州碼子容易學習，書寫便捷，一串數字能連筆寫出（阿拉伯數字就不能），而且寫法如同算珠，可以配合[算盤使用](https://zh.wikipedia.org/wiki/算盤 "wikilink")，所以曾經廣泛使用於商業中，在賬簿和發票等均有使用。現在這種數字在[中國大陸及](https://zh.wikipedia.org/wiki/中國大陸 "wikilink")[台灣幾近絕跡](https://zh.wikipedia.org/wiki/台灣 "wikilink")，但在[港](../Page/香港.md "wikilink")[澳地區的](../Page/澳門.md "wikilink")[街市](https://zh.wikipedia.org/wiki/街市 "wikilink")、舊式[茶餐廳](../Page/茶餐廳.md "wikilink")\[2\]及中藥房偶而仍然可見。在1990年代之前，香港[公共小型巴士](../Page/公共小型巴士.md "wikilink")普遍以此標示車資價錢，現時仍有少數路線仍以此標價\[3\]。香港小學數學課程中將之稱為**中國古代數字**\[4\]或**中國數碼**，並於小學五年級教授有關用法。**蘇州碼子**曾被用在[英國廣播公司電視連續劇](https://zh.wikipedia.org/wiki/英國廣播公司 "wikilink")《[新世紀福爾摩斯](https://zh.wikipedia.org/wiki/新世紀福爾摩斯 "wikilink")》第1季第2集《[銀行家之死](../Page/神秘符號.md "wikilink")》中的神秘符號，但是誤以豎寫表達（例如劇集中路邊小販的價錢牌用豎寫，實際只用橫寫）。

## 历史

蘇州碼子脫胎於中国文化历史上的[算籌](https://zh.wikipedia.org/wiki/算籌 "wikilink")，也是唯一还在被使用的算筹系统。花码由[南宋](../Page/南宋.md "wikilink")时期从算筹分化。同算筹一样，花码是一种进[十进位制](../Page/十进位制.md "wikilink")计数系统。与算筹不同的是算筹通常用在数学和工程上，花码通常用在商业领域里，主要用途是速记。[程大位](../Page/程大位.md "wikilink")的《-{[算法統宗](https://zh.wikipedia.org/wiki/算法統宗 "wikilink")}-》介紹了蘇州碼子，稱之為「**暗馬式**」和「**暗子馬數**」。\[5\]蘇州碼子從明代被蘇、杭一帶人們採用，在民間流行了數百年後，最終被阿拉伯數字取代其地位。\[6\]

## 符号

[蘇州碼子五及九的來源.svg](https://zh.wikipedia.org/wiki/File:蘇州碼子五及九的來源.svg "fig:蘇州碼子五及九的來源.svg") 花码使用特别符号来代表数字，符號是從南宋記錄[算籌的數碼演變而來](https://zh.wikipedia.org/wiki/算籌 "wikilink")。五至九的花碼，是零至四的花碼上各加一筆代表五的短豎（參考算籌橫式數碼），也就是短豎加「〇」是「〥」（五），短豎加「一」是「〦」（六），短豎加「二」是「〧」（七），短豎加「三」是「〨」（八），短豎加「〤」（四）是「〩」（九）。其中五和九的花碼「〥」「〩」變了形狀，以方便書寫。因為傳統畫圈是順時針方向從右到左，所以花碼「〥」於右上角收筆。\[7\]\[8\]注意數字六至八的花碼「〦」「〧」「〨」，書寫時第一筆應為短豎，不應寫作斜點（即使偶然有人如此寫），從而避免誤認為「二」「三」。一二三的花碼都是豎筆，為免這三個花碼相連時混淆，用橫筆「一二三」與豎筆花碼相間，第一個數字用豎筆，第二個用橫筆，第三個用豎筆，如此類推。比如，「二一」花碼作“〢一”，不用“〢〡”，避免與“〣”混淆，又如「四三一二」作「〤〣一〢」，「一九二二」作「〡〩〢二」。

## 记数法

苏州码子是一种[进位制](../Page/进位制.md "wikilink")记数系统，以位置表示大小。记数符号写成两行，首行记数值，第二行记[量级和](https://zh.wikipedia.org/wiki/量级 "wikilink")[计量单位](../Page/计量单位.md "wikilink")。例如：

|   |   |   |   |
| - | - | - | - |
| 〤 | 〇 | 〢 | 二 |
| 十 | 元 |   |   |

|   |   |
| - | - |
| 〨 | 〥 |
| 千 | 斤 |

  -
    左邊例子第一行记载的是数目的数值，“〤〇〢二”代表4022。第二行记载数目的[数量级](../Page/数量级.md "wikilink")和计量单位。此处数量级是十，代表第一行的第一位数字的数量级是十位。換言之這數字是“40.22元”，或“四十元二角二分”。

<!-- end list -->

  -
    右邊例子第一行「〨〥」代表85，第二行記數量級為千，表示第一行第一位數字的數量級是千位，因此這數字是「8500斤」。因為數量級已經表明，故此可以不寫末尾的「〇」，正如數目的讀法「八千五百」也省略末尾的零。

如果數量級是一，將數量級省去。如果數量級是萬，為求簡便，用筆畫簡單的俗字「-{万}-」表示，不用「-{萬}-」。\[9\]同樣地，當計量單位是「毫」時，用「毛」字代替，還有一些計量單位也有簡筆記法。例如：

|        |   |   |
| ------ | - | - |
| 〣      | ㄨ | 〢 |
| \-{万}- | 元 |   |

|   |   |
| - | - |
| 〩 | 〨 |
| 毛 |   |

  -
    左邊例子是「34200元」，右邊例子是「9.8毫」。

如果第一行只有一個數字，可以選擇將數量級記在右旁；當數量級為一時，可以將計量單位記在右旁。例如：

|   |   |
| - | - |
| 〤 | 十 |
| 元 |   |

|   |   |
| - | - |
| 〧 | 尺 |

  -
    左邊例子「40元」只有一個數字「〤」，將「十」字記在右旁（也可以在「〤」後添上「〇」，下行寫「十元」）。右邊例子「7尺」只有一個個位數字，將單位「尺」記在右旁。

雖然傳統中文是直行書寫，但花碼不用直寫，而是沿用了算籌的由左至右的橫向記法，與[算盤相同](https://zh.wikipedia.org/wiki/算盤 "wikilink")。\[10\]因此很多時候花碼寫得窄長而緊密。 此计数系统与[科学记数法](../Page/科学记数法.md "wikilink")很相近。

「10元」寫法，商用可簡作「十元」，一般寫法是上行寫「〡〇」，下行對應寫作「十元」， 另外，單位標示的「元、尺、百」，可以寫成「し、ス、る」。

#### 「杭州数字」

苏州码子由[Unicode](https://zh.wikipedia.org/wiki/Unicode "wikilink")3.0版本加入Unicode标准，当时被误解为[杭州风格](https://zh.wikipedia.org/wiki/杭州 "wikilink")。[Unicode](https://zh.wikipedia.org/wiki/Unicode "wikilink")4.0版本的勘误表认出了此误解\[11\]：

Unicode标准文献中所有关于的“杭州”错误都已经被修正成“苏州”，但是为了稳定，Unicode不允许对字符名称的改变，所以字符名称中的“杭州数字”也就将错就错了。\[12\]字符名称被有些软件作为[唯一标识符](https://zh.wikipedia.org/wiki/唯一标识符 "wikilink")，修改会造成[向后兼容问题](https://zh.wikipedia.org/wiki/向后兼容 "wikilink")。

关于这个误命名的应用之广泛，甚至在[英国广播公司](../Page/英国广播公司.md "wikilink")（BBC）于2010年出品的福尔摩斯改编剧[-{zh-cn:神探夏洛克;zh-tw:新世紀福爾摩斯;zh-hk:新福爾摩斯}-](../Page/神探夏洛克.md "wikilink")（Sherlock）中的《[-{zh-cn:神秘符号;zh-tw:銀行家之死;zh-hk:神秘符號}-](https://zh.wikipedia.org/wiki/神秘符号 "wikilink")》裡都得以体现：苏州码子大量地以“杭州”这个名称作为本剧第一季第二集的关键线索穿插其间。

### 相关标准

[中国大陆](../Page/中国大陆.md "wikilink")的 [GB 2312](../Page/GB_2312.md "wikilink") 标准未收录苏州码子相关的符号。[GB 18030](../Page/GB_18030.md "wikilink") 标准由于是一种 Unicode 变换格式，所以也收录了 Unicode 中与苏州码子相关的符号。

[台湾的](https://zh.wikipedia.org/wiki/台湾 "wikilink") [Big5](https://zh.wikipedia.org/wiki/Big5 "wikilink")、[CNS 11643](https://zh.wikipedia.org/wiki/CNS_11643 "wikilink") 收录了 1 到 9 的苏州码子，其中 1 到 3 为纵式。

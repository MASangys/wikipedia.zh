**蘇格蘭蓋爾語**（），或僅簡稱**蓋爾語**（；[IPA](../Page/國際音標.md "wikilink")：），屬於[凱爾特語族](https://zh.wikipedia.org/wiki/凱爾特語族 "wikilink")[蓋爾亞支](../Page/蓋爾亞支.md "wikilink")的一種語言。也被稱為**高地蓋爾語**或**高地蘇格蘭語**。

## 歷史

與蘇格蘭蓋爾語同屬蓋爾亞支的凱爾特語言尚有[愛爾蘭語](../Page/愛爾蘭語.md "wikilink")和[曼島語](https://zh.wikipedia.org/wiki/曼島語 "wikilink")，三者都是源自[古愛爾蘭語](../Page/古愛爾蘭語.md "wikilink")。蘇格蘭蓋爾語有時在[愛爾蘭或稱為](https://zh.wikipedia.org/wiki/愛爾蘭 "wikilink")「蘇格蘭語」（），然而这也容易造成混淆，因為實際上在蘇格蘭，所謂「蘇格蘭語」是用來指另一種屬於[日耳曼語族的](https://zh.wikipedia.org/wiki/日耳曼語族 "wikilink")「[低地蘇格蘭語](../Page/低地蘇格蘭語.md "wikilink")」。

蘇格蘭蓋爾語是[高地蘇格蘭人的傳統語言](https://zh.wikipedia.org/wiki/高地蘇格蘭人 "wikilink")；高地蘇格蘭人是現在主要住在[蘇格蘭高地](../Page/蘇格蘭高地.md "wikilink")的凱爾特族群，但在歷史上，蘇格蘭蓋爾語曾經是大半蘇格蘭地區的語言，約在5世紀左右由蘇格蘭人自[愛爾蘭帶到](https://zh.wikipedia.org/wiki/愛爾蘭 "wikilink")[大不列顛](../Page/大不列顛島.md "wikilink")（舊稱 Alba 或 Albion）的[加勒多尼亞](https://zh.wikipedia.org/wiki/加勒多尼亞 "wikilink")（約當涵蓋今天的整個蘇格蘭）；在這裡，蘇格蘭蓋爾語取代了皮克特人的[皮克特語](https://zh.wikipedia.org/wiki/皮克特語 "wikilink")。一直到15世紀末，[盎格魯人仍然以他們的語言稱蘇格蘭蓋爾語為](https://zh.wikipedia.org/wiki/盎格魯人 "wikilink")「蘇格蘭語」（[盎格魯語](https://zh.wikipedia.org/wiki/盎格魯語 "wikilink")：）。然而在16世紀初期，蘇格蘭蓋爾語卻變成盎格魯人口中的「愛爾蘭語」（低地蘇格蘭語/盎格魯語：），反而「蘇格蘭語」專門被用來指稱「[低地蘇格蘭語](../Page/低地蘇格蘭語.md "wikilink")」。無論如何，蘇格蘭蓋爾語已經在蘇格蘭文化佔了相當重的地位，對於那些蘇格蘭人，不管他們會不會說蘇格蘭蓋爾語，這個傳統語言都已經變成蘇格蘭王國的重要文化（雖然其他人可能仍只把它當成一個地方性的語言）。

蘇格蘭蓋爾語有豐富的口說（<span lang=gd>*beul aithris*</span>）與文學傳統；持續好幾百年，蘇格蘭蓋爾語都是蘇格蘭高地氏族（<span lang=en>the Highland clans</span>）承傳吟唱詩歌文化的語言，也因此蘇格蘭蓋爾語的保存對於蘇格蘭的政治相當重要的：蘇格蘭蓋爾語傳承了前封建時期的律法習俗，也就是 <span lang=gd>*tuatha*</span> 和 <span lang=gd>*duthchas*</span>：<span lang=gd>*tuatha*</span>代表「子民」，<span lang=gd>*duthchas*</span>代表「這塊土地的未來」；也因為靠著這樣母語的傳承，蘇格蘭高地人才能有一股堅持，反抗以低地為中心、說著盎格魯語的低地蘇格蘭政治，一直到之後蘇格蘭政府與[英格蘭的](https://zh.wikipedia.org/wiki/英格蘭 "wikilink")[倫敦](https://zh.wikipedia.org/wiki/倫敦 "wikilink")[西敏市](../Page/西敏市.md "wikilink")聯盟、合併，這樣的堅持都仍然還存在。縱使之後1746年的[卡洛登戰役與](https://zh.wikipedia.org/wiki/卡洛登戰役 "wikilink")「[高地驅趕](https://zh.wikipedia.org/wiki/高地驅趕 "wikilink")」對於蘇格蘭高地人殘害甚深，威脅到蘇格蘭蓋爾語，但前封建時期的態度仍然可以在19世紀末[高地土地聯盟的控訴與要求中端倪出](https://zh.wikipedia.org/wiki/高地土地聯盟 "wikilink")。

蘇格蘭蓋爾語更正確的稱法應為**高地蓋爾語**，以與現已滅亡的**低地蓋爾語**做區別。低地蓋爾語是[低地蘇格蘭人來到蘇格蘭低地之前](../Page/低地蘇格蘭語.md "wikilink")，存在於當地（蘇格蘭南部）的語言。但實際上來說，這兩者在地誌分佈上來說，並沒有很清楚的南北地理分界。

## 書寫

現代蘇格蘭蓋爾語共有18個[字母](../Page/字母.md "wikilink")：

以前的字母中沒有 *h* ，但現今則用 *h* 表示[輔音的](https://zh.wikipedia.org/wiki/輔音 "wikilink")[輕化](https://zh.wikipedia.org/wiki/辅音弱化 "wikilink")（以前是在輔音上頭加上一點來表示）。傳統上，蘇格蘭蓋爾語的字母部分以植物命名：

|          |           |          |
| -------- | --------- | -------- |
| A - 「榆樹」 | B - 「樺樹」  | C - 「榛樹」 |
| D - 「橡樹」 | E - 「白楊」  | F - 「赤楊」 |
| S - 「柳樹」 | T - 「金雀花」 | U - 「紫杉」 |

例如 A、B、C、D 分別是 「榆樹」、「樺樹」、「榛樹」、「橡樹」等等，但現在已經不依循。

[PartickPartaig.jpg](https://zh.wikipedia.org/wiki/File:PartickPartaig.jpg "fig:PartickPartaig.jpg")[格拉斯哥](https://zh.wikipedia.org/wiki/格拉斯哥 "wikilink")[帕提克車站的雙語標示](https://zh.wikipedia.org/wiki/帕提克車站 "wikilink")：上方綠字為蘇格蘭蓋爾文「Partaig」\]\] 和[愛爾蘭語](../Page/愛爾蘭語.md "wikilink")相同，蘇格蘭蓋爾語的輔音隨其前後[元音](../Page/元音.md "wikilink")而有不同發音：兩語都將元音分為'''寬'''元音（）與'''窄'''元音（），寬元音如 *a*、*o*、*u*，窄元音如 *e* 和 *i*。而輔音又依其前後元音的寬窄有相應發音變化。規則是：「窄對窄，寬對寬」；當輔音前後皆為窄元音時，該輔音變為窄輔音（[顎音化](../Page/顎音化.md "wikilink")）；當輔音前後皆為寬元音時，該輔音變為寬輔音（[軟齶音化](https://zh.wikipedia.org/wiki/軟齶音化 "wikilink")）。

因此依照前後元音的寬窄，*t* 就同時有寬與窄的不同，例如   裡的 *t* 前後皆為窄元音，其發音即為窄音 *t*；而   裡的 *t* 前後皆為寬元音，其發音即為寬音 *t*。

也因為這個規則，故每個輔音前後的元音的寬窄要一致，例如複數字尾通常為 ，如 「鞋」 的複數為  。但因為要符合輔音前後元音寬窄需一致的規則，故像 「屋子」 這個字的複數即為  ：兩者複數字尾  和  的發音是一樣的，後者  的 *e* 也只是為該原則。

然而[蘇格蘭考試委員會從](https://zh.wikipedia.org/wiki/蘇格蘭考試委員會 "wikilink")1976年開始，對拼法作了部分修正。例如現在[過去分詞的](https://zh.wikipedia.org/wiki/過去分詞 "wikilink")[字尾一律是](https://zh.wikipedia.org/wiki/字尾 "wikilink") （如 「被舉起的」，但以前作 ）。

另外由於在口語中，[非重音的元音常略](https://zh.wikipedia.org/wiki/重音 "wikilink")，因此非正式的書寫也常會省掉，如：

  -

    「我希望」 \>

雖然蘇格蘭蓋爾語的拼音規則複雜，但熟悉之後就會發現，它與英語的發音比起來實在規律太多，只要注意不要將他語的拼音習慣帶進蘇格蘭蓋爾語即可（例：人名   就常被英語慣用者念錯）。

## 發音

蘇格蘭蓋爾語字母的發音與其他歐語類似，例如像[愛爾蘭語](../Page/愛爾蘭語.md "wikilink")、[斯拉夫語族](https://zh.wikipedia.org/wiki/斯拉夫語族 "wikilink")、和[羅曼語族一樣](https://zh.wikipedia.org/wiki/羅曼語族 "wikilink")，它的子音 *t*、*d*，及大部分的 *n* 都是發[齒音](https://zh.wikipedia.org/wiki/齒音 "wikilink")，而非[英語或其他](https://zh.wikipedia.org/wiki/英語 "wikilink")[日耳曼語族的](https://zh.wikipedia.org/wiki/日耳曼語族 "wikilink")[齒齦音](https://zh.wikipedia.org/wiki/齒齦音 "wikilink")；跟[義大利語的](https://zh.wikipedia.org/wiki/義大利語 "wikilink") *r* 或[西班牙語的](https://zh.wikipedia.org/wiki/西班牙語 "wikilink") *rr* 相同，蘇格蘭蓋爾語的 *r* 發[齒齦顫音](https://zh.wikipedia.org/wiki/齒齦顫音 "wikilink")（除了輕化後的窄子音 *r* 以外）。

跟[冰島語](https://zh.wikipedia.org/wiki/冰島語 "wikilink")（以及[漢語拼音](https://zh.wikipedia.org/wiki/漢語拼音 "wikilink")）一樣，蘇格蘭蓋爾語的 *b*、*d*、*g* 全都是[無聲的非](https://zh.wikipedia.org/wiki/無聲 "wikilink")[送氣音](https://zh.wikipedia.org/wiki/送氣音 "wikilink")，而子音 *p*、*t*、*c* 則是無聲且強烈的送氣音（在字首為後送氣音，字尾為前送氣音）。較特別的是，在蘇格蘭蓋爾語裡，當字首的[重音節](https://zh.wikipedia.org/wiki/重音 "wikilink")[塞音](../Page/塞音.md "wikilink")之前為一個[鼻音子音時](https://zh.wikipedia.org/wiki/鼻音子音 "wikilink")，此重音節的塞音會變為有聲子音，如： 「房子」的發音為，但加上定冠詞變為  之後的發音是 ；又如 「菸草」的發音是。

此外輕化後的子音有特別的發音：*bh* 和 *mh* 發 ；*ch* 發  或 ；*dh*、*gh* 發  或 ；*fh* 永遠不發音，除了三個例外：、、和 ，分別發 。以上均在子音後加上 *h* 以表示輕化後的子音，但輕化的 *l*、*n* 和 *r* 並無特別的拼法。

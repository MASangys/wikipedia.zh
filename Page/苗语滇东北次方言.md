**苗语滇东北次方言**是[苗语](https://zh.wikipedia.org/wiki/苗语 "wikilink")[川黔滇方言的一支](../Page/苗语川黔滇方言.md "wikilink")，分布在[云南省东北的](../Page/云南省.md "wikilink")[昭通市](../Page/昭通市.md "wikilink")、[昆明市](../Page/昆明市.md "wikilink")、[曲靖市和](../Page/曲靖市.md "wikilink")[楚雄彝族自治州](../Page/楚雄彝族自治州.md "wikilink")，以及[贵州省西部的](../Page/贵州省.md "wikilink")[威宁彝族回族苗族自治县](../Page/威宁彝族回族苗族自治县.md "wikilink")、织金县、[赫章县](https://zh.wikipedia.org/wiki/赫章县 "wikilink")、[六盘水市和](../Page/六盘水市.md "wikilink")[紫云苗族布依族自治县](https://zh.wikipedia.org/wiki/紫云苗族布依族自治县 "wikilink")。说这种次方言的人口有30万。\[1\]以贵州[毕节市](../Page/毕节市.md "wikilink")[威宁县](https://zh.wikipedia.org/wiki/威宁县 "wikilink")[石门乡的](https://zh.wikipedia.org/wiki/石门乡_\(威宁县\) "wikilink")[石门坎苗话为标准音](../Page/石门坎_\(贵州\).md "wikilink")。

20世纪初，英国传教士伯格理为滇东北苗语创制了一套文字，即[老苗文](../Page/柏格理苗文.md "wikilink")。1950年代制定了《滇东北苗文改革方案（草案）》。这套新的苗文方案，虽然名义上是改革方案，实际上完全使用[拉丁字母](../Page/衍生拉丁字母.md "wikilink")，和老苗文没有明显的关系。1980年代，在老苗文的基础上又改造出一种“规范苗文”，在云南[昆明市](../Page/昆明市.md "wikilink")、[楚雄州](../Page/楚雄彝族自治州.md "wikilink")、[玉溪市](../Page/玉溪市.md "wikilink")、[曲靖市等地推行](../Page/曲靖市.md "wikilink")。目前三种苗文都在使用。

塞音和塞擦音的清音一般配合阴调（第1、3、5、7调），浊音配合阳调（第2、4、6、8调）。老苗文和新苗文都不区别不送气清音和浊音，但是新苗文完整的标出调类，所以一般不会混淆。但是也有一些例外情况。

浊擦音声母、较少见，新苗文里用相近的声母代替。声母用于现代汉语借词，对应于汉语拼音h ，新苗文也用h表示。

所有浊音声母在2、4II、8II调都带有浊送气音。

### 声调

滇东北次方言东部地区在8调的基础上，第4、6、8调部分或全部分裂成I、II两类，最多可以有11个调类。基本上名词、量词属于第I类，声调较高；其他词类属第二类，声调较低。新苗文和规范苗文里不区分这两类。贵州[威宁县石门坎和云南](https://zh.wikipedia.org/wiki/威宁县 "wikilink")[武定县](../Page/武定县.md "wikilink")[乌龙乡洒普山的声调如下](https://zh.wikipedia.org/wiki/乌龙乡 "wikilink")。

| 新苗文 | 调类               | 調值 |
| --- | ---------------- | -- |
| 石门坎 | 洒普山              |    |
| b   | 阴平               | 1  |
| x   | 阳平               | 2  |
| d   | 阴上               | 3  |
| l   | 阳上               | 4I |
| 4II | 12<sub>浊送气</sub> |    |
| t   | 阴去               | 5  |
| s   | 阳去               | 6I |
| 6II | 31               |    |
| k   | 阴入               | 7  |
| f   | 阳入               | 8I |
| 8II | 31<sub>浊送气</sub> |    |

滇东北次方言有连续变调现象。和川黔滇方言一些其他次方言类似，双音节词第二个音节在第一个音节是平声（1、2调）的情况发生变调。例如石门坎话在这种情况下，第二个音节2调变1调，3调变5调，5调变7调，4I、4II调变6I调，6I、6II、8I、8II调变4II调。另外，石门坎话在第一个音节是其他一些调（3至8调，非II类）的情况下，第二个音节在一定条件下由I类变为II类。\[2\]

石门坎11个声调有些（如第1调和第3调）虽然本调调值相同，但是连续变调的表现不同，因此是独立的声调。

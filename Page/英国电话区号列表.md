以下是**英国电话区号列表**。英国的电话号码分类计划由[英国政府下属部门](https://zh.wikipedia.org/wiki/英国政府 "wikilink")[通讯管理局](../Page/通讯管理局.md "wikilink")发布。本列表在官方标准的基础上加入已取消的号码及历史变迁。\[1\]\[2\]\[3\] 英国的[公共交换电话网](../Page/公共交换电话网.md "wikilink")使用[开放拨号系统](https://zh.wikipedia.org/wiki/电话号码分类计划#开放拨号系统 "wikilink")，电话区号并不完全按照[英国行政区划](../Page/英国行政区划.md "wikilink")来分配，例如[考文垂](../Page/考文垂.md "wikilink")的区号也在[瓦立克郡的部分区域使用](https://zh.wikipedia.org/wiki/瓦立克郡 "wikilink")，[曼彻斯特](../Page/曼彻斯特.md "wikilink")的区号仅包含其下属部分城镇。当某一区号的覆盖范围包括多个行政区时，该区号所覆盖的区域被称为“ELNS区域”（ELNS, Extended Linked Numbering Scheme）。

在英国国内拨打电话时，所有区号前都需加上长途冠码“0”（下文的区号已将长途冠码包含在内）。“0”最初是在用户直接拨号系统启用前用于呼叫[接线生的号码](https://zh.wikipedia.org/wiki/接线生 "wikilink")。从英国以外的国家拨号时则毋需加入长途冠码；在同一区号的地区内呼叫对方毋需加入长途冠码及区号，但有少数例外；使用[移动电话](../Page/移动电话.md "wikilink")拨号时则必须加入区号。

## 拨号冠码

| 冠码     | 电话服务种类                                                                                                                                                          |
| ------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **00** | 国际长途拨号                                                                                                                                                          |
| **01** | 按地区划分的电话号码                                                                                                                                                      |
| **02** | 按地区划分的电话号码（自2000年起使用）                                                                                                                                           |
| **03** | 不按地区划分的电话号码（收费与按地区划分的电话号码相同，自2007年起使用）                                                                                                                          |
| **04** | 保留                                                                                                                                                              |
| **05** | 公司、企业号码及[IP电话服务](https://zh.wikipedia.org/wiki/IP电话 "wikilink")（部分IP电话号码使用08冠码或直接使用按地区划分的电话号码），0500号码为[免费电话](https://zh.wikipedia.org/wiki/受话方付费电话 "wikilink")。 |
| **06** | 保留（可能将被用作个人号码服务（Personal Numbering Service），以取代“070”冠码）                                                                                                         |
| **07** | 移动电话号码（包括071xx、072xx、073xx、074xx、075xx、07624、077xx、078xx和079xx）；070号码为个人号码服务；076xx号码为[传呼机号码](https://zh.wikipedia.org/wiki/传呼机 "wikilink")。                     |
| **08** | 080号码为[免费电话](https://zh.wikipedia.org/wiki/受话方付费电话 "wikilink")；084及087为特别服务号码。                                                                                  |
| **09** | 优质通话服务（Premium Rate services, PRS）                                                                                                                              |

“2+8格式”指区号部分为2位数（不包含长途冠码0），去除冠码及区号的部分为8位数，即（020）xxxx xxxx格式（“020”的第一个“0”为长途冠码，不计入）。“5+4格式”指区号部分为5位数（不包含长途冠码0），去除冠码及区号的部分为4位数，即（0169 77）xxxx格式（“016977”的第一个“0”为长途冠码，不计入）。以此类推。\[81\]

在数学的一个抽象分支[范畴论](../Page/范畴论.md "wikilink")中，**范畴的等价**（）是两个范畴间的一个关系，在这种关系之下的范畴是“本质上一样的”。从数学的许多地方都有范畴等价的例子。建立一个等价涉及展示所考虑的数学结构间很强的相似性。在许多情形，这些结构表面或直觉上看并无关联，这样就使这种概念特别有用：它提供了在不同数学结构之间翻译的可能性，本质一语是指在翻译中保持的定理。

如果一个范畴等价于另一个范畴的[反范畴](https://zh.wikipedia.org/wiki/对偶_\(范畴论\) "wikilink")，则我们说“范畴的对偶性”，以及这两个范畴**对偶等价**。

范畴的等价由所涉范畴的一个函子组成，这个函子要求有一个“逆”函子。但与通常代数语境的[同构](../Page/同构.md "wikilink")不同，这个函子与它的逆不必是恒等映射，二只要每个对象[自然同构与在此复合函子下的像](https://zh.wikipedia.org/wiki/自然同构 "wikilink")。从而我们可以说这个函子是差一个同构下的逆。这实际上是[范畴的同构的概念](https://zh.wikipedia.org/wiki/范畴的同构 "wikilink")，其中要求逆函子的严格性质，但这比“等价”概念用得要少。

## 定义

正式定义为，给定两个范畴 **C** 与 **D**，一个范畴等价包括函子 *F* : *C* → *D*，函子 *G* : *D* →
*C*，以及两个自然同构 ε: *FG*→**I**<sub>*D*</sub> 与 η :
**I**<sub>*C*</sub>→*GF*。这里 *FG*: *D*→*D* 与 *GF*: *C*→*C* 分别为 *F* 与
*G* 的复合，而 **I**<sub>*C*</sub>: *C*→*C* 与 **I**<sub>*D*</sub>: *D*→*D*
分别为 *C* 与 *D* 的恒同函子。如果 *F* 与 *G* 是反变函子我们则说范畴的对偶。

通常我们不指出如上所有数据。例如，我们说范畴 *C* 与 *D* 是等价的（对偶等价）如果它们之间存在一个等价（对偶等价）。进一步，我们说
*F* 是一个范畴的等价如果如上逆函子 *G* 以及自然同构存在。但要注意 *F* 所具有的信息不足以构造 *G*
以及自然同构：存在许多不同的选择（见下面的例子）。

## 等价的刻画

可以证明函子 *F*: *C* → *D* 给出范畴的等价当且仅当它是：

  - [完全](https://zh.wikipedia.org/wiki/完全函子 "wikilink")，即给定 *C* 的任何两个对象
    *c*<sub>1</sub> 与 *c*<sub>2</sub>，由 *F* 给出的映射
    Hom<sub>*C*</sub>(*c*<sub>1</sub>,*c*<sub>2</sub>) →
    Hom<sub>*D*</sub>(*Fc*<sub>1</sub>,*Fc*<sub>2</sub>)
    是[满射](../Page/满射.md "wikilink")。
  - [忠实](https://zh.wikipedia.org/wiki/忠实函子 "wikilink")，即对 *C* 的任何两个对象
    *c*<sub>1</sub> 与 *c*<sub>2</sub>，由 *F* 给出的映射
    Hom<sub>*C*</sub>(*c*<sub>1</sub>,*c*<sub>2</sub>) →
    Hom<sub>*D*</sub>(*Fc*<sub>1</sub>,*Fc*<sub>2</sub>)
    是[单射](../Page/单射.md "wikilink")；
  - [本质满](https://zh.wikipedia.org/wiki/本质满函子 "wikilink")，即 *D* 中每个对象
    *d* 同构与某个形如 *Fc* 的对象。

这是一个相当有效和常用的判别法，因为不必真正构造出逆 *G* 以及 *FG*，*G*'
与恒同函子之间的自然同构。另一方面，尽管上面性质保证了范畴等价的存在性（假定背景集合论具有一个足够强的[选择公理](../Page/选择公理.md "wikilink")），缺少的数据没有完全确定，通常有许多选择。只要可能，给出缺少的构造是个好主意。正因为如此，具有这些性质的函子有时叫做**范畴的弱等价**（不幸地是这个术语与同伦论冲突）。

它与[伴随函子概念也有紧密联系](https://zh.wikipedia.org/wiki/伴随函子 "wikilink")。如下论断对函子
*F* : *C* → *D* 与 *G* : *D* → *C* 等价:

  - 存在从 *FG* 到 **I**<sub>*D*</sub> 与从 **I**<sub>*C*</sub> 到 *GF*
    的自然同构，分别叫做**余单位**与**单位**。
  - *F* 是 *G* 的一个左伴随且两个函子都完全且忠实。
  - *F* 是 *G* 的一个右伴随且两个函子都完全且忠实。

**茶在不同語言的稱呼**有所差異。在[茶的發源地](../Page/茶.md "wikilink")[中国](https://zh.wikipedia.org/wiki/中国 "wikilink")，各地[漢族語言对](https://zh.wikipedia.org/wiki/漢語方言 "wikilink")“茶”的發音不尽相同，以致茶传播至世界各地时的叫法也不同。

## 漢字名稱

| 名 | 普通話/國語發音   |
| - | ---------- |
| 茶 | chá/ㄔㄚˊ    |
| 荼 | shū/ㄊㄨˊ    |
| 茗 | míng／ㄇㄧㄥˊ  |
| 檟 | jiǎ／ㄐㄧㄚˇ   |
| 蔎 | shè／ㄕㄜˋ    |
| 荈 | chuǎn／ㄔㄨㄢˇ |

**茶**一字的早期寫法是**荼**，據說是作為藥品。後來有人去掉一橫造成“茶”字，分立出來表示今義，大約是在中唐\[1\]。茶从原産地[雲南地区往](https://zh.wikipedia.org/wiki/雲南省 "wikilink")[四川](../Page/四川省.md "wikilink")、江南以及[長江流域传播](https://zh.wikipedia.org/wiki/長江流域 "wikilink")，推定当时使用「荼」字并配以「dia／ㄉㄧㄚ」或「tia／ㄊㄧㄚ」的发音（[反切为](../Page/反切.md "wikilink")：澄麻切）。
據說早採的茶叫**「荼/茶」**，次摘的茶叫“**槚**”，第三次摘的叫“**蔎**”，晚採的叫“**茗**”，葉子已經老了的叫“**荈**”。一說蜀西南人稱茶為**蔎**。“茶”一字最早的書面記載是西漢王褒的《僮約》，提到了“武都買荼”、“烹荼盡具”。現今還在廣泛使用的，是“茶”、“茗”這兩個稱呼。近年來，又有“茗茶”一謂。\[2\]
[日語的](https://zh.wikipedia.org/wiki/日語 "wikilink")「茶」字發音包括：[吴音](https://zh.wikipedia.org/wiki/吴音 "wikilink")「」（ja）、[漢音](../Page/漢音.md "wikilink")「」（ta）、[唐音](../Page/唐音.md "wikilink")「」（sa）等。惯用音的「」（cha）为[院政時代的](https://zh.wikipedia.org/wiki/院政時代 "wikilink")《[色葉字類抄](https://zh.wikipedia.org/wiki/色葉字類抄 "wikilink")》所提及，可能在产生漢音和唐音之間的時期传入日本。[朝鮮語漢字音也存在](../Page/朝鮮語.md "wikilink")「」（da）和「」（cha）的发音，当表示植物、飲料的茶的場合则使用「」（cha）。[越南语的茶的字音包括](../Page/越南语.md "wikilink")[标准汉越音的](https://zh.wikipedia.org/wiki/汉越音 "wikilink")「」，和[漢喃音的](https://zh.wikipedia.org/wiki/喃音#汉喃音 "wikilink")「」、「」、「」、「」、「」。

## 漢字文化圈以外語言对「茶」的称呼

世界各種语言中具有「茶」的意思的[词语的起源](../Page/词语.md "wikilink")，主要为「ㄑㄧㄚ」（chya）音系統和「ㄊㄝ∕」（tê）音系統\[3\]。「ㄑㄧㄚ」音系統主要由陆路传播，[地理大发现以后的传播主要由](../Page/地理大发现.md "wikilink")[粤语读音](../Page/粤语.md "wikilink")「ㄑㄧㄚ」（ch'a或ts'a）和[闽南语的读音](https://zh.wikipedia.org/wiki/闽南语 "wikilink")「ㄊㄝ∕」（白話字：tê）而来。[葡萄牙主要从](../Page/葡萄牙.md "wikilink")[澳门进口茶](https://zh.wikipedia.org/wiki/澳门 "wikilink")，葡萄牙語的发音也为「ㄑㄧㄚ」（葡萄牙語的「chá」的發音為「ㄒㄧㄚ」）。而[荷兰则主要从福建厦门](../Page/荷兰.md "wikilink")、[台灣一帶进口茶](https://zh.wikipedia.org/wiki/台灣 "wikilink")，从荷兰输入茶的国家、如英國等，则多存在「ㄊㄝ∕」的发音，因此一般认为「ㄊㄝ∕」音多由海路传出，也造成了英文中「tea」這個詞的由來。在[汉字文化圈外](https://zh.wikipedia.org/wiki/汉字文化圈 "wikilink")，具有「茶」这个意思的单词的语言中，都不属于这两个系统的则十分罕见。

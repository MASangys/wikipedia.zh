**荃灣站**（）是[港鐵](../Page/港鐵.md "wikilink")[荃灣綫](../Page/荃灣綫.md "wikilink")的北面終點站，位於[新界](../Page/新界.md "wikilink")[荃灣區](../Page/荃灣區.md "wikilink")北部一帶，於1982年5月10日啟用。在[兩鐵合併](../Page/兩鐵合併.md "wikilink")前荃灣站是前[地鐵系統中最北端車站](../Page/香港地鐵.md "wikilink")。

## 車站設計

荃灣站是荃灣綫北面總站，位處[荃灣市中心北部](https://zh.wikipedia.org/wiki/荃灣市中心 "wikilink")，[青山公路](../Page/青山公路.md "wikilink")及[大河道](../Page/大河道.md "wikilink")之間，[西樓角路](../Page/西樓角路.md "wikilink")旁。車站旁邊是[港鐵荃灣車廠](../Page/港鐵荃灣車廠.md "wikilink")，上蓋是[港鐵公司住宅發展項目](https://zh.wikipedia.org/wiki/香港鐵路有限公司 "wikilink")[綠楊新邨](../Page/綠楊新邨.md "wikilink")。車站以紅色為主要色調。

在前地鐵公司網絡中，荃灣站的結構設計較為獨特，是少數的地面車站之一；加上車站直接設於西樓角路旁邊，使此站與[寶琳站](../Page/寶琳站.md "wikilink")、[欣澳站](../Page/欣澳站.md "wikilink")成為[兩鐵合併](../Page/兩鐵合併.md "wikilink")前可直接由月台走出車站的少數例子。此站另一獨特之處，是在於身為荃灣綫總站，卻使用了[對向式月台設計](https://zh.wikipedia.org/wiki/對向式月台 "wikilink")（原因是荃灣站根據原來規劃並非荃灣綫終站，詳見[\#歷史](https://zh.wikipedia.org/wiki/#歷史 "wikilink")），故車站的西面設有露天掉頭路軌，供列車前往對面月台接載乘客或返回荃灣車廠。

### 大堂

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Tsuen Wan Station 2019 03 part3.jpg | caption1 = 大堂（收費區外） | image2 = Tsuen Wan Station 2019 03 part4.jpg | caption2 = 大堂（收費區內） | image3 = Tsuen Wan Sta Footbridge.JPG | caption3 = 大堂外的行人通道 }}

大堂位於U1層，其中[大河道北高架橋在大堂中央的上方跨過此站](https://zh.wikipedia.org/wiki/大河道北 "wikilink")；其大堂東面的上方則設有[街市](../Page/傳統市場.md "wikilink")，後來街市納入[綠楊坊](../Page/綠楊坊.md "wikilink")範圍並改為[超級市場](../Page/超級市場.md "wikilink")。

大堂兩端的A、B出入口，可直接通往連接荃灣市中心各處的[荃灣行人天橋網絡](../Page/荃灣行人天橋網絡.md "wikilink")，方便乘客前往荃灣市中心各處。行人天橋網絡中就有直接連接大堂兩端的天橋，位於大堂外側；由於大堂中間被劃分為收費區，乘客需利用這條天橋往返大堂兩邊。在這條天橋的中間，又設有兩條樓梯連接位於車站上面通過的[大河道北行車天橋](https://zh.wikipedia.org/wiki/大河道北 "wikilink")，供乘客前往車路兩旁的[巴士站](../Page/巴士站.md "wikilink")轉乘巴士。為方便乘客前往目的地，天橋系統部分通道附有出入口編號，即A1－A4及B1－B3共7個出入口。

荃灣站大堂安裝了[空調](../Page/空氣調節.md "wikilink")，內裡提供不同類型的商店供乘客購物或進食，例如有[便利店](../Page/便利商店.md "wikilink")、[麵包糕餅店](https://zh.wikipedia.org/wiki/麵包糕餅店 "wikilink")、外賣[壽司店](https://zh.wikipedia.org/wiki/壽司 "wikilink")、[銀行](../Page/銀行.md "wikilink")及電力公司客戶服務中心等\[1\]。所有車站商店均設於非收費區。此外，荃灣站也有不少自助服務設施供乘客使用，包括[自動櫃員機](../Page/自動櫃員機.md "wikilink")和[自動售賣機](../Page/自動販賣機.md "wikilink")\[2\]。近B、C出入口大堂入閘機旁亦設有[香港郵政](../Page/香港郵政.md "wikilink")[郵箱](../Page/郵箱.md "wikilink")\[3\]\[4\]，方便乘客在車站內投寄[郵件](../Page/郵件.md "wikilink")。另外，大堂近A出入口的非付費區設有「[會員服務站](../Page/會員服務站.md "wikilink")」，供乘客享用港鐵友禮會的會員優惠\[5\]。

在[互聯網服務上](https://zh.wikipedia.org/wiki/互聯網 "wikilink")，大堂及月台均設有免費由[電訊盈科](../Page/電訊盈科.md "wikilink")提供的[Wi-Fi](../Page/Wi-Fi.md "wikilink")熱點\[6\]，乘客可在候車的時候透過任何配備Wi-Fi功能的電子器材存取互聯網。同時，港鐵於大堂中央的已付費區設有「iCentre」免費上網服務設施\[7\]，使乘客可在該處瀏覽網上資訊。

### 月台

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Tsuen Wan Station 2019 03 part2.jpg | caption1 = 1號月台（落客專用） }} 荃灣站月台層位於地面的G層。月台採用[側式月台](../Page/側式月台.md "wikilink")開放式設計，其車站佈局跟[港島綫](../Page/港島綫.md "wikilink")[杏花邨站](../Page/杏花邨站.md "wikilink")相似，荃灣站啟用早期月台沒有裝設空調，只有在牆身掛上「牛角[風扇](https://zh.wikipedia.org/wiki/風扇 "wikilink")」來調節月台溫度。及後於2011年港鐵月台翻新工程過後，加設通風系統。

由於此站為荃灣綫的終點站，1號月台為落客專用月台，因此列車在1號月台落客後，需要繼續前行離開車站，然後再反方向駛入2號月台接載乘客。由於此站接近[荃灣車廠](../Page/港鐵荃灣車廠.md "wikilink")，因此每當上下班繁忙時間之後，部分荃灣綫列車在此站1號月台落客完畢後，便會暫停載客，先前行離開車站，然後反方向駛往車站旁的荃灣車廠。兩個月台已於2011年裝設[月台閘門](../Page/月台閘門.md "wikilink")，與觀塘站自動閘門工程一起在2012年前完成，標誌著所有市區綫的地面車站均有自動閘門。

## 利用狀況

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Tsuen Wan Station Platform1 2019.jpg | caption1 = 月台通往大堂通道 }} 在荃灣綫啟用初期，荃灣站的位置較為荒蕪，缺乏發展，遠離當時位於[沙咀道](../Page/沙咀道.md "wikilink")一帶的[荃灣市中心](https://zh.wikipedia.org/wiki/荃灣市中心 "wikilink")，令車站乘客量頗低。然而，隨著越來越多市民搭乘地下鐵路，使荃灣站一帶繁盛起來，車站附近的[南豐中心](../Page/南豐中心.md "wikilink")、[綠楊新邨](../Page/綠楊新邨.md "wikilink")等多個商場及住宅相繼落成，使荃灣北部取代了荃灣中部傳統市中心的地位。此外，荃灣站乘客可使用[荃灣行人天橋網絡](../Page/荃灣行人天橋網絡.md "wikilink")前往荃灣市中心各地，而行人天橋網絡亦於2012年6月延長，進一步增加本站人流。

本站通車初期的[屯門](../Page/屯門.md "wikilink")、[元朗](../Page/元朗.md "wikilink")及[青衣新市鎮未有鐵路連接](https://zh.wikipedia.org/wiki/青衣新市鎮 "wikilink")，[九龍巴士](../Page/九龍巴士.md "wikilink")便開辦了大量巴士線連接新市鎮及荃灣綫車站，方便該區市民轉乘地下鐵路前往市區。其中為數不少的巴士路線便是以[荃灣站公共運輸交匯處](../Page/荃灣站公共運輸交匯處.md "wikilink")為終點站，由於新界西北通勤需求龐大，間接地把本站建立成轉乘巴士及地鐵的交通樞紐。直至[香港3號幹線](../Page/香港3號幹線.md "wikilink")、[西區海底隧道及](https://zh.wikipedia.org/wiki/西區海底隧道 "wikilink")[汀九橋](../Page/汀九橋.md "wikilink")相繼啟用後，促進多條直接來往新界西北及港九的點對點巴士路線的開辦，吸收了荃灣站轉乘巴士的客量，之後[西鐵在](https://zh.wikipedia.org/wiki/九廣西鐵 "wikilink")2003年通車，荃灣站的轉運作用已大幅下降，但因為新界西北的人口持續增長，現時仍有不少市民選擇使用此站前往大西北地區。

### 連接荃灣西站的行人天橋

荃灣站與[西鐵綫](../Page/西鐵綫.md "wikilink")[荃灣西站](../Page/荃灣西站.md "wikilink")並非正式的轉車站，乘客若在兩站間轉乘，無論使用[八達通](../Page/八達通.md "wikilink")還是單程車票，均需當作兩段車程處理，並繳納再入閘收費。雖然兩者並非轉乘車站，但是兩站之間有[專線小巴](https://zh.wikipedia.org/wiki/香港小巴 "wikilink")95K（八達通、[屯門－南昌全月通加強版或](https://zh.wikipedia.org/wiki/屯門－南昌全月通加強版 "wikilink")[屯門－紅磡全月通加強版可免費轉乘西鐵綫](https://zh.wikipedia.org/wiki/屯門－紅磡全月通加強版 "wikilink")）及301M（八達通轉乘西鐵綫優惠）連接。

至於步行全程來往兩站，原本大概需要30分鐘，不過隨著[荃灣行人天橋網絡](../Page/荃灣行人天橋網絡.md "wikilink")擴建計劃已經建成，步行時間已縮短至15至20分鐘。天橋網絡原本可將兩站直接以天橋接駁，惟因行人需要經過[荃灣運輸大樓才能抵達](https://zh.wikipedia.org/wiki/荃灣運輸大樓 "wikilink")[荃灣西站](../Page/荃灣西站.md "wikilink")，而運輸大樓已於2013年2月拆卸。擴建計劃則因為出現裂痕等安全問題，拖延至同年6月才完全開放，使乘客不能直接前往荃灣西站。\[8\]

現時使用荃灣行人天橋網絡系統前往荃灣西站，需要於[荃灣大會堂](../Page/荃灣大會堂.md "wikilink")或[荃灣廣場](../Page/荃灣廣場.md "wikilink")返回地面，再沿著海貴路或大河道行人通道才能到達荃灣西站A2或D出入口。

## 歷史

### 選址

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Mass Transportation Study 1967 zh.svg | caption1 = 地底集體運輸系統路綫圖1967年建議方案。車站曾選址沙咀道地底。 }} 1967年，顧問公司Freeman, Fox, and Partners於研究報告中建議荃灣、[大窩口兩站設於](../Page/大窩口站.md "wikilink")[沙咀道](../Page/沙咀道.md "wikilink")地底\[9\]。但因後來荃灣綫的車廠決定於[青山公路－荃灣段以北興建而將走綫北移](https://zh.wikipedia.org/wiki/青山公路－荃灣段 "wikilink")，兩站的位置因而改為現時位置。

### 荃灣站以西路段

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = 1970 MTR route map zh.png | caption1 = 地下鐵路「整個系統」路綫圖 （1970年建議方案） }} 1970年，顧問公司完成對興建地下鐵路系統的進一步研究，發表《集體運輸計劃總報告書》，具體提出建造鐵路的建議。研究報告建議興建的地下鐵路系統全長52.7公里，合共50個車站，並且分成3條主要行車綫：港九綫、港島綫及[東九龍綫](../Page/東九龍綫_\(1970年方案\).md "wikilink")，以及兩條港九綫支綫：荃灣支綫、觀塘支綫。根據原設計，荃灣站只是中途站，並打算在現今荃灣站西面[荃景圍](../Page/荃景圍.md "wikilink")，近[荃景花園](../Page/荃景花園.md "wikilink")一帶山中興建荃灣西站（與現時[西鐵綫](../Page/西鐵綫.md "wikilink")[荃灣西站](../Page/荃灣西站.md "wikilink")沒有關連，位置亦不相同）作為終點站；但因收地及人口不足問題，一直無法興建。亦因荃灣站當初定位為中途站的關係，使用了[對向式月台設計](../Page/側式月台.md "wikilink")。雖然該站最終未能落成，但興建荃灣綫的時候，仍有於荃景花園後方鋪設路軌，惟這段掉頭路軌的最遠處基本上不會有列車使用。

後來在地鐵的[西部走廊鐵路](../Page/西部走廊鐵路.md "wikilink")設計方案中，亦包括延長荃灣綫至荃景圍一帶作為終點站，並與[東涌綫](../Page/東涌綫.md "wikilink")新界西北分支連接為轉乘站，途中增設荃景花園站，但因後來香港政府決定改由[九廣鐵路公司](../Page/九廣鐵路公司.md "wikilink")興建西部走廊（即今西鐵綫），並在新填海地興建新的[荃灣西站](../Page/荃灣西站.md "wikilink")，荃灣綫延長計畫亦隨之取消。

### 遷移三棟屋村

在荃灣站建設期間，[荃錦公路](../Page/荃錦公路.md "wikilink")需要改道，而[三棟屋村](../Page/三棟屋村.md "wikilink")亦需要搬遷，原址在工程完成後則改建為[三棟屋博物館](../Page/三棟屋博物館.md "wikilink")。另外，車站所在地原有不少墓地存在，後因興建車站而搬遷至荃灣川龍附近的響石，成為了現在的響石墳場。

### 翻新大堂 增設計特色

1992年，整個大堂翻新後以中國傳統建築特色作為主題。車站大堂商舖群被做到一個四合院，採用黃色牆和綠色屋頂作為設計。在車站大堂其他地方，也可以看到一個個較粗大很多的圓柱躉同樣被鋪上不同顏色的紙皮石，砌出飛龍圖案。月台上的紅色圓柱則較幼小，與北京故宮太和殿的建築柱躉雷同。

但以上設計特色已隨著車站在2000年代初被多次翻新而消失。

### 變更及增設D及E出口

2004年7月後，當時地鐵公司調動大堂商店以及C出入口的位置。現時通往綠楊新邨的C出入口位置原為[恆生銀行](https://zh.wikipedia.org/wiki/恆生銀行 "wikilink")，舊有C出入口通道位置設於現時自動櫃員機的位置。而大堂亦增設升降機往2號月台。可供乘客直接由落客月台出站至西樓角路路面的D及E出入口，亦於2005年11月13日啟用。

### 月台閘門

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = MTR Tsuen Wan Station platform.JPG | caption1 = 未安裝月台閘門的2號月台（2008年） }} 此站啟用時未設有[月台閘門](../Page/月台閘門.md "wikilink")。港鐵公司於2008年1月決定於前[地鐵網絡的所有地面及架空車站加建約高](../Page/香港地鐵.md "wikilink")1.5米的月台閘門，以及改善現有通風設計，以解決乘客墜軌的問題。為減輕對日常列車服務之影響，有關工程在平日深夜終止服務的時間進行。此站與[觀塘綫](../Page/觀塘綫.md "wikilink")[觀塘站](../Page/觀塘站.md "wikilink")屬於第三期月台閘門加裝工程，已於2011年10月完成。

### 翻新月台

除了加裝月台閘門外，港鐵公司亦於前地鐵網綫的所有地面及架空車站進行了月台翻新工程，其中此站的工程已於2011年起展開。工程包括安裝假天花，改善照明，並更換了地板。至於原本為圓柱形的月台支柱，則於四周圍上白色及紅色人造纖維板，使支柱變成四方狀。

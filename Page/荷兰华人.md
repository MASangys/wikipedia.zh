**荷兰华人**（）是指居住在[荷兰](../Page/荷兰.md "wikilink")的华人，不管其国籍为何。而国籍为荷兰的华人则称为**华裔荷兰人**。截至2012年，根據官方統計數據顯示，有80198人來自[中華人民共和國或](https://zh.wikipedia.org/wiki/中華人民共和國 "wikilink")[中華民國](../Page/中華民國.md "wikilink")（[臺灣](../Page/臺灣.md "wikilink")），或者父母一方來自[大中華地區](https://zh.wikipedia.org/wiki/大中華地區 "wikilink")\[1\]。然而，這些統計數字並沒有包括到其他移民至[荷蘭的](https://zh.wikipedia.org/wiki/荷蘭 "wikilink")[海外華人](../Page/海外華人.md "wikilink")，因為最早的華人移民除了來自中國外，還包括來自其他海外華人社群，如荷蘭前殖民地[印度尼西亞的華人](../Page/印度尼西亞華人.md "wikilink")。 [2015_0221_CNY_celebration_The_Hague.jpg](https://zh.wikipedia.org/wiki/File:2015_0221_CNY_celebration_The_Hague.jpg "fig:2015_0221_CNY_celebration_The_Hague.jpg")慶祝[農曆新年](https://zh.wikipedia.org/wiki/農曆新年 "wikilink")\]\]

## 移民歷史

早期華人移民到荷蘭主要來自兩個來源：第一次世界大戰後開始抵達該國的商人，大部分來自[浙江](https://zh.wikipedia.org/wiki/浙江 "wikilink")[青田](https://zh.wikipedia.org/wiki/青田 "wikilink")，以及來自[英國華人社群的](https://zh.wikipedia.org/wiki/英國華人 "wikilink")[廣東籍船員](https://zh.wikipedia.org/wiki/廣東人 "wikilink")；在[大蕭條期間](https://zh.wikipedia.org/wiki/大蕭條 "wikilink")，許多船員被解僱，而成了商人。之後由於自願遷出和驅逐出境的因素，華人人數下降，至[二戰期間剩下的華人不到](https://zh.wikipedia.org/wiki/二戰 "wikilink")1000人。荷蘭的另一批早期華裔移民是學生，大部分來自前荷蘭殖民地[印度尼西亞而非中國](https://zh.wikipedia.org/wiki/印度尼西亞 "wikilink")，從1911年的20人中，人數逐年增加至二戰期間；在1957年，來自印度尼西亞的華人大約1400名，其中有1000名是學生。在1911年，來自印尼的華人學生成立了中華會，並與中國各種組織和歐洲各國政黨接觸。\[2\]大部分來自印度尼西亞的學生為[土生華人](../Page/峇峇娘惹.md "wikilink")，傾向使用而非母語漢語並在荷蘭中學接受教育。然而，隨著印尼與荷蘭的關係在1950年代末期和1960年代初期日益緊張，華人學生人數急遽下降。雖然來自印尼的華人學生人數下降，但由於[1965年印尼排華屠殺](https://zh.wikipedia.org/wiki/印尼大屠殺_\(1965-1966年\) "wikilink")，成千上萬的印尼華人被迫離開印尼。大多數回到中國，或者移民至美國及澳大利亞；而曾經在荷蘭受過教育的華人，優先以荷蘭為移民目的地，人數約有5000人，他們大都不諳中文，而以印尼語言作為母語，並將荷蘭語作為學術語言\[3\] ，1970至1980年代初，[香港](../Page/香港.md "wikilink")成為華人移民到荷蘭的重要的來源地，每年約有600至800人，至1980年代後期下降為300到400人。\[4\]中國大陸學生在1980年代將荷蘭當作最佳留學國家，一開始，這些是中國政府資助且由考試選拔的學生，並在[萊登大學等著名的荷蘭大學獲得入學資格](https://zh.wikipedia.org/wiki/萊登大學 "wikilink")。\[5\]然而，在1990年代，更多的私人資助的學生、拿獎學金的學生和短期交換學生開始到達。\[6\]在2002年，由中國大使館的數據顯示，荷蘭約有4000名中國學生。\[7\]

## 人口統計

截至2012年，根據[荷蘭中央統計局統計](https://zh.wikipedia.org/wiki/荷蘭中央統計局 "wikilink")：

| 澳門    | 83人（男性31人，女性52人）           |  |
| 臺灣    | 2,254人（男性830人，女性1,424人）    |  |

| 父母其中一方出生的國家/地區 | 人口                         |  |
| -------------- | -------------------------- |  |
| 中國大陸           | 17,564人（男性9,002人，女性8,562人） |  |
| 香港             | 8,440人（男性4,300人，女性4,140人）  |  |
| 澳門             | 36人（男性18人，女性18人）           |  |
| 臺灣             | 531人（男性273人，女性258人）        |  |

共計80198人。比1996年的人口增長了92.8％，這是統計數據的最早年份。然而，人口中的各個群體顯示出不同的增長趨勢。來自中國大陸人數比同期增長了152％，相比之下，來自香港的人數只有輕微的增長，完全是自然增長而非外來增長。荷蘭華人人口分布並不集中，一半以上從事的餐飲業，尤其是開中華餐館。[阿姆斯特丹唐人街](../Page/阿姆斯特丹唐人街.md "wikilink")是荷兰最大的唐人街，但純粹是一個商業區，而非住宅混合商業區，此外还有[鹿特丹唐人街](../Page/鹿特丹唐人街.md "wikilink")和[海牙唐人街](../Page/海牙唐人街.md "wikilink")。 荷蘭華人人口中，從前殖民地[蘇利南](../Page/蘇利南.md "wikilink")移居至荷蘭的[華裔](https://zh.wikipedia.org/wiki/華裔 "wikilink")[客家人](https://zh.wikipedia.org/wiki/客家人 "wikilink")，約佔荷蘭華人人口10％，荷蘭華裔除了來自大中華地區、印度尼西亞和蘇利南外，也有來自[新加坡](../Page/新加坡.md "wikilink")、[馬來西亞](https://zh.wikipedia.org/wiki/馬來西亞 "wikilink")、[越南](https://zh.wikipedia.org/wiki/越南 "wikilink")、[柬埔寨](../Page/柬埔寨.md "wikilink")、[老挝](../Page/老挝.md "wikilink")。

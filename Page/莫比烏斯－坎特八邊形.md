在[幾何學中](https://zh.wikipedia.org/wiki/幾何學 "wikilink")，**莫比烏斯－坎特八邊形**是一個複正多邊形，其位於\(\mathbb{C}^2\)[複](https://zh.wikipedia.org/wiki/複數_\(數學\) "wikilink")[希爾伯特平面中由八個頂點和八個三元稜組成](https://zh.wikipedia.org/wiki/希爾伯特空間 "wikilink")，是一個自身對偶的多邊形{{\#tag:ref|Coxeter, 1991,\[1\] p.30, 47}}。[考克斯特將其命名為莫比烏斯](https://zh.wikipedia.org/wiki/考克斯特 "wikilink")－坎特八邊形，用於共享結構，如。\[2\]

這種形狀由於1952年發現，其將此形狀根據其對稱性以3(24)3表示，考克斯特將這種對稱性計為<sub>3</sub>\[3\]<sub>3</sub>，其與24階的同構。\[3\]

## 性質

莫比烏斯－坎特八邊形是一種由8個頂點和8條稜所組成的幾何結構，其在[施萊夫利符號中可以用](https://zh.wikipedia.org/wiki/施萊夫利符號 "wikilink")<sub>3</sub>{3}<sub>3</sub>來表示、在考克斯特記號中可以用來表示。與一般的[八邊形不同](https://zh.wikipedia.org/wiki/八邊形 "wikilink")，莫比烏斯－坎特八邊形位於[複](https://zh.wikipedia.org/wiki/複數_\(數學\) "wikilink")[希爾伯特平面](https://zh.wikipedia.org/wiki/希爾伯特空間 "wikilink")，且構成這種形狀的稜每個稜階連接了三個頂點，稱為三元稜或三元邊（Trion）\[4\]，這種幾何結構在施萊夫利符號中可以用<sub>*3*</sub>{}來表示。{{\#tag:ref|Complex Regular Polytopes,\[5\] 11.1 *Regular complex polygons* p.103}}

### 頂點座標

莫比烏斯－坎特八邊形可以於\(\mathbb{C}^3\)空間中給出，其為：

|            |                          |                        |          |
| ---------- | ------------------------ | ---------------------- | -------- |
| (*ω*,−1,0) | (0,*ω*,−*ω*<sup>2</sup>) | (*ω*<sup>2</sup>,−1,0) | (−1,0,1) |
| (−*ω*,0,1) | (0,*ω*<sup>2</sup>,−*ω*) | (−*ω*<sup>2</sup>,0,1) | (1,−1,0) |

其中\(\omega = \tfrac{-1+i\sqrt3}{2}\)。

## 實空間的代表

在實空間中，莫比烏斯－坎特八邊形可以用[四維空間的](https://zh.wikipedia.org/wiki/四維空間 "wikilink")[正十六胞体](../Page/正十六胞体.md "wikilink")來代表，{{\#tag:ref|Shephard, G.C. 1952,\[7\] p.93|name="Shephard, G.C. 1952 p.93"}}其共用了相同的8個頂點。當莫比烏斯－坎特八邊形的8條三元邊被繪製為三條獨立的邊時，即可在當莫比烏斯－坎特八邊形中觀察到[正十六胞体](../Page/正十六胞体.md "wikilink")的24條邊。在下圖中這8個三角形被以每個個分成一組，分別塗上藍色和紅色。下圖中，B<sub>4</sub>投影在兩個顏色組之間以兩個擁有不同對稱性的方向進行投影。此外，所代表的實空間形狀也可以是一個β<sub>4</sub>的四維[正軸形](https://zh.wikipedia.org/wiki/正軸形 "wikilink")。\[8\]

## 背景

[1980年代](../Page/1980年代.md "wikilink")，中国开始实行[改革开放](../Page/改革开放.md "wikilink")。1984年至1988年，中国社会粮食消费量平均每年增加150亿公斤，而人均占有耕地仅为一亩半。耕地的产出率不高，面积还在逐年减少\[2\]。与此同时，1978年至1988年，中国[副食品中的鲜蛋消费增长](https://zh.wikipedia.org/wiki/副食品 "wikilink")170％、家禽290％、猪肉88％、水产54.3％\[3\]。副食品的供应明显不足，同时也无法靠主粮食弥补。 [“菜篮子工程”所指的菜篮子不仅仅包括蔬菜，而是指整个[副食品](https://zh.wikipedia.org/wiki/副食品 "wikilink")，更多的还是指肉、禽、蛋、奶|缩略图](https://zh.wikipedia.org/wiki/File:BikurimS.jpg "fig:“菜篮子工程”所指的菜篮子不仅仅包括蔬菜，而是指整个副食品，更多的还是指肉、禽、蛋、奶|缩略图") [菜市场的菜篮子](https://zh.wikipedia.org/wiki/File:Fruit_baskets.jpg "fig:菜市场的菜篮子")

## 时间线

### 第一阶段

  - 计划

1988年5月，[中华人民共和国农业部](../Page/中华人民共和国农业部.md "wikilink")向[国务院提出](../Page/中华人民共和国国务院.md "wikilink")“关于发展副食品生产保障城市供应的建议”（简称“菜篮子工程”）\[4\]。

8月份，该项目得到了[国家计委的批准](../Page/中华人民共和国国家计划委员会.md "wikilink")\[5\]。1989年1月开始正式在全国实施\[6\]。

| **计划** | **内容**                                                                                                                                                                                                                                                                                                                                                         | **备注** |
| ------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------ |
| 总体设想   | 通过发展生产，调整[副食品供给结构](https://zh.wikipedia.org/wiki/副食品 "wikilink")，引导消费，逐步实现[肉食品消费多样化](https://zh.wikipedia.org/wiki/肉 "wikilink")，增加节粮、高蛋白的[禽](https://zh.wikipedia.org/wiki/禽 "wikilink")、[兔](../Page/兔.md "wikilink")、[蛋](../Page/蛋.md "wikilink")、[奶](https://zh.wikipedia.org/wiki/奶 "wikilink")、[鱼](../Page/鱼.md "wikilink")的供给比重；在保证大路菜供应的基础上增加细菜供给\[7\]。 |        |
| 方法     | 运用系统工程的方法，在理顺副食品价格的基础上，改革生产流通体制，合理开发利用国土资源，调整副食品供给结构，推广实用技术，强化基础设施，逐步提高副食品供给水平\[8\]。                                                                                                                                                                                                                                                                           |        |
| 目标     | 目标到[1992年](../Page/1992年.md "wikilink")实现动物性食品总量在1987年的水平上增加1260万[吨](https://zh.wikipedia.org/wiki/吨 "wikilink")（肉类400万吨、蛋225万吨、奶240万吨、鱼260万吨\[9\]），使全国特别是大中城市的动物性食品供应有所改善，蔬菜实现均衡优质上市\[10\]。                                                                                                                                                                   |        |
| 基本原则   | 菜篮子大家提\[11\]。                                                                                                                                                                                                                                                                                                                                                  |        |
| 预算     | 预计总投资额142.45亿元\[12\]。                                                                                                                                                                                                                                                                                                                                          |        |

“菜篮子工程”所指的菜篮子的内容绝不仅仅是指[蔬菜](../Page/蔬菜.md "wikilink")，而是指整个副食品，更多的还是指肉、禽、蛋、奶\[13\]。

  - 完成情况

到1993年底，全国农副产品批发市场已达2080个，城乡集贸市场已达8.3万个，初步形成了全国大市场、大流通的新格局。1994年，“菜篮子工程”已处在由过去以生产基地建设为主转入生产基地与市场体系建设并举的新阶段。同年年底，全国肉类总产量达4499.3万吨，禽蛋1479万吨，水产品总产量达2146.4万吨，水果总产量达3499.1万吨，蔬菜面积达1.34亿亩。全国有27个省市初步建立了主要副食品的地方储备。重点解决了市场供应短缺问题\[14\]。

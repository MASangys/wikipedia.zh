**萨维奇定理**（）是[计算复杂性理论中的一个](../Page/计算复杂性理论.md "wikilink")[定理](../Page/定理.md "wikilink")，由[沃尔特·萨维奇](../Page/:en:Walter_Savitch.md "wikilink")（Walter
Savitch）于1970年证明。定理的结论为对于任何函数\(f(n)\)满足\(f(n)\geq \log n\)，下列关系成立：

\[\text{NSPACE}\left(f\left(n\right)\right) \subseteq \text{DSPACE}\left(\left(f\left(n\right)\right)^2\right).\]

亦即，如果一台[非确定型图灵机能够利用](../Page/非确定型图灵机.md "wikilink")\(f(n)\)空间解决某个问题，那么一台[确定型图灵机能够在至多](../Page/确定型图灵机.md "wikilink")\(f^2(n)\)空间解决相同的问题。尽管非确定性的引入能够在时间上带来指数的提升，但是，这种引入对于空间而言作用有限。

## 证明

萨维奇定理的证明是构造性的。证明过程为设计一个针对有向图连通性问题的算法（其它问题可以通过图灵机的格局图归约到此问题）。有向图连通问题可以简述为对于一个[有向图和给定的两个顶点](../Page/有向图.md "wikilink")*s*和*t*，是否存在从*s*到*t*的有向路径。对于*n*个顶点，存在一个算法在\(\mbox{O}\left((\log{n})^2\right)\)空间内解决这一问题。这一算法的基本思路是利用[递归解决一个更一般化的问题](../Page/递归.md "wikilink")：检查是否存在从*s*到*t*的一条至多包含*k*条边的有向路径，*k*是递归的输入参数。原始的有向图连通问题当\(k=n\)时与此问题等价。为了测试是否存在一条从*s*到*t*的长度为*k*的有向边，可以测试是否存在一条从*s*到*t*的以*u*为中点的有向边。如果存在，那么对从*s*到*u*和从*u*到*t*递归此算法。

用[伪代码表示如下](../Page/伪代码.md "wikilink")（[Python语法](../Page/Python.md "wikilink")）：

``` python
def k-edge-path(s,t,k):
    if k == 0:
        return s == t
    else if k == 1:
        return (s,t) in edges
    else for u in vertices:
        if k-edge-path(s,u,⌊k/2⌋) and k-edge-path(u,t,⌈k/2⌉):
            return true
    return false
```

这一递归过程的递归深度为\(O(\log n)\)层，每层需要\(\mbox{O}(\log{n})\)位来存储该层的函数参数和局部变量。因此，算法的总空间复杂度为\(\mbox{O}\left((\log{n})^2\right)\)。上述算法尽管是使用[高级语言进行描述](../Page/高级语言.md "wikilink")，然而，相同的算法使用[图灵机实现时所需要的空间上界在渐近意义下是等同的](../Page/图灵机.md "wikilink")。

由于有向图连通性问题为**[NL完全](../Page/NL完全.md "wikilink")**问题，因而任意**[NL](../Page/NL_\(复杂度\).md "wikilink")**中的语言都在复杂度类\(\text{DSPACE}\left(\left(\log n\right)^2\right)\)中（这一复杂度类也常常被写作[L](../Page/L_\(复杂度\).md "wikilink")<sup>2</sup>）.

## 推论

从萨维奇定理可以得到许多重要的推论：

  - [PSPACE](../Page/PSPACE.md "wikilink") =
    [NPSPACE](../Page/NPSPACE.md "wikilink")
      - 得出这一结论因为多项式函数的平方仍然是一个多项式。尽管对于空间，确定性类与非确定性类相等，但是一般认为，对于时间的确定性类[P和非确定性类](../Page/P_\(复杂度\).md "wikilink")[NP](../Page/NP_\(复杂度\).md "wikilink")，这种关系不成立，尽管这一假设尚是一个[悬而未决的问题](../Page/P/NP问题.md "wikilink")。
  - [NL](../Page/NL_\(复杂度\).md "wikilink") ⊆
    [L](../Page/L_\(复杂度\).md "wikilink")<sup>2</sup>
      - 直接规定定理结论中的\(f(n)= \log n\)即可。

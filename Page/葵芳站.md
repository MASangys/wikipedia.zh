**葵芳站**（）是[港鐵](../Page/港鐵.md "wikilink")[荃灣綫](../Page/荃灣綫.md "wikilink")的一個[架空車站](https://zh.wikipedia.org/wiki/架空車站 "wikilink")，位於[新界西](https://zh.wikipedia.org/wiki/新界西 "wikilink")[葵青區](../Page/葵青區.md "wikilink")，[中葵涌](../Page/中葵涌.md "wikilink")與[下葵涌](../Page/下葵涌.md "wikilink")之間的分界綫，但主要的是[下葵涌](../Page/下葵涌.md "wikilink")[醉酒灣](../Page/醉酒灣.md "wikilink")葵仁路旁，於1982年5月10日啟用。

## 結構

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Kwai Fong Station 2014 04.JPG | caption1 = 車站外觀 }} 葵芳站是一個架空車站。其與葵興站之間的一段的架空路軌設有混凝土隔音屏障，是首項地鐵在興建時已經規劃的減低噪音工程，減低列車嘈音對列車於葵芳邨中間通過時的影響以及防止葵芳邨居民將垃圾丟進軌道。

### 大堂

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Kwai Fong Station 2018 06 part4.jpg | caption1 = 大堂 }} 葵芳站大堂的非收費區是被收費區分隔，因此乘客不能由大堂北面（A、D出入口）直接走至大堂南面（B、C出入口）或E出入口大堂，需要途徑路面往返兩個大堂，此特點與[荃灣站](../Page/荃灣站.md "wikilink")相似。

葵芳站大堂現時有不同類型的車站商店，包括便利店、麵包糕餅店、外賣店、診所、藥房等，以及各樣自助服務，包括自動櫃員機和自動售賣機\[1\]。大堂內亦有「iCentre」免費上網服務設施供乘客瀏覽[萬維網的內容](https://zh.wikipedia.org/wiki/全球資訊網 "wikilink")，乘客亦可以設有[Wi-Fi](../Page/Wi-Fi.md "wikilink")功能的電子設備於「iCentre」範圍內享5節各15分鐘的免費無線寬頻服務\[2\]。此外，於非收費區近A和D出入口設有「-{會員服務站}-」，供乘客登記為港鐵友禮會會員、查閱獎分及換領獎賞\[3\]。

葵芳站曾經是港鐵數十年以來惟一車站大堂使用強力天花吊扇給大堂通風的一個車站，但在2018年，車站內天花吊扇不再開動，且於2019年左右，設於大堂兩個非付費區的多把天花吊扇被拆除

### 月台

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Kwai Fong Station 2018 06 part1.jpg | caption1 = 1號月台﹙往荃灣方向﹚ | image2 = Kwai Fong Station 2018 06 part5.jpg | caption2 = 2號月台﹙往中環方向﹚ }} 月台採用[側式月台](../Page/側式月台.md "wikilink")設計，兩個月台的候車位置被路軌分隔，並裝有月台閘門。乘客如果往返兩個月台，需要途徑大堂。根據車站啟用時的月台設計，在1號月台牆身較低位置，由於面向當時的葵芳[臨時房屋區](../Page/臨時房屋區.md "wikilink")（即現時[新都會廣場](../Page/新都會廣場.md "wikilink")及[葵涌廣場](../Page/葵涌廣場.md "wikilink")的位置），所以裝了全身落地玻璃板或設有幼鐵絲網欄的全身結構開口以便自然通風。但2號月台因為太靠近建築物（好爵中心），故此牆身沒有通風窗口或玻璃窗。兩個月台的牆身較高位置設有大型廣告燈箱，而每個燈箱以車站主色（綠色）的纖維板相間，纖維板上有車站名稱和月台號碼。後來在纖維板上方印上月台列車方向及目的地。而車站經歷多次月台翻新工程，現今月台的形像與車站啟用時大有不同。

葵芳站車站的整體佈局及設計跟葵興站非常相似。 [Kwai_Fong_Station_2018_06_part3.jpg](https://zh.wikipedia.org/wiki/File:Kwai_Fong_Station_2018_06_part3.jpg "fig:Kwai_Fong_Station_2018_06_part3.jpg") [Kwai_Fong_Station_2018_06_part6.jpg](https://zh.wikipedia.org/wiki/File:Kwai_Fong_Station_2018_06_part6.jpg "fig:Kwai_Fong_Station_2018_06_part6.jpg")

## 鄰接車站

葵芳站是荃灣綫的中途站，乘客可在此乘坐往中環或荃灣列車，往返[香港島](../Page/香港島.md "wikilink")、[九龍](../Page/九龍.md "wikilink")及[新界](../Page/新界.md "wikilink")等地。

## 利用狀況

葵芳站是葵青區[葵涌](../Page/葵涌.md "wikilink")的大型交通中轉站，亦有多條巴士線連接和新界西倒數旺地，車站又設有大型巴士總站，供乘客轉乘巴士前往新界西區；車站附近集合了葵涌工業區、大型商場 [新都會廣場](../Page/新都會廣場.md "wikilink")、[葵涌廣場](../Page/葵涌廣場.md "wikilink")、住宅樓宇及[葵青劇院](../Page/葵青劇院.md "wikilink")、[葵涌運動場](../Page/葵涌運動場.md "wikilink")等康樂設施，另外有不少就讀[香港專業教育學院青衣分校](../Page/香港專業教育學院青衣分校.md "wikilink")的學生，都是乘搭港鐵荃灣綫到此站轉乘88C小巴或香港專業教育學院青衣分校校巴上下課，因此此站成為區內最繁忙的港鐵車站之一。

## 接駁交通

## 历史

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = KwaiFongStation20070726.jpg | caption1 = 未安裝月台閘門的月台（2007年） | image2= KwaiFongStation 20060817.jpg | caption2 = 翻新前的大堂有特高樓底（2006年） }}

### 命名

葵芳站位於葵涌垃圾灣（即[醉酒灣](../Page/醉酒灣.md "wikilink")）堆填區，在1967年的地底集體運輸系統建議書中，此站命名為**垃圾灣**，在通車時據附近興建有政府廉租屋[葵芳邨](../Page/葵芳邨.md "wikilink")而命名為葵芳站。

### 與西鐵交匯的建議

現時[西鐵綫](../Page/西鐵綫.md "wikilink")（前稱九廣西鐵）在葵芳站南面100米處的葵福路地底路過，但不在葵青區設站。惟1990年代後期，[九廣鐵路公司](../Page/九廣鐵路公司.md "wikilink")在規劃西鐵時，曾有意見認為西鐵應在葵芳設站，與此站接駁。然而，九廣鐵路公司以興建技術因難及轉乘安排不便為由否決有關建議。\[4\]

### 增建E出入口

葵芳站1號月台以前沒有與往新都會廣場的行人天橋連接，當時往來新都會廣場的乘客需要經過大堂及出入口才可出入車站。後來因人流增多而決定興建新出入口接駁1號月台及行人天橋，於2006年5月24日下午4時啟用，並編定為E出入口。

### 月台閘門

現時葵芳站所有月台已完成裝上月台閘門，並已投入運作。月台閘門約高1.5米。整個計劃於平日深夜終止服務的時間加建。此站與[葵興站](../Page/葵興站.md "wikilink")及[觀塘綫](../Page/觀塘綫.md "wikilink")[九龍灣站](../Page/九龍灣站.md "wikilink")、[牛頭角站](../Page/牛頭角站.md "wikilink")屬於第二期進行月台閘門加裝工程，該站所有月台的月台閘門加裝工程經已於2011年8月完工。

### 翻新月台

早期的月台改善工程要牽涉在廣告燈箱的位置，原本月台設計是每個在月台的大型廣告燈箱與綠色站名纖維板梅花間竹地掛在牆身高位，1號月台共有34個廣告燈箱，2號月台共有36個廣告燈箱。後來於1990年代，基於工作安全理由，將部份在樓梯或扶手電梯之上的廣告燈箱封上綠色膠布，這些燈箱不再作廣告燈箱使用，這使1號月台可用廣告燈箱減至25個，2號月台減至24個燈箱。

與市區線很多非地下車站一樣，葵芳站月台由2011年起也開始展開翻新工程，當中包括安裝假天花、改善照明，亦翻新了深綠色站名纖維板，原「往荃灣/中環」的行車方向資訊則改為張貼在月台閘門之上。但由於假天花進一步阻擋大型廣告燈箱的視線，故此港鐵公司再大幅度減少可使用的月台廣告燈箱數目，使1號月台只剩下22個可用，2號月台剩下8個可用，這些可繼續使用的燈箱的位置都略為降低，而其他棄用的燈箱則以綠色膠布封掉。

## 社會事件

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Kwai Fong Station 2019 part2.jpg | caption1 = 車站玻璃外牆被嘖上字句 }}

### 反修例運動相關衝突及關閉車站事件

2019年8月11日晚上9時，警方將葵芳警署外的示威者驅趕至葵芳站後，大量防暴警察趕到車站入口處。期間警方在站內密閉空間施放[催淚彈](../Page/催淚彈.md "wikilink")和發射[橡膠子彈](../Page/橡膠子彈.md "wikilink")，濃煙攻陷整個大堂。現場留下橡膠子彈彈殼，有市民站內感到不適和刺鼻，亦有人被催淚彈擊中。而部分示威者走到樓上月台的列車離開和躲避。港鐵在晚上10時35分表示發生特別事故而關閉車站。\[5\]到凌晨發聲明指，由於當時站內有乘客和港鐵職員，有關事件及警方行動極有可能影響他們的安全，對此表示非常遺憾，港鐵已就此向警方表示非常關注，並促請警方於執法期間必須先顧及港鐵員工及乘客的安全。\[6\]

警方最初否認曾在站內施放催淚彈。\[7\]其後警察公共關係科高級警司江永祥在記者會稱現場警員認為葵芳站是「半開放式設計」，「並非處於地底」，而且當時情況「十分緊急」，故施放催淚煙。\[8\]

8月12日早上，有身穿黑衣的年輕人自發在早上上班繁忙時間，於葵芳站內外免費派發口罩，並呼籲乘客戴上，以免吸入殘餘化學物，甚至呼籲市民盡量不要使用葵芳站。而站內部分地磚沾有白色粉末，仍然有淡淡化學氣味，職員亦戴上口罩。\[9\]

到8月13日晚上，逾300名市民到葵芳站大堂聚集，要求港鐵譴責警方在站內放催淚彈以及徹底清潔通風系統。港鐵車務副總經理丁嘉俊到場聽取意見，表示當日已向警方反映對事件關注，而且車站會再進行清潔，如乘客懷疑受催淚彈影響，亦可以向港鐵索償。但市民仍然不滿他的回應，要求港鐵盡快呈交調查報告向公眾交代和立刻譴責警方。\[10\]\[11\]

8月22日晚上，再有過百名市民去該車站表示不滿，亦有人包围车站控制室，以强光照射当值职员。他們用膠紙遮擋閘口感應器，讓乘客「免費」進入，一名25歲示威者被一名18歲「外判保安」打傷，報案後警員於48分鐘後才到場\[12\]。翌日晚，港铁决定当晚9时暂时关闭葵芳站，荃湾线列车亦不会停该站，參與當晚「[香港之路](../Page/香港之路.md "wikilink")」的市民晚上到葵芳站再次包圍車站控制室，要求港鐵開站並高叫「我要搭地鐵」和「我要返屋企（我要回家）」\[13\]\[14\]。

8月25日當天舉行已獲發不反對通知書的「[荃葵青遊行](../Page/2019年8月25日香港反對逃犯條例修訂草案遊行.md "wikilink")」，港鐵指因應公眾活動，下午1時半起暫時關閉本站、[荃灣站](../Page/荃灣站.md "wikilink")與[西鐵綫](../Page/西鐵綫.md "wikilink")[荃灣西站](../Page/荃灣西站.md "wikilink")，荃灣綫列車以[大窩口站](../Page/大窩口站.md "wikilink")為終點站\[15\]。港鐵沒有安排接駁巴士接載受影響乘客。大批市民阻止港鐵關站，一度與警方爆發衝突，荃灣區議員[譚凱邦](../Page/譚凱邦.md "wikilink")呼籲港鐵及警方不要拉閘關站，譚的助理為其攝錄，但被警方拉進鐵閘內，閘外數十人高叫「放人」。民主黨議員[尹兆堅](../Page/尹兆堅.md "wikilink")要求港鐵開站讓群眾和平散去，否則遊行到達終點的群眾最近要走到大窩口站離開。\[16\]

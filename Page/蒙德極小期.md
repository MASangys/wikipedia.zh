**蒙德極小期**(、)大約是從1645年至1715年，當時的觀測者注意到[太陽黑子非常罕見](https://zh.wikipedia.org/wiki/太陽黑子 "wikilink")。太陽學家[愛德華·沃爾特·蒙德](../Page/愛德華·沃爾特·蒙德.md "wikilink")（1851-1928）在研究那段時期的記錄資料時，發現這段期間的太陽黑子非常稀少，因此以他的名字來命名這一段時期。以蒙德極小期內一段30年時間為例，天文學家只觀察到約50個太陽黑子，而相對於在平常的相同時段，可以觀察到40,000-50,000顆太陽黑子。

## 太陽黑子的觀察

蒙德極小期出現在1645年至1715年，相較於正常可以觀測到40,000-50,000顆太陽黑子，只觀察到50顆左右的太陽黑子。在1610年到1681年間，每10年內觀察到的太陽黑子數目如下：

| 十年期間 | 太陽黑子 |
| ---- | ---- |
| 1610 | 9    |
| 1620 | 6    |
| 1630 | 9    |
| 1640 | 2    |
| 1650 | 3    |
| 1660 | 1    |
| 1670 | 0    |
| 1680 | 1    |

在蒙德極小期的時段內，仍然有足夠多的黑子被看見，讓我們能從外推測11年週期的極大期分別是1674年、1684年、1695年1705年和1716年。

太陽黑子的活動集中在太陽的南半球，只有最後一個週期才有黑子出現在太陽的北半球。

依據[史波勒定律](https://zh.wikipedia.org/wiki/史波勒定律 "wikilink")，當週期開始時的太陽黑子會出現在較高的緯度；在極大期時太陽黑子的平均緯度是15°，然後平均值會繼續向低緯度移動至大約7°的位置。之後老週期的黑子開始消散，而新週期的黑子在高緯度區域開始出現。

## 小冰期

[
](https://zh.wikipedia.org/wiki/File:Maunder_Minimum_and_Little_Ice_Age.tif "fig: ")

蒙德極小期與[小冰期](../Page/小冰期.md "wikilink")的中間 － 最冷的一段時間 －
吻合，使當時的[歐洲和](https://zh.wikipedia.org/wiki/歐洲 "wikilink")[北美洲](../Page/北美洲.md "wikilink")，或許世界上大部分的地區，都處於嚴寒的冬季。黑子活動量的降低和寒冷的冬天之間是否有關聯，仍是持續受到爭辯的題目（參見[全球暖化](https://zh.wikipedia.org/wiki/全球暖化 "wikilink")）。

## 其他的觀測

[Carbon14_with_activity_labels.svg](https://zh.wikipedia.org/wiki/File:Carbon14_with_activity_labels.svg "fig:Carbon14_with_activity_labels.svg")
在蒙德極小期中減弱的太陽活動對抵達地球的[宇宙射線也有影響](https://zh.wikipedia.org/wiki/宇宙射線 "wikilink")。在[人工製品考古學中使用的](https://zh.wikipedia.org/wiki/考古學 "wikilink")[碳定年法就必須考慮](https://zh.wikipedia.org/wiki/碳定年法 "wikilink")[碳14在數量變化上的結果](https://zh.wikipedia.org/wiki/碳14 "wikilink")。

[Solar_Activity_Proxies.png](https://zh.wikipedia.org/wiki/File:Solar_Activity_Proxies.png "fig:Solar_Activity_Proxies.png")
太陽的活動也影響到[鈹-10的產生以及](../Page/鈹的同位素.md "wikilink")[宇宙性](https://zh.wikipedia.org/wiki/宇宙性 "wikilink")[同位素](../Page/同位素.md "wikilink")，對它們的研究可以作為太陽活動的替代者。

歷史上其他的太陽黑子極小期也直接或間接的從[冰核或](https://zh.wikipedia.org/wiki/冰核 "wikilink")[年輪的](https://zh.wikipedia.org/wiki/年輪 "wikilink")[碳-14](../Page/碳-14.md "wikilink")分析中被發現，這些週期有[史波勒極小期](https://zh.wikipedia.org/wiki/史波勒極小期 "wikilink")（1450-1540）和少被提及的[道爾頓極小期](../Page/道爾頓極小期.md "wikilink")（1790–1820）。看起來在過去的8,000年中有18個這樣的極小期，研究顯示太陽在這段期間有四分之一的時間是在極小期的狀態下。

一篇最近發表的論文，基於對[佛蘭斯帝德繪圖的分析](../Page/約翰·佛蘭斯蒂德.md "wikilink")，認為在極度小的蒙德極小期（1666-1700）時太陽的自轉速率曾減慢\[1\]。

在蒙德極小期[極光依然如常的被觀測到](https://zh.wikipedia.org/wiki/極光 "wikilink")，詳細的分析已被[Wilfried
Schröder和J](https://zh.wikipedia.org/wiki/Wilfried_Schröder "wikilink").
P. Legrand等人發表（1992年）\[2\] \[3\]。

以蒙德極小期為基礎的論文（Eddy, Legrand, Gleissberg, Schröder, Landsberg *et
al.*）已經在Wilfried Schröder(Ed)發行，標題為*Case studies on the Spörer,
Maunder and Dalton Minima*\[4\]。

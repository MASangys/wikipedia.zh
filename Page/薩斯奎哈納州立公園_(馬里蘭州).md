**薩斯奎哈納州立公園**（）是美國[馬里蘭州的州立公園](https://zh.wikipedia.org/wiki/馬里蘭州 "wikilink")，位在[薩斯奎哈納河](../Page/薩斯奎哈納河.md "wikilink")下游，馬里蘭州的[阿弗爾第格雷斯北方](https://zh.wikipedia.org/wiki/:en:Havre_de_Grace,_Maryland "wikilink")。公園主要區域在薩斯奎哈納河西岸，不過也管理河中島嶼與東岸土地。

公園範圍包括岩溪史蹟區（Rock Run Historical Area），其由岩溪磨坊（Rock Run Grist Mill）、重建之澤西收費亭（昔日收取通過[底帕席特港大橋之費用](https://zh.wikipedia.org/wiki/:en:Port_Deposit_Bridge "wikilink")）、[薩斯奎哈納與潮水運河之遺跡](https://zh.wikipedia.org/wiki/:en:Susquehanna_and_Tidewater_Canal "wikilink")，以及1804年岩溪大廈（1804 Rock Run Mansion，昔為磨坊主人 John Archer之住家）\[1\]\[2\] 公園裡還有由私人營運的[Steppingstone 博物館](https://zh.wikipedia.org/wiki/:en:Steppingstone_Museum "wikilink")，展示1880年到1920年之間農村的工藝品。

## 活動設施

公園提供在[薩斯奎哈納河](../Page/薩斯奎哈納河.md "wikilink")上行舟與釣魚，兩個露營區有69個營位，還有一個野餐區。公園有長的步道供健行、登山自行車、騎馬之用。

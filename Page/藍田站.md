**藍田站**是[港鐵](../Page/港鐵.md "wikilink")[觀塘綫](../Page/觀塘綫.md "wikilink")一個車站，位於[香港](../Page/香港.md "wikilink")[九龍](../Page/九龍.md "wikilink")[觀塘區](../Page/觀塘區.md "wikilink")[藍田](../Page/藍田_\(香港\).md "wikilink")[鯉魚門道](../Page/鯉魚門道.md "wikilink")以南-{[晒草灣](../Page/晒草灣.md "wikilink")}-的山坡，上蓋為[滙景花園](../Page/滙景花園.md "wikilink")第9至17座。藍田站於1986年6月7日動工，1989年8月6日未有隨[東區海底隧道](../Page/東區海底隧道.md "wikilink")鐵路部分通車，直至1989年10月1日正式啟用。

## 車站設計

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Lam Tin Station 2014 04.JPG | caption1 = 進入此站的架空路軌（圖中右方） }} 基於地勢，藍田站建於山坡之上，靠近觀塘一方路軌為架空，靠近油塘一方則建於地面，以月台所處樓層而論，本站屬於[架空車站](https://zh.wikipedia.org/wiki/架空車站 "wikilink")，但由於車站設計成無窗密封環境才使人誤會自己身處地底而已，不過可以在鯉魚門道觀察，看到車站是興建在山坡上，西北面為架空部分，東南面為地底部分，位於山丘之下。由於車站上蓋是[滙景花園](../Page/滙景花園.md "wikilink")的9至16座住宅大樓，故此車站月台和大堂的支柱比其它車站為粗，以支撐車站上方的物業。

由於設計計算錯誤，路軌旁的廣告燈箱比其他車站較高，與月台乘客的視角並非處於同一水平，而燈箱之間的距離也較大。此外，路軌與廣告燈箱之間亦留有較闊的空間作為逃生或供維修時使用，此站是第一個使用此設計的前地鐵車站，隨後前地鐵路線新建的車站和行車隧道均採用此設計。

### 大堂

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Lam Tin Station 2017 09 part3.jpg | caption1 = 大堂（收費區內） | image2 = Lam Tin Station 2019 02 part4.jpg | caption2 = 大堂（收費區外） }} 藍田站是港鐵、亦是前地鐵由1989年至2011年啟用超過20年以來，除了[恆生銀行外而未有其他類別商店的車站](https://zh.wikipedia.org/wiki/恆生銀行 "wikilink")，曾成為最少商店的車站。

隨港鐵獲得車站擁有權後，於2010年7月進行加建兩間商店工程。工程把近A出入口一邊的[廣告](../Page/廣告.md "wikilink")燈箱及車站啟用紀念牌扁搬至近D出入口出閘機一邊，並且把近A出入口一邊的車費表，向東移前至另一條柱，而[單程車票](../Page/港鐵單程票.md "wikilink")[售票機](https://zh.wikipedia.org/wiki/售票機 "wikilink")、[八達通](../Page/八達通.md "wikilink")查閱機及八達通增值機，向東稍為移前至近入閘機斜對面。另外，工程期間消防喉轆則臨時搬至售票機舊有的位置，完工後整合至加建商鋪範圍所預留的位置。所有設施於2010年10月中完成改動。

此外，藍田站也有不少自助服務設施供乘客使用，包括[自動櫃員機](../Page/自動櫃員機.md "wikilink")及[自動售賣機](../Page/自動販賣機.md "wikilink")\[1\]，以及「[會員服務站](../Page/會員服務站.md "wikilink")」機供乘客登記為港鐵友禮會會員、查閱獎分及換領獎賞\[2\]。

### 月台

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Lam Tin Station 2018 07 part2.jpg | caption1 = 2號月台（往黃埔方向） | image2 = Lam Tin Station 2018 06 part1.jpg | caption2 = 連接月台及大堂的扶手電梯 }} 藍田站設有2個月台，並以[島式月台](../Page/島式月台.md "wikilink")排列，並設有[月台幕門](../Page/月台幕門.md "wikilink")。由於此站與過海段同期興建，故此月台的焗漆板亦設有書法字，成為首個擁有書法字裝飾的觀塘綫車站（[油塘站](../Page/油塘站.md "wikilink")及[調景嶺站](../Page/調景嶺站.md "wikilink")為[將軍澳綫](../Page/將軍澳綫.md "wikilink")同期興建，故一般視為[將軍澳綫](../Page/將軍澳綫.md "wikilink")車站）。

在興建將軍澳綫之前，原本在觀塘綫的列車由藍田站開往[鰂魚涌站](../Page/鰂魚涌站.md "wikilink")方向時，會有一小段的露天路段才進入東區海底隧道，但是後來該露天路段因為將軍澳綫通車而改為備用路段，並命名為[藍田站後備行車隧道](../Page/藍田站後備行車隧道.md "wikilink")，可供工程車或車務調動使用。另外，當將軍澳綫未能提供過海服務時，觀塘綫列車可利用此隧道提供有限度的過海服務（如2013年12月16日，將軍澳綫電力故障導致全線暫停服務，觀塘綫部分班次通過此隧道以北角為終點站）。

[Lam_Tin_Station_2018_07_part3.jpg](https://zh.wikipedia.org/wiki/File:Lam_Tin_Station_2018_07_part3.jpg "fig:Lam_Tin_Station_2018_07_part3.jpg") [Lam_Tin_Station_2018_07_part4.jpg](https://zh.wikipedia.org/wiki/File:Lam_Tin_Station_2018_07_part4.jpg "fig:Lam_Tin_Station_2018_07_part4.jpg")

另外在[將軍澳綫](../Page/將軍澳綫.md "wikilink")通車後，由於觀塘綫由油塘站開往藍田站方向的路段是跨越備用行車隧道上方，加上要遷就觀塘綫和將軍澳綫在油塘站的走綫，導致設計為一急彎兼下坡路段。故列車由油塘站開往藍田站時當駛至路段一半時，會減速行駛以防止乘客跌倒，惟列車減速下坡前甚少有廣播提醒乘客緊握扶手，仍然偶有乘客因此而跌倒。

### 出入口

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Sceneway Plaza Level 3 Access.jpg | caption1 = [匯景廣場](../Page/匯景廣場.md "wikilink")3樓中庭。A出入口實際連接匯景廣場3樓中庭及4樓中庭，乘客需要穿越商場才能往來藍田站大堂及啟田道路面。 | image2 = Lam Tin Station Exit A 20151122.jpg | caption2 = A出入口 | image3 = Lam Tin Station 2019 02 part1.jpg | caption3 = B出入口 }} 藍田站設有5個出入口，主要連接[藍田](../Page/藍田_\(香港\).md "wikilink")[鯉魚門道](../Page/鯉魚門道.md "wikilink")及[滙景花園](../Page/滙景花園.md "wikilink")一帶。由於位於鯉魚門道以南[-{晒}-草灣山坡](../Page/晒草灣.md "wikilink")，與藍田山上啟田道的住宅區有一定距離，所以當時[地鐵公司決定與發展商興建行人通道連接藍田站大堂與A出口啟田道路面](https://zh.wikipedia.org/wiki/地鐵公司 "wikilink")，該行人通道先以密封天橋由藍田站大堂跨過鯉魚門道、再穿過[滙景廣場](../Page/滙景廣場.md "wikilink")3樓中庭及4樓中庭，然後沿兩條屬地鐵範圍的密封往上扶手電梯才可到達A出口啟田道路面的啟田大廈及[康田苑](../Page/康田苑.md "wikilink")一帶，此種需要取道商場通道方可進出車站設計為全港首創，但是，地鐵沒有興建任何升降機設施連接滙景廣場4樓中庭至啟田道路面。而滙景廣場3樓中庭的一條屬商場範圍的往下扶手電梯可到達位於鯉魚門道路面的[鯉安苑](../Page/鯉安苑.md "wikilink")及觀塘警署一帶。而藍田站D1出口[茜發道](../Page/茜發道.md "wikilink")路面設有蓋通道及天橋可到達位於[茶果嶺道](../Page/茶果嶺道.md "wikilink")路面的[麗港城](../Page/麗港城.md "wikilink")一帶。在各個出入口，均設有指示牌顯示出入口周邊的建築物及設施列表。

## 利用狀況

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Lam Tin Station 2018 06 part2.jpg | caption1 = 連接月台至大堂的升降機 }} 藍田站啟用初期，由於上蓋物業滙景花園仍未入伙，加上距離藍田山上的屋邨和屋苑較遠，故人流偏低。後來在通車一年後，隨著[麗港城](../Page/麗港城.md "wikilink")和[滙景花園](../Page/滙景花園.md "wikilink")入伙，人流逐漸上升。而藍田站在將軍澳綫通車前亦曾經是[將軍澳](../Page/將軍澳.md "wikilink")居民接駁地鐵的主要車站。同時大型日資[百貨公司](../Page/百貨公司.md "wikilink")[八佰伴](../Page/八佰伴.md "wikilink")曾在[滙景廣場](../Page/滙景廣場.md "wikilink")開業，人流隨即急增。儘管八佰伴已結束營業多時，而將軍澳居民亦因為[將軍澳綫](../Page/將軍澳綫.md "wikilink")通車而不再使用藍田站，但滙景廣場的人流仍然十分高，加上[藍田山上大量湧現的屋邨及屋苑居民使用](../Page/藍田_\(香港\).md "wikilink")，而且更是[觀塘](../Page/觀塘.md "wikilink")山上的[秀茂坪邨](../Page/秀茂坪邨.md "wikilink")、[寶達邨](../Page/寶達邨.md "wikilink")、[安達邨](../Page/安達邨.md "wikilink")、[安泰邨](../Page/安泰邨.md "wikilink")居民接駁港鐵的主要車站。因此現時藍田站的使用量仍然不俗。

## 歷史

[Mass_Transportation_Study_1967_zh.svg](https://zh.wikipedia.org/wiki/File:Mass_Transportation_Study_1967_zh.svg "fig:Mass_Transportation_Study_1967_zh.svg") [Lam_Tin_Station_Concourse.jpg](https://zh.wikipedia.org/wiki/File:Lam_Tin_Station_Concourse.jpg "fig:Lam_Tin_Station_Concourse.jpg")

### 命名及選址

在香港地鐵規劃「早期系統」時，離開觀塘站後下一站為**觀塘村站**，服務現時蓝田站周邊地區，車站選址鄰近鯉魚門道的空置校舍，早在1976年已被徵用。但隨着[前地鐵早期系統規模縮減](../Page/香港地鐵.md "wikilink")，上行總站只至觀塘。

直到1984年有[日本](../Page/日本.md "wikilink")財團建議興建行車鐵路兩用的[東區海底隧道](../Page/東區海底隧道.md "wikilink")，觀塘綫得以伸延至港島的[鰂魚涌](../Page/鰂魚涌站.md "wikilink")，走綫之間新設藍田站以服務[藍田一带居民](../Page/藍田_\(香港\).md "wikilink")，最終藍田站於1989年10月1日啓用。

### 車站擁有權

在1989年至2008年，由於地下鐵路公司（2000年改組為地鐵有限公司，即今[香港鐵路有限公司](https://zh.wikipedia.org/wiki/香港鐵路有限公司 "wikilink")）接受日本財團提出以[民間興建營運後轉移模式](https://zh.wikipedia.org/wiki/民間興建營運後轉移模式 "wikilink")（BOT，建造-營運-移交）形式建[東區海底隧道](../Page/東區海底隧道.md "wikilink")至觀塘路段，故通車後藍田站連上蓋項目[滙景花園](../Page/滙景花園.md "wikilink")擁有權屬於[新香港隧道有限公司](https://zh.wikipedia.org/wiki/新香港隧道有限公司 "wikilink")。根據當時的安排，地下鐵路公司雖然並未負擔東隧和此站的建築費用，但需每半年繳交一次隧道費和本站租金給東隧公司，至2008年鐵路隧道部分和本站專營權屆滿，移交政府為止，之後繼續由政府徵收此項租金。但因2000年地鐵有限公司上市，政府遂與東隧公司簽訂補充協議，東隧的鐵道隧道和本站擁有權於專營權屆滿後，以一千港元的象徵式代價，經由地政總署移交給地鐵公司（今稱[港鐵公司](https://zh.wikipedia.org/wiki/港鐵公司 "wikilink")），而非移交政府。\[3\]

### 啟用

藍田站於1989年8月9日啟用，當日英國[威爾斯王妃戴安娜在此站舉行開幕儀式](https://zh.wikipedia.org/wiki/戴安娜_\(威爾斯王妃\) "wikilink")，揭開車站啟用紀念牌匾。同日下午，車站月台路軌附近爆水喉，令兩邊路軌浸水，列車駛進站時都要慢駛，對很多乘客做成不便。結果在翌日（8月10日）地鐵方面再一次派單程票給乘客，前後總共派了三天，以作補償乘客的損失及再安排新的通車日。

直至同年10月1日，藍田站正式開放給予乘客出入，當日亦為觀塘綫首段（即[修正早期系統](../Page/修正早期系統.md "wikilink")北段）[石硤尾至](../Page/石硤尾站.md "wikilink")[觀塘通車](../Page/觀塘站.md "wikilink")10週年。

### 使用後備行車隧道

2013年12月16日中午12時40分，將軍澳綫調景嶺站架空電纜故障，油塘站至[寶琳站](../Page/寶琳站.md "wikilink")及[康城站服務癱瘓](../Page/康城站_\(香港\).md "wikilink")。港鐵抽調觀塘綫列車，以梅花間竹的方式提供有限度將軍澳綫過海服務。當每4分鐘、以[觀塘站](../Page/觀塘站.md "wikilink")作終點站的列車調頭往觀塘重新接載乘客時；每8分鐘一班的列車便會繼續駛往藍田站，並在接載乘客後使用此後備行車隧道直接駛往鰂魚涌站，以[北角站](../Page/北角站.md "wikilink")作終點站。

### 車站關閉

2019年8月24日上午10點半，港鐵宣佈因應當天[觀塘區遊行](../Page/2019年8月24日香港反對逃犯條例修訂草案遊行.md "wikilink")，當天中午12時正起，藍田站連同[九龍灣站](../Page/九龍灣站.md "wikilink")、[牛頭角站](../Page/牛頭角站.md "wikilink")和[觀塘站](../Page/觀塘站.md "wikilink")關閉，觀塘線[彩虹站](../Page/彩虹站.md "wikilink")至[調景嶺站](../Page/調景嶺站.md "wikilink")路線服務暫停，但沒有提供任何接駁巴士服務，直至晚上11時45分回復正常服務\[4\]。藍田站於正常開放時間關閉共約12小時，為首次非天氣因素關閉。

2019年10月1日晚上9點08分，港鐵宣佈藍田站關閉\[5\]。

### 車站破壞

2019年9月1日晚上，藍田站兩部售票機、一部增值機、一部查票機、兩部閘機及一部電視受到惡意破壞。

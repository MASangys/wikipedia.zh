## 方案

这套方案与《藏汉对照拉萨口语词典》的方案基本一致。区别主要有：后者不标鼻冠音，把写作â。

#### 聲母

聲母總共有36個，其中包括28個單輔音，7個複輔音，1個零聲母。

28個單輔音即是輔音中的28個音位。

7個複輔音為

**1.mb 2.nz 3.nd 4.nzh 5.nj 6.ngy 7.ngg**

### 元音

元音有13個

|   |    |    |
| - | -- | -- |
| i | ii |    |
| e | ee |    |
| a | aa | ae |
| u | uu | ue |
| o | oo | oe |
| â |    |    |

â有時不算作一個獨立的音位。

與《少数民族语地名汉语拼音字母音译转写法》比较

| - | -- | ---- |

| a | ar | ai/ä |
| u | ur | ü    |
| o | or | oi/ö |
| â |    |      |

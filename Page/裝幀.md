## 书籍结构

### 前页

前页，即正文前的书页，是书中的第一节，通常也是书中页数最少的部分。书页的页码通常采用小写罗马数字，每一页都计算在页码之内，但是空白页的编码不被打印。

前页通常只出现在丛书的第一册，虽然其他元素(例如目录，索引)可能会出现在每一册中。

下面的表的可能会对区分前页的不同类型有所帮助:

| 名称                                                            | Voice        | 目的                                                                                     |
| ------------------------------------------------------------- | ------------ | -------------------------------------------------------------------------------------- |
| [目录](https://zh.wikipedia.org/wiki/目录 "wikilink")             | 出版社          | 章节标题和子标题及他们对应页码的列表。包含所有的前面列举的前页项，以及正文和书后结文中的章节。显示的子标题的层数应该被限制以便控制目录不要过长，理想的长度应该是一页或两页。 |
| [序](https://zh.wikipedia.org/wiki/序 "wikilink")               | 除作者以外其他现实中的人 | 通常序言会给出一些作序人与书籍作者或书籍相关信息。新版作品的序通常解释新版与先前版在哪些方面有不同。                                     |
| [前言](https://zh.wikipedia.org/wiki/前言 "wikilink")             | 作者           |                                                                                        |
| [感言](https://zh.wikipedia.org/wiki/感言 "wikilink")             | 作者           |                                                                                        |
| [引言](https://zh.wikipedia.org/wiki/引言 "wikilink")             | 作者           | 描述写作的目的。                                                                               |
| 题词                                                            | 作者           |                                                                                        |
| [Prologue](https://zh.wikipedia.org/wiki/Prologue "wikilink") | 书中人物         |                                                                                        |

**西南石油大学**位于中国[四川省](../Page/四川省.md "wikilink")[成都市](../Page/成都市.md "wikilink")、[南充市](../Page/南充市.md "wikilink")，是中华人民共和国创建的第二所石油[本科院校](https://zh.wikipedia.org/wiki/本科 "wikilink")，是国家建设高水平大学公派研究生项目实施高校，是首批[雙一流高校](https://zh.wikipedia.org/wiki/雙一流 "wikilink")，入选“111计划”，“中西部高校基础能力建设工程”，是《国家中长期教育改革和发展规划纲要（2010-2020年）》提出的“特色重点学科项目”建设高校，是设有研究生院、国家大学科技园、国家重点实验室和联合国援建的“中国石油天然气油井完井技术中心”的省部共建大学。成都校区位于成都市新都区新都大道，南充校区位于南充市顺庆区油院路。\[1\]

## 概况

学校1978年开始招收硕士研究生，1986年获得博士学位授予权，1988年油气田开发工程（联合石油机械工程）成为首批国家重点学科，1989年油气藏地质及开发工程国家重点实验室批准建设，1991年设立博士后科研流动站。

## 历史沿革

| 年代        | 名称     | 隶属           |
| --------- | ------ | ------------ |
| 1958－1962 | 四川石油学院 | 四川省人民委员会     |
| 1962－1970 | 四川石油学院 | 中华人民共和国石油工业部 |
| 1970－1978 | 西南石油学院 | 四川省革命委员会     |
| 1978－1988 | 西南石油学院 | 中华人民共和国石油工业部 |
| 1988－1998 | 西南石油学院 | 中国石油天然气总公司   |
| 1998－2000 | 西南石油学院 | 中国石油天然气集团公司  |
| 2000－2005 | 西南石油学院 | 四川省人民政府      |
| 2005－现在   | 西南石油大学 | 四川省人民政府      |

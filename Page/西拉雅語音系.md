**西拉雅語音系**（）為[西拉雅語之](../Page/西拉雅語.md "wikilink")[音系](../Page/音位學.md "wikilink")。而其音系之[音位大都使用適當的](../Page/音位.md "wikilink")[Unicode符號來標示](https://zh.wikipedia.org/wiki/Unicode "wikilink")。在[台灣南島語的](https://zh.wikipedia.org/wiki/台灣南島語 "wikilink")[書寫系統的訂定上](https://zh.wikipedia.org/wiki/書寫系統 "wikilink")，元輔音之音系表原則上都是先「[發音部位](https://zh.wikipedia.org/wiki/發音部位 "wikilink")」(橫列)、後「[發音方法](https://zh.wikipedia.org/wiki/發音方法 "wikilink")」(縱列)、再考量「[清濁音](https://zh.wikipedia.org/wiki/清濁音 "wikilink")」，來訂定其音系之音位架構。\[1\]

## 語言差異

根據前中研院史語所副所長李壬癸博士與日本語言學家土田茲研究，西拉雅語、[大滿語及](https://zh.wikipedia.org/wiki/大滿語 "wikilink")[馬卡道語等南部平埔語言至少有四種音韻演變的差異及一種構詞上的差異](../Page/馬卡道語.md "wikilink")\[3\]\[4\]：

1.  西拉雅語和馬卡道語的 r 對應大滿語的 h 或消失，如西拉雅語的 *turu*（三）對應大滿語的 *tuhu*，或西拉雅語的
    *rima*（五）對應大滿語的 *hima*。
2.  西拉雅語和大滿語的 l 對應馬卡道語的 n、l 或 r，如西拉雅語的 *wali*（牙齒）對應馬卡道語的 *wari*。
3.  西拉雅語的 s 對應大滿語的 r 或 d，如西拉雅語的 *sa*（和）對應大滿語的 *ra* 或 *da*，或西拉雅語的
    *raos*（西邊）對應大滿語的 *raur*。
4.  西拉雅語在[元音之間的](../Page/元音.md "wikilink")[舌根音](https://zh.wikipedia.org/wiki/舌根音 "wikilink")
    k 和 g 在大滿語已脫落，如西拉雅語的 *ako-saij*（沒有）對應大滿語的 *au-saij*，或西拉雅語的
    *dagogh*（價錢）對應大滿語的 *daoh*。

構詞上，西拉雅語表示「未來」的動詞後綴為 -ali、-ili（新港社）或 -ati、-ili（卓猴社），而大滿語是
-ah（灣裡社、麻豆社），馬卡道語則是與西拉雅語同源的 -ani（下淡水社）。

整理如下：

|                | 西拉雅語     | 大滿語  | 馬卡道語      | 古南島語          |
| -------------- | -------- | ---- | --------- | ------------- |
| 音韻演變（1）        | r        | Ø\~h | r         | \< \*l        |
| 音韻演變（2）        | l        | l    | n         | \< \*N        |
| 音韻演變（3）        | s        | r, d | r, d      | \< \*D, \*d   |
| 音韻演變（4）        | \-k- -g- | Ø Ø  | \-k- ---- | \< \*k \< \*S |
| 構詞差異 （表「未來」詞綴） | \-ali    | \-ah | \-ani     |               |

南部三種平埔語言的主要差異

由各地語料上的差異規律，李壬癸發現麻豆社與灣裡社有相同的音韻與構詞變化，應同屬大滿語，而非西拉雅語。

李壬癸推論南部平原西拉雅語、大滿語及馬卡道語約在 3 千多年前就開始分化，只是17
世紀荷蘭人據臺後，採用臺南新港社西拉雅語作為通用語，造成各族群間語言逐漸同化。

在[数理逻辑和](../Page/数理逻辑.md "wikilink")[理论计算机科学中](https://zh.wikipedia.org/wiki/理论计算机科学 "wikilink")，一个[重写系统有](https://zh.wikipedia.org/wiki/重写系统 "wikilink")**规范化性质**，如果所有项都是**强规范化**的；就是说所有重写序列都最终终止于[规范形式的项](https://zh.wikipedia.org/wiki/规范形式 "wikilink")。

纯粹无类型 [lambda
演算不是强规范化的](https://zh.wikipedia.org/wiki/lambda_演算 "wikilink")。考虑项
\(\lambda x . x x x\)。它有如下重写规则: 对于任何项 *t*，

  -
    \((\mathbf{\lambda} x . x x x) t \rightarrow t t t\)

但是考虑在应用 \(\lambda x . x x x\) 于自身时所发生的:

|                                                      |                                                                                                                              |
| ---------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| \((\mathbf{\lambda} x . x x x) (\lambda x . x x x)\) | \(\rightarrow (\mathbf{\lambda} x . x x x) (\lambda x . x x x) (\lambda x . x x x)\)                                         |
|                                                      | \(\rightarrow (\mathbf{\lambda} x . x x x) (\lambda x . x x x) (\lambda x . x x x) (\lambda x . x x x)\)                     |
|                                                      | \(\rightarrow (\mathbf{\lambda} x . x x x) (\lambda x . x x x) (\lambda x . x x x) (\lambda x . x x x) (\lambda x . x x x)\) |
|                                                      | \(\ldots\,\)                                                                                                                 |

所以项 \((\lambda x . x x x) (\lambda x . x x x)\) 不是规范化的。

各种**有类型 lambda 演算**系统包括[简单类型 lambda
演算](https://zh.wikipedia.org/wiki/简单类型_lambda_演算 "wikilink")，[Jean-Yves
Girard](https://zh.wikipedia.org/wiki/Jean-Yves_Girard "wikilink")
的[系统F](../Page/系统F.md "wikilink")，和 [Thierry
Coquand](https://zh.wikipedia.org/wiki/Thierry_Coquand "wikilink")
的[构造演算都有规范化性质](../Page/构造演算.md "wikilink")。

带有规范化性质的 lambda
演算系统可以被看作带有所有程序都[终止性质的编程语言](https://zh.wikipedia.org/wiki/终止 "wikilink")。尽管这是一个非常有用的性质，它也有缺点:
带有规范化性质的编程语言不可能是[图灵完全的](https://zh.wikipedia.org/wiki/图灵完全 "wikilink")。这意味着有可计算函数不能在简单类型的
lambda 演算中定义(类似的有可计算函数不能在构造演算或系统 F 中计算)。

**角动量守恒定律**是指系统所受合外[力矩为零时系统的](../Page/力矩.md "wikilink")[角动量保持不變](../Page/角动量.md "wikilink")。
\(\frac{d\mathbf{L}}{dt} = \mathbf{r} \times \mathbf{F}\)
当[方程式右边](https://zh.wikipedia.org/wiki/方程式 "wikilink")[力矩为零时](../Page/力矩.md "wikilink")，可知[角动量不随时间变化](../Page/角动量.md "wikilink")。

角动量守恒定律是[自然界普遍存在的](https://zh.wikipedia.org/wiki/自然界 "wikilink")[基本定律之一](https://zh.wikipedia.org/wiki/基本定律 "wikilink")，角动量的守恒实质上对应着空间旋转[不变性](https://zh.wikipedia.org/wiki/不变性 "wikilink")。例如，当考虑到[太阳系中的](https://zh.wikipedia.org/wiki/太阳系 "wikilink")[行星受到](../Page/行星.md "wikilink")[太阳的万有引力这一](../Page/太阳.md "wikilink")[有心力时](https://zh.wikipedia.org/wiki/有心力 "wikilink")，由于[万有引力对太阳这个参考点力矩为零](https://zh.wikipedia.org/wiki/万有引力 "wikilink")，所以他们以太阳为参考点的角动量守恒，这也说明了行星绕太阳[公转单位时间内与太阳连线扫过的面积大小总是恒定值的原因](https://zh.wikipedia.org/wiki/公转 "wikilink")。另外，角动量守恒定律也是[陀螺效應的原因](https://zh.wikipedia.org/wiki/陀螺效應 "wikilink")。

需要注意的是，由于成立的条件不同，角动量是否守恒与[动量是否守恒没有直接的联系](../Page/动量.md "wikilink")。

以下作为一个实例在质量点日元施转运动证明质量点的角运动恒量定理。 首先使施转的质量点质量当做m，和r做那个半径，和v做那个质量点的速度的话，
mv<sup>2</sup>/r变成质量点m的离心力。
在这里半径r变化Δr了的时候，假如质点m的速度v变化Δv了的话，这个半径变化Δr时的工作量变化mv<sup>2</sup>/r×Δr成为，和质量点m的速度变化Δv时的动能变化m(v
+Δv)<sup>2</sup>/2 - mv<sup>2</sup>/2成为。 使在这个时候的这个施转派的能源变化当做ΔE比能守恒定律是 ΔE
= 0 的话，并且下一次算式被形成，

ΔE = mv<sup>2</sup>/r×Δr + m(v +Δv)<sup>2</sup>/2 - mv<sup>2</sup>/2 = 0
。

展开这次算式的话

ΔE = mv<sup>2</sup>/r×Δr + mvΔv + mΔv<sup>2</sup>/2 = 0 ，

mΔv<sup>2</sup>/2的项在这里是Δ项目的平方项，并且考虑能无视的话，

ΔE = mv<sup>2</sup>/r×Δr + mvΔv = 0，而且变形是

Δr/r + Δv/v = 0

因而如是变成，Δr/r = - Δv/v，使这个结果Δ项目变成无限小的话，dr/r = - dv/v 能够得到。 做积分这个两边的话，达到，

∫1/r dr = -∫1/v dv 其结果变成下次，

log r = - log v + C，在这里C是积分常数，并且当做 C = logC' 的话，这到，

log r + log v = logC' 因而能说 rv=C'，就是说 rv = 常数。

因而 mvr 也变成常数，并且角动量守恒定律被能源守恒定律引导。

然而，到这里考察质量点m的日元施转了，但是如果使半径r当做向量**r**，使速度v当做向量**v** 的话，针对随圆施转，也变成rv =
**r**×**v**（但是v是**v**的对**r**的垂直成分）。 能据说因而也被关于本来的向量定义的角动量**r**×m**v** 守恒。

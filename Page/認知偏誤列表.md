**[認知偏誤](../Page/認知偏誤.md "wikilink")**（又稱對比效應）是在某些特定情況下的特定的思考、行為傾向，会导致[理性或判断产生系统性偏誤](https://zh.wikipedia.org/wiki/理性 "wikilink")，這些現象廣泛受到[心理学](../Page/心理学.md "wikilink")與[行为经济学研究](../Page/行为金融学.md "wikilink")。

虽然这些偏誤是重复研究确認的，然而關於如何分類與解釋，一直存在爭論\[1\]。大腦有一些處理資訊的規則，為了增進決策與判斷效率，這些「心理捷徑」稱作[捷思](https://zh.wikipedia.org/wiki/捷思 "wikilink")，捷思有時會產生一些不好的效應，這些不好的效應稱作[認知偏誤](../Page/認知偏誤.md "wikilink")\[2\]\[3\]。判斷與决策的偏誤也可能是由[动机](../Page/动机.md "wikilink")導致，例如信念受到[一廂情願](../Page/一廂情願.md "wikilink")的想法扭曲。有些偏见有多種認知（“冷”）或動機（“热”）的解釋，兩種效應也可能同時發生\[4\]\[5\]。

一些偏誤是否該視為真正的[不理性](https://zh.wikipedia.org/wiki/不理性 "wikilink")、是否可能導致有用的態度與行為，是有爭議的。例如，要瞭解他人時，人們常會問[引導性問題](https://zh.wikipedia.org/wiki/引導性問題 "wikilink")，看起來是為了應證自己對他人的假設，是一種偏誤；但有些人認為這種確認性偏誤應該視作，用於建立與他人的連結\[6\]。

絕大多數關於認知偏誤的研究都是以人類為主體。除了人類以外，還能夠從老鼠、鴿子、猴子身上也能觀察到現象\[7\]。

## 決策、信念與行為偏誤

這些偏誤多半會影響信念的形成、商業與經濟決策、以及其他一般的人類行為。它們是可複製與再現的，面臨特定情境時，一般可預期人們會有相應的偏差傾向。

| **[安於現狀偏誤](https://zh.wikipedia.org/wiki/安於現狀偏誤 "wikilink")**                                                             | Status quo bias                                           | 傾向於維持現狀。（參見[損失趨避](https://zh.wikipedia.org/wiki/損失趨避 "wikilink")、[原賦效應](https://zh.wikipedia.org/wiki/原賦效應 "wikilink")、[系統正當化](https://zh.wikipedia.org/wiki/系統正當化 "wikilink")）\[68\]\[69\] |
| **[刻板印象](../Page/刻板印象.md "wikilink")**                                                                                    | Stereotyping                                              | 根據事物的所屬類別或群體判斷其特質，而忽略其獨特性。                                                                                                                                                                  |
| **[斯德哥爾摩症候群](../Page/斯德哥爾摩症候群.md "wikilink")**                                                                            | Stockholm syndrome                                        | 被害者認同加害者的某些觀點和想法，並覺得自己不再受到威脅。                                                                                                                                                               |
| ****                                                                                                                      | Subadditivity effect                                      | 評估可能性時，對整體直接評估低於對各部分分別評估再加總\[70\]。                                                                                                                                                          |
| **[主观验证](../Page/主观验证.md "wikilink")**或**[主觀確認](https://zh.wikipedia.org/wiki/主觀確認 "wikilink")**                          | Subjective validation                                     | 相信某事是對的，就感覺它是對的。也會把巧合的事當做有關聯。                                                                                                                                                               |
| **[倖存者偏差](../Page/倖存者偏差.md "wikilink")**                                                                                  | Survivorship bias                                         | 專注於從某個過程中存活下來的人或事尋找弱點意欲補強，卻忽略了最大的弱點更可能在未存活的人或事之中。                                                                                                                                           |
| ****                                                                                                                      | Time-saving bias                                          | 低速行進時，低估可節省的時間或高估會損失的時間；高速行進時，高估可節省的時間或低估會損失的時間。                                                                                                                                            |
| **[单位偏误](https://zh.wikipedia.org/wiki/单位偏误 "wikilink")**                                                                 | Unit bias                                                 | 認為計量單位反映合理程度。例如認為一瓶、一碗、一盤食物是最合理的食用量\[71\]。                                                                                                                                                  |
| ****                                                                                                                      | Well travelled road effect                                | 低估采用熟悉路线的时间，高估采用陌生路线的时间。                                                                                                                                                                    |
| ****                                                                                                                      | Whole only effect                                         | 選項為整套方案時，忽略個別部分有協商的可能。                                                                                                                                                                      |
| ****                                                                                                                      | Zero-risk bias                                            | 偏好把小風險降為零（例如 1% → 0%），勝過把大風險降低更多（例如 5% → 2%）。                                                                                                                                               |
| ****                                                                                                                      | Zero-sum heuristic                                        | 直覺判定一個情況是零和的，但實情未必如此。[零和一詞源于](https://zh.wikipedia.org/wiki/零和 "wikilink")[博弈论](../Page/博弈论.md "wikilink")，是指成功與失敗期望值的總和為零\[72\]\[73\]。                                                     |
| ****                                                                                                                      | misattribution of arousal                                 | 當產生莫名的生理反應或認知矛盾時，人們可能會以不相干的理由來合理化之。又稱為**吊橋效應**。                                                                                                                                             |

## 社會偏誤

这些偏誤大多是由归因偏誤導致。

| 名稱                                                                                                                  | 原文                                                                 | | 描述                                                                                               |
| ------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------- |
| ****                                                                                                                | Actor-observer bias                                                | 解释他人行为歸因於內在个性，解释自己行为則歸因於环境影响（​​参见[基本歸因錯誤](https://zh.wikipedia.org/wiki/基本歸因錯誤 "wikilink")）。       |
| ****                                                                                                                | Defensive attribution hypothesis                                   | 把自己的成功归结于内在因素，如自己的能力、品质、努力等；把自己的失败归结于外在因素，如运气坏、环境恶劣、任务困难等。                                         |
| **[达克效应](../Page/达克效应.md "wikilink")**                                                                              | Dunning–Kruger effect                                              | 能力過差的人由於無法分辨能力好與能力差的不同，因而認為大家都一樣\[74\]。                                                            |
| ****                                                                                                                | Egocentric bias                                                    | 过于依赖自己的观点和对自己的评价过高而不合实际情况                                                                          |
| **[外在激励偏差](https://zh.wikipedia.org/wiki/外在激励偏差 "wikilink")**                                                       | Extrinsic incentives bias                                          | 認為他人需要外在激勵才會做好，而自己可以靠內在激勵做好。[基本歸因錯誤的反例](https://zh.wikipedia.org/wiki/基本歸因錯誤 "wikilink")。          |
| **[錯誤共識效應](../Page/錯誤共識效應.md "wikilink")**                                                                          | False consensus effect                                             | 高估别人对自己的认同程度，認為所有人以同一方式思考\[75\]。                                                                   |
| **[基本歸因錯誤](https://zh.wikipedia.org/wiki/基本歸因錯誤 "wikilink")**                                                       | Fundamental attribution error                                      | 解释他人行为时，归因于内在特质（一定是他有这样的人格，才做出这样的行为），而非外在情境（也许是情势所迫，或这个场所有特殊的潜规则）\[76\]。                           |
| **[群体归因错误](https://zh.wikipedia.org/wiki/群体归因错误 "wikilink")**                                                       | Group attribution error                                            | 把個體的特質當作其所屬群體的特質，或把個體的行為當做其所屬群體的偏好。                                                                |
| **[團體迷思](../Page/團體迷思.md "wikilink")**                                                                              | Groupthink                                                         | 又稱集體錯覺。團體在決策過程中，由於成員傾向讓自己的觀點與團體一致，因而令整個團體缺乏不同的思考角度，不能進行客觀分析。                                       |
| **[月暈效應](https://zh.wikipedia.org/wiki/月暈效應 "wikilink")**或**[暈輪效應](https://zh.wikipedia.org/wiki/暈輪效應 "wikilink")** | Halo effect                                                        | 評價他人時，以整體觀感評斷特定的能力、素質。（参见[刻板印象](../Page/刻板印象.md "wikilink")）\[77\]                                 |
| ****                                                                                                                | Illusion of asymmetric insight                                     | 认为自己比其他人懂得更多\[78\]。                                                                                |
| **[外部机构错觉](https://zh.wikipedia.org/wiki/外部机构错觉 "wikilink")**                                                       | Illusion of external agency                                        | 由於對外部機構有主觀喜好，而認為其資訊是有影響力的、有洞察力的、仁慈的。                                                               |
| ****                                                                                                                | Illusion of transparency                                           | 認為自己很容易被他人看透，或認為自己很容易看透他人。                                                                         |
| ****或****或                                                                                                          | Better-than-average effect; Superiority bias; Illusory superiority | 高估自己的優點、低估自己的缺點\[79\]。                                                                             |
| **[群内偏见](https://zh.wikipedia.org/wiki/群内偏见 "wikilink")**                                                           | Ingroup bias                                                       | 認為對方屬於自己團體，便給予較好的待遇。                                                                               |
| **[道德运气](https://zh.wikipedia.org/wiki/道德运气 "wikilink")**                                                           | Moral luck                                                         | 評價他人行為的道德時，根據結果而非根據行為時的情境。                                                                         |
| ****                                                                                                                | Naïve cynicism                                                     | 認為其他人比自己更自我中心。                                                                                     |
| ****                                                                                                                | Naïve realism                                                      | 相信自己所見所聞即是真相、是客觀且不帶偏見的，認為這是顯而易見的事實，理性人一定會同意自己，不同意的人一定是資訊不足、懶惰、不理性或有偏見。                             |
| **[外团体同质性偏见](https://zh.wikipedia.org/wiki/外团体同质性偏见 "wikilink")**                                                   | Outgroup homogeneity bias                                          | 認為自己群體的成員比較多樣化，而外部群體的成員是相當單一的\[80\]。                                                               |
| ****                                                                                                                | Projection bias                                                    | 不自觉地以为他人（或未来的自己）和（現在的）自己有相似的情感、思想與价值观\[81\]。                                                       |
| **[自利偏誤](https://zh.wikipedia.org/wiki/自利偏誤 "wikilink")**                                                           | Self-serving bias                                                  | 将成功归因于自己，而将失败归因于环境。（參見[基本歸因錯誤](https://zh.wikipedia.org/wiki/基本歸因錯誤 "wikilink")）\[82\]             |
| ****                                                                                                                | Shared information bias                                            | 團隊討論傾向花較多時間與精力討論所有成員都知道的事（即共有資訊），而花較少時間討論較少成員知道的事\[83\]。                                           |
| ****或****                                                                                                           | System justification                                               | 人們傾向於喜好與捍衛現有的社會、經濟、政治安排，貶低其他方案，即使這樣會犠牲個人或群體利益。（參見安於現狀偏誤）                                           |
| ****                                                                                                                | Trait ascription bias                                              | 認為自己的個性、行為、緒緒是多變的，而他人是一成不變且容易預測的。（參見[外团体同质性偏见](https://zh.wikipedia.org/wiki/外团体同质性偏见 "wikilink")） |
| **[终极归因错误](https://zh.wikipedia.org/wiki/终极归因错误 "wikilink")**                                                       | Ultimate attribution error                                         | 把問題歸因於整個群體，而非群體中的個體。                                                                               |
| ****                                                                                                                | Worse-than-average effect                                          | 對於高難度的事，認為自己比一般人差。是[優於常人效應的反例](https://zh.wikipedia.org/wiki/優於常人效應 "wikilink")\[84\]。             |

## 記憶錯誤與偏誤

在[心理学](../Page/心理学.md "wikilink")和[认知科学](../Page/认知科学.md "wikilink")中，**記憶偏誤**是一種[認知偏誤](../Page/認知偏誤.md "wikilink")，導致[記憶](../Page/記憶.md "wikilink")的強化、弱化（包括能否想起的機會、想起所需的時間，或兩者皆有），或者記憶內容的改變。記憶偏誤有很多種，如下：

| 名稱                                                                                                                                                                            | 原文                                                     | | 描述                                                                                                          |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------- |
| ****                                                                                                                                                                          | Bizarreness effect                                     | 怪異的事比尋常的事更容易記住。                                                                                               |
| ****                                                                                                                                                                          | Choice-supportive bias                                 | 回顧自己做過的選擇時，傾向認為是明智的（参见[支持選擇偏誤](https://zh.wikipedia.org/wiki/支持選擇偏誤 "wikilink")）。                             |
| ****                                                                                                                                                                          | Change bias                                            | 參與一件事的改變後，不易回憶過去的情況\[85\]。                                                                                    |
| ****                                                                                                                                                                          | Childhood amnesia                                      | 難以回憶四歲以前的事。                                                                                                   |
| **[保守傾向](https://zh.wikipedia.org/wiki/保守傾向 "wikilink")**或****                                                                                                                | Conservatism; Regressive bias                          | 記憶會傾向中庸不極端，把高價值高或然率的事記成比實際低，把低價值低或然率的事記成比實際高\[86\]\[87\]。                                                     |
| ****                                                                                                                                                                          | Consistency bias                                       | 記憶中他人過去的態度與行為會變得像目前的態度與行為\[88\]。                                                                              |
| ****                                                                                                                                                                          | Context effect                                         | 某情境中的記憶在其他情境下較不易回想。                                                                                           |
| **[跨种族效应](https://zh.wikipedia.org/wiki/跨种族效应 "wikilink")**                                                                                                                   | Cross-race effect                                      | 難以分辨其他種族的人。                                                                                                   |
| ****                                                                                                                                                                          | Cryptomnesia                                           | 接觸後記得某些影像或語句，卻不記得是看來聽來的，甚至會誤以為是自己原創，造成剽竊的爭議\[89\]。                                                            |
| ****                                                                                                                                                                          | Egocentric bias                                        | 回憶中的自己會美化，例如記憶中自己的考試成績比實際更好。                                                                                  |
| ****                                                                                                                                                                          | Fading affect bias                                     | 負面、不愉快的記憶比正面、愉快的記憶更快淡忘\[90\]。                                                                                 |
| ****或****                                                                                                                                                                     | False memory; Confabulation                            | 產生虛構的事填補記憶空白，即使沒有欺騙意圖。例如偽造某人的幼年照片給他看，他往往能說出以前做那些事的過程與回憶，但實際上他根本沒做過。常見於記憶障礙的患者。                                |
| ****                                                                                                                                                                          | Self-generation effect                                 | 自己生產的資訊記得比較清楚，他人生產的則容易淡忘。                                                                                     |
| **[谷歌效应](../Page/谷歌效应.md "wikilink")**                                                                                                                                        | Google effect                                          | 容易從網路上搜尋到的資訊比較容易遺忘。本詞源自於著名的網路搜尋引擎[谷歌](https://zh.wikipedia.org/wiki/谷歌 "wikilink")。                           |
| **[幽默效应](https://zh.wikipedia.org/wiki/幽默效应 "wikilink")**                                                                                                                     | Humor effect                                           | 幽默的事比不幽默的事更容易记住，可能的解釋是幽默的事有較長的認知處理時間，或造成較多的情緒激發（）。                                                            |
| ****                                                                                                                                                                          | Illusion of truth effect                               | 人们倾向认为听过的是真的（即使没意识到听过），而不管該陳述實際上是否有效。换言之，人容易相信熟悉的話勝過陌生的。                                                      |
| ****                                                                                                                                                                          | Illusory correlation                                   | 若認為兩件事相關，回憶時就會覺得兩者同時發生的頻率較高\[91\]\[92\]。                                                                      |
| ****                                                                                                                                                                          | Lag effect                                             | 比起短延遲時間的多次重複接觸，長延遲時間的少次接觸有較好的記憶與學習效果。（亦見[間隔效應](https://zh.wikipedia.org/wiki/間隔效應 "wikilink")）\[93\]          |
| ****                                                                                                                                                                          | Leveling and Sharpening                                | 記憶細節在每次回憶中扭曲的現象，特別重要的事會銳化或得到選取而憶起，事件的細節及詳情則會鈍化或遺失。時間過得越久或回憶次數越多，這種偏誤就越加強\[94\]。                               |
| ****                                                                                                                                                                          | Levels-of-processing effect                            | 用不同方式「編碼」的資訊，容易記憶的程度也不相同。例如視覺、聽覺、觸覺、嗅覺\[95\]。                                                                 |
| ****                                                                                                                                                                          | List-length effect                                     | 列表中有越多東西，能記住的比例就越少，但能記住的項目總數會增加\[96\]。                                                                        |
| ****                                                                                                                                                                          | Misinformation effect                                  | 記憶會受到事後資訊干擾，而變得不準確\[97\]。錯誤訊息效應是[逆向干擾](../Page/干擾理論.md "wikilink")(retroactive interference)的主要例子。            |
| ****                                                                                                                                                                          | Modality effect                                        | 對於來自口語的訊息，最後聽到的東西印象會較深刻；對於來自文字的訊息則不明顯。                                                                        |
| ****                                                                                                                                                                          | Mood-congruent memory bias                             | 與當下心境相合的資訊會比較容易回想。                                                                                            |
| **[轮流发言效应](https://zh.wikipedia.org/wiki/轮流发言效应 "wikilink")**                                                                                                                 | Next-in-line effect                                    | 輪流發言時，後發言的人不易記住之前的人說過的話\[98\]。                                                                                |
| ****                                                                                                                                                                          | Part-list cueing effect; Part-set cueing effect        | 記憶的東西為一整組時，提示部分項目會使回想後續項目更困難\[99\]。                                                                           |
| ****                                                                                                                                                                          | Peak-end rule                                          | 人們對一件事的體驗通常是由最高峰的感覺與結尾的感覺決定，而不是由總體決定。                                                                         |
| ****                                                                                                                                                                          | Persistence of traumatic event                         | 經歷[創傷事件後](../Page/心理創傷.md "wikilink")，往往會不斷反覆回憶。（參見[創傷後壓力症候群](../Page/創傷後壓力症候群.md "wikilink")）                |
| **[图片优势效应](https://zh.wikipedia.org/wiki/图片优势效应 "wikilink")**                                                                                                                 | Picture superiority effect                             | 以圖片傳達概念較以文字傳達概念容易學習與回憶\[100\]\[101\]\[102\]\[103\]\[104\]\[105\]。                                             |
| ****                                                                                                                                                                          | Positivity effect                                      | 老年人回憶中的內容往往會往正面扭曲。                                                                                            |
| **[初位效應](https://zh.wikipedia.org/wiki/初位效應 "wikilink")**、**[近時效應](https://zh.wikipedia.org/wiki/近時效應 "wikilink")**、**[序位效應](https://zh.wikipedia.org/wiki/序位效應 "wikilink")** | Primacy effect; Recency effect; Serial position effect | 序列末尾的東西最容易回憶，其次是序列開頭的，序列中間的訊息最不易記住\[106\]。                                                                    |
| ****                                                                                                                                                                          | Processing difficulty effect                           | 處理難度高的資訊，由於花較多時間閱讀與思考，因而較易記住\[107\]。                                                                          |
| ****                                                                                                                                                                          | Reminiscence bump                                      | 回憶人生事件時，青春期和成年早期的事特別容易想到\[108\]。                                                                              |
| ****                                                                                                                                                                          | Rosy retrospection                                     | 美化过去的回忆，往往比实际情況更好。                                                                                            |
| ****                                                                                                                                                                          | Self-relevance effect                                  | 與自身相關的訊息較易回想，與他人相關的較不易。                                                                                       |
| ****                                                                                                                                                                          | Source confusion                                       | 把偶發事件的記憶和其他資訊混淆，造成扭曲的記憶\[109\]。                                                                               |
| ****                                                                                                                                                                          | Spacing effect                                         | 比起無間隔的重複接觸，有間隔的重複接觸有較好的記憶與學習效果。（亦見[延遲效應](https://zh.wikipedia.org/wiki/延遲效應 "wikilink")）\[110\]               |
| ****                                                                                                                                                                          | Spotlight effect                                       | 高估自身言行舉止受他人關注的程度。                                                                                             |
| **[后缀效应](https://zh.wikipedia.org/wiki/后缀效应 "wikilink")**                                                                                                                     | Suffix effect                                          | 在資訊尾端加上不須記憶的項目，會使前面須記憶的項目變得特別不易記住。這是[近時效應的一種應用](https://zh.wikipedia.org/wiki/近時效應 "wikilink")\[111\]\[112\]。 |
| ****                                                                                                                                                                          | Suggestibility                                         | 回想事情時，如旁人暗示了某些想法，想到的內容會受扭曲。                                                                                   |
| ****                                                                                                                                                                          | Telescoping effect                                     | 認為近期的事比實際久遠些，久遠的事比實際近期些。                                                                                      |
| ****                                                                                                                                                                          | Testing effect                                         | 反复书写比反复阅读更容易记住資訊\[113\]。                                                                                      |
| **[舌尖現象](https://zh.wikipedia.org/wiki/舌尖現象 "wikilink")**                                                                                                                     | Tip of the tongue phenomenon                           | 能想起一件事的一部分或相關的東西，卻難以想起全部。導致在向他人敘述時，說到一半突然卡住\[114\]。                                                           |
| **[逐字效应](https://zh.wikipedia.org/wiki/逐字效应 "wikilink")**                                                                                                                     | Verbatim effect                                        | 話語中的大意比完整字句（逐字）更容易記住\[115\]。                                                                                  |
| **[雷斯多夫效应](https://zh.wikipedia.org/wiki/雷斯多夫效应 "wikilink")**                                                                                                                 | Von Restorff effect                                    | 醒目的事或特別強調的事比其他的事更容易記住\[116\]。                                                                                 |
| **[蔡格尼效應](https://zh.wikipedia.org/wiki/蔡格尼效應 "wikilink")**                                                                                                                   | Zeigarnik effect                                       | 比起己完成的事，未完成或中斷的事情會記得更清楚。                                                                                      |

## 統計與機率偏誤

[機率性推論是指根據有關](https://zh.wikipedia.org/wiki/機率 "wikilink")[機率的資訊推論事情的可能性](https://zh.wikipedia.org/wiki/機率 "wikilink")，而機率的估算通常來自[統計資料](https://zh.wikipedia.org/wiki/統計 "wikilink")，因此兩者密不可分。以下列出不熟悉或誤用機率造成的錯誤認知或推理：

| 名稱                                                            | 原文                                   | | 描述                                                                                                  |
| ------------------------------------------------------------- | ------------------------------------ | ----------------------------------------------------------------------------------------------------- |
| **[基本比率謬誤](../Page/基本比率謬誤.md "wikilink")**或**基本比率忽視**         | Base rate fallacy; Base rate neglect | 只關注針對性的資訊，忽略一般性的資訊（基本比率），導致不恰當的認知。例如因為酒駕較容易肇事，就認為肇事者多半是酒駕，然而由於酒駕者佔所有駕駛者的比率甚小，肇事者之中酒駕者的比例不會那麼高\[117\]。 |
| **[合取謬誤](https://zh.wikipedia.org/wiki/合取謬誤 "wikilink")**     | Conjunction fallacy                  | 認為較特殊性的情況比較一般性的情況更可能發生。例如對某人的描述很像女性主義者會做的事時，認為某人是替婦女辯護的律師的可能性比某人是律師的可能性更高\[118\]。                     |
| **[賭徒謬誤](../Page/賭徒謬誤.md "wikilink")**                        | Gambler's fallacy                    | 認為某事多次發生則未來發生的機率會較小，或多次未發生則未來發生的機率會較大。                                                                |
| **[逆賭徒謬誤](https://zh.wikipedia.org/wiki/逆賭徒謬誤 "wikilink")**   | Inverse gambler's fallacy            | 認為機率很小的事發生了，一定是做了很多次。                                                                                 |
| **[熱手謬誤](../Page/熱手謬誤.md "wikilink")**                        | Hot Hand Fallacy                     | 認為某事多次發生則未來發生的機率會較大。一般是在某人近來做某事很成功時，認為他未來做同一件事也會很成功，而忽略其成功可能只是隨機與巧合導致。                                |
| **[迴歸謬誤](../Page/迴歸謬誤.md "wikilink")**                        | Regression fallacy                   | 非常態的甲事發生以後，用乙措施處理後甲事扭轉，便斷定乙措施可扭轉甲事。然而非常態的事發生後，本來就比較容易發生較接近常態的事。                                       |
| **[檢察官謬誤](../Page/檢察官謬誤.md "wikilink")**                      | Prosecutor's fallacy                 | 泛指多種根據不相關資訊認定被告「無辜的機率」很小的情況。                                                                          |
| **[辯護人謬誤](https://zh.wikipedia.org/wiki/辯護人謬誤 "wikilink")**   | Defendant's fallacy                  | 泛指多種根據不相關資訊認定被告「犯罪的機率」很小的情況。                                                                          |
| **[多重比較謬誤](https://zh.wikipedia.org/wiki/多重比較謬誤 "wikilink")** | Multiple Comparisons Fallacy         | 廣泛比較二個群體的各種特徵，從中找出有明顯差異的幾個，宣稱它就是造成二個群體不同的原因。                                                          |
| **[德州神槍手謬誤](../Page/德州神槍手謬誤.md "wikilink")**                  | Texas sharpshooter fallacy           | 即「先射箭再畫靶」，常指統計研究做出結果後，把其中的集群當做有統計意義，然而實際上此集群更可能是隨機產生。                                                 |
| ****                                                          | Insensitivity to sample size         | 評估統計數據時，未考慮小樣本比大樣本更容易觀察到極端結果。                                                                         |
| **[戲局謬誤](https://zh.wikipedia.org/wiki/戲局謬誤 "wikilink")**     | Ludic fallacy                        | 過度使用統計與機率預測未來。                                                                                        |
| **[麥納馬拉謬誤](../Page/麥納馬拉謬誤.md "wikilink")**                    | McNamara fallacy                     | 過度相信數據、依賴數據評估事情，忽略難以量化的事。                                                                             |

## 成因理論

以下列出解釋認知偏誤成因的常見理論：

  - **[有限理性](https://zh.wikipedia.org/wiki/有限理性 "wikilink")** - 理性與最佳化的限制
      - **[前景理论](../Page/展望理论.md "wikilink")**
      - **心理账户**
      - **適應偏誤** - 在資訊有限下做出決策而造成偏誤，並以可能的錯誤為代價
  - **属性替代** - 建立複雜、困難的判斷，卻不自覺地換成簡單地判斷\[121\]
  - **[归因理论](../Page/归因理论.md "wikilink")**
      - **显着性**
      - **素樸實在論**
  - **[认知失调](../Page/認知失調.md "wikilink")**，及相关的：
      - **印象管理**
      - **自我知觉理论**
  - **判斷與決策的捷思**，包括：
      - **可用性捷思** - 估計哪些比較容易從記憶中取得，例如生動的、不尋常的、或激起情緒的例子\[122\]
      - **代表性捷思** - 根據相似性判斷可能性\[123\]
      - **情感捷思** - 根據情緒反應決策，而非計算風險與效益\[124\]
  - [情感的一些其他理论](../Page/情绪.md "wikilink")，如：
      - **情緒二因論**
      - **體細胞標記假說**
  - **內省錯覺**
  - [統計的誤解與誤用](../Page/統計誤用.md "wikilink")；[數學盲](https://zh.wikipedia.org/wiki/數學盲 "wikilink")。

2012年一篇《[心理学公报](https://zh.wikipedia.org/wiki/心理学公报 "wikilink")》文章表明，至少有八種看似无关的偏誤可由同一種資訊理論的产生机制产生，此機制假設從人類記憶儲存與提取資訊時，雜亂的資訊處理過程導致了偏誤\[125\]。

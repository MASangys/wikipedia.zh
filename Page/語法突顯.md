**語法突顯**（，又称**語法高亮**、**语法加亮**\[1\]），是指根據術語（[關鍵字](../Page/保留字.md "wikilink")）類別來顯示不同的[顏色与](https://zh.wikipedia.org/wiki/顏色 "wikilink")[字體以增强可读性的一种编辑器特性](https://zh.wikipedia.org/wiki/字體 "wikilink")。

这种功能可改善人对编辑器中内容（通常为[源代碼](https://zh.wikipedia.org/wiki/源代碼 "wikilink")）的可讀性，進而降低出现誤讀或誤寫而出现错误的几率。

## 範例

下面是一例[C++](../Page/C++.md "wikilink")的語法突顯范例：

``` cpp
// Allocate all the windows
for (int i = 0; i < max; i++) {
    pWins[i] = new Window();
}
```

在這個範例中，編輯器辨認出[關鍵字為](https://zh.wikipedia.org/wiki/關鍵字 "wikilink")<font color="blue">for</font>，<font color="blue">int</font>和<font color="blue">new</font>，并辨認出[變數](../Page/變數.md "wikilink")的名字**i**，**pWins**和**max**，相應地將它們以高亮颜色顯示，[註解则以](https://zh.wikipedia.org/wiki/註解 "wikilink")[灰色](https://zh.wikipedia.org/wiki/灰色 "wikilink")﹝或異於關鍵字的颜色﹞顯示，以避免與程式碼相混淆。

基本上现代专业的[代碼編輯器都支援語法突顯功能](../Page/文本编辑器.md "wikilink")，並且支援重新定義語法突顯功能。亦有不少常规文本编辑器支持这种功能。

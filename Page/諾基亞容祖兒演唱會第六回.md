**Concert Number 6**
容祖兒演唱會，是[香港女歌手](../Page/香港.md "wikilink")[容祖兒於](https://zh.wikipedia.org/wiki/容祖兒 "wikilink")2010年11月19日至28日，以及12月3日、4日在[紅磡體育館舉辦的](https://zh.wikipedia.org/wiki/紅磡體育館 "wikilink")12場大型演唱會。

主辦單位[英皇娛樂集團及](https://zh.wikipedia.org/wiki/英皇娛樂集團 "wikilink")[TVB
Music](https://zh.wikipedia.org/wiki/TVB_Music "wikilink")，主題贊助廠商為[諾基亞](https://zh.wikipedia.org/wiki/諾基亞 "wikilink")。

## 簡介

Nokia Joey Yung Concert Number 6 2010
為[容祖兒第](https://zh.wikipedia.org/wiki/容祖兒 "wikilink")6個於紅磡香港體育館舉行的個人演唱會，主題曲為《[空港](https://zh.wikipedia.org/wiki/空港_\(容祖兒專輯\) "wikilink")》，舞台切合演唱會的主題——[機場](../Page/機場.md "wikilink")，設計為一條[飛機起飛](https://zh.wikipedia.org/wiki/飛機 "wikilink")、降落的[跑道](../Page/跑道.md "wikilink")。
Concert Number 6
亦為容祖兒暫時個人演唱會場數第二多的演唱會，日期為2010年11月19日至28日，以及12月3日、4日，合共十二場。

## 各部分之簡介

### Opening + Classic Songs

歌曲 : 逃、跑步機上、怯、誰來愛我、16號愛人

服飾 : 羽毛頭飾 + 黑色網狀緊身衣（羽毛邊）+ 黑色長褲（羽毛邊）

### 馬戲團

歌曲 : 世界小姐、我所知的兩三事、全身暑假

服飾 : 彩色頭飾 + 彩色旋轉木馬裙（設計：Manish Arora）

### Sexy Wearing

歌曲 : Medley - 流淚眼望流淚眼 / 勞斯萊斯、天之驕女、蛇、穿花蝴蝶、桃色冒險

服飾 : 黑色貓型頭飾 ( 設計: Victoria Grant ) + 白色毛衣 ( 設計: Chanel ) ( Medley、天之驕女 )
、黑色貓型頭飾 ( 設計: Victoria Grant ) + 黑色厘士裙 ( 蛇、穿花蝴蝶、桃色冒險 )

### 慘情四部曲

歌曲 : 飛、習慣失戀、痛愛、破相

服飾 : 紅色低胸長裙（設計：Kenneth Chan）/藍色背心低胸長裙（設計：YSL）

### Dancer Part

歌曲 : Scream 、夢路 、喜喜 、未知 、隆重登場

服飾 : LED閃燈帽 + LED閃燈露腰長袖衫褲（設計：Kenneth Chan）

### 空港

歌曲 : 空港、逃避你、抱抱、舊日回憶的山丘、搜神記、心淡

服飾 : 背心彩虹裙(皮/棉) + 木結他（設計：Kenneth Chan）

### 越唱越強

歌曲 : 越唱越強

服飾 : 水花裙（設計：Iris van Herpen、Kenneth Chan）

### Encore

歌曲 : 去火星戀愛、想得太遠、麻煩你、告解、爭氣

服飾 : 演唱會主題T-Shirt（設計：5preview）

**謬論**或**（思考）謬誤**是指不當的推理言論或推理思路\[1\]。惡意或別有所圖的謬論又稱作[詭辯](https://zh.wikipedia.org/wiki/詭辯 "wikilink")。\[2\]

知識或資料的錯誤，稱作**訛誤**\[3\]，但日常習慣上，謬論或謬誤有時也用於泛指錯誤的言論或想法\[4\]。

一個好的[論證必須](https://zh.wikipedia.org/wiki/論證 "wikilink")**[有效](https://zh.wikipedia.org/wiki/有效 "wikilink")**且**[健全](https://zh.wikipedia.org/wiki/健全 "wikilink")**。所謂有效是指如果[前提為真則結論必定為真](https://zh.wikipedia.org/wiki/前提 "wikilink")，所謂健全是指除了有效以外，所有前提都是真的，健全的論證保證結論為真。反之，如果前提不為真或前提為真不能保證結論為真，則不是好論證。

論證有謬誤，不表示其結論不真確（參見[謬誤論證](https://zh.wikipedia.org/wiki/謬誤謬誤 "wikilink")）；然而，對於尚未相信結論的人而言，論證過程有謬誤會導致他們無法被論證說服。

## 分類

### 傳統分類

[論證的謬誤可發生在三個地方](https://zh.wikipedia.org/wiki/論證 "wikilink")：一是陳述了錯誤的事實（[實質謬誤](https://zh.wikipedia.org/wiki/實質謬誤 "wikilink")），二是使用了不恰當的詞語（[言詞謬誤](../Page/言詞謬誤.md "wikilink")）；三是使用了不恰當的推理結構（形式）（[形式謬誤](https://zh.wikipedia.org/wiki/形式謬誤 "wikilink")）。其中一與二合稱[非形式謬誤](https://zh.wikipedia.org/wiki/非形式謬誤 "wikilink")，而二與三合稱[邏輯謬誤](https://zh.wikipedia.org/wiki/邏輯謬誤 "wikilink")\[5\]：

|                                                         |                                                       |
| ------------------------------------------------------- | ----------------------------------------------------- |
| [非形式謬誤](https://zh.wikipedia.org/wiki/非形式謬誤 "wikilink") |                                                       |
| [實質謬誤](https://zh.wikipedia.org/wiki/實質謬誤 "wikilink")   | [言詞謬誤](../Page/言詞謬誤.md "wikilink")                    |
|                                                         | [邏輯謬誤](https://zh.wikipedia.org/wiki/邏輯謬誤 "wikilink") |
|                                                         |                                                       |

  - **形式謬誤**：形式謬誤是因不恰當的[推論結構所造成的謬誤](https://zh.wikipedia.org/wiki/推論 "wikilink")，可藉由[符號化的](https://zh.wikipedia.org/wiki/符號化 "wikilink")[邏輯系統揭露此類謬誤](https://zh.wikipedia.org/wiki/邏輯系統 "wikilink")。
  - **言詞謬誤**：泛指各種因自然語言[語意不明確而形成的推理謬誤](https://zh.wikipedia.org/wiki/語意 "wikilink")，可藉由[語義](https://zh.wikipedia.org/wiki/語義 "wikilink")、[語用分析及符號化揭露此類謬誤](../Page/语用学.md "wikilink")。
  - **實質謬誤**：泛指各種推理結構正確，但論證前提不能恰當支持結論所造成的謬誤。
  - **邏輯謬誤**：邏輯謬誤包括了形式謬誤及言詞謬誤，可藉由語義、語用分析及符號化揭露此類謬誤。「邏輯謬誤」有時亦可廣義泛指所有的謬誤。
  - **非形式謬誤**：形式謬誤以外的其他謬誤統稱非形式謬誤，含括了所有不能純粹由符號化邏輯系統揭露的謬誤，這些謬誤需要關於現實世界的背景知識與常識才能揭露出來。

### 四不分類

[香港](../Page/香港.md "wikilink")[哲學家](https://zh.wikipedia.org/wiki/哲學家 "wikilink")[李天命](../Page/李天命.md "wikilink")提倡將謬誤分成四大類：[不一致](../Page/不一致的謬誤.md "wikilink")、[不相干](../Page/不相干的謬誤.md "wikilink")、[不充分](https://zh.wikipedia.org/wiki/不充分的謬誤 "wikilink")、[不當預設](https://zh.wikipedia.org/wiki/不當預設的謬誤 "wikilink")。\[6\]

1.  不一致：主張的[命題](https://zh.wikipedia.org/wiki/命題 "wikilink")（前提或無前提的結論）[自相矛盾或](https://zh.wikipedia.org/wiki/自相矛盾 "wikilink")[自我推翻](https://zh.wikipedia.org/wiki/自我推翻 "wikilink")。
2.  不相干：前提與結論無關，無論前提是否成立，都無法推理出結論。
3.  不充分：前提與結論有關，但前提無法充分支持結論。
4.  不當預設：把不應視為理所當然的前提視為理所當然。

**语法糖**（Syntactic
sugar），也译为**糖衣语法**，是由英国[计算机科学家](https://zh.wikipedia.org/wiki/计算机科学家 "wikilink")[彼得·蘭丁发明的一个术语](https://zh.wikipedia.org/wiki/彼得·蘭丁 "wikilink")，指[计算机语言中添加的某种语法](https://zh.wikipedia.org/wiki/计算机语言 "wikilink")，这种语法对语言的功能没有影响，但是更方便程序员使用。语法糖让程序更加简洁，有更高的可读性。

举例来说，许多[程序语言提供专门的语法来对](https://zh.wikipedia.org/wiki/程序语言 "wikilink")[数组中的元素进行引用和更新](../Page/数组.md "wikilink")。从理论上来讲，一个数组元素的引用涉及到两个[参数](https://zh.wikipedia.org/wiki/参数 "wikilink"):[数组和](../Page/数组.md "wikilink")[下标向量](https://zh.wikipedia.org/wiki/下标向量 "wikilink")，比如这样的表达式：`get_array(Array,
vector(i, j))`。然而，许多语言支持这样直接引用：`Array[i,
j]`。同理，数组元素的更新涉及到三个参数：`set_array(Array,
vector(i, j), value)`，但是很多语言提供直接赋值：`Array[i, j] = value`。

## 語法糖的例子

### for循环

一个例子是[C語言中的for循环](https://zh.wikipedia.org/wiki/C語言 "wikilink")：

``` c
for (i = 0; i < 10; i++) {
    // P
}
```

其效果和以下的while循环是一樣的，其中P是循环要處理的主體。

``` c
i = 0;
while (i < 10) {
    // P
    i++;
}
```

### 中置表示法

[Haskell雖然是](../Page/Haskell.md "wikilink")[函數程式語言](https://zh.wikipedia.org/wiki/函數程式語言 "wikilink")，但它提供了**a
+
b**這種「[中置表示法](https://zh.wikipedia.org/wiki/中缀表示法 "wikilink")」的語法，讓程式更容易撰寫，也比較容易閱讀。

### Lua

在[Lua中](../Page/Lua.md "wikilink")，一个变量的赋值方法如下：

``` Lua
foo = "bar"
```

而一个函数，作为基本类型之一，语法与定义变量的方法基本一致：

``` Lua
foo = function()
    print "bar"
end
```

但是此种写法对于有其他编程语言背景的人来说不够直观，故Lua提供了如下的语法糖：

``` Lua
function foo()
    print "bar"
end
```

它与上述方法效果完全一样，但是更加紧凑和易于理解。

## 衍生詞語

### 语法盐

**[语法盐](../Page/语法盐.md "wikilink")**（）指的是让写出坏代码更难的语法特性。这些特性强迫程序员做出一些基本不用于描述程序行为，而是用来证明他们知道自己在做什么的额外举动。

### 语法糖精

**语法糖精**（），或者说**语法糖浆**（），指的是未能让编程更加方便的附加语法。\[1\]\[2\]

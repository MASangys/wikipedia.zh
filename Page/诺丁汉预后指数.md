**诺丁汉预后指数**（），简称**NPI**，是[乳癌](../Page/乳癌.md "wikilink")患者预后的估算方法，其使用的参数肿瘤大小、涉及淋巴结数量、肿瘤分级为参数，可以估算乳癌患者预后\[1\]。该指数不适用于[远端转移病患](../Page/遠端轉移.md "wikilink")\[2\]。

## 计算

计算公式如下\[3\]：

  -
    NPI=\[0.2x S\]+N+G

其中：

  - S即[肿瘤大小](https://zh.wikipedia.org/wiki/肿瘤 "wikilink")，以厘米计\[4\]\[5\]
  - N即[淋巴结分期](https://zh.wikipedia.org/wiki/淋巴结 "wikilink")：1期则没有无淋巴结转移，2期即1\~3个淋巴结转移， 3期则淋巴结转移个数大于4\[6\]\[7\]
  - G即[肿瘤分级](https://zh.wikipedia.org/wiki/腫瘤分級 "wikilink")：I级即1，II级即2，III级即3\[8\]\[9\]

## 解读方法

| 分数             | 5年存活几率\[10\] |
| -------------- | ------------ |
| \>/=2.0\</=2.4 | 93%          |
| \>2.4\</=3.4   | 85%          |
| \>3.4\</=5.4   | 70%          |
| \>5.4          | 50%          |

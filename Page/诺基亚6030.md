**诺基亚
6030**是由[诺基亚公司出品的一款](../Page/诺基亚.md "wikilink")[移动电话产品](../Page/移动电话.md "wikilink")，是一款入门级商务手机。外壳颜色主要有[黑色和](../Page/黑色.md "wikilink")[银色两种](https://zh.wikipedia.org/wiki/银色 "wikilink")。支持
Java MIDP 2.0，支持通过“空中传送”下载基于 Java 的应用软件和游戏。

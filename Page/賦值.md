在[代数中](../Page/代数.md "wikilink")，**赋值**是[域元素的](../Page/域_\(數學\).md "wikilink")[阶](../Page/階_\(群論\).md "wikilink")（多少）或元素重复度一个度量。推广到[交换代数](../Page/交换代数.md "wikilink")，就是对[复分析中](../Page/复分析.md "wikilink")[极点](../Page/极点_\(复分析\).md "wikilink")，[零点重复度度量](../Page/零点.md "wikilink")，推广到[代数数论中的](../Page/代数数论.md "wikilink")[代数整数整性的度量](../Page/代数整数.md "wikilink")，在[代数几何中也有类似概念](../Page/代数几何.md "wikilink")，一个域与它的赋值被称为**赋值域**。

## 定義

一個[域](../Page/域_\(數學\).md "wikilink")\(K\)上取值在[有序交換群Γ的](../Page/有序交換群.md "wikilink")**賦值**是從\(K^*\)到Γ的映射\(v\)，滿足下述性質：

  - \(v(xy) = v(x)+v(y)\)（即：\(v\)是群同態）
  - \(x+y \neq 0 \Rightarrow v(x+y) \geq \mathrm{max}(v(x),v(y))\)

Γ稱作\(v\)的**值群**。

兩個賦值\(v_i: K^* \rightarrow \Gamma_i \; (i=1,2)\)被稱作**等價的**，若且唯若存在有序交換群的同構\(\phi: \Gamma_1 \rightarrow \Gamma_2\)使得\(v_2 = \phi \circ v_1\)。

為了操作上的便利，我們通常會將\(v\)的值域擴至\(\Gamma \cup \{\infty\}\)，並設\(v(0)=\infty\)。

## *p*進賦值

設*p*為正[質數](../Page/質數.md "wikilink")。對於所有非[零的](../Page/0.md "wikilink")[有理數](../Page/有理數.md "wikilink")，存在一且唯一一個整數\(n\)使得
\(x = \frac{u}{v} p^n\)
，其中\(u,v\)均非\(p\)的倍數。*p*進賦值就是[函數](../Page/函數.md "wikilink")
\(v_p: x \to n\)。它給出一個p進絕對值
\(\vert\cdot\vert _p:\,\mathbb{Q} \to \mathbb{R}\)，定義為

|                                                                                     |          |
| ----------------------------------------------------------------------------------- | -------- |
| \(\vert x \vert _p =
  \begin{cases}
     0 \\
     p^{-v_p(x)} \\
  \end{cases}\) | 若\(x=0\) |
| 若\(x \ne 0\)                                                                        |          |

*p*進賦值是個非阿基米得賦值。其值群是 \(\Z\)。

## 例子

  - 令\(X\)為[緊黎曼曲面](../Page/黎曼曲面.md "wikilink")，\(\mathbb{C}(X)\)為其上的[亞純函數域](../Page/亞純函數.md "wikilink")。固定一點\(x \in X\)。定義\(v_x(f)\)為\(f\)在\(x\)的重根數，便得到\(\mathbb{C}(X)\)上的賦值，其值群為\(\mathbb{Z}\)。對於高維情形則須考慮其因子，但此時需考慮點的[拉開](../Page/拉開.md "wikilink")，狀況較複雜。[扎里斯基正是為了研究](../Page/扎里斯基.md "wikilink")[代數曲面而開始研究賦值論](../Page/代數曲面.md "wikilink")。
  - 上述構造亦可套用到定義在任意域上的[代數曲線](../Page/代數曲線.md "wikilink")。
  - 利用[函數域與](../Page/函數域.md "wikilink")[數域的類比](../Page/數域.md "wikilink")，可在\(\mathbb{Q}\)上考慮[p進賦值](../Page/P進數.md "wikilink")。根據**[奥斯特洛夫斯基定理](../Page/奥斯特洛夫斯基定理.md "wikilink")**，\(\mathbb{Q}\)上的任意賦值皆等價於某個p進賦值。

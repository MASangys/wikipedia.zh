[FusionintheSun.svg](https://zh.wikipedia.org/wiki/File:FusionintheSun.svg "fig:FusionintheSun.svg")或更小的恆星上佔有主導的地位。\]\]

**質子﹣質子鏈反應**是恆星內部將[氫融合成](../Page/氫.md "wikilink")[氦的幾種](../Page/氦.md "wikilink")[核融合反應中的一種](../Page/核融合.md "wikilink")，另一種主要的反應是[碳氮氧循環](../Page/碳氮氧循環.md "wikilink")。**質子﹣質子鏈反應**在[太陽或更小的恆星上佔有主導的地位](../Page/太陽.md "wikilink")。

克服兩個氫原子核之間的靜電斥力需要很大的能量，並且即使在太陽高溫的核心中，平均也還需要10<sup>10</sup>年才能完成。由於反應是如此的緩慢，因此太陽迄今仍能閃耀著，如果反應稍為快速些，太陽早就已經耗盡燃料了。

通常，質子﹣質子熔合反應只有在溫度（即動能）高到足以克服它們相互之間的[庫侖斥力時才能進行](../Page/庫侖定律.md "wikilink")。質子﹣質子反應是太陽和其它恆星燃燒產生能量來源的理論，是在1920年代由[亞瑟·史坦利·艾丁頓主張和提出基本原則的](../Page/亚瑟·爱丁顿.md "wikilink")。當時，太陽的溫度被認為太低，以至於不足以克服[庫侖障壁](../Page/庫侖障壁.md "wikilink")。直到[量子力學發展之後](../Page/量子力學.md "wikilink")，發現質子可以經由[波函數的](../Page/波函數.md "wikilink")[隧道](../Page/量子穿隧效應.md "wikilink")，穿過排斥障礙而在比[傳統預測為低的溫度下進行融合反應](../Page/古典物理.md "wikilink")。

## pp鏈反應

第一個步驟是兩個氫原子核<sup>1</sup>[H](../Page/氫.md "wikilink")（[質子](../Page/質子.md "wikilink")）融合成為[氘](../Page/氘.md "wikilink")，一個質子經由釋放出一個
[e<sup>+</sup>和一個](../Page/正電子.md "wikilink")[中微子成為](../Page/中微子.md "wikilink")[中子](../Page/中子.md "wikilink")。

  -
    <sup>1</sup>H + <sup>1</sup>H → <sup>2</sup>H +
    [e<sup>+</sup>](../Page/正電子.md "wikilink") +
    [ν<sub>e</sub>](../Page/微中子.md "wikilink")

在這個階段中釋放出的中微子帶有0.42[MeV的能量](../Page/電子伏特.md "wikilink")。

第一個步驟進行的非常緩慢，因為它依賴的[吸熱的β正電子衰變](../Page/正電子發射.md "wikilink")，需要吸收能量，將一個質子轉變成中子。事實上，這是整個反應的瓶頸，一顆質子平均要等待10<sup>9</sup>年才能融合成氘。

正電子立刻就和[電子湮滅](../Page/電子.md "wikilink")，它們的質量轉換成兩個[γ射線的](../Page/γ射線.md "wikilink")[光子被帶走](../Page/光子.md "wikilink")。

  -
    e<sup>+</sup> + e<sup>−</sup> → 2[γ](../Page/γ射線.md "wikilink")
    （它們的能量為1.02MeV）

在這之後，氘先和另一個氫原子融合成較輕的[氦](../Page/氦.md "wikilink")[同位素](../Page/同位素.md "wikilink")，[<sup>3</sup>He](../Page/氦-3.md "wikilink")：

  -
    <sup>2</sup>H + <sup>1</sup>H → <sup>3</sup>He +
    [γ](../Page/γ射線.md "wikilink") （能量為5.49 MeV）

然後有三種可能的路徑來形成氦的同位素[<sup>4</sup>He](../Page/氦.md "wikilink")。在pp1分支，氦-4由兩個氦-3融合而成；在pp2和pp3分支，氦-3先和一個已經存在的氦-4融合成鈹。
在太陽，pp1最為頻繁，佔了86%，pp2佔14%，pp3佔0.11%。還有一種是極端罕見的pp4分支。

### pp1分支

  -
    <sup>3</sup>He +<sup>3</sup>He → <sup>4</sup>He + <sup>1</sup>H +
    <sup>1</sup>H + 12.86 MeV

完整的pp1鏈反應是放出的淨能量為26.7MeV。
pp1分支主要發生在一千萬至一千四百萬[K的溫度](../Page/熱力學溫標.md "wikilink")，當溫度低於一千萬K時，質子﹣質子鏈反應就不能製造出<sup>4</sup>He。

### pp2分支

|  |                                 |   |                                                                                        |
|  | ------------------------------- | - | -------------------------------------------------------------------------------------- |
|  | <sup>3</sup>He + <sup>4</sup>He | → | <sup>7</sup>[Be](../Page/鈹.md "wikilink") + [γ](../Page/γ射線.md "wikilink")             |
|  | <sup>7</sup>Be + e<sup>−</sup>  | → | <sup>7</sup>[Li](../Page/鋰.md "wikilink") + [ν<sub>e</sub>](../Page/微中子.md "wikilink") |
|  | <sup>7</sup>Li + <sup>1</sup>H  | → | <sup>4</sup>He + <sup>4</sup>He                                                        |

pp2分支主要發生在一千四百萬至二千三百萬[K的溫度](../Page/熱力學溫標.md "wikilink")。

90%的在<sup>7</sup>Be(e<sup>−</sup>,ν<sub>e</sub>)<sup>7</sup>Li\*的反應中產生的微中子，90%帶有0.861MeV的能量，剩餘的10%帶有0.383
MeV 的能量（依據鋰-7是在基態還是激發態而定）。

### pp3分支

|  |                                 |   |                                                                             |
|  | ------------------------------- | - | --------------------------------------------------------------------------- |
|  | <sup>3</sup>He + <sup>4</sup>He | → | <sup>7</sup>Be + [γ](../Page/γ射線.md "wikilink")                             |
|  | <sup>7</sup>Be + <sup>1</sup>H  | → | <sup>8</sup>[B](../Page/硼.md "wikilink") + [γ](../Page/γ射線.md "wikilink")   |
|  | <sup>8</sup>B                   | → | <sup>8</sup>Be + e<sup>+</sup> + [ν<sub>e</sub>](../Page/微中子.md "wikilink") |
|  | <sup>8</sup>Be                  | ↔ | <sup>4</sup>He + <sup>4</sup>He                                             |

pp3鏈反應發生在二千三百萬[K以上的溫度](../Page/熱力學溫標.md "wikilink")。

pp3鏈雖然不是太陽主要的能量來源（只佔0.11%），但在[太陽微中子問題上非常重要](../Page/太阳中微子问题.md "wikilink")，因為它產生的微中子能量是非常高的（高達14.06
MeV）。

### pp4或Hep

雖然預測上有這種反應，但因為極為罕見（在太陽中只佔千萬分之三的量），因此從未曾在太陽中被觀測到。在此種反應中，氦-3直接和質子作用成為氦-4，可以產生能量更高的微中子（高達18.8
MeV）。

  -
    <sup>3</sup>He + <sup>1</sup>H → <sup>4</sup>He + ν<sub>e</sub> +
    e<sup>+</sup>

### 能量釋放

比較最後產生的氦-4和4個質子的質量，顯示少了0.007或是0.7%的質量。這些質量被轉換成了能量，在各自的反應中以γ射線和微中子的形式釋放出去。在一個完整的反應鏈可以得到26.73MeV的能量。

只有以γ射線釋放的能量會和電子與質子作用來加熱太陽的內部。這些熱量支撐著太陽使它不致於因為本身的重量而崩潰。

微中子不會與一般的物質發生交互作用，而且不會支持太陽去對抗本身的重力崩潰。微中子在pp1、pp2和pp3鏈分別帶走2.0%、4.0%和28.3%的能量\[1\]。

## pep反應

[Solar_Neutrino_Generation.png](https://zh.wikipedia.org/wiki/File:Solar_Neutrino_Generation.png "fig:Solar_Neutrino_Generation.png")

[Proton_proton_cycle.svg](https://zh.wikipedia.org/wiki/File:Proton_proton_cycle.svg "fig:Proton_proton_cycle.svg")
[氘也能經由罕見的pep](../Page/氘.md "wikilink")（質子﹣電子﹣質子）反應（電子捕獲）產生：

  -
    <sup>1</sup>H + e<sup>−</sup> + <sup>1</sup>H → <sup>2</sup>H +
    ν<sub>e</sub>

在太陽，pep反應和pp反應的比率是1：400，但是pep反應產生的微中子擁有更高的能量：在pp反應的第一步產生的微中子能量是0.42MeV，而pep反應產生的微中子譜線能量集中在1.44MeV。

pep和pp反應可以被看成是相同的基本交互作用，以兩種不同的[費曼图表示](../Page/費曼图.md "wikilink")。此處電子穿越到反應的右邊成為一個反電子，這在2006年NDM的網站圖中表示的是恆星內的質子﹣質子和電子捕獲鏈反應。\[2\]

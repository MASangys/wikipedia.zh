[Pseudorapidity2.png](https://zh.wikipedia.org/wiki/File:Pseudorapidity2.png "fig:Pseudorapidity2.png")之中，極角θ為零通常指的是入射粒子束的方向。\]\] 在實驗[粒子物理學](../Page/粒子物理學.md "wikilink")當中，**贗快度**（pseudorapidity, \(\eta\)）是一個常用的空間座標，用來描述粒子運動方向，與入射粒子束的夾角。其定義為：

\[\eta = -\ln\left[\tan\left(\frac{\theta}{2}\right)\right],\]

  -
    其中\(\theta \,\)是與入射粒子束的夾角（極角）。

數量上，當粒子具有相對論性（即其運動速度接近光速）時，贗快度近似於[快度](../Page/快度.md "wikilink")*y*。快度在[狹義相對論中定義為](https://zh.wikipedia.org/wiki/狹義相對論 "wikilink")：

\[y = \frac{1}{2} \ln \left(\frac{E+p_L}{E-p_L}\right)\]

這裡\(p_L\)代表粒子沿粒子束方向的[動量](https://zh.wikipedia.org/wiki/動量 "wikilink")。不過，贗快度只與粒子運動方向與粒子束的夾角（極角）有關，與粒子所帶的能量無關。

在強子對撞機的環境下，人們偏好使用快度（贗快度）來取代極角\(\theta\)。簡單的來說，這是因為粒子產生量對不同的贗快度是常數函數。而且，對於兩個不同的粒子，其贗快度的差是與粒子束方向的[勞侖茲變換無關的](https://zh.wikipedia.org/wiki/勞侖茲變換 "wikilink")。

以下是一些代表性的數字：

| \(\theta \,\) *(角度)* | \(\eta\) *（贗快度）* |
| -------------------- | ---------------- |
| 0                    | 無限大              |
| 5                    | 3.13             |
| 10                   | 2.44             |
| 20                   | 1.74             |
| 30                   | 1.31             |
| 45                   | 0.88             |
| 60                   | 0.55             |
| 80                   | 0.175            |
| 90                   | 0                |

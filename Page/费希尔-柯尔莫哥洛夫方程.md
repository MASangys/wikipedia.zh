[FKPPwiki.jpg](https://zh.wikipedia.org/wiki/File:FKPPwiki.jpg "fig:FKPPwiki.jpg") **费希尔-柯尔莫哥洛夫方程**是以英国统计学家[罗纳德·费希尔和俄国数学家](https://zh.wikipedia.org/wiki/罗纳德·费希尔 "wikilink")[安德雷·柯尔莫哥洛夫命名的](https://zh.wikipedia.org/wiki/安德雷·柯尔莫哥洛夫 "wikilink")[非线性偏微分方程](../Page/非线性偏微分方程.md "wikilink")，常见于[热传导](../Page/热传导.md "wikilink")、[燃烧](../Page/燃烧.md "wikilink")理论、[生物学](../Page/生物学.md "wikilink")、[生态学](../Page/生态学.md "wikilink")等领域。某些文献\[1\]\[2\]中又称费希尔-柯尔莫哥洛夫方程为**柯尔莫哥洛夫--皮斯库诺夫方程**（Kolmogorov–Petrovsky–Piskunov equation），或**KPP方程**，**费希尔-KPP方程**。费希尔-柯尔莫哥洛夫方程是[费希尔方程](../Page/费希尔方程.md "wikilink")的推广形式。费希尔-柯尔莫哥洛夫方程的基本形式为{{\#tag:ref|Graham所著的《Traveling wave analysis of partial differential equations : numerical and analytical methods with MATLAB and Maple》一书中第八章提到的“Fisher–Kolmogorov Equation”实际上是第十章“Kolmogorov–Petrovskii–Piskunov Equation”（即下式）在 D = 1、a = 1、b = -1、m = q + 1 时的特殊情况。|group="注"}}：

\[\frac{\partial u}{\partial t} = D \frac{\partial^2 u}{\partial x^2} + a u + b u^m\]

其中，a、b、D、m为任意常数，且m不等于1。\[3\]\[4\]

通过重新定义时间的尺度，可以[不失一般性地令参数](https://zh.wikipedia.org/wiki/不失一般性 "wikilink") D 等于1，因此一些文章中直接将形如 \(u_t - u_{xx} + \mu u + \nu u^2 + \delta u^3=0\) 称为KPP方程\[5\]\[6\]。其他形似KPP方程的，例如 \({\partial u}/{\partial t} = \frac{D}{2} {\partial^2 u}/{\partial x^2} + f(u)\) \[7\] 和 \(u_t + (-\Delta)^\alpha u = \mu (x) u - u^2\) \[8\] 被称作“KPP型方程（KPP type equation）”或“费希尔-KPP型方程（Fisher-KPP type equation）”。

## 解析解

形如 \(\frac{\partial u}{\partial t} = D \frac{\partial^2 u}{\partial x^2} + a u + b u^m\) 的KPP方程有以下解析解\[9\]：

\[u(x,t)=[\beta +exp(\lambda t +\frac{\mu x}{\sqrt{D}}]^\frac{2}{1-m}\]

其中，

\[\lambda=\frac{a(1-m)(m+3)}{2(m+1)}\]

\[\mu=\sqrt{\frac{a(1-m)^2}{2(m+1)}}\]

\[\beta=\sqrt{-\frac{b}{a}}\]

**西臺語**（），亦稱**涅西特语**（）。[西臺人的母語](https://zh.wikipedia.org/wiki/赫梯人 "wikilink")，現已滅絕，起自西元前1800年
，終於前1100年
，或許更早之前即有。西臺人發源於中北[安納托利亞(土耳其)，](https://zh.wikipedia.org/wiki/安納托利亞 "wikilink")
後來在哈圖沙(現代的[Boğazkale](https://zh.wikipedia.org/wiki/Boğazkale "wikilink")）建立帝國。帝國瓦解後，西臺楔形文字就蕩然棄用，但是直至西元前700年，在安納托利亞和北敘利亞，仍有人說西臺語
。

赫梯語是證實的最早[印歐語言](https://zh.wikipedia.org/wiki/印歐語系 "wikilink")，在[原始印歐語假說提出之后僅一個多世紀就被重新發現了](../Page/原始印歐語.md "wikilink")。因為在它的結構和語音上的顯著不同，一些現代[語言學家](https://zh.wikipedia.org/wiki/語言學家 "wikilink")，最顯著的是提議了[印度-赫梯語假說的](https://zh.wikipedia.org/wiki/印度-赫梯語 "wikilink")[Edgar
H.
Sturtevant和](https://zh.wikipedia.org/wiki/Edgar_H._Sturtevant "wikilink")[Warren
Cowgill](https://zh.wikipedia.org/wiki/Warren_Cowgill "wikilink")，他們爭論說它應當被分類為印歐語言的姐妹語言而非后代語言。但是很多學者繼續接受傳統的十九世紀原始印歐語為主的觀點，并把赫梯語的不尋常特征解釋為主要由於後來的革新。

## 名字

“赫梯”（Hittite）是[現代名字](https://zh.wikipedia.org/wiki/現代名字 "wikilink")，依據Hatti王國與[希伯來聖經中提到的赫梯王國的同一而選取](../Page/塔納赫.md "wikilink")（仍有爭議）。

在赫梯遺跡找到的多語言文本中，用赫梯語寫的章節都前導著副詞nesili（或nasili、nisili）、“的講話”，這是在帝國興起前的一个重要城市。在某一場合下，標簽是Kanisumnili“人的講話”。

儘管赫梯帝國是由來自很多不同種族和語言背景的人構成，他們的長期書寫文本多數都使用赫梯語。不管在這個術語的適合性上的各種爭論，赫梯仍是最常用的術語，儘管一些作者確立使用術語——**涅西特语**。

## 破解

[Jørgen Alexander
Knudtzon在](https://zh.wikipedia.org/wiki/Jørgen_Alexander_Knudtzon "wikilink")1902年发表了一本有关在[埃及](../Page/埃及.md "wikilink")[阿玛纳發現的两封信的书](../Page/阿玛纳.md "wikilink")，这两封信是埃及國王和赫梯統制者之間的书信往来，在这本书中，Jørgen
Alexander Knudtzon提出了第一个关于赫梯語從屬關係的論斷。Knudtzon
論證赫梯語是印歐語系語言，很大程度上是以[構詞為基礎的](https://zh.wikipedia.org/wiki/構詞學 "wikilink")。儘管沒有雙語言文本，他能夠對這兩封信給出部分解釋是因為這個時代的外交信件的客套話本質。他的論證不被一般接受，部分因為他在赫梯語和印歐語言之間觀察到的構詞相似性也可以在印歐語言之外找到，部分因為這兩封信的解釋可合理的當作是不確定性的。

當[Hugo
Winckler在](https://zh.wikipedia.org/wiki/Hugo_Winckler "wikilink")[Boğazköy的現代鄉村發現了大量的用熟知的](https://zh.wikipedia.org/wiki/Boğazköy "wikilink")[阿卡德](https://zh.wikipedia.org/wiki/阿卡德 "wikilink")[楔形文字書寫的未知語言的泥板的時候](../Page/楔形文字.md "wikilink")，Knudtzon最終被證明是正確，這里是赫梯帝國的首都[哈圖沙的前期遺跡](https://zh.wikipedia.org/wiki/哈圖沙 "wikilink")。基於對這么廣泛的材料的研究，[Bedřich
Hrozný成功的分析了這門語言](https://zh.wikipedia.org/wiki/Bedřich_Hrozný "wikilink")。他在1915年出版的一篇論文中提出這門語言是印歐語言的論證，隨后發表了這門語言的語法。Hrozný對赫梯語的印歐語言從屬關係的論證是徹底現代的，儘管貧乏實例證明。他集中於構詞的特質特征上的驚人相似性，這不太可能是由于變更而獨自出現，也不太可能是借用來的。這包括在某些名詞詞幹中r/n變換（參見
[rhotacism](https://zh.wikipedia.org/wiki/rhotacism "wikilink")）和[元音變換](https://zh.wikipedia.org/wiki/印歐元音變換 "wikilink")，二者可以在對應“water”的詞的主格單數wadar和屬格單數wedenas之間的變換中見到。他還提出一組正規的語音對應。在由于[第一次世界大戰導致的短暫的最初拖延之后](https://zh.wikipedia.org/wiki/第一次世界大戰 "wikilink")，Hrozný的破解，試探性文法分析，和赫梯語的印歐語言從屬關係的實證，被迅速接受并被當代學者如[Edgar
H.
Sturtevant廣泛實例證明](https://zh.wikipedia.org/wiki/Edgar_H._Sturtevant "wikilink")，他著作了第一部科學上可接受的赫梯語文法，并帶有文獻選編和詞匯表。Sturtevant文法的1951年修訂版在今天仍是權威性的。

## 歸類

赫梯語屬于[安納托利亞語族](https://zh.wikipedia.org/wiki/安納托利亞語族 "wikilink")。純赫梯語是從[楔形文字泥板和赫梯國王樹立的碑銘中得知的](../Page/楔形文字.md "wikilink")。叫做“赫梯象形文字”的文字已經被證明用于書寫密切相關的[盧維語](https://zh.wikipedia.org/wiki/盧維語 "wikilink")，而非純赫梯語。后來的語言如[呂基亞語和](https://zh.wikipedia.org/wiki/呂基亞語 "wikilink")[呂底亞語也被證實於赫梯疆域內](https://zh.wikipedia.org/wiki/呂底亞語 "wikilink")。Palaic
語也用於赫梯疆域中，并只在赫梯文檔中的儀式文本中被證實。安納托利亞語言分支還包括[卡里亞語](https://zh.wikipedia.org/wiki/卡里亞語 "wikilink")、Pisidian語和Sidetic語。

在赫梯語和盧維語中有很多借詞，特別是來自非印歐語言的[胡里特语和](https://zh.wikipedia.org/wiki/胡里特语 "wikilink")[哈梯語](https://zh.wikipedia.org/wiki/哈梯語 "wikilink")(Hattic)的宗教詞匯。哈梯語是[哈梯人的語言](https://zh.wikipedia.org/wiki/哈梯人 "wikilink")，他們是在被[赫梯人吸收或取代之前的Hatti地域的土著居民](https://zh.wikipedia.org/wiki/赫梯人 "wikilink")。宗教和玄幻的赫梯文本經常用哈梯語、胡里特语和卢维语書寫，即使在赫梯語成為其他著述的規范之后。

赫梯語傳統上被劃分為舊赫梯語（OH）、中赫梯語（MH）和新赫梯語（NH；不要混淆於實際上是后赫梯的“[Neo-Hittite](https://zh.wikipedia.org/wiki/Neo-Hittite "wikilink")”時期），分別對應於赫梯帝國的舊王國、中王國和新王國（大約公元前1750–1500、1500–1430年和1430–1180年）。這種階段劃分的基礎一部分在於語言學一部分在於古文書學。正因為中王國的概念已經很大程度上被破除了，Melchert
論爭說作為語言學術語的MH是不明確的描述并應當被理解為在OH和NH之間的變遷時期。

### 喉音

赫梯語保留了一些在其他印歐語言中消失了的非常古老的特征。例如，赫梯語保留了三個[喉音中的兩個](../Page/喉音理論.md "wikilink")(詞首的
h<sub>2</sub> 和
h<sub>3</sub>)。[弗迪南·德·索緒爾在](../Page/弗迪南·德·索緒爾.md "wikilink")1879年以在其他印歐語言中的元音音質為基礎，已經假定了這些語音的存在性，它們在任何證實了的其他印歐語言中都沒有保留為獨立的語音，直到赫梯語的發現。在赫梯語中，這個音位被寫為
。赫梯語和多數其他安納托利亞語一樣，在這方面不同於任何其他印歐語言，在赫梯語中喉音的發現是對索緒爾的假設的最顯著的證實。

喉音的保留，和缺乏赫梯語共享其他早期印歐語言擁有的文法特征的任何證據，已經導致一些語言學家相信安納托利亞語族與原始印歐語的余下部分之間分離得比任何其他分裂都要早。一些人提議了[印度-赫梯語系或超語系](https://zh.wikipedia.org/wiki/印度-赫梯語 "wikilink")，它包括了安納托利亞語族在分界線的一邊而余下的印歐語言在另一邊。絕大多數學者繼續重構著原始印歐語，但是都相信安納托利亞語是印歐語中第一個脫離的分支。

## 文法

作為已證實的印歐語言中最古老的語言，赫梯語的價值很大程度上在於它缺乏在其他古老的印歐語言比如[梵語](https://zh.wikipedia.org/wiki/梵語 "wikilink")、[拉丁語和](https://zh.wikipedia.org/wiki/拉丁語 "wikilink")[古希臘語中展現的一些文法特征](https://zh.wikipedia.org/wiki/古希臘語 "wikilink")。特別是赫梯語沒有對立陽性-陰性的印歐性別系統；它轉而有一個基於動物性-非動物性對立的初步名詞類系統，讓人想起非班圖[尼日爾-剛果語言的名詞類系統](https://zh.wikipedia.org/wiki/尼日爾-剛果語系 "wikilink")。

### 構詞

#### 名詞

赫梯語名詞系統由下列[格構成](https://zh.wikipedia.org/wiki/格_\(語法\) "wikilink"):
[主格](https://zh.wikipedia.org/wiki/主格 "wikilink")、[賓格](https://zh.wikipedia.org/wiki/賓格 "wikilink")、[與格](https://zh.wikipedia.org/wiki/與格 "wikilink")-[方位格](../Page/方位格.md "wikilink")、[屬格](https://zh.wikipedia.org/wiki/屬格 "wikilink")、[向格](https://zh.wikipedia.org/wiki/向格 "wikilink")(allative)、[離格和](https://zh.wikipedia.org/wiki/離格 "wikilink")[工具格](../Page/工具格.md "wikilink")，并區分兩種[數](https://zh.wikipedia.org/wiki/數_\(語法\) "wikilink")（單數和複數）和兩個[性](https://zh.wikipedia.org/wiki/性_\(語法\) "wikilink")，動物性和非動物性（有時稱為動物性和無性）。在性之間的區別是非常初步的，帶有一般只在主格中的區分，并且相同的名詞有時被證實於這兩個性中。

在它的最基本形式中，赫梯語名詞變格功能如下，對動物性使用了 “man”作為例子，對無性使用了 “place”作為例子。

|            |         |         |
| ---------- | ------- | ------- |
|            | **動物性** | **無性**  |
|            | **單數**  | **複數**  |
| **主格**     | Pisnas  | Pisnēs  |
| **賓格**     | Pisnan  | Pisnus  |
| **屬格**     | Pisnas  | Pisnas  |
| **與格/方位格** | Pisni   | Pisnas  |
| **離格**     | Pisnats | Pisnats |
| **向格**     | Pisna   | \-      |
| **工具格**    | Pisnit  | \-      |

可以看出來，在複數中有比在單數中區分更少的格的傾向。在早期文本中有一些名詞形成帶有 -u
的[呼格](../Page/呼格.md "wikilink")，但是呼格不再多產即使是在我們最早的資料中，它的功能在多數文檔中被主格涵蓋。向格也在語言發展的后來階段脫離了使用，它的功能被與格方位格所包容。在早期的文本中不規則的找到了屬格複數尾綴
-an，和工具格複數尾綴 -it。一些名詞還形成獨立的根本沒有尾綴的[方位格](../Page/方位格.md "wikilink")。

#### 動詞

在同其他早期證實了的印歐語言比如古希臘語和梵語相比較的時候，赫梯語的動詞系統在構詞上是相對不復雜的。有兩種動詞變位的一般[動詞類](https://zh.wikipedia.org/wiki/變位 "wikilink")，mi-變位和hi-變位。有兩種[語態](https://zh.wikipedia.org/wiki/語態 "wikilink")（主動和中間-被動），兩種[語氣](https://zh.wikipedia.org/wiki/語氣 "wikilink")（直陳和祈使），和兩種[時態](https://zh.wikipedia.org/wiki/時態 "wikilink")（現在和過去）。此外，動詞系統展示了兩種[動詞不定式](https://zh.wikipedia.org/wiki/動詞不定式 "wikilink")，[動名詞](../Page/動名詞.md "wikilink")(supine)，和[分詞](https://zh.wikipedia.org/wiki/分詞 "wikilink")。Rose列出了132個hi-動詞并解釋hi/mi分立為文法語態系統的遺跡("centripetal
voice" vs. "centrifugal voice")。

### 句法

赫梯語句法展示了一種值得注意的典型安納托利亞語特征。一般而言，句子或子句的開始由要么句子連接質詞，要么前移的或主題化的形式來構成，向它添加固定次序的[附著詞素鏈](../Page/附著詞素.md "wikilink")。

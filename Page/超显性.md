**超显性**是指一对[等位基因在](https://zh.wikipedia.org/wiki/等位基因 "wikilink")[異型合子](https://zh.wikipedia.org/wiki/異型合子 "wikilink")（Heterozygote）中的[表现型或生理功能超出其任一](https://zh.wikipedia.org/wiki/表现型 "wikilink")[同型合子的现象](https://zh.wikipedia.org/wiki/同型合子 "wikilink")。即假设A与a是一对等位基因，若[基因型](../Page/基因型.md "wikilink")为Aa的个体的[表现型比基因型为AA或aa更突出](https://zh.wikipedia.org/wiki/表现型 "wikilink")，则称基因型Aa具有“超显性”。

如野生的[苍蝇的基因型为W](https://zh.wikipedia.org/wiki/苍蝇 "wikilink")<sup>+</sup>W<sup>+</sup>，而白眼的苍蝇的基因型为WW，两个同型合子进行[杂交产生的](https://zh.wikipedia.org/wiki/杂交 "wikilink")[子代的基因型可能为W](https://zh.wikipedia.org/wiki/子代 "wikilink")<sup>+</sup>W，这种基因型的苍蝇[复眼的颜色比](https://zh.wikipedia.org/wiki/复眼 "wikilink")[亲代更白](https://zh.wikipedia.org/wiki/亲代 "wikilink")，这就是超显性。

超显性可能是[杂种优势的一个原因](https://zh.wikipedia.org/wiki/杂种优势 "wikilink")，但杂种优势却不一定是超显性。

## Gillespie模型

[群体遗传学](../Page/群体遗传学.md "wikilink")家John H.
Gillespie于2004年建立了以下模型\[1\]：

|        |                            |                            |                            |
| ------ | -------------------------- | -------------------------- | -------------------------- |
| 基因型：   | A<sub>1</sub>A<sub>1</sub> | A<sub>1</sub>A<sub>2</sub> | A<sub>2</sub>A<sub>2</sub> |
| 相对适合度： | 1                          | 1-hs                       | 1-s                        |

表中的h代表[杂合效应](https://zh.wikipedia.org/wiki/杂合效应 "wikilink")，而s代表[隐性等位基因效应](https://zh.wikipedia.org/wiki/隐性等位基因效应 "wikilink")。
因此给s赋值（即 0\<s\<1），则由h则可得出下列信息:

|         |                                                                                         |
| ------- | --------------------------------------------------------------------------------------- |
| h=0     | A<sub>1</sub>[显性](../Page/显性.md "wikilink")，A<sub>2</sub>[隐性](../Page/隐性.md "wikilink") |
| h=1     | A<sub>2</sub>显性，A<sub>1</sub>隐性                                                         |
| 0\<h\<1 | [完全显性](https://zh.wikipedia.org/wiki/完全显性 "wikilink")                                   |
| h\<0    | 超显性                                                                                     |
| h\>1    | [显性不足](https://zh.wikipedia.org/wiki/显性不足 "wikilink")                                   |

## 人类遗传学

[人类的](https://zh.wikipedia.org/wiki/人类 "wikilink")[镰刀型红血球疾病是由一](https://zh.wikipedia.org/wiki/镰刀型红血球疾病 "wikilink")[隐性遗传病](https://zh.wikipedia.org/wiki/隐性遗传病 "wikilink")。导致镰状细胞性贫血的基因对拥有这种基因的人有害，基因杂合的人的寿命预期较短，一般不能达到50岁。但这对等位基因的[表达也可以使拥有该基因的人获得一定的抵抗](https://zh.wikipedia.org/wiki/基因表达 "wikilink")[疟疾](../Page/疟疾.md "wikilink")的能力。因此，在某些疟疾常发地区，镰状细胞性贫血症患者会因其较强的抗疟疾能力被[自然选择](../Page/自然选择.md "wikilink")出来。对于不具有抵抗疟疾能力的[显性纯合子或可能患上更严重的镰状细胞性贫血症的](https://zh.wikipedia.org/wiki/显性纯合子 "wikilink")[隐性纯合子](https://zh.wikipedia.org/wiki/隐性纯合子 "wikilink")，杂合子患上两种疾病的可能性都较低，具有更高的适应性，体现出超显性的优越性。在Gillespie模型中，上述情形对应的是表格中h\<0的情状。

**超波里特數**（）是指一種特別的[偽質數](https://zh.wikipedia.org/wiki/偽質數 "wikilink")，其本身及所有正因數都是，也就是每一個正[因數](../Page/因數.md "wikilink")*d*（包括本身）都可以整除

  -
    2<sup>*d*</sup> − 2.

例如341為超波里特數，其正因數為{1, 11, 31, 341}，而：

  -
    (2<sup>11</sup> - 2) / 11 = 2046 / 11 = 186
    (2<sup>31</sup> - 2) / 31 = 2147483646 / 31 = 69273666
    (2<sup>341</sup> - 2) / 341 = 13136332798696798888899954724741608669335164206654835981818117894215788100763407304286671514789484550

小於10000的超波里特數有：

| 1  | 341 = 11 × 31   |
| 2  | 1387 = 19 × 73  |
| 3  | 2047 = 23 × 89  |
| 4  | 2701 = 37 × 73  |

## 有超過2個質因數的超波里特數

可以找到有3個不同質因數的超波里特數，若可以找到三個超波里特數，兩兩之間有三個質數的公因數，將三個質數相乘即可得到有三個質因數的超波里特數。

例如：

  - 2701 = 37 \* 73為超波里特數
  - 4033 = 37 \* 109為超波里特數
  - 7957 = 73 \* 109為超波里特數

因此294409 = 37 \* 73 \* 109 也是超波里特數。

以下列出幾組質數，每組都是七個，相乘乘積即為七個質因數的超波里特數：

  - { 103, 307, 2143, 2857, 6529, 11119, 131071 }
  - { 709, 2833, 3541, 12037, 31153, 174877, 184081 }
  - { 1861, 5581, 11161, 26041, 37201, 87421, 102301 }
  - { 6421, 12841, 51361, 57781, 115561, 192601, 205441 }

例如1.118.863.200.025.063.181.061.994.266.818.401 = 6421 \* 12841 \* 51361 \* 57781 \* 115561 \* 192601 \* 205441為超波里特數，其120個因數都是波里特數，且有七個質因數。

**越南粉**（），或稱**河粉**，是[越南一種以](https://zh.wikipedia.org/wiki/越南 "wikilink")[大米製成的](https://zh.wikipedia.org/wiki/大米 "wikilink")[河粉](https://zh.wikipedia.org/wiki/河粉 "wikilink")。越南人常这种河粉佐以生芽菜、香葉，並配上切片牛肉或雞絲食用，與「麵餅」（，[越南法式麵包](https://zh.wikipedia.org/wiki/越式法包 "wikilink")）同被視為越南菜的代表菜色。現時在越南賣河粉的形式主要有兩類，有街邊檔和連鎖店兩種，

## 歷史

越南粉的早期文獻並不多見，一般相信它在20世紀才出現在[河內街頭](https://zh.wikipedia.org/wiki/河內 "wikilink")，1920年代河內開設首間越南粉餐館，約1950年代再傳入[西貢](../Page/胡志明市.md "wikilink")，

隨著大量，這種河粉1970年代後在不少國家落地生根。在香港，越南餐館多數由越南華族所經營，一般以西貢式為主，。而在臺灣，則大部分是越籍女性[外籍配偶經營](../Page/外籍配偶_\(臺灣\).md "wikilink")，規模多為攤販或小店。

在北美地區、澳洲悉尼及墨爾本、及法國巴黎，亦可輕易找到越南河粉，一部分越南餐館亦會同時兼營中菜。

## 語源

在食物形式上，越南粉的製造方法與口感，均與河粉類似。兩者同樣以[魚露作調味](../Page/魚露.md "wikilink")，並採用[牛丸](https://zh.wikipedia.org/wiki/牛丸 "wikilink")、[牛腩](../Page/牛腩.md "wikilink")、[牛肉片及](https://zh.wikipedia.org/wiki/牛肉片 "wikilink")[芽菜等材料](https://zh.wikipedia.org/wiki/芽菜 "wikilink")。

[Phingredients_in_Ho_Chi_Min_City.jpg](https://zh.wikipedia.org/wiki/File:Phingredients_in_Ho_Chi_Min_City.jpg "fig:Phingredients_in_Ho_Chi_Min_City.jpg")、[胡荽葉](../Page/芫荽.md "wikilink")、[羅勒](../Page/羅勒.md "wikilink")、紅椒、青檬\]\]
越南粉的湯底主要用洋蔥和牛骨餚製，部分越南地區亦會加入[冰糖調味](../Page/冰糖.md "wikilink")，令湯底有淡淡甜味。而今天在海外廣泛流傳的越南河粉屬於西貢式，其特點是大量採用新鮮香料，在湯料上加入新鮮[薄荷](../Page/薄荷.md "wikilink")、[胡荽葉](../Page/芫荽.md "wikilink")、[羅勒](../Page/羅勒.md "wikilink")（或稱九層塔），佐以魚露、青檸汁及新鮮紅椒。除了牛腩、雞絲、肉丸外，越南粉亦慣常配以生牛肉，即席在客人面前以熱湯燜熟，成為該菜一大特式。

除了廣東起源一說外，越南近年亦有人提出phở一詞源自法國[火上锅](../Page/火上锅.md "wikilink")（）一詞，但phở一字的聲調並非帶平調，而在越南話中的法語借用詞中，除非該字以t、p、c、ch作結尾，否均該字均屬平聲，phở一詞並不符合這種規律；法國菜中亦沒有河粉。但不少人相信作為農業社會的越南，進食生牛肉有可能是受到法國菜的影響。

## 種類

越南河主要成為北圻式（北部）、順化式（中部）及西貢式（南部）三大特式，其中北部的河粉較粗且闊，早期除了辣椒及青檬外，亦較少放別的香葉素菜；南部的河粉相對較幼細，近似與流行在馬來半島及泰國的[粿條](https://zh.wikipedia.org/wiki/粿條 "wikilink")，進食時亦較多加入新鮮香料及芽菜。其典型菜單如下，細心留意其越南字與漢譯比較時，與目前香港流行的麵食店有不少相近之處：

北部和南部越南河粉比較：

|    | 北部河粉            | 南部河粉                            |
| -- | --------------- | ------------------------------- |
| 米粉 | 使用較粗的米粉         | 使用較幼的米粉                         |
| 湯底 | 湯底較清澈和簡單        | 味道比較甜和辣                         |
| 肉類 | 喜歡使用雞肉或者簡單剁碎的牛肉 | 喜歡使用不同牛肉部分，如牛肉切片，骨髓，肌腱，腩肉，胸肉和肉丸 |
| 配料 | 使用較多的蔥          | 使用較多的豆芽和各種香草，包括泰國羅勒和香菜          |
| 醬汁 | 使用米醋，魚醬和辣椒醬調味   | 使用青檸，海鮮醬，辣椒醬和新鮮切片辣椒調味           |

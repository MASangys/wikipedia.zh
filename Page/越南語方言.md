**越南語方言**，大致分做3類。三者間的差異主要在音韻和詞彙上，文法上的差異非常小。 [缩略图](https://zh.wikipedia.org/wiki/File:Phương_ngữ_tiếng_Việt_-_Copy_-_Copy.png "fig:缩略图") 現代[越南語的發音以](https://zh.wikipedia.org/wiki/越南語 "wikilink")[河內腔](https://zh.wikipedia.org/wiki/河內 "wikilink")（北方方言）為標準。但是不少的海外越僑說的是[西貢](https://zh.wikipedia.org/wiki/西貢 "wikilink")（南方方言）腔的越南語。海外的越南語[媒體多數都用西貢腔廣播](https://zh.wikipedia.org/wiki/媒體 "wikilink")。河內腔跟西貢腔主要差別是在聲調和[捲舌音](../Page/捲舌音.md "wikilink")上。

## 音韻差異

在河內腔中，後面3組的字母並沒有發音上的差別：<span lang="vi" dir="ltr">tr</span>=<span lang="vi" dir="ltr">ch</span>=<span lang="vi" dir="ltr"></span>、<span lang="vi" dir="ltr">d</span>=<span lang="vi" dir="ltr">r</span>=<span lang="vi" dir="ltr">gi</span>=<span lang="vi" dir="ltr">\[z\]</span>、<span lang="vi" dir="ltr">s</span>=<span lang="vi" dir="ltr">x</span>=<span lang="vi" dir="ltr"></span>。但是在西貢腔中，<span lang="vi" dir="ltr">r</span>、<span lang="vi" dir="ltr">s</span>、<span lang="vi" dir="ltr">tr</span>需要捲舌，<span lang="vi" dir="ltr">ch</span>、<span lang="vi" dir="ltr">gi</span>、<span lang="vi" dir="ltr">x</span> 沒有捲舌。而且西貢腔將 <span lang="vi" dir="ltr">d</span> 唸做半元音。

| 捲舌音                                 | 平舌音                                 | 半元音                                | 現代標準音                                    |
| ----------------------------------- | ----------------------------------- | ---------------------------------- | ---------------------------------------- |
| <span lang="vi" dir="ltr">tr</span> | <span lang="vi" dir="ltr">ch</span> |                                    | <span lang="vi" dir="ltr">tr=ch=</span>  |
| <span lang="vi" dir="ltr">r</span>  | <span lang="vi" dir="ltr">gi</span> | <span lang="vi" dir="ltr">d</span> | <span lang="vi" dir="ltr">r=gi=d=</span> |
| <span lang="vi" dir="ltr">s</span>  | <span lang="vi" dir="ltr">x</span>  |                                    | <span lang="vi" dir="ltr">s=x=</span>    |

在聲調上，西貢腔將跌聲（陽上）歸併到問聲（陰上）。所以，西貢腔的越南語只有5個聲調。

[越南语的](../Page/越南语.md "wikilink")[代词分为](https://zh.wikipedia.org/wiki/代词 "wikilink")[人称代词](https://zh.wikipedia.org/wiki/人称代词 "wikilink")、[指示代词和](https://zh.wikipedia.org/wiki/指示代词 "wikilink")[疑问代词三类](https://zh.wikipedia.org/wiki/疑问代词 "wikilink")，起到代替名词短语的作用。有真正的代词与家族称谓名词两类，很多都是由[汉语中对](../Page/汉语.md "wikilink")[亲属的称呼而来](https://zh.wikipedia.org/wiki/亲属 "wikilink")，但意义发生改变。使用代词时需要考虑到所指人的[性别](../Page/性别.md "wikilink")、[年龄](../Page/年龄.md "wikilink")、[身份等](https://zh.wikipedia.org/wiki/身份 "wikilink")。

## 真正的代词

越南语中通常用的真正的代词也可以根据能否与表复数的 ** 或 ** 搭配而分为两类，可构成复数的人称代词见下：

|      | 单数         | 复数             |
| ---- | ---------- | -------------- |
| 第一人称 | *, , , , * | *, , , , , , * |
| 第二人称 | *, *       | *, , , *       |
| 第三人称 | **         | *, , *         |

下面就使用场合作简略的说明。

  - **（）：对朋友或熟人自称；对非亲属的长辈、晚辈自称。
  - **（）：对亲密朋友自称；小孩间自称。亲昵。
  - **（）：
      - （第一人称单数）傲慢的自称；对极亲密的朋友自称；在日记中或描述心理活动。
      - （第一人称复数）咱们義。还有“我”（国、军）、“国产”、“本土”之義。
      - （第三人称单数）置于家族称谓词后。
  - **（）：对轻蔑或憎恨的人自称；长辈对晚辈自称；对极亲密的朋友自称。
  - **（）：自称用 ** 的场合的对称。
  - **（）：可指人、动物或事物。指人时为蔑称，只有在称呼晚辈、轻蔑的人或敌人时用。也可作第三人称复数代词。
  - **（）：咱们。
  - **（）：不包括对方。也可作第一人称单数代词，表礼貌或谦逊。
  - *、*：不包括对方。使用场合与 *、* 相似。
  - *、、*：自称用 ** 和 ** 时的对称代词。** 可作第二人称单数代词，与 ** 相同。
  - **：只能指人。无褒贬之分。
  - **：有不尊重之意，一般用于称呼小孩、看不起的人或敌人。也可指动物和事物。
  - **：有不尊重之意，一般用于称呼小孩、看不起的人或敌人。也可指动物。

以下是不能与复数词搭配的代词。它们很多为旧称或带有文学色彩，特别是第一、二人称代词。

| ------------ | ------------ | -------- |
| 第一人称         | **（）（熟／书面）   | **（）（书面） |
| **（）（男／书面）   |              |          |
| **（）（女／书面）   |              |          |
| **（）（王对臣／旧）  |              |          |
| 第二人称         | **（）（女对男／书面） | –        |
| **（）（女对男／书面） |              |          |

| **（）（熟）      |              |          |

  - **：与 *nó* 相似，但只能指人，多指男性。
  - **：只能指人，不能指物。贬义，但感情色彩不如 ** 强烈。在南部语中可被复数化。
  - **：人、人家。泛指。

## 家族称谓词

一如其他[東亞語言](https://zh.wikipedia.org/wiki/東亞 "wikilink")，越南語家族称谓词也可作为人称代词。这同样是十分常见的称呼他人的方法，应用范围很广，而且很复杂。汉语中也有类似的用法，例如北京话中用“大哥”、“姐姐”作为与不相识的人搭话前的称呼等，用戀人之間用哥、妹相稱等。这一点与西方语言有很大不同；比如，在英语中，如用“[哥哥](https://zh.wikipedia.org/wiki/哥哥 "wikilink")”、“[妹妹](https://zh.wikipedia.org/wiki/妹妹 "wikilink")”作[恋人间的互称](https://zh.wikipedia.org/wiki/恋人 "wikilink")，反倒有些乱伦的意味。

同一个家族称谓词经常是既可作为第二人称，也可作为第一人称和第三人称代词。例如 **（男性单数）：

  - **　小朋友，过来和我玩。（第一人称）
  - **　我爱你。（男性说给女性，** 用作第一人称）——**（女性单数）同样是家族称谓词。
  - **　我爱你。（女性说给男性，** 用作第二人称）
  - **　他说完就站起来走了。（第三人称）

** 后接指示代词 **（那），即构成第三人称单数代词“他”。

## 其他类型的人称代词

越南语中，很多指人的名词也可作为代词使用。例如 **（）、**（）、**（，即“朋友”）都可用作第二人称单数代词：

  - *<u>Bạn</u> đang sống ở đâu?*　<u>你</u>住在哪？

第一人称单数代词 *、*
也是由名词发展而来，原意均为“[仆人](https://zh.wikipedia.org/wiki/仆人 "wikilink")”。

再如： 以下對話中提及三個人：雄（**）、丽（**）、明（**）

  -
    雄：**　丽在干什么呢？——即：你在干什么呢？
    丽：**　丽在给明打电话。雄知道明在哪吗？——即：我在给明打电话。你知道他在哪吗？
    雄：**　不知道，雄不知道明在哪。——即：不知道，我不知道他在哪。

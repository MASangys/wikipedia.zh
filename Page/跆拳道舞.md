**跆拳道舞**是一種在近來十年間開始新興的表演用[跆拳道](../Page/跆拳道.md "wikilink")衍生活動。它有三分二是跆拳道的[套拳](https://zh.wikipedia.org/wiki/套拳 "wikilink")、[腳法](https://zh.wikipedia.org/wiki/腳法 "wikilink")，三分一是舞蹈（例如是[嘻哈](../Page/嘻哈.md "wikilink")或其他流行[舞蹈](../Page/舞蹈.md "wikilink")）。在跳這種舞蹈時會配合[跆拳道音樂或其他](https://zh.wikipedia.org/wiki/跆拳道音樂 "wikilink")[流行音樂](https://zh.wikipedia.org/wiki/流行音樂 "wikilink")，使其內容更加豐富。其练习者更可以跟着不同的[音乐](../Page/音乐.md "wikilink")而跳不同的舞，以达到強身健體的目的，同時舒筋活血。场地方面亦沒有特別的限制，在任何的地方也可以進行該活動。這種舞蹈亦曾於「[國泰航空新春國際巡遊匯演之夜2008](https://zh.wikipedia.org/wiki/國泰航空新春國際巡遊匯演之夜2008 "wikilink")」由機構[韓國兒童跆拳道表現過](https://zh.wikipedia.org/wiki/韓國兒童跆拳道 "wikilink")，受到不少香港市民的讚賞。下表為跟[嘻哈舞作出比較](https://zh.wikipedia.org/wiki/嘻哈舞 "wikilink")：

|          | 跆拳道舞                                    | 嘻哈舞                                  |
| -------- | --------------------------------------- | ------------------------------------ |
| **表演者**  | 任何有跆拳道基础的人                              | 年輕人、青年人                              |
| **常見衣著** | [色帶或黑帶](../Page/跆拳道用品.md "wikilink")、道褲 | 寬腿褲、緊身[T恤](../Page/T恤.md "wikilink") |
| **地方**   | 沒有固定的地方                                 | 多在街上                                 |
| **音樂**   | 節奏溫和或強勁，兩者皆有                            | 節奏強勁、充滿動感                            |
| **歷史**   | 源於九〇年代                                  | 源於六〇年代尾                              |

## 衣著

在表現這隻舞蹈時對衣著普遍只有两个要求：第一是穿[道裤](https://zh.wikipedia.org/wiki/道裤 "wikilink")；第二是繫上[色带或黑帶](../Page/跆拳道等級列表.md "wikilink")。至于身上面的衣服，可以根据[舞蹈](../Page/舞蹈.md "wikilink")的风格而定。除以上規定之外，表演者還可以佩戴各式各样的附加[飾品](https://zh.wikipedia.org/wiki/飾品 "wikilink")，令自己的風格更加配合該舞蹈的需要、更加可以表達出[舞蹈](../Page/舞蹈.md "wikilink")中的意思。

## 訓練

這隻舞除了融合著[舞蹈](../Page/舞蹈.md "wikilink")，還融合著跆拳道的[套拳等技術](https://zh.wikipedia.org/wiki/套拳 "wikilink")；如果沒有經過合適的[跆拳道](../Page/跆拳道.md "wikilink")訓練便進行該舞蹈的練習會比較難，所以一般一般練習這個跆拳道舞的要求也都是[綠帶以上](../Page/跆拳道等級列表.md "wikilink")，以便導師進行教授。而跆拳道本身就是一个集柔軟、力量及速度为一体的格鬥[運動](https://zh.wikipedia.org/wiki/運動 "wikilink")，长期練習跆拳道舞對[身体的协调性和灵活性都很有幫助](https://zh.wikipedia.org/wiki/身体 "wikilink")。

在[图论](../Page/图论.md "wikilink")中，一张[无向图里](https://zh.wikipedia.org/wiki/无向图 "wikilink")，两[顶点之间的](../Page/顶点_\(图论\).md "wikilink")**距离**是指他们之间**最短路径**（）的长度，两[顶点之间的](../Page/顶点_\(图论\).md "wikilink")**距离**也被称为**测地距离**（）\[1\]。需要注意的是两个[顶点之间可能有多条最短路径](../Page/顶点_\(图论\).md "wikilink")\[2\]，如果两个顶点之间不存在路径（即他们属于不同的[连通分量](../Page/连通图.md "wikilink")），那么按照传统它们距离被定义为无穷大。

在[有向图中](https://zh.wikipedia.org/wiki/有向图 "wikilink")，如果从[顶点](../Page/顶点_\(图论\).md "wikilink") \(u\) 到顶点 \(v\) 存在**有向路径**（），那么距离 \(d(u,v)\) 被定义为从顶点 \(u\) 到顶点 \(v\) 之间**最短有向路径**的长度\[3\]。不同于无向图，在有向图中 \(d(u,v)\) 不一定和 \(d(v,u)\) 相等，甚至有可能出现从顶点 \(u\) 到顶点 \(v\) 存在有向路径，而从顶点 \(v\) 到顶点 \(u\) 却不存在有向路径的情况。

## 相关概念

一个[顶点](../Page/顶点_\(图论\).md "wikilink") \(v\) 的**偏心率** \(\epsilon(v)\) 被定义为 \(v\) 和其它顶点的距离的最大值，也即是这个点和离其最远点的距离\[4\]。

一张图的**半径** \(r\) 被定义为最小的偏心率：\(r = \min_{v \in V} \epsilon(v)\)\[5\]

一张图的**直径** \(d\) 被定义为最大的偏心率：\(d = \max_{v \in V}\epsilon(v)\)，也即最远的两点间的距离。若要求得一张图的直径，首先要求得任意两点之间的最短路径，在这些所有的最短路径中，取其长度最长者，即是这张图的直径\[6\]。

一张半径为 \(r\) 的图的**中心点**是所有的偏心率为 \(r\) 的[顶点](../Page/顶点_\(图论\).md "wikilink")。若 \(v\) 是一个中心点，那么可用数学符号表示为 \(\epsilon(v) = r\)。一张图可以有不止一个中心点\[7\]。

**边缘点**和中心点的定义类似。一张直径为 \(d\) 的图的**边缘点**是所有的偏心率为 \(d\) 的顶点。若 \(v\) 是一个边缘点，那么 \(\epsilon(v) = d\)。一张图可以有多个边缘点\[8\]。

### 例2 - 直径和半径

|                                       |         |       |         |         |         |         |
| ------------------------------------- | ------- | ----- | ------- | ------- | ------- | ------- |
| [顶点](../Page/顶点_\(图论\).md "wikilink") | \(V_1\) | \(V\) | \(V_3\) | \(V_4\) | \(V_5\) | \(V_6\) |
| 偏心率                                   | 3       | 3     | 2       | 2       | 2       | 3       |

在图2中，例如，距离[顶点](../Page/顶点_\(图论\).md "wikilink") \(V_1\) 的最远点是顶点 \(V_6\)，那么 \(\epsilon(V_1) = 3\)。 每一个顶点的偏心率如上表所示，所以图1的半径为2，而直径为3。

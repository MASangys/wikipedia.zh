**車輛首次登記稅**，又稱**汽車首次登記稅**，是一種向首次登記的汽車車主徵收的一種[稅](../Page/稅.md "wikilink")項。它是一種[間接稅](https://zh.wikipedia.org/wiki/間接稅 "wikilink")。

在[香港](../Page/香港.md "wikilink")，車輛首次登記稅是由[運輸署](../Page/運輸署.md "wikilink")徵收的。顧名思義，車輛首次登記稅只是向第一次登記的車輛徵稅，故買賣是不用繳稅的。

一般來說，車輛的[應課稅值的釐定基準](https://zh.wikipedia.org/wiki/應課稅值 "wikilink")，須根據該車輛的公佈零售價，或根據[香港海關](../Page/香港海關.md "wikilink")評估的臨時應課稅值來計算\[1\]。

以下為香港汽車首次登記稅稅率\[2\]：

|                             |        |
| --------------------------- | ------ |
| **汽車類別**                    | **稅率** |
| 私家車                         |        |
| \- 最初的$150,000應課稅價值         | 40%    |
| \- 其次的$150,000              | 75%    |
| \- 其次的$200,000              | 100%   |
| \- 剩餘的應課稅價值（即$500,000以上的餘額） | 115%   |
| 電單車及機動三輪車                   | 35%    |
| 客貨車以外的貨車                    | 15%    |
| 許可車輛總重不超過1.9公噸的客貨車          |        |
| \- 最初的$150,000              | 35%    |
| \- 其次的$150,000              | 65%    |
| \- 餘額                       | 85%    |
| 許可車輛總重超過1.9公噸的客貨車           | 17%    |
| 的士、小巴、巴士或特別用途車輛             | 3.7%   |

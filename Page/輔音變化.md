**輔音變化**（、音變）在語言學裡是指一個字的[輔音根據其本身字詞的](https://zh.wikipedia.org/wiki/輔音 "wikilink")[語形及前後的](https://zh.wikipedia.org/wiki/語形學 "wikilink")[語法發生變化的現象](https://zh.wikipedia.org/wiki/語法學 "wikilink")。一般所稱之「音變」大抵指的是「輔音的變化」。輔音變化的現象在世界上的許多語言裡都能發現，然而最典型，也最為代表性的輔音變化，是現今所有[凱爾特語族語言的字首輔音變化](https://zh.wikipedia.org/wiki/凱爾特語族 "wikilink")。字首輔音變化的現象也可在[日語](https://zh.wikipedia.org/wiki/日語 "wikilink")、[印度語](https://zh.wikipedia.org/wiki/印度語 "wikilink")、[馬來語](https://zh.wikipedia.org/wiki/馬來語 "wikilink")、[南派優印地安語](https://zh.wikipedia.org/wiki/猶他-阿茲提克語族 "wikilink")、及許多[西非的語言](https://zh.wikipedia.org/wiki/非洲的語言 "wikilink")（例如[富拉語](../Page/富拉語.md "wikilink")）中發現。[芬蘭語支](../Page/芬蘭語支.md "wikilink")的語言如[芬蘭語和](https://zh.wikipedia.org/wiki/芬蘭語 "wikilink")[愛沙尼亞語有字中輔音變化](https://zh.wikipedia.org/wiki/愛沙尼亞語 "wikilink")；[肯亞地區](https://zh.wikipedia.org/wiki/肯亞 "wikilink")[尼羅-撒哈拉語系的](https://zh.wikipedia.org/wiki/尼羅-撒哈拉語系 "wikilink")[盧奧語和](https://zh.wikipedia.org/wiki/盧奧語 "wikilink")[英語有詞幹字尾的輔音變化](https://zh.wikipedia.org/wiki/英語 "wikilink")。而[希伯來語則在字首](https://zh.wikipedia.org/wiki/希伯來語 "wikilink")、字中、字尾全具有輔音變化。

## 凱爾特語族

凱爾特語族以其字首輔音變化著名；而語族之內，各種語言所擁有的輔音變化類型數目又有所不同：[蘇格蘭蓋爾語和](https://zh.wikipedia.org/wiki/蘇格蘭蓋爾語 "wikilink")[曼島語有一種輔音變化](https://zh.wikipedia.org/wiki/曼島語 "wikilink")，[愛爾蘭語](../Page/愛爾蘭語.md "wikilink")有兩種，[布立吞亞支](../Page/布立吞亞支.md "wikilink")的[威爾斯語](https://zh.wikipedia.org/wiki/威爾斯語 "wikilink")、[布列塔尼語](https://zh.wikipedia.org/wiki/布列塔尼語 "wikilink")、及[康瓦爾語則各有三種](https://zh.wikipedia.org/wiki/康瓦爾語 "wikilink")（三種語言的各三種輔音變化類型是不完全一樣的）。此外，愛爾蘭語、康瓦爾語、和布列塔尼語又有一種**混雜變化**，也就是某種情況下，某些輔音和其餘輔音會被觸發的輔音變化類型是不同的。

這些凱爾特語會觸發輔音變化的條件多少有些不同，但有兩種情況是一定會觸發輔音變化的：

1.  [定冠詞](https://zh.wikipedia.org/wiki/定冠詞 "wikilink") + 陰性[名詞](../Page/名詞.md "wikilink")，此陰性名詞會被觸發輔音變化
2.  陰性名詞 + [形容詞](https://zh.wikipedia.org/wiki/形容詞 "wikilink")，此形容詞會被觸發輔音變化

此外，許多語言都會依其[所有代名詞人稱數目的不同而觸發不同的輔音變化](https://zh.wikipedia.org/wiki/所有代名詞 "wikilink")，以下以布列塔尼語、愛爾蘭語、及威爾斯語為例：

| ---------------------- | --------------------- | -------------------- | ----- |
| maouez                 | bean                  | gwraig               | 女人    |
| bras                   | mór                   | mawr                 | 偉大的   |
| ar **v**aouez **v**ras | an **bh**ean **mh**ór | y **w**raig **f**awr | 偉大的女人 |
| kazh                   | cat                   | cath                 | 貓     |
| e **g**azh             | a **ch**at            | ei **g**ath          | 他的貓   |

| o **c'h**azh           | a **gc**at            | eu **c**ath          | 他們的貓  |

</center>

## 日語

當日語中的兩個單字組成一個複合字時，第二個單字的字首會被觸發字首輔音變化，稱為[連濁](https://zh.wikipedia.org/wiki/連濁 "wikilink")（漢字："連濁"，れんだく），舉例如下：

1.  にぎり（羅馬拼音：nigiri） + すし（羅馬拼音：sushi）→ にぎり**ず**し（[握壽司](https://zh.wikipedia.org/wiki/壽司 "wikilink")，漢字："握り寿司"；羅馬拼音：nigirizushi）
2.  にごり（羅馬拼音：nigori） + さけ（羅馬拼音：sake） → にごり**ざ**け（濁[酒](https://zh.wikipedia.org/wiki/清酒 "wikilink")，漢字：濁り酒；羅馬拼音：nigorizake）

"にごりざけ" 中的 にごり 和 "れんだく" 中的 だく 的[漢字同為](https://zh.wikipedia.org/wiki/漢字 "wikilink")**濁**，因為日語裡，正是用**清，濁**來表示無聲輔音與有聲輔音。

另外，日語的方言亦因着輔音變化而出現，演變成「關東方言」及「關西方言」(參看[近畿方言](../Page/近畿方言.md "wikilink"))。

## [芬蘭語](https://zh.wikipedia.org/wiki/芬蘭語 "wikilink")

在芬蘭語和其他同支的語言（如愛沙尼亞語）裡，詞幹字中的輔音會有輔音交替的現象，其中一種輔音變化為弱化；以下是部分表列：

<center>

| 詞幹      | 弱化                                                    |
| ------- | ----------------------------------------------------- |
| pp      | p                                                     |
| tt      | t                                                     |
| kk      | k                                                     |
| p       | v                                                     |
| t       | d                                                     |
| k       | [母音間歇](https://zh.wikipedia.org/wiki/母音間歇 "wikilink") |
| mp      | mm                                                    |
| nt      | nn                                                    |
| nk      | ng                                                    |
| lt      | ll                                                    |
| rt      | rr                                                    |
| uku/yky | uvu/yvy                                               |

</center>

芬兰语中的辅音交替是以音节的开闭为依据的。在开音节中一般是强势形势，而在闭音节中则是弱势形势。此现象在芬兰语中比较复杂，下面仅举一小部分例子。 例如單數屬格的名詞与形容詞就常會發生此字中的弱化：

  - la**pp**u '紙条'（主格），la**p**un（屬格）
  - hal**p**a '便宜'（主格），hal**v**an（屬格）
  - ko**t**a '拉普人的帳棚'（主格），ko**d**an（屬格）
  - su**k**a '刷子'（主格），suan（屬格）
  - p**uku** '套裝'（主格），p**uvu**n（屬格）

更詳細的列表：[1](http://www.stanford.edu/~laurik/fsmbook/exercises/FinnishConsonantGradation.html)

另在口语中，常會發生字尾輔音的[變音](https://zh.wikipedia.org/wiki/變音 "wikilink")，但這在[標準語中並不會被讀出來](https://zh.wikipedia.org/wiki/標準語 "wikilink")。

## 盧奧語

[盧奧語名詞詞幹的字尾輔音會發生有聲與無聲輔音的交替](https://zh.wikipedia.org/wiki/盧奧語 "wikilink")，將獨立詞轉變為像"...的山丘"、"...的枝條" 之類含意的附屬詞：

  - '山丘'（獨立詞），go**d**（附屬詞）

  - '枝條'（獨立詞），lu**ð**（附屬詞）

  - '外表'（獨立詞），ki**t**（附屬詞）

  - '骨頭'（獨立詞），（附屬詞）

  - bu**k** '書本'（獨立詞），bu**g**（附屬詞）

  - '書本'（獨立詞），（附屬詞）

註：交替的情況中，也常有些獨立於輔音變化的母音交替現象

## 英語

英語也曾有豐富的詞幹字尾磨擦音的輔音變化現象，可在以下的名詞-動詞詞組及英語[複數型態的構成中端倪出](https://zh.wikipedia.org/wiki/複數 "wikilink")：

  - li**f**e - li**v**e
  - proo**f** - pro**v**e
  - stri**f**e - stri**v**e
  - thie**f** - thie**v**e

  - brea - breae
  - mou (n.) - mou (vb.)
  - shea - sheae
  - wrea - wreae
  - choie - chooe
  - houe (n.) - houe (vb.)

現代英語裡發生於單複數轉換的輔音交替現象正漸漸在縮水，而且在下面列出的語例中，現在的許多使用者也僅剩 \[f-v\] 的交替能正確發出。

## 現代希伯來語

現代希伯來語只有一種擦音化輔音變化交替組合，但它可以發生在詞幹字首、字中、或字尾。

<center>

| 詞幹 | 擦音化 |
| -- | --- |
| p  | f   |
| k  | x   |
| b  | v   |

</center>

例如動詞的時態和詞形變化之間的轉換，會發生輔音變化：

  - **k**atav '他寫了'，yi**x**tov '他將要寫'
  - ti**b**a '他沉了'（不及物動詞），ta**v**a '他使...下沉了'（及物動詞）

陽性與陰性名詞、單複數名詞之間的轉換及介詞之後，也會有輔音變化的現象：

  - mele**x** '國王', mal**k**a '皇后'
  - do**v** '熊'（單數），du**b**im '熊'（複數）
  - **b**ayit '房子'，be-**v**ayit '在房子裡'

但不是所有的字詞都會發生輔音變化，例如：

  - **x**atav '他劈砍了'，ya**x**tav '他將要劈砍'
  - to**v** '貨物'（單數），tu**v**im '貨物'（複數）
  - **k**ibuc '基布茲農場'，be-**k**ibuc '在基布茲農場裡'

## 托爾金的語言

[魔戒](../Page/魔戒.md "wikilink")作者[约翰·罗纳德·鲁埃尔·托尔金少年時期發明的](https://zh.wikipedia.org/wiki/约翰·罗纳德·鲁埃尔·托尔金 "wikilink")[高多格林語及之後魔戒中的](https://zh.wikipedia.org/wiki/高多格林語 "wikilink")[辛達林語](https://zh.wikipedia.org/wiki/辛達林語 "wikilink")，都受到威爾斯語的啟發，擁有輔音變化的現象。

侏儒語的輔音變化現象主要只有**軟化**一種（少數特定詞幹會觸發鼻音化交替），觸發時機主要為定冠詞、前置詞之後的名詞，例如：

  - i（單數定冠詞）+ **b**ess "妻子" → i‧**v**ess
  - na（單數屬格）+ **t**alwint "雙足" → na‧**d**alwint

早期諾多林語則有**軟化**、**塞音變化**、及**鼻音化**三種，主要觸發條件亦為定冠詞、前置詞與名詞的組合，以及複合字的構成，例如：

  - i（單數定冠詞）+ **b**ad "路" → i‧**bh**ad
  - o "在...之上" + **c**â "頂端" → o‧**ch**a
  - i(n)（複數定冠詞）+ **t**î "線段" → i‧**th**iath，i‧**nth**iath
  - tain "山" + **p**ethil → Tain**B**ethil "[泰尼魁提爾山](https://zh.wikipedia.org/wiki/泰尼魁提爾山 "wikilink")"

魔戒時期的辛達林，則有變化最多的輔音變化，除了軟化、鼻音化、塞音變化外，還有**混雜變化**以及**流音變化**；觸發情況也更為多變，除了定冠詞、介詞、前置詞與名詞的組合，以及複合字的構成外，還有名詞後的形容詞與動詞後的直接受格，例如：

  - tol "島嶼" + **c**alen "綠色的" → tol **g**alen "綠色的島嶼"
  - eglerio "讚美吧" + **T**aur a **P**erhael "佛羅多與山姆" → eglerio **D**aur a **B**erhael "讚美佛羅多與山姆吧！"

## 輔音變化與連音的區別

字首輔音變化常與[連音混淆](https://zh.wikipedia.org/wiki/連音 "wikilink")：前者是指因為**語法**環境而觸發交替的現象，而後者為基於**語音**環境而觸發的交替。

語例如下：

  - [西班牙語](https://zh.wikipedia.org/wiki/西班牙語 "wikilink")：[鼻音與間歇後的輔音](../Page/鼻音_\(辅音\).md "wikilink") \[b, d, g\] 會和[母音及](https://zh.wikipedia.org/wiki/母音 "wikilink")[流音](../Page/流音.md "wikilink")之後的  交替，例：un \[**b**\]arco "一艘船"，mi \[**β**\]arco "我的船"。
  - [蘇格蘭蓋爾語](https://zh.wikipedia.org/wiki/蘇格蘭蓋爾語 "wikilink")：重音處的塞音在鼻音之後會交替為有聲輔音，例：\[**k<sup>h</sup>**aht\] "貓"， **g<sup>h</sup>**aht\] "貓"（加定冠詞）。

因語法而觸發的輔音變化的源頭，通常本來是像這些因為語音環境而觸發的連音，例如上述提到英語裡的擦音交替（輔音變化），實際上可追溯到[古英語時](https://zh.wikipedia.org/wiki/古英語 "wikilink")，母音或有聲輔音之間之間的擦音有聲化，以及觸發於字首、字尾或其他無聲輔音後的擦音無聲化（連音）。語例：古英語的不定詞字尾為 *-(i)an*，而複數字尾為 *-as*，因此詞幹 *hūs* "房子"（單數）的 's' 發 ，而 *hūsas* "房子"（複數）和 *hūsian* "住"（不定詞）的 's' 發 。但之後到了英語裡，許多原來的字尾都脫落了，加上受到大量[法語借字的擦音音素拼寫法](https://zh.wikipedia.org/wiki/法語 "wikilink")，所以本來的連音到了現代英語，就變成依語法觸發的輔音變化了。

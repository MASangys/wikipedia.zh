**輕聲**（neutral
tone）是[漢語中所具有的一種特殊](https://zh.wikipedia.org/wiki/漢語 "wikilink")[變調現象](../Page/變調.md "wikilink")（多見於[普通話](https://zh.wikipedia.org/wiki/普通話 "wikilink")）。輕聲一般不被當作[聲調看待](../Page/聲調.md "wikilink")，因為它沒有固定的[調值](../Page/調值.md "wikilink")。

## 發音特性

將漢語中的音節弱化之後，聲調會變得又輕又短，此種音變即為輕聲。

輕聲的調值需要根據前一個音節決定調值，所以輕聲音節不可能獨立存在。

有些方言沒有輕聲跟非輕聲的對立，但有輕音跟非輕音之別，如[廣州話](https://zh.wikipedia.org/wiki/廣州話 "wikilink")；有些方言輕聲跟非輕聲對立，但無輕音跟非輕音之別，如[台灣話](https://zh.wikipedia.org/wiki/台灣話 "wikilink")；有些方言輕聲跟非輕聲對立，亦有輕音跟非輕音之別，如[北京話](https://zh.wikipedia.org/wiki/北京話 "wikilink")。

## 發音規則

***注意***：由於可能未有定論，以下內容僅供參考。

1.  在陰平後，唸中降調，似較短的去聲；
2.  在陽平後，唸中降調，似較短的去聲(但和陰平後的輕聲相比略有不同)；
3.  在上聲後，唸半高平調(或是略升的聲調)，似輕短的陰平；
4.  在去聲後，唸低降調，似較短的去聲。

簡而言之：當前字是上聲時，輕聲字的音高較高；當前字是其他聲調時，輕聲的音高較低。在陰平、陽平、去聲後的輕聲是降調，在上聲後的輕聲是平調或不明顯的升調。

## 輕聲字詞判別

以下是一些判別輕聲字的規則，輕聲字用**粗體**註明：

|                                       |                                                     |
| ------------------------------------- | --------------------------------------------------- |
| 判別條件                                  | 舉例                                                  |
| 單音節虛詞([助詞](../Page/助詞.md "wikilink")) | 好**的**、輕輕**地**、說**得**好、聽**著**、太棒**了**、你**呢**、對**嗎** |
| 某些方位詞                                 | 墻**上**、屋**裡**、外**面**、裡**邊**                         |
| 表趨向的動詞                                | 放**下**、起**來**、出**去**                                |
| 動詞和名詞的重疊成份                            | 看**看**、試**試**、哥**哥**、謝**謝**                         |
| 某些詞綴，多不帶有真正含意                         | 桌**子**、房**子**、死對**頭**                               |

此外，還有很多尚未確定的規則，以及一些約定俗成的輕聲詞，在此暫不細究。

**转移金属化**（）是一种[配体在金属原子间发生转移的](https://zh.wikipedia.org/wiki/配体 "wikilink")[有机金属反应](https://zh.wikipedia.org/wiki/有机金属 "wikilink")。该反应典型的形式为M<sub>1</sub>-R
+ M<sub>2</sub>-R' → M<sub>1</sub>-R' +
M<sub>2</sub>-R，R和R'包括但不限于[烷基](../Page/烷基.md "wikilink")、[芳基](https://zh.wikipedia.org/wiki/芳基 "wikilink")、[炔基](https://zh.wikipedia.org/wiki/炔基 "wikilink")、[烯丙基](https://zh.wikipedia.org/wiki/烯丙基 "wikilink")、[卤素或](../Page/卤素.md "wikilink")[拟卤素](../Page/拟卤素.md "wikilink")[基团](https://zh.wikipedia.org/wiki/基团 "wikilink")。鉴于[热力学和](../Page/热力学.md "wikilink")[动力学因素](../Page/化学动力学.md "wikilink")，这一反应通常是不可逆的。热力学上，金属的电负性有利于反应的进行；动力学上，涉及的两种金属如存在空轨道，也有利于反应进行。\[1\]
转移金属化包括不同的类型，即氧化还原-转移金属化和氧化还原转移金属化/配体交换。转移金属化过程中，金属-碳键被激活，新的金属-碳键形成。\[2\]转移金属化通常应用于[催化剂](../Page/催化剂.md "wikilink")、[主族元素化合物合成和](https://zh.wikipedia.org/wiki/主族元素 "wikilink")[过渡金属化合物的合成](../Page/过渡金属.md "wikilink")。

## 转移金属化反应的分类

### 氧化还原-转移金属化

氧化还原-转移金属化过程中，配体经分子间[机理](../Page/反应机理.md "wikilink")，从一种金属转移到另一种金属。本反应通常的形式为M<sub>1</sub><sup>n+</sup>-R
+ M<sub>2</sub> → M<sub>1</sub> +
M<sub>2</sub><sup>n+</sup>-R。反应中，一种金属中心被氧化，另一种被还原。金属和配体的电负性推动反应的进行。如果M<sub>1</sub>电负性弱于M<sub>2</sub>，那么热力学上R基团倾向于和M<sub>2</sub>配位。

### 氧化还原-转移金属化/配体交换

氧化还原-转移金属化/配体交换过程中，两种金属化合物的配体发生互换，分别和另一种金属中心结合。通常的反应式为M<sub>1</sub>-R +
M<sub>2</sub>-X → M<sub>1</sub>-X +
M<sub>2</sub>-R。R配体可以是烷基、芳基、炔基、烯丙基，X配体可以是卤素、拟卤素、烷基或芳基。反应可能经过两个中间步骤。第一步，R和X两个配体[桥接两种金属](https://zh.wikipedia.org/wiki/桥接配体 "wikilink")，形成稳定的[过渡态中间产物](../Page/过渡态.md "wikilink")。

第二个中间产物较为少见，形成的是一种阳离子，R配体和两种金属桥联，X配体成为阴离子。本反应以[协同反应方式进行](https://zh.wikipedia.org/wiki/协同反应 "wikilink")。和氧化还原-转移金属化类似，本反应受到电负性影响，X配体倾向于和高电负性的金属结合，如M<sub>2</sub>电负性高于M<sub>1</sub>，热力学上R配体和X配体更易发生交换。

两大类反应中，M<sub>1</sub>通常为IV B族或者V B族过渡金属，M<sub>2</sub>通常为主族或者III
B族金属。根据电负性，可以判断反应是否能够进行，反应将得到何种产物。例如，因为汞的电负性高于铝，可以判断[二苯基汞](../Page/二苯基汞.md "wikilink")（HgPh<sub>2</sub>）和[铝](../Page/铝.md "wikilink")（Al）反应会得到[汞](../Page/汞.md "wikilink")（Hg）和[三苯基铝](https://zh.wikipedia.org/wiki/三苯基铝 "wikilink")（AlPh<sub>3</sub>）。

## 转移金属化反应的应用

### 交叉偶联反应

转移金属化常被用作[交叉偶联反应](../Page/偶联反应.md "wikilink")[催化循环中的一个步骤](https://zh.wikipedia.org/wiki/催化循环 "wikilink")。某些交叉偶联反应包含转移金属化过程，如[施蒂勒反應](https://zh.wikipedia.org/wiki/施蒂勒反應 "wikilink")、[铃木反应](../Page/铃木反应.md "wikilink")、[薗头偶合反应](https://zh.wikipedia.org/wiki/薗头偶合反应 "wikilink")、[根岸偶联反应](../Page/根岸偶联反应.md "wikilink")。含[钯催化剂是交叉偶联催化剂中较为常用的一类](../Page/钯.md "wikilink")。交叉偶联反应通常的反应式为R'-X
+ M-R → R'-R +
M-X，用于形成碳-碳键。R和R'可以是任何含碳单元。不同类型的交叉偶联反应，所用的M不同。施蒂勒反應使用[锡](../Page/锡.md "wikilink")，[鈴木反應使用](https://zh.wikipedia.org/wiki/鈴木反應 "wikilink")[硼](../Page/硼.md "wikilink")，[薗頭偶合反應使用](https://zh.wikipedia.org/wiki/薗頭偶合反應 "wikilink")[铜](../Page/铜.md "wikilink")，[根岸偶聯反應使用](https://zh.wikipedia.org/wiki/根岸偶聯反應 "wikilink")[锌](../Page/锌.md "wikilink")。钯催化反应的转移金属化步骤中，通过加入R-M化合物，来形成R'-Pd-R化合物。

交叉偶联反应在[合成化学乃至](https://zh.wikipedia.org/wiki/合成化学 "wikilink")[药物化学中都有广泛应用](../Page/药物化学.md "wikilink")。一种抗[肿瘤药物](https://zh.wikipedia.org/wiki/肿瘤 "wikilink")，(±)-*epi*-假白榄酮，就是用施蒂勒反應合成的\[3\]；铃木反应可用于合成另一种抗肿瘤药物，oximidine
II\[4\]；薗頭偶合反應可用于合成抗癌药物恩尿嘧啶\[5\]；根岸偶聯反應可经转移金属化[级联反应合成一种](https://zh.wikipedia.org/wiki/级联反应 "wikilink")[类胡萝卜素](https://zh.wikipedia.org/wiki/类胡萝卜素 "wikilink")——β-胡萝卜素\[6\]。

### 镧系元素

镧系有机金属化合物可用氧化還原-轉移金屬化反应和氧化還原-轉移金屬化/配體交換反应合成。镧系元素的电负性很小。

有机汞化合物如二苯基汞（HgPh<sub>2</sub>），和[有机锂试剂或](../Page/有机锂试剂.md "wikilink")[格氏试剂不同](../Page/格氏试剂.md "wikilink")，在氧化還原-轉移金屬化反应和氧化還原-轉移金屬化/配體交換反应中通常呈[惰性](https://zh.wikipedia.org/wiki/惰性 "wikilink")，使得官能衍生物能够被合成。\[7\]芳基汞化合物常被用来合成镧系有机化合物。在氧化還原-轉移金屬化反應中，Hg(C<sub>6</sub>F<sub>5</sub>)<sub>2</sub>比HgPh<sub>2</sub>是更好的反应试剂，因为前者不需要激活金属原子这一步骤。\[8\]然而，含苯基的镧系化合物比含五氟苯基的镧系化合物有更好的热力学稳定性。HgPh<sub>2</sub>的加入，使镱化合物中的镱元素呈现出两种不同的价态：\[9\]

  -
    Yb(C<sub>10</sub>H<sub>8</sub>)(THF)<sub>2</sub> + HgPh<sub>2</sub>
    → Yb<sup>II</sup>Yb<sup>III</sup>Ph<sub>5</sub>(THF)<sub>4</sub>

在Ln(C<sub>6</sub>F<sub>5</sub>)<sub>2</sub>型化合物中（Ln可以是Yb、Eu或Sm），Ln-C键有较高的反应活性，在氧化還原-轉移金屬化/配體交換反应中有利。质子性基底可作为反应物和Ln(C<sub>6</sub>F<sub>5</sub>)<sub>2</sub>反应：Ln(C<sub>6</sub>F<sub>5</sub>)<sub>2</sub>
+ 2LH → Ln(L)<sub>2</sub> +
2C<sub>6</sub>F<sub>5</sub>H。可通过如下反应避免LH与不稳定的Ln(C<sub>6</sub>F<sub>5</sub>)<sub>2</sub>反应：

  -
    Ln + HgR<sub>2</sub> + 2 LH → Ln(L)<sub>2</sub> + Hg + 2 RH

有机锡化合物在氧化還原-轉移金屬化和氧化還原-轉移金屬化/配體交換反应中也是不活跃的试剂，在很多有机金属反应中都有应用，也可以通过一些反应来合成镧系金属化合物：\[10\]

  -
    Yb + Sn(N(SiMe<sub>3</sub>)<sub>2</sub>)<sub>2</sub> →
    Yb(N(SiMe<sub>3</sub>)<sub>2</sub>)<sub>2</sub> + Sn

### 锕系元素

氧化還原-轉移金屬化反应可以用来合成[锕系金属化合物](https://zh.wikipedia.org/wiki/锕系金属 "wikilink")，如用铀和汞卤化物来合成铀卤化物：U
+ HgX → UX + Hg (X可以是Cl，Br，或I)。\[11\]这一反应还可以用多种汞化合物，令不同的配体与金属进行配合：

  -
    2 U + 3 (C<sub>5</sub>H<sub>5</sub>)<sub>2</sub>Hg +
    HgCl<sub>2</sub> → 2 (C<sub>5</sub>H<sub>5</sub>)<sub>3</sub>UCl + 4
    Hg

### 碱土金属

[碱土金属可用氧化還原](../Page/碱土金属.md "wikilink")-轉移金屬化/配體交換反应合成，借助的方法与合成镧系金属化合物类似。二苯基汞的使用会使反应生成金属汞单质。由于汞对人和环境的毒性，汞单质的操作和处理较有难度。因此，三苯基铋（BiPh<sub>3</sub>）成为了合适的替代品。\[12\]汞和铋的电负性值相近，在氧化還原-轉移金屬化/配體交換反应中的性质也相似。BiPh<sub>3</sub>被用于合成碱土金属[酰胺化合物和碱土金属](../Page/酰胺.md "wikilink")[环戊二烯化合物](../Page/环戊二烯.md "wikilink")。这些合成反应使用HgPh<sub>2</sub>和BiPh<sub>3</sub>的不同在于BiPh<sub>3</sub>需要的反应时间更长。

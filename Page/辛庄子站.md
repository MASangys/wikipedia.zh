**辛庄子站**（**Xinzhuangzi Railway
Station**）是[京包铁路](../Page/京包铁路.md "wikilink")（原[京张铁路](../Page/京张铁路.md "wikilink")）上的一个小火车站，位于[河北省](../Page/河北省.md "wikilink")[张家口市](../Page/张家口市.md "wikilink")[下花园区境内](../Page/下花园区.md "wikilink")。

## 列车时刻

2007年4月18日[中国铁路第六次提速以后](../Page/中国铁路.md "wikilink")，每天只有一对客运列车经停辛庄子站，是[张家口站和](../Page/张家口站.md "wikilink")[北京北站之间互发的](../Page/北京北站.md "wikilink")6411/6412次列车。

|        |          |         |          |                 |         |           |
| ------ | -------- | ------- | -------- | --------------- | ------- | --------- |
| **车次** | **车辆类型** | **始发站** | **始发时间** | **在本站停车时间**     | **终点站** | **终到时间**  |
| 6411次  | 普通慢车     | 北京北站    | 06:25    | 12:13-12:14（当日） | 张家口站    | 13:10（当日） |
| 6412次  | 普通慢车     | 张家口站    | 15:30    | 16:37-16:54（当日） | 北京北站    | 22:13（当日） |

不同社会对待**近親性行為**的态度因年代、[性行为的类型](https://zh.wikipedia.org/wiki/性行为 "wikilink")、[亲属间的血缘关系](https://zh.wikipedia.org/wiki/血亲 "wikilink")、年龄和性别的变化而有很大差别。除法律禁令外，全球大部分的社会都存在某种程度的对近亲婚姻的反对。

有关近親性行為的法律可能包括限制[婚姻](../Page/婚姻.md "wikilink")的自由。当近親性行為的參與者包括儿童时，通常會被认为是一种[儿童性虐待](../Page/兒童性虐待.md "wikilink")\[1\]\[2\]。当近亲婚姻的双方均为成年人且双方均知情同意时通常被称为[合意的近亲性行为](https://zh.wikipedia.org/wiki/近親性交#合意的成年人之間 "wikilink")\[3\]\[4\]。

## 近亲的关系度

有关近亲性行为的法律有时会通过近亲间的关系度而表示。近亲关系度是通过统计共同祖先的辈分计算得来。通常来说，近亲关系密切程度可以总结为下表:

| 近亲关系度 | 关系                                                                                                          | 平均共同的DNA(r) % |
| ----- | ----------------------------------------------------------------------------------------------------------- | ------------- |
|       | 自交                                                                                                          | 99%           |
| 0     | [双胞胎](https://zh.wikipedia.org/wiki/雙胞胎#同卵雙生（identical_twins） "wikilink"); [克隆人](../Page/克隆人.md "wikilink") | 100%\[5\]     |
| 1     | [亲子](https://zh.wikipedia.org/wiki/亲子关系 "wikilink")\[6\]                                                    | 50%           |
| 2     | [兄弟姊妹(同父同母)](https://zh.wikipedia.org/wiki/兄弟姊妹 "wikilink")                                                 | 50%           |
| 2     | 3/4-siblings or sibling-cousins                                                                             | 37.5%         |
| 2     | [祖父母](../Page/祖父母.md "wikilink")                                                                            | 25%           |
| 2     | 半兄弟姊妹(同父异母或同母异父的兄弟姊妹)                                                                                       | 25%           |
| 3     | 姨; 侄女; 外甥                                                                                                   | 25%           |
| 4     | 双表兄弟                                                                                                        | 25%           |
| 3     | 曾祖父母                                                                                                        | 12.5%         |
| 4     | 表兄弟                                                                                                         | 12.5%         |
| 6     | quadruple second cousins                                                                                    | 12.5%         |
| 6     | triple second cousins                                                                                       | 9.38%         |
| 4     | half-first cousins                                                                                          | 6.25%         |
| 5     | first cousins once removed                                                                                  | 6.25%         |
| 6     | double second cousins                                                                                       | 6.25%         |
| 6     | second cousins                                                                                              | 3.13%         |
| 8     | third cousins                                                                                               | 0.78%         |
| 10    | fourth cousins                                                                                              | 0.20%\[7\]    |
|       |                                                                                                             |               |

大部分的法律禁止*r* = 25%或更高的亲缘关系的近亲性行为，而允许*r* = 12.5%或更小的亲缘关系的近亲性行为。在[美国](../Page/美国.md "wikilink")24个洲中，[表兄弟间的婚姻是被禁止的](https://zh.wikipedia.org/wiki/表兄弟 "wikilink")。

相关法律也有可能因[法律关系](../Page/法律关系.md "wikilink")过为亲近而禁止无血缘关系的亲属间的婚姻，例如[收养关系](../Page/收養.md "wikilink")。

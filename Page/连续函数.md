在[数学中](../Page/数学.md "wikilink")，**连续**是[函数的一种属性](../Page/函数.md "wikilink")。直观上来说，连续的函数就是当输入值的变化足够小的时候，输出的变化也会随之足够小的函数。如果输入值的某种微小的变化会产生输出值的一个突然的跳跃甚至无法定义，则这个函数被称为是**不连续**的函数（或者说具有**不连续性**）。

举例来说，考虑描述一棵树的[高度随](../Page/高度.md "wikilink")[时间而变化的函数](../Page/时间.md "wikilink")\(h(t)\)，那么这个函数是连续的（除非树被砍断）。又例如，假设\(T(P)\)表示地球上某一点\(P\)的空气温度，则这个函数也是连续的。事实上，[古典物理学中有一句格言](../Page/古典物理学.md "wikilink")：“自然界中，一切都是连续的。”相比之下，如果\(M(t)\)表述在时间t的时候银行账户上的钱币金额，则这个函数无论在存钱或者取钱的时候都会有跳跃，因此函数\(M(t)\)是不连续的。

## 实值连续函数

最基本也是最常见的连续函数是[定义域为](../Page/定义域.md "wikilink")[实数集的某个子集](../Page/实数.md "wikilink")、取值也是[实数的连续函数](../Page/实数.md "wikilink")。例如前面提到的花的高度，就是属于这一类型。这类函数的连续性可以用[直角坐标系中的](../Page/笛卡尔坐标系.md "wikilink")[图像来表示](../Page/函数图像.md "wikilink")。一个这样的函数是连续的，如果粗略地说，它的图像为一个单一的不破的曲线，并且没有**间断**、**跳跃**或**无限逼近的振荡**。

严格来说，设\(f\)是一个从实数集的子集\(\mathbf{I} \subset \mathbb{R}\)射到\(\mathbf{J} \subset \mathbb{R}\)的函数：\(f : \mathbf{I} \longrightarrow \mathbf{J}\)。\(f\)在\(\mathbf{I}\)中的某个[点](../Page/点.md "wikilink")\(c\)处是连续的当且仅当以下的两个条件满足：

1.  \(f\)在点\(c\)上有定义。
2.  \(c\)是\(\mathbf{I}\)中的一个[聚点](../Page/聚点.md "wikilink")，并且无论自变量\(x\)在\(\mathbf{I}\)中以什么方式接近\(c\)，\(f(x)\)的[极限都存在且等于](../Page/极限.md "wikilink")\(f(c)\)。

我们称函数**到处连续**或**处处连续**，或者简单的称为**连续**，如果它在其定义域中的任意一点处都连续。更一般地，当一个函数在定义域中的某个子集的每一点处都连续时，就说这个函数在这个子集上是连续的。

### \(\varepsilon-\delta\)定义

不用极限的概念，也可以用下面所谓的\(\varepsilon-\delta\)方法来定义实值函数的连续性。

仍然考虑函数\(f : \mathbf{I} \longrightarrow \mathbf{J}\)。假设\(c\)是\(f\)的定义域中的元素。函数\(f\)被称为是在\(c\)点连续当且仅当以下条件成立：

连续性的“\(\varepsilon-\delta\)定义”由[柯西首先给出](../Page/柯西.md "wikilink")。

更直观地，函数\(f\)是连续的当且仅当任意取一個\(\mathbf{J}\)中的点\(f(c)\)的[鄰域](../Page/鄰域.md "wikilink")\(\Omega\)，都可以在其定义域\(\mathbf{I}\)中选取点\(x\)的足够小的鄰域，使得\(x\)的鄰域在函數\(f\)上的映射下都會落在\(f(c)\)的鄰域\(\Omega\)之內。

以上是针对单变量函数（定义域在\(\mathbb{R}\)上的函数）的定义，这个定义在推广到多变量函数时也是成立的。[度量空间以及](../Page/度量空间.md "wikilink")[拓扑空间之间的连续函数定义见下一节](../Page/拓扑空间.md "wikilink")。

### 例子

  - 所有[多项式函数都是连续的](../Page/多项式.md "wikilink")。各类[初等函数](../Page/初等函数.md "wikilink")，如[指数函数](../Page/指数函数.md "wikilink")、[对数函数](../Page/对数函数.md "wikilink")、[平方根函数与](../Page/开平方.md "wikilink")[三角函数在它们的定义域上也是连续的函数](../Page/三角函数.md "wikilink")。
  - [绝对值函数也是连续的](../Page/绝对值.md "wikilink")。
  - 定义在非零实数上的[倒数函数](../Page/倒数.md "wikilink")*f* =
    1/*x*是连续的。但是如果函数的定义域扩张到全体实数，那么无论函数在零点取任何值，扩张后的函数都不是连续的。
  - 非连续函数的一个例子是分段定义的函数。例如定义*f*为：*f*(*x*) = 1如果*x* \> 0，*f*(*x*) = 0如果*x*
    ≤ 0。取ε =
    1/2，不存在*x*=0的δ-邻域使所有*f*(*x*)的值在*f*(0)的ε邻域内。直觉上我们可以将这种不连续点看做函数值的突然跳跃。
  - 另一个不连续函数的例子为[符号函数](../Page/符号函数.md "wikilink")。

### 连续函数的性质

如果两个函数\(f\)和\(g\)是连续的，\(\lambda\)为一个实数，那么\(\displaystyle f + g\)、\(\displaystyle \lambda f\)和\(\displaystyle fg\)都是连续的。所有连续函数的集合构成一个环，也构成一个[向量空间](../Page/向量空间.md "wikilink")（实际上构成一个[代数](../Page/代数_\(环论\).md "wikilink")）。如果对于定义域内的所有\(x\)，都有\(g ( x ) \neq 0\)，那么\(\frac{f}{g}\)也是连续的。

两个连续函数的[复合函数](../Page/复合函数.md "wikilink")\(f \circ g\)也是连续函数。

如果实函数\(f\)在[闭区间](../Page/闭区间.md "wikilink")\([ a, b]\)内连续，且\(k\)是某个\(f ( a )\)和\(f ( b )\)之间的数，那么存在某个\([ a , b ]\)内的\(c\)，使得\(f ( c ) = k\)。这个定理称为[介值定理](../Page/介值定理.md "wikilink")。例如，如果一个小孩在五岁到十岁之间身高从1米增长到了1.5米，那么期间一定有某一个时刻的身高正好是1.3米。

如果\(f\)在\([ a , b ]\)内连续，且\(f ( a )\)和\(f ( b )\)一正一负，则中间一定有某一个点\(c\)，使得\(f ( c ) = 0\)。这是介值定理的一个推论。

如果\(f\)在闭区间\([ a, b]\)内连续，则它一定取得最大值，也就是说，总存在\(c \in [ a, b]\)，使得对于所有的\(x \in [ a, b]\)，有\(f ( c ) \geqslant f(x)\)。同样地，函数也一定有最小值。这个定理称为[极值定理](../Page/极值定理.md "wikilink")。（注意如果函数是定义在开区间\(( a, b)\)内，则它不一定有最大值和最小值，例如定义在开区间(0,1)内的函数\(f ( x ) = \frac{1}{x}\)。）

如果一个函数在定义域中的某个点\(f ( c )\)[可微](../Page/可微.md "wikilink")，则它一定在点\(c\)连续。反过来不成立；连续的函数不一定可微。例如，[绝对值函数在点](../Page/绝对值.md "wikilink")\(c=0\)连续，但不可微。

## 度量空间之间的连续函数

现在考虑从[度量空间](../Page/度量空间.md "wikilink")\((X, d_X)\)到另一个度量空间\((Y, d_Y)\)的函数\(f\)。

\[f\]在\(c\in X\)是连续的，則对任何[实数](../Page/实数.md "wikilink")\(\varepsilon>0\)，存在一个实数\(\delta>0\)使得\(\forall x\in X\)，只要满足\(d_X(x, c)<\delta\)，就满足\(d_Y(f(x), f(c))<\varepsilon\)。

这个定义可以用[序列与](../Page/序列.md "wikilink")[极限的语言重述](../Page/极限.md "wikilink")：

  -
    如果函数\(f\)在点\(c\)连续，則对\(X\)中任何序列\(\{x_n\}\)，只要\(\lim\limits_{n\rightarrow\infty} x_n=c\)，就有\(\lim\limits_{n\rightarrow\infty}f(x_n)=f(c)\)。连续函数将极限变成极限。

后一个条件可以减弱为：

\[f\]在\(c\)点连续，当且仅当对\(X\)中任何序列\(\{x_n\}\)，只要\(\lim\limits_{n\rightarrow\infty} x_n=c\)，就滿足序列\(\{f(x_n)\}\)是一个[柯西序列](../Page/柯西序列.md "wikilink")。连续函数将收敛序列变成柯西序列。

## 拓扑空间之间的连续函数

如上连续函数的定义可以自然地推广到一个[拓扑空间到另一拓扑空间的函数](../Page/拓扑空间.md "wikilink")：函数\(f: X\rightarrow Y\)，这里\(X\)与\(Y\)是拓扑空间是连续的[当且仅当任何](../Page/当且仅当.md "wikilink")[开集](../Page/开集.md "wikilink")\(V\subseteq Y\)的[逆像](../Page/逆像.md "wikilink")\(f^{-1}(V)\)是\(X\)中开集。

## 历史

函数的连续性质在很长时间内被认为是当然的。

第一个比较严格的定义归功于[伯纳德·波尔查诺](../Page/伯纳德·波尔查诺.md "wikilink")\[1\]。他在1817年用德文写下的定义是这样的：函数\(f\)在\(x\)点是连续的，[当且仅当](../Page/当且仅当.md "wikilink")：

  -
    “……若\(h\)足够小时，\(f(x+h)-f(x)\)比任何事先给定的量都小”\[2\]。

然后波尔查诺在证明中值定理时用\(\epsilon\)来表示所谓“事先给定的量”。

六年以后，柯西在1823年也给了一个定义，但此定义还不如波尔查诺前面给出的定义清楚：

  -
    “……\(f(x+h)-f(x)\)的大小随着\(h\)的减小而不确定地减小。……变量（指\(x\)）的一个无穷小的增长会导致函数本身（指\(f(x)\)）的一个无穷小的增长”。

这里的无穷小指的是：一个量的“绝对值不断而无止境地减小以至于小于任何一个事先给定的量”。

现代的\(\epsilon - \delta\)定义只要把波尔查诺在其证明里的写法中“事先给定的量”用\(\epsilon\)来代替就可以了。这个现代定义第一次公开发表在刊物上是1874年由[魏尔斯特拉斯的一个学生](../Page/魏尔斯特拉斯.md "wikilink")[海涅根据魏尔斯特拉斯的讲义写的](../Page/海涅.md "wikilink")。

在[语言学中](../Page/语言学.md "wikilink")，**逆向、相反（converse）**或**关系的[反义词](../Page/反义词.md "wikilink")（relational
antonym）**是指从相反的角度 (如父母/孩子或借/借) 指关系的词对。\[1\]\[2\]

这叫做**逆向关系。**\[3\]
逆向可以理解为一对词，其中一个词表示两个物体之间的关系，而另一个词则表示当物体颠倒时存在相同的关系。\[4\]

它们之间存在“非此即彼”的关系。一个存在只是因为另一个存在。一存在仅仅是因为其他的存在。\[5\]

## 逆向单词列表

  - *拥有* 和 *属于* 是逆向的。"A拥有B"等于"B属于A."
  - *赢得* 和 *失去* 即如果有人的赢得，必须有人失去。
  - *part [分数](../Page/分數.md "wikilink")* 和 whole*整体* (if there is a
    fraction, then there is a whole)。
  - *上面* 和 *下面*
  - *雇主* 和 *雇员*
  - *父母* 和 *孩子*
  - *老师* 和 *学生*
  - *购买* 和 *出售*
  - *东* 和 *西*
  - *丈夫* 和 *妻子*
  - *捕食者* 和 *猎物*
  - *借给* 和 *借用*
  - *进攻* 和 *防御*
  - *奴隶* 和 *主人*

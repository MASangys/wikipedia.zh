**通用語言執行平台**（**Common Language Runtime**，簡稱**CLR**）是[微軟為他們的](https://zh.wikipedia.org/wiki/微軟 "wikilink")[.NET的](https://zh.wikipedia.org/wiki/.NET_Framework "wikilink")[虛擬機器](../Page/虛擬機器.md "wikilink")所選用的名稱。它是微軟對[通用语言架构](../Page/通用语言架构.md "wikilink")（CLI）的實作版本，它定義了一個程式碼執行的環境。CLR執行一種稱為[通用中间语言](../Page/通用中间语言.md "wikilink")的[字节码](https://zh.wikipedia.org/wiki/字节码 "wikilink")，這個是微軟的通用中间语言實作版本。

CLR執行在微軟的[視窗](../Page/視窗.md "wikilink")[作業系統上](https://zh.wikipedia.org/wiki/作業系統 "wikilink")。檢視通用语言架构可以找到該規格的實作版本列表。其中有一些版本是執行在非Windows的作業系統中。

## 主要功能

CLR的主要功能如下

1.  基礎類別庫支援 Base Class Library Support
2.  [内存管理](https://zh.wikipedia.org/wiki/内存管理 "wikilink") Memory Management
3.  线程管理 Thread Management
4.  垃圾回收 Garbage Collection
5.  安全性 Security
6.  类型檢查 Type Checker
7.  异常管理 Exception Manager
8.  [除錯管理](https://zh.wikipedia.org/wiki/除錯 "wikilink") Debug Engine
9.  中間碼(MSIL)到機器碼(Native)編譯
10. 類別裝載 Class Loader

開發人員使用高階程式語言撰寫程式。接下來[編譯器](../Page/編譯器.md "wikilink")將程式碼編譯成微軟的[中繼語言](https://zh.wikipedia.org/wiki/中间语言 "wikilink")(MSIL)。執行的時候CLR會將MSIL碼轉換為作業系統的原生碼（Native code）。CLR內建有[即時編譯](../Page/即時編譯.md "wikilink")）編譯器。

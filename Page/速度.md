**速度**（）是描述物体运动快慢和方向的[物理量](https://zh.wikipedia.org/wiki/物理 "wikilink")。物体在一段时间\(\Delta t\)内的**平均速度**\(\bar{\boldsymbol{v}}\)是它在这段时间里的[位移](../Page/位移.md "wikilink")\(\Delta \boldsymbol{r}\)和[时间间隔之比](../Page/时间.md "wikilink")：

\[\bar{\boldsymbol{v}} = \frac{ \Delta \boldsymbol{r} }{ \Delta t } \,\]

物体在某一时刻的**瞬时速度**\(\boldsymbol{v}\)则是定義為[位置矢量](https://zh.wikipedia.org/wiki/位置矢量 "wikilink")\(\boldsymbol{r}\)隨[時間](https://zh.wikipedia.org/wiki/時間 "wikilink")\(t\)的[變化率](../Page/导数.md "wikilink")：

\[\boldsymbol{v} = \frac{\mathrm{d}\boldsymbol{r}}{\mathrm{d}t} \,\]
物理学中提到物体的速度通常是指其瞬时速度。速度在[国际单位制中的单位是](../Page/国际单位制.md "wikilink")**[米每秒](https://zh.wikipedia.org/wiki/米_\(计量单位\) "wikilink")**，国际符号是**m/s**，中文符号是**米/秒**。[相对论框架中](../Page/相对论.md "wikilink")，物体的速度上限是[光速](../Page/光速.md "wikilink")。

在日常生活中，「[速率](../Page/速率.md "wikilink")」和「速度」混用，但两者在物理学中对应着不同的概念：速率是一个[标量](../Page/标量.md "wikilink")（只有大小、没有方向），它的量纲是[路程除以](https://zh.wikipedia.org/wiki/路程 "wikilink")[时间](../Page/时间.md "wikilink")；速度是一个[向量](../Page/向量.md "wikilink")（有方向），它的量纲是[位移除以](../Page/位移.md "wikilink")[时间](../Page/时间.md "wikilink")\[1\]。举例来说，假如一辆汽车以60公里每小时的速率朝正北方行驶，那么它的速度是一个大小等于60公里每小时、方向指向正北的矢量。物体的瞬时速率等于瞬时速度的大小，而平均速率则不一定等于平均速度的大小。

## 定义

物体在一段时间\(\Delta t\)内的**平均速度**\(\bar{\boldsymbol{v}}\)是它在这段时间里的[位移](../Page/位移.md "wikilink")\(\Delta \boldsymbol{r}\)和时间间隔之比：

\[\bar{\boldsymbol{v}} = \frac{ \Delta \boldsymbol{r} }{ \Delta t } \,\]
其中的位移\(\Delta \boldsymbol{r}\)是矢量，表示物体的位置\(\boldsymbol{r}\)从开始到结束的变化。平均速度的大小一般不是物体在这段时间里的平均速率。比如一个以恒定速率做圆周运动一周的物体，由于最后的位置和初始的位置相同，总位移是0，所以平均速度是0，但平均速率不等于0。平均速度总小于等于平均速率。

平均速度是对物体移动快慢和方向的粗略量度，物体在特定时刻运动的快慢和方向则用瞬时速度表示。物体在某一时刻\(t_0\)的**瞬时速度**\(\boldsymbol{v}(t_0)\)定義為物体在\(t_0\)左右的很小一段时间\(\Delta t\)中的平均速度在\(\Delta t\)趋向于0时的极限。用数学的语言来说，就是位置矢量\(\boldsymbol{r}\)在\(t_0\)时刻隨時間\(t\)的變化率，也就是对时间的[导数](../Page/导数.md "wikilink")：

\[\boldsymbol{v}(t_0) =\lim_{\Delta t \to 0}{{ \boldsymbol{r} (t_0+\Delta t)- \boldsymbol{r} (t_0)} \over \Delta t}= \left.\frac{\mathrm{d}\boldsymbol{r}}{\mathrm{d}t}\right|_{t=t_0} \,\]
瞬时速度的大小（矢量的模长）等于瞬时速率；瞬时速度的方向则是物体运动[曲线的](../Page/曲线.md "wikilink")[切线方向](https://zh.wikipedia.org/wiki/切线 "wikilink")：
\(\boldsymbol{v} = v \boldsymbol{e}_{T}.\)

其中\(v\)是物体在某一点的瞬时速率，\(\boldsymbol{e}_{T}\)是物体运动轨迹曲线在这一点的切向单位矢量。

## 例子

### 直线运动

直线运动是指物体（通常简化成[质点](https://zh.wikipedia.org/wiki/质点 "wikilink")）沿着直线运动，當中並無方向的變化。通常会为这个直线指定一个正方向和一个原点，以方便描述。如果物体的速度方向与正方向相同，则记其速度为其速率（正数），反之则记其速度为速率的[相反数](https://zh.wikipedia.org/wiki/相反数 "wikilink")（负数）。这种记法下，物体的位置、位移和速度都可以用[实数来表示](../Page/实数.md "wikilink")。假设物体在初始时刻\(t = 0\)的位置是\(x_0\)，速度为定值\(v\)，那么称其做**[匀速直线运动](https://zh.wikipedia.org/wiki/匀速直线运动 "wikilink")**。经过时间\(\Delta t\)以后，物体的位置\(x_f\)是：\(x_f = x_0 + v\Delta t\)，物体的位移是\(\Delta \boldsymbol{r} = v\Delta t\)。

如果物体的速度随时间均匀改变：\(v(t) = v_0 + at\)，那么称之为**匀变速直线运动**。经过时间\(\Delta t\)以后，物体的位置\(x_f\)是：\(x_f = x_0 +  v_0  \Delta t + \frac12 a\Delta t^2\)，物体的位移是\(\Delta \boldsymbol{r} =  v_0  \Delta t + \frac12 a\Delta t^2\)。

### 圆周运动

圆周运动是指物体沿圆周做运动。这时候物体的速度是沿圆周切线的方向的向量。当速度大小恒定时，但方向隨時間而變動，即非等速度運動，称为**[匀速圆周运动](https://zh.wikipedia.org/wiki/匀速圆周运动 "wikilink")**（其中的“匀速”指“匀速率”）。圆周运动的物体，其平均速度的大小和平均速率是不同的。假设物体以速率\(v\)做匀速圆周运动，那么它的平均速率永远是\(v\)，而它的平均速度的大小则是终点和起点构成的弦长度除以间隔的时间。

一般情况下，物体的位移是速度对时间的[积分](../Page/积分.md "wikilink")：\(\Delta \boldsymbol{r} = \int_{0}^{\Delta t}\boldsymbol{v}(t)\mathrm{d}t\)。如果物体在初始时刻\(t = 0\)的位置是\(\boldsymbol{x}_0\)，那么经过时间\(\Delta t\)以后，物体的位置\(\boldsymbol{x}_f\)是：\(\boldsymbol{x}_f = \boldsymbol{x}_0 + \int_{0}^{\Delta t}\boldsymbol{v}(t)\mathrm{d}t \,\)，平均速度是：\(\bar{\boldsymbol{v}} = \frac{1}{\Delta t}\int_{0}^{\Delta t}\boldsymbol{v}(t)\mathrm{d}t\)

## 速度的分解

研究不同的物理问题时，通常会依据研究对象的特性，使用不同的坐标系。在不同的坐标系下，速度有不同的分解方式，選擇座標的方式通常以分解向量的方便性為主。

### 直角坐标系

直角座標系是我們最常用到也最直觀的座標系統如果在三维空间中架设[直角坐标系O](https://zh.wikipedia.org/wiki/直角坐标系 "wikilink")-xyz，那么一个物体的位置\(\boldsymbol{r}\)可以表示成：

\[\boldsymbol{r} =x \boldsymbol{e}_{x} + y \boldsymbol{e}_{y} + z \boldsymbol{e}_{z}\]。

其中\(\boldsymbol{e}_{x}, \boldsymbol{e}_{y}, \boldsymbol{e}_{z}\)分别是x轴、y轴、z轴方向上的单位矢量。物体的速度等于位移对时间的导数：

\[\boldsymbol{v} = \frac{\mathrm{d} \boldsymbol{r} }{\mathrm{d}t} = \frac{\mathrm{d} x }{\mathrm{d}t}\boldsymbol{e}_{x} + \frac{\mathrm{d} y }{\mathrm{d}t} \boldsymbol{e}_{y} +  \frac{\mathrm{d} z }{\mathrm{d}t}  \boldsymbol{e}_{z} =  v_x  \boldsymbol{e}_{x} + v_y \boldsymbol{e}_{y} + v_z\boldsymbol{e}_{z}\]。
其中\(v_{x}, v_{y}, v_{z}\)是物体速度在三个坐标轴方向上的分量。速度的大小为：\(|\boldsymbol{v} | = \sqrt{v_{x}^2+v_{y}^2+v_{z}^2}\)。

### 极坐标

平面中运动的物体，其速度也可以采用[极坐标来表示](https://zh.wikipedia.org/wiki/极坐标 "wikilink")(依照方便來選擇座標系分解)。极坐标下的速度可以分解为两个部分：[径向速度](https://zh.wikipedia.org/wiki/径向速度 "wikilink")，即物体到原点的距离的变化率，以及角速度，即物体位置的[幅角随时间的变化率](../Page/幅角.md "wikilink")。假设物体的位置用极坐标表示为\((r, \theta)\)，定义其径向单位矢量和横向单位矢量为\(\boldsymbol{e}_{r}, \boldsymbol{e}_{\theta}\)，那么物体的位置矢量可以表示成：\(\boldsymbol{r} =r \boldsymbol{e}_{r}\)。

其速度是位移对时间的导数：

\[\boldsymbol{v} = \frac{\mathrm{d} \boldsymbol{r} }{\mathrm{d}t} = \frac{\mathrm{d}}{\mathrm{d}t} \left( r \boldsymbol{e}_{r} \right) = \frac{\mathrm{d}r}{\mathrm{d}t}  \boldsymbol{e}_{r} + r  \frac{\mathrm{d} \boldsymbol{e}_{r} }{\mathrm{d}t} = \frac{\mathrm{d}r}{\mathrm{d}t}  \boldsymbol{e}_{r} + r  \frac{\mathrm{d} \theta }{\mathrm{d}t}  \boldsymbol{e}_{\theta}\]。
其中\(\frac{\mathrm{d}r}{\mathrm{d}t}  \boldsymbol{e}_{r}\)称为径向速度，\(r  \frac{\mathrm{d} \theta }{\mathrm{d}t} \boldsymbol{e}_{\theta}\)称为横向速度。速度的极坐标描述是依赖于物体位置的描述，因为\(\boldsymbol{e}_{r}, \boldsymbol{e}_{\theta}\)都是随着物体位置的改变而改变的。物体做圆周运动时，\(r\)是常数，所以径向速度为零，即速度永远沿着圆周切线方向；而横向速度等于\(r  \frac{\mathrm{d} \theta }{\mathrm{d}t}  \boldsymbol{e}_{\theta}\)，即半径乘以角速度。

如果物体在三维空间中运动，可以加入纵坐标轴z，建立[圆柱坐标系](https://zh.wikipedia.org/wiki/圆柱坐标系 "wikilink")：\((\rho,\theta,z)\)，物体的位置写作：\(\boldsymbol{r} =\rho \boldsymbol{e}_{ \rho }+  z \boldsymbol{e}_{z}\)，其速度为：

\[\boldsymbol{v} = \frac{\mathrm{d} \rho }{\mathrm{d}t}  \boldsymbol{e}_{ \rho } + \rho \frac{\mathrm{d} \theta }{\mathrm{d}t}  \boldsymbol{e}_{\theta} +  \frac{\mathrm{d} z }{\mathrm{d}t}  \boldsymbol{e}_{z}\]。

### 球坐标

[3D_Spherical.svg](https://zh.wikipedia.org/wiki/File:3D_Spherical.svg "fig:3D_Spherical.svg")
三维空间中运动的物体，还可以用[球坐标系来研究](https://zh.wikipedia.org/wiki/球坐标系 "wikilink")。球坐标系中，物体的速度可以分解成三个部分。除了物体的径向速度外，还有方位角速度和顶角速度，即方位角\(\varphi\)和顶角\(\theta\)的变化率。假设物体的位置用球坐标表示为\((r , \theta , \varphi )\)，定义它的基矢：\(\boldsymbol{e}_{r}, \boldsymbol{e}_{\theta}, \boldsymbol{e}_{\varphi}\)，则物体的位置可以写成：\(\boldsymbol{r} =r \boldsymbol{e}_{r}.\)

其速度是位移对时间的导数：

\[\boldsymbol{v} = \frac{\mathrm{d} \boldsymbol{r} }{\mathrm{d}t} = \frac{\mathrm{d}}{\mathrm{d}t} \left( r \boldsymbol{e}_{r} \right) = \frac{\mathrm{d}r}{\mathrm{d}t}  \boldsymbol{e}_{r} + r  \frac{\mathrm{d} \boldsymbol{e}_{r} }{\mathrm{d}t} = \frac{\mathrm{d}r}{\mathrm{d}t}  \boldsymbol{e}_{r} + r \frac{\mathrm{d} \theta }{\mathrm{d}t}  \boldsymbol{e}_{\theta} + r \frac{\mathrm{d} \varphi}{\mathrm{d}t} \sin \theta \boldsymbol{e}_{\varphi}\]

### 自然坐标

如果已知物体运动轨迹，可以使用自然坐标系来描述物体的运动情况。与其他坐标系的不同是，自然坐标系中基矢的选择与物体的运动情况相关。这时候定义的基矢是：\(\boldsymbol{e}_{T}, \boldsymbol{e}_{N}, \boldsymbol{e}_{B}\)，分别是[切向单位矢量](https://zh.wikipedia.org/wiki/切向量 "wikilink")、[主法向单位矢量](https://zh.wikipedia.org/wiki/法向量 "wikilink")（轨迹曲线在该点的密切圆所在的平面上的法向量）和副法向单位矢量（与主法向和切向量垂直的法向矢量）。而物体的速度是沿切向的，所以速度的表达式是：

\[\boldsymbol{v} = v \boldsymbol{e}_{T}\]。 其中v就是瞬时速率。这是最简洁的坐标表达方式。

## 相对速度

相对速度是指一个物体相对另一个物体运动的速度。具体来说，假设在某个[参考系中](../Page/参考系.md "wikilink")，两个物体A和B的速度分别是\(\boldsymbol{v}_A\)和\(\boldsymbol{v}_B\)，那么A相对于B的速度\(\boldsymbol{v}_{A/B}\)就是A在B静止的参考系中的速度。经典物理学中（非[相对论框架](../Page/相对论.md "wikilink")），\(\boldsymbol{v}_{A/B} = \boldsymbol{v}_A - \boldsymbol{v}_B \,\)，而B相对于A的速度\(\boldsymbol{v}_{B/A}\)是：\(\boldsymbol{v}_{B/A} =\boldsymbol{v}_B - \boldsymbol{v}_A\)。，透過向量的轉換，利用相對速度來解決問題可以將問題簡化許多。

经典物理学中的相对速度变换公式是坐标系做[伽利略变换的结果](../Page/伽利略变换.md "wikilink")。相对论框架中，需要用[洛伦兹变换代替伽利略变换](../Page/洛伦兹变换.md "wikilink")，因此相对速度的变换公式也不同。假设物体A的速度是\(\boldsymbol{v} = (v_x, v_y, v_z)\)，物体B的速度是\(\boldsymbol{u} = (u_x, 0, 0)\)，那么物体A相对于物体B的速度是：

\[\boldsymbol{v}_{A/B} = \left(\frac{v_x - u_x}{1 - \frac{v_x u_x}{c^2}}, \frac{v_y}{1 - \frac{v_x u_x}{c^2}}\sqrt{1 - \frac{u_x^2}{c^2} },  \frac{v_z}{1 - \frac{v_x u_x}{c^2}}\sqrt{1 - \frac{u_x^2}{c^2} } \right)\]

例如两艘飞船各自以光速\(c\)的一半朝着相反的方向作直线运动，那么某一艘飞船上的人观察到另一艘飞船的相对速度就是：

\[v = \frac{\frac{c}{2} -(-\frac{c}{2})}{1 - \frac{(\frac{c}{2})(-\frac{c}{2})}{c^2}} = \frac{c}{1+\frac{1}{4}} = \frac45 c\]。
相对速度是0.8倍光速，而不是光速。

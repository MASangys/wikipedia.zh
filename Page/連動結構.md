**連動結構**（）是一種常見於[非洲](../Page/非洲.md "wikilink")、[亞洲和](https://zh.wikipedia.org/wiki/亞洲 "wikilink")[新幾內亞](../Page/新幾內亞.md "wikilink")語言的語法現象，和[從屬結構](https://zh.wikipedia.org/wiki/從屬結構 "wikilink")([subordination](https://zh.wikipedia.org/wiki/:en:Subordination_\(linguistics\) "wikilink"))這種其中一個句子從屬於另一個句子的結構不同的是，在連動結構中，兩個或多個動詞在同一個句子中並列，沒有任何一方從屬於另一方。\[1\]

## 連動結構示例

以下範例來自[努培語](https://zh.wikipedia.org/wiki/努培語 "wikilink")([Nupe language](https://zh.wikipedia.org/wiki/:en:Nupe_language "wikilink"))

(1)

|                                                             |
| ----------------------------------------------------------- |
| Asakura bé lá èbi.                                          |
| Asakura                                                     |
| 朝倉                                                          |
| [朝倉來拿刀](https://zh.wikipedia.org/wiki/朝倉涼子 "wikilink")\[2\] |

連動結構有以下特點：

(i) 動詞串列中的每個動詞其主詞都相同

(ii) 不同動詞的主詞[一致常交叉指涉](../Page/一致_\(語法\).md "wikilink")(cross-reference)。

以下是來自[巴瑞語](https://zh.wikipedia.org/wiki/巴瑞語 "wikilink")([Baré language](https://zh.wikipedia.org/wiki/:en:Baré_language "wikilink"))的例子： (2)

|                     |
| ------------------- |
| nu-takasã nu-dúmaka |
| nu-takasã           |
| 一人稱單數-騙             |
| 我在裝睡\[3\]           |

在其他例子中，句子只有一個主詞，但由兩個動詞共享，以下為來自[約魯巴語的範例](https://zh.wikipedia.org/wiki/約魯巴語 "wikilink")：

(3)

|             |
| ----------- |
| ó mú ìwé wá |
| ó           |
| 他           |
| 他把書帶了\[4\]  |

兩個動詞的主詞都理解為第三人稱單數

(iii) 唯一能出現在動詞序列中的各動詞間的句子成份(constituent)，是其中某一個動詞的受詞，且僅有一些具有連動結構的語言能如此，如例(3)所示一般。

(iv)整個結構僅有一個否定詞，如以下巴瑞語的範例所示：

(4)

|                                          |
| ---------------------------------------- |
| hena nihiwawaka nu-tšereka nu-yaka-u abi |
| hena                                     |
| 不                                        |
| 我沒去和我媽說\[5\]                             |

(v) 連動結構中每個成份的時態/語體/語氣之類等不彼此獨立。這種標記或相同，或相關，或單由所有動詞共享的標記。(就如其共享主詞一般。)\[6\]

## 和复合动词的不同處

連動結構通常和[复合动词](https://zh.wikipedia.org/wiki/复合动词 "wikilink")([compound verb](https://zh.wikipedia.org/wiki/:en:Compound_verb "wikilink"))有所區別：

  - 連動結構將數個動作(這些動作經常但不總是接續發生的)給「堆疊」在一個單一的句子中。像埃維語的(意即「轉身離開」，直譯為「轉、離開」)和[印地語的](https://zh.wikipedia.org/wiki/印地語 "wikilink")\\ फ़ोन उठा-कर कहा (*fon uṭhā-kar kahā*)(意即「拿起電話說」，直譯為「電話、拾起-CONJPART、說.過去時)等皆為其例。在漢語和許多東南亞語言中，第一個[及物動詞](../Page/及物動詞.md "wikilink")的直接受詞常會是第二個動詞的主詞，像在「[胡安咬死了](../Page/胡安_\(中土大陸\).md "wikilink")[卡黑洛斯](../Page/卡黑洛斯.md "wikilink")」這句中，「[卡黑洛斯](../Page/卡黑洛斯.md "wikilink")」是「咬」這個動詞的直接受詞，也同時是「死」這個動詞的主詞，若把此句中的每個字都換成印地語相對應的字的話，則聽者會認為死的是「[胡安](../Page/胡安_\(中土大陸\).md "wikilink")」，而非「[卡黑洛斯](../Page/卡黑洛斯.md "wikilink")」。

<!-- end list -->

  - 在复合动词(Compound verb，又稱「複謂語(complex predicate)」) 中，第一個動詞提供主要的語義和[論元](https://zh.wikipedia.org/wiki/論元 "wikilink")(argument)的結構，而第二個動詞，也就是常稱為「[功能动词](../Page/功能动词.md "wikilink")」的動詞，則提供輔助的說明並帶有詞形變化。其主動詞經常會處於連結[分詞](https://zh.wikipedia.org/wiki/分詞 "wikilink")(conjunctive participle)形(或如[印地語與](https://zh.wikipedia.org/wiki/印地語 "wikilink")[旁遮普語](../Page/旁遮普語.md "wikilink")般，處於沒有任何詞形變化詞綴的詞幹形)。

像例如印地語的सत्तू खा लिया (*sattū khā-liyā*)(意即「吃光sattu飯」，直譯為「炒穀、吃-拿了」)和 बच्चे.को खा-डाला (*bacce.ko khā-ḍālā*)(意即「吞食小孩」(暴力或不想要的動作)，直譯為「小孩、吃-丟了」)皆為其例，在上述印地語的範例中，खा (*khā*)是主動詞，而 लिया (*liyā*)(意即「拿了」) 和डाला (*ḍālā*)(意即「丟了」)則是其功能動詞。

連動結構和复合动词的不同處在於前者用不只一個動詞來表達不只一個動作，而後者用不只一個動詞來表達一個動作。复合动词在印地語和旁遮普語等一些[印度-雅利安语支](../Page/印度-雅利安语支.md "wikilink")的語言當中非常常見，它們在其他的一些印度-雅利安语支較少見，同時复合动词亦可見於[达罗毗荼语系](../Page/达罗毗荼语系.md "wikilink")的語言、[突厥語族的語言](https://zh.wikipedia.org/wiki/突厥語族 "wikilink")、[韓語](https://zh.wikipedia.org/wiki/韓語 "wikilink")、[日語](https://zh.wikipedia.org/wiki/日語 "wikilink")、一些[藏缅语族的語言](https://zh.wikipedia.org/wiki/藏缅语族 "wikilink")、一些[東北高加索語支](https://zh.wikipedia.org/wiki/東北高加索語支 "wikilink")(Northeast Caucasian languages)的語言和[奇曲瓦語](https://zh.wikipedia.org/wiki/奇曲瓦語 "wikilink")([Quichua](https://zh.wikipedia.org/wiki/:en:Quichua "wikilink")，一種克丘亞語系的語言)中；連動結構可見於上述所有的語言，且可見於[漢語](https://zh.wikipedia.org/wiki/漢語 "wikilink")、[孟高棉语族](../Page/孟高棉语族.md "wikilink")的一些語言、[壯侗語族](../Page/壯侗語族.md "wikilink")的一些語言、[克瓦語族](https://zh.wikipedia.org/wiki/克瓦語族 "wikilink")([Kwa languages](https://zh.wikipedia.org/wiki/:en:Kwa_languages "wikilink"))的一些語言，和許多的皮欽語和克里奧爾語中。(可見[V·S·奈波爾](../Page/V·S·奈波爾.md "wikilink")所用的千里達話 的序列動詞如insure-and-burn和choke-'n'-rob等)

## 注意

儘管這現象有些常見，目前對於連動結構好的解析方法沒有一定的共識，而這也是目前語法學家所爭執的事物之一。

就如塔勒曼(Tallerman) (1998)所指出的，連動結構對英語使用者而言，並非完全陌生的，它出現於一些自早期現代英語留存下來的用法中，像*Let's go eat.*(意即「讓我們去吃」)即其一例。\[7\]

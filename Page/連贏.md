連赢為賽馬術語，是[賽馬](../Page/賽馬.md "wikilink")[博彩的一個重要投注項目](https://zh.wikipedia.org/wiki/博彩 "wikilink")，在[香港](../Page/香港.md "wikilink")、[澳門](../Page/澳門.md "wikilink")、[新加坡](../Page/新加坡.md "wikilink")、[英國](https://zh.wikipedia.org/wiki/英國 "wikilink")、[美國](https://zh.wikipedia.org/wiki/美國 "wikilink")、[澳洲](https://zh.wikipedia.org/wiki/澳洲 "wikilink")、[日本](../Page/日本.md "wikilink")、[韓國等地方均有接受投注](https://zh.wikipedia.org/wiki/韓國 "wikilink")，澳門及香港粵語稱為「連贏」，投注者需選中一場賽事中的第一名及第二名，毋須順序。

## 投注金額

以澳門及香港為例，連贏每注最少澳門幣或港幣10元；而新加坡為每注最少2新加坡元。

## 投注方法

|                |                                                                                                                                  |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| **投-{注}-種類**   | **投-{注}-方法**                                                                                                                     |
| 「單式」投-{注}-     | 從自選任何馬匹編號中選出2匹不同馬匹編號。                                                                                                            |
| 「複式」投-{注}-     | 假設您的心水馬為7號、9號、11號，複式互串會將所選馬匹組成所有可能的組合，即分別為7+9、7+11 及 9+11，總共三注 ，好處是提高命中機會。                                                       |
| 「馬-{膽}-」投-{注}- | 當您對一匹馬有很大信心能跑入前兩名，您可選擇以該匹馬作「馬膽」，其他選擇則作為「配腳」。 假設重心馬為7號，以7號胆拖9號、11號，即分別為7+9 及 7+11，總共兩注，只要其中一個馬匹組合同時跑入前兩名，便為中彩，相比起「複式互串」好處是更能節省注項。 |

## 中獎方法

選中一場賽事中的第1名及第2名馬匹，毋須順序。

## 香港賽馬史上最冷連贏派彩

紀錄於2017年3月5日第九場產生，由39倍的「天合」及99倍的「中游新星」包辦首兩名，每港幣$10派$23,863。\[1\]

舊紀錄是2013年2月17日「魅力知己」搭「總統之光」所構成的連贏位，派彩$18,948。

**-{zh-cn:远程桌面协议;
zh-tw:遠端桌面協定;}-**（，缩写：）是一個多通道（multi-channel）的協定，讓使用者（用戶端或稱「本地電腦」）連上提供[微軟終端機服務的電腦](https://zh.wikipedia.org/wiki/微軟 "wikilink")（伺服器端或稱「遠端電腦」）。大部分的[Windows都有用戶端所需軟體](https://zh.wikipedia.org/wiki/Windows "wikilink")。其他作業系統也有這些用戶端軟體，例如[Linux](../Page/Linux.md "wikilink")、[FreeBSD](../Page/FreeBSD.md "wikilink")、[Mac
OS
X](https://zh.wikipedia.org/wiki/Mac_OS_X "wikilink")。伺服端電腦方面，則-{zh-cn:监听;
zh-tw:聽取;}-送到[TCP](https://zh.wikipedia.org/wiki/TCP "wikilink")
[3389埠的資料](https://zh.wikipedia.org/wiki/TCP/UDP端口列表#1024到49151號端口 "wikilink")。

## 功能

  - 多種顯示支援，包括8，15，16，24，32位元色。
  - 128位元加密，使用[RC4加密演算法](../Page/RC4.md "wikilink")。（此為內定的加密方式；比較舊版的用戶端可能使用較弱的加密強度）
  - 支援[TLS](https://zh.wikipedia.org/wiki/TLS "wikilink")（Transport
    Layer
    Security，前身為[SSL](https://zh.wikipedia.org/wiki/SSL "wikilink")）。
  - 聲音轉向（redirection）支援，使用者可以在遠端電腦執行有聲音的應用程式，但是將聲音導引至用戶端電腦來聽。
  - 檔案系統轉向支援，使用者可在使用遠端電腦的過程中，取用本地（用戶端）電腦上的檔案系統。
  - 印表機轉向支援，在使用遠端電腦時，可以使用本地（用戶端）電腦上的印表機輸出，包括直接連在用戶端電腦的印表機或網路共享印表機。
  - 通訊埠轉向支援，遠端電腦上的應用程式可以使用本地（用戶端）電腦上的-{zh-cn:[串行端口](../Page/串行端口.md "wikilink");
    zh-tw:序列埠;}-或-{zh-cn:[并行端口](../Page/并行端口.md "wikilink");
    zh-tw:平行埠;}-。
  - Windows的[剪貼簿資料可以在遠端及本地電腦之間互通](https://zh.wikipedia.org/wiki/剪貼簿 "wikilink")。

## 優點

與遠端桌面協定相容的用戶端可在多種作業系統上執行，許多[Linux系統上甚至將RDP用戶端功能列為核心功能之一](../Page/Linux.md "wikilink")。此外，使用者也不一定要有寬頻網路才能連上他們的遠端電腦桌面，RDP即使在56K撥接網路下，都還可以提供每秒更新五到六個畫面的效果。

## 版本

RDP的設計建構於[國際電信聯盟T](https://zh.wikipedia.org/wiki/國際電信聯盟 "wikilink").share協定（又稱為T.128），發展以來各個版本大致為

  - 4.0版：這是第一個版本。隨同終端機服務（Terminal Services）出現在[Windows NT 4.0
    Server](../Page/Windows_NT_4.0.md "wikilink")、Terminal Server
    Edition。
  - 5.0版：由[Windows 2000
    Server引入](../Page/Windows_2000.md "wikilink")。加入了許多新功能，包括列印到用戶端印表機，針對網路頻寬使用的改進等等。
  - 5.1版：由[Windows XP Professional](../Page/Windows_XP.md "wikilink")（XP
    Home不支援伺服端功能）引入。支援24位元顏色顯示及聲音的支援。该版本的客户端可以是Windows 2000，Windows
    9x，Windows NT 4.0\[1\]。
  - 5.2版：由[Windows Server
    2003引入](../Page/Windows_Server_2003.md "wikilink")，包括了console
    mode connections，session directory，以及用戶端資源的取用。[Windows
    CE](https://zh.wikipedia.org/wiki/Windows_CE "wikilink")
    5.0及6.0均有這個版本的用戶端部分，但Windows CE沒有作為伺服端的功能。该版本内置于Windows
    XP Professional x64 Edition和Windows Server 2003 x64或x86版中。
  - 6.0版：由[Windows
    Vista引入](../Page/Windows_Vista.md "wikilink")。该版本的客户端可以是Windows
    XP SP2，Windows Server 2003 SP1/SP2（x64或x86版），Windows XP Professional
    x64 Edition。包括許多重大改進，最值得注意的是可以從遠端使用單一應用程式，而非整個桌面；以及32位元顏色顯示的支援。
  - 6.1版：由[Windows Server
    2008和](../Page/Windows_Server_2008.md "wikilink")[Windows
    Vista](../Page/Windows_Vista.md "wikilink") SP1引入。该版本的客户端可以是Windows
    XP SP3。Windows XP SP2须安裝KB952155\[2\]。
  - 7.0版：由[Windows Server 2008
    R2或](../Page/Windows_Server_2008_R2.md "wikilink")[Windows
    7引入](https://zh.wikipedia.org/wiki/Windows_7 "wikilink")\[3\]。该版本的客户端可以是Windows
    XP SP3、Windows Vista SP1/SP2，但须安装KB969084\[4\]。该版本的客户端不支持Windows
    Server 2003 x86和Windows Server 2003 / Windows XP Professional x64
    editions。
  - 7.1版：必須要有[Windows
    7](https://zh.wikipedia.org/wiki/Windows_7 "wikilink") SP1或[Windows
    Server 2008 R2](../Page/Windows_Server_2008_R2.md "wikilink")
    SP1，主要增加了的功能。
  - 8.0版：由[Windows
    8或者](https://zh.wikipedia.org/wiki/Windows_8 "wikilink")[Windows
    Server 2012引入](../Page/Windows_Server_2012.md "wikilink")。Windows 7
    SP1和Windows Server 2008 R2 SP1要支持该协议须先安装KB2574819\[5\]，再安装KB2592687。
  - 8.1版：由Windows 8.1和Windows Server 2012 R2引入。Windows 7 SP1和Windows
    Server 2008 R2
    SP1要支持该协议须先安装KB2574819、KB2857650\[6\]，再安装KB2830477。之后最好再安装KB2913751。

## RDP 6.0的新功能

  - 遠端應用程式：用戶端電腦上特定檔案格式的相關應用程式可以在遠端電腦上。
  - 無縫隙視窗：在用戶端電腦上可以直接執行在遠端電腦上的應用程式。（不用先連上整個遠端電腦的桌面）
  - 終端伺服器閘道器：可以使用front-end IIS server，經由https，來連接back-end Terminal
    Services servers。
  - 支援遠端[Windows Aero畫面](../Page/Windows_Aero.md "wikilink")
  - 支援遠端的[Windows Presentation
    Foundation應用程式](../Page/Windows_Presentation_Foundation.md "wikilink")：相容的用戶端如果支援[.NET
    Framework
    3.0就能在本地電腦上顯示完整的Windows](../Page/.NET_Framework_3.0.md "wikilink")
    Presentation Foundation效果。
  - \-{zh-cn:外部设备; zh-tw:周邊裝置;}--{zh-cn:重定向;
    zh-tw:轉向;}-經過重新設計，可以使用更多樣的設備。
  - 經由[WMI](https://zh.wikipedia.org/wiki/WMI "wikilink")（Windows
    Management Instrumentation），所有終端服務都可以configure，都可scriptable。
  - 針對用戶端做頻寬調整。
  - 支援32位元顯示。
  - 支援雙顯示器，包括[寬螢幕顯示](../Page/显示器.md "wikilink")。
  - 支援IPv6协议，可以工作在纯IPv6的TCP/IP网络环境中。

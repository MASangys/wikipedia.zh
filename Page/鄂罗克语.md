**鄂罗克语**是[通古斯语的一支](https://zh.wikipedia.org/wiki/通古斯语 "wikilink")，为[鄂罗克人](../Page/鄂罗克人.md "wikilink")的民族语言。使用者分布于俄罗斯的[萨哈林州](../Page/萨哈林州.md "wikilink")[波罗奈斯克行政区和](https://zh.wikipedia.org/wiki/波罗奈斯克 "wikilink")[诺格利基行政区](https://zh.wikipedia.org/wiki/诺格利基 "wikilink")。为一种[濒危语言](https://zh.wikipedia.org/wiki/濒危语言 "wikilink")。

根据[2002年俄羅斯人口普查](../Page/2002年俄羅斯人口普查.md "wikilink")，俄罗斯境内有346名鄂罗克人，其中会说鄂罗克语的有64名。至2010年的普查时，只有47名鄂罗克人会说鄂罗克语了。鄂罗克人还生活于日本的[北海道](../Page/北海道.md "wikilink")，但会说鄂罗克语的人数不明，且非常少。\[1\]

## 语音

鄂罗克语的音节构造基本为(C)V(C)(C)结构。有[元音和谐](https://zh.wikipedia.org/wiki/元音和谐 "wikilink")，包括前后和谐和唇音和谐。

### 元音

鄂罗克语有15个元音音位，其中短元音7个（ɐ, ə, i, ɛ, ɔ, u, ʌ），长元音8个（ɐː, əː, eː, iː, ɛː, ɔː, uː, ʌː）。

| 开闭   | 前后        |
| ---- | --------- |
| 前元音  | 央元音       |
| 闭元音  | i, iː     |
| 半开元音 | ɛ, ɛː, eː |
| 开元音  |           |

### 辅音

鄂罗克语有18个辅音音位。从俄语借词中引进音位\[z\]，则音位增至19个。再加上在元音间才出现的音位变体\[q\]、\[ɢ\]和\[ɣ\]，则辅音增至22个。

## 语法

### 名词

鄂罗克语的名词[语法范畴](../Page/语法范畴.md "wikilink")包括[数](../Page/数_\(语法\).md "wikilink")、[格](../Page/格_\(语法\).md "wikilink")、[人称](../Page/人称.md "wikilink")等。有生命/无生命的对立不明显。名词主要分为两类：人及非人。

#### 数

名词分单数及复数。使用后缀 -л 或 -сал、-сэл、-сали、-сэли来表示复数。如：

  - **нари** — *人*
  - **нарил \~ нарисал** — *人（复数）*
  - **апу(н)** — *帽子*
  - **апусал** — *帽子（复数）*

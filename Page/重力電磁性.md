引力磁性在1893年由[奧利弗·黑維塞](../Page/奧利弗·黑維塞.md "wikilink")對[牛頓力學拓展時發展出來](https://zh.wikipedia.org/wiki/牛頓力學 "wikilink")，早於廣義相對論出現之前。廣義相對論建立後，以其為基礎發展出來的引力磁性理論，只與1893年的版本相差幾個變數，基礎框架仍可沿用。

## 背景

廣義相對論所描述的重力在弱場極限的情況下，可改寫形式為一特別的場，其中有兩個組成類似於電磁學中的電場與磁場。因此透過類比，將其稱為重力電場與重力磁場。重力磁性的表現為跟[速度](../Page/速度.md "wikilink")相依的[加速度](../Page/加速度.md "wikilink")，使得在旋轉巨大物體旁移動的小物體所感受到的加速度與純牛頓重力學（重力電性）預測的加速度有所偏差。細節包括自由落體產生旋轉，以及已經自轉的物理發生[進動](../Page/進動.md "wikilink")，這些現象都是[廣義相對論的實驗驗證內容](https://zh.wikipedia.org/wiki/廣義相對論的實驗驗證 "wikilink")。

間接檢驗引力磁性效應曾經使用相對論性噴流作為分析基準。[羅傑·潘洛斯](../Page/羅傑·潘洛斯.md "wikilink")曾經提出參考系拖曳機制將從黑洞中汲取[能量](../Page/能量.md "wikilink")與[動量](https://zh.wikipedia.org/wiki/動量 "wikilink")。\[1\][佛羅里達大學的Reva](https://zh.wikipedia.org/wiki/佛羅里達大學 "wikilink") Kay Williams則發展出潘洛斯機制的數學證明\[2\]。她的研究成果顯示：參考系拖曳的[冷澤-提爾苓效應可解釋](https://zh.wikipedia.org/wiki/冷澤-提爾苓效應 "wikilink")[天文觀測中高能](../Page/天文學.md "wikilink")[波霎亮度](https://zh.wikipedia.org/wiki/波霎 "wikilink")、活動星系核上等數據結果。

史丹佛大學一群研究人員目前正分析首批由Gravity Probe B針對引力磁性的實驗數據，用以檢驗引力磁性的實驗數據是否與理論吻合。阿帕契點天文台月地雷射測距站（APOLLO）也正在計畫觀測引力磁性效應。目前的數據僅顯示類似於引力磁性的效應確實存在，但與引力磁性理論的描述層級上有相當大的差距，吻合度並不太高，某些量子引力理論的自然推導結論完全可以補償並巧妙完善地說明觀測到的數據，可以完全不使用引力磁性理論來說明觀測數據。

## 數學形式

### 方程式

根據[廣義相對論](../Page/廣義相對論.md "wikilink")，由一旋轉物體（或任何的旋轉[質能](https://zh.wikipedia.org/wiki/質能 "wikilink")）所產生的[重力場](https://zh.wikipedia.org/wiki/重力場 "wikilink")，在弱場極限情形下可用一組類比電磁學[馬克斯威方程組的方程組來描述](https://zh.wikipedia.org/wiki/馬克斯威方程組 "wikilink")，以「重力電磁性方程組」稱之。兩者比較皆以[SI單位制可寫如下表](https://zh.wikipedia.org/wiki/SI "wikilink")：\[3\]\[4\]

| 重力電磁性方程組                                                                                                                                           | 電磁學馬克斯威方程組                                                                                                                 |
| -------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| \(\nabla \cdot \mathbf{E}_\text{g} = -4 \pi G \rho_\text{g} \\)                                                                                    | \(\nabla \cdot \mathbf{E} =  \frac{\rho}{\epsilon_0}\)                                                                     |
| \(\nabla \cdot \mathbf{B}_\text{g} = 0 \\)                                                                                                         | \(\nabla \cdot \mathbf{B} = 0 \\)                                                                                          |
| \(\nabla \times \mathbf{E}_\text{g} = -\frac{\partial \mathbf{B}_\text{g} } {\partial t} \\)                                                       | \(\nabla \times \mathbf{E} = -\frac{\partial \mathbf{B} } {\partial t} \\)                                                 |
| \(\nabla \times \mathbf{B}_\text{g} =  -\frac{4 \pi G}{c^2} \mathbf{J}_\text{g} + \frac{1}{c^2} \frac{\partial \mathbf{E}_\text{g}} {\partial t}\) | \(\nabla \times \mathbf{B} = \frac{1}{\epsilon_0 c^2} \mathbf{J} + \frac{1}{c^2} \frac{\partial \mathbf{E}} {\partial t}\) |

其中

  - **E**<sub>g</sub>：[重力場](https://zh.wikipedia.org/wiki/重力場 "wikilink")（重力電性），單位為m⋅s<sup>−2</sup>；
  - **E**：[電場](../Page/電場.md "wikilink")；
  - **B**<sub>g</sub>：重力磁場，單位為s<sup>−1</sup>；
  - **B**：[磁場](../Page/磁場.md "wikilink")；
  - *ρ*<sub>g</sub>：[質量密度](../Page/密度.md "wikilink")，單位為kg⋅m<sup>−3</sup>；
  - *ρ*：[電荷密度](../Page/電荷密度.md "wikilink")；
  - **J**<sub>g</sub>：質量流密度或[質量通量](https://zh.wikipedia.org/wiki/質量通量 "wikilink") (**J**<sub>g</sub> = *ρ<sub>g</sub>***v**<sub>ρ</sub>，而**v**<sub>ρ</sub>為產生重力磁場的質量流的[速度](../Page/速度.md "wikilink")，單位為kg⋅m<sup>−2</sup>⋅s<sup>−1</sup>；
  - **J**：[電流密度](https://zh.wikipedia.org/wiki/電流密度 "wikilink")；
  - *G*：[重力常數](https://zh.wikipedia.org/wiki/重力常數 "wikilink")，單位為m<sup>3</sup>⋅kg<sup>−1</sup>⋅s<sup>−2</sup>；
  - ε<sub>0</sub>：[真空電容率](https://zh.wikipedia.org/wiki/真空電容率 "wikilink")；
  - *c*：，根據廣義相對論，其值等於[真空中](https://zh.wikipedia.org/wiki/真空 "wikilink")[光速](../Page/光速.md "wikilink")，單位為m⋅s<sup>−1</sup>。

### 勞侖茲力

一個帶微小質量*m*的[測試粒子處於靜態系統中](https://zh.wikipedia.org/wiki/測試粒子 "wikilink")，其所受來自於重力電磁場的淨力（勞侖茲力）類比於電磁學的[勞侖茲力方程式](https://zh.wikipedia.org/wiki/勞侖茲力 "wikilink")：

| 重力電磁性方程式                                                                                                     | 電磁學方程式                                                                                   |
| ------------------------------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------- |
| \(\mathbf{F_\text{g}} = m \left( \mathbf{E}_\text{g} \ + \ 4 \mathbf{v} \times \mathbf{B}_\text{g} \right)\) | \(\mathbf{F_\text{e}} = q \left( \mathbf{E} \ + \ \mathbf{v} \times \mathbf{B} \right)\) |
|                                                                                                              |                                                                                          |

其中

  - **v**：測試粒子的[速度](../Page/速度.md "wikilink")；
  - *m*：測試粒子的[質量](https://zh.wikipedia.org/wiki/質量 "wikilink")；
  - *q*：測試粒子的[電荷](../Page/電荷.md "wikilink")。

### 坡印廷向量

重力電磁性亦有類似電磁學的[坡印廷向量](../Page/坡印廷向量.md "wikilink")：\[5\]

| 重力電磁性方程式                                                                                         | 電磁學方程式                                                           |
| ------------------------------------------------------------------------------------------------ | ---------------------------------------------------------------- |
| \(\mathcal{S}_\text{g} = -\frac{c^2}{4 \pi G} \mathbf{E}_\text{g} \times 4 \mathbf{B}_\text{g}\) | \(\mathcal{S} = c^2 \varepsilon_0 \mathbf{E} \times \mathbf{B}\) |
|                                                                                                  |                                                                  |

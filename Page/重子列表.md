[Proton_quark_structure.svg](https://zh.wikipedia.org/wiki/File:Proton_quark_structure.svg "fig:Proton_quark_structure.svg")是著名的[重子](../Page/重子.md "wikilink")之一，具有二個上[夸克](../Page/夸克.md "wikilink")（u）和一個下夸克（d）\]\]

**重子**由三個[夸克](../Page/夸克.md "wikilink")組成，相較之下[介子](../Page/介子.md "wikilink")則是由夸克-[反夸克對所組成](https://zh.wikipedia.org/wiki/反夸克 "wikilink")。\[1\][重子](../Page/重子.md "wikilink")與介子都屬於[强子](../Page/强子.md "wikilink")，意即單純由夸克或同時由夸克和反夸克所組成的粒子。重子的英文名稱「」是來自[希腊语中的](https://zh.wikipedia.org/wiki/希腊语 "wikilink")「」，意為「重的」，因為在命名當時，重子具有比其他物質粒子還要大的質量被認為是其特性之一。

直到近幾年，部分實驗證實了[五夸克態](../Page/五夸克態.md "wikilink")粒子的存在，亦即由四個夸克與一個反夸克所組成的重子。2015年7月13日，[歐洲核子研究組織](../Page/歐洲核子研究組織.md "wikilink")的[LHCb](../Page/LHCb.md "wikilink")團隊宣布，在[底Λ粒子衰變模式中找到了與五夸克態吻合的結果](../Page/Λ粒子.md "wikilink")。

因為重子是由夸克所組成的，它們也會參與[强相互作用](../Page/强相互作用.md "wikilink")。相對而言，不是由夸克所組成的[輕子](../Page/輕子.md "wikilink")就不會參與强相互作用了。著名的重子包含[質子](../Page/質子.md "wikilink")和[中子](../Page/中子.md "wikilink")，它們組成了[宇宙](../Page/宇宙.md "wikilink")中大多數的可見[物质](../Page/物质.md "wikilink")，然而另一個主要組成[原子](../Page/原子.md "wikilink")的粒子，也就是[电子](../Page/电子.md "wikilink")，則屬於輕子。每個重子都有各自相對應的[反粒子](https://zh.wikipedia.org/wiki/反粒子 "wikilink")，稱為反重子，其中，夸克被替換成了對應的反夸克。例如，一個質子是由兩個上夸克與一個下夸克所組成，而其對應的反粒子，亦即[反質子](../Page/反質子.md "wikilink")，則是由兩個反上夸克與一個反下夸克所組成。

## 重子列表

本列表列出了所有已知與被預測的重子，其角動量為*J* =  及 *J* = ，並有著正的[宇稱](../Page/宇稱.md "wikilink")值。

  - 只含有一種夸克的重子（例如 uuu 和 ddd）可以存在*J* =  的組態，然而*J* =  是[泡利不相容原理](../Page/泡利不相容原理.md "wikilink")所不允許的。
  - 含有二種夸克的重子（例如 uud 和 uus）可以存在*J* =  及 *J* =  的組態。
  - 含有三種夸克的重子（例如 uds 和 udc）可以存在*J* =  及 *J* =  的組態。這類型的重子可以具有二種*J* = 的組態。

在本列表中的符號有：*I*（[同位旋](https://zh.wikipedia.org/wiki/同位旋 "wikilink")）、*J*（[總角動量量子數](../Page/總角動量量子數.md "wikilink")）、*P*（[宇稱](../Page/宇稱.md "wikilink")，表示座標系鏡像變換時[特徵值的奇偶性](https://zh.wikipedia.org/wiki/特徵值 "wikilink")）、u（[上夸克](../Page/上夸克.md "wikilink")）、d（[下夸克](../Page/下夸克.md "wikilink")）、s（[奇夸克](../Page/奇夸克.md "wikilink")）、c（[粲夸克](../Page/粲夸克.md "wikilink")）、b（[底夸克](../Page/底夸克.md "wikilink")）、*Q*（[電荷](https://zh.wikipedia.org/wiki/荷_\(物理\) "wikilink")）、*B*（[重子数](https://zh.wikipedia.org/wiki/重子数 "wikilink")）、 *S*（[奇异数](https://zh.wikipedia.org/wiki/奇异数 "wikilink")）、*C*（[粲数](https://zh.wikipedia.org/wiki/粲数 "wikilink")）、*B′*（[底數](https://zh.wikipedia.org/wiki/底數_\(物理學\) "wikilink")）以及多種次原子粒子。

本列表並未列出反重子。然而，它們只是將所有夸克換成對應的反夸克，並將*Q*、*B*、*S*、*C*、*B′* 等數值變號。在本表格中，有著<sup>†</sup>記號的粒子表示該粒子被[标准模型](../Page/标准模型.md "wikilink")所預測，然而尚未被觀測到。<span style="color:orange">橙色</span>的數值表示實驗上尚未被完全確定，但其實驗結果吻合[夸克模型](../Page/夸克模型.md "wikilink")之預測。

### 重子共振粒子

本列表給出了已被所確認的重子共振粒子名稱、已知量子數與實驗狀態。重子[共振粒子是受激發的重子態](https://zh.wikipedia.org/wiki/共振_\(粒子物理学\) "wikilink")，有著較短的半衰期和較高的質量。儘管有著大量的研究，人們仍舊對重子激發譜的基礎自由度有著極薄弱的了解。每個粒子的已知自旋-宇稱值J<sup>P</sup>在下表亦有列出。對於強衰變粒子，J<sup>P</sup>值被視為其名稱的一部份，所有共振態的質量亦然。

以下列表中，使用\*號數量表示該粒子存在之確定性高低，如下：

|          |                                         |

| \*\*\*   | 幾乎可以確定其存在，然而需要更多確認，且/或其量子數、分支比等性質尚未被確定。 |
| \*\*     | 有一些存在的證據。                               |
| \*       | 存在的證據很少。                                |

[重庆市](../Page/重庆市.md "wikilink")[永川中学始创于](../Page/永川中学.md "wikilink")
1719 年的[桂山书院](../Page/桂山书院.md "wikilink")，从1900年兴办新学始，已有 100
多年历史。是原[四川省首批四十所](../Page/四川省.md "wikilink")[重点中学之一](../Page/重点中学.md "wikilink")，也是[重庆直辖后首批确认的](../Page/重庆直辖市.md "wikilink")[重庆市重点中学](../Page/重庆市.md "wikilink")。\[1\]

## 历史变迁\[2\]

| 时间    | 历史                                                                                                                                                                                                        |
| ----- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1719年 | 创立桂山书院。                                                                                                                                                                                                   |
| 1762年 | 学校更名为锦云学院。                                                                                                                                                                                                |
| 1900年 | [永川知县罗崇玲废锦云书院](../Page/永川区.md "wikilink")，聘黄秉湘为监督。仿[日本学制办学](../Page/日本.md "wikilink")，聘日籍教师山本策吉、神田正雄等，设[经学](../Page/经学.md "wikilink")、[英文](../Page/英文.md "wikilink")、[格致](../Page/科学.md "wikilink")、博物等科。 |
| 1906年 | “达用学堂”更名为“重庆府分立永川中学堂”。                                                                                                                                                                                    |
| 1909年 | 学校更名为“永川官立中学堂”。                                                                                                                                                                                           |
| 1939年 | 为避[侵华日军机空袭](../Page/侵华日军.md "wikilink")，永川县立初级中学迁址于[陈食](../Page/陈食街道.md "wikilink")。                                                                                                                      |
| 1946年 | 私立达用高级商业学校成立，有学生约四十人。学校由[陈食迁回城内原址](../Page/陈食街道.md "wikilink")。                                                                                                                                           |
| 1947年 | “[永川县立初级中学](../Page/永川区.md "wikilink")”更名为“永川县立中学”，将[永中建成完全中学](../Page/永川中学.md "wikilink")。                                                                                                               |
| 1950年 | 学校更名为“川东区永川中学”。永川中学与永川县立女子中学合并。                                                                                                                                                                           |
| 1951年 | 私立英井中学，私立达用高级商业学校并入永川中学。永川中学由原永川城内东正街迁回永川桐子坳英井中学校址至今。                                                                                                                                                     |
| 1953年 | “川东区永川中学”更名为“四川省永川中学校”。                                                                                                                                                                                   |
| 1997年 | 更名为“重庆市永川中学校”。                                                                                                                                                                                            |

《**重案傳真**》（），是[香港](../Page/香港.md "wikilink")[電視廣播有限公司](../Page/電視廣播有限公司.md "wikilink")於1992年首播的時裝警匪[電視劇](https://zh.wikipedia.org/wiki/電視劇 "wikilink")，由[許志安](../Page/許志安.md "wikilink")、[許紹雄](../Page/許紹雄.md "wikilink")、[劉錫明](../Page/劉錫明.md "wikilink")、[關寶慧](../Page/關寶慧.md "wikilink")等主演。

此劇于2018年10月16日在[TVB經典台](../Page/TVB經典台.md "wikilink")重播。

## 劇情簡介

女幹探鄺月娥 ( [關寶慧](../Page/關寶慧.md "wikilink")飾 ) 精明能幹，對家庭亦愛謢有加。惟妹妹鄺月美 ( [陳梅馨飾](https://zh.wikipedia.org/wiki/陳梅馨 "wikilink") ) 常發明星夢，做姊姊在多加提點之際，竟無意揭發一個娛樂圈騙色圈套。正當她欲將騙子繩之以法，卻赫然發現其妹已成受害人，頓時十分矛盾。

重案組沙展張近富 ( [許紹雄](../Page/許紹雄.md "wikilink")飾 )，接手調查一宗連續兇殺案，三名男死者生前均曾徵婚。近富為查案前往徵婚，結識了豔女郎姜莉 ( [林其欣](../Page/林其欣.md "wikilink")飾 )，一方面懷疑她就是變態殺手，另一方面卻對她產生情愫。正感懊惱不堪之際，竟發現兇手是另有其人。

幹探周德培 ( [許志安](../Page/許志安.md "wikilink")飾 ) 與好友余小明 ( [吳　剛飾](../Page/吳剛_\(香港演員\).md "wikilink") ) 本為警校同學，後小明因意外而沒法當上警察，改任差館雜役。後小明戀上一少女，為討歡心不惜冒警，結果闖下彌天大禍。德培及同僚奉命追緝，深感友情、公義兩難全。

為追查一宗劫案，探員方正 ( [劉錫明](../Page/劉錫明.md "wikilink")飾 ) 及周培德奉命監視疑犯的妹妹王天真 ( [甄楚倩](../Page/甄楚倩.md "wikilink")飾 )，但兩人卻同時被這位天真少艾的美女迷倒了。後他們隱瞞身份與她接觸，認定她是無辜的。不料此時犯案的大圈幫卻找上門，強搶天真的哥哥收藏在家中的贓物，令她身陷絕境。

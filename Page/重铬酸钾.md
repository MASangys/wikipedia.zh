**重铬酸钾**（；[化学式](../Page/化学式.md "wikilink")：**[K](../Page/钾.md "wikilink")<sub>2</sub>[Cr](../Page/铬.md "wikilink")<sub>2</sub>[O](../Page/氧.md "wikilink")<sub>7</sub>**；此处“**重**”的[普通话](../Page/普通话.md "wikilink")发音为：**chóng**）是一种有[毒且有](https://zh.wikipedia.org/wiki/毒 "wikilink")[致癌性的强](https://zh.wikipedia.org/wiki/致癌性 "wikilink")[氧化剂](../Page/氧化剂.md "wikilink")，[室温下为橙红色](https://zh.wikipedia.org/wiki/室温 "wikilink")[固体](../Page/固体.md "wikilink")。它被[国际癌症研究机构](../Page/国际癌症研究机构.md "wikilink")划归为[第一类致癌物质](https://zh.wikipedia.org/wiki/1类致癌物 "wikilink")，而且是强[氧化剂](../Page/氧化剂.md "wikilink")，在实验室和工业中都有很广泛的应用。

酸性溶液中重铬酸根还原为铬(III)离子的[半反應](../Page/半反應.md "wikilink")式为：

\[{\rm \ Cr_2O_7^{2-} + 14H^+ + 6 {\mathit{e}}^- \rightarrow 2Cr^{3+} + 7H_2O} ; \ \ \varphi^{\ominus} = 1.36 \ \rm V\]

## 制备

由[重铬酸钠](../Page/重铬酸钠.md "wikilink")与[氯化钾](../Page/氯化钾.md "wikilink")混合以后通过重结晶分离得到纯品。\[1\]\[2\]

## 物理性质

橙红色三斜晶系板状结晶体。有苦味及金属性味。密度2.676g/cm3。熔点398℃。稍溶于冷水，水溶液呈酸性，易溶于热水，不溶于乙醇。有剧毒。

水中溶解度：0℃，4.3；20℃，11.7；40℃，20.9；60℃，31.3；80℃，42.0；100℃，50.2

## 化学性质

重铬酸钾是一种强氧化剂，加热到241.6℃时三斜晶系转变为单斜晶系，强热约500℃时分解为[三氧化二铬](../Page/三氧化二铬.md "wikilink")和[铬酸钾](../Page/铬酸钾.md "wikilink")。不吸湿潮解，不生成水合物（不同于[重铬酸钠](../Page/重铬酸钠.md "wikilink")）。遇[浓硫酸有红色针状晶体](https://zh.wikipedia.org/wiki/浓硫酸 "wikilink")[铬酸酐析出](https://zh.wikipedia.org/wiki/铬酸酐 "wikilink")，对其加热则分解放出[氧气](../Page/氧气.md "wikilink")，生成[硫酸铬](https://zh.wikipedia.org/wiki/硫酸铬 "wikilink")，使溶液的颜色由橙色变成绿色。

### 复分解反应

在重铬酸钾的溶液中加入Ba<sup>2+</sup>、Pb<sup>2+</sup>和Ag<sup>+</sup>时，形成相应的铬酸盐沉淀\[3\]：

  -
    2 Ba<sup>2+</sup> + Cr<sub>2</sub>O<sub>7</sub><sup>2-</sup> + H<sub>2</sub>O ⇌ 2 BaCrO<sub>4</sub>↓ + 2H<sup>+</sup>
    2 Pb<sup>2+</sup> + Cr<sub>2</sub>O<sub>7</sub><sup>2-</sup> + H<sub>2</sub>O ⇌ 2 PbCrO<sub>4</sub>↓ + 2H<sup>+</sup>
    4 Ag<sup>+</sup> + Cr<sub>2</sub>O<sub>7</sub><sup>2-</sup> + H<sub>2</sub>O ⇌ 2 Ag<sub>2</sub>CrO<sub>4</sub>↓ + 2H<sup>+</sup>

### 氧化反应

在酸性溶液中，重铬酸钾是强氧化剂，可以和许多还原性物质反应，自身被还原为Cr<sup>3+</sup>\[4\]\[5\]：

  -
    K<sub>2</sub>Cr<sub>2</sub>O<sub>7</sub> + 6 KI + 7H<sub>2</sub>SO<sub>4</sub> → Cr<sub>2</sub>(SO<sub>4</sub>)<sub>3</sub> + 3 I<sub>2</sub> + 7 H<sub>2</sub>O + 4 K<sub>2</sub>SO<sub>4</sub>
    K<sub>2</sub>Cr<sub>2</sub>O<sub>7</sub> + 3 K<sub>2</sub>SO<sub>3</sub> + 4 H<sub>2</sub>SO<sub>4</sub> → Cr<sub>2</sub>(SO<sub>4</sub>)<sub>3</sub> + 4 K<sub>2</sub>SO<sub>4</sub> + 4 H<sub>2</sub>O
    K<sub>2</sub>Cr<sub>2</sub>O<sub>7</sub> + H<sub>2</sub>SO<sub>4</sub> + 3SO<sub>2</sub> → K<sub>2</sub>SO<sub>4</sub>·Cr<sub>2</sub>(SO<sub>4</sub>)<sub>3</sub> + H<sub>2</sub>O
    K<sub>2</sub>Cr<sub>2</sub>O<sub>7</sub> + 3 H<sub>2</sub>S + 4 H<sub>2</sub>SO<sub>4</sub> → Cr<sub>2</sub>(SO<sub>4</sub>)<sub>3</sub> + 3 S↓ + 7 H<sub>2</sub>O + K<sub>2</sub>SO<sub>4</sub>
    K<sub>2</sub>Cr<sub>2</sub>O<sub>7</sub> + 3 NaNO<sub>2</sub> + 4 H<sub>2</sub>SO<sub>4</sub> → 3 NaNO<sub>3</sub> + Cr<sub>2</sub>(SO<sub>4</sub>)<sub>3</sub> + K<sub>2</sub>SO<sub>4</sub> + 4 H<sub>2</sub>O
    K<sub>2</sub>Cr<sub>2</sub>O<sub>7</sub> + 6 KBr + 7 H<sub>2</sub>SO<sub>4</sub> → Cr<sub>2</sub>(SO<sub>4</sub>)<sub>3</sub> + 4 K<sub>2</sub>SO<sub>4</sub> + 3 Br<sub>2</sub> + 7 H<sub>2</sub>O
    5 K<sub>2</sub>Cr<sub>2</sub>O<sub>7</sub>(浓) + 6 KI —<sup><u>煮沸</u></sup>→ 8 K<sub>2</sub>CrO<sub>4</sub> + Cr<sub>2</sub>O<sub>3</sub> + 3 I<sub>2</sub>

因此[分析化学](../Page/分析化学.md "wikilink")上，重铬酸钾被用来作氧化剂来测定铁的含量：

  -
    K<sub>2</sub>Cr<sub>2</sub>O<sub>7</sub> + 6 FeSO<sub>4</sub> + 7 H<sub>2</sub>SO<sub>4</sub> → 3 Fe<sub>2</sub>(SO<sub>4</sub>)<sub>3</sub> + Cr<sub>2</sub>(SO<sub>4</sub>)<sub>3</sub> + K<sub>2</sub>SO<sub>4</sub> + 7 H<sub>2</sub>O

需要注意的是，重铬酸钾可以和热的浓盐酸反应，放出[氯气](https://zh.wikipedia.org/wiki/氯气 "wikilink")：

  -
    K<sub>2</sub>Cr<sub>2</sub>O<sub>7</sub> + 14 HCl(浓) → 2 CrCl<sub>3</sub> + 2 KCl + 3 Cl<sub>2</sub>↑ + 7 H<sub>2</sub>O

若鹽酸濃度較低，則可能生成橘紅色的氯鉻酸鉀 :

  -

K<sub>2</sub>Cr<sub>2</sub>O<sub>7</sub> + 2 HCl(較稀) → 2 KCrO<sub>3</sub>Cl + H<sub>2</sub>O

在高温下，重铬酸钾可以被[硫](../Page/硫.md "wikilink")、[碳](../Page/碳.md "wikilink")等单质还原：\[6\]

  -
    K<sub>2</sub>Cr<sub>2</sub>O<sub>7</sub> + S → Cr<sub>2</sub>O<sub>3</sub> + K<sub>2</sub>SO<sub>4</sub>
    K<sub>2</sub>Cr<sub>2</sub>O<sub>7</sub> + 2 C → Cr<sub>2</sub>O<sub>3</sub> + K<sub>2</sub>CO<sub>3</sub> + CO↑

在有机反应中，重铬酸钾-硫酸可以将[伯醇氧化成](https://zh.wikipedia.org/wiki/伯醇 "wikilink")[醛](../Page/醛.md "wikilink")，若继续反应，则[醛](../Page/醛.md "wikilink")进一步被氧化为[羧酸](../Page/羧酸.md "wikilink")。而[叔醇的反应则是先被](https://zh.wikipedia.org/wiki/叔醇 "wikilink")[硫酸](../Page/硫酸.md "wikilink")脱水，再被重铬酸钾氧化。[苯酚](../Page/苯酚.md "wikilink")等酚类也能被酸化的重铬酸钾氧化，如产生黄色的[对苯醌](https://zh.wikipedia.org/wiki/对苯醌 "wikilink")。\[7\]

### 过氧化物的形成

重铬酸钾能和冷的[过氧化氢](../Page/过氧化氢.md "wikilink")反应，形成深藍色且不穩定的[过氧化铬](../Page/过氧化铬.md "wikilink")：

  -
    Cr<sub>2</sub>O<sub>7</sub><sup>2-</sup> + 4 H<sub>2</sub>O<sub>2</sub> + 2 H<sup>+</sup> → 2 CrO<sub>5</sub> + 5 H<sub>2</sub>O

该化合物可以被[乙醚](../Page/乙醚.md "wikilink")或[異戊醇](../Page/異戊醇.md "wikilink")萃取。

### 铬酸根的形成

重铬酸钾在碱性条件下会转化为铬酸钾：

  -
    K<sub>2</sub>Cr<sub>2</sub>O<sub>7</sub> + 2 KOH → 2 K<sub>2</sub>CrO<sub>4</sub> + H<sub>2</sub>O

和浓、热的氨水反应，冷却后得到复盐：

  -
    K<sub>2</sub>Cr<sub>2</sub>O<sub>7</sub> + 2 NH<sub>3</sub>·H<sub>2</sub>O(浓、热) —<sup><u>冷却</u></sup>→ 2 NH<sub>4</sub>KCrO<sub>4</sub> + H<sub>2</sub>O\[8\]

和碳酸钾反应，也能形成铬酸钾：

  -
    K<sub>2</sub>Cr<sub>2</sub>O<sub>7</sub> + K<sub>2</sub>CO<sub>3</sub> → 2 K<sub>2</sub>CrO<sub>4</sub> + CO<sub>2</sub>↑

### 铬酰盐的形成

重铬酸钾在硫酸的作用下与其他物质反应可以形成铬酰盐，如：

  -
    K<sub>2</sub>Cr<sub>2</sub>O<sub>7</sub> + 2 CaF<sub>2</sub> + 3 H<sub>2</sub>SO<sub>4</sub>(发烟) —<u><sup>Δ</sup></u>→ 2 CrO<sub>2</sub>F<sub>2</sub> + 2 CaSO<sub>4</sub> + K<sub>2</sub>SO<sub>4</sub> + 3 H<sub>2</sub>O
    K<sub>2</sub>Cr<sub>2</sub>O<sub>7</sub> + 4 KCl(适量) + 3 H<sub>2</sub>SO<sub>4</sub>(过量) —<u><sup>Δ</sup></u>→ 2 CrO<sub>2</sub>Cl<sub>2</sub>↑ + 3 K<sub>2</sub>SO<sub>4</sub> + 3 H<sub>2</sub>O

## 应用

[有机合成](../Page/有机合成.md "wikilink")中，根据条件不同，重铬酸钾可将[醇](../Page/醇.md "wikilink")氧化为[醛](../Page/醛.md "wikilink")、[羧酸](../Page/羧酸.md "wikilink")或[酮](../Page/酮.md "wikilink")，同时颜色由橙红色变为铬离子的绿色。也可用于鉴别醛和酮。

在实验室用来配置[铬酸](../Page/铬酸.md "wikilink")洗液清洗玻璃器皿。酸化的重铬酸钾可将乙醇氧化并变色，以检验司机是否酒后驾驶。\[9\]

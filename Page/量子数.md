**量子數**描述[量子系統中動力學上各守恒數的值](https://zh.wikipedia.org/wiki/量子力學 "wikilink")。它們通常按性質描述[原子](../Page/原子.md "wikilink")中[電子的各](https://zh.wikipedia.org/wiki/電子 "wikilink")[能量](../Page/能量.md "wikilink")，但也會描述其他物理量（如[角動量](https://zh.wikipedia.org/wiki/角動量 "wikilink")、[自旋](../Page/自旋.md "wikilink")等）。由於任何量子系統都能有一個或以上的量子數，列出所有可能的量子數是件沒有意義的工作。

## 有多少個量子數？

「要多少個量子數才能描述任何已知系統？」這道問題並沒有一致的答案，儘管要解決每一個系統都必需要對系統進行全面分析。任何系統的動力學都由一量子[哈密頓算符](https://zh.wikipedia.org/wiki/哈密頓算符 "wikilink")，**H**，所描述。系統中有一量子數對應能量，即哈密頓算符的[特徵值](https://zh.wikipedia.org/wiki/特徵值 "wikilink")。對每一個算符**O**而言，還有一個量子數可與哈密頓算符交換（即滿足**OH = HO**這條關係式）。這些是一個系統中所能有的所有量子數。注意定義量子數的算符**O**應互相獨立。很多時候，能有好幾種選擇一組互相獨立算符的方法。故此，在不同的條件下，可使用不同的量子數組來描述同一個系統。

## 原子內的單個電子

最被廣為研究的量子數組是用於一[原子](../Page/原子.md "wikilink")的單個[電子](https://zh.wikipedia.org/wiki/電子 "wikilink")：不只是因為它在[化學](../Page/化學.md "wikilink")中有用（它是[週期表](https://zh.wikipedia.org/wiki/週期表 "wikilink")、[化合價及其他一系列特性的基本概念](https://zh.wikipedia.org/wiki/化合價 "wikilink")），還因為它是一個可解的真實問題，故廣為教科書所採用。

在非相對論性[量子力學中](https://zh.wikipedia.org/wiki/量子力學 "wikilink")，這個系統的哈密頓算符由電子的[動能及](https://zh.wikipedia.org/wiki/動能 "wikilink")[勢能](https://zh.wikipedia.org/wiki/勢能 "wikilink")（由電子及原子核間的[庫侖力所產生](https://zh.wikipedia.org/wiki/庫侖定律 "wikilink")）。動能可被分成，有環繞[原子核](../Page/原子核.md "wikilink")的電子[角動量](https://zh.wikipedia.org/wiki/角動量 "wikilink")，**J**的一份，及餘下的一份。由於勢能是球狀對稱的關係，其完整的哈密頓算符能與**J**<sup>2</sup>交換。而**J**<sup>2</sup>本身能與角動量的任一分量（按慣例使用**J**<sub>z</sub>）交換。由於這是本題中唯一的一組可交換算符，所以會有三個量子數。

依慣例，它們被稱為：

  - [主量子數](../Page/主量子數.md "wikilink")（n=1，2，3，4 …）代表除掉**J**<sup>2</sup>以後**H**的特徵值。這個數因此會視電子與原子核間的距離（即半徑座標**r**）而定。平均距離會隨着**n**增大，因此不同量子數的量子態會被說成屬於不同的電子層。
  - [角量子數](https://zh.wikipedia.org/wiki/角量子數 "wikilink")（l=0，1 … n-1）（又稱**方位角量子數**或**軌道量子數**）通過關係式\(L^2 = \hbar^2 l(l+1)\)來代表軌道角動量。在化學中，這個量子數是非常重要的，因為它表明了一軌道的形狀，並對[化學鍵及](https://zh.wikipedia.org/wiki/化學鍵 "wikilink")[鍵角有重大影響](https://zh.wikipedia.org/wiki/鍵角 "wikilink")。有些時候，不同角量子數的軌域有不同代號，**l=0**的軌域叫s軌域，**l=1**的叫p軌域，**l=2**的叫d軌域，而**l=3**的則叫f軌域。
  - [磁量子數](https://zh.wikipedia.org/wiki/磁量子數 "wikilink")（m<sub>l</sub>= -l，-l+1 … 0 … l-1，l）代表[特徵值](https://zh.wikipedia.org/wiki/特徵值 "wikilink")，\(L_z = m_l \hbar\)\[1\]。這是軌道[角動量沿某指定軸的投影](https://zh.wikipedia.org/wiki/角動量 "wikilink")。

從[光譜學中所得的結果指出一個軌道最多可容納兩個電子](https://zh.wikipedia.org/wiki/光譜學 "wikilink")。然而兩個電子絕不能擁有完全相同的量子態（[泡利不相容原理](../Page/泡利不相容原理.md "wikilink")），故也絕不能擁有同一組量子數。所以為此特別提出一個假設來解決這問題，就是設存在一個有兩個可能值的第四個量子數。這假設以後能被相對論性量子力學所解釋。

  - [自旋量子數](https://zh.wikipedia.org/wiki/自旋量子數 "wikilink")（m<sub>s</sub>= -1/2 或 +1/2）代表電子的固有[角動量](https://zh.wikipedia.org/wiki/角動量 "wikilink")。這是[自旋](../Page/自旋.md "wikilink") s=1/2沿某指定軸的射影。

例：用於描述[氟](../Page/氟.md "wikilink")（F）原子最外層電子（即[價電子](https://zh.wikipedia.org/wiki/價電子 "wikilink")，位於[原子軌道](https://zh.wikipedia.org/wiki/原子軌道 "wikilink")2p）的各量子數值為：n=2，l=1，m<sub>l</sub>=1或0或-1，m<sub>s</sub>=-1/2或1/2。

注意[分子軌道需要使用完全不同的量子數組](https://zh.wikipedia.org/wiki/分子軌道 "wikilink")，因為其[哈密頓算符及對稱跟上述相當不同](https://zh.wikipedia.org/wiki/哈密頓算符 "wikilink")。

## 適用於自旋－軌道交互作用的量子數

當考慮到[自旋-軌道作用](../Page/自旋-軌道作用.md "wikilink")時，l、m及s就再不能與[哈密頓算符](https://zh.wikipedia.org/wiki/哈密頓算符 "wikilink")[交換](../Page/交換律.md "wikilink")，因而它們的值會隨時間改變。故應該使用另一組[量子數](https://zh.wikipedia.org/wiki/量子數 "wikilink")。這組包括了

  - [總角動量量子數](../Page/總角動量量子數.md "wikilink")（j=1/2，3/2 … n-1/2）通過關係式\(J^2 = \hbar^2 j(j+1)\)代表着總[角動量](https://zh.wikipedia.org/wiki/角動量 "wikilink")。
  - [總角動量沿某指定軸的投影](https://zh.wikipedia.org/wiki/次要總角動量量子數 "wikilink")（m<sub>j</sub>=-j，-j+1 … j-1，j），此數與m類似，且滿足關係式\(m_j = m_l + m_s\)。
  - [宇稱](../Page/宇稱.md "wikilink")。它是經反射所得的[特徵值](https://zh.wikipedia.org/wiki/特徵值 "wikilink")，當態之l為偶數時其值為正（即+1），奇數時其值為負（即-1）。前者亦被稱為**偶宇稱**，後者則為**奇宇稱**。

例：考慮以下八個態，定義它們的量子數：

1.  l = 1，m<sub>l</sub> = 1，m<sub>s</sub> = +1/2
2.  l = 1，m<sub>l</sub> = 1，m<sub>s</sub> = -1/2
3.  l = 1，m<sub>l</sub> = 0，m<sub>s</sub> = +1/2
4.  l = 1，m<sub>l</sub> = 0，m<sub>s</sub> = -1/2
5.  l = 1，m<sub>l</sub> = -1，m<sub>s</sub> = +1/2
6.  l = 1，m<sub>l</sub> = -1，m<sub>s</sub> = -1/2
7.  l = 0，m<sub>l</sub> = 0，m<sub>s</sub> = +1/2
8.  l = 0，m<sub>l</sub> = 0，m<sub>s</sub> = -1/2

系統的[量子態](../Page/量子態.md "wikilink")能被這八個態的線性組合所描述。但由於[自旋-軌道作用](../Page/自旋-軌道作用.md "wikilink")的關係，如欲使用八個由[哈密頓算符的](https://zh.wikipedia.org/wiki/哈密頓算符 "wikilink")[特徵向量](https://zh.wikipedia.org/wiki/特徵向量 "wikilink")（即每一個代表一個態且不會因時間而跟其他態混合）所組成的態來描述同一個系統，應考慮以下這八個態：

1.  j = 3/2， m<sub>j</sub> = 3/2，奇宇稱 （從上態1得）
2.  j = 3/2， m<sub>j</sub> = 1/2，奇宇稱 （從上態2及3得）
3.  j = 3/2， m<sub>j</sub> = -1/2，奇宇稱 （從上態4及5得）
4.  j = 3/2， m<sub>j</sub> = -3/2，奇宇稱 （從上態6得）
5.  j = 1/2， m<sub>j</sub> = 1/2，奇宇稱 （從上態2及3得）
6.  j = 1/2， m<sub>j</sub> = -1/2，奇宇稱 （從上態4及5得）
7.  j = 1/2， m<sub>j</sub> = 1/2，偶宇稱 （從上態7得）
8.  j = 1/2， m<sub>j</sub> = -1/2，偶宇稱 （從上態8得）

## 基本粒子

  -
    <small> 如欲見更完整的基本粒子量子態敍述，可參閱[標準模型理論及](https://zh.wikipedia.org/wiki/標準模型理論 "wikilink")[味 (粒子物理學)](../Page/味_\(粒子物理學\).md "wikilink")</small>

基本粒子包含不少量子數，一般來說它們都是粒子本身的。但需要明白的是，基本粒子是[粒子物理學](../Page/粒子物理學.md "wikilink")上[標準模型的量子態](https://zh.wikipedia.org/wiki/標準模型理論 "wikilink")，所以這些粒子量子數間的關係跟模型的哈密頓算符一樣，就像[波耳原子量子數及其哈密頓算符的關係那樣](https://zh.wikipedia.org/wiki/波耳原子 "wikilink")。亦即是說，每一個量子數代表問題的一個對稱性。這在[場論中有着更大的用處](https://zh.wikipedia.org/wiki/量子場論 "wikilink")，被用於識別[時空及](https://zh.wikipedia.org/wiki/時空 "wikilink")[內對稱](https://zh.wikipedia.org/wiki/內 "wikilink")。

一般跟[時空對稱有關係的量子數有](https://zh.wikipedia.org/wiki/時空對稱 "wikilink")[自旋](../Page/自旋.md "wikilink")（跟旋轉對稱有關）、[宇稱](../Page/宇稱.md "wikilink")、[C-宇稱](https://zh.wikipedia.org/wiki/C-宇稱 "wikilink")、[T-宇稱](https://zh.wikipedia.org/wiki/T-宇稱 "wikilink")（跟時空上的[龐加萊對稱有關係](https://zh.wikipedia.org/wiki/龐加萊對稱 "wikilink")）。一般的內對稱有[輕子數](../Page/輕子數.md "wikilink")、[重子數及](https://zh.wikipedia.org/wiki/重子數 "wikilink")[電荷](../Page/電荷.md "wikilink")數。條目[味有這些量子數的更詳細列表](../Page/味_\(粒子物理學\).md "wikilink")。

值得一提的是較次要但常被混淆的一點。大部分守恒量子數都是可相加的。故此，在一基本粒子反應中，反應前後的量子數總和應相等。然而，某些量子數（一般被稱為宇稱）是可相乘的；即它們的積是守恒的。所以可相乘的量子數都屬於一種對稱（像守恒那樣），而在這種對稱中使用兩次對稱變換式跟沒用過是一樣的。它們都屬於一個叫**Z**<sub>2</sub>的抽象[群](../Page/群.md "wikilink")。

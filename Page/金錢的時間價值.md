**金錢的時間價值**、**货币的时间价值**或**資金的時間價值**（[英文](https://zh.wikipedia.org/wiki/英文 "wikilink")：**t**ime **v**alue of **m**oney，**TVM**）是[金融](../Page/金融.md "wikilink")的基礎概念。[金錢具有時間價值](https://zh.wikipedia.org/wiki/金錢 "wikilink")，是基於人們希望現在而不是未來取得金錢，因而當金錢用於[存款或](https://zh.wikipedia.org/wiki/存款 "wikilink")[投資時](https://zh.wikipedia.org/wiki/投資 "wikilink")，理應獲得[利息](../Page/利息.md "wikilink")，這樣，金錢的未來[終值](https://zh.wikipedia.org/wiki/終值 "wikilink")（future value）應大於[現值](../Page/現值.md "wikilink")（present value）。

金錢不會自動隨時間而增值，金錢的時間價值經過實際的投資才能實現。 [歐文·費雪](https://zh.wikipedia.org/wiki/歐文·費雪 "wikilink")（Irving Fisher）的[利息論則認為](https://zh.wikipedia.org/wiki/利息論 "wikilink")，利息是由不耐（impatience）及投資機會（opportunity to invest）產生。人們希望現在而不是未來取得金錢，是時間價值本身推動勞動，並以勞動實現時間價值。

## 金錢的時間價值公式

### 現值－終值

\[\left( F / P  \right)  \ = \   (1+r)^n\]

\[\left( P / F  \right)  \ = \  { 1 \over  (1+r)^n }\]

其中，P是[現值](../Page/現值.md "wikilink")（present value），F是[終值](https://zh.wikipedia.org/wiki/終值 "wikilink")（future value），r是每期必要[報酬率](https://zh.wikipedia.org/wiki/報酬率 "wikilink")（required rate of return per time period），n是期數（number of time periods）。

### 終值－年金

\[\left( F / A \right)  \ = \  { (1+r)^n - 1 \over r }\]

\[\left( A / F  \right)  \ = \  { r \over (1+r)^n - 1  }\]

其中，F是終值（future value），A是[年金](../Page/年金.md "wikilink")（annuity），r是每期必要報酬率（required rate of return per time period），n是期數（number of time periods）。

### 現值－年金

\[\left( P / A  \right)  \ = \  { (1+r)^n - 1 \over r (1+r)^n }\]

\[\left( A / P  \right)  \ = \  {  r (1+r)^n  \over (1+r)^n - 1   }\]

其中，P是現值（present value），A是年金（annuity），r是每期必要報酬率（required rate of return per time period），n是期數（number of time periods）。

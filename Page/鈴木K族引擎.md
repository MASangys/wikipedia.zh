**鈴木K系列引擎**是[日本](../Page/日本.md "wikilink")[鈴木公司自](https://zh.wikipedia.org/wiki/鈴木公司 "wikilink")1994年9月起開發製造的[往復式](https://zh.wikipedia.org/wiki/往復式發動機 "wikilink")[汽油引擎](../Page/汽油引擎.md "wikilink")，前代產品為[鈴木F系列引擎](https://zh.wikipedia.org/wiki/鈴木F系列引擎 "wikilink")，後繼產品則是[鈴木R系列引擎](https://zh.wikipedia.org/wiki/鈴木R系列引擎 "wikilink")。

## 概要

相較於舊世代F族引擎之[汽缸本體](../Page/汽缸本體.md "wikilink")以鑄鐵製成，K族引擎改採[鋁合金](../Page/鋁合金.md "wikilink")[壓鑄](../Page/壓鑄.md "wikilink")而成，故較為輕量化。此外，驅動[凸輪軸的方式為正時鏈條](https://zh.wikipedia.org/wiki/凸輪軸 "wikilink")，這是該公司汽車用引擎中首度使用。此族引擎的[汽門](../Page/汽門.md "wikilink")驅動方式全部採[DOHC形式](https://zh.wikipedia.org/wiki/顶置凸轮轴#双顶置凸轮轴（DOHC） "wikilink")，至於汽門數目則採用相當於進氣2、排氣2的4汽門方式。此族引擎多數搭載於鈴木產製的[輕型車](../Page/輕型車.md "wikilink")、[緊湊型車等](../Page/紧凑型轿车.md "wikilink")，自2013年8月份起K6A型亦供應給[英國](https://zh.wikipedia.org/wiki/英國 "wikilink")[卡特漢姆汽車](https://zh.wikipedia.org/wiki/卡特漢姆汽車 "wikilink")，以搭載於入門等級的復古[跑車](https://zh.wikipedia.org/wiki/跑車 "wikilink") 160/165上\[1\]，卡特漢姆汽車亦將其馬力輸出調高至80ps。

## K6A型

排氣量658c.c.的**K6A型**，缸徑和衝程是68.0mm×60.4mm，壓縮比可分成自然進氣（10.5：1）、[渦輪增壓](../Page/渦輪增壓器.md "wikilink")（8.4 / 8.6 / 8.9：1）、直噴渦輪增壓（9.0：1）、[CNG](../Page/壓縮天然氣.md "wikilink")（12.5：1）等。至於馬力和扭力表現則為：

  - 自然進氣
      - 54ps / 6,500rpm、6.4kg·m / 3,500rpm（[VVT](../Page/可变气门正时.md "wikilink")）
      - 54ps / 6,500rpm、6.2kg·m / 4,000rpm
      - 49ps / 5,800rpm、6.3kg·m / 4,000rpm
      - 46ps / 6,000rpm、5.8kg·m / 3,500rpm（[稀薄燃燒](https://zh.wikipedia.org/wiki/稀薄燃燒 "wikilink")）
      - 44ps / 5,500rpm、5.8kg·m / 3,500rpm
      - 37ps / 5,500rpm、5.6kg·m / 3,500rpm（[油電混合車](../Page/混合動力車輛.md "wikilink")）
  - 渦輪增壓附中冷器
      - 64ps / 6,000rpm、9.7kg·m / 3,000rpm
      - 60ps / 6,000rpm、8.5kg·m / 3,000rpm（M-渦輪增壓）
      - 64ps / 6,500rpm、10.5kg·m / 3,500rpm（S-渦輪增壓）
      - 64ps / 6,500rpm、10.5kg·m / 3,500rpm（DI缸內直噴-渦輪增壓）
      - 64ps / 6,500rpm、10.8kg·m / 3,500rpm
      - 64ps / 6,500rpm、11.0kg·m / 3,500rpm（VVT）
      - 80ps / 7,000rpm、10.9kg·m / 3,400rpm
  - CNG
      - 50ps / 6,500rpm、5.9kg·m / 3,500rpm（VVT）

## K10A型

排氣量996c.c.的**K10A型**，缸徑和衝程是68.0mm×68.6mm，分成自然進氣VVT（壓縮比10.0：1）和[渦輪增壓](../Page/渦輪增壓器.md "wikilink")（附中冷器，壓縮比8.4：1）等二種版本，其馬力輸出為：

1.  自然進氣VVT：最大馬力70ps / 7,000rpm、扭力峰值9.0kg·m / 3,500rpm。
2.  渦輪增壓：最大馬力100ps / 6,500rpm、扭力峰值12.0kg·m / 4,000rpm。

車型：

1.  1997年-1999年：[鈴木Wagon R Wide](https://zh.wikipedia.org/wiki/鈴木Solio#第一代_MA61S/MB61S型（1997年-1999年） "wikilink")

## K10B型

**K10B型**的排氣量是996c.c.，缸徑與衝程為73.0mm×79.4mm，壓縮比11.0：1，透過[VVT技術可輸出最大馬力](../Page/可变气门正时.md "wikilink")68ps / 6,000rpm、扭力峰值9.2kg·m / 4,800rpm。車型：

1.  2008年迄今：[鈴木Splash](../Page/鈴木Splash.md "wikilink")
2.  2008年-2014年：第一代[鈴木Celerio](../Page/鈴木Celerio.md "wikilink")
3.  2009年-2013年：日產Pixo
4.  2009年-2014年：[馬魯蒂Zen Estilo](https://zh.wikipedia.org/wiki/鈴木MR_Wagon#印度馬魯蒂Zen_Estilo "wikilink")
5.  2012年迄今：[鈴木Alto K10](../Page/鈴木Alto.md "wikilink")

## K10C型

## K12B型

[Suzuki_K12B_Dual_Jet.jpg](https://zh.wikipedia.org/wiki/File:Suzuki_K12B_Dual_Jet.jpg "fig:Suzuki_K12B_Dual_Jet.jpg") **K12B型**的排氣量是1,242c.c.，屬直列四缸之構造，缸徑73.0mm、衝程74.2mm，壓縮比11.0：1（Dual Jet雙噴射系統則為12.0：1\[2\]）。按不同型式，馬力輸出可分成：

1.  VVT：最大馬力90ps / 6,000rpm及最大扭力12.0kg·m / 4,400rpm
2.  進排氣VVT：最大馬力91ps / 6,000rpm及最大扭力12.0kg·m / 4,800rpm
3.  雙噴射系統：最大馬力91ps / 6,000rpm及最大扭力12.0kg·m / 4,400rpm

車型：

## K12C型

## K14B型

排氣量1,372c.c.的**K14B型**直列四缸引擎乃外銷市場專用，缸徑和衝程分別是73.0mm×82.0mm，壓縮比10.0：1，在VVT[可變氣門正時的加持下可輸出](https://zh.wikipedia.org/wiki/可變氣門正時 "wikilink")95ps / 6,000rpm的最大馬力、13.2kg·m / 4,000rpm的扭力峰值。此外，原廠與[中國](../Page/中國.md "wikilink")合資的[昌河鈴木衍生出下列亞型](../Page/昌河汽车.md "wikilink")：

  - **K14B-D型**：具有VVT可變氣門正時技術，搭載於昌河鈴木北斗星，最大馬力71Kw / 6,000rpm，扭力峰值120N·m / 3,600rpm。
  - **K14B-F型**：搭載於昌河福瑞達M50S，最大馬力71Kw / 6,000rpm，扭力峰值128N·m / 4,000rpm。
  - **K14B-G型**：搭載於昌河鈴木利亞納及派喜，最大馬力74Kw / 6,000rpm，扭力峰值135N·m / 3,600rpm。

## K14C-DITC型

排氣量1,373c.c.的**K14C-DITC型**直列四缸引擎乃外銷市場專用，原廠稱作「booster jet」。缸徑和衝程分別是73.0mm×82.0mm，壓縮比9.9：1，在[渦輪增壓與缸內直噴技術的加持下可輸出最大馬力](../Page/渦輪增壓器.md "wikilink")140ps / 5,500rpm、最大扭力22.4kg·m / 1,700-4,000rpm。車型：

1.  2015年迄今：[第四代鈴木Vitara](https://zh.wikipedia.org/wiki/鈴木Escudo#第四代（2015年迄今） "wikilink")

## K15B型

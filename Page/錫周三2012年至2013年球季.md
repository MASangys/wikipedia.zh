**錫周三2012年至2013年球季**（）是[谢周三足球俱乐部自](../Page/谢周三足球俱乐部.md "wikilink")1892年加入[英格蘭足球聯賽後第](https://zh.wikipedia.org/wiki/英格蘭足球聯賽 "wikilink")110個球季，是自[去年取得](../Page/錫周三2011年至2012年球季.md "wikilink")[英甲亞軍直接升班後](https://zh.wikipedia.org/wiki/英甲 "wikilink")，重返第二級聯賽的首個球季。

## 記事

  - 2012年7月5日助理領隊特里·伯顿（Terry Burton）重返母會[-{zh-hans:阿森纳;
    zh-hk:阿仙奴;}-出任預備隊教練](https://zh.wikipedia.org/wiki/阿森纳足球俱乐部 "wikilink")\[1\]。
  - 2012年12月20日聘請史超活·格雷（Stuart Gray ）加入教練團隊\[2\]。

## 球衣

本季球衣繼續由[彪馬](https://zh.wikipedia.org/wiki/彪馬 "wikilink")（Puma）製造及供應，主場是傳統的藍白直間，改回襯黑褲藍襪；作客為全黃款式，配有藍袖；保留去季作客球衣作為第三選擇。而球衣[贊助](https://zh.wikipedia.org/wiki/贊助 "wikilink")[廣告則仍然與同城的](../Page/廣告.md "wikilink")[-{zh-hans:谢菲尔德联;
zh-hk:錫菲聯;}-共同合作](../Page/谢菲尔德联足球俱乐部.md "wikilink")，主場為汽车代理商「Gilder
Honda」，而作客球衣為醫療保險公司「Westfield Health」。  |  |  |  |}

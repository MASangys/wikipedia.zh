**錫蘭象**（[学名](https://zh.wikipedia.org/wiki/学名 "wikilink")：）又名**斯里蘭卡象**，是体型最大的[亞洲象亞種](https://zh.wikipedia.org/wiki/亞洲象 "wikilink")，也是亞洲象的指名亞種。

## 食物與生活習性

錫蘭象是[食草動物](https://zh.wikipedia.org/wiki/食草動物 "wikilink")，牠主要食[草](https://zh.wikipedia.org/wiki/草 "wikilink")，[葉](../Page/葉.md "wikilink")，[樹皮](https://zh.wikipedia.org/wiki/樹皮 "wikilink")，掉落在地上的[果實](https://zh.wikipedia.org/wiki/果實 "wikilink")、棕櫚葉（如椰樹樹葉）。長大後每日需要吃大約 200 [公斤食物](https://zh.wikipedia.org/wiki/公斤 "wikilink")。

## 分佈與保育

錫蘭象歷史分佈区包括所有[斯里蘭卡](../Page/斯里蘭卡.md "wikilink")的生態區域：低地熱帶雨林、山區森林及錫蘭乾燥地區的森林，雖然亦有零星分佈在熱帶雨林，但主要生活在山區森林。

錫蘭象現正瀕臨[絕種](https://zh.wikipedia.org/wiki/絕種 "wikilink")。當地的[象牙](../Page/象牙.md "wikilink")貿易已有2000年歷史。由1800年至1900年初，很多錫蘭象被人大量捕殺。1999年至2006年底期間，每年都有接近100隻野生大象被獵殺。現在只有很少野生錫蘭象住在保护区外。只有5%到7%的野生大象有象牙。

| 年份   | 野生大象數量           |
| ---- | ---------------- |
| 1800 | 12000-14000\[1\] |
| 1900 | 10000-12000      |
| 1920 | 7000-8000        |
| 1970 | 5000             |
| 1999 | 4000             |
| 2003 | 3500             |
| 2004 | 3350             |
| 2006 | 3150             |
| 2007 | 2900-3000        |

斯里蘭卡设立了多个國家公園及自然保护区来保護野生動物，包括錫蘭象。平納維拉大象孤兒院醫治受傷大象，養育失去母親的幼象。大約70隻大象住在此孤兒院。\[2\]

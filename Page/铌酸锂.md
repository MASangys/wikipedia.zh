**铌酸锂**（化学式：LiNbO<sub>3</sub>）是一种[偏铌酸盐](../Page/铌酸盐.md "wikilink")。其单晶是光波导，移动电话，压电传感器，光学调制器和各种其它线性和非线性光学应用的重要材料。

## 性质

铌酸锂是一种不溶于水的无色固体。它具有一个缺乏[反演对称性的](https://zh.wikipedia.org/wiki/反演对称性 "wikilink")[三方晶系](../Page/三方晶系.md "wikilink")，并显示出[铁电性](../Page/铁电性.md "wikilink")、[泡克耳斯效应](https://zh.wikipedia.org/wiki/泡克耳斯效应 "wikilink")、[压电效应和](https://zh.wikipedia.org/wiki/压电效应 "wikilink")[光弹性](../Page/光弹性.md "wikilink")。铌酸锂具有负的单轴[双折射](https://zh.wikipedia.org/wiki/双折射 "wikilink")，这略微取决于温度以及晶体的[化学计量比](https://zh.wikipedia.org/wiki/化学计量比 "wikilink")。它可以透过波长为350至5200[纳米](../Page/纳米.md "wikilink")之间的波。

铌酸锂可通过掺杂[氧化镁](../Page/氧化镁.md "wikilink")在它的[光学损伤阈上](https://zh.wikipedia.org/wiki/光学损伤阈 "wikilink"),来增强它对光损伤的抗性（这种损伤常称为光反射损伤）。其他可用的掺杂物还有、、、、、、、和。

## 生长

铌酸锂的[单晶可以使用](https://zh.wikipedia.org/wiki/单晶 "wikilink")[柴可拉斯基法](../Page/柴可拉斯基法.md "wikilink")制作。\[1\]
[Lithium_Niobate_Wafer.jpg](https://zh.wikipedia.org/wiki/File:Lithium_Niobate_Wafer.jpg "fig:Lithium_Niobate_Wafer.jpg")在晶体生长完之后，它会被在不同方向上切成薄片。常见的取向有Z向，X向，Y向，以及沿上述几个轴旋转一定角度切割。\[2\]

## 纳米粒子

铌酸锂和[五氧化二铌](../Page/五氧化二铌.md "wikilink")的纳米颗粒可在低温下制备。\[3\]完整的实验报告表明NbCl<sub>5</sub>被[LiH还原](https://zh.wikipedia.org/wiki/LiH "wikilink")，随后原位自发氧化成低价铌纳米氧化物。这些铌氧化物暴露于空气气氛从而产生纯的Nb<sub>2</sub>O<sub>5</sub>。最后，稳定的Nb<sub>2</sub>O<sub>5</sub>在过量的LiH的受控水解中被转化为铌酸锂LiNbO<sub>3</sub>纳米颗粒。\[4\]
直径约为10nm的铌酸锂球形纳米颗粒可以通过将孔状二氧化硅基质浸渍于LiNO<sub>3</sub>和
NH<sub>4</sub>NbO(C<sub>2</sub>O<sub>4</sub>)<sub>2</sub>的混合水溶液中，接着在IR炉中加热10分钟来制备。\[5\]

## 应用

铌酸锂被广泛使用在电信市场，比如在[移动电话](../Page/移动电话.md "wikilink")和中。它是制造[表面声波设备的可选材料](https://zh.wikipedia.org/wiki/表面声波 "wikilink")。对于某些用途，它可以被，所取代。其他用途是在[激光](../Page/激光.md "wikilink")、[非线性光学](https://zh.wikipedia.org/wiki/非线性光学 "wikilink")、[泡克耳斯盒](https://zh.wikipedia.org/wiki/泡克耳斯效应 "wikilink")、、[Q开关激光器以及其它](https://zh.wikipedia.org/wiki/Q开关 "wikilink")[声光效应器件](https://zh.wikipedia.org/wiki/声光效应 "wikilink")、千兆赫频率[光开关](../Page/光开关.md "wikilink")等。它是用于制造的优异材料。

它也被用于光学的空间低通（[抗混叠](https://zh.wikipedia.org/wiki/抗混叠 "wikilink")）滤波器的制作。

## 周期性极化铌酸锂（PPLN）

**周期性极化铌酸锂**（**PPLN**）是一种以磁畴为单位设计制造的铌酸锂晶体，主要用于实现[非线性光学的](https://zh.wikipedia.org/wiki/非线性光学 "wikilink")[准相位匹配](../Page/准相位匹配.md "wikilink")。[铁电体畴交替指向](https://zh.wikipedia.org/wiki/铁电体 "wikilink")*+c*和*-c*方向，周期通常为5至35[µm](../Page/微米.md "wikilink")。该范围中周期较短的铁电体被用于，而周期较长的则被用于[光参量振荡](../Page/光参量振荡器.md "wikilink")。可以通过使用周期性结构的电极进行电极化来实现。晶体受控加热可以用于在介质中微调[相位匹配](https://zh.wikipedia.org/wiki/相位匹配 "wikilink")，这是因为分散体随温度变化而微小变化。

周期性极化使用铌酸锂的非线性张量最大值，d<sub>33</sub>= 27
pm/V。准相位匹配给出的最大效率为完整的d<sub>33</sub>的2/π（64%），约为17
pm/V。

用于的其它材料是宽[带隙无机晶体状的](https://zh.wikipedia.org/wiki/带隙 "wikilink")[KTP](https://zh.wikipedia.org/wiki/KTP "wikilink")（可以产生周期性极化KTP，即PPKTP）、，以及一些有机材料。

周期性极化技术也可以用来形成表面的。\[6\]\[7\]

然而，由于它的低光反射破坏阈值，PPLN只在非常低的功率水平发现有限的应用。不过掺杂氧化镁的铌酸锂是通过周期性极化的方法制造的。因此，周期性极化的掺杂氧化镁的铌酸锂（PPMgOLN）将应用扩大到中等功率水平。

## 塞耳迈耶尔方程

关于非寻常波折射率的[塞耳迈耶尔方程被用于找到晶体的掺杂周期和准位相匹配的大致温度Jundt](https://zh.wikipedia.org/wiki/塞耳迈耶尔方程 "wikilink")\[8\]的结论是

\(n^2_e =
 5.35583 + 4.629 \times 10^{-7} f
+ {0.100473 + 3.862 \times 10^{-8} f \over \lambda^2 - (0.20692 - 0.89 \times 10^{-8} f)^2 }
+ { 100 + 2.657 \times 10^{-5} f \over \lambda^2 - (11.34927 )^2 }
- 1.5334 \times 10^{-2} \lambda^2\)

这一公式在温度20\~250°C，波长0.4\~5[微米](../Page/微米.md "wikilink")范围内适用，对于更长的波长，有\[9\]

\(n^2_e =
 5.39121 + 4.968 \times 10^{-7} f
+ {0.100473 + 3.862 \times 10^{-8} f \over \lambda^2 - (0.20692 - 0.89 \times 10^{-8} f)^2 }
+ { 100 + 2.657 \times 10^{-5} f \over \lambda^2 - (11.34927 )^2 }- (1.544 \times 10^{-2} + 9.62119 \times 10^{-10} \lambda) \lambda^2\)

其适用范围是*T*=25\~180°C波长λ在2.8到4.8微米之间。

上述方程中*f*=（*T*-24.5）（*T*+570.82），λ单位是微米，*T*单位是摄氏度。

一个更加综合且适合各种掺杂MgO比例的LiNbO3的非寻常波折射率公式是:

\(n^2 =
 a_1 + b_1 f
+ {a_2 + b_2 f \over \lambda^2 - (a_3 + b_3 f)^2 }
+ {a_4 + b_4 f \over \lambda^2 - a_5^2 }
- a_6 \lambda^2\),

其中：

|                 |                       |                         |
| --------------- | --------------------- | ----------------------- |
| 因素              | 5%MgO掺杂（CLN）          | 1%MgO掺杂（SLN）            |
|                 | *n*<sub>e</sub>       | *n*<sub>o</sub>         |
| *a*<sub>1</sub> | 5.756                 | 5.653                   |
| *a*<sub>2</sub> | 0.0983                | 0.1185                  |
| *a*<sub>3</sub> | 0.2020                | 0.2091                  |
| *a*<sub>4</sub> | 189.32                | 89.61                   |
| *a*<sub>5</sub> | 12.52                 | 10.85                   |
| *a*<sub>6</sub> | 1.32×10<sup>-2</sup>  | 1.97×10<sup>-2</sup>    |
| *b*<sub>1</sub> | 2.860×10<sup>-6</sup> | 7.941×10<sup>-7</sup>   |
| *b*<sub>2</sub> | 4.700×10<sup>-8</sup> | 3.134×10<sup>-8</sup>   |
| *b*<sub>3</sub> | 6.113×10<sup>-8</sup> | \-4.641×10<sup>-9</sup> |
| *b*<sub>4</sub> | 1.516×10<sup>-4</sup> | \-2.188×10<sup>-6</sup> |

这些数据仅对符合对应化学计量比的铌酸锂适用。\[10\]

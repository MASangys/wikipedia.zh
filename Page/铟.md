**銦**是一種化學[元素](https://zh.wikipedia.org/wiki/元素 "wikilink")，[符號為In](https://zh.wikipedia.org/wiki/符號 "wikilink")，[原子序數](https://zh.wikipedia.org/wiki/原子序數 "wikilink")49。銦是[鹼金屬除外最柔軟的金屬](https://zh.wikipedia.org/wiki/鹼金屬 "wikilink")，外觀如[錫般呈銀白色](https://zh.wikipedia.org/wiki/錫 "wikilink")，它是一種[後過渡金屬](https://zh.wikipedia.org/wiki/後過渡金屬 "wikilink")，在地殼中的成分佔0.21[ppm](https://zh.wikipedia.org/wiki/ppm "wikilink")。銦的熔點比[鈉和](https://zh.wikipedia.org/wiki/鈉 "wikilink")[鎵高](https://zh.wikipedia.org/wiki/鎵 "wikilink")，但低於[鋰和](https://zh.wikipedia.org/wiki/鋰 "wikilink")[錫](https://zh.wikipedia.org/wiki/錫 "wikilink")。化學上，銦類似鎵和鉈，其性質主要介於兩者之間\[1\]。銦在1863年由[斐迪南˙賴希和](https://zh.wikipedia.org/wiki/斐迪南˙賴希 "wikilink")[希羅尼穆斯˙特奧多爾˙里赫特透過](https://zh.wikipedia.org/wiki/希羅尼穆斯˙特奧多爾˙里赫特 "wikilink")[光譜方法發現](https://zh.wikipedia.org/wiki/光譜方法 "wikilink")，他們因為其靛藍色的譜線而如此命名，隔年，銦才被分離出來。

銦是[硫化鋅礦石的次要成分](https://zh.wikipedia.org/wiki/硫化鋅 "wikilink")，且為[鋅精煉的副產物](https://zh.wikipedia.org/wiki/鋅 "wikilink")。它主要運用在[半導體工業](https://zh.wikipedia.org/wiki/半導體工業 "wikilink")、低熔點金屬[合金如焊料](../Page/合金.md "wikilink")、軟金屬高真空密封墊以及製造塗覆在玻璃上的透明氧化銦錫(ITO)導電膜。銦是一種[科技關鍵元素](https://zh.wikipedia.org/wiki/科技關鍵元素 "wikilink")。

雖然銦的化合物在注入血液時有些毒性，它並無生物角色。大多數的職業暴露是經由攝入與吸入，前者無法良好吸收，後者則會中度吸收。

## 性質

### 物理

銦是一種銀白色、高延展性且帶有光澤的後過渡金屬\[2\]。它質軟（[莫氏硬度](https://zh.wikipedia.org/wiki/莫氏硬度 "wikilink")1.2），如鈉般可以用刀切割，也會在紙上留下可見的線\[3\]。它是[週期表](https://zh.wikipedia.org/wiki/週期表 "wikilink")[第13族的成員](https://zh.wikipedia.org/wiki/第13族 "wikilink")，性質主要介於與它上下比鄰的[鎵和](https://zh.wikipedia.org/wiki/鎵 "wikilink")[鉈之間](https://zh.wikipedia.org/wiki/鉈 "wikilink")。如同[錫](https://zh.wikipedia.org/wiki/錫 "wikilink")，當銦被彎曲時，可聽見高音頻的[鳴聲](https://zh.wikipedia.org/wiki/鳴聲 "wikilink")──由於[孿晶](https://zh.wikipedia.org/wiki/孿晶 "wikilink")\[4\]的爆裂聲；如同鎵，銦能浸潤玻璃，而與兩者相同的是，銦具有低[熔點](https://zh.wikipedia.org/wiki/熔點 "wikilink")156.60°C
(313.88 °F)，比同族中較輕的鎵高，但比同族中較重的鉈低，也比錫來得低\[5\]。沸點為2072 °C (3762
°F)，比鉈高但低於鎵，此與熔點的普遍趨勢相反，但與其他後過渡金屬族的趨勢相同，原因是具有較少數未定域電子而造成的弱金屬鍵\[6\]。

銦的密度是7.31 g/cm<sup>3</sup>，亦大於鎵但小於鉈。當低於臨界溫度3.41
K時，銦會成為[超導體](../Page/超導體.md "wikilink")。銦以體心[四方晶系形成結晶體](../Page/四方晶系.md "wikilink")，其[空間群為I](https://zh.wikipedia.org/wiki/空間群 "wikilink")4/mmm
(lattice parameters: a = 325 pm, c = 495
pm):\[7\]此為輕微扭曲的[面心立方結構](https://zh.wikipedia.org/wiki/面心立方 "wikilink")，即每個銦原子有四個相距324pm和八個相距較遠(336pm)的相鄰原子\[8\]。銦在液態汞中擁有比任何其他金屬更佳的溶解度（在0°C大於50%的質量百分率）\[9\]。銦展現了延展黏塑的反應，其張力或壓縮與尺寸大小無關。然而，它在彎曲和壓痕時確有[尺寸效應](https://zh.wikipedia.org/wiki/尺寸效應 "wikilink")，與長度規模50–100
µm的量級有關\[10\]，遠比其他金屬大。

### 化學

銦有49個電子，電子組態是[Kr](https://zh.wikipedia.org/wiki/[krypton "wikilink")\]4d<sup>10</sup>5s<sup>2</sup>5p<sup>1</sup>。
在化合物中，銦最常提供最外層的三顆電子成為銦(III)（In3+）。在一些例子中，5s的電子對並未提供而成為銦(I)（In+），此[單價態的穩定性歸因於](https://zh.wikipedia.org/wiki/單價 "wikilink")[惰性電子對效應](https://zh.wikipedia.org/wiki/惰性電子對效應 "wikilink")，即在較重的元素中可觀察到因[相對論效應使](https://zh.wikipedia.org/wiki/相對論效應 "wikilink")5s軌域穩定，鉈（較銦重的同族元素）展現了甚至更強的效應，使得氧化成鉈(I)比氧化成鉈(III)更可能發生\[11\]
，然而鎵（較銦輕的同族元素）通常則只以+3的氧化態出現，因此，雖然鉈(III)是中強的[氧化劑](https://zh.wikipedia.org/wiki/氧化劑 "wikilink")，銦(III)並不是，而且許多銦(I)的化合物是強大的[還原劑](https://zh.wikipedia.org/wiki/還原劑 "wikilink")\[12\]。
儘管銦將s-電子囊括在化學鍵中所需的能量是13族金屬中最低的，但因鍵能在同族中越往下越減少，導致形成額外兩個鍵所得到的能量，很多時候並不足以補足失去5s-電子生成+3氧化態所需的能量\[13\]。氧化銦(I)和氫氧化銦(I)是較為鹼性的，而氧化銦(III)和氫氧化銦(III)則較為酸性\[14\]。

在各種不同反應中\[15\]研究所得之銦的各種[標準電極電位數值](https://zh.wikipedia.org/wiki/標準電極電位 "wikilink")，反映了+3氧化態降低的穩定性\[16\]：

|         |                                   |                   |
| ------- | --------------------------------- | ----------------- |
| −0.40   | In<sup>2+</sup> + e<sup>−</sup>   | ↔ In<sup>+</sup>  |
| −0.49   | In<sup>3+</sup> + e<sup>−</sup>   | ↔ In<sup>2+</sup> |
| −0.443  | In<sup>3+</sup> + 2 e<sup>−</sup> | ↔ In<sup>+</sup>  |
| −0.3382 | In<sup>3+</sup> + 3 e<sup>−</sup> | ↔ In              |
| −0.14   | In<sup>+</sup> + e<sup>−</sup>    | ↔ In              |

銦金屬不與水反應，但是會被較強的氧化劑如鹵素氧化成銦(III)化合物。它不形成[硼化物](https://zh.wikipedia.org/wiki/硼化物 "wikilink")、[矽化物或](../Page/矽化物.md "wikilink")[碳化物](../Page/碳化物.md "wikilink")，而氫化物氫化銦(III)（InH<sub>3</sub>）頂多在低溫的醚溶液中短暫存在，相當不穩定以至於會自發性的無配位聚合\[17\]
。銦在水溶液中相當鹼性，只展現些微的兩性特性，並且不同於它較輕的同族元素鋁和鎵，在鹼性溶液中是難溶的\[18\]。

### 同位素

銦有39個已知的同位素，分布在[質量數](../Page/質量數.md "wikilink")97至135。只有兩個同位素以[原始核種自然出現](https://zh.wikipedia.org/wiki/原始核種 "wikilink")：銦-113以及銦-115，前者是唯一的穩定同位素，後者則具有4.41年的半衰期，規模較宇宙年齡大4個數量級，比[天然釷要大將近](https://zh.wikipedia.org/wiki/天然釷 "wikilink")30000倍\[19\]
，<sup>115</sup>In的半衰期之所以極長是因為生成<sup>115</sup>Sn的β衰變是[自旋禁止的](https://zh.wikipedia.org/wiki/自旋禁止 "wikilink")\[20\]。銦-115占了全部銦的95.7%。銦是穩定同位素在自然界的豐度少於長壽原始放射性同位素的三種已知元素之一（其他是[碲和](../Page/碲.md "wikilink")[錸](https://zh.wikipedia.org/wiki/錸 "wikilink")）\[21\]。

最穩定的[人造同位素是銦](https://zh.wikipedia.org/wiki/人造 "wikilink")-111，擁有半衰期約2.8天，其他所有同位素的半衰期都少於5小時。銦也具有47個介態，其中銦-114m1（半衰期約49.51天）是最穩定的，除了原始同位素外，比其他銦同位素的基態都穩定。所有的這種銦同位素透過[核異構素躍遷進行衰變](https://zh.wikipedia.org/wiki/核異構素躍遷 "wikilink")。比<sup>115</sup>In輕的銦同位素主要以[電子捕捉和](https://zh.wikipedia.org/wiki/電子捕捉 "wikilink")[正子發射衰變成](https://zh.wikipedia.org/wiki/正子發射 "wikilink")[鎘同位素](https://zh.wikipedia.org/wiki/鎘 "wikilink")，而<sup>115</sup>In與更重者則主要以[β−衰變形成錫同位素](https://zh.wikipedia.org/wiki/β−衰變 "wikilink")\[22\]。

## 化合物

### 銦(III)

氧化銦(III)（In<sub>2</sub>O<sub>3</sub>）是由銦金屬在空氣中燃燒，或其氫氧化物或硝酸鹽加熱形成\[23\]。氧化銦具有類似[鋁的結構且為兩性的](https://zh.wikipedia.org/wiki/鋁 "wikilink")，即與酸和鹼皆可以反應。銦與水反應生成可溶性的氫氧化銦(III)，同樣也是兩性的；與鹼金屬產生銦酸(III)鹽；與酸則產生銦(III)鹽。

  -
    In(OH)<sub>3</sub> + 3 HCl → InCl<sub>3</sub> + 3 H<sub>2</sub>O

和硫、硒、碲所形成之類似的倍半硫屬化合物也是已知的\[24\]。銦生成預期中的三鹵化物，其氯化、溴化和碘化會生成無色的
[InCl<sub>3</sub>](https://zh.wikipedia.org/wiki/indium\(III\)_chloride "wikilink")、[InBr<sub>3</sub>以及黃色的InI](https://zh.wikipedia.org/wiki/indium\(III\)_bromide "wikilink")<sub>3</sub>。這些化合物是[路易士酸](https://zh.wikipedia.org/wiki/路易士酸 "wikilink")，類似為人熟知的三鹵化鋁。就如同相近的鋁化合物，InI<sub>3</sub>是聚合體\[25\]。
銦與[氮族元素直接反應會生成灰色或半金屬III](../Page/氮族元素.md "wikilink")-IV的[半導體](https://zh.wikipedia.org/wiki/半導體 "wikilink")，他們之中有許多會在潮濕空氣中緩慢分解，使得這些半導體化合物需要謹慎的儲藏，以避免和大氣接觸。氮化銦容易被酸和鹼攻擊\[26\]。

### 銦(I)

銦(I)的化合物並不常見，其氯化物、[溴化物和碘化物都是深色的](https://zh.wikipedia.org/wiki/溴化物 "wikilink")，與製備來源的三鹵化物不同；氟化物只以不穩定的氣態化合物為人所知\[27\]
。黑色的粉狀氧化銦(I)是在氧化銦(III)加熱至700 °C時分解產生\[28\]。

### 其他氧化態

較少見的，銦會以+2甚至是分數型的氧化態形成化合物。通常此類材料以銦－銦鍵結為特色，最顯著的是在鹵化物In<sub>2</sub>X<sub>4</sub>
與\[In<sub>2</sub>X<sub>6</sub>\]<sup>2−</sup>中\[29\]
以及各種副硫屬化合物如In<sub>4</sub>Se<sub>3</sub>.\[30\]。許多其他化合物已知會與銦(I)和銦(III)結合，如In<sup>I</sup><sub>6</sub>(In<sup>III</sup>Cl<sub>6</sub>)Cl<sub>3</sub>、\[31\]
In<sup>I</sup><sub>5</sub>(In<sup>III</sup>Br<sub>4</sub>)<sub>2</sub>(In<sup>III</sup>Br<sub>6</sub>)和\[32\]
In<sup>I</sup>In<sup>III</sup>Br<sub>4</sub>.\[33\]。

### 有機銦化合物

有機銦化合物以銦－碳鍵結為特色，多數是銦(III)的衍生物，但[環戊二烯銦(I)是例外](https://zh.wikipedia.org/wiki/環戊二烯銦\(I\) "wikilink")，它是第一個被知道的有機銦(I)化合物\[34\]
，且是聚合型的，由交替的銦原子和[環戊二烯錯合物的鋸齒狀鏈組成](https://zh.wikipedia.org/wiki/環戊二烯錯合物 "wikilink")\[35\]
。可能最為人熟知的有機銦化合物是三甲基銦（In(CH<sub>3</sub>)<sub>3</sub>），用來製備某些半導體材料\[36\]\[37\]。

## 歷史

1863年，德國化學家[斐迪南˙賴希和](https://zh.wikipedia.org/wiki/斐迪南˙賴希 "wikilink")[希羅尼穆斯˙特奧多爾˙里赫特在試驗來自薩克森](https://zh.wikipedia.org/wiki/希羅尼穆斯˙特奧多爾˙里赫特 "wikilink")[弗類貝格周圍礦井的礦石](https://zh.wikipedia.org/wiki/弗類貝格 "wikilink")，他們將礦物[黃鐵礦](https://zh.wikipedia.org/wiki/黃鐵礦 "wikilink")、[砷黃鐵礦](https://zh.wikipedia.org/wiki/砷黃鐵礦 "wikilink")、[方鉛礦和](https://zh.wikipedia.org/wiki/方鉛礦 "wikilink")[閃鋅礦溶解在](https://zh.wikipedia.org/wiki/閃鋅礦 "wikilink")[氫氯酸中並提純粗](https://zh.wikipedia.org/wiki/氫氯酸 "wikilink")[氯化鋅](https://zh.wikipedia.org/wiki/氯化鋅 "wikilink")。賴希是[色盲](../Page/色盲.md "wikilink")，所以他雇用里赫特作助手以偵測有色譜線。由於他們知道源自該區的礦石有時含[鉈](https://zh.wikipedia.org/wiki/鉈 "wikilink")，於是想尋找綠色的鉈放射光譜線，但取而代之的，卻找到了一條明亮的藍線，因為該藍線並不符合任何已知元素，他們於是假定在礦物中出現了新的元素。他們命名此元素為銦，源自在光譜中看見的[靛藍色](https://zh.wikipedia.org/wiki/靛藍 "wikilink")，從拉丁文indicum而來，意思是印度。\[38\]\[39\]\[40\]\[41\]
1864年，里赫特繼續分離此金屬\[42\]。在1867年的[世界博覽會](../Page/世界博覽會.md "wikilink")，展出了一錠0.5公斤的銦塊\[43\]
。後來，由於里赫特自稱是唯一的發現者，賴希和里赫特就此失和。\[44\]

**银氰酸钾**是无机化合物，为无色晶体，易溶于水。

## 制备

  - 在[氧化剂](../Page/氧化剂.md "wikilink")的存在下，用[氰化钾](../Page/氰化钾.md "wikilink")溶液将银溶解：

\[\mathsf{4Ag + 8KCN + O_2 + 2H_2O \ \xrightarrow{}\ 4K[Ag(CN)_2] + 4KOH }\]

\[\mathsf{2Ag + 4KCN + H_2O_2 \ \xrightarrow{}\ 2K[Ag(CN)_2] + 2KOH }\]

  - [氰化钾](../Page/氰化钾.md "wikilink")溶液溶解[氯化银](../Page/氯化银.md "wikilink")：

\[\mathsf{AgCl + 2KCN \ \xrightarrow{}\ K[Ag(CN)_2] + KCl }\]

## 化学性质

  - 加热分解：

\[\mathsf{2K[Ag(CN)_2] \ \xrightarrow{250-420^oC}\ 2KCN + 2Ag + C_2N_2\uparrow }\]

  - 与强酸反应：

\[\mathsf{2K[Ag(CN)_2] + 2H_2SO_4 \ \xrightarrow{80^oC}\ Ag_2SO_4\downarrow + K_2SO_4 + 4HCN\uparrow }\]

  - 复分解：

\[\mathsf{2K[Ag(CN)_2] + K_2S \ \xrightarrow{}\ Ag_2S\downarrow + 4KCN }\]

  - 置换出金属银：

\[\mathsf{2K[Ag(CN)_2] + Zn \ \xrightarrow{}\ K_2[Zn(CN)_4] + 2Ag\downarrow }\]

## 用途

  - [镀银时的电解质](https://zh.wikipedia.org/wiki/镀银 "wikilink")。
  - 作为杀菌剂，生产中的防腐剂。\[1\]
  - 化学合成。

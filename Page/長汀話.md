**長汀話**（）是[客家語的一個方言](https://zh.wikipedia.org/wiki/客家語 "wikilink")，屬的[汀州片](https://zh.wikipedia.org/wiki/汀州片 "wikilink")，主要通行於中國[福建省](../Page/福建省.md "wikilink")[龍巖市的](https://zh.wikipedia.org/wiki/龍巖市 "wikilink")[長汀縣一帶](https://zh.wikipedia.org/wiki/長汀縣 "wikilink")。

長汀話以城關口音為代表標準方言，全縣各地都通用。在歷史上，長汀縣曾是[汀州府的駐地](https://zh.wikipedia.org/wiki/汀州府 "wikilink")，因此大多數學者把長汀話當作客家語汀州片（閩西客家語）的代表方言，與[上杭話](https://zh.wikipedia.org/wiki/上杭話 "wikilink")、[武平話](https://zh.wikipedia.org/wiki/武平話 "wikilink")、[永定話皆可互通](https://zh.wikipedia.org/wiki/永定話 "wikilink")，但和[連城話溝通存在一定困難](https://zh.wikipedia.org/wiki/連城話 "wikilink")。

與其他一些漢語方言不同的是，長汀話在近七八十年以來並無明顯的變化，老年人與年輕人的讀音大致相同，唯部份字詞出現了新派讀音。

## 內部差異

長汀境內方言，可以按照入聲韻尾的有無及其他語音特點，可分為兩大片、五個小區。長汀城關話為代表口音，全縣城鄉之間基本可以通話。

| 分片       | 範圍                    |
| -------- | --------------------- |
| 東北片（無入聲） | 城區                    |
| 汀東區      | 包括館前、童坊、新橋、庵傑、鐵長五個鄉鎮。 |
| 半溪區      | 包括南山鄉的橋下、鐘屋、官坊、塘背等村。  |
| 西南片（有入聲） | 汀南區                   |
| 汀西區      | 古城鄉                   |

## 語音特點

### 聲母

1.  古全濁塞音與塞擦音，不論平仄今音絕大部分讀成送氣清音。
2.  一部分古非組字念重唇音。
3.  古精、知二、莊組讀，知三、章組讀。
4.  分尖團音，精組細音前讀，見組細音前則讀。
5.  古來母三等一部分字讀。

### 韻母

汀北片沒有前鼻音韻尾，只有後鼻音尾，汀南片則兼具兩者。

### 新派老派之分

長汀話沒有明顯的新派或老派的差異，只有少數字音，年輕人有些不同，如：

| 例子     | 老派 | 新派 |
| ------ | -- | -- |
| 戰爭的“戰” | 讀  | 讀  |
| 清朝的“清” | 讀  | 讀  |
| 抽煙     | 食煙 | 抽煙 |
| 開水     | 滾水 | 開水 |

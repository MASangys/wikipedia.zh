**長沙灣站**（）是[港鐵](../Page/港鐵.md "wikilink")[荃灣綫](../Page/荃灣綫.md "wikilink")位於九龍[深水埗區](../Page/深水埗區.md "wikilink")[長沙灣](../Page/長沙灣.md "wikilink")[長沙灣道](../Page/長沙灣道.md "wikilink")的一個地底車站，於1982年5月17日啟用，承建商是[西松建設](../Page/西松建設.md "wikilink")株式會社。

### 大堂

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Cheung Sha Wan Station 2017 12 part4.jpg | caption1 = 大堂 | image2 = Cheung Sha Wan Station Artwall 2015.jpg | caption2 = 大堂設由万里子於2002年12月完成的藝術品「茶壺、碗、杯及匙」 }} 長沙灣站設有不同類型的車站商店，包括[便利店](../Page/便利商店.md "wikilink")、[麵包糕餅店](https://zh.wikipedia.org/wiki/麵包糕餅店 "wikilink")、[外賣店](https://zh.wikipedia.org/wiki/外賣店 "wikilink")、[時裝店等](https://zh.wikipedia.org/wiki/時裝店 "wikilink")。站內亦設有各樣自助服務，包括[自動櫃員機](../Page/自動櫃員機.md "wikilink")、[自動售賣機和自動照相機](../Page/自動販賣機.md "wikilink")\[1\]。此外，站內設有[香港郵政](../Page/香港郵政.md "wikilink")[郵箱](../Page/郵箱.md "wikilink")，方便乘客投寄[郵件](../Page/郵件.md "wikilink")\[2\]\[3\]，以及「[會員服務站](../Page/會員服務站.md "wikilink")」，供乘客登記為港鐵友禮會會員、查閱獎分及換領獎賞\[4\]。

## 鄰接車站

長沙灣站是荃灣綫的中途站，乘客可在此乘坐往[中環或](../Page/中環站.md "wikilink")[荃灣列車](../Page/荃灣站.md "wikilink")，往返[香港島](../Page/香港島.md "wikilink")、[九龍](../Page/九龍.md "wikilink")及[新界](../Page/新界.md "wikilink")等地。

## 利用狀況

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Cheung Sha Wan Station 2017 09 part3.jpg | caption1 = 勿衝門動漫版本標語 }} 長沙灣站主要服務周邊鄰近住宅區及學校，如[元州邨](../Page/元州邨.md "wikilink")、[李鄭屋邨](../Page/李鄭屋邨.md "wikilink")、[蘇屋邨](../Page/蘇屋邨.md "wikilink")、[麗閣邨](../Page/麗閣邨.md "wikilink")、[長沙灣邨](../Page/長沙灣邨.md "wikilink")等公共屋邨，因此，使用該站的人流都是居住於鄰近屋苑的居民、中小學學生或於鄰近辦公室上班的市民。除了上下班及上下課繁忙時間外，利用該站出入閘的人流僅屬一般。

## 歷史

[Mass_Transportation_Study_1967_zh.svg](https://zh.wikipedia.org/wiki/File:Mass_Transportation_Study_1967_zh.svg "fig:Mass_Transportation_Study_1967_zh.svg")

### 命名

長沙灣站位於九龍深水埗以北至[美孚新邨](../Page/美孚新邨.md "wikilink")以南一帶的一片於19世紀末填海土地上的南面，該處原為一長長的沙灘及海邊，故此地名為[長沙灣](../Page/長沙灣.md "wikilink")。\[5\]

由於長沙灣地區發展前本為一長形海灣，因此幅距甚大，區內有不少建築物以[深水埗](../Page/深水埗.md "wikilink")或長沙灣命名，如[長沙灣道](../Page/長沙灣道.md "wikilink")、[長沙灣廣場](../Page/長沙灣廣場.md "wikilink")、[深水埗運動場](../Page/深水埗運動場.md "wikilink")、[長沙灣政府合署](../Page/長沙灣政府合署.md "wikilink")等，有些並不接近長沙灣站。亦因此令不少市民誤會，誤以為建築物名稱有「長沙灣」即等如在長沙灣站下車。故此，港鐵公司在長沙灣站內貼出告示，提示乘客如要到[長沙灣廣場](../Page/長沙灣廣場.md "wikilink")，應在荔枝角站下車。

{{ multiple image | align = right | direction = vertical | width = 220 | image1 = Cheung Sha Wan Station 2017 12 part5.jpg | caption1 = 地面升降機 }}

### 加建出入站升降機

長沙灣站自啟用以來一直沒有連接地面的[升降機](https://zh.wikipedia.org/wiki/升降機 "wikilink")，為提高車站可達度，港鐵公司首次與[房屋署](../Page/房屋署.md "wikilink")合作，於2011年5月開始在前[長沙灣工廠大廈](../Page/長沙灣工廠大廈.md "wikilink")範圍（現[元州邨](../Page/元州邨.md "wikilink")元謙樓）建造兩部連接車站大堂、地面及平台的升降機，升降機已於2012年4月26日投入服務，成為「用心聽‧用心做」計劃下首部落成啟用的升降機，也是首個與[香港公共房屋](../Page/香港公共房屋.md "wikilink")租住住宅大廈融合的港鐵站出入口\[6\]\[7\]。

## 未來發展

### 更換升降機

往來大堂及月台升降機將於2017年9月16日暫停使用，以更換為較新型號，並預計於2018年6月底重投服務\[8\]。

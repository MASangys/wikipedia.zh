**長野線**（）是一條連結[日本](../Page/日本.md "wikilink")[大阪府](https://zh.wikipedia.org/wiki/大阪府 "wikilink")[羽曳野市](../Page/羽曳野市.md "wikilink")至[河內長野市](../Page/河內長野市.md "wikilink")[河內長野站](https://zh.wikipedia.org/wiki/河內長野站 "wikilink")，屬於[近畿日本鐵道的鐵路線](../Page/近畿日本鐵道.md "wikilink")。

## 路線資料

  - 路線距離（）：12.5公里
  - [軌距](https://zh.wikipedia.org/wiki/軌距 "wikilink")：1067毫米
  - 站數：8個（包括起終點站）
  - 複線路段：古市－富田林間
  - 電氣化路段：全線電氣化（直流1500V）
  - [閉塞方式](../Page/閉塞_\(鐵路\).md "wikilink")：自動閉塞式
  - 最高速度：100公里/小時

全線由大阪統括部（舊天王寺營業局）管轄。

## 历史

  - 1898年4月14日 : 古市站 - 富田林站間開業
  - 1902年3月25日 : 富田林站 - 瀧谷不動站開業
  - 1902年12月12日 : 瀧谷不動站 - 長野站（現、河內長野站）開業兼全面通車
  - 1904年10月7日 : 学校前站（現、富田林西口站）開業
  - 1906年7月18日 : 学校前站廢止
  - 1911年6月1日 : 汐之宮站開業
  - 1911年8月1日 : 宮之前站、川西站開業
  - 1911年8月15日 : 西浦站開業
  - 1916年12月1日 : 学校前站再開業
  - 1920年4月1日 : 西浦站、宮之前站、川西站廢止
  - 1920年8月1日 : 廿山站（現、川西站）再開業
  - 1923年10月16日 : 全線電化
  - 1923年12月28日 : 旭岡站開業
  - 1945年6月1日 : 旭岡站、富田林西口站休止
  - 1946年7月1日 : 富田林西口站営業再開
  - 1957年10月18日 : 古市站 - 喜志站間複線化
  - 1974年7月20日 : 旭岡站廢止
  - 1987年10月25日 : 喜志站 - 富田林站間複線化

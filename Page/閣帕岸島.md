**閣帕岸島**（**Koh Phangan**， [泰文](https://zh.wikipedia.org/wiki/泰文 "wikilink"):เกาะพะงัน）又譯**帕岸島**，是一個位於[泰國東南方](https://zh.wikipedia.org/wiki/泰國 "wikilink")[暹邏灣中的](https://zh.wikipedia.org/wiki/暹邏灣 "wikilink")[島嶼](../Page/島嶼.md "wikilink")。在行政區劃上屬於[素叻他尼府](../Page/素叻他尼府.md "wikilink")**閣帕岸縣**。

它以[滿月派對](../Page/滿月派對.md "wikilink")而聞名全球，也是許多[自助旅行者的旅遊點](https://zh.wikipedia.org/wiki/自助旅行者 "wikilink")。帕岸島有兩個姐妹島：南邊的[閣沙梅島](https://zh.wikipedia.org/wiki/閣沙梅島_\(素叻他尼府\) "wikilink")（蘇美島）及北邊的[閣道島](../Page/閣道島.md "wikilink")（龜島）。

閣帕岸島與[印度](../Page/印度.md "wikilink")[果阿及](https://zh.wikipedia.org/wiki/果阿 "wikilink")[西班牙](../Page/西班牙.md "wikilink")[伊維薩島合稱](https://zh.wikipedia.org/wiki/伊維薩島 "wikilink")[電子音樂](../Page/電子音樂.md "wikilink")三聖地。

  - 面積：約168平方公里
  - 省份：[素叻他尼府](../Page/素叻他尼府.md "wikilink")
  - 距[蘇美島](https://zh.wikipedia.org/wiki/蘇美島 "wikilink")：約15公里
  - 距離泰國素叻府[棟莎港](https://zh.wikipedia.org/wiki/棟莎港 "wikilink")（Dongsak）：約60公里
  - 人口：11,846（2004年）
  - 主要城鎮：[通撒拉](https://zh.wikipedia.org/wiki/通撒拉 "wikilink")

## 行政區劃

閣帕岸島連同閣道島劃為閣帕岸縣（Amphoe Ko Pha Ngan）。本縣再被劃為三個區（tambon）：

| 號   | 名字   | 泰文名       |  |
| --- | ---- | --------- |  |
| 1\. | 閣帕岸島 | เกาะพะงัน |  |
| 2\. | 班泰   | บ้านใต้   |  |
| 3\. | 閣道島  | เกาะเต่า  |  |

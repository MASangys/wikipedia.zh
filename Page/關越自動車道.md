**關越自動車道**（）是起於[日本](../Page/日本.md "wikilink")[東京都](../Page/東京都.md "wikilink")[練馬區](https://zh.wikipedia.org/wiki/練馬區 "wikilink")[練馬交流道經](https://zh.wikipedia.org/wiki/練馬交流道 "wikilink")[埼玉縣](../Page/埼玉縣.md "wikilink")、[群馬縣](../Page/群馬縣.md "wikilink")至[新潟縣](https://zh.wikipedia.org/wiki/新潟縣 "wikilink")[長岡市](../Page/長岡市.md "wikilink")的[高速公路](../Page/日本高速公路.md "wikilink")（[高速自動車國道](https://zh.wikipedia.org/wiki/高速自動車國道 "wikilink")），略稱為**關越道**（，KAN-ETSU EXPWY）或**關越**（）。高速公路路線編號編為「**E17**」\[1\]。

## 概要

貫通[三國山脈](https://zh.wikipedia.org/wiki/三國山脈 "wikilink")，為連結東京與新潟縣的高速道路，與[上越新幹線](../Page/上越新幹線.md "wikilink")同為[首都圈與](../Page/首都圈_\(日本\).md "wikilink")[日本海](../Page/日本海.md "wikilink")側連結的高速交通網。分別於[藤岡JCT及](https://zh.wikipedia.org/wiki/藤岡JCT "wikilink")[高崎JCT與](https://zh.wikipedia.org/wiki/高崎JCT "wikilink")[上信越自動車道](../Page/上信越自動車道.md "wikilink")（關越自動車道上越線）及[北關東自動車道](../Page/北關東自動車道.md "wikilink")連接。

在[水上IC](https://zh.wikipedia.org/wiki/水上IC "wikilink")-[湯澤IC之間的](https://zh.wikipedia.org/wiki/湯澤IC_\(新潟縣\) "wikilink")[關越隧道](../Page/關越隧道.md "wikilink")為全日本第二長的[公路隧道](https://zh.wikipedia.org/wiki/公路隧道 "wikilink")，僅次於[首都高速中央環狀線的](https://zh.wikipedia.org/wiki/首都高速中央環狀線 "wikilink")[山手隧道](../Page/山手隧道_\(東京都\).md "wikilink")。

## 設施

  - 交流道（IC）編號欄背景色<span style="color:#BFB">■</span>區間是已通車區間，設施欄背景色<span style="color:#CCC">■</span>區間是未通車區間與未啟用設施，未通車區間名稱全部是暫稱。
  - (數字)為其他路線的編號，<數字>為預定編號。
  - [智能IC](https://zh.wikipedia.org/wiki/智能IC "wikilink")（SIC）背景色為<span style="color:#eda5ff">■</span>。
  - 路線名無特別註明者為[市道](https://zh.wikipedia.org/wiki/市町村道 "wikilink")。
  - BS（巴士站）：○為使用中、◆為停用中，未註記者即未設置。
  - 停车区：停車場，SA：服務區，IC：交流道，JCT：系統交流道。

## 歷史

  - 1963年7月20日 : 關越自動車道路線基準制定。
  - 1966年7月1日 : 關越自動車道為[國土開發幹線自動車道](../Page/國土開發幹線自動車道.md "wikilink")預定路線。
  - 1971年12月20日 : 國道254號**東京川越道路**練馬IC-川越IC間通車。
  - 1973年4月1日 : 東京川越道路昇格編入**關越自動車道**。
  - 1975年8月8日 : 川越IC-東松山IC間通車。
  - 1978年9月21日 : 長岡IC-新潟黑埼IC（現[新潟西繞道](https://zh.wikipedia.org/wiki/新潟西繞道 "wikilink")[黑埼IC](https://zh.wikipedia.org/wiki/黑埼IC "wikilink")）間通車。
  - 1980年7月17日 : 東松山IC-前橋IC間、藤岡JCT-藤岡IC間通車。
  - 1980年9月27日 : 長岡JCT通車，與北陸自動車道連接。
  - 1982年3月30日 : 越後川口IC-長岡IC間通車。
  - 1982年12月2日 : 小出IC-越後川口IC間通車。
  - 1983年10月26日 : 六日町IC-小出IC間通車。
  - 1984年11月8日 : 湯澤IC-六日町IC間通車。
  - 1985年10月2日 : 前橋IC-湯澤IC間通車，**全線通車**（月夜野IC-土樽停车区間暫定使用2車道）。
  - 1988年11月28日 : 月夜野IC-谷川岳停车区間拓寬為4車道。
  - 1991年10月22日 : 谷川岳停车区-土樽停车区（關越隧道）間拓寬為4車道。
  - 1993年3月27日 : 藤岡JCT-藤岡IC間的關越自動車道編入上信越自動車道，與上信越自動車道連接。
  - 1994年3月30日 : 大泉JCT-藤岡JCT間拓寬為6車道。大泉JCT通車，與東京外環自動車道連接。
  - 1995年8月7日 : 堀之內IC通車。
  - 1996年3月26日 : 藤岡JCT-前橋IC間拓寬為6車道，鶴島JCT通車與首都圈中央連絡自動車道（圈央道）連接。
  - 1998年3月26日 : 昭和IC通車。
  - 2001年3月31日 : 高崎JCT通車，與北關東自動車道連接。
  - 2004年3月27日 : 嵐山小川IC通車。
  - 2004年10月23日 : [新潟縣中越地震](../Page/新潟縣中越地震.md "wikilink")發生，小出IC-長岡IC間停止通行。
  - 2004年11月5日 : 小出IC-長岡IC間復舊（六日町IC-長岡IC間暫時復舊2車道）。
  - 2004年12月10日 : 駒寄停车区智能IC社會實驗開始（至2006年3月31日（當初實驗期間至2005年3月21日））。
  - 2005年4月17日 : 三芳停车区智能IC社會實驗開始（至2006年3月31日（當初實驗期間至2005年8月31日））。
  - 2005年6月1日 : 大和停车区智能IC社會實驗開始（至2006年3月31日（當初實驗期間至2005年8月31日））。
  - 2005年10月1日 : [日本道路公團](../Page/日本道路公團.md "wikilink")民營化，成為[東日本高速道路](../Page/東日本高速道路.md "wikilink")管理路線。
  - 2005年12月26日 : 六日町IC-長岡IC間完全復舊。
  - 2006年10月1日 : 三芳智能C、駒寄智能IC、大和智能IC開始使用。
  - 2007年7月16日 : [新潟縣中越沖地震發生](https://zh.wikipedia.org/wiki/新潟縣中越沖地震 "wikilink")，小千谷IC-長岡IC間停止通行及速度管制。
  - 2009年9月24日 : 長岡南越路智能IC開始使用。

**晟碟 ( 新帝 ) 公司**（，），原名 **SunDisk**，是設計與銷售[快閃記憶體產品的](https://zh.wikipedia.org/wiki/快閃記憶體 "wikilink")[美國](https://zh.wikipedia.org/wiki/美國 "wikilink")[跨國公司](../Page/跨國公司.md "wikilink")，於1988年由Eli Harari與[非揮發性記憶體](../Page/非揮發性記憶體.md "wikilink")技術專家Sanjay Mehrotra創立，1995年11月在[NASDAQ上市](https://zh.wikipedia.org/wiki/NASDAQ "wikilink")。新帝生產許多不同類型的快閃記憶體，包括多種記憶卡與一系列[USB](../Page/USB.md "wikilink")隨身碟。晟碟 ( 新帝 ) 公司均有銷售高端與低端的快閃記憶體。

晟碟 ( 新帝 ) 公司的總部位於[加州](https://zh.wikipedia.org/wiki/加州 "wikilink")[苗必達市](https://zh.wikipedia.org/wiki/苗必達市 "wikilink")，並在世界各地擁有辦事處與生產設施，如在[愛爾蘭共和國](https://zh.wikipedia.org/wiki/愛爾蘭共和國 "wikilink")[都柏林](../Page/都柏林.md "wikilink")的歐洲總部。2015年10月21日，西部數據（[威騰電子](../Page/威騰電子.md "wikilink")）宣布將以190億美元收購晟碟 ( 新帝 ) 公司\[1\]，並於2016年5月12日以160億美元完成併購\[2\]\[3\]。

### 收購

  - 晟碟 ( 新帝 ) 公司以13億[美元](../Page/美元.md "wikilink")收購了[USB](../Page/USB.md "wikilink")[快閃記憶棒製造商](https://zh.wikipedia.org/wiki/快閃記憶體 "wikilink") [msystems](https://web.archive.org/web/20061103084504/http://www.m-systems.com/)\[4\]。

  - 2014年6月16日，晟碟 ( 新帝 ) 公司宣布以11亿美元收购[Fusion-io](https://zh.wikipedia.org/wiki/Fusion-io "wikilink")。 Fusion-io主要为数据中心生产闪存产品和软件。 过此项收购，晟碟 ( 新帝 ) 公司将有机会更专注于企业用户市场。\[5\]。

## MP3授權爭議

2006年9月4日，在[德國](https://zh.wikipedia.org/wiki/德國 "wikilink")[柏林](../Page/柏林.md "wikilink")舉行的[IFA展覽上當局因](../Page/柏林國際廣播展.md "wikilink")[意大利](../Page/意大利.md "wikilink")[專利公司](https://zh.wikipedia.org/wiki/專利 "wikilink")[Sisvel取得禁令而查收了所有在晟碟](https://zh.wikipedia.org/wiki/Sisvel "wikilink") ( 新帝 ) 公司[攤位的](https://zh.wikipedia.org/wiki/攤位 "wikilink")[MP3播放器](https://zh.wikipedia.org/wiki/MP3播放器 "wikilink")\[7\]。Sisvel曾在[曼海姆](../Page/曼海姆.md "wikilink")提出另一[訴訟](../Page/訴訟.md "wikilink")聲稱新帝在[MP3](../Page/MP3.md "wikilink")使用格式的同時沒有支付所需的授權費。2006年9月8日一個柏林的法庭推翻了該禁令，新帝將其播放器放回展覽。\[8\]。

2007年3月16日，晟碟 ( 新帝 ) 公司發表新聞稿宣佈已達成協議並為所有現時及未來的MP3應用購得授權\[9\]。

**晟碟 ( 新帝 ) 公司**（，），原名
**SunDisk**，是一家設計與銷售[快閃記憶體產品的](../Page/快閃記憶體.md "wikilink")[美國](../Page/美國.md "wikilink")[跨國公司](../Page/跨國公司.md "wikilink")，於1988年由Eli
Harari與[非揮發性記憶體技術專家Sanjay](../Page/非揮發性記憶體.md "wikilink")
Mehrotra創立，1995年11月在[NASDAQ上市](../Page/NASDAQ.md "wikilink")。新帝生產許多不同類型的快閃記憶體，包括多種記憶卡與一系列[USB隨身碟](../Page/USB.md "wikilink")。晟碟
( 新帝 ) 公司均有銷售高端與低端的快閃記憶體。

晟碟 ( 新帝 )
公司的總部位於[加州](../Page/加州.md "wikilink")[苗必達市](../Page/苗必達市.md "wikilink")，並在世界各地擁有辦事處與生產設施，如在[愛爾蘭共和國](../Page/愛爾蘭共和國.md "wikilink")[都柏林的歐洲總部](../Page/都柏林.md "wikilink")。2015年10月21日，西部數據([威騰電子](../Page/威騰電子.md "wikilink"))宣布將以190億美元收購晟碟
( 新帝 ) 公司\[1\]，並於2016年5月12日以160億美元完成併購\[2\]\[3\]。

[FlashCP是一個用於便攜裝置上數位材料](../Page/FlashCP.md "wikilink")（如[電子書](../Page/電子書.md "wikilink")）的[數位著作權管理技術](../Page/數位著作權管理.md "wikilink")，主要針對學生，容許用戶在執行複製限制下傳送版權材料。晟碟
( 新帝 )
公司在2005年收購[以色列公司](../Page/以色列.md "wikilink")[MDRM時一同獲得了該技術](../Page/MDRM.md "wikilink")。

Drive，是[Cruzer產品線的一部份](../Page/Cruzer.md "wikilink")。另外

Plaza}-](http://www.SanDiskPlaza.SanDisk.com) 下載。-{SanDisk
Plaza}-是一個發展迅速的網上商店，供應數位書籍、音樂、遊戲與教育工具，其線上產品的定價不同，許多類別為免費。數位內容在下載後可在連線或離線時使用。

## MP3授權爭議

2006年9月4日，在[德國](../Page/德國.md "wikilink")[柏林舉行的](../Page/柏林.md "wikilink")[IFA展覽上當局因](../Page/柏林國際廣播展.md "wikilink")[意大利](../Page/意大利.md "wikilink")[專利公司](../Page/專利.md "wikilink")[Sisvel取得禁令而查收了所有在晟碟](../Page/Sisvel.md "wikilink")
( 新帝 )
公司[攤位的](../Page/攤位.md "wikilink")[MP3播放器](../Page/MP3播放器.md "wikilink")\[7\]。Sisvel曾在[曼海姆提出另一](../Page/曼海姆.md "wikilink")[訴訟聲稱新帝在](../Page/訴訟.md "wikilink")[MP3使用格式的同時沒有支付所需的授權費](../Page/MP3.md "wikilink")。2006年9月8日一個柏林的法庭推翻了該禁令，新帝將其播放器放回展覽。\[8\]。

2007年3月16日，晟碟 ( 新帝 ) 公司發表新聞稿宣佈已達成協議並為所有現時及未來的MP3應用購得授權\[9\]。

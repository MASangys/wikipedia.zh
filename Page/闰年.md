**閏年**是比普通年分多出一段時間的年分，在各種[曆法中都有出現](https://zh.wikipedia.org/wiki/曆法 "wikilink")，目的是為了彌補人為規定的紀年與[地球公轉產生的差異](https://zh.wikipedia.org/wiki/地球公轉 "wikilink")⋯⋯。\[1\]

## 閏年規則

目前使用的[格里高利历闰年规则如下](https://zh.wikipedia.org/wiki/格里高利历 "wikilink")\[2\]：

1.  公元年分除以4不可整除，為平年。
2.  公元年分除以4可整除但除以100不可整除，为闰年。
3.  公元年分除以100可整除但除以400不可整除，为平年。
4.  公元年分除以400可整除，为闰年。

计算方法： function output=CGUMIRS(y)

if rem(y ,4) == 0 && rem(y ,100) \~= 0

`      fprintf('%g year is leap year\n',y)`

elseif rem(y,400)==0

`      fprintf('%g year is leap year\n',y)`

else

`   fprintf('%g year is ordinary year\n',y)`

end

每逢閏年，2月分有29日，平年的2月分為28日。

因此，[1977年](../Page/1977年.md "wikilink")、[2021年](../Page/2021年.md "wikilink")為平年，[1980年](../Page/1980年.md "wikilink")、[2020年](../Page/2020年.md "wikilink")逢4的倍数為閏年，[1900年](../Page/1900年.md "wikilink")逢100的倍数但非400的倍數故為平年，[2000年](../Page/2000年.md "wikilink")逢400的倍数又為閏年。

此外，如依照现有太阳年的长度与上述闰年规则，每[8000年又约差一日](https://zh.wikipedia.org/wiki/8000年 "wikilink")，因此[约翰·赫歇尔提议每逢](https://zh.wikipedia.org/wiki/约翰·赫歇尔 "wikilink")4000的倍数不闰，如[公元](../Page/公元.md "wikilink")4000年。但距此年分来临尚有约二千年之遥，因此还未曾真正纳入规则或实施过。又由于地球公转速度的不稳定与众多影响因素，届时是否需要纳入此规则有待商榷。

## 公元前的閏年

公元前之閏年出現在前1, 前5, 前9, 前13, ...，或記作1 BC，5 BC，9 BC，13 BC，...，或在數軸上記作0，-4，-8，-12，...。 判斷是否閏年，須將年份值減1再以「除以4」計算，或以數軸記法表示時直接計算（正負性不影響是否整除）。（因爲沒有[公元0年这一年](https://zh.wikipedia.org/wiki/0年 "wikilink")（除非臨時約定，並注明對應等式），所以公元前1, 2, 3, 4, ... 年应该在數學數軸上對應著（但不是，或不應該在曆法上稱）公元0, -1, -2, -3, ... 年，而公元前1, 5, 9, 13, ... 年在數學數軸上對應著0, -4, -8, -12, ... 年，为4的倍数）。 記住：臨時約定的數學“0”年，等於天文學固有的“0”年，等於曆法公元前一年。

## 各種曆法中的閏年

### 陽曆中的「閏年」

[阳历](../Page/阳历.md "wikilink")中有闰日的年分叫闰年，相反就是[平年](https://zh.wikipedia.org/wiki/平年 "wikilink")，平年為365天，闰年为366天。在[公历](https://zh.wikipedia.org/wiki/公历 "wikilink")[儒略曆](../Page/儒略曆.md "wikilink")及[纪年](../Page/纪年.md "wikilink")中，平年的二月为28天，闰年的二月为29天。**闰年平月2月29日为闰日**。

### 中國農曆的「閏年」

中国旧历[农历纪年中](https://zh.wikipedia.org/wiki/农历 "wikilink")，有**闰月**的一年称为闰年。一般年分为12个月，353（罕见）或354（多见）或355（少见）天，闰年则为13个月，383（少见）或384（多见）或385（罕见）天。

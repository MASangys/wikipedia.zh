**阴阳合历**，又稱**太陰太陽曆**，是为許多[文化](../Page/文化.md "wikilink")採用的[曆法](https://zh.wikipedia.org/wiki/曆法 "wikilink")，其日期採朔望月以指示[月球的相位](../Page/月相.md "wikilink")，[年](../Page/年.md "wikilink")則與太陽相關，且一年的月數必須是整數。大多採用太陽年以[回歸年定義](https://zh.wikipedia.org/wiki/回歸年 "wikilink")，也有採用[恆星年定義者](https://zh.wikipedia.org/wiki/恆星年 "wikilink")。採用陰陽合曆的主要目的是配合季節，因此需要安排閏月來調整，大多數的年有12個月，閏年就會有13個月。 在[天文學](../Page/天文學.md "wikilink")中是指兼顾[月相](../Page/月相.md "wikilink")週期和太陽週期運動所安排的[曆法](https://zh.wikipedia.org/wiki/曆法 "wikilink")。平年有12个[朔望月](https://zh.wikipedia.org/wiki/朔望月 "wikilink")，十九年為一章，四章為一篰（詳見史記曆書），其中安置七个閏月置閏，使年的平均值大约与[回归年](../Page/回归年.md "wikilink")相当。俗稱農民曆的[夏历就是阴阳历的一种](https://zh.wikipedia.org/wiki/夏历 "wikilink")。

農民曆是構成陰陽合曆的部分，連續一章一套可名為陰陽章曆，連續三章一套可名為陰陽篰曆。阴阳合历制中單一各別的年，其月數、日數不夠穩定，依曆制的定義，欠缺年曆年年穩定循環的性質，沒有年曆的身份，不能直接稱某一本農民曆是陰陽合曆，也不便於以回歸年為循環的農作應用，一般稱農民曆可以兼顧陰曆和陽曆的優點，且簡單、易懂、好記，其實是欠缺系統思考導致的誤解。

現今公曆以回歸年為基礎，如果加註四分月相令使用者清楚理解，且兼顧一年一循環的節氣，如此仍可稱之為陰陽合曆。

## 含義

單純的“陽曆”是指曆法中只保證一年的時間與地球繞日運行周期基本一致，不考慮月份，中国的[24节气就是纯阳历](https://zh.wikipedia.org/wiki/24节气 "wikilink")；單純的“陰曆”是指曆法中只保證一個[月](../Page/月.md "wikilink")的時間與[月亮運行周期基本一致](../Page/月球.md "wikilink")，不考慮年長，而陰陽曆則既保證“年”與地球繞日周期的一致，又保證“月”與月亮周期的一致。以中國的[農曆](../Page/農曆.md "wikilink")為例，大小月分別為30天和29天，平均每月為29.5天，與月亮運行周期一致（參見[朔望月](https://zh.wikipedia.org/wiki/朔望月 "wikilink")）；[平年為](https://zh.wikipedia.org/wiki/平年 "wikilink")354天或355天或353天，[閏年為](https://zh.wikipedia.org/wiki/閏年 "wikilink")383天或384天或385天，[每十九年平均為](https://zh.wikipedia.org/wiki/農曆#十九年七閏 "wikilink")365.247天，與地球繞日運行周期一致（參見[回歸年](https://zh.wikipedia.org/wiki/回歸年 "wikilink")）。

## 例子

[希伯來曆](../Page/希伯來曆.md "wikilink")、[佛曆](../Page/佛曆.md "wikilink")、[印度曆](https://zh.wikipedia.org/wiki/印度曆 "wikilink")、[庫德曆](https://zh.wikipedia.org/wiki/庫德曆 "wikilink")、[孟加拉曆](https://zh.wikipedia.org/wiki/孟加拉曆 "wikilink")、[藏曆](../Page/藏曆.md "wikilink")，還有傳統的中國[農曆](../Page/農曆.md "wikilink")、日本[和曆](../Page/和曆.md "wikilink")、[越南曆](https://zh.wikipedia.org/wiki/越南曆 "wikilink")、[蒙古曆](https://zh.wikipedia.org/wiki/蒙古曆 "wikilink")，再加上古代[希臘](https://zh.wikipedia.org/wiki/希臘曆 "wikilink")、[科利尼](https://zh.wikipedia.org/wiki/科利尼曆 "wikilink")、和[巴比倫曆都是陰陽曆](https://zh.wikipedia.org/wiki/巴比倫曆 "wikilink")。也有些在[南阿拉伯的古老](https://zh.wikipedia.org/wiki/南阿拉伯 "wikilink")[前伊斯蘭曆是陰陽曆的系統](https://zh.wikipedia.org/wiki/伊斯蘭曆#前伊斯蘭曆 "wikilink")\[1\]。中國、科利尼和希伯來\[2\]的陰陽曆或多或少的是追隨著回歸年；而佛曆和印度曆追隨的是恆星年。因此，前三種曆法有季節的想法在內，而後二者則以某個滿月所在的星座來定年，藏曆則受到中國和印度曆的影響。據現代學者在皈依基督教之前[日耳曼民族也使用陰陽曆](https://zh.wikipedia.org/wiki/德國曆 "wikilink")。

## 全阳历与阴历

[伊斯蘭曆的日期與太陽無關](https://zh.wikipedia.org/wiki/伊斯蘭曆 "wikilink")，所以是[陰曆](https://zh.wikipedia.org/wiki/陰曆 "wikilink")，而不是陰陽曆。民用版的[儒略曆](../Page/儒略曆.md "wikilink")和[格里曆](../Page/格里曆.md "wikilink")都是[陽曆](https://zh.wikipedia.org/wiki/太陽曆 "wikilink")，因為它們的日期與[月相](../Page/月相.md "wikilink")無關。

## 閏月的推定

當要確認是否要插入[閏月時](https://zh.wikipedia.org/wiki/閏月 "wikilink")，一些曆法依賴直接的觀察，一些則比較[太陽和](https://zh.wikipedia.org/wiki/太陽 "wikilink")[月球的](../Page/月相.md "wikilink")[黃經度數](https://zh.wikipedia.org/wiki/黃經 "wikilink")。夏威夷人則是觀察特定的幾顆恆星和月球的關係。

另一方面，計算的陰陽曆，要累積幾個月才需要在那些年插入閏月是有一個固定的規則。要建構這樣的一個曆法（在原則上），回歸年的長度除以[朔望月的平均長度是](https://zh.wikipedia.org/wiki/朔望月 "wikilink")：

  -
    12.368266......

將這個十進位值轉為[連分數的最佳近似值為](https://zh.wikipedia.org/wiki/連分數 "wikilink")（\[12; 2, 1, 2, 1, 1, 17, ...\]），所以在下面的清單中列出了朔望月在分子，回歸年在分母的整數的比值：

|        |       |              |                              |       |                     |
| -----: | ----: | :----------- | ---------------------------- | :---- | ------------------: |
|   12 / |   1 = | 12           | \= \[12\]                    | （誤差 = | −0.368266... 朔望月/年） |
|   25 / |   2 = | 12.5         | \= \[12; 2\]                 | （誤差 = |  0.131734... 朔望月/年） |
|   37 / |   3 = | 12.333333... | \= \[12; 2, 1\]              | （誤差 = | −0.034933... 朔望月/年） |
|   99 / |   8 = | 12.375       | \= \[12; 2, 1, 2\]           | （誤差 = |  0.006734... 朔望月/年） |
|  136 / |  11 = | 12.363636... | \= \[12; 2, 1, 2, 1\]        | （誤差 = | −0.004630... 朔望月/年） |
|  235 / |  19 = | 12.368421... | \= \[12; 2, 1, 2, 1, 1\]     | （誤差 = |  0.000155... 朔望月/年） |
| 4131 / | 334 = | 12.368263... | \= \[12; 2, 1, 2, 1, 1, 17\] | （誤差 = | −0.000003... 朔望月/年） |

注意：沒有計算的曆法的年平均長度正好等於真實回歸年的長度。使用不同的曆法有不同的年平均長度和平均的每月長度，所以曆法的月和月球朔望月之間的差距並不等於上面給出的值。

8年的循環（99個朔望月，包括99 – 8 X12 = 3個閏月）是[古雅典曆法使用的](https://zh.wikipedia.org/wiki/阿蒂卡曆 "wikilink")[八年法](https://zh.wikipedia.org/wiki/八年法 "wikilink")（octaeteris）。8年的週期也曾在第三世紀初在羅馬用來[計算復活節](../Page/復活節計算表冊.md "wikilink")（或舊的計算法）。

19年的循環（235過朔望月，包括235 – 19 X 12 = 7個閏月）是算數的陰陽曆，也是經典的[默冬章](../Page/默冬章.md "wikilink")。它結合了8年和11年的週期，使誤差逼近於，這個循環可以截斷為11年（跳過包括3個閏月的8年），之後又可以恢復為19年的巡環。[默冬章](../Page/默冬章.md "wikilink")的週期是整數的天數，然而實際的陰曆月並不是整數的天數。它可以適用於一年365.25天，採用4 X 19年的[卡利巴斯週期](https://zh.wikipedia.org/wiki/卡利巴斯週期 "wikilink")（Callippic cycle）的年平均值。

羅馬從第三世紀至457年，採用84年的週期計算復活節的日期。[島嶼基督教一直沿用至](../Page/凱爾特基督教.md "wikilink")768年，當時的[Bangor主教](https://zh.wikipedia.org/wiki/Bishop_of_Bangor "wikilink")[Bishop Elfodd說服他們採用](https://zh.wikipedia.org/wiki/Elfodd "wikilink")[St Augustine's mission改良的曆法](https://zh.wikipedia.org/wiki/Gregorian_mission "wikilink")。84年的週期相當4 X 19年的卡利巴斯週期（包括 4 X 7個閏月）加上8年的週期（包括3個閏月），總共1039個月（包括31個閏月）。這給出的年平均為12.3690476..個月，一個週期為30681天。但比1039個朔望月短少1.28天，比84個回歸年超出0.66天，比84個恆星年短少0.55天。

繼默冬章之後的下一個近似值（連分數所產生的）所採用的是非常敏感的太陰週期（朔望月）與年，特別是年的數值（例如334年的週期）。其他幾種為特定的目的而定義的一年，其近似值可能更精確：例如，353年的週期，包括130個閏月，總月數為4366個朔望月（12.36827195..），對北半球的春分年會更準確；611年的週期，包括225個閏月，總數7557個朔望月（12.36824877..）對北半球的夏至年有常好的近似；160年的週期，包括59個閏月，總數1979個朔望月（12.36875）對恆星年（12.3687462856[朔望月](https://zh.wikipedia.org/wiki/月#月的長度 "wikilink")）有非常好的近似。

## 計算閏月

在所有的陰陽曆，都可以通過以下的計算方法，用近似長度的月和年的天數，獲得插入閏月的粗略頻率：

  - 年：365.25天，月：29.53天
  - 365.25/(12 X 29.53) = 1.0307
  - 1/0.0307 = 32.57 ：在兩個閏月之間的正常月數
  - 32.57/12 = 2.7 在閏年之間的年數

一個代表平常月分和閏月的序列是ccLccLcLccLccLccLcL，這就是經典的默冬章（19年7閏）。佛曆和希伯來曆限制閏年只能插入一個閏月；因此在兩個閏月之間的正常月數通常是36個月，偶爾也會只有24個月。由於中國和印度的陰陽曆依據太陽的真實運動，可以插在任何一個月的前面或後面，但通常不會在太陽移動最快的那幾個月（通過近日點的前後，現在大約是1月3日）。這使得兩個閏月之間的正常月數通常大於34個月，而同時較少的月數大約是29個月。

## 與時間無關的陰陽曆

太陽年不能包含整數月數的另一種處理方法是在一年當中通過若干不屬於任何月份的日子。一些[海岸薩利希人就是用這樣的曆法](https://zh.wikipedia.org/wiki/海岸薩利希人 "wikilink")。例如，[奇黑利斯](https://zh.wikipedia.org/wiki/Sts'Ailes "wikilink")（Chehalis）以[奇努克鮭魚到來產卵的陰曆月](https://zh.wikipedia.org/wiki/奇努克鮭魚 "wikilink")（在格里曆的10月）開始計算10個月，然後留下不定的日數，直到下一次[奇努克鮭魚的返回](https://zh.wikipedia.org/wiki/奇努克鮭魚 "wikilink")\[3\]。

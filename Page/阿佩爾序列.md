在[數學中](https://zh.wikipedia.org/wiki/數學 "wikilink")，**阿佩爾序列**是得名于十九世紀[法國](https://zh.wikipedia.org/wiki/法國 "wikilink")[數學家](https://zh.wikipedia.org/wiki/數學家 "wikilink")[保羅·埃米尔·阿佩爾](https://zh.wikipedia.org/wiki/保羅·埃米尔·阿佩爾 "wikilink")（Paul
Émile
Appell）\[1\]的一類[多項式序列](https://zh.wikipedia.org/wiki/多項式序列 "wikilink")
{*p*<sub>*n*</sub>(*x*)}<sub>*n* = 0, 1, 2, ...</sub>。阿佩爾序列滿足以下關係：

\[{d \over dx} p_n(x) = np_{n-1}(x),\]

其中的 *p*<sub>0</sub>(*x*) 是非零常數。

除了一些平凡的例子如 { *x*<sup>*n*</sup> }
以外，最值得注意的阿佩爾序列是[埃爾米特多項式](https://zh.wikipedia.org/wiki/埃爾米特多項式 "wikilink")、[伯努利多項式以及](../Page/伯努利多項式.md "wikilink")[歐拉多項式](https://zh.wikipedia.org/wiki/歐拉多項式 "wikilink")。所有的阿佩爾序列都是[謝弗序列](https://zh.wikipedia.org/wiki/謝弗序列 "wikilink")，但要注意的是絕大多數謝弗序列都不是阿佩爾序列。

## 等價的阿佩爾序列定義方式

最常見的阿佩爾序列的定義就是以上的

  - 對所有的 *n* = 1, 2, 3, ...,
    \[{d \over dx} p_n(x) = np_{n-1}(x),\]
      -
        並且 *p*<sub>0</sub>(*x*) 是一個非零常數

的關係式。此外，以下的條件也可以被驗證是與之等價的：

1.  純數數列 {*c*<sub>*n*</sub>}<sub>*n* = 0, 1, 2, ...</sub> 滿足
    *c*<sub>0</sub> ≠ 0，並且
    \[p_n(x) = \sum_{k=0}^n {n \choose k} c_k x^{n-k};\]
2.  純數數列 {*c*<sub>*n*</sub>}<sub>*n* = 0, 1, 2, ...</sub> 滿足
    *c*<sub>0</sub> ≠ 0，並且
    \[p_n(x) = \left(\sum_{k=0}^\infty {c_k \over k!} D^k\right) x^n,\]
      -
        其中 \(D = {d \over dx};\)
3.  對所有的 *n* = 0, 1, 2, ...,
    \[p_n(x+y) = \sum_{k=0}^n {n \choose k} p_k(x) y^{n-k}.\]

## 遞歸公式

假設

\[p_n(x) = \left(\sum_{k=0}^\infty {c_k \over k!} D^k\right) x^n = Sx^n,\]

其中後一個等式是在以*x*為不定元的多項式構成的[線性空間中的線性算子](https://zh.wikipedia.org/wiki/線性空間 "wikilink")
*S* 的定義式。并定義：

\[T = S^{-1} = \left(\sum_{k=0}^\infty {c_k \over k!} D^k\right)^{-1} = \sum_{k=1}^\infty {a_k \over k!} D^k\]

為 *S* 的逆算子，其中的係數 *a*<sub>*k*</sub>
是[形式冪級數的逆係數](https://zh.wikipedia.org/wiki/形式冪級數 "wikilink")。這樣得到

\[Tp_n(x) = x^n.\,\]

在[影子演算的約定中](https://zh.wikipedia.org/wiki/影子演算 "wikilink")，算子 *T*
一般被用來代表阿佩爾序列 {*p*<sub>*n*</sub>}，可以定義對數算子：

\[\log T = \log\left(\sum_{k=0}^\infty {a_k \over k!} D^k \right)\]

運用通常的 log(1 + *x*) 的冪級數展開表達式以及通常的複合形式冪級數定義後，可以得到：

\[p_{n+1}(x) = (x - (\log T)')p_n(x).\,\]

當阿佩爾序列是[埃爾米特多項式的時候](https://zh.wikipedia.org/wiki/埃爾米特多項式 "wikilink")，這個關係式也可以變化為埃爾米特多項式的遞推公式。

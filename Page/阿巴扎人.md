**阿巴扎人**（；；[切爾克斯語](https://zh.wikipedia.org/wiki/Circassian_language "wikilink"):
*Абазэхэр*；；），又譯“阿巴札人”或“阿巴茲人”，是主要分布於[俄羅斯聯邦](https://zh.wikipedia.org/wiki/俄羅斯聯邦 "wikilink")[卡拉恰伊-切尔克斯共和国和](../Page/卡拉恰伊-切尔克斯共和国.md "wikilink")[阿迪格共和国境內的一個民族](../Page/阿迪格共和国.md "wikilink")，信仰[遜尼派伊斯蘭教](../Page/遜尼派.md "wikilink")。根據2010年俄羅斯人口普查\[1\]，有43,341阿巴扎人居住在俄羅斯。国外則分布在土耳其\[2\]、埃及\[3\]
、叙利亚、约旦、亞美尼亞及烏克蘭。

## 民族分布、人口與語言

### 民族分布

現今的[阿巴扎人主要分布在](../Page/阿巴扎人.md "wikilink")[土耳其](../Page/土耳其.md "wikilink")、[埃及](../Page/埃及.md "wikilink")，以及以俄羅斯為聯邦主體的[卡拉恰伊-切爾克斯共和國](https://zh.wikipedia.org/wiki/卡拉恰伊-切爾克斯共和國 "wikilink")、[斯塔夫羅波爾邊疆區](https://zh.wikipedia.org/wiki/斯塔夫羅波爾邊疆區 "wikilink")、[卡巴爾達-巴爾卡爾共和國](../Page/卡巴爾達-巴爾卡爾共和國.md "wikilink")\[4\]三國，少部分則分布在[亞美尼亞及](https://zh.wikipedia.org/wiki/亞美尼亞 "wikilink")[烏克蘭](https://zh.wikipedia.org/wiki/烏克蘭 "wikilink")。

在俄羅斯南部[卡拉恰伊-切爾克斯共和國生活的阿巴扎人的](https://zh.wikipedia.org/wiki/卡拉恰伊-切爾克斯共和國 "wikilink")14個鎮和村莊有Kara-Pago，Kubina，Psikh，El'burgan，Inzhich-Chukun，Koi-dan，Abaza-Khabl'，Malo-Abazinka，Tapanta，Krasnovostochni，Novokuvinski，Starokuvinski，Abazakt和Ap-sua\[5\]等

### 人口

根據2010年[俄羅斯人口普查](https://zh.wikipedia.org/wiki/俄羅斯人口 "wikilink")\[6\]，有43,341[阿巴扎人居住在](../Page/阿巴扎人.md "wikilink")[俄羅斯](https://zh.wikipedia.org/wiki/俄羅斯 "wikilink")。其餘多數的[阿巴扎人則散居於](../Page/阿巴扎人.md "wikilink")[土耳其](../Page/土耳其.md "wikilink")\[7\]
及[埃及](../Page/埃及.md "wikilink")\[8\]，而居住在中東國家（於[約旦和](https://zh.wikipedia.org/wiki/約旦 "wikilink")[敘利亞](https://zh.wikipedia.org/wiki/敘利亞 "wikilink")）的[阿巴扎人](../Page/阿巴扎人.md "wikilink")，大部分為[俄羅斯帝國時期](https://zh.wikipedia.org/wiki/俄羅斯帝國 "wikilink")[高加索戰爭的](../Page/高加索戰爭.md "wikilink")[阿巴扎人移民後裔](../Page/阿巴扎人.md "wikilink")。

### 語言

**阿巴扎语**（Абаза Бызшва 或 Abaza Byzšwa； ；），又名**阿巴札语**、**阿巴茲语**，
是[阿巴扎人的语言](../Page/阿巴扎人.md "wikilink")，主要使用于[俄罗斯联邦的](https://zh.wikipedia.org/wiki/俄罗斯联邦 "wikilink")[卡拉恰伊-切尔克斯共和国的高加索山区](../Page/卡拉恰伊-切尔克斯共和国.md "wikilink")。它有两种方言，即[阿什科勒瓦方言和作为标准书面语的](https://zh.wikipedia.org/wiki/阿什科勒瓦方言 "wikilink")[塔潘达方言](https://zh.wikipedia.org/wiki/塔潘达方言 "wikilink")。

从1932年至1938年，阿巴扎语一直是用[拉丁字母书写的](../Page/拉丁字母.md "wikilink")。自1938年起，在[俄罗斯境内的阿巴扎语使用者开始用](../Page/俄罗斯.md "wikilink")[基里尔字母来拼写该语言](https://zh.wikipedia.org/wiki/基里尔字母 "wikilink")。而在[土耳其](../Page/土耳其.md "wikilink")，阿巴扎语依然被人们用[拉丁字母来书写](../Page/拉丁字母.md "wikilink")。目前在[俄罗斯境内共有大约三万五千名使用阿巴扎语使用者](../Page/俄罗斯.md "wikilink")，在[土耳其另有大约一万名使用者](../Page/土耳其.md "wikilink")，另外在[德国和](../Page/德国.md "wikilink")[美国等地也有一些阿巴扎人说这种语言](../Page/美国.md "wikilink")。

阿巴扎语和其同属于[高加索语系](https://zh.wikipedia.org/wiki/高加索语系 "wikilink")[西北高加索语族的其他亲属语言一样](https://zh.wikipedia.org/wiki/西北高加索语族 "wikilink")，是一种高度[黏着的语言](../Page/黏着语.md "wikilink")。与其亲属语言一样，它也有着庞大的辅音音位系统（六十三个辅音音位），且元音音位系统很小（仅有两个元音音位）。它和[阿布哈兹语很接近](https://zh.wikipedia.org/wiki/阿布哈兹语 "wikilink")，两者是互通的，即两种语言的使用者都可以理解对方的语言。但阿巴扎语也有一些地方表现出了[阿布哈兹语所不具备的特点](https://zh.wikipedia.org/wiki/阿布哈兹语 "wikilink")，例如阿巴扎语中有[浊咽擦音](../Page/浊咽擦音.md "wikilink")，比方说在“三月”
（阿巴扎语：гӀапынхъамыз，国际音标：\[ʕaːpənqaːməz\]）这个词中。因此，阿巴扎语被语言学家认为是最难学的一种[高加索语言](../Page/高加索诸语言.md "wikilink")。[W·S·艾伦](https://zh.wikipedia.org/wiki/W·S·艾伦 "wikilink")、[布莱恩·欧贺林和](https://zh.wikipedia.org/wiki/布莱恩·欧贺林 "wikilink")[约翰·科拉鲁索等人都曾对阿巴扎语进行过研究](https://zh.wikipedia.org/wiki/约翰·科拉鲁索 "wikilink")。\[12\]對於外人來說，講阿巴扎語發出的聲音是強烈的；對於阿巴扎人來說，由於阿巴扎語較為困難，因此在學習其他語言的時候是比較容易的。

### 语法

#### 名词

阿巴札语的[名词有泛指和确指的变化](https://zh.wikipedia.org/wiki/名词 "wikilink")，名词原型可表示泛指，比如
кыт 指任意一个村子， цӀла 指任意一棵树。在其后加 кӀ 也可表示泛指，意为“某”， кыткӀ 某个村子， цӀлакӀ
某棵树。表示确指时要在名词前加 а ，如： акыт 那个村子， ацӀла 那棵树。

阿巴札语通过在名词前加词缀的方式表示[领属性](https://zh.wikipedia.org/wiki/领属性 "wikilink")[人称](../Page/人称.md "wikilink")。所有名词基本都符合以下变化：

|     |       |         |         |         |         |         |
| --- | ----- | ------- | ------- | ------- | ------- | ------- |
| 人称： | 第一人称  | 第二人称（阳） | 第二人称（阴） | 第三人称（阳） | 第三人称（阴） | 第三人称（中） |
| 单数  | с(ы)  | у(ы)    | б(ы)    | й(ы)    | л(ы)    | а       |
| 复数  | хӀ(ы) | шв(ы)   | шв(ы)   | р(ы)    | р(ы)    | р(ы)    |

例如单词“кыт”（村子）：

|     |     |        |         |         |         |         |         |
| --- | --- | ------ | ------- | ------- | ------- | ------- | ------- |
| 人称： | 词干  | 第一人称   | 第二人称（阳） | 第二人称（阴） | 第三人称（阳） | 第三人称（阴） | 第三人称（中） |
| 单数  | кыт | сыкыт  | уыкыт   | быкыт   | йыкыт   | лыкыт   | акыт    |
| 复数  | кыт | хӀыкыт | швыкыт  | швыкыт  | рыкыт   | рыкыт   | рыкыт   |

### [土耳其](../Page/土耳其.md "wikilink")

[Turkey_(orthographic_projection).svg](https://zh.wikipedia.org/wiki/File:Turkey_\(orthographic_projection\).svg "fig:Turkey_(orthographic_projection).svg")
土耳其的阿巴扎人分散在土耳其全國各地區，其中又以土耳其中北部的黑海沿岸為主要集中帶。

#### 地理位置

土耳其處於歐亞連接處的重要地理位置，歐洲土耳其占全國的3％，陸地總面積達780,576平方公里，略小於巴基斯坦。北臨黑海，與羅馬尼亞、烏克蘭和俄羅斯相望；南臨地中海，遙望賽普勒斯、埃及和以色列；東面與喬治亞、亞美尼亞、亞塞拜然、伊朗、伊拉克和敘利亞多國接壤；西臨愛琴海，並在西北部與希臘以及保加利亞接壤。土耳其位於歐洲部分的領土是巴爾幹半島的東色雷斯，其他部分則屬小亞細亞（又稱安那托利亞）。安那托利亞半島形狀大致呈長方形，占土耳其總面積97％。位於歐洲部分的東色雷斯所占面積雖然少於3％，卻容納了土耳其10%以上的人口，其中大部分居住在伊斯坦堡。東色雷斯與小亞細亞被博斯普魯斯海峽、達達尼爾海峽和馬爾馬拉海分開。

#### 自然環境

##### 地形河流與山脈

土耳其大部分屬高原地形，其多樣的景觀是複雜的地球運動的產物，地殼持續變動，並且伴隨著頻繁的地震\[14\]和偶爾的火山噴發。[博斯普魯斯海峽和](https://zh.wikipedia.org/wiki/博斯普魯斯海峽 "wikilink")[達達尼爾山脈的通過土耳其的斷層線造出了](https://zh.wikipedia.org/wiki/達達尼爾山脈 "wikilink")[黑海](../Page/黑海.md "wikilink")。北安納托利亞斷層線則從西向東延伸到全國北部。

  - 土耳其的亞洲部分

主要由安納托利亞半島組成，其中包括狹窄沿海平原的高原中部高原，北部的科羅爾山脈和龐蒂山脈以及南部的金牛座山脈。位於亞美尼亞高原西部高原的東土耳其擁有更多的山區景觀，並擁有幼發拉底河，底格里斯河和阿拉斯等河流的來源，並包含阿拉拉特山，土耳其最高點為5,137米（16,854英尺）
，和湖泊最大的湖泊耳其東南部位於上美索不達米亞北部平原。 \[15\]

##### 氣候

  - 毗鄰[愛琴海和](https://zh.wikipedia.org/wiki/愛琴海 "wikilink")[地中海的土耳其沿海地區](../Page/地中海.md "wikilink")

溫和的地中海氣候，炎熱而乾燥的夏季，溫和至涼爽，潮濕的冬天。

  - 與[黑海接壤的沿海地區](../Page/黑海.md "wikilink")

溫帶洋洋的氣候，溫暖潮濕的夏天，涼爽，寒冷，潮濕的冬天。土耳其黑海沿岸的降水量最多，是土耳其全年唯一接受高降水的地區該海岸東部平均每年為2200毫米（87英寸），是全國最高的降水量。

  - 連接愛琴海和黑海的馬爾馬拉海毗鄰的沿海地區

在溫帶的[地中海氣候和溫帶的](https://zh.wikipedia.org/wiki/地中海氣候 "wikilink")[海洋氣候之間具有過渡氣候](https://zh.wikipedia.org/wiki/海洋氣候 "wikilink")，溫暖至炎熱，中度乾燥的夏季，涼爽至寒冷，潮濕的冬天。幾乎每個冬天，雪落在馬爾馬拉海和[地中海對內陸的影響](https://zh.wikipedia.org/wiki/黑海的沿海地區，但通常不會在幾天內融化。然而，愛琴海沿岸的雪是罕見的，在地中海的沿海地區非常罕見。_靠近海岸的山脈防止了[[地中海 "wikilink")，使得土耳其內陸的安納托利亞中部高原成為一個具有鮮明對比季節的[大陸性氣候](https://zh.wikipedia.org/wiki/大陸性氣候 "wikilink")。

  - 高原東部的冬天特別嚴寒

[東安納托利亞州可能會發生](https://zh.wikipedia.org/wiki/東安納托利亞州 "wikilink")-30至-40°C（-22至-40°F）的溫度。雪可能至少在一年的120天之內。

  - 在西部

冬季氣溫平均低於1°C（34°F）。夏天炎熱而乾燥，天氣溫度通常在30°C（86°F）以上。年降水量平均約為400毫米（16英寸），實際數量由海拔決定。

  - 最乾旱的科尼亞平原和馬拉蒂亞平原

年降雨量通常小於300毫米（12英寸）。五月通常是最潮濕的月份，而七月和八月是最乾燥的月份。

##### 多樣化物種

  - Pontic山脈

形成了具有多種溫帶雨林類型，動植物群的生態區。土耳其的非凡生態系統和棲息地多樣性已經產生了相當多的物種多樣性。安納托利亞是農業出現以來已經種植的許多植物的家園，而現在為人類提供主食的許多植物的野生祖先仍然在土耳其生長。土耳其的動物多樣性甚至比其植物群多。整個歐洲的動物物種數量約為6萬隻，而土耳其則有8萬多種（超過10萬種亞種）。

  - 北安納托利亞針葉樹和落葉林

覆蓋了土耳其北部的龐蒂山脈，而高加索混交林則延伸到東端。該地區是歐亞野生動物的所在地，如歐亞大麻雀，金鷹，東方皇家鷹，較小斑點的鷹，白種黑猩猩，紅色絲絲蛋白和壁紙。龐蒂山與黑海之間狹窄的沿海地帶是Euxine-Colchic落葉林的所在地，其中包含世界上幾個溫帶雨林。土耳其松木主要發現在土耳其和其他東地中海國家。鬱金香的幾種野生種類是安那托利亞的原生植物，這種花被首次引入西歐，在16世紀從奧斯曼帝國獲取的物種。

### 埃及

[EGY_orthographic.svg](https://zh.wikipedia.org/wiki/File:EGY_orthographic.svg "fig:EGY_orthographic.svg")

#### 地理位置

埃及面積達1,001,450平方公里，疆域橫跨亞、非兩洲，當中大部分位於非洲東北部，另外蘇伊士運河以東的西奈半島位於亞洲西南部。埃及有著獨特的地緣位置：在陸路上，它連接亞、非兩洲；在海路上，透過蘇伊士運河及紅海亦連接了地中海及印度洋。所以有歷史紀錄以來，埃及都是兵家必爭之地。現時埃及的西面與利比亞為鄰，南與蘇丹交界，東臨紅海並與巴勒斯坦、以色列接壤，北臨地中海。

#### 自然環境

##### 地形

埃及全境有95%為沙漠。最高峰凱瑟琳山海拔2,629米。世界第一長河尼羅河從南到北流貫全境，境內長1350公里，兩岸形成寬約3—16公里的狹長河谷，並在首都開羅以北形成2.4萬平方公里的三角洲。這兩岸的河谷及三角洲地帶，雖然只佔埃及全國總面積的4%，但卻是埃及99%的人口聚居所在。蘇伊士運河是連接亞、非、歐三洲的交通要道。主要湖泊有大苦湖和提姆薩赫湖，以及亞斯文高壩形成的非洲最大的人工湖——納賽爾水庫（5000平方公里）。

##### 氣候

全境乾燥少雨。尼羅河三角洲和北部沿海地區屬地中海型氣候，1月平均氣溫12℃，7月26℃；年平均降水量50—200毫米。其餘大部分地區屬熱帶沙漠氣候，炎熱乾燥，沙漠地區氣溫可達40℃，年平均降水量不足30毫米。每年4—5月間常有「五旬風」，飛沙走石，使農作物受害。

##### 自然資源

  - 東部沙漠

磷石及石油。

  - 西奈半島

錳礦，蘇爾、阿爾等地有油田，為埃及能源所在。

### [亞美尼亞](https://zh.wikipedia.org/wiki/亞美尼亞 "wikilink")

[Armenia_(orthographic_projection).svg](https://zh.wikipedia.org/wiki/File:Armenia_\(orthographic_projection\).svg "fig:Armenia_(orthographic_projection).svg")

#### 地理位置

亞美尼亞位於西亞外高加索的南部、亞美尼亞高原的東北部，面積29800平方公里。其西與土耳其，北與喬治亞、東與亞塞拜然、南與伊朗接壤。

#### 自然環境

##### 地形

境內多高山，90%多的地區海拔1000米以上，崇山峻嶺向南一直綿延到土耳其境內。地理最高點是西北高地上的阿拉加茨山，海拔4090米。東部有塞凡窪地，窪地中的塞凡湖面積1360平方公里，為亞美尼亞境內最大的湖。主要河流為阿拉克斯河。

##### 氣候

氣候屬亞熱帶高山氣候，乾燥寒冷是其主要特色。1月平均氣溫攝氏-2～12度；7月平均氣溫攝氏24～26度。

### [烏克蘭](https://zh.wikipedia.org/wiki/烏克蘭 "wikilink")

[Europe-Ukraine_(disputed_territory).svg](https://zh.wikipedia.org/wiki/File:Europe-Ukraine_\(disputed_territory\).svg "fig:Europe-Ukraine_(disputed_territory).svg")

#### 地理位置

烏克蘭位於歐洲東部，面積為60.37萬平方公里（約占原蘇聯面積的3%），東西長1300公里，南北長900公里。南臨亞速海和黑海，陸上東北接俄羅斯，北鄰白俄羅斯，西連斯洛伐克、波蘭和匈牙利，南同摩爾多瓦、羅馬尼亞接壤。

#### 自然環境

##### 地形

烏克蘭大部分地區屬東歐平原，最高峰為西部喀爾巴阡山脈的戈維爾拉山（海拔2061米），最長的河流是聶伯河。

##### 氣候

氣候屬溫帶大陸性氣候，克里米亞半島南岸為亞熱帶氣候。\[16\]

##### 自然資源

烏克蘭自然資源豐富。它不僅擁有大片肥沃的「黑土帶」（占全世界「黑土帶」總面積的40%），而且還蘊藏著70餘種礦藏資源，如瀝青、無煙煤、鐵、錳、鉻、鈦、鉛、鋅、鋁、汞、鎳和一定量的天然氣和石油，頓巴斯是烏克蘭最大的煤田。已探明儲量為420億噸。凱瑞沃羅格的鐵礦儲量為260億噸。\[17\]

  - 森林資源

占全國領土面積的14%，跨越三個植被帶：森林沼澤帶、森林草原帶和草原帶。且具有自然保護區和天然國家公園23個（面積為77.19萬公頃，其中自然保護區14個，地球生物層保護區3個，天然國家公園6個。在克里木半島南部黑海沿岸約10公里的狹長地帶，具有獨特的自然景觀，落葉林與長綠的草地灌木叢並存。由於它優美的環境、溫和的氣候，使其成為世界著名旅遊度假聖地之一。主要樹種有：松樹、柞樹、雲杉、冷杉、椴樹、槭樹、白樺樹等。植物的部分，約有3萬種低級和高級植物，植物資源豐富。其中，藻類植物約4000種；菌類和粘菌類植物15000種以上；苔蘚類植物1000多種，導管植物4523種。

  - 動物資源

鄰海（黑海、亞述海）領海水域在內，大約有44800種動物。根據動物分類，在烏克蘭，原生動物有1200種以上；腔腸動物有40種；櫛水母網動物1種；扁蟲類1299種；紐蟲動物門有33種；原始腔腸動物有1457種；鉤頭蟲動物有58種；環節動物門動物有400多種；節肢動物門動物有39146種；軟體動物門動物有369種；有觸手亞網動物33種；棘皮動物門動物14種；脊索動物門動物有694種。\[18\]

## 歷史沿革

### 民族起源

  - 希羅多德

[阿巴扎部落始被發現於西元前五世紀初的古代作家的作品內](https://zh.wikipedia.org/wiki/阿巴扎 "wikilink")，例如：古希臘歷史學家[希羅多德](../Page/希羅多德.md "wikilink")（5世紀B.C.）在他的古代世界地圖中列出了居住在[黑海岸的人們的](../Page/黑海.md "wikilink")[阿巴扎部落](https://zh.wikipedia.org/wiki/阿巴扎 "wikilink")：2500年前，希臘歷史學家[希羅多德](../Page/希羅多德.md "wikilink")（Herodotus）指出，希臘海員在黑海東岸遇到一個名為阿巴扎的部落。後來阿巴扎的後裔在東北方向遷入高加索山脈，並在過去的1000年間變成了穆斯林。\[19\]

  - 斯特拉波

亦曾提及在[科爾基斯共同體下的](https://zh.wikipedia.org/wiki/科爾基斯 "wikilink")[阿巴扎人和其他相關部落](https://zh.wikipedia.org/wiki/阿巴扎 "wikilink")。\[20\]

  - 阿巴扎人與阿布哈茲人

阿巴扎人是白種人，與阿布哈茲人一樣。阿巴扎人是阿布哈茲部落的後裔，從原[阿布哈茲部落分出](../Page/阿布哈茲.md "wikilink")，為阿布哈茲部落的一個分支。
\[21\] \[22\]

### 民族遷徙

#### 9世紀前

阿巴茲人是阿布哈茲部落的後裔，可能早在第二個千年B.C.E.居住在黑海附近的土地上。居住於[阿布哈茲西部的Sadzen地區](../Page/阿布哈茲.md "wikilink")。
\[23\] \[24\]

#### 9世紀

到了9世紀C. E.已經分離成兩個不同的部落，塔塔坦和Shkaraua。\[25\] \[26\]

#### 13世紀

塔塔坦向南移動到北高加索。 \[27\] \[28\]

#### 14-15世紀

14世紀時，Shkaraua也跟著移動到北高加索，但此時已有一些Shkaraua被阿布哈茲人和其他被稱為切爾克斯的白種人所吸收。北高加索的阿巴扎部落於14-15世紀，從[阿布哈茲遷至阿巴尼亞](../Page/阿布哈茲.md "wikilink")。\[29\]
\[30\]15世紀時受到鄂圖曼土耳其帝國的入侵，全面性的信仰基督教。

#### 頂峰

阿巴茲人在十五和十六世紀達到了頂峰，但於17世紀受到卡巴爾德人的影響。後來的俄羅斯人逐漸控制了高加索地區。\[31\] \[32\]

#### 18-19世紀

俄羅斯斯拉夫人、切爾克斯人和土耳其人競爭高加索地區，由俄羅斯人控管了穆斯林阿巴扎人，許多阿巴茲人被搬遷到俄羅斯和土耳其，且許多斯拉夫人定居在阿巴扎人的祖國。\[33\]
\[34\]並於19世紀中期爆發高加索戰爭，這次戰爭主要的改變了居住在[北高加索地區的](https://zh.wikipedia.org/wiki/北高加索 "wikilink")[阿巴扎人的分布](../Page/阿巴扎人.md "wikilink")，成現今阿巴扎人分布在各國的狀況：一部分阿巴扎人搬到俄羅斯，另一部分受到基督教俄羅斯的威脅的伊斯蘭阿巴扎人，接受了穆斯林土耳其的提供避難所，並搬到土耳其及前[奧斯曼帝國的各個地區](https://zh.wikipedia.org/wiki/奧斯曼帝國 "wikilink")。\[35\]

### 地區性的重大事件

#### 鄂圖曼土耳其帝國的入侵

在十五世紀，奧斯曼土耳其人征服了俄羅斯和格魯吉亞的高加索山地區。當時，許多本土的阿巴扎人從基督教轉變為伊斯蘭教。\[36\]

#### 高加索戰爭

19世紀的[高加索戰爭](../Page/高加索戰爭.md "wikilink")([俄羅斯帝國在](https://zh.wikipedia.org/wiki/俄羅斯帝國 "wikilink")19世紀中期入侵[北高加索地區](https://zh.wikipedia.org/wiki/北高加索 "wikilink"))，許多阿巴扎人在此次高加索戰爭中死亡，這次戰爭也改變了居住在[北高加索地區的](https://zh.wikipedia.org/wiki/北高加索 "wikilink")[阿巴扎人的分布](../Page/阿巴扎人.md "wikilink")，使部分[阿巴扎人移民至其他中東地帶的國家](../Page/阿巴扎人.md "wikilink")(如：[約旦](https://zh.wikipedia.org/wiki/約旦 "wikilink")、[敘利亞等](https://zh.wikipedia.org/wiki/敘利亞 "wikilink"))，而成為了現今的[阿巴扎人的分布狀況](../Page/阿巴扎人.md "wikilink")。根據紅皮書的記錄，在戰爭結束後，阿巴扎人以及更多他們的卡拉克表兄弟遭到大規模驅逐到[奧斯曼帝國](https://zh.wikipedia.org/wiki/奧斯曼帝國 "wikilink")，五萬人中只有9000人留在家園。而近年來，在土耳其的阿巴扎人已經開始被吸收入土耳其主流文化，因此在土耳其成為一個傳統文化漸漸消失的獨立的族裔群體。\[37\]

#### 蘇聯

在二十世紀，阿巴茲人受到進一步的壓力。 在蘇聯上台之後的內戰期間，阿巴西人為紅衛兵和白衛兵而戰。
蘇聯時期，阿巴扎人面臨蘇聯對他們的進一步的驅逐出境，集體化和鎮壓傳統習俗及其伊斯蘭宗教。\[38\]

## 社會家庭與婚姻

### 土耳其的阿巴扎人的政治體制

現今在[卡拉恰伊-切爾克斯共和國的阿巴扎人沒有明確的行政領土中心](https://zh.wikipedia.org/wiki/卡拉恰伊-切爾克斯共和國 "wikilink")、政治中心，他們的村莊位於不同的行政管理區。而打造阿巴扎地區的計畫長期以來一直在討論中，但其中有預算問題以及潛在的土地索賠問題存在。\[39\]

### 土耳其的阿巴扎人的社會家庭與婚姻

#### 社會家庭

阿巴扎社會是一個以父為尊的社會。\[40\]

#### 婚姻

傳統：土耳其阿巴扎社會有安排婚姻的習慣。
現今：男女可自由選擇自己的伴侶，大多數阿巴扎婦女在二十多歲時結婚，但男人經常等到三十幾歲甚至四十歲。\[41\]

  - 禁忌

禁止與所有可能的親戚結婚：不得與自己祖父母同姓氏的人結婚。\[42\]

  - 綁架婚

如果發生年輕女子同意年輕男子的求婚，但女子的父母不同意的情況，年輕男子會和他的朋友綁架了年輕女子，把她帶到他的家裡，在那裡舉行結婚儀式。無論新娘是否被綁架，她的家人都不參加婚禮。當丈夫的家人宴請和慶祝時，年輕女子被要求保持沉默和僻靜。\[43\]

## 產業與生活

### 傳統生產體系

傳統的阿巴扎人的生活方式，是在高地中較低的地方種植（原來是小米，後來是玉米），飼養牲畜（綿羊，牛和馬，尤其在18世紀，阿巴扎人口很多，養活了牲畜，並以其血統的馬群而聞名）。\[44\]

#### 第一級產業

  - 傳統活動

飼養牛隻，游牧

  - 農耕：春耕，於冬開始處理土地的結石，並連根將樹木拔起。
  - 重要職業園藝：野生蘋果，梨，山茱萸，伏牛，榛子。
  - 最古老的阿巴扎職業之一-養蜂\[45\]

##### 產業分工

男\-狩獵、開墾、木材、石材、金屬加工、鐵匠-鍛造農具與武器（步槍和匕首用刀片）。

女-種植、畜牧、羊毛加工、獸皮處理、織布。\[46\]

#### 飲食

阿巴扎自古從事農業（蔬菜），畜牧業（羊肉、牛肉、乳製品），家禽養殖，這反映在阿巴扎人的菜色上面。\[47\]

  - 家禽菜餚

雞肉或火雞肉煮熟的國菜kvtIuzhdzyrdza（字面意思是“雞汁”）。\[48\]

  - 傳統農業和畜牧業產品

使阿巴扎人的傳統菜餚使用了大量的動物脂肪，特別是黃油和酥油，奶油，酸奶油，酸奶。\[49\]

  - 調味料

多數阿巴扎人與許多北高加索民族一樣，主要使用辣椒粉、搗爛的大蒜加鹽和香草製成的混合物，香草又特別以蒔蘿和百里香為主。夏普逢低的阿巴扎人則是使用加鹽的搗爛的大蒜，配上酸奶、酸奶油、紅辣椒製成的混合物。\[50\]

  - 普遍的酒精飲料bahsyma（希迪）。\[51\]

#### 服裝

切爾克斯外套
十九世紀上半葉，除了較為有錢的人，幾乎所有的服裝都由阿巴扎家庭自製。一般情況下，製作衣服的材料，是阿巴扎人財富和社會地位的指標之一，其中又以鮮豔的色彩為高貴，而工人多穿著粗布的深色衣服。阿巴扎人的服裝、髮型及頭飾深受年齡和婚姻狀況規範。\[52\]

  - 珠寶

阿巴扎人的珠寶主要由銀製成，金則使用於鍍金。\[53\]

#### 住屋

  - 農村的房子

古民居為圓形，有矩形單室和矩形多室兩種，以柳條製成。十九世紀末，阿巴扎人的屋子開始出現了鐵和石板屋頂，以及磚和原木等建材。\[54\]

### 現今在土耳其的阿巴扎人

對土耳其阿巴扎人的生活知之甚少;然而，較為已知的是，在土耳其的阿巴扎人的文化已大部分被同化為土耳其文化，過著與土耳其其他農村土著人民一樣的生活。\[55\]

#### 第一級產業

絕大多數的土耳其阿巴扎人是農民和動物生產者，十分之九的阿巴扎人生活在農村。\[56\]

##### 種植

主要作物：[菸草](https://zh.wikipedia.org/wiki/菸草 "wikilink")。[Tobacco_blossom_1580.jpg](https://zh.wikipedia.org/wiki/File:Tobacco_blossom_1580.jpg "fig:Tobacco_blossom_1580.jpg")
其它：茶葉，水果和蔬菜。\[57\]

##### 養殖

牛的生產是阿巴扎人的另一個重要經濟。\[58\]

#### 住屋

土耳其阿巴扎人的房子是由磚或混凝土建造的，大多數家庭都有陽台。\[59\]

## 信仰與習俗

### 信仰

#### 基督教

15世紀前主要信仰宗教。\[60\] \[61\]

#### 伊斯蘭教遜尼派

在15世紀和16世紀，鄂圖曼帝國入侵後，由於鄂圖曼帝國的宗教政策，使得阿巴扎人從基督教轉信為伊斯蘭教遜尼派，並以伊斯蘭教遜尼派為主要信仰宗教。\[62\]
\[63\]

#### 現今阿巴扎人在各地區的信仰

  - 俄羅斯境內

俄羅斯境內有96%阿巴扎人信奉伊斯蘭教遜尼派、1%信仰基督教以及0.02%的福音派人士。\[64\]

  - 土耳其境內

土耳其境內有99.8%阿巴扎人信奉伊斯蘭教遜尼派、其餘0.2%則信仰基督教。\[65\]

  - 埃及境內

埃及境內有99.9%阿巴扎人信奉伊斯蘭教遜尼派、其餘0.1%則信仰基督教。\[66\]

### 社會習俗

土耳其阿巴扎人對接待客人非常重視，客人與父親或祖父一樣尊重，並坐在餐桌上的榮譽地點。客人的到來伴隨著儀式盛宴。在葡萄酒，主人和客人身後，您可以品嚐到各式各樣的烤麵包片。以這種熱情的方式好客是家庭的自豪感。\[67\]

#### 節慶

在土耳其的阿巴扎人試圖通過節日和組織來保護他們的傳統文化。\[68\]

#### 儀式

與民間傳說<納爾特史詩>相關。\[69\]

#### 民間傳說

民間傳說是阿巴扎人民的重要的精神文化。\[70\]

  - 燕子傳說

嚴禁蹂躪燕子巢，蹂躪燕子巢的行為被認為是一個大罪；飛入家裡的燕子預示著家庭的幸福和快樂。

在古代，七頭怪獸發出訊息給世界的各個角落的動物，如鳥類和昆蟲，使牠們知道牠們的肉非常好吃，而且血液是最甜蜜的。然而蛇急於跑去告訴七頭怪物，人類的肉和血液才是最美味、最甜蜜的。燕子於是用嘴打擊蛇，並且刺砍牠。後來，蛇就失去了說話的能力，只能發出噓聲，讓這個可怕的消息沒有傳到七頭怪物那，使人類都得救了。\[71\]

  - 青蛙傳說

而且阿巴扎人從不殺害青蛙，因為青蛙代表的是雨水來臨的預兆。\[72\]

  - 關於馬的傳說

在阿巴扎的民間傳說裡，馬被賦予了神奇的特質，因為馬總是在人類最危險的時刻即刻給予救援。\[73\]

  - 其它

阿巴扎創建並保存神話般的史詩，包含了有關魔術、福利、寓言與動物等的故事，其中，最流行的泛高加索的故事，就是[納爾特史詩](https://zh.wikipedia.org/wiki/納爾特史詩 "wikilink")。在童話裡關注的主要是正義與邪惡及各種手段的懲罰、夢幻般的史詩主要關注的是勞動問題。故事中，正面的人物主要是牧民、耕作者、牧人、獵人和刺繡者。許多阿巴扎童話故事的結尾都是：「......我們的生活開始富足且快樂。」。\[74\]

### 音樂

自古以來，阿巴扎人藉由歌曲表達他們自己的願望、想法和感受，並有著的不同流派。民歌主要根據其內容和形式的性質區分，內容分有農力勞動pripevk、遊戲、儀式、歡呼、圓舞、舞蹈史詩（敘事）、抒情、幽默、歷史、英雄（感嘆）等；形式則以演奏的樂器和成人或兒童歌曲做區分。\[75\]

#### 樂器

根據十九世紀的阿巴扎書面報告資料，阿巴扎人的古樂器有：（myshIkvabyz），兩弦的小提琴（aphartsa），像豎琴的工具（ANDA），管了槍（kIyzhkIyzh），巴拉萊卡的桶型的木棒槌（pharchIak），阿巴扎人最古老的樂器：嗩吶和atsIarpIyna。\[76\]

### 工藝品

因為有勞動分工的情形，阿巴扎人的工藝品很早就出現。主要包含：鍛造、羊毛加工。又，加工的珠寶屬於最古老的阿巴扎工藝品之一：阿巴扎人的珠寶主要由銀製成，金則使用於各個部分的鑲鍍。\[77\]

  - 女性的責任

羊毛加工與獸皮的處理。

  -   - 製品：羊毛斗篷、休閒裝、綁腿、帽子、腰帶、鞋、氈墊、毛毯，相關皮革工藝（毛皮大衣和帽子的毛皮，皮革製成的鞋，皮，馬鞍袋，線束。羊皮）及其他各種針織品等。\[78\]

<!-- end list -->

  - 男性的責任

木材、金屬及石材的加工與鍛造（榮譽的鐵匠）。。\[79\]

  -   - 製品：製造和修理鐮刀、叉子、鐵鏟、鋤、馬蹄鐵、線束、鏈條、刀、剪刀等的金屬部件。許多鐵匠也身兼為製造武器的軍械士，在阿巴扎，武器的生產有著悠久的歷史，可以追溯到遙遠的過去，主要為步槍及匕首用的刀片，皆以銀與金裝飾。後來的阿巴扎軍械工也會製造不同口徑的子彈。\[80\]

  - 飾品

女人的乳房飾、戒指和耳環。\[81\]

## 現況\[82\]

由[卡拉恰伊-切爾克斯共和國的一名自由記者](https://zh.wikipedia.org/wiki/卡拉恰伊-切爾克斯共和國 "wikilink")，法蒂瑪·蒂斯洛娃（Fatima
Tlisova）所寫的《阿巴扎的緩慢消失》一文指出，阿巴扎的語言及文化正逐漸消失。距離高加索山腳下[卡拉恰伊-切爾克斯共和國的首府](https://zh.wikipedia.org/wiki/卡拉恰伊-切爾克斯共和國 "wikilink")[切爾克斯克不遠的](https://zh.wikipedia.org/wiki/切爾克斯克 "wikilink")，毗鄰Elburgan村莊或aul的森林正被砍伐，而當地的阿巴扎人正慢慢失去他的的身分和文化。阿巴扎第二大村莊Elburgan的長老馬桑‧卡莫夫指出：去年只有三個小孩子出生在那裏。而現代阿巴扎人正在失去身份的主要原因在於，他們中很少有人說母語，特別是在城鎮。在村莊，老一輩繼續說Abaza，但對於30歲以下的人來說，俄語現在是主要的交流語言。

距離[切爾克斯克](https://zh.wikipedia.org/wiki/切爾克斯克 "wikilink")45公里的Inzhich-Chukun村的校長阿爾伯特·揚杜巴耶夫（Albert
Jandubayev）說，如果要生存下去，阿巴扎語就需要得到當局的直接支持,並補充道：「課程內容的任何改變，都需要額外的資助，例如為教師提供額外的工作，或至少加班超額工資。但即使一個鄉村校長準備用不多的學校預算來支付這些費用，也不能保證他的行動將獲得上級批准，這就是為什麼阿巴扎人學習母語較為不易。[卡拉恰伊-切爾克斯共和國是俄羅斯聯邦最多樣化的地區之一](https://zh.wikipedia.org/wiki/卡拉恰伊-切爾克斯共和國 "wikilink")，擁有約16個民族。根據共和國的憲法，其中五個國家具有“土著民族”的地位，分別是[俄羅斯人](../Page/俄羅斯人.md "wikilink")、[卡拉奇人](https://zh.wikipedia.org/wiki/卡拉奇人 "wikilink")、[卡拉斯人](https://zh.wikipedia.org/wiki/卡拉斯人 "wikilink")、[阿巴扎人和](../Page/阿巴扎人.md "wikilink")[諾蓋人](../Page/諾蓋人.md "wikilink")。

現今的阿巴扎已被列入“紅皮書”，紅皮書為愛沙尼亞學者編寫的瀕危“俄羅斯帝國人民”的清單。

阿巴扎人儘管為數不多，在[卡拉恰伊-切爾克斯共和國內仍是一個強大的政治力量](https://zh.wikipedia.org/wiki/卡拉恰伊-切爾克斯共和國 "wikilink")，並且能夠影響總統選舉。
“阿巴扎問題”是迄今為止共和國總統選舉運動的要點之一。據說，阿巴扎以網路犯罪控制了[切爾切克斯幾乎一半的業務](https://zh.wikipedia.org/wiki/切爾切克斯 "wikilink")，並且有許多富有的阿巴扎企業家，但商人卻不願意支持阿巴茲文化，目前並無一個有效的組織為阿巴扎人籌募資金。在莫斯科的聯邦當局和切爾切克斯的共和國當局沒有在小民族的文化生存中投入資金。

擁有自治共和國最成功的企業之一的阿巴扎的Anatoly
Tlyabichev，Rezinotekhnik工廠告訴IWPR，“人們經常要求我一次過的各種幫助，醫療，出版書籍或自我宣傳沒有人用真正的計劃向我申請恢復阿巴茲語言和文化，即使這樣的計劃出現，也不能保證這筆錢將按預期的方式支付，所以我寧願給予直接援助到特定的收件人。”

2001年[卡拉恰伊-切爾克斯共和國的法律承諾保護小土著人民的權利](https://zh.wikipedia.org/wiki/卡拉恰伊-切爾克斯共和國 "wikilink")。但是，阿巴扎社會主席Shchors
Chagov表示：“2003年的預算，以及2004年聯邦或區域一級的預算都沒有在這個法律框架下為小國提供資金。”

阿巴扎人在[卡拉恰伊-切爾克斯共和國沒有明確的行政領土中心](https://zh.wikipedia.org/wiki/卡拉恰伊-切爾克斯共和國 "wikilink")、政治中心，他們的村莊位於不同的行政管理區。而打造阿巴扎地區的計畫長期以來一直在討論中，但其中有預算問題以及潛在的土地索賠問題存在。

當地議會副議長烏亞文·尤加姆科夫是在美國工作多年的阿巴扎人，並在邁阿密擁有舞蹈學校。 \[83\]

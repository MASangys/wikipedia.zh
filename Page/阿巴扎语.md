**阿巴扎语**（Абаза Бызшва 或 Abaza Byzšwa； ；），又名**阿巴札语**、**阿巴茲语**，
是[阿巴扎人的语言](../Page/阿巴扎人.md "wikilink")，主要使用于[俄罗斯联邦的](https://zh.wikipedia.org/wiki/俄罗斯联邦 "wikilink")[卡拉恰伊-切尔克斯共和国的高加索山区](../Page/卡拉恰伊-切尔克斯共和国.md "wikilink")。它有两种方言，即[阿什科勒瓦方言和作为标准书面语的](https://zh.wikipedia.org/wiki/阿什科勒瓦方言 "wikilink")[塔潘达方言](https://zh.wikipedia.org/wiki/塔潘达方言 "wikilink")。

从1932年至1938年，阿巴扎语一直是用[拉丁字母书写的](../Page/拉丁字母.md "wikilink")。自1938年起，在[俄罗斯境内的阿巴扎语使用者开始用](../Page/俄罗斯.md "wikilink")[基里尔字母来拼写该语言](https://zh.wikipedia.org/wiki/基里尔字母 "wikilink")。而在[土耳其](../Page/土耳其.md "wikilink")，阿巴扎语依然被人们用[拉丁字母来书写](../Page/拉丁字母.md "wikilink")。目前在[俄罗斯境内共有大约三万五千名使用阿巴扎语使用者](../Page/俄罗斯.md "wikilink")，在[土耳其另有大约一万名使用者](../Page/土耳其.md "wikilink")，另外在[德国和](../Page/德国.md "wikilink")[美国等地也有一些阿巴扎人说这种语言](../Page/美国.md "wikilink")。

阿巴扎语和其同属于[高加索语系](https://zh.wikipedia.org/wiki/高加索语系 "wikilink")[西北高加索语族的其他亲属语言一样](https://zh.wikipedia.org/wiki/西北高加索语族 "wikilink")，是一种高度[黏着的语言](../Page/黏着语.md "wikilink")。与其亲属语言一样，它也有着庞大的辅音音位系统（六十三个辅音音位），且元音音位系统很小（仅有两个元音音位）。它和[阿布哈兹语很接近](https://zh.wikipedia.org/wiki/阿布哈兹语 "wikilink")，两者是互通的，即两种语言的使用者都可以理解对方的语言。但阿巴扎语也有一些地方表现出了[阿布哈兹语所不具备的特点](https://zh.wikipedia.org/wiki/阿布哈兹语 "wikilink")，例如阿巴扎语中有[浊咽擦音](../Page/浊咽擦音.md "wikilink")，比方说在“三月”
（阿巴扎语：гӀапынхъамыз，国际音标：\[ʕaːpənqaːməz\]）这个词中。因此，阿巴扎语被语言学家认为是最难学的一种[高加索语言](../Page/高加索诸语言.md "wikilink")。[W·S·艾伦](https://zh.wikipedia.org/wiki/W·S·艾伦 "wikilink")、[布莱恩·欧贺林和](https://zh.wikipedia.org/wiki/布莱恩·欧贺林 "wikilink")[约翰·科拉鲁索等人都曾对阿巴扎语进行过研究](https://zh.wikipedia.org/wiki/约翰·科拉鲁索 "wikilink")。

## 语法

### 名词

阿巴札语的[名词有泛指和确指的变化](https://zh.wikipedia.org/wiki/名词 "wikilink")，名词原型可表示泛指，比如
кыт 指任意一个村子， цӀла 指任意一棵树。在其后加 кӀ 也可表示泛指，意为“某”， кыткӀ 某个村子， цӀлакӀ
某棵树。表示确指时要在名词前加 а ，如： акыт 那个村子， ацӀла 那棵树。

阿巴札语通过在名词前加词缀的方式表示[领属性](https://zh.wikipedia.org/wiki/领属性 "wikilink")[人称](../Page/人称.md "wikilink")。所有名词基本都符合以下变化：

|     |       |         |         |         |         |         |
| --- | ----- | ------- | ------- | ------- | ------- | ------- |
| 人称： | 第一人称  | 第二人称（阳） | 第二人称（阴） | 第三人称（阳） | 第三人称（阴） | 第三人称（中） |
| 单数  | с(ы)  | у(ы)    | б(ы)    | й(ы)    | л(ы)    | а       |
| 复数  | хӀ(ы) | шв(ы)   | шв(ы)   | р(ы)    | р(ы)    | р(ы)    |

例如单词“кыт”（村子）：

|     |     |        |         |         |         |         |         |
| --- | --- | ------ | ------- | ------- | ------- | ------- | ------- |
| 人称： | 词干  | 第一人称   | 第二人称（阳） | 第二人称（阴） | 第三人称（阳） | 第三人称（阴） | 第三人称（中） |
| 单数  | кыт | сыкыт  | уыкыт   | быкыт   | йыкыт   | лыкыт   | акыт    |
| 复数  | кыт | хӀыкыт | швыкыт  | швыкыт  | рыкыт   | рыкыт   | рыкыт   |

阿巴扎语名词复数的构成较为复杂，构成情况大致有以下几种：

1.  表人名词后加чва, хъацӏа男人-ахъацӏачва，чвагъва庄稼人-ачвагъвачва
2.  表人或物名词后加ква, хъацӏа男人-ахъацӏаква,урам街道-аурамква
3.  表人名词后加чваква,пхьагӀв学生-апхьагӀвчваква
4.  词根以с结尾的名词加ра，词根有音变：хӀвыс牛犊-хӀваара，сыс羊羔-сара。在这种形态之后也可再加词缀ква,
    ахӀваараква。
5.  表家族时在人名后加ргӀа,кӀамы卡姆-кӀамыргӀа卡姆家族

以нхара（工作）的第一人称单数为例：

|        |                                                     |                                                         |                                                         |                                                         |                                                           |                                                           |                                                         |                                                         |
| ------ | --------------------------------------------------- | ------------------------------------------------------- | ------------------------------------------------------- | ------------------------------------------------------- | --------------------------------------------------------- | --------------------------------------------------------- | ------------------------------------------------------- | ------------------------------------------------------- |

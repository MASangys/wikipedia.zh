阿留特語語法（Alyutor
grammar）為基於[古西伯利亞語言的](../Page/古西伯利亞語言.md "wikilink")[阿留特語之](../Page/阿留特語.md "wikilink")"語法"。阿留特語有如下之語法語詞分類：名詞、形容詞、數詞、代詞、動詞、分詞、副詞、後置詞、連接詞，及"分詞"。

## 名詞

名詞會因數字，格位，定義性及語法人稱等而變化。

有三種有兩種語法[數的特徵](../Page/数_\(语法\).md "wikilink")：單數、雙數，以及複數。

### 格位位置

  - [通格是指名詞的引用形式](../Page/通格.md "wikilink")。它作為不及物動詞子句的論元("主體")與及物子句的受體，出現在"句法所有格"(syntactic
    possessives)及[呼格上](../Page/呼格.md "wikilink")。
  - 作格用於及物動詞的主事者("主體")，作為[工具格](../Page/工具格.md "wikilink")，並作為[反被動語態句子的](../Page/反被動語態.md "wikilink")[論元](https://zh.wikipedia.org/wiki/論元_\(語音學\) "wikilink")(argument)。
  - [方位格用於位置及方向](../Page/方位格.md "wikilink")([相似格](https://zh.wikipedia.org/wiki/相似格 "wikilink")(essive
    case)與[方向格](https://zh.wikipedia.org/wiki/方向格 "wikilink")(lative
    case))，以及"被驅離(driven awayt)"之論元。
  - [與格用於受助人](https://zh.wikipedia.org/wiki/與格 "wikilink")、受益人、定向對象([向格](../Page/向格.md "wikilink")(allative
    case)情況下)，以及體驗式動詞的主題。
  - [方向格](https://zh.wikipedia.org/wiki/方向格 "wikilink")(lative
    case)用於指朝向一個目標的運動。
  - [經由格](https://zh.wikipedia.org/wiki/經由格 "wikilink")(prolative
    case)用於經由運動而運動之間的變化源於([過格](https://zh.wikipedia.org/wiki/過格 "wikilink")(perlative
    case)及[出格](../Page/出格.md "wikilink")(elative case))。
  - [等同格](https://zh.wikipedia.org/wiki/等同格 "wikilink")(equative
    case)用為"如X"(like X)，"為X"(as X)等之含義，通常配合動詞使用如"成為"(to
    become)、"變成"(to turn into)、"作為"(to work as)等。
  - 接觸格用於對物件進行接觸。
  - [使役格](https://zh.wikipedia.org/wiki/使役格 "wikilink")(causative
    case)用於使役或激勵行動之名詞短語。

### 語法人稱

語法上名詞的第一和第二人稱後綴是用來等同於參與者的話語名詞。他們只出現在[通格上](../Page/通格.md "wikilink")，對元音結尾的名詞的接上/j/；以輔音結尾的名詞接上/i/。

|      | 單數      | 雙數     | 複數     |
| ---- | ------- | ------ | ------ |
| 第1人稱 | \-j-ɣəm | \-muri | \-muru |
| 第2人稱 | \-j-ɣət | \-turi | \-turu |

  - …ʡopta am-ʡujamtawilʔ-ə-muru "是的，我們的人民"
  - japlə=q ʡujamtawilʔ-iɣəm "我就是一個人"

## 動詞

有幾種[限定動詞](https://zh.wikipedia.org/wiki/限定動詞 "wikilink")( finite
verb)及[非限定动词的形態](https://zh.wikipedia.org/wiki/非限定动词 "wikilink")。而且有幾種動詞[变位形式](../Page/变位.md "wikilink")。

### 多人稱動詞變位

限定動詞與他們的核心[論元](https://zh.wikipedia.org/wiki/論元 "wikilink")(Argument
(linguistics))在人稱及數方面一致；一致性是經由前綴及後綴展現。

### 單人稱動詞變位

單人稱動詞變化包括兩個[變格](https://zh.wikipedia.org/wiki/變格 "wikilink")，其一是第三人稱單數以動詞以\<*ɣa-...-lin*\>架構變化；其二是動詞以\<'*'n-...-qin*\>架構變化。

### 非人稱動詞變位

非人稱動詞變位包括"言語謂詞"(以[環綴形式](../Page/環綴.md "wikilink")(a…ka)形成)，及[祈使語氣](../Page/祈使語氣.md "wikilink")(以環綴形式(ɣa…a/ta)形成)。

### 非限定形式

這些結構包括[不定式](https://zh.wikipedia.org/wiki/不定式 "wikilink")，不定目的式(supine)，[動名詞](../Page/動名詞.md "wikilink")，及[分詞](https://zh.wikipedia.org/wiki/分詞 "wikilink")。

## 數詞

阿留特語數詞系统是[二十进制](../Page/二十进制.md "wikilink")、[括號內為](https://zh.wikipedia.org/wiki/括號 "wikilink")[IPA發音](https://zh.wikipedia.org/wiki/IPA "wikilink")。數詞分單一數詞及複合數詞。數詞一到五、十、十五，及二十均為單一數詞，而其它的數詞則基於單一數詞之結合而變為複合數詞。

## 語句類型

阿留特語屬於[多式综合语](../Page/多式综合语.md "wikilink")。

|                                                       |  |                                  |  |                                |
| :---------------------------------------------------- | :- | :------------------------------- | :- | :----------------------------- |
| ŋan(.ina)                                             |  | ulʲlʲaʔu.tku=ʔuttə-k             |  | na-n.illitə-tkə-ni-na…         |
| that+3PL                                              |  | walk.into.woods.masked=stick-LOC |  | LOW.A-hang-IPF-3.SG.A+3P-3PL.P |
| 'Those things on a stick, which wear masks, hung ...' |  |                                  |  |                                |

詞型屬於[黏著](https://zh.wikipedia.org/wiki/黏著 "wikilink")(Agglutination)型態，具有延伸的前綴及後綴。

|                                           |  |                        |  |            |  |           |
| :---------------------------------------- | :- | :--------------------- | :- | :--------- | :- | :-------- |
| qəlʲippə                                  |  | tətu-kki               |  | ɣeqəta     |  | a-mal-ka. |
| bread+NOM+SG                              |  | eat.with.something-INF |  | ASSOCASSOC |  | good      |
| 'Bread (eaten) with butter is excellent.' |  |                        |  |            |  |           |

其論元結構為[作通格語言](../Page/作通格語言.md "wikilink")。

|                      |  |        |  |                       |
| :------------------- | :- | :----- | :- | :-------------------- |
| ə-nannə              |  | ɣəmmə  |  | ina-ɣal-e.            |
| he-ERG               |  | me+ABS |  | 1SG.P-walk.past-3SG.A |
| 'He walked past me.' |  |        |  |                       |

[語序是可變的](../Page/語序.md "wikilink")，所以很難說哪一個類型的語序。動詞的通格語序<AVO>及<VAO>或許是最常用。

|                                          |  |                  |  |             |  |              |  |            |
| :--------------------------------------- | :- | :--------------- | :- | :---------- | :- | :----------- | :- | :--------- |
| tita·qa                                  |  | qutkinʲnʲaqu-nak |  | maŋ.ki·ʔana |  | ɣalin        |  | ənnə-ʔən.  |
| once                                     |  | (name)-ERG+SG    |  | somewhere   |  | RESRES+3SG.P |  | fishABS+SG |
| 'Once Qutkinnyaqu saw a fish somewhere.' |  |                  |  |             |  |              |  |            |

|                                       |  |                  |  |               |  |          |
| :------------------------------------ | :- | :--------------- | :- | :------------ | :- | :------- |
| ɣalin                                 |  | qutkinʲnʲaqu-nak |  | təlɣə-lŋən    |  | ŋan.tiŋ. |
| RESRES+3SG.P                          |  | (name)-ERG+SG    |  | finger-ABS+SG |  | there    |
| 'Qutkinnyaqu stuck his finger there.' |  |                  |  |               |  |          |

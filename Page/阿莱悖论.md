**阿莱悖论**（）是[决策论中的一个](../Page/决策论.md "wikilink")[悖论](../Page/悖论.md "wikilink")，由法國經濟學家[莫里斯·阿莱斯在](../Page/莫里斯·阿莱斯.md "wikilink")1952年提出。阿萊設計出這個悖論，來證明[預期效用理論](../Page/預期效用.md "wikilink")，以及預期效用理論根據的[理性選擇公理](../Page/理性選擇.md "wikilink")，本身存在邏輯不一致的問題。[丹尼尔·卡内曼與](../Page/丹尼尔·卡内曼.md "wikilink")[阿摩司·特沃斯基提出](../Page/阿摩司·特沃斯基.md "wikilink")[確定性效應](../Page/確定性效應.md "wikilink")，來解釋阿萊悖論形成的原因。

## 概論

|         |         |
| :-----: | :-----: |
| **选择1** | **选择2** |
| **赌局A** | **赌局B** |
| **赢得**  | **几率**  |
|   1百万   |  100%   |
|    0    |   1%    |
|   5百万   |   10%   |

1952年，法国经济学家、诺贝尔经济学奖获得者[莫里斯·阿莱斯作了一个著名的实验](../Page/莫里斯·阿莱斯.md "wikilink")：

对100人测试所设计的赌局：

  - 赌局A：100％的机会得到100万元。
  - 赌局B：10％的机会得到500万元，89％的机会得到100万元，1％的机会什么也得不到。

实验结果：绝大多数人选择A而不是B。即赌局A的[期望值](../Page/期望值.md "wikilink")（100万元）虽然小于赌局B的期望值（139万元），但是A的[效用值大于B的效用值](../Page/效用.md "wikilink")，即：

  -
    \(1.00U(\$1\text{ M}) > 0.89U(\$1\text{ M}) + 0.01U(\$0\text{ M}) + 0.1U(\$5\text{ M})\,\)......【1】

然后阿莱使用新赌局对这些人继续进行测试，

  - 赌局C：11％的机会得到100万元，89％的机会什么也得不到。
  - 赌局D：10％的机会得到500万元，90％的机会什么也得不到。

实验结果：绝大多数人选择D而非C。即赌局C的[期望值](../Page/期望值.md "wikilink")（11万元）小于赌局D的期望值（50万元），而且C的[效用值也小于D的效用值](../Page/效用.md "wikilink")，即：

  -
    \(0.89U(\$0\text{ M}) + 0.11U(\$1\text{ M}) < 0.9U(\$0\text{ M}) + 0.1U(\$5\text{ M})\,\)......【2】

## 數式證明

而由【2】式得：

  -
    \(0.11U(\$1\text{ M}) < 0.01U(\$0\text{ M}) + 0.1U(\$5\text{ M})\,\)

<!-- end list -->

  -
    \(1.00U(\$1\text{ M}) - 0.89U(\$1\text{ M}) < 0.01U(\$0\text{ M}) + 0.1U(\$5\text{ M})\,\)

<!-- end list -->

  -
    \(1.00U(\$1\text{ M}) < 0.89U(\$1\text{ M}) + 0.01U(\$0\text{ M}) + 0.1U(\$5\text{ M})\,\)......【3】

【3】与【1】式矛盾，即**阿莱悖论**。

阿莱悖论的另一种表述是：按照[期望效用理论](../Page/期望效用理论.md "wikilink")，风险厌恶者应该选择A和C；而风险喜好者应该选择B和D。然而实验中的大多数人选择A和D。

## 阿莱悖论的解释

出现阿莱悖论的原因是[確定性效應](../Page/確定性效應.md "wikilink")（Certainty
effect），即人在决策时，对结果确定的现象过度重视。

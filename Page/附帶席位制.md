**附帶席位制**（*Additional Member System*，AMS），又稱「額外席位制」，是[單一選區兩票制](../Page/單一選區兩票制.md "wikilink")的一種，與[聯立制](https://zh.wikipedia.org/wiki/聯立制 "wikilink")、[並立制相比](https://zh.wikipedia.org/wiki/並立制 "wikilink")，也是接近[比例代表制](../Page/比例代表制.md "wikilink")的選舉制度。

## 概要

  - 原指德國聯立制、後來日本並立制的性質，即除單一選區外，另有比例代表制席次。換言之，就是單一選區兩票制。
  - 英國用來專指在[威爾斯](../Page/威爾斯.md "wikilink")、[蘇格蘭](../Page/蘇格蘭.md "wikilink")、[倫敦實行的議會選舉制度](https://zh.wikipedia.org/wiki/倫敦 "wikilink")；日本也有同樣選舉改革方案，稱為。\[1\]\[2\]\[3\]\[4\]

## 席次分配

參考[威爾斯國民議會](../Page/威爾斯國民議會.md "wikilink")官網範例，各區域有四個附加席位名額，假設該區域分成八個選區、四個政黨參選，依[頓特法除以](../Page/漢狄法.md "wikilink")「[領先者當選席次](https://zh.wikipedia.org/wiki/領先者當選 "wikilink")＋1」，總除數則自第二輪開始採前一輪所獲得一席累計比較，從當選最少席次分配：

| 項目      | 政黨A             | 政黨B                 | 政黨C                 | 政黨D                 |
| ------- | --------------- | ------------------- | ------------------- | ------------------- |
| 政黨票數    | 50,000          | 62,000              | 48,000              | 36,000              |
| 領先者當選席次 | 3               | 3                   | 2                   | 0                   |
| 總除數     | 4               | 4                   | 3                   | 1                   |
| 分配席次計算  |                 |                     |                     |                     |
| 第一輪     | 50,000÷4=12,500 | 62,000÷4=15,500     | 48,000÷3=16,000     | 36,000÷1=**36,000** |
| 第二輪     | 50,000÷4=12,500 | 62,000÷4=15,500     | 48,000÷3=16,000     | 36,000÷2=**18,000** |
| 第三輪     | 50,000÷4=12,500 | 62,000÷4=15,500     | 48,000÷3=**16,000** | 36,000÷3=12,000     |
| 第四輪     | 50,000÷4=12,500 | 62,000÷4=**15,500** | 48,000÷4=12,000     | 36,000÷3=12,000     |
| 總計      |                 |                     |                     |                     |
| 領先者當選席次 | 3               | 3                   | 2                   | 0                   |
| 附加席位    | 0               | 1                   | 1                   | 2                   |
| 總席次     | 3               | 4                   | 3                   | 2                   |

如領先者當選席次相同，政黨票較高者優先。由於從單一選區席次最少者開始分配，同德國聯立制有利於小黨的優點，卻不會產生[超額席次](https://zh.wikipedia.org/wiki/超額席次 "wikilink")，兼具日本並立制優點。

## 採用國家

[英國目前在地方議會選舉使用](https://zh.wikipedia.org/wiki/英國 "wikilink")，國會下議院選舉仍維持[單一選區制](https://zh.wikipedia.org/wiki/單一選區制 "wikilink")。

[日本](../Page/日本.md "wikilink")曾於1993年由民間提出此種選舉改革方案，但執政[自民黨想維持](../Page/自由民主党_\(日本\).md "wikilink")[一黨獨大的優勢](https://zh.wikipedia.org/wiki/一黨獨大 "wikilink")，最終採用[並立制](https://zh.wikipedia.org/wiki/並立制 "wikilink")；[民主黨](https://zh.wikipedia.org/wiki/民主党_\(日本\) "wikilink")2009年執政後，2012年6月18日提出選舉改革方案，在並立制中導入「連用制」（即附帶席位制），隨著同年12月16日大選[自民黨重新執政而不了了之](../Page/自由民主党_\(日本\).md "wikilink")。

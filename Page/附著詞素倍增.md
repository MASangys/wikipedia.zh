**附著詞素倍增**〈Clitic doubling；pronominal
reduplication〉。在[语言学上它是一種](../Page/语言学.md "wikilink")[附著詞素](../Page/附著詞素.md "wikilink")[代詞出現在](../Page/代詞.md "wikilink")[动词](../Page/动词.md "wikilink")[片語與指涉到的所有](../Page/片語.md "wikilink")[名詞](../Page/名詞.md "wikilink")[片語之結合](../Page/片語.md "wikilink")[語態裡](../Page/語態.md "wikilink")。（相對於一些情形：像是[代名詞與所有的](../Page/代名詞.md "wikilink")[名詞](../Page/名詞.md "wikilink")[片語是以](../Page/片語.md "wikilink")[互补分布出現的語態](../Page/互补分布.md "wikilink")）

## 語言中之附著詞素倍增

附著詞素倍增在許多[語言裡都被發現到](../Page/語言.md "wikilink")，保括[阿尔巴尼亚语](../Page/阿尔巴尼亚语.md "wikilink")、[马其顿语](../Page/马其顿语.md "wikilink")、[保加利亚语](../Page/保加利亚语.md "wikilink")、[希腊语](../Page/希腊语.md "wikilink")、[羅馬尼亞語](../Page/羅馬尼亞語.md "wikilink")、[索馬利亞語](../Page/索馬利亞語.md "wikilink")、及[西班牙语](../Page/西班牙语.md "wikilink")；在每種語言裡，它都會遵循不同的[語法規則](../Page/語法.md "wikilink")。

標準的[马其顿语裡](../Page/马其顿语.md "wikilink")，附著詞素倍增與[限定](../Page/限定詞.md "wikilink")[直接受詞](../Page/宾格.md "wikilink")，及[間接受詞都是必要用到的](../Page/与格.md "wikilink")。這種情形是對比於標準的[保加利亞語中](../Page/保加利亞語.md "wikilink")，附著詞素倍增是可以選擇用或不用的。而在非標準的[馬其頓語及](../Page/馬其頓語.md "wikilink")[保加利亞語](../Page/保加利亞語.md "wikilink")[方言裡](../Page/方言.md "wikilink")，使用附著詞素倍增是有不同的規則。

## 西班牙語中之附著詞素倍增

[西班牙語是一種以附著詞素倍增而聞名的](../Page/西班牙語.md "wikilink")[語言](../Page/語言.md "wikilink")。在某些架構上，間接受詞必須以完全[名詞](../Page/名詞.md "wikilink")[片語與附著詞素倍增來表示](../Page/片語.md "wikilink")。

  -
    *Le di un regalo a mi madre.*〈我送過一件禮物給我母親。〉
    *A mis invitados siempre les ofrezco café.*〈我常常提供咖啡給客人。〉
    *No les des comida a los animales.*〈不要餵食物給動物吃。〉

以上的句子，如果逐字的譯為[英語這樣句子將會變得很冗長](../Page/英語.md "wikilink")：

|        |        |      |          |     |      |         |
| ------ | ------ | ---- | -------- | --- | ---- | ------- |
| *Le*   | *di*   | *un* | *regalo* | *a* | *mi* | *madre* |
| to.her | I.gave | a    | gift     | to  | my   | mother  |
| 對.她    | 我.送過   | 一件   | 禮物       | 給   | 我的   | 母親      |

|     |       |             |           |         |           |        |
| --- | ----- | ----------- | --------- | ------- | --------- | ------ |
| *A* | *mis* | *invitados* | *siempre* | *les*   | *ofrezco* | *café* |
| to  | my    | guests      | always    | to.them | I.offer   | coffee |
| 給   | 我的    | 客人          | 常常        | 對.他們    | 我.提供      | 咖啡     |

|      |         |       |          |     |       |            |
| ---- | ------- | ----- | -------- | --- | ----- | ---------- |
| *No* | *les*   | *des* | *comida* | *a* | *los* | *animales* |
| not  | to.them | give  | food     | to  | the   | animals    |
| 不要   | 對.它們    | 餵     | 食物       | 給   | 這些    | 動物         |

當談話涉及的對象是人或是其它的動態[實體時](../Page/實體.md "wikilink")，會使用間接[賓語附著詞素倍增的](../Page/賓語.md "wikilink")[語態](../Page/語態.md "wikilink")。不過如此的用法不是一成不變的，比如一個人可以說：〈*Siempre
ofrezco café a mis invitados*〉卻不須要使用倍增語態。

直接[賓語或許可以來倍增](../Page/賓語.md "wikilink")，不過這樣並不是一種普遍情形。而且決定來使用倍增表示型態時，經常視之為一種[語體的風格](../Page/語體.md "wikilink")（在[方言的使用上亦是如此](../Page/方言.md "wikilink")）。

  -
    *Lo vi a tu papá en la tienda.*〈在商店裡我看過你老爸。〉
    *El otro día la conocí a su esposa.*〈前些天我看過他老婆。〉

〈以上句型中，*lo* 及 *la* 經常被忽略。〉

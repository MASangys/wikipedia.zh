**附著語素**（；源自“倾斜”\[1\]）是一个[词法学](../Page/词法学.md "wikilink")概念，上指在[句法作用有如一](../Page/语法学.md "wikilink")[自由詞素](https://zh.wikipedia.org/wiki/自由詞素 "wikilink")，在[語音學上卻表現出](https://zh.wikipedia.org/wiki/語音學 "wikilink")[规范语素特性的语素](https://zh.wikipedia.org/wiki/规范词素 "wikilink")，其[發音](../Page/發音.md "wikilink")必须依附于前面或后面的[被附着词](../Page/詞_\(語言\).md "wikilink")（host）。\[2\]常见的附着语素有後附語素和前附語素， 某些[語言](../Page/語言.md "wikilink")中还存在中间语素。

附着语素常见于代词、限定词和介词，但其实它可以出现在任何语法范畴当中。正写法有时可以作为判断附着语素的参考，但附着语素并不总写成词缀形态，有些附着语素会用连字号或撇号分隔，甚至分开写成两个词语。

## 分類

### 後附語素

後附語素（Enclitic），也称後接語素、前属语素，置于被附着词（host）之后。[印欧语系](../Page/印欧语系.md "wikilink")的例子如：

  - 拉丁语的 -que“與”、-ve“或則”、-ne（一般疑问词）等。如 -que：

|     |                                                               |                |          |
| --- | ------------------------------------------------------------- | -------------- | -------- |
| 1\. | Senatus                                                       | Populus**que** | Romanus. |
|     | 元老院                                                           | 人民**=与**       | 罗马的      |
|     | [元老院与罗马人民](https://zh.wikipedia.org/wiki/元老院与罗马人民 "wikilink") |                |          |

  - [古希腊语的](https://zh.wikipedia.org/wiki/古希腊语 "wikilink") -τε (-te)“與”、-δε (-de)“但是”、-γαρ (-gar)“為了”、-ουν (-oun)“因此”等。如 -τε：

|     |           |          |       |        |
| --- | --------- | -------- | ----- | ------ |
| 2\. | άνθρωποί  | **(τε)** | θεοί  | **τε** |
|     | ánthrōpoí | **(te)** | theoí | **te** |
|     | 人         | **(=与)** | 上帝    | **=与** |
|     | 人与上帝      |          |       |        |

  - [荷兰语](https://zh.wikipedia.org/wiki/荷兰语 "wikilink")：'t（中性及第三人称代词的定冠词）、'k（第一人称代词的定冠词）、-je（第二人称代词的定冠词）、-ie（第三人称代词的定冠词）等。

<!-- end list -->

  - [俄語](https://zh.wikipedia.org/wiki/俄語 "wikilink")：-ли（一般疑问句）、-же（强调语气助词）、-не（否定词）、-бы（虚拟语气助词）等。

<!-- end list -->

  - 英语：[助动词的省略形式](../Page/助動詞.md "wikilink")、[属格](../Page/属格.md "wikilink") 's 等皆为后附语素，如 I**'m**、yo****u**'re**、****I**'d**、sh****e**'s**等。否定词的省略形式（如 could**n't**）则有争议，它被认为是一个[后缀](https://zh.wikipedia.org/wiki/后缀 "wikilink")（[规范词素](https://zh.wikipedia.org/wiki/规范词素 "wikilink")）而非[附着词素](https://zh.wikipedia.org/wiki/附着词素 "wikilink")。\[3\]

[印歐語系當中](../Page/印欧语系.md "wikilink")，有些附着语素可以被追溯到[原始印歐語](../Page/原始印歐語.md "wikilink")上，比如：*-k<sup>w</sup>e* 為****[拉丁語](https://zh.wikipedia.org/wiki/拉丁語 "wikilink") -****que、[希臘](https://zh.wikipedia.org/wiki/希臘語 "wikilink")****[語](https://zh.wikipedia.org/wiki/希臘語 "wikilink") τε(te)，及[梵语](../Page/梵语.md "wikilink") -च (-ca) 的原始形式。非印欧语系者如：

  - [日語](https://zh.wikipedia.org/wiki/日語 "wikilink")：大部分日语介词和系词都属于附着语素，如属格介词“の”（no），或****者[话题标记](../Page/话题优先语言.md "wikilink")****“は”（wa****）等。例如：

|     |                            |                          |         |
| --- | -------------------------- | ------------------------ | ------- |
| 3\. | あの人**は**                   | 私**の**                   | 甥です。    |
|     | ano-hito **wa**            | watashi **no**           | oi desu |
|     | 那个人**=<small>TOP</small>** | 我**=<small>GEN</small>** | 外甥 是    |
|     | 那个人是我的外甥。                  |                          |         |

  - [朝鮮語](../Page/朝鮮語.md "wikilink")：系動詞“이다”（ida）、形容詞“하다”（hada）以及一些[名詞](../Page/名詞.md "wikilink")及[動詞性的助词](https://zh.wikipedia.org/wiki/動詞 "wikilink")“는”（neun）。\[4\]然而，其它的一些分析指出[名詞](../Page/名詞.md "wikilink")性助词並沒有附着词素的性质，其作用更近乎短语词缀。\[5\]

### 前附語素

前附語素（Proclitic），也称前接語素、[连音](../Page/连音.md "wikilink")语素，置于被附着词之前。

  - [英語元音前的](https://zh.wikipedia.org/wiki/英語 "wikilink")[冠词](../Page/冠词.md "wikilink")都属于前附語素。如：

|     |                                           |       |
| --- | ----------------------------------------- | ----- |
| 4\. | **an**                                    | apple |
|     | **<small>SG</small>.<small>DET</small>=** | 苹果    |
|     | 一个苹果                                      |       |

  - [法语](../Page/法语.md "wikilink")：

|     |                                       |                                                                                      |                       |
| --- | ------------------------------------- | ------------------------------------------------------------------------------------ | --------------------- |
| 5a. | il                                    | **s**'est                                                                            | réveillé.             |
|     | 3<small>SG</small>.<small>PRO</small> | **3<small>SG</small>.<small>REFL</small>=**存在.<small>IND</small>.<small>PRES</small> | 醒.<small>PAST</small> |
|     | 他醒了。                                  |                                                                                      |                       |

|     |                                       |                                                                 |
| --- | ------------------------------------- | --------------------------------------------------------------- |
| 5b. | je                                    | **t**'aime.                                                     |
|     | 1<small>SG</small>.<small>PRO</small> | **2<small>SG</small>=**爱.<small>IND</small>.<small>PRES</small> |
|     | 我爱你。                                  |                                                                 |

### 中間語素

中間語素（Mesoclitic）出現在被附着词语干與其它的[詞綴](../Page/詞綴.md "wikilink")之間。如[葡萄牙语](https://zh.wikipedia.org/wiki/葡萄牙語 "wikilink")：

|     |                                                          |                             |
| --- | -------------------------------------------------------- | --------------------------- |
| 6\. | Ela                                                      | evá-**lo**-ia.              |
|     | 3<small>SG</small>.<small>FEM</small>.<small>PRO</small> | 拿**=它=**<small>COND</small> |
|     | 她会拿走。                                                    |                             |

### 内附语素

内附语素（Endoclitic），即將[词根中斷為兩部份並將内接语素置於其間](https://zh.wikipedia.org/wiki/词根 "wikilink")。内附语素由于直接挑战了，故長久以來一直认为这种附着语素不太可能会存在，不過后来在[东北高加索语系](../Page/东北高加索语系.md "wikilink")的找到存在的证据。\[6\]內接詞也出現在[普什图语](../Page/普什图语.md "wikilink")。\[7\]

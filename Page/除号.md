**除号**（符号：**÷**，，复数：obeli）是由一根短横线和横线两侧的两点构成的符号，主要用来表示数学的[除法](../Page/除法.md "wikilink")运算。

## 历史

[Obelos.png](https://zh.wikipedia.org/wiki/File:Obelos.png "fig:Obelos.png") 英语中除号的单词“obelus”来自于[希腊语](https://zh.wikipedia.org/wiki/希腊语 "wikilink")，表示尖锐的棍棒，这与英语中表示[方尖碑](../Page/方尖碑.md "wikilink")（Obelisk）的词源相同。

这个符号原本在古代手稿中应用于标记被损坏或虚假的文章，由阿里斯塔克发明以标记[荷马](../Page/荷马.md "wikilink")作品中的疑问，常被用来标记[福音书](../Page/福音书.md "wikilink")手稿中那些有疑问的章节，比如遭受质疑的[约翰福音中的章节](https://zh.wikipedia.org/wiki/约翰福音 "wikilink")。而如今这种用法已经被弃置不用\[1\]。

1544年，[德国](../Page/德国.md "wikilink")数学家于其出版的《整数算术》（*Arithmetica integra*）中以一个或一对括号作除号，如以“8 ) 24”或“8 ) 24 (”表示“24 ÷ 8”；[威廉·奥特雷德](../Page/威廉·奥特雷德.md "wikilink")则以“a ) b ( c”来表示“b ÷ a = c”；J.马洪（1701年）则以“D ) A + B - C”表示“( A + B - C )÷D”\[2\]。至1545年，施蒂费尔又改以大写德文字母D表示除，其后，斯蒂文也采用了这符号，他以“5②Dsec①Mter②”表示\(\frac{5x^2}{y}z^2\)，而戈里马德（1751年）则以反写字母“Ɑ”表示除。另外，昆尼亚于1790年出版的《数学原理》中，以平放的小写字母p表示除\[3\]。

现在被广泛使用的除法运算符号“÷”被称为雷恩记号，它首次作为除号被使用，是在瑞士学者1659年所著的一本代数學书中\[4\]。但其實真正的引入也有可能归功于这本书的编辑约翰·佩尔\[5\]。1668年，该书被译成英文，才逐渐被人们所认识和接受，得以流行起来，沿用至今。

因为“÷”号在欧洲大陆曾长期被用来表示减法，为了与减法区别，德国数学家[戈特弗里德·莱布尼茨](../Page/戈特弗里德·莱布尼茨.md "wikilink")於他的一篇論文《組合的藝術》（*Dissertatio de arte combinatoria*）內首以“∶”作除号，与当时流行的比号一致。现在世界上有些国家仍然用“∶”做除号\[6\]。

至于为何要使用“÷”作为除号则有两种说法：一种说法是，该符号代表除法以分数的形式来表示，一的上方和下方各加“.”，分别代表分子分母；另一种说法是，不以分数表示时，横线上下的“.”是用来与“－”区别的符号\[7\]。

## 使用

[Skjermbilete_2012-11-03_kl._02.48.36.png](https://zh.wikipedia.org/wiki/File:Skjermbilete_2012-11-03_kl._02.48.36.png "fig:Skjermbilete_2012-11-03_kl._02.48.36.png")2010财年官方的购销表《«Næringsoppgave 1»》中，“÷”以减号的变体被使用\]\] 除号在算术中用来表示两个数相除。例如：

\[a \div b = c\] </span>上面等式中，a为被除数，b为除数，得到c为[商](https://zh.wikipedia.org/wiki/商數 "wikilink")。

在部分文化中，“÷”可以被用作表示范围（如1÷10，表示1到10的范围），甚至可以被用作减号\[8\]。

## 输入

在操作系统[Microsoft Windows中](https://zh.wikipedia.org/wiki/Microsoft_Windows "wikilink")，除号可以通过按住[Alt键](https://zh.wikipedia.org/wiki/Alt键 "wikilink")，同时按数字小键盘0247输入，或者同时按、[shift键和](https://zh.wikipedia.org/wiki/shift键 "wikilink")“+ =”键输入。

在操作系统[Mac OS中](https://zh.wikipedia.org/wiki/Mac_OS "wikilink")，可通过Option键加上“/”键输入。

在[Unicode中](https://zh.wikipedia.org/wiki/Unicode "wikilink")，“÷”的编码是U+00F7。在[HTML](../Page/HTML.md "wikilink")中，它可以通过编码`÷`或者`÷`（在HTML 3.2），或者是`÷`输入。在[LaTeX中](https://zh.wikipedia.org/wiki/LaTeX "wikilink")，可以使用`\div`来表示除号。

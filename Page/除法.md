[数学中](../Page/数学.md "wikilink")，尤其是在基本[计算裏](../Page/算术.md "wikilink")，**除法**可以看成是「**[乘法的反运算](../Page/乘法.md "wikilink")**」，也可以理解为「**重复的[减法](../Page/减法.md "wikilink")**」。除法运算的本质就是「把参与运算的除数变为\(1\)，得出被除数的值」。

例如：\(6 \div 3 = 2\)，就好像\(6-3-3 = 0\)， \(\begin{cases}
6-3 = 3 \\
3-3 = 0
\end{cases}\)，\(6\)被\(3\)減了兩次後，就變成了\(0\)。

如果

\[a \times b = c\]
而且\(b\)[不等于](../Page/不等于.md "wikilink")[零](../Page/零.md "wikilink")，那么

\[a = c \div b\]

其中，a称为**[商数](../Page/商数.md "wikilink")**，b称为**除数**，c称为**被除数**。

如果除式的商數（\(a\)）必須是[整數](../Page/整數.md "wikilink")，则称为[带餘除法](../Page/带餘除法.md "wikilink")，\(a \times b\)与\(c\)相差的数值，称为**[餘數](../Page/餘數.md "wikilink")**（\(d\)）。

\[c \div b = a \dots d\]

這也意味著

\[c = a \times b + d\]

在[高等数学](../Page/高等数学.md "wikilink")（包括在[科学与](../Page/科学.md "wikilink")[工程学中](../Page/工程学.md "wikilink")）和[计算机编程语言中](../Page/程序设计语言.md "wikilink")，\(c \div b\)写成\(c/b\)。如果我们不需要知道确切值或者留待以后引用，这种形式也常常是称之为[分数的最终形式](../Page/分数.md "wikilink")。其中尋找商數的[函數為](../Page/函數.md "wikilink")\(\operatorname{div}\)，尋找餘數的函數則為\(\operatorname{mod}\)。

在大部分的非[英语语言中](../Page/英语.md "wikilink")，\(c:b\)代表\(c \div b\)的**比**，讀做c比b；\(c/b\)則代表\(c \div b\)的**比值**。用法请参照[比例](../Page/比例.md "wikilink")。

## 整除

**整除**是[数学中两个](../Page/数学.md "wikilink")[自然数之间的一种](../Page/自然数.md "wikilink")[关系](../Page/关系.md "wikilink")。自然数\(a\)可以被自然数\(b\)整除，是指\(b\)是\(a\)的[因數](../Page/因數.md "wikilink")，且a是b的[整数](../Page/整数.md "wikilink")[倍数](../Page/倍数.md "wikilink")，也就是\(a\)**除以**\(b\)没有[餘数](../Page/餘数.md "wikilink")。

\[a \div b = q \dots 0\] 因數判別法可參照[整除規則](../Page/整除規則.md "wikilink")。

### 表示法

\(b \mid a\)表示\(b\)整除\(a\)，即\(a\)是\(b\)的倍数，\(b\)是\(a\)的[因数](../Page/因数.md "wikilink")。

### 举例

\(15\)可以被\(5\)整除，记作\(5 \mid 15\)。

\(20\)不能被\(6\)整除（因为餘数为\(2\)），记作\(6 \nmid 20\)。在\(\mid\)上加一条[斜线即表示不整除](../Page/斜线.md "wikilink")⋯⋯

## 除法计算

根据[乘法表](../Page/乘法表.md "wikilink")，两个整数可以用[长除法](../Page/长除法.md "wikilink")（直式除法）笔算。如果被除数有[分数部分](../Page/分数.md "wikilink")（或者说时[小数点](../Page/小数点.md "wikilink")），计算时将小数点带下来就可以；如果除数有小数点，将除数与被除数的小数点同时移位，直到除数没有小数点。

[算盘也可以做除法运算](../Page/算盘.md "wikilink")。

### 長除法

[長除法俗稱](../Page/長除法.md "wikilink")「**長除**」，適用於正式除法、[小數除法](../Page/小數.md "wikilink")、[多項式除法](../Page/多項式.md "wikilink")（即[因式分解](../Page/因式分解.md "wikilink")）等較重視計算過程和[商數的除法](../Page/商數.md "wikilink")，過程中兼用了[乘法和](../Page/乘法.md "wikilink")[減法](../Page/減法.md "wikilink")。

使用長除法計算\(1260257 \div 37\)：

\end{array}
\! \! \! \! \!
\begin{array}{r}

150 \quad \ \ \\

### 短除法

[短除法是長除法的簡化版本](../Page/短除法.md "wikilink")。在短除法裏，被除數放中央，旁以一L型符號表示除法，被除數左側為除數，下側為商，省去了長除法逐層計算的過程。

  - 使用短除法計算\(3 \div 7\)的近似值：

\[\begin{array}{r}
7 \ | \! \underline{\, \ 3.00000000000000000 \dots \ } \\
0.42857142857142857 \dots \
\end{array}\]

  - 使用短除法計算\(420\)的[質因數分解](../Page/質因數分解.md "wikilink")：

\[\begin{array}{r}
2 \ | \! \underline{\, \ \ \ 420 \ } \\
2 \ | \! \underline{\, \ \ 210 \ } \\
3 \ | \! \underline{\, \ 105 \ } \\
5 \ | \! \underline{\, \ 35 \ } \\
7 \
\end{array}\]

\[420 = 2^2 \times 3 \times 5 \times 7\]

  - 使用短除法計算\(420,270\)的[最大公因數及](../Page/最大公因數.md "wikilink")[最小公倍數](../Page/最小公倍數.md "wikilink")：

\[\begin{array}{r}
2 \ | \! \underline{\, \ \ \ 420 \quad 270 \ } \\
3 \ | \! \underline{\, \ \ 210 \quad 135 \ } \\
5 \ | \! \underline{\, \ 70 \quad \ \ 45 \ } \\
14 \quad \ \ \ \ 9 \
\end{array}\]

\[\begin{cases}
\gcd(420, 270) = 2 \times 3 \times 5 = 30 \\
\operatorname{lcm}(420, 270) = 2 \times 3 \times 5 \times 14 \times 9 = 3780
\end{cases}\]

## [多項式的除法](../Page/多項式.md "wikilink")

和[整数之间的](../Page/整数.md "wikilink")[带余除法类似](../Page/带余除法.md "wikilink")，一元多项式之间也可以进行带余除法。可以证明，设有多项式\(A\)和非零多项式\(B\)，则存在唯一的多项式\(Q\)和\(R\)，满足：

\[A = BQ + R\] 而多项式\(R\)若非零多项式，則其冪次严格小于\(B\)的冪次。

作为特例，如果要计算某个多项式\(P\)除以一次多项式\(X-a\)得到的餘多项式，可以直接将\(a\)代入到多项式\(P\)中。\(P\)除以\(X-a\)的餘多项式是\(P(a)\)。

具体的计算可以使用类似直式除法的方式。例如，计算\(X^3 - 12X^2 - 42\)除以\(X-3\)，列式如下：

\[\begin{matrix}
\qquad\quad\;\, X^2 \; - 9X \quad - 27\\
\qquad\quad X-3\overline{\vert X^3 - 12X^2 + 0X - 42}\\
\;\; \underline{\;\;X^3 - \;\;3X^2}\\
\qquad\qquad\quad\; -9X^2 + 0X\\
\qquad\qquad\quad\; \underline{-9X^2 + 27X}\\
\qquad\qquad\qquad\qquad\qquad -27X - 42\\
\qquad\qquad\qquad\qquad\qquad \underline{-27X + 81}\\
\qquad\qquad\qquad\qquad\qquad\qquad\;\; -123
\end{matrix}\]

因此，商式是\(\ X^2 - 9X - 27\)，餘式是\(\ -123\)。

## 重要性質

通常不定义[除以零这种形式](../Page/除以零.md "wikilink")。亦即當除以0 或分數的分母為0
時，該式或該數**無意義**。

在[線性代數中](../Page/線性代數.md "wikilink")（以及其在[量子力學中的運用](../Page/量子力學.md "wikilink")），**升算符**或**降算符**——集合起來稱為**階梯算符**——為可以將另一算符的[本徵值分別做增加或減少的算符](../Page/本徵值.md "wikilink")。在量子力學中，有時候升算符稱為[創生算符](../Page/創生算符.md "wikilink")，而降算符稱為[消滅算符](../Page/消滅算符.md "wikilink")。階梯算符在量子力學中的著名應用是出現在[量子諧振子以及](../Page/量子諧振子.md "wikilink")[角動量的形式中](../Page/角動量.md "wikilink")。

假設一阶梯算符X與一任意算符N有[對易關係如下](../Page/對易關係.md "wikilink")：

\[[N,X] \equiv NX - XN = cX\,\]
*c*為某一[純量](../Page/純量.md "wikilink")。則算符*X*的作用會表現為：將算符*N*的一個本徵態\(|n\rangle\)，其本徵值移動了*c*：

|                 |                                    |
| --------------- | ---------------------------------- |
| \(NX|n\rangle\) | \({}= (XN+[N,X])|n\rangle\)        |
|                 | \({} = (XN + cX)|n\rangle\)        |
|                 | \({} = XN|n\rangle + cX|n\rangle\) |
|                 | \({} = Xn|n\rangle + cX|n\rangle\) |
|                 | \({} =(n+c)X|n\rangle\)            |

換句話說，若\(|n\rangle\)是算符\(N\)的一個本徵態，帶有本徵值*n*，則\(X|n\rangle\)也是*N*的一個本徵態，帶有本徵值*n*
+
*c*。對*N*來說，升算符是一個算符*X*使得*c*是正實數，而降算符則是使*c*是負實數。若*N*是[厄米算符](../Page/厄米算符.md "wikilink")（Hermitian
operator），則*c*必須要是實數，而*X*的[厄米伴算符](../Page/厄米伴算符.md "wikilink")（Hermitian
adjoint）*X*<sup>†</sup>遵守如下對易關係：

\[[N,X^\dagger] = -cX^\dagger \,\]。特別是若*X*對*N*來說是降算符，則*X*<sup>†</sup>對*N*來說會是個升算符，反之亦然。

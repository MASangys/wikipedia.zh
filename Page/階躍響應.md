[High_accuracy_settling_time_measurements_figure_1.png](https://zh.wikipedia.org/wiki/File:High_accuracy_settling_time_measurements_figure_1.png "fig:High_accuracy_settling_time_measurements_figure_1.png")（overshoot）、[振鈴](../Page/振鈴.md "wikilink")（ringing），在[安定時間](../Page/安定時間.md "wikilink")（settling time）後回到穩定值。\]\]

**階躍響應**是指系統在其輸入為[單位階躍函數時](https://zh.wikipedia.org/wiki/單位階躍函數 "wikilink")，其輸出的變化\[1\]。在[電子工程或](https://zh.wikipedia.org/wiki/電子工程 "wikilink")[自動控制領域中](https://zh.wikipedia.org/wiki/自動控制 "wikilink")，階躍響應是指[系統](../Page/系統.md "wikilink")的輸入在很短時間由0變成1時，其輸出的[時域特性](https://zh.wikipedia.org/wiki/時域 "wikilink")。此概念可以延伸到使用抽象数学概念的[动力系统](../Page/动力系统.md "wikilink")，以演化参数表示其特性。

分析系統的階躍響應有助於了解系統的特性，因為當輸入在長時間穩態後，有快速而大幅度的變化，可以看出系統各個部份的特性。而且也可以知道系統的穩定性。

## 反馈放大器的阶跃响应

[Block_Diagram_for_Feedback.svg](https://zh.wikipedia.org/wiki/File:Block_Diagram_for_Feedback.svg "fig:Block_Diagram_for_Feedback.svg")

本节介绍了一个简单的[负反馈放大器](../Page/负反馈放大器.md "wikilink")的阶跃响应如图1所示。反馈放大器由一个增益为 *A*<sub>OL</sub> 的主开环放大器和反馈因子为 β 的反馈回路。以下會分析此回授放大器，確認其阶跃响应和控制響應的時間常數之間的關係，也看阶跃响应和回授量之間的關係。

负反馈放大器增益为（见[负反馈放大器](../Page/负反馈放大器.md "wikilink")）：

\[A_{FB} = \frac {A_{OL}} {1+ \beta A_{OL}}\]，

其中

  -
    *A*<sub>OL</sub> = 开环增益
    *A*<sub>FB</sub> = 闭环增益（存在负反馈时的增益）
    β = 反馈因子。

### 有一个主导极点

在许多情况下，可以用时间常数 τ 的单一主导极点很好地模拟正向放大器，它的开环增益为：

\[A_{OL} = \frac {A_0} {1+j \omega \tau}\]，

零频率增益为 *A*<sub>0</sub>，角频率 ω = 2π*f*。这种正向放大器有单位阶跃响应

\[S_{OL}(t) = A_0(1 - e^{-t / \tau})\]，

是从零到新平衡值 *A*<sub>0</sub> 的指数趋近。

单极点放大器的传递函数导出闭环增益：

\[A_{FB} = \frac {A_0} {1+ \beta A_0} \times  \frac {1} {1+j \omega \frac { \tau } {1 + \beta A_0} }\]

此闭环增益与开环增益是形式相同：均为单极滤波器。其阶跃响应的形式相同：一个趋于新平衡值的指数衰减。但闭环阶跃函数的时间常数为 τ / (1 + β *A*<sub>0</sub>)，因此因此它比前向放大器的响应快，是其 1 + β *A*<sub>0</sub> 倍：

\[S_{FB}(t) =  \frac {A_0} {1+ \beta A_0}(1 - e^{-t (1 + \beta A_0)/ \tau})\]，

由于反馈因子 β 的增加，阶跃响应会更快，直到最初假设的一个主导极点不再准确。如果有第二个极点，则随着闭环时间常数区域第二个极点的时间常数，需要进行双极点分析。

### 双极点放大器

在开环增益有两个极点的情况下（两个[时间常数](https://zh.wikipedia.org/wiki/时间常数 "wikilink")，τ<sub>1</sub>和τ<sub>2</sub>），阶跃响应更为复杂。开环增益为：

\[A_{OL} = \frac {A_0} {(1+j \omega \tau_1) (1 + j \omega \tau_2)},\]

零频率增益为 *A*<sub>0</sub>，角频率 ω = 2π*f*。

#### 分析

双极点放大器的传递函数可以导出闭环增益：

\[A_{FB} = \frac {A_0} {1+ \beta A_0} \times \frac {1} {1+j \omega \frac { \tau_1 + \tau_2 } {1 + \beta A_0} + (j \omega )^2 \frac { \tau_1 \tau_2} {1 + \beta A_0} }\]

[Conjugate_poles_in_s-plane.PNG](https://zh.wikipedia.org/wiki/File:Conjugate_poles_in_s-plane.PNG "fig:Conjugate_poles_in_s-plane.PNG")

放大器的时间相关性通过切换变量 *s* = *j*ω 很容易发现，于是增益变为：

\[A_{FB} = \frac {A_0} { \tau_1 \tau_2 } \times \frac {1} {s^2 +s \left( \frac {1} {\tau_1} + \frac {1} {\tau_2} \right) + \frac {1+ \beta A_0} {\tau_1 \tau_2}}\]

这个表达式的极点（即分母的零点）位于：

\[2s = - \left( \frac {1} {\tau_1} + \frac {1} {\tau_2} \right)\]

\[\pm \sqrt { \left( \frac {1} {\tau_1} - \frac {1} {\tau_2} \right) ^2 -\frac {4 \beta A_0 } {\tau_1 \tau_2 } },\]

这说明对于足够大 *βA*<sub>0</sub> 平方根就会变成虚数，极点的位置是共轭复数，*s*<sub>+</sub> 与 *s*<sub>−</sub>；参见图2：

\[s_{\pm} = -\rho \pm j \mu, \,\]

其中

\[\rho = \frac {1}{2} \left( \frac {1} {\tau_1} + \frac {1} {\tau_2} \right ),\]

而

\[\mu = \frac {1} {2} \sqrt { \frac {4 \beta A_0} { \tau_1 \tau_2} - \left( \frac {1} {\tau_1} - \frac {1} {\tau_2} \right)^2 }\] 使用极坐标系，根的半径的模为 |*s*|（图2）：

\[| s | = |s_{ \pm } | =  \sqrt{ \rho^2 +\mu^2},\]

而角坐标 φ 为：

  -
    \(\cos \phi = \frac { \rho} { | s | } , \sin \phi = \frac { \mu} { | s | }\)

[拉普拉斯变换表告诉我们这样一个系统的时间响应是由两个函数的组合而成的](https://zh.wikipedia.org/wiki/拉普拉斯变换 "wikilink")：

\[e^{- \rho t} \sin ( \mu t) \quad\text{and} \quad\] \(e^{- \rho t} \cos ( \mu t),\)

也就是说，解在时间上阻尼振荡。特别是，该系统的单位阶跃响应为：\[2\]

\[S(t) = \left(\frac {A_0} {1+ \beta A_0}\right)\left(1 -   e^{- \rho t} \  \frac {  \sin \left( \mu t + \phi \right)}{ \sin( \phi )}\right)\ ,\]

化简为

\[S(t) = 1 -   e^{- \rho t} \  \frac {  \sin \left( \mu t + \phi \right)}{ \sin( \phi )}\\]

当 *A*<sub>0</sub> 趋于无穷大时，反馈系数 β 为1。

注意到响应的阻尼是由 ρ 決定的，也就是由开环放大器的时间常数确定的。相反，振荡的频率是由 μ 确定的，也就是由, 通过 β*A*<sub>0</sub> 由反馈参数确定的。因为 ρ 涉及到时间常数的倒数之和，所以可以发现 ρ 主要受到两个时间常数中*较短*的那个影响。

#### 结果

[Step_response_for_two-pole_feedback_amplifier.PNG](https://zh.wikipedia.org/wiki/File:Step_response_for_two-pole_feedback_amplifier.PNG "fig:Step_response_for_two-pole_feedback_amplifier.PNG") 图3显示了参数 μ 取3个不同值时，单元阶跃输入的时间响应。可以看出随着 μ 增加，振荡频率也会增加，但振荡被包含在由指数型函数 \[ 1 − exp (−ρt) \] 和 \[ 1 + exp(−ρt) \] 确定的两条渐近线以内。这两条渐近线是由 ρ 决定的，所以也就由开环放大器的时间常数决定，与反馈无关。

终值的振荡现象被称为[振鈴](../Page/振鈴.md "wikilink")。[过冲](../Page/过冲.md "wikilink")是指摆动最大值高于终值，显然会随着 μ 增加。同样，下冲是指摆动最小值低于终值，同样也会随着 μ 增加。[安定時間](../Page/安定時間.md "wikilink")是指从终值出发，降到低于某个特定水平（终值的10%）用的时间。

稳定时间对 μ 的依赖性不明显，而双极点系统的近似可能不能达到用稳定时间对反馈的依赖性作出现实中的结论的准确性。但渐近线 \[ 1 − exp (−ρt) \] 与 \[ 1 + exp (−ρt) \] 显然影响稳定时间，它们被开环放大器的时间常数控制，特别是在2个时间常数中的时间较短的。这表明开环放大器的设计必须满足稳定时间的规定。

此分析的有两个主要结论：

1.  反馈控制了给定开环放大器并给定开环时间常数 τ<sub>1</sub> 与 τ<sub>2</sub> 时终值上下振荡的幅度。
2.  开环放大器决定了稳定时间。它确定了图3中的时标，开环放大器越快，时标越快。

顺便说一句，可以注意到实际中与线性双极点模型的偏离主要来自两个方面：其一，实际放大器的极点多於两个，零点也是；其二，实际放大器是非线性的，所以它们的阶跃响应会随着信号幅度变换。 [Overshoot_control.PNG](https://zh.wikipedia.org/wiki/File:Overshoot_control.PNG "fig:Overshoot_control.PNG")

#### 控制过冲

以下會說明如何用適當的參數選擇來減少過沖。

利用以上的公式，可以將階躍響應微分找最大值來計算過沖量。其過沖量最大值*S*<sub>max</sub> 為 ：\[3\]

\[S_\max= 1 + \exp \left( - \pi \frac { \rho }{ \mu } \right)\]

階躍響應的終值為1，因此其指數即為過沖量。可以看出若μ = 0，其過沖量為0，也就是：

\[\frac {4 \beta A_0} { \tau_1 \tau_2} = \left( \frac {1} {\tau_1} - \frac {1} {\tau_2} \right)^2\]

令*x* = ( τ<sub>1</sub> / τ<sub>2</sub> )<sup>1 / 2</sup> ，可以求解二個時間常數之間的比例，結果為

\[x = \sqrt{ \beta A_0 } + \sqrt { \beta A_0 +1 } \,\]

因為β *A*<sub>0</sub> \>\> 1，因此平方根中的1可以省略，得到

\[\frac { \tau_1} { \tau_2} = 4 \beta A_0\]

換句話說，第一個時間常數需遠大於第二個時間常數。有時系統為了一些特性，需要允許一些過沖量，以下的關係中引入一個因子α：

\[\frac { \tau_1} { \tau_2} = \alpha \beta A_0,\]

α可以依允許的過沖量來設計。

圖4就是描述其程序。比較上圖（α = 4）及下圖（α = 0.5）可以看出α較小，可以加快響應的速度，但也讓過沖量變大。中間的圖α = 2為[幅度最平坦的滤波器](https://zh.wikipedia.org/wiki/巴特沃斯滤波器#幅度最平坦的滤波器 "wikilink")，在[波德圖](../Page/波德圖.md "wikilink")上沒有尖點。此設計有[經驗法則](../Page/經驗法則.md "wikilink")內建的安全預度，可以處理像重零點、重極點、非線性（例如和信號振幅相依的特性）及製造的變異，這些都可能造成過大的過沖量。極點擺放位置（也就是α）的調整是的主題，其中一個方式是。

#### 稳定时间控制

图3中阶跃响应中振铃的幅度是由阻尼因数 exp ( −ρ t ) 决定的。也就是说，如果我们指定出可接受的阶跃响应离终值的偏移量 Δ，即：

\[S(t)  \le 1 + \Delta, \,\]

在时间长于稳定时间 *t*<sub>S</sub> 这个前提下，无论 β *A*<sub>OL</sub> 的值为多少时这个条件都能满足。稳定时间 *t*<sub>S</sub> 为：\[4\]

\[\Delta = e^{- \rho t_S }\text{ or }t_S = \frac { \ln \left( \frac{1} { \Delta} \right) } { \rho } = \tau_2 \frac {2 \ln \left( \frac{1} { \Delta} \right) } { 1 + \frac { \tau_2 } { \tau_1} } \approx 2 \tau_2 \ln \left( \frac{1} { \Delta} \right),\]

因為過沖的條件，τ<sub>1</sub> = *αβA*<sub>OL</sub> τ<sub>2</sub>，因此τ<sub>1</sub> \>\> τ<sub>2</sub> 成立。一般稳定时间條件是指稳定时间和其單位增益的頻寬成反比的情形，原因是1/(2π τ<sub>2</sub>)接近放大器在典型下的頻寬。不過此結果比[經驗法則](../Page/經驗法則.md "wikilink")的結果更準確。例如，若Δ = 1/e<sup>4</sup> = 1.8 %，其稳定时间為*t*<sub>S</sub> = 8 τ<sub>2</sub>。

一般而言，對過沖量的控制會決定二個時間常數的比例，稳定时间*t*<sub>S</sub> 會決定 τ<sub>2</sub>\[5\]\[6\] \[7\]。

#### 相位裕度

[Phase_for_Step_Response.PNG](https://zh.wikipedia.org/wiki/File:Phase_for_Step_Response.PNG "fig:Phase_for_Step_Response.PNG") 其次，極點比例τ<sub>1</sub>/τ<sub>2</sub>也和回授放大器的相位裕度有關\[8\]。圖5是二個極點放大器的[波德圖](../Page/波德圖.md "wikilink")，頻率到第二個極點的位置。圖5的假設是頻率*f*<sub>0 dB</sub>在位在*f*<sub>1</sub> = 1/(2πτ<sub>1</sub>)的最小極點及位在*f*<sub>2</sub> = 1/(2πτ<sub>2</sub>)的第二極點之間。如圖5所示，若 α ≥ 1，此假設即成立。

利用圖5，頻率（用*f*<sub>0 dB</sub>表示)為迴路增益 β*A*<sub>0</sub>滿足單位增益或是0 dB條件的位置，可以定義為：

\[| \beta A_\text{OL} ( f_\text{0 db} ) | = 1\\]

波德增益圖中增益下降的斜率是 20 dB/decade，頻率每增加十倍，增益下降的比例相同：

\[f_\text{0 dB} = \beta A_0 f_1\,\]

相位裕度是頻率在*f*<sub>0 dB</sub> 處，相位和−180°之間的距離，因此裕度為：

\[\phi_m = 180 ^\circ - \arctan (f_\text{0 dB} /f_1) - \arctan ( f_\text{0 dB} /f_2)\\]

因為*f*<sub>0 dB</sub> / *f*<sub>1</sub> = *βA*<sub>0</sub> \>\> 1，有關*f*<sub>1</sub>的項為90°，因此相位裕度為：

\[\phi_m = 90 ^\circ - \arctan ( f_\text{0 dB} /f_2) \,\]

\[= 90 ^\circ -  \arctan \left( \frac {\beta A_0 f_1} {\alpha \beta A_0 f_1 } \right)\]

\[= 90 ^\circ - \arctan \left( \frac {1} {\alpha } \right) = \arctan \left(  \alpha  \right)\]

若α = 1，則 φ<sub>m</sub> = 45°，若α = 2，則φ<sub>m</sub> = 63.4°. Sansen\[9\]建議α = 3，對應的φ<sub>m</sub> = 71.6°「是一種很好的啟始條件。」

若τ<sub>2</sub>縮短，α會增加。穩定時間*t*<sub>S</sub> 也會減小。若τ<sub>1</sub>變大，α也會增加，穩定時間會略有變動。若用到技巧，τ<sub>1</sub>和 τ<sub>2</sub>都會變化。

若放大器有二個以上的極點，圖5的波德圖仍然可以計算相位裕度，只要將*f*<sub>2</sub>視為「等效的第二極點」位置即可\[10\]。

## 數學定義

本節以抽象概念下的\(\textstyle\mathfrak{S}\)，來提供形式性的數學定義。所有符號及假設列在下方。

  - \(\textstyle t\in T\)是系統的，為方便說明，簡稱為[時間](https://zh.wikipedia.org/wiki/時間 "wikilink")。
  - \(\textstyle\boldsymbol{x}|_t\in M\)是系統在時間\(t\,\)時的，為方便說明，稱為輸出。
  - \(\textstyle\Phi:T\times M\longrightarrow M\)是動態系統的。
  - \(\textstyle\Phi(0,\boldsymbol{x})=\boldsymbol{x}_0\in M\)是動態系統的。
  - \(\textstyle H(t)\,\)是[单位阶跃函数](../Page/单位阶跃函数.md "wikilink")。

### 非線性動態系統

若針對一個一般的動態系統，其階躍響應可定義如下：

\[\boldsymbol{x}|_t = \Phi_{\{H(t)\} \left(t,{\boldsymbol{x}_0} \right)}\,\]

其階躍響應是系統輸入為單位階躍函數時的演化函數。表示式中*H*(*t*)為下標。

### 線性動態系統

對於一個[線性非時變系統](../Page/线性时不变系统理论.md "wikilink")，令\(\textstyle\mathfrak{{S}}\ \equiv \ S\)，其階躍響應可以用[單位階躍函數](https://zh.wikipedia.org/wiki/單位階躍函數 "wikilink")*H* (*t*)和系統[衝激響應](https://zh.wikipedia.org/wiki/衝激響應 "wikilink") *h* (*t*) 的[卷積來表示](https://zh.wikipedia.org/wiki/卷積 "wikilink")：

\[a(t) = {h*H}(t) = {H*h}(t) = \int_{-\infty }^{+\infty} h(\tau) H(t - \tau ) \, d\tau = \int_{-\infty}^t h(\tau) \, d\tau\]

對線性非時變系統而言就是將後面的式子積分。相對的，對於線性非時變系統，階躍響應的微分即為衝激響應：

  -

      -
        \(h(t) = \frac{d}{dt}\, a(t)\)

不過此關係在非線性系統或是[時變系統](../Page/時變系統.md "wikilink")並不成立\[11\]。

對於一個[線性非時變系統](../Page/线性时不变系统理论.md "wikilink")，其階躍響應可以用[單位階躍函數](https://zh.wikipedia.org/wiki/單位階躍函數 "wikilink")*H*(*t*)和系統[衝激響應](https://zh.wikipedia.org/wiki/衝激響應 "wikilink") *h*(*t*) 的[卷積來表示](https://zh.wikipedia.org/wiki/卷積 "wikilink")：

\[a(t) = {h*H}(t) = {H*h}(t) = \int_{-\infty }^{+\infty} h(\tau) H(t - \tau ) \, d\tau = \int_{-\infty}^t h(\tau) \, d\tau\]

在統計學中，**集中趨勢**（central tendency）或**中央趨勢**，在口語上也經常被稱為**平均**，表示一個[機率分佈的中間值](https://zh.wikipedia.org/wiki/機率分佈 "wikilink")。\[1\]最常見的幾種集中趨勢包括[算數平均數](https://zh.wikipedia.org/wiki/算數平均數 "wikilink")、[中位數](../Page/中位數.md "wikilink")及[眾數](https://zh.wikipedia.org/wiki/眾數 "wikilink")。集中趨勢可以由有限的數組（如一群[樣本](https://zh.wikipedia.org/wiki/樣本 "wikilink")）中或理論上的機率分配（如[常態分佈](https://zh.wikipedia.org/wiki/常態分佈 "wikilink")）中求得。有些人使用集中趨勢（或**集中性**）這個詞彙以表示「數量化的資料之中央值的趨勢」。\[2\]\[3\]在這種意義下，我們可以利用資料的[離散程度](https://zh.wikipedia.org/wiki/離散程度 "wikilink")（例如[標準偏差或](https://zh.wikipedia.org/wiki/標準偏差 "wikilink")[四分差等相似的統計量](https://zh.wikipedia.org/wiki/四分差 "wikilink")）判別其集中趨勢的程度。

集中趨勢（central tendency）一詞於1920年代後期出現。\[4\]

## 集中趨勢的統計量

一維資料的集中趨勢可能有以下數種統計方法。在某些情況下，經[轉型](https://zh.wikipedia.org/wiki/轉型 "wikilink")（）後的資料才採用以下的方法。

  - [算數平均數](https://zh.wikipedia.org/wiki/算數平均數 "wikilink") : 觀測值的總和除以觀測值的個數，即\(\tfrac{x_1 + x_2 + x_3 \ldots + x_n}{n}\)。常簡稱為平均數，也往往是背後機率分佈的期望值之不偏估計。
    [中位數](../Page/中位數.md "wikilink") : 將所有觀測值按大小排序後在順序上居中的數值。
    [眾數](../Page/众数_\(数学\).md "wikilink"): 出現最多次的觀測值。
    [幾何平均數](https://zh.wikipedia.org/wiki/幾何平均數 "wikilink") : 觀測值的乘積之觀測值個數方根，即\((x_1 \times x_2 \times x_3 \ldots \times x_n)^{\frac{1}{n}}\)
    [調和平均數](https://zh.wikipedia.org/wiki/調和平均數 "wikilink") : 觀測值個數除以觀測值倒數的總和，即\(\frac{n}{\frac{1}{x_1} + \frac{1}{x_2} + ... + \frac{1}{x_n}}\)
    [加權平均數](../Page/加權平均數.md "wikilink") : 考慮不同群資料貢獻程度不同時的算數平均數
    [截尾平均數](https://zh.wikipedia.org/wiki/截尾平均數 "wikilink")（） : 忽略特定比例或特定數值之外的極端值後所得的平均數。例如，（）正是忽略25%前及75%後的資料後所得的算數平均數。
    （） : 最大值與最小值的算數平均數，即\(\frac{\min (x) + \max (x)}{2}\)。
    （） : 第一[四分位數與第三](https://zh.wikipedia.org/wiki/四分位數 "wikilink")[四分位數的算數平均數](https://zh.wikipedia.org/wiki/四分位數 "wikilink")，即\(\frac{Q_1 + Q_3}{2}\)。
    [三均值](../Page/三均值.md "wikilink")（） : 考慮三個四分位數的加權平均數，即\(\frac{Q_1 + 2Q_2 + Q_3}{4}\)。
    （） : 以最接近的觀測值取代特定比例的極端值後取得的算數平均數。舉例來說，考慮10個觀測值（由小到大排列為\(x_1\)至\(x_{10}\)）的情況下，10%的極端值調整平均數為

      -
        \(\frac{\overbrace{x_2 + x_2} + x_3 + x_4 + x_5 + x_6 + x_7 + x_8 + \overbrace{x_9 + x_9}}{10}\)，

    其中分別以\(x_2\)和\(x_9\)取代了\(x_1\)和\(x_{10}\)。

以上的統計量在多維變數中仍可單獨地被套用在各個維度上進行，但並不能保證在轉軸後仍維持一致的結果。

## 平均數、中位數與眾數的關係

[缩略图](https://zh.wikipedia.org/wiki/File:Median_exp.svg "fig:缩略图")

在左右對稱的機率分佈中，不同的集中趨勢統計量有相同結果，但在偏度遠離0時則可能不一致。在單峰型的機率分佈（）中，平均數（*μ*）、中位數（*ν*）與眾數（*θ*）的關係如下：\[5\]

  -
    \(\frac{| \theta - \mu |}{ \sigma } \le \sqrt{ 3 }\)，

<!-- end list -->

  -
    \(\frac{| \nu - \mu |}{ \sigma } \le \sqrt{ 0.6 }\)，

<!-- end list -->

  -
    \(\frac{| \theta - \nu |}{ \sigma } \le \sqrt{ 3 }\)，

其中*σ*為[標準偏差](https://zh.wikipedia.org/wiki/標準偏差 "wikilink")。至於任一機率分佈，\[6\]\[7\]

  -
    \(\frac{| \nu - \mu |}{ \sigma } \le 1\)。

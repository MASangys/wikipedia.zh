[Set_partition.svg](https://zh.wikipedia.org/wiki/File:Set_partition.svg "fig:Set_partition.svg")表示。\]\] 在[数学](../Page/数学.md "wikilink")中，[集合](https://zh.wikipedia.org/wiki/集合 "wikilink")*X*的**划分**是把*X*分割到覆盖了*X*的全部元素而又不重叠的“**部分**”或“**块**”或“**单元**”中。更加形式的说，这些“单元”對于被划分的集合是既[全无遗漏又](https://zh.wikipedia.org/wiki/周延事件 "wikilink")[互斥的](https://zh.wikipedia.org/wiki/互斥 "wikilink")。

## 定义

集合*X*的划分是*X*的[非空](https://zh.wikipedia.org/wiki/非空集合 "wikilink")[子集](../Page/子集.md "wikilink")的集合，使得每個*X*的元素*x*都只包含在这些子集的其中一个内。

等价的说，*X*的子集的集合*P*是*X*的划分，如果

1.  *P*的元素都不是[空集](../Page/空集.md "wikilink")。（注：某些定义不需要这个要求）
2.  *P*的元素的[并集](../Page/并集.md "wikilink")等于*X*。（我们称*P*的元素**覆盖***X*。）
3.  *P*的任何两个元素的[交集](../Page/交集.md "wikilink")为空。（我们称*P*的元素是[两两不相交](../Page/不交集.md "wikilink")。）

*P*的元素有时叫作划分的**块**或**部分**。\[1\]

## 例子

  - 所有[单元素集合](https://zh.wikipedia.org/wiki/单元素集合 "wikilink"){*x*}都有唯一一个划分，就是{ {*x*} }。
  - 对于任何集合*X*，*P* = {*X*}是*X*的一个划分。
  - [空集](../Page/空集.md "wikilink")有唯一一个划分，就是没有块的划分。
  - 对于集合*U*的任何非空[真子集](https://zh.wikipedia.org/wiki/真子集 "wikilink")*A*，*A*和它的[补集](../Page/补集.md "wikilink")一起是*U*的一个划分。
  - 如果我们不使用前面定义中的公理1，则上述例子可以推广为任何（空和非空）子集与它的补集一起是一个划分。
  - 集合{ 1, 2, 3 }有五个划分。
      - { {1}, {2}, {3} }，有时標示为1/2/3。
      - { {1, 2}, {3} }，有时標示为12/3。
      - { {1, 3}, {2} }，有时標示为13/2。
      - { {1}, {2, 3} }，有时標示为1/23。
      - { {1, 2, 3} }，有时標示为123。
  - 注意
      - 如果我们使用了前面定义中的公理1，则{ {}, {1,3}, {2} }不是一个划分（因为它包含空集）；否则它是{1, 2, 3}的一个划分。
      - { {1, 2}, {2, 3} }不是（任何集合的）一个划分，因为元素2包含在多于一个不同的子集中。
      - { {1}, {2} }不是{1, 2, 3}的一个划分，因为没有块包含3；但它是{1, 2}的一个划分。

## 划分和等价关系

如果给定在集合*X*上的一个[等价关系](../Page/等价关系.md "wikilink")，则所有[等价类](../Page/等价类.md "wikilink")的集合形成*X*的一个划分。反过来说，如果给定在*X*上的一个划分*P*，我们可以在*X*上定义等价关系\~，使得*x* \~ *y*[当且仅当](../Page/当且仅当.md "wikilink")存在*P*的一个成员包含*x*和*y*二者。“等价关系”和“划分”的概念因此本质上是等价的。\[2\]

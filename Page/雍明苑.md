**雍明苑**（）是[香港房屋委員會](../Page/香港房屋委員會.md "wikilink")的[居屋屋苑之一](https://zh.wikipedia.org/wiki/居屋 "wikilink")，位於[新界](../Page/新界.md "wikilink")[西貢區](../Page/西貢區.md "wikilink")[將軍澳](../Page/將軍澳.md "wikilink")至善街5號\[1\]，鄰近[寶盈花園](../Page/寶盈花園.md "wikilink")及[播道書院](../Page/播道書院.md "wikilink")。屋苑將於2020年1月完工\[2\]，由房屋署總建築師（5）設計，並由[保華建業集團](../Page/保華建業集團.md "wikilink")承建\[3\]。雍明苑會是西貢區內第19個居屋屋苑。由於雍明苑兩幢大廈估計會在2020年初落成，因此已撥入居屋2019計劃發售，並將於2019年5月接受申請。\[4\]

## 樓宇

屋苑共設兩座26層高樓宇，提供1,395個單位。單位密度方面，澤明閣1-12樓每層30伙，而13樓以上為每層22伙；浚明閣2-12樓每層38伙（1樓設34伙），較天水圍[屏欣苑](../Page/屏欣苑.md "wikilink")屏彥閣的每層34伙更多；而13樓以上為每層20伙。此屋苑設有可間三房單位，但僅限於浚明閣3、4室\[5\]。所有樓宇已於2018年7月初步命名，詳情如下\[6\]：

| 樓宇名稱（座號） | 樓宇類型                                                | 落成年份  |
| -------- | --------------------------------------------------- | ----- |
| 澤明閣（A座）  | [非標準設計大廈](../Page/非標準設計大廈.md "wikilink")（十字長型）      | 2020年 |
| 浚明閣（B座）  | [非標準設計大廈](../Page/非標準設計大廈.md "wikilink")（其他類型）\[7\] |       |
|          |                                                     |       |

所有單位將撥入居屋2019年中發售，預計2019年5月開始接受申請。

順帶一提，所有樓宇均將採用[東芝公司生產的升降機](https://zh.wikipedia.org/wiki/東芝公司 "wikilink")。澤明及浚明閣分別設有4台及6台（主翼4台、輔翼2台）升降機，1-12樓分別有兩台及四台（含輔翼）升降機到達，13樓以上一律有兩台升降機可達。

## 設施

屋苑設有兒童遊樂場、羽毛球場、園景設施及停車位。另外，屋苑內亦將有一座3層高商場（設6間店舖、1所有6間課室的幼稚園，以及房委會轄下部門辦公室），以及一座4層高服務設施大樓（設社區會堂、長者服務中心、綜合家庭服務中心，以及特殊幼兒中心等社福中心）\[8\]。

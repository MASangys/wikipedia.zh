**雜項符號和箭頭**是一個[Unicode區段](../Page/Unicode區段.md "wikilink")，內容包含了包含箭頭、幾何形狀與其他填充圖案。

## 區段

## 繪文字

本區段包含了七個[繪文字](../Page/繪文字.md "wikilink")：
U+2B05–U+2B07、U+2B1B–U+2B1C、U+2B50 與 U+2B55。\[1\]\[2\]

除此之外，區段中還有十四個，用來為上述七個繪文字指定不同的樣式（U+FE0F VS16）或文字呈現（U+FE0E VS15）。\[3\]

|              |      |      |      |      |      |      |      |
| ------------ | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| U+           | 2B05 | 2B06 | 2B07 | 2B1B | 2B1C | 2B50 | 2B55 |
| 預設顯示         | 文本   | 文本   | 文本   | 繪文字  | 繪文字  | 繪文字  | 繪文字  |
| 基本碼點         | ⬅    | ⬆    | ⬇    | ⬛    | ⬜    | ⭐    | ⭕    |
| 基本+VS15（文本）  | ⬅︎   | ⬆︎   | ⬇︎   | ⬛︎   | ⬜︎   | ⭐︎   | ⭕︎   |
| 基本+VS16（繪文字） | ⬅️   | ⬆️   | ⬇️   | ⬛️   | ⬜️   | ⭐️   | ⭕️   |

《**雪山飛狐**》（）是[香港](../Page/香港.md "wikilink")[電視廣播有限公司](../Page/電視廣播有限公司.md "wikilink")根據[金庸](../Page/金庸.md "wikilink")所著的[武俠小說](../Page/武俠小說.md "wikilink")《[雪山飛狐](https://zh.wikipedia.org/wiki/雪山飛狐 "wikilink")》及《[飛狐外傳](https://zh.wikipedia.org/wiki/飛狐外傳 "wikilink")》合而為一改編而拍攝製作的古裝武俠[劇集](https://zh.wikipedia.org/wiki/劇集 "wikilink")，全劇共40集，監製[王天林](../Page/王天林.md "wikilink")，于1985年10月14日至12月6日期间在无线电视翡翠台首播，此劇為無綫電視18週年台慶劇，此劇于1988年1月下午2:15-3:15, 1990年1月深夜, 2014年1月2日至5月2日星期一至五上午10:00-10:30在翡翠台重播。

## 故事大綱

明末闖王破京，吳三桂引清兵入關，將闖王圍困於九宮山。闖王派出身邊三大衛士分別姓苗、范、田去突圍召援，祗留下第四名姓胡衛士護駕。不料苗、范、田三名衛士領得援軍前來救駕時，發現闖王已失踪，而姓胡衛士卻投靠了吳三桂，三人立志追殺姓胡衛士報仇。其實，姓胡衛士是假意效忠吳三桂，靜待機會助闖王復國。此後，苗、范、田和胡家後人因不知內情而互相仇殺，及至胡家第四代胡一刀 ( [呂良偉](../Page/呂良偉.md "wikilink")飾 ) ；胡一刀因上代誓言保密百年期滿，本欲將真相告知三家以了結仇怨，但三家後人因爭奪明室寶藏，結果葬身冰洞。胡一刀始終無法說出真相，唯有將事實寫下並封於洞內以警惕世人，而胡一刀亦打算與夫人 ( [戚美珍](../Page/戚美珍.md "wikilink")飾 ) 退隱江湖。

及後田家第四代田歸農 ( [曾江](../Page/曾江.md "wikilink")飾 ) 佈局令胡一刀與苗家後人苗人鳳 ( [謝賢](../Page/謝賢.md "wikilink")飾 ) 決戰，並暗中下毒殺害胡一刀夫婦二人，胡夫人臨終前將初生的胡斐交予忠仆平四 ( [龍天生](../Page/龍天生.md "wikilink")飾 ) ，平四帶胡斐逃亡，並誓要查明真相。 胡斐 ( [呂良偉](../Page/呂良偉.md "wikilink")分飾 ) 專心習武，練成胡家刀法。他在追殺苗人鳳途中邂逅神秘女子袁紫衣 ( [周秀蘭](../Page/周秀蘭.md "wikilink")飾 ) ，因求藥醫治苗人鳳而結識了藥王之徒程靈素 ( [景黛音](../Page/景黛音.md "wikilink")飾 ) ，兩人更成莫逆之交。期間，苗人鳳女苗若蘭 ( [曾華倩](../Page/曾華倩.md "wikilink")飾 ) 亦對胡斐產生愛慕之情……最終胡斐得知父死于苗人鳳刀下，兩人遂展開生死決鬥……

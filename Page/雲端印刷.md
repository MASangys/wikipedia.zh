**雲端印刷**（Web-to-Print，縮寫 W2P）是指透過[網際網路發生與印刷有關的的商業行為活動](https://zh.wikipedia.org/wiki/網際網路 "wikilink")，又稱為**網路印刷**。影響雲端印刷發展的一個重要因素是[雲端運算](../Page/雲端運算.md "wikilink")。

## 源起

將雲端計算的概念運用於印刷產業上，使流程簡化、管理便捷、成本降低，印刷管理流程更加便捷，可以在任何時間、以任何工具、把任何格式的[檔案](../Page/檔案.md "wikilink")、在任何系統列印。雲端運算不只對消費者有所幫助，對於印刷企業本身也是有顯著影響，即為「架構即服務」概念：印刷企業不用自建硬體主機和作業系統，透過網路租用[虛擬主機平臺即可提供雲端印刷服務](https://zh.wikipedia.org/wiki/虛擬主機 "wikilink")，能夠省去軟硬體維護及管理的人力和時間，降低印刷企業發展雲端印刷的門檻，並同時確保資料的穩固性及安全性。

### 網路印刷平臺應具備之功能

網路印刷平臺應具備之功能，以展示、媒合、分享與管理四大方向。首先需建置基本服務：包含了建立線上編輯、線上預覽、線上估價、線上下單與線上看樣系統；展示功能方面，包含製造廠商提供的印刷品模組展示、設計者提供的素材展示、媒合成果的展示、合作提案以及案例分享等；在媒合面，主要規劃有合作提案、創意提案、熱門標籤與媒合訊息等功能；分享有案例分享、線上客服以及諮詢服務。

## 特色

  - 印件檔案儲存在網路上，稿件更新與修改再網際網路上進行，可以隨時隨地至網路上下單購買。
  - [電子商務流程的透明化](https://zh.wikipedia.org/wiki/電子商務 "wikilink")，從下訂購買到印刷產品出貨皆能隨時掌握相關訊息。
  - 提供客戶更完善的線上設計與編輯服務，並提供各種參考設計版型，滿足顧客的個性化創意需求。

### 網路印刷與傳統印刷產業比較

| 項目                                                    | 雲端印刷       | 傳統印刷產業        |
| ----------------------------------------------------- | ---------- | ------------- |
| 估價                                                    | 線上透明化估價    | 營業員電話詢問       |
| 訂單進度追蹤                                                | 全天網路及時追蹤   | 僅上班時間以電話詢問    |
| 個人化                                                   | 完全可以個人化    | 無             |
| 編輯自主性                                                 | 高          | 低             |
| [客製化產品](https://zh.wikipedia.org/wiki/客製化 "wikilink") | 單位成本廉價     | 單位成本高         |
| 供需條件                                                  | 長短版、依需印刷皆可 | 長版印刷為主        |
| 資源運用效率                                                | 高          | 低             |
| 營業利潤                                                  | 獲利穩定       | 市場削價競爭激烈，獲利微薄 |
| 營業地點                                                  | 線上服務、及時宅配  | 產業營業部門        |
| 營業成本                                                  | 低          | 高             |
| 營業時間                                                  | 24小時全天服務   | 門市作業時間        |

### 網路印刷技術相較於傳統印刷技術之優勢

| 項目                                 | 市場應用面                                                                                |
| ---------------------------------- | ------------------------------------------------------------------------------------ |
| 隨需印刷(POD)                          | 主要針對頻繁修訂和需要更新的[出版物](../Page/出版物.md "wikilink")，而且特別適用於印刷編入很多圖片之出版物。                  |
| 可變資料印刷(Variable-Date printing,VDP) | 因傳統印刷無法處理的變動性資料，在網路印刷中可輕易實現，且在每一頁上的圖文都可以在每一次印刷中連續變化。適用於印刷個性化資訊。                      |
| 發行和印刷                              | 網路印刷減少了喘統印刷［先印後送］所帶來的增加儲存、運輸成本及延誤交件時間等問題，提高了生產效率和經濟效益。同一份檔案可以同時在不同地點印刷，確保再版印刷與第一版相同。 |

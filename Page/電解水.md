**电解水**通常是指含[鹽](https://zh.wikipedia.org/wiki/鹽 "wikilink")（如[氯化鈉](https://zh.wikipedia.org/wiki/氯化鈉 "wikilink")）的[水](../Page/水.md "wikilink")经过[电解](../Page/电解.md "wikilink")之後所生成的產物。電解過後的水本身是中性，可以加入其他[離子](https://zh.wikipedia.org/wiki/離子 "wikilink")，或者可经过[半透膜分離](https://zh.wikipedia.org/wiki/半透膜 "wikilink")\[1\]而生成两种性質的水。其中一种是碱性離子水，另一种是酸性離子水。以氯化鈉為水中所含[電解質的電解水](https://zh.wikipedia.org/wiki/電解質 "wikilink")，在電解後會含有[氫氧化鈉](https://zh.wikipedia.org/wiki/氫氧化鈉 "wikilink")、[次氯酸](../Page/次氯酸.md "wikilink")與[次氯酸鈉](../Page/次氯酸鈉.md "wikilink")（如果是純水經過電解，則只會產生[氫氧根離子](https://zh.wikipedia.org/wiki/氫氧根離子 "wikilink")、[氫氣](../Page/氫氣.md "wikilink")、[氧氣與](https://zh.wikipedia.org/wiki/氧氣 "wikilink")[氫離子](https://zh.wikipedia.org/wiki/氫離子 "wikilink")）。

在某些條件下，電解後產生的酸性電解水有殺菌用途\[2\]\[3\]。依據電解原理在電極（陽極反應：H<sub>2</sub>O<sub>(l)</sub> → 2H<sup>+</sup><sub>(aq)</sub> + 1/2 O<sub>2(g)</sub> + 2e<sup>-</sup>，陰極反應：2H<sub>2</sub>O<sub>(l)</sub> + 2e<sup>-</sup> → 2OH<sup>-</sup><sub>(aq)</sub> + H<sub>2(g)</sub>，全反應：2H<sub>2</sub>O→2 H<sub>2</sub> + O<sub>2</sub>）生成的氧氣，在較低[pH值](https://zh.wikipedia.org/wiki/pH "wikilink")（例，pH\<2.7）情況時，會與[氯](../Page/氯.md "wikilink")化合生成[次氯酸根或](https://zh.wikipedia.org/wiki/次氯酸根 "wikilink")[亞氯酸根離子水溶液](https://zh.wikipedia.org/wiki/亞氯酸根 "wikilink")。

此外，雖然有些廣告宣稱鹼性電解水具有「中和酸性體質」的用途，但是實際上電解產生的鹼性水到達[胃](../Page/胃.md "wikilink")部時，會被具有強酸性的[胃酸](../Page/胃酸.md "wikilink")變成酸性\[4\]。

## 飲用水設備

製備電解水的機構稱之為[電解槽](https://zh.wikipedia.org/wiki/電解槽 "wikilink")，其內部主要構成部品是[電極板與](https://zh.wikipedia.org/wiki/電極板 "wikilink")[離子膜](https://zh.wikipedia.org/wiki/離子膜 "wikilink")，兩者都是目前許多科技產品應用的技術。一般常見的電解水製造設備，簡稱[電解水機](../Page/電解水機.md "wikilink")或[電解離子水生成器](https://zh.wikipedia.org/wiki/電解離子水生成器 "wikilink")（Ionic Water Generator）\[5\]。 依據公共自來水質溶存主要成份而言，「硫酸鹽」、「碳酸氫鹽」或少量「氯鹽」經實際功率、流量及其它電解條件的配合，可以製備出單獨溶存較高「鈣離子」或「鈉離子」濃度且適合飲用的「鹼性離子水」，此即是訴求飲水的「機能性」的飲水設備。

## 應用發展

在1990年代之前，由於電解槽電解極板的材質等問題，操作使用電解水機電解水仍面臨諸多硬體方面的疑難白金電極板的應用，終於在新近10年間，才造成電解水機製造飲用電解水發展的盛況。

初期，以鹼性水之[鈣離子及](https://zh.wikipedia.org/wiki/鈣離子 "wikilink")[負電位為主要訴求](https://zh.wikipedia.org/wiki/負電位 "wikilink")。由於日本本土自來水質很軟，仍必須添加鈣素（[碳酸鈣](../Page/碳酸鈣.md "wikilink")成份）才能有效促進電解水之製造；但在台灣，除少數地區水質較軟外，多數地區自來水質都是中度硬水或[硬水](../Page/硬水.md "wikilink")，反而造成電解水機操作使用約半年至一年時間而已，終究因為電解槽內，特別是離子膜表面嚴重卡鈣的問題，以致電解效能急遽下降或失效，電解水的療效，也因之變得無法加以驗證。

## 電解鹼性離子水質分析例

### 氧化還原電位

化學書目有提到，在25℃ 時以氫為準之[標準還原電位E](https://zh.wikipedia.org/wiki/標準還原電位 "wikilink")°。近年來歐美均採用[國際純粹及應用化學聯合會](https://zh.wikipedia.org/wiki/國際純粹及應用化學聯合會 "wikilink")（IUPAC）規定的標準還原電位，而標準氧化電位與標準還原電位的數值相同而正負符號相反。

在25℃ 時以氫為準之標準還原電位E°，如下表所示：

| 電 極                                   | 半電池之還原電位                                                | E°(V，伏特) |
| ------------------------------------- | ------------------------------------------------------- | -------- |
| K<sup>+</sup>/ K                      | K<sup>+</sup> + e<sup>-</sup> → K                       | －2.925   |
| Na<sup>+</sup> / Na                   | Na<sup>+</sup> + e<sup>-</sup> → Na                     | －2.714   |
| Mg<sup>2+</sup> / Mg                  | 1/2 Mg<sup>2+</sup> ＋e<sup>-</sup> → 1/2Mg              | －2.37    |
| 參考電極 H<sup>+</sup>；H<sub>2</sub> / Pt | H<sup>+</sup> + e<sup>-</sup> → 1/2 H<sub>2</sub>       | 0.000    |
| Cu<sup>2+</sup> / Cu                  | 1/2 Cu <sup>2+</sup> + e<sup>-</sup> → 1/2 Cu           | ＋0.337   |
| Cl<sup>-</sup>；Cl<sub>2</sub>(g) / Pt | 1/2 Cl<sub>2</sub> + e<sup>-</sup> → 1/2 Cl<sup>-</sup> | ＋1.360   |

在電解過程中，陽離子向直流電極之（－）極靠近析出，飲用電解還原水之簡單試驗結果，如下：

| 電解電流（A） | 1.5A | 2.0A | 2.5A |
| ------- | ---- | ---- | ---- |
| ORP（mV） | －250 | －500 | －750 |

觀察到的現象就是，電解鹼性離子水之所謂負電位，與電解電流值呈現明顯正相關（只是電位值是負數），即電流愈強，負電位值負愈大（最大負值約為－888mV）。這是還原反應陽離子析出的[半反應](../Page/半反應.md "wikilink")，也是電力能量與化學能/熱能的平衡結果。

### pH值

一般常溫常壓情況下，pH=7的中性水溶液，例如溶存200ppm中性鹽或純水，可表示水中解離的[氫離子和](https://zh.wikipedia.org/wiki/氫離子 "wikilink")[氫氧根離子濃度的乘積](https://zh.wikipedia.org/wiki/氫氧根離子 "wikilink")。

在電解過程中，[陽離子向直流電極之負極靠近析出](https://zh.wikipedia.org/wiki/陽離子 "wikilink")，飲用電解還原水之簡單試驗結果，如下：

| 電解電流（A） | 1.5A | 2.0A | 2.5A |
| ------- | ---- | ---- | ---- |
| pH      | 8.5  | 9.5  | 10.5 |

電解電流愈強，鹼性愈高，當然是正相關。透過pH定義的了解，電解鹼性離子水之pH值愈高，表示其水溶液之氫氧根離子濃度較氫離子濃度高；意味著陽離子析出濃度也較高。

### 鹼性資源

在電解過程中，陽離子向直流電極之負極靠近析出，飲用電解還原水之簡單試驗結果，如下：

| 電解電流（A）     | 1.5A | 2.0A   | 2.5A |
| ----------- | ---- | ------ | ---- |
| 陽離子濃度（mg/L） | 10以下 | 15\~25 | 30以上 |

## 酸性離子水

採用氯化鈉（[電解質](https://zh.wikipedia.org/wiki/電解質 "wikilink")）水溶液，電解可以製備pH＜4.0的次氯酸水溶液。電解過程，正極生成的氯氣，在較低pH值（＜4.0）時，溶於水產生[次氯酸](../Page/次氯酸.md "wikilink")。 依據「化學」學理及實際電解的操作試驗，「酸性離子水」的pH＜2.7的情況，「次氯酸根離子」將再度被「正極」析出的「氧氣」所氧化，因此生成更強的「氯的含氧酸」，即「亞氯酸根離子」成份。此時，進行酸性離子水之「ORP」測定時，其「正電位」可達（+1100mV）以上，在日本，這種特殊的電解氧化酸性水稱為「電解強酸性離子水」。

## 傳統電解水製備氫氣與氧氣

添加電解質導通電流，將水分子電解解離。直流電極之負極，析出[氫氣](../Page/氫氣.md "wikilink")；正極則析出[氧氣](https://zh.wikipedia.org/wiki/氧氣 "wikilink")。可以利用排水集氣法收集氫氣與氧氣，其體積比為2：1。上述電解質可以是強酸強鹼化合之離子化合物，如[氯化鈉](https://zh.wikipedia.org/wiki/氯化鈉 "wikilink")，但是為了避免陽極生成[氯氣](https://zh.wikipedia.org/wiki/氯氣 "wikilink")，應以[氫氧化鈉做為電解質為宜](https://zh.wikipedia.org/wiki/氫氧化鈉 "wikilink")。

氫氣不但被認作燃料，還被擬想為電的直接取代品，它可以[天然氣的狀態經地下管道四處輸送](https://zh.wikipedia.org/wiki/天然氣 "wikilink")。不同於電的是它可以儲存（通常在低溫下以液態），氣體可以不斷地供應，當能量的需要量高，尚可預先儲備。氫將以燃燒的方式產生熱能，或用於[燃料電池產生電能](https://zh.wikipedia.org/wiki/燃料電池 "wikilink")。

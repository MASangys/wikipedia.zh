為[日本麻將的特殊點數算法](https://zh.wikipedia.org/wiki/日本麻將 "wikilink")，通常簡稱為青天井。「天井」指天花板，「青天井」在日語中有時可指藍天，因為在這個規則下點數的累計**沒有上限**，故稱。

## 簡介

通常日本麻將點數到達一定程度時，[點數會變成固定數目](https://zh.wikipedia.org/wiki/日本麻將計分方法#满貫以上的情况的基本点 "wikilink")，例如滿貫為2000點、跳满為3000點……等。青天井規則則是去除該限制，無論翻数多少，均以同樣公式來計算點數，換句話說，每多一翻，點數就多一倍\[1\]。
下表為簡單的點數對比：

|       | 平時的基本點       | 青天井規則下的基本點     |
| ----- | ------------ | -------------- |
| 1翻30符 | 240 ≈ 300點   | 240 ≈ 300點     |
| 2翻30符 | 480 ≈ 500點   | 480 ≈ 500點     |
| 3翻30符 | 960 ≈ 1000點  | 960 ≈ 1000點    |
| 4翻30符 | 1920 ≈ 2000點 | 1920 ≈ 2000點   |
| 5翻30符 | 2000點（滿貫）    | 3840 ≈ 3900點   |
| 6翻30符 | 3000點（跳滿）    | 7680 ≈ 7700點   |
| 7翻30符 | 3000點（跳滿）    | 15360 ≈ 15400點 |

## 得點計算式

和普通規則一樣，以基本點來做計算。

  -
    基本點 = 符×2<sup>（翻数+2）</sup>

而支付點數如下，支付點數以100為單位[無條件進位](https://zh.wikipedia.org/wiki/無條件進位 "wikilink")。

|         |                        |
| ------- | ---------------------- |
| 閒家的自摸和了 | 莊家支付基本點×2，剩下兩名閒家支付基本點。 |
| 閒家的榮和   | 點炮者支付基本點×4。            |
| 莊家的自摸和了 | 其餘三人支付基本點×2。           |
| 莊家的榮和   | 點炮者支付基本點×6。            |

## [役滿的得點計算](https://zh.wikipedia.org/wiki/役滿 "wikilink")

青天井規則下，役滿的得點有兩種可能：

  - 役滿得點固定，莊家1500萬點；閒家1000萬點
  - 役滿視為13翻，且可和其他役複合

而[國士無雙的符固定視為](https://zh.wikipedia.org/wiki/國士無雙 "wikilink")30符（榮和：副底20符+門前加符10符=30符；自摸：副底20符+自摸2=22 ≈ 30符）

    [寶牌表示牌為](https://zh.wikipedia.org/wiki/懸賞牌_\(日本麻將\)#懸賞表示牌與懸賞牌 "wikilink")

則役有[大三元](https://zh.wikipedia.org/wiki/大三元 "wikilink")、[混一色](../Page/混一色.md "wikilink")、[對對和](https://zh.wikipedia.org/wiki/對對和 "wikilink")、[寶牌](../Page/懸賞牌_\(日本麻將\).md "wikilink")8，合計25翻；而符為副底20符 + 單騎五筒2符 + 中的暗刻8符 + 發的暗槓32符 + 白的明刻4符 + 北的明刻4符，合計70符。 則得點為70符×2<sup>（25翻+2）</sup> = 9395240960點，進位至9395241000點。

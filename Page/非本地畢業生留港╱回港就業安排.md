《**非本地畢業生留港╱回港就業安排**》（），為[香港入境事務處自](https://zh.wikipedia.org/wiki/香港入境事務處 "wikilink")2008年起實施的一項政策，容許在港讀書的非[本地畢業生留港工作](../Page/本地人_\(香港\).md "wikilink")，藉此補足[香港](../Page/香港.md "wikilink")的[勞動力](https://zh.wikipedia.org/wiki/勞動力 "wikilink")\[1\]。2018年經「非本地畢業生留港／回港就業安排」獲派來港的人士當中，約91%為內地居民\[2\]。

## 統計數字

根據「非本地畢業生留港／回港就業安排」的申請和獲批數字\[3\]：

|       |        | 申請數目   | 獲批數目  |
| ----- | ------ | ------ | ----- |
| 2018年 | 內地居民   | 9,297  | 9,206 |
| 非內地居民 | 957    | 944    |       |
| 總數    | 10,254 | 10,150 |       |

根據「非本地畢業生留港／回港就業安排」來港人士取得居留權數字：

| 年份    | 獲批數目  |
| ----- | ----- |
| 2016年 | 1,979 |
| 2017年 | 2,172 |
| 2018年 | 2,455 |

是一種[地震度量](../Page/地震度量.md "wikilink")，用於[地震学中描述地震的大小](https://zh.wikipedia.org/wiki/地震学 "wikilink")。表面波規模是根據測量主要傳播於地球表層的[雷利波](https://zh.wikipedia.org/wiki/雷利波 "wikilink")（Rayleigh
wave）而制定，曾经是[中华人民共和国分類地震的](https://zh.wikipedia.org/wiki/中华人民共和国 "wikilink")[國家標準](../Page/中华人民共和国国家标准.md "wikilink")（GB
17740-1999）。\[1\]

表面波規模最初發展於1950年代，由發展[芮氏規模的研究者發展](https://zh.wikipedia.org/wiki/芮氏規模 "wikilink")，目的在於提高大地震的解析能力。\[2\]在那時候記錄的地震規模，通常是用芮氏規模。

## 定義

計算表面波規模的方程式是：\[3\]\[4\]

\(M = \log_{10}\left(\frac{A}{T}\right)_{\text{max}} + \sigma(\Delta)\)

A表示表面波中最大質點位移（兩個水平位移的向量和），單位為[微米](../Page/微米.md "wikilink")，T表示對應的週期，單位為[秒](../Page/秒.md "wikilink")，Δ表示震中距，單位為[度](https://zh.wikipedia.org/wiki/度_\(角\) "wikilink")，以及

\(\sigma(\Delta) = 1.66\cdot\log_{10}(\Delta) + 3.5\)

根據GB 17740-1999，兩個水平位移必須是在相同時間或八分之一個週期內測量，如果兩個位移有不同的週期，必須使用加權總合

\(T = \frac{T_{N}A_{N} + T_{E}A_{E}}{A_{N} + A_{E}}\)

A<sub>N</sub>表示南北方向的位移，單位為微米，A<sub>E</sub>表示東西方向的位移，單位為微米，T<sub>N</sub>表示對應A<sub>N</sub>的週期，單位為秒，T<sub>E</sub>表示對應A<sub>E</sub>的週期，單位為秒。

## 其他研究

Vladimír Tobyáš與Reinhard Mittag提出了表面波規模與芮氏規模間的關係：\[5\]

  -
    \(M_s = -3.2 + 1.45 M_{L}\)

其他方程式則有陈俊杰提出的三個修訂的公式：\[6\]

  -
    \(M_s = \log_{10}\left(\frac{A_{max}}{T}\right) + 1.54\cdot \log_{10}(\Delta) + 3.53\)

  -
    \(M_s = \log_{10}\left(\frac{A_{max}}{T}\right) + 1.73\cdot \log_{10}(\Delta) + 3.27\)

以及

  -
    \(M_s = \log_{10}\left(\frac{A_{max}}{T}\right) - 6.2\cdot \log_{10}(\Delta) + 20.6\)

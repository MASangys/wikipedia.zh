**順安邨**（）[香港](../Page/香港.md "wikilink")的一座[公共屋邨](https://zh.wikipedia.org/wiki/公共屋邨 "wikilink")，位於[九龍](../Page/九龍.md "wikilink")[觀塘區](../Page/觀塘區.md "wikilink")[四順](../Page/四順.md "wikilink")，於1978年入伙，由[香港房屋委員會](../Page/香港房屋委員會.md "wikilink")管理，香港的公共屋邨爱好者把它称为「九龙版[大兴邨](https://zh.wikipedia.org/wiki/大兴邨 "wikilink")」（因为楼宇类型和大兴邨相同，但规模较大兴邨小）。

| 安逸樓  |                                                                |                 |    |
| 安群樓  | [第七型徙置大廈](https://zh.wikipedia.org/wiki/舊長型 "wikilink")        | 1980            |    |

順帶一提，本邨安群樓是香港最後一座落成的第七型徙置大廈，亦是香港最後一幢按徙置事務處標準設計的公屋樓宇；此外，此邨是全港最後採用十字型設計的屋邨。

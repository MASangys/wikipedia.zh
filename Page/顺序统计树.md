在[计算机科学](../Page/计算机科学.md "wikilink"),
**顺序统计树**是[二叉搜索树的变种](../Page/二叉搜索树.md "wikilink")。除了插入、查询和删除，这种数据结构还支持以下两种操作：

  - [Select(*i*)](../Page/选择算法.md "wikilink") — 在树中查询第*i*小的元素
  - Rank(*x*) – 查找元素*x*的排名

这两种操作的平均时间复杂度是\(O(\log n)\)。当所用数据结构是[平衡二叉树时](../Page/平衡二叉树.md "wikilink")，这是最坏复杂度。

## 算法实现

对于树中的每个节点，需要额外维护以这个节点为根的子树大小（该节点下点的个数）。

``` c
size[x] = size[left[x]] + size[right[x]] + 1;
```

根据定义，树为空时，其大小为0`size[nil] = 0`。Select操作实现如下：

``` c++
int Select(int t, int i) {
    if (i == size[left[t]] + 1) return key[t];
    if (i < size[left[t]]) return Select(left[t], i);
    else return Select(right[t], i - size[left[t]] - 1);
}
```

Rank操作实现如下：

``` c++
void Rank(int root, int x) {
    int rank = size[left[x]] + 1;
    for (y = x; ; y = parent[y]) {
        if (key[y] < key[x])
            rank += size[left[y]] + 1;
        if (y == root) break;
    }
}
```

通过改进顺序统计树，能够实现其他数据结构（例如, 维护节点的高度能实现[AVL树](../Page/AVL树.md "wikilink"),
维护节点颜色能实现[红黑树](../Page/红黑树.md "wikilink")）。
直接使用节点大小的信息，也能实现[加权平衡树](../Page/加权平衡树.md "wikilink")。\[1\]

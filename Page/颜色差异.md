**颜色差异**（****），亦称**距离**，是[色彩学上的一个关注点](https://zh.wikipedia.org/wiki/色彩学 "wikilink")。它量化了一个概念。在未量化之前，人们只能用形容词来大概描述这个概念，这使得对颜色要求严格的工作者们很不方便。颜色差异可以通过色彩空间内的[欧氏距离简单计算得出](https://zh.wikipedia.org/wiki/欧氏距离 "wikilink")，也可以使用[CIE较为复杂](https://zh.wikipedia.org/wiki/CIE "wikilink")、均匀的人类知觉公式计算。

## 欧氏距离

很多日常使用的“颜色差异”，是直接通过在一个“设备无关”的[色彩空间里](https://zh.wikipedia.org/wiki/色彩空间 "wikilink")，进行[欧氏距离的计算得到的](https://zh.wikipedia.org/wiki/欧氏距离 "wikilink")。给定一个RGB（红绿蓝）的色彩空间，最简单的差异计算方式就是在这个三维空间里求两个点间的距离：

\[\mathrm{juli} = \sqrt{ (R_2-R_1)^2+(G_2-G_1)^2 + (B_2-B_1)^2 }\]

如果还要简单一点，经常可以去掉平方根：

\[\mathrm{juli}^2 = { (R_2-R_1)^2+(G_2-G_1)^2 + (B_2-B_1)^2 }\]

这样求得的距离可以用来比较两个颜色中哪一个更接近某个给定的颜色。如果把这些平方距离加在一起，会得到一组距离间的[方差](../Page/方差.md "wikilink")。

有不少人尝试将RGB三值加上权重，希望可以让得到的结果更加符合人类感官。一种做法是使用2、4、3：

\[\sqrt{ 2 \times \Delta R^2 + 4 \times \Delta G^2 + 3 \times \Delta B^2 }\], 对于0\~255的数值，有一种近似是：\[1\]

\[\bar r = {{C_{1,R}  + C_{2,R} } \over 2}\]

\[\Delta R = C_{1,R}  - C_{2,R}\]

\[\Delta G = C_{1,G}  - C_{2,G}\]

\[\Delta B = C_{1,B}  - C_{2,B}\]

\[\Delta C = \sqrt {\left( {2 + {{\bar r} \over {256}}} \right) \times \Delta R^2  + 4 \times \Delta G^2  + \left( {2 + {{255 - \bar r} \over {256}}} \right) \times \Delta B^2 }\]

## Delta E

[国际照明委员会](../Page/国际照明委员会.md "wikilink")（CIE）称他们的度量标准为“”（也作“”，另有“”、“”、“Delta E”）。其中，[delta](https://zh.wikipedia.org/wiki/delta_\(letter\) "wikilink") 是一个常用来表示“差异”的[希腊字母](../Page/希腊字母.md "wikilink")，**E**表示，也就是德文中的“感觉”。富有影响的[赫尔曼·冯·亥姆霍兹](../Page/赫尔曼·冯·亥姆霍兹.md "wikilink")和[埃瓦尔德·赫林曾使用过它](https://zh.wikipedia.org/wiki/埃瓦尔德·赫林 "wikilink")。\[2\]\[3\]

两个颜色之间的[JND值](https://zh.wikipedia.org/wiki/JND "wikilink")，也就是有多大的差距才能刚好被察觉，目前还没有定论。尽管并没有实践经验支持，'1.0'这个值经常被提到并用作[JND](https://zh.wikipedia.org/wiki/JND "wikilink")。可是，1994年，Mahy et al研究并评估出了一个2.3的JND值。然而，支撑该理论的[CIELAB色彩空间中](https://zh.wikipedia.org/wiki/CIELAB色彩空间 "wikilink")，知觉的非均匀特性（换言之，有些颜色在改变时，人眼对其十分敏感，而有的就分辨得不太清楚）导致这种理论不太站得住脚。这使得[CIE用了接下来的几年时间修缮他们的定义](https://zh.wikipedia.org/wiki/CIE "wikilink")，最终产生了更完善的（据[CIE](https://zh.wikipedia.org/wiki/CIE "wikilink")）1994年和2000年的公式。\[4\] 这些非均匀特性很重要，因为[人眼对某些特定的颜色更敏感](https://zh.wikipedia.org/wiki/颜色视觉#颜色感知的生理学 "wikilink")。一个好用的度量方法，应该把这些因素都考虑在内，才能使“[刚刚好能被察觉到的差异](https://zh.wikipedia.org/wiki/JND "wikilink")”这种提法有意义。不然的话，可能出现这样的情况：对于两组分别含有一对不同的颜色、同样适用某个Δ*E*值的颜色组，其中一组颜色的差异能被人眼觉察出来，另一组却不能。\[5\] L、a和b三个值（见下方公式）一般是介于(-1,1)之间的双精度数值。Lab图像在保存时，L被存储为无符号（即“不分正负”）8位元整数，a、b则被存储为有符号的8位元整数，以保持文件大小。

### CIE76

1976年的这个色彩差异公式是首个用较为均匀的Lab空间计算色彩差异值的公式。Lab比RGB等空间在感官上均匀一点，所以得到的结果会更好。不过后来人们发现[Lab色彩空间](../Page/Lab色彩空间.md "wikilink")，尤其是在饱和度较高的区域里，并没有设计时预想的那么“感官上均匀”，所以几次更新了公式。CIE76公式会高估较为饱和的颜色之间的差异。

应用\(({L^*_1},{a^*_1},{b^*_1})\)和\(({L^*_2},{a^*_2},{b^*_2})\)两个[L\*a\*b\*色彩空间的颜色](https://zh.wikipedia.org/wiki/L*a*b* "wikilink")：

\[\Delta E_{ab}^* = \sqrt{ (L^*_2-L^*_1)^2+(a^*_2-a^*_1)^2 + (b^*_2-b^*_1)^2 }\]

本公式对应的[最小可觉差](https://zh.wikipedia.org/wiki/最小可觉差 "wikilink")：\(\Delta E_{ab}^* \approx 2.3\) \[6\]

### CIE94

在汽车喷漆行业里，人们测量了一堆人眼容差情况的数据。随着这些“应用特定”的数据的发布，人们将1976年的定义进行了进一步的发展，以更好地应对感知非均匀特性。该公式仍然采用的是Lab色彩空间。\[7\]

1994版Δ*E*是在[L\*C\*h\*色彩空间下定义的](https://zh.wikipedia.org/wiki/L*C*h*色彩空间 "wikilink")。它的亮度、色品（或称“浓度”）、色调是从[L\*a\*b\*坐标系里计算来的](https://zh.wikipedia.org/wiki/CIELAB "wikilink")。对于一个基准色\((L^*_1,a^*_1,b^*_1)\)和另一个颜色\((L^*_2,a^*_2,b^*_2)\)，它们的差异是：\[8\]\[9\]\[10\]

\[\Delta E_{94}^* = \sqrt{ \left(\frac{\Delta L^*}{k_L S_L}\right)^2 + \left(\frac{\Delta C^*_{ab}}{k_C S_C}\right)^2 + \left(\frac{\Delta H^*_{ab}}{k_H S_H}\right)^2 }\]

其中：

\[\Delta L^* = L^*_1 - L^*_2\]

\[C^*_1 = \sqrt{ {a^*_1}^2 + {b^*_1}^2 }\]

\[C^*_2 = \sqrt{ {a^*_2}^2 + {b^*_2}^2 }\]

\[\Delta C^*_{ab} = C^*_1 - C^*_2\]

\[\Delta H^*_{ab} = \sqrt{ {\Delta E^*_{ab}}^2 - {\Delta L^*}^2 - {\Delta C^*_{ab}}^2 } = \sqrt{ {\Delta a^*}^2 + {\Delta b^*}^2 - {\Delta C^*_{ab}}^2 }\]

\[\Delta a^* = a^*_1 - a^*_2\]

\[\Delta b^* = b^*_1 - b^*_2\]

\[S_L = 1\]

\[S_C = 1+K_1 C^*_1\]

\[S_H = 1+K_2 C^*_1\]

另外，其中的*k<sub>C</sub>*和*k<sub>H</sub>*一般都取1，系数（权值）*k<sub>L</sub>*、*K*<sub>1</sub>和*K*<sub>2</sub>则依应用场合不同而取不同的值。如：

|         | 图形艺术  | 纹理、纺织品 |
| ------- | ----- | ------ |
| \(k_L\) | 1     | 2      |
| \(K_1\) | 0.045 | 0.048  |
| \(K_2\) | 0.015 | 0.014  |

从几何学上讲，\(\Delta H^*_{ab}\)这个数量即是这两种颜色的相同色度圈的弦长的算术平均值。 \[11\]

### CIEDE2000

鉴于1994年的公式并没有充分解决[感知非均匀特性的问题](https://zh.wikipedia.org/wiki/感知非均匀特性 "wikilink")，CIE再次修缮了定义，并加入了5个修订系数：\[12\]\[13\]

  - 色调旋转项(R<sub>T</sub>)，用来应对常出问题的蓝色区域（色相角度275°左右）；\[14\]
  - 中性色调补偿（对应L\*C\*h差异）
  - 亮度补偿（S<sub>L</sub>）
  - 色度补偿（S<sub>C</sub>）
  - 色调补偿（S<sub>H</sub>）

\[\Delta E_{00}^* = \sqrt{ \left(\frac{\Delta L'}{k_L S_L}\right)^2 + \left(\frac{\Delta C'}{k_C S_C}\right)^2 + \left(\frac{\Delta H'}{k_H S_H}\right)^2 + R_T \frac{\Delta C'}{k_C S_C}\frac{\Delta H'}{k_H S_H} }\]

  -

      -
        <small>**注：**下面的公式请使用角度而非弧度，尤其在*R<sub>T</sub>*上需要注意。</small>
        <small>*k<sub>L</sub>*、*k<sub>C</sub>*、*k<sub>H</sub>*一般取1。</small>

\[\Delta L^\prime = L^*_2 - L^*_1\]

\[\bar{L} = \frac{L^*_1 + L^*_2}{2} \quad \bar{C} = \frac{C^*_1 + C^*_2}{2}\]

\[a_1^\prime = a_1^* + \frac{a_1^*}{2} \left( 1 - \sqrt{\frac{\bar{C}^7}{\bar{C}^7 + 25^7}} \right) \quad
  a_2^\prime = a_2^* + \frac{a_2^*}{2} \left( 1 - \sqrt{\frac{\bar{C}^7}{\bar{C}^7 + 25^7}} \right)\]

\[\bar{C}^\prime = \frac{C_1^\prime + C_2^\prime}{2} \mbox{ and }
  \Delta{C'}=C'_2-C'_1 \quad
  \mbox{where }
    C_1^\prime = \sqrt{a_1^{'^2} + b_1^{*^2}} \quad
    C_2^\prime = \sqrt{a_2^{'^2} + b_2^{*^2}} \quad\]

\[h_1^\prime=\text{atan2} (b_1^*, a_1^\prime) \mod 360^\circ, \quad
  h_2^\prime=\text{atan2} (b_2^*, a_2^\prime) \mod 360^\circ\]

  -

      -
        <small>**Note:** The inverse tangent (tan<sup>−1</sup>) can be computed using a common library routine `atan2(b, a′)` which usually has a range from −π to π radians; color specifications are given in 0 to 360 degrees, so some adjustment is needed. The inverse tangent is indeterminate if both *a′* and *b* are zero (which also means that the corresponding *C′* is zero); in that case, set the hue angle to zero. See .</small>

\[\Delta h' = \begin{cases}
    h_2^\prime - h_1^\prime & \left| h_1^\prime - h_2^\prime \right| \leq 180^\circ \\
    h_2^\prime - h_1^\prime + 360^\circ & \left| h_1^\prime - h_2^\prime \right| > 180^\circ, h_2^\prime \leq h_1^\prime \\
    h_2^\prime - h_1^\prime - 360^\circ & \left| h_1^\prime - h_2^\prime \right| > 180^\circ, h_2^\prime > h_1^\prime
  \end{cases}\]

  -

      -
        <small>**Note:** When either *C′*<sub>1</sub> or *C′*<sub>2</sub> is zero, then Δh′ is irrelevant and may be set to zero. See .</small>

\[\Delta H^\prime = 2 \sqrt{C_1^\prime C_2^\prime} \sin (\Delta h^\prime/2), \quad \bar{H}^\prime=\begin{cases}
    (h_1^\prime + h_2^\prime + 360^\circ)/2 & \left| h_1^\prime - h_2^\prime \right| > 180^\circ \\
    (h_1^\prime + h_2^\prime)/2 & \left| h_1^\prime - h_2^\prime \right| \leq 180^\circ
  \end{cases}\]

  -

      -
        <small>**Note:** When either *C′*<sub>1</sub> or *C′*<sub>2</sub> is zero, then ′ is *h′*<sub>1</sub>+*h′*<sub>2</sub> (no divide by 2; essentially, if one angle is indeterminate, then use the other angle as the average; relies on indeterminate angle being set to zero). See  stating most implementations on the internet at the time had "an error in the computation of average hue".</small>

\[T = 1 - 0.17 \cos ( \bar{H}^\prime - 30^\circ )
        + 0.24 \cos (2\bar{H}^\prime)
        + 0.32 \cos (3\bar{H}^\prime + 6^\circ )
        - 0.20 \cos (4\bar{H}^\prime - 63^\circ)\]

\[S_L = 1 + \frac{0.015 \left( \bar{L} - 50 \right)^2}{\sqrt{20 + {\left(\bar{L} - 50 \right)}^2} } \quad
  S_C = 1+0.045 \bar{C}^\prime \quad
  S_H = 1+0.015 \bar{C}^\prime T\]

\[R_T = -2 \sqrt{\frac{\bar{C}'^7}{\bar{C}'^7+25^7}} \sin \left[ 60^\circ \cdot \exp \left( -\left[ \frac{\bar{H}'-275^\circ}{25^\circ} \right]^2 \right) \right]\]

## 应用

### 差异分类

在讨论人类感知时，一般使用计算色彩差异。如前所述，一般认为在系统中，1.0是刚好注意得到的量。取决于产品的受众，有时候会改用2.5作为可以接受的限制。\[15\]例如一般人会说“叶子和草地是绿的”，但仔细看就会有从黄绿到蓝绿的差异。

这类区别在很多产品的[品质控制中非常重要](https://zh.wikipedia.org/wiki/品质控制 "wikilink")：在生产和运输中经常会有一些随机元素，造成实际上显示的颜色和之前校准出的结果有一些区别。在最理想的情况下，要做到“一般的观察者”不能察觉这类变化。[胶印](../Page/胶印.md "wikilink")过程中经常可以造成2到4个单位的；家用打印机造成的色差可能更大。\[16\]\[17\]

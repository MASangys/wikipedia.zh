《**飛躍霓裳**》（）是[香港](../Page/香港.md "wikilink")[電視廣播有限公司](../Page/電視廣播有限公司.md "wikilink")的時裝電視劇，全劇共20集，監製[梅小青](../Page/梅小青.md "wikilink")。

## 劇情簡介

生於基層家庭的林欣 ( [羅慧娟](../Page/羅慧娟.md "wikilink")飾 )，一向讀書名列前茅，因應考高等程度會考時意外失手而未能升讀大學，於是投身社會，在時裝製衣公司工作。姜舜苓 ( [戚美珍](../Page/戚美珍.md "wikilink")飾 ) 是欣的上司，工作能幹，對初出茅廬的欣照顧有加，讓欣在公司學習時裝設計，欣視之為恩師。

欣有一個要好的男朋友方浩明 ( [邵仲衡](../Page/邵仲衡.md "wikilink")飾 )，自從欣投身社會之後，與正在就讀大學的浩明產生了隔膜，浩明與欣考進大學的好友阮潔莊 ( [梁珮玲飾](https://zh.wikipedia.org/wiki/梁珮玲 "wikilink") ) 交往甚密，後來二人更奉子成婚，使到欣非常傷心。欣鄰居關學文 ( [黎明](../Page/黎明.md "wikilink")飾 ) 前來開解欣。原來學文對欣早有愛意，但由於性格內向，一直不敢向欣表白，最後更遠走加拿大。

欣在工作上認識了上司舜苓的男友攝影師夏祖蔭 ( [任達華](../Page/任達華.md "wikilink")飾 )，三人在工作上並肩作戰，後來舜苓在工作上被打壓，帶同欣與祖蔭另起爐灶。舜苓自立門戶後被舊公司施手段打壓，陷入財政危機。舜苓不惜以肉體換取訂單，祖蔭發現後鄙視舜苓，與她分手，同時追求欣，不久欣與祖蔭墮入愛河，卻被舜苓誤會欣橫刀奪愛，加上舜苓與欣在工作上出現分歧，在工作上處處打擊欣。欣最後接受舊公司邀請坐上舜苓以前的崗位，被舜苓視為背叛。

舜苓事業愛情皆失意，更認定欣害死她患有精神病的母親，於是向欣報復，更牽連欣的兄妹。祖蔭與欣交往後兩人性格不合，適值欣家中正處多事之秋，祖蔭却獲得一難得機會到非洲拍攝特輯，並要求欣同往，欣認為祖蔭不能體諒，決定與他分手。最後舜苓被欣兄長林偉 ( [詹秉熙](../Page/詹秉熙.md "wikilink")飾 ) 報復以致毀容，而偉亦因傷人入獄。此事後，舜苓與欣決定不再為之前的恩怨糾纏，作出和解。

因公司改組，欣負責的設計部被合併，欣雖被升為設計顧問，但覺得公司不再重視時裝設計，決定辭職。事業和愛情皆失意的欣，重遇由加拿大回港的學文。這時候，欣發現一直以來默默支持自己的學文，才是自己的理想對象\[1\]。

## 劇集歌曲

《你愛的一個無名氏》是此劇的主題曲，由[顧嘉輝作曲](https://zh.wikipedia.org/wiki/顧嘉輝 "wikilink")、[向雪懷](../Page/向雪懷.md "wikilink")填詞、[溫兆倫](../Page/溫兆倫.md "wikilink")主唱，歌曲收錄於溫兆倫的個人音樂專輯《[沒有你之後](https://zh.wikipedia.org/wiki/沒有你之後 "wikilink")》；《既是朋友》是此劇的插曲，由顧嘉輝作曲、向雪懷填詞、溫兆倫主唱，歌曲收錄於溫兆倫的個人音樂專輯《[戀愛故事](https://zh.wikipedia.org/wiki/戀愛故事 "wikilink")》。

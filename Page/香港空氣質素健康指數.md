**空氣質-{}-素健康指數**（）是[香港](../Page/香港.md "wikilink")[空氣污染情況的一項指標](../Page/空氣污染.md "wikilink")，由香港[環境保護署於](https://zh.wikipedia.org/wiki/環境保護署 "wikilink")2013年12月30日起推出以香港市民[健康為本的空氣質](../Page/健康.md "wikilink")-{}-素健康指數，取代由1995年沿用、使用了18年的[空氣污染指數](../Page/香港空氣污染指數.md "wikilink")\[1\]\[2\]。此指數計算了[臭氧](../Page/臭氧.md "wikilink")、[二氧化硫](../Page/二氧化硫.md "wikilink")、[二氧化氮和](../Page/二氧化氮.md "wikilink")[粒子引起的健康風險](https://zh.wikipedia.org/wiki/粒子 "wikilink")，每小時更新\[3\]。另外當環保署預測部分監測站的指數將會達到10+之「嚴重」水平時，會透過香港政府新聞網發新聞稿，通知市民有關事項\[4\]。

## 指數計算方法

根據監測站所錄得的[二氧化碳](../Page/二氧化碳.md "wikilink")、[二氧化氮](../Page/二氧化氮.md "wikilink")、[懸浮粒子](../Page/懸浮粒子.md "wikilink")（包括可吸入懸浮粒子（**PM<sub>10</sub>**）及微細懸浮粒子（**PM<sub>2.5</sub>**））及[臭氧四種](../Page/臭氧.md "wikilink")[污染物濃度](https://zh.wikipedia.org/wiki/污染物 "wikilink")，再參考[醫院管理局門診數據](../Page/醫院管理局.md "wikilink")，推算小童及長者因空氣污染的求診風險和在三小時內的平均濃度，然後估算對市民造成的短期健康風險，發出相應的指數和風險級別，並提供健康忠告。空氣質-{}-素健康指數劃分為1至10和10+級，並分為五個健康風險級別，低、中、高、甚高和嚴重。\[5\]

## 分級

香港空氣質-{}-素健康指數分為以下11級\[6\]：

| 健康風險 | 空氣質-{}-素健康指數級別 | 增加的健康風險 (%) | 健康忠告          |
| ---- | -------------- | ----------- | ------------- |
| 低    |                | 1-3         | 0.88-5.64     |
| 中    |                | 4-6         | \>5.64-11.29  |
| 高    |                | 7           | \>11.29-12.91 |
| 甚高   |                | 8-10        | \>12.91-19.37 |
| 嚴重   |                | 10+         | \>19.37       |
|      |                |             |               |

## 氣象因素的影響

香港的空氣污染除受本地污染源影響，內地的發展和[沙漠化往往亦加劇了香港的污染](../Page/沙漠化.md "wikilink")。香港[煙霞嚴重日子多在弱北風剛到達香港或當影響香港的東北季候風緩和時發生](https://zh.wikipedia.org/wiki/煙霞 "wikilink")，因氣團來至內陸，內陸污染物被帶到香港，而風勢微弱亦有利污染物積聚。另外，當有[熱帶氣旋在香港以東的台灣附近](../Page/熱帶氣旋.md "wikilink")，由於北半球熱帶氣旋逆時針旋轉的特性，在其外圍環流影響下，香港會轉吹西北風及下沉氣流令大氣變得較為穩定，特別在風勢微弱的日子，有利污染物積聚，香港的空氣污染亦會特別嚴重\[8\]。而在夏季（5月至8月），香港多受海洋氣流（一般為南至東南風）或西南季候風影響，帶來較潔淨的海洋空氣，香港的空氣污染水平顯著較低\[9\]。

## 重大事件

  - 環保署使用新的「空氣質-{}-素健康指數」第二日出現誤報，署方一度公布大埔錄得最差的「10+」指數，後來刪除了，指正進行系統調整，有環保團體指監測站儀器老化，影響量度準確度。環保署於中午十二時半，透過政府新聞處公布的空氣質-{}-素健康指數，表示在中午十二時，大埔的指數是最高的「10+」，健康風險屬「嚴重」級別，而其他一般監測站指數只是5或6。署方指因系統調整中，大埔的指數未反映真實情況，亦未知何時會恢復正常讀數。直到同日傍晚，環保署恢復公布大埔區的指數。
  - 受塔門監測站所處的建築物重鋪屋頂及裝修工程影響，該站的運作於2015年11月30日至2016年2月29日期間暫停運作。另因2016年3月17日間歇性暫停供電，該站當日亦暫停運作。
  - 2016年7月8日及9日受[熱帶氣旋尼伯特外圍下沉氣流影響](../Page/颱風尼伯特_\(2016年\).md "wikilink")，空氣質素欠佳，當中在2016年7月9日下午4時環保署16個監測站均錄得指數為10+，健康風險屬「嚴重」級別，是空氣質素健康指數設立以來首次全部監測站同一時間達最高水平。比起2015年8月8日受[熱帶氣旋蘇迪羅外圍下沉氣流影響期間](../Page/颱風蘇迪羅_\(2015年\).md "wikilink")，空氣更為混濁，當時雖然全部監測站均錄得10+，但並非全部同一時間錄得。

**[香港迪士尼樂園](../Page/香港迪士尼樂園.md "wikilink")**（[英文](https://zh.wikipedia.org/wiki/英文 "wikilink")：**Hong Kong Disneyland**）位於[香港](../Page/香港.md "wikilink")[新界](../Page/新界.md "wikilink")[大嶼山](../Page/大嶼山.md "wikilink")的[香港迪士尼樂園度假區](../Page/香港迪士尼樂園度假區.md "wikilink")。

以下是香港迪士尼樂園內的商店。

## 現有商店

### 美國小鎮大街

#### 百貨店

**百貨店**（**Emporium**）百貨店位於熙來攘往的美國小鎮大街上，是樂園內最大的商品店。你可以在這裡選購獨家限定紀念品、包裝精美的糖果餅乾，或由專人為你繡上名字的米奇老鼠耳朵帽子，快來將美好的迪士尼體驗帶回家吧！ {{ - }}

#### 馬迪小店

**馬迪小店**（**Marty's General Store**）走進位於小鎮大街酒店內的馬迪小店，你不但可選購一系列迪士尼精品，更可於這裡挑選樂園內拍攝的「迪士尼處處拍」相片，保留珍貴回憶。 {{ - }}

#### 小鎮沖印店

**小鎮沖印店**（**Town Square Photo**）小鎮沖印店的外形猶如百年酒店，你在這裡可以選購在樂園內拍攝的「迪士尼處處拍」相片，保留珍貴回憶。千萬別錯過迪士尼專業攝影師為你捕捉的每個奇妙時刻！ {{ - }}

#### 廣場時尚店

**廣場時尚店**（**Center Street Boutique**）廣場時尚店包羅各種名貴及高級禮品，包括精美的玻璃製品、迷人的珠寶，以至獨家或限量發行的紀念品及服飾，讓你在美國小鎮大街上享受時尚購物體驗。快來把這些閃爍悅目的精緻紀念品帶回家吧！ {{ - }}

#### 小鎮影院 :「我與DUFFY遊歷之旅」

**小鎮影院 :「我與DUFFY遊歷之旅」**（**Main Street Cinema: My Journeys with Duffy**）此商店匯集歷來最齊全的迪士尼小熊Duffy與ShellieMay精品，而Duffy的最新好友貓畫家小東尼精品亦會在本店正式登場。光臨本店的賓客更有機會與Duffy見面，參與迪士尼精彩購物體驗！ {{ - }}

### 探險世界

#### 布教授商店

**布教授商店**（**Professor Porter's Trading Post**）在探險世界尋幽探秘時，別忘了找找這間隱蔽的竹林小屋，嘗試獨特的購物體驗！店內搜羅各式森林狩獵戰利品，例如你至愛的迪士尼朋友造型帽子、T恤以至毛公仔，件件都令人愛不釋手！ {{ - }}

#### 加利布尼市集

**加利布尼市集**（**Karibuni Marketplace**） {{ - }}

### 幻想世界

#### 童話藝坊

**童話藝坊**（**Storybook Shoppe**）來這間華麗的中世紀商品店，把全新的迷人公主禮服帶回家，準備參加下一次的皇室盛宴吧！你更可以把其他高貴的公主系列精品，包括皇冠、首飾及洋娃娃等，一併帶回家慢慢搭配，每位小公主實在不容錯過！ {{ - }}

#### 小熊維尼精品店

**小熊維尼精品店**（**Pooh Corner**）走進百畝林，在這個童話故事書一樣的天地，眾多小熊維尼精品在你面前，將不知不覺間奪取你的芳心。快來選購琳琅滿目的玩具、禮品和商品，為自己和摯愛送上點點心意！ {{ - }}

#### 梅林法寶店

**梅林法寶店**（**Merlin's Treasures**）這間迷人的古雅小屋坐落於幻想世界的一隅，讓你繼續沉醉在「米奇幻想曲」的奇妙體驗中，發掘各式各樣的紀念品及令人著迷的神奇珍寶，無論你是來自哪個年代的魔法師，來到此處都必定會大開眼界！ {{ - }}

#### 梅林神奇影像

**梅林法寶店**（**Merlin's Magic Portraits**）快前來位於幻想世界的相片銷售處選購「迪士尼處處拍」相片！迪士尼的專業攝影師會捕捉你在樂園的所有奇妙時刻，你毋需唸出任何咒語，仿佛在揮動魔術棒的瞬間，便可以即時查看及選購在樂園指定地點所拍攝的照片，將影像帶回家中留念。 {{ - }}

### 明日世界

#### 太空驛站

**太空驛站**（**Space Traders**）在太空驛站降落！這裡蘊藏新奇有趣的外太空珍寶讓你帶回家，你更可以購買在「星戰極速穿梭」全速飛行時被拍下的精彩照片呢！ {{ - }}

#### 史達科技旗艦店

**史達科技旗艦店**（**Expo Shop**）身為「鐵甲奇俠」迷的你，定要造訪「史達科技旗艦店」，內裡不但提供逾70款香港迪士尼樂園獨家的「鐵甲奇俠」商品以及超過250款漫威主題商品可供選購！ {{ - }}

#### 科技館禮品店

**科技館禮品店**（*'Pavilion Gifts*）科技館禮品店設於神盾局科技館大樓北翼，賓客可在此選購各種體現神盾局致力以「科學捍衛世界和平」精神的商品，以幫助正義摧毀邪惡。其他以漫威為相關主題的商品也在此供賓客選購。 {{ - }}

### 反斗奇兵大本營

#### 安仔玩具箱

**安仔玩具箱**（**Andy's Toy Box**）「反斗奇兵大本營」裡各式各樣的遊戲和玩具層出不窮。看！薯蛋頭先生還在繪畫設計藍圖呢！安仔剛剛出門了，他精心佈置的玩具箱難得無人看管，快趁現在找他的玩具老友記一起玩餐飽吧！ {{ - }}

#### 安仔火車頭

**安仔火車頭**（**Andy's Engine**） {{ - }}

### 灰熊山谷

#### 小灰熊禮品車

**小灰熊禮品車**（**Bear Necessities**）\[1\]經過這個西部販賣點時，別忘了為你的馬車裝滿由灰熊山採礦公司獨家發售的補給品，包括幸運護身符和「黃金」飾品，當然，你也可以順道選購其他服飾、精品和紀念品，為旅程留下美好的回憶。 {{ - }}

### 迷離莊園

#### 迷離珍藏店

**迷離珍藏店**（**The Archive Shop**）別忘記到迷離珍藏店，在亨利爵士稀奇古怪的寶物中挑選屬於你的珍藏，為奇幻歷程畫上圓滿句號！這裡出售的迷離莊園系列紀念品多不勝數，由精美的服飾、玩具以至其他工藝品，你找到你的心頭好了嗎？ {{ - }}

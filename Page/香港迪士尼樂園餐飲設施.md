**[香港迪士尼樂園](../Page/香港迪士尼樂園.md "wikilink")**（英語：**Hong Kong Disneyland**）位於[香港](../Page/香港.md "wikilink")[新界](../Page/新界.md "wikilink")[大嶼山](../Page/大嶼山.md "wikilink")的[香港迪士尼樂園度假區](../Page/香港迪士尼樂園度假區.md "wikilink")。

以下是香港迪士尼樂園內的餐飲設施。

## 現有餐飲設施

### 美國小鎮大街

#### 大街餐廳

**大街餐廳**（**Main Street Corner Cafe**）快來這個家庭式食府，品嚐美國中部及中式風味佳餚。 {{ - }}

#### 廣場飯店

**廣場飯店**（**Plaza Inn**）走進這座維多利亞式的外牆建築，欣賞高貴典雅的中式內部裝潢，盡情發掘粵菜之美。 {{ - }}

#### 大街市集

**大街市集**（**Main Street Market**）走進這極具懷舊氣息的市集時，千萬別錯過品嚐各款經典美食，如火雞腿、窩夫及雪條。滋味小吃加上濃厚的懷舊氛圍，必會令你留下深刻印象！ {{ - }}

#### 大街餅店

**大街餅店**（**Main Street Bakery**）大街上總是傳來陣陣新鮮糕餅、三文治、薄餅和甜品的誘人香氣，在大街上逛累了，不妨進來這間美國舊式餅店，喝杯咖啡或紅茶，稍事休息。 {{ - }}

#### 市集咖啡店

**市集咖啡店**（**Market House**）賓客可於此與家人及朋友品嚐各款星巴克手調飲品，為奇妙之旅添上更豐富的味覺享受。 {{ - }}

#### 爆谷、冰凍雪條、三文治、咖啡售賣點

**爆穀、冰凍雪條、三文治、咖啡售賣點**（**Popcorn, Frozen Lollipops, Sandwiches and Coffee Cart**）剛出爐的爆谷散發著誘人的香氣, 你當然不能錯過各款迪士尼朋友造型的冰凍雪條, 而且有多種口味可供選擇. 買件傳統三文治, 呷一口熱咖啡, 充電後再繼續探險吧\! 餐飲車的設計融合了小鎮大街的懷舊風格, 但提供的食物和飲品絕對是最新鮮的\! {{ - }}

### 探險世界

#### 碧林餐廳

**碧林餐廳**（**Tahitian Terrace**）一齊來揚帆遠航到南太平洋的熱帶天堂，大啖印度風味素食、清真食品和南亞美食，盡享異國美饌！ {{ - }}

#### 河景餐廳

**河景餐廳**（**River View Cafe**）你可以在這個露天餐室品嚐到各款東南亞美食和中式菜餚，同時欣賞冒險河的瑰麗景色。 {{ - }}

#### 加利布尼市集

**加利布尼市集**（**Karibuni Marketplace**）美味的韓式魷魚及火雞腿，配上冰凍飲品，每款都會帶給你不一樣的味覺享受。 {{ - }}

#### 韓式魷魚、火雞腿、冰凍飲品、冰凍雪條售賣點

**韓式魷魚、火雞腿、冰凍飲品、冰凍雪條售賣點**（**Korean Squid, Turkey Leg, Refreshing Drinks, Frozen Lollipops Cart**） {{ - }}

### 幻想世界

#### 皇室宴會廳

**皇室宴會廳**（**Royal Banquet Hall**）皇室宴會廳提供美式、日式和中式美食，令你猶如成為了皇室成員，在氣派的中世紀宴會廳中享用各式佳餚。 {{ - }}

#### 笑匠歡宴坊

**笑匠歡宴坊**（**Clopin's Festival of Foods**）在愉快歡樂的氣氛中吃盡「大中華風味」美食，廣東燒味、麵食和小炒等，在這裡一一為你送上！ {{ - }}

#### 小小世界雪糕屋

**小小世界雪糕屋**（**Small World Ice Cream**）在炎熱的午後來一口透心涼的爽滑雪糕，即時為你消暑降溫！即製香脆窩夫筒、鮮果窩夫籃等等口味，款款美味誘人，讓你任何時候都保持冰爽清新！ {{ - }}

#### 滷水雞腿、粟米、冰凍雪條售賣點

**滷水雞腿、粟米、冰凍雪條售賣點**（**Soya Chicken Leg, Corn on the Cob, Frozen Lollipops Cart**）這裡提供多款口味的小食，必定能夠讓你吃個痛快！先來一客滷水雞腿，再來個香甜粟米棒，最後加一支冰凍雪條作甜品吧！ {{ - }}

#### 爆谷、棉花糖、冰凍雪條售賣點

**爆谷、棉花糖、冰凍雪條售賣點**（**Popcorn, Cotton Candy, Frozen Lollipops Cart**）快來看看！剛出爐的爆谷散發著誘人的香氣，可愛棉花糖是否也讓你很想嚐一口呢？你當然不能錯過多款迪士尼朋友造型的冰凍雪條，保證讓你無時無刻都感受到迪士尼的奇妙魅力。 {{ - }}

### 明日世界

#### 火箭餐廳

**火箭餐廳**（**Starliner Diner**）這個星際太空站提供美式炸雞籃、芝士漢堡包、香煎雞扒飯及香煎牛扒，讓你體驗來自外太空的美食。 {{ - }}

#### 彗星餐廳

**彗星餐廳**（**Comet Cafe**）從星際旅程中放鬆心情，盡情享用燒味、粉麵及特色小炒等港式美食。 {{ - }}

#### BB-8小食車

**BB-8小食車**（**BB-8 Snack Cart**）當你進入BB-8小食車的軌跡，電光火石間，各款美食便盡在眼前，你可以即時以光速帶走你想要的小食！從甜點到冰凍特飲，這裡應有盡有，不論你來自浩瀚銀河的哪一個角落，總有一款符合你的口味。 {{ - }}

#### 爆谷、棉花糖、冰凍雪條售賣點

**爆谷、棉花糖、冰凍雪條售賣點**（**Popcorn, Cotton Candy, Frozen Lollipops Cart**）這個戶外流動餐飲車提供多款別緻小食，新鮮的爆谷、可愛的棉花糖以及多款迪士尼朋友造型的冰凍雪條，而且有多款口味可供選擇，玩累了，不妨先來這裡補充體力吧！ {{ - }}

### 反斗奇兵大本營

#### 翠絲小食車

**翠絲小食車**（**Jessie's Snack Roundup**）小食車裡滿載翠絲為你炮製的美味小吃，包括大受遊客歡迎的玉桂條、水果杯和軟雪糕等。總有一款合你心意！ {{ - }}

### 灰熊山谷

#### 幸運黃金石小食亭

**幸運黃金石小食亭**（**Lucky Nugget Saloon**）在美食中尋寶！淘出金黃色的炸雞薯條、炸魚薯條，或者發掘凱撒沙律的爽脆鮮美。肚子餓得像灰熊般大叫？快來品嚐充滿西部風味的餐點或小吃，墊墊肚子吧！ {{ - }}

#### 爆谷售賣點

**爆谷售賣點**（**Popcorn Cart**）無論你喜歡甜甜的焦糖味，或蜜糖芥末香味，金黃色的爆谷都會是你至愛的經典小吃。就像金礦一樣，雖然份量足夠和大家分享，但你只想把一整盤獨吞！ {{ - }}

### 迷離莊園

#### 大冒險家餐廳

**大冒險家餐廳**（**Explorer's Club Restaurant**）置身五大異國特色主題用餐區，盡享四國風味佳餚，展開一段美食大冒險。 {{ - }}

#### 冰凍雪條售賣點

**冰凍雪條售賣點**（**Frozen Lollipops Cart**）來這裡吃個米奇老鼠雪糕批或其他迪士尼朋友造型的雪條降降溫吧！每款均設有多種口味可供選擇，滿足不同口味。這裡亦提供各式飲品，讓你暢飲解渴，稍事休息後才繼續盡情探索迷離莊園的奧秘。 {{ - }}

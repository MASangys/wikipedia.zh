**香港開源年會**（，簡稱
HKOSCon），是在[香港舉辦的開放源碼年度](../Page/香港.md "wikilink")[研討會](https://zh.wikipedia.org/wiki/研討會 "wikilink")，旨在推動[開放源碼開發](https://zh.wikipedia.org/wiki/開放源碼 "wikilink")，促進社群交流，並鼓勵工商業界使用開放源碼[軟件及方案](https://zh.wikipedia.org/wiki/軟件 "wikilink")。

活動每年都會就不同主題邀請業界代表參與，分享[開放源碼趨勢及技術發展](https://zh.wikipedia.org/wiki/開放源碼 "wikilink")，資訊科技業界及其他技術組織亦參與協辦，再加上大專院校、熱心支持者、贊助商捐贈和支援，造就每年一次活動。由最初的演講、講座、，拓展至加入工作坊、[非會議](https://zh.wikipedia.org/wiki/非會議 "wikilink")、[聚會和開放源碼方案推廣攤位的規模](https://zh.wikipedia.org/wiki/聚會 "wikilink")。

## 年會

### 2013

於 2013
年[開源香港與開放系統研用協會及](https://zh.wikipedia.org/wiki/開源香港 "wikilink")[香港Linux用家協會合辦第一屆香港開源年會](../Page/香港Linux用家協會.md "wikilink")。會議於10月19日
\[1\] 在[香港城市大學舉行](../Page/香港城市大學.md "wikilink")，全日三軌會議超過 150
人參與，主講題目包括[開放數據](https://zh.wikipedia.org/wiki/開放數據 "wikilink")、開放互聯網、[開源硬件及探討開放源碼在程式設計](../Page/開源硬件.md "wikilink")、教育、遊戲及政策層面所扮演的角色。在徵求提案階段大會收到遠自歐洲的講者投稿，最終採納
[Free Software Initiative of
Japan](https://en.wikipedia.org/wiki/Free_Software_Initiative_of_Japan)、台灣[中央研究院及](https://zh.wikipedia.org/wiki/中央研究院 "wikilink")[臺灣大學的稿件](https://zh.wikipedia.org/wiki/臺灣大學 "wikilink")，連同香港本地代表，合共有
20 份講稿被納入議程。

### 2014

於 2014 年接受[香港互聯網協會舉辦的](../Page/香港互聯網協會.md "wikilink") World Internet
Developer Summit ( WiDS ) 邀請作為三日活動當中其中一項活動並於3月29日 \[2\]
在[香港科學園舉行](../Page/香港科學園.md "wikilink")，主辦方為[香港創意開放科技協會](https://cota.hk/)，並由[開源香港及](https://zh.wikipedia.org/wiki/開源香港 "wikilink")[香港Linux用家協會協辦](../Page/香港Linux用家協會.md "wikilink")。當日舉辦了
23 場會議，其中 18 位講者來自外地，13 個位講者來自本地，最終參與人數為 228 人。

### 2015

從 2013 及 2014
會期一天演變為兩天的活動，由香港創意開放科技協會、[香港Linux用家協會](../Page/香港Linux用家協會.md "wikilink")、[開源香港及](https://zh.wikipedia.org/wiki/開源香港 "wikilink")[香港科技園公司共同主辦](../Page/香港科技園公司.md "wikilink")，於6月26及27日
\[3\] 在[香港科學園舉行](../Page/香港科學園.md "wikilink")。是年會議口號為 Connect Global
Open Source Technology in Hong
Kong.，以[雲端](https://zh.wikipedia.org/wiki/雲端 "wikilink")、[自造者](https://zh.wikipedia.org/wiki/自造者 "wikilink")、社群、新創企業
( Cloud // Maker // Community // Startup ) 四大主題為基礎，27 位海外講者及 25
位本地講者在兩日五軌會議合共舉辦 52 節演講及工作坊，並由參展商及社群設立多個展覽攤位，參與人數總共為
538 人。\[4\] 是年有來自美國的 [LWN.net](../Page/LWN.net.md "wikilink") 創辦人
Jonathan Corbet 及來自德國的 [GNOME](../Page/GNOME.md "wikilink") 基金會董事會前成員
Tobias Mueller 的主題演講。來自俄羅斯國立托姆斯克大學及 [Alt
Linux](https://en.wikipedia.org/wiki/ALT_Linux) 的失明開發者 Michael Pozhidaev
分享開發給失明及視障人士使用的開源方案，多位來自歐洲各地、日本及香港的自造者分享不同講題，例如西班牙的廢物再造開源飛行器，意大利的開源汽車，保加利亞的
[Internet of
Things](https://zh.wikipedia.org/wiki/Internet_of_Things "wikilink")、以及香港的開源教育三維列印機等等。\[5\]

### 2016

是年開始香港開源年會固定由香港創意開放科技協會、[香港Linux用家協會](../Page/香港Linux用家協會.md "wikilink")、[開源香港共同主辦](https://zh.wikipedia.org/wiki/開源香港 "wikilink")，於6月24-26日
\[6\] 在[香港數碼港舉行](https://zh.wikipedia.org/wiki/香港數碼港 "wikilink")。口號為 Go
Beyond
Open-source，圍繞著[物聯網](https://zh.wikipedia.org/wiki/Internet_of_Things "wikilink")、[大數據](../Page/大數據.md "wikilink")、[雲端和](https://zh.wikipedia.org/wiki/雲端 "wikilink")
DevOp ( Internet of Things // Data // Cloud // DevOps ) 四大主題，三天四軌會議有 23
位外籍講者及 14 位本地講者參與 ，參與人數總共為 355 人。\[7\] 主題演講包括來自澳洲的系統工程師 Joshua Rich 介紹
[Elasticsearch](../Page/Elasticsearch.md "wikilink") 開源搜尋引擎 \[8\] 及台灣知名
Linux 核心開發高手黃敬群 ( Jserv ) 主講《回顧台灣自由開源軟體發展》。\[9\]

### 2017

是年會議於6月9-10日 \[10\]
在[香港數碼港舉行](https://zh.wikipedia.org/wiki/香港數碼港 "wikilink")，口號為
Embracing Open Source's
Success，專注示範及推廣世界最新開放源碼軟件及硬件技術。第一天主題為商業及教育，第二天為社群交流，活動亦吸引企業到場招聘。\[11\]

### 2018

是年會議於6月9-10日 \[12\] 在[香港科學園舉行](../Page/香港科學園.md "wikilink")，口號為 Be
prepared for Open Source awesomeness，並開放供大眾免費參與。\[13\] 18 位外籍講者及 27
位本地講者合共舉辦 43 節演講及工作坊，參與人數總共為 377 人。\[14\] 講者包括來自香港的古樂新分享以 HTTP
Live Streaming（HLS）傳輸協議配合開源的 video.js 技術於網站上播放直播 \[15\]
及台灣「數碼農民」陳幸延以人工智能耕種。\[16\]

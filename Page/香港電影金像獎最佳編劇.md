**香港電影金像獎最佳編劇**是[香港電影金像獎](../Page/香港電影金像獎.md "wikilink")下設的一個獎項，該獎項主要授予年度最出色的香港電影編劇。

## 統計

| **統計**       | **名字**                           | **數據** | **備註** |
| ------------ | -------------------------------- | ------ | ------ |
| 獲獎最多的編劇      | [莊文強](../Page/莊文強.md "wikilink") | 3次     |        |
| 入圍最多次的編劇     | [游乃海](../Page/游乃海.md "wikilink") | 11次    | 2次得獎   |
| 入圍最多但從未獲獎的編劇 | [阮世生](../Page/阮世生.md "wikilink") | 8次     |        |
|              |                                  |        |        |

**香道**（こうどう）為[日本](../Page/日本.md "wikilink")傳統藝道，依一定的禮法焚[香木](https://zh.wikipedia.org/wiki/香木 "wikilink")，鑑賞香氣。也稱作**香遊**。主要分成鑑賞香木香氣的「**聞香**」和判別香氣的遊戲「**組香**」的二種要素。

## 歷史

根據[日本書紀](../Page/日本書紀.md "wikilink")，[推古天皇](../Page/推古天皇.md "wikilink")3年（595年），[香木漂着](https://zh.wikipedia.org/wiki/香木 "wikilink")[淡路島](../Page/淡路島.md "wikilink")\[1\]。日本香文化源自古[印度](../Page/印度.md "wikilink")，經由[中國](../Page/中國.md "wikilink")，和[佛教](../Page/佛教.md "wikilink")共同傳入日本。[平安時代](https://zh.wikipedia.org/wiki/平安時代 "wikilink")，脫離宗教儀禮，演變為聞香鑑賞，有[薰物合等宮廷遊戲的舉行](https://zh.wikipedia.org/wiki/薰物合 "wikilink")\[2\]。

香道作為藝道確立在[室町時代的](https://zh.wikipedia.org/wiki/室町時代 "wikilink")[東山文化期](https://zh.wikipedia.org/wiki/東山文化 "wikilink")，幾乎在[茶道](https://zh.wikipedia.org/wiki/茶道 "wikilink")、[華道](../Page/華道.md "wikilink")集大成的同時期，香道的禮法也確立為接近現在的形態。當時，香木的分類法「六國五味」等也體系化。

香道的古典書有[建部隆勝的](https://zh.wikipedia.org/wiki/建部隆勝 "wikilink")『[香道秘傳書](https://zh.wikipedia.org/wiki/香道秘傳書 "wikilink")』和[蜂谷宗悟的](https://zh.wikipedia.org/wiki/蜂谷宗悟 "wikilink")『[香道軌範](https://zh.wikipedia.org/wiki/香道軌範 "wikilink")』，皆在[天正年間編纂](../Page/天正_\(日本\).md "wikilink")。

## 香十德

關於香道的十德。

1.  感格鬼神
2.  清淨心身
3.  能除汚穢
4.  能覺睡眠
5.  靜中成友
6.  塵裏偸閑
7.  多而不厭
8.  寡而為足
9.  久藏不朽
10. 常用無障

## 六國五味

香木的香質分成辛・甘・酸・鹹・苦的「五味」。含有樹脂的質和量的差異分成以下的6種類，稱為六國。

| 木所   | 讀音      |
| ---- | ------- |
| 伽羅   | きゃら     |
| 羅國   | らこく     |
| 真那伽  | まなか     |
| 真南蛮  | まなばん    |
| 佐曾羅  | さそら     |
| 寸聞多羅 | すも（ん）たら |

## 聞香

依一定的禮法聞香稱作「**聞香**」。禮法因流派而有不同。

## 組香

組香是依一定的規則判別香氣的遊戲。從文學的要素到一般教養等規則多種多樣規則，比賽回答的成否、優劣。

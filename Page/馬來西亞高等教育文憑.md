**马来西亚高等教育文凭**（亦称：马来西亚高级教育文凭；；）是一个拥有与国际[GCE
A-Level同等水平待遇的文凭](https://zh.wikipedia.org/wiki/GCE_A-Level "wikilink")。
STPM由[马来西亚考试委员会举办](https://zh.wikipedia.org/wiki/马来西亚考试委员会 "wikilink")，[马来西亚教育部与](../Page/马来西亚教育部.md "wikilink")[剑桥大学考试委员会共同监督](../Page/剑桥大学考试委员会.md "wikilink")。
STPM是[马来西亚的大学预科公共考试](../Page/马来西亚.md "wikilink")，也是[马来西亚中学阶段最后一个全国统一考试](https://zh.wikipedia.org/wiki/马来西亚中学 "wikilink")。

## 简介

STPM的前身是「高等教育文凭」（Higher School
Certificate，HSC），HSC是[英国](https://zh.wikipedia.org/wiki/英国 "wikilink")[GCE
A-Level考试的前称](https://zh.wikipedia.org/wiki/GCE_A-Level "wikilink")，同时也是[澳洲部分州属的公共考试名称](https://zh.wikipedia.org/wiki/澳洲 "wikilink")。
STPM是由[马来西亚考试委员会](https://zh.wikipedia.org/wiki/马来西亚考试委员会 "wikilink")（；）自1982年开始独立举办，[剑桥大学考试委员会](../Page/剑桥大学考试委员会.md "wikilink")
（，UCLES）从旁监督和给予意见。马来西亚考试委员会也同时举办[马来西亚大学英文水平鉴定考试](../Page/马来西亚大学英文水平鉴定考试.md "wikilink")（,
MUET）。与[马来西亚教育文凭](https://zh.wikipedia.org/wiki/马来西亚教育文凭 "wikilink")（SPM）,
[中三评估](../Page/中三评估.md "wikilink")（PT3）和[小六检定考试](https://zh.wikipedia.org/wiki/小六检定考试 "wikilink")（UPSR）不同的是，此三项考试皆由[马来西亚考试局举办而非马来西亚考试委员会](https://zh.wikipedia.org/wiki/马来西亚考试局 "wikilink")。但是，马来西亚考试局和马来西亚考试委员会皆隶属[马来西亚教育部](../Page/马来西亚教育部.md "wikilink")。

### 文凭认证

STPM与[马来西亚大学先修班考试](https://zh.wikipedia.org/wiki/马来西亚大学先修班考试 "wikilink")（Matriculation）是备受马来西亚国立大学首要认可的公共考试。
STPM备受国内外大学的认可，并认定STPM为与国际[GCE
A-Level同一水准的考试](https://zh.wikipedia.org/wiki/A-Level "wikilink")。所以，STPM考生可透过本身的STPM成绩申请国内或国外著名的上等大学。普遍上，国内外大学都会要求STPM考生同时报考MUET。

### 学期制考试

从2012年开始，STPM将会以学期制的考试（Modular System）制度来取代现有的期末考制度（Terminal
System）\[1\]。马来西亚考试委员会相信此项措施将会吸引更多的本地学生报考STPM。此外，[剑桥大学考试委员会也会继续和从](../Page/剑桥大学考试委员会.md "wikilink")2012年开始从旁监督在新制度下的STPM考试\[2\]。

## 中六

[马来西亚教育是基于](../Page/马来西亚教育.md "wikilink")[英国的](https://zh.wikipedia.org/wiki/英国 "wikilink")[教育系统所形成的](../Page/英国教育.md "wikilink")，所以STPM完全衔接中学五年的教育，故被称为「中六」（Form
6）。马来西亚教育部会选择合适的中学来开办中六课程。

中六课程被分为三个学期：第一学期（Semester 1）、第二学期（Semester 2）及第三学期（Semester
3），一期半年，总共学习一年半。

### 入学要求

所有欲想申请修读中六的学生都必须以[马来西亚教育文凭](https://zh.wikipedia.org/wiki/马来西亚教育文凭 "wikilink")（SPM）的成绩进行申请，其对SPM成绩的要求包括：

  - [马来文考获良好等级](https://zh.wikipedia.org/wiki/马来文 "wikilink")（Credit）或以上。
  - [历史科考获良好等级](../Page/马来西亚历史.md "wikilink")（Credit）或以上。
  - 至少考获不包括马来文与历史科在内的3个良好等级（Credit）。
  - 不包括马来文与历史科在内最好的三个科目中的总和评级数值（Sum GPS）不可超过12分。
  - 中四与中五活跃于课外活动。
  - 在开课年时年龄介于17至20岁。

## 考试科目

在新制度下，考生可选考最多五个科目，包括普通常识科（，）\[3\]。虽然普通常识非必考科，但大部分考生会选择报考此科目以符合本地国立大学的入学需求。此外，校本评估制度（，简称：「SBA」）也会被纳入STPM的其中一项评分标准。根据马来西亚考试局的资料，SBA会占据各科总分的20%至40%，视乎不同科目而定。但是，重考生必须选择笔试以代替原有的校本评估项目。

以下将列出STPM所有供选考的科目。而由于STPM为大学预科考试，因此此考试并没有设置任何必考科目。\[4\]

## 等级和成绩平均绩点（CGPA）

STPM采用成绩平均绩点（CGPA）来计算考生的成绩\[5\]。同时，考生的成绩也会根据11个等级来评级，它们分别是A, A-, B+, B,
B-, C+, C, C-, D+ and D,
而F则是不及格的等级。每个等级的平均绩点则从最高的4.0至最低的1.0，考获F等级的考生不获任何绩点，即0分\[6\]\[7\]。

| 各卷等级 | 科目等级 | 科目平均绩点（CGPA）\[8\] | 分数 (%)   | 等级                 |
| ---- | ---- | ----------------- | -------- | ------------------ |
| A    | A    | 4.00              | 75 - 100 | 及格（Principal Pass） |
| A-   | A-   | 3.67              | 70 -74   |                    |
| B+   | B+   | 3.33              | 60 - 69  |                    |
| B    | B    | 3.00              | 55 - 59  |                    |
| B-   | B-   | 2.67              | 50 - 54  |                    |
| C+   | C+   | 2.33              | 45 - 49  |                    |
| C    | C    | 2.00              | 40 - 44  |                    |
| C-   | C-   | 1.67              | 35 - 39  | 部分及格（Partial Pass） |
| D+   | D+   | 1.33              | 30 - 34  |                    |
| D    | D    | 1.00              | 25 - 29  |                    |
| F    | F    | 0.00              | 0 - 25   | 不及格（Failed）        |

考生必须考获至少C等级
(绩点：2.00)才能算及格，而考获D至C-的考生则被定义为“部分及格”。不过，本地某些私立大学，如拉曼大学则把“部分及格”的标准定义在D至C，而非C-。

考生所考获的成绩将会列印在成绩单和正式的文凭上。所有的等级和积分也会按照该届考生的整体表现进行调整。成绩平均绩点
(CGPA)的计算方法是以等级最高的四个科目来计算，而平均绩点也会被四舍五入至小数点后二位。举个例子，若一个考生报考5个科目，其中4个科目全部考获全部A，而另外一科考获F
(不及格)，那么他的CGPA仍然会是4.0满分。

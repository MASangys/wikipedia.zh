**高斯單位制**（）是一種[計量單位的制度](https://zh.wikipedia.org/wiki/計量單位 "wikilink")，屬於[米制](../Page/米制.md "wikilink")，從[厘米-克-秒制衍生](../Page/厘米-克-秒制.md "wikilink")。厘米-克-秒制有幾組互相衝突的電磁單位，其中高斯單位最常見。

高斯單位制外最常用的別種選擇是[國際單位制](https://zh.wikipedia.org/wiki/國際單位制 "wikilink")。在大多述領域，國際單位制是主要使用的單位制。人们渐渐摒棄高斯單位制，改用國際單位制\[1\]\[2\]。高斯單位制與國際單位制之間的單位轉換並不像平常單位轉換那樣簡易。例如，[電磁學的](https://zh.wikipedia.org/wiki/電磁學 "wikilink")[馬克士威方程組](../Page/馬克士威方程組.md "wikilink")，依使用不同單位制而形式不同：[電容率或](https://zh.wikipedia.org/wiki/電容率 "wikilink")[磁導率在高斯單位制是無](https://zh.wikipedia.org/wiki/磁導率 "wikilink")[因次的物理量](https://zh.wikipedia.org/wiki/因次 "wikilink")，在國際單位制可能有因次。

高斯單位制必須與國際單位制掛鉤才有實驗意義，因為只有國際單位制精確定義了各個物理量。在某些領域高斯單位制的公式較為簡潔，例如在經常需要處理球對稱情況的天體物理學。但在經常需要處理直線情況的工程領域，國際單位制的公式更加簡潔。

## 各種不同的厘米-克-秒制

在厘米-克-秒制內，有好幾種電磁單位系統，高斯單位制只是其中的一種。其它有[靜電單位制](https://zh.wikipedia.org/wiki/靜電單位制 "wikilink")（）、[電磁單位制](https://zh.wikipedia.org/wiki/電磁單位制 "wikilink")（）、[勞侖茲-黑維塞單位制](https://zh.wikipedia.org/wiki/勞侖茲-黑維塞單位制 "wikilink")（）。

另外還有一類稱為[自然單位制的制度](https://zh.wikipedia.org/wiki/自然單位制 "wikilink")，包括[原子單位制](https://zh.wikipedia.org/wiki/原子單位制 "wikilink")、[普朗克單位制等等](../Page/普朗克單位制.md "wikilink")。從某些方面來看，在厘米-克-秒制與國際單位制之間，這些自然單位制比較接近前者。例如，假設選擇自然單位制或厘米-克-秒制，則[高斯定律方程式裏](../Page/高斯定律.md "wikilink")，都有一個因子\(4\pi\)，而[庫侖定律方程式裏則無](https://zh.wikipedia.org/wiki/庫侖定律 "wikilink")；假設選擇國際單位制，則高斯定律方程式裏沒有因子\(4\pi\)，而庫侖定律方程式裏則有。另外，高斯單位制比國際單位制少一個基本單位，即[電荷的單位不是基本單位](../Page/電荷.md "wikilink")，而自然單位制的基本單位又少了許多。

現今，國際單位制是最常使用的單位制。在工程學領域與實用領域，幾乎普遍採用國際單位制，這已是很多年的事實\[3\]。在科技文獻裏，像[理論物理學與](https://zh.wikipedia.org/wiki/理論物理學 "wikilink")[天文學的文獻](../Page/天文學.md "wikilink")，直到最近幾年，高斯單位制是主要單位制，但現在也越來越少使用\[4\]\[5\]。

自然單位制比較常見於更理論、更抽象的物理領域，特別是[粒子物理學與](../Page/粒子物理學.md "wikilink")[弦理論](../Page/弦理論.md "wikilink")。

## 高斯單位制與國際單位制之間重要差別

### 有理化單位制

高斯單位制與國際單位制之間，一個差別是在一些方程式裏的因子\(4\pi\)。國際單位制被分類為「有理化單位制」\[6\]\[7\]，因為，馬克士威方程組裏沒有因子\(4\pi\)，但是，庫侖定律和[必歐-沙伐定律的方程式裏](https://zh.wikipedia.org/wiki/必歐-沙伐定律 "wikilink")，都含有因子\(4\pi\)。採用高斯單位制的狀況完全相反。馬克士威方程組裏含有因子\(4\pi\)，但是，庫侖定律和必歐-沙伐定律的方程式裏，都沒有因子\(4\pi\)。因此，高斯單位制被分類為「非理化單位制」。

### 電荷的單位

對於高斯單位制與國際單位制，電荷單位的定義有很大的區別。國際單位制特別為[電現象設置一個基本單位](https://zh.wikipedia.org/wiki/電現象 "wikilink")──[安培(ampere)](../Page/安培.md "wikilink")，這動作的後果是，電荷是一種物理數量的一種獨特因次，（1[库伦（coulomb）](https://zh.wikipedia.org/wiki/庫侖 "wikilink")=1ampere×1second）不能用機械單位（kg、m、s）來表達。庫侖定律方程式為

\[F = \frac{1}{4\pi\epsilon_0} \frac{Q_1Q_2}{r^2}\]；

其中，\(F\)是庫侖力，\(\epsilon_0\)是[電常數](https://zh.wikipedia.org/wiki/電常數 "wikilink")，\(Q_1\)和\(Q_2\)是兩個相互作用的電荷，\(r\)是這兩個電荷之間的距離。

電常數\(\epsilon_0\)的因次為ampere<sup>2</sup> s<sup>4</sup> kg<sup>−1</sup>
m<sup>−3</sup>。藉著電常數\(\epsilon_0\)的單位，庫侖定律方程式兩邊的因次可以一致化。假若沒有\(\epsilon_0\)，則方程式兩邊的因次不一致；而在高斯單位制內，\(\epsilon_0\)並不存在。

在高斯單位制內，電荷的單位[statC](https://zh.wikipedia.org/wiki/靜庫倫 "wikilink")，可以完全以機械單位寫為

  -
    1 statC = 1 g<sup>1/2</sup> cm<sup>3/2</sup> s<sup>−1</sup>

庫侖定律方程式相當簡單：

\[F = \frac{Q_1Q_2}{r^2}\]。

假設電荷的單位是statC，半徑的單位是cm，則作用力的單位是[达因（dyne）](https://zh.wikipedia.org/wiki/達因 "wikilink")。

### 磁物理量的單位

與國際單位制不同，在高斯單位制內，電場\(\mathbf{E}\)與[磁感應強度](https://zh.wikipedia.org/wiki/磁感應強度 "wikilink")（B場）\(\mathbf{B}\)的因次一樣。這總計為B場在兩個單位制內[光速因子](../Page/光速.md "wikilink")\(c\)的差異。同樣的因子也發生於其他磁物理量，像[磁場強度](https://zh.wikipedia.org/wiki/磁場強度 "wikilink")（H場）與磁化強度。例如，對於傳播於真空的平面電磁波，在高斯單位制內，\(|\mathbf{E}(\mathbf{r},t)|=|\mathbf{B}(\mathbf{r},t)|\)；但在國際單位制內，\(|\mathbf{E}(\mathbf{r},t)|=c|\mathbf{B}(\mathbf{r},t)|\)。

### 電極化向量、磁化向量

與國際單位制不同，在高斯單位制內，電場\(\mathbf{E}\)、電位移\(\mathbf{D}\)、電極化強度\(\mathbf{P}\)、磁感應強度\(\mathbf{B}\)、磁場強度\(\mathbf{H}\)、磁化強度\(\mathbf{M}\)，都具有同樣的因次。另外一個重點是，在高斯單位制與國際單位制內，物質的電極化率與磁化率都不具因次，然而，它們在兩個單位制內的數值都不一樣。稍後，會列出它們的方程式。

### 介電質與磁物質

假設最簡單介電質案例，即線性、各向同性、均勻的介電質，[電容率只是常數](https://zh.wikipedia.org/wiki/電容率 "wikilink")。

| 高斯單位                                       | 國際單位                                              |
| ------------------------------------------ | ------------------------------------------------- |
| \(\mathbf{D} = \mathbf{E}+4\pi\mathbf{P}\) | \(\mathbf{D} = \epsilon_0 \mathbf{E}+\mathbf{P}\) |
| \(\mathbf{P} = \chi_e\mathbf{E}\)          | \(\mathbf{P} = \chi_e\epsilon_0\mathbf{E}\)       |
| \(\mathbf{D} = \epsilon\mathbf{E}\)        | \(\mathbf{D} = \epsilon\mathbf{E}\)               |
| \(\epsilon = 1+4\pi\chi_e\)                | \(\epsilon/\epsilon_0 = 1+\chi_e\)                |

  - \(\chi_e\)是[電極化率](../Page/電極化率.md "wikilink")，\(\epsilon\)是電容率。
  - \(\epsilon\)在高斯單位制內，與\(\epsilon/\epsilon_0\)在國際單位制內，都不具因次，但數值相同。
  - \(\chi_e\)在兩種單位制內，都不具因次，但數值不相同。

假設最簡單的磁物質案例，即線性、各向同性、均勻的磁物質，[磁導率只是常數](https://zh.wikipedia.org/wiki/磁導率 "wikilink")。

| 高斯單位                                       | 國際單位                                           |
| ------------------------------------------ | ---------------------------------------------- |
| \(\mathbf{B} = \mathbf{H}+4\pi\mathbf{M}\) | \(\mathbf{B} = \mu_0 (\mathbf{H}+\mathbf{M})\) |
| \(\mathbf{M} = \chi_m\mathbf{H}\)          | \(\mathbf{M} = \chi_m\mathbf{H}\)              |
| \(\mathbf{B} = \mu\mathbf{H}\)             | \(\mathbf{B} = \mu\mathbf{H}\)                 |
| \(\mu = 1+4\pi\chi_m\)                     | \(\mu/\mu_0 = 1+\chi_m\)                       |

  - \(\chi_m\)是磁化率，\(\mu\)是磁導率。
  - \(\mu\)在高斯單位制內，與\(\mu/\mu_0\)在國際單位制內，都不具因次，但數值相同。
  - \(\chi_m\)在兩種單位制內，都不具因次，但數值不相同。

在這表格裏，字母\(c\)代表數值29,979,245,800 ≈
3×10<sup>10</sup>。這是[光速在高斯單位制內的數值](../Page/光速.md "wikilink")（光速為3×10<sup>10</sup>cm/s）。採用符號"↔"來強調，國際單位與高斯單位之間的關係，是對應關係，而不是相等關係，因為兩種單位的因次互不相容。例如，從表格的最上一橫行，在國際單位制內，帶有1
[C電量的粒子](https://zh.wikipedia.org/wiki/庫侖 "wikilink")，轉換到在高斯單位制內，則帶有（10<sup>−1</sup>
c [statC電量](../Page/靜庫侖.md "wikilink")，或3×10<sup>9</sup> statC電量）。

另外一個令人驚訝的事實為，[電阻率的計量單位為秒](https://zh.wikipedia.org/wiki/電阻率 "wikilink")。舉一個實例：設定一個[平行板電容器](https://zh.wikipedia.org/wiki/平行板電容器 "wikilink")，其介電質的[電容率為](https://zh.wikipedia.org/wiki/電容率 "wikilink")1、[電阻率為](https://zh.wikipedia.org/wiki/電阻率 "wikilink")\(X\)秒，則在充電之後，這電容器會隨著時間的流易而放電，因為電流會漏過介電值，造成漏電，而放電的[半壽命期為](https://zh.wikipedia.org/wiki/半衰期 "wikilink")\(\approx 0.05X\)秒。這結果與電容器的大小、形狀、電量都無關。這實例顯示出電阻率與時間單位之間的基本關聯。

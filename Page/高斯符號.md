**高斯符號**是一个数学符号，形式为方括号\[x\]，表示不大於（等于或小于）[数x的最大](../Page/数.md "wikilink")[整數](../Page/整數.md "wikilink")，即x-1＜\[x\]≤x。

高斯符號首次出現是在[高斯的數學巨著](../Page/高斯.md "wikilink")《[算术研究](../Page/算术研究.md "wikilink")》。

运算示例：\[3.14159\]=3，\[2\]=2，\[-2.5\]=-3。

在计算机科学中，高斯符號常表示为INT()函数。

后来[肯尼斯·艾佛森在](../Page/肯尼斯·艾佛森.md "wikilink")1962年時於其著作《A Programming
Language》中把高斯符号称作**[取底符号](../Page/取底符号.md "wikilink")**（\(\lfloor x\rfloor\)，floor），并同时引进**取顶符号**（\(\lceil x\rceil\)，ceil）(用以表示不小於x的整數中最小的一個)。之后大家就普遍使用取底、取顶这种说法了。

## 高斯符號的一些性质

  - \(\lfloor x\rfloor \le x < \lfloor x \rfloor + 1\)

<!-- end list -->

  -
    当且仅当*x*是整数时，左面的等号成立。

<!-- end list -->

  - 对于所有实数*x*，有：

\[\left\lfloor \frac{x}{2} \right\rfloor = \frac{1}{4} ((-1)^{\lfloor x \rfloor} -1 + 2 \lfloor x \rfloor)\]

\[\left\lfloor \frac{x}{3} \right\rfloor = \frac{-2}{\sqrt{3}} \sin(\frac{2\pi}{3}\lfloor x \rfloor +\frac{\pi}{3}) + 1\]

  - 当*n*为正整数时，有：

\[\left\lfloor \frac{x}{n} \right\rfloor = \frac{x-x(rem ~ n)}{n}\]

  - 当*x*和*n*是正数时，有：

\[\left\lfloor \frac{n}{x} \right\rfloor \geq \frac{n}{x} - \frac{x-1}{x}\]

  - 对于任何整数*k*和任何实数*x*，有：

\[\lfloor {k+x} \rfloor = k + \lfloor x\rfloor.\]

  - 如果*x*是实数，*n*是整数，我们有\(n \le x\) [当且仅当](../Page/当且仅当.md "wikilink")
    \(n \le \lfloor x \rfloor\)。
  - 利用高斯符號，可以产生许多[素数公式](../Page/素数公式.md "wikilink")（但没有实际用途）。
  - 对于非整数的实数*x*，高斯函数具有以下的[傅里叶级数展开式](../Page/傅里叶级数.md "wikilink")：

\[\lfloor x\rfloor = x - \frac{1}{2} + \frac{1}{\pi} \sum_{k=1}^\infty \frac{\sin(2 \pi k x)}{k}.\]

  - 如果*m*和*n*是[互素的正整数](../Page/互素.md "wikilink")，那么：

\[\sum_{i=1}^{n-1} \lfloor im / n \rfloor = (m - 1) (n - 1) / 2\]

**高氯酸銣**是一種[無機化合物](https://zh.wikipedia.org/wiki/無機化合物 "wikilink")，屬於[高氯酸鹽](https://zh.wikipedia.org/wiki/高氯酸鹽 "wikilink")，其化學式為RbClO<sub>4</sub>。高氯酸銣是一種[氧化劑](https://zh.wikipedia.org/wiki/氧化劑 "wikilink")。

它有兩種晶型。其中，低於279°C為[斜方晶系](https://zh.wikipedia.org/wiki/斜方晶系 "wikilink")[晶體](https://zh.wikipedia.org/wiki/晶體 "wikilink")，晶格常數
a=0.927 nm，B =0.581 nm，C
=0.753 nm；超過279°C，它具有[立方結構](https://zh.wikipedia.org/wiki/立方 "wikilink")，晶格常數
a=0.770 nm\[1\]

在水中的溶解度:\[2\]

| 溫度 (°C)          | 0    | 8.5  | 14    | 20    | 25   | 50    | 70   | 99    |
| ---------------- | ---- | ---- | ----- | ----- | ---- | ----- | ---- | ----- |
| 溶解度 (g / 100 ml) | 1.09 | 0.59 | 0.767 | 0.999 | 1.30 | 3.442 | 6.72 | 17.39 |
|                  |      |      |       |       |      |       |      |       |

## 制备

高氯酸铷可由铷的化合物和[高氯酸或](../Page/高氯酸.md "wikilink")[高氯酸钡反应得到](../Page/高氯酸钡.md "wikilink")：

  -
    Rb<sub>2</sub>CO<sub>3</sub> + 2 HClO<sub>4</sub> → 2
    RbClO<sub>4</sub> + H<sub>2</sub>O + CO<sub>2</sub>↑
    Rb<sub>2</sub>SO<sub>4</sub> + Ba(ClO<sub>4</sub>)<sub>2</sub> → 2
    RbClO<sub>4</sub> + BaSO<sub>4</sub>↓

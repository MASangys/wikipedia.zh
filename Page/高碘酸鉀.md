**高碘酸鉀**是一種[無機化合物](https://zh.wikipedia.org/wiki/無機化合物 "wikilink")，為一種[高碘酸](../Page/高碘酸.md "wikilink")[盐](https://zh.wikipedia.org/wiki/盐 "wikilink")，其[化學式為](https://zh.wikipedia.org/wiki/化學式 "wikilink"):KIO<sub>4</sub>。這種化學品內含有[鉀](https://zh.wikipedia.org/wiki/鉀 "wikilink")、[氧](../Page/氧.md "wikilink")、[碘](../Page/碘.md "wikilink")，其中[氧](../Page/氧.md "wikilink")、[碘組成](../Page/碘.md "wikilink")[高碘酸根離子](../Page/高碘酸.md "wikilink")。

**高碘酸鉀**微溶於水（這是溶解度很低的鉀鹽，因為有一個很大的陰離子，因此，其水溶液呈弱鹼性。**高碘酸鉀**加熱時（尤其是以二氧化錳(氧化錳(IV))作為催化劑時），它將會分解，並釋放出氧氣。

低溶解度的**高碘酸鉀**有助於[鉀的測定](https://zh.wikipedia.org/wiki/鉀 "wikilink")。

## 名稱的念法

請注意，在英文中，Potassium periodate發音是per-iodate，而不是period-ate。

而台灣繁體中文應寫為**過碘酸鉀**，而不是**高碘酸鉀**。

## 代表性

為了表示對實驗室使用碘酸鉀，氧化借助鉀的鹼性範圍（方便鉀肥，氫氧化鉀）：

\[\mathrm{IO_{3}^{-} + S_{2}O_{8}^{2-} + 2 \ OH^- \longrightarrow IO_4^- + 2 \ SO_4^{2-} + H_2O}\]

## 制備

在[氯水中加入](https://zh.wikipedia.org/wiki/氯水 "wikilink")[碘酸鉀和](https://zh.wikipedia.org/wiki/碘酸鉀 "wikilink")[氫氧化鉀](https://zh.wikipedia.org/wiki/氫氧化鉀 "wikilink")，然後煮沸，會出現沉澱，再用冰水洗滌和過濾。另外會產生[氯化鉀](https://zh.wikipedia.org/wiki/氯化鉀 "wikilink")\[1\]\[2\]：

\[\mathrm{KIO_3 + 2 \ KOH + Cl_2 \longrightarrow KIO_4 + 2 \ KCl + H_2O}\]

## 性質

高碘酸鉀是一種強氧化劑。\[3\]

\[\mathrm{KIO_4 + 2\ KI + H_2O \longrightarrow KIO_3 + I_2 + 2 KOH}\]

\[\mathrm{KIO_4 + 2\ KI + H_2O \longrightarrow KIO_3 + I_2 + 2 KOH}\]

其晶體同晶的高氯酸鉀。

當**高碘酸鉀**溶解於[氫氧化鉀生成](https://zh.wikipedia.org/wiki/氫氧化鉀 "wikilink")[超碘酸酐鉀](https://zh.wikipedia.org/wiki/超碘酸酐鉀 "wikilink")\[4\],
與硝酸反應可回到簡單的碘酸鉀。\[5\]\[6\]

\[\mathrm{2\ KIO_4 + 2\ KOH \longrightarrow K_4I_2O_9 + H_2O}\]

\[\mathrm{K_4I_2O_9 + 2\ HNO_3 \longrightarrow 2KIO_4 + 2KNO_3 + H_2O}\]

-----

\[\mathrm{2\ K_2H_3IO_6 \ \xrightarrow {100 ^{o}C} \ K_4I_2O_9 + 3\ H_2O}\]\[7\]

在水溶液離子平衡：\[8\]

\[\mathrm{\lbrack H_3IO_6 \rbrack ^{2-} + H^+ \rightleftharpoons \lbrack IO_4 \rbrack ^{-} + 2\ H_2O}\]

\[\mathrm{2\ \lbrack H_3IO_6 \rbrack ^{2-} \rightleftharpoons 2\ \lbrack HIO_5 \rbrack ^{2-} + 2\ H_2O}\]

\[\mathrm{2\ \lbrack HIO_5 \rbrack ^{2-} \rightleftharpoons \lbrack H_2I_2O_{10} \rbrack ^{4-} \rightleftharpoons \lbrack I_2O_9 \rbrack ^{4-} + H_2O}\]

## 用途

除了為氧化劑外，高碘酸鉀也作為分析試劑用於測定[鈰](https://zh.wikipedia.org/wiki/鈰 "wikilink")\[9\]。

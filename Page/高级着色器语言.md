**高级着色器语言**（即 **HLSL**，缩写自 \[1\] 或
\[2\]），是由[微軟擁有及開發的一種](https://zh.wikipedia.org/wiki/微軟 "wikilink")，最初的开发是为了辅助
Direct3D 9 的着色器汇编语言，后成为 Direct3D 10 以来所必须的语言。

HLSL只能供微軟的[Direct3D以及](../Page/Direct3D.md "wikilink")[XNA使用](https://zh.wikipedia.org/wiki/XNA "wikilink")。HLSL是[GLSL的先辈](../Page/GLSL.md "wikilink")，同時不能與[OpenGL標準兼容](../Page/OpenGL.md "wikilink")。他跟[Nvidia的](https://zh.wikipedia.org/wiki/Nvidia "wikilink")[Cg非常相似](https://zh.wikipedia.org/wiki/Cg语言 "wikilink")。

HLSL的主要作用為將一些複雜的[圖像處理](https://zh.wikipedia.org/wiki/圖像處理 "wikilink")，快速而又有效率地在[顯示卡上完成](https://zh.wikipedia.org/wiki/顯示卡 "wikilink")，與組合式或低階Shader
Language相比，能降低在編寫複雜特殊效果時所發生編程錯誤的機會。

## 範例

  - 灰階貼圖用

<!-- end list -->

  - 灰階值 = 0.3 x 紅 + 0.59 x 綠 + 0.11 x 藍

<!-- end list -->

``` c
sampler2D Texture0;

float4 ps_main( float2 texCoord  : TEXCOORD0 ) : COLOR
{
   float4 _inColor = tex2D( Texture0, texCoord );
   float gray = 0.3*_inColor.x + 0.59*_inColor.y + 0.11*_inColor.z;
   float4 _outColor = float4(gray, gray, gray, 1.0);

   return _outColor;
}
```

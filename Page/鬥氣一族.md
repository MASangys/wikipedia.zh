《**鬥氣一族**》（）是[香港](../Page/香港.md "wikilink")[電視廣播有限公司](../Page/電視廣播有限公司.md "wikilink")製作的時裝電視劇，全劇共20集，監製[曾勵珍](../Page/曾勵珍.md "wikilink")。後來的喜劇巨星[周星馳](../Page/周星馳.md "wikilink")在1988年時尚未大紅大紫，在本劇僅是配角。

此劇於2009年7月在[無綫收費電視](https://zh.wikipedia.org/wiki/無綫收費電視 "wikilink")[經典台](https://zh.wikipedia.org/wiki/無綫經典台 "wikilink")「我們的...[吳君如](../Page/吳君如.md "wikilink")」欄目每周六至日連續播映四集。2019年於[翡翠台](../Page/翡翠台.md "wikilink")午夜劇集時段再次重播，並附加上中文字幕。

## 故事大綱

雞販鄧仁威 ( [夏雨饰](../Page/夏雨_\(香港\).md "wikilink") ) 與美國華僑鄧仁傑 ( [劉丹飾](https://zh.wikipedia.org/wiki/劉慶基 "wikilink") ) 為同父異母兄弟。仁威父親早年拋棄仁威母子，與仁傑母子移民美國，因此仁威對回港發展的仁傑充滿敵意。由於兄弟二人的父親留下一座複式單位，仁傑夫婦二人與仁威一家四口被逼同住一屋，發生了不少趣事。

仁威与仁傑兄弟二人雖势成水火，但仁威妻李小娇 ( [盧宛茵](../Page/盧宛茵.md "wikilink")饰 ) 与仁傑妻愛倫 ( [黃愷欣](../Page/黃愷欣.md "wikilink")饰 ) 卻一见如故，成为闺中好友；而愛倫的生意伙伴鍾娴 ( [陳嘉儀](../Page/陳嘉儀.md "wikilink")飾 ) 竟是仁傑的情妇；仁威幼子鄧珏骞 ( [李家聲](../Page/李家聲.md "wikilink")饰 ) 又恋上了仁傑與嫻所生的私生女鍾薏 ( [羅明珠](../Page/羅明珠.md "wikilink")饰 )，令兩家人关系错综复杂。後來仁威与仁傑因遗产问题而对簿公堂，不欢而散，直至仁威发觉仁傑被骗，两人的关系才有所突破。

同時，仁威長子鄧家發 ( [周星馳](../Page/周星馳.md "wikilink")飾 ) 自小已隨父在街市開檔賣雞，習染市井之氣，性格粗豪、好賭，他經常因小事與街市隔鄰檔鋪的獨女陳初一 ( [吳君如](../Page/吳君如.md "wikilink")飾 ) 爭執。初一自小被當作男性撫育，養成獨立豪爽的性格，不易被欺負，她與家發經常吵架，是街坊眼中的鬥氣冤家。

本是同根生，究竟仁威與仁傑這對異母兄弟，最終能否冰釋前嫌，一家團聚？而家發與初一這對鬥氣冤家又會否開花結果呢？

## 軼事

劇中雞檔位於[香港](../Page/香港.md "wikilink")[九龍](../Page/九龍.md "wikilink")[長沙灣](../Page/長沙灣.md "wikilink")[順寧道](https://zh.wikipedia.org/wiki/順寧道 "wikilink")329號興隆樓地下，雞檔為真實營業店舖，已隨社會的變遷在1990年代結束營業。2019年為唐順興燒味分店。觀眾從劇中可見1988年長沙灣順寧道一帶面貌。

**魯凱語音系**（）為[魯凱語之音系](../Page/魯凱語.md "wikilink")。而其音系之[音位大都使用適當的](../Page/音位.md "wikilink")[Unicode符號來標示](https://zh.wikipedia.org/wiki/Unicode "wikilink")。在[台灣南島語的](https://zh.wikipedia.org/wiki/台灣南島語 "wikilink")[書寫系統的訂定上](https://zh.wikipedia.org/wiki/書寫系統 "wikilink")，元輔音之音系表原則上都是先「[發音部位](https://zh.wikipedia.org/wiki/發音部位 "wikilink")」（橫列）、後「[發音方法](https://zh.wikipedia.org/wiki/發音方法 "wikilink")」（縱列）、再考量「[清濁音](https://zh.wikipedia.org/wiki/清濁音 "wikilink")」，來訂定其音系之音位架構。\[1\]

## 語言分區

萬山魯凱語是分歧性最高的魯凱方言。[李壬癸院士](../Page/李壬癸.md "wikilink")(2001)將魯凱語分為幾個分支如下：\[2\]

  - **魯凱語**
      - *萬山* (Mantauran) 250–300人
      - (主要分支)
          - 馬加–多納
              - *馬加* (Maga)
              - *多納* (Tona)
          - 霧台–大南(主要魯凱語)
              - *霧台* (Budai)
              - *大南* (Tanan；亦稱 *Taromak*)

李壬癸院士認為魯凱語是第1個從[原始南島語分裂出來的南島語言](../Page/原始南島語.md "wikilink")。下列為李壬癸院士所評估出來的台灣南島語的分裂時期。Li
(2008:215)\[3\]

1.  原始南島語：公元前 4,500年
2.  魯凱語：公元前 3,000年
3.  鄒語：公元前 2,500年 (大約在公元前
    1,000年分裂為[北鄒語群及](../Page/鄒語群.md "wikilink")[南鄒語群](../Page/鄒語群.md "wikilink"))
4.  其他大多數台灣南島語分裂時期：公元前 2,000年 至 0年
5.  西部平原台灣南島語：公元 1,000年

## 元輔音

大多數的魯凱語方言有4個元音，以及[翹舌音和](https://zh.wikipedia.org/wiki/翹舌音 "wikilink")[齒間音](https://zh.wikipedia.org/wiki/齒間音 "wikilink")(Interdental
consonant)等輔音。同樣霧台魯凱語\[4\]有4個元音，。不是一個[schwa](../Page/中央元音.md "wikilink")(schwa)音，而是一個全元音。字詞在音位上是以一輔音加上一[迴聲元音](https://zh.wikipedia.org/wiki/迴聲元音 "wikilink")(echo
vowel)來結尾、回聲元音屬於其中之一，\[5\]
而不是如[形態音位的](https://zh.wikipedia.org/wiki/形態音位學 "wikilink")(morphophonology)的元音經常在派生上遺失掉。當詞幹最後的元音是時須用。\[6\]\[7\]

霧台魯凱語輔音為：

| --- | --- | ---- | --- | --- | --- | --- |
| 鼻音  | m   |      | n   |     |     | ŋ   |
| 爆破音 | p b |      | t d | ɖ   |     | k ɡ |
| 塞擦音 |     |      | ts  |     |     |     |
| 擦音  | v   | θ ð  | s   |     |     |     |
| 顫音  |     |      | r   |     |     |     |
| 近音  | w   |      | l   | ɭ   | j   |     |

在[排湾语及](https://zh.wikipedia.org/wiki/排湾语 "wikilink")[中文的影響下](https://zh.wikipedia.org/wiki/中文 "wikilink")，年輕一輩有時發
 音為  音。在大南魯凱語(Tanan)裡，年輕一輩同樣地融合  音及  音為  音。

在萬山魯凱語(Mantauran)裡，清塞音含有擦音於：\*b 對 ，\*d 及 \*ɖ 對 ，和 \*g 對
等之情況中。此些語音現象的研究來自齊莎莉的發表論文(Zeitoun
2007)。總的有輔音 15 個、及元音 4 個如下：

  - 15個輔音，寫為 *p, t, k, , v, s, h, dh, c, m, n, ng, l, r, lr*
  - 4個元音，寫為 *a, i, e, o*

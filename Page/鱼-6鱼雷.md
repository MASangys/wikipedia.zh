**H/YZQ-006型热动力声线联导反潜鱼雷**\[1\]，旧型号名**鱼-6**重型鱼雷，是中国研制的一种533毫米重型线导鱼雷。鱼-6是美国[Mk 48鱼雷的中国对应型号](../Page/Mk_48型魚雷.md "wikilink")，\[2\]并且在设计上相当的参考了Mk 48。但鱼-6并非Mk 48的简单仿制型。该型号的总设计师是董春鹏。\[3\]

## 概要及研发历程

鱼-6鱼雷的研制起始于1980年代初。目标是通过[逆向工程](../Page/逆向工程.md "wikilink")一枚从[南海得到的Mk](https://zh.wikipedia.org/wiki/南中国海 "wikilink")48鱼雷来研制一种新型反潜/反舰鱼雷。项目在不久后因为资金困难被搁置，但关于鱼雷动力系统的研制成果被保留了下来。\<ref name = ""gmm"\>[Yu-6 Torpedo](http://www.globalmil.com/military/navy/china/systems/torpedoes/2010/0325/184.html) - [环球防务](http://www.globalmil.com/)</ref>项目在1995年被再次启动。\[4\]主承包商为[中国船舶重工集团公司](https://zh.wikipedia.org/wiki/中国船舶重工集团公司 "wikilink")705研究所（即西安精密机械研究所）。\[5\][上海交通大学亦参与了部分研制](https://zh.wikipedia.org/wiki/上海交通大学 "wikilink")。\<ref name = ""gmm"/\>该型号研制的难度大，使用的新技术占到2/3。\[6\]因为缺少供仿制的样本，实际上研制单位更多是参考美方的公开资料进行研制，且鱼雷本体参考了俄罗斯的设计，\[7\]因此鱼-6与Mk 48有很大的不同。

鱼-6是一种533毫米，反潜/反舰重型鱼雷。具有线导、声自导和尾流跟随三种制导模式。\<ref name = ""gmm"/\>鱼-6是中国首个采取开放式软件架构和模块化设计的鱼雷。其导引头采取了[80486级别的主处理器](https://zh.wikipedia.org/wiki/80486 "wikilink")。\<ref name = ""gmm"/\>

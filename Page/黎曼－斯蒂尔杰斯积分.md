**黎曼－斯蒂尔杰斯积分**（）是數學中的一種「[積分](https://zh.wikipedia.org/wiki/積分 "wikilink")」概念，是對[黎曼积分](../Page/黎曼积分.md "wikilink")的推廣。

黎曼－斯蒂尔杰斯积分有數種定義方式，但不是每種定義方式都是彼此等價的。

## 定義

和黎曼積分一樣，黎曼－斯蒂尔杰斯积分的定義依賴對區間分割的定義。

### 区间的分割

一个[闭区间](https://zh.wikipedia.org/wiki/闭区间 "wikilink")\([a,b]\)的一个**分割**P是指在此区间中取一个有限的点列\(a = x_0 < x_1 < x_2 < \ldots < x_n = b\)。每个闭区间\([x_i, x_{i+1}]\)叫做一个**子区间**。这些子區間长度的最大值 \(\lambda\) 為：\(\lambda := \max \{\, x_{i+1}-x_i \mid 0 \le i \le n - 1 \,\}\)。

定义**取样分割**。一个[闭区间](https://zh.wikipedia.org/wiki/闭区间 "wikilink") \([a,b]\) 的一个取样分割是指分割 \(P=\{ a = x_0 < x_1 < \cdots < x_n = b\}\) 再加上一組有限點，\(\{a\le t_0\le t_1\le,\cdots,\le t_{n-1}\le b\}\)，其中 \(t_i\in[x_i, x_{i+1}]\) 對所有 \(0\le i\le n-1\)。

**精细化分割**：设\(x_0,\ldots,x_n\)以及\(t_0,\ldots,t_{n-1}\)构成了[闭区间](https://zh.wikipedia.org/wiki/闭区间 "wikilink")\([a,b]\)的一个取样分割，\(y_0,\ldots,y_m\)和\(s_0,\ldots,s_{m-1}\)是另一个分割。如果对于任意\(0 \le i \le n\)，都存在\(r(i)\)使得\(x_i = y_{r(i)}\)，并存在\(r(i) \le j \le r(i+1)\)使得\(t_i = s_j\)，那么就把分割：\(y_0,\ldots,y_m\)、\(s_0,\ldots,s_{m-1}\)称作分割\(x_0,\ldots,x_n\)、\(t_0,\ldots,t_{n-1}\)的一个**精细化分割**。简单来说，就是说分割\(y_0,\ldots,y_m\)、\(s_0,\ldots,s_{m-1}\)是在分割\(x_0,\ldots,x_n\)、\(t_0,\ldots,t_{n-1}\)的基础上添加一些分点和标记。(即是說「設\(P = \{ a = x_0, x_1, x_2, \ldots, x_{n-1}, x_n = b \}\)是閉區間\([a,b]\)的一個分割，若分割\(P'\)是分割\(P\)的一個精細化分割，則\(P \subseteq P'\)，也就是說，分割\(P\)是分割\(P'\)的子集」)

于是我们可以在此区间的所有取样分割中定义一个[偏序关系](../Page/偏序关系.md "wikilink")，称作“精细”。如果一个分割是另外一个分割的精细化分割，就说前者比后者更“精细”。

### 黎曼－斯蒂尔杰斯和

对一个在闭区间\([a,b]\)有定义的实值函数 \(f\)，其對於函數 \(g\) 关于分割

\[P=\{a=x_0<x_1<\cdots <x_n=b\}\]

的**黎曼－斯蒂尔杰斯和**，規定為下式：

\[S(P,f,g) = \sum_{i=0}^{n-1} f(c_{i}) \cdot\left(g(x_{i+1})-g(x_{i})\right)\]

和式中的 \(c_{i}\in [x_{i+1}, x_i],\,0\le i\le n-1\)。

### 黎曼－斯蒂尔杰斯積分

當注意的是。這兩個定義在黎曼－斯蒂尔杰斯积分的情況下，並不完全等價，以第一種定義可推出其存在的積分，必能以第二種定義推出其存在，但以第二種定義方式可推出其存在的積分不一定能以第一種定義的方式來計算。

#### 第一種定義

\(A\in\R\) 是函数 \(f\) 在闭区间 \([a,b]\) 上對函數 \(g\) 的黎曼－斯蒂尔杰斯積分的值，若且唯若对任意的 \(\epsilon > 0\)，存在 \(\delta > 0\)，使得对任意的分割 \(\scriptstyle P=\{a=x_0< x_1<\cdots< x_n=b\}\)，只要這分割的子区间最大長度滿足 \(\lambda \le \delta\) 且對任意的 \(\scriptstyle c_i\in [x_{i+1}, x_i],\;  0\le i \le n-1\)，有：

\[\left| S(P,f,g) - A \right| < \epsilon.\,\]

#### 第二種定義

\(A\in\R\) 是函数 \(f\) 在闭区间 \([a,b]\) 上對函數 \(g\) 的黎曼－斯蒂尔杰斯积分的值，若且唯若对于任意的 \(\epsilon > 0\)，存在分割 \(P_\epsilon\)，使得對任何比 \(P_\epsilon\) 還要「精细」的分割 \(\scriptstyle P=\{a=x_0<x_1<\cdots <x_n=b\}\) 跟任意選取的 \(\scriptstyle c_i\in [x_{i+1}, x_i],\; 0\le i \le n-1\)，都有：

\[\left| S(P,f,g) - A \right| < \epsilon.\,\]

若一個函數\(f\)在闭区间\([a,b]\)上對函數\(g\)的黎曼－斯蒂尔杰斯积分存在，且值為\(A\)，則可寫作\(A = \int_{a}^{b} f(x)\,dg(x).\)

## 與黎曼積分間的關聯

若\(\,g(x) = x\,\)時，\(\,f\,\)在闭区间\(\,[a,b]\,\)上對函數\(\,g\,\)的黎曼－斯蒂尔杰斯积分

\[\int_{a}^{b} f(x) \, dg(x)\] 即為\(f\)在闭区间\([a,b]\)上的[黎曼積分](https://zh.wikipedia.org/wiki/黎曼積分 "wikilink")

\[\int_{a}^{b} f(x) \, dx\]， 故從黎曼－斯蒂尔杰斯积分可引出黎曼積分。

若\(\,g(x)\,\)可微且其對\(\,x\,\)微分後的函數\(\,g'(x)\,\)在闭区间\(\,[a,b]\,\)連續，則\(\,f\,\)在闭区间\(\,[a,b]\,\)上對函數\(\,g\,\)的黎曼－斯蒂尔杰斯积分

\[\int_{a}^{b} f(x) \, dg(x)\] 與黎曼積分

\[\int_{a}^{b} f(x) g'(x) \, dx\] 相等。

## Layout template

    {{Portal:Gymnastics/Selected article/Layout
      |image=
      |size=
      |caption=
      |text=
      |link=
    }}

### 用法

  -
    image= the image to be used in the form imagename.jpg ("image:"
    prefix is not required)
    size= images are sized to the default of 150px if this is left blank
    caption= text that will appear if a reader moves their mouse pointer
    over the image
    text= the text to appear below the image, remember to wikilink
    relevant articles and **bold** the main subject
    link= the main subject article, this is the page a reader will go to
    when they click *read more*, do not wikilink as this is done
    automatically

If there is no available image then leave image= size= and caption=
blank </noinclude>
